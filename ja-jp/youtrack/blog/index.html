A Reminder: Discontinuing Support for Internet Explorer

Posted on June 18, 2019 by Natasha Katson

Dear users,

Last February, we announced that we will discontinue support for Internet Explorer 11 starting from YouTrack 2019.2. The release is planned for July 2019, so we kindly remind you to switch to one of the other supported web browsers before the release.

What does this mean?

In earlier versions of YouTrack, we have only supported versions of Internet Explorer earlier than Internet Explorer 11. Starting with YouTrack 2019.2, we will no longer guarantee the stability of YouTrack functionality in any version of IE and will no longer fix bugs that only affect this web browser.

Please reach out to us if you have any questions or difficulties – we are here to help!
