AppCodeを選択する理由

あなたの素晴らしいアイデアを迅速（Swift）に実行

インテリジェントな
開発環境

AppCodeがコード構造を深く理解しているため、定型的な作業を担当させることで、余分なキー入力が不要になります。

効率的なプロジェクトナビゲーション

プロジェクト内の任意のファイル、クラス、シンボルにすぐに移動でき、階層ビューと構造ビューを使用してプロジェクト構造内をナビゲートできます。

スマート補完

AppCodeには二種類のコード補完があります。入力時の基本的な補完と、サジェストの正確なフィルタリング用のより高度なSmartType補完です。

信頼度の高いリファクタリング

安全で正確で信頼性の高いリファクタリングによって、いつでもコードの修正と改善を行えます。

徹底的なコード解析

AppCodeは常にコードの品質を監視しています。 これはエラーや臭いを警告し、自動的に解決するためのクイックフィックスを提案いたします。

AppCodeはObjective-C、Swift、C/C++用の豊富なコードインスペクションと、その他サポート対象言語用の多数のコードインスペクションを提供します。 コードインスペクションはすべて、オンザフライで動作します。

生産性の高い
ユニットテスト

AppCodeは最初から、テストフレームワークとして、XCTest、Quick、Kiwi、Catch、Boost.Test、Google Testをサポートしております。 ユニットテストのメソッドを取得するコード生成アクションにより、生産性が向上します。 ⌘Nを使って、コンテキストに応じてテストメソッドを取得します。

一つのIDEで多言語に対応

AppCodeは、JavaScript、XML、HTML、CSS及びXPathと同様に、Objective-C、Swift、C及びC++(モダンC++標準、libc++、Boostを含む)をネイティブにサポートします。

  • Objective-C
  • Swift
  • C++

Renameリファクタリング

変数、定数、関数、型名、クラス名を変更すると、AppCodeがコードベース全体のすべての使用箇所を更新するのでご安心いただけます。

Override/Implement

メソッドを実装もしくはオーバーライドする際の定型作業を削減します。AppCodeがデフォルトコードを含むスタブを作成いたします。

  • Objective-C
  • Swift
  • C++
  • Objective-C
  • Swift
  • C++

使用箇所からの作成

関数、定数、変数、またはプロパティをコード内で使用するだけで、その宣言が作成されることを願ったことがございますか？ 単に⌥⏎を押すだけで、適切な場所に自動的に作成されます！

完全な
エクスペリエンス

AppCodeはXcodeと100%互換性があり、相互運用が可能で、追加の設定は必要ありません。さらに、CocoaPods、Revealアプリケーション、最も一般的なバージョン管理システム、および多くの課題トラッカーとシームレスに統合いたします。
