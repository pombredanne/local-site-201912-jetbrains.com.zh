Интеграция с облачными платформами

Amazon EC2

TeamCity позволяет использовать преимущества облачных вычислений, размещая билд-агенты в Amazon EC2.

При планировании сборок TeamCity учитывает виртуальные машины, автоматически приостанавливая и возобновляя их в зависимости от статуса очереди сборок и нагрузки.

2019.1 Используйте запросы AWS Spot Fleet в качестве более продвинутого и экономичного способа запуска сборок в AWS.

Плагин Microsoft Azure

Используете облачную платформу Microsoft Azure? Интегрируйте ее с TeamCity и масштабируйте свою билд-ферму по мере необходимости. Вам больше не придется рассчитывать нагрузку на билд-агенты при выпуске релизов. Как только вам потребуется дополнительная мощность, TeamCity автоматически запустит агенты в Azure, а потом остановит их, когда необходимость в них отпадет. Плагин «Хранилище артефактов в Azure» позволяет использовать хранилище BLOB-объектов Azure для хранения внешних артефактов сборки.

Плагин Google Cloud

TeamCity обеспечивает тесную интеграцию с сервисами Google Cloud. Плагин «Агенты Google Cloud» позволяет использовать Google Compute Engine для запуска облачных экземпляров при необходимости масштабирования пула облачных билд-агентов. Он также поддерживает использование прерываемых виртуальных машин, обеспечивающих повышенную рентабельность. Плагин «Хранилище артефактов Google» позволяет использовать Google Cloud Storage для хранения внешних артефактов сборки.

Подробнее

VMware vSphere

Интеграция TeamCity с VMware vSphere позволяет использовать удаленные агенты TeamCity, установленные на виртуальных машинах VMware, для более эффективного выполнения сборок: TeamCity анализирует очередь сборок и запускает виртуальные машины с агентами TeamCity, соответствующими сборкам в очереди.

Подробнее

Плагин Kubernetes

Kubernetes — один из самых популярных способов запуска Docker-контейнеров. Некоторые команды и организации имеют свой кластер Kubernetes, настроенный для использования в производственной среде. Официальный плагин «Поддержка Kubernetes для TeamCity» от JetBrains позволит вам запускать билд-агенты в соответствующей инфраструктуре.
