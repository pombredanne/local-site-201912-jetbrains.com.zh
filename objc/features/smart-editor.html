智能编辑器

代码完成

AppCode提供两种代码完成功能：基本的按需输入完成功能和更高级的SmartType完成功能（ ^⇧Space ），用于更精确地过滤建议。

两种完成类型都支持CamelHumps和Middle Matching ;也就是说，您可以通过仅输入大写字母或完全输入其名称的任何部分来完成任何项目。

导入优化

使用AppCode可以很容易地从未使用的导入中清除整个项目-只需调用Code | Optimize imports从菜单Code | Optimize imports或按⌥⌘O 。所有未使用或重复的导入将在编辑器中突出显示，从而使代码更清晰。

自动导入

当您开始使用尚未导入的类时，AppCode将建议自动添加相应的导入，甚至在您完成代码后也添加导入。

代码格式化

代码格式化在您键入时起作用，也可以通过按⌥⌘L隐式调用选定的代码片段，文件或整个目录。

您可以在Preferences | Editor | Code style设置常规和特定于语言的格式Preferences | Editor | Code style 。对于影响代码的所有代码样式设置，AppCode都会显示实时预览，并突出显示受最新更改影响的行。

代码重新排序

AppCode使您可以使用一些简单的快捷方式快速更改代码元素的顺序。例如，你可以移动的代码块了或⌘⇧↑/↓下来，或动线，⌥⇧↑/↓。

进阶格式选项

为了在某些选定文件中保持特定格式，AppCode可以自动检测代码样式并使用现有文件缩进进行编辑。

AppCode还允许您更改仅适用于所选代码段的设置。选择代码并使用快速修复调整代码样式设置 （ ⌥Enter ）

多个插入号和选择

使用AppCode，您可以通过同时编辑文件中的多个位置来提高工作效率。按⌥⇧并用鼠标选择插入符号的位置，或按^ G将当前单词的下一个出现添加到选择中（要从选择中删除，请按^⇧G ）。然后，只需在所有这些位置同时编写代码即可。自动完成和实时模板也将照常工作。要恢复单插入符模式，请使用Esc 。

展开/收缩代码选择

AppCode具有简单的快捷方式，可以轻松地扩展选择：从符号到表达式，从表达式到行，从行到逻辑代码块，依此类推。只需按⌥↑多次。或者，要将选择范围缩小到较小的逻辑部分，请按⌥↓ 。

选择一个块后，可以对其应用块注释（ ⌥⌘/ ）或行注释（ ⌘/ ）。而且，如果您想注释一行，甚至不需要选择该行，只需在其上插入插入符号，然后按⌘/即可 。

快速文档

使用AppCode，您可以导航到方法的实现，而无需导航到该方法或离开当前上下文。只需将插入号放在方法调用上，然后按⌥Space 。

要查看相关文档，只需按F1 。该文档显示在一个弹出窗口中，其中包含指向其他资源的可单击链接。另外，快速文档甚至可以向您显示宏替换或Swift推断的类型。

参数信息

如果您不确定要传递给方法或函数的确切参数，可以使用Parameter Info（ ⌘P ）来提供帮助。它显示参数类型和可用的函数签名。
