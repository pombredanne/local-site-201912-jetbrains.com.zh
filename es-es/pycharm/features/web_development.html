Django

Django es el galardonado líder de los marcos de trabajo web para Python, y PyCharm lo soporta desde hace tiempo. Ejecutar, depurar, navegar y trabajar de forma productiva... PyCharm es su puerta de acceso a Django.

Ejecute y depure con facilidad

El tipo de configuración de ejecutar/depurar de Django permite configurar e iniciar fácilmente el servidor Django en una productiva ventana de herramienta, tanto local como remotamente.

Plantillas Django

Resaltado de sintaxis y errores, finalización de código, navegación entre vistas y plantillas y depuración visual de plantillas insertando puntos de interrupción directamente en el editor.

Navegación rápida

Salte con facilidad desde un símbolo en un plantilla hasta el código Python para modelos, métodos de visualización, formularios parámetros de ruta, ajustes y más.

Edición productiva

Ahorre tiempo con acciones de editor tales como formateo la sangriá automática de código, fragmentos de código/plantillas personalizables, emparejado automático de símbolos como llaves, anotación de comentarios al código, "Rodear con etiqueta" y autocompletado con inferencia de tipo.

Asistencia a la codificación

Asistencia especial a la codificación dentro de archivos estándar: urls.py y settings.py: archivos, carpetas, ver métodos, resaltado e inyección de regexp. Use intenciones para que PyCharm cree vistas y plantillas a partir del uso.

Soporte de ORM y formularios

Trabaje más rápido con modelos y formularios usando el autocompletado en vistas y plantillas, inferencia de tipo, navegación, refactorización y uso final.

Más para Django: autocomplete nombres de recursos estáticos en plantillas (y reciba advertencias), ventana de tarea interactiva especial para manage.py con autocompletado dirigido por el proyecto, plantillas Jinja2 y Mako en Django, intenciones de código tales como convertir una vista de función a/desde una vista basada en clase, inspecciones de plantillas para advertir errores, ayudantes i18n, plantillas personalizables para Django, diagramas de modelo y más.

Finalización de código

Navegación

SQLAlchemy

Más para Flask: plantillas personalizables para generar rutas Flask rápidamente y tupo de proyecto dedicado que genera un proyecto Flask simple preconfigurado.

Remoto

Los flujos de trabajo modernos para desarrollo usan tecnologías remotas o virtuales, ya sea para centralizar el desarrollo o para proporcionar aislamiento. PyCharm ofrece un gran compatibilidad para el desarrollo full stack usando Vagrant, SSH y Docker.

Intérpretes remotos

Ejecute y depure código local en entornos remotos/de contenedor. Compatible con Docker, Vagrant, SSH y Windows Subsystem para Linux. Interfaz de usuario PyCharm para explorar los contenedores Docker locales.

Desarrollo remoto

Configure su proyecto con un escenario de despliegue basado en SFTP o FTP. Carga automática/manual, gestión de credenciales, mapeo de carpetas locales a remotas y más.

Terminales y herramientas SSH

Inicie sesión en el sistema remoto usando un terminal basado en IDE similar al terminal local, de acuerdo con la configuración remota del proyecto. Defina tareas remotas que se ejecutan sobra SSH desde dentro de PyCharm.

Más

Python full stack es un tema importante, y PyCharm tiene mucho más para ofrecer:

  • Un completo cliente HTTP basado en archivos para pruebas automatizadas.
  • Compatibilidad con el marco de trabajo Python web2py: tipo de configuración de ejecución/depuración dedicado, soporte para lenguaje de plantillas, navegación entre vistas y plantillas y una plantilla de proyecto web2py.
  • Soporte para Google App Engine (GAE): configure los ajustes de GAE desde un panel, cargue aplicaciones desde un menú de herramientas, vea archivos de registro, consola dedicada para ejecutar comandos appcf.py, y una plantilla de proyecto GAE.
  • Inyección de lenguaje para incrustar la compatibilidad SQL en cadenas Python, con autocompletar no solo en los comandos SQL sino también en el esquema de su fuentes de datos definida.

Bases de datos

PyCharm analiza internamente todos los objetos de sus bases de datos y los muestra agrupados por esquemas en carpetas. También ofrece una interfaz de usuario para agregar y editar tablas, columnas, índices, restricciones, etc.

Consultas

Realice consultas en una consola dedicada con capacidad para explorar la respuesta, historial local y un práctico visor diff para comparar los distintos resultados.

Navegación

La ágil navegación lo lleva hasta cualquier objeto, sin importar si se acaba de crear en su código o si ya se ha leído desde una base de datos. La navegación al símbolo le permite encontrar objetos por su nombre.

Editor de tablas

El potente editor de tablas le permite añadir, eliminar, editar y clonara filas de datos. Explore los datos por claves foráneas y use la búsqueda de texto para encontrar cualquier cosa en los datos que se muestran en el editor de tablas.

Finalización de código

PyCharm ofrece una finalización de código sensible al contexto y consciente del esquema, que le ayuda a escribir código más rápido. La finalización tiene en cuenta la estructura de tablas, claves foráneas y hasta los objetos de base de datos creados en el código que usted edita.

Importar/Exportar

Mueva datos hacia dentro y hacia afuera de su base de datos de múltiples maneras, usando formatos conocidos como CSV/JSON/XML, otras fuentes de datos como SQLite, exportaciones de consultas recientes o hasta facilidades de extracción de datos basadas en la programación.
