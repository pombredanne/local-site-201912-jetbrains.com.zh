仪表板

在此页：

  • 多个仪表板

的 仪表板是一种方便的方法，可用于概述任务的当前状态并监视团队的进度。

登录到集线器时，默认情况下会显示仪表板。还有一个默认仪表板，可用于向来宾用户显示信息。有关更多信息，请参见 默认仪表板 。

要从集线器中的另一个页面或另一个连接的服务中打开仪表板，请单击在主导航区右上角的头像旁边，单击“ 服务菜单”图标，然后选择 仪表板 。

仪表板在卡上显示小部件。仪表板上可用的窗口小部件类型与连接到集线器的服务有关。

仪表板支持以下类型的小部件：

服务小部件描述
团队城市 建立文物显示来自特定构建配置的工件列表。
调查显示在TeamCity中分配给您的活动调查的列表。
项目状态显示特定项目中构建的状态。
上游 提交显示特定项目中的VCS提交列表。
评论显示与特定搜索查询匹配的代码审阅列表。
YouTrack 问题显示与搜索查询，保存的搜索，项目或标签匹配的问题列表。
报告书显示预定义报告的计算。
毂 快速笔记显示您想在仪表板上显示的任何消息。

您可以根据需要向仪表板添加任意数量的每种类型的小部件。

要将小部件添加到仪表板：

  1. 点击 添加小部件按钮。
  2. 选择要添加到仪表板的窗口小部件的类型。
  3. 选择要用作小部件数据源的服务。
  4. 配置窗口小部件的设置。

    —小部件已添加到您的仪表板。

多个仪表板

您可以创建和查看多个仪表板，每个仪表板都有自己的小部件集。避免您为不同项目设置和监视仪表板。

可用仪表板的数量显示在活动仪表板的名称下。还显示列表中上一个和下一个仪表板的名称。

按Shift + 左或Shift + 右可在仪表板之间切换。

按Shift + 向下 键以扩展仪表板下拉列表。

要添加另一个仪表板：

  1. 点击当前信息中心名称旁边的向下箭头，然后选择 创建新的仪表板选项。

    或者，按键盘上的Alt + Shift + N 。

  2. 在页面中输入新仪表盘的名称 命名仪表板对话框。
  3. 点击 创建仪表板按钮。

    —新的空白仪表板已添加到集线器。

共享仪表板

除了您自己创建的仪表板之外，您还可以查看与您共享的仪表板。

当您的团队成员直接与您或您所属的组共享一个仪表板时，您会在当前仪表板名称旁边看到一个链接。单击此链接时，将显示共享仪表板的名称以及共享仪表板的用户的名称。

  • 要将此仪表板添加到列表中，请选择共享的仪表板，然后单击 添加到我的仪表板按钮 。
  • 要从信息中心删除通知链接，请点击 驳回下拉列表链接。

    即使尚未将共享仪表板添加到列表中，您仍然可以查看它。

整理仪表板

要组织仪表板，您要做的就是将卡拖到仪表板上，并按照您喜欢的顺序放置。

您可以在仪表板上调整卡的大小。拖动卡的边缘以设置高度和宽度。卡的大小会自动对齐到网格的最近交点。

要更新窗口小部件的内容，请单击标题栏中的刷新图标。

仪表板的其他操作

除了可以将小部件添加到仪表板的选项之外，还可以执行以下操作：

选项描述
分享您可以配置仪表板并与其他用户或组共享。有关更多信息，请参见 共享仪表板 。
改名您可以随时更改仪表板的名称。要访问此选项，请单击 编辑>重命名 。
删除如果您不想再使用列表中的仪表板，则可以将其删除。要访问此选项，请单击 编辑>重命名 。

小部件的其他操作

您可以使用多种操作来自定义仪表板并与团队共享其内容。

要打开可用于窗口小部件的操作列表，请单击标题栏中的向下箭头。

如果窗口小部件包含您希望团队中的每个成员一起使用的信息，则可以共享该窗口小部件。

共享小部件：

  1. 单击要共享的小部件标题栏中的向下箭头。
  2. 选择 共享小部件选项。
  3. 复制对话框中的链接，并将其发送给您的队友。

    如果与您共享窗口小部件的人无权查看窗口小部件中使用的数据，则窗口小部件为空。

您可以在仪表板上创建现有小部件的副本。当您想添加具有微小更改的类似小部件时，此选项非常有用。

克隆小部件：

  1. 单击要克隆的小部件标题栏中的向下箭头。
  2. 选择 克隆小部件选项。

    —小部件的副本已添加到您的仪表板。

您可以将小部件从当前仪表板移动到另一个仪表板。

要将小部件移动到另一个仪表板：

  1. 单击要移动到另一个仪表板的小部件标题栏中的向下箭头。
  2. 选择 移至另一个仪表板选项。
  3. 在里面 将小部件移动到另一个仪表板对话框，选择目标仪表板的名称。
  4. 点击 移动按钮。

    —小部件已移动到选定的仪表板。

如果要修改仪表板上小部件的设置，则可以随时对其进行编辑。

要编辑现有的小部件：

  1. 单击要编辑的小部件标题栏中的向下箭头。
  2. 选择 编辑选项。
  3. 修改小部件的设置或内容。
    • 对于报告窗口小部件，选择要在窗口小部件中显示的报告。
    • 对于问题窗口小部件，请输入新的搜索查询或选择其他保存的搜索，项目或标签。点击 应用按钮以更新小部件。
    • 对于注释小部件，请在输入字段中编辑文本。点击 应用按钮以更新小部件。

    —小部件将使用新设置进行更新。

如果您不再希望在仪表板上看到窗口小部件，则可以将其删除。

要从仪表板上删除窗口小部件：

  1. 单击要从仪表板删除的小部件标题栏中的向下箭头。
  2. 点击 删除选项。
  3. 点击确认对话框中的“ 删除”按钮。

    —小部件已从仪表板上删除。
