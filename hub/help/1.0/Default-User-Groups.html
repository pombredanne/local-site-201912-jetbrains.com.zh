默认用户组

JetBrains Hub中有两个默认用户组：“所有用户”和“新用户”。

“所有用户”组

所有注册用户将自动包含在“所有用户”组中。该组具有在“全局”项目中授予的默认观察员角色 ，该角色允许查看所有公共项目中的任何公共问题和评论，因此，默认情况下，集线器中的所有注册用户都具有观察员角色。管理员只能编辑分配给该组的角色，而不能更改组名或禁用“自动加入”选项。同样，“所有用户”组也无法删除。

“新用户”组

创建用户帐户后，默认情况下会自动将其添加到“新用户”组中。该组具有默认的“报告者”角色 ，该角色允许用户（除了查看公开的问题和评论）还可以报告新问题，向问题添加评论，查看和编辑自己的用户个人资料。管理员可以根据需要编辑组的参数或将其删除。
