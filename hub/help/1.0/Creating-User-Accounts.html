创建用户帐户

在Hub中，添加新用户帐户的主要工作流程很可能是从连接的服务导入用户帐户。但是，Hub还为您提供了手动创建用户帐户的方法。

邀请新用户注册

邀请一个或几个人在Hub中注册用户帐户 ：

  1. 打开用户页面。
  2. 点击“ +新用户... ”按钮。
  3. 单击邀请用户按钮。
  4. 提供以空格分隔的电子邮件地址列表，应邀请所有者在集线器中注册帐户：
  5. 单击邀请以使用指定的电子邮件向用户发送邀请。

为什么在发送邀请进行注册时实际上在Hub中创建帐户

完成后，您将自动导航到邀请的用户个人资料。如果您一次邀请了多个用户，则将打开第一个用户的个人资料页面。打开配置文件意味着该帐户实际上存在于Hub中。

对于受邀请的用户，该帐户已经创建的事实并不会更改设置帐户的过程。另一方面，您（管理员）可以配置帐户的参数（如果已经存在）。例如，您可以在发送邀请后立即配置用户的访问权限。

创建新用户帐户

手动创建新的用户帐户：

  1. 打开用户页面。
  2. 点击“ +新用户... ”按钮。
  3. 在“新用户”对话框中，单击“ 创建帐户”按钮。
  4. 填写必填字段：用户的显示名称和电子邮件地址：
  5. 单击创建以创建具有指定属性的用户帐户。

创建新帐户后，您将自动导航到用户的个人资料。
