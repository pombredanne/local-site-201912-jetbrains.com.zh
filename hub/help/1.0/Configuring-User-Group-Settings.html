Configuring User Group Settings

You can configure parameters of a user group via its profile. To open group's profile you need to click its name in the list of groups (Administration > Groups page).

Group profile allows you to view and configure the following parameters:

  • The list of members of the group.
  • Group's general settings (such as name, description, etc.)
  • Group access permissions.

For details about configuring group's settings:
