Python开发人员调查2017年结果

2017年底，Python软件基金会与JetBrains共同进行了一次正式的Python开发人员调查。我们着手确定最新趋势，并深入了解当今Python开发世界的外观。来自将近150个国家/地区的9,500多名开发人员参加了此次活动，以帮助我们绘制Python社区的准确概况。

现在，让我们回顾一下前面提到的开发统计信息的类型。实际上，有26％的人将 Web开发作为主要活动，而18％的人主要从事数据分析，而9％的人主要从事机器学习，实际上这一比例是1：1 ！只有九分之一的受访者正确地猜到了这一点。

有趣的事实

大多数受访者低估了参与数据科学的开发人员总数。人们普遍认为Web开发是Python的主要应用程序。尽管几年前确实如此，但如今Python数据科学家的数量正在迅速增长，并且已经与Web开发人员的数量相当。

Python 3和Python 2

我们问：“您使用哪个版本的Python最多？”。Python 3是75％的强大领导者，只有25％的人将Python 2用作主要的解释器。

有趣的事实

Python 3的主要用途正在迅速增长。根据2016年的最新研究，使用Python 2的用户占60％，而使用Python 3的用户则占40％。Python 2的使用正在下降，因为它尚未得到积极开发，未获得新功能，并且其维护将于2020年停止 。

可能的原因

一个可能的解释是，某些Web开发人员在过渡到Python 3时仍然需要维护许多遗留代码。另一方面，许多数据分析师和机器学习专家最近才加入Python生态系统，并直接使用最新的Python 3。

另一个有趣的发现是，主要从事Web开发的人员中只有70％使用的是Python 3，而对于数据分析人员来说，这一比例上升到77％，对于机器学习专家来说甚至更高，达到了83％。
