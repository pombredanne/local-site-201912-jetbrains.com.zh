HTML / CSS / JavaScript编辑器

注意： PhpStorm包含WebStorm的所有与HTML，CSS和JavaScript相关的功能。

基于DOM，特定于浏览器的完成

用于标签，关键字，标签，变量，参数和函数的JavaScript，HTML和CSS代码完成是基于DOM的，并且支持诸如Firefox和Chrome之类的流行浏览器的细节。

* .js文件中的标准功能和用户定义功能，HTML事件处理程序以及所有其他适当的上下文都可以使用完成功能。

支持Sass，SCSS，Less，Stylus，Compass，CoffeeScript，TypeScript，Dart和其他尖端的Web开发技术。

JavaScript调试器

使用JetBrains PhpStorm，您可以在调试JavaScript的同时充分利用各种智能功能：
  • HTML和JavaScript的断点
  • 可自定义的断点属性：挂起模式，条件，通过次数等
  • JavaScript调试器UI中的框架，变量和监视视图
  • JavaScript表达式的运行时评估
  • JavaScript中的单元测试

现场编辑

实时编辑预览可以节省大量时间和按键。在编辑器和浏览器中打开一个页面，以便您可以同时看到两者，并在运行JavaScript调试会话时激活“实时编辑”功能。开始编辑代码，并在浏览器中立即查看更改，即使您正在浏览代码完成弹出窗口中的列表选项时也是如此。

PhpStorm会自动保存您的更改，浏览器会智能地动态更新页面，显示您的修改。

文件观察者

利用File Watchers将现代Web开发语言自动编译/转换为JavaScript（来自CoffeeScript，TypeScript，Dart等）或CSS（来自Sass，SCSS，Less，Stylus，Compass等）。

高速编码

该编辑器使您可以利用Emmet的所有功能（以前称为Zen Coding）。使用Live Templates定义您自己的缩写（摘要），或使用针对PHP，HTML，CSS，JS和其他语言的预定义。

验证和快速修复

PhpStorm检测并建议自动修复以下问题：缺少必需的属性，无效的属性或非法值，对链接中文件的错误引用，重复的属性，无效的CSS选择器格式，无效的CSS属性，未使用的CSS类定义，无效的本地锚点和更多。

JavaScript重构

为JavaScript提供的重构功能使您可以轻松地修改代码结构以及撤消修改。可用于JS的一些重构包括：重命名，提取变量/函数，内联变量/函数，移动/复制，安全删除，将嵌入式脚本提取到文件中。

JSLint / JSHint

JavaScript代码质量工具集成在JavaScript编辑器中，可在编辑代码时对其进行检查。打开“设置”，键入JSLint / JSHint，启用检查，然后在键入时将对照这些检查检查您的代码。

Node.js

PhpStorm通过存储库中提供的免费插件完全支持Node.js。它提供代码完成，突出显示，导航，错误检查，调试，使用npm管理软件包等。

ECMAScript Harmony支持

对于那些希望领先一步的人，PhpStorm提供了ECMAScript Harmony的支持。要尝试新功能，请在IDE设置中将ECMAScript Harmony设置为JavaScript语言版本。

JavaScript模板和Web组件支持

PhpStorm支持EJS模板引擎（语法突出显示，键入帮助，格式设置等），并为车把和Moustache模板提供格式设置和语法突出显示，以及自动标签关闭。

Web组件也有初始支持。

角度支撑

PhpStorm中的角度支持包括针对TypeScript或ECMAScript 2015的指令和绑定的代码完成和导航，以及了解新的事件属性。

流程支持

您可以使用Flow添加到JavaScript的类型注释。使用带注释的函数或方法时，您将在代码完成时获得键入的参数信息。

TSLint

PhpStorm提供与TSLint的集成，TSLint是用于TypeScript代码的linter。键入时，启用它可以在编辑器中从TSLint看到警告和错误。

Vue.js

享受Vue模板语言以及.vue文件的脚本和样式块中您选择的语言的编码帮助。IDE自动完成Vue组件并为其添加所需的导入。
