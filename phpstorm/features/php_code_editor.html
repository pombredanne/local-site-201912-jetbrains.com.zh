PHP代码编辑器

智能代码编辑器

PhpStorm具有适用于PHP的丰富，智能的代码编辑器，实际上可以“获取”您的代码并深刻理解其结构，并为现代和遗留项目支持PHP 5.3、5.4、5.5、5.6、7.0、7.1。IDE提供智能代码完成，语法突出显示，扩展代码格式配置，即时错误检查，代码折叠，支持语言混合等功能。

语言功能支持

支持最新的PHP 7 ，包括对匿名类，返回类型声明，空合并运算符，组使用声明，上下文敏感词法分析器等的支持。PHP 7.1兼容性检查将帮助您将项目更新到最新的PHP版本。还支持PHP 5.3、5.4、5.5、5.6和7.0。

PHP重构

重命名和移动重构适用于文件，函数，常量，类，属性，方法，参数以及局部和全局变量。轻松，安全地进行全局项目更改。本地更改可立即就地进行。

还提供以下重构：提取变量/常量/字段/方法/接口，内联变量，向上推成员/向下拉成员，更改签名，将类移动到另一个名称空间，移动静态成员，进行静态重构。

代码补全

自动代码完成（在键入时调用）完成类，方法，变量名和PHP关键字的确定，并根据字段和变量的类型确定它们的常用名称。代码完成还支持PHP代码中的数组索引。

代码格式化

具有代码样式配置和其他功能的代码格式化程序可帮助您编写易于维护的简洁代码。PhpStorm包含符合PSR1 / PSR2，Symfony2，Zend，Drupal和其他标准的内置代码格式。

代码（重新）编排

代码（重新）编排器是控制现有代码和生成代码结构的强大工具。例如，您可以指定要在字段之后排序的方法，或者将相关方法保持在一起。

PHPDoc支持

PhpStorm编辑器在您的代码中尊重PHPDoc ，并根据注释提供适当的代码完成建议。在为代码编辑PHPDoc时，将从相应的代码块中自动完成变量名称和类型。PHP代码重构也考虑使用PHPDocs使它们保持最新。

参数提示

通过PhpStorm中的参数提示，代码变得更易读和更易理解。将显示用作方法参数的文字和空值的参数提示，以帮助您更好地理解特定方法的语义。

代码质量分析

数以百计的代码检查会在您键入代码时验证您的代码，并检查整个项目是否存在错误或代码异味。

大多数检查的快速修复功能使立即修复或改进代码变得很容易。 Alt + Enter显示每次检查的适当选项。

PHP代码嗅探器和混乱检测器

只需打开PHP文件，即可使用PHP Code Sniffer（phpcs）和PHP Mess Detector（phpmd）快速检查代码。每当您更改文件中的某些内容时都会触发检查，因此非常容易修复由PHP Code Sniffer和PHP Mess Detector报告的问题。

树枝，刀片和聪明的模板

在处理模板文件时，使用Twig，Blade或Smarty模板引擎来利用生产力功能：针对不同结构的可配置语法高亮；语法错误突出显示；功能和属性完成；自动插入成对的花括号，引号和结束标记；和更多。

Phar包

PhpStorm透明地识别Phar打包的代码。

智能重复代码检测器

已为所有主要支持的语言（包括PHP，HTML和CSS）添加。使用代码|找到“重复项”以开始针对复制/粘贴编码的任务。将为您提供重构候选列表-借助重构，可以很容易地使代码保持DRY。

回调参考

代码编辑器了解字符串文字中的回调引用，并提供高级帮助，例如用法搜索和重命名重构。

命名空间

PhpStorm提供了对名称空间的智能处理，包括自动导入，优化导入，可折叠导入等。

高效的项目分析和导航

其全面的源代码分析使PhpStorm甚至可以为未注释的代码提供复杂的代码完成功能，例如，当从函数的主体中推断出函数返回类型并从构造函数代码中提取返回语句或类属性类型（和声明）时。

文件结构视图以及类，方法和调用层次结构视图允许更快的代码查看和导航。

数据流分析

数据流分析可帮助您更好地了解项目代码，解释代码的复杂部分，查找源代码中的瓶颈等。

轻松的代码导航和搜索

转到类/文件/符号 -快速项目导航的关键操作。使用通配符和缩写词可以更快地找到所需的项目。

转到声明 -导航到使用简单热键或Ctrl + Click /⌘Click声明相关功能，变量或标签的位置。

查找用法 -帮助您查找代码，当前文件或整个项目中任何符号（例如，类，方法，字段等）的用法。
