VCS集成

PhpStorm支持最流行的版本控制系统，包括Git，Subversion，Mercurial，Perforce，CVS，TFS等。在PhpStorm中，所有繁琐的任务（例如添加，删除和删除文件）都是自动执行的。内置的可视合并工具可以快速直观地解决所有冲突。当您在编辑器装订线中键入内容时，将突出显示本地所做的更改，从而提供直观的导航和2单击回滚以显示单个更改。

立即开始工作

打开一个现有文件夹，从VCS中检出代码，甚至指定FTP下载和自动同步文件。您仅需片刻即可开始运行。

在本地保存项目文件或按需编辑项目文件，并使用自动同步将其部署回远程服务器。

VCS功能

  • 集成变更列表-将您的变更分组为多个变更列表，以更好地组织
  • 搁置的更改-保留一些更改以供日后恢复
  • 存储库更改视图—查看其他团队成员已提交的内容
  • UML变更视图
  • 传入更改视图-查看尚未集成到本地副本中的代码更改
  • 更改通知-上次同步后，如果您正在使用的文件已更改，则会收到警报。

数据库与SQL

注意： PhpStorm包含与数据库和SQL相关的DataGrip的所有功能，并允许您直接在IDE中使用数据库。

数据库支持

PhpStorm提供了用于在项目中使用数据库和SQL的工具和代码辅助功能。连接到数据库，编辑架构和表数据，运行查询，甚至使用UML图分析架构。

SQL编辑器

可以将SQL代码注入其他语言或在SQL编辑器中进行编辑，并提供语法突出显示，智能代码完成，动态代码分析，代码格式设置和导航功能。

其他显着特征

PhpStorm支持最流行的版本控制系统，包括Git，Subversion，Mercurial，Perforce，CVS，TFS等。

当地历史

IDE会跟踪对源文件所做的任何更改，即使其他应用程序进行了更改，也可以保护您免受意外损失或修改。您可以随时检查特定文件或目录的历史记录，并回滚到其任何以前的版本。

编码样式支持

PhpStorm包含与PSR1 / PSR2，Symfony2，Zend，Drupal和其他标准兼容的内置代码格式，并且使您可以轻松地为PHP CodeSniffer指定自己的特定于项目的编码标准。在PhpStorm中重新格式化代码可帮助您的代码轻松满足编码标准。

PHP UML

PhpStorm通过为PHP代码提供可编辑的UML类图来帮助开发人员理解和更改其代码。您可以快速检查您的应用程序结构，创建和编辑类及其关系。在上下文菜单中查找“图…”。可以通过快捷菜单直接从图中应用许多重构。

与问题跟踪器集成

当您的待办事项列表位于问题/错误跟踪系统中时，您可以配置PhpStorm处理问题，而无需从IDE进行切换。 PhpStorm支持以下错误跟踪系统：JetBrains YouTrack，Atlassian JIRA，Lighthouse，Pivotal Tracker，GitHub，Redmine，Trac等。

外挂程式

PhpStorm插件存储库包含250多个各种IDE插件，其中包括对主要PHP框架，其他VCS，与各种工具的集成以及诸如Vim仿真之类的编辑器增强功能的支持。

可以从IDE设置中的插件管理器浏览和管理PhpStorm插件存储库。

访问PhpStorm 插件存储库网站。

Docker支持

用于PhpStorm的Docker插件使您可以直接从PhpStorm向现有项目添加Docker支持，查看日志，管理Docker容器。您还可以使用Xdebug或Zend Debugger调试Web应用程序。

流浪汉

Vagrant是原生集成在PhpStorm中的便捷工具，旨在帮助您创建可复制的开发环境。它允许您通过共享简单的脚本来共享开发虚拟机。然后其他开发人员可以通过运行“ up” vagrant命令来启动同一台计算机。

作曲家

现在支持Composer（PHP的依赖性管理器）：使用Composer创建新项目，在现有项目中创建Init composer，使用用户友好的UI管理依赖性和添加软件包，通过命令行工具使用Composer，等等。

内置REST客户端

REST Client集成在IDE中，使您可以直接从PhpStorm测试RESTful Web服务。只需通过HTTP对具有各种参数的RESTful API调用不同的请求（例如GET，POST，PUT等），即可获取响应和响应头。

带有远程工具的内置SSH控制台

内置的SSH控制台使您可以连接到任何远程计算机，并通过SSH执行各种操作。此外，可以将SSH远程工具配置为通过一次击键就可以从IDE直接运行任何远程工具。

命令行工具

PhpStorm具有命令行工具支持，并为各种外部命令行工具提供了预配置的命令完成功能，包括Composer，Symfony控制台，Zend Framework 2工具（ZFTool），Zend Framework 1工具，Drush for Drupal，基于Symfony控制台的工具（ Laravel，教义）。也可以添加任何其他自定义工具。

适用于PHP的Google App Engine

为适用于PHP的Google App Engine开发应用程序，并将例行部署任务委托给IDE。

创建您的应用程序，运行它并将其部署到GAE服务器，而无需离开PhpStorm。

远程PHP解释器

使用远程解释器而不是本地解释器，您可以在类似于生产的环境中运行应用程序和基于PHP的工具，无论是真正的服务器还是使用Vagrant创建的虚拟化服务器。

网络支持

PhpStorm会自动完成并检查构建文件中的标准标记，属性，目标名称和路径属性值。

跨平台体验

只需单个许可证密钥，即可在Windows，macOS和Linux上使用相同的集成开发环境。

SFTP / FTP / FTPS保持活动

PhpStorm自动将保持活动消息发送到SFTP / FTP / FTPS服务器和远程解释器，这样在每次操作后都不会关闭连接。

模板项目

您可以将任何项目另存为模板，并基于它创建一个新项目。工具菜单中有一个动作，将项目另存为模板。保存后，该模板即可在“新建项目”窗口中作为用户定义的模板使用。
