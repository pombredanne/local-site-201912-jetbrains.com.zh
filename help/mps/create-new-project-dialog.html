Create New Project Dialog

File | New Project

Welcome Screen | Create New Project

Use this dialog to create empty projects or generate framework-specific project stubs for developing applications.

When you create a new project, you have to select a project type, specify the project name and project location, configure optional details. For more information, see MPS project structure.

  1. Choose a project type in the left-hand pane of the dialog: Language, Solution or Empty project.

  2. Specify Project Name, Project Location and, in the More settings field, Language or Solution Name.

  3. In Language project, you also can create a Runtime Solution and Sandbox Solution.
