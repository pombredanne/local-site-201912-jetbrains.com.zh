创建新项目对话框

档案|新项目

欢迎屏幕|建立新专案

使用此对话框可以创建空项目或生成特定于框架的项目存根以开发应用程序。

创建新项目时，必须选择一个项目类型，指定项目名称和项目位置，并配置可选详细信息。有关更多信息，请参见MPS项目结构 。

  1. 在对话框的左侧窗格中选择一个项目类型：“语言”，“解决方案”或“空项目”。

  2. 指定“ 项目名称” ，“ 项目位置” ，然后在“ 更多设置”字段中指定“ 语言”或“ 解决方案名称”。

  3. 在语言项目中，您还可以创建运行时解决方案和沙箱解决方案。
