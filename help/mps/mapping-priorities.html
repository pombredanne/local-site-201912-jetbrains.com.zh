映射优先级

映射优先级是一组映射优先级规则，用于指定生成器规则（映射配置）组之间的优先级顺序。

映射优先级在生成器模块属性对话框中指定。
另请参阅： 发电机

每个映射优先级规则由中间的左侧部分，右侧部分和优先级符号组成。
例如：

* <[sample.language / generator：*]

优先级规则的左侧部分只能引用此生成器中的映射配置。

符号

指

*

此生成器中的所有映射

modelName 。*

模型中的所有映射

modelName 。 mappingName

映射配置

{ ...}

一组映射。
条目可以是
modelName 。 *或
modelName 。 mappingName

优先级规则的右边部分可以引用此生成器以及范围内可见的其他生成器中的映射配置（请参见Generator ）。

符号

指

*

此生成器中的所有映射

modelName 。*

模型中的所有映射

modelName 。 mappingName

映射配置

{ ...}

一组映射。
条目可以是
modelName 。 *或
modelName 。 mappingName

[ languageName / generatorName ：*]

所有映射都在外部生成器中

[ languageName / generatorName ： modelName ]

外部生成器模型中的所有映射

[ languageName / generatorName ： modelName 。 mappingName ]

外部映射配置

[ languageName / generatorName ：{ ...}]

所有映射都在外部生成器中

[*：*]

所有可能的映射

优先符号

符号

描述

<

左边的映射要先处理右边的映射

<=

左侧的映射不迟于右侧的映射进行处理

==

左部分和右部分中的映射一起处理
