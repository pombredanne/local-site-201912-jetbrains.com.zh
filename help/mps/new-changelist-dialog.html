New Changelist Dialog

Version Control tool window

Alt+Insert

Use this dialog to create a new changelist.

Item

Description

Name

Type the name of the new changelist.

Comment

Type optional comment. When the new changelist will be submitted to the repository, this comment will appear in the Comment text area of the Commit Changes dialog.

Make this changelist active

Select this checkbox to have MPS automatically give the active status to the new changelist immediately after the changes are restored in it.

When this checkbox is cleared, the current active changelist remains active.
