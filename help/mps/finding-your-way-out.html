寻找出路

本文档应为您提供有关如何解决MPS问题以及如何寻找解决方案的即时分步建议。它是有组织的模式和操作方法的集合，结合了我们自己的经验。

反光编辑器

投影编辑器本质上以受控方式向用户展示模型。根据语言设计者的意图，该语言可能向用户隐藏某些信息或某些节点，并禁止某些方式来操纵代码。同样，如果编辑器定义在某种意义上是损坏的或不完整的，则编辑器可能不允许用户以她希望的方式修改代码。反射性编辑器提供了抑制语言编辑器的方法，而是以默认的树状形式显示模型。这样，开发人员可以完全直接访问模型。

F5使编辑器恢复正常。

节点浏览器

Control + X键盘快捷键为用户提供了一种可视化AST的方式，该AST表示已在编辑器中选择的代码段。

检查节点的类型

了解您要查找的元素的类型可能会给您非常有用的见解。您需要做的就是按Control + Shift + T ，MPS会弹出一个对话框，其中包含carret下元素的类型。

检查carret下节点的概念

Control + Shift + S / Cmd + Shift + S键盘快捷键将使您获得当前正在查看或所选节点的概念的定义。

检查carret下节点的编辑器

Ctrl + Shift + E / Cmd + Shift + E键盘快捷键将使您进入编辑器的定义，以定义您当前正在查看或选择的概念。如果您想熟悉一个概念的具体语法及其提供的所有选项，这可能特别有用。

类型系统跟踪

当您遇到类型问题时，“ 类型系统跟踪”工具将使您深入了解类型的计算方式，从而可以帮助您发现问题的根源。在Type-system Debugging中检查详细信息。

调查结构

当您学习一种新语言时，该语言的结构方面通常是开始研究的最佳位置。方便快捷地浏览概念和搜索用法的快捷方式肯定会派上用场。


您绝对应该熟悉Control + B / Cmd + B （转到定义）， Control + N / Cmd + N （转到概念） ，Control + Shift + S / Cmd + Shift + S （转到概念声明）和Alt + F7 （查找用法）可让您的调查变得顺畅而高效。

导入元素

您正在尝试使用元素或语言功能，但是，MPS无法识别语言构造或在代码完成对话框中不提供该元素。因此，您无法以所需的方式更新代码。这是一个典型的初学者问题的征兆-缺少导入和使用的语言。

  • 为了使用某种语言的语言构造，必须在“ 使用的语言”中列出该语言 。

  • 为了能够输入模型中的元素，必须首先导入模型。

  • 另外，为了使您的语言增强另一种语言的功能，必须在“ 扩展语言”中列出该语言 。

保存瞬态模型

如果从生成器中获取错误，则可以考虑启用“ 保存瞬态模型”功能。这将保留代码生成的所有中间阶段，以供您检查。


为什么我会收到此错误/警告？

您会发现MPS对某些代码不满意，并且想找出原因。使用Control + Alt +单击/ Cmd + Alt +单击打开包含详细信息的对话框。


转到规则按钮将带您进入触发错误/警告的规则。
