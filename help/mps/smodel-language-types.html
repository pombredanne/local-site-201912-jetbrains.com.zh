SModel语言类型

种类

SModel语言具有以下类型：

  • node-对应一个AST节点（例如， 节点myIf = ... ）

  • nlist- 对应于AST节点列表（例如nlist body = ... ）

  • node-ptr- 表示一个节点的指针， 提供存储库时可以解析为节点

  • 模型-对应于MPS模型的实例

  • model-ptr-表示模型的指针，可以在提供存储库时解析为模型

  • 搜索范围-对应于节点引用的搜索范围，即引用的允许目标集

  • reference-对应于表示参考实例的AST节点

  • 概念-对应于org.jetbrains.mps.openapi.language。代表概念的SConcept概念（例如concept = concept / IfStatement / ）

  • conceptNode- （已弃用）对应于表示概念的AST节点（例如conceptNode = conceptNode / IfStatement / ）

  • 枚举成员-对应于一个代表枚举成员的AST节点（例如，枚举成员focus = ... ）

大多数SModel语言操作都适用于所有这些类型。
