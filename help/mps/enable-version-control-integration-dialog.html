Enable Version Control Integration Dialog

VCS | Enable Version Control Integration

In this dialog, choose one of the registered Version Control Systems to use in your project and assign it to the Project Root.

The dialog and the menu item are available only if the project does not use any Version Control System.

Item

Description

Select a version control system to associate with the project root

From this drop-down list, select one of the supported version control systems that you want to associate with the project root.
