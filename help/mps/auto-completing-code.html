代码完成

本节介绍了各种上下文感知的代码完成技术，这些技术可帮助您加快编码过程。

基本完成

基本代码完成功能可帮助您完成可见性范围内的类，方法和关键字的名称。当您调用代码完成功能时，MPS会分析上下文并建议从当前插入符号位置可以到达的选择。

非英语键盘布局也可以使用基本完成功能。

代码完成功能可用于自定义文件类型 。但是，MPS无法识别此类文件的结构，因此会建议选项，无论它们在当前情况下是否合适。

如果将基本代码完成功能应用于字段，参数或变量声明的一部分，则MPS会根据项目类型建议一个可能的名称列表。

调用基本完成

  1. 开始输入名称。

  2. 按Ctrl+Space或选择代码|完成|从主菜单基本 。

  3. 如有必要，请按Ctrl+Space第二次（或按Ctrl+Alt+Space ）。

智能完成

智能代码完成功能会过滤建议列表，并仅显示适用于当前上下文的类型。

调用智能完成

  1. 开始输入。

  2. 按Ctrl+Shift+Space 。

    突出显示了针对当前上下文的最合适建议。

  3. 如有必要，请按Ctrl+Shift+Space再来一次。

声明完成

您可以使用语句完成来创建语法正确的代码构造。它插入必要的语法元素（括号，花括号和分号），并使您处于可以开始键入下一条语句的位置。

完成代码构造

开始输入代码构造并按Ctrl+Shift+Enter 。

MPS自动完成构造并添加所需的标点符号，然后将插入号放置在下一个编辑位置。

完成提示和技巧

缩小建议列表

您可以通过键入单词的任何部分（甚至是中间位置的字符）或在点分隔符后调用代码完成来缩小建议列表的范围。MPS将显示建议，其中包括您在任何位置输入的字符。

这使得不需要使用通配符：

接受建议

您可以通过以下方式之一接受列表中的建议：

  • 按Enter或双击列表项以将其插入插入符号的左侧。

  • 按Tab替换插入符号右侧的字符。

  • 使用Ctrl+Shift+Enter使当前的代码构造在语法上正确（平衡括号，添加缺少的花括号和分号，等等）。

完成捷径

您可以对最常用的语句之一使用以下实时模板快捷方式：

  • sout ：版画System.out.printIn()

  • soutm ：添加当前的类和方法名称

  • soutp ：添加方法参数名称和值

  • soutv ：添加最后一个变量值

查看代码层次结构

从建议列表中选择一个条目后，您可以查看代码层次结构：

  • Ctrl+H ：视图类型层次结构

故障排除

如果无法完成代码，则可能是由于以下原因之一：

  • 省电模式已打开（ 文件|省电模式 ）。开启它可以消除背景操作，包括错误突出显示，即时检查和代码完成，从而将笔记本电脑的功耗降至最低。

  • 未为您的项目配置SDK。

  • 您的文件不驻留在内容根目录中，因此它没有获得代码完成所需的类定义和资源。

  • 包含要在完成建议列表中显示的类和函数的文件被标记为纯文本文件。

  • 包含要显示在完成建议列表中的功能的外部库不会作为依赖项或全局库添加 。
