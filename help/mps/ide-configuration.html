IDE configuration

Many aspects of MPS can be configured through the Settings/Preferences dialog Ctrl+Alt+S.

To quickly navigate to particular configuration items you may use the convenient text search box in the upper left corner. Since the focus is set to the text field by default, you can just start typing. Notice that the search dives deep into the individual screens:

Plugins

MPS is modular and bundles several plugins. If you open the MPS Plugin Manager you will see a list of plugins available in your installation.


Additionally installed plugins with supported languages are also listed here.

If some plugins are not necessary for your current work they can be switched off, which may have an increase in the overall performance of the platform.
