项目工具窗口

查看|工具窗口|项目
Alt+1

通过此工具窗口，您可以从各种角度查看项目并执行任务，例如创建新项目（目录，文件，类等），在编辑器中打开文件，导航到感兴趣的代码片段等。

此工具窗口中的大多数功能都可以在内容窗格和关联的快捷方式中作为上下文菜单命令来访问。

观看次数

工具窗口为您提供以下视图：

  • 项目视图。在此视图中，显示了所有项目项及其依赖项（SDK和库）。重点在于目录结构（尽管也显示了软件包）。

从标题栏左侧的列表中选择所需的视图，或者如果视图由选项卡表示，则单击相应的选项卡。

要配置视图，请使用标题栏上下文菜单中的相应选项。还可以通过单击来访问必要的选项在标题栏上。

标题栏上下文菜单

通过右键单击标题栏出现的上下文菜单提供项目视图的设置。

下表列出并简要说明了可用的命令和选项。

捷径

描述

自动滚动到源

如果启用此选项，MPS将自动在编辑器中打开所选项目。否则，您需要双击项目以将其打开。

从源代码自动滚动

如果启用此选项，则MPS会自动在此窗口的当前视图中找到您在编辑器中打开的文档。

固定，对接，浮动，窗口显示，拆分模式

这些选项使您可以控制工具窗口的一般外观和行为。

从侧边栏删除

此命令隐藏工具窗口，删除用于显示和隐藏工具窗口的按钮，并从快速访问菜单中删除工具窗口（ 要么 ）。

要再次打开工具窗口（并恢复关联的功能），请使用主菜单：“ 查看” |“更改”。工具窗口| 。

组选项卡

如果启用此选项，则标题栏左侧会有一个列表，您可以从中选择所需的视图。如果关闭此选项，则视图由显示在标题栏左侧的选项卡表示。

搬去

要将工具窗口与其他工具窗口栏关联，请选择此命令，然后选择目标工具窗口栏（“ 顶部” ，“ 左侧” ，“ 底部”或“ 右侧” ）。

调整大小

要通过移动工具窗口的边框来调整其大小，请选择此命令，然后选择必要的“ 拉伸至”选项。

请注意，此命令不适用于浮动模式。

隐藏

 Shift+Escape

使用此命令隐藏工具窗口。

内容窗格

内容窗格显示与项目及其内容关联的文件夹，或按DB数据源分组的数据库控制台。

内容窗格项的上下文菜单命令

使用项目的上下文菜单可以访问该项目的所有可用功能。

项目

捷径

描述

新

 Alt+Insert

使用此命令来创建新项目（目录，文件或类）。在所选的一个（项目或目录）中。

切

 Ctrl+X

使用此命令将选定的一个或多个项目从当前位置移动到剪贴板。

复制

 Ctrl+C

使用此命令将选定的一个或多个项目复制到剪贴板。

糊

 Ctrl+V

使用此命令将剪贴板的内容插入到所选位置。

跳转到源

 F4

使用此命令在编辑器中打开选定的文件。如果文件已经打开，则相应的编辑器选项卡将变为活动状态。

复制路径

 Ctrl+Shift+C

使用此命令可将一个或多个所选项目的完整路径复制到剪贴板。

查找用法

 Alt+F7

使用此命令查找所选项目的用法。（将打开“ 查找用法”对话框 。）

在路径中查找

 Ctrl+Shift+F

使用此命令执行文本搜索。（将在“在路径中查找”对话框中打开。）

替换为路径

 Ctrl+Shift+R

使用此命令执行文本搜索和替换。（“ 在路径中替换”对话框将打开。）

重构

使用此命令可执行可用于所选项目的重构之一。

例如，您可以重命名，移动和复制文件和目录。

删除

 Delete

使用此命令删除所选项目。小心使用！

运行'
调试' '

 Ctrl+Shift+F10

运行测试配置。

当地历史

使用此命令可查看所选文件或目录的本地历史记录 ，或为项目的当前版本创建标签。

同步' '

使用此命令将所选项目与其保存在文件系统中的版本同步。

（如果从外部更改文件或目录的内容，则在某些情况下，除非使用此命令，否则MPS可能不知道相应的更改。）

在资源管理器里显示

使用此命令打开文件浏览器（例如Windows资源管理器或Finder），并在其中显示所选的项目。
