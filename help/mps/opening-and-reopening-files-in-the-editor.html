在编辑器中打开和重新打开文件

打开文件进行编辑

打开文件进行编辑

MPS解析的文件的最大大小由idea.max.intellisense.filesize在idea.properties文件中设置。

位于MPS安装文件夹的bin目录中的idea.properties文件不应被编辑。无需编辑原始的idea.properties文件， 而是在以下位置创建一个idea.properties文件，将其打开以进行编辑并添加所需的属性：

  • 对于Windows ： ％USERPROFILE％\。％prod1％config

  • 对于* NIX ： 〜/.% prod1％/ config

  • 对于macOS ： 〜/ Library / Preferences /％prod1％/ config

请注意，文件越大，其编辑器的工作速度就越慢，并且总体系统内存要求也越高。

  1. 请执行以下任一操作：

    • 在一个工具窗口中双击所需的文件。

    • 在工具窗口之一中选择所需的文件，然后按F4 。

    • 在工具窗口之一中选择所需的文件，然后在上下文菜单中选择“ 跳转到源文件”。

  2. 如果注册了文件类型，则文件在MPS的编辑器中以静默方式打开。

    如果文件类型注册在“ 在关联的应用程序中打开的文件”类别下，它将在其关联的应用程序中打开，而不是在MPS编辑器中打开。默认情况下，MPS建议使用许多此类文件类型，例如.doc ， .chm或.pdf 。

    如果文件类型未知，MPS建议选择是要注册新文件类型，还是在其关联的应用程序中打开该文件。在“ 注册新文件类型关联”对话框中指定选择：

    您可以在“设置/首选项” |“设置”中注册所需的文件类型。 编辑器文件类型 。

打开外部文件

请执行以下任一操作：

  • 选择文件|在主菜单上打开 ，然后在打开的对话框中选择所需的文件。

  • 将所需的文件从资源管理器（Windows），文件浏览器（Linux）或Finder中拖放到编辑器中。该文件将打开，以便在新选项卡中进行编辑。

重新打开文件

重新打开文件

  • 要打开最近打开的文件，请选择“ 查看” |“ 打开 ”。主菜单上的“ 最近的文件 ”或按Ctrl+E 。然后从“ 最近的文件”弹出窗口中选择所需的文件。

  • 要打开最近更新的文件，请在主菜单上选择“ 查看” |“打开”。最近更改的文件或按Ctrl+Shift+E 。然后从打开的“ 最近编辑的文件”弹出窗口中选择所需的文件。

在“ 编辑器设置”页面的“ 最近使用的文件限制”文本框中，定义最近使用的文件的最大数量。

在单独的窗口中打开文件

在单独的MPS窗口中打开文件

请执行以下任一操作：

  • 将编辑器选项卡拖放到当前MPS窗口之外。

  • 按Shift+F4在“项目”工具窗口中选择的文件。

  • Shift+mouse double click在“项目工具”窗口中的文件名上。
