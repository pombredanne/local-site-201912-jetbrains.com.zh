如何安装语言插件

MPS可以使用其他语言轻松扩展。语言打包为普通的zip文件，您可以将其解压缩到MPS 插件目录中，并在重新启动时加载MPS。

安装语言插件的最便捷方法是通过“ 插件管理器” ，该插件可在“ 设置/首选项”对话框中找到。 Ctrl+Alt+S 。

您可以安装以前收到的zip文件（“ 从磁盘安装插件”选项），也可以单击“ 浏览存储库”按钮，然后从已上传到MPS插件存储库的插件列表中选择所需的插件。

在JetBrains插件存储库中已公开提供了一些MPS和IntelliJ IDEA的语言插件（由JetBrains和第三方提供）：

  • 文档中的“ MPS语言存储库”页面。

  • MPS语言插件站点。

  • 在MPS站点中开发的IntelliJ IDEA语言插件 。
