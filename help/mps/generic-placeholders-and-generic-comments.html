通用占位符

通用占位符表示两个节点之间的空白，可以将其添加到任何节点集合中。Control / Cmd + S + Enter键组合可将占位符插入集合中的当前位置。占位符的行为是透明的-您仍可以在占位符节点上调用完成菜单以替换它的节点，或按Enter键将普通节点添加到下一个同级位置。

使用任何语言的通用占位符，即使该语言不支持这种概念，用户也可以在代码中插入任意的视觉分隔符（空行）。

一般评论

通用占位符本身可能包含内容。MPS使用jetbrains.mps.lang.text语言或通用 devkit为占位符提供文本内容。这将为您提供完全可编辑的多行文本语言，并支持基本样式（粗体，斜体和下划线），可单击的超链接和嵌入式节点（代码）。

包含jetbrains.mps.lang.text语言或通用 devkit后，请在占位符上按“ [ ”（方括号）。您将在一个节点上输入和编辑文本。文本为多行，由单词组成。任何单词都可以设置为粗体 （按Control + B ）， 斜体 （按Control + I ）和带下划线 （按Control + U ）。要添加链接，请按Alt + Ente r并调用“ 添加链接”意图。
