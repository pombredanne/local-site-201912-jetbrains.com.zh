上下文动作工具

上下文操作工具可适应某些DSL用户对鼠标导航的偏爱。它在侧栏中列出了在给定上下文中适用的操作，这些操作可能是按层次组织的。

提供了有关示例机器人Kaja语言使用和定义上下文动作工具的简短视频。

常见的行为是在单击一次激活的光标位置执行某些操作。上下文动作工具非常适合发现探索性工作，因为它清楚地传达了在这种情况下可以执行的操作。

通过在工具窗口中选择一个选项，用户可以触发关联的操作，该操作通常会在当前焦点的位置修改模型。

该窗口具有与编辑者无关的位置，并且包含操作的分层菜单。如果菜单的深度大于一个级别，则通过将菜单名称与/分隔符（例如Toplevel / SecondlevelA / ThirdlevelA ）。
 

边栏的内容由语言设计人员通过新的“ 转换菜单语言”指定 。

该语言支持菜单的模块化及其易于重用，因此将超级概念或子概念的菜单组合起来应该非常简单。导入jetbrains.mps.editor.contextActionsTool.lang.menus语言后，您将能够为操作，组和其他相关元素指定上下文操作工具 位置 。该语言还使您可以为给定的条目指定图标和工具提示 。

由于工具窗口提供了足够的垂直空间，并且由于可以将条目分组为可折叠的子菜单，因此您也可以考虑通过子菜单操作将现有的“替换”菜单包括在工具窗口中。

 

有关实际语言语法的完整详细信息，请参见“ 转换菜单语言”页面。
