操纵工具窗口

显示工具窗口

请执行以下任一操作：

  • 在工作区的左下角，指向然后在显示的菜单中选择工具窗口。

  • 点击上的相应工具窗口按钮工具窗栏 。

  • 选择查看|工具窗口| 在主菜单中。

  • 如果工具窗口具有关联的快速访问编号，请按Alt （例如，对于“ 项目”工具窗口，请按Alt + 1 。

隐藏单个工具窗口

请执行以下任一操作：

  • 点击上的相应工具窗口按钮工具窗栏 。

  • 请点击在工具窗口标题栏上。

  • 右键单击相应的工具窗口按钮，然后从上下文菜单中选择“ 隐藏 ”。

  • 右键单击工具窗口标题栏，然后从上下文菜单中选择“ 隐藏 ”。

  • 选择查看|工具窗口| 在主菜单中。

  • 如果工具窗口具有关联的快速访问编号，请按Alt （例如，对于“ 项目”工具窗口，请按Alt + 1 。

  • 如果您要隐藏的工具窗口当前处于活动状态，请按Shift+Escape 。

隐藏连接到同一工具窗口栏的所有工具窗口

请执行以下任一操作：

  • 按下并按住不动Alt ，然后点击在附加到相应工具窗口栏的任何工具窗口的标题栏上。

  • 选择窗口|活动工具窗口|在主菜单中隐藏Side Tool Windows 。此命令隐藏与活动工具窗口或活动工具窗口中的最后一个工具窗口相连的所有工具窗口。

隐藏所有工具窗口

请执行以下任一操作：

  • 选择窗口|活动工具窗口|在主菜单中隐藏所有Windows 。

  • 按Ctrl+Shift+F12 。

切换到上一个活动工具窗口

请执行以下任一操作：

  • 选择窗口|活动工具窗口|跳至主菜单中的“ 上一个工具窗口 ”。

  • 按F12 。

如果当前隐藏了所有工具窗口，则将显示最后一个活动的工具窗口并将其激活。

隐藏或显示工具窗口栏

如果在MPS窗口中需要更多空间，则可以隐藏所有工具窗口栏：

  • 在工作区的左下角，单击 。

如果工具窗口栏是隐藏的，则可以将它们永久或短暂地重新显示在屏幕上：

  • 要还原工具窗口栏，请单击在工作区的左下角。

  • 要在短时间内查看工具窗口栏，请双击并按住Alt 。工具窗口栏出现在屏幕上，使工具窗口按钮可访问。释放键时，工具窗口栏再次被隐藏。

隐藏工具窗口按钮

MPS使隐藏单个工具窗口按钮成为可能，而无需实际卸载相应的插件。

从视图中删除工具窗口按钮

  1. 确保工具窗口栏可见 。

  2. 右键单击要隐藏的工具窗口按钮。

  3. 选择隐藏

要恢复隐藏的工具窗口按钮，请选择“ 视图” |“视图”。从主菜单中选择“ 工具窗口 ”，然后单击带有隐藏的工具栏按钮的窗口。

将工具窗口附加到其他工具窗口栏

  • 将相应的工具窗口按钮拖动到所需的工具窗口栏（顶部，左侧，底部或右侧）。

  • 右键单击相应的工具窗口按钮或工具窗口标题栏以打开上下文菜单。选择“ 移至” ，然后选择目标工具窗口栏（“ 顶部” ，“ 左侧” ，“ 底部”或“ 右侧” ）。

调整工具窗口大小

  • 将鼠标指针悬停在工具窗口边框上。当指针变为双向箭头时，请沿所需方向拖动边框。

  • 您还可以通过向左或向右或上下移动步长来调整工具窗口的大小。可以使用以下替代方法：

    • 右键单击相应的工具窗口按钮或标题栏，然后选择“ 调整大小” 。然后选择可用的“ 延伸到”选项之一。

    • 使感兴趣的工具窗口处于活动状态，然后执行以下任一操作：

      • 选择窗口|活动工具窗口|调整大小 ，然后选择必要的“ 延伸到”选项。

      • 按Ctrl+Shift结合相应的箭头键。

增加一次显示的工具窗口数量

要增加一次显示的工具窗口的数量，应适当设置不同工具窗口的查看模式 。考虑以下：

  • 通常，为了使工具窗口始终可见（即，即使处于非活动状态），也应固定工具窗口。

  • 可见的浮动窗口和窗口模式下的浮动窗口的数量没有限制因素。

  • 为了能够一次看到两个停靠在同一工具窗口栏上的窗口（而不是一个），其中一个窗口应关闭拆分模式 ，而另一个应打开。

  • 最初，使用三个（四个）工具窗口栏。您可以通过移动某些工具窗口来“激活”第四工具窗口栏（顶部）。

保存和恢复工具窗口的排列

您可以通过选择窗口|保留当前设置工具窗口的方式。在主菜单中将“ 当前布局”存储为“默认 ”。

以后，可以通过选择“ 窗口” |“返回”来保存工作空间。恢复默认布局 Shift+F12 。
