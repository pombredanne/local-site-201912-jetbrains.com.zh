Global Libraries

File | Settings | Global Libraries for Windows and Linux
MPS | Preferences | Global Libraries for macOS

Use this dialog to specify the languages and solution, which should be available on the IDE level.

Item

Description

Add

Click this button to add a new library. First, specify the library name, and then, in the corresponding dialog, select the desired directory.

Remove

Click this button to delete the selected library from the list.

Edit

Click this button to change the selected library.
