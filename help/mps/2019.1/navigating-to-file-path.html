Navigating to File Path

MPS enables you to open any file or directory in a file manager (like File Explorer or Finder) or in a Terminal session.

To view the absolute path to the file you are currently working on:

  • Hover the mouse cursor over the relevant tab in the editor.

To open any part of the file's path in a file manager:

  1. Hold Ctrl/Cmd and click the relevant tab in the editor.

  2. In the drop-down list, select the necessary item.

To open any file or directory in a file manager:

  • Right-click the relevant tab in the editor or an item in the Project tool window and click Reveal in Explorer or Reveal in Finder.

To navigate to any file or directory in a Terminal session:

  • Right-click the relevant tab in the editor or an item in the Project tool window and click Open in terminal.
