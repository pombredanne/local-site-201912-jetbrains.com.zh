Ctrl+Alt+Shift

This section lists and describes the keyboard shortcuts that include the Ctrl+Shift+Alt keys.

Shortcut

Function

Use this shortcut to...

Ctrl+Shift+Alt+C

Copy Relative Path

Copy a reference (a relative path) of a symbol to the Clipboard.

Ctrl+Shift+Alt+N

Go to Symbol

Navigate to a symbol with the specified name.

Ctrl+Shift+Alt+V

Paste Simple

Paste the last entry from the Clipboard as plain text.
