Navigating to a Node or File by Name

Overview

Navigate commands enable you to quickly jump to the desired files or nodes specified by names. MPS suggests a look-up list of matching names, from which you can select the desired one, and open it in the editor. This navigation honors CamelCase and snake_case capitalization. Refer to the tips for detailed list of available techniques.

Navigating by name

To jump to a node or a file with the specified name

  1. On the main menu, point to Navigate, and then choose Node, or File respectively, or use the following shortcuts:

    • Node: Ctrl+N

    • File: Ctrl+Shift+N

  2. In the popup, start typing the desired name.

    So doing, you can enter characters located anywhere inside the desired name. As you type, the suggestion list shrinks, displaying the matching names only.

Tips and tricks

While working in the navigation popup, use the following helpful techniques:

  • Narrow down the search scope by selecting the file types to search in. Just click the filter , and clear the checkboxes next to the file types you are not interested in.

  • Include non-project files in the look-up list and thus make available matching files from SDKs and libraries.

  • If the look-up list is too long, type more characters to shrink it, or click the ellipsis sign at the end of the list, to reveal its next portion.

  • Use * wildcard to represent any number of characters, though it is quite enough to type characters located in the middle of the desired name.

  • If while typing in one of the Navigate to Class/File/Symbol popups you notice that you need another one, just invoke the necessary dialog box. The text you have already entered will not disappear.

  • Press Alt+F1 to invoke the Select Target popup, and choose the desired IDE component.

  • When there is a detached editor frame with a certain file, you can opt to open this file in the main MPS frame by pressing Enter, or activate the detached frame by pressing Shift+Enter.
