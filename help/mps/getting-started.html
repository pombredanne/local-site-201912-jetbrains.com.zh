MPS 2019.2帮助

入门

有关MPS的最重要信息是此联机帮助 。要从MPS访问它，请执行以下一项操作：

  • 从主菜单中，选择帮助|帮忙 。

  • 按F1在键盘上。

  • 请点击在对话框或工具窗口中。

如果您没有Internet权限来查看联机帮助，则可以使用MPS帮助插件，该插件通过内置的Web服务器提供帮助页面以供离线使用。

发行新的主要版本时，将更新脱机帮助插件。在发行周期中添加到联机帮助的更改可能在脱机帮助中不可用。

联系支持

如果在联机帮助中找不到所需的信息，则可以联系JetBrains支持团队。

  • 浏览MPS知识库 。

  • 在社区论坛中创建主题。

  • 从主菜单中，选择帮助|与支持人员联系以直接向支持团队提出请求。

报告错误

如果您遇到错误或想要提出新功能，请使用MPS问题跟踪器 。

  • 从主菜单中，选择帮助|提交错误报告 。

在提交之前，最好在问题跟踪器中搜索类似的报告和功能请求，以避免重复。如果找到类似的票证，则可以对其添加评论或投票，以引起开发团队的更多关注。

如果需要附加MPS日志文件，请使用相关的“ 帮助”菜单项找到它：

  • 帮助|在资源管理器中显示日志 （适用于Windows和Linux）

  • 帮助|在Finder中显示登录 （适用于macOS）

分享意见

您可以使用反馈表告诉我们您对JetBrains产品的满意或不满意之处。

  • 从主菜单中，选择帮助|提交反馈 。

学到更多

有几种方法可以了解有关MPS的更多信息：

一天的小贴士

每次启动MPS时都会显示并提供有用的提示。要取消显示它们，可以清除“ 在启动时显示提示”复选框。

要打开“ 每日提示”对话框，请选择“ 帮助” |“帮助”。主菜单中的每日提示 。

生产力指南

显示具有统计信息和提示的有用功能的列表。

要打开它，请选择帮助|。主菜单中的“ 生产率指南” 。

要发现新功能，请按组对列表进行排序，并在与您更常使用的功能相同的组中记录很少使用的功能。单击该功能以查看其用法说明。

按键图参考

列出了一些最有用的键盘快捷键，您可以将其放在办公桌上以备参考。

要打开它，请选择帮助|。主菜单中的“ 键盘映射参考 ”。

有关更多信息，请参见掌握MPS键盘快捷键 。
