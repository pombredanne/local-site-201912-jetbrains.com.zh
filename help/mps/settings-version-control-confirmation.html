创建文件时

在本节中，指定是否以及如何将在MPS中创建的文件置于版本控制下：

  • 在添加到版本控制之前显示选项 ：在打开的对话框中指定选项后，新创建的文件将置于版本控制下。

  • 静默添加 ：新创建的文件会自动添加到版本控制中，而不会发出任何通知。

  • 不要添加 ：新创建的文件保持未版本控制，以后可以将它们置于版本控制下。

删除文件时

在此部分中，指定删除文件时是否以及如何从版本控制中删除文件：

  • 从版本控制中删除之前显示选项 ：在打开的对话框中选择本地删除的文件后，还将从VCS中删除该文件。

  • 静默删除 ：在不要求确认的情况下，将从VCS中删除本地删除的所有文件。

  • 不要删除 ：在本地删除的文件仍受版本控制。

当空的更改列表变为非活动状态时

在本节中，指定删除空变更列表时的MPS行为。

  • 删除前显示选项 ：MPS在删除失去活动状态的空变更列表之前要求确认。

    如果选择删除这样的变更列表，MPS建议选择另一个要标记为活动的变更列表。

  • 静默删除 ：MPS会自动删除变为非活动状态的空更改列表， 默认更改列表除外。

  • 不要删除 ：空的更改列表在失去其活动状态时不会被删除。

调用这些命令时的“显示选项”对话框

在此区域中，指定在调用下面选择的命令时是否希望MPS要求您确认。

  • 命令的可用性取决于特定的版本控制系统。

  • 如果您通过选中“以后不再显示此对话框”复选框来禁止显示特定于操作的选项对话框，请使用此区域中的控件来还原原始设置。

显示清除只读状态对话框

选中此复选框可使MPS在编辑器中打开文件并尝试对其进行修改时明确要求取消只读状态。

可以通过两种方式清除只读状态：

  • 使用当前的VCS：文件已添加到变更列表

  • 使用文件系统：文件未添加到变更列表。

此选项与将签出与打开分开进行编辑的版本控制系统相关。

在分支切换时还原Workapce

如果要在切换到该分支时还原其工作空间（即一组打开的文件，与某个分支关联的当前运行配置和断点），请选择此选项。

创建后在资源管理器中显示补丁

使用此列表定义创建补丁程序时的MPS行为。可用的选项有：

  • 是 ：如果选择此选项，则本机文件管理器将始终打开以显示补丁文件。

  • 否 ：如果选择此选项，则本机文件管理器将不会打开。

  • 询问 ：如果选择了此选项，MPS将显示一个对话框，通知成功创建补丁，并建议在本机文件管理器中找到补丁文件。
