首次运行MPS

首次运行MPS时，或者从早期版本升级MPS时，需要一些步骤才能完成安装，自定义实例并开始使用IDE。

以下视频教程将指导您完成这些步骤：

导入MPS设置

在“ 完整安装”对话框中，选择是否要导入IDE设置：

如果这是您的MPS的第一个实例，请选择“不导入设置”选项。

您可以稍后使用文件|手动导出和导入设置 。导入设置和文件|从主菜单中导出设置命令。

在MPS中启动项目

在欢迎屏幕上，您可以执行以下操作：

  • 创建一个新项目
  • 打开一个现有的项目或文件
  • 从版本控制系统中检出现有项目

也可以将现有项目目录或单独的文件拖到“欢迎”屏幕，然后在MPS中打开它。
