MPS 2019.2帮助

用户界面概述

首次运行MPS或未打开任何项目时，MPS将显示“ 欢迎”屏幕 。当您打开一个项目时，MPS将显示主窗口。

这是默认的（即用的）IDE接口布局。根据您安装和启用的插件集，IDE的外观和行为可能有所不同。

  1. 主菜单和工具栏 ：包含影响整个项目或项目部分的命令，例如打开，创建项目，重构代码，运行和调试应用程序，在版本控制下管理文件等。

    主工具栏复制了主菜单的基本命令，以加快访问速度。默认情况下，主工具栏是隐藏的。要显示它，请选择查看|外观|主菜单中的工具栏 。

  2. 导航栏 ：“ 项目工具窗口”的快速替代方法。使用它可以浏览项目并打开文件进行编辑 。

    使用视图|外观|导航栏隐藏或显示导航栏。按Alt+Home将焦点更改到导航栏。

  3. 状态栏 ：指示项目，整个IDE的状态，并显示各种警告和信息消息。

  4. 编辑器 ：读取，创建和修改您的代码。

  5. 工具窗口 ：辅助窗口，提供对项目管理，搜索，运行和调试，与版本控制系统集成以及其他任务的访问。
