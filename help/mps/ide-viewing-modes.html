Viewing modes

MPS provides special viewing modes for specific usage patterns. For example, if you need to focus on the code or present your code to an audience.

Toggle viewing modes: View | Appearance.

  • In Full Screen mode, MPS expands the main window to occupy the entire screen. All operating system controls are hidden, but you can access the main menu if you hover the mouse pointer over the top of the screen.

  • In Distraction-free mode, the editor occupies the entire main window and the source code is centered. All other elements of the UI are hidden (tool windows, toolbars, and editor tabs) to help you focus on the source code of the current file. You can still use shortcuts to open tool windows, navigate, and perform other actions.

  • In Presentation mode, MPS expands the editor to occupy the entire screen and increases the font size to make it easier for your audience to see what you are doing. Other elements of the UI are hidden, but you can bring them up with corresponding shortcuts or using the main menu if you hover the mouse pointer over the top of the screen.

Productivity tips

Use the quick switcher

  1. Press Ctrl+` (View | Quick Switch Scheme).

  2. In the Switch popup, select View Mode, and then select the required viewing mode.

Create shortcuts

  • You can map actions that toggle viewing modes to shortcuts as described in Configuring keyboard shortcuts.
