基础语言

BaseLanguage是MPS的Java对应版本，因为它与Java共享几乎相同的构造集。BaseLanguage是MPS和同时扩展最广泛的语言中最常见的代码生成目标。

为了简化与Java的集成，可以为MPS中的所有模块指定类路径。然后，将在类路径上找到的类自动导入到@java_stub模型中，因此可以直接在使用BaseLanguage的程序中使用。

MPS经常扩展的概念包括：

  • 表达方式 。构造，其计算结果为某些结果，例如1，“ abc”等。

  • 声明 。构造，可以包含在诸如if / while / synchronized语句之类的方法级别上。

  • 输入 。变量的类型，例如int，double。

  • 操作 。构造，可以放置在类似node.parent的点后。此处的父元素是IOperation 。

  • AbstractCreator 。构造，可用于实例化各种元素。

BaseLanguage被创建为Java 6的副本。Java 7和8兼容性对BaseLanguage的扩展已逐渐添加。

  • Java 7语言构造包含在jetbrains.mps.baselanguage.jdk7语言中

  • Java 8语言扩展包含在jetbrains.mps.baselanguage.jdk8语言中

  • 您可能想查看专用于MPS与Java互操作性的文档
