折叠代码元素

您可以折叠（折叠）代码片段，将它们减少到单个可见行。这样，您可以隐藏当前不重要的细节。如果需要，可以折叠（折叠）的代码片段。

折叠后的代码片段通常显示为阴影椭圆（ ）。

如果折叠的代码块中包含错误，则MPS以红色突出显示折叠的片段。

代码折叠方式

您可以使用以下方法折叠和扩展代码片段：

  • 代码折叠开关（ ， 要么 ）。这些切换显示在编辑器中相应折叠区域的左侧。如果某个区域展开， 指示区域的开始，而位于其末端。对于折叠区域，切换显示为 。

    如果您按住Alt键并单击 ， 要么 ，则该区域会与其所有下属区域一起递归折叠或扩展。

  • 折叠菜单命令和相关的键盘快捷键。可以从主菜单（ Code | Folding ）或在编辑器中作为上下文菜单访问“ 折叠”菜单。快捷方式显示在菜单的右侧。请参阅折叠菜单的命令和相关的快捷方式 。

  • 折叠后的片段本身：单击扩展相应的片段。另请参阅查看折叠的代码片段 。

折叠预定义和自定义区域

您可以折叠和展开：

  • 对应于导入声明，方法主体，文档注释等的预定义区域 。预定义的区域大致对应于默认情况下“ 折叠”下列出的区域 编辑器|一般| MPS设置的“代码折叠”页面Ctrl+Alt+S 。

    对于预定义区域，可立即使用折叠开关，而无需执行任何其他操作。

  • 任何选定的代码片段。通过代码|创建和删除选择的自定义折叠区域。 折叠折叠选择/删除区域命令Ctrl+. 。

指定代码折叠首选项

您可以指定：

  • 是否应显示代码折叠切换。

  • 默认情况下应折叠哪些折叠区域。

相应的设置已开启 编辑器|一般| MPS设置的“代码折叠”页面Ctrl+Alt+S 。

查看折叠的代码片段

要查看折叠片段的内容，请指向省略号表示该片段。

查看折叠区域的开始

要查看折叠区域的开始位置（如果当前尚不可见），请指向该区域末端的折叠开关。
