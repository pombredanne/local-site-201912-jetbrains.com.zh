命名

  • 首选长名称 ：在查找列表中突出显示最长的名称，以完成代码。如果禁用，则突出显示最短名称。

  • 名称前缀和名称后缀 ：指定在使用MPS代码生成功能生成用于命名新符号的建议时要使用的前缀和后缀。对于空白字段，使用不带前缀或后缀的默认名称建议。添加前缀值时，MPS会自动将建议的基本名称的首字母转换为大写。

    例如，如果静态字段的前缀定义为s字段的类型是Counter ，则建议的静态字段名称将为sCounter 。

    此前缀不影响getter和setter方法名称的生成。因此，在这种情况下，访问者名称为getCounter和setCounter 。
