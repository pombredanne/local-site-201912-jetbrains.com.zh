启动和停止Hub ZIP

本页描述如何停止和启动集线器服务器。

在以下情况下，您需要停止集线器服务器：

  • 您要更改中心数据库的位置。

  • 您要修改集线器服务器的集线器属性或JVM选项。

  • 您要升级集线器安装。

如果安装了带有ZIP分发的集线器，则可以从命令行运行集线器。您用于启动和停止Hub的脚本位于/bin/目录。的 directory是安装过程中解压缩ZIP分发包的位置。

要在macOS或Linux平台上停止和启动Hub，请以非root用户身份执行命令。

此页面上的所有命令均适用于Linux和macOS。如果您使用的是Windows服务器，请更换.sh与.bat并使用反斜杠。

在命令行界面中输入命令来停止和启动集线器服务。使用以下命令来管理集线器实例的状态：

命令

描述

hub.sh run

启动集线器服务。使用此命令，控制台将被阻止，直到使用停止服务为止。 hub.sh stop命令。

hub.sh start

启动集线器服务。使用此命令，仅在启动过程中阻止控制台。

hub.sh stop

停止集线器服务。

hub.sh restart

重新启动集线器服务。

hub.sh help

显示可以在命令行界面中执行的所有命令的列表。
