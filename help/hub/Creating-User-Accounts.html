创建用户帐户

将新用户帐户添加到集线器的最常见方法是启用外部身份验证模块。身份验证模块使用户可以使用他们在第三方应用程序中管理的凭据登录集线器。每个身份验证模块都可以让您自动为使用外部凭据登录的未注册用户创建中心帐户。您还可以将新用户添加到特定组中，以自动授予访问权限并为连接的服务分配许可证。有关更多信息，请参阅身份验证模块 。

此页面介绍了可用于手动创建集线器用户帐户的选项。

创建一个用户帐号

要提供对单个用户的访问权限，请在中心中创建一个新的用户帐户。

手动创建新的用户帐户：

需要权限：“读取用户基本信息”，“创建用户”

  1. 在管理菜单的访问管理部分，选择用户 。

  2. 单击新用户按钮。

  3. 在“ 新用户”对话框中，单击“ 创建用户”按钮。

    • 该对话框显示用于创建新用户帐户的输入字段。

  4. 输入以下设置的值：

    设置

    描述

    全名

    设置新用户的全名。

    电子邮件

    输入新用户的电子邮件地址。该电子邮件地址必须有效。

    密码

    输入新用户帐户的密码。

    确认

    确认新用户帐户的密码。

    强制更改密码

    选择此选项可强制用户更改密码。下次使用这些凭据登录时，系统会提示用户更改密码。

  5. 单击创建按钮。

    • 创建具有指定设置的用户帐户。

    • 新用户的个人资料页面打开。在这里，您可以配置用户帐户的访问权限。

创建帐户后，需要确保用户可以访问Hub所管理的项目和服务。

  • 选择“ 组”选项卡，然后将用户添加到一个或多个组。有关更多信息，请参见管理组成员资格 。

  • 选择“ 角色”选项卡，并向用户授予组或项目中的一个或多个角色。有关更多信息，请参见管理用户访问 。

  • 如果用户需要访问限制了可以使用该应用程序的用户数量的服务，请向该用户帐户授予许可证。您可以在“ 服务”页面上向用户和组授予许可证。有关更多信息，请参阅管理服务许可证 。

邀请新用户注册

在Hub中创建用户帐户的一种方法是邀请新用户注册自己的帐户。

邀请用户注册时，实际上是为新用户创建一个帐户和个人资料。受邀用户可以确认帐户并设置自己的密码。

创建帐户的事实使您可以作为管理员在发送邀请之前配置访问权限。

在邀请用户注册之前，必须在Hub中启用电子邮件通知。有关更多信息，请参阅系统设置 。

要邀请一个或多个用户在Hub中注册帐户，请执行以下操作：

需要权限：“读取用户基本信息”，“创建用户”

  1. 在管理菜单的访问管理部分，选择用户 。

  2. 单击新用户按钮。

  3. 在“ 新用户”对话框中，单击“ 邀请用户”按钮。

    • 该对话框显示一个输入字段，用于输入新用户的电子邮件地址。

  4. 在“ 电子邮件”输入字段中，输入要邀请其在集线器中注册帐户的用户的电子邮件地址列表，以空格分隔。

  5. 单击邀请按钮。

    • 邀请被发送到每个被邀请用户的电子邮件地址。邀请中包含指向登录页面的链接，用户可以在其中设置密码并访问集线器。

允许用户自行注册

要让用户注册自己的帐户，请在集线器身份验证模块中启用注册。

要配置集线器身份验证模块：

需要权限：低级管理员写

  1. 在管理菜单的访问管理部分，选择身份验证模块 。

  2. 从列表中选择集线器模块。

  3. 将注册选项设置为已启用 。

    • 显示了可用于管理在YouTrack中注册自己的帐户的用户的其他设置。

  4. 单击保存按钮。

    • 用户可以在Hub中创建自己的帐户。

为了防止您的集线器安装受到机器人的攻击，可以在注册页面上启用reCAPTCHA。要了解如何设置此功能，请参阅CAPTCHA设置 。

启用使用来自外部身份验证提供程序的凭据登录

YouTrack支持使用来自外部身份验证源（例如LDAP或各种OpenID提供程序）的帐户授权用户。要使用户能够使用来自外部提供商的帐户登录，必须启用并配置相应的身份验证模块。

文档的“身份验证模块”部分介绍了受支持的身份验证模块的设置说明。
