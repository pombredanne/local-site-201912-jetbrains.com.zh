更新用户资料

作为管理员，您可以访问中心中每个注册用户的完整用户配置文件。要打开用户个人资料，请在“ 用户”列表中选择一个“ 登录名 ”。

用户配置文件的“ 常规”选项卡显示基本信息，例如全名，登录名，头像和电子邮件地址。具有“ 读取自我”和“ 更新自我”权限的用户可以自己查看和编辑此信息。这些权限默认情况下分配给集线器中的所有默认角色。这意味着您几乎不需要代表其他用户来管理用户配置文件数据。

具有“ 读取用户基本”和“ 更新用户”权限的用户可以更新其他用户的用户配置文件数据。

作为管理员，您可以在用户配置文件的“ 常规”选项卡上执行多项操作。在这里，您可以：

  • 禁止用户 。

  • 删除用户帐户 。

  • 发送请求以验证用户电子邮件地址。如果用户没有至少一个已验证的电子邮件地址，则该用户将不会收到有关恢复忘记密码的说明。

  • 将SSH公钥添加到帐户中以授予对已连接服务的访问权限。

  • 将其他凭据添加到用户配置文件 。

  • 管理现有的登录名。

  • 禁用两因素身份验证 。

用户页面上的其余选项卡使您可以管理组成员身份并授予访问权限。

  • 有关组成员资格的更多信息，请参阅管理组成员资格 。

  • 有关访问权限的更多信息，请参见管理用户访问 。
