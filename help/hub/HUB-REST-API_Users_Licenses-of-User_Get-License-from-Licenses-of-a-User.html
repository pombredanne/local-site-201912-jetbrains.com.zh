Get License from Licenses of a User

Request

GET <REST API URL>/users/{user id}/licenses/{license id}?fields=string

Parameter

Description

fields

Optional. Returns only the specified subset of the fields for each License. Use Fields Syntax to define the subset.

Response

Response Code

Meaning

200 OK

Successful request.

400 Bad Request

At least one of the request parameters is invalid. For example, a required field in the passed JSON object is missing. For details, check the error message in the response.

403 Forbidden

The requester has no access to the requested resource.

404 Not Found

The requested resource was not found.

500 Internal Server Error

Failed to process request because of the server error. For details, check the error message in the response.

Response body

{ "licenseKey": string, "licenseName": string, "service": service, "users": [user, ...], "autoJoinGroups": [userGroup, ...], "licenseInfo": licenseInfo, "availableLicenses": int, "id": string /* from settings */, "aliasIds": [string, ...] /* from settings */, "aliases": [alias, ...] /* from settings */ }
