Hub 2019.1帮助

从用户许可证中获取许可证

请求

GET /users/{user id}/licenses/{license id}?fields=string

参数

描述

fields

可选的 。仅返回每个许可证的字段的指定子集。使用字段语法定义子集。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "licenseKey": string, "licenseName": string, "service": service, "users": [user, ...], "autoJoinGroups": [userGroup, ...], "licenseInfo": licenseInfo, "availableLicenses": int, "id": string /* from settings */, "aliasIds": [string, ...] /* from settings */, "aliases": [alias, ...] /* from settings */ }
