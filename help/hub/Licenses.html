管理服务许可证

服务的“ 许可证”选项卡提供了可用于管理服务许可证的工具。

此页面上描述的过程仅适用于外部集线器安装以及与Upsource安装捆绑在一起的内置集线器服务。在使用内置集线器服务或YouTrack InCloud实例的YouTrack Standalone安装中，请勿尝试更改这些设置。

  • 页面顶部显示基本许可证属性。这包括可以被授予许可证的用户数量和有效期。

  • 对于具有用户限制的许可证，您可以使用一些选项来管理分配了使用该应用程序的许可证的用户和组。

授予每用户许可证

对于限制使用该应用程序的用户数量的服务，您可以向特定用户和组成员授予许可证。

当您向组授予许可证时，该组成员的所有用户都将获得许可证。在分配所有可用许可证之前，将自动为添加到组的用户授予该服务的许可证。如果组启用了自动加入选项，则将用户添加到集线器后会自动为其授予许可证。

授予许可证：

需要权限：更新用户，更新组，读取服务

  1. 单击授予许可证按钮。

  2. 在添加用户和组输入字段中，输入用户或组的名称。过滤用户和组的列表以匹配输入字段中的文本。

  3. 从下拉列表中选择一个用户或组。对于列表中的每个组，将显示该组成员的用户数。

  4. 继续添加用户和组，直到您已将许可证分配给需要它们的所有用户。可用许可证的数量显示在输入字段上方。

  5. 单击授予许可证按钮。

    • 许可证被授予选定的用户和选定组的成员。

    • “ 可用许可证”设置中的许可证数量已更新。

    • 如果可用许可证的数量少于所选用户的数量，则将许可证随机授予所选用户和所选组的成员。未获得许可证的组成员显示在没有许可证标题的用户下方。

详细信息侧边栏显示项目授予每个用户的角色。使用此信息来确定用户是否需要该服务的许可证。

吊销许可证

如果您的用户或组不再需要应用程序许可证，则可以从单个用户或整个组中撤消许可证。

要撤消一个或多个用户的许可证：

需要权限：更新用户，更新组，更新服务

  1. 从列表中选择一个或多个用户。

    • 如果列表按组显示，则一次只能选择一个用户。要选择多个用户，请单击“ 用户”以切换显示模式。

    • 使用按用户搜索控件来过滤列表。

  2. 单击撤消许可证按钮。

  3. 在“ 吊销许可证”对话框中确认操作。

    • 许可证已从所选用户撤消。

    • “ 可用许可证”设置中的许可证数量已更新。

要从一个或多个组中撤消许可证：

需要权限：更新组，更新服务

  1. 从列表中选择一个或多个组。仅当列表按组显示时，此选项才可用。

  2. 单击撤消许可证按钮。

  3. 在“ 吊销许可证”对话框中，选择用于从组及其成员吊销许可证的选项。

  4. 单击撤消按钮。

    • 许可证已从所选组及其成员中撤消。

    • “ 可用许可证”设置中的许可证数量已更新。

从组中删除许可证

将许可证分配给组是简化许可证分配的好方法。但是，如果您发现不再希望向新的组成员授予许可证，或者用完了要分配的许可证，则可以删除该组而无需从其成员撤消该许可证。

要从一个或多个组中删除许可证：

需要权限：更新组，更新服务

  1. 从列表中选择一个或多个组。仅当列表按组显示时，此选项才可用。

  2. 单击撤消许可证按钮。

  3. 在“ 吊销许可证”对话框中，选择选项以从组中吊销许可证，但直接向其成员授予许可证。

  4. 单击撤消按钮。

    • 选定的组将从列表中删除。

    • 所选组的成员直接获得许可。

    • “ 可用许可证”设置中的许可证数量已更新。

编辑许可证

如果您已续订许可证或升级了用户包，则可以在此页面上更新许可证。

要编辑许可证：

需要权限：更新服务

  1. 单击编辑许可证链接。

  2. 在“ 编辑许可证”对话框中，在“ 许可证名称”输入字段中编辑许可证的名称 。例如，将名称从Upsource更改为Upsource 500-user 。

  3. 将许可证密钥粘贴到“ 许可证密钥”输入字段中。

  4. 单击保存按钮。

    • 许可证被上载到集线器服务器并应用于服务。

    • 许可证的属性已更新。

删除许可证

您也可以从服务中删除许可证。仅在将重复服务迁移到集线器并且需要手动配置许可证时，才使用此功能。

删除许可证：

需要权限：更新服务

  1. 单击编辑许可证链接。

  2. 在“ 编辑许可证”对话框中，单击“ 删除”按钮。

  3. 在对话框中确认操作。

    • 许可证已从服务中删除。

    • 已被授予使用该服务许可证的所有用户吊销该许可证。
