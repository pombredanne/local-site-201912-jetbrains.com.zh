获取QueryAssist

请求

GET /events/queryAssist?query=string&caret=int

参数

描述

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于事件，支持以下字段和元组：

after ： Instant —给定时刻之后的事件

afterId ： number —给定id之后的事件（不包括）

作者 ： 用户 -事件触发的用户，组或服务

author ： group —由用户，组或服务触发的事件

author ： service —由用户，组或服务触发的事件

before ： 即时 —给定时刻之前的事件

beforeId ： number —给定id之前的事件（包括）

entityType ： EntityType —具有给定类型的实体的事件

event ： auditEventType-创建，更新或删除给定类型的事件

target ： value —具有给定事件目标的事件

target ： authModule —具有给定事件目标的事件

target ： role —具有给定事件目标的事件

target ： service —具有给定事件目标的事件

target ： user —具有给定事件目标的事件

target ： group —具有给定事件目标的事件

caret

类型为int的查询参数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "query": string, "caret": int, "styleRanges": [queryStyleRange, ...], "suggestions": [querySuggestItem, ...] }
