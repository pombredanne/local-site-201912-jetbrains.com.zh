Get All Untrusted Redirect U R Is of a Service

Request

GET /services/{service id}/untrustedredirecturis?fields=string&$skip=int&$top=int
Parameter Description
fields Optional query parameter. Specifies a subset of the fields to be returned. Use Fields Syntax to define the subset.
$skip Optional query parameter. Number of untrusted redirect u r is to skip before returning the first entry in the page.
$top Optional query parameter. Maximum number of untrusted redirect u r is to return in the page.

Response

Response Code Meaning
200 OK Successful request.
400 Bad Request Some of the parameters of the request are invalid (for example, required fields in the passed JSON object are missing). The response contains message describing the error.
403 Forbidden Requester has no access to the request.
404 Not Found Requested resource wasn't found.
500 Internal Server Error Failed to process request because of the server error. The response contains message describing the error.

Response body

page

{ "skip": int, /* Requested number of untrusted redirect u r is to skip before returning the first entry in the page */ "top": int, /* Requested maximum number of untrusted redirect u r is to return in the page */ "total": int, /* Total number of untrusted redirect u r is. Returned only if the total field is included as a value of the fields parameter in the query */ "untrustedredirecturis": [untrustedRedirectURI, ...] }
