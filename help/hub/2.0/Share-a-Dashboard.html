Share a Dashboard

You can configure a dashboard and share it with another user or group.

This feature is especially useful if you manage a team. You can create a dashboard with widgets that contain information that is useful for everyone on the team. When you share the dashboard, every member of your team has access to the same information.

If you don't want to share the whole dashboard, you can also share an individual widget.

To share a dashboard:

  1. Create a new dashboard or select an existing dashboard.
  2. Configure the dashboard to include the widgets you want to share with your team.
  3. Click the Share button.
  4. In the Share Dashboard dialog, select one or more groups or users from the drop-down list.

    To remove a group or user from the list, click the x icon.

  5. If you want other users to be able to modify the dashboard, select the Allowed to edit dashboard option.

    To share the dashboard in read-only mode, deselect this option.

  6. Click the Share dashboard button.

    — The users who members of the selected groups and individual users on the list see a link to the dashboard on their Dashboard page.

Manage Users for Shared Dashboards

You can update the list of users and groups who have access to the shared dashboard at any time.

  • If the dashboard is shared with a single user or group, the name of the user or group is displayed as a link in the toolbar of the dashboard.
  • If the dashboard is shared with multiple users or groups, the link displays the number of users and groups who can access the dashboard.

Click this link to manage the users and groups who have access to the dashboard.

The following options are available:

OptionDescription
Add user or groupClick the Add user or group link to share the dashboard with additional users or groups.
Remove user or groupClick the X icon next to a user or group to remove them from the list.
Allowed to editSelect the Allowed to edit option to allow the selected users or groups to modify the dashboard.

Use this option to grant write access to specific users.
