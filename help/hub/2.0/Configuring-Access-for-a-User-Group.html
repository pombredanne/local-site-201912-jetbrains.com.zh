View Access Permissions of a Group

The access permissions for a group are displayed on the Access tab of the group profile. Here, you can view which roles and permissions are assigned to the group.

To view the access rights for a group:

  1. Open the Access tab of the group profile.
  2. Click one of the following buttons to see the access rights that are available to the group:
    • Access rights by project — displays a list of the projects for which the group has access.
    • Roles — displays a list of roles that are granted to the group. Select the role to see the project in which the role is granted and how the role is applied.
    • Permissions — displays a list of permissions that are granted to the group. Select a permission to view the project in which the permission is granted.

Permissions

Global:

  • Read User
  • Read Role

Per-project:

  • Read Project
  • Read User Group

Grant a Role to a Group

You can grant a role to a group directly. Members of the group are granted all access permissions assigned to this role.

To assign an individual role to a group:

  1. Click the Groups link in the main navigation.
  2. In the list of user groups, locate a group that you want to assign a role to and click its name to open the group profile.
  3. Select the Access tab.
  4. Click the Grant Access button.
  5. In the Grant Access dialog, choose a role to assign to the group.
  6. Select a project where you want to apply the role. If you want to apply the role to all projects in the system, select the Global project.
  7. Select an option that determines how the role is assigned.
  8. Click the Grant Access button.

    The selected role is granted to the user group and all its permissions are granted to the users who are members of the group.

Permissions

Per-project:

  • Read User Group
  • Add Role in Project
  • Remove Role in Project

Grant a Permission to a Group

If there is a specific permission you would like to grant to all members of a group, yYou can grant the permission to the group directly.

To grant a permission to a group:

  1. Click the Groups link in the main navigation.
  2. Find a group using the search box and click its name in the list to open the group profile.
  3. Select the Access tab.
  4. Click theGrant Access button.
  5. In the Grant Access dialog, click the Permission button.
  6. Select a permission to grant to the group in the drop-down list. To narrow down the list, start typing the name of the permission.
  7. Select the projects where you want to apply the permission.
  8. Select an option that determines how the permission is assigned (for example, by selecting which role to assign to the project).
  9. Click the Grant Access button.

Permissions

Per-project:

  • Read User Group
  • Add Role in Project
  • Remove Role in Project
