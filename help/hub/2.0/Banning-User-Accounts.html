Ban User Accounts

You can ban a user account, so the banned can no longer access Hub to perform any operation.

Banned users are not counted as active users for license restrictions. If, for example, you have a license for 10 user accounts and already have 10 registered users, you can ban one account and create a new one. Of course, that's not the only case for banning a user, but one of the most useful.

To ban a user account:

  1. In the Users list, select the user accounts that you want to ban.

    Use the search box to filter the list of users. You can search for accounts by login, name, email, assigned roles or permissions. Start typing in the search box or click Ctrl + Space to open the query assistant and use the suggested search criteria.

  2. Click the Ban button in the toolbar.
    • The selected accounts are banned immediately.

Permissions

Global:

  • Read User
  • Update User

You can unban previously banned accounts at any time.

To unban a user account:

  1. Select one or more banned accounts in the Users list.
  2. Click the Unban button in the toolbar.
