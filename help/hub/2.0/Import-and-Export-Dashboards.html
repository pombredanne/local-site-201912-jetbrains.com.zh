Import and Export Dashboards

You can export a dashboard configuration. The configuration contains the settings for all of the widgets on the dashboard.

You can import the configuration into another dashboard yourself or send the configuration to another user.

To export a dashboard configuration:

  1. Click the Export link at the bottom of the page.
  2. Copy the code that is displayed in the dialog.

    You can then import the configuration into another dashboard or send the code to another user.

To import a dashboard configuration:

  1. Open the dashboard into which you want to import the configuration.
  2. Click the Import link at the bottom of the page.
  3. Paste the code from the export dialog or from a message if the configuration was exported by another user.
  4. Click the OK button.

    The widgets from the exported configuration are imported into the active dashboard.
