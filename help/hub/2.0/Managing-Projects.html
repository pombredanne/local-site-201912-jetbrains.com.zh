Projects

In the Projects area in Hub, you can manage and configure your projects. The following operations are available:

  • Merge projects
  • Archive and restore projects
  • Manage the resources that are available to the project, including the project team
  • Access a history events related to the project
Even though you can create a new project on this page, we recommend that you use the Project Wizard for this task. The Project Wizard provides shortcuts that let you create a project and manage connected services quickly and easily. For more information, see Project Wizard

To access the Projects page in Hub, click the Projects link in the main menu.
