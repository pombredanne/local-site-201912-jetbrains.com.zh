Configuring Project Settings

To configure a project in Hub, click the Projects link in the main menu to view the page with the list of all registered projects in Hub.

To overview parameters of a project, select it in the list and view details in the right side-panel:

To edit project settings, click the project name in the list to open its page:

Project Settings in Hub

Settings Tab

SettingDescription
NameSets the name of the project.
DescriptionOptional description of the project.
ImageSets the image that is displayed in for the project in Hub.
AuditLinks to the Audit Events page in Hub. There, you can view a list of changes that were applied to this project.
Archive statusShows the current status of the project: whether it is not archived (active) or archived. Here, you can also archive an active project or re-enable an archived one.

Resources Tab

The Resources tab lists all Hub Concepts assigned to the project.

Here, you can either add a new resource to the project or move a resource to another project.

Access Tab

In the Access tab, you can view the list of users that have access to the project and why they have it.

Switch between the views to review access to the project by users, granted role, or permission.

Click the Grant access... to grant access to the current project to a user.
