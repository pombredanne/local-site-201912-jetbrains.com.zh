Merge Projects

If you have multiple projects that use the same resources, or have finished a project and want to re-allocate its resources to another project, you can merge the projects.

To merge multiple projects:

  1. Click the Projects link in the main menu.
  2. In the list of projects, select two or more projects that you want to merge.
  3. Click the merge icon in the toolbar.
  4. In the Merge Projects dialog, select the Key and Name of the project that you want to keep after the merge.
  5. If you want to automatically archive the merged project, select the Archive it option.
  6. Click the Merge Projects button.
    • All of the selected projects are merged into a single project with the specified key and name.
    • All of the resources that are associated with the merged project, including groups, repositories, projects in YouTrack and Upsource are transferred to the resulting project.
