Quick Notes Widgets

A note widget displays whatever message you want to show on your dashboard.

This feature is especially useful if you share your dashboard with other users. You can use this widget as a bulletin board for posting updates that are important for your team.

To add a note widget to the dashboard:

  1. Click the Add widget button on the dashboard.
  2. From the drop-down list, select Notes.

    — A card is added to the dashboard with the title Notes.

  3. Enter your note in the input field. You can use wiki markup to format your text. Click the more link to view the supported formatting options.
  4. To preview the text in the widget, click the Show preview link.
  5. Click the Create widget button.

    — The note widget is added to the dashboard.
