Configure User Group Settings

You can configure the parameters of a group on its profile page. To open the group profile, click the Groups link in the main navigation and select the name of the group in the list.

The group profile lets you view and configure the following parameters:

  • The list of members of the group.
  • The general settings of the group, such as the name and description.
  • The access permissions for the group.

See the following pages for details about group configuration settings:

  • Configure Access for a Group
  • Edit Basic Group Settings
