Audit Events

Every event in Hub — creating, deleting, updating an account, project, service and so on — is logged and stored.

To review any data update in Hub, open the More Settings > Audit Events page:

You can also download information about any event as an XML file for further investigation. To do so, select one or several events in the list and click the Download as XML button.
