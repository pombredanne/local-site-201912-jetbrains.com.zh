Hub 2019.1帮助

获取用户组的所有子组

请求

GET /usergroups/{user group id}/subgroups?fields=string&$skip=int&$top=int&query=string&orderBy=string

参数

描述

fields

可选的 。仅返回每个子组的字段的指定子集。使用字段语法定义子集。

$skip

可选的 。让您设置要返回第一个子组之前要跳过的子组数。

$top

可选的 。使您可以设置要返回的最大子组数。

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于子组，支持以下字段和元组：

访问 （ 项目 ： 项目 ， 具有 ： 角色 ）—在给定项目中具有给定角色的组

access （ project ： project ， 具有 ： Permission ）-在给定项目中具有给定权限的组

访问 （ 项目 ： 项目 ）-在给定项目中具有某些角色的组

访问 （ 带有 ： 角色 ）-在某些项目中具有给定角色的组

访问 （ 具有 ： 权限 ）-在某些项目中具有给定权限的组

has ： ownRole —具有自己角色的组

具有 ： 用户 -与用户组

has ： 子组 —具有子组的组

has ： icon —带有图标的组

id ： value —具有给定ID的组

is ： allUsers-所有用户组

is ： autoJoin —启用了自动加入的组

is ： required2FA —具有必需的两因素身份验证的组

name ： groupName-具有给定名称的组

ownRole ： 角色 —具有给定角色作为自己角色的组

parent ： group —给定组的子组

project ： project —给定项目中的组

subgroup ： group —包含给定组作为子组的组

团队 ： 团队 -包含在给定项目团队中的组

user ： user —立即包含给定用户的组

orderBy

可选的 。返回按指定字段排序的子组列表。有关详细信息，请参见排序语法 。子组支持按以下字段排序： 名称
