YouTrack小部件

当YouTrack作为中心中的服务连接时，以下窗口小部件可在仪表板上使用。

问题列表小部件

问题列表窗口小部件显示与搜索查询，保存的搜索，项目或标签匹配的问题列表。

要将问题小部件添加到仪表板：

  1. 单击仪表板上的添加小部件按钮。

  2. 从下拉列表中，选择YouTrack Issues List 。

  3. 输入可选的小部件标题。

  4. 如果您的Hub实例连接到多个YouTrack服务器，请选择一个YouTrack服务用作窗口小部件的数据源。

  5. 通过执行以下操作之一来配置窗口小部件：

    • 选择上下文（可选），然后在输入字段中输入搜索查询。

    • 在“ 项目”选项卡上选择一个项目。

    • 在标签选项卡上选择一个标签。

    • 在“ 保存的搜索”选项卡上选择一个保存的搜索。

    保存的搜索，项目和标签列表仅显示在YouTrack侧栏中可见（或固定）的项目。

  6. 设置小部件的刷新率。默认刷新率是4分钟。

  7. 点击创建小部件按钮

    • 问题窗口小部件已添加到仪表板。

    • 如果未指定标题，则卡的标题将设置为所选的搜索查询，保存的搜索，项目或标签。

当您单击窗口小部件的标题时，YouTrack将显示问题列表，其中包含为窗口小部件配置的搜索查询的结果。

设置后，将根据自定义刷新率刷新窗口小部件内容。否则，小部件每10分钟刷新一次。要手动刷新问题窗口小部件，请单击标题栏中的刷新图标。

报告小部件

报告窗口小部件显示预定义报告的计算。您只能为现有的YouTrack报告添加报告小部件。您可以为已创建或已与您共享的报告添加报告小部件。

如果没有可用的报告，请单击“ 创建新报告”链接以打开YouTrack并创建报告。

报表窗口小部件支持可以在YouTrack的“报表”区域中创建的所有报表类型，但“ 甘特图 ”除外。不支持在敏捷板上以图表形式显示的报告，但是，您可以使用相似的设置来创建单独的“ 燃尽”和“ 累积流”报告，并将其添加到仪表板中。

要将报告小部件添加到仪表板：

  1. 单击仪表板上的添加小部件按钮。

  2. 从下拉列表中选择报告 。

    • 卡片将添加到仪表板，标题为“现有报告中的添加小部件” 。

  3. 单击链接以展开其他设置。提供以下选项：

    • 输入可选的小部件标题。

    • 以秒为单位设置窗口小部件的自定义刷新率。报表小部件的默认刷新率是10分钟（600秒）。

  4. 如果您的集线器实例连接到多个YouTrack服务器，请选择YouTrack服务用作窗口小部件的数据源。

  5. 在卡上选择一个报告。要限制报告列表，请在输入字段中输入文本。

    该列表显示您拥有的或与您共享的报告，但“ 甘特图”报告除外。

    • 报表小部件已添加到仪表板。

    • 如果您未指定标题，则卡的标题将设置为所选报告的名称。

如果要添加在小部件的报告列表中不可用的报告，请在YouTrack中创建报告。

设置后，将根据自定义刷新率刷新窗口小部件内容。否则，小部件每10分钟刷新一次。

对于报表窗口小部件，此操作并不总是重新计算报表。根据报告参数重新计算报告。例如，如果将报告设置为每天重新计算，则报告小部件每10分钟刷新一次。当根据报告参数重新计算报告时，小部件将在下次刷新小部件时显示更新的报告。

对于没有重新计算设置的“快速报告”，刷新窗口小部件时将重新计算报告。对于这些报告，请单击标题栏中的刷新图标以重新计算数据。
