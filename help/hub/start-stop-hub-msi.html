启动和停止集线器MSI

本页描述如何停止和启动集线器服务器。

在以下情况下，您需要停止集线器服务器：

  • 您要更改中心数据库的位置。

  • 您要修改集线器服务器的集线器属性或JVM选项。

  • 您要升级集线器安装。

当您通过MSI发行版安装集线器时，集线器将作为Windows服务运行。Windows服务的显示名称是JetBrains Hub 。初始配置后，该应用程序将在服务器的后台运行。

要启动和停止Windows服务，请使用服务Microsoft管理控制台（MMC） （“ 控制面板”>“管理工具”>“服务” ）。
