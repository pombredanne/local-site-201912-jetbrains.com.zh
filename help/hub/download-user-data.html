下载用户资料

集线器存储可用于标识用户的信息，例如用户的全名和电子邮件地址。根据适用于居住在欧盟的数据主体的通用数据保护条例（GDPR），用户有权访问其个人数据。具有注册帐户的用户可以自己查看和更新其大部分个人数据。他们还可以直接从其个人资料下载个人数据的副本。

被禁止的用户无法登录和访问此信息。在这种情况下，具有“ 低级管理员读取”权限的用户可以下载个人数据的副本，然后将其提供给请求它的用户。

当您下载用户的个人数据时，将从Hub数据库中收集以下信息并以CSV格式生成：

  • 全名

  • 登录

  • 电子邮件

  • Jabber帐户

  • VCS用户名

  • 用户个人资料数据的历史更改-每个记录都包括显示值的日期范围

  • 来自用于登录集线器的任何身份验证模块的最新登录信息的访问详细信息—每个记录包括上次访问的日期和时间，IP地址，设备和操作系统

要为另一个用户下载个人数据的副本，请执行以下操作：

需要权限：读取用户完整，低级管理员读取

  1. 在管理菜单的访问管理部分，选择用户 。

  2. 在列表中找到要导出数据的用户。使用过滤器查找与特定属性匹配的用户。

  3. 单击用户名称以打开用户个人资料页面。

  4. 点击个人数据设置旁边的CSV格式下载按钮。

    • 用户的个人数据的副本将下载到您的本地目录中。

    • 该文件被分配了具有以下格式的名称 data.csv 。
