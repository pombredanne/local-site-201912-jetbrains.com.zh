获取QueryAssist

请求

GET /projectroles/queryAssist?query=string&caret=int

参数

描述

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于ProjectRoles，支持以下字段和元组：

group ： group —给定组拥有的项目角色

id ： value —具有给定ID的项目角色

许可权 ： 许可权 —获得给定许可权的项目角色

project ： project —为给定项目授予的项目角色

角色 ： 角色 -以给定角色授予的项目角色

team ： project —给定项目团队拥有的项目角色

user ： user —给定用户拥有的项目角色

caret

类型为int的查询参数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "query": string, "caret": int, "styleRanges": [queryStyleRange, ...], "suggestions": [querySuggestItem, ...] }
