TeamCity小部件

当TeamCity作为中心中的服务连接时，以下窗口小部件可在仪表板上使用。

构建工件小部件

构建工件小部件显示特定构建配置的构建工件的列表。

要将构建工件小部件添加到仪表板：

  1. 单击仪表板上的添加小部件按钮。

  2. 从下拉列表中，选择Build Artifacts 。

    • 标题为Build Artifacts的卡被添加到仪表板。

  3. 单击链接以展开其他设置。提供以下选项：

    • 输入可选的小部件标题。

    • 以秒为单位设置窗口小部件的自定义刷新率。构建工件小部件的默认刷新率是10分钟（600秒）。

  4. 如果您的Hub实例连接到多个TeamCity服务器，请选择一个TeamCity服务用作窗口小部件的数据源。

  5. 从下拉列表中选择一个构建配置。

  6. 选择以下一个或多个选项，以进一步过滤构建工件的列表：

    • 要显示上一次成功构建的工件，请启用“ 显示上一次成功构建”选项。

    • 要显示来自上一个固定构建的工件，请启用“ 显示上一个成功的构建”选项。

    • 要显示带有一个或多个标签的构建中的工件，请在输入字段中输入标签列表。

  7. 点击创建小部件按钮

    • 构建工件小部件将添加到仪表板。

    • 如果您未指定标题，则所选的构建配置将显示在小部件标题中。

单击窗口小部件的标题时，TeamCity将显示窗口小部件中显示的构建配置的“ 概述”选项卡。

设置后，将根据自定义刷新率刷新窗口小部件内容。否则，小部件每10分钟刷新一次。要手动刷新构建工件小部件，请单击标题栏中的刷新图标。

调查小工具

调查窗口小部件显示在TeamCity中分配给您的调查列表。

要将调查小部件添加到仪表板：

  1. 单击仪表板上的添加小部件按钮。

  2. 从下拉列表中选择调查 。

    • 卡片将添加到仪表板，标题为“ 调查” 。

  3. 单击链接以展开其他设置。提供以下选项：

    • 输入可选的小部件标题。

    • 以秒为单位设置窗口小部件的自定义刷新率。调查窗口小部件的默认刷新率是10分钟（600秒）。

  4. 如果您的Hub实例连接到多个TeamCity服务器，请选择TeamCity服务用作窗口小部件的数据源。

  5. 单击创建窗口小部件按钮。

    • 调查窗口小部件已添加到仪表板。

设置后，将根据自定义刷新率刷新窗口小部件内容。否则，小部件每10分钟刷新一次。要手动刷新调查窗口小部件，请单击标题栏中的刷新图标。

项目状态小部件

项目状态窗口小部件显示特定项目的构建列表。

要将项目状态小部件添加到仪表板：

  1. 单击仪表板上的添加小部件按钮。

  2. 从下拉列表中选择项目状态 。

    • 带有标题为Project Status的卡被添加到仪表板。

  3. 单击链接以展开其他设置。提供以下选项：

    • 输入可选的小部件标题。

    • 以秒为单位设置窗口小部件的自定义刷新率。项目状态小部件的默认刷新率是3分钟（180秒）。

  4. 如果您的Hub实例连接到多个TeamCity服务器，请选择TeamCity服务用作窗口小部件的数据源。

  5. 从下拉列表中选择一个项目。

  6. 选择以下一个或多个选项以进一步过滤构建列表：

    • 要显示特定构建配置的构建，请从下拉列表中选择一个或多个构建配置。默认情况下，显示所有构建配置。

    • 要包括绿色构建，请启用“ 显示绿色构建”选项。否则，在窗口小部件中仅显示失败的构建。

    • 要隐藏子项目，请启用“ 隐藏子项目”选项。否则，子项目将显示在窗口小部件中。

  7. 单击创建窗口小部件按钮。

    • 项目状态窗口小部件已添加到仪表板。

    • 如果您未指定标题，则所选项目的名称将添加到卡的标题中。

设置后，将根据自定义刷新率刷新窗口小部件内容。否则，小部件每3分钟刷新一次。要手动刷新项目状态窗口小部件，请单击标题栏中的刷新图标。
