字段语法

字段请求语法基于Google Partial请求语法。以下文字适用于Google API文档中的Hub API引用。

默认情况下，服务器在处理请求后发回资源的所有立即字段。对于嵌套对象，仅基本信息（例如， id ）默认返回。为了提高性能，您可以请求服务器仅返回您真正需要的字段，并获得部分响应 。

要请求部分回复，请使用fields请求参数以指定要返回的字段。您可以将此参数用于任何返回响应数据的请求。

例

以下示例显示了fields参数和示例API。

简单要求

简单请求：此HTTP GET请求省略了fields参数并返回完整资源。

获取https://sso.jetbrains.com/api/rest/users/6109acd8-598b-44db-9e85-e99f3adca355

完整的资源响应：完整的资源数据包括以下字段以及为简洁起见而省略的许多其他字段。

要求部分回应

要求部分答复：

对相同资源的以下请求使用fields参数来显着减少返回的数据量以及获取有关嵌套对象的详细信息所需的请求数。

GET https://sso.jetbrains.com/api/rest/users/6109acd8-598b-44db-9e85-e99f3adca355?fields=name,banned,avatar/pictureUrl,groups(id,name）

部分反应

响应上述请求，服务器发回仅包含基本用户信息和命名用户组列表的响应。

请注意，响应是一个JSON对象，其中仅包含所选字段及其封闭的父对象。

接下来将介绍如何格式化fields参数的详细信息，然后是有关在响应中确切返回的内容的更多详细信息。

字段摘要参数语法

字段请求参数值的格式大致基于XPath语法。下面总结了受支持的语法，以下部分提供了其他示例。

  1. 使用逗号分隔的列表来选择多个字段。

  2. 使用a/b选择一个领域b嵌套在字段下a ;使用a/b/c选择一个领域c嵌套在b 。

  3. 使用子选择器通过将表达式放在括号中来请求一组特定的数组或对象子字段"( )" 。

  4. 如果需要，请在字段选择中使用通配符。
