获取QueryAssist

请求

GET /services/queryAssist?query=string&caret=int

参数

描述

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于服务，支持以下字段和元组：

applicationName ： serviceApplicationName —具有给定应用程序名称的服务

baseUrl ： 值 —具有给定基本URL的服务

defaultRole ： roleName —定义给定角色的服务

groupUriPattern ： value —具有给定组URI模式的服务

has ： applicationName —具有定义的applicationName的服务

具有 ： 供应商 — 已定义供应商的服务

has ： version —具有定义版本的服务

has ： homeUrl —定义了homeUrl的服务

has ： baseUrl —具有定义的baseUrl的服务

has ： userUriPattern —具有定义的用户uri的服务

has ： groupUriPattern —定义了组uri的服务

has ： redirectUri —具有定义的重定向URI的服务

具有 ： 权限 -提供某些权限的服务

has ： defaultRole —定义一些默认角色的服务

具有 ： 资源 -提供一些资源的服务

具有 ： 许可证 -带有许可证的服务

homeUrl ： 值 —具有给定家庭URL的服务

id ： value —具有给定ID的服务

是 ： 受信任 —所有受信任的服务

key ： value —具有给定密钥的服务

name ： serviceName-具有给定名称的服务

许可 ： 许可 -具有给定家庭URL的服务

redirectUri ： 值 —具有给定重定向URI的服务

resource ： resource —提供给定资源的服务

userUriPattern ： value —具有给定用户URI模式的服务

vendor ： serviceVendor —给定供应商提供的服务

version ： serviceVersion —具有给定版本的服务

caret

类型为int的查询参数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "query": string, "caret": int, "styleRanges": [queryStyleRange, ...], "suggestions": [querySuggestItem, ...] }
