Hub 2019.1帮助

创建新的仪表板

请求

POST /dashboards?fields=string

参数

描述

fields

可选的 。仅返回每个仪表板的指定字段子集。使用字段语法定义子集。

请求正文

仪表板

扩展uuid

{ "name": string, "owner": user, "data": raw, "permission": string, "access": string, "permissions": [dashboardPermission, ...], "favorite": boolean, "ordinal": int, "id": string /* from uuid */, "aliasIds": [string, ...] /* from uuid */, "aliases": [alias, ...] /* from uuid */ }

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。
