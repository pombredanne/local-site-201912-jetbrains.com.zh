要求更改密码

作为管理员，您可以要求特定用户更改其密码。该操作在用户个人资料页面上可用。

  • 此操作适用于单个用户帐户。如果要要求所有用户更改密码，则可以在集线器身份验证模块中提交密码更改请求。有关更多信息，请参阅要求中心登录需要新密码 。

  • 仅在查看其他用户的集线器帐户时，请求密码更改的选项才可见。您自己的帐户不会看到此选项。如果要更改自己的密码，请使用登录页面上的“ 重置密码”选项。

  • 密码更改请求仅要求用户更改集线器帐户登录的密码。从其他身份验证模块登录的密码不受影响。

在许多情况下，您可能考虑强迫用户更改密码。例如，您意识到用户已经与组织的其他成员共享了集线器登录的密码。您可以强制该用户更改密码，并鼓励该用户将该信息保密。

您可能还会遇到对所有内容使用相同密码的用户。如果其他服务存在重大安全漏洞，并且密码被盗，则强制更改密码可以帮助保护您的中心帐户安全。

您还可以请求所有用户更改其集线器登录名的密码。如果要要求所有用户更新其密码，请改用此选项。有关更多信息，请参阅要求中心登录需要新密码 。

要强制用户在下次登录时更改其密码，请执行以下操作：

需要权限：更新用户

  1. 从管理菜单的访问管理部分，选择用户 。

  2. 单击您要更改密码的用户帐户的全名链接。使用标题中的过滤器来查找与特定属性匹配的用户。

  3. 在页面的“ 登录名”部分中，找到Hub的登录名。

  4. 点击需要更改密码链接。

    • 密码更改请求已在集线器中注册。

    • 下次登录时，要求所选用户更改集线器帐户的密码。

    • 在用户使用新密码登录集线器帐户之前，“ 要求更改密码”选项被禁用。
