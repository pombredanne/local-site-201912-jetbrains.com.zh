JetBrains帐户验证模块

使用JetBrains帐户身份验证模块，可以从JetBrains注册产品或服务的用户使用其JetBrains帐户凭据登录集线器和已连接的服务。

启用JetBrains帐户身份验证

要使具有现有JetBrains帐户的用户登录到集线器，请启用JetBrains帐户身份验证模块。

要启用JetBrains帐户身份验证模块：

需要权限：低级管理员写

  1. 从管理菜单的访问管理部分，选择身份验证模块 。

  2. 从“ 新建模块”下拉列表中，选择“ JetBrains帐户” 。

    • JetBrains帐户身份验证模块已添加到集线器。

    • 身份验证模块已启用并可以使用。

  3. 启用模块后：

    • 配置可选设置，例如，启用注册或在用户注册新帐户时将用户添加到组中。有关更多信息，请参阅设置 。

    • 单击测试登录按钮以测试连接。在对话框中输入现有JetBrains帐户的登录名和密码。

设定值

将身份验证模块添加到集线器时，将自动配置设置。

设置

描述

类型

显示在集线器中启用身份验证的服务类型。

名称

存储认证模块的名称。使用此设置可以将该模块与“身份验证模块”列表中的其他身份验证模块区分开。

服务器网址

存储JetBrains帐户服务的服务器地址，即https://account.jetbrains.com 。

更改密码URI

存储重定向URI，该重定向URI处理更改或恢复JetBrains帐户密码的请求，即change-password 。

SSL密钥

选择一个SSL密钥以加密集线器和目录服务之间的连接。有关在Hub中管理SSL密钥的更多信息，请参见SSL密钥 。

注册

为在Hub中注册自己的用户创建帐户。如果禁用此选项，则只有具有现有集线器帐户的用户才能登录。

用户创建

为使用他们的JetBrains帐户登录的未注册用户创建中心帐户。集线器使用电子邮件地址来确定用户是否具有现有帐户。

自动加入群组

当他们在集线器中注册自己时，自动将新用户添加到组中。您可以选择一个或多个组。自动加入组的新用户将继承分配给该组的所有权限。

我们建议您将用户添加到至少一个组中。否则，仅授予新用户当前分配给“ 所有用户”组的权限。不要使用此选项将用户添加到授予不必要的敏感数据访问权限的组。

连接超时

设置等待建立与授权服务的连接的时间。默认设置为5000毫秒（5秒）。

读取超时

设置等待从授权服务读取和检索用户配置文件数据的时间。默认设置为5000毫秒（5秒）。

审计

链接到中心中的“ 审核事件”页面。在这里，您可以查看已应用于此身份验证模块的更改列表。
