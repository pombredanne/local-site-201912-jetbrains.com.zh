获取混合查询助手

请求

GET /usergroups/mixed/queryAssist?query=string&caret=int

参数

描述

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于用户组，支持以下字段和元组：

访问 （ 项目 ： 项目 ， 具有 ： 角色 ）—在给定项目中具有给定角色的组

access （ project ： project ， 具有 ： Permission ）-在给定项目中具有给定权限的组

访问 （ 项目 ： 项目 ）-在给定项目中具有某些角色的组

访问 （ 带有 ： 角色 ）-在某些项目中具有给定角色的组

访问 （ 具有 ： 权限 ）-在某些项目中具有给定权限的组

has ： ownRole —具有自己角色的组

具有 ： 用户 -与用户组

has ： 子组 —具有子组的组

has ： icon —带有图标的组

id ： value —具有给定ID的组

is ： allUsers-所有用户组

is ： autoJoin —启用了自动加入的组

is ： required2FA —具有必需的两因素身份验证的组

name ： groupName-具有给定名称的组

ownRole ： 角色 —具有给定角色作为自己角色的组

parent ： group —给定组的子组

project ： project —给定项目中的组

subgroup ： group —包含给定组作为子组的组

团队 ： 团队 -包含在给定项目团队中的组

user ： user —立即包含给定用户的组

caret

类型为int的查询参数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "query": string, "caret": int, "styleRanges": [queryStyleRange, ...], "suggestions": [querySuggestItem, ...] }
