查询语法

总览

本文档为集线器查询语法提供了BNF描述。

语法

:: = EOF :: = “或” :: = “和” :: =（“非”）？ :: = | | | | :: = “：” :: = “（” *“）” :: =“（” | EOF“）”

语法不区分大小写。

  • —任意文本。

  • —完全匹配的文字。

  • —字段名称（ID，名称等）。它涵盖了所有可用的JSON字段名+一些其他字段，例如“ is”，“ in”，“ has”等。

  • —不是单个单词值的属性值（管理员，全局，{我的全局权限}等）应该用“ {”“}”括起来

  • —按字段的条件匹配（访问（项目：“ myProject”，带有：“ myRole”））

描述

名称： admin

是： 被禁止

访问（项目： MyProject ，具有： read-issue-permission ）

查找对项目“ MyProject”具有访问权限并且对“ read-issue-permission”具有权限的所有用户。
