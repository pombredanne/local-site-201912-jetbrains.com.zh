Keyboard Shortcuts

JetBrains team tools are designed to be keyboard-centric, and Hub is no exception. Hub provides several shortcuts for navigating through the user interface.

These shortcuts are mapped to keys on a keyboard with a functional layout for US English. Keys that differ for US and UK variants are not mapped to any shortcuts. If your keyboard has a visual layout for a different language, you can switch the language preference for your operating system to English and use these shortcuts with the default mapping.

The shortcuts on this page are grouped by the context where they can be applied when you work with Hub.
