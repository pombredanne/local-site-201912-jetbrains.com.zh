Add a Service to the Services Menu

The Services menu is shown in the upper-right corner of the header. When Upsource and YouTrack are connected to your Hub service, this menu is also displayed in the header for these applications. The Services menu lets users switch between applications from a single location.

You can display links to any service that is connected to Hub, including third-party services like GitHub.

The procedures that are described on this page only apply to external Hub installations and the built-in Hub service that is bundled with an Upsource installation. Do not attempt to change these settings in a YouTrack Standalone installation that uses a built-in Hub service or a YouTrack InCloud instance.

You can change which services are displayed in the Services menu and manage who can access each service. This behavior is controlled by the Access setting on the Settings tab for each service. When you add a new service to Hub, the Access setting is empty. This means that the service is not visible to any of the users who can log in to Hub.

To change the visibility of a service in the Services menu, add groups to the Access setting. For more information, see Settings.

For example, you want all users to see YouTrack, but Upsource and TeamCity should be visible to developers only. You have a group that is named Company Developers that contains the user accounts for all of your developers.

To configure the visibility settings for this use case:

  • Add the Company Developers group to the Access setting for the Upsource and TeamCity services.

  • Add the All Users group to the Access setting for the YouTrack service.
