Uninstall Hub MSI

To uninstall Hub service from your system, you need to remove binaries and the application data.

To uninstall Hub MSI:

  1. Stop the Hub service.

  2. Use your operating system tools to verify that all Java processes that are launched by Hub are no longer running. If there are Hub-related Java processes that are still running, kill the outstanding process trees.

  3. Locate the application data directories on the host machine. You can find the actual location of those in the file:

    %programdata%\JetBrains\Hub\conf\internal\bundle.properties

  4. Uninstall the Hub application using the standard MS Windows application management tools.

  5. Delete the content of %programdata%\JetBrains\Hub directory.

  6. Delete the content of data and logs folders, if these folders are located outside of the default %programdata%\JetBrains\Hub directory. Also, make sure that these folders are not used by another Hub instance.
