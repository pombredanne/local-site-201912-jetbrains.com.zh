Uninstall Hub ZIP

To uninstall Hub service from your system, you need to remove binaries and the application data.

To uninstall Hub ZIP from the system:

  1. Stop the Hub service.

  2. Locate the application data folders. You can find actual location of data, logs, temp folders in the <hub_home>/conf/internal/bundle.properties file.

  3. Delete the <hub_home> directory. This is the directory where you unpacked the ZIP file during installation.

  4. Before you remove the application data, make sure that Hub's database and logs are not reused by another Hub instance.

  5. If data and logs folders are located outside of the hub_home directory, then delete these folders.
