获取QueryAssist

请求

GET /projectteams/queryAssist?query=string&caret=int

参数

描述

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于ProjectTeam，支持以下字段和元组：

访问 （ 项目 ： 项目 ， 具有 ： 角色 ）-在给定项目中具有给定角色的团队

access （ 项目 ： project ， 具有 ： permission ）—在给定项目中具有给定权限的团队

访问 （ 项目 ： 项目 ）—在给定项目中具有某些角色的团队

访问 （ 带有 ： 角色 ）-在某些项目中具有指定角色的团队

access （ 具有 ： 许可 ）-在某些项目中具有给定许可的团队

可访问 （ 具有 ： 许可 ）-拥有给定许可可访问的团队

group ： group —包含给定组的团队

has ： ownRole —具有自己角色的团队

具有 ： 用户 -与用户组成的团队

has ： 组 -包含组的团队

id ： value —具有给定ID的项目团队

name ： projectName —具有给定名称的项目团队

ownRole ： 角色 -具有给定角色作为自己角色的团队

project ： project —给定项目的团队

user ： user —包含给定用户的团队

caret

类型为int的查询参数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "query": string, "caret": int, "styleRanges": [queryStyleRange, ...], "suggestions": [querySuggestItem, ...] }
