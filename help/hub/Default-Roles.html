默认角色

Hub提供了一组预定义的默认角色： System Admin ， Project Admin ， Developer和Observer 。有关授予默认角色的默认权限的比较 ，请参阅默认角色的权限比较 。

您不能删除系统管理员 ， 项目管理员或开发人员角色。但是，您可以更新分配给这些角色的权限集，更改其名称并修改其描述。

如果您要创建一组可用于组或用户帐户的自定义权限，则可以创建新角色或编辑预定义角色 。

系统管理员

系统管理员角色供负责管理集线器安装的用户使用。

该角色被分配了中心中的所有可用权限。

项目管理员

专案管理员角色专供管理专案的使用者使用。默认情况下，以下权限分配给此角色：

  • 阅读项目基础
  • 阅读项目全文
  • 更新专案
  • 阅读角色
  • 创建用户
  • 阅读用户基本
  • 阅读自我
  • 更新自我
  • 创建组
  • 读组
  • 更新组
  • 删除群组

开发者

开发人员角色供项目团队成员的用户使用。默认情况下，此角色分配有以下权限：

  • 阅读项目基础
  • 阅读自我
  • 更新自我

观察者

观察者角色供监视项目进度的用户使用。基本上，此角色启用了对项目的只读访问。默认情况下，此角色分配有以下权限：

  • 阅读用户基本
  • 阅读自我
  • 更新自我
