创建和编辑角色

如果要创建可用于组或用户帐户的自定义权限集，则可以创建新角色或编辑分配给现有角色的权限。

如果要创建与现有角色相似但不相同的角色，请改用“ 克隆”操作来创建该角色。有关更多信息，请参见克隆角色 。

创建角色

如果要提供对用户或组的特定访问权限，但又不想修改任何预定义角色，请创建一个新角色。

要创建新角色：

需要权限：创建角色，读取服务

  1. 在管理菜单的访问管理部分，选择角色 。

  2. 单击新角色按钮。

  3. 输入新角色的名称，然后单击“ 创建”按钮。

    • 新角色的设置页面将打开“ 常规”选项卡。

  4. 输入新角色的可选说明。

  5. 要配置角色可用的权限集，请选择“ 权限”选项卡。启用要分配给角色的每个权限。有关“ 权限”选项卡的详细说明，请参阅“ 管理权限” 。

    • 所有对权限的更改都会自动应用并保存。

一旦创建并配置了所有要使用的角色，就可以创建或配置用户组 ，并在每个项目的基础上为其分配角色 。

编辑现有角色

如果您希望为已经分配了特定角色的用户启用特定权限，则可以修改现有角色。

编辑现有角色：

需要权限：更新角色，读取服务

  1. 在管理菜单的访问管理部分，选择角色 。

  2. 从列表中选择一个角色。

  3. 在设置页面中，编辑名称或描述。

  4. 要配置角色可用的权限集，请选择“ 权限”选项卡。启用或禁用角色的特定权限。有关“ 权限”选项卡的详细说明，请参阅“ 管理权限” 。

    • 所有对权限的更改都会自动应用并保存。

您可以随时将默认角色恢复为其默认权限。有关更多信息，请参见重置服务的默认权限 。
