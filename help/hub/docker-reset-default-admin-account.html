重置默认管理员帐户

首次安装Hub时，请设置默认管理员帐户的登录名和密码。此帐户在全局项目中被分配了系统管理员角色。

如果您忘记了默认管理员帐户的凭据，或者继承了另一位管理员配置的安装，则可以将该帐户的登录名和密码重置为其默认值。您还可以在还原帐户时为默认管理员设置系统属性。

当启用了双重身份验证并且默认管理员不再有权访问与该帐户配对的身份验证应用程序时，您也可以使用此过程。将默认管理员帐户重置为默认状态也会禁用两因素身份验证。

如果在Docker容器中运行Hub，请使用以下过程。

此过程中使用的命令适用于Linux和macOS。对于Windows安装，请使用反斜杠。

重置默认管理员帐户凭据：

  1. 使用以下命令停止Hub：

    docker exec 停止

  2. 跑过configure-next-start命令：

    docker运行--rm -it -v ：/ opt / hub / conf \ jetbrains / hub： \ configure-next-start \ --J-Djetbrains.hub.admin.restore = true

    确保在输入之前输入两个破折号J选项。此方法仅设置当前运行的系统属性，并且不会更新配置文件。管理员帐户凭据不会为以后重新启动集线器服务而重置。

    如果要为默认管理员帐户设置任何系统属性，请将它们添加到命令中。有关系统属性的列表，请参见“默认管理员帐户”的“系统属性”部分。

  3. 运行docker容器：

    docker容器启动

    • 集线器服务开始使用指定的设置。

    • 默认管理员凭据将重置为其默认值或命令中指定的值。

  4. 使用默认管理员帐户登录到Hub。

  5. 如果需要，请更改帐户密码。

默认管理员帐户的系统属性

属性

描述

类型

jetbrains.hub.admin.restore

将默认的“ admin”帐户及其权限重置为默认方案。如果禁用了内置授权，则使用此参数启动集线器服务将启用它。

布尔型

jetbrains.hub.admin.uuid

设置默认管理员帐户的UUID。仅当与jetbrains.hub.admin.restore 。

串

jetbrains.hub.admin.name

设置默认管理员帐户的全名。仅当与jetbrains.hub.admin.restore 。

串

jetbrains.hub.admin.login

设置默认管理员帐户的登录名。仅当与jetbrains.hub.admin.restore 。

串

jetbrains.hub.admin.email

设置默认管理员帐户的电子邮件地址。仅当与jetbrains.hub.admin.restore 。

串

jetbrains.hub.admin.password

设置默认管理员帐户的密码。仅当与jetbrains.hub.admin.restore 。

串
