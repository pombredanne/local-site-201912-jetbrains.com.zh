搜索查询语法

本页提供有关集线器查询语法的BNF描述。

语法

:: = EOF :: = “或” :: = “和” :: =（“非”）？ :: = | | | | :: = “：” :: = “（” *“）” :: =“（” | EOF“）”

语法不区分大小写。

  • —任意文本。

  • —要匹配的确切文字短语。

  • —字段名称。例如， id ， name ， login ， 等等。这涵盖了所有可用的JSON字段名称，以及许多其他名称，例如“ is”，“ in”，“ has”等。

  • —属性值。例如， admin ， global ， {my global permission} 。将包含几个单词的每个值括在大括号中{ } 。

  • —按字段的条件匹配。例如， access(project: "myProject", with: "myRole") 。

样品

询问

描述

name: admin

返回所有具有visibleName “管理员”。

is: banned

返回所有禁止用户的列表。

access(project: MyProject, with: {Read Auth Module})

返回可以访问权限为“ Read Auth Module”的项目“ MyProject”的用户列表。
