创建新的小部件

请求

POST /widgets?fields=string

参数

描述

fields

可选的 。仅返回每个小部件的字段的指定子集。使用字段语法定义子集。

请求正文

小部件

含义

200 OK

成功的请求。

400错误的要求

403禁止

请求者无权访问所请求的资源。
