ZIP安装

可以从JetBrains网站上下载Hub ZIP发行版。

Hub ZIP是跨平台发行版。您可以使用此发行版在Linux，macOS和Windows上安装和运行Hub。

对于所有操作系统，使用此发行版安装Hub的过程都是相似的。您需要做的就是选择适当的应用程序，然后运行操作系统支持的脚本。

此页面上的所有命令均用于在Linux和macOS上安装。如果要在Windows服务器上安装集线器，请更换.sh与.bat并使用反斜杠。

要使用ZIP发行版安装Hub：

  1. 从JetBrains网站下载ZIP发行版。

  2. 在服务器上创建一个目录，用作Hub的主目录 。从ZIP存档中提取文件，然后选择此目录作为目标文件夹。

  3. 在命令行界面中，将目录更改为/bin 。

  4. 输入命令以开始安装：

    • 如果输入./hub.sh run ，控制台将被阻止，直到服务停止。

    • 如果输入./hub.sh start ，仅在启动过程中阻止控制台。

    如果您的主机上没有安装Web浏览器，请按照以下命令操作--no-browser选项。例如， ./hub.sh start --no-browser 。

    要查看所有可用的命令选项，请输入./hub.sh help 。

  5. 在基于Web的配置向导中继续安装。

    • 如果您在主机上安装了浏览器，则配置向导将自动打开。

    • 如果主机上未安装Web浏览器，请在另一个工作站上的Web浏览器中打开向导的URL。向导URL打印在控制台输出中：

      启动后， JetBrains Hub 将在[ / bundle / starting]上可用

  6. 在配置向导中 ，单击“ 设置” 。

  7. 在“ 确认设置”页面上，确认或更改基本系统设置。在这里，您还可以启用内置TLS，以保护服务器的网络连接。有关详细过程，请阅读“ 在安装过程中启用HTTPS”页面。

    设置

    描述

    HTTP | HTTPS

    允许启用或禁用内置TLS并配置与服务器的安全网络连接。如果启用HTTPS，则会显示TLS特定设置。

    有关详细信息，请参见“ TLS属性”页面。

    基本网址

    最终用户访问集线器的URL。

    应用监听端口

    安装后，集线器服务器侦听的端口。

    语言

    用户界面中显示的默认语言。

  8. 展开“ 高级设置”部分，以确认或更改集线器存储产品数据的位置。

    设置

    描述

    数据目录位置

    集线器存储内容数据的目录。

    备份位置

    集线器存储数据库备份的目录。

    日志位置

    集线器存储日志文件的目录。

    临时位置

    集线器存储特定于产品的临时文件的目录。

    要更改任何这些文件类型的位置，请单击“ 更改”按钮，然后选择一个新位置。

  9. 要与JetBrains共享使用情况统计信息，请保持启用“ 匿名发送使用情况统计信息”选项。此功能可帮助JetBrains跟踪使用情况统计信息，以使Hub更好。我们绝不与任何第三方共享收集的数据。

  10. 完成后，单击“ 下一步” 。

  11. 在“ 配置中心服务”页面上，输入并确认系统管理员帐户的密码。默认用户名是admin 。

    要将其他用户名分配给管理员帐户，请编辑“ 创建管理员登录名”设置。

    （可选）取消选择“ 启用访客身份登录”选项以禁止访客用户帐户。

  12. 完成后，单击“ 下一步” 。

  13. 在“ 确认许可证”页面上，验证许可证名称和密钥。

  14. 完成后，点击完成 。

    • 该配置将应用于您的集线器服务器。

    设置完成之前，请勿在浏览器中关闭页面。集线器服务器准备就绪时：

    • 如果您禁用guest用户帐户，您将被重定向到登录页面。

    • 如果启用guest用户帐户，您将被重定向到仪表板。

而已。您的集线器服务已安装并可以使用。
