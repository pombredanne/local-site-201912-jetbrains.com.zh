管理用户许可证

在配置文件页面上，您可以为用户授予服务许可，以限制可以使用该应用程序的用户数量。

授予许可证：

需要权限：更新用户

  1. 打开用户的个人资料页面。

  2. 单击“ 许可证”部分中的“ 授予许可证”按钮。

    • 显示“ 授予许可证”对话框。

  3. 在“ 对于服务”下拉列表中，选择服务的名称。您可以使用输入字段来过滤服务列表。

  4. 单击授予许可证按钮。

    • 所选服务的许可证已授予用户。

吊销许可证：

需要权限：更新用户

  1. 单击列表中目标许可证的“ 吊销许可证”按钮。

  2. 在显示的对话框中，确认操作。

    • 许可证从选定的用户和选定的组的成员中撤消。

    • 服务页面上“ 可用许可证”设置中的许可证数量已更新。
