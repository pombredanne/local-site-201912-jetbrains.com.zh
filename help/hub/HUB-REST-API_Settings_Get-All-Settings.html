Hub 2019.1帮助

获取所有设置

请求

GET /settings?fields=string&$skip=int&$top=int&query=string

参数

描述

fields

可选的 。仅返回每个设置的字段的指定子集。使用字段语法定义子集。

$skip

可选的 。可让您设置一些要返回第一个设置之前要跳过的设置。

$top

可选的 。使您可以设置要返回的最大设置数。

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于设置，支持以下字段和元组：

id ： value —具有给定ID的设置

类型 ： 许可证 -获取许可证设置

类型 ： 公共 —获取公共设置

类型 ： 内部 —获取内部设置

类型 ： smtp —获取smtp设置

类型 ： jabber —获取jabber设置

类型 ： 备份 —获取jabber设置

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "skip": int, "top": int, "total": int, "settings": [settings, ...] }
