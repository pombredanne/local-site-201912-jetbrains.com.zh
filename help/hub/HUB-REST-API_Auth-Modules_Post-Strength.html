Post Strength

Request

POST <REST API URL>/authmodules/password/strength?user=string&username=string

Parameter

Description

user

Query parameter of type string.

username

Multivalued query parameter of type string.

Request Body

plainpassword

extends password

{ "value": string, "oldValue": string /* from password */ }

Response

Response Code

Meaning

200 OK

Successful request.

400 Bad Request

At least one of the request parameters is invalid. For example, a required field in the passed JSON object is missing. For details, check the error message in the response.

403 Forbidden

The requester has no access to the requested resource.

404 Not Found

The requested resource was not found.

500 Internal Server Error

Failed to process request because of the server error. For details, check the error message in the response.

Response body

{ "referenceScore": int, "score": int, "maxScore": int, "referenceEntropy": int, "entropy": int, "maxEntropy": int, "feedbackMessage": string }
