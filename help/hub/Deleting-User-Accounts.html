删除用户帐号

如果集线器中的用户帐户不再需要访问权限，则可以将其从集线器中删除。

删除用户

从集线器删除用户帐户时，系统会要求您选择替代用户。

  • 替换用户继承在连接的服务中设置为已删除用户的引用。例如，如果在YouTrack中将已删除的用户设置为问题的报告者 ，则“ 报告者”值将设置为替换用户。

  • 替换用户不继承已删除用户帐户的凭据。已删除用户的凭据将从集线器中删除。删除的用户将无法再登录到集线器或任何连接的服务。

删除一个或多个用户：

需要权限：删除用户

  1. 在管理菜单的访问管理部分，选择用户 。

  2. 在列表中选择一个或多个用户。使用过滤器查找与特定属性匹配的用户。

  3. 单击工具栏中的删除按钮。

    • 将打开“ 删除用户”对话框。

  4. 在“ 替换用户”列表中，选择一个用户帐户。

  5. 单击删除按钮。

    • 所选用户将从集线器中删除。

    • 删除的用户无法再登录到集线器或任何连接的服务。

    • 替换用户继承设置为已删除用户的所有引用。

您也可以从用户个人资料中删除单个用户帐户。

要删除单个用户帐户：

需要权限：删除用户

  1. 打开用户个人资料。

  2. 单击删除用户按钮。

    • 将打开“ 删除用户”对话框。

  3. 在“ 替换用户”下拉列表中，选择一个用户帐户。

  4. 单击删除用户按钮。

    • 所选用户将从集线器中删除。

    • 删除的用户无法再登录到集线器或任何连接的服务。

    • 替换用户继承设置为已删除用户的所有引用。

帐户删除的替代方法

删除帐户不是唯一的选择。您也可以禁止一个帐户或将其合并到另一个帐户 。

合并用户帐户时，请记住以下要点：

  1. 合并的结果是，所有角色，组成员身份和访问权限都将转移到单个用户帐户中。

  2. 如果将任何合并和删除的用户帐户设置为在连接的服务中具有特定角色，则将这些值替换为结果用户帐户。例如，在关联的YouTrack项目中将用户设置为受让人，记者或评论者时，或在关联的Upsource项目中将用户设置为审阅者时。

  3. 当您将具有较少权限的帐户与具有更多权限的帐户合并时，合并的用户帐户将获得更高级的权限。

在某些情况下，您可能想阻止访问而没有禁止，删除或合并该帐户。为此，请删除用户的登录凭据。

删除用户帐户的登录凭据：

需要权限：更新用户

  1. 在管理菜单的访问管理部分，选择用户 。

  2. 在用户列表中，选择要删除的帐户以及要将其合并到的帐户。

  3. 合并两个帐户 。

  4. 打开合并帐户的管理员配置文件。

  5. 在与该帐户关联的凭据列表中，删除要从系统中删除的用户的凭据。
