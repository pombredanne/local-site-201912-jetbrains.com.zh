更改MSI的数据库位置

集线器存储数据的默认目录包含以下子目录：

名称

描述

hub

包含中心服务的数据库

internal

可能包含其他内部产品数据

将现有数据库复制到新位置时，请确保新位置使用此处显示的相同文件夹结构。

您可以使用以下过程来更改MSI分发的数据库位置。要在MSI分发中执行这些命令，请以管理员身份打开“ 命令提示符”窗口。

要更改数据库位置：

  1. 备份数据库。以防万一。

  2. 停止集线器服务。有关特定说明，请参阅启动和停止Hub MSI 。

  3. 将现有数据库移到新数据库目录。确保本地用户JetBrainsHub拥有对新数据目录及其所有内容的完全访问权限。

  4. 输入以下内容configure命令来设置新数据目录的位置：

    bin \ hub.bat配置--data-dir =

    对于MSI安装， \bin目录是MSI分发的安装位置。默认安装目录为C:\Hub

  5. 启动集线器服务。有关特定说明，请参阅启动和停止Hub MSI 。

其他目录

如果将数据库移动到另一台服务器，则可能还需要更改集线器存储日志和临时文件的位置。

要更改这些文件的位置，请遵循描述的更改数据库位置的相同步骤，并在脚本文件中执行以下命令：

目录

命令

后备

 configure --backups-dir=

日志

 configure --logs-dir=

温度

 configure --temp-dir=
