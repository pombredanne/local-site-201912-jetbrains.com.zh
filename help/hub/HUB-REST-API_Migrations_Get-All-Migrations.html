Hub 2019.1帮助

获取所有迁移

请求

GET /migrations?fields=string&$skip=int&$top=int

参数

描述

fields

可选的 。仅返回每次迁移的字段的指定子集。使用字段语法定义子集。

$skip

可选的 。使您可以设置许多迁移，以在返回第一个迁移之前跳过。

$top

可选的 。使您可以设置要返回的最大迁移数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "skip": int, "top": int, "total": int, "migrations": [migrations, ...] }
