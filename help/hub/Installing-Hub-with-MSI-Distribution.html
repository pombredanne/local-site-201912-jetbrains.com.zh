MSI安装

Hub MSI发行版可从JetBrains网站上下载。

以下软件与Hub MSI发行版捆绑在一起：

  • Jetty Servlet引擎和HTTP服务器

  • Java SE运行时环境8 x64

MSI发行版不包括JRE x86。如果要在x86环境中安装Hub，则需要下载并安装与您的操作系统兼容的Java Runtime Environment。

MSI安装程序需要.NET Framework 2.0或更高版本。

要将集线器安装为Windows服务：

  1. 从JetBrains网站下载MSI发行版。

  2. 打开Windows Installer程序包（MSI文件）。

    • 将打开“ JetBrains集线器设置向导” 。

  3. 请按照安装向导中的说明进行操作。

    • 设置向导完成后，基于Web的配置向导将在默认浏览器中打开。

      如果主机上未安装Web浏览器，请从设置向导复制URL，然后在具有Web浏览器的工作站上打开页面。

    安装向导可以检测到以前安装的集线器中的数据。有关更多信息，请参见找到先前的安装数据 。

  4. 单击完成按钮以关闭安装向导。

  5. 切换到Web浏览器，然后在基于Web的配置向导中继续进行安装。

  6. 在配置向导中 ，单击“ 设置” 。

  7. 在“ 确认设置”页面上，确认或更改基本系统设置。

    设置

    描述

    基本网址

    最终用户访问集线器的URL。

    应用监听端口

    安装后，集线器服务器侦听的端口。

    语言

    用户界面中显示的默认语言。

  8. 展开“ 高级设置”部分，以确认或更改集线器存储产品数据的位置。

    设置

    描述

    数据目录位置

    集线器存储内容数据的目录。

    备份位置

    集线器存储数据库备份的目录。

    日志位置

    集线器存储日志文件的目录。

    临时位置

    集线器存储特定于产品的临时文件的目录。

    要更改任何这些文件类型的位置，请单击“ 更改”按钮，然后选择一个新位置。

  9. 要与Jetbrains共享使用情况统计信息，请保持启用匿名发送使用情况统计信息选项。此功能可帮助JetBrains跟踪使用情况统计信息，以使Hub更好。我们绝不与任何第三方共享收集的数据。

  10. 完成后，单击“ 下一步” 。

  11. 在“ 配置中心服务”页面上，输入并确认系统管理员帐户的密码。默认用户名是admin 。

    要将其他用户名分配给管理员帐户，请编辑“ 创建管理员登录名”设置。

    （可选）取消选择“ 启用访客身份登录”选项以禁止访客用户帐户。

  12. 完成后，单击“ 下一步” 。

  13. 在“ 确认许可证”页面上，验证许可证名称和密钥。

  14. 完成后，点击完成 。

    • 该配置将应用于您的集线器服务器。

    设置完成之前，请勿在浏览器中关闭页面。集线器服务器准备就绪时：

    • 如果您禁用guest用户帐户，您将被重定向到登录页面。

    • 如果启用guest用户帐户，您将被重定向到仪表板。

而已。您的Hub实例已安装并可以使用。

找到先前的安装数据

在安装过程中，安装向导可以检测到先前安装的集线器中的数据。这些文件可能是由另一个用户卸载或安装的安装遗留的。

您可以从以下选项中选择以继续安装：

  • 执行全新安装。

  • 使用配置文件和先前安装中剩余的数据来安装Hub。

除非您熟悉以前的安装的详细信息，否则我们建议您执行全新安装。先前安装版本中的文件将被删除。
