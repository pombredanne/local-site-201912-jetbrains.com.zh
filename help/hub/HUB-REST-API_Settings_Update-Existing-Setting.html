更新现有设置

请求

POST /settings/{setting id}

请求正文

设定

由backupSettings ， importSettings ， internalSettings ， jabberSettings ， license ， publicSettings ， smtpSettings ， throttlingSettings 扩展

{ "id": string, "aliasIds": [string, ...], "aliases": [alias, ...] }

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。
