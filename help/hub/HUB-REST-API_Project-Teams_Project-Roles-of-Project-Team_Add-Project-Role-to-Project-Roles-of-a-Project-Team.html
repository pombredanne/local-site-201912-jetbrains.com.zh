Add Project Role to Project Roles of a Project Team

Request

POST <REST API URL>/projectteams/{project team id}/projectroles?fields=string

Parameter

Description

fields

Optional. Returns only the specified subset of the fields for each ProjectRole. Use Fields Syntax to define the subset.

Request Body

projectRole

extended by sourcedProjectRole

{ "id": string, "aliasIds": [string, ...], "aliases": [alias, ...], "role": role, "project": project, "owner": authorityHolder, "teamMember": boolean }

Response

Response Code

Meaning

Successful request.

400 Bad Request

403 Forbidden
