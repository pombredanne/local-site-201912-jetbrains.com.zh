为MSI配置网络接口绑定

默认情况下，集线器中使用的Jetty servlet引擎和HTTP服务器绑定在0.0.0.0网络接口。出于安全原因，在某些环境中可能会限制此操作。

您可以使用configure命令更改listen-address集线器服务器的属性。代表运行集线器服务的同一OS用户执行此命令。此命令创建配置文件和文件夹。集线器服务用户必须有权访问此命令创建的文件和文件夹。

要为MSI分发自定义网络接口绑定，请执行以下操作：

  1. 停止集线器服务 。

  2. 以管理员身份打开“ 命令提示符”窗口。

  3. 在命令行中，将目录更改为\bin 。

    默认安装目录位置是C:\Hub 。

  4. 输入以下命令：

    hub.bat配置--listen-address =

  5. 启动集线器服务 。
