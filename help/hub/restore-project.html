Restore a Project

Unlike deleting a project, archiving is a reversible operation. You can easily restore an archived project when you need to.

To restore a project:

Requires permissions: Update Project

  1. Click the Projects link in the header to open the Projects list.

  2. Click the project link in the Archived section at the bottom of the list.

    • The Edit Project page for the selected project is displayed.

  3. Open the Projects page and select the name of an archived project.

  4. Click the Restore button in the header.

    • The archived project is restored.
