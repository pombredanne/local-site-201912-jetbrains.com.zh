Hub 2019.1帮助

登录到集线器

集线器支持两种授权方法，可用于登录和访问外部客户端的数据。

方法

描述

永久令牌授权

永久令牌为脚本和应用程序中的REST API调用启用基于令牌的授权。此功能提高了集线器与外部服务集成的安全性。

我们建议您使用永久令牌对Hub REST API进行授权。

OAuth 2.0授权

OAuth 2.0是一种协议，使您的应用程序可以从Hub请求访问令牌。集线器支持一系列授权流程。您在客户端应用程序中使用的流程取决于客户端可用的信息以及客户端从Hub访问资源的位置（从Web浏览器或基于服务器的客户端应用程序）。

当您的应用程序需要客户端身份验证时，请使用OAuth 2.0。
