Hub 2019.1帮助

获取所有证书

请求

GET /certificates?fields=string&$skip=int&$top=int&query=string&orderBy=string&disabled=boolean

参数

描述

fields

可选的 。仅返回每个证书的字段的指定子集。使用字段语法定义子集。

$skip

可选的 。可让您设置要返回的第一个证书之前要跳过的证书数量。

$top

可选的 。使您可以设置要返回的最大证书数。

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于证书，支持以下字段和元组：

id ： value —具有给定ID的可信SSL证书

is ： 启用 -启用证书

name ： certificateName —具有给定名称的可信SSL证书

orderBy

可选的 。返回按指定字段排序的证书列表。有关详细信息，请参见排序语法 。证书支持按以下字段排序： 名称

disabled

布尔类型的查询参数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "skip": int, "top": int, "total": int, "certificates": [certificate, ...] }
