获取QueryAssist

请求

GET /roles/queryAssist?query=string&caret=int

参数

描述

query

可选的 。用于指定搜索查询以过滤返回的结果。有关支持的逻辑操作语法的定义，请参见查询语法。对于角色，支持以下字段和元组：

具有 ： 权限 -具有某些权限的默认角色

id ： value —具有给定ID的服务默认角色

key ： value —具有给定密钥的默认角色

name ： value —具有给定名称的默认角色

权限 ： 权限 -包含给定权限的默认角色

具有 ： 权限 -具有某些权限的角色

id ： value —具有给定ID的角色

key ： roleKey —具有给定密钥的角色

name ： roleName-具有给定名称的角色

权限 ： 权限 -包含给定权限的角色

权限 （ 键 ： permissionKey ， 服务 ： service ）—包含给定权限的角色

许可 （ 服务 ： 服务 ）—至少具有一项服务许可的角色

caret

类型为int的查询参数。

响应

回应码

含义

200 OK

成功的请求。

400错误的要求

至少一个请求参数无效。例如，缺少传递的JSON对象中的必填字段。有关详细信息，请检查响应中的错误消息。

403禁止

请求者无权访问所请求的资源。

找不到404

找不到请求的资源。

500内部服务器错误

由于服务器错误，无法处理请求。有关详细信息，请检查响应中的错误消息。

反应体

{ "query": string, "caret": int, "styleRanges": [queryStyleRange, ...], "suggestions": [querySuggestItem, ...] }
