排序语法

本文档为集线器排序语法提供了BNF描述。

语法

:: = + EOF; :: = （“：”（“ asc” |“ desc”））？

语法不区分大小写。

  • —排序依据的字段名称。例如， id ， email ， name ， 等等。

支持的字段和排序顺序

支持排序的字段列表取决于请求的资源。反过来，每个字段都有其自己的默认排序顺序。您可以在对所需资源的“全部获取”请求的响应中接收默认的排序顺序。

名称

按默认顺序按字段“名称”排序。

名称：asc

按字段名称升序排序。

组名：desc

首先按“组”排序，然后对具有相同组的条目按名称按降序排序。
