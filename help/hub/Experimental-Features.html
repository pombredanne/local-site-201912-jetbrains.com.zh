Hub 2019.1帮助

实验特征

在2017年，我们转向了较短的发布周期。与其长时间发布主要版本，不如更频繁地构建，测试和发布更新。

通过连续交付，我们不再需要支持“抢先体验”计划。相反，我们使用该选项在每个版本中包含实验功能。这些功能处于开发的早期或中期。这种策略使我们能够在使用某个功能时获得有关功能的反馈，从而更多地了解您希望在最终产品中看到的内容。

要使用和测试新功能，可以为集线器安装启用实验性功能。如果您是管理员，只需在“ 功能配置”页面上启用要尝试的功能 。此页面显示了可以在系统范围内启用的功能列表，以及可以为特定组启用的其他功能。

要访问此页面，请从“ 管理”菜单的“ 服务器设置”部分中选择“ 系统设置 ” ，然后单击页面底部的“ 打开功能配置页面”链接。

使用实验功能时，请注意以下注意事项：

  • 实验功能可以包括已知和未发现的错误。

  • 支持该功能的API也正在开发中，并且可能会发生变化。

  • 该文档可能不完整，也可能包含错误。

如果遇到任何错误，请随时报告Hub项目中的问题 。

启用实验功能

在用户可以访问和使用实验性功能之前，您需要在集线器服务中启用该功能。

启用实验功能

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 系统设置” 。

  2. 滚动到页面底部，然后单击打开功能配置页面链接。

    • 将打开“ 功能配置”页面。

  3. 在页面上找到要启用的功能。

    • 对于在系统范围内启用的功能，请选择功能，然后单击工具栏中的“ 启用”按钮。

    • 对于为特定组启用的功能：

      1. 选择功能。

      2. 单击选择组按钮。

      3. 在显示的对话框中，选择应为其启用功能的组。

      4. 单击保存按钮。

    —该功能已在集线器中激活并可以使用。

当前功能

默认情况下，以前以实验为基础提供的所有功能均已提供。当前版本中没有实验功能。
