为Docker配置网络接口绑定

默认情况下，集线器中使用的Jetty servlet引擎和HTTP服务器绑定在0.0.0.0网络接口。出于安全原因，在某些环境中可能会限制此操作。

如果您在Docker容器中运行Hub服务，则无需更改listen-address 。但是，您仍然可以使用configure命令更改listen-address属性。

要为Docker映像自定义网络接口绑定，请执行以下操作：

  1. 停止Hub docker容器 ：

    docker exec 停止

  2. 跑过configure命令：

    docker运行--rm -it -v ：/ opt / hub / conf \ -v ：/ opt / hub / logs \ jetbrains / hub： \ configure --listen-address =

  3. 启动Hub docker容器：

    码头工人开始
