Users

In the Users area in Hub, you can manage accounts for users and their access rights to connected services. The following operations are available:

  • Create User Accounts
  • Merge User Accounts
  • Ban User Accounts
  • Configure User Account Settings
  • Delete User Accounts
  • User Profile

To access the Users page in Hub, click the Users link in the main menu.
