System Settings

On this page:

  • System Language
  • SMTP Settings
  • Jabber Settings

This page describes how to administer the basic settings of JetBrains Hub, including the configuration of email and Jabber notifications.

To access the system settings page, select System Settings in the Server Settings section of the Administration menu.

System Language

Out of the box, the Hub user interface is available in English (default), French, German, Spanish and Russian.

To change the language on the user interface:

  1. Select the language you prefer in the System Language drop-down list.
  2. Click the Save button.

    The change is applied immediately.

SMTP Settings

Here, you can configure SMTP connection to enable Hub sending messages to its users by email.

Enable email!

If the SMTP connection is disabled or configured incorrectly, your users do not receive any messages from Hub. This includes invitations to register an account, contact verification emails, and password recovery instructions!

To enable email notifications:

  1. Click the Enable Email button
  2. Enter values for the following settings:
    SettingDescription
    HostEnter the server address of your SMTP server.
    PortEnter the connection port for the SMTP server.
    ProtocolSelect the protocol to use for your SMTP server (either with or without SSL or TLS support).
    SSL KeySelect a previously uploaded SSL key or trusted certificate for secure connection to your mail server. For details, see SSL Key Stores.
    FromEnter an email address to use as the 'From' address for email notifications.
    UsernameEnter the username of the account that is used to access the SMTP server.
    PasswordClick the Change password button to specify the password for account that is used to access the SMTP server.
  3. Test the connection.
    • Click the Send Test Message button.
    • Enter an email address to which the test message should be sent.
    • Click the Send button.

Jabber Settings

Here, you can configure the connection to a Jabber server. When configured, Hub can send direct messages to users who are registered to use the Jabber service.

To enable Jabber notifications:

  1. Click the Enable Jabber button
  2. Enter values for the following settings:
    SettingDescription
    HostEnter the server address of your Jabber service.
    Service NameEnter the service name of your Jabber service provider.
    PortEnter the connection port for the Jabber server.
    UsernameEnter the username of the account that is used to access the Jabber server.
    SASL AuthenticationEnable this option if required by the Jabber service provider. This option is disabled by default.
    PasswordClick the Change password button to specify the password for account that is used to access the Jabber server.
  3. Test the connection.
    • Click the Send Test Message button.
    • Enter an Jabber account to which the test message should be sent.
    • Click the Send button.
