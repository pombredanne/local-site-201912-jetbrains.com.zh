Quick Navigation

With the new global navigation (aka Go To) feature in Hub, you can swiftly navigate to any page that is available for you. Just open the global navigation dialog, and start entering the name of an entity that you wish to view.

To invoke the global navigation, press the Ctrl+K or Cmd+K (macOS) shortcut or the icon in the main menu.

  • Queries are case-insensitive and take into account the order of words.

  • Only those entities to which you have the access are displayed in the list.

  • If you see the N more mark for a category title, it means that you might want to adjust your query to get more precise results.
