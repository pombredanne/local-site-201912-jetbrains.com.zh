将凭据添加到用户配置文件

您可以将任何一组凭据添加到用户配置文件中作为中心登录。如果用户删除或禁用了现有登录名，则可以为该用户提供一组新的凭据，并将其手动添加到用户配置文件中。然后，用户可以使用新凭据登录集线器，但是要求他们立即更改密码。

将凭据添加到用户配置文件时，集线器会根据现有集线器登录名列表和存储在连接的身份验证模块中的凭据来检查这些凭据。

  • 如果找不到匹配项，则可以选择将这些凭据作为新的集线器登录名添加到当前配置文件中。要成功添加这些凭据，密码必须满足为集线器身份验证模块设置的密码强度要求。

  • 如果集线器在连接的身份验证模块中找到了属于另一个用户的现有集线器凭据或匹配的凭据，则无法将这些凭据添加到当前帐户。如果您确定这些凭据属于当前用户帐户，请使用合并功能合并两个帐户。

要将凭证添加到用户配置文件：

需要权限：更新用户

  1. 打开用户个人资料。

  2. 在页面的“ 登录”部分中，单击“ 添加凭据”按钮。

    • 将打开“ 添加凭据”对话框。

  3. 输入要添加到此帐户的用户名/电子邮件地址和密码。

  4. 单击验证凭据按钮。

    • 集线器检查数据库中是否有使用这些凭据的集线器帐户和身份验证模块。

    • 该对话框显示从Hub数据库返回的结果。

    • 如果未找到匹配项，请启用将凭据另存为集线器登录选项。

  5. 单击添加凭据按钮。

    • 所选凭据将作为登录名添加到当前用户配置文件。
