建立专案

在“ 项目”页面上，可以快速有效地创建新的Hub项目。

要创建一个新项目：

需要权限：创建项目，阅读项目基础

  1. 单击主菜单中的“ 项目”链接。

  2. 单击标题中的“ 新建项目”按钮。

  3. 在名称和键输入字段中输入项目的名称和键 。名称和密钥对于Hub都必须唯一。

    项目密钥是Hub中项目的易于识别的标识符。在连接的服务中创建项目时，集线器中的项目密钥用作相关项目ID的前缀。

  4. 在“ 描述”输入字段中输入项目的可选描述。

  5. 单击创建项目按钮。

    • 该项目将在项目概述页面中打开。

创建项目后，可以将服务添加到项目中 。
