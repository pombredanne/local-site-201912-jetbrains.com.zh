系统设置

通过“ 系统设置”页面，您可以管理JetBrains Hub的服务器范围的设置。

要访问此页面，请在“ 管理”菜单的“ 服务器设置”部分中选择“ 系统设置 ” 。

设置

描述

系统语言

设置在集线器中显示的服务器范围的系统语言。集线器用户界面支持英语（默认），法语，德语，日语，西班牙语和俄语。

还可以使用YouTrack用户社区支持的其他语言。这些语言在列表中带有社区标签。有关更多信息，请参见基于社区的语言支持 。

唯读模式

将集线器数据库置于只读模式。在只读模式下，集线器继续为连接的服务中的用户提供访问权限，但不接受更改。如果您遇到集线器问题，则可以在调查和隔离问题的同时激活只读模式。

公司标志

设置在集线器登录页面上显示的图像。

匿名方案

确定匿名用户帐户时如何转换个人数据。

  • 与散列选项，用户数据是通过用SHA-256哈希值替换它假名。虽然假名化数据不再以人类可读的状态存储，但仍可以将哈希值与原始值链接在一起。使用此选项时，当您在Hub中向用户提供用户协议或发布描述您如何管理个人数据的法律文档时，请考虑向用户披露此信息。

  • 使用随机选项，个人数据将替换为随机值。这种匿名化不可逆转地破坏了识别数据主体的任何方式。随机数据无法恢复为可读状态。

记录事件类别

启用特定事件类别的日志记录。该设置支持DEBUG和TRACE日志记录级别。使用此设置可以解决集线器安装问题，而不必重新启动服务。

提供最有价值信息的事件类别因安装而异。集线器支持团队会根据特定的支持要求提供最适合您的安装的类别。

  • 启用DEBUG级别的日志记录，添加事件类别名称。

  • 启用TRACE级别的日志记录，将事件类别名称附加为:trace 。

实验特征

打开实验功能页面。在此页面上，您可以管理您的Hub安装中启用了哪些实验功能。有关更多信息，请参见实验功能 。
