管理群组访问

使用“ 角色”选项卡，您可以查看和管理组成员可用的权限。组成员根据一个或多个项目中分配给该组的角色继承权限。

您还可以通过将组直接添加到项目团队来使用组来授予组成员对项目的访问权限。有关更多信息，请参见管理项目团队 。

查看组的访问权限

组的访问权限显示在组配置文件的“ 角色”选项卡上。在这里，您可以查看将哪些角色分配给该组。

此选项卡上有两个视图。默认视图显示分配给组的角色列表，由Project列出。

  • 第一部分显示直接分配给该组的角色。

  • 第二部分显示了作为项目团队成员分配给该组的角色。

  • 最后几节显示该组从任何父组继承的角色，包括分配给“ 所有用户”组的角色。

您还可以选择按角色查看列表。

该信息分为与Project相同的部分。

在两个视图中都可以执行以下操作：

  • 使用搜索框过滤列表。您可以使用一系列过滤条件，包括用户，组，角色和权限。输入格式的搜索条件field: value 。

    例如，如果要查看组成员有权更新的项目，请输入permission: (Update Project)并应用过滤器。

  • 单击详细信息按钮以显示或隐藏侧栏。边栏显示分配给角色的一组权限。权限按服务分组。

要查看组的访问权限：

需要权限：读取用户基本，读取角色，读取组，读取项目完整

  1. 打开组配置文件的“ 角色”选项卡。

  2. 使用搜索框按角色，项目甚至特定权限过滤列表，以查找组是否具有对资源或操作的所需访问权限。

向组授予角色

您可以直接向组授予角色。授予组成员所有分配给该角色的访问权限。

要将个人角色分配给组：

需要权限：读取用户基本，读取角色，读取组，更新项目

  1. 在管理菜单的访问管理部分，选择组 。

  2. 从列表中选择一个组。

  3. 选择角色选项卡。

  4. 单击授予角色按钮。

  5. 在“ 授予角色”对话框中，选择要分配给组的角色。

  6. 选择一个或多个要在其中应用角色的项目。如果要将角色应用于集线器中的所有项目，请选择“ 全局”项目。

  7. 单击授予按钮。

    • 所选角色将授予用户组。

    • 分配给角色的所有权限都被授予属于该组成员的用户。

撤消组中的角色

如果一组用户不再需要角色授予的访问权限，则可以撤消角色分配。此过程仅适用于直接分配给组的角色。如果将角色作为另一个组的子组授予该组，则有两个选择：

  • 从父组中撤消该角色。

  • 通过将子组嵌套在“ 所有用户”组下，将其从父组中删除。

要撤消组中的角色，请执行以下操作：

需要权限：读取用户基本，读取角色，读取组，更新项目

  1. 在管理菜单的访问管理部分，选择组 。

  2. 使用搜索框找到所需的组。

  3. 从列表中选择一个组。

  4. 选择角色选项卡。

  5. 找到要从选定组中删除的角色分配。使用搜索框按名称，项目或权限查找角色。

  6. 在角色分配下，选择在其中授予角色的项目。

  7. 单击撤消角色按钮。

  8. 在确认对话框中确认操作。

    • 该角色将从所选项目的组中撤消。

    • 组的成员将失去分配给项目中角色的权限。

对于作为项目团队成员分配给该团队的角色，您可以通过从该团队或项目团队中删除来撤消访问权限。在列表中选择角色，然后单击从团队中删除按钮。
