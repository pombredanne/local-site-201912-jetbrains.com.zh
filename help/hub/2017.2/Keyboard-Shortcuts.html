Keyboard Shortcuts

JetBrains team tools are designed to be keyboard-centric, and Hub is no exception. Hub provides several shortcuts for navigating through the user interface.

These shortcuts are mapped to keys on a keyboard with a functional layout for US English. Keys that differ for US and UK variants are not mapped to any shortcuts. If your keyboard has a visual layout for a different language, you can switch the language preference for your operating system to English and use these shortcuts with the default mapping.

The shortcuts on this page are grouped by the context where they can be applied when you work with Hub.

Use the following shortcut to open the list of keyboard shortcuts from anywhere in Hub.

Windows ShortcutMac ShortcutDescription
Shift + F1command + /View keyboard shortcuts.

Tables and Lists

Use these shortcuts to select and update items that are displayed in lists and tables. Note the differences between one or more items that are selected in the list and the current item. The current item is the single item that is highlighted in the table or list. Selected items have a check mark in the checkbox.

Windows ShortcutMac ShortcutDescription
Down ArrowDown ArrowToggle focus from the filter to the first item. Set focus to the item below the current item.
Up ArrowUp ArrowToggle focus from the first item to the filter. Set focus to the item above the current item.
Shift + Up ArrowShift + Up ArrowToggle the selection of the current item and move to the item above.
Shift + Down ArrowShift + Down ArrowToggle the selection of the current item and move to the item below.
