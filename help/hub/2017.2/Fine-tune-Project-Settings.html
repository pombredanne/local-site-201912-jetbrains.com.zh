Manage the Project TeamHub Project Settings

Fine-tune Project Settings

When you add services to a project, the project page provides direct access to the pages that let you configure the settings for each connected service.

The drop-down list displays the name of the page and the server address of the connected service. Select an item from the list to open the selected page in connected service.

To edit Hub project settings:

  1. Click the Projects link in the main menu to open the list of projects in Hub.
  2. Click the project name in the list to open its page:
  3. Click the Fine-tune project settings button and select the type of settings that you need to update to open the relevant page.

See the Hub Project Settings page for the description of settings and options that you can fine-tune in Hub.
