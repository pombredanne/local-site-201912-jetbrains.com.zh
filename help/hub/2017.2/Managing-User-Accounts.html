Users

In the Users area in Hub, you can manage accounts for other users and grant them permission to access and work in connected services. The following operations are available:

  • Create User Accounts
  • Merge User Accounts
  • Ban User Accounts
  • Update User Profiles
  • Add Credentials to a User Profile
  • Manage User Licenses
  • Manage Group Memberships
  • Manage User Access
  • Delete User Accounts

To access the Users page in Hub, select the Users option from the Access Management section of the Administration menu.
