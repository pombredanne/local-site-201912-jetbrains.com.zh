Supported Environments

This page describes the supported environments for JetBrains Hub service.

Java Version

JetBrains Hub requires Java 7+.

Supported Browsers

Hub runs in a standard web browser. With the speed at which browsers are updated, it is not always possible to pre-certify every browser. We resolve functional issues in new releases of supported browsers as they arise.

We encourage you to update your browser to the latest version. You benefit from the latest security and stability fixes.

Hub is supported in the two latest stable versions of following web browsers.

  1. Google Chrome
  2. Mozilla Firefox
  3. Apple Safari
  4. Microsoft Internet Explorer (Hub supports IE 9.0+)
  5. Opera

If you use Internet Explorer...

If you use IE 9.0+ but still find visual inconsistencies or possible problems while viewing Hub, please make sure that Document Mode is set to 'IE9 Mode' or higher in your Internet Explorer. Also, MS IE Document Modes can be enabled globally for the company's intranet. Thus, if your personal Document Mode settings in IE don't solve the problem, you might need to contact your system administrator to check the global intranet settings.
