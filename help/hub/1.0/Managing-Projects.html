Managing Projects

Projects administering section in Hub suits best for more low-level administering operations with projects like: merging projects, various batch operations, archiving and restoring projects, or viewing history of events related to a project, and so on. Though you can create a new project in Hub itself, we recommend using Project Wizard for this task as it makes creating and configuring projects easier and more user-friendly.

You can access the Projects section in Hub by clicking Projects link in the main Hub menu.
