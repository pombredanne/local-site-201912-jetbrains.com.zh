How to connect YouTrack and Upsource to Hub

The general steps of connecting ("integrating") YouTrack and Upsource to Hub is described below for various combinations of YouTrack and Upsource installations.
  • You have an existing YouTrack and an existing Upsource installation
  • You have existing YouTrack and going to install new Upsource
  • You have existing Upsource and going to install new YouTrack
  • You are going to install both new YouTrack and Upsource

You have an existing YouTrack and an existing Upsource installation

  1. Install, configure and run a standalone Hub instance
  2. Migrate your current YouTrack instance to using this standalone Hub
  3. Migrate your current Upsorce installation to using this standalone Hub
  4. In the standalone Hub, configure or merge projects created from YouTrack and Upsource

You have existing YouTrack and going to install new Upsource

  1. Install, configure and run a standalone Hub instance
  2. Migrate your current YouTrack instance to using this standalone Hub
  3. During initial configuration of the new Upsource, set it up to use your standalone Hub instance in the User Management settings section.
  4. In the standalone Hub,use the Project Wizard to create and add resources from Upsource to the projects created from existing YouTrack.

You have existing Upsource and going to install new YouTrack

  1. Install, configure and run a standalone Hub instance
  2. Migrate your current Upsource instance to using this standalone Hub
  3. During initial configuration of the new YouTrack, set it up to use your standalone Hub instance in the User Management settings section.
  4. In the standalone Hub,use the Project Wizard to create and add resources from Youtrack to the projects created from existing Upsource.

You are going to install both new YouTrack and Upsource

  1. Install, configure and run a standalone Hub instance
  2. Install YouTrack and during the initial configuration of the new YouTrack, connect it to your standalone Hub instance in the User Management settings section.
  3. Install Upsource and during the initial configuration of the new Upsorce, connect it to your standalone Hub instance in the User Management settings section.
  4. In the standalone Hub,use the Project Wizard to create new projects and add to them resources from both YouTrack and Upsource.
