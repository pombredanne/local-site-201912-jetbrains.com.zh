Installing Hub with ZIP Distribution

Introduction

Hub ZIP is a cross-platform distribution. It allows installing and running Hub on MS Windows, Linux and Mac OS.

The installation procedures for all operation systems are similar. The major difference between installing Hub is the two scripts hub.bat for MS Windows and hub.sh for Linux and Mac OS, and the tool which you use to start the installation script:

  • MS Windows — Command line tool
  • Linux — Command line console
  • OS X — Terminal

All commands listed on this page are Linux/Mac OS commands. If youâ€™re working on a Windows server, simply replace .sh to .bat.

Installation Procedure

To install Hub using ZIP distribution:

  1. Download Hub distribution from our site.
  2. Select a directory which will be used as the home directory for your Hub server. Unzip the downloaded ZIP file to this folder.
  3. In a command line tool, change directory to <Hub home directory>/bin.
  4. Run Hub with the ./hub.sh run command.

    Use the ./hub.sh help to view all available command options.

    The web-based Configuration Wizard will be opened in your default browser. It will help you set up and start your new Hub.
  5. Provide the main System Settings of the Hub service:
    • Base URL: the URL by which the Hub server will be accessible to the end-users.
    • Application Listen Port: the default port that Hub will use for HTTP communications.
  6. In the Advanced Settings section, you can configure location of the Hub system directories: directories to store Hub's database, database backups, logs, and temporary files.
    When done, click Next to configure User Management settings.
  7. In the User Management section, specify the password for the default system administrator account admin.
  8. Send usage statistics anonymously - keep this option selected to help us make Hub better. We never share collected data with any third party.
  9. Click Next to review your Hub license. At the moment, Hub is free and you can use it without limitations in number of users When done, click Install.
When the configuring process is finished, you will be navigated to the login page of the newly installed Hub server.
