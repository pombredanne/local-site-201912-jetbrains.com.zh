Configuring Access for a User Group

    Viewing Access Permissions of a Group

    Open the Access tab of a user group profile. Here you can view which roles and permissions are assigned to the group.

    Switch between three views to see the various slices of the access configuration of the account:

    • Access rights by project — see to which projects the group has the access.
    • Roles — see the list of roles granted to the group. Select the role to view in which project it was granted and by what means (e.g. the account is a member of the New Users group, and thus inherits the role).
    1. Permissions — see the whole set of permissions granted to the group. Select a permission to view details and how it was granted.

    Granting Access to a User Group

    In addition to granting permissions via groups, you can assign roles or permissions to users individually.

    Granting a Role to a Group

    To assign an individual role to a group:

    • In the Administration area, click Groups.
    • In the list of user groups, locate a group that you want to assign a role to, and click its name to open the group's profile.
    • Open the Access tab.
    • Click +Grant Access... button.
    • In the Grant Access dialog, choose a role to assign to the group.
    • Select a project where you want to apply the role. Select "Global" if you want to apply the role in all projects in the system.
    • Select an option how the role should be assigned.
    • Click Grant Access button.

    Selected role is granted to the user group and all its permissions are granted to its members.

    Granting a Permission to a Group

    To grant a permission to a group:

    • In the Administration area, click Groups.
    • Find a group using the search box and click its name in the list to open the group's profile.
    • Open the Access tab.
    • Click +Grant Access... button.
    • In the Grant Access dialog, switch to Permission view.
    • Select permission to grant to the group in the drop-down list. Start typing the permission's name to narrow the list.
    • Select a projects where you want to apply the permission.
    • Select an option how the permission should be assigned (for example, via creating a new user group and adding current account to it, or by selecting an existing one).
    • Click Grant Access button.
