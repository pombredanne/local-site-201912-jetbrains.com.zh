Configuring User Account Settings

Each user account has its own administrative profile, which is available for the administrators only and differs from the general User Profile that a user can view and configure himself.

Account profile allows you to view and configure the following parameters of an account:

  • General settings which include login and full name, user's avatar, email and jabber accounts, as well as the list of merged login names for the account.
  • Group membership for the account.
  • Access permissions.

To open an account profile, you need to click user's name in the Users list (Administration > Users).

Configuring General Account Settings

General settings include login and full name, user's avatar, email and jabber accounts.

Also General settings contain the list of merged login names for the account.

Using the general settings you can send request to verify user's email. If an account has not at least one verified email, the user won't be able to receive notifications or restore forgotten password.

You can also ban (or reactivate) the account right here, in the General settings.

Below the current ban status of the account, you can find the Additional profile settings link which directs to the user profile of the account.

Configuring Group Membership

To view and configure account's group membership, open the Groups tab of the accounts profile.

For the details about configuring group membership, please refer to the Configuring Group Membership for a User Account page.
