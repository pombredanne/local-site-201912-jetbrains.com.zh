Configuration Parameters

Configuration parameters can be passed to Hub as:

  1. Servlet context parameters
  2. Servlet configuration parameters
  3. JNDI objects
  4. System properties (for example, -Djetbrains.jetpass.key=...)

Refer to the Providing Hub Configuration Parameters page for more details.
