Merging Projects

To merge several projects

  1. Open Projects section in Hub.
  2. In the list of projects, select two or more projects, which you need to merge. Click the merge icon in the toolbar.

  3. In the Merge Projects dialog, select project Name and Key of the resulting project, the project that will remain after the merge.

    Optionally, select the Archive it option to automatically archive the resulting merged project.

  4. When ready, click Merge Projects.

All selected projects will be merged into one with the specified Key and Name. All resources associated with merged projects (groups, repositories, projects in YouTrack or Upsource) will be transferred to the resulting project.
