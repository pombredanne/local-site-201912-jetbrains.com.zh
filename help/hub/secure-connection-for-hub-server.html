集线器服务器的安全连接

有多种方法可以在集线器服务器及其客户端之间使用TLS（SSL）设置加密的HTTPS连接。

  • 使用第三方TLS终止代理服务器。有关详细信息，请参见反向代理配置 。

  • 对集线器服务器使用内置的TLS。您可以将Hub配置为以HTTPS模式运行：

    • 在配置向导中 ，在安装或升级过程中

    • 从命令行 ，为现有的Hub安装启用HTTPS

无论当前设置如何，都可以随时从第三方TLS终止代理切换到集线器内置TLS，反之亦然 。
