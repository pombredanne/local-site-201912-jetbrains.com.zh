Secure Connection for Your Hub Server

There are several ways to set up an encrypted HTTPS connection with TLS(SSL) between your Hub server and its clients.

  • Use a third-party TLS-terminating proxy server. See Reverse Proxy Configuration for details.

  • Use built-in TLS for Hub server. You can configure Hub to run in the HTTPS mode:

    • in Configuration Wizard, during the installation or upgrade

    • from the command line, enabling HTTPS for your existing Hub installation

You can switch from a third-party TLS-terminating proxy to the Hub built-in TLS and vice-versa at any time regardless of your current settings.
