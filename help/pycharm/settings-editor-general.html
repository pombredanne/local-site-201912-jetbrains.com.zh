描述

老鼠

双击选择时，请遵守“ CamelHumps”一词设置

选中此复选框可使PyCharm在通过双击选择单词时调用CamelHumps选择。

仅当启用“使用'CamelHumps'单词”选项时，此功能才有效。

使用Ctrl / Command +鼠标滚轮更改字体大小（缩放）

如果选中此复选框，可以在按住鼠标左键的同时滚动鼠标滚轮来更改特定的编辑器字体大小。 Ctrl键。

此复选框还会影响快速文档查找中的字体大小。

如果清除该复选框，请在按住鼠标左键的同时滚动鼠标滚轮Ctrl键滚动编辑器。

在编辑器中启用拖放功能

如果选中此复选框，则可以在编辑器中拖放代码片段。

软包装

软包装文件

选中此复选框可将自动换行应用于特定文件类型（例如，在降价文件中编写文档时可能会有所帮助）。

在字段中，输入文件扩展名，以分号分隔。

您可以在编辑器中启用或禁用指定的配置。

右键单击左装订线，然后从上下文菜单中选择或清除“ Soft-Wrap Current Editor”选项。请记住，这些设置仅影响当前编辑器，而不影响文件。

要快速访问设置，请从选项列表中选择“ 配置软包装” 。

对包装的零件使用原始线的缩进

选中此复选框可在调整编辑器或控制台的大小时使用自定义缩进进行自动换行。在右侧的“ 其他班次”文本字段中指定缩进值。

仅显示当前行的软包装指示器

如果选中此复选框，则自动包装字符将仅在活动逻辑行中显示。

否则，自动换行符将显示在每行的末尾和下一行的开始处。

虚拟空间

允许在行尾放置插入号

如果清除此复选框，则插入记号永远不会停在一行中的最后一个符号之后。

允许将插入号放置在标签内

选择此复选框以允许将插入号放置在制表符中。原因是每个选项卡字符在编辑器中均显示为一组“虚拟”空格字符。

在文件底部显示虚拟空间

如果选中此复选框，则当前编辑的行（即使是最后一行）也可以滚动到屏幕顶部。PyCharm添加了必要数量的虚拟线。

其他

在保存上删除尾随空格

选择PyCharm在文件保存行的末尾处理尾随空格的方式：

  • 修改的行 ：仅在修改的行的末尾删除尾随空格。

  • 全部 ：在所有行中删除尾随空格。

  • 无 ：不去除尾随空格。

在适用的情况下，不会删除影响程序输出的大量尾随空格。例如，不会删除Groovy中多行字符串中的尾随空格，依此类推。

始终在插入符号行上保留尾随空格

如果选择此选项，则在保存操作中插入符号的行上的尾随空格将不会被删除（例如，当您切换到另一个窗口时）。

确保在保存时文件末尾有换行

选择此复选框可使PyCharm在保存过程中自动在文件末尾添加一个空行。

显示有关鼠标移动的快速文档

选择此复选框以显示插入符号的快速文档 。在指定的延迟后，将弹出快速文档弹出窗口。

高亮显示装订线

如果要在左侧编辑器装订线中用颜色条纹突出显示添加/修改的行，请选中此复选框。

仅使用空格修改的行的颜色不同

仅当启用了高亮显示装订线中的修改的线选项时，此选项才可用。

如果您希望仅添加/删除空格的行与经过重大修改的行以不同的颜色突出显示，请选中此复选框。

插入符号运动的重点

突出显示匹配的花括号

选中此复选框可使PyCharm 高亮显示成对的开/关括号，当您将插入号放置在打开前或关闭后的位置时。它也适用于HTML和XML标签。

突出显示当前范围

选中此复选框可使PyCharm突出显示当前插入符号位置中键入的代码的可用范围。

突出显示插入符号处元素的用法

选择此复选框可使PyCharm突出显示插入符号当前所在元素的所有用法。

格式化

重新格式化代码后显示通知

选中此复选框可在每次尝试重新格式化代码时显示有关代码更改的通知以及“ 重新格式化代码”对话框的快捷方式。否则，PyCharm将静默重新格式化代码。

优化导入操作后显示通知

选择此复选框以显示有关代码更改的通知。否则，PyCharm将默默优化导入。

卷动

平滑滚动

如果启用此选项，则编辑器将在您导航到元素时滚动页面，而不仅仅是跳转到目标位置。

首选滚动编辑器画布以使插入符号线居中

单击此选项以选择滚动编辑器画布并将插入符号保持在适当位置。

将插入符号保持在适当位置并滚动编辑器画布在调试会话过程中可能会有所帮助。在单步执行代码行时，编辑器画布将滚动，而插入标记处的行始终位于屏幕中央。

首选移动插入符号行以最小化编辑器滚动

单击此选项选择移动插入符。

在调试会话中单步执行代码行时，插入记号向下移动，并且编辑器画布不会滚动，直到插入记号行到达屏幕底部为止。

重构

启用就地模式

选择或清除此复选框以启用或禁用Python的就地重构。

与重构相关的就地意味着通过在编辑器中直接键入来指定重构所需的全部或大部分信息。所有受影响的代码片段都将突出显示，并在您键入时更改。如果合适，在相应的选项框中选择其他重构选项。

就地重构模式可用于以下重构：

  • 提取常数
  • 提取字段
  • 提取变量
  • 改名

如果未选中此复选框，则在相应的对话框中指定所有重构的重构设置。

预选旧名称

如果选中此复选框，则为该符号调用重命名重构时， 将选择该符号的旧名称。

如果清除该复选框，则不会选择要重命名的符号。

显示局部变量的内联对话框

如果要显示“内联局部变量”重构的确认对话框，请选中此复选框。

限度

剪贴板中保留的最大内容数

在此字段中，指定剪贴板中可以保留多少个代码块。

最近档案限制

在此字段中，指定可以在最近文件列表中包含多少个文件名。

富文本副本

默认情况下复制为富文本

选中此复选框可将富文本格式从编辑器复制到可识别RTF的任何其他编辑器。
请注意，如果您从编辑器的上下文菜单中选择“ 复制为纯文本” ，反之亦然，则可以覆盖此选项，使用上下文菜单中的“ 复制为富文本”选项将覆盖编辑器设置中未选中的复选框。

配色方案

使用此列表为文本副本选择配色方案。您可以从以下选项中选择：
  • 默认
  • 主动方案
  • 达库拉
突出显示错误

误差条纹标记最小高度（像素）

在此字段中，指定错误和警告条纹的最小大小。

自动修复延迟（毫秒）

在此字段中，指定PyCharm开始解析输入的文本之前的时间段。

“下一个错误”操作仅适用于高优先级问题

选中此复选框，以使PyCharm在执行“ 导航|导航”时仅通过优先级最高的问题（例如错误） 。下一个/上一个突出显示的错误命令F2 / Shift+F2 。

清除此复选框，以使PyCharm依次解决所有现有问题（例如，错误和警告）。
