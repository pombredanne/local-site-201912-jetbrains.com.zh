PyCharm 2019.2帮助

支持的语言

现代应用程序的开发涉及使用多种语言，因此PyCharm是用于多语言编程的IDE。

PyCharm支持以下语言：

  • 蟒蛇

    • 应该将Python 下载并安装在您的计算机上。

    • PyCharm支持以下Python版本：

      • Python 2： 2.7版

      • Python 3：从3.5版到3.8版

    • PyCharm支持wxPython ， PyQt4和PyGTK 。

  • JavaScript 。请参阅JavaScript部分。

  • CoffeeScript 。请参阅CoffeeScript部分。

  • 标记语言和样式表：

    • XML格式

    • HTML / XHTML

    • YAML
    • 的CSS

    • 无礼 / 少

    • 触控笔

PyCharm支持以下框架，工具和模板语言：

  • Django最高版本为1.3。

  • Google App Engine 。

  • 烧瓶 。

  • 建立 。

    PyCharm检测buildout在现有项目中提供支持，并可以打开它们。但是，您必须提前提供扩展环境。

  • Web2Py。有关详细信息，请参阅Web2Py部分。

  • 金字塔。有关详细信息，请参见“ 金字塔 ”部分。

  • 玛雅人 。该支持包括Maya中的远程调试和启动脚本。

  • 模板语言：

    • 真子
    • Jinja2
    • 变色龙

    请参阅模板部分。

  • SQLAlchemy 。该支持包括：

    • 代码见解

    • 可以在图中查看数据库结构。请参阅使用图节。

    • 代码完成并解决。

PyCharm中的编码帮助包括：

  • 语法和错误突出显示以维护绿色代码

  • 用于创建存根类，脚本等的文件模板

  • 实时模板 ，可将缩写词扩展为复杂的代码结构

  • 代码完成 ，根据上下文完成类，方法，字段，参数和其他实体的名称

  • 代码生成以生成特定于语言的代码结构

  • 意向操作和快速修复以优化您的代码

  • 代码重构可在整个代码库中重命名，移动，复制和删除实体

  • 导入协助以导入丢失的库

  • 语言注入，用于在字符串文字，标签或属性中使用受支持的语言

  • 用于查看API文档，方法中的参数名称和提示，静态表达式数据等的代码参考信息

除了编码帮助，PyCharm还支持特定于语言的工具，用于构建，运行，测试和部署应用程序。
