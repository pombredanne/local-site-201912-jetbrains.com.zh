PyCharm重构教程

本教程的内容

本教程以使用有理数的简单类为例，展示了PyCharm中可用的一些重构。

先决条件

确保满足以下先决条件：

  • 您正在使用PyCharm 2016.2或更高版本。

  • 已经创建一个项目。

在您的项目中创建一个Python文件Rational.py并添加以下代码：

简化有理数

让我们通过将分子和分母除以最大公约数来简化有理数：

从集合中导入namedtuple类Rational（namedtuple（'Rational'，['num'，'denom']））：def __new __（cls，num，denom）：如果denom == 0：引发ValueError（'Denominator不能为null' ）如果denom <0：num，denom = -num，-denom x = abs（num）y = abs（denom）而x：x，y = y％x，x factor = y返回super（）.__ new __（cls ，num // factor，denom // factor）def __str __（self）：返回'{} / {}'。format（self.num，self.denom）

提取方法

现在，让我们将对最大公约数的搜索提取为一个单独的方法。为此，请选择语句

x = abs（num）y = abs（denom）而x：x，y = y％x，x因数= y

然后按Ctrl+Alt+M 。在打开的对话框中 ，输入方法名称gcd然后单击确定 ：

@staticmethod def gcd（denom，num）：x = abs（num）y = abs（denom）而x：x，y = y％x，x factor = y返回因子

内联局部变量并更改方法签名

让我们摆脱变量factor ，通过使用内联变量重构。为此，将插入符号放在变量处，然后按Ctrl+Alt+N 。所有检测到factor内联变量。

接下来，使用Change signature更改参数名称。为此，将插入号放在方法声明行上，然后按Ctrl+F6 。在打开的对话框中，重命名参数denom和num至x和y分别单击更改参数的顺序。

您最终得到以下代码：

@staticmethod def gcd（x，y）：x = abs（x）y = abs（y）而x：x，y = y％x，x返回y

提取方法和快速修复

接下来，我们提取一个表达式Rational(other, 1)变成一个单独的方法。为此，将插入符号置于上述表情处，按Ctrl+Alt+M在打开的对话框中 ，键入新的方法名称from_int 。

最后，将插入符号置于方法处from_int声明，按Alt+Enter ，从建议列表中选择“ 将方法设为静态 ”，然后按Enter ：

@staticmethod def from_int（other）：返回Rational（其他，1）

最后，让我们更改参数名称other至number 。为此，将插入号放在参数上，然后按Shift+F6 。
