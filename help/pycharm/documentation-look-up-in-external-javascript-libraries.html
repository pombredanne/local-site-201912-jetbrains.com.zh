在外部库中查找

在PyCharm中，您还可以查看在外部JavaScript库和框架中定义的符号的文档。按下时Shift+F1将光标置于符号上，PyCharm会调用相应的链接，并在默认的PyCharm浏览器中打开文档页面。

启用外部JavaScript库查找功能

  1. 下载所需的库或框架。

  2. 在PyCharm级别将下载配置为库 。

  3. 指定到外部文档的链接。

查看有关外部JavaScript库中定义的符号的文档
将光标置于符号上，然后按Shift+F1或选择查看|主菜单上的外部文档 。
