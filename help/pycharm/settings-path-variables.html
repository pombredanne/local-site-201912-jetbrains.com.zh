示例：创建一个新的路径变量

您可能有一个位于项目目录之外的库。该库已附加到您的项目中，并且该库的路径包含在项目的.iml文件中。但是，该路径不应是绝对路径，因为此库在队友计算机上的位置可能不同。在这种情况下，您可以创建PATH_TO_LIB变量：

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到外观和行为|路径变量 ，然后单击 。

  2. 在“ 添加变量”对话框中，键入PATH_TO_LIB变量，其值指向磁盘上的库位置。

  3. 通过版本控制系统共享.iml文件。

  4. 在您的队友从VCS更新项目后，他们将更改PATH_TO_LIB变量值，使其指向其计算机上的库位置。
