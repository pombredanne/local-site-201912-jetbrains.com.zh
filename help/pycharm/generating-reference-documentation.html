PyCharm 2019.2帮助

生成参考文档

此功能在PyCharm的教育版中不可用。

PyCharm使用以下文档生成器来帮助生成格式化的API文档：

  • DocUtils
  • 狮身人面像

注意以下几点：

  • 文档生成器应正确安装在您的计算机上。有关详细信息，请参阅其各自的下载和安装页面。

  • PyCharm可识别文档字符串格式，并使用在“设置”对话框的“ Python集成工具”页面中定义的文档源目录。

使用DocUtils生成参考文档

生成docutils文档

  1. 选择DocUtil任务运行/调试配置，然后根据需要进行更改：指定配置名称，输入和输出目录以及可选键。

  2. 启动运行/调试配置，如“ 运行应用程序 ”部分中所述。

使用Sphinx生成参考文档

为Sphinx文档创建初始基础结构

  1. 确保Sphinx软件包已与项目解释器一起安装（ “设置/首选项” |“ 项目” | 项目翻译 ）。有关如何在PyCharm中安装软件包的更多信息，请参阅安装，卸载和升级软件包 。

  2. 在主菜单上，选择“ 工具”。 Sphinx快速入门 。

  3. 如果未在“ Python集成工具”页面中指定Sphinx工作目录 ，则会打开“ 设置工作目录”对话框，建议您指定文档的路径。

    如果在项目中指定了Sphinx工作目录，则不会出现此对话框。

  4. 在PyCharm中打开的控制台中，回答sphinx-quickstart实用程序提供的问题。特别是，指定将存储生成的conf.py文件的源目录。如果回答这个问题，您只需按Enter ，PyCharm将使用您在上一步中指定的路径，或“ Python集成工具”页面的Sphinx工作目录字段中指定的路径。

对于特定目录，仅执行一次sphinx-quickstart实用程序。如果要在其他目录中生成Sphinx基础结构，请在“ Python集成工具”页面的Sphinx工作目录字段中指定它。

生成Sphinx文档

  1. 选择Sphinx任务运行/调试配置，然后根据需要进行更改：指定配置名称，输入和输出目录。

  2. 启动运行/调试配置，如“ 运行应用程序 ”部分中所述。
