PyCharm 2019.2帮助

Ctrl + Alt

本节列出并描述了键盘快捷方式，包括Ctrl+Alt按键：

  • Ctrl + Alt +字母数字键
  • Ctrl + Alt +导航键
  • Ctrl + Alt +功能（F）键

Ctrl + Alt +字母数字键

捷径

功能

使用此快捷方式可以...

Ctrl+Alt+B

导航到实施

导航到插入符号处的项目实施 。

Ctrl+Alt+C

提取常数

将所选表达式替换为常量 （静态最终字段）（重构）。

Ctrl+Alt+F

提取字段

将选定的表达式结果放入字段 （重构）。

Ctrl+Alt+H

呼叫阶层

浏览所选方法的调用层次结构。请参阅页面查看源代码的结构和层次结构

Ctrl+Alt+I

自动缩进线

根据代码样式设置缩进当前行或所选块。

Ctrl+Alt+J

环绕实时模板

用实时模板之一包围选择。

Ctrl+Alt+L

重新格式化代码

重新排列所选的代码块。

Ctrl+Alt+M

提取方法

根据所选代码创建一个方法 （重构）。

Ctrl+Alt+N

排队

内联所选方法/变量 （重构）。

Ctrl+Alt+P

提取参数

将选定的表达式转换为方法参数 （重构）。

Ctrl+Alt+T

环绕着

用包围所选的代码片段if ， while ， try/catch或其他构造。

Ctrl+Alt+V

提取变量

将选定的表达式结果放入变量（重构）。参见页面提取变量 。

Ctrl+Alt+Y

同步化

检测所有外部更改的文件，然后从磁盘重新加载它们。

Ctrl + Alt +导航键

捷径

功能

使用此快捷方式可以...

Ctrl+Alt+Enter

在当前行之前开始新行

在当前行之前开始新行。

Ctrl+Alt+Down
Ctrl+Alt+Up

导航到下一个/上一个发生

导航到下一个/上一个找到的项目。

Ctrl+Alt+Left

背部

撤消上一次导航操作。查看有关导航项的页面

在macOS计算机上，您也可以使用三指从右向左滑动手势。

Ctrl+Alt+Right

向前

重做上一个未完成的导航操作。查看有关导航项的页面

在macOS计算机上，您也可以使用三指从左到右滑动手势。

Ctrl+Alt+Home

导航到相关符号

在具有各种关系的文件之间导航。

Ctrl + Alt +功能（F）键

捷径

功能

使用此快捷方式可以...

Ctrl+Alt+F6

切换到另一个覆盖套件。

打开Coverage Suites弹出窗口，然后选择要运行的套件。

Ctrl+Alt+F7

显示用法

在插入符号处显示符号的用法。

请参见查看符号用法

Ctrl+Alt+F8

快速评估表达

评估一个任意表达式而无需调用“ 评估表达式”对话框。

Ctrl+Alt+F9

强制运行到光标

转到插入符号所在的行，忽略断点。请参见逐步执行程序页面。
