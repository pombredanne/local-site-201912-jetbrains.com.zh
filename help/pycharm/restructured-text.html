PyCharm 2019.2帮助

reStructuredText支持

除了支持reStructuredText Docstring格式（PEP 287） ，PyCharm还提供了用于重组文本的便捷编辑器。

与Markdown编辑器类似，您可以修改reStructuredText文件并在“ 预览”面板中验证结果。您可以使用相应的图标在可用的编辑模式之间切换。编辑和预览模式反映了当前选择的PyCharm 用户界面外观 。

您可以使用重组后的文本打开任何现有文件，也可以从头开始创建一个新文件。

创建一个.rst文件

  1. 在“ 项目”工具窗口中右键单击，然后选择“ 新建” |“新建”。文件 。在打开的对话框中，指定扩展名为.rst的任何文件名，然后单击“ 确定” 。

  2. PyCharm可以识别文件类型，并用相应的图标标记。

  3. 新创建的文件在“ 编辑器”窗口中打开。如果未添加任何文本，则禁用预览 。

让我们添加一些内容，以了解PyCharm如何协助编辑重组文本。

编辑文字

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），查看reStructuredText的默认配色方案（“ 编辑器” |“配色方案” |“ reStructuredText” ）。如果需要，设置标记元素的替代颜色和效果。

    单击确定以保存更改并关闭窗口。

  2. 添加任何节结构标记并切换到拆分视图模式以预览结果。请注意，PyCharm支持标记检查，因此，如果检测到任何错误，则会在“ 编辑器”面板中突出显示有问题的行，并在“ 预览”面板中显示相应的错误消息。

  3. 使用.. code-block ::指令添加代码片段。PyCharm支持指令名称的代码完成。

    您可以通过在指令中指定特定的语言来启用代码语法突出显示： .. code-block :: python 。另外，请确保您的项目解释器安装了Pygments软件包。

  4. 如果需要，您可以快速标记文本中的链接并推迟添加参考目标。PyCharm会通知您，并突出显示缺少引用的链接。

    您需要做的就是将正确的目标引用添加到reStructuredText文件，例如， .. _pytest：https://docs.pytest.org/ 。

如果在预览内容时遇到任何问题，请尝试更改“ 预览”面板的类型。 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），导航到“ 语言和框架| ReStructrued Text ，然后从“ 预览”面板列表中选择JavaFX或Swing ，单击“ 确定” ，然后继续编辑.rst文件。
