向上拉成员，向下拉成员

上拉成员重构允许您将类成员移动到超类。

提拔成员重构可以创建抽象方法。如果项目使用解释器Python 2.x，则只能抽象实例方法。如果项目使用Python 3.x，则可以抽象任何方法。

请注意，PyCharm会自动添加抽象方法所需的import语句。

下推成员重构通过将类成员移至子类来帮助清理类层次结构。然后将成员仅重定位到直接子类中。

拉起成员

  1. 选择要移动到超类的类。

  2. 在主菜单或上下文菜单上，选择“ 重构” |“更改” 。拉成员 。出现“上拉成员”对话框。

  3. 选择目标对象（超类）。

  4. 在“ 成员”部分中，选择要移动的成员。

  5. 要将方法移动为抽象方法，请选中该方法旁边的“ 使抽象 ”列中的复选框。

  6. 单击“ 重构”将所选成员拉到其目的地。

下推成员

  1. 在编辑器中，打开您需要下推其成员的类。

  2. 在主菜单或上下文菜单上，选择“ 重构” |“更改” 。压低成员 。下推成员对话框显示要下推的成员列表。

  3. 在要下推的成员区域中，选择要移动的成员。请注意，插入标记处的成员已被选中。

    如果推成员可能会导致问题，则会以红色突出显示通知您。这意味着，如果无人照管情况，则重构后将出现错误。PyCharm会在“检测到问题”对话框中提示您，您可以在其中选择忽略或解决问题。

  4. 预览并应用更改。
