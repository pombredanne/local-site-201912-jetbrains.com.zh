登录到IntelliJ Configuration Server对话框

  • 在你开始前
  • 总览
  • 控制项

在你开始前

如管理插件中所述，在“ 插件”页面上安装IntelliJ Configuration Server插件。

总览

IntelliJ Configuration Server用于存储团队要使用的常见IDE设置。这样可以确保所有团队成员具有统一的IDE外观。

对话框打开：

  • 安装插件后首次启动PyCharm时。在这种情况下，对话框将在下次启动时包含一个附加区域。

  • 当您单击IntelliJ配置服务器状态按钮时在会话期间单击状态栏上的，然后在打开的对话框中单击登录按钮。

如有必要，使用此对话框登录IntelliJ Configuration Server并指定代理设置以访问服务器。

控制项

项目

描述

登录

在此字段中，输入您的JetBrains帐户登录名。

密码

在此字段中，输入您的JetBrains帐户密码。

创建或管理您的JetBrains帐户

单击此链接创建一个JetBrains帐户，或者如果您已经拥有它，则编辑您的帐户。

使用HTTP代理

如果要通过代理服务器访问IntelliJ配置服务器，请选中此复选框。

主机名

在此字段中，输入代理主机名。

端口号

在此字段中，键入代理服务器端口号。

代理验证

如果要使用登录名和密码来访问HTTP代理服务器，请选中此复选框。

登录

在此字段中，输入您的代理登录。

密码

在此字段中，输入您的代理密码。

记住密码

如果选中此复选框，PyCharm会记住您的代理密码，以后您登录时会自动填写“ 密码”字段。

在下一个启动区域

仅当PyCharm首次启动时，该区域才会显示在对话框中。使用此区域可配置您以后如何登录IntelliJ Configuration Server。

可以使用以下控件：

项目

描述

显示登录对话框

选择此选项可在下一次PyCharm启动期间显示“ 登录到IntelliJ Configuration Server”对话框。

静默登录

选择此选项可登录服务器，而无需输入登录名和密码。

不登入

如果您不打算登录IntelliJ Configuration Server，请选择此选项。

您可以在会话期间的任何时间重新配置此行为。

登录

单击此按钮登录IntelliJ Configuration Server。

跳跃

单击此按钮可启动PyCharm会话，而无需登录IntelliJ Configuration Server。

您可以在会话期间的任何时间登录到服务器。
