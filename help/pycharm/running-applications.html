运行应用程序

PyCharm支持运行整个应用程序以及特定的脚本。

PyCharm使用“ 运行/调试配置”中定义的设置。项目中存在的所有运行配置在“ 选择运行/调试配置”列表中可用。

如果在运行/调试配置中启用了运行前启动工具的选项，则PyCharm会运行这些工具，并在成功后运行该应用程序。否则，程序将立即启动。

如果要查看所有当前正在运行的应用程序的列表，请选择“运行” |“运行”。从主菜单显示运行列表 。有关详细信息，请参考查看运行过程部分。

请注意，开始运行会话后， 标记“ 运行工具”窗口的图标，并在“运行/调试配置选择器”中切换到指示运行过程处于活动状态。

运行脚本

  • 选择运行|在主菜单上运行或按Shift+Alt+F10 ，然后选择所需的运行/调试配置。

    这样，就有可能运行任何可用的运行/调试配置-必须从显示项目中存在的所有运行/调试配置的弹出菜单中选择所需的运行/调试配置。

    有关详细信息，请参见使用“运行”弹出菜单 。

  • 对于主要条款： 在左侧装订线中，单击图标 ，然后选择所需的命令。

  • 选择运行从上下文菜单中：

  • 按Ctrl+Shift+F10 。

  • 在“项目”工具窗口中选择所需的模块，然后选择“运行” 从选择的上下文菜单中。

使用“运行”弹出菜单

通过选择“ 运行” | “ 运行”来调用“ 运行”弹出菜单。在主菜单上运行命令，或按Shift+Alt+F10 。

在此弹出菜单中，您可以：

  • 调用“ 编辑配置”对话框 。

  • 启动前编辑所选配置F4 。

  • 即时删除配置Delete 。

  • 从运行切换到调试，反之亦然（按住Shift ）。

  • 访问先前选择的配置1 。

也可以通过按以下按钮快速访问此弹出菜单F9 ，当您不运行任何调试会话时。

重新运行应用程序

  1. 在“ 运行”窗口中，选择打开所需应用程序的选项卡。

  2. 在“ 运行”窗口的工具栏中，单击“ 重新运行”按钮 ，或按Ctrl+F5 。

    如果要重新运行而不失去工作位置，请按Shift+F10 。
