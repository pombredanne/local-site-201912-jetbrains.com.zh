PyCharm 2019.2帮助

配置调试器选项

PyCharm支持调试Python和Django应用程序，类和文件。调试功能已包含在PyCharm中，您只需要配置其设置。

根据启用的插件，PyCharm还可以支持其他语言的调试，例如JavaScript。

PyCharm内置了JavaScript调试功能，您只需配置其设置即可 。

PyCharm支持调试在内置或外部Web服务器上运行的应用程序。只能使用Google Chrome和Chrome系列的其他浏览器进行调试。

配置调试器设置

要配置调试所需的设置，请执行以下常规步骤

  1. 在“ 项目结构”中 ，配置要传递给解释器的根，依赖项和库。

  2. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），配置调试器选项：

    • 在“ 构建，执行和部署”部分下，单击“ 调试器” ，然后配置调试器选项。

    • 在“ 构建，执行和部署”部分下，单击“ Python Debugger” ，然后配置Python调试器选项。
