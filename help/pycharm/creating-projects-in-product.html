创建一个教育项目

PyCharm使创建各种类型的项目成为可能。

PyCharm Educational（Edu）版本提供了此功能。您可以通过EduTools插件为PyCharm Community和PyCharm Professional启用它。

创建课程

创建课程

  1. 请执行以下任一操作：

    • 在“ 欢迎使用”屏幕上的“ 快速入门”部分下，单击链接创建新项目 。

    • 从主菜单中，选择文件|新项目...。

  2. 在“ 新建项目”对话框中，单击“ 课程创建 ”选项，然后执行以下操作：

    • 指定项目位置。

    • 从Python解释器列表中，选择要用于新项目的解释器。

      请注意，该列表显示了所有先前配置的解释器和虚拟环境。如果在列表中找不到所需的口译员，请单击配置所需的本地Python解释器或虚拟环境。

    • 指定课程名称和描述。这些字段是必填字段。

      作者的姓名是自动指定的-这是系统用户名。您可以接受默认值，也可以输入其他作者姓名。

  3. 点击创建 。

有关详细信息，请参阅教育工作者的快速入门指南 。

创建一个教育项目

创建教育项目

  1. 请执行以下任一操作：

    • 在“ 欢迎使用”屏幕上的“ 快速入门”部分下，单击链接创建新项目 。

    • 从主菜单中，选择文件|新项目...。

  2. 在“ 选择项目类型”页面中，单击“ 教育 ”选项，然后执行以下操作：

    1. 指定项目位置。您可以手动执行，也可以单击然后从“ 选择基本目录”对话框中选择所需的位置。

    2. 请点击 （在浏览按钮旁边），然后在“ 选择Python解释器”对话框中，从“ 解释器”列表中，选择要用于新项目的解释器。

      请注意，该列表显示了所有先前配置的解释器和虚拟环境。如果在列表中找不到所需的口译员，请单击配置所需的本地Python解释器或虚拟环境 。

    3. 选择您要通过的课程。请执行以下任一操作：

      • 接受默认建议的课程（ Python简介 ）。

      • 通过单击下拉箭头选择一种预定义课程。如果列表未反映当前状态，请单击刷新它。

      • 请点击然后选择添加本地课程 。然后从文件选择器对话框中选择所需的zip存档。

      • 请点击然后选择登录到Stepik 。

      • 如果要通过自适应课程，请从列表中选择一个，例如Adaptive Python 。

  3. 点击创建 。

有关详细信息，请参阅《 学习者入门指南 》。
