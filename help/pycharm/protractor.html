量角器

量角器是Angular和AngularJS应用程序的测试框架。 量角器针对在真实浏览器中运行的应用程序执行测试，从而确保测试结果的正确性和可信赖性。PyCharm与Protractor集成在一起，因此您可以从IDE内部运行和调试测试。您可以在树形视图中查看测试结果，并从那里轻松导航到测试源。

在开始之前，请确保您的计算机上装有Node.js。

安装量角器

您可以在本地，项目中或全局安装Protractor。最好是全局安装。

全局安装量角器

  1. 在嵌入式终端 （ Alt+F12 ），输入：

    npm install -g protractor

  2. 要下载必要的二进制文件，请键入webdriver-manager update 。另请参阅量角器官方网站上的入门 。

您还可以按照npm和Yarn中的说明 ，在Node.js和NPM页面上安装量角器包。

运行测试

量角器测试仅通过运行/调试配置启动。

创建量角器运行配置

  1. 打开“ 运行/调试配置”对话框（主菜单上的“运行” |“编辑配置 ”），单击在左侧窗格中，然后从列表中选择“ 量角器 ”。将打开“ 运行/调试配置：量角器”对话框。

  2. 指定要使用的Node.js解释器。这可以是本地Node.js解释器，也可以是Windows Subsystem for Linux上的Node.js。

  3. 指定量角器包的位置以及protractor.conf.js配置文件的路径。如果您遵循标准安装，则PyCharm会检测所有这些路径并将其显示在相应的字段中。

通过运行配置运行测试

  1. 手动启动Selenium Server或将其配置为在protractor.conf.js中自动启动。有关详细信息，请参见量角器官方网站。

  2. 从主工具栏上的列表中选择量角器运行/调试配置，然后单击在列表的右边。

  3. 查看和分析来自服务器的邮件 “ 运行”工具窗口的“选项卡” 。

导航

使用PyCharm，您可以在文件和相关测试文件之间跳转，也可以从“ 测试运行器”选项卡中的测试结果跳转到测试。

  • 在编辑器中打开文件，然后选择“ 转到|测试或转到|从上下文菜单中测试受试者 ，或者只需按Ctrl+Shift+T 。

  • 要从测试结果跳转到测试，请在“ 测试运行器”选项卡中选择测试名称，然后从上下文菜单中选择“ 跳转到源代码 ”。测试文件将在编辑器中打开，并将光标置于测试定义上。

调试测试

仅通过运行/调试配置才能启动用于量角器测试的调试会话。

开始调试测试

  1. 如上所述创建量角器运行/调试配置。

  2. 从主工具栏上的列表中选择量角器运行/调试配置，然后单击在列表的右边。

  3. 在打开的“ 调试工具”窗口中，照常进行： 逐步执行测试 ， 停止并继续执行测试，在挂起时检查测试 ，依此类推。
