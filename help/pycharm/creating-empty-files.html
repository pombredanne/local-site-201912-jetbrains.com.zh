创建空文件

通常，开发应用程序时创建的所有文件都是基于模板的。但是，有时您可能想要创建空文件。

  1. 请执行以下任一操作：

    • 在项目工具窗口中Alt+1 ，选择要在其中创建新文件的目录或软件包，然后选择“ 文件” |“文件”。主菜单上的新功能。

    • 右键单击相应的目录或软件包，然后从上下文菜单中选择“ 新建 ”。

    • 按Alt+Insert 。

  2. 在“ 新建文件”对话框中的“ 输入新文件名”下的字段中，键入文件名和扩展名。

    您可以在新文件名之前指定整个目录结构。如果嵌套目录尚不存在，则将创建它们：

  3. 如果您指定的扩展名没有与PyCharm识别的任何文件类型相关联，则会显示“ 注册新文件类型关联”对话框。

    在此对话框中，可以将扩展名与一种公认的文件类型相关联。为此，请在“ 打开PyCharm中的匹配文件”下选择文件类型，然后单击“ 确定” 。结果，扩展名与指定的文件类型相关联。

    如果没有适用于新扩展名的文件类型，则可能要创建一个新文件类型并将扩展名与其关联。有关更多信息，请参见设置文件类型关联 。

  4. 单击确定 。
