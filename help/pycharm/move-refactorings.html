移动和复制重构

移动重构允许您在项目中移动类，函数，模块，文件和目录。这样，PyCharm会跟踪这些运动并自动更正源代码中对已移动符号的所有引用。

提供以下移动重构：

  • “ 移动文件”重构将文件移动到另一个目录。

  • “ 移动目录”重构将目录移动到另一个目录。

  • “ 移动模块成员”重构可移动Python模块的顶级符号。

  • Make本地函数/方法顶层重构将方法或本地函数转换为顶层函数，并将其移至指定文件。

将文件或目录移动到另一个目录

  1. 在“ 项目”工具窗口中选择文件或目录。

  2. 从主菜单或编辑器上下文菜单中，选择“ 重构” |“重构”。移动或按F6 。

  3. 在“ 到目录”字段中，指定要将所选文件或文件夹移动到的文件夹。从列表中选择一个现有文件夹，或键入要创建的父文件夹的完整路径。

  4. 要根据重构结果更新对选定文件或文件夹的引用，请选中“ 搜索引用”复选框。

将功能/方法移至顶层

通过将对实例属性或变量的引用从封闭范围转换为参数，并相应地更新现有用法，此重构将局部函数或方法移到了顶层。

要将函数或方法移至顶层：

  1. 将插入符号放在本地函数或方法名称上。

  2. 从主菜单或编辑器上下文菜单中，选择“ 重构” |“重构”。移动或按F6 。

  3. 在打开的“ 制作方法顶层”对话框中，指定移动的目的地。您可以手动输入，或单击浏览按钮并在“ 选择目标文件”对话框中找到目标文件。

  4. 点击重构在查找工具窗口进行重构，或预览 ，以显示预览。如果对预览结果满意，请单击“ 执行重构”确认移动。

复制重构

通过复制重构，可以在不同或相同目录中创建文件或目录的副本。

执行复制重构

  1. 在工具窗口（例如“ 项目”工具窗口）中选择感兴趣的项目 。或者，在编辑器中打开必要的类或文件。

  2. 请执行以下任一操作：

    • 选择重构|从主菜单或上下文菜单中复制 。

    • 按F5 。

    • 在“ 项目”工具窗口中，按住Ctrl ，然后将该项目拖到目标位置。

  3. 在打开的“ 复制”对话框中，指定要创建的副本的名称和位置，然后单击“ 确定” 。
