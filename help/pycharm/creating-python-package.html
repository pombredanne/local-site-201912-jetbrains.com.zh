创建Python包

PyCharm使得可以递归创建Python包，从而创建整个包结构。

Python包节点标记为图标。

创建一个新的Python包

  1. 在项目工具窗口中，选择目标目录。

  2. 从选择的上下文菜单中，选择“ 新建” |“新建”。 Python套件 ，或按Alt+Insert调用上下文菜单：

  3. 在打开的“ 新软件包”对话框中，指定软件包名称。
    您还可以指定嵌套包；在这种情况下，名称应以点分隔：


    单击确定 。 PyCharm创建新的包或包结构：
