JavaScript语言版本

从此列表中，选择JavaScript语言版本，该版本代表要在项目中使用的语言功能集。可用的选项有：

  • ECMAScript 3
  • ECMAScript 5.1
  • JavaScript 1.8.5

  • ECMAScript 6 ：此版本增加了对ECMAScript 2015-2017中引入的功能的支持，以及对该标准的一些最新建议。

  • React JSX ：此版本在ECMAScript 6之上增加了对JSX语法的支持

  • Flow ：此版本增加了对Flow语法的支持。

配置代码完成设置

单击此链接以打开“ 代码完成”页面，并在JavaScript上下文中配置完成。

流包或可执行文件

在这个领域里， 指定node_modules \ flow-bin node_modules / flow-bin软件包或Flow二进制可执行文件的路径。要使用node_modules \ .bin \ flow node_modules / .bin / flow，请确保将Node.js的路径添加到PATH环境变量。

仅当从“ JavaScript语言版本”列表中选择“ 流”时，此字段才可用。

使用Flow服务器执行以下操作：

在这方面通过选中或清除以下复选框，指定编码帮助的基础：

  • 类型检查： 选中此复选框后，将基于从Flow服务器接收的数据提供语法和错误突出显示。清除复选框后，仅基本的内部PyCharm高亮显示可用。

  • 导航，代码完成和类型提示： 选中此复选框后，用于参考解析和代码完成的建议列表g既包含从与Flow集成中检索到的建议，也包含由PyCharm计算得出的建议。清除该复选框后，将仅通过PyCharm计算来解析引用。

仅在指定Flow可执行文件的路径时，此复选框才可用。

自动保存所有修改的文件

保持选中此复选框以确保连续应用Flow，因为Flow仅在保存所有其他修改的文件之后才检查当前文件。
