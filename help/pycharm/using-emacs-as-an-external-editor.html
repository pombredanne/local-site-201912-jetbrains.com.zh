在PyCharm中使用Emacs作为外部编辑器

本教程的内容

本简短教程旨在逐步指导您将Emacs定义为PyCharm的外部编辑器。

Emacs的基础知识超出了本教程的范围。

先决条件

确保：

  • 您正在使用PyCharm 2.7或更高版本。本教程是使用PyCharm版本5创建的。牛

  • Emacs已下载并正确安装在您的计算机上。

将Emacs配置为外部工具

打开设置/首选项对话框。为此，例如，您可以选择File |设置 （在Windows和* nix上）或PyCharm |首选项 （在macOS上），或单击主工具栏上的按钮。

然后，在“ 工具”节点下，打开“ 外部工具 ”页面。在此页面上，您必须将Emacs安装指定为当前文件的外部编辑器。这是怎么做的...

  1. 首先，在“ 外部工具”页面中，单击 。将打开“ 创建/编辑工具”对话框。

  2. 在此对话框中，执行以下操作：

    • 输入工具名称（Emacs）和可选描述（Open Emacs）

    • 指定组的名称，Emacs将在该名称下显示在“ 工具”菜单中。在此示例中，组名称为Editors 。此步骤是可选的-如果您未指定组名，则Emacs将原样显示在“ 工具”菜单中。

    • 清除“ 打开控制台 ”复选框。

    • 指定Emacs二进制文件位置。您可以手动输入，也可以单击省略号按钮，然后在文件系统中找到所需的二进制文件。

    • 由于要在Emacs中打开当前文件，因此将文件路径作为参数传递给程序：在Parameters字段中，键入$FilePath$ 。

    • 最后，指定工作目录-在我们的示例中，这是$ProjectFileDir$

    • 单击确定 。

  3. 应用更改并关闭“设置/首选项”对话框。
