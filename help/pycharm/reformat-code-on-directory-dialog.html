在目录对话框中重新设置代码格式

代码重新格式化代码
Ctrl+Alt+L

当您在目录上选择“ 重新格式化代码”操作时，将出现此对话框。

项目

描述

目录

这是一个只读字段，显示所选目录的名称。

选件

此区域显示代码重新格式化的选项。
您可以从以下选项中选择：

  • 包括子目录：选中此复选框可重新格式化嵌套目录中文件的源代码。仅当嵌套目录存在时，此复选框才会出现。

  • 优化导入：选中此复选框可从选定范围内的代码中删除未使用的导入语句。

  • 重新排列条目：选中此复选框可以重新排序源代码条目。

  • 清理代码：选择此选项可运行代码清理检查。

  • 仅VCS更改了文本：如果选中此复选框，则重新格式化将仅应用于已在本地更改但尚未签入到存储库中的文件。

    此复选框仅适用于受版本控制的文件。

筛选器

此区域显示可应用于代码重新格式化操作的过滤器选项。
您可以从以下选项中选择：

  • 范围：选中此复选框，从列表中选择要应用重新格式化选项的范围。如果要配置自定义范围，请单击按钮打开“ 范围”对话框。

  • 文件掩码：选中此复选框，从列表中选择要对其应用格式设置选项的文件扩展名。

跑

单击此按钮开始在指定范围内重新格式化源代码。
