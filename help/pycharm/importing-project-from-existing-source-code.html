从现有源代码导入项目

您可以围绕外部创建的现有源代码（即在另一个IDE或专用编辑器中）设置项目。PyCharm分析代码库，并在.idea目录中添加设置。

从现有源代码创建项目

从现有源代码创建新项目

  1. 在主菜单中，选择“ 文件” |“文件”。打开 。

  2. 在打开的对话框中 ，选择包含所需源代码的目录。请注意，外部创建的应用程序带有常规目录图标标记 。

  3. 单击确定 。

  4. 指定是要在单独的窗口中打开新项目还是要关闭当前项目并重用现有项目。有关详细信息，请参见“ 打开多个项目 ”部分。
