PyCharm 2019.2帮助

PhoneGap /科尔多瓦

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

PhoneGap ， Apache Cordova和Ionic是用于使用单个HTML，CSS和JavaScript / TypeScript代码库开发移动应用程序并针对包括Android在内的各种移动平台的框架。

在你开始前

  1. 如“ 管理插件”中所述，在“ 插件”页面上安装并启用PhoneGap / Cordova存储库插件。

  2. 下载并安装Node.js。

  3. 根据目标平台和所使用的操作系统下载并安装仿真器工具。从Cordova官方网站了解更多信息。

安装PhoneGap / Cordova / Ionic

  • 在嵌入式终端 （ Alt+F12 ），键入以下命令之一：

    • npm install -- global phonegap

    • npm install -- global cordova

    • npm install -- global ionic

您还可以按照npm和Yarn中的说明将程序包安装在Node.js和NPM页面上 。

创建一个新的PhoneGap / Cordova / Ionic应用程序

如果还没有应用程序，则可以从样板模板生成具有PhoneGap / Cordova / Ionic特定结构的PyCharm项目。或者，创建一个空的PyCharm项目，并按照下面的“ 从现有Meteor应用程序开始”中的描述在其中配置PhoneGap / Cordova / Ionic支持。

从模板创建PhoneGap / Cordova / Ionic应用程序

  1. 选择文件|新品|项目从主菜单或单击欢迎屏幕上创建新项目 。将打开“ 新建项目”对话框。

  2. 在左侧窗格中，选择PhoneGap / Cordova App 。

  3. 在右侧窗格中：

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. 指定可执行文件phonegap.cmd或cordova.cmd或ionic.cmd的位置 （请参阅安装PhoneGap / Cordova / Ionic ）。

  4. 当您单击Create时 ，PyCharm会生成具有框架特定结构的PhoneGap / Cordova / Ionic应用程序的框架。

创建一个空的PyCharm项目

  1. 选择文件|新品|项目从主菜单或单击欢迎屏幕上创建新项目 。将打开“ 新建项目”对话框。

  2. 在左侧窗格中，选择空项目 。在右侧窗格中，指定应用程序文件夹，然后单击创建 。

从现有的PhoneGap / Cordova / Ionic应用程序开始

  1. 通过选择文件|在PyCharm中打开所需的PhoneGap / Cordova / Ionic应用程序。 打开主菜单或点击欢迎屏幕 打开 。

  2. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |科尔多瓦 。将打开“ 科尔多瓦”页面。

  3. 如果PyCharm未自动检测到可执行文件，请检查可执行文件phonegap.cmd或cordova.cmd或ionic.cmd的位置，或指定其路径。

    PyCharm会检测到已安装的版本，并将其显示在PhoneGap / Cordova版本只读字段中，

  4. 在“ 科尔多瓦工作目录”字段中， 指定要存储要运行的PhoneGap / Cordova / Ionic应用程序文件的文件夹。

  5. 在“ 插件”区域中， 通过安装所需的软件包，配置要在开发中使用的插件列表。该列表显示了全局和项目级别上当前安装在计算机上的所有PhoneGap / Cordova / Ionic插件。

    • 要安装插件，请单击 。在打开的“ 可用软件包”对话框中，选择所需的软件包。

      要全局安装软件包，请选择选项复选框，然后键入--global在该领域。单击安装软件包 。

    • 要删除插件，请在列表中选择它，然后单击 。

    • 要将插件升级到最新的可用版本，请在列表中选择插件，然后单击 。

    有关插件及其使用的信息，请参见Apache Cordova插件和PhoneGap插件 。

创建和启动PhoneGap / Cordova / Ionic运行配置

PhoneGap / Cordova / Ionic应用程序根据专用的运行/调试配置执行。

  1. 从主菜单中，选择运行|编辑配置 。在打开的“ 编辑配置”对话框中，单击在工具栏上，然后从上下文菜单中选择PhoneGap / Cordova 。

  2. 在打开的“运行/调试配置：PhoneGap / Cordova”对话框中，指定以下内容：

    1. 配置的名称。

    2. 在“ PhoneGap / Cordova可执行路径”字段中， 指定可执行文件phonegap.cmd ， cordova.cmd或ionic.cml的位置 （请参阅安装PhoneGap / Cordova / Ionic ）。

    3. 在“ PhoneGap / Cordova工作目录”字段中， 指定要存储要运行的PhoneGap / Cordova / Ionic应用程序文件的文件夹。

    4. 在命令列表中， 选择要运行的命令。列表的内容取决于实际使用的框架，即取决于PhoneGap / Cordova可执行路径字段中指定的可执行文件。可用的选项有：

      • 对于PhoneGap ：

        • 仿真

        • 跑

        • 准备

        • 服务

        • 远程构建

        • 远程运行

        有关特定于PhoneGap的命令的说明，请参见PhoneGap CLI 。

      • 对于科尔多瓦 ：

        • 仿真

        • 跑

        • 准备

        • 服务

        请参阅Cordova CLI以获取特定于Cordova的命令及其说明的列表。

      • 对于离子 ：

        • 仿真

        • 跑

        • 准备

        • 服务

        有关特定于离子的命令的列表，请参见Ionic CLI 。

    5. 在平台列表中， 选择应用程序所针对的平台。可用的选项有：
      • 安卓系统

      • ios要模拟此平台，您需要全局安装ios-sim命令行工具 。您可以通过Node Package Manager（npm）进行操作 ，请参阅npm和Yarn或通过运行sudo npm install ios-sim -g命令，具体取决于您的操作系统。

      • 亚马逊-菲洛斯
      • 火狐
      • 黑莓10
      • ubuntu
      • wp8
      • Windows8的
      • 浏览器

      从平台指南中了解有关目标平台的更多信息。

    6. 对于Cordova和Ionic ，指定要在其上运行该应用程序的目标虚拟或物理Android设备：选中“ 指定目标”复选框，然后从列表中选择所需的设备。

      该列表显示了我们计算机上当前配置的所有虚拟和物理设备。有关详细信息。从Cordova官方网站上了解有关设置模拟器的更多信息。

      如果PyCharm显示以下错误消息： 无法在path中检测到ios-sim ，请确保已安装ios-sim请参阅开始之前 。

  3. 要运行PhoneGap / Cordova / Ionic应用程序， 从主工具栏上的列表中选择所需的运行配置，然后转到“运行” |“运行”。跑从主菜单或单击 。
