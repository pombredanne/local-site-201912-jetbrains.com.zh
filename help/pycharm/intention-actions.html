意向动作

在编辑器中工作时，PyCharm会分析您的代码，搜索优化代码的方法，并检测潜在的和实际的问题。

IDE一旦找到更改代码的方法，就会显示黄色的灯泡图标在当前行旁边的编辑器中。通过单击此图标，您可以查看可用于此代码单元的意图操作 。意图动作涵盖了从警告到优化建议的多种情况。您可以查看意图的完整列表并禁用它们 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ）。

代码检查还提供快速修复，但是它们具有不同的设置。有关更多信息，请参阅代码检查 。

带有感叹号的红色灯泡表示PyCharm已检测到问题，并提示您选择关联的修复程序- 快速修复程序 。

应用意图动作

  1. 点击灯泡图标（或按Alt+Enter ）以打开建议列表。

  2. 从列表中选择一个动作，然后按Enter 。

    这适用于Python 3.6！

    例如，PyCharm提供了一种转化意图.format()呼叫和printf样式格式转换为f字符串文字。此意图称为“ 转换为f字符串文字” ，其工作方式如下：

    之前

    后

    var ='你好％s！' ％ '约翰'
    var = f'hello {“ John”}！'
    var ='hello {}！'.format（'John'）
    var = f'hello {“ John”}！'

    f字符串文字在编辑器中的显示方式取决于相应的配色方案设置。 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），导航到“ 编辑器” |配色方案| Python ，选择String |组件列表中的f-string ，然后根据需要修改可用的设置。

查看所有意图动作

您可以查看可用意图动作的完整列表，并禁用不需要的动作：

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |用心 。

  2. 对于每个意图动作，您都可以看到说明和示例。要禁用意图，请清除其旁边的复选框。

  3. 应用更改并关闭对话框。

即时停用意图动作

  1. 点击灯泡图标（或按Alt+Enter ）以打开建议列表。

  2. 选择要禁用的操作，然后单击向右箭头。

  3. 选择禁用 。
