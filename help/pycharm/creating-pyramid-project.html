创建金字塔项目

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

金字塔工程支架旨在用于金字塔应用程序的高效开发。PyCharm负责创建特定的目录结构和设置。

若要进行一个金字塔项目，请按照下列步骤

  1. 在主菜单中，选择“ 文件” |“文件”。新建项目... ，或在欢迎屏幕中单击新建项目按钮。“新建项目”对话框打开。

    在“ 新建项目”对话框中，执行以下操作：

    • 指定项目类型Pyramid 。

    • 指定项目位置。

    • 接下来，点击单击相应的单选按钮以展开“ 项目解释器”节点，并选择新环境或现有解释器。以下步骤取决于您的选择：

      • 使用新环境 ：如果已选择此选项，请选择用于创建虚拟环境的工具。为此，请单击列表，然后选择Virtualenv ， Pipenv或Conda 。

        接下来，指定新虚拟环境的位置和基本解释器。如有必要，请单击“ 继承全局站点包”和“对所有项目可用”复选框。

      • 现有解释器 ：如果选择了此选项，请从列表中选择所需的解释器，或者（如果找不到所需的解释器），请单击并选择翻译。有关详细信息，请参见配置Python解释器 。

        当PyCharm停止支持任何过时的Python版本时，相应的项目解释器将标记为不支持。

    请点击 （ 更多设置 ），然后指定以下内容：

    • 从列表中，选择要使用的支架。可能的选项是：

      • started
      • alchemy
      • zodb

    • 从列表中，选择要使用的模板语言 。

    • 在模板文件夹字段中， 指定模板的存储目录以及从中加载模板的目录。您可以指定尚不存在的目录的名称。在这种情况下，将创建目录。

    点击创建 。

  2. 如果弹出警报，

    单击否 。我们正在创建一个新项目。

    PyCharm创建一个项目，安装Pyramid及其依赖项，并生成特定的目录结构，您可以在Project工具窗口中进行浏览。打开项目目录中的任何文件。如果对包装的要求不满意 ，PyCharm建议解决或忽略它们：

    您应该安装依赖项才能运行开发服务器。

    创建金字塔项目时，必须运行setup.py develop安装要开发的项目。PyCharm可能会通知您这样做，如以下屏幕截图所示：

    或者，您可以选择工具|运行setup.py Task ...并输入develop 。出现另一个弹出窗口， 运行设置任务开发 。单击确定 。

    如果创建了基于炼金术的项目，则需要初始化数据库，打开终端（确保虚拟环境处于活动状态）并运行以下命令：

    initialize_PROJECTNAME_db development.ini

    （更换PROJECTNAME并附上您确切的项目名称。）
