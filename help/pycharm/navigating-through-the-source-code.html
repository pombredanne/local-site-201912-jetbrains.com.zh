源代码导航

您可以使用不同的操作和弹出窗口在编辑器中快速浏览代码。有关在编辑器和工具窗口之间导航的详细信息，请查看编辑器基础知识 。

您可以从观看视频教程开始，以检查PyCharm中可用的导航选项。

查找文件路径

  1. 在编辑器中，按Ctrl+Alt+F12或在上下文菜单中，选择“ 在Finder中显示” 。

  2. 在“ 在查找器中显示”弹出窗口中，选择要在路径查找器中打开的文件或目录，然后按Enter 。

使用“选择范围”弹出窗口进行导航

您可以在“ 项目”工具窗口中自动找到文件。

  1. 如果在编辑器中打开了文件，请按Alt+F1打开“ 选择范围”弹出窗口。

  2. 在弹出窗口中，选择“ 项目视图” ，然后按Enter 。PyCharm在“ 项目”工具窗口中找到您的目标。

使用自动滚动查找文件

您可以使用“ 自动滚动到源”和“ 从源自动滚动”操作在“ 项目”工具窗口中找到文件。

  1. 在“ 项目”工具窗口中，右键单击“ 项目”工具栏，然后从上下文菜单中选择“ 从源代码自动滚动” 。之后，PyCharm将跟踪当前在活动编辑器选项卡中打开的文件，并自动在“ 项目”工具窗口中找到它。

  2. 您还可以选择“ 自动滚动到源”选项。在这种情况下，当您在“ 项目”视图中单击文件时，PyCharm会自动在编辑器中将其打开。

使用书签进行导航

  • 要创建匿名书签，请将插入符号放在所需的代码行中，然后按F11 。

  • 要创建带有助记符的书签，请将插入号放置在所需的代码行中，按Ctrl+F11并选择一个数字或字母作为助记符。

  • 要显示下一个或上一个书签，请在主菜单中选择“ 导航”。书签|下一个书签或导航|书签|上一个书签 。

  • 要打开“ 书签”对话框，请按Shift+F11 。您可以使用此对话框来管理书签，例如，删除书签，对书签排序或为其提供简短说明。

  • 要导航到带有字母助记符的现有书签，请按Shift+F11然后按一下您需要的字母。PyCharm将您带回到编辑器和相应的书签。

  • 要导航到带有数字助记符的现有书签，请按Ctrl和书签的编号。

  • 每个创建的书签都会反映在“ 收藏夹”中 Alt+2工具窗口，您还可以使用该窗口导航到书签。

转到声明及其类型

您可以从符号的用法导航到符号的初始声明和符号类型。

  • 将插入符号放在所需的符号上，然后按Ctrl+B 。

    或者，保持Ctrl按下，将鼠标指针悬停在该符号上。当符号变成超链接时，其声明将显示在工具提示中。单击超链接而不释放键，以在编辑器中打开声明。

  • 对于类型声明，请按Ctrl+Shift+B 。

前往实施

您可以使用装订线图标或按相应的快捷方式来跟踪类的实现和重写方法。

  • 单击其中一个 / ， / 装订线图标，然后从列表中选择所需的上升或下降类别。

  • 要导航到超级方法，请按Ctrl+U 。

  • 要导航到实现，请按Ctrl+Alt+B 。

在错误或警告之间导航

  • 要跳至代码中的下一个或上一个发现的问题，请按F2要么Shift+F2分别。或者，从主菜单中选择“ 导航|导航”。下一个/上一个突出显示的错误 。

    PyCharm在代码发布之前立即将插入号放置。

  • 配置PyCharm在代码问题之间导航的方式：它可以在所有代码问题之间跳转或跳过次要问题，并且仅在检测到的错误之间导航。右键单击滚动条区域中的代码分析标记，然后从上下文菜单中选择一种可用的导航模式：

    • 要让PyCharm跳过警告，信息和其他次要问题，请选择仅转到高优先级问题 。

    • 要让PyCharm在所有检测到的代码问题之间跳转，请选择转到下一个问题 。

在“结构”视图弹出窗口中找到一个代码元素

您可以使用结构视图弹出窗口在正在处理的文件中找到代码元素。

  1. 要打开结构视图弹出窗口，请按Ctrl+F12 。

  2. 在弹出窗口中，找到您需要的项目。您可以开始为PyCharm输入元素的名称以缩小搜索范围。按Enter返回到编辑器和相应的元素。

    在弹出窗口中，您可以对文件成员进行排序，查看匿名类和继承的成员。
