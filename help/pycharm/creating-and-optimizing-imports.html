自动导入

当您引用尚未导入的类时，PyCharm可帮助您找到该文件并将其添加到导入列表中。您可以导入单个类或整个程序包，具体取决于您的设置。

将import语句添加到imports部分，但是插入符号不会从当前位置移动，并且当前的编辑会话不会挂起。此功能称为“ 导入助手” 。

要配置自动导入， 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|自动导入 ，并使用相应的XML，Python和TypeScript / JavaScript 选项 。

使用导入助手是在PyCharm中处理导入的首选方法，因为不支持通过命令行进行导入优化。

同样的可能性也适用于XML文件。当您输入带有未绑定名称空间的标签时 ，导入助手会建议创建一个名称空间并提供适当选择的列表。

优化进口

迟早一些导入的类或程序包对代码而言变得多余。PyCharm提供了“ 优化导入”功能，该功能使您可以在方便时从当前文件或当前目录中的所有文件中一次删除未使用的导入。这可以帮助您避免项目中未使用，过多和重复的导入。可以删除整个项目或仅当前文件中未使用的导入语句。

除了清除未使用的导入中的代码外，PyCharm还根据Python代码样式指南格式化现有的导入语句。在这种情况下，PyCharm将import语句拆分为单独的行，并将其排序为组（有关详细信息，请参见Imports部分）。

另外，导入在各个组中按字母顺序和区分大小写进行排序：

您可以在Python代码样式设置（ “设置/首选项” |“编辑器” |“代码样式” |“ Python” ）的“ 导入”选项卡中修改排序策略。有关更多信息，请参见Python代码样式设置 。

在整个项目中优化进口

  1. 将焦点切换到“项目”工具窗口，然后执行以下操作之一：

    • 在主菜单上，选择“代码”。优化进口 。

    • 按Ctrl+Alt+O 。

    将打开“ 优化导入”对话框。

  2. 如果您的项目处于版本控制下，则启用“ 仅VCS更改的文件 ”选项。根据需要选择或清除此选项。

  3. 点击运行 。

处理未使用的导入的一种方法是使用将突出显示符号设置在突出显示的未使用的导入时出现的快速修复。但是，可以如下所述在更大的范围内优化导入。

优化当前文件中的导入

  1. 在主菜单上，选择“代码”。优化进口 。

  2. 按Ctrl+Alt+O 。

  3. 将插入符号放在导入语句中，单击 ，然后选择删除未使用的导入 。

  4. 打开“ 重新格式化文件”对话框 Ctrl+Shift+Alt+L并选择优化导入复选框。

在代码完成时添加导入语句

当您完成导出的JavaScript或TypeScript符号时，PyCharm将添加导入语句。

您可以在完成时禁用自动导入，而使用快速修复：

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|自动导入 。

  2. 在打开的“ 自动导入”页面上，使用TypeScript / JavaScript区域中的复选框启用或禁用代码完成时的导入生成。
