创建本地服务器配置

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

本地服务器是在本地或已安装文件夹中运行的服务器，其文档根目录 不是项目根目录的父级。您进行开发，然后将源复制到服务器。

要在此设置中配置对服务器的访问，需要指定以下内容：

  1. 服务器配置根文件夹和访问它的URL地址。

  2. 项目根文件夹 ，将数据从项目根文件夹复制到的服务器上的文件夹以及访问服务器上复制的数据的URL地址之间的对应关系。这种对应称为映射 。

指定服务器配置的名称，类型和可见性

  1. 通过执行以下操作之一打开“ 部署”页面：

    • 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择部署下建立，执行，部署 。

    • 选择工具|部署|从主菜单进行配置 。

  2. 在列出所有现有服务器配置的左侧窗格中，单击“ 添加”。 并在弹出菜单中选择本地或已安装文件夹 。

  3. 在打开的“ 创建新服务器”对话框中，键入要创建的服务器的名称，然后单击“ 确定” 。关闭“ 创建新服务器”对话框，然后返回到“ 部署”节点的“ 连接”选项卡。

  4. 单击用作默认值 在以下情况下，让PyCharm静默应用当前配置：

    • 自动上传更改的文件 。

    • 手动上传和下载文件，而无需选择目标主机。

    • 将本地文件和文件夹与其远程版本进行比较。

    • 编辑远程主机上的单个文件 。

  5. 仅在此项目中使用可见复选框配置服务器访问配置的可见性。

    • 选中复选框以将配置的使用限制为当前项目。这样的配置不能在当前项目外部重用。它不会出现在其他项目的可用配置列表中。

      服务器记录与项目一起存储在.idea目录中，从而允许通过VCS在团队成员之间共享它们。

      在左侧窗格的服务器访问配置列表中，仅在当前项目中可见的配置标记有图标。

    • 清除复选框后，该配置在所有PyCharm项目中均可见。它的设置（包括SFTP服务器的SSH凭据）可以在多个项目中重复使用。

指定服务器配置根目录和访问它的URL地址

  1. 切换到“ 映射”选项卡。

  2. 在文件夹字段中， 指定服务器配置根目录 。

    服务器配置根目录是服务器上文件树中可通过服务器配置访问的最高文件夹。最简单的方法是使用Web服务器配置文件中定义的Web服务器的文档根目录。但是，您可以在文档根目录下指定任何其他现有文件夹。

  3. 在“ Web服务器URL”字段中， 指定服务器配置根目录的URL地址。该URL地址将是构建应用程序URL地址的起点。根据您对服务器配置根目录的选择，执行下列操作之一：

    • 键入与Web服务器配置文件中定义的与Web服务器的文档根目录关联的URL地址。

    • 以以下格式输入网址/ 。

    HTTP和HTTPS协议均受支持。

将项目文件夹映射到服务器上的文件夹和URL地址以访问它们

配置映射 ，即设置项目文件夹，服务器上用于将项目文件复制到的文件夹和URL地址之间的对应关系，以访问服务器上复制的数据。最简单的方法是将整个项目根文件夹映射到服务器上的文件夹。如果在“ 选项”对话框中选择了“ 创建空目录”复选框，则在服务器上将重复这种情况下的项目文件夹结构。有关更多详细信息，请参阅配置与Web服务器的同步和自定义上载/下载 。

  1. 切换到“ 映射”选项卡。

  2. 在本地路径字段中， 指定所需本地文件夹的完整路径。在最简单的情况下，它是项目的根目录。

  3. 在“ 部署路径”字段中， 指定服务器上的文件夹，PyCharm将在该文件夹中从“ 本地路径”字段中指定的文件夹上载数据。输入相对于服务器配置根目录的文件夹路径。

    如果指定名称的文件夹尚不存在，只要您在“ 选项”对话框中选中了“ 创建空目录”复选框，PyCharm便会创建该文件夹。有关更多详细信息，请参见自定义上传/下载 。

  4. 在“ Web路径”字段中， 键入相对于服务器配置根目录的服务器上文件夹的路径。实际上，在“ 部署路径”字段中键入您输入的相对路径。

指定服务器配置根目录的示例

例如，服务器的默认文档根目录是htdocs文件夹，访问数据的默认URL地址是http：// localhost 。假设您正在使用XAMPP软件包，它已安装在C：/驱动器的根目录中。

如果您决定直接在服务器文档根目录下复制项目文件，则服务器配置根目录将为C：\ xampp \ htdocs ，其URL将为http：// localhost： 。

您可以在服务器上建立更复杂的文件夹结构，例如，在服务器文档root下具有MySite1和MySite2文件夹。在这种情况下，您将必须决定在当前配置中使用哪个文件夹，将其设为MySite2 。因此， 服务器配置根目录将为C：\ xampp \ htdocs \ MySite2 ，其URL地址将为http：// localhost： \ MySite2 。

使用服务器上的文件夹映射项目文件夹的示例

例如，如果您的项目位于C：\ My_Projects \ Mapping_project下 ，则服务器文档根目录为C：\ xampp \ htdocs ， 服务器配置根目录为C：\ xampp \ htdocs \ MySite2 ，其URL地址为http：/ / localhost： \ MySite2 ，按如下所示填写字段：

  1. 在“ 本地路径”字段中，键入C：\ My_Projects \ Mapping_project 。

  2. 在“ 部署路径”字段中，键入Mapping_project 。

  3. 在“ Web路径”字段中，键入Mapping_project 。
