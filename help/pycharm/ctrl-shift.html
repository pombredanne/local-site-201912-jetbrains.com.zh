Ctrl + Shift

本节列出并描述了键盘快捷方式，包括Ctrl+Shift按键：

  • Ctrl + Shift +字母数字键
  • Ctrl + Shift +导航键
  • Ctrl + Shift +符号键
  • Ctrl + Shift +数字键
  • Ctrl + Shift +功能（F）键

Ctrl + Shift +字母数字键

捷径

功能

使用此快捷方式可以...

Ctrl+Shift+A

寻找行动

查找操作，绕过菜单。请参阅查找动作 。

Ctrl+Shift+B

导航到类型声明

导航到插入符号处的变量或方法调用的类型声明 。

N/A

导航到最近更改的文件

显示最近更新的文件列表。

Ctrl+Shift+F

在路径中查找

在指定的路径中启动文本搜索 。

Ctrl+Shift+H

方法层次结构

浏览所选类的层次结构。

Ctrl+Shift+J

连接线

将选定的行连接为一个，或将插入符号当前所在的行与下一行连接。

Ctrl+Shift+N

导航到文件

跳转到项目中的指定文件 。

Ctrl+Shift+R

替换为路径

在指定的路径中启动文本替换 。

Ctrl+Shift+U

拨动套

切换所选文本片段的大小写 。

Ctrl+Shift+V

从历史粘贴

从最近的剪贴板粘贴。参见页面选择代码结构

Ctrl+Shift+W

在插入符号处取消选择单词

删除“ 在插入符号处选择单词”操作所进行的顺序选择。

Ctrl+Shift+Z

重做

重做上一个撤消操作 。

Ctrl + Shift +导航键

捷径

功能

使用此快捷方式可以...

Ctrl+Shift+End

移至以选择结尾的文本

从当前插入符号位置选择文本到文本结尾，然后将插入符号移动到文本结尾。请参阅使用源代码页。

Ctrl+Shift+Home

从选择开始移到文本

从当前插入符号位置选择文本到文本的开头，然后将插入符号移动到文本的开头。请参阅使用源代码页。

Ctrl+Shift+Right

移至Word End with Selection

从当前插入符号的位置选择文本到单词的结尾，然后将插入符号移动到单词的结尾。请参阅使用源代码页。

Ctrl+Shift+Left

移至Word从选择开始

从当前插入符号的位置选择文本到当前单词的开头，然后将插入符号移动到该单词的开头。请参阅使用源代码页。

Ctrl+Shift+V

从历史粘贴

从最近的剪贴板粘贴。参见页面选择代码结构

Ctrl+Shift+SpaceSmartType代码完成

完整的代码，根据期望的类型过滤查找列表。

Ctrl+Shift+Page Down

导航到选择的页面底部

将插入记号向下移动到页面底部，选择文本。请参阅使用源代码页。

Ctrl+Shift+Page Up

导航到带有选择的页面顶部

将插入符号向上移动到页面底部，选择文本。请参阅使用源代码页。

Ctrl+Shift+Down

下移线

将插入符下移一行，以保持句法正确性。参见页面代码行 。

Ctrl+Shift+Up

向上移动

插入标记处向上移，以保持语法正确性。参见页面代码行 。

Ctrl+Shift+Backspace

最后编辑位置

跳至上一次编辑的位置。

Ctrl + Shift +符号键

捷径

功能

使用此快捷方式可以...

Ctrl+Shift+[

从选择开始移至代码块

将插入符号移动到当前代码块的开头，从初始插入符号位置选择代码。请参阅使用源代码页。

Ctrl+Shift+]

选择后移至代码块结尾

将插入符号移动到当前代码块的末尾，从初始插入符号位置选择代码。请参阅使用源代码页。

Ctrl+Shift+/

带块注释

带块注释的注释/取消注释代码。参见页面代码行 。

Ctrl+NumPad Plus

展开全部

展开所有折叠块。请参见代码折叠 。

Ctrl+Shift+NumPad -

全部收缩

折叠所有折叠块。请参见代码折叠 。

Ctrl + Shift +数字键

捷径

功能

使用此快捷方式可以...

Ctrl+Shift+Numpad/

带块注释

带块注释的注释/取消注释代码。参见页面代码行 。

Ctrl+Shift+Numpad+
Ctrl+NumPad Plus

展开全部

展开所有折叠块。请参见代码折叠 。

Ctrl+Shift+Numpad-

折叠所有折叠块。请参见页面代码折叠 。

Ctrl + Shift +功能（F）键

捷径

功能

使用此快捷方式可以...

Ctrl+Shift+F4

关闭活动标签

关闭工具窗口中的活动选项卡。参见页面导航 。

Ctrl+Shift+F7

突出显示文件/突出显示方法出口点中的用法

突出显示插入符号当前所在位置的符号的用法 。如果将插入号放置在方法的出口之一上，例如return ，所有方法出口点都突出显示。

Ctrl+Shift+F8查看断点

查看/管理所有断点/观察点
