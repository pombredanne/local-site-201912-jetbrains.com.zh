使用TextMate套件

项目可以包含PyCharm未知的文件类型。虽然PyCharm带有对许多编程和脚本语言的内置支持。如果要突出显示特定于项目的语言的语法，请使用功能强大的PyCharm与文本编辑器TextMate的集成。

所有可用的捆绑软件都列在“ 编辑器” |“ TextMate捆绑包页面在“ 设置/首选项”对话框中（ Ctrl+Alt+S ）。

学习如何使用TextMate语法描述语言超出了本文档的范围。有关TextMate的更多详细信息，请参阅产品文档 。

先决条件

确保：

  • 您已经下载了要使用的捆绑软件。例如，您可以找到要在GitHub或Subversion上安装的捆绑软件。

  • 在开始使用TextMate捆绑软件之前，请确保已启用TextMate捆绑软件支持插件。该插件与PyCharm捆绑在一起，并且默认情况下处于激活状态。如果未激活插件，请在“ 插件”页面上启用它 设置/首选项对话框（ Ctrl+Alt+S ），如“ 管理插件”中所述 。

    如果您正在使用Windows的Docker，请在Docker设置的`` 常规''部分中启用tcp：// localhost：2375上没有TLS选项的Expose守护程序 。

导入捆绑

假设您希望PyCharm突出显示OCaml文件的语法。为此，您已经下载了OCaml TextMate软件包 。现在它位于您的硬盘上，您只需要将此捆绑包导入PyCharm。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择编辑器| TextMate捆绑包 。
