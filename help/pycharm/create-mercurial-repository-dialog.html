创建Mercurial存储库对话框

VCS |创建Mercurial存储库

使用此对话框可以在您选择的文件夹中创建本地Mercurial存储库。

项目

描述

为整个项目创建存储库

选择此选项可在项目根目录中初始化存储库。如果要将整个项目置于Mercurial控制下，则此选项很有用。

选择创建存储库的位置

选择此选项可在项目根目录下的一个文件夹中初始化存储库。使用此选项，可以使项目的文件夹处于不同版本控制系统的控制之下。在下面的文本框中，指定要在其中创建存储库的文件夹。手动输入路径或单击浏览 然后在打开的对话框中选择所需的文件夹。
