Navigation bar

View | Navigation Bar
Alt+Home

The Navigation Bar is a quick alternative to the Project Tool Window. On the left-hand side of the bar, you can navigate through the project and open files for editing .

On the right-hand side, you can access the run/debug configuration selector, run and debug buttons, version control buttons (if the version control integration is enabled) and the search everywhere magnifying glass .
