Ctrl+Alt+Shift

This section lists and describes the keyboard shortcuts that include the Ctrl+Shift+Alt keys.

Shortcut

Function

Use this shortcut to...

Ctrl+Shift+Alt+C

Copy Relative Path

Copy a reference (a relative path) of a symbol to the Clipboard.

Ctrl+Shift+Alt+N

Go to Symbol

Navigate to a symbol with the specified name.

Ctrl+Shift+Alt+H

Pop up Hector

Open the Highlighting level pop-up window.

Ctrl+Shift+Alt+U

Show Uml Diagram

Open UML Class diagram for a class or package.

Ctrl+Shift+Alt+V

Paste Simple

Paste the last entry from the Clipboard as plain text.

Ctrl+Shift+Alt+L

Show Reformat File Dialog

Show reformatting dialog.

Ctrl+Shift+Alt+I

Run Inspection by Name

Execute an inspection by its name.

Ctrl+Shift+Alt+Insert

New Scratch File

Create a new scratch file with the selected language.
