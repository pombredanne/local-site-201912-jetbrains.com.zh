Find Usages. Variable Options

Edit | Find | Find Usages
Alt+F7

This section describes Field, Variable, or Parameter usage search options in the Find Usages dialog.

The dialog opens when you click in the Show Usages pop-up window which lists all the occurrences of the symbol at caret.

Item

Description

Skip results tab with one usage

Select this checkbox to be navigated directly to the found usage without the Find tool window displayed, when only one usage is found.

Scope

In this area, specify the scope of search. Select a pre-defined scope from the drop-down list or click the Browse button to open the Scopes dialog box, where you can define a custom scope.

Open in new tab

Select this checkbox have the results of each search shown in a separate tab of the Find Results window. If the checkbox is cleared, the search results will overwrite the contents of the current tab.
