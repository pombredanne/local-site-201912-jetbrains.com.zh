IntelliLang

The following is only valid when IntelliLang Plugin is installed and enabled!

This feature is not available in the Educational edition of PyCharm.

This a third-party documentation.

The IntelliLang IntelliLang plugin offers a number of features related to the use of custom languages in PyCharm.

PyCharm implements the language injections functionality with a bundled plugin, which can be completely disabled by clearing the IntelliLang IntelliLang check box on the the Plugins page of PyCharm settings (Ctrl+Alt+S).
