PyCharm 2019.2帮助

查看源代码的结构和层次结构

此功能在PyCharm的教育版中不可用。

通过PyCharm，您可以在“层次结构”工具窗口中检查类，方法和调用的层次结构，并在“结构”工具窗口中探索源文件的结构。

  • “ 视图”菜单中提供了“ 层次结构”和“ 结构”工具窗口。

  • 仅当构建层次结构时，“ 层次结构”工具窗口才可用。

  • 层次结构是在“ 导航”菜单中构建的。

在这一部分：

  • 查看层次结构
  • 查看源文件的结构
  • 建筑类层次结构
  • 保留层次结构选项卡
