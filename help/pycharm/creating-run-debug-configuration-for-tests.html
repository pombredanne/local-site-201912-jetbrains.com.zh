创建测试的运行/调试配置

您可以运行测试（测试用例，测试套件等），可以使用运行/调试配置来运行测试，其方式类似于运行普通应用程序。PyCharm提供了一个框架，用于为测试目的创建特殊的运行/调试配置，可以将测试指定为目标。

除了创建和编辑运行/调试配置部分中描述的常规过程之外，PyCharm还提供了一个快捷方式，可让您为容器中的所有测试，单个测试用例甚至测试方法创建运行/调试配置。 。

创建测试配置

  1. 在“ 项目”工具窗口中，右键单击所需的目标，例如目录或单个测试。如果是单个测试，则可以在编辑器中将其打开，然后右键单击背景。

  2. 从选择的上下文菜单中，选择创建对于单个测试：

    对于测试方法，请在编辑器中打开该类，然后右键单击该方法中的任何位置。上下文菜单建议使用命令Create <方法名称> 。

  3. 在打开的对话框中，指定运行/调试配置参数，应用更改并关闭对话框。

    Ctrl+Space代码完成可在“运行/调试”配置对话框中使用，例如在“ 工作目录”字段中。
