设置属性对话框（Subversion）

VCS |颠覆|设置属性

使用此对话框可以为SVN版本控制下的文件和文件夹定义SVN特定的属性（忽略列表，外部等）。

项目

描述

物业名称

在文本字段中输入自定义属性名称，或使用下拉列表选择预定义属性之一。

设置属性值

单击此单选按钮可在文本区域中设置指定属性名称的值。可以在多行中输入接受多个值的属性，例如忽略列表。

删除财产

单击此单选按钮，从列表中删除选定的属性。

递归更新属性

如果要将属性应用于所选目录下的每个文件和目录，请选中此选项。
