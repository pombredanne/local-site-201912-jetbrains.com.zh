数据编辑器提供了用于处理表数据的用户界面。在数据编辑器中，您可以对数据进行排序，过滤，添加，编辑和删除以及执行其他相关任务。

重新载入页面

 Ctrl+F5

重新加载表视图的数据，以使您在编辑器中看到的数据与数据库的内容同步。另外，如果要在更改后应用新的页面大小限制设置，请使用“ 重新加载页面”按钮。

添加新行

 Alt+Insert

在表中添加一行。

要保存新行，请单击提交 。

有关使用行的更多信息，请参见使用表 。

删除行

 Ctrl+Y

删除所选的行。

要选择行，请单击装订线中的数字。要选择多行，请按Ctrl然后单击必要的行。

发射和发射隔离

选择数据库事务的隔离级别以及提交事务的方式。

  • 汽车。 当您将本地更改提交到数据库服务器时，当前事务将自动提交。

  • 手册。 提交给数据库服务器的更改将累积在可以提交或回滚的事务中。

提交

 Ctrl+Enter

将本地更改提交到数据库服务器。请参阅将更改提交到数据库 。

承诺

提交当前交易。另请参见Tx和Tx隔离 。

回滚

回滚当前事务。另请参见Tx和Tx隔离 。

取消查询

 Ctrl+F2

终止当前查询的执行。

与之比较

将当前表与数据库控制台工具窗口中的表进行比较。有关比较表的更多信息，请参阅比较表数据 。

数据提取器

选择数据的输出格式。

另外，您可以配置以下选项：

  • 跳过计算列（SQL） ：不包括未物理存储在表中的虚拟列（例如，标识列）。

  • 跳过生成的列（SQL） ：对于INSERT和UPDATE语句，在复制或保存数据时不包括自动增量字段。

  • 添加表定义（SQL） ：对于INSERT和UPDATE语句，在复制或保存数据时添加表定义。

  • 配置CSV格式 ：打开“ CSV格式”对话框，您可以在其中管理定界符分隔的值格式（例如CSV，TSV）。

  • 转到“脚本目录” ：使用脚本将目录数据转换为不同的输出格式，打开目录。

转储数据

将表数据复制到剪贴板或保存到文件。

转储数据提交

使用此命令将表数据保存在文件中。在打开的对话框中，指定文件的位置和名称。

导出到数据库

将数据导出到另一个表，架构或数据库。选择目标模式（将创建一个新表）或表（数据将添加到所选表中）。在打开的对话框中，指定数据映射信息和目标表的设置。

查看查询

使用此按钮可以查看用于生成当前表视图的查询。要关闭显示查询的窗格，请按Escape 。

显示选项菜单列表包括以下选项：

  • 移调 ：切换移调的表格视图。在转置视图中，行和列互换。

  • 重置视图 ：如果您之前自定义了表格视图（例如，隐藏列或排序的数据），则还原初始表格视图。

  • 通过ORDER BY排序：在数据库级别对数据进行排序。“ 通过ORDER BY排序”选项仅适用于数据编辑器中的表，不适用于“ 服务”工具窗口中的表结果集。

    如果“ 通过ORDER BY排序”选项处于启用状态，则将根据相应的数据库系统对整个表的数据进行排序。在“ 输出”选项卡中，您可以看到所有相应的排序操作 。

    如果要尽量减少与数据库的交互，请不要打开此选项。例如，当表很大或数据库连接速度很慢时。

    如果关闭此选项，则数据将在本地进行排序，并且仅针对当前显示的行进行排序。

  • 行过滤器 ：显示或隐藏过滤器框。

  • 设置 ：打开“ 设置”对话框的“ 数据库”页面 ，以查看或编辑数据库控制台，数据编辑器和“ 数据库工具”窗口的设置。

编辑

 F2

在选定的一个或多个单元格中编辑一个值。或者，您可以双击该单元格并开始输入一个值。“ 编辑”命令不适用于只读值。对于只读值，请使用“ 查看”命令。

  • 要打开自动完成列表，请按Ctrl+Space 。

  • 要输入修改的值，请按Enter 。

  • 要取消编辑，请按Escape 。

有关编辑单元格的更多信息，请参见单元格 。

编辑最大化

 F2

展开选定的单元格以编辑该单元格中的值。

在最大化的单元格中，按Enter开始新的一行。要输入值，请按Ctrl+Enter 。要恢复初始值并退出编辑模式，请按Escape 。

有关编辑单元格的更多信息，请参见单元格 。

默认设置

 Ctrl+Alt+D

设置默认值或您为列指定的值。要查看默认列值，请在“ 数据库”工具窗口（“ 视图” |“工具窗口” |“数据库” ）中右键单击一个表，然后选择“ 修改表” 。双击列条目，然后在“ 默认”字段中查看该值。

设置为空

 Ctrl+Alt+N

用NULL替换一个或多个值。要确保该列接受NULL值，请在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中右键单击一个表，然后选择“ 修改表” 。双击列条目，然后查看是否清除了“ 不为空”复选框。

载入档案

将文件加载到该字段中。

还原

 Ctrl+Z

还原选择范围内的更改。有关还原更改的更多信息，请参阅将更改提交到数据库 。

克隆行

 Ctrl+D

创建所选行的副本。

快速文档

 Ctrl+Q

打开快速文档视图。要关闭视图，请按Escape 。有关快速文档的更多信息，请参阅使用快速文档视图 。

转置

切换转置的表格视图。在转置视图中，行和列互换。

转到|行

 Ctrl+G

跳转到指定的行。在转到行对话框中，指定列号和行号（使用column:row图案）。

转到|相关资料

 F4

跳转到相关记录。命令选项是“ 转到” | “命令”选项的组合。 参考数据并转到|引用数据 。在导航到相关对象中了解有关相关数据的更多信息。

如果没有相关记录，则该命令不可用。

转到|参考资料

 Ctrl+B

切换到当前记录引用的记录。如果引用了多个记录，请在出现的弹出窗口中选择目标记录。在导航相关数据之间阅读更多有关相关数据的信息 。

如果没有引用的记录，则该命令不可用。

转到|参考资料

 Alt+F7

使用此命令或快捷方式可以查看引用当前记录的记录。在导航相关数据之间阅读更多有关相关数据的信息 。在出现的弹出窗口中，目标记录有两种类别：

  • 第一个引用行 ：显示相应表中的所有行，并选择引用当前行的第一个行。

  • 所有引用行 ：仅显示引用当前行的行。

如果没有引用当前记录的记录，则该命令不可用。
