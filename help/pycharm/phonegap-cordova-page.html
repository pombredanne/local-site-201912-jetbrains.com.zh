PhoneGap / Cordova可执行路径

在这个领域里， 指定可执行文件phonegap.cmd ， cordova.cmd或ionic.cml的位置 （请参阅安装PhoneGap / Cordova / Ionic ）。

PhoneGap / Cordova版本

PhoneGap / Cordova工作目录

在这个领域里， 指定要存储要运行的PhoneGap / Cordova / Ionic应用程序文件的文件夹。

自动在打开的项目中排除“平台”目录

选中此复选框可在您打开项目时将平台目录自动标记为排除。结果，PyCharm在索引，解析和代码完成期间会忽略它。

外挂程式

在这方面通过安装所需的软件包，配置要在开发中使用的插件列表。该列表显示了全局和项目级别上当前安装在计算机上的所有PhoneGap / Cordova / Ionic插件。

  • 要安装插件，请单击 。在打开的“ 可用软件包”对话框中，选择所需的软件包。

    要全局安装软件包，请选择选项复选框，然后键入--global在该领域。单击安装软件包 。

  • 要删除插件，请在列表中选择它，然后单击 。

  • 要将插件升级到最新的可用版本，请在列表中选择插件，然后单击 。

有关插件及其使用的信息，请参见Apache Cordova插件和PhoneGap插件 。
