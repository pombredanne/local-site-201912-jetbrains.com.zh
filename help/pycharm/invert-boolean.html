反转布尔

此重构旨在将布尔变量的值及其所有用法从True转换为False，反之亦然。

例

之前

后

def foo {}：abc =真返回abc
def foo {}：abc =假，返回不返回abc

要反转布尔变量，请按照下列步骤操作

  1. 将插入符号放在布尔变量的名称处。

  2. 请执行以下任一操作：

    • 在主菜单上，选择“ 重构”。反转布尔值 。

    • 在插入符号位置的上下文菜单中，选择“ 重构” |“展开”。反转布尔值 。

  3. 在打开的对话框中，单击“ 预览”以在“查找工具”窗口中查看建议的更改，或单击“ 重构”以静默反转检测到的事件。
