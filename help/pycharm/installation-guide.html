安装PyCharm

PyCharm是一个跨平台的IDE，可在Windows，macOS和Linux操作系统上提供一致的体验。

PyCharm提供三种版本： 专业版 ， 社区版和教育版（Edu） 。Community和Edu版本是开源项目，它们是免费的，但功能较少。PyCharm Edu提供课程并帮助您学习Python编程。专业版是商业版本，并提供了一套出色的工具和功能。有关详细信息，请参见版本比较矩阵 。

使用工具箱应用安装

建议使用JetBrains工具箱应用程序安装JetBrains产品的工具。使用它可以安装和维护不同的产品或同一产品的多个版本，包括抢先体验计划 （EAP）版本，在必要时进行更新和回滚，以及轻松删除任何工具。工具箱应用程序维护所有项目的列表，以在正确的IDE和版本中快速打开任何项目。

安装工具箱应用

  1. 从工具箱应用程序网页下载安装程序.exe 。

  2. 运行安装程序，然后按照向导步骤操作。

运行Toolbox App后，在通知区域中单击其图标，然后选择要安装的产品和版本。

从工具箱应用程序登录到您的JetBrains帐户，它将为您安装的任何IDE自动激活可用的许可证。

安装工具箱应用

  1. 从工具箱应用程序网页下载磁盘映像.dmg 。

  2. 挂载图像并将JetBrains Toolbox应用拖到Applications文件夹中。

运行Toolbox App后，在主菜单中单击其图标，然后选择要安装的产品和版本。

从工具箱应用程序登录到您的JetBrains帐户，它将为您安装的任何IDE自动激活可用的许可证。

安装工具箱应用

  1. 从工具箱应用程序网页下载tarball .tar.gz 。

  2. 将压缩包解压缩到支持文件执行的目录。

从提取的目录中执行jetbrains-toolbox二进制文件以运行Toolbox App，然后选择要安装的产品和版本。首次运行Toolbox App后，它将自动将Toolbox App图标添加到主菜单。

独立安装

手动安装PyCharm以管理每个实例和所有配置文件的位置。例如，如果您有一个要求特定安装位置的策略。

  1. 下载安装程序 .exe 。

    要验证安装程序的完整性，请使用“ 下载”页面中链接的SHA校验和。

  2. 运行安装程序，然后按照向导步骤操作。

    注意安装向导中的以下选项

    • 64位启动器 ：向桌面添加启动图标。

    • 以项目形式打开文件夹 ：在文件夹上下文菜单中添加一个选项，该选项允许以PyCharm项目形式打开所选目录。

    • .py ：与Python文件建立关联，以在PyCharm中打开它们。

    • 将启动器目录添加到PATH ：允许从控制台运行该PyCharm实例，而无需指定其路径。

有关更多信息，请参阅首次运行PyCharm 。

使用快照包安装

  1. 对于Ubuntu 16.04及更高版本，您可以使用快照包来安装PyCharm。

    运行以下命令：

    sudo snap install --classic

    例如：

    sudo snap install pycharm-community --classic

  2. 跑pycharm-professional ， pycharm-community ， 要么pycharm-educational在终端。

使用tar归档文件进行安装

  1. 如果没有快照软件包，请在当前的“ 下载”文件夹不支持文件执行的情况下，将pycharm-*。tar.gz文件解压缩到另一个文件夹：

    焦油xfz pycharm-*。tar.gz -C

    根据文件系统层次结构标准（FHS）的推荐安装位置是/ opt 。要将PyCharm安装到此目录中，请输入以下命令：

    须藤tar xfz pycharm-*。tar.gz -C / opt /

    要验证下载的存档的完整性，请使用“ 下载”页面中链接的SHA校验和。

  2. 切换到bin子目录：

    cd / pycharm-* / bin
    例如，
    cd / opt / pycharm-* / bin

  3. 从bin子目录运行pycharm.sh 。

    sh pycharm.sh

Windows上的静默安装

无需任何用户界面即可执行静默安装。网络管理员可以使用它在许多机器上安装PyCharm，并避免打扰其他用户。

要执行静默安装，请使用以下开关运行安装程序：

  • /S ：启用静默安装

  • /CONFIG ：指定静默配置文件的路径

  • /D ：指定安装目录的路径

    此参数必须是命令行中的最后一个，并且即使路径包含空格也不应包含任何引号。

例如：

pycharm-professional.exe / S /CONFIG=d:\temp\silent.config / D = d：\ IDE \ PyCharm
要么
pycharm.exe / S /CONFIG=d:\temp\edu_silent.config / D = d：\ IDE \ PyCharm Edu

要在安装过程中检查问题，请添加/LOG在日志文件路径和名称之间切换/S和/D参数。安装程序将生成指定的日志文件。例如： pycharm-professional.exe /S /CONFIG=d:\temp\edu_silent.config /LOG=d:\JetBrains\PyCharm\install.log /D=d:\IDE\PyCharmpycharm-professional.exe /S /CONFIG=d:\temp\edu_silent.config /LOG=d:\JetBrains\PyCharm\install.log /D=d:\IDE\PyCharm

静默配置文件

您可以在https://download.jetbrains.com/python/silent.config和https://download.jetbrains.com/python/edu_silent.config下载PyCharm的静默配置文件。

静默配置文件定义用于安装PyCharm的选项。使用默认选项，仅对当前用户执行静默安装mode=user 。如果要为所有用户安装PyCharm，请使用文本编辑器打开静默配置文件，更改安装模式选项的值mode=admin并以管理员身份运行安装程序。

默认的静默配置文件对于每个JetBrains产品都是唯一的。您可以根据需要对其进行修改，以启用或禁用各种安装选项。
