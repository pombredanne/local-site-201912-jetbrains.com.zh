IntelliJ配置服务器设置

单击IntelliJ Configuration Server状态图标时，将打开此对话框。 / 在会话期间在状态栏上单击。

项目

描述

显示登录对话框

选择此选项可在下一次PyCharm启动期间显示“ 登录到IntelliJ Configuration Server”对话框。

静默登录

选择此选项可登录IntelliJ Configuration Server，而无需输入登录名和密码。

不登入

如果您不打算登录IntelliJ Configuration Server，请选择此选项。

您可以在会话期间的任何时间重新配置此行为。

登录/注销

单击此按钮登录IntelliJ Configuration Server。
