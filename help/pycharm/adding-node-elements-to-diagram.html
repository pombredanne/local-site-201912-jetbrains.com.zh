向图添加节点元素

您可以使用上下文菜单操作或拖放技术将现有节点元素添加到UML类图的背景。

如果模型中的节点元素与添加的元素之间存在关系，则这些关系将显示在图中。

在此页：

  • 添加节点元素
  • 添加笔记

添加节点元素

  1. 在“ 输入要添加的班级名称”对话框中，开始输入所需的班级名称。PyCharm自动显示具有匹配名称的建议列表。

    请注意，可以通过选择“ 包括非项目类”复选框来包括项目范围之外的类 。

  2. 从建议列表中选择所需的班级，然后按Enter 。

添加笔记

  1. 右键单击要注释的节点元素。

  2. 在上下文菜单上，选择“ 新建|注意 。

  3. 在文本框中，键入所需的文本。注意Enter在文本框中开始新行。要完成备忘，请单击确定 ，或按Ctrl+Enter 。
