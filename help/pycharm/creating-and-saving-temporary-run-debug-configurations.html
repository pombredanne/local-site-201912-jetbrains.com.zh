创建和保存临时运行/调试配置

有时，您可能需要运行或调试脚本，应用程序或测试，而无需创建专用的运行配置。在这种情况下，您可以使用PyCharm提供的临时运行配置 。

每次您从上下文菜单或使用装订线图标运行测试类或方法时，都会创建一个临时配置要么 。当运行或调试会话结束时，新配置在“运行/调试配置”选择器中显示为灰色。

临时运行/调试配置将添加到可用配置列表中，并且与永久运行/调试配置的工作方式相同。您可以使用“ 运行/调试配置”对话框更改其设置，还可以选择将其保存为永久文件。

临时配置无法在XCode中查看和编辑。您可以保存一个临时配置，以使其在其他XCode方案中可用。

创建临时运行/调试配置

  1. 在“ 项目”工具窗口中选择所需的脚本或在编辑器中将其打开。

  2. 请执行以下任一操作：

    • 在编辑器中右键单击代码，然后选择“运行”。 或调试 。

    • 按Ctrl+Shift+F10 。

    PyCharm创建一个临时配置，当运行或调试会话结束时，该临时配置将显示在“ 运行/调试配置”选择器中。

保存临时运行/调试配置

  • 在“ 运行/调试配置”选择器中，选择“ 保存”。 。

  • 在“ 编辑配置”对话框中，从列表中选择临时配置，然后单击“确定”。 。

  • 从编辑器或项目工具窗口的上下文菜单中，选择“ 保存”。 。
