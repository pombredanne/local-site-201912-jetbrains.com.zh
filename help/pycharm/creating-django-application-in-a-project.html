在项目中创建Django应用程序

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

要将新站点添加到您现有的Django项目中，请使用startapp manage.py实用程序的任务。

向现有项目添加新的Django应用程序

  1. 在主菜单上，选择“ 工具”。运行manage.py任务

  2. 在“ 输入manage.py任务名称”对话框中，键入startapp 。请注意，在输入第一个字母后，该建议列表会出现在对话框下，并在您键入时缩小以仅显示完全匹配项。

  3. 在打开的对话框中，输入新Django应用程序的名称。
