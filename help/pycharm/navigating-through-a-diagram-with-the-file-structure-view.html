使用结构视图浏览图

要浏览图表，请遵循以下常规步骤

  1. 按Ctrl+F12 。弹出图元素菜单。

  2. 使用箭头键选择所需的图表元素，然后按Enter 。所选图节点元素变为活动状态。

  3. 按Enter再一次。元素的第一个成员获得焦点。

  4. 可以执行以下操作：

    • 紧迫Enter在图元素及其成员之间切换焦点。

    • 当图表元素具有焦点时，请使用箭头键在元素之间跳转。

    • 当成员具有焦点时，请使用垂直箭头键在元素中导航至所需成员。

    • 选择元素或成员后， 导航至基础源代码 。
