PyCharm 2019.2帮助

与答案占位符一起使用

PyCharm Educational（Edu）版本提供了此功能。您可以通过EduTools插件为PyCharm Community和PyCharm Professional启用它。

总览

答案占位符是一个框架，它将代替原始练习代码的一部分，学生必须在其中编写解决方案。

当占位符已经存在时，上下文菜单命令“ 添加答案占位符”将替换为一组命令，这些命令使您可以编辑占位符，删除占位符或删除所有答案占位符。

如果学生认为自己无法解决问题，则可以寻求帮助。

以下功能适用于教育者！

创建答案占位符

创建答案占位符

  1. 选择要用一些文本替换的练习代码片段。

  2. 右键单击选择，然后从上下文菜单中选择“ 添加答案占位符” 。

  3. 在“ 添加答案占位符”对话框中，指定将替换学生项目中所选片段的文本。

    如有必要，添加提示 。

编辑答案占位符

编辑答案占位符

  1. 将插入符号放置在要更改的答案占位符上。

  2. 从上下文菜单中，选择“ 编辑答案占位符” 。

  3. 如有必要，更改任务窗口文本和提示文本。

  4. 准备好后，单击“ 确定” 。

删除答案占位符

删除答案占位符

  1. 将插入符号放置在要删除的答案占位符上。

  2. 从上下文菜单中，选择“ 删除答案占位符” 。

    如果不再需要所有答案占位符 ，请选择删除所有答案占位符 。

占位符被无提示删除。

此操作仅适用于学生！

放弃

获得答案

  • 在主菜单中，选择“ 帮助” |“选择” 。填写答案占位符 。

当前页面上的所有答案占位符都填充有正确的解决方案。
