字串格式

选择要由PyCharm识别的文档字符串的格式。

根据所选的文档字符串格式，PyCharm将生成存根文档注释并在显示快速文档中呈现文本：

  • 普通 ：按时Enter或打开引号后的空格，将生成一个空的存根；快速文档显示为纯文本。

  • reStructuredText ：按下时Enter或在打开引号后加空格，则根据reStructuredText格式生成存根文档注释；快速文档由Docutils提供。

  • Epytext ：按下时Enter或在打开引号后加空格，则根据epytext格式生成存根文档注释；快速文档由epydoc呈现。

  • NumPy ：按下时Enter或打开引号后的空格，则根据NumPy格式生成存根文档注释；快速文档由Napoleon和Docutils提供。

  • 谷歌 ：在按Enter或在打开引号后加空格，则根据Google格式生成存根文档注释；快速文档由Napoleon和Docutils提供。

所有类型的文档字符串均具有：

  • 正确生成文档字符串

  • 应用意图动作和快速修复后进行更新

  • 编码协助

  • 自动完成节标题

请注意，文档字符串中提供的信息用于代码洞察。

分析文档字符串中的Python代码

如果选中此复选框，PyCharm将突出显示代码示例并执行语法检查和代码检查。
如果未选中此复选框，则不会分析文档字符串中的代码片段。
