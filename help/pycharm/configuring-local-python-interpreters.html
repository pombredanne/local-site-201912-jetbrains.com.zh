配置系统解释器

要配置系统Python解释器，请按照以下步骤操作：

  1. 确保已在计算机上下载并安装了Python 。

    如果您使用的是Windows，则可以从Microsoft Store下载Python，并将其安装为项目解释器。从Microsoft Store下载Python应用程序后，它就可以在Python可执行文件列表中使用。

    请注意，从Microsoft Store安装中添加的解释器有一些限制。由于Microsoft Store应用程序的限制，Python脚本可能无法完全访问共享位置，例如TEMP和注册表。

  2. 通过以下两种方式打开“ 添加Python解释器”对话框：

    • 当您在编辑器中时 ，最方便的方法是在状态栏中使用Python Interpreter 小部件 。单击小部件，然后选择添加解释器...

    • 通过按打开设置/首选项对话框 Ctrl+Alt+S或选择文件|设置 Windows和Linux或PyCharm | macOS的首选项 。

      在“ 设置/首选项”对话框中Ctrl+Alt+S ，选择项目|项目口译员 。点击图标，然后选择添加 。

  3. 在“ 添加Python解释器”对话框的左侧窗格中，选择“ 系统解释器” 。

  4. 在“ 解释器”字段中，键入所需解释器可执行文件的标准路径，或单击“确定”。 在打开的“ 选择Python解释器”对话框中，选择所需的Python可执行文件，然后单击“ 确定” 。

    您将需要管理员权限才能安装，删除和升级系统解释器的软件包。尝试通过意图操作安装解释程序包时，您可能会收到以下错误消息：

    根据提示，考虑为项目使用虚拟环境。

  5. 单击确定以完成任务。

对于任何已配置的Python解释器，您可以：

  • 安装和卸载解释器路径
  • 安装和卸载解释程序包
