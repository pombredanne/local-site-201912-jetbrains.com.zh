把手和胡子

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

借助PyCharm，您可以在hbs或mustache文件中使用Handlebars表达式和Mustache模板 。PyCharm根据“设置/首选项”对话框的“ 模板”页面上指定的默认或自定义设置来处理此类文件的内容。

在你开始前

  • 如“ 管理插件”中所述，在“ 插件”页面上安装并启用Handlebars / Mustache存储库插件。

在PyCharm中配置车把和胡须

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |模板 。

  2. 在打开的“ 模板”页面上，切换到“ 把手/胡子”区域。

  3. 自动插入第二个右花括号}在输入第一个结束符时选择“ Handlebars”表达式 ，请选中“ 自动插入结束符”复选框。

    PyCharm还可以识别三重藏匿处 {{{防止转义表达式内的值。在这种情况下，只要您输入第一个右花括号，PyCharm就会自动插入两个右花括号。

  4. 在代码生成，重构或按下时自动重新格式化Handlebars表达式和Mustache模板 Ctrl+Alt+L ，请选中启用格式复选框。否则，PyCharm始终保持其原始格式。

  5. 选择用于使用样式进行注释的语言。当您按以下内容输入行或注释时Ctrl+/要么Ctrl+Shift+/ ，PyCharm会插入用于所选语言的注释定界符，例如：

    • {{!----}}用于车把

    • /**/用于JavaScript

    • 用于HTML

  6. 要配置“车把/胡须”语法突出显示， 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |配色方案|车把/小胡子 。选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
