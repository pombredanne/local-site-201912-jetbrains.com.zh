在项目中查找和替换文本

您可以在项目中搜索文本字符串，使用不同的范围来缩小搜索过程，从搜索中排除某些项目，查找用法和出现的地方。

在项目中查找搜索字符串

  1. 从主菜单中，选择编辑|查找|在路径中查找 Ctrl+Shift+F 。

  2. 在搜索字段中，输入搜索字符串。或者，在编辑器中，突出显示要查找的字符串，然后按Ctrl+F 。PyCharm将突出显示的字符串放入搜索字段。

    要查看以前的搜索列表，请按Alt+Down 。

    如果需要，请指定其他选项。

    PyCharm列出了搜索字符串和包含它们的文件。

    要进行多行搜索，请点击图标输入新行，然后按Ctrl+Alt+Down / Ctrl+Alt+Up浏览事件。

  3. 在对话框的预览区域中检查结果，您可以在其中替换搜索字符串或选择其他字符串，然后按Ctrl+Shift+F再次开始新的搜索。

  4. 如果需要在单独的工具窗口中查看事件列表，请单击“ 在查找窗口中打开” 。您可以使用此窗口及其选项对结果进行分组，预览并进一步使用它们。

缩小搜索范围

您可以在“ 在路径中查找”对话框中使用其他选项来调整搜索过程。

  1. 在“ 在路径中查找”对话框中，选择“ 单词”或“ 区分大小写”之类的选项以在项目中查找确切的单词，或匹配字母的大小写。

    请参阅带有正则表达式的文档进行搜索，以了解如何在搜索中使用正则表达式 。

  2. 点击图标以过滤您的搜索。例如，您可以过滤搜索以省略评论或仅搜索评论。

  3. 选择显示的选项之一，例如“ 模块”或“ 目录”以限制搜索。此外，您可以选择“ 范围”选项，该选项为您提供了搜索的预定义范围列表。

    例如，您可以仅将搜索限制为项目中的打开文件。

    如果不使用选项卡，则“ 最近查看的文件”或“ 最近更改的文件 ”范围可能会非常有用。您也可以创建自己的自定义范围，点击图标打开“ 作用域”对话框。

搜索特定的文件类型

使用“ 文件掩码”选项将搜索范围缩小到特定文件类型。您可以从列表中选择现有文件类型，添加新文件类型或添加其他文件掩码语法来搜索具有特定模式的文件类型。

  1. 在“ 在路径中查找”对话框中，选中“ 文件掩码”复选框，然后从文件类型列表中选择所需的文件。

    PyCharm将其搜索限制为指定的类型。

  2. 如果在列表中找不到所需的文件类型，请在“ 文件掩码”字段中输入文件类型。

    除了其他通配符 *支持。您还可以通过使用逗号分隔指定几种文件类型。

搜索项目中的用法

您可以在整个项目或设置的范围中搜索符号的用法。

  1. 选择您要查找其用法的符号。请注意，您还可以将搜索范围扩展到文件使用情况。

  2. 从主菜单中选择编辑|查找|查找用法 Alt+F7 。

  3. 在“ 查找工具”窗口中检查结果。

    请点击打开“ 查找用法”对话框。

    要查看文件中所选符号的用法，请右键单击文件中的上下文菜单，然后选择“ 查找用法” Ctrl+F7 。

    请点击打开“ 查找用法”对话框。

    您也可以从以前的“ 查找用法”操作中提取结果。从主菜单中，选择编辑|查找|最近查找用法和用法查询。

    在“ 查找工具”窗口中时，可以使用“ 预览”区域检查找到用法的位置，查看方法的调用层次结构，字段的数据流等等。

    如果PyCharm没有返回任何结果，它将显示一条消息，提示您选择更多选项。您可以点击链接或按Ctrl+Shift+Alt+F7打开“ 查找用法”对话框，您可以在其中设置搜索的新范围。

    例如，您可以将搜索范围设置为仅生产文件或仅打开文件。

    要设置自定义范围 ，请点击 。

  4. 完成设置新范围后，点击查找 。

如果您希望PyCharm在单独的窗口中显示所选符号的用法，请按Ctrl+Alt+F7 。您可以使用此窗口进行快速导航。

再次按相同的快捷方式以查看默认范围内的用法。

要查看以前的“ 查找用法”操作的结果，请选择“ 编辑” |“其他”。查找|从主菜单中找到“最近使用情况 ”，然后选择使用情况查询。

禁用用法的自动突出显示

将插入符号放置在符号上时，IDE会在当前文件中突出显示该符号的所有用法。使用F3和Shift+F3在突出显示的符号之间导航的快捷方式。

在节电模式下，用法的突出显示被禁用。

如有必要，可以禁用自动突出显示。

  • 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |常规 ，然后清除在插入符号处突出显示元素的用法复选框。

禁用自动突出显示后，将插入标记放在必要的符号处，然后按Ctrl+Shift+F7 。这将突出显示当前文件中该符号的所有用法。

替换项目中的搜索字符串

  1. 按Ctrl+Shift+R打开“ 替换为路径”对话框。

  2. 在顶部字段中，输入搜索字符串。在底部字段中，输入您的替换字符串。

    例如，如果要将大型项目的变量名替换为新名称，请使用replace in路径而不是重命名重构，因为您的变量也可以出现在配置文件中。

  3. 单击可用的替换命令之一。
