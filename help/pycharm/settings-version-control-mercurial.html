HG可执行文件的路径

指定Mercurial可执行文件的位置。手动输入路径，或单击“ 浏览”按钮然后在打开的对话框中选择路径。

如果遵循标准安装过程，则默认位置是/ opt / local / bin或Linux和macOS的/ usr / local / bin ，以及Windows的/ Program Files / TortoiseHG 。

建议您将Mercurial可执行文件的路径添加到PATH 可变的 。在这种情况下，您只能指定可执行文件名称，而不需要可执行文件位置的完整路径。

测试

单击此按钮以验证Mercurial可执行文件的路径。

在所有根上执行分支操作

仅当您有一个多根项目时，即在一个项目中有多个Mercurial存储库时，此选项才可用。

如果要分支操作（例如， checkout ， merge ，等等。）同步应用于项目中的所有存储库。

检查传入和传出变更集

如果您希望PyCharm在后台模式下检测传入和传出的更改，请选择此选项。PyCharm将每隔5分钟自动请求服务器获取传入和传出的变更集。

忽略注释中的空白差异

如果要在注释时忽略空格，请选择此选项，从而获得更有意义的注释，并消除无意义的注释。
