PyCharm 2019.2帮助

辅助功能

通过PyCharm，您可以启用各种辅助功能来满足您的需求。您可以使用屏幕阅读器或调整字体大小，颜色和某些UI元素的行为，以简化使用PyCharm的过程。

设置屏幕阅读器

目前，我们完全支持Windows上的PyCharm屏幕阅读器。

对于macOS ，请打开VoiceOver并安装并设置PyCharm 。但是，要获得全屏阅读器的支持，建议您使用Windows。

为Windows启用屏幕阅读器

  1. 下载并启用您喜欢的屏幕阅读器。检查以下推荐的屏幕阅读器：

    • NVDA ：使用NVDA 2015或更高版本。如果使用的是32位版本的NVDA，则必须在计算机上安装32位JRE，因为此版本的NVDA需要C：\ Windows \ SysWOW64 \ WindowsAccessBridge-32。DLL与PyCharm一起使用。如果NVDA找不到此文件，则NVDA事件日志窗口将显示一条消息。

    • JAWS ：下载所需的版本，然后重新启动计算机以启用JAWS屏幕阅读器。

  2. 确保已为屏幕阅读器安装了Java Access Bridge和正确的Java版本，如下所示：

    • 要启用Java Access Bridge ，请打开命令提示符并键入[JRE_HOME] \ bin \ jabswitch -enable ，其中[JRE_HOME]是计算机上JRE的目录。对于Java版本1.8，Java Access Bridge是JDK的一部分，您无需单独下载或启用它。

    • 如果您的屏幕阅读器是32位的，请安装32位的JRE 1.7或更高版本。如果您的屏幕阅读器是64位的，请安装64位的JRE 1.7或更高版本。

    您的计算机可能具有Java Access Bridge某些重要组件的多个版本，并且它们可能在各个版本之间不兼容。您需要验证Java Access Bridge配置是否正确。

    如果您的屏幕阅读器是32位的，请确保C：\ Windows \ SysWOW64 \ WindowsAccessBridge-32。DLL存在并且具有7.xxx或更高版本。该文件的描述应为“ Java TM Platform SE 7”。

安装和设置PyCharm

  1. 下载并安装 PyCharm。

  2. 要在首次启动PyCharm之前启用屏幕阅读器支持，请执行以下操作：

    • 打开包含个人设置的配置目录，例如键盘映射，配色方案等。

      句法
      ％HOMEPATH％\。 \ config
      例
      C：\ Users \ JohnS \。PyCharm2019.2 \配置
      句法
      〜/图书馆/首选项/
      例
      〜/图书馆/偏好/PyCharm2019.2
      句法
      〜/。 /配置
      例
      〜/。PyCharm2019.2 /配置

    • 创建一个名为idea.properties的文件。

    • 添加ide.support.screenreaders.enabled=true属性创建的文件。

  3. 启动PyCharm。“ 设置/首选项” | “ 支持”中的“屏幕阅读器”选项外观与行为|外观将启用。

自定义IDE

您可以根据可访问性需求自定义IDE。

调整颜色以消除红绿色视力障碍

如果您有红绿色视觉缺陷，则可以调整颜色。在这种情况下，代码片段（例如通常以红色突出显示的错误或通常为绿色的字符串）将改变颜色（红色将变为橙色，绿色将变为蓝色）。测试运行器中进度条的颜色也将进行调整，以便可以轻松识别。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到外观和行为|外观 。

  2. 从右侧的选项中，选择“ 调整红绿色视力障碍的颜色（盲目，反盲）”选项，然后单击“ 确定”保存更改。

配置代码元素，编辑器，超链接等的颜色

您可以调整代码元素，错误，编辑器元素和工具窗口的颜色。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |配色方案|一般

  2. 从右侧的选项列表中，选择要调整颜色的对象。例如，您可以选择代码并为注入的语言片段或匹配的花括号调整颜色，等等。单击确定保存更改。

您还可以调整调试器，控制台和IDE的其他部分的颜色：在“设置/首选项” |“选项”中，选择选项列表中的相应节点。 编辑器配色方案 。

覆盖默认的UI字体

您可以覆盖UI元素的默认字体。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到外观和行为|外观 。

  2. 从右侧的选项中，选择使用自定义字体 。从字体列表中，选择所需的字体，然后在“ 大小”字段中指定字体的大小。

    单击确定保存更改。

调整工具窗口大小

您可以使用快捷方式垂直或水平调整实际工具窗口的大小。

  1. 要垂直向上或向下调整大小，请按Ctrl+Shift+Up要么Ctrl+Shift+Down 。

  2. 要左右水平调整大小，请按Ctrl+Shift+Left要么Ctrl+Shift+Right 。

在编辑器中调整文字大小

您可以在编辑器中更改字体和文本大小。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般

  2. 在编辑器中工作时，从右侧的选项中选择“ 使用Ctrl +鼠标滚轮更改字体大小（缩放）”选项以快速更改文本大小（转动鼠标滚轮）。

  3. 如果需要指定确切的字体大小，请选择“ 编辑器” |“字体”。字体 。

  4. 从右侧的选项中，指定字体，其大小，行距和其他可用选项。单击确定保存更改。

自定义快捷方式

您可以为常用操作配置自定义快捷方式。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到Keymap 。

  2. 从右侧的选项列表（例如菜单，操作和工具）中，选择所需的操作。

  3. 右键单击所选项目，然后从上下文菜单中选择要执行的操作，例如“ 添加键盘快捷方式” ，“ 添加鼠标快捷方式 ”或“ 添加缩写” 。

  4. 在打开的对话框中，指定一个快捷方式。如果需要，请选择“ 第二笔画”选项，然后为快捷方式指定其他键。单击确定保存更改。

    用鼠标单击“ 确定 ”。如果按Enter PyCharm会认为这是捷径。

您可以忽略冲突，并为多个操作分配快捷方式。但是，强烈建议您避免使用同一快捷方式绑定两个操作，因为未定义这些操作的优先级。

自定义智能键行为

您可以配置智能键的行为。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|智能钥匙 。

  2. 从右侧的选项中，选择或清除智能键选项，例如，您可以清除自动插入右括号或引号的“ 插入成对括号”或“ 插入成对引号”选项，因为当您使用屏幕阅读器。单击确定保存更改。

禁用自动完成代码

当您使用屏幕阅读器在编辑器中工作时，可以禁用自动代码完成功能以避免自动插入代码元素。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|代码完成 。

  2. 清除“ 智能类型完成”选项。

自定义代码折叠

您可以控制代码折叠行为，并指定应该或不应该折叠的内容。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|代码折叠 。

  2. 从右侧的选项中，选择默认情况下应折叠的内容。

自定义代码样式

您可以配置空格，制表符和缩进。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |代码样式| [语言] 。

  2. 从右侧的选项中，单击“ 制表符和缩进”以配置制表符，或单击“ 空格”以配置在何处以及如何使用空格。

  3. 单击确定保存更改。

管理鼠标指针聚焦模式

您可以配置打开对话框时鼠标指针的行为。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到外观和行为|外观 。

  2. 从右侧的选项中，选择将鼠标光标自动定位在默认按钮上 。

    对话框打开时，将鼠标指针放在默认按钮上。

  3. 单击确定保存更改。

在编辑器中读取装订线图标和行号

您可以配置屏幕阅读器以读取行号，VCS批注，调试器以及位于编辑器左装订线中的其他图标。

  1. 在编辑器中打开文件。

  2. 按Shift+Alt+6和F同时关注排水沟。PyCharm从您的插入符号当前所在的行开始读取。

  3. 使用向上和向下箭头键在各行之间移动。如果需要移动到行中的下一个或上一个装订线元素，请分别使用向右和向左箭头键。

    当焦点位于装订线时，屏幕阅读器可以读取装订线图标工具提示（如果有）。

    要访问工具提示，请按双快捷键Shift+Alt+6, T 。要浏览工具提示的内容（逐个符号），请使用向右和向左箭头键。

  4. 按Escape将焦点切换回编辑器。

设置高对比度的颜色主题

您可以设置高对比度界面主题以在PyCharm中工作。界面主题定义窗口，对话框和控件的外观。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到外观和行为|外观 。

  2. 在“ 外观”页面上的“ UI选项”区域的“ 主题”列表中，选择“ 高对比度” ，然后单击“ 确定”以应用更改。

设置高对比度配色方案

您可以为编辑器设置高对比度配色方案。PyCharm使用颜色方案来帮助您在编辑器中定义首选的颜色和字体。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |配色方案 。

  2. 在“ 配色方案”页面上，从“ 方案”列表中选择“ 高对比度” 。

  3. 单击确定以应用更改。
