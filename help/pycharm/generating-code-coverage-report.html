生成代码覆盖率报告

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

在运行至少一项覆盖率测试之后，代码覆盖率报告将变为可用。

生成报告

  1. 在主菜单中，选择工具|生成覆盖率报告 ，或单击在“ 覆盖率”工具窗口中 。

    请注意，仅当您在覆盖范围内运行必要的测试后，此选项才可用。

  2. 在“ 生成覆盖率报告”对话框中，指定将存储生成的报告的目录，并选择选中“ 在浏览器中打开生成的HTML”复选框。

  3. 点击保存 。

请注意，如果您要保存在同一窗口中打开的多个项目之一的代码覆盖率报告，则检查建议的目标位置很重要，因为PyCharm会建议以前使用的位置。
