SQL文件

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

仅当安装并启用了数据库工具和SQL插件时，以下内容才有效。

运行一个SQL文件

  1. 右键单击数据源。

  2. 选择“运行SQL脚本” 。

  3. 在“ 选择路径”窗口中，导航到要应用的SQL文件。

为多个数据源运行一个SQL文件

  1. 在“ 项目”工具窗口中选择必要的SQL文件。

  2. 右键单击该SQL文件，选择“运行”。 （或按Ctrl+Shift+F10 ）。

  3. 在“ 执行目标”窗口中，选择必要的数据源。

运行单个语句

  1. 在PyCharm中打开一个SQL文件。

  2. 单击您要执行的语句。如果要执行多个语句，请选择（突出显示）必要的语句。

  3. 按Ctrl+Enter或从上下文菜单中选择执行 。

  4. 选择要在其中执行该语句的数据库控制台 。

在控制台菜单中，您可以将其他控制台与文件关联，或删除文件与控制台之间的关联（ 分离控制台 ）。否则，可以通过从文件的上下文菜单中选择“ 附加控制台”来将控制台与文件关联。

您一次只能为一个数据源运行单个语句。

语句可以包含参数。在运行这些语句之前，请指定这些参数的值。

创建一个SQL文件

  1. 在“ 项目”工具窗口（“ 视图” |“工具窗口” |“项目” ）中，右键单击要在其中创建SQL文件的目录，然后选择“ 新建” |“创建”。 SQL文件 。

  2. 在“ 新建文件”对话框中，指定文件名。

  3. 单击确定 。

另外，您也可以导航到File | File |新增| SQL文件

指定SQL方言

为了获得更好的编码帮助，请将SQL方言分配给SQL文件。SQL方言是特定于DBMS的SQL版本。

通过PyCharm，您可以为单个文件，目录，项目或计算机中的所有SQL和DDL文件指定SQL方言。

  1. 右键单击SQL文件中的编辑区域，然后选择Change Dialect（ ） 。

  2. 从方言列表中选择一种方言，并将其应用于当前的SQL文件。如果要将方言应用于一组文件，请选择“ SQL方言” 。

  3. 在“ SQL方言”对话框中，单击“ 添加”按钮并选择要更改其方言的目录或文件。

  4. 点击打开 。

  5. 单击“ SQL方言”单元格，然后选择所需的方言。

  6. 应用更改，然后单击“ 确定” 。
