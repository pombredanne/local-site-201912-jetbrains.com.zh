最近的文件

在大多数情况下，您会在少量正在处理的文件之间切换。其他文件则较少。PyCharm具有使此工作流程非常方便的功能。

假设您正在编辑一个函数，同时您正在为此函数编写测试-这意味着您要来回切换。

这是导航到最新文件的地方：按下Ctrl+E 。请注意，最新文件首先列出，默认情况下处于选中状态-您只需要按Enter 。

最近的文件弹出窗口中提供了快速搜索。只需开始输入您要查找的内容，您就会看到与搜索字符串匹配的结果：

请注意，您还可以跳到工具窗口（例如，打开新的终端会话）。

如果您对最近编辑 （而不只是访问）的文件感兴趣怎么办？在这种情况下，请使用键盘快捷键N/A 。请注意，还可以进行快速搜索和导航到工具窗口。
