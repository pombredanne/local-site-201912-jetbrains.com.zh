从数据编辑器调用“导入文件”对话框

将分隔符分隔的值从数据编辑器导出到数据库时， 将打开此对话框。

  • 表名，结构和数据映射
  • DDL预览
  • 编码，将错误写入文件并将不可转换的值插入为null

表名，结构和数据映射

项目

描述

表

表的名称。

评论

表注释。

列/键等

列的数据映射，以及列的定义，约束和索引。
  • 要开始编辑列的信息，请双击列列表中的相应行。

    注意映射到字段。该字段使您可以指定要导入的文件中的哪个数据列用作数据库中相应列的数据源。如果清除此字段，则不会将任何数据添加到数据库的目标列。

  • 要删除列，请选择相应的行，然后单击 。

DDL预览

将要运行的一个或多个语句。您可以直接在此窗格中编辑语句。

编码，将错误写入文件并将不可转换的值插入为null

项目

描述

编码方式

源文件中数据的字符编码。

将错误写入文件

如果选中此复选框，则导入期间发生的错误将写入指定的文本文件。对于每个错误，都将指示错误发生的位置。

将不可转换的值插入为null

如果选中该复选框，则将为无法转换的数据将NULL插入表中。
