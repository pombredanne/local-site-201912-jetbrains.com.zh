仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

PyCharm与大多数流行的JavaScript单元测试框架集成，并为编写JavaScript单元测试提供了丰富的编码帮助。要获得支持框架的编码帮助，请下载所需的框架并将其配置为PyCharm JavaScript库 。

创建JavaScript单元测试

  1. 在src文件夹旁边创建一个单独的测试文件夹（ File | New | Directory ）。该文件夹的名称取决于您使用的测试框架。

  2. 在“ 项目结构”页面上，将新文件夹标记为测试文件夹 ，请参阅在内容根 目录中 配置文件夹 。

  3. 对于每个生产文件，通过在测试文件夹的上下文菜单上选择（ File | New | JavaScript File ，创建一个单独的测试文件。测试文件名的格式取决于您使用的测试框架。
