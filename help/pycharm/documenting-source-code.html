基本

PyCharm提供了用于创建文档注释的便捷功能。

您的源代码中的文档注释可用于代码参考信息，并在按时开放供审阅Ctrl+Q 。

当您创建其他标签时 ，PyCharm会提供代码完成提示可能的标签名称。

Python文档

可以根据在“设置/首选项”对话框的“ Python集成工具”页面中选择的语法创建文档注释，例如reStructuredText或epytext 。
如果此功能适用于某个函数，则PyCharm会根据所选文档字符串格式生成标签，例如：

  • 对于reStructuredText： :param函数签名中声明的每个参数的标签，以及:return标签。

  • 对于epytext： @param函数签名中声明的每个参数的标签，以及@return标签。

reStructuredText和epytext标记中的标签将相应突出显示。

如果已配置 ，则可以使用以下命令生成文档注释存根： type和rtype标签。

在Python文件中，PyCharm可以识别以Python docstrings表示的文档注释。

在开始之前 ，请确保所需的文档字符串格式，例如， epytext要么reStructuredText在“设置/首选项”对话框的“ Python集成工具”页面中被选中。

另请注意，PyCharm从conf.py捕获自定义角色。在配置包含* .rst文件的目录时，请指向带有conf.py的目录（“ Python 集成工具|带有* .rst文件的目录路径” ）。
