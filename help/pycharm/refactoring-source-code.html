重构代码

在这一部分中，您将找到：

  • 一般重构程序
  • 可用重构的过程和示例

要执行重构，请遵循以下常规步骤

  1. 选择（或将鼠标悬停在）符号或代码片段上以进行重构。可用重构的集合取决于您的选择。您可以在以下PyCharm组件中选择符号：

    • 专案检视

    • 结构工具窗口

    • 编辑

    • UML类图

  2. 请执行以下任一操作：

    • 在“ 重构 ”主菜单上或从所选内容的上下文菜单中，选择所需的重构或按相应的键盘快捷键（如果有）。

    • 从主菜单中，选择“ 重构|重构此 ，或按Ctrl+Shift+Alt+T ，然后从弹出窗口中选择所需的重构。

  3. 在打开的对话框中，指定重构选项。

  4. 要立即应用更改，请根据重构类型，单击“ 重构”或“ 确定” 。

  5. 对于某些重构，可以选择在实际执行重构之前预览更改。在这种情况下，相应对话框中的预览按钮可用。

    要预览可能的更改并进行必要的调整，请单击预览 。PyCharm在“ 查找”工具窗口的专用选项卡上显示将要进行的更改。

    此步骤可能采取的措施之一是从重构中排除某些条目。为此，请在列表中选择所需的条目，然后按Delete 。

    如果重构后预计会发生冲突，PyCharm将显示一个对话框，其中简要介绍所遇到的问题。在这种情况下，请执行以下任一操作：

    • 通过单击继续按钮忽略冲突。结果，将执行重构，但是，这可能导致错误的结果。

    • 通过单击“ 在视图中显示”按钮预览冲突。PyCharm在“ 查找”工具窗口的“ 冲突”选项卡上显示所有冲突的条目，使您可以导航到有问题的代码行并进行必要的修复。

    • 取消重构并返回到编辑器。

  6. 对建议的结果满意后，单击“ 执行重构”以应用更改。
