仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

添加一行

  1. 点击添加新行图标（ ）。或者，右键单击该表，然后从上下文菜单中选择“ 添加新行 ”。

  2. 按Alt+Insert 。

请注意，上下文菜单“ 克隆行”命令Ctrl+D可以用作替代。

删除一行

  1. 选择您要删除的一个或多个行。

    要选择行，请单击装订线中的数字。要选择多行，请按Ctrl然后单击必要的行。

  2. 点击删除行图标（ ）。或者，按Ctrl+Y要么Delete 。

删除表中的所有行

右键单击表，然后导航到“ 数据库工具” |“数据库”。截断 。

克隆一行

您可以克隆选定的行。该行的副本将添加到表的末尾。

要克隆行，请右键单击该行，然后选择克隆 。或者，选择该行并按Ctrl+D 。

在行的子集之间导航

根据将页面大小限制为参数的值，结果集可能分为几页。例如，您将“ 限制页面大小为”参数设置为100，但是查询返回了200行。您将有两行行，每页上有100行。要在页面之间导航，请使用以下控件：

  • ：导航到结果集的第一页。

  • ：导航到结果集的上一页。（ Ctrl+Alt+Up ）

  • ：导航到结果集的下一页。（ Ctrl+Alt+Down ）

  • ：导航到结果集的最后一页。

使所有行同时可见

  1. 打开设置Ctrl+Alt+S并导航到插件|数据库|数据视图 。

  2. 清除将页面大小限制为复选框，然后单击确定 。

  3. 点击刷新按钮或按Ctrl+F5刷新表格视图。

转到指定的行

要导航到具有指定编号的行，请右键单击表格，然后选择“ 转到|行 Ctrl+G 。在转到行对话框中，指定行号，然后单击确定 。

导航到相关对象

您可以从表中检索所选对象的行，并将焦点移到该对象。例如，在项目树中选择一个字段，然后按Ctrl+Alt+Home 。焦点移到新选项卡中的相应列。

要导航到结果集中的对象，请选择一个对象，然后转到“ 导航|导航”。相关符号 Ctrl+Alt+Home 。

在相关数据之间导航

您可以在不同类型的相关数据之间导航：

  1. 引用数据 ：当前对象引用的数据。

  2. 引用数据 ：引用当前对象的数据。

  3. 右键单击表中的列，然后导航到“ 转至| | | |”。参考资料 Ctrl+B或转到|参考资料 Ctrl+Alt+Home 。

  4. 如果出现“ 选择目标”弹出窗口，请选择表和要显示的行：

    • 第一个引用行 ：表中的所有行。选择引用当前行的第一行。

    • 仅引用行 ：仅引用当前行的行。

  5. 按Enter

设置结果集中的行数

默认情况下，发出查询时，返回的行数限制为500。引入此限制是为了避免过载（例如，当SELECT语句返回一百万行时）。

要更改此限制，请打开设置设置Ctrl+Alt+S并导航到工具|数据库|数据视图 。在限制页面大小为字段中，指定一个新数字。要禁用该限制，请清除“ 限制页面大小为”复选框。

另外，您可以在编辑器中单击“ 更改页面大小”列表，然后选择一个预定义值。
