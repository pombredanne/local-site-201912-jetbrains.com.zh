如果要按字母顺序对建议列表中的项目进行排序，而不是按相关性对其进行排序，请选择此选项。

在以下位置显示文档弹出窗口

选中此复选框，PyCharm会自动为建议列表中的每个项目显示一个弹出窗口，并带有当前在搜索列表中突出显示的类，方法或字段的文档。

在右侧的文本字段中，指定弹出窗口应显示的延迟时间（以毫秒为单位）。

仅基于类型的完成

默认情况下，PyCharm会建议符号的完成方式，无论其类型如何。使用这种方法，在复杂情况下，列表显示了多个完成变体。

为了使完成更加精确，请选中该复选框。完成列表将在很大程度上取决于PyCharm推断。结果，在推理不佳的情况下，列表可能保持空白。

建议变量名称

如果需要，请选中此复选框 PyCharm在声明新变量时建议其名称。这些建议基于项目，使用的库以及标准API中定义的类，类型和接口的名称。

默认情况下，清除该复选框。

使用参数名称建议类型注释

选择此复选框可为每个完成建议提供有关其类型的信息。

该复选框仅在选中“ 建议变量名称”复选框时可用。

完成扩展方法主体以进行覆盖

默认， 当您想从父类或接口重写方法并从完成建议列表中选择此方法时，PyCharm会自动添加参数，生成一个super()调用，并在可能的情况下添加类型信息。

清除“ 为替代覆盖完成时扩展方法主体”复选框，以禁止在完成过程中自动生成替代覆盖的方法主体。
