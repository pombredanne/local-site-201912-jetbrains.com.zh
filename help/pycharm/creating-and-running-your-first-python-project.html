PyCharm 2019.2帮助

步骤1。创建并运行您的第一个Python项目

在你开始前

确保满足以下先决条件：

  • 您正在使用PyCharm CE或Professional。

  • 您已经安装了Python本身。如果您使用的是macOS或Linux，则您的计算机已经安装了Python。您可以从python.org获取Python。

选择口译员

选择一个项目使用哪个口译员是一个重要的决定。Python是一种脚本语言，这意味着您的代码将由Python解释器转换为机器代码。

您可以在计算机上安装多个版本的Python，并且需要选择该项目所需的版本。有关详细信息，请参见“ 配置Python解释器”部分。请注意，您以后总是可以改变主意，并为项目指定其他解释器。

创建虚拟环境

当您使用外部库（来自PyPI或其他地方）时，还需要管理这些库的版本。用于此目的的Pythonic解决方案是virtualenvs（有时缩写为venv）。Virtualenvs帮助您将不同项目的依赖关系分开。尽管在此项目中我们没有使用任何依赖关系，但是如果您将来希望添加依赖关系，则为您创建的每个新项目创建一个virtualenv被认为是最佳实践。

在PyCharm中创建一个简单的Python脚本

要开始使用PyCharm，让我们编写一个Python脚本。

让我们开始我们的项目：如果在“ 欢迎”屏幕上 ，单击“ 创建新项目” 。如果您已经打开了一个项目，请选择“ 文件” |“项目” 。新项目 。

PyCharm建议使用几个项目模板来创建各种类型的应用程序（Django，Google AppEngine等）。当PyCharm从项目模板创建新项目时，它将生成相应的目录结构和特定文件，以及任何所需的运行配置或设置。

在本教程中，我们将创建一个简单的Python脚本，因此我们将选择Pure Python 。该模板将为我们创建一个空项目。

选择项目位置。为此，请单击按钮旁边的位置字段，并指定为您的项目目录。

Python最佳实践是为每个项目创建一个virtualenv。为此，请展开“ 项目解释器：新Virtualenv环境”节点，然后选择用于创建新虚拟环境的工具。让我们选择Virtualenv工具，并指定用于新虚拟环境的位置和基本解释器。如有必要，请选择下面的两个复选框。

然后单击“ 新建项目”对话框底部的“ 创建”按钮。

如果您已经打开了一个项目，请单击“ 创建 PyCharm”后询问您是在当前窗口中还是在新窗口中打开一个新项目。选择“ 在当前窗口中打开” -这将关闭当前项目，但是以后您可以重新打开它。有关详细信息，请参见打开多个项目页面。

创建一个Python文件

在“ 项目工具”窗口中选择项目根目录，然后选择“ 文件” |“文件”。从主菜单新建...或按Alt+Insert 。

从弹出菜单中选择选项Python文件 ，然后键入新文件名。

PyCharm创建一个新的Python文件并打开以进行编辑。

运行/调试配置

刚运行脚本时，PyCharm为我们创建了一个临时 运行/调试配置 。首先保存此配置：转到编辑器右上角的运行配置下拉菜单，然后选择保存配置 。

然后，选择“ 编辑配置”以查看此处发生的情况。

不要为“ 模板”节点下列出的默认“运行/调试配置”设置工作目录。这可能会导致新创建的“运行/调试配置”中的目标无法解决。

如果您想更改PyCharm执行程序的方式，则可以在此处配置各种设置，例如：命令行参数，工作目录等。有关更多详细信息，请参见运行/调试配置 。

如果您想使用此“运行”配置启动脚本，请使用下拉菜单旁边的按钮。
