仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

查看代码覆盖率可帮助您检测不受仿真影响的部分源代码。

查看代码覆盖率结果

  1. 请执行以下任一操作：

    • 使用coverage运行所需的类 ， 选择要显示的套件 ，然后在编辑器中打开类。

    • 在主菜单中，选择工具|显示代码覆盖率数据 。

    • 按Ctrl+Alt+F6 。

  2. 查看覆盖范围结果：

    • 在项目工具窗口中：

    • 在专用的Coverage工具窗口中 ：

  3. 在编辑器中打开要浏览的文件。

  4. 使用左侧装订线中的颜色指示器检测未发现的代码行。

  5. 要找出一条线被击中了多少次，请在装订线区域中单击该线。

    打开的弹出窗口显示插入符处该行的统计信息。对于有条件的行，弹出窗口还提供统计信息：

    使用以下工具栏按钮：

    • ：跳到下一个/上一组覆盖或未覆盖的线。

    • ：打开“ 配色方案”设置，您必须在其中选择节点Line Coverage 。

    • ：隐藏覆盖范围信息。

  6. 创建缺少的测试，如创建测试 创建JavaScript单元测试一节中所述。

    对于JavaScript，可以按照“ 创建JavaScript单元测试”部分中的描述生成测试 。
