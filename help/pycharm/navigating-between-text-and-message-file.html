使用装订线图标从模板导航到本地化文件。要从本地化文件跳转到相应的模板，请使用Ctrl+Click 。

导航到语言环境

  1. 请点击所需标签旁边的装订线中的图标。

  2. 如果从多个语言环境引用了一个标记，请从弹出菜单中选择一个：

    选定的django.po文件在编辑器中打开，插入标记位于msgid对应的标签。

根据Django文档，每个django.po文件都包含注释，每个注释的路径均位于每个注释上方msgid 。

导航到模板

  1. 在所需的django.po文件中，将插入标记放置在相关语言环境上方的注释处：

  2. 在主菜单上，选择“ 导航”。宣言 Ctrl+B ，或使用“ 导航”部分中所述的任何其他方法。

    相应的模板文件在编辑器中打开。

  3. 或者，您可以按Ctrl键，并将鼠标指针悬停在相关区域设置上方的注释上方。注释变为超链接。单击它导航到模板文件。

查看对本地化标签的引用

  1. 将鼠标指针悬停在所需标签旁边的装订线图标上。
