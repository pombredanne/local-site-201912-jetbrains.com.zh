按下此按钮时，所有软件包均显示为单层视图。

当按下此按钮时，在工具窗口中选择的类的源代码会自动在单独的编辑器选项卡中打开，并获得焦点。

当按下此按钮时，当某个类的源代码在编辑器中成为焦点时，相应的节点将在工具窗口中自动突出显示。

单击此按钮生成代码覆盖率报告，并将其保存到指定目录。有关详细信息，请参见生成代码覆盖率报告 。

每次运行Karma测试时，实际上都会在磁盘上生成一份覆盖率报告。覆盖率报告的格式可以在配置文件中配置，例如：

// karma.conf.js module.exports = function（config）{config.set（{...//（可选）配置报告程序coverageReporter：{类型：'html'，目录：'coverage /'} ...}）;};

下列type可接受的值：

  • html产生一堆带有注释源代码的HTML文件。

  • lcovonly产生一个lcov.info文件。

  • lcov生成HTML + .lcov文件。默认情况下会应用此格式。

  • cobertura生成cobertura-coverage.xml文件，以方便Hudson集成。

  • text-summary生成覆盖范围的紧凑文本摘要，通常覆盖控制台。

  • text产生一个详细的文本表，覆盖所有文件。
