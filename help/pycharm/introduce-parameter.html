提取参数

提取参数重构用于向函数声明中添加新参数并相应地更新函数调用。

提取Python参数

默认情况下，在PyCharm中启用就地重构 。因此，如果您尚未更改此设置，则在编辑器中就地执行Python的Extract Parameter重构：

  1. 在编辑器中，将插入号放置在要用参数替换的表达式内。

  2. 请执行以下任一操作：

    • 按Ctrl+Alt+P 。

    • 选择重构|提取物|主菜单中的参数 。

    • 选择重构|提取物|上下文菜单中的参数 。

  3. 如果当前插入符号位置检测到多个表达式，则出现“ 表达式”列表。在这种情况下，请单击表达式以将其选中。或者，按Up要么Down导航到感兴趣的表达，然后按Enter选择它。

  4. 在带有红色边框的框中键入参数名称。

  5. 要完成重构，请按Tab要么Enter 。

    如果您尚未完成重构并想取消所做的更改，请按Escape 。

    请注意，有时您可能需要多次按相应的键。

使用“提取参数”对话框提取参数

为了能够使用“ 提取参数”对话框（而不是就地执行重构），请确保在编辑器设置中关闭了“ 就地重构”选项。

在这种情况下，您可以按照以下步骤执行提取参数重构：

  1. 在编辑器中，将插入号放置在要用参数替换的表达式内。

  2. 请执行以下任一操作：

    • 按Ctrl+Alt+P 。

    • 选择重构|提取物|主菜单中的参数 。

    • 选择重构|提取物|上下文菜单中的参数 。

  3. 如果当前插入符号位置检测到多个表达式，则出现“ 表达式”列表。在这种情况下，请单击表达式以将其选中。或者，按Up要么Down导航到感兴趣的表达，然后按Enter选择它。

  4. 在打开的“ 提取参数”对话框中：

    1. 在名称字段中指定参数名称。

    2. 如果在函数体中找到一个以上的表达式出现，则可以选择用对新参数的引用替换仅选中的出现或所有找到的出现。使用替换所有出现复选框来指定您的意图。

  5. 预览并应用更改 。
