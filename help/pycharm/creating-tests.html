创建测试

PyCharm提出了一种为类和单个方法创建测试的方法。

为类或方法创建测试

  1. 在编辑器中，将插入符号放置在类声明处或方法中的某个位置。

  2. 请执行以下任一操作：

    • 从主菜单中，选择导航|测试 。

    • 从上下文菜单中，选择“ 转到|测试 。

    • 按Ctrl+Shift+T 。

    PyCharm显示可用测试的列表。

  3. 如果所需的测试尚不存在，请点击创建新测试 。

    将打开“ 创建测试”对话框。

  4. 在“ 创建测试”对话框中，指定以下设置：

    • 目标目录，将在其中生成新的测试类。

    • 测试文件的名称和测试类的名称。

    • 选中要包含在测试类中的方法旁边的复选框。
      请注意，如果插入符号已放置在方法内，则列表中仅包含此方法名称。

  5. 准备好后，单击“ 确定” 。PyCharm在指定位置生成测试文件。
