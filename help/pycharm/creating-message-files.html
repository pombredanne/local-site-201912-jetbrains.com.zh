仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

PyCharm将语言环境存储在语言环境目录的与语言相关的子目录中。要创建语言环境，请运行makemessages manage.py实用程序的任务。

创建一个消息文件

  1. 在主菜单上，选择“ 工具”。运行manage.py task ，或按Ctrl+Alt+R 。

  2. 在manage.py任务窗口中，输入makemessages --locale 然后按Enter 。

    对要创建的每个语言环境重复此步骤。

    如果有标记为本地化的字符串，PyCharm将为每个语言环境生成一个目录和django.po文件：

    如果没有这样的字符串，则仅创建一个空的目录结构。
