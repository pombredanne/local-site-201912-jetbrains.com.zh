先决条件

行为或生菜应该下载并安装在您的计算机上！

要安装行为或生菜，请按照下列步骤操作：

  1. 打开设置/首选项对话框Ctrl+Alt+S ，选择项目|项目口译员 。

  2. 按照“ 安装，卸载和升级软件包 ”部分中所述添加行为或生菜 软件包 。

创建特征文件

创建特征文件

  1. 在“项目”工具窗口中，右键单击应在其中创建特征文件的目录。

  2. 从目标目录的上下文菜单中，选择“ 新建” |“新建”。小黄瓜功能文件 ，并指定文件名。

  3. 在功能文件中，键入您的方案。由于没有步骤定义，因此这些步骤将突出显示为未解决。

  4. 创建步骤定义 。
