启用Django支持

此复选框的默认状态取决于项目类型 。对于空项目，将禁用Django支持。对于Django项目，默认情况下启用；您可以根据需要清除此复选框。在这种情况下，其他字段将不可用。

Django项目根目录

默认情况下，此字段显示存储应用程序的settings.py和manage.py文件的目录。如果需要，您可以指定其他位置。

设定值

单击浏览按钮以选择所需的设置文件。

使用以下方法之一：

  • 该文件可以是名称匹配* settings * .py的任何文件，位于Django项目根目录下。

  • 指向任何Python包并将设置存储在__init__.py中 。
    当您想将设置拆分为多个模块并导入它们时，后一种方法很有用。

默认情况下，PyCharm显示位于Django项目根目录下的settings.py文件。

不要使用Django测试运行器

默认情况下，未选中此复选框。You can select it if you want to use any of the alternative test runners specified in the PyCharm Integrated Tools dialog window (File | Settings | Tools | Python Integrated Tools for Windows and Linux and PyCharm | Preferences | Tools | Python Integrated Tools for macOS).
