PyCharm 2019.2帮助

Sass，Less和SCSS

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

PyCharm与编译器集成，可以将Sass ， Less和SCSS代码转换为CSS 。 要在PyCharm中使用编译器，您需要根据相关的预定义模板将其配置为File Watcher 。

您也可以从命令行使用编译器，或者将其配置为第三方工具，有关详细信息，请参阅配置第三方工具 。

在开始之前，请确保您的计算机上装有Node.js。

安装Sass / SCSS

  • 在嵌入式终端 （ Alt+F12 ），输入：

    npm install -g sass

    从Sass官方网站了解更多信息。

减少安装

  • 在嵌入式终端 （ Alt+F12 ），输入：

    npm install --global less

    从Less官方网站上了解更多信息。

您还可以按照npm和Yarn中的说明 ，在Node.js和NPM页面上安装less软件包。

将代码编译成CSS

要即时编译代码，您需要将编译器配置为PyCharm File Watcher 。

创建文件监视程序

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），点击“ 工具”下的“ 文件监视程序 ”。打开的“ 文件监视程序”页面显示已配置的文件监视程序的列表。

  2. 请点击或按Alt+Insert 。根据要使用的工具，从列表中选择Less ， Sass或SCSS预定义模板。

  3. 在“ 程序”字段中，根据所选的预定义模板指定编译器存档的路径。

    • Lessc的lessc 。

    • 上海社会科学院社科院/ SCSS。

    如果您对npm遵循标准安装过程，PyCharm会自行查找所需文件并自动填写字段。否则，请手动输入路径或单击然后在打开的对话框中选择文件位置。

  4. 按照文件监视程序中所述进行操作。

编译代码

当您打开文件时，PyCharm会检查当前项目中是否有适用的文件监视程序。如果已配置但禁用了此类文件监视程序，PyCharm将显示一个弹出窗口，通知您有关已配置文件监视程序的信息并建议启用它。

如果在当前项目中配置并启用了适用的文件监视程序，则PyCharm会在“ 新建监视程序”对话框中指定的事件后自动启动它。

  • 如果选中了“ 自动保存编辑的文件以触发监视程序”复选框，则在对源代码进行任何更改后都会立即调用文件监视程序。

  • 如果清除了自动保存编辑的文件以触发监视程序复选框，则文件监视程序会在保存时启动（ File | Save All ， Ctrl+S ）或将焦点从PyCharm移开（停用帧时）。

PyCharm使用生成的输出创建一个单独的文件。该文件的名称为源Sass ， Less或SCSS文件，扩展名为.css 。在“ 新建监视程序”对话框的“ 刷新的输出路径”字段中定义了生成文件的位置。但是，在项目树中 ，它们显示在源文件下，该文件现在显示为节点。

配置语法突出显示

您可以根据自己的喜好和习惯来配置“少/低/少/ SCSS感知”语法高亮显示。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |配色方案|较少/无礼/ SCSS 。

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
