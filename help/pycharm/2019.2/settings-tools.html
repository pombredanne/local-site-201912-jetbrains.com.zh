When you select the Tools category in the left-hand pane, its main subcategories are listed in the right-hand part of the dialog.

  • Web Browsers
  • File Watchers
    • New Watcher Dialog
  • External Tools
    • Create/Edit/Copy Tool Dialog
  • Terminal
  • Database
    • Data Views
    • User Parameters
    • CSV Formats
  • SSH Terminal
  • Datalore
  • Diagrams
  • Diff & Merge
    • External Diff Tools
  • Education
  • Python External Documentation
  • Python Integrated Tools
  • Remote SSH External Tools
    • Create/Edit/Copy Tool Dialog (Remote SSH External Tools)
  • Server Certificates
  • Settings Repository
  • Startup Tasks
  • Vagrant

This category lists only languages and frameworks, which support is bundled with PyCharm. If you want to work with any specific language, try one of the corresponding available PyCharm plugins. See Managing plugins for more information.
