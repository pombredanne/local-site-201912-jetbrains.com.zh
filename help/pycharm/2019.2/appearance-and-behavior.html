Appearance and Behavior

File | Settings | Appearance and Behavior for Windows and Linux
PyCharm | Preferences | Appearance and Behavior for macOS
Ctrl+Alt+S

When you select the Appearance and Behavior category in the left-hand pane, its main subcategories are listed in the right-hand part of the dialog.

  • Appearance
  • Menus and Toolbars
    • Choose Actions to Add Dialog
  • System Settings
    • Passwords
      • Password Manager Database Updated
      • Setup Master Password Dialog
      • Change Master Password Dialog
      • Enter Master Password Dialog
      • Master Password / Reset Master Password Dialog
    • HTTP Proxy
    • Data Sharing
    • Updates
  • File Colors
  • Scopes
  • Notifications
  • Quick Lists
  • Path Variables

This category lists only languages and frameworks, which support is bundled with PyCharm. If you want to work with any specific language, try one of the corresponding available PyCharm plugins. See Managing plugins for more information.
