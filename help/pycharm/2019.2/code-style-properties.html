File | Settings | Editor | Code Style | Properties for Windows and Linux
PyCharm | Preferences | Editor | Code Style | Properties for macOS

Use this section to configure formatting options for Properties files. When you change these settings, results of a change are displayed in the Preview pane.

Controls

Item

Description

Align properties in columns

Select this checkbox to align properties and their values.

Insert space around key-value delimiter

Select this checkbox to insert whitespace symbols around the key-value delimiters.

Key-value delimiter

Select the desired key-value delimiter from the drop-down list.

Keep blank lines

Select this checkbox to preserve blank lines as is.
