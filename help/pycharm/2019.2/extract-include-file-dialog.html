Extract Include File Dialog

Refactor | Extract Include File

Item

Description

Name for extracted include file

In this field, specify the name of the include file to extract the selected source code to.

No extension is allowed.

Extract to directory

In this field, specify the directory to create the include file in. Either accept the predefined directory, or type another one manually, or click Browse and choose the desired folder in the dialog that opens.
