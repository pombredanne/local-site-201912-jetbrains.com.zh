Create and run setup.py

PyCharm provides an action that helps create setup.py script, intended for building, distributing, and installing modules.

Once setup.py is created, the corresponding action becomes disabled, but instead appears the action that allows running tasks of this utility.

Creating setup.py

  1. In the Project tool window, choose the project you want to package.

  2. From the main menu, choose Tools | Create setup.py.

  3. In the New Setup Script dialog, specify package name, version and the other required information:

  4. Click OK when ready. PyCharm creates setup.py and opens it in the editor.

Running setup.py

  1. From the main menu, choose Tools | Run setup.py.

  2. In the Enter setup.py task name dialog, type the letters of the task names. Note that asterisk wildcard and initial letters of the snake_case names are honored.

    As you type, the suggestion list shrinks to show the matching names only. Choose the desired task, and press Enter.
