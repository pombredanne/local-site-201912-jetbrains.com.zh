Languages and Frameworks

File | Settings | Languages and Frameworks for Windows and Linux
PyCharm | Preferences | Languages and Frameworks for macOS
Ctrl+Alt+S

When you select the Languages and Frameworks category in the left-hand pane, its main subcategories are listed in the right-hand part of the dialog.

  • JavaScript
    • JavaScript. Libraries
      • JavaScript. Usage Scope
    • Bower
    • Yeoman
    • Meteor
    • Cordova
    • Dart
  • Schemas and DTDs
    • Default XML Schemas
    • XML Catalog
  • BDD
  • Django
  • Google App Engine
  • Markdown
  • Node.js and NPM
  • Style Sheets
  • Puppet
  • Python Template Languages
  • SQL Dialects
  • SQL Resolution Scopes
  • TypeScript

This category lists only languages and frameworks, which support is bundled with PyCharm. If you want to work with any specific language, try one of the corresponding available PyCharm plugins. See Managing plugins for more information.
