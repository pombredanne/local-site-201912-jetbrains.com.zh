Extract Superclass Dialog

Refactor | Extract | Superclass

Item

Description

Extract superclass from

This read-only field displays the name of the class, from which a superclass should be extracted.

Superclass name

In this text field, type the Superclass name.

Directory for new class

In this text field, specify the path to the target directory, where the new class will be stored.

Use code completion while you type the path:

You can also click the ellipsis button, or press Shift+Enter, and select the desired path in the Select Path dialog.

Members to form superclass

This area displays the list of members detected in the original class. Select the checkboxes next to the members to be included in the new superclass.
