此对话框是启用以下功能的程序包管理器：

  • 查看软件包，可在远程存储库中找到

  • 在本地安装选定的软件包。

  • 添加和删除存储库

项目

描述

搜索领域

在此处输入搜索字符串。这样，软件包列表将缩小以显示仅包含输入字符串的软件包名称。

  • ：单击此按钮以使用搜索历史记录。

  • ：单击此按钮以清除搜索字段。

包

该区域显示了可访问的远程存储库中的软件包列表。对于每个软件包，您可以查看其名称和存储库。其他信息（描述和版本）显示在右侧。

描述

对于任何选定的软件包，此字段显示简要说明，最新版本，作者姓名，网站和电子邮件地址。

指定版本

如果选中此复选框，则将下载并安装指定版本的软件包。如果未选中此复选框，则将安装最新版本。

选件

选中此复选框以输入pip install命令。

您可以通过键入以下内容探索可能的选项pip install - help在命令行中。

安装套件

单击此按钮以安装选定的软件包。

管理储存库

单击此按钮以打开“ 管理存储库”对话框。默认情况下，仅pypi存储库可用。该存储库无法删除。
可以添加更多存储库，并删除选定的存储库。
