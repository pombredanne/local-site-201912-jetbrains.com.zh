添加一列

  1. 右键单击表，然后选择新建|列 。

  2. 指定列名称和设置。

  3. 点击执行 。

删除栏

  1. 右键单击列，然后选择新建|放下 。或者，按N/A 。

  2. （可选）点击设置图标（ ）配置删除过程。

  3. 单击确定 。

修改栏

  1. 在“ 数据库”工具窗口中，右键单击一列，然后选择“ 修改列” 。

  2. 在“ 修改表”对话框中，指定所需的列设置。

重新排序栏

要重新排序列，请对标题行中的相应单元格进行拖放。

隐藏栏

要隐藏列，请右键单击相应的标题单元格，然后选择“ 隐藏列” 。

或者，您可以右键单击标题行中的任何单元格，然后选择“ 列列表” Ctrl+F12 。在“ 列列表”窗口中，选择列名称，然后按Space 。您可以隐藏所有列。要搜索列列表，只需开始在“ 列列表”窗口中输入列名称。

显示列

  1. 右键单击标题行中的任何单元格，然后选择“ 列列表” 。或者，按Ctrl+F12 。

  2. 选择隐藏的列（删除线列名称），然后按Space 。

或者，您可以重置表的视图。结果，数据变得未排序，列以它们最初定义的顺序出现，并显示所有列。有关重置表格视图的更多信息，请参阅重置初始表格视图 。

为列注入语言

您可以为整个列分配一种正式的语言（例如HTML，CSS，XML，RegExp等）。因此，当您在属于该列的单元格中有一段代码时，可以使用代码生成，代码完成和其他功能（IDE功能）。

要为整个列注入语言，请右键单击相应的标题单元格，然后选择“ 编辑为” 。在支持的语言列表中，选择要注入的语言。

创建一个索引

数据库索引是一种用于加快数据库表中的定位和访问操作的结构。通过使用索引，可以减少处理查询时所需的磁盘访问次数。您可以为数据库表的一个或多个列创建索引。

  1. 在数据库工具窗口（ 视图|工具窗口|数据库 ）中，展开数据源树，直到列的节点。

  2. 右键单击表或列，然后选择“ 新建” |“列”。索引 。

  3. 在“ 列”窗格中，单击“ 添加”按钮（ ）。

  4. 在“ 名称”字段中，指定要添加到索引的列的名称。

  5. 点击执行 。

创建外键

外键是一个表中的一列或一组列，可唯一标识另一列中的行。包含外键的表是子表，包含候选键的表是引用表或目标表。

  1. 在数据库工具窗口（ 视图|工具窗口|数据库 ）中，展开数据源树，直到子表的节点。

  2. 用鼠标右键单击子表，然后选择新建|外键 。

  3. 在“ 目标表”窗格中，指定目标表的名称。

  4. 在“ 列”窗格中，单击“ 添加”按钮 。

  5. 在“ 发件人”字段中，指定子表中列的名称。

  6. 在“ 收件人”字段中，指定目标表中列的名称。

  7. 点击执行 。

修改生成的索引和键名的模板

创建索引以及主键和外键约束时，它们的默认名称是根据相应的模板生成的。例如，对于主键，模板为{table}_{columns}_pk 。

要查看和修改这些模板，请打开设置Working_with_source_code.xml＃L797-L848 Ctrl+Alt+S并导航到编辑器|代码样式| SQL |通用SQL 。单击代码生成选项卡。

模板可以包含变量和文本。生成名称时，将原样复制指定的文本。例如，当您应用{table}_pk中的模板actor表中，生成的主键名称将为actor_pk 。

要查看有关可能的变量及其用法的信息，请单击字段并按Ctrl+Q 。

{unique?u:}检查索引是否唯一，并插入相应的字符序列。如果索引是唯一的，则模板会生成一个名称，该名称具有以下指定的字符序列?和: 。为了{unique?u:}模板，它是u 。如果索引不是唯一的，则序列之间:和}被插入。为了{unique?u:}模板，没什么。

例

你有persons列表FirstName和LastName 。的{table}_{columns}_{unique?u:}index模板为非唯一索引生成以下名称： persons_FirstName_LastName_index 。
