PyCharm 2019.2帮助

首次运行PyCharm

首次运行PyCharm时，或从以前的版本升级PyCharm时，需要一些步骤才能完成安装，自定义实例并开始使用IDE。

导入PyCharm设置

在“ 完整安装”对话框中，选择是否要导入IDE设置：

如果这是您的PyCharm的第一个实例，请选择“不导入设置”选项。

您可以稍后使用文件|手动导出和导入设置 。导入设置和文件|从主菜单中导出设置命令。

选择用户界面主题

选择是要使用默认的Darcula主题还是Light主题。

下载并安装其他插件

如有必要，请从PyCharm插件存储库下载并安装未与IDE捆绑在一起的其他插件。

在PyCharm中启动一个项目

在欢迎屏幕上，您可以执行以下操作：

  • 创建一个新项目
  • 打开一个现有的项目或文件
  • 从版本控制系统中检出现有项目

也可以将现有项目目录或单独的文件拖到“欢迎”屏幕，然后在PyCharm中将其打开。

打开PyCharm并创建项目后，就可以创建并运行第一个Python应用程序了 。
