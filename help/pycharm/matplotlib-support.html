科学模式

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

PyCharm中的科学模式为交互式科学计算和数据可视化提供支持。必须正确安装以下软件包：

  • Matplotlib
  • 脾气暴躁的

要了解如何将软件包安装到您的PyCharm解释器中，请阅读安装，卸载和升级软件包一节 。

当一个numpy如果检测到导入，PyCharm将显示提示您启用科学模式的标语：

科学模式

启用此模式后，UI进行了以下更改：

视图|默认情况下， 未选中“ 科学模式”命令。

  • “ 视图”菜单中的“ 科学模式”检查命令会自动选择。

  • 显示了Python控制台 。

  • 出现“ 文档工具”窗口 （“快速文档”弹出框的固定版本），显示插入符号处的嵌入式文档：

  • DataFrame / Array视图出现在Python控制台和“ 调试工具”窗口中 。

在“科学”模式下，图形将在SciView窗口的自己的选项卡中打开，您可以调整其大小，放大和缩小等等。您可以通过切换在工具窗口中显示图复选框（ 设置/首选项|工具| Python Scientific ）来更改此行为。

科学项目

通过在“ 新建项目”向导中选择相应的项目类型，可以获取所有预定义的科学模式设置。

选择Scientific项目类型时，需要确保已安装Conda解释器。完成此操作后，您只需指定项目名称即可。如果需要，您还可以更改数据文件夹的名称。
