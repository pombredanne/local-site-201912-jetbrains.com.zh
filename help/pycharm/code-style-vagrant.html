制表符和缩进

项目

描述

使用制表符

  • 如果选中此复选框，则使用制表符：

    • 在按Tab键

    • 缩进

    • 用于代码重新格式化

  • 清除复选框后，PyCharm将使用空格而不是制表符。

智能标签

  • 如果选中此复选框，则由代码块嵌套定义的缩进部分由制表符和（如有必要）空格组成，而由对齐方式定义的缩进部分仅由空格组成。

  • 如果清除此复选框，则仅使用选项卡。这意味着将将一组适合指定制表符大小的空格自动替换为一个制表符，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中包含的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组元素之间插入的空格数（如果选择了“ 使用制表符”复选框，则为制表符 ）。

缩进空白行

如果选中此复选框，PyCharm会将缩进保留在空行上，就好像它们包含一些代码一样。
如果清除此复选框，PyCharm将删除制表符和空格。

使用相对于表达式开始的缩进

使用此复选框可以在两种可能的缩进行为之间切换：

  • 如果未选中此复选框，则将根据新行开始的最接近的祖先块对代码块进行格式化。

  • 如果选中此复选框，则代码块将按列设置格式。

在访问修饰符后缩进方法

选择public ， protected要么private复选框，以在相应的访问修饰符之后创建一个额外的缩进。

其他

项目

描述

保留现有的换行符

如果选中此复选框，则在编辑器中查看Ruby文件时会使用换行符。

对齐多行列表项

如果选中此复选框，则顺序行中的呼叫将与: 。

对齐作业或哈希的正确部分

如果选中此复选框，则赋值和散列将与值对齐。

散列中大括号周围的空间

如果选中此复选框，则散列中的开头大括号后跟一个空格，而结束大括号前跟一个空格。

大括号中的花括号周围的空间

如果选中此复选框，则块中的大括号后跟一个空格，而大括号后跟一个空格。

哈希火箭周围的空间

如果选中此复选框， =>字符用空格包围。

范围运算符周围的空间

如果选中此复选框，则范围字符...被空格包围。

赋值运算符周围的空间

如果选中此复选框，则赋值运算符将被空格包围，即使它们是多行的也是如此。

关系运算符周围的空间

如果选中此复选框，则关系运算符（例如== ， ~= ）被空格包围。

其他运算符周围的空间

如果选中此复选框，则运算符（例如+ ）被空格包围。

在可见性修改器周围强制换行

默认情况下启用此选项。当您不需要修改访问权限时，请用新行将其取消选择。

转换{}块以执行...按Enter键结束

默认情况下启用此选项。如果要通过按Enter限制转换{}块，请取消选择它。

在声明中保留空白行

指定重新格式化后要保留的声明中的空行数。

方法周围的空白行

指定重新格式化后要保留的方法周围的空白行数。
