烧瓶

仅专业版支持此功能。该版本是商业版本，它提供了一组出色的功能：有关更多详细信息，请参见比较矩阵 。

PyCharm支持Flask开发，包括：

  • 专用项目类型 。

  • 支持内置的Flask调试器。

  • Flask命令行界面（CLI）。

  • Jinjia2，Django和Mako 模板支持。

  • Flask组中的实时模板可创建Flask路线的存根。

  • 在视图和模板之间导航 。

  • 代码完成并解决。

在专业版中默认启用Flask集成。如果您需要禁用Flask集成（例如，忽略Flask路由或禁止生成新的Flask Run / Debug配置），请在“ 语言和框架|框架”中取消选中相应的复选框。项目“首选项/设置”中的“ 烧瓶”对话框窗口。
