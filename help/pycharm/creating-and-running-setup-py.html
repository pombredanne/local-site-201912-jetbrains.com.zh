创建并运行setup.py

PyCharm提供了一个有助于创建setup.py脚本的操作，该脚本用于构建，分发和安装模块。

创建setup.py后 ，相应的操作将被禁用，但会显示允许该实用程序运行任务的操作。

创建setup.py

  1. 在项目工具窗口中，选择要打包的项目。

  2. 在主菜单中，选择“ 工具” |“工具”。创建setup.py 。

  3. 在“ 新建安装脚本”对话框中，指定程序包名称，版本和其他必需信息：

  4. 准备好后，单击“ 确定” 。PyCharm创建setup.py并在编辑器中将其打开。

运行setup.py

  1. 在主菜单中，选择“ 工具” |“工具”。运行setup.py 。

  2. 在“ 输入setup.py任务名称”对话框中，键入任务名称的字母。注意星号通配符和snake_case名称的首字母均被接受。

    键入时，建议列表将缩小以仅显示匹配的名称。选择所需的任务，然后按Enter 。
