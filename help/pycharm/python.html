要在PyCharm中使用Python ，请至少安装一个Python解释器。

支持的版本

  • Python 2： 2.7版

  • Python 3：从3.5版到3.8版

以下功能是Python 3.8支持的一部分

  • PEP-570 –仅Python位置参数
  • PEP-572 –作业表达
  • PEP-591 –在输入中添加最终限定词
  • Python错误跟踪器任务＃36817 –支持= f字符串中的表达式

PyCharm集成了Python 3.7的以下功能：

  • PEP-561 –分发和包装类型信息
  • PEP-544 –协议：结构化子类型化（静态鸭式输入）
  • PEP-557 –数据分类
  • PEP-560 –键入模块和通用类型的核心支持
  • PEP-562 –模块__getattr__和__dir__
  • PEP-563 –注释的延迟评估

PyCharm在以下支持下提供对Python 3.5和（自2016.3起）Python 3.6的支持：

  • PEP-484-类型提示
  • PEP-448-其他拆包概述
  • PEP-492-具有异步和等待语法的协程
  • PEP-526-变量注释的语法
  • PEP-498-文字字符串插值
  • PEP-515-数字文字中的下划线
  • PEP-525-异步发电机
  • PEP-530-异步理解
