咕unt工具窗口

Gruntfile.js的上下文菜单-显示Grunt任务
查看|工具窗口|咕unt声

使用此工具窗口运行Grunt任务 。执行任务的结果显示在“ 运行工具”窗口中 。 工具窗口显示Grunt输出，报告发生的错误，列出未找到的软件包或插件，等等。最后执行的任务的名称显示在工具窗口的标题栏上。

在你开始前

  1. 下载并安装Node.js。

  2. 全局安装grunt-cli软件包（Grunt命令行界面），并将grunt软件包作为开发依赖项安装。有关详细信息，请参见安装Grunt 。

运行任务和目标

打开Grunt工具窗口

  • 在“ 项目”工具窗口中选择所需的Gruntfile.js文件，或在编辑器中将其打开，然后从上下文菜单中选择“ 显示Grunt任务 ”。

当您在RubyMine会话中首次构建任务树时，尚未打开Grunt工具窗口。

建立任务树

  • 在“ 咕unt”工具窗口中，单击在工具栏上，然后从列表中选择所需的Gruntfile.js文件。默认情况下，RubyMine在项目的根目录中显示Gruntfile.js文件。

  • 如果您还有另一个Gruntfile.js文件，请单击“选择Gruntfile.js” ，然后在打开的对话框中选择所需的Gruntfile.js文件。RubyMine在其标题上添加一个具有所选Gruntfile.js文件路径的新节点，并在新节点下构建任务树。

重建一棵树

  • 切换到所需的节点，然后单击在工具栏上。

按名称对树中的任务进行排序

  • 请点击在工具栏上，从菜单中选择“ 排序依据 ”，然后选择“ 名称” 。
    默认情况下，一棵树按照在Gruntfile.js中定义的顺序显示这些任务（选项Definition order ）。

运行任务或目标

  • 双击任务或目标。

  • 在树中选择任务或目标，然后按Enter或选择运行从上下文菜单中。

运行默认任务

  • 在树中选择根节点，然后在所选内容的上下文菜单上选择“运行默认值 ”。

运行多个任务或目标

  • 使用多选模式：按住Shift （用于相邻项目）或Ctrl （对于非相邻项目）键，然后选择所需的任务或目标，然后从所选内容的上下文菜单中选择“运行”或“ 调试 ”。

跳转到任务或目标的定义

  • 在树中选择所需的任务或目标，然后从所选内容的上下文菜单中选择“ 跳转到源 ”。

工具列

添加Gruntfile

单击此按钮可以为另一个Gruntfile.js文件构建任务树。从列表中选择所需的Gruntfile.js文件。RubyMine添加一个新节点并在其下构建任务树。

删除Gruntfile

单击此按钮以删除所选节点下的任务树。

重新加载任务

单击此按钮可以重新构建所选节点下的任务树。更新相应的Gruntfile.js文件后，您可能需要重新构建树，因为Grunt不会动态对树进行更改。

全部收缩

单击此按钮可隐藏所有任务树，并且仅显示Gruntfile.js节点。

单击此按钮可配置当前视图并更改工具窗口的查看模式 ，有关详细信息，请参见工具窗口查看模式 。请注意，大多数菜单项都是可以打开或关闭的选项。启用的选项的名称左侧带有一个复选标记。Grunt特定的选项是：

  • Grunt设置：选择此菜单项以打开Grunt设置对话框，然后重新配置Grunt和Node解释器的当前设置，请参阅任务运行器 。

  • 排序方式：选择此菜单项可配置任务在树中的显示顺序。 请点击在工具栏上，从菜单中选择“ 排序依据 ”，然后选择“ 名称” 。
    默认情况下，一棵树按照在Gruntfile.js中定义的顺序显示这些任务（选项Definition order ）。

隐藏

单击此按钮可以隐藏工具窗口。要使其再次显示，请选择“ 视图”工具窗口|在主菜单上发出咕unt声 。再次出现工具窗口，显示所有先前构建的任务树。

一棵树的上下文菜单

咕unt声设置

选择此菜单项以打开“ Grunt设置”对话框并查看或编辑Node.js配置

跳转到源

选择此菜单项以打开为其构建当前树的Gruntfile.js文件。

重新加载任务

选择此菜单项可重新构建选定节点下的任务树。

复制路径

选择此菜单项可将当前树构建到剪贴板的路径保存到Gruntfile.js文件。

删除Gruntfile.js

选择此菜单项可删除所选节点下的任务树。

任务或目标的上下文菜单

跑

选择此菜单项以运行选定的任务或目标。

除错

选择此菜单项可调试选定的任务或目标。

编辑设定

选择此菜单项以打开“ 运行/调试配置”对话框，并编辑所选任务或目标的预定义设置。

跳转到源

选择此菜单项以打开为其构建当前树的Gruntfile.js文件，并导航到所选任务或目标的定义。
