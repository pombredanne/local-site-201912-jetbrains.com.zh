导入表对话框

将数据从一个表导出到另一表时，或将表导出到另一数据库或架构时，此对话框打开。

在此对话框中，指定数据映射信息和目标表的设置。

表名，结构和数据映射

项目

描述

表

目标表的名称。

评论

表注释。

列/键等

列的数据映射，以及列的定义，约束和索引。

  • 要开始编辑列的信息，请双击列列表中的相应行。

    注意映射到字段。该字段使您可以指定要导入的表中的哪一列应用作目标表中相应列的数据源。如果清除此字段，则不会将任何数据添加到目标列。

  • 要删除列，请选择相应的行，然后单击 。

数据和DDL预览

项目

描述

资料预览

您将要导入的数据的预览。如果您使用“ 导出到数据库”命令开始导入，则此选项卡不可用（ ）在数据编辑器或数据库控制台的“结果”窗格中。

DDL预览

将要运行的一个或多个语句。您可以在预览窗格中直接编辑语句。

将错误写入文件，并将不可转换的值插入为null

项目

描述

将错误写入文件

如果选中此复选框，则导入期间发生的错误将写入指定的文本文件。对于每个错误，都将指示错误发生的位置。

将不可转换的值插入为null

如果选中此复选框，则对于无法导入的数据，会将NULL插入表中。
