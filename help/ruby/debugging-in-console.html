教程：在IRB / Rails控制台中进行调试

运行IRB或Rails控制台后，RubyMine创建一个临时的运行/调试配置 。通过此配置，您可以重新运行控制台或以调试模式启动它。在本教程中，我们将向您展示如何在IRB控制台中加载和调试代码的特定部分。

本教程中使用的示例Ruby应用程序有两个文件-script.rb和song.rb。我们将调试新创建的Song对象的to_s方法。

require'./song'将Song.new（“ My Way”，“ Sinatra”）。to_s放入
class Song def initialize（name，artist）@name = name @artist = artist end def to_s“歌曲：＃{@ name}-＃{@ artist}”结束

执行以下步骤在IRB控制台中调试此程序代码。

  1. 打开song.rb文件，并在to_s方法中设置断点，如下所示。

  2. 在调试模式下运行IRB控制台配置。为此，请在主工具栏上选择此配置，然后单击按钮。

  3. 在打开的song.rb文件中，将插入符号放置在任意行，然后选择“ 工具” |“工具”。将文件/选择加载到IRB / Rails控制台 Shift+Alt+L从主菜单。这会将代码从文件加载到IRB控制台。

  4. 打开script.rb文件，然后选择to_s方法调用，如下所示。

    然后，选择工具|将文件/选择加载到IRB / Rails控制台 Shift+Alt+L从主菜单。RubyMine将此选择加载到IRB控制台，并在断点处停止程序执行。

    在调用的“ 调试工具”窗口中，您可以调查应用程序状态：框架，变量值等。在“ 检查暂停的程序”中了解更多信息。
