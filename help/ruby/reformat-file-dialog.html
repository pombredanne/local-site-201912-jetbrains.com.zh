重新格式化文件对话框

代码重新格式化代码

Ctrl+Shift+Alt+L

按下时出现对话框Ctrl+Shift+Alt+L在当前文件的编辑器中。如果选择代码|从主菜单重新设置代码格式或按Ctrl+Alt+L ，RubyMine尝试自动重新格式化指定范围的源代码。

有关如何重新格式化或排除文件格式的更多详细信息，请参阅“重新格式化并重新排列代码” 。

项目

描述

仅VCS更改了文字

如果选中此复选框，则重新格式化将仅应用于在本地已更改但尚未签入到存储库中的文件。
此复选框仅适用于受版本控制的文件。

所选文字

选择此选项可重新格式化当前选定的源代码片段。

整个档案

选择此选项可重新格式化当前文件中的所有源代码。

优化进口

选中此复选框可从所选范围内的代码中删除未使用的导入语句。

重新排列代码

选中此复选框可以根据“代码样式”设置的“ 排列”选项卡中指定的配置对源代码条目重新排序。

该复选框不适用于Python文件。

代码清理

选择此选项可运行代码清除检查。RubyMine查找并自动修复可能有问题的代码片段。

跑

单击此按钮开始在指定范围内重新格式化源代码。

您可以定义规则并在编写代码时格式化代码。要为Ruby定义格式设置规则， 打开 设置/首选项对话框Ctrl+Alt+S并导航到编辑器|代码样式|红宝石。有关为Ruby配置代码样式的更多信息，请参见配置SQL代码样式 。
