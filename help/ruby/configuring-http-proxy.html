配置HTTP代理

Subversion将http代理设置存储在用户运行时配置区域（在Unix系统上为〜/ .subversion / auth /或在Windows上为/ Application Data / Subversion / auth / ）的服务器文件中。

您可以通过两种方式配置Subversion代理设置：

  • 手动编辑服务器文件 。

  • 直接从RubyMine配置代理设置。请执行下列操作：

    1. 打开 版本控制| RubyMine设置的Subversion页面Ctrl+Alt+S ，然后打开“ 网络”选项卡。

    2. 单击“ 编辑网络选项”按钮，然后在打开的“ 与网络层相关的编辑子版本选项”对话框中指定代理设置。
