RubyMine 2019.3帮助

运行Ruby脚本

RubyMine允许您从编辑器或使用配置的一组参数（ 运行/调试配置）快速运行或调试Ruby脚本。例如，您可以为Ruby运行/调试配置指定以下参数：脚本参数，环境变量，其他命令行参数，等等。

在本主题中，我们将向您展示如何：

  • 在没有运行配置的情况下首次运行Ruby脚本

  • 使用运行配置运行Ruby脚本

  • 配置其他参数以运行脚本

运行脚本而不进行配置

要首次运行指定的Ruby脚本（例如script.rb ），请执行以下操作之一：

  • 按Ctrl两次并键入以下命令：

    红宝石script.rb

    按Enter 。

    要在项目根目录之外运行暂存文件或脚本，请按住Alt在运行命令之前按键（在这种情况下，对话框标题将更改为在上下文中运行 ）。

  • 用打开的脚本右键单击编辑器 ，然后从上下文菜单中选择“运行'脚本” 。

  • 在项目工具窗口中右键单击脚本，然后从上下文菜单中选择“运行'脚本'” 。

运行Ruby脚本后，RubyMine自动创建一个特殊的配置文件-临时运行/调试配置 。您可以自定义此配置的设置 ，例如，传递命令行参数，配置环境变量等。然后，您可以保存自定义配置，以在将来快速运行此配置 。

使用运行/调试配置运行脚本

您可以通过以下方式之一使用现有的运行/调试配置来运行Ruby脚本：

  • 按Ctrl两次调用“ 运行任何东西”弹出窗口。开始输入所需的配置名称，从列表中选择它，然后按Enter 。

  • 点击工具栏上的按钮可运行当前选定的运行/调试配置。

  • 选择运行|跑 Shift+F10从主菜单输入命令，然后选择所需的配置。

配置用于运行脚本的参数

首次运行Ruby脚本时 ，RubyMine会自动创建一个相应的Ruby 临时配置 ，可以将其保存。如有必要，您可以从预定义的模板中手动创建Ruby运行/调试配置。

要自定义运行/调试配置，请执行以下操作：

  1. 打开“ 运行/调试配置”对话框：

    • 选择运行|从主菜单编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按Shift+Alt+F10 ，然后按0或从弹出窗口中选择配置，然后按F4 。

  2. 在打开的“运行/调试配置”对话框中，在Ruby组中选择所需的配置，然后指定其设置。

    例如，您可以在“ 配置”选项卡中指定以下设置：

    选项

    描述

    Ruby脚本

    指定要执行的Ruby脚本的标准路径，或单击 ，然后在打开的对话框中选择所需的Ruby脚本。

    脚本参数

    指定将在脚本文件名之后传递的命令行参数。这些参数将在全球范围内供您的Ruby程序使用ARGV数组。例如：

    • 将以下值作为脚本参数传递： "Jet" "Brains" 。

    • 运行以下Ruby脚本：

      放置“您好，＃{ARGV.join}！我是露比”

      程序将返回Hello, JetBrains! I'm Ruby! 。

    工作目录

    指定正在运行的脚本使用的工作目录。例如，当正在运行的脚本通过相对路径加载其他脚本时，此选项有效。

    环境变量

    将环境变量列表指定为名称/值对，以分号分隔。或者，单击省略号按钮以创建变量并在“ 环境变量”对话框中指定其值。

    Ruby参数

    指定要传递给Ruby解释器的命令行参数 。

    Ruby SDK

    指定用于运行脚本的Ruby解释器 。您可以选择项目默认的Ruby SDK，也可以从已配置的Ruby SDK列表中选择另一个。
