RubyMine 2019.3帮助

查看和快速处理变更列表

如果您正在使用的服务器上，并在本地工作副本颠覆1.5或更高版本，您可以通过实现扩展的合并信息的功能优势合并信息的窗格 版本控制窗口Alt+9 ，“ 存储库”选项卡 。

使用此功能，您可以定义一组要显示的变更列表，而不必在一定时间内浏览所有变更列表。这是通过在存储库中指定一对分支（源和目标）来完成的，随后RubyMine仅显示源分支中已集成到目标分支中的变更列表。

此外，您可以指定各种筛选选项以最大程度地减少无关的变更列表的数量。

最后，集成和管理集成状态也可以直接从“ 合并信息”窗格中获得。

扩展的浏览功能包括：

  • 定义要显示的变更列表集
  • 过滤掉无关的变更列表
  • 从版本控制工具窗口集成文件和变更列表
  • 查看和管理集成状态

如果使用的是SVN 1.4或更低版本， 则要启用“ 合并信息”功能，需要首先更改本地工作副本的格式 。

在启用扩展的“合并信息”功能之前，请确保您正在使用SVN Server 1.5或更高版本。
