安装Ruby / Rails

要使用Ruby编程语言，您需要下载并安装所需的Ruby发行版： 下载Ruby 。RubyMine自动检测安装在本地计算机上并由版本管理器维护的解释 器 。支持以下解释器：

  • Ruby MRI（1.8.7版和更高版本）

  • 红宝石

  • Rubinius（无需调试 ）

  • TruffleRuby（无需调试 ）

对于Rails项目，支持以下框架版本：

  • Rails 3.2.x需要Ruby 1.8.7或更高版本。

  • Rails 4需要Ruby 1.9.x或更高版本。

  • Rails 5需要Ruby 2.2.2或更高版本。

  • Rails 6需要Ruby 2.5.0或更高版本。
