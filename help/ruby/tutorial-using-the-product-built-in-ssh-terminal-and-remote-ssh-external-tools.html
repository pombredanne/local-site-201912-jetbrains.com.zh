使用RubyMine内置的SSH终端和远程SSH外部工具

仅当安装并启用了SSH Remote Run插件时，以下内容才有效。

本教程的内容

本教程描述了如何利用RubyMine内置的SSH终端和远程工具。

SSH基础知识不在本教程的讨论范围之内。

先决条件

在开始之前，请确保：

  • 您正在使用RubyMine 2016.1或更高版本。本教程是使用RubyMine 2016.2创建的

  • 您有权访问SSH服务器。

让我们看看如何使用RubyMine的内置SSH客户端。

提供连接信息

使用远程SSH外部工具

让我们定义一个外部工具，该工具将通过SSH运行命令，例如显示日历。

配置SSH外部工具

打开“设置/首选项”对话框（ 在主工具栏上），然后在“ 工具”节点下，选择“ 远程SSH外部工具 ”页面。请点击创建一个新的远程工具，并在“ 创建工具”对话框中指定新的工具设置：

  • 名称字段有助于指定新工具的一些可见名称。接下来的两个字段是可选的：

    • 在描述字段中，输入一些有意义的描述。

    • “ 组 ”字段表示将在“ 工具”菜单中创建的具有指定名称的组，新的SSH外部工具将放置在该组下。

  • 在“ 显示在”区域中，指定要在哪里查看新工具。在这种情况下，仅选中复选框主菜单 。

  • 在“ 连接设置”区域中，单击“ 部署服务器 ”单选按钮，然后在每次运行时选择“ 选择服务器” 。这意味着每次您要运行此外部工具时，都必须指定连接设置。

  • 最后，在“ 工具设置”区域中，指定要远程执行的工具。在下面的屏幕截图中，我们正在运行bash命令；参数和工作目录是可选的。我们还可以使用宏来注入当前命令名称。

单击“ 确定”关闭“ 创建工具”对话框，并返回到“ 远程SSH外部工具 ”页面。我们可以在列表中看到新工具：
