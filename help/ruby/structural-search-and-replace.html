结构搜索和替换

常规的搜索过程没有考虑源代码的语法和语义。即使您使用正则表达式，RubyMine仍会将您的代码视为正则文本。结构搜索和替换 （ SSR ）操作使您可以考虑代码结构来搜索代码中的特定代码模式或语法结构。

RubyMine根据您创建的搜索模板和您应用的条件来查找并替换源代码的片段。

从结构上搜索目标

  1. 从主菜单中，选择编辑|查找|搜索结构上开结构搜索对话框。

    在“ 结构搜索”对话框中，可以快速切换到“ 结构替换”对话框。请点击 ，然后选择切换到替换 。

  2. 在“ 结构搜索”对话框中，执行下列操作之一：

    • 从头开始创建自己的模板 。

      在编辑器区域中，输入代码模式（ $variable$代表您的代码），请点击 ，然后选择保存模板以将其保存以备将来使用。

      RubyMine将创建的模板添加到“ 现有模板”对话框中的现有模板列表（“ 用户定义”节点）中。

    • 使用现有模板之一作为原型 。

      请点击 ，选择“ 现有模板”以打开预定义模板的列表，然后选择适当的模板。

      考虑以下添加到HTML文件的样式定义：

      HTML文件的标签。

      在现有模板列表中，选择XML / HTML下的html属性值 。

      单击“ 确定”返回到“ 结构搜索”对话框，以使用所选模板。

      RubyMine立即在编辑器中突出显示找到的代码。

  3. “ 结构搜索”对话框显示所选模板以及该模板中使用的过滤器的值。您可以编辑现有过滤器或添加新条件 ，例如，添加一些正则表达式或脚本约束 。将插入号放在代码变量处，然后使用过滤器区域管理过滤器。

  4. 在过滤器区域中，单击添加新条件。例如，如果需要添加正则表达式，请选择Text 。您还可以根据变量添加其他条件 。

  5. 在此示例中，设置以下条件：

    • 为$ tag $ –text=div

    • 为$ attribute $ –text=id

    • 为$ value $ –text=element[1-9]

  6. 指定以下选项以进一步缩小搜索范围：

    • 文件类型 -使用列表选择哪些文件类型应作为搜索的一部分。在我们的例子中是Ruby 。

    • 范围 -指定搜索位置：在项目，模块，目录或自定义范围内。

    • 搜索目标 -您可以选择完全匹配以匹配确切的模板或其他可用的代码元素（在我们的示例中为）。

  7. 点击查找 。

RubyMine在“ 查找”工具窗口中显示结果。

从结构上替换目标

  1. 从主菜单中，选择编辑|查找|结构上更换 。

  2. 在“ 替换结构”对话框中，将新模板或现有模板添加到搜索和替换模板区域。您可以使用与搜索模板相同的方式保存替换模板。

  3. 如果需要在替换模板中为变量添加过滤器，请在所需的变量处插入插入符号，然后使用过滤器区域来管理过滤器。

  4. 在过滤器区域中，根据您选择的过滤器 ，指定条件。

  5. 要缩小替换结果的范围，请选择以下选项：

    • 缩短全限定名 -用短名称和导入替换全限定类名。

    • 重新格式化 -自动格式化替换的代码。

    • 使用静态导入 -尽可能使用静态导入进行替换。

    指定必要的选项后，点击查找 。RubyMine在“ 查找”工具窗口中显示结果。

  6. 在“ 查找工具”窗口中，您可以进一步处理结果，一次一地替换找到的项，一次全部替换，或者预览可能的更改。

考虑下面的示例，该示例转换class归因于p标记为小写。在搜索模板字段中，我们创建了一个模板，用于搜索所有p具有大写值的标签（ [A-Z].*并匹配大小写 ） class属性。我们仅将搜索范围缩小到class值（ 搜索目标= b ）。在替换模板字段中，我们创建一个新变量$d$并为其分配一个Groovy脚本（ b.getText().toLowerCase() ）。搜索后，我们选择全部替换以执行替换。

共享搜索模板

您可以通过导出或导入与他人共享搜索模板。

  1. 在“ 结构搜索”对话框（“ 编辑” |“查找” |“结构搜索” ）中， 创建一个新的搜索模板或使用现有的模板 。

  2. 要导出模板，请单击 ，然后选择将模板导出到剪贴板 。RubyMine将模板的XML表示添加到剪贴板（按Ctrl+Shift+V查看剪贴板的内容）。您可以在聊天，电子邮件或论坛中与其他开发人员共享此表示。

    要导入模板，请复制（ Ctrl+C ）从任何地方（电子邮件，聊天或论坛）共享的XML代码，然后在“ 结构搜索”对话框中，单击然后选择从剪贴板导入模板 。RubyMine采用XML代码表示形式，并将其转换为包含变量和范围（如果存在）的模板。
