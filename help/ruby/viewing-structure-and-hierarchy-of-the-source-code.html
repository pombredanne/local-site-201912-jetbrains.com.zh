代码层次结构

RubyMine使您可以在“层次结构”工具窗口中检查类和调用的层次结构。

建立类的层次结构

  1. 通过以下方式之一选择所需的类：

    • 在编辑器中，将插入号放在类名上。

    • 在“结构”工具窗口中选择类（ Alt+7 ）。

    • 在“项目”工具窗口的“ Rails”视图中选择类。

  2. 从主菜单中，选择导航|键入层次结构或按Ctrl+H 。

  3. 使用要么按钮分别显示父类或子类的层次结构。

建立方法调用的层次结构

  1. 通过以下方式之一选择所需的方法：

    • 在编辑器中，将插入符号放在方法声明或用法处。

    • 在“结构”工具窗口中选择方法（ Alt+7 ）。

    • 在“项目”工具窗口的“ Rails”视图中选择方法。

  2. 从主菜单中，选择导航|呼叫阶层或按Ctrl+Alt+H 。

  3. 使用按钮以显示您正在研究的方法所使用的方法。
