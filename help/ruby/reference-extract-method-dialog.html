提取方法对话框

重构|提取方法
Ctrl+Alt+M

在JavaScript上下文中，对话框的标题可能会更改为Extract Function 。

项目

描述

名称

在此文本框中，根据所选的源代码指定要生成的函数或方法的名称。

能见度

在此区域中，指定要生成的方法的可见性范围。

如果存在私有或受保护的部分，则将新生成的方法添加到那里。如果它们不存在，则会生成它们（不使用public部分，请参见https://github.com/bbatsov/ruby-style-guide#consistent-classes ）。

参量

在此区域中，选择要传递给新方法/功能的参数。

  • 如果未选择对新方法的功能至关重要的参数，则RubyMine将无法继续进行重构。

上/下移动

使用这些按钮可以更改参数的顺序。

签名预览

在此只读字段中，查看新方法/函数的声明。

输出变量

此只读字段显示在方法主体中更改的局部变量或参数，并将由方法返回。
