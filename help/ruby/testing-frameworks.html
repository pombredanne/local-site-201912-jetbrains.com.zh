支持的测试框架/库

RubyMine提供了与用于测试Ruby / Rails应用程序的最流行框架的集成。您可以运行多个或单个测试，在一个测试和一个测试主题之间导航，生成缺少的测试，等等。

支持以下测试框架：

  • 最小测试

    Minitest是一个测试框架，支持TDD和BDD方法，模拟和基准测试。在RubyMine中运行Minitest测试之前，请确保满足以下先决条件：

    • 将“ minitest”和“ minitest-reporters” gem添加到Gemfile中，并安装到项目的SDK中 。

    • 以下代码已添加到您的测试中：

      需要'minitest / reporters'MiniTest :: Reporters.use！

      例如，在Rails应用程序中，应将此代码添加到test / test-helper.rb文件中。

  • 规范

    RSpec是一个流行的BDD框架，用于测试Ruby / Rails应用程序。根据您的项目类型（Ruby或Rails），使用rspec或rspec-rails gem。

  • 黄瓜

    Cucumber是使用Gherkin语言编写测试方案的BDD测试工具。要了解有关RubyMine支持的Cucumber功能的更多信息，请参见Cucumber 。

  • 测试::单位

    Test :: Unit是用于测试Ruby / Rails代码的传统TDD框架。

除了上面列出的框架之外，RubyMine还支持以下测试库：

  • a

    除了标准的Minitest和RSpec测试之外，RubyMine还允许您运行使用Shoulda的测试。

  • 工厂启动

    FactoryBot是一个用于将Ruby对象设置为代替夹具的测试数据的库。RubyMine允许您安全地重命名或删除工厂， 完成工厂属性和方法（例如create和build ），从模型导航到工厂，反之亦然，等等。

  • SimpleCov

    SimpleCov是RubyMine用于测量代码覆盖率的分析工具。从代码覆盖率主题中了解如何执行此操作。
