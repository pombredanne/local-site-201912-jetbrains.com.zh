创建新的RVM宝石集

档案|设置| Ruby SDK和Gems- 对于Linux
RubyMine |首选项| Ruby SDK和Gems- 对于macOS

RubyMine允许您在“设置/首选项”对话框中创建RVM宝石集。请参阅“设置/首选项”对话框中的“创建RVM gemset ”部分。

控制项

项目

描述

Ruby SDK

在此字段中，指定所需的Ruby SDK。如果为您的项目选择了RVM SDK，则将使用它。如果选择了SDK或非RVM SDK（例如，系统， chef要么remote ），则使用默认的SDK。

RVM宝石集

指定所需的宝石集名称。
