调试服务器和客户端代码

使用RubyMine，您可以调试Node.js应用程序的服务器端代码及其客户端JavaScript代码。为此，除了Node.js配置之外，您还需要创建一个单独的JavaScript Debug配置，并将这两个配置一起运行。您可以通过两种方式做到这一点：

  • 从“ 编辑配置”对话框中创建一个JavaScript调试配置（ “运行” |“编辑配置” ），然后单击“确定”。 要启动它，请参阅调试在外部Web服务器上运行的应用程序以获取详细信息。

  • 在创建或编辑Node.js配置时，从“ 实时编辑”选项卡创建JavaScript调试配置。在这种情况下，每次您启动Node.js配置时，JavaScript Debug配置都会自动启动。

从Node.js配置创建JavaScript调试配置

  1. 从主菜单中，选择运行|编辑配置 。

  2. 从列表中，选择Node.js运行配置以与JavaScript Debug配置一起启动。在打开的对话框中，切换到“ 浏览器” /“实时编辑”选项卡。

  3. 选择启动后，在启动调试会话时自动启动浏览器。

  4. 在下面的字段中，输入URL地址以打开应用程序。

  5. 从启动后复选框旁边的列表中选择要使用的浏览器。

    • 要使用系统默认浏览器，请选择默认 。

    • 要使用自定义浏览器，请从列表中选择它。请注意，只有Chrome才完全支持Live Edit 。

    • 要配置浏览器，请单击并在打开的“ Web浏览器”对话框中调整设置。有关更多信息，请参见Web浏览器 。

  6. 选中使用JavaScript调试器复选框。
