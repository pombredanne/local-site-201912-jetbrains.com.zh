创建和发布您的第一个Ruby gem

本分步教程将向您展示如何创建一个简单的Ruby gem并将其发布到RubyGems.org 。

在开始本教程之前，请执行以下操作：

  • 安装Git 。

  • 在RubyMine中注册您的GitHub帐户。

  • 为您的平台安装Ruby发行版 。

您可以通过以下链接下载完成的项目： https : //github.com/rubyminedoc/hello_rubymine

创建一个宝石应用程序

要创建新的Gem应用程序，请按照以下步骤操作：

  1. 运行RubyMine并在“ 欢迎屏幕 ”上单击“ 创建新项目 ” 。

  2. 在“ 新建项目”对话框中，在左窗格的“ Ruby”组中选择“ 宝石” ，然后指定以下设置：

    • 位置 ：指定项目的位置和名称（本例中为hello_rubymine ）。

    • Ruby SDK ：选择系统上安装的必需Ruby解释器。

    • （可选）如果您项目的SDK中未安装Bundler gem，请点击安装 。在调用的对话框中，选择所需的版本，然后单击“ 安装” 。

    指定所有选项后，在“ 新建项目”对话框中单击“ 创建 ”。RubyMine将创建一个新的Gem应用程序。

添加代码

gem的代码位于lib目录中。我们新创建的项目在此文件夹中包含hello_rubymine.rb文件。让我们为gem添加工作代码：

  1. 要打开hello_rubymine.rb ，请按Ctrl+Shift+N ，开始输入hello_rubymine.rb ，选择此文件，然后单击Enter 。

  2. 在打开的lib / hello_rubymine.rb文件中，添加以下代码：

    需要“ hello_rubymine / version”模块HelloRubymine def self.greet（name）放置“ Hello，＃{name}！我是露比“结束

提供宝石规格

每个gem项目都有* .gemspec文件，其中包含有关gem的信息。在我们的项目中，此信息存储在hello_rubymine.gemspec文件中。执行以下步骤以提供所需的数据：

  1. 按Ctrl+Shift+N ，开始输入hello_rubymine.gemspec ，选择hello_rubymine.gemspec文件，然后单击Enter

  2. 在打开的hello_rubymine.gemspec文件中，指定所需的gemspec属性 。

    对于示例gem，我们指定了以下字段：

    • authors ：宝石的作者。

    • email ：电子邮件地址。

    • summary ：简短的宝石描述

    • homepage ：gem主页的URL。在将gem的代码发布到GitHub后，我们将在稍后指定此地址。

    • metadata["source_code_uri"] ：gem的源代码URI。我们将使用与homepage 。

    • metadata["allowed_push_host"] ：我们从* .gemspec中删除了此字符串，以便发布到RubyGems。

在GitHub上共享gem

在这一部分中，我们将把gem的源代码发布到GitHub：

  1. 从主菜单中，选择VCS |。导入版本控制|在GitHub上共享项目 。

  2. 在调用的对话框中，指定要在GitHub上创建的存储库的名称。然后，在“ 共享方式”字段中选择所需的帐户，然后单击“ 共享”按钮。

  3. 在下一个对话框中，将提示您选择用于初始提交的文件。保留默认的文件集，指定提交消息，然后单击“ 添加” 。

    然后，等待项目推送到GitHub。

  4. 在GitHub上创建项目后， 提供 homepage和metadata["source_code_uri"]项目* .gemspec文件中的属性（在我们的示例中为https://github.com/rubyminedoc/hello_rubymine ）。然后， 提交并推送在* .gemspec中所做的更改。

建立宝石

在* .gemspec中进行所有更改之后 ，我们可以构建一个gem：

  1. 在主菜单中，转到“ 工具” |“工具”。宝石|建立宝石 。

  2. 在调用的对话框中，单击“ 生成”按钮以生成 gem。RubyMine将在“ 运行”工具窗口中显示有关创建的gem的信息。

    您可以在“项目”视图中看到创建的宝石（ Alt+1 ）。

安装宝石

构建好gem之后，我们可以将其安装到本地SDK进行测试：

  • 按Ctrl两次，然后输入以下命令：

    宝石安装hello_rubymine

    按Enter安装宝石。

使用IRB测试宝石

在本章中，我们将使用IRB控制台来测试我们的gem。 greet方法：

  1. 在主菜单中，转到“ 工具” |“工具”。运行IRB控制台 。

  2. 类型require 'hello_rubymine'然后按Enter将我们的宝石加载到IRB。

  3. 然后，输入HelloRubymine.greet("JetBrains")在控制台中，然后按Enter再次确保我们的宝石按预期工作。

发布宝石

最后，我们将我们的gem发布到RubyGems.org 。请注意，您需要在RubyGems.org上创建一个帐户。

  1. 在主菜单中，选择工具|宝石|推宝石 。

  2. 在“ 运行工具”窗口中，指定您的RubyGems凭据。

    您的宝石将发布到RubyGems.org 。
