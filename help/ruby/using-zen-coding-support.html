Emmet支持

Emmet和Emmet 1.1版是工具包，可让您更快地开发HTML，CSS或JSX代码。您可以使用Emmet代码模板而无需离开RubyMine。要将模板扩展为标记，只需键入其缩写并按扩展键或快捷方式即可。默认情况下，缩写用Tab ，但是您可以按照配置缩写扩展键中所述选择另一个键 。

RubyMine支持以下功能，例如用于编写RGBA颜色的新语法，隐含，默认和布尔属性，“ 更新标签”操作等。

启用和配置Emmet

借助RubyMine，您可以使用本机Emmet模板以及200多个其他HTML，CSS和XSL实时模板，这些模板在下面的Zen CSS ， Zen HTML和Zen XSL节点下列出。 编辑器| RubyMine设置的“ 实时模板”页面Ctrl+Alt+S 。

在RubyMine中启用和配置本机Emmet

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器| Emmet

  2. 在打开的Emmet页面上，默认选择键以扩展Emmet的缩写Tab被选中。

  3. 要在各种语言环境中启用和配置Emmet，请使用Emmet页面（ Editor | Emmet ）上的相应设置。

RubyMine允许您使用和自定义Emmet实时模板，也可以添加自定义模板。假设您有一个包含以下模板文本的模板条目 ：

在Emmet中使用实时模板

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|实时模板 。

  2. 在Emmet页面上，选中要使用的模板旁边的复选框。

  3. 当您在列表中选择一个模板时，焦点将移至“ 模板文本”区域，该区域中的字段显示所选模板的设置。

  4. 在“ 模板文本”字段中，将所需的文本和变量添加到模板主体中。

  5. 单击编辑变量按钮。在打开的“ 编辑模板变量”对话框中，在“ 默认值”字段中指定默认变量值，并在必要时选中“ 如果定义则跳过”复选框。

配置缩写扩展键

默认情况下，按以下键可扩展Emmet本机缩写和其他实时模板Tab 。对于其他实时模板，RubyMine可让您重新定义默认的扩展密钥。请注意，此自定义设置不会覆盖本机Emmet支持的默认设置。您将可以使用这两个键之一扩展模板。

为本机Emmet缩写配置扩展密钥

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器| Emmet 。

  2. 在打开的Emmet页面上，选择新的扩展密钥，而不是默认的Tab从扩展缩写为列表中。

为Emmet实时模板配置扩展密钥

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|实时模板 。

  2. 在“实时模板”页面上，展开Zen HTML ， Zen CSS或Zen XSL模板组，然后选择所需的模板。焦点移到“ 模板文本”区域。

  3. 从“ 展开方式”列表中，选择用于扩展模板的键。

用Emmet模板围绕代码块

  1. 在编辑器中，选择要包围的代码块，然后按Ctrl+Alt+J或选择代码|环绕|主菜单中的实时模板 。

  2. 从“ 选择模板”列表中，选择Emmet ：

  3. 键入要使用的Emmet缩写，然后按Enter 。

在编辑点之间导航

在HTML和XML中，您可以在编辑点之间，即适用Emmet模板的那些代码点之间导航。

  • 要将光标移到上一个编辑点，请选择“ 导航” |“导航”。上一个Emmet编辑点 ，或按Shift+Alt+[ 。

  • 要将光标移到下一个编辑点，请选择“ 导航” |“导航”。下一个Emmet编辑点 ，或按Shift+Alt+] 。
