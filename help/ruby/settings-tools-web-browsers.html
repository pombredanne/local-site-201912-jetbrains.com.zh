在此页面上，您可以配置以下内容：

  • 将Web浏览器的安装与RubyMine集成在一起，从RubyMine激活或停用启动的Web浏览器。

  • 指定是通过运行其可执行文件还是通过默认系统命令来启动浏览器。

  • 指定默认的RubyMine浏览器 ，默认情况下，即没有明确指定浏览器时，RubyMine将在其中请求打开HTML文件

有关更多信息，请参见Web浏览器 。

浏览器

在本部分中，指定哪些浏览器可用于预览HTML输出。这部分显示预定义列表中的浏览器以及先前配置的自定义浏览器安装（如果有）。

RubyMine随附了最流行的浏览器的预定义列表，您可能希望在运行，调试或预览HTML文件的输出期间从IDE自动安装和启动这些浏览器。RubyMine假定您按照标准过程安装浏览器，并为每个安装分配一个别名 ，该别名代表浏览器的可执行文件或macOS应用程序的默认路径。如果在实际的浏览器安装中，可执行文件的路径不同，则需要在“ 路径”字段中显式指定它。

除了预定义的浏览器之外，您还可以使用工具栏上的控件配置所需数量的自定义浏览器。

默认浏览器

从列表中选择默认浏览器。可能的选项是：

  • 系统默认值 ：选择此选项以接受操作系统默认的Web浏览器作为RubyMine的默认值。

  • 首次列出 ：选择此选项可使RubyMine启动列表中的第一个浏览器。使用更改订单或浏览器和工具栏上的图标。

  • 自定义路径 ：选择此选项可将另一个Web浏览器指定为RubyMine的默认浏览器。输入浏览器可执行文件的路径或单击然后在打开的对话框中选择路径。

在编辑器中显示浏览器弹出窗口

  • 如果选中此复选框，则具有启用的浏览器的弹出窗口将显示在HTML文件中。

  • 如果未选中此复选框，则不会显示此弹出窗口，从而帮助您阅读或编辑代码。
