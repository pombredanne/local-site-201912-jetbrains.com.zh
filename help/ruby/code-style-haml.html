使用此部分为Haml文件配置代码样式选项。更改这些设置时，更改结果将显示在“ 预览”窗格中。

制表符和缩进

使用制表符

  • 如果选中该复选框，则制表符将用于缩进和代码重新格式化。

  • 如果清除该复选框，则RubyMine使用空格而不是制表符。

智能标签

压痕由两部分组成。一部分来自嵌套代码块，另一部分由对齐确定。

  • 如果选中此复选框，则嵌套产生的零件将同时包含制表符和空格（如果需要），而由对齐方式定义的零件仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组适合指定制表符大小的空格后，将自动用制表符替换，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中可以容纳的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组的元素之间插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

缩进空行

如果选中此复选框，则RubyMine会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，则RubyMine会删除制表符和空白行。

设置为

如果适用，该链接将显示在页面的右上角。单击此链接，然后选择要用作当前语言代码样式基础的语言。

要返回最初的代码样式设置集并放弃所做的更改，请单击“ 重置” 。
