结构工具窗口，文件结构弹出窗口

结构工具窗口

此工具窗口显示当前在编辑器中打开并具有焦点的文件的结构，或在“项目”工具窗口中选择的文件的结构。

对于图，此工具窗口显示图预览。

查看|工具窗口|结构体
Alt+7

文件结构弹出

此弹出窗口显示文件的结构，当前在编辑器中打开该文件并具有焦点。

导航|档案结构
Ctrl+F12

两种视图均有助于快速浏览文件结构 。

本节介绍工具窗口标题栏上的按钮以及标题栏的上下文菜单选项。打开和关闭这些选项，可以隐藏或显示某些类型的元素，并配置它们的显示方式。

标题栏上的按钮在所有语言环境中都是通用的。

  • 红宝石
  • 规范
  • 黄瓜功能文件，ERB，HAML
  • HTML，XML
  • JavaScript，TypeScript，CoffeeScript

有关“ 结构”工具窗口中文件树视图中的成员图标的信息，请参阅“ 图标参考 ”部分。

标题栏

标题栏上的按钮在所有语言环境中都是通用的。

项目

工具提示

描述

展开全部Ctrl+NumPad Plus

单击此按钮以展开工具窗口中的所有节点。

全部收缩Ctrl+NumPad -

单击此按钮可折叠工具窗口中的所有节点。

单击此按钮可以打开上下文菜单，并通过打开或关闭菜单项来配置工具窗口的外观，其查看模式以及呈现当前文件结构的方式。

Shift+Escape

使用此图标或快捷方式隐藏工具窗口。

当与Alt键，单击此图标将隐藏附加到同一工具窗口栏的所有工具窗口 。

对于视图文件.html.erb ， .html.haml ，“结构”工具窗口显示对应于组成语言（ Ruby ， HTML ， ERB，HAML ）的多个选项卡。
