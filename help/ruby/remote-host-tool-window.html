远程主机工具窗口

工具|部署|浏览远程主机

使用此工具窗口可以查看目标FTP / FTPS / SFTP服务器的文件夹结构以及上传到它们的数据。

访问远程主机工具窗口

  • 查看|工具窗口|远程主机 -使用“ 工具” | “ 工具” |“打开”后，可以通过这种方式访问工具窗口。部署|浏览远程主机命令。

  • 仅当启用了FTP / SFTP连接捆绑插件时，该工具窗口才可用。该插件默认情况下处于活动状态。如果没有，请在“ 设置”对话框的“ 插件”页面中将其激活。

上下文菜单

项目

描述

在这里上传

选择此选项可使当前打开的项目中的文件根据所选配置上载。如果从文件的上下文菜单中调用该操作，则会上传相应的本地文件。如果从文件夹调用该操作，则会上载整个文件夹。

从这里下载

选择此选项可将所选文件或文件夹下载到当前打开的文件或文件夹。现有的本地文件将被更新，丢失的文件将被创建。

新

选择此选项可在所选文件夹中创建一个新的远程文件或文件夹。该选项仅在文件夹的上下文菜单中可用。

改名

选择此选项可以重命名选定的文件或文件夹。

删除

选择此选项以删除选定的文件或文件夹。

切

选择此选项可将所选文件或文件夹的名称复制到剪贴板，并从树中删除文件或文件夹。

复制

选择此选项可将所选文件或文件夹的名称复制到剪贴板。

糊

选择此选项可将先前复制的文件或文件夹名称插入树中。

编辑本地文件

当选择文件并选择此选项时，RubyMine在编辑器中打开其本地副本，并将焦点移至相应的位置。仅当当前在编辑器中打开具有相应本地文件的项目时，此选项才可用。

与本地同步

选择此选项可将所选的远程文件夹与其本地版本进行比较。 在打开的文件夹差异查看器中 ，浏览差异并同步文件（如果适用）。请参阅在差异查看器中比较两个文件夹 。

有关详细信息，请参见将已部署的文件和文件夹与其本地版本进行比较。

与本地版本比较

选择此选项可将所选的远程文件与其本地版本进行比较。 在打开的“ 差异查看器文件”对话框中，浏览差异并根据需要应用差异。 按钮。有关详细信息，请参见查看文件之间的差异 。

有关详细信息，请参见将已部署的文件和文件夹与其本地版本进行比较。

编辑远程文件

选择此选项可在RubyMine编辑器中编辑所选文件，而不将其添加到当前打开的项目中。

有关详细信息，请参见在远程主机上编辑单个文件 。

复制路径

排除路径

选择此选项可从上传/下载中排除所选文件夹。

有关详细信息，请参见从上载/下载中排除文件和文件夹 。

复制路径

选择此选项可将服务器上所选文件或文件夹的绝对路径复制到剪贴板。
