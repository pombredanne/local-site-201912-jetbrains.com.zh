创建并运行您的第一个Ruby项目

本教程将向您展示如何在RubyMine中创建和运行简单的交互式Ruby程序。

在开始本教程之前，请执行以下操作：

  • 下载并为您的平台安装Ruby发行版 。

  • 安装并设置 RubyMine。

我们将使用安装在macOS上的RubyMine执行所有步骤。

下载完成的项目： https : //github.com/rubyminedoc/ruby_helloworld

创建一个空的应用程序

要从头开始创建Ruby程序，请执行以下操作：

  1. 运行RubyMine并在“ 欢迎屏幕 ”上单击“ 创建新项目 ” 。

  2. 在“ 新建项目”对话框中，确保在左侧窗格中选择了“ 空项目” 。然后，指定以下设置：

    • 位置 ：指定项目的位置和名称（在本例中为ruby-helloworld ）。

    • Ruby SDK ：选择系统上安装的必需Ruby解释器。

    指定所有选项后，在“ 新建项目”对话框中单击“ 创建 ”。

创建一个Ruby文件

创建项目后，您将在左侧的“项目”工具窗口中看到其结构。要添加Ruby文件，请执行以下操作：

  1. 在“项目”工具窗口中选择一个项目根目录，然后按Alt+Insert 。

  2. 在调用的弹出窗口中，选择“ Ruby File / Class” ，然后按Enter 。

  3. 在调用的弹出窗口中，指定脚本的名称（在本例中的脚本 ），然后单击确定 。

  4. 在编辑器中，插入以下代码：

    输入“请输入您的姓名” name = gets.chomp输入“您好，＃{name}！我是露比”

运行一个应用程序

要运行创建的脚本，请执行以下操作：

  1. 按Ctrl两次并键入以下命令：

    红宝石script.rb

    按Enter 。

  2. 在“运行工具”窗口中，键入任何名称，然后按Enter查看程序的响应。

运行/调试配置

运行Ruby脚本后，RubyMine自动创建一个特殊的配置文件- 运行/调试配置 。这样的配置文件用于运行，调试和测试应用程序，并指定脚本名称，工作目录，启动前要执行的操作等。

要编辑配置，请转到编辑器顶部的下拉菜单，然后点击编辑配置 。

在调用的“运行/调试配置”对话框中，您可以指定启动脚本所需的设置。

有关更多详细信息，请参见使用运行/调试配置 。
