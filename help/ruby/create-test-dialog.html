创建测试对话框

导航|测试-创建新测试

Ctrl+Shift+T

如果尚不存在此类测试，请使用此对话框为类或方法生成测试类。

项目

描述

目标目录

点击浏览按钮打开目录选择器对话框，然后选择目标目录，将在其中生成所需的测试文件。

测试文件名

在此文本字段中，指定将在其中生成测试类的文件的名称，或接受默认名称。

测试类别名称

在此文本字段中，指定测试类的名称，或接受默认名称。

测试方法

此列表显示了所讨论类的所有方法（如果对话框是从类声明中调用的）或唯一方法的名称（如果对话框是从该方法内部调用的）。

选中方法名称旁边的复选框，以将这些测试方法包括在生成的测试类中。
