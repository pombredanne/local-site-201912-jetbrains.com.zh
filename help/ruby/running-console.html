IRB控制台

RubyMine允许您运行IRB控制台并与您的应用程序进行交互，而无需离开IDE。您可以使用完成和导航功能，使用其他选项运行控制台，等等。

在使用IRB控制台之前，请确保为您的项目配置了Ruby解释器。

运行IRB控制台

要在RubyMine中运行IRB控制台，请执行以下步骤：

  • 请执行以下任一操作：

    • 按Ctrl两次，开始输入IRB控制台 ，在弹出窗口中选择相应的项目，然后按Enter 。

    • 在主菜单中，转到“ 工具” |“工具”。运行IRB控制台 。

RubyMine将使用选定的项目SDK加载IRB控制台。

请注意，可以在远程SDK上启动IRB控制台。

使用其他选项运行控制台

首次运行控制台后，RubyMine创建专用的运行配置 。如有必要，您可以更改创建的配置以提供其他选项：将代码预加载到IRB中，添加其他库或gem，等等。例如，要将代码从script.rb项目文件加载到IRB控制台，请执行以下步骤：

  1. 通过以下方式之一打开“ 运行/调试配置”对话框：

    • 按Ctrl+Shift+A ，开始输入Edit Configurations ，从列表中选择它，然后按Enter 。

    • 展开主工具栏上的运行/调试配置选择器，然后选择编辑配置 。

  2. 在调用的“运行/调试配置”对话框中，在“ IRB控制台”组中选择创建的IRB控制台配置，并通过以下方式指定“ IRB参数”选项：

    -r ./脚本

    单击确定 。

从控制台的编辑器运行源代码

RubyMine允许您从IRB控制台的编辑器中运行源代码。为此，请执行以下步骤：

  1. 在编辑器中打开所需的Ruby文件（如有必要，请选择要执行的代码片段）。

  2. 在主菜单中，选择“ 工具” |“工具”。将文件/选择加载到IRB / Rails控制台中 ，或按Shift+Alt+L 。

使用IRB控制台

您可以在控制台中执行以下操作：

  • 执行命令

    在控制台中键入命令，然后按Enter执行它们。

  • 完整名称

    使用基本代码完成 Ctrl+Space完成类，方法等的名称。

  • 浏览历史

    使用Up和Down箭头键可滚动浏览命令的历史记录并执行所需的命令。

  • 导航到源代码

    要导航到符号和符号类型的初始声明，请将插入标记放在所需的符号上，然后按Ctrl+B 。

  • 查看快速文档

    要获得任何符号或方法签名的快速信息 ，请将插入符号放在所需的符号上，然后按Ctrl+Q 。
