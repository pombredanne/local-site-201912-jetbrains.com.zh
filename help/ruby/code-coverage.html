RubyMine 2019.3帮助

代码覆盖率

使用RubyMine，您可以使用SimpleCov分析工具来衡量测试覆盖了多少代码。您可以针对任何受支持的测试框架进行覆盖测试 ，在单独的工具窗口和编辑器中分析覆盖文件和行的百分比，生成HTML报告，等等。

先决条件

在RubyMine的使用代码覆盖之前，请确保simplecov宝石加到Gemfile中并安装到项目的SDK。从“ 使用捆绑程序安装宝石”主题中学习如何使用RubyMine 安装宝石 。

  • 执行测试后自动

  • 手动地

进行覆盖测试

RubyMine允许您通过以下方式运行具有覆盖率的测试：

使用预定义的配置在覆盖范围内运行测试

要使用已经创建的运行配置在覆盖范围内运行测试，请执行以下一项操作：

  • 按Shift+Alt+F10 ，选择所需的配置，按右箭头键，选择Cover ，然后按Enter 。

  • 要从主工具栏运行具有覆盖率的测试：

    1. 从主工具栏上的列表中选择所需的运行配置。

    2. 点击右边的按钮。

在“项目”视图中进行覆盖测试

要在“ 项目”视图中运行一个或多个涵盖范围的测试，请执行以下操作：

  1. 右键单击特定的测试文件或包含所需测试（例如test ， spec或features ）的目录。

  2. 从上下文菜单中，选择相应的运行命令，例如， 使用Coverage运行“所有测试： ” 。

在编辑器的覆盖范围内运行测试

要从编辑器运行测试，请执行以下步骤：

  1. 单击所需测试或测试类旁边的左侧装订线中的“运行”按钮。

  2. 从上下文菜单中，选择所需的运行命令，例如Run''with Coverage 。

RubyMine将运行测试并在“ 运行”工具窗口中显示结果。Coverage结果将显示在多个位置： Coverage工具窗口，编辑器和Project视图。在查看覆盖率结果中了解更多信息。

如果您不是第一次进行覆盖测试，RubyMine将在分析后为您提供以下选择：

  • 不要应用收集的覆盖率 ：RubyMine保存结果，但不显示结果。要了解如何显示保存的结果，请参阅管理覆盖率结果 。

  • 添加到活动套件中 ：RubyMine将新的覆盖结果与现有覆盖结果合并并显示它们。

  • 替换活动套件 ：RubyMine用新的套件替换现有的覆盖结果并显示它们。

您将来可以禁用此对话框，并在必要时更改“ 设置/首选项”中的默认行为（ Ctrl+Alt+S ），在构建，执行，部署|覆盖页面。

查看覆盖范围结果

在进行覆盖测试之后，RubyMine通过以下方式显示结果：

  • 在项目视图中。

    在这里，您可以浏览项目并检查涵盖文件和行的百分比。

  • 在“ 覆盖率”工具窗口中。

    您可以启用选项以在编辑器中自动打开所选文件。的按钮可让您根据覆盖率结果生成HTML报告 。

  • 在编辑器中。

    使用装订线中的颜色指示器检测未覆盖的代码行。

    要找出行被击打了多少次，请单击相应的颜色指示器。

    在打开的弹出窗口中，您可以单击通过单击“ 隐藏coverage”链接配置代码coverage颜色或隐藏coverage结果。要了解如何显示隐藏的覆盖范围结果，请参阅管理覆盖范围结果 。

从“ 创建测试”主题中学习如何创建缺失的测试。

管理覆盖范围结果

在使用coverage运行测试之后，RubyMine将结果保存在coverage套件中。Coverage Suite允许您显示或隐藏特定的Coverage结果，合并多个报告以及删除不必要的结果。

查看特定套件的覆盖范围结果

  1. 在主菜单中，选择工具|显示代码覆盖率数据 Ctrl+Alt+F6 。

  2. 在“ 选择要显示的覆盖套件”对话框中，选择或取消选择所需的套件，然后单击“ 显示所选内容” 。

    RubyMine 打开所选测试套件的覆盖率结果。

从文件添加coverage套件

考虑从另一台计算机获取包含代码覆盖率信息的文件的情况。您可以从磁盘加载该文件并查看覆盖率结果。

  1. 在主菜单中，选择工具|显示覆盖率数据 Ctrl+Alt+F6 。

  2. 请点击然后在打开的对话框中选择所需的.rcov文件。

生成HTML报告

RubyMine允许您为收集的覆盖率结果生成HTML报告。

为此，请按照以下步骤操作：

  1. 在主菜单中，选择工具|生成覆盖率报告 ，或单击在“ 覆盖率”工具窗口中。

  2. 在“ 生成覆盖率报告”对话框中，指定将存储生成的报告的目录。（可选）选择在浏览器中打开生成的HTML 。点击保存 。
