FAQ

How to get RubyMine free or at a discount?

If you want to get a RubyMine license free or at a discount, check out the offers on the following page: Toolbox Subscription - Special Offers. If you have any questions, contact our sales support.

Is RubyMine available via the JetBrains Toolbox as part of the 'All Products Pack'?

Yes, it is available both as a standalone IDE and as part of the 'All Products Pack'.

What is the difference between RubyMine and the Ruby plugin for IntelliJ IDEA?

RubyMine and the Ruby plugin for IntelliJ IDEA have the same functionality. However, because of the differences in release cycles, the latest features and bug fixes become available in the Ruby plugin a little later than they do in RubyMine.

In addition to Ruby/Rails, which web technologies does RubyMine support?

RubyMine supports code insight for the most popular languages used in web development, including JavaScript, TypeScript, and CoffeeScript, stylesheet languages, and so on. Moreover, you can work with web frameworks such as Angular, React, and Vue.js by installing the corresponding plugins.

Does RubyMine include support for databases?

Yes, RubyMine allows you to connect to databases, navigate to any object, and execute queries.

How can I work with RubyMine offline?

A lot of features in RubyMine require access to the Internet. Check out our Working offline guide for detailed information about working offline, for example, in an isolated environment.

Does RubyMine support RuboCop?

Yes, RubyMine supports integration with RuboCop/Standard and enables you to fix its offenses right inside the IDE. Moreover, RubyMine respects all settings from .rubocop.yml. You can learn more from the RuboCop topic.
