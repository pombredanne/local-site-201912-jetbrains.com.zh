Sharing configurations

RubyMine allows you to share your configurations with team members through VCS. To do this, perform the following steps:

  1. Open the Run/Debug Configuration dialog:

    • Select Run | Edit Configurations from the main menu.

    • With the Navigation bar visible (View | Appearance | Navigation Bar), choose Edit Configurations from the run/debug configuration selector.

    • Press Shift+Alt+F10, then press 0 or select the configuration from the popup and press F4.

  2. In the Run/Debug Configuration dialog, select the required configuration and enable the Share through VCS option. RubyMine will create a file containing configuration settings in the .idea/runConfigurations directory.

  3. (Optional) If the .idea directory is added to VCS ignored files, click Remove from ignored files in the invoked popup. If you use Git for your project, RubyMine will make the entire .idea directory visible. To make .idea ignored again and share .idea/runConfigurations only, change the .gitignore file as follows:

    /.idea/* !/.idea/runConfigurations
