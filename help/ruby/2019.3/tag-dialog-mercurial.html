Tag Dialog (Mercurial)

Use this dialog to create a global tag that identifies the tip of a repository, which is the most recently changed heade in this repository. The created tag will be stored in the file .hgtags and tracked by Mercurial.

Item

Description

Select repository to tag

From this list, choose the repository whose tip you want to tag. The list shows the location of the repositories under the project root.

Tag name

In this field, type the name of the tag. By this name, you will be able to find the tag in the Log tab of the Version Control window Alt+9.
