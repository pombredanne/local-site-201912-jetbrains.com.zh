RubyMine 2019.3 Help

Create New RVM Gemset

File | Settings | Ruby SDK and Gems - for Linux
RubyMine | Preferences | Ruby SDK and Gems - for macOS

RubyMine allows you to create an RVM gemset in the Setting/Preferences dialog. Refer to the section Creating RVM gemset in the Setting/Preferences dialog.

Controls

Item

Description

Ruby SDK

In this field, specify the desired Ruby SDK. If an RVM SDK has been selected for your project, then it is used. If an SDK or a non-RVM SDK has been selected (for example, system, chef or remote), then the default SDK is used.

RVM Gemset

Specify the desired gemset name.
