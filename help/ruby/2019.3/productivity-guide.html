Help | Productivity Guide

The Productivity Guide shows usage statistics for RubyMine features.

The table in the upper part lists the features. Select a feature to see its description in the lower part. To sort the information, click a cell in the header row.

Item

Description

Feature

The name of the feature.

Group

The group to which the feature belongs.

Used

How many times you used the feature.

Last used

When the feature was last used.
