Working with RubyMine Features from Command Line

Besides working from within RubyMine, you can perform the following actions from the command line:

  • Opening Files from Command Line
  • Formatting Files from Command Line
  • Launching Code Inspection from Command Line
  • Viewing Difference Between Files from Command Line
  • Merging Files from Command Line

Enable a command-line launcher on Windows

  • Add the path to the RubyMine bin folder to the Path environment variable (for example, C:\Program Files\JetBrains\RubyMine\bin). After you configured the Path environment variable, you can run the RubyMine commands.

Enable a command-line launcher on Linux and macOS

On macOS and Linux, create the command-line launcher to integrate RubyMine with your shell. Ensure that the created launcher script is within the search path of your shell so that it can be invoked system-wide.

  1. In the main menu, choose Tools | Create Command-line Launcher. In the Create Launcher Script dialog, provide the name and the path of the launcher, or accept the suggested values.

  2. Outside of RubyMine, add the path and the name of the launcher script to your path.

On Linux, installing RubyMine as a snap package automatically creates the command-line launcher named rubymine. The Tools | Create Command-line Launcher command is therefore not available.

To launch RubyMine from the command line

  • To launch RubyMine from the command line, type the <RubyMine> command in the command line and press Enter. <RubyMine> is the platform-specific launcher.
