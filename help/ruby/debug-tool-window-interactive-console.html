调试工具窗口。互动式控制台

此选项卡已标记 。它使您能够使用交互式命令行控制台。
请注意，当控制台繁忙时，该选项卡可用于调试测试，Rake任务等。当调试任何其他应用程序时，它内置在控制台中 。

项目

工具提示和快捷方式

描述

上/下堆栈跟踪
Ctrl+Alt+Up Ctrl+Alt+Down

单击此按钮可在堆栈跟踪中向上或向下导航，并使光标跳至源代码中的相应位置。

使用软包装

单击此按钮可切换输出的自动换行模式。

滚动到最后

单击此按钮导航到堆栈跟踪的底部，并使光标跳至源代码中的相应位置。

打印

单击此按钮可将控制台文本发送到默认打印机。

全部清除

单击此按钮可从控制台中删除所有文本。也可以从控制台的上下文菜单中使用此功能。

浏览记录
Ctrl+Alt+E

按下此按钮以显示“ 调试控制台历史记录”对话框，在该对话框中，可以查看控制台条目并使用箭头键浏览它们，然后通过按来将所选项目添加到控制台中。 Enter 。

单击“ 确定”关闭对话框。
