Gulpfile.js的上下文菜单-显示Gulp任务
查看|工具窗口|古尔普

使用此工具窗口运行Gulp.js任务在“ 运行”工具窗口中显示执行任务的结果。 工具窗口显示Gulp.js输出，报告发生的错误，列出未找到的软件包或插件，等等。最后执行的任务的名称显示在工具窗口的标题栏上。

在你开始前

  1. 下载并安装Node.js。

  2. 全局安装gulp-cli软件包（Gulp命令行界面），并将gulp软件包安装为开发依赖项。有关详细信息，请参见安装Gulp.js。

运行任务

打开Gulp工具窗口

  • 在“ 项目”工具窗口中选择所需的Gulpfile.js文件，或在编辑器中将其打开，然后从上下文菜单中选择“ 显示Gulp任务 ”。

当您在当前RubyMine会话中首次构建任务树时， Gulp工具窗口尚未打开。

默认情况下，RubyMine无法在Gulpfile.js中识别ES6，并且无法构建任务树。要解决此问题，请更新默认的Gulp.js运行配置 。

从ES6 Gulpfile.js构建任务树

  1. 从主菜单中，选择运行|编辑配置 。

  2. 在“ 模板”节点下，单击Gulp.js。

  3. 在打开的“运行/调试配置：Gulp.js”对话框中，键入--harmony在“ 节点选项”字段中，然后单击“ 确定” 。

建立任务树

  • 在“ Gulp工具”窗口中，单击在工具栏上，然后从列表中选择所需的Gulpfile.js文件。默认情况下，RubyMine在项目的根目录中显示Gulpfile.js文件。

  • 如果还有另一个Gulpfile.js文件，请单击“选择Gulpfile.js” ，然后在打开的对话框中选择所需的Gulpfile.js文件。RubyMine在其标题上添加一个具有所选Gulpfile.js文件路径的新节点，并在新节点下构建任务树。

重建一棵树

  • 切换到所需的节点，然后单击在工具栏上。

按名称对树中的任务进行排序

  • 请点击在工具栏上，从菜单中选择“ 排序依据 ”，然后选择“ 名称” 。
    默认情况下，一棵树以在Gulpfile.js中定义的顺序显示这些任务（选项Definition order ）。

运行任务

  • 双击任务。

  • 在树中选择任务，然后按Enter或选择运行从上下文菜单中。

运行默认任务

  • 在树中选择根节点，然后从所选内容的上下文菜单中选择“运行默认值 ”。

运行几个任务

  • 使用多选模式：按住Shift （用于相邻项目）或Ctrl （对于非相邻项目）键并选择所需的任务，然后从所选内容的上下文菜单中选择“运行”或“ 调试 ”。

导航到任务的定义

  • 在树中选择所需的任务，然后从所选内容的上下文菜单中选择“ 跳转到源 ”。

工具列

添加Gulpfile

单击此按钮可以为另一个Gulpfile.js文件构建任务树。从列表中选择所需的Gulpfile.js文件。RubyMine构建一个任务树并将其显示在单独的节点下。

删除Gulpfile

单击此按钮以删除所选节点下的任务树。

重新加载任务

单击此按钮可以重新构建所选节点下的任务树。更新相应的Gulpfile.js文件后，您可能需要重新构建树，因为Gulp.js不会动态将更改应用于树。

全部收缩

单击此按钮可隐藏所有任务树，并且仅显示Gulpfile.js节点。

单击此按钮可配置当前视图并更改工具窗口的查看模式 ，有关详细信息，请参见工具窗口查看模式 。请注意，大多数菜单项都是可以打开或关闭的选项。启用的选项的名称左侧带有一个复选标记。Gulp特定的选项是：

  • Gulp设置：选择此菜单项以打开Gulp设置对话框，然后重新配置Gulp和Node解释器的当前设置，请参阅任务运行器 。

  • 排序方式：选择此菜单项可配置任务在树中的显示顺序。 请点击在工具栏上，从菜单中选择“ 排序依据 ”，然后选择“ 名称” 。
    默认情况下，一棵树以在Gulpfile.js中定义的顺序显示这些任务（选项Definition order ）。

隐藏

单击此按钮可以隐藏工具窗口。要使其再次显示，请选择“ 视图”工具窗口|在主菜单上吞食 。再次出现工具窗口，显示所有先前构建的任务树。

一棵树的上下文菜单

口吃设置

选择此菜单项以打开“ Gulp设置”对话框并查看或编辑Node.js配置

跳转到源

选择此菜单项以打开为其构建当前树的Gulpfile.js文件。

重新加载任务

选择此菜单项可重新构建选定节点下的任务树。

复制路径

选择此菜单项将路径保存到Gulpfile.js文件，根据该路径将当前树构建到剪贴板。

删除Gulpfile.js

选择此菜单项可删除所选节点下的任务树。

任务的上下文菜单

跑

选择此菜单项以运行所选任务。

除错

选择此菜单项以调试所选任务。

编辑设定

选择此菜单项以打开“ 运行/调试配置”对话框，并编辑所选任务的预定义设置。

跳转到源

选择此菜单项以打开为其构建当前树的Gulpfile.js文件，并导航到所选任务的定义。
