常用选项

项目

描述

将应用程序重点放在断点上

如果选中此复选框，则在遇到断点时，RubyMine将在编辑器中显示此断点的位置，并尝试将其框架置于最前面。

在断点上显示调试窗口

如果选中此复选框，RubyMine会在遇到断点时激活“ 调试工具”窗口 。

在进程终止时隐藏调试窗口

当调试程序终止时，自动隐藏“ 调试”窗口 。

将执行点滚动到中心

如果选中此复选框，则具有当前执行点的行将保留在屏幕中间。

单击行号以执行游标

如果选中此复选框，则可以在编辑器中单击行号以在该行中运行程序执行 。

删除断点

选择您要如何删除断点：

  • 通过用鼠标左键单击它们

  • 通过将它们拖动到编辑器或用鼠标中键单击它们。在这种情况下，单击断点将在启用和禁用之间切换其状态。

您还可以选择是否要在删除条件断点或记录断点时显示确认对话框。

调试时自动附加到子流程

读取非暂停线程的框架（Ruby 2.0不支持）

选择此复选框将能够查看当前断点事件未暂停的线程的帧。

忽略非项目来源

选中此复选框可自动跳过外部库代码，并将重点放在您自己的代码上。
无论使用什么步进命令，RubyMine都不会停止任何非项目代码。此选项还可以防止调试时在编辑器选项卡中打开外部文件。

此选项仅适用于Ruby2.x 。 Ruby 1.8和1.9不支持它。

内置服务器

项目

描述

港口

使用此旋转框可以指定运行内置Web服务器的端口。默认情况下，此端口设置为端口63342 ，RubyMine通过该端口接受服务的连接。您可以将端口号设置为以1024或更高版本开头的任何其他值。

可以接受外部连接

如果选中此复选框，则可以从另一台计算机访问在指定端口上运行的内置服务器上的文件。
如果清除此复选框（默认情况下），则调试器仅侦听本地连接。

允许未签名的请求

 出于安全原因，默认情况下，从RubyMine外部对内置服务器上的页面的任何请求均被拒绝，并显示以下授权弹出窗口：要访问请求的页面，请单击“ 将授权URL复制到剪贴板” ，然后将生成的令牌粘贴到浏览器的地址栏中。

但是，此行为可能很烦人，例如，如果不可能进行手动干预，则可能会阻止调试会话。要取消显示授权弹出窗口，请选中“ 允许未签名的请求”复选框。
