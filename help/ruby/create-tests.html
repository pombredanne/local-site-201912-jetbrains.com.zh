RubyMine 2019.3帮助

创建测试

RubyMine提供了以各种方式创建缺失测试的功能，例如，您可以：

  • 从模板创建测试

  • 使用Rails生成器生成测试

  • 从测试主题导航到测试时创建测试

从模板创建测试

要从模板创建新测试，请执行以下操作：

  1. 请执行以下任一操作：

    • 在项目工具窗口中Alt+1 ，选择要在其中创建新文件的目录，然后选择“ 文件” |“文件”。主菜单上的新功能。

    • 右键单击相应的目录，然后从上下文菜单中选择“ 新建 ”。

    • 按Alt+Insert 。

  2. 从列表中选择所需的测试模板，例如RSpec Template 。

  3. 在打开的对话框中，指定所需的名称。例如，对于RSpec模板，您需要指定文件和行为名称。单击确定 。

  4. RubyMine将创建一个具有相应初始内容的测试文件，然后在编辑器中将其打开。您可以在此处添加所需的代码。

    如有必要，您可以在“设置/首选项”中修改预定义的模板Ctrl+Alt+S在编辑器上|文件和代码模板页面。

生成Rails应用程序的测试

RubyMine提供了几种生成Rails元素（控制器，模型等）缺失测试的方法。

导航到它时创建一个测试

  1. 在编辑器中，打开所需的Rails元素，例如控制器或模型。

  2. 从主菜单中，选择导航|测试或按Ctrl+Shift+T 。

  3. 如果不存在相应的测试，则会显示“ 生成测试”弹出窗口。选择所需的测试框架并按Enter 。

  4. （可选）如果项目的SDK中缺少相关的测试框架，RubyMine将建议安装缺少的gem（例如，“ rspec-rails”）。

    单击捆绑它 ，然后在打开的捆绑安装对话框中单击安装 。

    IDE将安装缺少的gem，并在必要的目录下生成测试（例如， spec或test取决于所选的测试框架）。

与Rails元素一起创建测试

Rails允许您与其他应用程序元素（例如控制器，模型等）一起生成测试。通过创建Rails应用程序元素学习如何在RubyMine中创建Rails应用程序元素 。

请参阅配置生成器以了解如何配置所需的测试框架。

使用Rails生成器创建RSpec测试

RSpec 生成器可以独立运行以创建测试。您可以按照以下步骤进行操作：

  1. 在主菜单中，选择工具|运行Rails Generator （ Ctrl+Alt+G ）。

  2. 在调用的弹出窗口中，开始键入所需的生成器名称。例如， rspec:model下图中的将会用于创建模型测试。

  3. 指定生成器参数。例如，对于用户模型，我们通过user作为争论。单击确定 。
