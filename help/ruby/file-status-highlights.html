文件状态亮点

在RubyMine中，每个文件根据您所使用的版本控件中的状态用颜色标记。

在编辑器中，检查文件中的每一行的版本控制状态，并在左侧装订线区域中将其标记为颜色 。

您可以自定义默认颜色在里面 设置/首选项对话框Ctrl+Alt+S

  • 对于文件：在“ 版本控制”下|文件状态颜色 。

  • 对于编辑器中的行：在Editor |下。配色方案| VCS 。
