依存关系

该区域显示计算机上当前安装的所有与Bower相关的软件包的列表。

  • 软件包：此只读字段显示软件包的名称，与在命令行模式下安装软件包时应引用的名称完全相同。

  • 版本：此只读字段显示计算机上安装的软件包的版本。

  • 最新：此只读字段显示软件包的最新发行版本。如果软件包不是最新的，则标记为 。

  • 请点击安装新软件包。在打开的“ 可用软件包”对话框中，选择相关的软件包。准备好后，单击“ 安装软件包” 。

  • 请点击删除选定的软件包。

  • 请点击将所选软件包的当前版本替换为最新发行版本。仅当所选项目不是最新时才启用该按钮。
