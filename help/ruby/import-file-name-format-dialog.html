格式

选择文件格式，然后检查表预览。如果尚未达到预期效果，请调整设置。

如果您更改了设置并要保存更改，请单击并选择以下选项之一：

  • 保存更改：设置以“相同名称”保存，而不创建新格式。实际上，格式是一组命名的设置。

  • 另存为：设置以“其他名称”保存：创建新格式，您可以为该新格式指定名称。

值分隔符

选择或键入用于分隔各个值的字符。

行分隔符

选择或键入应被视为行分隔符的字符。

空值文字

如果单元格包含的文本将用作值null （未知值）。

添加行前缀/后缀

行前缀和后缀是字符序列，除了行分隔符外，它们还指示行的开头和结尾。

如有必要，请单击链接，然后在显示的字段中指定行前缀和后缀。

报价单

报价区域下的每一行都是一个报价模式（请参见报价值 ）。报价模式包括：

  • 左引号字符，在值前插入一个引号。

  • 右引号，在值后插入一个；通常，与左引号相同。

  • 引号字符是值的一部分时的转义方法或字符。例如转义符：重复表示如果值中包含引号字符，则将其加倍。（您可以指定自己的转义符。）

报价值

指定在哪种情况下应使用引号（即用引号引起来）。

  • 需要时：仅在包含值和/或行分隔符的情况下才引用该值。

  • 始终：在其文本表示形式中引用任何值。

修剪空白

如果未选中此复选框，则在值分隔符之前和之后的Unicode空格字符将被视为相应值的一部分。如果选中此复选框，则将忽略或删除相应的空格字符。

第一行是标题

如果选中此复选框，则将第一行视为包含列名。在“ 标题格式”下显示的设置与上述含义相同，但应用于第一行。

第一列是标题

如果选中此复选框，则将第一列视为包含行名。
