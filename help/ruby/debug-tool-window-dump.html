调试工具窗口。倾倒

当转储按钮时，此选项卡出现在“ 调试器”工具栏上单击，或选择“运行” |“运行”。在主菜单上获取线程转储 。

使用此选项卡可以方便地查看外部线程转储或从调试器获取的转储。该选项卡分为两个部分。左边的一个显示所有线程，右边的一个显示所选线程的堆栈跟踪。

在“ 转储”选项卡中，对所有线程进行了排序，以便最有意义和最有用的线程位于列表顶部。为方便起见，线程以不同的灰色文本显示，死锁以红色突出显示。

过滤
Ctrl+F

单击此按钮启用按堆栈跟踪中的单词过滤线程转储。搜索字段显示在线程列表中，您可以在其中键入搜索字符串。

请注意， 普通搜索 Ctrl+F在“转储”选项卡右侧的选定线程的堆栈跟踪中，也提供了该功能。

复制到剪贴板

单击此按钮将整个线程转储复制到剪贴板。

按名称对线程排序/按类型对线程排序

单击这些按钮可以在按字母顺序或按类型排序的线程之间切换。

导出到文本文件
Alt+O

单击此按钮可将当前线程导出到指定的文本文件。
