放置左括号

在该行的第一个关键字下对齐或缩进左括号。选择“ 在同一行上”以使关键字的开头括号保持在该行上。

放置元素

更改括号中元素的位置。

  • 对齐同一行 ：对齐子句的所有成员，使第一个成员与括号保持在同一行。

  • 不缩排包装 ：将子句的所有成员移至新行，而无需添加缩排。

  • 包装对齐 ：对齐并将子句的所有成员移至新行。

  • 包装缩进 ：添加缩进并将子句的所有成员移至新行。

放置右括号

更改右括号的位置。

  • 最后 ：将右括号与子句的最后一个元素放在同一行。

  • 首先 ：将结束括号移到带有子句最后一个元素的新行。

  • 在开孔下 ：将右括号放在开括号下。

  • 在元素下 ：将右括号放在子句的对齐元素下。

对齐类型

在子句中对齐类型（例如， serial ， char ， varchar ， 等等）。

对齐默认值

对齐默认约束。

对齐空值

对齐NULL和NOT NULL约束。

短时收合

连接多行短代码部分。将要折叠的部分的长度由RubyMine自动确定。

包装变更说明

将ALTER指令移至新行。

对齐变更指令

对齐所有ALTER指令。要使用此选项，请将“ 包装更改说明”选项设置为“ 请勿更改” 。
