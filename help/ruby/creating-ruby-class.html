创建Ruby文件，类和模块

RubyMine使您可以创建空的Ruby文件或生成类和模块。要创建新的Ruby文件，类或模块，请执行以下操作：

  1. 在项目工具窗口中，选择要在其中创建类的目录，然后按Alt+Insert 。

  2. 从弹出菜单中，选择Ruby File / Class 。

  3. 在“ 新建Ruby文件/类”弹出窗口中，执行下列操作之一：

    • 要创建一个新的Ruby文件，请输入其名称并确保已选择File 。

    • 要创建Ruby类，请切换到Class并指定类名称。

      请注意，Ruby类和模块的名称应以大写字母开头。

    按Enter 。
