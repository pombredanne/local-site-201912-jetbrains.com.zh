调试工具窗口。镜框

通过“ 框架”窗格，您可以访问应用程序的线程列表。

要检查线程，请从窗格顶部的列表中选择它。线程的状态和类型由一个特殊的图标和该线程名称旁边的文字注释指示。对于每个线程，您可以在编辑器中查看堆栈框架，检查框架，在框架之间导航以及自动跳转到框架的源代码。

要检查存储在框架中的值，请使用“调试”工具窗口的“ 变量”窗格 。

线程状态

Java提供了线程状态，以反映程序中线程当前正在发生的情况。

线程状态

描述

监控

线程正在Java监视器上等待。

没有开始

该线程尚未启动。

正在运行

线程处于活动状态并正在运行。

睡眠

线程正在休眠，因为Thread.sleep()要么JVM_Sleep()被称为。

未知

线程状态未知。

等待

线程正在等待Object.wait()要么JVM_MonitorWait()被称为。

松比

线程已完成执行。
