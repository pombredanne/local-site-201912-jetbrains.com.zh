教程：调试Rails运行器脚本

Rails运行器允许您非交互地在Rails上下文中运行Ruby代码。在本教程中，我们将向您展示如何在RubyMine中调试Rails运行器脚本。

例如，您的项目包含要在以下路径调试的Ruby脚本： lib / script.rb 。在调试之前，您需要创建运行/调试配置，以使用Rails运行器运行此脚本。

创建Rails运行器配置

  1. 在主菜单中，转到“运行”。编辑配置 ，单击 ，然后从列表中选择Gem Command 。

  2. 在“ 运行/调试配置”对话框中，指定以下设置：

    • 名称 ：输入运行/调试配置名称，例如railsRunner 。

    • 宝石名称 ：在此处指定rails宝石。

    • 可执行文件名称 ：选择rails可执行文件。

    • 参数 ：指定用于使用Rails运行器运行脚本的参数： r lib/script.rb 。

    单击确定 。

调试Rails运行器配置

要调试新创建的railsRunner配置，请执行以下操作：

  1. 在lib / script.rb文件中设置一个断点。

  2. 按Ctrl两次，然后在弹出窗口中输入配置名称： rails runner 。

  3. 按住Shift键（对话框标题更改为Debug ），然后按Enter 。RubyMine将在断点之前暂停脚本执行。
