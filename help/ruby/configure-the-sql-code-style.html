配置SQL代码样式

更改代码样式设置

自定义SQL代码的格式设置规则：对齐，换行和缩进。

  1. 单击文件|设置|编辑器代码样式| SQL 。

  2. 在可用的选项卡上选择所有必需的代码样式设置。

  3. 点击应用 。

当您应用规则时，RubyMine将更新SQL代码并在“ 预览”窗格中突出显示所有修改的部分。

为方言创建代码样式

  1. 开启设定|编辑器代码样式| SQL 。

  2. 点击齿轮图标（ ），然后选择复制 。

  3. 输入新样式的名称，然后按Enter 。

  4. 选择所有必需的代码样式设置，然后单击Apply 。

配置每个数据源的代码样式

在为方言创建样式之后，可以将此样式应用于使用该方言的数据源。

  1. 右键单击数据源，然后选择属性 。

  2. 单击选项选项卡。

  3. 在代码样式下拉菜单中，选择要应用的样式。

  4. 点击应用 。

从上下文菜单将SQL样式应用于文件

您可以从上下文菜单中格式化文件而无需打开它们。

在“ 文件”工具窗口（“ 视图” |“工具窗口” |“文件” ）中，右键单击目录，然后选择“ 重新设置代码格式” 。
