AngularJS

AngularJS也称为Angular 1是用于开发单页Web应用程序的框架。RubyMine建议针对预定义和自定义的AngularJS感知完成选项ng指令以及控制器和应用程序名称，以及对花括号表达式内的数据绑定的代码见解{{}} 。您可以使用内置的AngularJS实时模板，并在HTML中的控制器名称和JavaScript中的控制器名称之间或之间导航。 ngView要么&routeProvider和模板。对于AngularJS实体，请使用“ 转到符号”导航。

在你开始前

  1. 下载并安装Node.js。

  2. 如管理插件中所述，在“ 插件”页面上安装并启用Angular和AngularJS存储库插件。

创建一个新的AngularJS应用程序

您可以使用AngularJS种子项目或创建一个空的RubyMine项目，然后通过下载AngularJS框架或使用Bower包管理器手动将AngularJS安装在其中。

使用种子项目创建应用程序

  1. 选择文件|新增|项目从主菜单或单击欢迎屏幕上创建新项目 。将打开“ 新建项目”对话框。

  2. 在左侧窗格中，选择AngularJS 。

  3. 在右侧窗格中， 指定将存储与项目相关的文件的文件夹的路径。

  4. 当您点击创建时 ， RubyMine基于AngularJS种子项目，使用所有必需的配置文件生成特定于AngularJS的项目结构

要下载AngularJS依赖项，请执行以下一项操作：

  • 在嵌入式终端 （ Alt+F12 ），输入：

    npm install

  • 从项目根目录中package.json文件的上下文菜单中选择Run'npm install' 。

AngularJS依赖项包含AngularJS代码以及支持开发和测试的工具。

创建一个空的RubyMine项目

  1. 选择文件|新品|项目从主菜单或单击欢迎屏幕上创建新项目 。将打开“ 新建项目”对话框。

  2. 在左侧窗格中，选择空项目 。

  3. 指定将存储与项目相关的文件的文件夹的路径。

  4. 当您单击Create时 ，RubyMine创建并打开一个空项目。

在空项目中手动安装和配置AngularJS

  1. 在http://angularjs.org/下载AngularJS框架。

  2. 打开将在其中使用AngularJS的空项目。

  3. 将AngularJS配置为RubyMine JavaScript库，以使RubyMine识别AngularJS特定的结构并提供完整的编码帮助：

    1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” | JavaScript |图书馆 。将打开“ JavaScript库”页面 。

    2. 在“ 库”区域中，单击“ 添加”按钮。

    3. 在打开的“ 新建库”对话框中，指定库的名称。

    4. 请点击在库文件列表旁边，然后从上下文菜单中选择“ 附加文件”或“ 附加目录 ”，具体取决于您需要单独的文件还是整个文件夹。

    5. 在打开的对话框中选择Angular.js或Angular.min.js或整个目录。RubyMine返回到“ 新库”对话框，其中“ 名称”只读字段显示所选文件或文件夹的名称。

    6. 在“ 类型”字段中，指定已下载并要添加的版本。

      • 如果添加了Angular.js ，请选择“ 调试” 。该版本对开发环境特别是调试很有帮助。

      • 如果添加了缩小的 Angular.min.js ，请选择Release 。该版本在生产环境中很有用，因为文件大小明显较小。

    从配置JavaScript库中了解更多信息。

通过Bower在空项目中安装AngularJS

  1. 打开将在其中使用AngularJS的空项目。

  2. 安装鲍尔如描述鲍尔 。

  3. 在嵌入式终端 （ Alt+F12 ），输入bower install angular将软件包安装为项目依赖项。

您也可以安装angular如“ 管理Bower程序包”中所述，在“设置/首选项”对话框的“ Bower”页上找到该程序包 。

从现有的AngularJS应用程序开始

如果您的项目中已经有Angular源（例如，在bower_components文件夹中），则只需打开项目并开始工作即可。如果这些源未从项目中排除 ，则只需要将AngularJS配置为JavaScript库 。

从版本控制中签出应用程序源

  1. 在“ 欢迎”屏幕上单击“从版本控制获取”，或选择“ VCS |从主菜单的版本控制中获取 。

  2. 在调用的对话框中，从列表中选择您的版本控制系统，然后指定存储库以签出应用程序源。

使用AngularJS路由器状态图

您可以看到一个图，该图说明了使用ui-router的 AngularJS应用程序中的视图，状态和模板之间的关系。

生成并查看图

  • 在编辑器中打开所需的文件，然后选择“ 图|图表”。从上下文菜单中显示AngularJS ui-router状态图 。RubyMine生成一个图，并在一个单独的编辑器选项卡中显示它。

从图中的元素导航到实现该元素的代码

  • 选择它，然后从上下文菜单中选择“ 跳转到源 ”。
