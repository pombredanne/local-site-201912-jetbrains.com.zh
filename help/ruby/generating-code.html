完整的配对元素

RubyMine可以自动在代码中添加各种结束元素，例如方括号，引号，XML和HTML标记或将Ruby代码包装在视图中的标记（<％％>）。

要启用或禁用是否添加结束元素，请打开“ 设置/首选项”对话框Ctrl+Alt+S ，单击“ 编辑器”下的“ 常规 ” ，然后单击“ 智能键” 。例如，您可以使用以下选项：

  • 插入对括号
  • 插入对报价

根据用法生成方法

您可以为要使用但尚未定义的方法生成一个空的存根。

为此，请执行以下步骤：

  1. 键入引用不存在的方法的名称。RubyMine突出了参考。

  2. 按Alt+Enter ，选择创建一个从建议列表方法'，然后按Enter 。

创建属性

RubyMine使您可以使用attr_reader ， attr_writer和attr_accessor属性为实例变量快速添加访问器方法。

为此，请执行以下步骤：

  1. 将插入符号放在班级中，然后按Alt+Insert 。

  2. 在打开的弹出窗口中，选择Reader ， Writer或Accessor 。

  3. 在调用的对话框中选择所需的实例变量，然后单击“ 确定” 。

请注意，如果在创建Reader之后创建变量的Accessor ，RubyMine将自动用attr_accessor替换相应的attr_reader 。

覆盖超类的方法

您可以通过在子类中生成相应的方法存根来覆盖父类的任何方法。

执行以下步骤以执行此操作：

  1. 将插入号放入子类中，然后执行以下操作之一：

    • 在“ 代码”菜单上，单击“ 替代方法” （ Ctrl+O ）。

    • 在代码菜单上，单击生成 （ Alt+Insert ），然后选择覆盖方法 。

  2. 在“ 选择要覆盖的方法”对话框中选择所需的方法。

  3. 实现创建的方法。

使用左装订线中的按钮打开覆盖的方法声明。

具有特定语言构造的环绕代码片段

RubyMine根据源代码的语言为周围的代码片段提供各种结构的标准模板。例如，您可以将代码包装到if…end或除非……条件语句中。

包围一段代码：

  1. 选择所需的代码片段。

  2. 在“ 代码”菜单上，单击“ 环绕声” （ Ctrl+Alt+T ）。

  3. 从列表中选择必要的环绕声语句，然后按Enter 。

解开和删除语句

RubyMine使您可以快速从包装语句中解开或提取表达式。

此操作可用于：

  • 红宝石

  • 的JavaScript

  • XML和HTML标签

拆开或删除语句：

  1. 将插入符号放在要提取或解包的表达式上。

  2. 选择代码|从主菜单解包/删除或按Ctrl+Shift+Delete 。RubyMine显示一个弹出窗口，其中包含当前上下文中可用的所有操作。

  3. 单击所需的操作，然后按Enter 。

使用实时模板创建代码构造

实时模板可用于将常用的构造插入到您的源代码中，例如循环，条件，各种声明，打印语句，标签等。

要了解更多信息，请参阅实时模板 。
