使用此部分为YAML文件配置代码样式选项。更改这些设置时，更改结果将显示在“ 预览”窗格中。

制表符和缩进

缩进

在此字段中，为每个缩进级别指定要插入的空格数。

缩进空行

如果选中此复选框，则RubyMine会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，则RubyMine删除空白行上的空格。

缩进序列值

如果选中，则相对于父键项缩进YAML序列值。

包装和大括号

在此标签中，定制RubyMine在重新格式化源代码时将应用于各种代码构造的异常，括号放置和对齐选项。在“ 预览”窗格中检查结果。

对齐优先于缩进选项。

视觉指南

在此字段中，指定多个右边距。您可以保留默认值，也可以输入空白。如果要指定多个边距，请输入用逗号分隔的数字。

重新格式化时保持

默认情况下，“ 换行符”复选框处于选中状态。在这种情况下，RubyMine在重新格式化源代码时将忽略换行符。如果您的代码包含比标准约定短的行，则可以通过在重新格式化之前禁用“ 换行符”复选框来转换它们。

对齐地图中的值

使用此列表可以指定RubyMine如何对齐键值对。可用的选项有：

  • 不对齐 ：连续行中的属性将不对齐。

  • 在冒号上 ：连续行中的属性将与冒号对齐。

  • 在值上 ：连续行中的属性将与值对齐。

序列值

使用以下选项为YAML序列值指定格式：

  • 新行上的序列 ：如果选中，则嵌套序列块中的每个新序列都放在单独的行上：

    ---嵌套序列块

    否则，嵌套序列将保留在一行上：

    ---嵌套序列块

  • 新行上的块映射 ：如果选中，则序列内的块映射将放在单独的行上：

    - 核心价值

    否则，它们将保持在一行上：

    - 核心价值

  • 在enter处自动插入连字符 ：如果选中，当您按下RubyMine时，它将自动对下一个序列值进行连字符Enter 。
