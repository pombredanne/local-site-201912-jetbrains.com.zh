调试控制台

当您为Ruby脚本，Rails应用程序或测试运行调试会话时 ，RubyMine在控制台中显示输出和错误消息。在达到断点或手动暂停调试会话之后，您可以使用类似于IRB的交互式控制台检查已暂停程序的状态。在此控制台中，您可以检查变量值，检查复杂对象，调用方法等。

在本主题中，我们将向您展示如何打开和使用此控制台。

您可以在IRB控制台中加载和调试代码的特定部分，而无需运行程序。在教程：在IRB / Rails控制台中进行调试中了解更多信息。

打开交互式控制台

根据调试会话的类型，RubyMine提供了两种在控制台中与应用程序进行交互的方式：

  • 如果您调试Ruby脚本或Rails应用程序，请在“ 调试工具”窗口中打开“ 控制台”选项卡，并确保按下按钮。

  • 如果调试测试，Rake任务，则“ 控制台”选项卡将忙于测试输出。在这种情况下，请切换到“ 交互式控制台”选项卡以与您的应用程序进行交互。

使用交互式控制台

您可以在控制台中执行以下操作：

  • 执行命令

    在控制台中键入命令，然后按Enter执行它们。

  • 完整名称

    使用基本代码完成 Ctrl+Space完成类，方法等的名称。

  • 浏览历史

    使用Up和Down箭头键可滚动浏览命令的历史记录并执行所需的命令。

  • 导航到源代码

    要导航到符号和符号类型的初始声明，请将插入标记放在所需的符号上，然后按Ctrl+B 。

  • 查看快速文档

    要获得任何符号或方法签名的快速信息 ，请将插入符号放在所需的符号上，然后按Ctrl+Q 。
