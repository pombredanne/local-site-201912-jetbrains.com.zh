生成档案

RubyMine与Warbler集成，提供了生成应用程序.war存档的可能性。与Warbler集成需要JRuby。

安装莺鸟宝石后， warble命令从控制台变为可用。除此之外， 工具| Build WAR命令从主菜单出现。

请注意， Build WAR file命令可识别项目中解释程序和gem的类型，并建议配置适当的版本。

生成网络档案

  1. 在主菜单上，选择“ 工具”。建立。WAR文件 。

  2. 在输出文件名字段中指定目标档案的名称 。

  3. 在构建中。在“ WAR文件”对话框中，选中要包含在目标归档中的目录和文件旁边的复选框。

  4. 根据需要启用或禁用以下行为选项： 可执行 -如果选中此复选框，则目标归档将是可执行的。
    预编译 -如果选中此复选框，则Ruby文件将被编译为类文件。
    包括宝石库 -如果选中此复选框，则宝石库将包含在存档中。
    打开调用/执行跟踪，启用完整回溯 -如果选中此复选框，将启用跟踪模式。

  5. 选择一个或多个要包含在Web存档的WEB-INF目录中的文件，然后按WEB-INF配置按钮。

  6. 单击确定以生成存档。
