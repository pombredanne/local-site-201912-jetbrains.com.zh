兑换

转换重构允许将方法转换为函数或块，反之亦然，以及将属性转换为实例变量。

这些重构是从“ 重构”菜单中调用的。您也可以出于相同目的使用更改签名重构。

转换重构仅适用于Objective-C。

转换为方法

“ 转换为方法”重构使您可以将函数或块转换为方法。

  1. 在编辑器中，将插入符号放在要转换为方法的函数或块上。

  2. 从主菜单或上下文菜单中，选择“ 重构|转换为Method 。

  3. 在打开的对话框中，根据需要进行更改：

  4. 要执行重构，请点击重构 。要查看预期的更改并在执行重构之前进行必要的调整，请单击预览 。

转换为功能

“ 转换为函数”重构使您可以将方法或块转换为函数。

  1. 在编辑器中，将插入符号放在要转换为函数的方法或块上。

  2. 从主菜单或上下文菜单中，选择“ 重构|转换为Function 。

  3. 在打开的对话框中，根据需要进行更改：

  4. 要执行重构，请点击重构 。要查看预期的更改并在执行重构之前进行必要的调整，请单击“ 预览” 。

转换成块

通过转换为块重构，可以将函数或方法转换为块。

  1. 在编辑器中，将插入符号放在要转换为块的方法或函数上。

  2. 从主菜单或上下文菜单中，选择“ 重构|转换为Block 。

  3. 在打开的对话框中，根据需要进行更改：

  4. 要执行重构，请点击重构 。要查看预期的更改并在执行重构之前进行必要的调整，请单击“ 预览” 。

转换为财产

通过转换为属性重构，可以将实例变量转换为属性。

  1. 将插入符号放在要转换为属性的实例变量上。

  2. 从主菜单或上下文菜单中，选择“ 重构|转换为Property 。

  3. 在打开的对话框中，选择要转换的实例变量：

  4. 单击“ 确定”执行重构。

之前

后

@interface AlertParams：NSObject {//这些实例变量//将转换为属性@public NSString * _title; NSString * _message; } @结束
@interface AlertParams：NSObject //新属性@property（非原子的，强壮的）NSString * title; @属性（非原子的，强的）NSString *消息； @结束

转换为实例变量

转换为实例变量重构使您可以将属性转换为实例变量。

  1. 将插入符号放在编辑器窗口中。

  2. 从主菜单或上下文菜单中，选择“ 重构|转换为实例变量 。

  3. 在打开的对话框中，选择要转换的属性：

  4. 单击“ 确定”执行重构。
