AppCode 2019.2帮助

密码

在此页面上，配置您希望AppCode如何存储用于访问远程Git存储库的密码。这些设置在AppCode级别有效，也就是说，它们适用于您从AppCode使用的所有远程Git存储库的密码。

项目

描述

忘记密码

选择此选项后，AppCode不会存储密码，并在每次尝试访问远程Git存储库时要求您输入密码。

记住密码，直到关闭应用程序

选择此选项后，AppCode仅在一个会话期间存储密码。会话结束后，将从内存中清除密码。

在磁盘上记住（受主密码保护）

选择此选项可使AppCode将所有密码保存到受单个主密码保护的密码数据库中的远程存储库中。当您尝试访问远程Git存储库时，AppCode不会要求提供相应的密码，而是要求您指定主密码并自动检索存储库密码。

重置主密码

单击此按钮可打开“ 主密码/重置主密码”对话框，并首次指定主密码或重置现有的主密码 。

如果重置主密码 ，则会清除所有先前保存的主密码 。

修改主密码

单击此按钮可以在“ 更改主密码”对话框中更改现有的主密码 。

AppCode保留对主密码进行更新的历史记录，直到您重置它为止。

主密码

单击此按钮设置或重置主密码。将打开“ 更改主密码”对话框 。
