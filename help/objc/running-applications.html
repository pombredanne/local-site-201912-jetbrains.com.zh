运行应用程序

AppCode可以运行整个应用程序以及单元测试。

AppCode利用“ 运行/调试配置”中定义的设置。项目中存在的所有运行配置在“ 选择运行/调试配置”列表中可用。

如果在运行/调试配置中启用了在运行前启动外部工具的选项，则AppCode将运行外部工具，并在成功后运行该应用程序。否则，程序将立即启动。

如果要查看所有当前正在运行的应用程序的列表，请选择“运行” |“运行”。从主菜单显示运行列表 。

请注意，开始运行会话后， 标记“ 运行工具”窗口的图标，并在“运行/调试配置选择器”中切换到指示运行过程处于活动状态。

运行一个应用程序

  • 在左侧装订线中，单击图标 ，然后选择所需的命令。

  • 在主工具栏上，选择所需的运行配置，然后执行以下操作之一：

    • 选择运行|从主菜单运行“选定的配置” 。

    • 请点击 。

    • 按⌃R 。

  • 按⌃⌥R ，从列表中选择所需的运行配置，然后按⏎ 。

    在此弹出菜单中，您还可以：

    • 调用“ 编辑配置”对话框 。

    • 启动前编辑所选配置⌘↓ 。

    • 从运行切换到调试，反之亦然（按住⇧ ）。

    • 访问先前选择的配置1 。

    • 如果要查看所有当前正在运行的应用程序的列表，请选择“运行” |“运行”。从主菜单显示运行列表 。

可以选择模拟器并以相同的方式运行。
