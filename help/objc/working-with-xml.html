AppCode 2019.2帮助

XML格式

AppCode带来了强大的XML编辑支持，其中包括：

  • 验证和语法突出显示。

  • 代码完成 ⌃Space 。

  • 缩进 （ ⌃⌥I ， ⌥⌘L ）。

  • 格式化 ⌥⌘L根据XML代码样式 。

  • 意向动作 ⌥⏎ 。

  • 导入未绑定的名称空间 。

  • 查看代码结构 ⌘7 。

  • 在源代码中导航 ⌘B 。

  • 快速文档 F1 。

  • 搜索用法 ⌥F7 。

  • 评论和取消评论行（ ⌘/ ， ⌥⌘/ ）。

  • 展开和删除标签 ⇧⌘⌦ 。

  • 从实例文档生成模式，反之亦然。

  • 生成DTD文件 。

AppCode使用Xerces 2.11 ，这是由Apache Software Foundation Group开发的XML解析器。

导入XML名称空间

  1. 打开所需的文件进行编辑，然后开始输入标签。如果未绑定名称空间，则会出现以下提示：

  2. 按⌥⏎ 。如果有多个选择，请从列表中选择所需的名称空间。

配置语法突出显示

您可以根据自己的喜好和习惯配置可识别XML的语法。

  1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |配色方案| XML 。

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
