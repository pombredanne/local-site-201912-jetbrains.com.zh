XML

AppCode brings powerful XML editing support that includes:

  • Validation and syntax highlighting.

  • Code completion ⌃Space.

  • Indentation (⌃⌥I, ⌥⌘L).

  • Formatting ⌥⌘L according to the XML code style.

  • Intention actions ⌥⏎.

  • Importing unbound namespaces.

  • Viewing code structure ⌘7.

  • Navigation in the source code ⌘B.

  • Quick documentation F1.

  • Search for usages ⌥F7.

  • Commenting and uncommenting lines (⌘/, ⌥⌘/).

  • Unwrapping and removing tags ⇧⌘⌦.

  • Generating schemas from instance documents and vice versa.

  • Generating DTD files.

AppCode uses Xerces 2.11, an XML parser developed by Apache Software Foundation Group.

  2. Press ⌥⏎. If there are multiple choices, select the desired namespace from the list.

Configuring syntax highlighting

You can configure XML-aware syntax highlighting according to your preferences and habits.

  1. In the Preferences dialog (⌘,), go to Editor | Color Scheme | XML.

  2. Select the color scheme, accept the highlighting settings inherited from defaults or customize them as described in Configuring Colors and Fonts.
