从Subversion存储库中检出文件

通过从Subversion存储库中检出文件，可以获得存储库的本地工作副本 ，您可以对其进行编辑。进行必要的更改后，您可以通过提交或签入对存储库的更改来发布结果。

要从Subversion存储库中检出文件，请执行以下操作：

  1. 在主菜单上，选择VCS |。从版本控制中检出|颠覆 。

  2. 在“从Subversion签出”对话框中，展开所需的存储库位置，然后选择要签出的元素。

  3. 点击结帐按钮。

  4. 在打开的对话框中，指定将在其中创建存储库文件的本地副本的目标目录，然后单击“ 确定” 。

    如果要签出现有项目的源，则目标文件夹应在项目下方。

  5. 在“ SVN检出选项”对话框中，指定以下设置：

    • 要检出的修订（HEAD或选定的修订）。

    • 是否需要签出嵌套目录。

    • 是否需要包括外部位置。

    单击确定 。

AppCode建议根据从版本控制中检出的源来创建项目。
