代码生成

AppCode提供了多种生成通用代码结构和重复元素的方法，可帮助您提高生产率。这些可以是预定义模板，也可以是自定义模板，它们会根据上下文，各种包装程序和字符自动配对而不同地应用。

此外，AppCode还提供代码完成和Emmet支持。

在主菜单中，选择代码|生成 ⌘N打开带有您可以生成的可用结构的弹出菜单。

生成初始化器

AppCode允许您为类生成初始化程序，并选择初始化参数。

为一个类生成一个初始化器

  1. 在代码菜单上，单击生成 ⌘N 。

  2. 在“ 生成”弹出窗口中，单击“ Swift的初始化程序”或“ Objective-C的initWith ”。

  3. 如果该类包含字段，请选择要初始化的字段，然后单击“ 确定” 。

以下代码片段显示了为类生成初始化程序的结果：

为isEqual（_ :)和哈希生成代码

AppCode可以为标准生成替代isEqual(_:)方法和hash属性。

阅读更多isEqual(_:)和hash ：

  • Swift： isEqual（_ :)和哈希

  • Objective-C： isEqual（_ :)和哈希

生成isEqual（_ :)并为类散列

  1. 在代码菜单上，单击生成 ⌘N 。

  2. 在“ 生成”弹出窗口中，单击等于和hashValue （对于Swift）或-isEqual：和-hash （对于Objective-C）。

  3. 选择用于确定相等性的字段，然后单击下一步 。

  4. 选择用于计算哈希码值的字段。您只能从上一步（用于确定相等性）中选择的字段中选择。单击下一步 。

覆盖超类的方法

您可以通过在子类中生成相应的方法存根来覆盖父类的任何方法。

执行以下步骤以执行此操作：

  1. 将插入号放入子类中，然后执行以下操作之一：

    • 在“ 代码”菜单上，单击“ 替代方法” （ ⌃O ）。

    • 在代码菜单上，单击生成 （ ⌘N ），然后选择覆盖方法 。

  2. 在打开的对话框中选择所需的方法。

  3. 实现创建的方法。

使用左装订线中的按钮打开覆盖的方法声明。

实现接口或抽象类的方法

如果一个类被声明为实现接口或抽象类，则它必须实现父类或基本接口的方法。AppCode为实现的方法创建存根。

  1. 从类的上下文菜单中选择“ 生成 ” ⌘N并从弹出窗口中选择“ 实施方法” ，或按⌃I 。

  2. 在打开的对话框中，选择要实现的方法，然后单击确定 。

通过使用生成属性

您可以为要使用但尚未定义的属性生成一个空的存根。

为此，请执行以下步骤：

  1. 键入引用不存在的属性的名称。AppCode突出显示参考。

  2. 按⌥⏎ ，选择创建从建议列表属性'，然后按⏎ 。

使用实时模板生成代码

AppCode为许多常见的代码构造提供了许多预定义的实时模板 。您还可以定义自定义模板，以涵盖特定于您的工作流程的用例。

您可以通过插入和扩展实时模板以及使用环绕模板包装代码片段来创建代码构造。

插入实时模板

  1. 将插入符号放置在您希望模板扩展的位置。

  2. 输入模板缩写，然后按调用键（ ⇥默认）。

    或者，要使用代码完成功能，请按⌘J或从上下文菜单中选择“ 插入实时模板” ，然后从建议列表中选择所需的模板。要查看所选建议的快速文档 ，请按F1 。

  3. 如果所选模板需要用户输入，则第一个输入字段会用红色框标记。在此框中键入您的值，然后按⏎要么⇥完成输入并传递到下一个输入字段。在完成最后一个输入字段后，插入符号将移至构造的末尾，编辑器将返回常规操作模式。

用实时模板包围代码片段

您可以将代码仅放在那些实时模板中， $SELECTION$使用变量。此变量指示应包装的代码。要将变量添加到模板，请转至“首选项” |“模板”。编辑器实时模板 ，选择您的语言和模板，然后将变量放置在模板文本字段中的所需位置。

  1. 在编辑器中，选择要包装的代码，然后按⌥⌘J或从上下文菜单或选择中选择“ 实时模板环绕” 。

  2. 在建议列表中，选择所需的模板。

查看可用的实时模板列表

  • 在“ 首选项”对话框中（ ⌘, ），点击编辑器下的实时模板 。打开的实时模板页面显示按语言分组的所有已配置实时模板。有关更多信息，请参见创建实时模板 。

用代码环绕

AppCode根据源代码的语言为周围的代码片段提供具有各种结构的标准模板。例如，您可以将代码包装到if…end或除非……条件语句中。

包围一段代码：

  1. 选择所需的代码片段。

  2. 在“ 代码”菜单上，单击“ 环绕声” （ ⌥⌘T ）。

  3. 从列表中选择必要的环绕声语句，然后按⏎ 。

完整的配对元素

AppCode可以自动在代码中添加各种结束元素，例如方括号，引号，XML和HTML标签。

要启用或禁用是否添加结束元素，请打开“ 首选项”对话框⌘, ，单击“ 编辑器”下的“ 常规 ” ，然后单击“ 智能键” 。例如，您可以使用以下选项：

  • 插入对括号
  • 插入对报价

解开和删除语句

AppCode使您可以快速从封装语句中解开或提取表达式。

拆开或删除语句：

  1. 将插入符号放在要提取或解包的表达式上。

  2. 选择代码|从主菜单解包/删除或按⇧⌘⌦ 。AppCode显示一个弹出窗口，其中包含当前上下文中可用的所有操作。

  3. 单击所需的操作，然后按⏎ 。
