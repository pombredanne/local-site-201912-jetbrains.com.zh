AppCode Preferences

AppCode | Preferences ⌘,

The settings that pertain to the current project are marked with the icon.

The Preferences dialog is a central location for all global AppCode IDE settings and AppCode-specific project settings.

To configure the Xcode-compatible project settings, either use AppCode settings editor (choose File | Project Settings in the main menu) or open the project in Xcode (choose File | Open Project in Xcode in the main menu) and use the native Xcode settings editor.

On this page find the descriptions of the main controls of the dialog.

Item

Description

Search

Enter a search keyword in the text area. While typing the search string, the list of options in the dialog reduces to the matching occurrences.

Click this button to clear the search area.

OK

Apply changes and close the dialog.

Cancel

Discard changes and close the dialog.

Apply

Apply changes and leave the dialog opened.
