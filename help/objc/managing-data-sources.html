管理数据源

数据源是您的应用程序的数据位置。在AppCode中，您可以定义以下数据源：

  • 数据库数据源：包含数据的可操作数据库（请参阅连接数据库 ）。

  • 数据定义语言（DDL）文件：包含数据定义语言语句的SQL文件（请参阅添加DDL数据源 ）。

创建数据源

从数据库文件创建数据库数据源

  1. 如果项目文件夹中有数据库文件，请打开“ 项目”工具窗口。否则，请打开文件管理器。

  2. 打开数据库工具窗口（“ 视图” |“工具窗口” |“数据库” ）。

  3. 选择要用作数据源的文件，然后将它们拖到“ 数据库”工具窗口中。为每个文件创建一个单独的数据源。

  4. 如果没有必需的数据库驱动程序文件，请下载它们。请点击在数据库工具窗口的工具栏上。或者，从上下文菜单中选择“ 属性 ”。

  5. 在“ 数据源和驱动程序”窗口中，单击“ 下载”链接。

  6. 为确保成功连接到数据源，请单击“ 测试连接” ，然后单击“ 确定” 。

创建一个DDL数据源

  1. 打开“ 数据库”工具窗口（“ 视图” |“工具窗口” |“数据库” ）。

  2. 单击数据源属性图标（ ），然后选择DDL数据源 。

  3. 指定数据源名称。

  4. 在“ DDL文件”表单中，单击“ 添加”图标（ ），然后选择必要的SQL文件。

另外，您可以将DDL文件从“ 文件”工具窗口或文件管理器拖到“ 数据库”工具窗口。

自定义数据源

配置数据源设置

  1. 在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中，选择要更改的数据源。

  2. 单击数据源属性图标（ ）（或按⌘I ）。

  3. 在“ 数据源和驱动程序”窗口中，编辑数据源属性。有关数据源属性的更多信息，请参阅数据源设置和DDL数据源设置 。

组数据源

如果您有许多不同的数据库，则可以在“ 数据库”工具窗口（“ 视图” |“工具窗口” |“数据库” ）中对它们的数据源进行分组。要对数据源进行分组，请选择所需的数据源，然后按F6 ，或从上下文菜单中选择“ 移至组 ”。

或者，如果要创建一个新组，只需将一个数据源拖到另一个上即可。如果要将数据源放入现有组中，请将其拖到那里。

在“ 移至组”菜单中，以下选项可用于数据源组：

  • 为数据源创建一个新组。

  • 将数据源移动到根列表。此选项从组中删除数据源。

  • 将数据源移动到现有组。

删除数据源

要删除数据源，请在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中，按⌦ 。或者，您可以从上下文菜单中选择“ 删除 ”，或单击“ 编辑” |“ 删除 ” 。删除 。

另外，要删除数据源，可以使用“ 数据源和驱动程序”窗口（“ 文件” |“数据源” ）。在“ 数据源和驱动程序”窗口中，选择数据源，然后按⌦ 。或者，您可以单击在工具栏上，或从上下文菜单中选择“ 删除 ”。
