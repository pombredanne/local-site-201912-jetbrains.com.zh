Export Threads

Run | Export Thread
Debug tool window | Threads | context menu | Export Threads

Use this dialog to export a thread to the specified text file.

Item

Description

Export to file

Type the target file to store the thread report.

Use this button to navigate to the desired location in the Select Path dialog.

Save

Click to save the current threads in a text file indicated in the Export to file field.

Copy

Click to copy the thread status to the Clipboard.

Cancel

Use to discard all changes and close the dialog.
