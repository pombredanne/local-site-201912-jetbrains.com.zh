制表符和缩进

项目

描述

使用制表符

  • 如果选中此复选框，则使用制表符：

    • 在按⇥键

    • 缩进

    • 用于代码重新格式化

  • 清除复选框后，AppCode将使用空格而不是选项卡。

智能标签

  • 如果选中此复选框，则由代码块嵌套定义的缩进部分由制表符和（如有必要）空格组成，而由对齐方式定义的缩进部分仅由空格组成。

  • 如果清除此复选框，则仅使用选项卡。这意味着将将一组适合指定制表符大小的空格自动替换为一个制表符，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中包含的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组元素之间插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

缩进空行

如果选中此复选框，则AppCode将在空白行上保持缩进，就好像它们包含一些代码一样。
如果清除此复选框，AppCode将删除制表符和空格。

缩进多行字符串

如果选中此复选框，则多行字符串将缩进。

指令

遵循代码缩进 ：如果选中此复选框，则将缩进条件编译块以匹配周围的代码。
缩进子代 ：如果选中此复选框，则条件编译块中的语句将缩进。
选择“ 遵循代码缩进”时，此选项可用。

空间

使用此选项卡可以指定代码中需要空格的位置。要让AppCode自动在某个位置插入空格，请在列表中选中该位置旁边的复选框。结果显示在“ 预览”窗格中。

包装和大括号

在此标签中，自定义代码样式选项，AppCode将在重新格式化源代码时应用这些样式。左侧窗格包含异常列表（ 重新格式化时保留 ），以及各种代码结构（列表，语句，操作，注释等）的放置和对齐选项。右侧窗格显示预览。

对齐优先于缩进选项。

右边距（列）

使用“ 硬包装在”字段可以指定元素右侧所需的页边空白空间。如果选择默认选项，则使用全局设置中右边距的值。

包装打字

使用打字设置环绕到指定编辑的文本是怎样安装在指定的硬包装的 。您可以选择以下选项之一：

  • 默认值 ：在这种情况下，AppCode使用全局设置中指定的“ 键入时自动换行”选项。

  • 是 ：在这种情况下，AppCode使用“ 右边界”字段中指定的值。

  • 否 ：在这种情况下，此选项已关闭，并且一行可以超出右边界中指定的值。

视觉指南

使用“ 视觉参考线”字段来指定多个右边距。您可以保留默认值，也可以输入空白。如果要指定多个边距，请输入用逗号分隔的数字。

重新格式化时保持

使用复选框配置AppCode在重新格式化源代码时将产生的异常。例如，默认情况下，“ 换行符”复选框处于选中状态。如果您的代码包含比标准约定短的行，则可以在重新格式化源代码之前通过禁用换行符复选框来转换它们。

项目

描述

换行

如果选择此选项，则重新格式化代码后将保留换行符。如果取消选择该选项，则比标准约定短的行将合并为一行。

第一栏评论

如果选择此选项，则在重新格式化代码后，放置在第一列的注释将不会更改其位置。如果取消选择该选项，则此类注释将与下一行代码对齐。

一行中的简单块和（拖曳）闭包

如果选择此选项，则重新格式化代码后，块和尾随闭包将保留为一行。如果取消选择该选项，它们将扩展为多行。

一行中的简单方法和功能

如果选择此选项，则在重新格式化代码后，简单的方法和功能将保留为一行。如果取消选择该选项，它们将扩展为多行。

一行中的简单闭包参数

如果选择此选项，则括号中的简单闭包参数()重新格式化代码后，它们将保留为一行。如果取消选择该选项，它们将扩展为多行。

包装选项

包装样式适用于在左侧窗格中指定的各种代码构造（例如，方法调用参数或赋值语句）。

项目

描述

包装方式

从此列表中选择所需的包装样式：

  • 不包裹 ：选择此选项时，不应用特殊的包裹样式。

    选择此选项后，嵌套的对齐方式和花括号设置将被忽略。

  • 如果很长则自动换行：选择此选项可使行超出正确的边距并用适当的缩进包裹。

  • 总是换行 ：选择此选项可使列表中的所有元素都换行 ，以便每行有一个具有适当缩进的元素。

  • 如果时间长，则切短 ：选择此选项可使列表中的元素超出右边距的范围，从而使每行有一个带有适当缩进的元素。

多行时对齐

如果选中此复选框，则代码构造将从下一行的同一列开始。否则，代码构造的位置由当前的缩进级别确定。

多行时对齐括号

如果选中此复选框，则带括号的代码构造将在下一行的左括号的列处开始。否则，代码构造的位置由当前的缩进级别确定。

对齐列

如果选中此复选框，则变量名称，类型和初始化程序将按列对齐，每个变量在下一行，例如：

之前

var myName：字符串var myTitle：字符串var myOrderNumber：整数

后

var myName：字符串var myTitle：字符串var myOrderNumber：整数

新行上的“ else”

使用此复选框可将相应的语句或字符移至下一行。

特别待遇

大括号放置样式

使用此列表可以在类声明 ， 方法声明和其他类型的声明中指定左括号的位置。可用的选项有：

  • 行尾 ：选择此选项可将开括号放在声明行的末端。

  • 如果被换行，则换行 ：选择此选项可将开括号放在多行声明行之后的行首。

  • 下一行 ：选择此选项可将开头括号放在声明行之后的行开头。

  • 转移下一行 ：选择此选项可在将声明行转移到相应的缩进高度后，将开括号放在该行。

  • 每次移动下一行 ：选择此选项可在将声明行移动到相应的缩进级别之后将开括号放在该行，并将下一行也移动到下一个缩进级别。
