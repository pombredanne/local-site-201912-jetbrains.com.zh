HTML，CSS和JavaScript的实时编辑

使用实时编辑功能，您对HTML，CSS或JavaScript代码所做的更改将立即显示在浏览器中，而无需重新加载页面。Live Edit仅在调试会话期间可用。

Live Edit适用于包含或生成HTML，CSS或JavaScript的其他文件类型。您还可以在调试Node.js应用程序时使用Live Edit 。

默认情况下，仅对HTML和CSS文件启用Live Edit 。

激活和配置实时编辑

  1. 如“ 管理插件”中所述，在“ 插件”页面上安装LiveEdit插件。

  2. 在“ 首选项”对话框中（ ⌘, ），转到“ 构建，执行，部署|调试器实时编辑 。

  3. 在打开的“ 实时编辑”页面上，选中“ 更改时在Chrome中更新应用程序”复选框。这样可以即时预览HTML和CSS。

  4. 要在JavaScript中启用实时编辑 ，请选择JavaScript，HTML和CSS选项。

  5. 指定在编辑器中更改代码与在浏览器中显示此更改之间的时间延迟：接受默认值300 ms或使用相应字段旁边的旋转框指定自定义值。

  6. 要配置突出显示，请选中“ 插入符号更改中的浏览器中突出显示当前元素”复选框。否则，在调试会话期间，您必须保持⇧然后在编辑器中单击该元素，以在浏览器中突出显示该元素。

  7. 选择热交换失败时重新启动复选框如果没有这些更改无法应用更改，则允许AppCode自动重新加载页面。
