File types

To view the list of file types recognized by AppCode, in the Preferences dialog (⌘,), go to | Editor | File Types. If a file in your project is marked with the icon, it indicates that AppCode can't recognize it. In this case, you can register and configure a new file type.

The set of file types recognized by AppCode depends on enabled plugins.
