TextMate

对于AppCode不支持的多种语言，例如PHP，Python，Ruby和Java，如果您偶尔需要检查此类代码，则IDE仍会提供语法高亮显示以改善您的编码体验。要了解如何使用TextMate语法描述语言，请参见TextMate官方网站 。

这些语言的语法高亮显示基于TextMate语法，而AppCode随附了针对不同语言的语法文件集合。所有可用的捆绑软件都列在“ 编辑器” |“ TextMate捆绑包页面 首选项对话框（ ⌘, ）。

您还可以下载和使用其他语言的自定义TextMate捆绑软件。

在你开始前， 确保已启用TextMate Bundles捆绑插件。如果插件被禁用，请按照“ 管理插件”中所述激活它。
