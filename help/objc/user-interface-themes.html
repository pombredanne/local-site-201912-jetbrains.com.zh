User interface themes

The interface theme defines the appearance of windows, dialogs, and controls.

The interface theme is not the same as the color scheme, which defines the colors and fonts to format various text resources: the source code, search results, and so on.

Change the UI theme

  1. In the Preferences dialog (⌘,), select Appearance & Behavior | Appearance.

  2. Select the UI theme from the Theme list:

    • Darcula: Default dark theme

    • Light: More traditional light theme

    • High contrast: Theme designed for users with color vision deficiency

It is also possible to create your own UI themes for AppCode and customize the built-in themes. For more information, see IntelliJ Platform SDK Documentation.
