表的基本操作

建立表格

  1. 在“ 数据库”工具窗口中，选择一个数据源，然后导航到“ 文件” |“文件”。新增|表 。

  2. 在“ 创建新表”对话框中，指定表设置（列，键，索引，外键）。

  3. 单击确定 。

删除表格

要删除表格，请右键点击表格，然后选择DROP （ ⌦ ）。

您可以为“ SQL预览”字段中生成的代码设置其他选项：

  • 使用模式名称限定对象 ：将模式名称添加到表名称。当您有两个和更多具有相同方案名称的表时，可以限定一个表。此选项具有以下参数：

    • 自动 ：如果在不同方案中具有两个以上相同的表名，则自动限定表名。

    • 从不 ：从不限定表名。

    • 始终 ：始终限定表名。

  • 使用IF EXISTS语法 ：确保该表存在。

  • 使用DROP CASCADE语法 ：删除依赖于表的对象（例如视图）。

如果要编写DROP语句并在以后使用，请使用“ 使用模式名称限定对象”选项。如果您打算立即执行DROP语句，请忽略此选项，因为已经确定了该表。

修改表格

  1. 在数据库工具窗口中，右键单击一个表，然后选择修改表 。

  2. 在“ 修改表”对话框中，指定所需的表设置。

查看表格数据

  1. 打开数据库工具窗口（“ 视图” |“工具窗口” |“数据库” ）。

  2. 在数据库树中，导航到要打开的表。

  3. 双击表格⌘↓ 。

为表启用只读模式

为了防止数据编辑器中的表被意外修改，可以将表设为只读。要启用只读模式，请单击“ 单击以切换”只读属性图标（ ）在编辑器的右下角。

要关闭只读模式，请单击“ 单击以再次切换”只读属性图标。

使用数据库控制台或在数据库工具窗口中时，仍可以修改数据编辑器中具有只读状态的表。

用查询过滤表数据

您可以通过在“ 行过滤器”字段中编写查询来过滤表中的数据。

  1. 如果行过滤器不可用，请单击设置图标（ ），然后选择“ 行过滤器” 。

  2. 在“ 行过滤器”字段中，键入查询。查询语法与WHERE子句，但没有WHERE关键词。例如， first_name LIKE 'Joh%' AND last_name LIKE 'lol%' 。

    您可以在LIKE表达式。例如，百分号%零个或多个字符并加下划线_一个字符。

  3. 按⏎ 。要重置过滤器，请点击清除图标（ ），或删除“ 行过滤器”字段的内容，然后按⏎ 。

使用快速过滤选项

您可以手动指定过滤条件，也可以使用快速过滤器选项。快速过滤器选项是当前列名称的过滤条件。条件取决于当前单元格中的值。

  1. 右键单击一个单元格，然后导航至筛选依据 。

  2. 选择您要应用的选项。

运行搜索时过滤行

您可以在运行搜索时过滤行⌘F在桌子上。

  1. 按⌘F ，然后选择过滤行 。

  2. 开始输入搜索查询（例如， John ）。

比较两个表结构

您可以比较两个表结构，并查看列，键，索引和其他结构表元素之间的差异。要比较选定表中的数据 ，请参阅比较表数据 。

  1. 在数据库工具窗口（ 视图|工具窗口|数据库 ）中，选择两个表。

  2. 右键单击所选内容，然后导航到“ 比较” ⌘D 。

或者，要比较两个表，请按⌘D 。

比较表格数据

您可以比较存储在两个表中的数据。

  1. 双击两个要比较的表。

  2. 在编辑器中，点击比较方式按钮（ ），然后选择第二个表格。

    如果需要，在比较对话框中更改Tolerance参数的值。Tolerance参数定义两个结果集之间允许的最大差异数。例如，如果您希望两行的数据在同一列中不同，则认为两行相等，请输入1在“ 公差”字段中。

转置表格

您可以将表数据从行旋转到列，也可以从列旋转到行。在转置视图中，行和列互换。

要转置表格，请点击设置图标（ ），然后选择“ 转置” 。

将表格导出到文件

当您将表数据复制到剪贴板或将其保存在文件中时，数据将转换为可用的输出格式之一。作为输出格式，您可以选择SQL INSERT要么UPDATE语句， TSV或CSV ，HTML表或JSON数据。

打开一个表或结果集，单击“ 转储数据”图标（ ），然后选择“ 到文件” 。

将表格导出到剪贴板

在工具栏上，单击“ 转储数据”图标（ ），然后选择“ 到剪贴板” 。

配置数据输出格式和选项

您可以配置如何使用预定义的输出格式，自定义格式和脚本将提取的数据转换为文本。要配置提取数据的输出，请右键单击表中的任何单元格，然后指向Data Extractor 。

在此菜单中，您可以选择输出格式（例如，作为SQL INSERT语句，HTML表或CSV格式）或将数据转换为特定格式的脚本。以下附加选项也可用：

跳过计算列（SQL）
启用以排除具有计算值的列（影响SQL输出格式）。
跳过生成的列（SQL）
启用以排除具有自动增量值的列（影响SQL输出格式）。
添加表定义（SQL）
启用以包括CREATE TABLE语句（影响SQL输出格式）。

此外，您可以配置以下参数：

  • 配置CSV格式。 打开“ CSV格式”对话框，可用于管理定界符分隔的值格式（例如CSV或TSV）。

  • 转到脚本目录。 用脚本打开目录，该脚本将表数据转换为各种输出格式。

将数据导出到数据库

  1. 确保在数据库工具窗口中将数据库添加为数据源 。

  2. 右键单击表中的任何单元格，然后单击“ 导出到数据库” 。

  3. 指定数据库，目标架构（以使用导出的数据创建新表）或表（以将导出的数据添加到现有表）。

  4. 配置目标表的数据映射和设置。

在表格之间复制和粘贴单元格

您可以复制⌘C并粘贴⌘V所选单元格和单元格范围-在同一表格内或从一个表格到另一表格。粘贴时，AppCode会根据需要自动转换数据类型。

将表复制到另一个架构

您可以在当前数据源的架构中创建表的副本，或者选择另一个数据源并在此处创建副本。

  1. 右键单击表，然后选择“ 将表复制到” 。或者，按F5 。

  2. 输入架构名称，然后单击“ 确定” 。

  3. （可选）在“ 导入”对话框中，修改表设置。

  4. 按确定 。

重新加载表格视图的数据

如果要将在编辑器中看到的数据与数据库内容同步，则需要为表视图重新加载数据。或者，当您想在更改页面大小限制设置后应用它。

要刷新表视图，请考虑以下操作：

  • 单击同步图标在工具栏上。

  • 右键单击该表，然后从上下文菜单中选择“ 重新加载页面 ”。

  • 按⌘R 。

重置初始表视图

点击设置图标（ ），然后选择重置视图 。结果，数据变得不排序，列按其最初定义的顺序出现，并显示所有列。

查看生成结果集的查询

要查看用于生成结果表的查询 ，请单击工具栏上的“ 查看查询 ”。如有必要，您可以选择查询文本并将其复制到剪贴板⌘C 。

要关闭显示查询的窗格，请按⎋ 。
