创建并保存临时运行/调试配置

每次您从上下文菜单或使用装订线图标运行测试类或方法时，都会创建一个临时配置要么 。当运行或调试会话结束时，新配置在“运行/调试配置”选择器中显示为灰色。

默认情况下，每个项目只能创建五个临时配置，因此当您创建第六个配置时，将删除第一个创建的配置，依此类推。您可以在“ 编辑配置” | “ 临时配置限制”字段中更改此限制。 模板 。

临时配置无法在XCode中查看和编辑。您可以保存一个临时配置，以使其在其他XCode方案中可用。

保存临时运行/调试配置

  • 在“ 运行/调试配置”选择器中，选择“ 保存”。 。

  • 在“ 编辑配置”对话框中，从列表中选择临时配置，然后单击“确定”。 。

  • 从编辑器或项目工具窗口的上下文菜单中，选择“ 保存”。 。
