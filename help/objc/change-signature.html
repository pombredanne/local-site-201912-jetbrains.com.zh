变更签名

更改签名重构结合了可以应用于方法/功能签名的几种不同的修改。您可以使用此重构来：

  • 更改方法/功能名称

  • 更改方法/函数返回类型

  • 添加新参数并删除现有参数

  • 重新排序参数

  • 更改参数名称和类型

更改方法/功能签名时，AppCode会搜索该方法/功能的所有用法，并更新所有可以安全修改以反映更改的方法/功能的调用，实现和替代。

仅Objective-C支持更改签名重构。

    更改方法/功能签名

    1. 将插入符号放在要重构的方法/函数的名称上。

    2. 按⌘F6 。或者，选择Refactor |从主菜单或上下文菜单中更改签名 。

    3. 在“ 更改签名”对话框中，根据需要对方法/功能签名进行必要的更改：

      • 更改可调用过程的类型。在“ 可调用类型”中 ，选择“ 函数” ，“ 方法 ”或“ 块” 。

      • 更改方法/功能名称。要更改名称，请在“ 名称”字段中编辑文本。

      • 通过编辑“ 返回类型”字段的内容来更改方法/函数的返回类型 。

      • 管理方法/功能参数。要配置参数，请使用表格和“ 参数”区域中的按钮：

        • 要添加新参数，请单击并在相应的表格行中指定新参数的属性。

        • 要删除参数，请选择任意行，然后单击 。

        • 要重新排列参数，请使用向上键 （ ）和下 （ ）图标。

    4. 单击重构 。

    5. 要查看预期的更改并在重构之前进行调整，请单击预览 。

    例子

    将函数转换成块

    1. 将插入符号放在isPasswordValid功能并按⌘F6 。

    2. 在打开的对话框中，在“ 可折叠类型”列表中选择“ 阻止 ”。

    3. 单击重构 。

    之前

    后

    //此函数将转换为块BOOL isPasswordValid（NSString * password）{return password.length> 4; }-（void）performLogin {// ...//函数调用if（isPasswordValid（password）== NO）{// ...}}
    -（void）performLogin {// ...//新块if（^ BOOL（NSString * password）{// ...}}

    重命名函数及其参数

    1. 将插入符号放在isPasswordValid功能并按⌘F6 。

    2. 在打开的对话框中，在“ 名称”字段中键入一个新名称。

    3. 单击参数，然后在出现的“ 名称”字段中键入新名称。

    4. 单击重构 。

    之前

    后

    //此函数及其参数//将重命名为BOOL isPasswordValid（NSString * password）{return password.length> 4; }-（void）performLogin {// ...//函数调用if（isPasswordValid（password）== NO）{// ...}}
    //更改后的函数BOOL isIdValid（NSString * id）{return id.length> 4; }-（void）performLogin {// ...//函数调用if（isIdValid（password）== NO）{// ...}}
