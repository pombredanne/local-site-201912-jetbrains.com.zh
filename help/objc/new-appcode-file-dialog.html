新建文件对话框

在“ 新建文件”下拉菜单（“ 文件” |“新建...” ）中选择文件类型后，将出现“ 新建...”对话框。
对话框标题反映了所选的文件类型，例如New Swift Type或New C ++ Class 。
根据文件类型的不同，对话框中字段的数量和类型可能会有所警惕。

项目

描述

名称

在此文本字段中，为要创建的文件指定名称。

夹

在此文本字段中，指定文件的路径。在文本字段中输入或按按钮从对话框中选择。

组

从此下拉列表中，选择与文件关联的组。

目标

选择（或在必要时取消选择）包含正在创建的文件的目标。

类型

选择C ++类源/头文件.mm / .h或.cpp / .h的类型

类

从下拉列表中选择文件类型。
