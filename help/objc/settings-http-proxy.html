HTTP代理

指定AppCode用于访问Internet的代理设置。

项目

描述

没有代理

无需代理即可直接连接。

自动检测代理设置

使用代理自动配置（PAC）文件。

  • 自动代理配置URL ：如果未自动检测到PAC文件，则手动指定其位置。

    如果PAC文件编码为带有BOM的UTF-8，它将无法正常工作。确保编码为无BOM的UTF-8。

  • 清除密码 ：清除指定代理的密码。

手动代理配置

手动指定代理设置。

  • HTTP ：使用HTTP代理。

  • SOCKS ：使用SOCKS代理。有关更多信息，请参见套接字安全协议 。

  • 主机名 ：指定代理主机名或IP地址。

  • 端口号 ：指定代理端口号。

  • 没有代理 ：指定网址或IP地址的格式，不应该为其指定代理。

  • 代理身份验证 ：如果您的代理需要身份验证，请选中此复选框。

  • 登录名 ：指定用于连接代理的用户。

  • 密码 ：指定与用户关联的密码（登录）。

  • 记住密码 ：如果您希望AppCode 记住密码，请选中此复选框。否则，每次AppCode连接到代理时，都会要求您提供密码。
