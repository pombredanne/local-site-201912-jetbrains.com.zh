JavaScript文档查找

AppCode向您显示标准JavaScript API，项目中的符号及其依赖关系以及外部库中定义的符号的参考。您可以在“ 文档”弹出窗口中查看符号的文档 ，获取有关方法参数的信息，并在外部站点上打开文档（如果有指向它们的链接）。

查看插入符号处的文档

  • 按F1或选择查看|从主菜单快速查找文档 。该符号的引用显示在“ 文档”弹出窗口中。

    • 对于项目符号或项目依赖项中的符号，AppCode根据相应的JSDoc注释生成文档。

    • 对于标准JavaScript对象或方法，AppCode从内置TypeScript定义文件 d.ts中显示相应的JSDoc注释。这些文件与AppCode捆绑在一起并定期更新。

    • 如果在d.ts文件中未找到注释，则AppCode将显示相应MDN文章的摘要。

要查看MDN文档中插入符号的信息

  • 在文档弹出窗口中F1 ，请点击底部的链接：

  • 按⇧F1或选择查看|主菜单中的外部文档 。

AppCode在默认的AppCode浏览器中打开MDN文章。

查看第三方JavaScript库的文档

  1. 下载所需的库或框架，将其配置为外部JavaScript库，并指定其外部文档的链接，有关详细信息，请参阅配置自定义第三方JavaScript库 。

  2. 将插入号置于相关符号处，然后按⇧F1或选择查看|主菜单中的外部文档 。

查看npm软件包的文档

  • 在package.json文件或require要么import声明，将插入号放在包装名称下，然后按F1

  • 要在浏览器中打开npm官方网站上的文档，请按⇧F1 。

自动显示文档

  1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |一般|代码完成 。

  2. 在打开的“代码完成”页面上，选中“ 显示文档”弹出复选框，并指定经过时间。

要更改快速文档的字体大小，请执行以下一项操作：

  • 请点击在快速文档窗口的右上角，然后移动滑块。

  • 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |常规，然后选择使用命令+鼠标滚轮更改字体大小（缩放）复选框。有关详细信息，请参见有用的编辑器配置 。
