档案颜色

使用此页面可以设置不同的背景颜色，以区分特定范围的项目文件。设置适用于以下UI元素：

  • 编辑器标签的标题。

  • 当一个人按文件或类的名称搜索时， 导航列表

  • 项目工具窗口

管理范围

单击此按钮以打开“ 范围”对话框，您可以在其中定义各种操作的自定义范围。

加

单击此按钮以打开“ 添加颜色标签”对话框，您可以在其中配置新的色标关联。

去掉

单击此按钮删除选定的色域关联。

要么

向上或向下移动

使用这些按钮对色域关联进行排序，从而确定它们的应用顺序。

分享

单击此按钮可以在团队成员之间共享选定的范围颜色关联。

所选的关联将相应地移动到“ 共享颜色”区域中的列表。

共享颜色

使用此区域中的控件来配置要在团队所有成员之间共享的色域关联。文件颜色作为项目设置存储在.idea / fileColors.xml中 。

定义后，色域关联将无法更改。要将颜色重新分配给合并范围，请删除现有的关联并定义一个新的关联。
