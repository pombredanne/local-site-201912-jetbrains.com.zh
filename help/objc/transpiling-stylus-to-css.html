AppCode 2019.2帮助

触控笔

AppCode使您可以将手写笔代码即时转换为CSS。

在开始之前，请确保您的计算机上装有Node.js。

安装手写笔

  • 在嵌入式终端 （ ⌥F12 ），输入：

    npm install --global stylus

    从Stylus官方网站了解更多信息。

您还可以按照npm和Yarn中的说明 ，在Node.js和NPM页面上安装手写笔软件包。

编写手写笔代码

要动态编译代码，您需要将编译器配置为AppCode File Watcher 。

创建文件监视程序

  1. 在“ 首选项”对话框中（ ⌘, ），点击“ 工具”下的“ 文件监视程序 ”。打开的“ 文件监视程序”页面显示已配置的文件监视程序的列表。

  2. 点击或按⌘N并从列表中选择Stylus预定义模板。

编译代码

当您打开文件时，AppCode会检查当前项目中是否有适用的文件监视程序。如果已配置但禁用了这种文件监视程序，则AppCode会显示一个弹出窗口，通知您有关已配置的文件监视程序并建议启用它。

如果在当前项目中配置并启用了适用的文件监视程序，则AppCode会在“ 新建监视程序”对话框中指定的事件后自动启动它。

  • 如果选中了“ 自动保存编辑的文件以触发监视程序”复选框，则在对源代码进行任何更改后都会立即调用文件监视程序。

  • 如果清除了自动保存编辑的文件以触发监视程序复选框，则文件监视程序会在保存时启动（ File | Save All ， ⌘S ）或将焦点从AppCode移开（框架停用时）。

AppCode使用生成的输出创建一个单独的文件。该文件具有源手写笔文件的名称，扩展名为.css 。在“ 新建监视程序”对话框的“ 刷新的输出路径”字段中定义了生成文件的位置。但是，在项目树中 ，它们显示在源文件下，该文件现在显示为节点。

配置语法突出显示

您可以根据自己的喜好和习惯配置支持Stylus的语法突出显示。

  1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |配色方案|手写笔 。

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
