重构

重构是在不创建新功能的情况下改进源代码的过程。重构可帮助您保持代码的牢固， 干燥和易于维护。

调用重构

  1. 选择一个要重构的项目。您可以在“ 项目”工具窗口中选择文件/文件夹，也可以在编辑器中选择表达式/符号。

  2. 按⌃T打开可以选择的重构列表。或者，您可以使用键盘快捷键进行特定的重构。

如果您需要撤消重构，请按⌘Z 。

预览重构更改

对于某些重构，AppCode允许您在应用更改之前预览更改。

  1. 通过按下调用重构⌃T并从列表中选择所需的重构。

  2. 要查看可能的更改，请在对话框中单击“ 预览 ”。

  3. 在“ 查找工具”窗口中检查将要进行的更改。您可以排除⌦或删除⌘X您认为不必要的更改。

  4. 单击“ 执行重构”以继续更改。

解决冲突

当AppCode遇到重构问题时，它将打开一个对话框，其中列出了冲突列表及其简短说明。

  • 单击继续以忽略该问题，然后继续在“ 查找”工具窗口中打开预览。

  • 单击“在视图中显示冲突”以在“ 查找”工具窗口中打开冲突条目，然后进一步处理它们。

    例如，您可以尝试通过按以下操作从重构中排除条目： ⌦或单击“ 取消”并返回到编辑器。

配置重构设置

  1. 在“ 首选项”对话框中（ ⌘, ），选择编辑器|一般

  2. 在“ 常规”页面上的“ 重构”部分中，调整重构选项，然后单击“ 确定” 。

您还可以在“ 编辑器” |“调整”中调整重构意图。用心 。

AppCode支持的最受欢迎的重构

改名 ⇧F6

重命名元素及其用法。

变更签名 ⌘F6

更改方法的签名。

移动并复制

F6 / F5

移动/复制元素。

安全删除 ⌘⌦

确保您不删除源代码中引用的文件。

提取方法 ⌥⌘M

将代码片段提取到方法中，并帮助您减少代码重复。

提取常数 ⌥⌘C

声明一个新常量，并使用所选表达式对其进行初始化。

提取参数 ⌥⌘P

向方法声明中添加新参数，并相应地更新方法调用。

提取变量 ⌥⌘V

声明一个新变量，并使用所选表达式对其进行初始化。

排队 ⌥⌘N

内联元素。与提取相反。
