创建复合运行/调试配置

假设您想同时启动多个运行/调试配置。例如，您可能要运行几种不同类型的配置或一系列测试配置的序列。您可以使用复合运行/调试配置来配置此行为。

当您使用复合配置运行或调试代码时，实际上会启动一系列配置，按所列顺序包括这些配置。

创建复合运行/调试配置

  1. 打开“ 运行/调试配置”对话框 。您可以从运行|访问编辑配置主菜单或使用工具栏上的运行配置切换器。

  2. 在打开的对话框中，单击并从添加新配置菜单中选择化合物 。

  3. 在名称字段中，指定配置的名称。

  4. 选择通过VCS共享，以使其他团队成员可以使用此运行/调试配置。

  5. 要将新的运行/调试配置包括在复合配置中，请单击并从列表中选择所需的一个。

  6. 如有必要，选择将在其上运行运行/调试配置的设备或模拟器：

    如果为每种配置选择了特定的设备/模拟器，则选择器中将显示“ Multiple selected ”：

    如果您未指定任何设备/模拟器，则默认情况下会选择最低模拟器版本。但是，您可以从选择器中选择所需的设备或模拟器：

  7. 单击应用和确定按钮以完成。
