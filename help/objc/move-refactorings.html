Move and copy

Copy

The Copy refactoring lets you create a copy of a file or directory in a different or the same directory.

  1. Select a file/directory in the Project tool window.

  2. Select Refactor | Copy from the main or context menu or press F5.

  3. In the Copy window, specify the name and location for your copy and click OK.

Move

The Move refactoring lets you move declarations and class members to another file/class and files/directories to another directory.

Move a file/directory

  1. Select a file/directory in the Project tool window.

  2. Select Refactor | Move from the main or context menu or press F6.

  3. In the Move dialog, specify a directory where the selected file will be moved to:

    To have references to the selected file or folder updated according to the refactoring result, select the Search for references checkbox.

    Select Open moved files in editor if you want AppCode to open the files that have been moved.

    Click OK.

Move a declaration/member

  1. Place a caret at the class, protocol, or member you want to move.

  2. Select Refactor | Move from the main or context menu or press F6.

  3. In the invoked dialog, select declarations/members you want to move:

    Click Refactor. If necessary, click Preview to preview the potential changes.
