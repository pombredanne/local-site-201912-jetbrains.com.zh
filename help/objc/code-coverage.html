以代码覆盖率运行测试

代码覆盖率使您可以查看在单元测试期间执行了多少代码，因此您可以了解这些测试的有效性。AppCode使用llvm-cov收集coverage数据。

代码覆盖率结果显示在覆盖工具窗口，在项目视图项目工具窗口 ，并在编辑器。

覆盖范围测量结果包括一个覆盖范围套件 。您可以将新模拟的结果与任何现有套件合并。在这种情况下，如果一条线被至少一个模拟覆盖，则该线将被视为覆盖。

以代码覆盖率运行

在项目中使用代码覆盖率的一般步骤

  1. 指定您要如何处理coverage结果 。

  2. 为目标代码创建测试 。

  3. 使用主菜单中的专用命令运行| 覆盖 运行以Coverage运行 ，或单击以Coverage运行按钮 。

  4. 一旦执行覆盖运行，您就可以执行以下操作：

    • 使用各种覆盖套件 。

    • 查看代码覆盖率数据 。
