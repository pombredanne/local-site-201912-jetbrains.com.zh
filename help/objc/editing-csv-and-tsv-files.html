将CSV和TSV文件编辑为表格

AppCode使您可以将带分隔符的文本文件 （CSV，TSV和其他以分隔符分隔的格式）编辑为表格。如果满足以下条件，则可以使用数据编辑器：

  • 捆绑的数据库工具和SQL插件已启用。有关更多信息，请参阅管理插件 。

  • 文件扩展名与文本文件类型相关联。有关更多信息，请参见设置文件类型关联 。

打开数据编辑器

  1. 在分隔的文本文件中右键单击，然后单击“ 编辑为表格” 。

  2. 在打开的对话框中，指定格式设置，然后单击“ 确定” 。

    该对话框具有两种预定义的格式（CSV和TSV），您可以创建自定义格式。例如，您可能需要用逗号分隔的值和分号作为行分隔符。

数据编辑器会根据您配置的设置在单独的选项卡中打开。如果要使用其他格式设置，请重复上述步骤，然后再次打开数据编辑器。

右键单击任何单元格或列标题，以访问用于修改表的可用命令。

排序数据

  • 单击列标题可在排序方向之间切换：升序，降序和初始未排序状态。

    当您按多列排序时，数字表示排序级别（优先级）。

隐藏栏

  • 右键单击列标题，然后单击“ 隐藏列” 。

    要查看所有列的列表，请右键单击任何列标题，然后单击列列表 。隐藏的列的名称被删除掉。要在列的隐藏状态和显示状态之间切换，请在列表中选择它，然后按Space 。

启用编码协助

  • 右键单击列标题或单个单元格，然后单击“ 编辑为”以选择一种语言，并在修改内容时提供编码帮助。

转置表

  • 右键单击表中的任何单元格，然后单击“ 转置”以切换行和列。

从表中提取数据

如果您需要使用其他表中的数据，AppCode提供了几种复制或保存数据的可能性。

将所选单元格复制到剪贴板

  • 右键单击选定的单元格，然后单击“ 复制”或按⌘C 。

将所有表格数据复制到剪贴板

  • 用鼠标右键单击表中的任何单元格，指向转储数据 ，然后单击到剪贴板 。

将所有表数据保存到文件

  • 用鼠标右键单击表中的任何单元格，指向转储数据 ，然后单击到文件 。

您可以配置如何使用预定义的输出格式，自定义格式和脚本将提取的数据转换为文本。要配置提取数据的输出，请右键单击表中的任何单元格，然后指向Data Extractor 。

在此菜单中，您可以选择输出格式（例如，作为SQL INSERT语句，HTML表或CSV格式）或将数据转换为特定格式的脚本。以下附加选项也可用：

跳过计算列（SQL）
启用以排除具有计算值的列（影响SQL输出格式）。
跳过生成的列（SQL）
启用以排除具有自动增量值的列（影响SQL输出格式）。
添加表定义（SQL）
启用以包括CREATE TABLE语句（影响SQL输出格式）。

将数据导出到数据库

  1. 确保在数据库工具窗口中将数据库添加为数据源 。

  2. 右键单击表中的任何单元格，然后单击“ 导出到数据库” 。

  3. 指定数据库，目标架构（使用导出的数据创建新表）或表（将导出的数据添加到现有表）。

  4. 配置目标表的数据映射和设置。
