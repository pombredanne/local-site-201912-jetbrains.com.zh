结构工具窗口，文件结构弹出窗口

结构工具窗口

此工具窗口显示当前在编辑器中打开并具有焦点的文件的结构，或在“项目”工具窗口中选择的文件的结构。

对于图，此工具窗口显示图预览。

查看|工具窗口|结构体
⌘7

文件结构弹出

此弹出窗口显示文件的结构，当前在编辑器中打开该文件并具有焦点。

导航|档案结构
⌘F12

两种视图都有助于快速浏览文件的结构 。

本节介绍工具窗口标题栏上的按钮以及标题栏上下文菜单上的选项。打开和关闭这些选项，可以隐藏或显示某些类型的元素，并配置它们的显示方式。

标题栏上的按钮在所有语言环境中都是通用的。上下文菜单上的选项集取决于上下文。

  • 标题栏
  • Objective-C，C ++
  • HTML，XML
  • JavaScript，TypeScript，CoffeeScript

有关“ 结构”工具窗口中文件树视图中成员图标的信息，请参阅“ 图标参考 ”一节。

显示继承

单击此按钮可显示类继承的所有方法和字段，并可从该类访问。继承的成员显示为灰色，以区别于当前类中定义的成员。

按类型排序

单击此按钮可以按照以下分组顺序按元素类型对元素进行排序：

  • C ++结构

  • Objective-C类

  • 构造函数

  • 破坏者

  • 功能/方法

  • 属性

  • 领域

  • 类型定义（typedef）

  • 全局变量

  • 指令（#define）。

按字母顺序排序
