Breadcrumbs

Use this page to configure editor breadcrumbs for the supported languages.

Item

Description

Show breadcrumbs

When this checkbox is selected, the path from the root element in the current document to the code element where your caret is will be shown in the editor:

Placement

Choose whether you want to have breadcrumbs at the top or at the bottom of the editor. The default option is Bottom.

Languages

Select the checkboxes next to the language contexts where you want to have breadcrumbs.

Manage colors

Click this link to open Color Scheme page and configure colors for breadcrumbs.
