调试工具窗口。变数

在本主题中：

  • 工具列
  • 上下文菜单
  • 变量类型

通过“ 变量”窗格，您可以检查存储在应用程序对象中的值。

在“ 框架”窗格中选择堆栈框架后，“ 变量”窗格将显示其范围内的所有数据（方法参数，局部变量和实例变量）。在此窗格中，您可以设置对象的标签，检查对象，评估表达式，将变量添加到监视等等。

工具列

仅当隐藏“ 监视”窗格时，才会显示此工具栏，以便在“ 变量”窗格中显示已配置的监视 。隐藏/显示“ 手表”窗格是通过切换按钮：

  • 按下按钮（默认状态）后，“ 监视”窗格被隐藏，工具栏显示在“ 变量”窗格中。重点在于“ 调试器”选项卡 。

  • 释放按钮后，工具栏将移至“ 监视”窗格。

项目

捷径

描述

Insert

单击此按钮创建新手表。

⌦

单击此按钮可从列表中删除选定的手表。

⌃↑⌃↓

使用这些按钮可以更改手表的顺序。

⌘D

使用此按钮可以创建所选手表的副本。

在变量标签中显示手表

使用此切换按钮可以隐藏或显示“ 监视”窗格。默认情况下，该按钮被按下并显示在“ 变量”窗格的工具栏上。因此，“ 监视”窗格被隐藏，并且监视在“ 变量”窗格中显示 。

  • 要单独显示“ 监视”窗格并在其中查看配置的监视，请释放“ 在变量中显示监视”选项卡切换按钮。出现“ 监视”窗格，并在工具栏上显示“在变量中显示监视”选项卡切换按钮。

  • 要隐藏“ 监视”窗格并在“ 变量”窗格中查看监视，请按 监视窗格的工具栏上的切换按钮。切换按钮返回到“ 变量”窗格工具栏上的默认位置。

上下文菜单

项目

捷径

描述

检查

不适用

该命令可用于字段，局部变量和引用表达式，并打开一个非模态“ 检查”窗口，您可以在其中专注于特定引用。您可以根据需要打开任意多个“ 检查”窗口。“ 检查”窗口中的视图与“ 监视”窗格中的视图相同，但所需的屏幕空间较小。

设定值

F2

使用此命令可以更改字段或变量的运行时值。

复制值

⌘C

使用此命令将所选变量的值复制到剪贴板。如果选择了多个项目，则不仅会复制变量的值，还会复制变量的结构，因此，当您将所选内容复制粘贴到文本文件时，缩进将模仿调试器的树输出以产生易于阅读的内容输出。

或者，将鼠标光标悬停在一个值上，然后在工具提示中查看其内容。

复制JSON

该菜单项仅在JavaScript上下文中可用。选择此命令以JSON格式复制所选值。

比较价值与剪贴板

不适用

使用此命令将所选值与剪贴板中当前的值进行比较。

复制名称

不适用

使用此命令将所选变量的名称复制到剪贴板。

评估表达

⌥F8

使用此命令可以在打开的对话框中评估所选变量 。

添加到手表

不适用

该命令可用于除静态节点以外的所有节点。使用此命令来创建引用该节点的表达式，并将该表达式添加到“ 监视”窗格中。

显示参照对象

不适用

使用此命令可显示引用当前所选变量的对象列表。

跳转到源

⌘↓

此命令在编辑器中打开所选变量或字段的源代码，并将插入号放置在相应的行。

跳转到类型源

⇧F4

使用此命令导航到所选变量或字段的类的定义。

变量类型

每个变量左侧的图标指示其类型：

  • ： 静态的

  • ：领域

  • ：数组

  • ：原始

  • ：对象

  • ：参数
