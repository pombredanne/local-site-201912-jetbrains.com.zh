从命令行使用AppCode

除了在AppCode中工作之外，您还可以从命令行执行以下操作：

  • 在命令行中查看文件之间的差异
  • 从命令行合并文件
  • 从命令行格式化文件

在Linux和macOS上启用命令行启动器

在macOS和Linux上，创建命令行启动程序以将AppCode与Shell集成。确保创建的启动器脚本在外壳程序的搜索路径内，以便可以在系统范围内调用它。

  1. 在主菜单中，选择“ 工具” |“工具”。创建命令行启动器 。在“ 创建启动器脚本”对话框中，提供启动器的名称和路径，或接受建议的值。

  2. 在AppCode之外，将启动脚本的路径和名称添加到您的path 。

从命令行启动AppCode

  • 要从命令行启动AppCode，请键入在命令行中输入命令，然后按⏎ 。 是特定于平台的启动器。
