创建文件，组和文件夹引用

您可以将各种格式的文件添加到项目中，并按组进行组织 。使用可用的AppCode或Xcode模板即可轻松创建文件。

如有必要，您还可以添加空文件 ，创建新的文件模板和文件类型 。

建立档案

AppCode提供了它支持的大多数语言的 文件模板 。这使您可以使用适合文件目的的初始内容来创建文件。例如，存在用于Swift类型和Objective-C类的文件模板，JavaScript文件，XSLT样式表等。此外，您可以使用所有默认的Xcode模板在AppCode中创建文件。

通常，基于模板的文件的文件扩展名是自动设置的，因此您无需指定它。但是，如果文件类型允许多个扩展名，则可以选择一个。例如，当创建一个新类时，您将可以在Objective-C类的.m扩展名和Objective-C ++类的.mm扩展名之间进行选择。

从AppCode模板创建文件

  1. 请执行以下任一操作：

    • 在项目工具窗口中⌘1 ，选择要在其中创建新文件的组，然后选择“ 文件” |“文件”。主菜单上的新功能。

    • 右键单击相应的组，然后从上下文菜单中选择“ 新建 ”。

    • 按⌘N 。

  2. 从列表中选择所需的文件类型。

    如果AppCode的扩展名未知，则该列表中可能缺少现有的自定义文件模板。在这种情况下， 请将扩展名与识别的文件类型之一相关联 。

  3. 在打开的对话框中，在相应的字段中键入文件名（不带扩展名）。

    根据选择的模板，对话框包含各种选项。例如，对于代码文件，您可以选择位置，组和目标 。

    根据需要指定其他信息。例如，如果相应的文件模板包含自定义变量的值并且当前未设置其值，则可能会要求您定义自定义变量的值。

从Xcode模板创建文件

  1. 在主菜单中，选择“ 文件” |“文件”。新品| Xcode模板中的文件 。或者，在“ 项目”工具窗口中⌘1右键单击一个组，然后从Xcode Template中选择File 。

  2. 在打开的对话框中，在左窗格中选择一种文件类型。在右侧窗格中，选择一个模板：

    单击下一步继续。

  3. 指定新文件的文件名和位置。您可以使用在“ 位置”字段中选择的默认位置，也可以指定另一个位置 。选择所需的组和目标，然后单击完成 。

创建一个空文件

  1. 请执行以下任一操作：

    • 在项目工具窗口中⌘1 ，选择要在其中创建新文件的组，然后选择“ 文件” |“文件”。主菜单上的新功能。

    • 右键单击相应的组，然后从上下文菜单中选择“ 新建 ”。

    • 按⌘N 。

  2. 在“ 创建新文件”对话框中，键入文件名和扩展名。（可选）您可以更改文件在磁盘上的位置，将文件添加到的组以及目标 。

    如果您指定的扩展名没有与AppCode识别的任何文件类型相关联，则当您尝试打开此文件时，将出现“ 注册新文件类型关联”对话框。在此对话框中，您可以将扩展名与识别的文件类型之一相关联 。

    单击确定 。

建立群组

组是Xcode项目模型的概念。使用AppCode创建的组与Xcode完全兼容。

组允许将顺序带到项目文件中，这是它们的唯一目的。您可以按照自己喜欢的任何方式安排项目项目。例如，您可以在单独的组中组织Model，View和Controller类，或为测试创建组层次结构。

在“ 项目”工具窗口中显示一个组⌘1并在导航栏中带有文件夹图标 （如果有基础文件夹）或带有浅色文件夹图标 （如果没有基础文件夹）。

建立群组

  1. 在项目工具窗口中⌘1 ，选择目标目录节点。

  2. 右键单击所选内容或按⌘N调用上下文菜单，然后选择“ 新建|组 。

  3. 在“ 新建组”对话框中，指定组名。默认情况下，AppCode为组创建基础目录，以使组和目录结构保持同步。如有必要，您可以创建一个没有此目录的组。为此，请清除“ 创建文件夹”复选框。

    为了更好地了解Xcode / AppCode项目中的组和目录之间的区别，请参阅本文 。

  4. 单击确定 。该组一旦出现在“ 项目”工具窗口中，就可以将文件和组拖入其中。

通过选择创建组

  1. 在项目工具窗口中⌘1 ，通过单击选择几个项目（文件和/或组） ⌘按下。

  2. 右键单击所选内容或按⌘N调用上下文菜单，然后选择“ 新建|从选择中分组 。请注意，所有选择的项目都应在项目层次结构的同一级别上。

  3. 在“ 新建组”对话框中，指定组名，然后单击“ 确定” 。

创建文件夹引用

与AppCode中的组相反， 文件夹引用是物理结构。文件夹引用使您可以将文件组织到文件系统上的文件夹中，并在AppCode中保持相同的文件夹结构。因此，您可以从IDE外部管理资产文件。

使用该文件夹作为不需要编译的项目资源（例如，图标）的存储位置。IDE不会将此类文件夹中的源文件视为项目的一部分。

创建文件夹参考

  1. 在项目工具窗口中⌘1 ，选择所需的组。

  2. 右键单击所选内容或按⌘N调用上下文菜单，然后选择“ 新建|文件夹参考 。

  3. 在“ 新建文件夹引用”对话框中，指定文件夹名称。如果需要，您还可以在此处更改组和目标。

  4. 单击“ 确定”创建文件夹引用和相应的物理文件夹。
