使用制表符

  • 如果选中此复选框，则使用制表符：

    • 在按⇥键

    • 缩进

    • 用于代码重新格式化

  • 清除复选框后，AppCode将使用空格而不是选项卡。

智能标签

  • 如果选中此复选框，则由代码块嵌套定义的缩进部分由制表符和（如有必要）空格组成，而由对齐方式定义的缩进部分仅由空格组成。

  • 如果清除此复选框，则仅使用选项卡。这意味着将将一组适合指定制表符大小的空格自动替换为一个制表符，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中包含的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组元素之间插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。
