JavaScript Postfix模板

后缀代码完成功能使您可以在刚刚键入的表达式周围添加模板代码。当您在圆点后输入缩写（后缀）并按扩展键（ ⇥默认情况下），或者在代码完成弹出窗口中选择缩写时。例如， .if应用于表达式的后缀将其用if声明。

AppCode附带了一组预定义的后缀模板，可让您为JavaScript和TypeScript定义自己的自定义模板。请注意，JavaScript预定义模板也可以在TypeScript文件中使用。

可以复制，更新和删除自定义模板。对于预定义的模板，您只能更改其后缀，例如，将长键替换为较短的后缀。

激活后缀完成

  1. 在“ 首选项”对话框中（ ⌘, ），单击“ 编辑器”下的“ 常规 ” ，然后单击“ 后缀完成” 。

  2. 将打开“ 后缀完成”页面，显示可用后缀的列表以及其旁边的相应模板。当您选择一个后缀时，“ 描述”窗格将说明相应的转换，该转换显示扩展模板前后的代码片段。

  3. 选中启用后缀模板复选框，然后选中您需要激活的后缀旁边的复选框。

应用后缀模板

  1. 键入您的表达式后跟一个点。

  2. 输入后缀，然后按扩展键（默认情况下， ⇥ ），或从建议列表中选择后缀。如有必要，请选择要包围或替换的表达式。

    所有后缀模板的默认扩展密钥是⇥ 。请参阅更改默认扩展键以了解如何选择其他扩展键 。

创建一个自定义模板

  1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |一般|后缀完成 。

  2. 在打开的“ 后缀完成”页面上，单击 ，然后选择模板可以使用的语言。您可以选择JavaScript或TypeScript，JavaScript模板也可以在TypeScript文件中使用。

  3. 在打开的“ 创建新模板”对话框中，键入模板后缀，选择语言级别，键入模板代码，然后添加$EXPR$在您需要插入初始表达式的地方。加$END$您希望光标位于末尾的位置。

您还可以创建一个新模板，该模板与现有模板略有不同，例如，其语言环境或光标的最终位置。AppCode可让您复制原始模板并在该副本中进行必要的更改。

从现有模板创建新的自定义模板

  1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |一般|后缀完成 。

  2. 在打开的“ Postfix完成”页面上，选择要从中创建新模板的自定义模板，然后单击在工具栏上。

  3. 在打开的“ 编辑模板”对话框中，根据需要编辑模板，然后键入新的后缀。

更改默认扩展键

  • 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |一般| Postfix完成 ，然后从扩展模板范围列表中选择一个新密钥。

禁用后缀完成

  • 要禁止扩展所有已配置的后缀模板，请清除“ 启用后缀模板”复选框。

  • 要禁止扩展特定模板，请清除其后缀旁边的复选框。
