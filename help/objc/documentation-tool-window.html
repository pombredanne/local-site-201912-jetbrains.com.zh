文档工具窗口

查看|工具窗口|文献资料
F1

在以下情况下，将显示此工具窗口：

  • 通过单击可固定快速文档查找 在查找窗口的右上角。

  • 通过单击固定快速定义工具提示 在工具提示的右上角。

结果，工具窗口出现在“ 视图”菜单中的可用工具窗口列表中。

图标

捷径

描述

 

←要么→

切换到上一个或下一个文档页面（例如，在使用超链接之后）。

您也可以使用三指从右到左和从左到右的滑动手势。

⌘↓

切换到与当前显示的文档页面相对应的项目（例如，来源）。

单击此图标执行以下设置：

  • 显示字体大小滑块。根据需要移动滑块以在快速文档窗口中增加或减小字体大小。

  • 从源启用或禁用文档自动更新。

  • 控制窗口的位置。
