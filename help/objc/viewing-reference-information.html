代码参考信息

本节介绍如何查看符号的定义，显示文档参考以及如何使用视图参数信息功能。

    定义

    在AppCode中，您可以查看项目中在何处以及如何定义符号，例如标签，类，字段，方法或函数。为此，IDE具有“ 快速定义”弹出窗口。

    • 要查看符号的定义，请在编辑器中选择它，然后按⌥Space （或单击查看|快速定义 ）。

    或者，使用⌘按下键，将光标悬停在任何符号上。AppCode将符号显示为链接，并在工具提示中显示其定义。单击此链接跳转到符号的定义。

    点击右上角的图标在“ 查找工具”窗口中打开定义。要打开定义的源代码进行编辑并关闭弹出窗口，请单击图标⌘↓ 。要打开定义而不关闭弹出窗口，请单击⌘⏎ 。

    参数信息

    “ 参数信息”弹出窗口显示方法和函数调用中的参数名称。在编辑器中输入尖括号或从建议列表中选择方法后，AppCode会在1秒（1000毫秒）内自动显示一个弹出窗口，其中包含所有可用的方法签名。

    如果弹出窗口已关闭，或者将IDE 配置为不自动显示弹出窗口，则可以显式调用弹出窗口 。为此，请按⌘P （或单击查看|参数信息 ）。

    配置参数信息弹出窗口

    1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |一般|代码完成 。

    2. 在“ 以...毫秒为单位显示参数信息弹出窗口”字段中，以毫秒为单位指定弹出窗口应出现的时间。

    如果您不希望弹出窗口自动显示，请清除“ 以...毫秒为单位显示参数信息弹出窗口”复选框。

    快速文档

    您可以通过“ 快速文档”弹出窗口获取任何符号或方法签名的快速信息 。请注意，该符号必须随附根据AppleDoc / HeaderDoc / Doxygen标记创建的文档注释，或SDK中的文档。

    • 要查看插入符号处的文档，请按F1 （或单击查看|快速文档 ）。

      你可以按F1再次在弹出窗口和工具栏之间切换。

    点击图标以更改字体大小，显示快速文档工具栏或转到源代码。

    自动显示快速文档

    1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |一般

    2. 在“ 其他”区域中，选中“ 在鼠标移动时显示快速文档”复选框，然后指定延迟（以毫秒为单位），之后应显示弹出窗口。

    外部文件

    外部文档会在Web浏览器中打开必要的信息，以便您可以导航到相关符号并同时保留信息以备将来参考。

    AppCode显示Foundation Framework Classs的文档。

    查看外部文档

    • 要在Web浏览器中的当前插入符号位置查看符号的文档，请按⇧F1或选择查看|主菜单中的外部文档 。

    选择浏览器以获取外部文档

    1. 在“ 首选项”对话框中（ ⌘, ），转到“ 编辑器” |外部文件 。

    2. 选择默认Web浏览器或Dash 。

    上下文信息

    如果当前方法或类声明不可见，则可以通过按以下工具提示来查看它： ⌃⇧Q 。
