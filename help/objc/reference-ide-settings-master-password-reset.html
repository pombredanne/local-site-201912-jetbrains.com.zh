Master Password / Reset Master Password Dialog

File | Settings | Passwords - Reset Master Password for Windows and Linux
AppCode | Preferences | Passwords - Reset Master Password for macOS
⌘,

The dialog opens when you click the Reset Master Password button on the Passwords page of the Settings Preferences dialog, or the Reset Password button in the Change Master Password dialog box.

In this dialog, specify or reset the master password for accessing the database where passwords for remote repositories are stored.

When you reset the master password, AppCode clears the history of all the previously existed master passwords.

Item

Description

New Password

In this field, type the master password to use.

Confirm New Password

In this field, repeat the master password to use.

Set Password

Click this button to have AppCode save and use the specified master password.The button is available in the Master Password dialog, which opens when you set the master password for the first time.
