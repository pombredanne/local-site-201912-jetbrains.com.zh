查看代码覆盖率结果

查看代码覆盖率可帮助您检测不受仿真影响的部分源代码。

查看代码覆盖率结果

  1. 创建一个测试运行/调试配置 ，在“ 运行/调试配置”列表中选择它，然后单击“覆盖运行”按钮 。

  2. 点击运行|显示代码覆盖率数据 。或者，按⌥⌘F6 。

  3. 查看覆盖范围结果：

    • 在项目工具窗口中：

    • 在专用的Coverage工具窗口中 ：

  4. 在编辑器中打开要浏览的文件。

  5. 使用左侧装订线中的颜色指示器检测未发现的代码行。

    要找出一条线被击中了多少次，请在装订线区域中单击该线。
