View code coverage results

Viewing code coverage helps you detect pieces of your source code that are not affected by simulation.

View code coverage results

  1. Create a testing run/debug configuration, select it in the Run/Debug Configurations list, and click the Run with Coverage button .

  2. Click Run | Show Code Coverage Data. Alternatively, press ⌥⌘F6.

  3. View coverage results:

    • In the Project tool window:

    • In the dedicated Coverage tool window:

  4. Open in the editor the files you want to explore.

  5. Use the color indicators in the left gutter to detect the uncovered lines of code.

    To find out how many times a line has been hit, click the line in the gutter area.
