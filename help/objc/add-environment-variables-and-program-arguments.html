添加环境变量和程序参数

通过将程序参数和环境变量添加到运行/调试配置中，可以影响应用程序的运行时行为。

您可以在“ 运行/调试配置”对话框的相应字段中指定程序参数和环境变量。要打开对话框，请从主菜单中选择“运行” |“运行”。编辑配置或从运行/调试工具栏上的选择器中选择“ 编辑配置” 。

添加程序参数

  1. 在“ 运行/调试配置”对话框中，选择要为其传递参数的配置。

  2. 在“ 程序参数”字段中输入参数 。参数应以空格分隔。

    请点击扩展文本字段，以便您可以查看和编辑整个参数列表。

添加环境变量

  1. 在“ 运行/调试配置”对话框中，选择要为其添加环境变量的配置。

  2. 输入变量名称和值： = 。如果添加多个变量，则应使用分号将它们分开。

    或者，单击并将变量名称和值添加到“ 用户环境变量”列表中。

    在打开的对话框中，您还可以看到可用系统环境变量的列表。如果不想将系统环境变量用于选定的运行/调试配置，请清除“ 包括系统环境变量”复选框。
