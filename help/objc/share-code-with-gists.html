与要点共享代码

要点让您共享代码片段，整个文件甚至应用程序。您还可以在运行，调试或测试代码时使用摘要来保存和共享控制台输出。

每个要点都是一个可以被其他人克隆或分叉的存储库。

创建要点

  1. 在编辑器中选择一个代码片段，或在“ 项目”工具窗口中选择文件和文件夹。要将控制台输出保存到要点，请右键单击工具窗口或包含该输出的选项卡中的任何位置。

  2. 在选择的上下文菜单上，选择创建要点 。

  3. 在“ 创建要点”对话框中，在“ 文件名”下指定要点的名称 ，然后输入要发布的代码的描述。

  4. 选择“ 秘密”选项以创建秘密要点。秘密要点不可搜索，也不会显示在Discover中 。它们只能用于您自己的目的，因为您不能共享它们。

    如果要创建公共要点，请确保取消选中此选项。可以搜索公共要点，它们显示在“ 发现”中 ，人们可以在其中浏览新出现的要点。如果您希望其他人能够找到并查看您的代码，请使用公共摘要。

    一旦创建了要点，就不能将其从秘密转换为公开，反之亦然。

  5. 如果要在默认浏览器中打开新创建的要点，请选择“ 在浏览器中打开”选项，然后单击“ 确定” 。
