AppCode 2019.2帮助

首次运行AppCode

首次运行AppCode或从早期版本升级AppCode后，需要一些步骤才能完成安装，自定义实例并开始使用IDE。

导入AppCode设置

在“从以下位置导入AppCode设置”对话框中，选择是否要导入IDE设置：

如果这是您的AppCode的第一个实例，请选择“不导入设置”选项。

您可以稍后使用文件|手动导出和导入设置 。导入设置和文件|从主菜单中导出设置命令。

选择用户界面主题

如果要使用浅色或深色界面主题以及AppCode默认键盘映射，请选择默认或Darcula选项。如果要设置类似于Xcode默认首选项的界面主题和键盘映射，请选择Xcode 。

禁用不必要的插件

AppCode包括可与不同版本控制系统和应用程序服务器集成的插件，添加了对各种框架和开发技术的支持，等等。为了提高性能，您可以禁用不需要的插件。如有必要，您可以稍后在“ 首选项”对话框中启用它们⌘,在插件下。有关更多信息，请参阅管理插件 。

您可以单击每个插件组的“ 全部禁用”链接以全部禁用它们，或单击“ 自定义”以禁用该组插件中的单个插件。

下载并安装其他插件

如有必要，请从AppCode插件存储库下载并安装未与IDE捆绑在一起的其他插件 。

在AppCode中启动项目

在欢迎屏幕上，您可以执行以下操作：

  • 创建一个新项目
  • 打开一个现有的项目或文件
  • 从版本控制系统中检出现有项目

也可以将现有项目目录或单独的文件拖到“欢迎使用”屏幕，然后在AppCode中打开它。
