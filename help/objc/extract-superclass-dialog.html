Extract Superclass Dialog

Refactor | Extract Superclass

Item

Description

Superclass name

In this field, type the name for the new superclass.

Package for new superclass

In this list, specify the package for the new superclass. If necessary, click Browse and choose the target package in the dialog that opens.

Members to Form Superclass

In this area, specify the members of the class to be moved or delegated to the new superclass. To have an element included in the interface, select the checkbox next to it.
