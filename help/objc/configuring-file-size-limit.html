最大档案长度

默认情况下，AppCode中的Objective-C / C / C ++源/头文件最大长度限制为500 000个字符。如果超过了该限制，则将禁用所有代码洞察功能，例如代码完成和语法突出显示。在这种情况下，IDE会通知您当前配置的限制和实际文件长度。

在最大文件长度对话框中更改文件大小限制

  1. 单击IDE通知右上角的链接：

  2. 在打开的对话框中，指定文件长度限制，然后按OK继续。

请注意，IDE会在该操作后重建项目的符号。

使用注册表更改文件大小限制

  1. 在主菜单中，选择“ 帮助” |“选择” 。查找动作或按⇧⌘A 。

  2. 在打开的对话框中，键入注册表 ，然后按⏎ 。

  3. 在出现的对话框中，开始输入cidr.max.intellisense.file.length 。在“ 值”字段中指定所需的文件大小限制。单击关闭继续。

    在这种情况下，您需要通过调用文件|手动使项目符号无效。使缓存无效|重新启动 。
