尽可能移动为枚举常量

移动常量时，此选项很有用（ static final字段）到enum在以下情况下输入enum type具有一个带有适当类型的一个参数的构造函数。

说，我们在移动MOUSE_EVENT从班上Events 。

class Events {public static final String MOUSE_EVENT =“ mouseEvent”; }

到枚举ActionType

枚举ActionType {;字符串typeName; ActionType（字符串名称）{typeName =名称； }}

如果启用该选项，我们将得到以下结果：

枚举ActionType {MOUSE_EVENT（“ mouseEvent”）;字符串typeName; ...}

如果关闭该选项，则结果将是：

枚举ActionType {;公共静态最终字符串MOUSE_EVENT =“ mouseEvent”;字符串typeName; ...}
