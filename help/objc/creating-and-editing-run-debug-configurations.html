创建和编辑运行/调试配置

您可以从导航栏（“ 查看” |“外观” |“导航栏” ）上的选择器访问可用的运行/调试配置：

滚动到列表底部以找到以下配置：

AppCode中的运行配置与Xcode方案等效，并且与Xcode方案保持同步。

AppCode提供“ 运行/调试配置”对话框作为处理运行/调试配置的工具：创建配置文件或更改默认配置文件。

创建运行/调试配置

  1. 打开“ 运行/调试配置”对话框：

    • 选择运行|从主菜单编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按⌃⌥R ，然后按0或从弹出窗口中选择配置，然后按⌘↓ 。

  2. 在“ 运行/调试配置”对话框中，单击在工具栏上或按⌘N 。该列表显示了默认的运行/调试配置。选择所需的配置类型。

    右侧窗格中显示的字段显示所选配置类型的默认设置。

  3. 在“ 配置”选项卡上填写以下字段：

    • 名称 ：输入配置名称。

    • 目标 ：从列表中选择所需的目标 。

    • 配置 ：选择配置类型- 调试或发布 。

    • 可执行文件 ：为您的配置选择适当的可执行文件。

    如有必要，添加程序参数和环境变量 。

    根据配置类型指定其他参数。有关可用选项的更多信息，请参考“ 运行/调试配置对话框”部分。

  4. 在启动前部分中，定义要在运行之前启动任何外部工具还是其他运行/调试配置。要跳过构建阶段，请从启动前列表中删除“ 构建” 。

    将运行/调试配置添加到启动前列表中时，可以选择将在其上运行的设备或模拟器：

    1. 请点击然后选择“运行其他配置” 。

    2. 从打开的列表中选择配置以及用于它的设备或模拟器。

  5. 应用更改并关闭对话框。

编辑现有的运行/调试配置

  1. 请执行以下任一操作：

    • 从主菜单中，选择运行|编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按⌃⌥R ，然后按0以显示“ 编辑配置”对话框，或从弹出窗口中选择配置，然后按⌘↓ 。

  2. 在相应的运行/调试配置对话框中，根据需要更改参数。

如果在早期版本的AppCode中创建了一个项目，则将这样的项目添加到已在同一窗口中打开的另一个项目时，其运行/调试配置可能会丢失。

为避免丢失运行/调试配置，建议您以最新版本打开一次这样的项目，并且仅在将其添加到另一个项目之后再打开。

启用服务工具窗口

您可以在专用工具窗口中一次管理多个运行/调试配置。例如，您可以启动，暂停和停止几个应用程序，跟踪它们的状态，并检查特定于应用程序的详细信息。

  1. 从主菜单中，选择运行|编辑配置或使用工具栏上的选择器。

  2. 在“ 运行/调试配置”对话框中，从左侧列表中选择“ 模板 ”。

  3. 在右侧窗格中，展开“ 服务”部分中的“ 可用配置” 。

  4. 请点击在“ 服务”中的“ 可用配置”部分中，选择所需的运行配置类型。

    您可以一一添加或删除多种配置类型。

    请注意，工具窗口将仅显示已为其创建一个或多个配置的配置类型。

    因此，如果添加项目中不存在任何配置的配置类型，则在创建该类型的配置之前，该类型将不会显示在工具窗口中。

  5. 应用更改并关闭对话框。

    所选配置将在“ 服务”工具窗口上可用。要打开此工具窗口，请选择查看|工具窗口|主菜单中的服务 。
