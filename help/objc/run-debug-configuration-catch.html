运行/调试配置：捕获

使用此页面可以为Catch测试（一种C ++测试框架）配置运行/调试选项。

在这里，您可以找到所有运行/调试配置通用的选项说明。

配置板

项目

描述

测试种类

  • 选择标签/测试单选按钮以针对特定标签或所有标签运行测试。

  • 选择“ 模式”单选按钮以针对特定模式运行所有测试。

图案

仅当选择“ 图案”单选按钮时，此选项才可用。
要运行特定的测试，请指定其名称，并用逗号分隔：
仅此测试和此测试
模式字段中的上一行指定了要运行的两个测试。

如果逗号是文件名的一部分，请使用斜杠符号对其进行编码。

要在文件中运行所有测试，请指定文件名，在文件名前加一个井号，并用方括号括起来：
[#basic_check]
有关模式语法的详细信息，请参阅原始文档 。

标签

在此文本框中，指定测试用例的标签或将其保留为空白以使用所有标签。
