Change Master Password Dialog

AppCode | Preferences | Passwords - Change Master Password
⌘,

The dialog opens when you click the Change Master Password button on the Passwords page of the Settings Preferences dialog. In this dialog, update the master password for accessing the database where passwords for remote repositories are stored.

AppCode saves the history of updates of the master password, until you reset it.

Item

Description

Old Password

In this field, type the current master password.

New Password

In this field, type the master password to use.

Confirm New Password

In this field, repeat the master password to use.

Change Password

Click this button to have AppCode use the specified master password and add the previous password to the history.

Reset Password

Click this button to open the Reset Master Password dialog, where you can set a new master password and clear the history of previous master passwords.
