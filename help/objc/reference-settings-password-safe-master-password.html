更改主密码对话框

AppCode |首选项|密码-更改主密码
⌘,

当您单击“ 设置 首选项”对话框的“ 密码”页面上的“ 更改主密码”按钮时，该对话框打开。在此对话框中，更新用于访问存储远程存储库密码的数据库的主密码 。

AppCode会保存主密码的更新历史记录，直到您重置它为止。

项目

描述

旧密码

在此字段中，输入当前的主密码 。

新密码

在此字段中，输入要使用的主密码 。

确认新密码

在此字段中，重复使用主密码 。

更改密码

单击此按钮可使AppCode使用指定的主密码 ，并将以前的密码添加到历史记录中。

重设密码

单击此按钮以打开“ 重置主密码”对话框，您可以在其中设置新的主密码并清除以前的主密码的历史记录。
