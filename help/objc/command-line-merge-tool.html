从命令行合并文件

您可以将AppCode用作用于合并文件的命令行工具。

通常，您需要合并同一文件的三个版本：本地版本，存储库或上游版本，以及基本修订版，这是两个不同版本的来源。

  • 在命令行中，运行merge使用以下语法的命令：

    merge path1 path2 path3 output

也可以仅将三个参数传递给merge工具。在这种情况下， output将作为基本修订。

  • 在命令行中，运行merge使用以下语法的命令：

    合并path1 path2输出

所选文件将在“ 合并”对话框中打开，使您可以分别接受或放弃每个更改。

命令选项如下：

选项

描述

特定于平台的AppCode启动器。

path1

文件本地副本的路径。

path2

文件存储库版本的路径。

path3

（可选）基本修订的路径path1和path2 。

output

用于保存合并结果的文件的路径。

例子

句法
％prod％.exe合并path1 path2 path3输出
例
％prod％.exe合并C：\ MyProject \ Readme.md.bak C：\ MyProject \ Readme.md C：\ Archive \ Readme.md C：\ NewProject \ Readme.md
句法
％prod％合并path1 path2 path3输出
例
％prod％merge〜/ MyProject / Readme.md.bak〜/ MyProject / Readme.md〜/ Archive / Readme.md〜/ NewProject / Readme.md
