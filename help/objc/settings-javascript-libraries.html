从此下拉列表中，选择要配置为库的框架。

版

在此文本框中，指定要使用的所选框架的版本。

能见度

在此区域中，指定希望库可用于与文件和文件夹关联的位置。可用的选项有：

  • 当前项目：选择此选项后，库只能与当前项目中的文件和文件夹关联。如果以后尝试将框架与另一个项目一起使用，则必须重新配置该库。

  • 全局：选择此选项可将库与您的任何AppCode项目关联。
