重新格式化文件对话框

⌥⇧⌘L

按下时出现对话框⌥⇧⌘L在当前文件的编辑器中。如果选择代码|从主菜单重新设置代码格式或按⌥⌘L ，AppCode尝试自动重新格式化指定范围的源代码。

有关如何重新格式化或排除文件格式的更多详细信息，请参阅“重新格式化并重新排列代码” 。

项目

描述

仅VCS更改了文字

如果选中此复选框，则重新格式化将仅应用于在本地已更改但尚未签入到存储库中的文件。
此复选框仅适用于受版本控制的文件。

所选文字

选择此选项可重新格式化当前选定的源代码片段。

整个档案

选择此选项可重新格式化当前文件中的所有源代码。

优化进口

选中此复选框可从所选范围内的代码中删除未使用的导入语句。

代码清理

选择此选项可运行代码清除检查。AppCode查找并自动修复可能有问题的代码片段。

您可以定义规则并在编写代码时格式化代码。要为Objective-C定义格式设置规则， 打开“ 偏好设置”对话框（ ⌘, ）并导航到“编辑器” |代码样式|目标C。有关为Objective-C配置代码样式的更多信息，请参见配置SQL代码样式 。
