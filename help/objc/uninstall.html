AppCode 2019.2 Help

Uninstall AppCode

The proper way to remove AppCode depends on the method you used to install it.

Toolbox App

If you installed AppCode by using the Toolbox App:

  • Open the Toolbox App, click the screw nut icon for the necessary instance, and select Uninstall.

Standalone instance

To remove a standalone AppCode instance, use the standard procedure for uninstalling applications on your operating system and remove the IDE configuration and system directories.

  1. Delete the application in the Applications directory.

  2. Remove the following directories:

    Syntax
    ~/Library/Preferences/<product><version>
    Example
    ~/Library/Preferences/AppCode2019.2
    Syntax
    ~/Library/Caches/<product><version>
    Example
    ~/Library/Caches/AppCode2019.2
