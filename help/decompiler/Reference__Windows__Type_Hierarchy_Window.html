层次结构窗口

Windows |层次结构
检查层次结构

“ 层次结构”窗口可以显示代码的几种类型的层次结构：

  • 类型层次结构
  • 类型依赖图
  • 装配依赖图
  • 参考层次
  • 窗口标签

类型层次结构

此视图显示包括成员的类型的层次结构以及特定类型的所有子类型和超类型的列表。您可以分析层次结构，并通过双击导航到任何类型的声明。

工具栏控件

控制

名称

描述

刷新

更新窗口内容以反映代码中或外部所做的任何更改。

基于此

选定的类型成为层次结构树的根，并相应地重建树。新的基本类型以粗体突出显示。

全部展开 / 全部折叠

展开/折叠当前选项卡中的所有节点。

上一页 / 下一页
Shift+F8 / F8

导航到上一个/下一个项目，并在源代码中滚动。

显示成员

在使用列表指定的位置（在窗口的底部或右侧）隐藏或显示预览窗格。

出口

点击这个按钮以文本格式导出当前在窗口中显示的数据，或使用下拉选择器以XML或HTML格式导出数据。出现的“ 导出数据”对话框将帮助您将数据保存到文件或将其复制到剪贴板。

类层次结构

该树显示中心类型的所有子类型和超类型。中心类型可以位于树中的任何位置，而不必位于根部。您可以使用“ 以此为基准”按钮选择其他中心类型 ，然后将相应地重建树。

子类型层次结构

此树显示中心类型的所有子类型。中心类型位于树的根部。

超类型层次结构

该树以升序显示中心类型的所有超类型（每个子节点是其父节点的超类型）。中心类型位于树的根部。

实施层次

此树显示中心类型的所有后代。

接口叶

该树显示所选接口的叶子。

显示此解决方案中的元素

隐藏外部库中的符号。

在图上显示

单击此按钮可在类型依赖关系图视图中显示窗口中显示的所有类型。

参考项目

单击此按钮以显示当前项目引用的所有项目和程序集。

引用项目

单击此按钮以显示所有引用当前项目的项目。层次结构以升序显示（每个子节点都是引用其父项的项目）。当前项目位于树的根。

通过...分组

允许按不同类别对窗口中的项目进行分组。

所有实例成员 （预览窗格）

在预览窗格中显示所有实例成员。

仅多态成员 （预览窗格）

将预览窗格限制为仅显示所选层次结构节点的多态成员。

类型依赖图

类型依赖关系图视图使您可以直观地研究解决方案中类型之间的依赖关系。您可以添加来自不同项目或已编译程序集的任意数量的类型，并可视化它们之间的不同种类的依赖关系。有关更多信息，请参见探索类型依赖图 。

参考层次

此视图可帮助您探索任何程序集的引用层次结构，并跟踪所有程序集依赖项。检测到递归依存关系，并在参考条目的右侧标记字形。有关更多信息，请参见探索参考层次 。

窗口标签

在窗口中打开类型或样式层次结构或类型相关性图时，它们将显示在新选项卡中，这些选项卡显示在左侧。如果窗口的宽度允许，先前打开的选项卡将移至右侧，否则它们将自行卸载以节省内存-但您始终可以通过单击选项卡右侧出现的箭头并单击所需的选项卡来还原它们：

为了防止标签页卸载，请点击图钉图标标签标题旁边。
