Configure Code Viewer

The basic settings of the Code Viewer include:

  • Word wrap
  • Highlighting of white spaces
  • Displaying line numbers

You can configure code presentation in many other ways. For more information, see Working with Code.

To wrap code lines to the Code Viewer width

  • In the main menu, choose View | Options | Word Wrap.

  • Press Ctrl+R,R

  • Tick the Word wrap checkbox on the Environment | Viewer page of dotPeek options.

To show or hide white spaces

  • In the main menu, choose View | Options | View White Space.

  • Press Ctrl+R,W

  • Tick the View white space checkbox on the Environment | Viewer page of dotPeek options.

To show or hide line numbers

  • In the main menu, choose View | Options | Line Numbers.

  • Press Ctrl+R,N

  • Tick the Show line numbers checkbox on the Environment | Viewer page of dotPeek options.
