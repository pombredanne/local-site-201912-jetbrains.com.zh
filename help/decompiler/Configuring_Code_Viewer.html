dotPeek 2019.2帮助

配置代码查看器

代码查看器的基本设置包括：

  • 自动换行
  • 突出显示空白
  • 显示行号

您可以通过许多其他方式配置代码表示。有关更多信息，请参见使用代码 。

将代码行包装为“代码查看器”宽度

  • 在主菜单中，选择查看|选项|自动换行 。

  • 按Ctrl+R,R

  • 在“ 环境”上，选中“自动换行”复选框。 dotPeek选项的查看器页面。

显示或隐藏空白

  • 在主菜单中，选择查看|选项|查看空白 。

  • 按Ctrl+R,W

  • 勾选环境| 视图空白复选框。 dotPeek选项的查看器页面。

显示或隐藏行号

  • 在主菜单中，选择查看|选项|行号 。

  • 按Ctrl+R,N

  • 勾选环境| 显示行号复选框。 dotPeek选项的查看器页面。
