查看|概述

默认情况下，在代码查看器中启用了代码概述，您可以分别调整每个选项卡的概述设置。

打开新文档时，“代码查看器”中的所有代码均可见。如有必要，可以折叠代码区域以将其隐藏。在“代码查看器”的每个选项卡中，垂直线出现在左边缘。带减号（-）的正方形标识可折叠代码区域的开始。当您单击减号时，代码区域的文本将替换为三个句点框（…），并且减号变为加号（+）。当您单击加号时，将显示折叠的代码，并且加号更改为减号。
