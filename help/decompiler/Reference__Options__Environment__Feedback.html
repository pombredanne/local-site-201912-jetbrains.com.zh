Product Feedback

Tools | Options | Environment | Product Feedback

A while after you have installed dotPeek, it might invoke a balloon in the system tray to invite you to join the dotPeek Feedback Program in order to help dotPeek developers improve the product based on anonymous usage statistics.

When you click the balloon, the Product Feedback prompt appears, letting you make a decision whether to participate or not.

This page serves two purposes:

  • It sets forward comprehensive conditions of participation in the program.

  • It provides the Participate anonymously in Feedback program checkbox to either join the program or resign at any time.
