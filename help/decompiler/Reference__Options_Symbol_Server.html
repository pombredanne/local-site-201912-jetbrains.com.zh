符号服务器

工具|选项|符号服务器

在dotPeek选项的此页面中，您可以定义dotPeek符号服务器的设置。

项目

描述

生成用于以下目的的符号文件（.pdb）： 使用单选按钮，选择应为其生成符号文件的程序集。 前四个选项的含义很明显。最后一个具有最大的灵活性-选择此选项后，您只能选择调试所需的程序集， 然后在Assembly Explorer中打开它们 。

必要时，您可以使用程序集列表来加载所需的程序集。

港口

dotPeek符号服务器端口。如果默认端口号被其他应用程序占用，则可能需要更改它。
