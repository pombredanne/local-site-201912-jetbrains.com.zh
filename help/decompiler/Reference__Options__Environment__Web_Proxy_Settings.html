Web Proxy Settings

Tools | Options | Environment | Web Proxy Settings

In this page of dotPeek options, you can adjust Web proxy settings for dotPeek's Internet activities. By default, dotPeek follows your system settings and if any proxy server is defined in your system, dotPeek will use it too for fetching sources from symbol servers, detecting updates, and so on.

Item

Description

Use system Web proxy settings when downloading from Internet

You can clear this checkbox to override system Web proxy settings for dotPeek's Internet connection.

Edit

This button appears when you choose to use system proxy server settings.
By clicking on this button you open the Internet Properties dialog, where you can change system Web proxy settings.
