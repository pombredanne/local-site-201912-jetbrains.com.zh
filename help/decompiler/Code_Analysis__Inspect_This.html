Inspect This

This command is a shortcut to dotPeek's code exploration and inspection features:

  • Explore Type Hierarchy
  • Type dependency diagram

You can invoke this command from the code viewer, from the File Structure window and other tool windows. Depending on the context, this shortcut helps you access all code-exploration features available in that context.

To view contextually available code exploration features

  1. Place the caret at a symbol in the code viewer or select the symbol in a tool window.

  2. Press Ctrl+Shift+Alt+A

  3. The Inspect This list appears.

  4. Choose an item from the list and click it or press Enter.
