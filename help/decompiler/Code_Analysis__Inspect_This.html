检查一下

此命令是dotPeek的代码探索和检查功能的快捷方式：

  • 探索类型层次
  • 类型依赖图

您可以从代码查看器，“ 文件结构”窗口和其他工具窗口调用此命令。根据上下文，此快捷方式可帮助您访问该上下文中可用的所有代码探索功能。

查看上下文可用的代码探索功能

  1. 将插入号放置在代码查看器中的符号上，或在工具窗口中选择该符号。

  2. 按Ctrl+Shift+Alt+A

  3. 将显示“ 检查此列表”。

  4. 从列表中选择一个项目，然后单击它或按Enter 。
