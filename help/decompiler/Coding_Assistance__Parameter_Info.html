参数信息

查看|参数信息
Ctrl+P

每当您研究函数调用时，dotPeek都会帮助您查看有关函数所有重载的允许参数的详细信息。在工具提示中，您将看到所有公共签名，包括带有参数的简短签名以及从函数文档中获得的简短说明（如果有）。

使用参数信息

要研究现有函数调用的替代签名，请将插入号设置在函数括号内，然后按Ctrl+P或从主菜单中选择。

显示参数信息工具提示时，可以按Ctrl+P / Ctrl+Shift+Alt+Space或使用箭头按钮在签名列表中滚动并显示当前所选签名的摘要。

此功能的灵感来自JetBrains ReSharper （Microsoft Visual Studio的开发人员生产力工具）。
