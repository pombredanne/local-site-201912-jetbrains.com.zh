查看编译器生成的代码

如果您探索自己的程序集，则可能有兴趣检查编译器生成的代码。例如，查看编译器如何处理lambda，闭包和自动属性等。为此，dotPeek 提供了“ 显示编译器生成的代码”选项，该选项全局适用于“ 代码查看器”中所有打开的反编译文档以及“ 程序集资源管理器”窗口中的类型和成员。

默认情况下，此选项是关闭的，并且dotPeek尝试显示可用的原始代码结构。

显示编译器生成的代码选项仅适用于反编译的文档。如果文档是源文件，则此选项不会更改任何内容。您可以通过// Decompiled with JetBrains decompiler在第一行中注释。有关查看代码的不同方式的更多信息，请参见编译代码的演示选项 。

显示或隐藏编译器生成的代码

  • 单击打开/关闭编译器生成的代码 在工具栏上。

  • 选中dotPeek选项的Decompiler页面上的Show compiler-generate code复选框。
