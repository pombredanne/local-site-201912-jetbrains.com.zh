在文件夹中查找装配体

档案|浏览文件夹

dotPeek允许在Assembly Explorer窗口中加载整个文件夹。加载文件夹后，dotPeek处理其所有子文件夹以寻找能够反编译的文件，并在Assembly Explorer中显示文件夹的层次结构。

由于dotPeek会递归处理所选文件夹，因此请确保不要添加可能包含太多程序集的文件夹（例如Program Files ）。在dotPeek中加载所有程序集可能需要很长时间。

除了浏览文件夹之外，您还可以打开一个特定的程序集文件 。

使用.NET程序集打开文件夹

  • 在主菜单中，选择“ 文件”浏览文件夹 ，然后选择一个目标文件夹。

  • 单击浏览文件夹 在Assembly Explorer工具栏上，然后选择一个目标文件夹。

  • 将目标文件夹从Windows资源管理器拖到Assembly Explorer窗口中 。

该文件夹将添加到Assembly Explorer窗口中 。您可以通过展开其中的子文件夹和所有装配节点来进行探索。双击代码符号以反编译并在代码查看器中查看相应的代码。有关更多信息，请参见使用代码 。

文件夹中找到的所有程序集都可以在程序集资源管理器的根目录下显示。为此，右键单击装配节点，然后在上下文菜单中选择“在根级别显示”。

如果您想随时清除反编译缓存，则可以删除％LOCALAPPDATA％\ JetBrains \ Shared \ vAny \ DecompilerCache中的数据 。
