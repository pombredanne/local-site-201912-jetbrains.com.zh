dotPeek 2019.2帮助

介绍

dotPeek是一个免费的.NET反编译器和程序集浏览器。dotPeek背后的主要思想是免费向.NET社区中的每个人提供高质量的反编译。

dotPeek会反编译任何.NET程序集，并将其显示为C＃或IL代码。支持的文件类型包括：

  • 库（.dll）

  • 可执行文件（.exe）

  • Windows 8元数据文件（.winmd）

  • 封存（.zip）

  • NuGet软件包（.nupkg）

  • Microsoft Visual Studio扩展程序包（.vsix）

为了帮助您使用已编译的程序集，dotPeek提供了许多功能来：

  • 探索.NET程序集
  • 使用反编译的代码，原始源代码和IL代码
  • 导航和搜索
  • 调试编译的代码

dotPeek借鉴了JetBrains ReSharper的大部分功能， JetBrains ReSharper是Microsoft Visual Studio的开发人员生产力工具。

Windows资源管理器集成

dotPeek允许在Windows资源管理器中打开组件，方法是右键单击它们，然后选择“ 使用JetBrains dotPeek浏览” 。如果使用安装程序部署dotPeek，则会自动启用Windows资源管理器集成。但是，如果您使用的是可移植发行版，则需要在“ 环境” | “ 环境” |“ 与Windows资源管理器集成”复选框中打钩。 dotPeek选项的常规页面，用于启用Windows资源管理器集成。
