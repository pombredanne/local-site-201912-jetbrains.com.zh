使用代码

可以反汇编程序集并将其显示为C＃代码，IL代码，或者可以使用PDB文件和符号服务器从本地或远程位置获取原始源代码。有关更多信息，请参见编译代码的呈现选项 。

dotPeek将代码反编译为C＃4.0。如果有async / await修饰符，它将代码反编译为C＃5.0。如果有异常过滤器或自动属性初始化程序，则使用C＃6.0呈现反编译的代码。

反编译或源代码显示在“ 代码查看器”选项卡中，该选项卡具有Visual Studio中编辑器选项卡的外观，并具有可配置的行号，自动换行，概述，空白标记等选项。

以下各节描述了使用代码的各个方面：

  • 编译代码的演示选项
  • 查看编译器生成的代码
  • 查看元数据令牌
  • 查看中间语言（IL）
  • 代码概述
  • 语法高亮
  • 参数信息
  • 快速文档
  • 将符号信息复制到剪贴板
  • 扩展/缩小选择
  • 选择包含声明
  • 高亮匹配定界符
  • 突出显示当前行
  • 检查一下
  • 探索类型层次
  • 探索类型依赖图
  • 访问反编译文件
