前往集会

导航|前往集会
Ctrl+Shift+N

此命令将您导航到其中的任何程序集或反编译文件当前在“ 程序集资源管理器”窗口中加载的所有程序集以及从这些程序集引用的所有程序集。

请注意，此命令在全局范围内起作用，也就是说，您无需将重点放在代码查看器上即可调用它。

转到程序集或反编译文件

  1. 在主菜单中，选择“ 导航|前往集会或按Ctrl+Shift+N 。

  2. 在出现的输入程序集名称弹出窗口中，开始输入项目名称。键入时，查找列表将变窄，并建议与输入的子字符串匹配的名称。

  3. 请执行以下任一操作：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找结果中显示 ， 按Shift+Enter要么+在数字键盘上。

此处支持不区分大小写的CamelHumps ：例如，您可以通过键入“ LecS”轻松找到LocalElementConflictSearcher.cs 。

您也可以在指定名称时使用通配符：“ *”（星号）表示零个或多个字符； '+'（加号），'：'（分号），'。' （点）和'\'（反斜杠）将类型的完全限定名称分开。

此功能的灵感来自JetBrains ReSharper （Microsoft Visual Studio的开发人员生产力工具）。
