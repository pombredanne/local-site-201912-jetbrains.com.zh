Quick Find window

View | Quick Find
Ctrl+F

Using this window you can look for text string in the current document.

You can switch the search scope for the current document or for the current selection in the document and change standard find options. If you want to search with a regular expression, wildcard, acronym or shorthand, select the corresponding option in the Search type selector.

To search in a wider scope, use the Search Everywhere/Go to Type or Go to String commands.
