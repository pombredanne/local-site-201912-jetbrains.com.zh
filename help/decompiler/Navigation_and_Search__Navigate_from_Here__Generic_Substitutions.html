Navigate To Generic Substitutions

Navigate | Navigate To | Generic Substitutions
Ctrl+Shift+G | Generic Substitutions

This command helps you determine exactly what types are used to substitute generic type parameters in usages of the generic symbols in the currently loaded assemblies. The command becomes available when you invoke Navigate To on a declaration or a usage of any generic type parameter, or or any generic substitution. You can invoke this command from the code viewer, from the File Structure window and other tool windows.

All found substitutions are displayed in the Find Results window, where you can further analyze the results.

This feature is inspired by and borrowed from JetBrains ReSharper, a developer productivity tool for Microsoft Visual Studio.
