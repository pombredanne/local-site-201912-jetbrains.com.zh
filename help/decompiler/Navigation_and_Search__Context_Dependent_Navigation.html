dotPeek 2019.2帮助

浏览当前上下文

dotPeek提供了许多功能，可帮助您根据当前上下文（即根据打开的文档及其在其中的插入符号位置）导航代码。

导航文件结构

Windows |档案结构
Ctrl+F11

借助dotPeek，您可以使用“文件结构”窗口浏览当前文档的结构，从而大大简化了大型文件的浏览。该窗口与代码查看器同步：当您切换到另一个代码查看器选项卡时，该窗口将显示相应文件的结构。

  1. 按Ctrl+F11或选择 Windows |主菜单中的文件结构 ％menuPath％ 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  2. 当前文档的内容以树形结构显示，您可以浏览并浏览该文档。您可以双击任何符号，以使代码查看器中的插入符号切换到相应位置。

  3. 要将更改代码查看器中插入符号位置的方式与“ 文件结构”窗口中突出显示的符号同步，请单击 在编辑器中跟踪插入符 在工具栏上。

  4. 要将您在“ 文件结构”窗口中选择的符号与代码查看器中的插入符号位置同步，请单击 自动滚动到源 在工具栏上。

在“文件结构”窗口中，您可以访问更多操作。要检查可用命令，可以右键单击任何符号。

  • 从符号导航
  • 查找符号的用法 Alt+F7

要在“文件结构”窗口中搜索项目，只需开始输入名称或CamelHumps缩写。

导航到包含声明

导航|包含声明
Ctrl+[

使用单个快捷键，您可以在编辑器中从其体内的任何位置跳转到类型或函数的声明。

dotPeek提供了一项相关功能- 选择包含声明 （ Ctrl+Shift+[ ）。

  1. 将插入号设置为类型或函数体内的任何位置。

  2. 按Ctrl+[或选择 导航|从主菜单包含声明 ％menuPath％ 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. dotPeek会将插入符号设置为类型或函数的声明。

  4. 如果在另一个类型中声明了该类型或函数，则可以重复第二步以导航到包含类型的声明。

在成员和标签之间导航

导航|下一个/上一个成员
Alt+Down / Alt+Up

使用“ 下一个/上一个成员”命令，您可以在代码文件中的成员和嵌套类型之间或标记文件中的标记之间快速导航。

  1. 当焦点位于编辑器中时，按Alt+Down / Alt+Up或选择导航|主菜单中的下一个/上一个成员 。

  2. 插入符号将移至下一个/上一个成员/标签的声明。如果存在内部类型，则插入符号还将在每个内部类型的成员处停止。

在解决方案/组装资源管理器中查找当前文档

导航|在装配体浏览器中找到
Shift+Alt+L

此便捷的导航功能使您可以在Assembly Explorer窗口中展开树，自动滚动并突出显示与当前文档相对应的项目。

如果光标在特定符号上，则与该符号对应的项目在Assembly Explorer中突出显示；如果光标在文档中的其他位置，则突出显示文档的根类型。

您也可以从使用该符号的源代码中直接在Assembly Explorer中找到库符号：当插入号位于符号上时，按或选择 导航|在主菜单中导航到… ，然后选择在装配体浏览器中定位 。

  1. 在“ 代码查看器”中 ，将插入记号设置为要在Assembly Explorer中找到的符号。

  2. 按Shift+Alt+L或选择导航|在主菜单中的Assembly Explorer中找到 ，

  3. dotPeek将调出Assembly Explorer窗口以聚焦并向您显示该项目。

浏览相关文件

导航|相关文件…
Ctrl+Shift+Alt+G

该命令可以将您带到其内容与当前文件某种程度上相关的所有文件。

导航到相关文件对于Web应用程序可能非常有用：例如，它使您能够从Web表单跳转到其主页，用户控件，引用的图像或JavaScript和CSS文件。在ASP.NET MVC应用程序中，它简化了从视图到控制器以及向后的导航。

  1. 将插入符号放在文件中。

  2. 按Ctrl+Shift+Alt+G或选择 导航|相关文件…主菜单中的％menuPath％ 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 如果存在一些相关文件，则会打开“ 选择文件”列表。从列表中选择一个文件，然后单击它以在编辑器中将其打开。

通过数字导航代码行

查看|前往线
Ctrl+G

“ 转到行”命令使您可以快速跳至当前文档中的任何代码行。

要在代码查看器中显示行号，请按Ctrl+R,N或选择查看|选项|行号。在菜单中。

  1. 按Ctrl+G或选择查看|从菜单转到行 。

  2. 在出现的对话框中，您将看到当前文档中的总行数。目标行设置为当前行。键入目标行号，然后单击“ 确定” 。
