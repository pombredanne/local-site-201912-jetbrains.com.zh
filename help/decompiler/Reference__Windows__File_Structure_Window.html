文件结构窗口

Windows |档案结构
Ctrl+F11

借助dotPeek，您可以使用“文件结构”窗口浏览当前文档的结构，从而大大简化了大型文件的浏览。该窗口与代码查看器同步：当您切换到另一个代码查看器选项卡时，该窗口将显示相应文件的结构。

在“文件结构”窗口中，您可以访问更多操作。要检查可用命令，可以右键单击任何符号。

  • 从符号导航
  • 查找符号的用法 Alt+F7

要在“文件结构”窗口中搜索项目，只需开始输入名称或CamelHumps缩写。
