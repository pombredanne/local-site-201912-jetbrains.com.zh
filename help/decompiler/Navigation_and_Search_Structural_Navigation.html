Structural Navigation

dotPeek allows you to use the Tab and Shift+Tab keys to quickly move text selection to the next or previous code element without having to use the mouse or the cursor keys. This way you can quickly jump to the next or previous logical piece of code.

This feature is inspired by and borrowed from JetBrains ReSharper, a developer productivity tool for Microsoft Visual Studio.
