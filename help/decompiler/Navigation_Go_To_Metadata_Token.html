转到元数据令牌

导航|转到元数据令牌

如果加载了相应的程序集，此命令将通过其元数据令牌查找并显示反编译符号。

如果您知道特定符号的标记（例如，来自另一个反编译器的标记），那么这将很有帮助，然后您可以在dotPeek中快速找到该符号。

您还可以使用“在任何地方搜索”来按标记搜索，并使用标记来过滤“ 程序集资源管理器”窗口中的条目

如果有必要，可以在工具栏上单击显示标记值的显示标记值大会Explorer窗口的树和在反编译代码的注释。有关更多信息，请参见查看元数据令牌。

导航到元数据令牌

  1. 在主菜单中，选择“ 导航|转到元数据令牌 。

  2. 在显示的“ 输入元数据令牌”弹出窗口中，输入一个十六进制令牌或0x前缀或不带前缀。所有加载的程序集中具有匹配标记的符号将显示在列表中。

  3. 要缩小搜索结果的范围，请在空格后输入模块名称。

  4. 请执行以下任一操作：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找结果中显示 ， 按Shift+Enter要么+在数字键盘上。
