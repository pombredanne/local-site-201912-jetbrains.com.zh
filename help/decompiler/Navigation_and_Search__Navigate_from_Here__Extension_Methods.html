Navigate To Extension Methods

Navigate | Navigate To | Extension Methods
Ctrl+Shift+G | Extension Methods

This command allows you to navigate from a declaration or usage of a type to all extension methods defined for this type in the currently loaded assemblies. It is available when you invoke Navigate To on declarations and usages of types. You can invoke this command from the code viewer, from the File Structure window and other tool windows.

If the command returns a single result, dotPeek navigates directly there. In case of multiple results, they will be listed in a drop-down where you can do one of the following:

  • Click the desired item to open it in the editor.

  • Select the desired item by pressing Up and Down and then press Enter to open it in the editor.

  • Start typing to filter results. Note that you can use CamelHumps here. If necessary, press Esc once to clear the filtering.

  • To view and analyze the list of matched items in the Find Results window, click Show in Find Results , press Shift+Enter or + on the numeric keypad.

This feature is inspired by and borrowed from JetBrains ReSharper, a developer productivity tool for Microsoft Visual Studio.
