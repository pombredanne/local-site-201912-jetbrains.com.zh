浏览编译的代码

作为使用已编译代码的工具，dotPeek可帮助您搜索和浏览已编译代码。

代码 查看器中的代码表示取决于 设置 ，PDB信息的可用性以及所应用的导航命令 。

有两种针对库符号的导航命令：

  • 明确指定在何处以及如何表示库符号的命令：

    • 在装配体浏览器中找到
    • 导航到反编译源
    • 导航到元数据视图
    • 导航到符号文件中的源

  • 导致其他已编译符号的命令在未指定如何表示目标代码的情况下已编译符号。例如，

    • 前往宣言
    • 转到符号类型
    • 转到基本符号

对于第二种命令，您可以指定dotPeek将如何表示库符号。您可以在以下位置配置默认导航的反编译器页面 dotPeek选项。

如果您想随时清除反编译缓存，则可以删除％LOCALAPPDATA％\ JetBrains \ Shared \ vAny \ DecompilerCache中的数据 。
