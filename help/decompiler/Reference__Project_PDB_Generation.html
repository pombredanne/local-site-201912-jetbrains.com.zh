项目/ PDB生成状态窗口

Windows | Pdb生成状态

该工具窗口记录并显示与生成符号文件和项目有关的所有事件。在以下情况下，此窗口中会出现新条目：

  • dotPeek 用作符号服务器 。在这种情况下，每个调试器生成符号文件的请求都将显示为新事件。

  • 您手动生成程序集的PDB文件 。

  • 您将程序集导出到Visual Studio项目 。

如果在生成过程中发生一些错误，则会显示相应的错误消息。
