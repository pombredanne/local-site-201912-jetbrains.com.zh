配置dotPeek

开箱即用的dotPeek功能具有默认配置，该默认配置基于.NET世界中的约定和最佳实践。但是，每个功能都可以根据您的需要和偏好进行灵活调整。

配置dotPeek首选项

  1. 在主菜单中，选择“ 工具” |“工具”。选项 。

  2. 在出现的“ 选项”对话框中，在左窗格中选择一个节点，然后在右窗格中配置设置。
    使用左上角的搜索框查找特定的首选项。

  3. 单击确定 。

在这个部分：

  • 配置代码查看器
  • 配置键盘快捷键
