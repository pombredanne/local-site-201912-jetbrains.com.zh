Configure dotPeek

Out of the box, dotPeek features have default configurations, which are based on conventions and best practices in the .NET world. However, every feature can be flexibly adjusted according to your needs and preferences.

To configure dotPeek preferences

  1. In the main menu, choose Tools | Options.

  2. In the Options dialog that appears, select a node in the left pane and configure settings in the right pane.
    Use the search box in the left top corner to find a specific preference.

  3. Click OK.
