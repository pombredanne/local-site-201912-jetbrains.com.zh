打开和关闭装配体

档案|打开
Ctrl+O

使用dotPeek，您可以打开任何.NET程序集，进行反编译并浏览其内容。 可以反汇编程序集并将其显示为C＃代码，IL代码，或者可以使用PDB文件和符号服务器从本地或远程位置获取原始源代码。有关更多信息，请参见编译代码的呈现选项 。

您也可以从全局程序集缓存（GAC）中打开程序集，并在特定文件夹中找到程序集 。

支持的程序集文件类型包括：

  • 库（.dll）

  • 可执行文件（.exe）

  • Windows 8元数据文件（.winmd）

  • 封存（.zip）

  • NuGet软件包（.nupkg）

  • Microsoft Visual Studio扩展程序包（.vsix）

打开装配体

  • 在主菜单中，选择“ 文件”打开 ，然后选择一个程序集文件。

  • 按Ctrl+O然后选择一个程序集文件。

  • 点击打开 在“ 程序集资源管理器”工具栏上，然后选择一个程序集文件。

  • 将程序集文件（或文件的选择）从Windows资源管理器拖到“ 程序集资源管理器”窗口 。

  • 在“ 进程资源管理器”窗口中选择所需的程序集，然后单击“ 将程序集添加到程序集资源管理器” 在窗口的工具栏上。

  • 对于.dll文件：通过在Windows资源管理器中双击一个文件（前提是dotPeek选项的“ 环境|常规”页面上启用了Windows资源管理器集成）对于.dll和.exe文件：通过在Windows资源管理器中右键单击一个文件，然后选择使用JetBrains dotPeek浏览 。

打开的装配体将显示其名称和版本，并将其添加到“ 装配体浏览器”窗口中 。您可以通过扩展部件节点及其子节点来探索它。双击代码符号可反编译并查看相应的代码。有关更多信息，请参见使用代码 。

除了打开程序集文件之外，您还可以浏览一个包含程序集文件的文件夹 。

由于dotPeek不会对打开的程序集进行任何修改，因此可以通过从Assembly Explorer窗口中删除来关闭它们。

关闭装配体

  • 右键单击一个装配，然后在上下文菜单中选择“ 从列表中删除项目” 。

  • 在装配体浏览器中选择一个装配体，然后按Delete 。

  • 要关闭所有部件，请单击“ 清除部件资源管理器”。

如果您想随时清除反编译缓存，则可以删除％LOCALAPPDATA％\ JetBrains \ Shared \ vAny \ DecompilerCache中的数据 。
