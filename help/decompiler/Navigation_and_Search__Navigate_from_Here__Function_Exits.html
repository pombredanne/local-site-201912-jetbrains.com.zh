导航到功能出口

导航|导航到|功能出口
Ctrl+Shift+G | 功能出口

该命令为您提供了一种导航到功能退出点的简便方法。你可以找到，如果你这个命令来调用定位到一个属性的功能，getter或setter的名字，构造函数的名字， return和throw陈述。

dotPeek对待return和throw语句以及具有以下功能的右括号void函数退出时返回类型。

如果只有一个出口点，则dotPeek会直接导航到那里。如果存在多个出口点，则每个出口点将以蓝色突出显示。

请注意，默认情况下，将尖号设置为其中的一个时，dotPeek会自动突出显示所有上下文出口。

此功能的灵感来自JetBrains ReSharper （Microsoft Visual Studio的开发人员生产力工具）。
