DM服务器

选中复选框以能够使用SpringSource dm Server并开发以dm Server为目标的应用程序。

项目

描述

服务器

从列表中选择要使用的服务器（如果在IntelliJ IDEA中已经定义了相应的服务器）。

要在IntelliJ IDEA中定义服务器，请单击“ 添加”并指定服务器设置。

刻面

选择模块将实现的部署工件类型（IntelliJ IDEA术语中的Spring DM facet类型）：

  • 捆绑包， OSGi捆绑包。IntelliJ IDEA将创建：

    • 捆绑软件的META-INF \ MANIFEST.MF文件。

    • dm捆绑软件工件配置。

    • dm Server API的项目级库。该库将被添加到模块依赖项中。

    • 面向dm服务器的运行/调试配置。

    如有必要，请指定其他选项 。

  • PAR或计划， dm Server PAR或计划。指定相关的设置 。

  • 配置，一个配置工件。指定工件名称 。

有关dm Server部署工件的更多信息，请参见SpringSource dm Server程序员指南中的 “部署体系结构”和“开发应用程序” 。

捆绑选项

Spring DM支持

选中此复选框以启用Spring支持（以IntelliJ IDEA术语创建Spring facet）。如此一来，IntelliJ IDEA将创建以下文件：

  • META-INF \ spring \ module-context.xml
  • META-INF \ spring \ osgi-context.xml

在此步骤中，建议您不要下载实现Spring的库文件。但是，在使用“ 项目结构”对话框中的相应快速修复程序创建模块之后，您将能够执行此操作。

网络模块

选中此复选框以启用通用Web应用程序开发支持（以IntelliJ IDEA术语创建Web构面）。指定关联的设置：

  • 版。 选择要支持的Servlet规范版本。

  • 创建web.xml。 对于3.0或更高版本：选择此复选框以创建部署描述符文件web.xml 。（对于早期版本，总是创建此文件。）

结果，IntelliJ IDEA将创建web \ WEB-INF \ web.xml （对于3.0或更高版本-如果已指定）。

PAR或计划选项

名称

对于PAR，这是应用程序标识符Application-SymbolicName ，对于一个计划-计划名称（ name的属性元件）。

版

应用程序或计划版本（ Application-Version或者version的属性元件）。

计划

选择此选项以创建计划。IntelliJ IDEA将创建：

  • 计划 XML文件。

  • dm计划工件规范。

平台存档（PAR）

选择此选项以创建一个PAR。 IntelliJ IDEA将创建：

  • PAR的META-INF \ MANIFEST.MF文件。

  • dm Platform Archive工件规范。

范围

对于计划：选中此复选框以使计划成为作用域（对应于scoped="true"在元件）。

原子

对于计划：选中此复选框以使计划成为原子计划（对应于atomic="true"在元件）。

嵌套捆绑

使用此区域中的控件来管理PAR或计划中的其他dm Server部署工件。（在IntelliJ IDEA中，如果这些模块具有合适的dm Server方面，则这些模块由同一项目中的其他模块表示。PAR可以包括OSGi捆绑包和配置构件；计划-OSGi捆绑包，配置工件，PAR和其他计划）。

  • 加。 使用此按钮将工件添加到列表中。在打开的对话框中选择必要的工件（IntelliJ IDEA模块）。

  • 去掉。 使用此按钮可以从列表中删除选定的工件。

  • 起来 对于计划：使用此按钮可将所选工件在列表中向上移动一行。（工件的顺序定义了它们的部署顺序。）

  • 下。 对于计划：使用此按钮可将所选工件在列表中向下移动一行。

  • 版本。 对于计划：使用此按钮可以为所选工件指定版本或版本范围（对应于version的属性元件）。

Spring MVC，Spring Batch或其他Spring框架

选择一个复选框以添加对特定Spring框架（Spring MVC，Spring Batch等）的支持。

您需要一个实现所选框架的库 。您可以选择使用现有库，创建和使用新库，或者下载库文件（如果计算机上尚不可用）。

使用图书馆

选择要使用的库（如果在IntelliJ IDEA中已经定义了相应的库）。

创建

如果您的计算机上已经有相应的jar文件，则可以将这些文件放在库中。单击创建，然后在打开的对话框中选择所需的文件。

（可选）单击“ 配置”以编辑选定的库。
