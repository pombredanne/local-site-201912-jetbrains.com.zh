监视PHP应用程序的代码覆盖率

仅当安装并启用PHP插件时，以下内容才有效。

PHPUnit的 代码覆盖率向您显示了多少代码被测试覆盖，并且在编辑器中直观地显示了覆盖和未覆盖的标记。

在这个部分：

  • 先决条件
  • 配置代码覆盖率
  • 测量代码覆盖率

先决条件

  1. PHPUnit工具已安装在您的计算机上，并已在IntelliJ IDEA中启用 。

  2. 在当前的PHP解释器中配置了Xdebug或Zend Debugger调试引擎。

配置代码覆盖率

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，在“ 生成，执行，部署”下单击“ 覆盖率 ”。

  2. 在“ 覆盖率”页面上指定覆盖率选项。

测量代码覆盖率

  1. 手动编写测试。

  2. 创建类型为PHPUnit 的运行配置 。

  3. 在主工具栏上的“运行/调试配置”下拉列表中选择“ PHPUnit运行”配置，然后单击“ 运行覆盖率”按钮 。

  4. 查看代码覆盖率结果 。
