模组

档案|项目结构模组
Ctrl+Shift+Alt+S |模组
|模组

当您在“ 项目结构”对话框中选择“模块” 类别时 ， 元素选择器窗格中将显示现有模块组，模块，构面以及Flash 配置的构建配置的分层视图。（Facet和构建配置显示为模块元素。）

使用工具栏图标，上下文菜单命令或键盘快捷键来管理显示的元素集（ 请参见下文 ）。

要查看或编辑元素（模块，构面或构建配置）的设置，请选择所需的元素，然后使用选择器窗格右侧的页面。

在这个部分：

  • 模块页面
    • 来源标签
    • 路径标签
    • 依赖选项卡
    • 插件部署选项卡
  • 构面页面
    • Android Facet页面
    • AspectJ Facet页面
    • Android级方面页面
    • EJB方面页面
    • Google App Engine方面页面
    • GWT方面页面
    • Hibernate和JPA Facet页面
    • Java EE应用程序方面页面
    • JSF Facet页面
    • OSGi Facet页面
    • 缝面页面
    • Struts 2刻面页面
    • 挂毯刻面页面
    • 网页页面
      • Web资源目录路径对话框
    • Web服务方面页面
    • Web服务客户端方面页面
  • Flash模块的模块页面
  • 闪存模块的构建配置页面
    • 常规选项卡
    • 依赖项选项卡
    • 编译器选项选项卡
    • 空气包装标签
    • Android标签
    • iOS标签

 Alt+F7
