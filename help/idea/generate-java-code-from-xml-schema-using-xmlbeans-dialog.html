使用XmlBeans对话框从XML模式生成Java代码

工具| XmlBeans |使用XmlBeans从XML模式生成Java代码

使用此对话框可通过XmlBeans数据绑定器配置基于XML Schema的Java代码存根的生成。

进入对话框

  1. 此功能通过WebServices捆绑插件提供，默认情况下启用。如果没有，请按照“ 管理插件 ”部分中所述启用它。

  2. 当在活动编辑器选项卡中打开的文件包含XML架构时，菜单项和对话框可用。

项目

描述

模式路径

在此字段中，指定要用作生成基础的文件。默认情况下，该字段显示当前文件的完整路径。要使用另一个架构，请单击浏览。 并在打开的“ 选择XML模式/ Wsdl文件生成”对话框中选择所需的文件。

输出路径

使用此字段来指定用于将生成的和已编译的Java代码放入其中的jar的名称。默认情况下，IntelliJ IDEA建议创建一个新的types.jar 。要覆盖现有的jar ，请单击“ 浏览”。 然后在打开的对话框中选择所需的jar 。

添加必要的库以便生成的代码进行编译和工作

选中此复选框可将其他XmlBeans库自动添加到将放置生成的源代码的模块的类路径中。

这些是生成的存根所依赖的库。

状态

在配置生成过程时，查看此只读字段中的信息以跟踪和改善差异。
