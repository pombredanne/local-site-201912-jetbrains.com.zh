手动将WS库添加到Web服务客户端模块

如果先前已经下载了必需的库，则可以使用此方法。

向现有模块添加必要的库

  1. 打开“模块设置”对话框。

  2. 选择所需的模块后，单击“ 依赖关系”选项卡。

  3. 再次单击“ 添加”按钮，然后从上下文菜单中选择“ 单项模块库 ”。在打开的对话框中 ，选择javaee.jar库和所需的特定于Web服务的库。

    1. javaee.jar库的位置是在IntelliJ IDEA安装期间定义的。

    2. Web服务特定库的位置是在其下载期间定义的。

  4. 准备好后，单击“ 确定” 。
