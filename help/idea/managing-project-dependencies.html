IntelliJ IDEA 2019.3帮助

管理项目依赖性

IntelliJ IDEA使得跟踪项目中未满足的依赖关系成为可能，并提供与主要依赖关系管理方法的集成，包括对requirements.txt和Pipfile的使用。

  • requirements.txt包含应用程序依赖的一组软件包。将具有该名称的文件添加到根项目目录后， Python Integrated tools会自动检测到该文件。

    有关详细说明，请参见使用requirements.txt 。

  • Pipfile仅适用于具有Pipenv虚拟环境的项目。它还可以管理项目依赖项集。但是，此工具通过创建项目环境的精确快照（包括哈希码和确切的软件包版本）为安全敏感的部署提供了更多功能。

    有关详细说明，请参见使用pipfile管理依赖项 。
