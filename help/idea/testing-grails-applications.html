测试Grails应用程序

IntelliJ IDEA支持创建和运行单元测试和集成Grails测试，并提供每种测试类型的运行配置。

要创建Grails测试，请执行以下任一操作：

  • 对要创建测试的域类使用脚手架 ：

  • 运行Grails目标 ：按Ctrl+Alt+G ，然后在文本字段中输入目标名称。使用代码完成来缩小匹配目标的范围：

创建Grails测试运行/调试配置

  1. 在Grails视图中 ，右键单击测试目录之一，指向上下文菜单上的Create Run Configuration ，然后选择建议的选项之一：
    • 在...中测试
    • Grails测试：...
  2. 在打开的“运行/调试配置”对话框中，指定运行/调试配置设置：测试类型，Grails脚本等。

    您可以指定多个用空格分隔的测试名称，所有这些测试将在单个运行配置中运行。

运行Grails测试

  1. 在Grails视图中 ，选择所需的测试类别。

  2. 在打开的菜单中，选择运行 ，然后选择所需的配置类型：

    所选的运行/调试配置在“运行”工具窗口中执行。
