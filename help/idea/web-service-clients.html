介绍

使用IntelliJ IDEA，您可以开发以下最常见类型的Web服务的客户端：

  • GlassFish / JAXWS2。X RI / Netro 1。X / JWSDP2.0，用于开发JAX-WS Web服务客户端。

  • Apache Axis，用于开发Apache Axis Web服务客户端。

  • JAX-RS，用于开发RESTful Web服务客户端。

您还可以启用并手动配置对以下其他WS引擎的支持 ：

  • Apache Axis2
  • https://jax-rpc.dev.java.net/

  • XFire 1。X / CXF

  • JBossWS
  • WebSphere 6。X
