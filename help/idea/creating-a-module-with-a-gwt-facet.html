使用GWT构面创建模块

当创建具有专用GWT构面的模块时，IntelliJ IDEA会配置该模块并自动添加所有必需的库。

创建具有GWT构面的模块

根据情况，您可以选择创建一个新项目或向现有项目添加模块。

  1. 请执行以下任一操作：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件” |“ 新建” 。 新增|项目 。

      结果，“ 新建项目”向导打开。

    • 如果要向现有项目中添加模块，请执行以下操作：打开要向其中添加模块的项目，然后选择“ 文件” |“添加” 。新增|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页的左侧窗格中，选择Java Enterprise 。在页面的右侧，指定要使用的JDK并选择要支持的Java EE版本。

  3. 在其他库和框架下 ，选中Google Web Toolkit复选框。

    在GWT SDK字段中，指定GWT SDK的位置。（要下载最新版本的GWT SDK，请使用“ 下载GWT”链接。）

    如果要创建示例GWT应用程序 ，请选中“ 创建示例应用程序”复选框，并在下面的字段中指定应用程序名称。

    单击下一步 。

  4. 指定名称和位置设置。

    点击完成 。
