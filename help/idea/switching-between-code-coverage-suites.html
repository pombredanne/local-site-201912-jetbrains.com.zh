管理覆盖套件

覆盖范围测量结果包括覆盖范围套件。您可以将新模拟的结果与任何现有套件合并。在这种情况下，如果一条线被至少一个模拟覆盖，则该线将被视为覆盖。

每当执行具有代码覆盖率度量的测试或应用程序时，就会生成覆盖率套件。可能有无限数量的覆盖套件。

选择一个覆盖套件

在您至少运行了一项覆盖测试之后，覆盖套件列表将变为可用。

  1. 从主菜单中，选择“ 分析|显示覆盖率数据 Ctrl+Alt+F6 。

  2. 在“ 选择要显示的覆盖套件”对话框中，选中必要套件旁边的复选框，然后单击“ 显示所选内容” 。

    在编辑器中，IntelliJ IDEA打开所选测试套件的测试覆盖率结果。

  3. 要隐藏覆盖率结果，请选中必要类别旁边的复选框，然后单击“ 无覆盖率” 。

上传或删除Coverage Suite

考虑从构建服务器获取包含代码覆盖率信息的文件时的情况。您可以从磁盘加载此文件，并使其可供查看。另外，您可以带些时间检查生产的覆盖套件。
