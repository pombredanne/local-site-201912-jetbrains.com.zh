创建GWT UiRenderer和ui.xml文件

IntelliJ IDEA可以为您生成GWT UiRenderer和ui.xml文件。在ui.xml中定义声明式布局时，可以使用各种类型的编码帮助，例如代码完成和导入语句的生成 。

IntelliJ IDEA支持ui.xml中的声明式布局与相应的Java类之间的集成，包括查找用法 。

如果您正在使用ClientBundle界面，IntelliJ IDEA用图标标记其方法在左装订线区域。单击此图标导航到相应的资源。

只能在客户端中创建GWT UiRenderer。

生成GWT UiRenderer

  1. 在“ 项目”视图中 ，右键单击客户端程序包。

  2. 从上下文菜单中，选择“ 新建| Google Web工具包| GWT Renderer和ui.xml文件 。

  3. 在打开的“ 创建新的GWT Renderer和ui.xml文件”对话框中，指定UiRenderer和ui.xml文件的名称。
    单击确定 。
