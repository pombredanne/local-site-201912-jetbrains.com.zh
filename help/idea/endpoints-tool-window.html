端点工具窗口

端点工具窗口提供了项目中用于HTTP和Web套接字协议的客户端和服务器API的聚合视图。您还可以将视图扩展到在IntelliJ IDEA中打开的所有项目。

图标

行动

描述

从开放项目中显示

包括在IntelliJ IDEA中打开的其他项目中可用的端点。其他项目的端点以黄色突出显示。

从图书馆展示

包括在库中实现的端点。库中的端点以黄色突出显示。

显示详细资料

打开一个额外的窗格，其中包含端点详细信息：相应的类，方法等。

显示空组

包括所有可用的框架，模块和项目，即使未为其定义端点。

通过...分组

在列表中选择如何对端点进行分组：

  • 全部清除：列出所有检测到的端点而不进行分组

  • 项目 ：将所有检测到的端点按定义它们的项目分组

  • 模块 ：将所有检测到的端点按定义它们的模块分组

  • 端点类型 ：按类型（ 服务器 ， 客户端 ， API ）将所有检测到的端点分组

  • 框架 ：通过相应框架将所有检测到的端点分组

  • 端点源 ：按其源（使用相对路径定义的类或文件）将所有检测到的端点分组。
