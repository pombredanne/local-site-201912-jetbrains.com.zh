介绍

Web应用程序一词通常用于具有HTML，JSP，JavaScript等的分层文件夹集合。代表用户界面的文件和资源，以及实现应用程序功能的Java类。这些文件夹是根据为部署和执行应用程序的服务器指定的标准排列的。为了确保Web应用程序的成功部署和可访问性，您需要在开发阶段就已经根据目标服务器标准设置其结构。

接下来，最有可能需要配置Web应用程序元素，设置初始化参数并定义与EJB，WebServices等的交互。您将需要各种库来配置这些组件和关系，以及一个包含这些组件，设置和约束的定义的应用程序描述符。

使用IntelliJ IDEA，您可以根据所需标准自动设置基本应用程序结构，并在Web应用程序开发的所有阶段获得广泛的编码帮助。

开发Web应用程序

要在IntelliJ IDEA中开发Web应用程序，请执行以下常规步骤：

  1. 启用Web应用程序开发支持以获取必要的库，基本文件夹层次结构和应用程序描述符。

  2. 填充Web模块 ：创建所需的Java类，配置servlet，过滤器，侦听器和引用。

  3. 配置所需的静态Web内容资源 。

  4. 配置Web应用程序部署 。

  5. 部署并运行该应用程序。
