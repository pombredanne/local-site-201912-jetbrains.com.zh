创建表单初始化代码

完成GUI设计器窗体的构建后 ，至少应具有一个绑定到窗体的类，并且该类中的组件类型字段绑定到各种窗体组件。

如果将二进制类文件指定为输出选项，则将在类文件中生成所有运行时初始化代码。如果您的表单具有绑定类，那么您将在此处看不到自动生成的初始化代码。如果您选择了Java源代码作为输出选项，则表单的绑定类将包含自动生成的$$$setupUI$$$()方法。

不要手动编辑GUI初始化程序源代码！在进行任何编译时，IntelliJ IDEA都会在本节中自动生成新代码，并且所有更改都将丢失。

有时您可能需要提供自己的初始化代码。例如，您希望GUI组件由具有某些参数的非默认构造函数实例化。在这种情况下，IntelliJ IDEA将不会生成该组件的实例化，您有责任在createUIComponents()方法。否则，将报告空指针异常。请遵循下面介绍的一般技术。

要为某个组件创建自定义GUI初始化程序源代码，请遵循以下一般步骤

  1. 选择所需的组件。

  2. 在检查器中，选中“ 自定义创建 ”选项。

  3. 选择组件后，按F4 ，或在上下文菜单上选择“ 跳转到源 ”。

  4. 在文本编辑器中，找到方法createUIComponents() ，然后输入所需的源代码。此方法中的代码在编译时不会删除。

例
