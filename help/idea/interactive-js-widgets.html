使用互动式小部件

IntelliJ IDEA支持交互式小部件，这些小部件提供了笔记本内核中运行的Python代码与浏览器中运行的JavaScript之间的集成。有关交互式小部件的种类和特性的更多详细信息，请参考Jupyter小部件文档 。

将小部件代码添加到Jupyter笔记本中

要将交互式小部件添加到笔记本，请首先添加一个代码单元。按Shift+Alt+A （Windows）或⌥ ⇧ A （macOS）在当前选定的单元格上方插入新的单元格，或Shift+Alt+B （Windows）或⌥ ⇧ B （macOS）在所选单元格下方创建一个新单元格。有关更多详细信息，请参见笔记本中的“编辑源代码” 。

编辑交互式窗口小部件的代码时，请使用代码完成功能来方便地添加构造和快速修复以添加丢失的导入语句。

要预览小部件，您需要执行相应的代码单元。您可以看到在编辑器中内联并在Jupyter工具窗口的“ 变量”选项卡中列出的窗口小部件变量的值。

除了使用交互式小部件创建代码单元外，您还可以将JavaScript调用注入Python代码单元中，并将小部件变量传递给JavaScript代码。考虑一个将ColorPicker小部件的选定颜色值传递给JavaScript调用以绘制文档背景的示例：

处理安全要求

如果您在项目外部创建的笔记本文件中打开任何JavaScript代码，则该笔记本被视为不受信任。根据为Jupyter笔记本电脑建立的安全模型 ，永远不会执行不受信任的JavaScript代码。因此，当您尝试在这样的笔记本中执行JavaScript时，会收到错误消息： Notebook is not trusted. JavaScript hasn't been executed 。

选中Jupyter工具栏上的Trusted复选框，以明确允许在Jupyter笔记本中执行JavaScript。
