IntelliJ IDEA 2019.3帮助

AspectJ

默认情况下，IntelliJ IDEA Ultimate中的AspectJ支持已启用，并包括以下功能：

  • 能够以两种形式创建方面：as .aj文件和包含带有注释类的java文件@Aspect 。

  • 编写方面代码时的编码帮助（包括代码完成 ）。对于注释样式方面，将提供完整的编码帮助。对于代码样式方面，援助仅限于类型间声明 。

  • 能够执行基本方面方面的重构，例如Rename和Move ，以及为类型间声明进行重构中的Push ITD 。

  • 与AcpectJ编译器ajc集成，您可以直接在IDE中运行它。该编译器是AspectJ发行版的一部分，您可以从AspectJ网站上下载该编译器。

  • 能够在项目级别配置ajc ，并可以在单个模块级别微调其用法。

必要时启用AspectJ插件

捆绑的AspectJ和Spring AOP / @ AspectJ插件应默认启用。如果它们被禁用，请执行以下操作：

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，选择插件 。

  2. 单击已安装选项卡，然后搜索aspectj 。

  3. 启用捆绑的AspectJ和Spring AOP / @ AspectJ插件。

将带有AspectJ运行时的库添加到项目中

如下载页面所述下载并安装AspectJ。AspectJ运行时Aspectjrt.jar应该位于/ lib下的安装目录中。

  1. 打开项目结构对话框Ctrl+Shift+Alt+S ，然后选择库 。

    如果要为所有项目添加新库，请选择“ 全局库” 。

  2. 请点击然后选择Java 。

    您还可以选择From Maven并搜索Aspectjrt.jar以从Maven存储库添加AspectJ运行时。

  3. 指定Aspectjrt.jar在文件系统中的位置。

  4. 选择应向该库添加模块，然后单击“ 确定” 。

添加AspectJ依赖项

  1. 打开项目结构对话框Ctrl+Shift+Alt+S ，然后选择Facets 。

  2. 请点击然后选择AspectJ 。

  3. 选择要使用此构面的模块，然后单击“ 确定” 。

  4. 为相关模块选择新的AspectJ构面，并将依赖项添加为相应方面的路径。

创建一个方面

添加必要的AspectJ 库和依赖项之后 ，就可以开始开发方面了。

  1. 在“ 项目”工具窗口中，选择要在其中创建外观的目录，然后按Alt+Insert ，然后选择Aspect 。

  2. 指定方面文件的名称和类型，然后单击确定 。

    • Aspect将使用指定的名称创建一个.aj文件。

    • @Aspect创建一个.java文件，其中包含带有注释的类@Aspect 。

如有必要，您可以重构方面的源代码 。

使用AspectJ编译器编译方面ajc ， 选择它作为项目编译器 。 ajc在Aspectjtools.jar中 ，该文件位于/ lib下的安装目录中。
