设计器工具窗口

使用此专用工具窗口可以无需手动编辑布局定义文件即可构建应用程序的设计，并检查如何在各种目标环境中呈现应用程序设计，而无需在任何物理或虚拟设备上运行应用程序。

或者，可以手动编辑布局定义文件（可能使用IntelliJ IDEA提供的特定于Android的重构），并预览立即反映在专用“ 预览工具”窗口中的更改 ，您可以在其中调整布局以适应各种平台和设备。要切换到手动模式，请单击“ 文本”选项卡，或从“ 设计”窗格或“ 组件树 ”的上下文菜单中选择“ 转到声明” 。

布局定义活动或应用程序小部件 （片段）的用户界面。布局在XML 资源定义文件中声明。有关如何创建资源文件夹和资源定义文件的说明，请参见创建资源 。

Android UI设计器包含以下窗格：

  • 设计窗格
  • 组件树
  • 物产
  • 调色板

设计窗格

设计窗格位于UI设计器的中心部分（假定使用默认的工具窗口布局）。当您打开布局定义文件进行编辑时，默认情况下该窗格显示在编辑器选项卡中。如果您要手动编辑布局定义文件，然后通过单击“ 设计”选项卡切换到可视模式，则窗格将在该选项卡中打开，其中已编辑的布局定义文件被打开，

要在手动和可视编辑模式之间切换，请使用窗格/编辑器底部的“ 文本”和“ 设计”选项卡：

该窗格显示了一个矩形画布，该画布与当前布局定义文件和“ 组件树”视图同步，因此对画布的任何更改都将在此反映出来。

要将组件添加到画布，请执行以下操作之一：

  • 在“ 面板”窗格中选择所需的元素，然后将其拖放到“ 设计”窗格中的画布上。

  • 单击“ 调色板”窗格中的所需元素，然后单击画布上的一个区域。

以任一种方式添加的每个组件也将添加到“ 组件树”中，并在布局定义文件中声明。

您也可以在画布中为组件指定最基本的属性，而无需切换到“ 属性”窗格。为此，选择组件，双击所选区域，然后在打开的弹出窗口中填写字段。

画布具有一个上下文菜单 ，该菜单提供对剪贴板，布局操作，重构等的访问。

您可以查看各种设备上的构建布局：使用Controls上的控件 配置目标环境仿真 。

有关“ 设计”窗格中可用的不同配置选项的详细信息，请参考以下部分：

  • 上下文菜单
  • 工具列

上下文菜单

可以从“ 设计”窗格中的任何区域（在画布内部和外部）使用上下文菜单。所有操作都与组件树和XML定义文件同步，因此所有更改都会立即反映在它们中。组件树窗格的上下文菜单中也提供了相同的操作。

控制项

使用此区域中的控件预览不同配置的布局。

项目

工具提示

描述

配置以在IDE中呈现此布局

从此列表中，选择要预览和编辑的布局配置，创建新的布局配置，或选择其他预览选项。可用的选项有：

  • 创建横向变化 ：选择此选项可创建布局的横向版本。相应的布局定义文件将在res \ layout-land文件夹中生成。创建此变体后，此菜单选项将替换为“ 切换到布局区域”选项，该选项将打开layout-land \ .xml文件进行编辑。

  • 创建layout-xlarge变体 ：选择此选项可为更大的屏幕尺寸（至少960x720 dp） 创建布局的变体。相应的布局定义文件将在res \ layout-xlarge文件夹中生成。创建此变体后，此菜单选项将替换为“ 切换到layout-xlarge”选项，该选项将打开layout-xlarge \ .xml文件进行编辑。

  • 切换到布局 ：仅当您创建了多个布局版本时，此选项才可用。选择它以返回到原始布局定义文件。

  • 创建其他 ：选择此选项可创建布局的另一个变体。在打开的“ 选择布局目录”对话框中，指定将存储布局定义的文件夹，然后选择确定特定设备配置的资源限定符 。选择相关的限定词，然后单击 。然后在打开的对话框中指定限定符的值。限定词将添加到“ 选择的限定词”列表中。

  • 预览代表样本选择此选项可显示多个设备配置并在最重要的屏幕尺寸上预览布局。

  • 预览所有屏幕尺寸 ：选择此选项可显示多种设备配置并预览所有可用屏幕尺寸的布局。

  • 预览所有语言环境 ：选择此选项可在将要使用您的应用程序的所有语言环境中预览布局。

  • 预览从右到左的布局 ：选择此选项可以并排预览两个方向（从左到右和从右到左）的布局。

  • 预览Android版本 ：选择此选项可预览所有已安装的Android API版本上的布局。

  • 包括预览 ：选择此选项可预览嵌套在另一个布局中的布局。仅当当前布局包含在另一个布局中时，此选项才可用。

  • 预览布局版本 ：选择此选项可显示多个设备配置并预览所有可用版本中的布局。

  • 无 ：选择此选项可返回默认视图。

  • 切换布局模式 ：选择此选项可在不同的预览选项之间切换。

进入下一个状态

从此下拉列表中，选择预览方向（ 纵向或横向 ），UI模式（ 常规，汽车坞，桌面坞，电视，设备 ），然后在夜间 （屏幕变暗）和非夜间 （标准亮度）模式之间切换。有关UI模式的详细信息，请参考UiModeManager 。

从此下拉列表中，选择一个现有语言环境或为您的应用程序添加语言环境。语言环境是目标国家和语言的组合，可以根据本地规则和偏好显示日期和一些其他数据。您还可以在所有可用语言环境和两个方向（从左到右和从右到左）中预览布局。

组件

描述

版面

Android布局定义了应用程序界面的视觉结构。它们是确定其子级在屏幕上显示方式的类。

小部件

Android小部件是用户界面中的交互式组件。您可以从各种小部件中进行选择，例如按钮，复选框，测试字段等。

文字栏位

约会时间

专家

除了最常用的组件之外，本节还包含其他预定义的UI元素。

自订

在此部分中，您可以选择并添加在项目或Android SDK中定义的UI组件。您可以插入小部件的精心组合，甚至可以将整个布局嵌入到当前布局中。IntelliJ IDEA检查项目和Android SDK中适当类型的资源定义。提供以下选项：

  • 包括 ：单击此项目以选择要嵌入的布局。在打开的“ 资源”对话框中，选择“ 项目”选项卡以从项目中选择布局定义，或者选择“ 系统”选项卡以在SDK中进行搜索。

  • 片段 ：单击此项目以选择要嵌入的片段。

  • requestFocus ：单击此项以包括requestFocus元素view宾语。的requestFocus元素将其父级初始焦点放在屏幕上。

  • CustomView ：单击此项目以选择要嵌入的用户定义视图 。
