尽可能使用界面

尽可能使用接口重构将从基类/接口派生的指定方法的执行委派给实现相同接口的祖先类或内部类的实例。

  1. 在编辑器中，将插入记号放在应将方法委托给其父类或接口的类上。

  2. 在主菜单或上下文菜单上，选择“ 重构|尽可能使用接口 。

  3. 在打开的对话框中，选择将替换当前类用法的父对象。

  4. 替换中的当前类名instanceof语句，请在instanceof中选中使用接口/超类选项。

    请注意，如果您使用instanceof语句，并且未选中此复选框，则您可能会收到错误的代码，例如：

    if（secondInterface instanceof类）

    该代码将编译，但可能会产生不希望的结果。

  5. 预览并应用更改。
