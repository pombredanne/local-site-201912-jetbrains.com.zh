发送反馈

IntelliJ IDEA 2019.3帮助

启用休眠支持

  • 总览
  • 确保启用了Hibernate Support插件
  • 在创建项目或模块时启用Hibernate支持
  • 为现有模块启用Hibernate支持

总览

为了能够使用Hibernate支持，您应该：

  • 确保启用了Hibernate Support 插件 。（此插件与IDE捆绑在一起，并默认启用。）

  • 在模块级别启用Hibernate支持。您可以在创建新项目或模块时执行此操作。您还可以为现有模块启用Hibernate支持。在所有这些情况下，IntelliJ IDEA都会（以下选项将作为选项提供）：

    • 创建一个Hibernate配置文件hibernate.cfg.xml和一个带有main()为您的管理实体输出记录的方法。

    • 下载实现框架的库文件，并将它们添加到相应模块的依赖项中。

    • 为数据库表生成实体类和对象/关系映射（如果有可用的适当数据源 ）。

    • 创建一个Hibernate facet。您将可以使用该方面来管理配置hibernate.cfg.xml文件。

    • 使“ 持久性工具”窗口可用。

确保启用了Hibernate Support插件

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，选择插件 。

  2. 在对话框的右侧，在“ 插件”页面上 ，键入hib在搜索框中。结果，只有名称和说明包含的插件hib显示。

  3. 如果未选中“ Hibernate支持 ”右侧的复选框，则将其选中。

  4. 单击确定 。

  5. 如果建议，请重新启动IntelliJ IDEA。

在创建项目或模块时启用Hibernate支持

  1. 请执行以下任一操作：

    • 如果要创建新项目，请：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件” |“ 新建” 。 新品|项目 。

      结果，“ 新建项目”向导打开。

    • 如果要向现有项目添加模块，请执行以下操作：打开要向其中添加模块的项目，然后选择“ 文件” |“添加”。新品|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页的左侧窗格中，选择Java 。在页面的右侧，指定要使用的JDK 。

  3. 在“ 其他库和框架”下 ，选中“ 休眠”复选框。

  4. 如有必要，请选择创建配置文件hibernate.cfg.xml和一个主类（带有main()输出管理实体记录的方法）。

    如果在IntelliJ IDEA中定义了一个数据库作为数据源，则可以选择导入数据库架构。这将导致为它们创建必要的实体类和对象/关系映射。

  5. 选择所需的库选项，并在必要时指定关联的设置。您可以选择：
    • 下载Hibernate实现文件，并将这些文件放在库中 。

      为此，请在“ 库”下选择“ 下载” 。（可选）单击“ 配置”以编辑库设置。（将打开“ 下载选项”对话框 。）

    • 使用已经定义了IntelliJ IDEA的库。

      为此，请单击“ 使用库” ，然后从列表中选择所需的库。（可选）单击“ 配置”以编辑库设置。（将打开“ 编辑库”对话框 。）

    • 使用计算机上可用的适当JAR文件创建新库。

      为此，单击使用库 ，然后单击创建 。在打开的对话框中选择所需的JAR文件。（对于多项选择，请保留Ctrl键）。（可选）单击“ 配置”以编辑库设置。（将打开“ 创建库”对话框 。）

    • 将库的设置推迟到以后。在这种情况下，请选择稍后设置库 。

    单击下一步 。

  6. 指定名称和位置设置。

    点击完成 。

如果选择导入数据库架构，则将打开“ 导入数据库架构”对话框 ，您可以指定要生成的实体类和相关设置。

为现有模块启用Hibernate支持

  1. 在“ 项目工具”窗口中 ，右键单击所需的模块，然后选择“ 添加框架支持” 。

  2. 在打开的“ 添加框架支持”对话框的左侧窗格中，选中“ 休眠”复选框。

  3. 如有必要，请选择创建配置文件hibernate.cfg.xml和一个主类（带有main()输出管理实体记录的方法）。

    如果在IntelliJ IDEA中定义了一个数据库作为数据源，则可以选择导入数据库架构。这将导致为它们创建必要的实体类和对象/关系映射。

  4. 选择所需的库选项，并在必要时指定关联的设置。您可以选择：
    • 下载Hibernate实现文件，并将这些文件放在库中 。

      为此，请在“ 库”下选择“ 下载” 。（可选）单击“ 配置”以编辑库设置。（将打开“ 下载选项”对话框 。）

    • 使用已经定义了IntelliJ IDEA的库。

      为此，请单击“ 使用库” ，然后从列表中选择所需的库。（可选）单击“ 配置”以编辑库设置。（将打开“ 编辑库”对话框 。）

    • 使用计算机上可用的适当JAR文件创建新库。

      为此，单击使用库 ，然后单击创建 。在打开的对话框中选择所需的JAR文件。（对于多项选择，请保留Ctrl键）。（可选）单击“ 配置”以编辑库设置。（将打开“ 创建库”对话框 。）

    • 将库的设置推迟到以后。在这种情况下，请选择稍后设置库 。

  5. 单击确定 。

如果选择导入数据库架构，则将打开“ 导入数据库架构”对话框 ，您可以指定要生成的实体类和相关设置。
