Thymeleaf支持概述

IntelliJ IDEA中的Thymeleaf支持包括：

  • 表达式的代码完成和th:*属性。

  • 导航到声明功能（ 导航|声明或Ctrl+B ），您可以从模板中的引用跳转到相应的getter方法， .properties文件中的消息或其他适当的代码片段。

  • 导航到类型声明功能（ 导航|类型声明或Ctrl+Shift+B ），以切换到相应的类型定义。

  • 重命名重构（ Refactor | Rename或Shift+F6 ），用于引用的属性（和getter方法），迭代和状态变量等。

  • 查找未解决的引用和表达式语法错误的代码检查 。

  • 各种查找功能，例如查找用法（ 编辑|查找|查找用法或Alt+F7 ）。

  • 意向行为 ，如创建未解决的消息，引用或导入类添加属性 import的声明org.thymeleaf.*类。

  • 在Web浏览器中预览可以直接从编辑器访问的原型（模板的静态部分）。

启用Thymeleaf插件

为了能够在IntelliJ IDEA中使用Thymeleaf支持，请确保已启用Thymeleaf 插件 。（此插件与IDE捆绑在一起，并默认启用。）

请参阅管理插件 。

Thymeleaf对项目和模块的支持

您可以在创建项目或模块或现有项目或模块时添加Thymeleaf支持。在所有这些情况下，IntelliJ IDEA都会下载Thymeleaf库文件（您可以选择哪些库文件是必需的，哪些不是必需的），并将它们添加到相应模块的依赖项中。

您也可以通过打开适当的pom.xml文件来创建“ Thymeleaf项目”。在这种情况下，项目中的依赖项将由Maven管理。有关更多信息，请参见Maven 。

创建项目或模块时添加Thymeleaf支持

  1. 请执行以下任一操作：

    • 如果要创建新项目，请：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件” |“ 新建” 。 新品|项目 。

      结果，“ 新建项目”向导打开。

    • 如果要向现有项目添加模块，请执行以下操作：打开要向其中添加模块的项目，然后选择“ 文件” |“添加”。新品|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页的左侧窗格中，选择Java 。在页面的右侧，指定要使用的JDK 。

  3. 在“ 其他库和框架”下 ，选中“ Thymeleaf”复选框。

    在页面下部，单击配置 ，然后在打开的对话框中选择要下载的库文件。

  4. 您可能还希望通过选中“ Web应用程序”复选框来启用Web应用程序开发支持。（有关更多信息，请参见启用Web应用程序支持 。）

    单击下一步 。

  5. 指定名称和位置设置。

    点击完成 。

为现有项目或模块添加Thymeleaf支持

  1. 打开项目工具窗口（例如， 视图|工具窗口|项目 ）。

  2. 右键单击项目或模块文件夹，然后选择添加框架支持 。

  3. 在打开的“ 添加框架支持”对话框的左侧窗格中，选中“ Thymeleaf”复选框。

  4. 在对话框的右侧，单击配置 ，然后在打开的对话框中选择要下载的库文件。

  5. 您可能还希望通过选中“ Web应用程序”复选框来启用Web应用程序开发支持。（有关更多信息，请参见启用Web应用程序支持 。）

  6. 在“ 添加框架支持”对话框中单击“ 确定 ”。
