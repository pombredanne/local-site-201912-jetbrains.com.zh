将GWT构面添加到模块

将专用GWT构面添加到模块时，IntelliJ IDEA会重新配置模块并自动添加所有必需的库。

将GWT构面添加到现有模块

  1. 打开项目工具窗口 （例如， 视图|工具窗口|项目 ）。

  2. 右键单击该模块，然后选择添加框架支持 。

  3. 在“ 添加框架支持”对话框的左侧窗格中，选中“ Google Web Toolkit”复选框。

  4. 在GWT SDK字段中，指定GWT SDK的位置。（要下载最新版本的GWT SDK，请使用“ 下载GWT”链接。）

  5. 如果要创建示例GWT应用程序 ，请选中“ 创建示例应用程序”复选框，并在下面的字段中指定应用程序名称。

  6. 在“ 添加框架支持”对话框中单击“ 确定 ”。
