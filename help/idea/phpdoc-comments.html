PHPDoc评论

仅当安装并启用PHP插件时，以下内容才有效。

对于文档注释，IntelliJ IDEA提供了默认情况下启用的完成功能。当您键入IntelliJ IDEA时，将创建PHPDoc块的存根。 /**打开标签并按Enter ，或按Alt+Insert并指定代码构造（类，方法，函数等）进行记录。根据您的选择，IntelliJ IDEA将创建所需的标签或添加一个空的文档存根。

如果您需要其他特定于PHP的标记 ，则IntelliJ IDEA提供代码完成功能，以建议与当前上下文相关的标记名称。如果某个标签具有多个值，则代码补全将提供可用值的列表。

在PHPDoc注释中，IntelliJ IDEA支持符合ZEND，PEAR和其他编码标准的格式化选项。

源代码中的PHPDoc注释可用于“ 快速文档查找” ，该文档可帮助您快速获取任何已记录符号的信息。您可以通过按以下方式在“ 文档”工具窗口中打开它们进行审阅Ctrl+Q 。

启用文档注释

  1. 打开 编辑器|一般| IntelliJ IDEA设置的“ 智能键”页面Ctrl+Alt+S 。

  2. 在“ 输入”部分中，选中或清除“ 插入文档注释存根”复选框。

  3. 以下内容仅在安装并启用Python插件后才有效。

    对于Python，请滚动到文档注释存根选项中的 “ 插入类型占位符”，然后根据需要选中或清除该复选框。有关详细信息，请参阅选项说明 。

为代码构造生成一个PHPDoc块

  1. 要调用PHPDoc块的生成，请执行以下一项操作：

    • 将插入号放置在所需的代码构造（类，方法，函数等）之前，键入开始块注释/** ，然后按Enter 。

    • 在编辑器上下文菜单中，选择“ 生成” |“生成”。生成PHPDoc块，然后选择代码构造以为其生成PHPDoc注释。

    • 按Alt+Insert ，然后选择生成PHPDoc块 ，然后选择代码构造以为其生成PHPDoc注释。

    IntelliJ IDEA分析指定的代码结构，在可能的情况下提取参数，返回值，变量或字段的数据，并在此基础上生成文档块的存根。

  2. 描述列出的参数，并在必要时返回值。IntelliJ IDEA根据代码检查设置检查和处理注释中的语法。

在PHPDoc注释块中创建标签

IntelliJ IDEA分析指定的代码结构，在可能的情况下提取参数，返回值，变量或字段的数据，并在此基础上生成文档块的存根。如有必要，您可以填写缺少的信息。

  1. 在PHPDoc块中，选择所需的空行，然后按Ctrl+Space 。

  2. 从建议列表中选择相关标签。

  3. 如果输入的标签具有多个值，请按Ctrl+Space然后从建议列表中选择所需的值。

在PHPDoc注释中配置格式

您可以配置PHPDoc注释的外观，类名的表示形式以及默认标签的排序顺序。 请注意，属性标记不再可配置，默认的@var标记将自动插入。有关详细信息，请参见https://github.com/phpDocumentor/fig-standards/pull/55 。

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|代码样式| PHP 。

  2. 切换到PHPDoc选项卡，通过选择或清除复选框来配置所需的外观选项。

  3. 通过选中或清除“ 使用完全限定的类名”复选框，指定IntelliJ IDEA如何显示属性，函数参数， 返回值和引发值等的类名 。

  4. 如有必要，通过选择对PHPDoc标签进行排序，定义应如何对生成的PHPDoc标签进行排序。

    • 使用和从列表中添加或删除PHPDoc标记条目。

    • 使用和排列PHPDoc标记条目的顺序。

    请注意，未添加到列表中的PHPDoc标记将在列出的标记下方生成。

使用PHPDoc代码检查

IntelliJ IDEA提供了一组针对PHPDoc块的预定义代码检查。这些检查检查类，方法，函数，变量和常量是否随PHPDoc注释一起提供，以及注释中的标记是否与记录的项目匹配。

启用或禁用PHPDoc检查

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，选择编辑器|检查 。

  2. 在打开的“ 检查”页面上，展开“ PHP”下的“ PHPDoc”节点。

  3. 在打开的预定义检查列表中，通过选中或清除旁边的复选框来启用或禁用检查。

检查是否为特定类型的代码构造提供了PHPDoc注释

  1. 启用“ 缺少PHPDoc评论”检查。

  2. 在“ 选项”区域中，选中所需元素类型旁边的复选框：类，方法，函数，变量或常量。

    若要在方法或函数不包含任何参数或返回值的情况下禁止报告缺少PHPDoc注释的错误，请选中忽略不带@ param / @ return的PHPDoc复选框。
