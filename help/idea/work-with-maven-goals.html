IntelliJ IDEA 2019.3帮助

Maven目标

IntelliJ IDEA使您可以运行，调试和管理项目中的Maven目标。

运行Maven目标

您可以使用多种方法来运行Maven目标，例如使用“ 运行任何东西”窗口，使用Maven工具窗口中的上下文菜单或为一个或多个Maven目标创建运行配置 。

从“任何运行”窗口中运行Maven目标

  1. 在Maven工具窗口的工具栏上，单击按钮。或者，按Ctrl两次以打开“任何运行”窗口。

  2. 在“ 运行任何内容”窗口中，开始输入要执行的目标的名称。该窗口还显示最近的Maven目标条目的列表。

    如果您有一个多模块项目，并且需要从特定模块执行目标，请首先输入模块的名称，然后输入目标的名称。例如， mvn module1 test

  3. IntelliJ IDEA运行选定的目标并在“ 运行工具”窗口中显示结果。

从上下文菜单运行Maven目标

  1. 在Maven工具窗口中，单击“ 生命周期”以打开Maven目标的列表。

  2. 右键单击所需目标，然后从上下文菜单中选择“运行'目标名称” 。IntelliJ IDEA运行指定的目标，并将其添加到“ 运行配置”节点。

通过运行配置运行一个Maven目标或一组目标

IntelliJ IDEA允许您为一个特定目标或一组多个目标创建运行配置。

  1. 在Maven工具窗口中，单击“ 生命周期”以打开Maven目标的列表。

  2. 右键单击要为其创建运行配置的目标。（要选择几个Maven目标，请按Ctrl并突出显示期望的目标。）

  3. 从列表中选择创建“目标名称” 。

  4. 在“ 创建运行/调试配置：目标名称”对话框中，指定目标设置（您可以指定任何Maven命令和参数），然后单击“ 确定” 。

    IntelliJ IDEA在“ 运行配置”节点下显示目标。

  5. 双击目标以运行它，或右键单击目标，然后从上下文菜单中选择“运行” 。

配置Maven目标的触发器

IntelliJ IDEA使您可以在项目执行之前运行Maven目标，或使用目标激活配置来设置其他条件。

  1. 在Maven工具窗口中，单击“ 生命周期”以打开目标列表。

  2. 在打开的列表中，右键单击要为其设置触发器的目标。

  3. 从上下文菜单中，选择一个激活阶段。

    例如，当您使用“ 在构建之前执行”操作作为触发器时，具有此类触发器的目标会在运行项目的构建操作（ Build | Build Project ）之前执行。如果将构建操作委托给Maven ，则Maven install命令被执行。当您使用“ 在重建之前执行”时 ，带有此类触发器的目标将在运行项目的重建操作（“ 构建” |“重建项目” ）之前执行。如果将重建操作委托给Maven ，则Maven clean,install命令被执行。

    激活名称将添加到Maven工具窗口中的选定目标。

您还可以创建将依赖于Maven目标的运行/调试配置。

  1. 从主菜单中，选择运行|编辑配置以打开项目的运行/调试配置。

  2. 在“ 运行/调试配置”对话框的“ 启动之前”部分中，单击 。

  3. 在打开的列表中，选择“运行Maven目标” 。

  4. 在“ 选择Maven目标”对话框中，指定一个项目和要在启动项目之前执行的目标。

  5. 单击确定 。

将Maven目标与键盘快捷键相关联

您可以将Maven目标与键盘快捷键相关联，并通过一次按键执行目标。

  1. 在Maven工具窗口中，右键单击所需的目标。

  2. 从上下文菜单中，选择“ 分配快捷方式” 。将打开“ 键盘映射”对话框。

  3. 在“ 键盘映射”对话框中的“ Maven”节点下，导航到您的目标。

  4. 右键单击目标，然后从打开的列表中选择要分配的快捷方式类型。

  5. 在打开的对话框中，根据快捷方式的类型，配置您的快捷方式，然后单击“ 确定” 。

    在Maven工具窗口中，针对您的目标显示了快捷方式。

在“ 键盘映射”对话框中，可以添加要为其分配快捷方式的新目标。

  1. 在“ 键盘映射”对话框中的“ Maven”节点下，单击“选择阶段/目标以分配快捷方式” 。

  2. 在打开的对话框中，选择所需的目标，然后单击确定 。

    目标将添加到Maven节点下的列表中。现在，您可以配置快捷方式了 。

调试Maven目标

您可以为一个或多个Maven目标创建常规调试配置 。您还可以在Maven工具窗口中选择一个目标并启动调试会话 。

调试Maven目标

您可以为单个Maven目标或可能包含多个Maven目标的Maven运行配置启动调试会话。

  1. 打开Maven工具窗口。

  2. 在“ 生命周期”节点下，选择要为其启动调试会话的目标。（在“ 运行配置”节点下查找现有的Maven运行配置，以为创建的配置启动调试会话。）

  3. 右键单击目标，然后从上下文菜单中选择“ 调试[目标名称]” 。IntelliJ IDEA启动调试会话。
