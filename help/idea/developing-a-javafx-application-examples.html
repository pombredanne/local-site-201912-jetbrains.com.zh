开发基本的JavaFX应用程序

在本教程中，我们将IntelliJ IDEA创建的示例应用程序转换为基本的JavaFX Hello World应用程序。

本教程假定您熟悉IntelliJ IDEA和基本的编码帮助功能。如果要了解有关代码完成，实时模板和其他有用的IDE功能的更多信息，请参阅“ 创建第一个Java应用程序”部分 。

重命名Controller类

要根据您的需要调整示例应用程序，您可能要开始重命名文件。例如，让我们重命名Controller上课SampleController 。

  1. 在编辑器中，将插入号放置在类名称上，然后从主菜单中选择“ 重构|改名 Shift+F6 。

    Controller现在突出显示。

  2. 类型Sample之前Controller然后按Enter表示您已完成重构。

  3. 在编辑器中切换到sample.fxml并注意GridPanelfx:controller属性已从更改为"sample.Controller"至"sample.SampleController" 。

重命名重构可帮助您更改项目中符号，文件，目录的名称。

添加一个按钮

在我们应用程序的界面中，让我们定义一个要显示的按钮Hello World!单击它之后。

在开始和结束之间添加以下两个元素文件sample.fxml中的标签：

sayHelloWorld以红色显示， helloWorld突出显示。这意味着IntelliJ IDEA无法解析相应的引用。让我们解决这个问题。

完成SampleController的代码

让我们定义字段helloWorld在里面SampleController类。我们还将添加相应的事件处理程序方法sayHelloWorld这将为helloWorld标签。

  1. 在sample.fxml中 ，将插入符号放置在helloWorld然后按Alt+Enter 。

  2. 选择创建字段“ helloWorld” 。

    IntelliJ IDEA切换到IDE已添加字段声明的SampleController.java helloWorld 。按Enter退出重构模式。

    还要注意刚刚添加的导入语句import javafx.scene.control.Label;和图标在字段声明的左侧。这是一个导航图标；单击它返回到sample.fxml 。

  3. 将插入符号放在sayHelloWorld然后按Alt+Enter 。

  4. 选择创建方法'void sayHelloWorld（ActionEvent）' 。

    IntelliJ IDEA将相应的方法声明添加到SampleController.java中 。

  5. 按Shift+Enter退出重构模式并开始新的一行。

  6. 键入以下代码来设置标签的文本：

    helloWorld.setText（“ Hello World！“）;

上下文操作 （例如快速修复和意图操作）可帮助您优化代码并修复潜在的和实际的问题。

运行应用程序

在此步骤中，代码已准备就绪。让我们运行该应用程序以查看结果。

  1. 请点击在工具栏上或按Shift+F10 。

    现在，应用程序窗口将显示“ 说“您好，世界””按钮。

  2. 单击此按钮以显示Hello World！ 文本。

使用CSS设置UI样式

让我们通过添加样式表并定义格式样式来更改UI的外观。

  1. 在文件sample.fxml中 ，添加对我们尚未创建的sample.css的引用。

    添加stylesheets开头的属性标签：

    stylesheets =“ / sample / sample.css”

  2. 按Alt+Enter并选择创建文件sample.css

  3. 创建.css文件后，添加以下样式定义。

    第一种样式使应用程序窗口中的背景“简洁”，第二种样式-设置字体的大小。 Hello World!文字为20像素。

    .root {-fx-background-color：浓汤； } .label {-fx-font-size：20; }

  4. 请点击在工具栏上或按Shift+F10运行该应用程序。
