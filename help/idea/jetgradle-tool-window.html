摇篮工具窗口

创建或打开 Gradle项目时，会自动启用Gradle工具窗口。工具窗口将显示Gradle链接的项目，它们的任务，依赖关系以及对基础build.gradle文件所做的所有更改。

摇篮

项目

描述

单击此按钮可打开菜单，用于配置当前视图和更改工具窗口的查看模式。菜单项是可以打开或关闭的选项。活动项目的名称左侧包含一个复选标记。

菜单中的可用选项如下：

  • 分组模块 ：如果要在项目中分组模块，请选择此选项。

  • 分组任务 ：如果要分组任务，请选择此选项。例如，构建任务将包含在构建目录中。

  • 显示忽略 ：选择此选项以显示忽略的项目 。

  • 显示工具栏 ：选择此选项可显示Gradle项目的工具栏。

  • 查看模式固定模式，对接模式，浮动模式，拆分模式-这些选项使您可以控制工具窗口的常规外观和行为，请参见工具窗口查看模式。

  • 组选项卡 ：如果工具窗口中有多个视图，则取消选择此选项可在单独的选项卡上查看视图。

  • 移至 ：选择此选项可将Gradle工具窗口移至顶部，左侧或右侧。

  • 调整大小 ：选择此选项可调整Gradle工具窗口的大小。

使用此图标或快捷方式隐藏工具窗口Shift+Escape 。

当与Alt键，单击此图标将隐藏附加到同一工具窗口栏的所有工具窗口 。

使用此按钮执行Gradle任务 。当您单击此图标时，“ 运行任何内容”窗口将打开。输入您要执行的任务的名称。要执行多个任务，请使用空格输入任务名称以分隔每个新任务。

使用这些按钮可以展开或折叠所有节点。

单击此按钮可通过“要导入的项目数据”对话框将模块或数据导入到Gradle项目。这对于多模块项目可能很有用。

单击此按钮可以在离线模式下使用Gradle项目。在这种情况下，Gradle将使用缓存中的依赖项。Gradle将不会尝试访问网络来执行依赖关系解析。如果缓存中不存在所需的依赖项，则生成执行将失败。另外，如果启用了离线模式，则项目的打开和同步可能会失败。

单击此图标生成Gradle依赖关系图 。

Gradle任务的上下文菜单命令

命令

描述

跑

选择此命令以使用特定于任务的运行/调试配置来运行所选任务 。

除错

选择此命令可使用特定于任务的运行/调试配置来调试所选任务 。

创建

选择此命令可为选定的Gradle任务创建运行/调试配置 。

选择

选择此命令可激活所选Gradle任务的配置。如果所选任务没有现有配置，则禁用此选项，并用“ 创建”选项代替。

同步前执行，同步后执行

选择这些命令以设置所选任务的相应标志。在这种情况下，节点名称旁边会出现“ 同步 前”和“ 同步后”注释。

构建前执行，构建后执行

选择这些命令以设置所选任务的相应标志。在这种情况下，“ 生成前”和“ 生成后”注释会出现在节点名称旁边。

重建前执行，重建后执行

选择这些命令以设置所选任务的相应标志。在这种情况下，“ 重建之前”和“ 重建 之后”注释会出现在节点名称旁边。

分配快捷方式

选择此命令可以将选定的阶段与键盘快捷键相关联 。在这种情况下，带有快捷方式的注释会出现在节点名称旁边。
