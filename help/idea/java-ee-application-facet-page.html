部署描述符

形成您的应用程序的部署描述符列表。

Alt+Insert 。创建application.xml并将其添加到列表中。如果列表中已经存在相应的描述符，则该功能无效。

Alt+Delete 。从列表中删除选定的描述符。

Enter 。用另一个相同类型的描述符替换选定的描述符。

添加Application Server特定的描述符。 创建特定于应用程序服务器的部署描述符（例如glassfish-application.xml ， jboss-app.xml ）并将其添加到列表中。
