Use this page to edit the settings for your Java EE Application facet.

Item

Description

Name

The name of the facet.

Deployment Descriptors

 Form the list of deployment descriptors for your application.

 Alt+Insert. Create application.xml and add it to the list. The function is not active if the corresponding descriptor is already present in the list.

 Alt+Delete. Remove the selected descriptor from the list.

 Enter. Replace the selected descriptor with another one of the same type.

Add Application Server specific descriptor. Create an application server-specific deployment descriptor (for example glassfish-application.xml, jboss-app.xml) and add it to the list.
