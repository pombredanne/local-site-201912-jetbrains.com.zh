包装返回值

Wrap Return Value重构允许您选择一种方法，并为其返回值创建一个包装类，或者使用一个现有的兼容包装类。所选方法的所有返回将被适当地包装，并且对该方法的所有调用将使其返回被包装。

如果您的设计以您希望某个方法返回比原计划更多的信息的方式更改，则包装方法的返回值将很有用。包装后，可以扩展包装器类，从而允许从该方法返回更多数据。同样，通常包装原始返回值，从而允许根据需要将接口和实现分离。

也可以从UML类图中进行此重构。

  1. 在编辑器中打开所需的类，然后将插入号放在要包装其返回值的方法上。

  2. 选择重构|在主菜单或上下文菜单上包装返回值 。

  3. 在打开的对话框中，为新包装器类指定名称和程序包，或选择一个现有的兼容包装器类。

  4. 预览并应用更改。
