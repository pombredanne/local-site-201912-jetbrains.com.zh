隐式提示

IntelliJ IDEA允许您启用，展开和折叠用于隐式转换和参数的编辑器提示，以帮助您阅读代码。

  1. 在主工具栏上，选择“ 视图”显示隐式提示 。

  2. 在编辑器中，右键单击提示，然后从弹出菜单中选择适当的操作，以展开现有提示，禁用模式或查看隐式参数 。

或者，在编辑器中，您可以按Ctrl+Shift+Alt++启用隐式提示。如果再次按同一快捷方式，则IntelliJ IDEA会扩展隐式提示以向您显示更多详细信息。按Ctrl+Shift+Alt+-折叠提示。

查找隐式用法

您可以使用“ 查找用法”操作从隐式定义导航到其用法。

此功能依赖于字节码索引，还可以用于按原样查找源代码中不存在的以下隐藏元素：

  • apply / unapply方法调用

  • 单一抽象方法 （SAM）类型实例化

  • foreach / map / flatMap / filter / withFilter通过理解打电话

  1. 在编辑器中，选择隐式定义，然后从上下文菜单中选择“ 查找用法” Alt+F7 。

  2. 在“ 查找工具”窗口中查看结果。

Java到Scala代码转换

IntelliJ IDEA使您可以将Java代码转换为Scala。

  1. 复制您的Java代码（表达式，方法，类）并将其粘贴到Scala文件中。

  2. IntelliJ IDEA显示“ 从Java转换代码”对话框，建议进行转换。单击确定 。

如果您不想使用复制/粘贴操作，则可以在编辑器中打开Java文件，然后选择“ 重构” | “复制” 。转换为Scala或按Ctrl+Shift+G 。

在这种情况下，IntelliJ IDEA将使用转换后的代码创建一个Scala文件。

将Scala代码反编译为Java

您可以将Scala代码反编译为Java，以查看特定代码段如何在Java中转换和实现。

  1. 在“ 项目”工具窗口中，右键单击要反编译的Scala类。

  2. 从上下文菜单中，选择将Scala反编译为Java 。

IntelliJ IDEA将代码转换为Java，并在编辑器中打开转换后的文件。

您也可以在编辑器中打开Scala类，并使用其上下文菜单进行转换。
