指定Servlet名称和目标包

定义servlet名称和将在其中创建servlet的目标包

  1. 从“ 项目工具”窗口中打开“ 新建Servlet”对话框。

  2. 在里面在文本框中，指定新servlet名称的根部分。

  3. 在“ 包”文本框中，指定将在其中生成新servlet的目标包的名称。如有必要，请使用按钮。将打开“ 选择Servlet软件包”对话框。

  4. 选择相关的软件包或通过单击创建一个新的软件包 。单击确定。您将返回到“ 新建Servlet”对话框。

  5. 在“ Servlet类”文本框中，键入实现新Servlet的类的名称。如有必要，请使用按钮。将打开“ 选择Servlet类”对话框。

  6. 选择相关的类，然后单击“ 确定” 。您将返回到“ 新建Servlet”对话框。

  7. 单击确定 。
