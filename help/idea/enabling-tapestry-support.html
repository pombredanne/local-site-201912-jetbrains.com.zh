启用挂毯支持

您可以在创建项目或模块或现有模块时启用Tapestry支持。

  • 在创建项目或模块时启用Tapestry支持
  • 为现有模块启用Tapestry支持

在创建项目或模块时启用Tapestry支持

  1. 请执行以下任一操作：

    • 如果要创建新项目，请：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件” |“ 新建” 。 新品|项目 。

      结果，“ 新建项目”向导打开。

    • 如果要向现有项目中添加模块，请执行以下操作：打开要向其中添加模块的项目，然后选择“ 文件” |“添加” 。新品|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页的左侧窗格中，选择Java Enterprise 。在页面的右侧，指定要使用的JDK并选择要支持的Java EE版本。

  3. 在“ 其他库和框架”下 ，选择“ 挂毯”复选框。

  4. 您将需要一个实现Tapestry的库。您可以选择使用现有的库，创建和使用新的库，下载库文件（如果计算机上尚不可用）或将库的设置推迟到以后。

    使用图书馆

    选择要使用的库（如果在IntelliJ IDEA中已经定义了相应的库）。

    创建

    如果您的计算机上已经有相应的jar文件，则可以将这些文件放在库中。单击创建，然后在打开的对话框中选择所需的文件。

    （可选）单击“ 配置”以编辑选定的库。

    下载

    选择此选项以下载必要的库文件。

    （可选）单击“ 配置”以编辑库设置和内容。

    稍后设置图书馆

    选择此选项可以将库的设置推迟到以后。

    单击下一步 。

  5. 指定名称和位置设置。

    点击完成 。

  6. 在打开的对话框中指定应用程序过滤器名称和根软件包 。

结果，将Tapestry实现库添加到模块依赖项中，并创建了Tapestry构面。

为现有模块启用Tapestry支持

  1. 打开项目工具窗口（例如， 视图|工具窗口|项目 ）。

  2. 右键单击感兴趣的模块，然后选择添加框架支持 。

  3. 在打开的“ 添加框架支持”对话框的左侧窗格中，选择“ 挂毯”复选框。

  4. 您将需要一个实现Tapestry的库。您可以选择使用现有的库，创建和使用新的库，下载库文件（如果计算机上尚不可用）或将库的设置推迟到以后。

    使用图书馆

    选择要使用的库（如果在IntelliJ IDEA中已经定义了相应的库）。

    创建

    如果您的计算机上已经有相应的jar文件，则可以将这些文件放在库中。单击创建，然后在打开的对话框中选择所需的文件。

    （可选）单击“ 配置”以编辑选定的库。

    下载

    选择此选项以下载必要的库文件。

    （可选）单击“ 配置”以编辑库设置和内容。

    稍后设置图书馆

    选择此选项可以将库的设置推迟到以后。

  5. 在“ 添加框架支持”对话框中单击“ 确定 ”。

  6. 在打开的对话框中指定应用程序过滤器名称和根软件包 。

结果，将Tapestry实现库添加到模块依赖项中，并创建了Tapestry构面。
