选择目标变更列表对话框

使用此对话框可以回滚特定变更列表中的变更。

项目

描述

现有变更表

选择此选项可恢复现有更改列表之一中的搁置更改。从列表中选择目标更改列表。

新变更清单

选择此选项可创建一个新的更改列表，并从其中的架子恢复更改。

  • 名称：在此文本框中，输入要创建的变更列表的名称。

  • 注释：在此文本框中，输入新变更列表的可选描述。

  • 启用此更改列表：选中此复选框使IntelliJ IDEA在更改恢复后立即自动将活动状态赋予新的更改列表。 清除此复选框后，当前活动的更改列表将保持活动状态。

  • 跟踪上下文：选中此复选框使IntelliJ IDEA在停用时保留与新变更列表关联的任务的上下文，并在变更列表变为活动状态时还原该上下文。有关详细信息，请参见任务和上下文 。

从架子上删除成功应用的文件

  • 清除此复选框使IntelliJ IDEA仍在“ 货架”选项卡中显示已经搁置的更改，以便在必要时可以再次应用它们。

  • 选中此复选框后，取消搁置的更改不会显示在“ 货架”选项卡中。
