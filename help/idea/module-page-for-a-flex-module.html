Flash模块的模块页面

档案|项目结构模块|

当您在元素选择器窗格中选择Flash模块时，将在“ 项目结构”对话框的右侧打开此页面。

使用此页面可以编辑模块名称并配置模块内容。

通过添加和删除模块内容根目录以及通过将单独的文件夹（在内容根目录内）分配给源文件夹，测试源文件夹以及排除文件夹来配置模块内容。

  • 名称栏位
  • 左侧窗格
  • 右侧窗格

名称栏位

项目

描述

名称

使用此字段来编辑模块名称。

左侧窗格

左侧窗格显示模块内容根目录的文件夹树。如果模块具有多个内容根，则显示的结构与在右侧窗格中选择的内容根相对应。

属于不同类别的文件夹具有不同的图标。

下表列出了可用的工具栏按钮（“ 图标”列）并说明了它们的功能（“ 描述”列）。注意，相应的功能也可以作为上下文菜单命令来访问。这些列在“ 命令”列中。

大多数图标/命令都可以用作切换按钮，并且可以用于取消相应的分配（将文件夹设置为“普通文件夹”）。

图标

命令

描述

资料来源

使用此图标或命令将选定的一个或多个文件夹分配给源文件夹类别。

测验

使用此图标或命令将选定的一个或多个文件夹分配给测试源。

排除在外

使用此图标或命令可以排除所选文件夹。

排除文件。 排除名称与至少一种指定模式匹配的文件和文件夹。

右侧窗格

右侧窗格显示模块内容根目录 。（内容根是模块根文件夹。与模块有关的所有内容都存储在其内容根目录中。一个模块可以具有多个内容根。）

对于每个内容根，都提供了模块文件夹的分类视图。类别是源文件夹，测试源文件夹等。

“普通”文件夹未在此视图中显示。

类别中的各个文件夹由其路径标识。文件夹路径都是相对于模块根文件夹（内容根）的。

从功能上讲，文件夹路径是超链接，可让您跳到左侧窗格中显示的树中的相应文件夹。

如果一个模块具有多个内容根，则在右侧窗格中选择一个内容根也会在左侧窗格中切换树视图。即，当您单击内容根区域内的某个位置时，此特定内容根的文件夹结构将显示在左侧窗格中。

下表列出了右侧窗格（图标）中可用的控件并描述了它们的功能。

图标

工具提示

描述

添加内容根

使用此图标添加内容根。在打开的对话框中选择要添加为内容根目录的文件夹。

删除内容条目

使用此图标可以从内容根列表中删除相应的内容根。

取消标记

使用此图标可从相应类别中删除文件夹。结果，该文件夹变为“普通文件夹”，即不属于任何特定类别。
