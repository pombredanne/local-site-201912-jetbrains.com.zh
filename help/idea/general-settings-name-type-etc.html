输出目录

构建后的构件（ Build | Build Artifacts ）将放置在该字段中指定了路径的目录中。

如果不会单独使用工件，而仅将其用作其他工件的一部分，则该字段可以保留为空。在这种情况下，选择“ 构建” |“创建”后 ，工件将不会出现在建议列表中。 Build Artifacts ，选择“ Build” |“ Build”后 ，不会在build.xml中为其生成目标。生成Ant Build 。
