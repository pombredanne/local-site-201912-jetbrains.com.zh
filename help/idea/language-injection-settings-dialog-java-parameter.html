语言注入设置对话框：Java参数

档案|设置|语言注入— Windows和Linux的Java参数
IntelliJ IDEA |首选项|语言注入— macOS的Java参数
Ctrl+Alt+S

此页面描述控件。有关如何使用语言注入的信息，请参阅语言注入 。

单击时将打开对话框在“ 语言注入”页面上，然后从上下文菜单中选择“ Java参数 ”，或选择一个条目并单击 。

使用此对话框可以配置Java参数的语言注入。如果由于某种原因无法使用注入注释，则该对话框提供了使用IntelliLang功能的功能。这主要适用于配置第三方/库方法以及仍必须使用Java 1.4的项目。

项目

描述

语言

在此区域中，指定要注入的语言和注入上下文。

  • ID ：选择要插入的语言的ID。

  • 前缀 ：指定构成注入上下文的前缀。

  • 后缀 ：指定后缀以构成注入上下文。

请注意， 前缀和后缀字段是可选的。

类方法

请点击并选择感兴趣的类别。结果，显示了该类的方法及其参数。

选择要启用进样的参数。

高级

指定其他设置。

  • 值模式 ：为注入上下文键入正则表达式。

  • 单个文件 ： 如果关闭该选项，则与值模式匹配的片段将被分别视为不同的“文件”，例如，从片段编辑器的角度来看。

    如果启用此选项，则所有相应的片段都将合并在一起以形成一个单元或“文件”。

    给定价值模式

    xxx (.+) yyy (.+) zzz

    和片段

    xxx select * yyy from family zzz ，

    select *和from family如果关闭该选项，则被视为两个独立的片段（或“文件”）。如果该选项打开， select * from family被视为一个单位或“文件”。
