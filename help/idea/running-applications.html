运行应用程序

IntelliJ IDEA支持使用main（）方法运行整个应用程序以及类。

IntelliJ IDEA利用“ 运行/调试配置”中定义的设置。项目中存在的所有运行配置在“ 选择运行/调试配置”列表中可用。

如果在运行/调试配置中启用了在运行前启动构建或工具的选项，则IntelliJ IDEA将运行该构建或工具，并在成功后运行该应用程序。否则，程序将立即启动。

如果要查看所有当前正在运行的应用程序的列表，请选择“运行” |“运行”。从主菜单显示运行列表 。有关详细信息，请参考查看运行过程部分。

请注意，开始运行会话后， 标记“ 运行工具”窗口的图标，并在“运行/调试配置选择器”中切换到指示运行过程处于活动状态。

运行一个应用程序

  • 在装订线中，单击 ，然后选择所需的命令。

  • 在主工具栏上，选择所需的运行配置，然后执行以下操作之一：

    • 选择运行|从主菜单运行“选定的配置” 。

    • 请点击 。

    • 按Shift+F10 。

  • 按Shift+Alt+F10 ，从列表中选择所需的运行配置，然后按Enter 。

    您还可以通过按以下按钮快速访问此弹出菜单F9 ，当您不运行任何调试会话时。

    在此弹出菜单中，您还可以：

    • 调用“ 编辑配置”对话框 。

    • 启动前编辑所选配置F4 。

    • 即时删除配置Delete 。

    • 从运行切换到调试，反之亦然（按住Shift ）。

    • 访问先前选择的配置1 。

    • 访问上下文相关的配置（ 2要么3 ）。

    • 如果要查看所有当前正在运行的应用程序的列表，请选择“运行” |“运行”。从主菜单显示运行列表 。有关详细信息，请参考查看运行过程部分。

使用main（）方法运行一个类

  • 在编辑器中打开该类，然后执行以下操作之一：

    • 在装订线中，单击 ，然后选择所需的命令。

    • 选择运行从上下文菜单中

    • 按Ctrl+Shift+F10

  • 在“项目”工具窗口中选择类，然后选择“运行” 从选择的上下文菜单中。
