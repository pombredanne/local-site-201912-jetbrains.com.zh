描述

调试时自动附加到子流程

如果选中此复选框，则IntelliJ IDEA将自动附加正在调试的进程的所有子进程。因此，如果父流程具有子流程，则它们的断点将始终有效。

收集运行时类型信息以了解代码

如果选中此复选框，则函数调用的类型将在调试期间保留，并传递给类型检查器。

清除缓存

单击此按钮可删除有关在运行时收集的参数类型的信息。

兼容Gevent

如果选中此复选框，则调试器将与Gevent-monkeypatched代码兼容。

此参数适用于Python> = 2.7，Python> = 3.5

兼容PyQt

如果在解释器上安装了PyQt，但未将其导入应用程序代码中，则可能会发生一些导入错误。取消选择此选项可修复这些错误。

如果您的解释器上安装了多个兼容PyQt的后端，则必须从列表中选择PyQt或PySide2后端。默认情况下，“ 自动”选项处于启用状态，这意味着将使用首先找到的后端。
