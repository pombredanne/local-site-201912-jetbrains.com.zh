创建EJB

IntelliJ IDEA提供了以下创建bean的方法：

  • 使用IntelliJ IDEA提供的广泛编码帮助，通过直接编辑 ejb-jar.xml文件进行手动操作。

  • 使用上下文菜单命令。

  2. 创建部分。

要创建Enterprise Java bean，请遵循以下常规步骤

  1. 打开EJB工具窗口 。

  2. 右键单击所需的模块，在上下文菜单中选择“ 新建 ”，然后在子菜单上选择bean类型。

    将打开“ 新建Bean”对话框。

    或者，在模块的上下文菜单上选择“ 跳转到源代码 ”以打开专用编辑器，在“ 常规”选项卡中单击“ 新建” ，或在“ EJB关系”选项卡中右键单击图背景，然后选择所需的bean类型。

  3. 指定bean名称，类和包 。

  4. 定义特定于bean的设置：

    • 对于实体Bean（1.x和2.x）和会话Bean（1.x和2.x）， 配置本地和远程接口 。

    • 对于实体Bean（1.x和2.x）， 配置主键和CMP Bean字段 。

    • 对于消息bean， 配置消息侦听器 。

    • 对于会话Bean， 配置服务终结点 。
