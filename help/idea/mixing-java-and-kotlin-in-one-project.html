在一个项目中混合Java和Kotlin

确保已安装并启用Kotlin插件。

将Java文件转换为Kotlin文件

  • 在主菜单中，选择代码|将Java文件转换为Kotlin文件 Ctrl+Shift+Alt+K 。生成的Kotlin文件将代替原始Java文件。

有关更多信息，请在一个项目教程中参考Mixing Java and Kotlin 。

在Kotlin项目中创建Java文件

  1. 创建一个新的Kotlin-JVM或Kotlin-JavaScript项目。

  2. 在“ 项目工具窗口”中 ，选择要在其中创建Java类的目标包或目录。

  3. 按Alt+Insert ，从弹出窗口中选择selecyt Java Class ，选择其种类并指定其名称。

在Java项目中创建Kotlin文件

  1. 在Java项目中，选择目标位置，然后按Alt+Insert然后从弹出菜单中选择Kotlin文件 。

  2. 如果这是您首次在Java项目中创建Kotlin文件，则标语会通知您有关问题。

    单击配置链接，然后选择所需的配置器。

  3. 在打开的“ 创建Kotlin Java运行时库”对话框中，执行以下操作：

    • 选择要配置的模块（如果项目中有多个模块）

    • 指定应使用哪个Kotlin运行时库。

请注意，将来无需执行配置操作。
