创建和删除Web应用程序元素-一般步骤

本主题描述Web模块中元素的一般过程。

向Web模块添加元素

  1. 通过选择查看|打开Web工具窗口。工具窗口|网页 。工具窗口将显示所有具有Web构面的模块。

  2. 右键单击所需的Web模块节点，然后选择“ 新建” |“新建”。上下文菜单上的Servlet / Filter / Listener 。

  3. 在打开的“ 新建Servlet /过滤器/侦听器”对话框中，键入元素的名称。

  4. 指定将在其中创建元素的包。

  5. 指定实现该元素的类。

  6. 单击确定 。创建一个新类并在编辑器中打开。

  7. 按照相关的特定于元素的主题中的说明进行操作。

从Web应用程序中删除元素

  1. 在Web工具窗口中，选择要删除的元素。

  2. 在上下文菜单上，选择“ 删除” ，然后在确认对话框中单击“ 是 ”。
