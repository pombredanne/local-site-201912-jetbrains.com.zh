介绍

本节介绍如何：

  • 获取XML模式的Java表示形式

  • 具有基于Java类生成的XML模式

  • 将 XML实例文档解组为Java内容树，反之亦然。

使用IntelliJ IDEA，可以使用以下数据绑定器之一完成此转换：

  • XML绑定的Java体系结构（JAXB）
  • XMLBeans

使用JAXB和XmlBeans

要使用JAXB和XmlBeans，请执行以下初步步骤：

  1. 下载并安装XMLBeans工具。

  2. 在“设置/首选项”对话框的“ 插件”页面上，确保已启用Web服务捆绑插件。

  3. 在“设置/首选项”对话框的“ Web服务”页面上，指定XmlBeans的安装目录以及以下Web服务引擎之一：
    • 地铁

    • GlassFish 。

    • XML Web Services参考实现的Java API（JAX-WS RI）
    • Java Web服务开发包（JWSDP）
