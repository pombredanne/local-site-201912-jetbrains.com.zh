OSGi Framework Instance Dialog

IntelliJ IDEA lets you use this dialog to configure OSGi framework instance. The OSGI Framework Instance dialog appears when you try to add a new OSGI framework instance from the Add OSGi Framework list, on the OSGi Framework Instances page.

Item

Description

Home Directory

Use this field to specify the location of your new OSGi framework.

Name

Use this field to specify the name of the framework. By default, this field contains a type of your OSGi framework that you are trying to add.

Version

Use this field to specify the version of the framework. As long as the home directory is specified this field is filled in automatically.
