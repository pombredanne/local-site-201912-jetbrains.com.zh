将TypeScript编译为JavaScript

由于浏览器和Node.js仅处理JavaScript，因此您必须在运行或调试它之前编译TypeScript代码。

编译还可以生成源映射 ，该源映射设置TypeScript代码与实际执行的JavaScript代码之间的对应关系。

IntelliJ IDEA带有内置的TypeScript编译器。默认情况下，它在TypeScript文件旁边输出生成的JavaScript文件和源映射。

在你开始前

打开 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” | TypeScript ，并确保已选中“ TypeScript语言服务”复选框。

创建一个tsconfig.json文件

默认情况下，内置编译器不会创建源映射，该源映射将使您在调试会话期间逐步执行TypeScript代码。默认情况下，编译器还会处理当前打开的TypeScript文件或从TypeScript页面的Compile Scope字段中选择的范围中的所有TypeScript文件，如TypeScript Language Service中所述 。

使用tsconfig.json文件，您可以修改此默认行为以生成源映射并仅编译来自自定义范围的文件。

  1. 在“ 项目”工具窗口中，选择TypeScript代码所在的文件夹（通常是项目根文件夹），然后选择“ 新建” |“新建”。 tsconfig.json来自上下文菜单的文件 。

  2. 要在编译过程中生成源地图，请确保sourceMap属性设置为true 。

编译TypeScript代码

  1. 要编译整个应用程序的TypeScript代码，请在TypeScript工具窗口中单击然后从列表中选择全部编译 。

  2. 要编译一个文件，请在编辑器中将其打开，单击然后从列表中选择文件的路径。

  3. 要从自定义范围编译文件，请确保文件已在files tsconfig.json的属性， 如上所述 。在“ TypeScript工具”窗口中，单击并选择tsconfig.json的路径。
