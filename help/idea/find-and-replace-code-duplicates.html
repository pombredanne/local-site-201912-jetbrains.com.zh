查找和替换代码重复项

使用“ 查找和替换代码重复项”操作，您可以查找与所选方法或常量字段相似的代码重复，并将其替换为对原始方法或常量的调用。它还可以让您指定搜索范围。

  1. 将插入号放置在您要搜索其重复项的方法或常量字段中。

  2. 从主菜单或上下文菜单中，选择“ 重构|查找和替换代码重复项 。

  3. 在打开的对话框中，选择IntelliJ IDEA查找代码重复项的范围。

  4. 对于找到的每个重复代码，IntelliJ IDEA会提示您确认替换。
