将类公开为Web服务对话框

Alt+Enter - 将类公开为Web服务

该对话框仅可通过专门的意图操作使用。要调用它，请将插入号放在类名上，然后按Alt+Enter或点击黄色灯泡图标 。

使用该对话框为整个类配置Web服务WSDL生成，其所有方法都作为Web服务操作公开和部署。对话框的内容取决于Web服务类型。

项目

描述

服务名称

使用此列表可以指定要发布的服务的名称（例如， ROOT上下文或mycontext对于/mycontext ）。

阿帕奇轴心

服务类别名称

在此字段中，指定要作为Web服务公开的类。

所有

服务命名空间

在此字段中，键入Web服务名称空间前缀。

阿帕奇轴心

服务风格

使用此列表可以指定要生成的WSDL文档的样式。可用的选项有：
  • RPC-选择此选项以生成rpc WSDL。

    默认情况下选中此选项。

  • 文档 -选择此选项以生成文档 WSDL。

  • 裹 -选择此选项，必须使用包装的方法所产生的WSDL。

    如果选择此选项，则会在“ 使用绑定中的项目”列表中自动选择“ 文字” 。

阿帕奇轴心

物品的使用

从此列表中，选择应如何使用生成的WSDL文档。可用的选项有：

  • 文字 -选择此选项后，请求的XML表示形式由XML模式定义。

  • 编码 -选择此选项可在生成的WSDL中指定SOAP编码。
