IntelliJ IDEA 2019.3帮助

OSGi

IntelliJ IDEA允许您使用OSGi插件创建基于OSGi的应用程序。IntelliJ IDEA还允许您导入Bnd / Bndtools项目 。您可以创建具有OSGi支持的项目，将具有OSGI支持的模块添加到现有项目，或将OSGi支持添加到现有模块。

创建一个新项目

  1. 启动IntelliJ IDEA。

    如果“欢迎”屏幕打开，则单击“ 创建新项目” 。

    否则，从主菜单中选择File |新品|项目 。

  2. 在“ 新建项目”向导中，从左侧列表中选择Java 。

  3. 从Project SDK列表中，选择要在项目中使用的JDK。

    如果列表为空，请单击“ 新建”并指定Java主目录的路径（例如/Library/Java/JavaVirtualMachines/jdk-12.0.1.jdk ）。

    如果您的计算机上没有Java，请参阅下载JDK软件包 。

  4. 从“ 其他库和框架”列表中，选择OSGi 。

  5. 如果您的计算机上没有必要的库，请在“ 库”部分中选择“ 下载” 。

    否则，选择“ 使用库” ，然后单击“ 创建” ，然后在打开的对话框中选择必要的文件。

    单击下一步 。

  6. 为项目命名并在必要时更改默认项目位置。点击完成

将OSGi支持添加到现有模块

  1. 在“项目”工具窗口中，右键单击要向其添加OSGi支持的模块，然后选择“ 添加框架支持” 。

  2. 在打开的对话框中，从左侧窗格中选择OSGi 。

  3. 如果您的计算机上没有必要的库，请在“ 库”部分中选择“ 下载” 。

    否则，选择“ 使用库” ，然后单击“ 创建” ，然后在打开的对话框中选择必要的文件。

    单击确定 。
