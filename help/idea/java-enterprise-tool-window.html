Java企业工具窗口

要打开此工具窗口：
查看|工具窗口| Java企业版
另请参阅显示工具窗口 。

Java Enterprise工具窗口使您可以从Java EE角度查看项目。

该工具窗口包括一组窗格，其中所选项目的“ Java EE内容”显示在右侧窗格中。如果合适，将在最右边的窗格中显示某项的Javadoc HTML文档（或有时为图表）。

工具栏按钮用于显示或隐藏项目的类别（例如，模块或CDI bean），或用于更改项目的显示方式。

上下文菜单包括用于在窗格之间切换（ Previous或Next ）和在编辑器中打开项目的命令（ Edit ）。

您可以通过指定相应的搜索字符串来过滤大多数窗格中的信息。

  • 工具栏图标
  • 上下文菜单命令
  • 筛选信息

工具栏图标

除了极少数例外，工具栏图标是可以打开或关闭的切换按钮。

前两个图标（ 和 ）始终可用。其余图标仅对某些Java EE技术和框架（例如CDI，WebSocket）显示。

图标

描述

图标的状态定义了Java EE技术和框架的显示级别。

启用后，技术将显示在第一层（即，最左侧的窗格中）。在这种情况下打开，并且模块显示在右侧窗格中，您可以选择一种技术，例如，查看在哪些模块中使用了所选技术。

禁用时，技术显示在第二层。也就是说，您的模块显示在最左侧的窗格中，而技术显示在最右侧的窗格中。在这种情况下，您可以选择一个模块，然后查看所选模块中使用了哪些技术。

如果亮起时，此图标用于显示或隐藏模块。在这种情况下，模块（如果显示）显示在左侧第二个窗格中。

对于CDI：使用此图标可以显示或隐藏CDI bean。

对于CDI：使用此图标可以显示或隐藏生产者方法和字段。

取决于技术：
  • 对于CDI：使用此图标可以在最右边的窗格中显示图表或文档之间进行切换。

    如果选中，将显示所选项目的依赖项注入图。否则，将显示Javadoc HTML文档。

  • 对于WebSocket：使用此图标可以更改端点类的显示和排序方式。

    如果禁用，则按以下顺序显示有关类的信息：首先显示类名，然后显示端点URL（如果有）。在这种情况下，这些类按其名称排序。

    如果选中，则将首先显示端点URL，然后是类名称。在这种情况下，这些类由端点URL排序。

  • 对于RESTfull Web服务：使用此图标可以更改资源类及其方法的显示和排序方式。

    如果未选中，则按以下顺序显示有关类和方法的信息：首先显示名称，然后显示资源URL。在这种情况下，类和方法按其名称排序。

    如果选中，则首先显示资源URL，然后按URL排序信息。

    注意，可以通过以下方式显示或隐藏资源类 。要显示或隐藏方法，请使用 。

对于CDI：使用此图标可以显示或隐藏InjectionPoint类型。

请注意，如果开启时，注射点显示在最右侧窗格中的图表上，并且该图标变为非活动状态。

对于WebSocket：使用此图标可以显示或隐藏带有注释的类@ServerEndpoint 。

对于WebSocket：使用此图标可以显示或隐藏带有注释的类@ClientEndpoint 。

对于RESTful Web服务：使用此图标可以显示或隐藏资源类。

如果未选中，则有一个窗格显示类。选择一个类时，其方法将显示在右侧窗格中。因此，在这种情况下，方法将按定义它们的类进行分组。

如果选中，则不显示类。属于不同类的方法都在同一窗格中同时显示。

对于RESTful Web服务：单击此图标，然后单击必要的选项以显示或隐藏带有注释的方法。 @GET ， @POST ， @PUT ， 要么@DELETE 。
