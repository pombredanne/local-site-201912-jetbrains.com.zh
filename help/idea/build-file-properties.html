蚂蚁构建工具窗口|构建文件的上下文菜单|性质

使用此对话框可以配置Ant构建脚本的行为。

项目

描述

最大堆大小

使用此字段可以更改分配给Ant构建过程的内存量。如果Ant进程内存不足，请增加此值。

最大堆栈大小（MB）

使用此字段可以更改堆栈存储器的大小。默认值为2 MB。

在后台进行构建

如果启用此选项，则构建过程将在后台运行。否则，将显示模式进度对话框。

编译需要大量的处理器资源，并且可能导致性能严重下降。当构建过程在后台运行时，您会发现IntelliJ IDEA无法响应您的操作。

如果没有发生错误，请关闭消息视图

如果启用此选项，则不会为成功的构建打开“ Ant消息”窗口。

属性标签

使用此选项卡可以指定应传递给构建脚本的运行时属性。这些属性等同于-D命令行选项。属性定义为一对“名称-值”。该值可以是硬编码的，也可以在运行时从宏动态评估。

加

使用此按钮可以在运行时属性列表中创建一个新条目。

去掉

使用此按钮可以从列表中删除选定的属性。

执行标签

使用此选项卡可以配置IntelliJ IDEA如何启动构建脚本。

与蚂蚁一起跑

在此选项组中，您可以确定要运行所选build.xml的Ant版本。

使用项目默认的Ant

单击此单选按钮以运行默认的Ant版本。

使用自定义蚂蚁

单击此单选按钮以运行您选择的Ant版本。从组合框中，您可以选择已配置并注册了IntelliJ IDEA的任何Ant版本。使用省略号按钮修改Ant配置。

默认设置

单击此按钮以配置默认的Ant。

蚂蚁命令行

使用此字段提供命令行参数。您可以包含任何参数，除了-logger 。参数应以破折号开头，并以空格分隔。

在JDK下运行

使用此字段来定义用于运行Ant目标的JDK。默认情况下，您可以选择项目或模块JDK。使用省略号按钮配置JDK。

其他类路径选项卡

使用此选项卡添加Ant在运行时加载的库和目录。

加

单击此按钮可将存档目录添加到类路径。

全部添加到目录

单击此按钮可将目录及其所有内容添加到类路径。

上/下移动

使用这些按钮可以更改类路径条目的顺序。对话框中的条目顺序定义了Ant加载资源的顺序。

去掉

单击此按钮可从列表中删除所选条目。

过滤器标签

应用过滤后，可使用此选项卡配置在Ant Build工具窗口中显示哪些构建目标。该选项卡显示所选构建文件的构建目标的列表。左列显示构建目标的名称，右列包含对目标的可选描述。如果选中了一个目标，则在按下后它将显示在Ant Build工具窗口中 ;否则，该目标将被隐藏。
