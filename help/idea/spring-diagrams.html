弹簧图

弹簧图可帮助您分析应用程序中的依赖关系。您可以双击图上的bean或文件以在编辑器的专用选项卡中将其打开。

要使用图，必须启用捆绑的UML插件。

Spring Bean依赖关系图

Spring Beans Dependencies图使您可以查看和分析项目中Bean之间的依赖关系。

对于Spring Boot，您还可以构建Runtime Beans图 。

打开图

  1. 右键单击要为其构建图的Bean。您也可以为一个类构建一个图表，以查看该类中的bean及其依赖关系。

  2. 从菜单中选择图 。

  3. 选择显示图弹出窗口 Ctrl+Alt+U在本地弹出窗口中打开图，或显示图 Ctrl+Shift+Alt+U在编辑器中打开图。

  4. 请点击 Spring Bean的依赖关系 。

    如果只想查看图中豆之间的直接依赖关系，请选择“ 邻居”模式 。

Spring应用程序上下文依赖关系图

Spring Application Context Dependencies图表使您可以查看多个配置文件之间的依赖关系，并分析它们如何包含和相互引用。

图上的红色箭头标记了错误（例如循环依赖项），使您可以轻松地从Spring配置中发现并删除它们。

打开图

  1. 右键单击要为其构建图的配置文件。

  2. 从菜单中选择图 。

  3. 选择显示图弹出窗口 Ctrl+Alt+U在本地弹出窗口中打开图，或显示图 Ctrl+Shift+Alt+U在编辑器中打开图。

  4. 请点击 Spring应用程序上下文依赖关系 。

蓝色实线代表@ComponentScan注释依赖项。

黄色实线代表@Enable注释依赖项。

虚线绿色代表@Import注释依赖项。

红线代表错误。

虚线虚线代表其他依赖项。

在Spring工具窗口中浏览依赖项

要访问弹簧工具窗口，请转到视图|工具窗口|春天 。

Spring工具窗口可帮助您在Spring组件和依赖关系之间导航。您可以查看项目中使用的Spring bean的定义，并查看它们与其他bean的关系。

Beans选项卡显示了项目中使用的Spring Bean的定义，它们的文档以及它们与其他Bean的关系。每个bean都标有一个图标- （在XML文件中定义的bean）； （通过以下方式声明的可自动发现的豆类： @Component注释）； （Spring添加的未明确定义的服务bean）。

使用左框架上的图标来更改Bean的显示方式，查看文档或构建图形。

MVC选项卡允许您查看Spring MVC框架的控制器映射。您也可以使用 Request Method选项可通过HTTP方法过滤映射。

如果您的配置包含Spring Data，则“ 数据”选项卡将为您显示存储库列表，它们的查询和它们的投影。
