创建Grails应用程序模块

您可以使用Grails模块创建新项目，也可以将新Grails模块添加到现有项目中。

创建Grails应用程序模块

  1. 请执行以下任一操作：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件” |“ 新建” 。 新增|项目 。

      结果，“ 新建项目”向导打开。

    • 如果要向现有项目中添加模块，请执行以下操作：打开要向其中添加模块的项目，然后选择“ 文件” |“添加” 。新增|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页的左侧窗格中，选择Grails 。在页面的右侧，指定以下设置：

    • 您将要使用的Project SDK 。

    • Grails SDK主页 -选择由库表示的本地Grails安装。

    • 创建create-app或create-plugin-根据要创建的内容选择以下选项之一。

    • 选项 -指定其他选项。

    单击下一步 。

  3. 指定名称和位置设置。

    点击完成 。
