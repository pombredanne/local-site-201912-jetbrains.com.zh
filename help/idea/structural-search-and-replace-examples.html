结构搜索和替换示例

一个陈述

$ Statement $;

将出现次数增加到一定数量，您可以找到包含最多指定数量元素的语句序列。

方法调用

$ Instance $。$ MethodCall $（$ Parameter $）

该模板与方法调用表达式匹配。如果出现次数为零，则意味着可以省略方法调用。

@不推荐使用的$ Instance $。$ MethodCall $（$ Parameter $）

您可以使用此模板查找已弃用的方法，并将其用作创建其他带注释的方法模板的原型。专门用于搜索对不赞成使用的方法的方法调用，您可以选择已经存在的对不赞成使用的方法的模板方法调用 。

有关如何创建搜索模板，请参阅结构搜索并替换 。

如果陈述

如果（$ Expr $）{$ ThenStatements $; } else {$ ElseStatements $; }

搜索注释和/或字符串文字

考虑一个想要查找包含“ foo”的注释或文字的人。搜索模板就像$SomethingWeWantToFind$要么"$SomethingWeWantToFind$" 。如果要查找包含某些特定单词（例如foo作为单词）的注释/字符串，则应将其指定为文本约束。

添加尝试/捕获/最终代码

如果要用一个替换语句try/catch/finally构造，可以建议以下一对搜索和替换模板。搜索模板为：

$ Statements $;

具有指定的最大出现次数作为约束。

替换模板为：

尝试{$ Statements $; } catch（Exception ex）{}

查找一个类的所有后代或实现某个接口的所有类

考虑以下搜索模板：

class $ Clazz $扩展了$ AnotherClass $ {}

要么

类$ Clazz $实现$ SomeInterface $ {}

作为变量$ AnotherClass $或$ SomeInterface $的文本约束，请指定基类或已实现接口的名称。

找到所有这样的方法

要查找同一接口方法的不同实现，请使用以下搜索模板：

class $ a $ {public void $ show $（）; }

指定文字约束$show$变量，然后启用选项。 此变量是搜索的目标 。

使用@Modifier查找包的本地方法和实例方法

IntelliJ IDEA建议为类的包本地和实例字段提供预定义的模板。这些模板使用@Modifier批注，当无法使用自然语言手段表达搜索目标时，该注释有助于描述搜索目标。

但是，如果需要搜索包本地方法或实例方法，则必须使用@Modifier注释自己创建相应的搜索模板。

要使用可见性修饰符包local和instance来指定查找所有方法的条件，请使用以下搜索模板：

类$ Class $ {@Modifier（“ packageLocal”）@Modifier（“ Instance”）$ ReturnType $ $ MethodName $（$ ParameterType $ $ Parameter $）; }}

在搜索中使用“包含在约束中”字段

现有示例使用以下模板：

LOG.debug（$ params $）;

放置if('_a) { '_st*; }哪里_a和_st是变量和*表示“ 包含在约束中”字段中出现零次或多次出现，并且选择“ 完全匹配”变量的“ 反转条件”复选框将导致搜索不包含在“ if声明。

搜索XML和HTML标记，属性及其值

搜索标签的最简单模板是<$tag$/> 。

通过对变量施加约束$tag$ ，您可以指定要查找的标签。例如，如果您指定li ，你们都会li标签。

考虑以下用于XML和HTML搜索的模板： <$tag$ $attribute$=$value$ /> 。例如，如果您指定文本过滤器id为了$attribute$变量和\d+正则表达式作为$value$变量，您可以在id属性。

使用脚本过滤器处理XML和HTML内容

删除所有id属性大于2的行

在搜索模板字段中，我们创建了一个模板，用于搜索所有li带有数字值的标签（ \d+ ）在id属性。我们将搜索扩展到具有此类值的整个字符串（ 搜索目标=完全匹配 ）。

我们使用以下Groovy脚本过滤这些行： d.getText().replaceAll (/"/, '').toInteger() > 2 。该脚本读取内容d变量并以字符串形式返回（例如， "1" ）。然后，脚本将替换所有引号，并将字符串值转换为整数并将其与2 。

在“ 替换模板”字段中，我们什么都没有删除整个字符串。搜索后，我们选择全部替换以执行替换。

将p标签中class属性的大写值转换为小写

在搜索模板字段中，我们创建了一个模板，用于搜索所有p具有大写值的标签（ [A-Z].*并匹配大小写 ） class属性。我们仅将搜索范围缩小到class值（ 搜索目标= b ）。

在替换模板字段中，我们创建一个新变量$d$并为其分配一个Groovy脚本（ b.getText().toLowerCase() ）。搜索后，我们选择全部替换以执行替换。
