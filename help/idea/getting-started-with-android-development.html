在你开始前

  • 下载并解压缩Android SDK 。强烈建议Android SDK主目录的路径不包含空格。

    Android SDK不能替代Java SDK（JDK）。无论如何，您都需要为项目下载并配置Java SDK 。

  • 添加SDK软件包。有关详细说明和下载链接，请参阅“ 添加SDK软件包” 。

  • 在IntelliJ IDEA中配置Android SDK，请参阅SDK 。

  • 确保捆绑的Android支持插件已激活。此插件在IntelliJ IDEA级别提供Android支持，并且默认情况下处于启用状态。如果不是，请按照“ 管理插件”中所述启用它。

  • 根据您的任务， 选择 Android开发所需的模块类型 。

  • 从头开始使用Android模块创建项目，将Android模块 添加到项目中，或将Android Facet附加到现有Java模块上。

    根据所选的模块类型，IntelliJ IDEA会使用res和gen文件夹自动设置正确的模块结构，下载必要的库，并生成各种Android特定的描述符。

从IntelliJ IDEA版本2016.1开始，不支持非Gradle Android项目。

选择您需要的模块类型

IntelliJ IDEA允许您在创建项目时选择应用程序模块（形状因数），将不同的Android模块添加到现有项目中，或将Android构面添加到模块中。

创建一个Android项目

  1. 在菜单中，选择文件|新品| Project打开新建项目向导 。在左侧窗格中，选择Android 。

  2. 在右侧窗格中配置新项目，然后单击下一步 。

  3. 在向导的下一页上，选择应用程序模块类型及其最小的SDK。单击下一步 。

  4. 在向导的下一页上，选择Android活动 ，然后单击下一步 。

  5. 在向导的下一页上，指定“ 活动”设置 ，然后单击“ 完成” 。

将Android模块添加到项目中

  1. 选择文件|新品|从主菜单或新 模块 |从“ 项目”工具窗口的上下文菜单中选择模块 。

    将打开“ 新建模块”向导 。

  2. 在左侧窗格中，选择Android 。在右侧窗格中，选择要添加的模块，然后单击下一步 。

  3. 在向导的下一页上，指定应用程序或库名称，模块名称，程序包名称以及此类应用程序所需的最低SDK。单击下一步 。

  4. 在向导的下一页上，为模块选择一个活动，然后单击下一步 。

  5. 在向导的下一页上，指定设置以自定义所选活动 ，然后单击完成 。

将Android Facet附加到现有Java模块

如果要将Android Facet附加到现有Java模块，则需要执行此过程。Android模块默认情况下会应用此方面。

  1. 打开项目结构对话框（例如Ctrl+Shift+Alt+S ）。

  2. 在“ 项目设置”下 ，选择“ 模块” 。

  3. 选择要向其添加Android Facet的模块，然后单击 ，然后选择Android 。

  4. 在打开的Facet'Android'页面上，在“ 结构”选项卡中指定关键应用程序组件的位置： AndroidManifest.xml文件， 应用程序资源 ， 应用程序资产和Android本机库。 。

    如有必要，您可以编辑默认路径。要返回默认的Android方面设置，请点击重置默认路径 。

  5. 要使模块的源代码和资源可用于其他项目 ，请选中此页面顶部的“ 库”模块复选框。

配置特定于Android的XML定义文件的代码样式

Android开发涉及使用专用XML文件，例如布局和资源定义文件，清单文件等。您可以让IntelliJ IDEA将标准XML代码样式应用于此类文件，或为其配置自定义代码样式设置。

  1. 通过按打开设置/首选项对话框 Ctrl+Alt+S或选择文件|设置 Windows和Linux或IntelliJ IDEA的| macOS的首选项 。展开“ 编辑器”节点，然后单击代码样式下的XML 。

  2. 在打开的“代码样式：XML”页面上，切换到“ Android”标签。

  3. 请执行以下任一操作：

    • 要为特定于Android的XML文件定义自定义代码样式，请选中“ 使用Android XML文件的自定义格式设置”复选框，并使用“ 代码样式”中所述的选项卡控件配置要应用于各种类型的Android XML文件的设置： XML-Android 。

    • 要使IntelliJ IDEA根据页面其他选项卡中定义的标准XML代码样式设置来格式化Android专用的XML文件，请清除“ 为Android XML文件使用自定义格式设置”复选框。
