Changing Default Facet Settings

For certain facet types (e.g. GWT), you can specify default settings. These settings are applied when a new facet of the corresponding type is created.

Note that there are no default settings for most of the facet types.

To change default facet settings

  1. Open the Project Structure dialog (Ctrl+Shift+Alt+S).
  2. In the left-hand pane, click Facets.
  3. In the central pane, select the required facet type.

    Note that the facet types are shown for existing facets only, and appear at the top hierarchical level. The existing facets are shown within the corresponding type, at the second level.

  4. The default settings (if any) are shown in the right-hand part of the dialog. If there is the Defaults tab there, this is the one that you want.

    Configure the default settings as necessary.

  5. Click Apply or OK in the Project Structure dialog.
