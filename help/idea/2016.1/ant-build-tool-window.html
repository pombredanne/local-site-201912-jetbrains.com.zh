Ant Build Tool Window

View | Tool Windows | Ant Build

Use this tool window to add Ant build scripts to your project, control behavior of the build, and run build targets.

  • Prerequisites
  • Ant Build
  • Ant build results

Prerequisites

Before you start working with Ant Build tool window, make sure that Ant support plugin is enabled. The plugin is bundled with IntelliJ IDEA and activated by default. If it is not, enable the plugin as described in the section Enabling and Disabling Plugins.

Ant build results

Results of running Ant targets or the entire builds are shown in the Messages tool window.
