Viewing Pages with Web Contents

With IntelliJ IDEA, you can perform two opposite operations:

  • Preview the output of your Web application in the browser to check whether the pages are rendered correctly. IntelliJ IDEA can display a page preview in a browser of your choice or you can switch between several browsers. IntelliJ IDEA currently supports previews in the following browsers:
    • Mozilla Firefox
    • Internet Explorer
    • Safari
    • Chrome
    • Opera

    No preliminary steps are required to view HTML code. However, to preview PHP pages in the browser, you need to configure synchronization with a server first.

  • View the HTML source code of a Web page in the IntelliJ IDEA editor.
