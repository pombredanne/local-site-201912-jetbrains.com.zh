Opening JPA or Hibernate ER Diagram

Entity Relationship diagram (ER diagram) enables you to review your model, manage fields, relationships between entities, and mapping files.
For the detailed information about controls and context menu commands, refer to Diagram Toolbar and Context Menu. For the description of the basic techniques, refer to the section General Techniques of Using Diagrams.

To open an ER diagram for a persistence unit

  1. In the Persistence tool window, select the desired persistence unit or session factory.
  2. In the context menu, select  ER Diagram.
