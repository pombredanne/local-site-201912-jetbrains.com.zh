EJB Tool Window | context menu of a session bean | Jump to Source

This section describes the fields that are specific for Session beans.

ItemDescription
Session Type Select session type (stateful or stateless) depending on the conversational state the bean should have.
Transaction TypeSpecify an object the transactions should be managed with. The possible options are a bean and its container.
