Android Activity Settings Page

Use this page to specify the Android Activity settings.

ItemDescription
Activity NameUse this field to specify the name of your activity. This field appears if you selected either Blank Activity option or Fullscreen Activity option.
Layout NameUse this field to specify the name of the layout. This field appears if you selected either Blank Activity option or Fullscreen Activity option.
Fragment Layout NameUse this field to specify the name of the fragment layout. This field appears if you selected the Blank Activity option.
Navigation TypeUse this drop-down list to select a type of navigation for the activity. This drop-down list appears if you selected the Blank Activity option.
Object KindThis field appears if you selected the Master/Detail Flow option.
Object Kind PluralThis field appears if you selected the Master/Detail Flow option.
PreviewThis area shows the preview of the selections you have made on this page.
Description This area shows a short description of the selected activity.
