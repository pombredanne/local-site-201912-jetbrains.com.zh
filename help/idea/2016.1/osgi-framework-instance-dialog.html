OSGi Framework Instance Dialog

IntelliJ IDEA lets you use this dialog to configure OSGi framework instance. The OSGI Framework Instance dialog appears when you try to add a new OSGI framework instance from the Add OSGi Framework drop-down list, on the OSGi Framework Instances page.

ItemDescription
Home DirectoryUse this field to specify the location of your new OSGi framework.
NameUse this field to specify the name of the framework. By default, this field contains a type of your OSGi framework that you are trying to add.
VersionUse this field to specify the version of the framework. As long as the home directory is specified this field is filled in automatically.
