Adding Frameworks Support

You can extend capabilities of an existing module by adding support for various development frameworks and technologies.

To add frameworks support to a module, do one of the following

  • In the Project tool window, right-click the desired module node, and choose Add Framework Support from the context menu. Then, in the Add Frameworks Support Dialog, select the check boxes for the technologies you want to be enabled in your module, and specify the associated settings.
  • Open the module settings and click . Choose the desired facet from the list. Configure the facet settings as required.
