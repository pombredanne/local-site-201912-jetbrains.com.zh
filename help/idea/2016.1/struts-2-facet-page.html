Struts 2 Facet Page

File | Project Structure | Modules - module - Struts 2 facet
Ctrl+Shift+Alt+S

The page is available only for modules with enabled Struts 2 support.

Use this page to configure the settings for an individual Struts 2 facet attached to a particular module.

The following tabs are available:

  • File Sets tab
  • Features tab

File Sets tab

ItemDescription
 (Alt+Insert)Use this icon or shortcut to create a new file set or add a file to the selected file set. (The Edit File Set dialog will open.)
 (Enter) Use this icon or shortcut to edit the selected file set. (The Edit File Set dialog will open.)
 (Alt+Delete)Use this icon or shortcut to remove the selected file or file set.

ItemDescription
