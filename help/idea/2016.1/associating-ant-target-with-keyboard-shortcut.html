Associating Ant Target with Keyboard Shortcut

You can associate a build target with a keyboard shortcut and execute commonly-used targets with a single key-stroke. If an Ant build file is added to the project, its targets appear under the Ant Targets node in the Keymap dialog box.

To associate a keyboard shortcut with a build target

  1. In the Ant Build Tool Window, right-click the desired build target.
  2. On the context menu, choose Assign Shortcut. Keymap dialog is opened.
  3. Configure keymap, as described in the section Configuring Keyboard Shortcuts.
