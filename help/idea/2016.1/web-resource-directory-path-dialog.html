Web Resource Directory Path Dialog

File | Project Structure | Modules - module - Web facet - Web Resource Directories - New

Use this dialog box to add a Web resource directory path to a module with a Web facet and map this path to the deployment directory root.

ItemDescription
Web resource directory pathIn this text box, specify the local directory where the desired Web resource resides. Type the path manually or click the Browse button and choose the necessary location in the dialog that opens.
Relative path in deployment directoryIn this text box, type the directory to deploy the Web resource to relative to the deployment directory root. For example, if you type a forward slash /, the files from the Web resource directory will be copied into the deployment root directory.
