Android-Gradle Foreground Configuration Page

Use this page to specify settings for the Android-Gradle screen configuration.

ItemDescription
ForegroundYou can select one of the following options:
  • Image
  • Clipart
  • Text
Image fileThis field appears after the Image option is selected.
ClipartThis option appears after the Clipart option is selected. Use Choose button to select the appropriate clipart image.
TextThis field appears after the Text option is selected. Use this field to enter a text that you need.
FontThis drop-down list appears after the Text option is selected. Use this list to select the appropriate font.
Trim surroundings blank spaceUse this check box to trim surroundings blank space.
Additional paddingUse this slider to change the padding.
Foreground scalingYou can select one of the following options:
  • Crop
  • Center
ShapeYou can select one of the following options:
  • None
  • Square
  • Circle
Background colorThis box shows the background color. To change the background color, right-click on the box and select the color you need. For more information about colors, see Color Picker options.
Foreground colorThis box appears after the Clipart option or the Text option is selected. To change the foreground color, right-click on the box and select the color you need. For more information about colors, see Color Picker options.
PreviewThis area shows a preview of your selections on the page.
