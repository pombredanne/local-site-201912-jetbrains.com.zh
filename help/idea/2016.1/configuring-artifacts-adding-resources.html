Configuring Artifacts: Adding Resources

To add resources to an artifact

  1. Open the Output Layout tab.
  2. In the Available Elements pane, select the element you want to add to the artifact that is being configured.
  3. Drag the selected element to the desired target node (archive or folder) in the Artifact Layout pane.

    Alternatively, you can double-click the selected element or choose Put on the context menu of the selection. In this case, IntelliJ IDEA adds the selected element to the default location depending on the type of the artifact and the element.

    To view the contents of elements in the layout tree, select the Show Content of Elements check box. Specify the elements to show contents of by selecting the relevant options on the context menu.
