Using the Flow Type Checker

IntelliJ IDEA provides basic support of Flow static type checker that brings type annotations to JavaScript. This support involves recognition and syntax highlighting of Flow structures on all operating systems. If you are using Unix or OS X, you can also run Flow in the command-line mode.

For information on installing Flow, see Getting Started with Flow.

To have IntelliJ IDEA recognize Flow structures, provide correct syntax highlighting, and avoid false-positive error highlighting, change the JavaScript language level to Flow:

  1. Open the Settings / Preferences Dialog by choosing File | Settings for Windows and Linux or IntelliJ IDEA | Preferences for OS X, and click JavaScript under Languages&Frameworks.
  2. On the JavaScript page that opens, choose Flow from the JavaScript Language Version drop-down list.
