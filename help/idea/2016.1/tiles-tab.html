Tiles Tab

View | Tool Windows | Struts Assistant

The tab is available from the Struts Assistant tool window.

Use this tab to manage the Struts tiles in your application.

Root node tiles-defs.xml

ItemName and tooltipDescription
tiles-defs.xml The root node of the application tiles' structure tree.
Context menu (same for all nodes)
F4 Jump to sourceSelect this menu item to switch to the code block in the tiles-defs.xml file, text view, that corresponds to the selected item in the tree.
Expand AllClick this button to expand all nodes.
Collapse AllClick this button to collapse all nodes.

Tile definition

ItemName and tooltipDescription
Tile The node under which an application tile is defined.
Context menu
Add PutSelect this menu item to add a new <put> element to the tile definition.
Add Put ListSelect this menu item to add a list of <put> elements to the tile definition.
Remove DefinitionSelect this menu item to remove the tile definition.
