Validation Tab

File | Project Structure | Artifacts
Ctrl+Shift+Alt+S | Artifacts
| Artifacts

In this tab, appoint server-specific validators to perform real-time validation of applications, in case your server poses additional requirements to Web components you are deploying.

This server-specific validators are presently supported for the Weblogic application servers only.

The tab is not available for artifacts of the types Jar and Other.

For the details on common options, click here.

ItemDescription
Application Server Specific ValidatorsThis read-only field shows a list of current application server validation extensions to apply.
Click this button to select the class that implements the relevant validator in the dialog box, that opens.
Click this button to remove the selected entry from the list.
