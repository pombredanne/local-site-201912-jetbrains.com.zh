Artifacts to Deploy dialog

This dialog lets you manage the list of deployment artifacts for an application server run/debug configuration.

Shown under Available and Chosen are the artifacts defined in your project. By moving the artifacts between the panes, you compose the list of the artifacts to be deployed on the corresponding application server. (Under Chosen, the artifacts selected for deployment are shown.)

Item Description
Add Click this button to move the artifacts selected in the Available pane to the Chosen pane.
Remove Click this button to move the artifacts selected in the Chosen pane to the Available pane.
Add all Click this button to move all the artifacts to the Chosen pane.
Remove all Click this button to remove all the artifacts from the Chosen pane.
