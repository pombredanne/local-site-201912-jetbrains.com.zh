Passwords

File | Settings | Appearance and Behavior | System Settings | Passwords for Windows and Linux
IntelliJ IDEA | Preferences | Appearance and Behavior | System Settings | Passwords for OS X
Ctrl+Alt+S

Specify whether IntelliJ IDEA should remember or save your passwords - ones you use to access password-protected resources such as version control repositories.

If necessary, set a master password. If you do so, instead of the individual passwords - if they are stored in the IntelliJ IDEA password database - you'll be able to use your master password.

Note that the password storage policy defined on this page doesn't affect your database passwords. Whether those passwords should be saved is specified for each of the passwords separately, in the settings for the corresponding data sources. See, for example, DB data source settings.

  • Password storage policy
  • Disk storage protection

Password storage policy

Select one of the following options:

  • Do not remember passwords. IntelliJ IDEA will ask you for a password each time it needs to access the corresponding password-protected resource.
  • Remember passwords until the application is closed. IntelliJ IDEA will ask you for a password only the first time it needs to use that password. Then, until you quit the IDE, you won't be asked for that password again.
  • Save on disk with master password protection. The passwords will be stored encrypted in the IntelliJ IDEA password database. To access this database, a master password will be used.

    Instead of the passwords themselves, IntelliJ IDEA will ask you for the master password. For each of the passwords stored in the database, this will happen only once during each session.

Disk storage protection

Master Password. Click this button to set or change the master password. (The Setup Master Password or the Change Master Password dialog will open.)

The master password is used to access the IntelliJ IDEA password database and also to strengthen the encryption of the stored passwords.

Enabled/Disabled. The text to the right of the Master Password button reflects the state of the master password. Enabled means that the master password is set and is not empty. Disabled means that the password is not set or is empty.
