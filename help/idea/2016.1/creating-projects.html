Creating Projects

IntelliJ IDEA suggests several ways of creating a project. You can create a project from scratch, or create a project by importing existing source files. These may be the sources originating from Eclipse, Flash Builder, Gradle, or Maven, or a collection of sources of "unspecified origin".
