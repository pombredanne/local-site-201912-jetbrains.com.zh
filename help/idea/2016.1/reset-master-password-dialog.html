Reset Master Password Dialog

This dialog opens when you click Reset in the Enter Master Password or the Change Master Password dialog.

If you reset the master password, all your passwords currently stored in the password database will be lost.

Specify your new master password.

You can set the password empty. As a result, IntelliJ IDEA won't ask you for the password when accessing the password database. However, the security of your passwords in that case will be lower.

To come back to the previous dialog, click Enter or Change.

Item Description
New password Specify your new master password.
Confirm password Type the new master password once more.
Encrypt with OS user credentials For Windows only: if you select this check box, the master password itself will be stored encrypted in the password database. (Your Windows logon credentials will be used for encryption.)

As a result, IntelliJ IDEA won't ask you for the master password when accessing the password database.

Enter or Change If you decided not to reset the password (e.g. you remembered it), click this button to come back to the previous dialog. This may be the Enter Master Password or the Change Master Password dialog.
