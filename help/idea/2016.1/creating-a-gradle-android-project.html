Creating a Gradle-Android Project

You can create a Gradle-Android project or add a new Gradle-Android module to an existing project.

To create a Gradle-based Android project

  1. Do one of the following:
    • If you are going to create a new project: click Create New Project on the Welcome screen or select File | New | Project.

      As a result, the New Project wizard opens.

    • If you are going to add a module to an existing project: open the project you want to add a module to, and select File | New | Module.

      As a result, the New Module wizard opens.

  2. On the first page of the wizard, in the left-hand pane, select Android. In the right-hand part of the page, select Gradle: Android Module. Click Next.
  3. Specify your Android module settings, and click Next.
  4. Specify the settings for your Android-Gradle Foreground configuration, and click Next.
  5. Select an Android application template from the list, and click Next.
  6. Specify the settings for the selected Android Activity template, and click Next.
  7. Specify the name and location settings. For more information, see Project Name and Location or Module Name and Location.

    Click Finish.
