Packaging a Module into a JAR File

Before packaging a Java module into a JAR file, you need to configure an artifact of the type JAR. This artifact can define the output of one or more modules.

To build a JAR file from a module

  1. On the main menu, choose Build | Build Artifact.
  2. From the drop-down list, select the desired artifact of the type JAR.

    The list shows all the artifacts configured for the current project. To have all the configured artifacts built, choose the Build all artifacts option.
