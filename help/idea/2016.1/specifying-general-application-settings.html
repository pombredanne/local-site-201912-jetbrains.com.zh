Specifying General Application Settings

Specify the name of the application, provide a brief description, and define in which environment the application can run.

To specify the general application settings

  1. Open the web.xml application descriptor.
  2. Switch to the General tab.
  3. In the Display name text box, specify the short display name for the Web application.
  4. In the Description text box, type some descriptive text about the application.
  5. To enable running the application in a distributed environment, select the Distributable check box.

Alternatively, you can switch to the Text tab and specify all these settings manually.
