Configuring Artifacts: Arranging Elements

To arrange the elements included in an artifact

  1. Open the Output Layout tab.
  2. In the Output Layout pane, define the desired order of elements within an artifact. If necessary, extract fragments into separate artifacts or disassemble artifacts. To do so, use the toolbar buttons and options of the context menu.
