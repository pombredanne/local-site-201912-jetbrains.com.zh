Time Tracking

File | Settings | Tools | Tasks | Time Tracking for Windows and Linux
IntelliJ IDEA | Preferences | Tools | Tasks | Time Tracking for OS X
Ctrl+Alt+S

Use this page to enable time tracking and configure its settings.

ItemDescription
Enable Time TrackingSelect this check box to enable time tracking in the current project. When this check box is selected, the Time Tracking tool window becomes available on the right tool window bar, and on the View | Tool Windows menu.
Suspend delayIn this field, enter the "accuracy" of time tracking. If no user activity detected during this time, task is considered suspended.
