Facet Page

File | Project Structure | Modules
Ctrl+Shift+Alt+S

The Facet page opens in the right-hand part of the Project Structure dialog when you select a facet in the element selector pane.

Facet settings vary depending on the facet type:

  • Android Facet Page
  • AspectJ Facet Page
  • Android-Gradle Facet Page
  • Bean Validation Facet Page
  • Google App Engine Facet Page
  • GWT Facet Page
  • Hibernate and JPA Facet Pages
  • JSF Facet Page
  • OSGi Facet Page
  • Seam Facet Page
  • Spring Facet Page
  • Struts Facet Page
  • Struts 2 Facet Page
  • Tapestry Facet Page
  • Web, EJB and Java EE Application Facet Pages
  • Web Services Facet Page
  • Web Services Client Facet Page
