Spring Boot

The Spring Boot support in IntelliJ IDEA includes:

  • Integration of Spring Initializr that allows creating a custom Spring Boot starter project using a pre-configured project template directly from the New Project wizard.
  • Advanced coding assistance for editing your application configuration which includes code completion, error highlighting, navigation and quick fixes.
  • A dedicated Run/Debug Configuration that lets you quickly override the Spring Boot settings.
