为Ant构建目标配置触发器

您可以配置在某些事件之前或之后运行Ant目标的触发器，例如，编译，运行或测试。这些触发器分别称为编译触发器和执行触发器 。

设置编译触发器

  1. 在“ 蚂蚁构建工具窗口”上 ，右键单击所需的构建目标。

  2. 在目标的上下文菜单中选择执行上 。

  3. 在子菜单上，根据需要选择“ 编译前”或“ 编译后” 。请注意，您可以设置两个选项。

设置执行触发器

  1. 在“蚂蚁构建”工具窗口上，右键单击所需的构建目标。

  2. 在目标的上下文菜单中选择执行上 。

  3. 在子菜单上，选择“运行/调试之前” 。出现“运行/调试之前执行目标”对话框，显示可能的执行类型列表。

  4. 在对话框中，选中应触发所选Ant目标的执行类型旁边的复选框，然后单击OK 。
