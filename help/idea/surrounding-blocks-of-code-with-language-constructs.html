环绕声代码片段

IntelliJ IDEA根据源代码的语言为周围的代码片段提供了具有各种结构的标准模板。这包括if...else条件语句do...while和for循环，组合try...catch...finally ，XML标签，折叠区域和其他结构。

用语言构造包围代码块

  1. 选择所需的代码片段。

  2. 在代码菜单上，单击环绕 Ctrl+Alt+T 。

  3. 从列表中选择必要的环绕声语句。

您可以编辑环绕语句中使用的代码模板。 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|文件和代码模板 。

例如，您可以配置最终修饰符，名称或什至是例外的类型try...catch有条件的声明。
