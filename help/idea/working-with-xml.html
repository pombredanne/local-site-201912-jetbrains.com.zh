XML格式

IntelliJ IDEA带来了强大的XML编辑支持，其中包括：

  • 验证和语法突出显示。

  • 代码完成 Ctrl+Space 。

  • 缩进 （ Ctrl+Alt+I ， Ctrl+Alt+L ）。

  • 格式化 Ctrl+Alt+L根据XML代码样式 。

  • 意向动作 Alt+Enter 。

  • 导入未绑定的名称空间 。

  • 查看代码结构 Alt+7 。

  • 在源代码中导航 Ctrl+B 。

  • 快速文档 Ctrl+Q 。

  • 搜索用法 Alt+F7 。

  • 评论和取消评论行 （ Ctrl+/ ， Ctrl+Shift+/ ）。

  • 展开和删除标签 Ctrl+Shift+Delete 。

  • 从实例文档生成模式，反之亦然。

  • 生成DTD文件 。

IntelliJ IDEA使用Xerces 2.11 ，这是由Apache Software Foundation Group开发的XML解析器。

配置语法突出显示

您可以根据自己的喜好和习惯配置可识别XML的语法。

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|配色方案| XML 。

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
