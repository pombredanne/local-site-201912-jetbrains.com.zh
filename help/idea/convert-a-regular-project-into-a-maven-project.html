将常规项目转换为Maven项目

您可以打开现有的非Maven项目，并通过IntelliJ IDEA UI添加Maven支持。

添加Maven支持

  1. 打开一个现有项目，例如Java项目。

  2. 在“ 项目”工具窗口中，右键单击您的项目，然后选择“ 添加框架支持” 。

  3. 在打开的对话框中，从左侧的选项中选择“ Maven ”，然后单击“ 确定” 。

    IntelliJ IDEA将默认的POM添加到项目中，并在“ 项目”工具窗口中生成标准的Maven布局。IntelliJ IDEA还在Maven工具窗口中使用Lifecycle和Plugins创建相应的结构。

  4. 打开生成的POM并指定一个groupId 。的artifactId和version是自动指定的。

    每次更改POM时，IntelliJ IDEA都会显示一个弹出窗口，提示您导入更改。

此时，您可以使用Maven进一步开发您的项目。我们建议您在POM中进行所有项目更改，因为IntelliJ IDEA认为pom.xml是唯一的事实来源。

创建一个可执行的JAR

您可以结束以下可选步骤来创建可执行JAR。

  1. 请点击建立项目。IntelliJ IDEA生成目标文件夹。请注意，IntelliJ IDEA仅编译源文件，不创建JAR文件或清单文件。

  2. 在资源目录中创建清单文件。

  3. 在您的POM中指定清单文件信息，以便您可以使用Maven生成可执行的jar文件。

  4. 在Maven工具窗口的“ 生命周期”列表中，双击install命令以生成jar文件。IntelliJ IDEA在目标文件夹中生成适当的信息，并在“ 项目”工具窗口中生成可执行的JAR。

    您可以右键单击生成的JAR，然后选择“运行”以执行文件。

如果现有项目包含多个模块，那么将这样的项目转换为Maven项目将变得非常困难。在这种情况下，我们建议您创建一个外部POM，在其中描述您的项目并像打开常规Maven项目一样打开 POM。
