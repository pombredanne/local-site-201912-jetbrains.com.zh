资源包

资源束是一组具有相同基本名称和特定于语言的后缀的属性文件 。例如，如果您创建file_en.properties和file_de.properties ，则IntelliJ IDEA将识别它们并将其组合到资源束中。

IntelliJ IDEA使用图标标记资源包 。

创建一个新的资源包

  1. 在“ 项目”工具窗口中，选择应在其中创建新资源包的目录。

  2. 在“ 文件”菜单上，指向“ 新建” ，然后单击“ 资源包” ，或按Alt+Insert然后单击“ 资源包” 。

  3. 在打开的对话框中，执行以下操作：

    1. 指定资源束的基本名称。

    2. 如有必要，请选中使用基于XML的属性文件复选框。

    3. 添加所需的语言环境。为此，请单击并键入所需语言环境的逗号分隔后缀。

    4. 准备好后，单击“ 确定” 。

新的节点Resource Bundle' '出现在Project Tool窗口中 ：

默认情况下，捆绑软件包含所有指定语言环境的属性文件。您可以解除关联，仅显示属性文件，而不显示捆绑软件。

解除资源束的关联

  1. 右键单击要解除关联的资源包。

  2. 在上下文菜单中，单击“ 解除资源捆绑” 。

将几个属性文件合并到资源束中

  1. 选择要合并的属性文件。

  2. 右键单击所选内容。

  3. 在上下文菜单中，单击“ 合并到资源包” 。

  4. 指定资源束的基本名称。

编辑资源包

一旦创建了多个具有相同名称，不同区域设置后缀的属性文件 ，IntelliJ IDEA就会自动识别它们并将它们在“项目”视图中分组为一个资源束。

IntelliJ IDEA包括一个资源束编辑器，用于方便地编辑属性文件中的可本地化字符串。

要打开资源包编辑器，请执行以下任一操作：

  • 在“ 项目”工具窗口中，右键单击资源束，然后选择“ 跳转到源代码” 。

  • 在“ 项目”工具窗口中选择一个资源束，然后按F4 。

  • 打开属于捆绑包的属性文件，然后单击编辑器底部的“ 资源捆绑包”选项卡：

编辑属性键

  1. 打开一个属性文件。

  2. 根据需要添加，更改或删除密钥。所做的更改将反映在Resource Bundle编辑器中。

使用资源包编辑器更改属性值，这将确保您同时编辑整个属性文件集。IntelliJ IDEA在捆绑软件的每个文件中创建相应的记录。

编辑属性值

  1. 在资源包编辑器的左窗格中选择属性键。

  2. 在目标语言环境框架中，根据需要编辑值。相应的属性文件将相应更新。

编辑资源包时，请记住以下几点：

  • 没有值或在其中一个属性文件中省略的属性以红色突出显示。

  • 在转义序列之间进行转换（例如， \u00df ）和Unicode文字（相应的国家字符，例如ß ）中的属性文件，然后在资源包编辑器中，选中“设置/首选项”对话框的“ 文件编码”页面上的“ 透明的本机到ASCII转换”复选框。

  • 可以使用大写和小写十六进制序列对非ASCII符号进行编码（例如， \u00E3和\u00e3 ）。默认情况下，仅启用大写序列。要使用小写的十六进制序列，请设置idea.native2ascii.lowercase将idea.properties文件中的属性true 。

    有关更多信息，请参阅配置平台属性 。
