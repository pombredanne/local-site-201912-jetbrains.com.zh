Importing Adobe Flash Builder Projects

  1. Launch IntelliJ IDEA.

    If the Welcome screen opens, click Import Project.

    Otherwise, from the main menu, select File | New | Project from Existing Sources.

    You can add a Flash Builder project to another existing project as a module. In this case, select File | New | Module from Existing Sources.

  2. In the dialog that opens, select the directory in which your sources, libraries, and other assets are located and click Open.

  3. Select Import project from external model | Flash Builder and click Next.

  4. Configure the project:

    • Flash Builder workspace or project: specify the location of a file or directory that you want to use as a source of import (a Flash Builder workspace or project directory, a .project file, a .fxp or .fxpl file, a .zip file that contains an ActionScript project.

    • Project name: specify the name of the project after the import.

    • Project files location: specify the directory in which you want to store the project. If you are importing an .fxp, .fxpl, or zip archive, it will be extracted into this folder or its subfolder.

    • Project format: it's recommended that you use the directory-based format.

    Click Next.

  5. If you are importing a Flash Builder workspace, the page shows the detected projects. Select the ones that you want to import.

    Click Finish.
