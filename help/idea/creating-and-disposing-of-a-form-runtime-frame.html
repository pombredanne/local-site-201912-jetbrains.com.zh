创建和处理表单的运行时框架

为了使您的表单正常工作，您必须为其提供一个运行时框架。GUI表单的main（）方法负责创建和处理此类框架。

为表单创建main（）方法

  1. 打开以编辑要在其中创建main（）方法的表单的绑定类。

  2. 按Alt+Insert 。

  3. 在弹出菜单上，单击Form main（） 。将以下方法添加到源代码中：

    公共静态无效main（String [] args）{JFrame frame = new JFrame（“ ”）; frame.setContentPane（new （）. contentPane ）; frame.setDefaultCloseOperation（JFrame。EXIT_ON_CLOSE）; frame.pack（）; frame.setVisible（true）; }
