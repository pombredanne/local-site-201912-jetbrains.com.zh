运行笔记本并分析数据

要预览和分析数据集，您需要运行笔记本的可执行段。

运行笔记本

您可以一次或全部一次运行段落。执行任何段落时，请注意代码依赖性。例如，如果当前段落依赖于上一段中初始化的变量，则需要首先执行它们。

  • 请点击在笔记本编辑器工具栏上可以执行笔记本的所有段落。执行进度将显示在工具栏上。

  • 请点击装订线中的图标以执行笔记本的特定段落。

执行完成后，执行状态将显示在工具栏和装订线中：

  • ：执行已成功完成

  • ：执行失败

  • ：执行已中止

如果成功执行，请预览段落代码下方显示的输出：

当段落包含任何启动Spark作业的RDD操作时， Spark作业链接将显示在预览区域中，例如， count要么saveAsTextFile方法。单击此链接可在浏览器中打开专用页面，并预览完成状态，事件时间线和DAG可视化。

当您执行SQL语句或运行show Zeppelin或Spark对象的方法，结果显示在预览区域的“ 表格”和“ 图表”选项卡中。

查看表格

默认情况下，“ 表”选项卡显示所有已处理的数据。

整理表格中的数据

  • 单击列标题以对其中的值进行排序。

  • 请点击过滤所选列中的数据。

  • 请点击在页面中组织表格。切换此按钮并指定要在页面上显示的表行数：10、15、30或100。

导出表

  1. 请点击将表保存为.csv文件。

  2. 输入文件名，然后单击保存 。

查看图表

图表的默认类型由服务器上的图表设置定义。但是，您可以配置和修改预定义设置。

配置图表

  1. 请点击更改图表的初始设置。

  2. 单击与图表类型相对应的任何图标，将绘制新图表。例如，单击添加新的散点图。

  3. 向下滚动图表设置以配置新添加的图表。将要绘制的变量拖到特定字段：

  4. 要从绘图区域删除任何图表，请单击在图表设置上。

导出图表

  1. 请点击将生成的图形输出保存为.png格式。

  2. 输入文件名，然后单击保存 。
