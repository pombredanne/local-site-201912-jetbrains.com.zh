创建测试的运行/调试配置

  1. 选择运行|从主菜单编辑配置 。

  2. 在“ 运行/调试配置”对话框中，单击在工具栏上或按Alt+Insert 。

  3. 从列表中选择您使用的测试框架（例如， JUnit或 TestNG）。

  4. 将打开一个新的配置页面。命名此配置。

  5. 从“ 测试种类”列表中，选择测试的范围：对于单个方法或类，对于包或目录中的所有测试。

  6. 根据所选的测试类型，指定要使用此配置运行的项目。

    例如，如果您选择了Method ，请指定IntelliJ IDEA将在此配置中运行的类和方法。

  7. （可选）切换到“ 代码覆盖率”选项卡，然后选择一个覆盖率运行器 。

  8. 应用更改并关闭对话框。

有关运行/调试配置对话框中特定选项的更多信息，请参考相应的页面JUnit或TestNG 。

您甚至可以在不创建运行/调试配置的情况下运行测试。请点击Ctrl+Shift+F10在工具栏上； IntelliJ IDEA会生成一个临时运行/调试配置，您可以在以后保存和使用该配置。
