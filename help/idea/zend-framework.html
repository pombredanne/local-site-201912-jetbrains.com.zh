Command Line Tool Support: Zend Framework 1

File | Settings | Tools | Command Line Tool Support for Windows and Linux
IntelliJ IDEA | Preferences | Tools | Command Line Tool Support for macOS

The dialog opens when you click and choose Zend Framework 1 Tool in the Command Line Tools dialog. Use the dialog to configure Zend Framework 1 Tool support in IntelliJ IDEA.

Use the dialog to configure Zend Framework support in IntelliJ IDEA.

The tool works only with local PHP interpreters.

Item

Description

Path to zf tool

In this field, specify the location of the <Zend-Framework-home>/zf.bat file or <Zend-Framework-home>/zf file. IntelliJ IDEA parses the contents of the specified file for Zend Framework commands.
