定义Bean类和包

您可以通过编辑源代码或使用IntelliJ IDEA提供的工具来定义bean类和包。

定义bean类和包

  1. 打开“新建Bean”对话框 。

    或者， 打开所需的bean进行编辑 ，然后单击“ 更改EJB类”按钮。

  2. 在里面字段中，指定将在“ 全局代码样式”对话框的“ Java EE名称”选项卡中配置的前缀和后缀作为补充的Bean名称。

  3. 在Package字段中，指定Bean将驻留的软件包的名称。在EJB类字段中，指定此bean的类名称。
