使用此对话框可以定义搜索的敏感性，并设置限制，这将有助于您避免报告每个相似的代码构造。您的首选项是在特定于语言的上下文中指定的。

CoffeeScript

  • 匿名变量 ：选中此复选框时，使用不同变量名的两个相同函数被视为重复，例如：

    var test01 = function（a，b）{return（a * b）} var test01 = function（a，b）{return（a * b）}

  • 匿名函数
  • 匿名文字

  • 不要显示比以下简单的重复 ：设置结果窗口中显示的重复语言结构的大小。默认情况下，不包括少于10个单元的构造（并且不能更改此限制）。

  • 匿名化不常见的子表达式比以下更简单 ：设置可被视为相似的语言构造中子元素的值，以在结果窗口中将其显示为重复。数字越大，IntelliJ IDEA采取的构造就越大。

    基于使用加性算法计算的元素大小，将值设置为任意权重。元素越大，计算值越高。
