使用正则表达式查找和替换字符串

当您要搜索和替换特定的文本模式时，请使用正则表达式 。它们可以帮助您进行模式匹配，解析，结果过滤等。一旦学习了正则表达式语法，就可以将其用于几乎所有语言。

  1. 按Ctrl+R打开搜索和替换窗格。

    如果您需要搜索和替换多个文件，请按Ctrl+Shift+R 。有关更多详细信息，请参阅在项目中搜索和替换目标 。

  2. 在顶部字段中输入搜索字符串，在底部字段中输入替换字符串。

    默认情况下，正则表达式复选框处于选中状态。如果需要检查regex语法参考，请单击Regex旁边的问号。

  3. 当您搜索包含特殊正则表达式符号的文本字符串时，IntelliJ IDEA会自动以反斜杠转义它们\在搜索字段中。

    请记住，如果您复制（ Ctrl+C ）字符串，然后粘贴（ Ctrl+V ）（在搜索字段中），则不会考虑正则表达式符号。

    但是，当您专门搜索诸如.[{()\^$|?*+ ，您需要使用反斜杠将其转义\ ，因此它们可以被识别。

    例如，如果您需要查找. ，类型\.在搜索字段中。

  4. 当您在搜索字段中输入一系列字符时，IntelliJ IDEA也可以匹配字母大小写。

    例如，如果只想搜索大写字符，请在搜索字段中键入以下内容：

    \ b [AZ]

    要搜索和替换更复杂的模式，请使用结构搜索和替换 。

  5. 如果清除了“区分大小写”复选框，则IntelliJ IDEA会同时搜索大小写 。选中“区分大小写”复选框以匹配指定范围的大小写。

  6. 浏览事件时，IntelliJ IDEA将显示替换提示，因此您可以在单击“ 替换”按钮之前查看可能的结果。

有关更多详细信息，请参见RegEx语法 。

使用正则表达式捕获组和反向引用

您可以将正则表达式放在方括号内以便对其进行分组。每个组都有一个以1开头的数字，因此您可以在替换模式中引用（反向引用）它们。请注意，组0引用了整个正则表达式。

让我们考虑以下代码：

  1. 打开搜索和替换窗格Ctrl+R 。

  2. 在搜索字段中，输入括号()表示捕获组 ，例如： \stitle="(.*)?"\s*(/>*)

  3. 在替换字段中，以1开头的数字反向引用这些组，例如：

    2元$ 1

  4. IntelliJ IDEA根据您的搜索规范突出显示找到的事件，并显示带有替换字符串的提示。

切换字符大小写

您可以使用正则表达式来更改与某些条件匹配的字符的大小写。

  1. 打开搜索和替换窗格Ctrl+R 。确保选中正则表达式复选框。

  2. 在搜索字段中输入搜索模式。

  3. 在替换字段中，根据您要实现的目标，输入以下语法之一：

    • \l将字符更改为小写，直到字符串中的下一个字符。
      例如， Bar变成bar 。

    • \u将字符更改为大写，直到字符串中的下一个字符。
      例如， bar变成Bar 。

    • \L将字符更改为小写，直到文字字符串的末尾\E 。
      例如， BAR变成bar 。

    • \U将字符更改为大写，直到文字字符串的末尾\E 。
      例如， bar变成BAR 。

有关更多详细信息，请参考RegEx语法参考表。
