Syntax Highlighting

Special syntax highlighting in Tapestry templates allows you to make template visual interpretation much more comprehensive.

  • In the Settings/Preferences dialog Ctrl+Alt+S, go to Editor | Color Scheme | Tapestry.

    You can configure the plugin to highlight every Tapestry specific tag and tag parameter in your template. If you do, you will be able to easily identify in your template what are normal tags and Tapestry tags.

If invalid tag attributes are reported it's because you have an inspection that checks if you're using any HTML tag parameter that your not supposed to use. When using Tapestry you will have to use special tag parameters so if you don't want to get this warning, turn off the "HTML tags and attributes conventions" inspection.
