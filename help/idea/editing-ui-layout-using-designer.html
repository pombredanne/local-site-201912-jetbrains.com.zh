使用Designer编辑UI布局

在这个阶段，您的样本的用户界面HelloDroid应用程序基于非常简单的布局。UI布局在res / layout文件夹中的activity_main.xml文件中定义。默认情况下，IntelliJ IDEA提供布局文件的图形视图，但是它也允许您切换到基于文本的视图，在该视图中您可以手动编辑布局文件。

让我们使用内置的UI Designer修改自动生成的用户界面，并查看如何呈现应用程序布局而不在任何物理或虚拟设备上运行它。

1。打开布局文件

在“ 项目”视图中 ，导航到res \ layout文件夹，然后选择activity_main.xml文件。“ 设计”窗格将在编辑器中打开。

该窗格显示了一个矩形画布，该画布与当前布局定义文件和“ 组件树”视图同步，因此对画布的任何更改都将在此反映出来。

2。删除现有的文本元素

让我们删除现有的text元素。为此，请单击以在视图中选择文本标签，然后右键单击它以调用上下文菜单。选择删除以清除用户界面：

3。添加一个ImageView小部件

现在添加一个ImageView小部件：从“ 小部件”面板中选择ImageView组件，然后单击要在其中插入小部件的画布。或者，您可以将小部件拖动到“ 设计”窗格中：

此时，视图包含图像的占位符，但尚无与之关联的图像。要将图像添加到项目，首先需要在res下创建一个drawable文件夹。

4。创建“可绘制”文件夹

右键单击“ 项目”视图中的res节点，然后选择“ 新建” |“新建”。 Android资源目录 。从资源类型列表中，选择drawable 。如有必要，请选择任何可用的限定词。

仅当您希望为不同的屏幕，UI模式，密度或语言环境使用不同的资源集时，才使用“小”等限定符。如果要使用一组资源，则不需要限定符。

要添加图像，只需在Windows资源管理器中选择图像文件并将其拖到IntelliJ IDEA中的drawable文件夹中即可。

5，将图像文件链接到“ ImageView”小部件

现在，您需要链接已添加到ImageView小部件的图像文件。在UI设计器中，您需要选择小部件以编辑其属性。您可以通过单击它来选择小部件，但是，有时这可能是一个非常困难的任务。如果您添加了图像小部件，但未附加任何图像，则该小部件将呈现为非常薄的框，可能很难用鼠标选择。这是抢救组件树并使您轻松选择所需视觉元素的时候。

在添加，编辑或删除UI布局的图形组件时，可以使用与Designer相同的方式使用组件树 。您可以将小部件拖到“ 组件树”上，然后从显示的层次结构中删除或编辑元素。

  1. 在“ 组件树”窗格中，选择imageView组件，然后在其属性表中找到src条目：

  2. 点击浏览 然后在打开的对话框中选择要附加到小部件的图像：

    图像将被附加到小部件。您可以在设计器中选择它并调整其大小和位置：

6。添加一个“ TextView”组件

现在让我们添加一个TextView组件。在选用板中，找到“ 小部件”下的“ 纯文本视图”组件，并将其拖动到图像下方的视图中：

默认情况下，纯文本视图显示一些文字文本： New Text 。要更改它并将其链接到一些可本地化的字符串，您需要创建一个新的文本资源。

7。创建一个“字符串”资源

  1. 在“ 组件树”中 ，选择TextView元素，然后在下面的属性表中找到text属性。

  2. 点击浏览 调用资源对话框，该对话框允许您选择一个现有的字符串值或添加一个新的字符串值。

    请注意，字符串存储在res |下的strings.xml文件中。值文件夹。如有必要，您可以直接编辑strings.xml文件。

  3. 单击新资源|新的字符串值 。在打开的对话框中，输入资源名称并指定将在TextView小部件中显示的文本：

8。为文字添加样式

为了使文本看起来更具吸引力，您需要设置一些其他属性。您可以通过编辑TextView组件的属性来实现。让我们执行以下操作：

  1. 将标签水平居中：将重力属性设置为center_horizontal 。

  2. 稍微填充文本：找到padding属性并将all的值设置为10dp 。

  3. 更改字体颜色：编辑textColor属性。您可以将属性值设置为颜色字符串，例如#ffd764例如，或者您可以让它引用颜色资源。要添加颜色资源，请单击浏览。 并创建一个名为welcomeText的价值#ffd764 。

  4. 更改字体大小：编辑TextSize属性。您可以设置一个值，或以与上面的TextColor属性相同的方式将其链接到大小资源。

    请注意，您不能将尺寸表示为纯数字。您必须始终添加dp后缀。

9。在各种条件下预览布局

通过“设计器”工具窗口顶部的控件，您可以在不同条件下预览UI布局：横向或纵向，不同的屏幕尺寸，使用不同的主题，针对不同的语言环境等。这为您提供了一种快速简便的方法，以查看在许多常见情况下整个应用程序UI的外观。

例如，执行以下操作：

  • 点击图标，然后从下拉菜单中选择预览所有屏幕尺寸 。IntelliJ IDEA将在最常见的屏幕类型上显示UI布局的预览。

  • 要在不同设备上预览应用程序布局，请单击具有当前设备名称的图标（在本例中为 ），然后从列表中选择。

  • 请点击在布局和纵向预览模式之间切换。
