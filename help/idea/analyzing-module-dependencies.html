分析模块依赖性

模块依赖关系分析显示指定范围内存在的所有模块，在“ 项目结构”对话框的“ 依赖关系”选项卡中指定的这些模块之间的关系以及模块之间的循环依赖关系。

您可能希望使用这种类型的分析来确保您先前定义的依赖项仍存在于项目中。

分析模块依赖性

  1. 在主菜单上，选择“ 分析|分析模块依赖性 。

  2. 指定分析范围。您可以选择选择整个项目或特定模块。

  3. 在“ 模块依赖性工具”窗口中检查依赖性

  4. 在树形视图中选择一个模块，然后使用 “模块依赖关系”工具窗口的工具栏按钮，以查找依赖于所选模块的模块 。
