生成Ant构建文件

IntelliJ IDEA提供了为项目自动生成Ant生成文件的可能性，或者为组成该项目的每个模块自动生成几个生成文件。生成构建文件后，可以将其添加到Ant Build工具窗口 。您可以将此类构建文件作为常规文件进行编辑和管理。

生成Ant构建文件

  1. 在主菜单上，选择“ 生成” |“创建”。生成Ant Build 。

  2. 在“ 生成蚂蚁构建”对话框中 ，指定生成选项。

    • 选择生成单文件或多文件构建。

    • 选择保留或覆盖先前生成的文件。

    请参阅主题生成蚂蚁构建对话框以获取控件的详细说明。

  3. 单击确定 。
