管理Struts 2元素

Struts 2元素（例如动作，结果和拦截器以及它们之间的交互）在 struts.xml配置文件的部分。当您为模块启用Struts 2支持时 ，将自动创建该文件。

IntelliJ IDEA为您提供了一个友好的界面，可在文本视图和图表中编辑struts.xml 。

在视图之间切换

  • 在“ 项目工具”窗口中打开struts.xml ，然后单击屏幕底部的“ 图形 ”。

创建一个新的Struts 2应用程序元素

  1. 在“ 项目工具”窗口中右键单击struts.xml文件，选择“ 新建” ，然后单击要创建的元素。

    将打开“ 新建...”对话框。

  2. 指定元素数据，然后单击“ 确定” 。

    创建新元素并将其显示在项目树中。相应的源代码在struts.xml中生成。

删除Struts 2应用程序元素

  • 在项目工具窗口中 ，右键单击要删除的元素，然后选择删除 。

    或者，切换到struts.xml文件并手动进行必要的更改。
