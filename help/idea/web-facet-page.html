使用此页面来编辑Web构面的设置。

项目

描述

名称

构面的名称。

部署描述符

形成您的应用程序的部署描述符列表。

Alt+Insert 。创建web.xml并将其添加到列表中。如果列表中已经存在相应的描述符，则该功能无效。

Alt+Delete 。从列表中删除选定的描述符。

Enter 。用另一个相同类型的描述符替换选定的描述符。

添加Application Server特定的描述符。 创建特定于应用程序服务器的部署描述符（例如glassfish-web.xml ， jboss-web.xml ）并将其添加到列表中。

网络资源目录

指定包含Web应用程序资源（如网页，图像等）的目录 （“ Web资源目录”列）及其在相应工件中的位置（“ 相对于部署根目录的路径”列）。

为了使资源目录包含在构件中，您应该确保构件配置包含Web构面资源元素。（所有资源目录都作为一个整体包含在工件中。）在这种情况下，/在路径相对于部署根列对应于在工件构造元件的根部。

源根

选择要包含在工件中的源根 。注意：

  • 如果工件配置中存在' '编译输出元素，则所有源根的类都包括在相应的工件中，而与此处选择的源根无关。

  • 要仅包括在此页面上选择的源根目录中的类，请在配置工件时使用以下内容： | JavaEE Facet类|网址（ ） 。结果，工件配置中的相应元素将显示为' 'module：'Web'facet classes 。
