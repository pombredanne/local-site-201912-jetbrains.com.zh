IntelliJ IDEA 2019.3帮助

JPA支持概述

仅Ultimate Edition支持此功能。

为了使用Java Persistence API （JPA），IntelliJ IDEA提供了：

  • Java EE：EJB，JPA，Servlets 插件 。该插件与IDE捆绑在一起，并且默认情况下启用。请参阅确保插件已启用 。

  • 能够为模块打开JPA支持。您可以在创建项目或模块时，或针对现有项目或模块时执行此操作。请参阅启用JPA支持 。

  • 用于管理JPA配置persistence.xml和对象/关系映射orm.xml文件的JPA方面。请参见Hibernate和JPA Facet页面 。

  • “ 持久性”工具窗口将显示您的JPA项目项，并允许您创建新的配置文件和持久性类，在编辑器中导航到相关的源代码，打开图和控制台等。请参阅使用持久性工具窗口 。

  • 通过导入数据库模式，EJB部署描述符文件ejb-jar.xml或Hibernate对象/关系映射文件.hbm.xml为其生成托管实体类和对象/关系映射的能力。请参阅生成管理实体类和O / R映射 。

  • 实体关系（ER）图。

    可从“ 持久性”工具窗口访问ER图（请参阅打开实体关系图 ），并提供与工具窗口大致相同的功能集。

    这些功能可作为上下文菜单命令访问。实体类和主图区域（对应于）的上下文菜单不同。

  • 一个JPA控制台，可让您编写和运行JPQL查询，并分析查询结果。请参阅使用JPA控制台 。

  • Java代码中的代码完成 ，包括JPA批注及其属性，以及JPA配置和O / R映射XML文件。

  • JPA特定的代码检查 。

  • 编辑器左边缘的导航标记，例如，用于从实体类跳转到映射文件中的相应片段。
