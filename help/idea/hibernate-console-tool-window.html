Hibernate控制台工具窗口

从“ 持久性工具”窗口 （用于会话工厂或其中的任何节点）：

  • 在标题栏上

  • 从上下文菜单中进行控制台

  • Ctrl+Shift+F10

数据库控制台的输入窗格将作为单独的编辑器选项卡打开。在输入窗格中，您可以编写SQL语句。

执行语句时，“ 服务”工具窗口将打开。服务工具窗口显示可用的连接会话，“ 输出”和“ 结果”选项卡。有关“ 服务”工具窗口的更多信息，请参阅“ 服务工具”窗口 。

如果执行的语句检索数据（例如， SELECT ），结果将显示在“ 结果”标签中，该标签的标题为合格的表名。有关为结果选项卡创建自定义标题的更多信息，请参阅对带有结果的选项卡使用自定义标题 。

在执行参数化语句之前，您会看到“ 参数”窗口，可以在其中指定参数值。

在此页：

  • 工具栏图标和快捷方式
  • 输出窗格
  • 结果窗格
  • 参数窗格

另请参阅使用Hibernate控制台 。

重新载入页面

 Ctrl+F5

重新加载表视图的数据，以使您在编辑器中看到的数据与数据库的内容同步。另外，如果要在更改后应用新的页面大小限制设置，请使用“ 重新加载页面”按钮。

数据提取器

选择数据的输出格式。

另外，您可以配置以下选项：

  • 跳过计算列（SQL） ：不包括未物理存储在表中的虚拟列（例如，标识列）。

  • 跳过生成的列（SQL） ：对于INSERT和UPDATE语句，在复制或保存数据时不包括自动增量字段。

  • 添加表定义（SQL） ：对于INSERT和UPDATE语句，在复制或保存数据时添加表定义。

  • 配置CSV格式 ：打开“ CSV格式”对话框，您可以在其中管理定界符分隔的值格式（例如CSV，TSV）。

  • 转到“脚本目录” ：使用脚本将目录数据转换为不同的输出格式，打开目录。

全部复制到剪贴板/全部保存到文件

  • 全部复制到剪贴板。 使用此命令将表数据复制到剪贴板上。

  • 将所有保存到文件。 使用此命令将表数据保存在文件中。在打开的对话框中，指定文件的位置和名称。

注意：

  • 应用当前活动的数据提取器。

  • 如果当前仅显示相应行的子集，则所有适当的行都将复制到剪贴板或保存在文件中。（当前显示的行数可能受“结果集页面大小”参数限制。）

保存LOB

将表中当前选择的二进制大对象（ LOB ）保存到文件中。

参数窗格

“ 参数”窗格显示在输入窗格中检测到的参数，并允许您编辑它们的值。要打开或关闭此窗格，请使用在工具栏上。

参数值可以指定为文本或数字，也可以指定为包含对象引用和方法调用的Groovy表达式。例如， date查询中的参数

从订单中选择o在哪里o.date>：date

可以指定为

新的java.sql。日期（System.currentTimeMillis（）-24 * 3600 * 1000）

要开始编辑值，请切换到相应的表格单元格并开始键入。要表明您已完成值的编辑，请按Enter或切换到其他单元格。要退出编辑模式并恢复初始值，请按Escape 。

当您在表中选择一行时，相应的参数将在输入窗格中突出显示。
