Ruby版本管理器

以下仅在安装并启用Ruby插件时有效。

IntelliJ IDEA使您可以使用由以下版本管理器维护的Ruby解释器：

请注意，所有受支持的版本管理器都不兼容，因此不应同时使用。

  • RVM
  • 班本
  • 克鲁比

  • 航空自卫队与航空自卫队，红宝石插件

Linux，MacOS和Windows 10至WSL支持所有版本管理器。

如果本地计算机上安装了这些版本管理器之一，则IntelliJ IDEA会自动检测到它，并允许您在可用的Ruby解释器（以及RVM和rbenv的gemset）之间进行切换。

rbenv需要rbenv-gemsets插件才能使用gemsets。

IntelliJ IDEA还使您能够使用通过这些版本管理器维护的远程Ruby解释器 。在这种情况下，在配置远程解释器时，必须指定版本管理器可执行文件的路径。例如，下表列出了Linux的典型路径。

版本管理员

可执行路径

RVM

单用户模式： /home/jetbrains/.rvm/bin/rvm （对于jetbrains用户）

多用户模式： / usr / local / rvm / bin / rvm

混合模式： / usr / share / rvm / bin / rvm

班本

/home/jetbrains/.rbenv/bin/rbenv （对于jetbrains用户）

/ usr / bin / rbenv

克鲁比

/usr/local/share/chruby/chruby.sh

自卫队

/home/jetbrains/.asdf/bin/asdf （适用于jetbrains用户）
