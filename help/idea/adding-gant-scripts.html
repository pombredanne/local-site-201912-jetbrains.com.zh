添加Gant脚本

IntelliJ IDEA允许您为Groovy，Grails和Griffon项目添加Gant脚本。

对于Groovy项目，您需要在“ 项目设置” |“项目”中指定Gant主目录的位置。加入Gant脚本之前， 甘特 。

添加Gant脚本

  1. 在“项目”工具窗口中，右键单击要在其中创建Gant脚本文件的目录。

  2. 选择新建|来自上下文菜单的Gant脚本 。

  3. 在“ 新建Gant脚本”对话框中，指定脚本的名称，然后单击“ 确定” 。

  4. IntelliJ IDEA使用默认目标创建一个Gant文件。
