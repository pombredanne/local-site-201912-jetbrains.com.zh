介绍

IntelliJ IDEA支持Web服务的有效开发，打包和部署。

支持的标准是：

  • XML Web Services的Java API（JAX-WS）
  • 阿帕奇轴心
  • JAX-RS

您还可以启用并手动配置对以下其他WS引擎的支持 ：

  • Apache Axis2

  • XFire 1。X / CXF （支持2.7版）

  • JBossWS
  • WebSphere 6。X

IntelliJ IDEA提供以下开发和打包功能：

  • 根据您指定的专用Web服务方面，设置相关的模块结构并下载所有必需的资源。

  • 即时生成必要的部署描述符，映射和清单文件。

  • 打包Web服务文件，并遵循Enterprise Web Services 1.1规范定义的规则，使用EAR文件结构。

Web服务开发功能通过Web Services捆绑插件提供 ，默认情况下已启用。如果不是，请使用“设置/首选项”对话框的“ 插件”页面启用它。

开发Web服务

要开发Web服务，请遵循以下常规步骤：

  1. 在Java模块中， 启用对相关Web服务的支持 。

  2. 用必要的类和方法填充模块。

  3. 编译开发的类并将它们公开为Web服务。

    为了能够在部署Web服务本身之前开发Web服务的客户端，请生成WSDL文档 。

  4. 配置要部署的工件 。

    不要忘记在工件中包括Web和Web Services方面资源。

  5. 创建运行配置 。在“ 部署”选项卡上，创建要部署的工件的列表。为它们各自指定应用程序上下文。

  6. 运行应用程序。

  7. 在“ 运行”工具窗口的“ 部署控制台”中查看和管理已部署的Web服务 。
