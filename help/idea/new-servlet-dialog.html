新的Servlet对话框

档案|新增| Servlet

指定新servlet的设置。

此页面描述控件。有关如何配置Java EE的信息，请参阅启用Java EE应用程序支持。

项目

描述

名称

Servlet名称的“根部分”。

该名称（及其前缀和后缀）将在 web.xml或servlet类中的元素@WebServlet注解。

您可以指定servlet名称的前缀和后缀（“ 设置”对话框中“ 编辑器” |“代码样式” |“ Java”页面的“ Java EE名称”选项卡上的标记“ 前缀”和“ 后缀”字段）。在这种情况下，前缀和后缀会自动添加到指定名称的前面和后面。

包

应该在其中创建新的servlet类的包的名称。

请点击 。然后选择一个现有的程序包或单击创建一个新的程序包。

类

servlet类的完全限定名称。

使用包名称，类名称前缀，servlet名称和类名称后缀生成默认的类名称。

类名的前缀和后缀在“ 设置”对话框中设置 。（“ 编辑器” |“代码样式” |“ Java”页面的“ Java EE名称”选项卡上的“ Servlet类 前缀和后缀”字段。）

如果要指定现有的类，请单击然后在打开的对话框中选择类。

创建Java EE 6带注释的类

如果未选中该复选框，则将servlet名称-类映射添加到web.xml中 。

如果选中此复选框，则新的Servlet类将为@WebServlet -带注释，并且不会对web.xml进行任何更改。

如果项目中没有web.xml ，则默认情况下会选中此复选框，并且您不能取消选择它。
