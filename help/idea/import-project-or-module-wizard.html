汇入专案

本节描述了开箱即用的可用功能。如果您使用的是框架插件，请参阅“ 语言和特定于框架的准则”部分 。

  1. 启动IntelliJ IDEA。

    如果“欢迎”屏幕打开，请单击“ 导入项目” 。

    否则，从主菜单中选择File |新品|现有的项目 。

  2. 在打开的对话框中，选择源，库和其他资产所在的目录，然后单击“ 打开” 。

  3. 选择项目类型：

    • Android Gradle
    • Bnd / Bndtools
    • 日食
    • Flash Builder
    • 摇篮
    • 马文

    • 从现有来源创建项目 （如果您的项目有另一个来源）

从Android Gradle导入项目

  1. 启动IntelliJ IDEA。

    如果“欢迎”屏幕打开，请单击“ 导入项目” 。

    否则，从主菜单中选择File |新品|现有的项目 。

  2. 在打开的对话框中，选择源，库和其他资产所在的目录，然后单击“ 打开” 。

  3. 选择从外部模型导入项目| Android Gradle ，然后点击完成 。

IntelliJ IDEA在IDE中打开并同步项目。如果需要调整Gradle设置选项，请参阅访问Gradle设置 。
