调试工具窗口。变数

在本主题中：

  • 工具列
  • 上下文菜单
  • 变量类型

通过“ 变量”窗格，您可以检查存储在应用程序对象中的值。

在“ 框架”窗格中选择堆栈框架后，“ 变量”窗格将显示其范围内的所有数据（方法参数，局部变量和实例变量）。在此窗格中，您可以设置对象的标签，检查对象，评估表达式，将变量添加到监视等等。

工具列

仅当隐藏“ 监视”窗格时，才会显示此工具栏，以便在“ 变量”窗格中显示已配置的监视 。隐藏/显示“ 手表”窗格是通过切换按钮：

  • 按下按钮（默认状态）后，“ 监视”窗格被隐藏，工具栏显示在“ 变量”窗格中。重点在于“ 调试器”选项卡 。

  • 释放按钮后，工具栏将移至“ 监视”窗格。

项目

捷径

描述

Insert

单击此按钮创建新手表。

Delete

单击此按钮可从列表中删除选定的手表。

Alt+UpAlt+Down

使用这些按钮可以更改手表的顺序。

Ctrl+D

使用此按钮可以创建所选手表的副本。

在变量标签中显示手表

使用此切换按钮可以隐藏或显示“ 监视”窗格。默认情况下，该按钮被按下并显示在“ 变量”窗格的工具栏上。因此，“ 监视”窗格被隐藏，并且监视在“ 变量”窗格中显示 。

  • 要单独显示“ 监视”窗格并在其中查看配置的监视，请释放“ 在变量中显示监视”选项卡切换按钮。出现“ 监视”窗格，并在工具栏上显示“在变量中显示监视”选项卡切换按钮。

  • 要隐藏“ 监视”窗格并在“ 变量”窗格中查看监视，请按 监视窗格的工具栏上的切换按钮。切换按钮返回到“ 变量”窗格工具栏上的默认位置。

上下文菜单

项目

捷径

描述

检查

不适用

该命令可用于字段，局部变量和引用表达式，并打开一个非模态“ 检查”窗口，您可以在其中专注于特定引用。您可以根据需要打开任意多个“ 检查”窗口。“ 检查”窗口中的视图与“ 监视”窗格中的视图相同，但所需的屏幕空间较小。

标记对象

F11

使用此命令添加对象标签。

设定值

F2

使用此命令可以更改字段或变量的运行时值。

复制值

Ctrl+C

使用此命令将所选变量的值复制到剪贴板。如果选择了多个项目，则不仅会复制变量的值，还会复制变量的结构，因此，当您将所选内容复制粘贴到文本文件时，缩进将模仿调试器的树输出以产生易于阅读的内容输出。

或者，将鼠标光标悬停在一个值上，然后在工具提示中查看其内容。

将值复制为

不适用

该菜单项仅在PHP上下文中可用。

选择此命令以下列格式之一将所选变量复制到剪贴板：

  • print_r
  • var_export
  • json_encode

复制JSON

该菜单项仅在JavaScript上下文中可用。选择此命令以JSON格式复制所选值。

比较价值与剪贴板

不适用

使用此命令将所选值与剪贴板中当前的值进行比较。

复制名称

不适用

使用此命令将所选变量的名称复制到剪贴板。

评估表达

Alt+F8

使用此命令可以在打开的对话框中评估所选变量 。

添加到手表

不适用

该命令可用于除静态节点以外的所有节点。使用此命令来创建引用该节点的表达式，并将该表达式添加到“ 监视”窗格中。

显示参照对象

不适用

使用此命令可显示引用当前所选变量的对象列表。

跳转到源

F4

此命令在编辑器中打开所选变量或字段的源代码，并将插入号放置在相应的行。

跳转到类型源

Shift+F4

使用此命令导航到所选变量或字段的类的定义。

查看文本/查看/编辑文本（用于字符串值）

不适用

使用此命令显示所选变量的文本表示形式（如果有）。

请注意，对于字符串值，“ 查看文本”将变为“ 查看/编辑文本” 。这样就可以在文本区域中更改字符串值。

查看为

不适用

使用此命令选择希望变量值的显示方式。对于数字值，可以在十进制和十六进制格式之间切换。对象通常由类名称和实例标识符表示，但是您可以选择以字符串格式显示对象。

  • 自动 ：此选项适用于可能具有不同布局的项目。如果选中，则IntelliJ IDEA将浏览所有可用的渲染器并搜索当前所选项目的第一个合适的布局。如果未找到，则应用默认布局。

  • 十六进制 ：此选项可用于数字变量。如果选择该变量，则以十六进制格式显示。

  • 原始 ：此选项可用于原始变量。它显示适用于相应原始类型的值。

  • Binary ：此选项可用于二进制数据。如果选中，则变量显示为二进制文字。

  • 时间戳记 ：此选项适用于long数据。如果选择，该变量将显示为时间戳。

  • 布尔值 ：适用于布尔数据。如果选中，该变量将显示为布尔值true要么false 。

  • toString（） ：此选项可用于所有引用类型，其中toString()被覆盖（数组除外），并显示节点的toString()树中的值。

  • Array ：此选项可用于数组。

  • Object ：这是可用于所有非基本变量类型的默认布局。

  • 地图 ：此选项显示为地图。

  • 集合 ：此选项显示为集合。

  • File ：此选项可用于文件，是默认布局java.io.File类型变量。

  • 用户定义的渲染器 ：此选项可用于用户在Debugger | Java Type Renderers ，或在“ 自定义数据视图”对话框中。菜单中显示了相应的渲染器名称。

调整范围

不适用

此命令可用于数组和列表，并允许您在指定的索引范围内查看数组的内容。

显示类型

不适用

如果要显示变量的类型，请选择此选项。

静音渲染器

不适用

选择此选项可禁用Java类型渲染器的自动计算，以减少开销。

自定义数据视图

不适用

选择此选项可打开“ 自定义数据视图”对话框，您可以在其中配置数据的表示方式。

变量类型

每个变量左侧的图标指示其类型：

  • ： 静态的

  • ：领域

  • ：数组

  • ：原始

  • ：对象

  • ：参数
