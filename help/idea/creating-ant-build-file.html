创建Ant构建文件

本节介绍如何手动创建Ant构建文件。

IntelliJ IDEA提供了用于编辑构建文件的框架，但是开发人员有责任使用目标填充构建文件。

创建Ant构建文件

  1. 在“项目”窗口中，选择应在其中创建构建文件的目录。

  2. 右键单击目录，然后选择“ 新建|在快捷菜单上选择文件 ，或按Alt+Insert 。

  3. 在“ 新建文件”对话框中，指定带有xml扩展名的新文件的名称，例如build.xml 。新文件在编辑器中打开。

  4. 指定构建目标 。您可以在fileset要么dirset定义指令

    您必须在build.xml文件中至少输入一个根标签，否则该文件将不会添加到Ant Build工具窗口中。

  5. 将构建文件添加到项目中 。
