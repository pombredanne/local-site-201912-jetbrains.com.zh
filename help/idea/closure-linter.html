封闭棉短绒

您可以直接从IntelliJ IDEA配置和使用Closure Linter 。

在你开始前

  1. 按照Python官方网站上的说明下载并安装Python。

  2. 按照Closure官方网站上的说明下载并安装Closure Linter。

启用Closure Linter并在IntelliJ IDEA中配置其行为

  1. 打开 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” | JavaScript |代码质量工具|关闭Linter ，选择启用复选框。之后，页面上的所有控件均可用。

  2. 指定Closure Linter可执行文件的路径：

    • Windows的\ Scripts \ gjslint.exe

    • / usr / local / bin / gjslint，用于Linux和macOS

  3. 指定先前创建的配置文件的路径。
