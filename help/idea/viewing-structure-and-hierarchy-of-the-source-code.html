查看源代码层次结构

使用IntelliJ IDEA，您可以检查类，方法和调用的层次结构，并探索源文件的结构。

建立层次结构

  • 类型层次结构显示类的父类和子类。

  • 方法层次结构显示了子类，其中方法覆盖了所选的子类，以及父类或接口，其中所选方法被覆盖了。

    在层次结构树中，显示IntelliJ IDEA 表示不是抽象但没有定义方法的子类。当方法未在类中定义但在超类中定义时，将显示IntelliJ IDEA 。

  • 调用层次结构显示方法的调用者（超类型）或被调用者（子类型）。

    如果在字段上调用调用层次结构，它将为您显示使用所选字段的方法列表。

构建后，可以立即在“ 层次结构”工具窗口中查看和检查层次结构 。默认情况下，每个新构建的层次结构都会覆盖当前选项卡的内容。您可以保留当前选项卡，并在新选项卡中构建下一个层次结构。

建立类型层次

  1. 在“ 项目”工具窗口中选择所需的类，或在编辑器中将其打开。

  2. 从主菜单中，选择导航|输入层次结构或按Ctrl+H 。

建立方法层次

  1. 在编辑器中打开文件，然后将插入号放在所需方法的声明处。

    或者，在“ 项目”工具窗口中选择所需的方法。

  2. 从主菜单中，选择导航|方法层次结构或按Ctrl+Shift+H 。

建立通话层次

  1. 在编辑器中打开文件，然后将插入号放在所需方法或字段的声明或用法处。

    或者，在“ 项目”工具窗口中选择所需的方法或字段。

  2. 从主菜单中，选择导航|呼叫阶层或按Ctrl+Alt+H 。

保留层次结构选项卡

  • 在“ 层次结构”工具窗口中，单击“ 固定选项卡”按钮在工具栏上。

查看层次结构

打开层次结构工具窗口

  1. 确保您已经构建了要显示的层次结构，请参见上面的构建层次结构 。

  2. 选择查看|工具窗口|主菜单中的层次结构 。

在标签之间导航

  • 单击当前显示的选项卡，然后从列表中选择要显示的下一个选项卡。

在视图之间切换

  • 使用IntelliJ IDEA，您可以构建和探索升序或降序层次结构，即被调用方或调用方方法，父级或子级类等。

    请点击要么分别显示调用者方法或被调用者方法。

使用此列表来限制当前层次结构的范围：

  • 项目 ：跟踪整个项目中方法的用法。
