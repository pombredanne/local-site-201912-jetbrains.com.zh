Grails入门

IntelliJ IDEA支持Grails 3.0版和更高版本。

有关使用较早版本的Grails的信息，请参阅“ Grails 1/2入门”部分。

创建Grails项目的最简单方法是使用Grails Application Forge生成一个项目，您可以在其中快速填写模板，然后将生成的模板项目导入IntelliJ IDEA。在这种情况下，Application Forge会根据所选功能和配置文件自动设置所有必需的依赖项。

打开生成的模板Grails项目

从Grails 3.0版本开始，Grails项目使用Gradle构建系统。您可以使用Gradle打开您的项目。IntelliJ IDEA导入项目并创建特定的文件夹结构和所有必要的工件。

  1. 从主菜单中，选择文件|打开并打开 build.gradle文件或包含build.gradle文件的目录。

  2. 在打开的对话框中，单击“ 以项目形式打开” 。

IntelliJ IDEA使用预定义的结构，Gradle构建文件和一组库来创建功能齐全的Grails项目。

如果您需要有关在IntelliJ IDEA中使用Gradle的更多信息，请参阅Gradle部分。

在IntelliJ IDEA中使用Application Forge创建Grails项目

您可以使用与IntelliJ IDEA集成的Application Forge，并通过项目向导创建Grails项目。

  1. 如果当前没有打开任何项目，请在欢迎屏幕上单击“ 创建新项目 ”。否则选择File |新品|项目 。

  2. 在打开的页面的左窗格中，选择Application Forge 。

  3. 从右侧的选项中，指定项目SDK（JDK）。如果在IntelliJ IDEA中已经定义了必要的JDK，请从列表中选择它。否则，请单击“ 新建” ，然后从列表中选择“ JDK”。然后，在打开的对话框中 ，选择所需JDK的安装文件夹（此时，您的计算机上必须已经安装了相应的JDK）。

  4. 从可用模板中选择Grails选项，然后单击Next 。

  5. 在下一页上，指定项目信息，然后单击完成 。

IntelliJ IDEA使用Gradle构建文件和所有需要的库生成Grails项目。

使用Grails SDK创建Grails项目

您可以使用特定的Grails SDK在IntelliJ IDEA中创建一个新的Grails项目。在开始使用Grails框架创建项目之前，请确保已在计算机上下载了所需的Grails SDK版本 。

  1. 打开项目向导 ，在左侧窗格中选择Grails 。

  2. 在页面的右侧，指定以下信息：

    • 您将用于项目的Project JDK。

    • Grails SDK主页-由库表示的本地Grails版本安装。

    • 创建create-app或create-plugin-根据您要创建的内容选择以下选项之一。在我们的情况下create-app 。

    • 选项-在此字段中使用其他选项。例如，您可以指定一个配置文件，例如--profile=web对于Grails。

    单击下一步 。

  3. 在向导的下一页上，指定项目的信息，例如名称和位置，然后单击Finish 。

IntelliJ IDEA使用适当的目录和工件创建Grails应用程序。

探索Grails应用程序

在Grails项目结构中存在一些显着差异，例如，适用于具有较早版本（1和2）的Grails项目：

  • build.gradle -Grails 3.0及更高版本使用Gradle进行构建。

  • config目录-config目录的内容使您可以使用YAML文件或Groovy文件进行配置。日志记录配置也可用。

  • init目录-此目录包含主应用程序文件，可让您使用默认设置运行应用程序。

  • 支持Grails 视图 。

    所有项目视图设置也可用于Grails视图。

  • 在“ 项目”视图中可以使用Grails操作。所有工件图标都更改为Grails图标（ “项目名称” |“新建” ）。

有关差异的更多信息，请参阅Grails文档 。

运行Grails应用程序

当您创建一个新的或打开现有的Grails项目时，IntelliJ IDEA会自动与您的项目一起创建运行/调试配置。在这种配置中指定的信息足以运行该应用程序。您可以点击在主工具栏上执行它。如果要编辑创建的运行/调试配置，请从主菜单中选择“运行” |“调试”。编辑配置 。

如果由于某种原因，创建的运行/调试配置没有在“ 应用程序”字段中指定的应用程序名称，请尝试重新导入项目。

您也可以从编辑器运行应用程序：

  1. 在项目工具窗口中，单击init文件夹。

  2. 从列表中，选择Application.groovy以在编辑器中打开文件。

  3. 在编辑器的左装订线中，单击并在打开的窗口中单击Run'Application main（）' 。

您的应用程序在默认浏览器中启动，地址栏中带有以下URL：

http：// localhost：8080 /

调试Grails应用程序

IntelliJ IDEA允许您使用Application.groovy 调试 Grails应用程序 。

  1. 在项目工具窗口中，打开init目录，然后右键单击Application.groovy

  2. 从列表中选择Debug Grails：“名称”

您也可以使用编辑器启动调试过程。

IntelliJ IDEA在“ 调试工具”窗口中显示结果。

使用Gradle任务调试Grails应用程序

IntelliJ IDEA还允许您使用Gradle任务调试Grails应用程序。

  1. 打开Gradle工具窗口 。

  2. 从任务列表中，单击应用程序 ，然后在打开的列表中，右键单击bootRun 。

  3. 从打开的列表中，选择Debug Grails：'name' 。

IntelliJ IDEA在“ 调试工具”窗口中显示结果。

Grails编码协助

您可以在Grails项目中使用以下编码帮助：

  • IntelliJ IDEA提供了所有相关文件之间的导航，例如域类，视图，服务和控制器。按Ctrl+Alt+Home在编辑器中，然后选择您的目标。

  • IntelliJ IDEA在注入的依赖项之间提供导航。

  • IntelliJ IDEA还提供了控制器方法和gsp文件之间的导航。

  • IntelliJ IDEA使您可以打开并检查域类的层次结构。按Ctrl+H在编辑器中打开“ 层次结构”工具窗口。

  • IntelliJ IDEA还允许您检查域类的依赖关系。
