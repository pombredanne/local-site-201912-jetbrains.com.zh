项目

描述

输入目录

在此字段中，指定要为其生成文档的源的目录的标准路径。

输出目录

在此字段中，输入将存储生成的文档的目录的标准路径。或者，单击打开对话框 ，您可以在其中选择所需的路径或创建新目录。

私人的

选择此复选框以设置private归因于true 。在这种情况下，所有私有成员将包含在生成的GroovyDoc中。

使用

选择此复选框以设置use归因于true 。在这种情况下，您可以使用外部文件。

源码包

使用按钮和创建和管理包含源文件的软件包列表。

在浏览器中打开生成的文档

如果希望IntelliJ IDEA在浏览器中自动打开生成的文档，请选中此复选框。
