创建并运行setup.py

以下内容仅在安装并启用Python插件后才有效。

IntelliJ IDEA提供了一个有助于创建setup.py脚本的操作，该脚本用于构建，分发和安装模块。创建setup.py后 ，相应的操作将被禁用

创建setup.py

  1. 在“ 项目”工具窗口中，选择包。然后，从主菜单中选择“ 工具” |“工具”。创建setup.py 。

  2. 在“ 新建安装脚本”对话框中，指定程序包名称，版本，作者，作者电子邮件，程序包描述以及任何许可证详细信息：

  3. 准备好后，单击“ 确定” 。IntelliJ IDEA创建setup.py并在编辑器中将其打开。

    请注意，IntelliJ IDEA使用有限的参数集创建setup.py文件。您可能需要手动添加其余所需的详细信息。有关setup.py参数的更多详细信息，请参阅《 Python打包用户指南》 。

当您处理setup.py附带的任何项目时，可以运行一些任务来构建，安装和分发setup.py文件中描述的软件包。IntelliJ IDEA可帮助您创建以下格式的命令行序列：

setup.py

运行setup.py任务

  1. 在主菜单中，选择工具|运行setup.py 。

  2. 在“ 输入setup.py任务名称”对话框中，键入任务名称的字母。

    注意星号通配符和snake_case名称的首字母均被接受。键入时，建议列表将缩小以仅显示匹配的名称。选择所需的任务，然后按Enter 。

  3. 在“ 命令行”字段中，输入要执行的任务的参数。

    如果不需要任何参数，则可以将此字段留空。

  4. 单击“ 确定”后 ，命令行开始在“ 运行工具”窗口中执行。

    有关setup.py命令参数的更多信息，请参见使用Setuptools构建和分发软件包 。
