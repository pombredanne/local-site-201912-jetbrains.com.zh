启用应用程序服务器集成插件

仅Ultimate Edition支持此功能。

默认情况下，所有可用的服务器集成插件均处于启用状态。但是，在开始使用应用程序服务器之前，始终需要确保启用了一个或多个必要的插件。

服务器集成插件名称通常包括服务器名称和单词“集成或支持”，例如GlassFish集成，dmServer支持，Tomcat和TomEE集成等等。

  1. 打开设置/首选项对话框Ctrl+Alt+S 。

  2. 在对话框的左侧，选择插件 。

  3. 在对话框的右侧，在“ 插件”页面上 ，输入所需的服务器名称（例如tomcat ）。结果，只有名称和描述包含键入文本的插件才会显示在插件列表中。

  4. 如果未选中插件名称右侧的复选框，则将其选中。

  5. 在“ 设置”对话框中，单击“ 确定 ” 。

  6. 如果建议，请重新启动IntelliJ IDEA。
