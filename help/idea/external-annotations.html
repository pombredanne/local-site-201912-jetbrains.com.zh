外部注释

在源代码中存储注释并不总是很方便。例如，如果您在一个团队中的每个人都使用不同的IDE的项目中工作，使用库类或希望使代码中没有注释，那么您是否在工作？这是您可以使用外部注释的时间。

外部注释是常规注释，它们存储在源代码之外的XML文件中，该文件名为注解.xml 。

在代码中，外部注释用装订线图标，并用灰色突出显示。默认情况下，IntelliJ IDEA在编辑器中显示外部注释。

启用外部注释

要使用外部注释，您需要在设置中启用此选项。如果跳过此步骤，则可以在外部注释库代码，但是将无法在源代码中使用外部注释。

  1. 按Ctrl+Alt+S打开“ 设置/首选项”对话框。

  2. 转到编辑器|代码样式| Java |代码生成，然后选择使用外部注释复选框。

  3. 应用更改并关闭对话框。

将注释设为外部

  • 要将常规注解快速转换为外部注解，请在其上放置插入符号，然后按Alt+Enter （或点击意图操作图标），然后选择“ 外部注释” 。

新增注解

  1. 按Alt+Enter在您要注释的元素上（或单击意图操作图标）。

  2. 选择注释 （例如， Annotate field 'out' ），然后选择一个注释。

    如果只能使用一个注释，请选择“ 注释为@” 选项（例如， Annotate class 'String' as @Deprecated ）。

  3. 在下一个对话框中，点击外部添加 。

  4. 指定要在其中存储带有外部注释的文件的目录（外部注释根）。

    该模块中的每个下一个外部注释将存储在同一文件夹中。如果您有多个注释根，则IntelliJ IDEA会询问您要使用哪个注释根。

配置注释根目录

注释根目录是一个文件夹，您可以在其中存储项目当前模块中使用的外部注释。您可以更改，删除和添加新的根：

  1. 按Ctrl+Shift+Alt+S打开项目结构对话框。

  2. 转到模块|路径 。

  3. 找到“ 外部注释”区域，您可以在其中管理附加到模块的外部注释。

注释也可以在项目级别上配置。在“ 项目结构”对话框中，单击“ SDK”。注解 。

隐藏外部注释

默认情况下，外部注释显示在编辑器中。您可以从代码中隐藏它们：

  1. 按Ctrl+Alt+S然后选择编辑器|一般|外观 。

  2. 清除“ 显示嵌入式注释”复选框。

  3. 应用更改并关闭对话框。

隐藏外部注释时，仅装订线中的图标表示其在代码中的存在。

从Maven下载外部注释

  • 您可以将外部注释存储为Maven工件，并将它们与库一起下载。这样做，请从“ Maven存储库 ” 下载对话框中选择“ 注解”复选框。

有关从Maven下载库的更多信息，请参阅从Maven 下载库。

IntelliJ IDEA将使用以下命令搜索所有工件annotations分类器并下载它们（如果存储库中可用）。
