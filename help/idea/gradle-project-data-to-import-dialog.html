Gradle项目数据导入对话框

单击时出现此对话框在工具栏上的Gradle工具窗口中。使用此对话框可以将外部模块添加到项目中。对于多模块项目可能会有所帮助。

项目

描述

请选择要包含在项目中的模块/数据

使用此区域可以查看当前项目的可用外部模块或数据列表。

单击此图标以从列表中选择所有模块。

单击此图标以清除所选模块的列表。

单击此图标仅查看选定的模块。

选择必填

如果单击此图标，则在不确定模块之间的依赖性时，IntelliJ IDEA将选择适当的模块。
