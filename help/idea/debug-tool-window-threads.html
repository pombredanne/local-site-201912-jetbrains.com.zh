调试工具窗口。线程数

“ 线程”窗格将进程的所有线程显示为树状视图，并允许对其进行浏览， 自定义线程视图并导出到文件 。

线程状态

Java提供了线程状态，以反映程序中线程当前正在发生的情况。

线程状态

描述

监控

线程正在Java监视器上等待。

没有开始

该线程尚未启动。

正在运行

线程处于活动状态并正在运行。

睡眠

线程正在休眠，因为Thread.sleep()要么JVM_Sleep()被称为。

未知

线程状态未知。

等待

线程正在等待Object.wait()要么JVM_MonitorWait()被称为。

松比

线程已完成执行。

线程图标

每个线程附近的图标指示相对于当前调试会话的线程状态。

图标

描述

当前线程处于挂起状态。

活动线程。

达到当前断点的线程。

挂起的线程。线程在调试器暂停时被标记为暂停。

冻结线程。当线程被手动暂停时，它们被标记为冻结。

上下文菜单选项

项目

描述

暂停

选择此选项可以挂起所选线程。当线程被挂起时，此菜单选项切换为恢复 。

打断

使用此命令中断活动线程。

落架

使用此命令删除选定的帧 ，即在调试时返回上一帧 。仅当有两个或更多帧时，此选项才可用。

添加步进过滤器

使用此命令在打开的对话框中添加步进过滤器。

导出线程

使用此项目可以打开“ 导出线程”对话框 ，该对话框允许您将线程导出到指定的文本文件。

自定义线程视图

使用此项目可以管理“ 框架”选项卡的内容。例如，您可以选择显示线程组，行号等。有关详细信息，请参见对话框说明 。
