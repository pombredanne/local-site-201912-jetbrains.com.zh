运行/调试配置：Flash远程调试

使用此对话框可以创建或编辑Flash Remote Debug 运行/调试配置 ，这些配置使您可以调试已经编译过的应用程序（SWF文件），并在必要时打包并准备在本地或远程计算机上运行，移动设备。

  • 主要设定
  • 启动前选项
  • 工具列

另请参阅使用Flash远程调试配置 。

主要设定

项目

描述

名称

使用此字段来编辑运行/调试配置的名称。

分享

选中此复选框可通过版本控制共享运行配置。

如果未选中该复选框，则运行配置设置将存储在.idea / workspace.xml或.iws文件中。

如果选中此复选框，则设置将存储在.idea / runConfigurations或.ipr文件中的单独xml文件中。

请参阅项目 。

仅单实例

如果选中此复选框，则一次仅运行配置的一个实例。

构建配置

选择要使用的构建配置 。

请注意，在此运行/调试配置类型的上下文中，仅以下构建配置属性相关：

  • Flex SDK。使用相应SDK中包含的调试器。

  • 依赖关系。除模块源代码外，还将在可用源代码的构建配置依赖项中搜索源文件。

启动前选项

指定在开始运行/调试配置之前应执行的任务。指定的任务按照它们在列表中出现的顺序执行。

项目

捷径

描述

Alt+Insert 单击此图标可将任务添加到列表中。选择要添加的任务：

  • 运行外部工具。 选择此选项可运行IntelliJ IDEA外部的应用程序。在打开的对话框中，选择应运行的一个或多个应用程序。如果尚未在IntelliJ IDEA中定义必要的应用程序，请添加其定义。有关更多信息，请参阅外部工具和外部工具 。

  • 建立工件。 选择此选项有一个神器内置或假象。在打开的对话框中，选择应构建的工件。

    另请参阅使用工件 。

  • 运行其他配置。 选择此选项可以执行另一个运行/调试配置。在打开的对话框中，选择要运行的配置。

    仅当您在当前项目中已经具有至少一个运行/调试配置时，此选项才可用。

  • 运行Ant目标。 选择此选项可以运行Ant目标。在打开的对话框中，选择要运行的目标。有关更多信息，请参见Ant 。

  • 生成CoffeeScript源映射。 选择此选项可生成CoffeeScript源的源映射。在打开的对话框中，指定CoffeeScript源文件的位置。

    有关更多信息，请参见CoffeeScript 。

  • 运行Maven目标。 选择此选项可以运行Maven目标。在打开的对话框中，选择要运行的目标。

    有关更多信息，请参见Maven 。

Alt+Delete

单击此图标从列表中删除所选任务。

Enter

单击此图标以编辑所选任务。在打开的对话框中进行必要的更改。

Alt+Up

单击此图标可将所选任务在列表中向上移动一行。

Alt+Down

单击此图标可将所选任务在列表中向下移动一行。

显示此页面

选中此复选框可在实际启动运行/调试配置之前显示运行/调试配置设置。

工具列

项目

描述

要么Insert

使用此图标或快捷方式创建新的运行/调试配置。

要么Delete

使用此图标或快捷方式删除所选的运行/调试配置。

要么Ctrl+D

使用此图标或快捷方式可以创建所选运行/调试配置的副本。

单击此按钮可编辑运行/调试配置的默认设置。

 

使用这些按钮可以在列表中上下移动所选的运行/调试配置。

列表中的配置顺序定义配置在主工具栏上相应列表中显示的顺序。
