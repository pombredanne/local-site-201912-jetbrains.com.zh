配置JavaFX Scene Builder

在设置中指定了Scene Builder应用程序的路径后，IntelliJ IDEA允许您直接从IDE在JavaFX Scene Builder中打开.fxml文件。

在IntelliJ IDEA中配置JavaFX Scene Builder

  1. 下载并安装最新版本的Scene Builder 。

  2. 在里面 设置/首选项对话框Ctrl+Alt+S ，选择“ 语言和框架” | JavaFX 。

  3. 请点击在“ SceneBuilder的路径”字段中。

  4. 在打开的对话框中，选择Scene Builder可执行文件，然后单击“ 确定” 。

  5. 应用更改并关闭对话框。

在Scene Builder中打开文件

在编辑器中打开.fxml文件时，编辑区域下面有两个选项卡：“ 文本”选项卡用于开发标记，“ 场景构建器”选项卡用于在Scene Builder中编辑文件。

  • Scene Builder2.x。然后。

    如果您使用的是2.x版的Scene Builder。然后默认情况下，这些文件将在Scene Builder选项卡上的Scene Builder中打开。

  • Scene Builder 1.x

    如果使用版本1.x的Scene Builder ，则“ Scene Builder”选项卡为空。在这种情况下，请在“ 项目”工具窗口中右键单击所需的.fxml文件， 然后从上下文菜单中选择“ 在SceneBuilder中打开” 。
