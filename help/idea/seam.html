介绍

IntelliJ IDEA中的Seam支持使您可以通过专用的Seam构面使用此框架提供的功能来开发，运行，调试和部署应用程序。通常，Seam facet与应用程序性质所需的其他方面（例如，Web，EJB，JSF，Java EE或Hibernate）一起在模块中使用。IntelliJ IDEA中的Seam使实现事务模型，Java Persistence API，JSF和jPDL技术变得容易。

接缝支持包括：

  • 自动检测接缝组件。

  • 检测和可视化Seam组件的依赖关系。

  • 可以检测和下载丢失的库和档案。

  • 接缝注释。

  • 注入的表达语言。

  • 编码辅助（代码完成，意图操作和快速修复），适用于Seam批注，组件，导航和页面流定义文件以及注入的表达式。

  • 导航，搜索和重构。

除了上述功能外，IntelliJ IDEA还支持Seam提供的业务流程和页面流：

  • 可以使用文本编辑器或“导航图”来定义页面导航规则。

  • 可以使用文本编辑器或设计器定义jPDL页面流。

使用Jboss Seam开发应用程序

要使用Jboss Seam开发应用程序，请遵循以下常规步骤

  1. 在模块中配置Seam支持 。

  2. 设置所需的数据源，请参阅管理数据源 。

  3. 用必要的类填充您的应用程序。

  4. 使用components.xml文件和Seam批注定义Seam组件 。

  5. 使用其他注释（EJB，持久性等）注释类。

  6. 定义页面导航规则 。

  7. 定义页面流 。

在具有Seam facet的应用程序中，您可以：

  • 除了常规的导航方式之外 ，您还可以在对话的各方之间跳转，或从观察者跳转至事件 。

  • 探索Seam组件的依赖关系 。
