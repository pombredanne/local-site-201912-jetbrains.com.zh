构建工具窗口

查看|工具窗口|建立

生成工具窗口可帮助您查看委托给Maven或Gradle的生成的输出以及项目同步的结果。

当您使用IntelliJ IDEA构建项目时，IntelliJ IDEA在消息工具窗口中显示输出消息（如果有）。

生成输出选项卡

IntelliJ IDEA在“ 构建输出”选项卡中显示构建结果 。根据项目，构建输出显示已执行的任务，目标和命令。

默认情况下，该选项卡还显示所有成功的步骤和警告。

建立一个项目

  1. 从主菜单中，选择Build |建设项目 Ctrl+F9 。

  2. 在“ 生成工具”窗口中检查结果。

同步标签

当您打开Gradle或Maven项目时，同步将自动完成。点击构建 状态栏上的图标可在“ 构建”工具窗口中查看同步结果。

如果需要，可以手动触发项目的同步。

重新导入项目

  1. 打开Gradle或Maven工具窗口。

  2. 右键单击要同步的项目，然后从上下文菜单中，根据项目，选择Reimport Gradle Project或Reimport Maven Project 。

  3. 在“ 生成工具”窗口中检查结果。

有关更多信息，请参考Maven和Gradle部分。

错误处理

如果在构建过程中发生错误，则IntelliJ IDEA会显示一条错误消息，其中包含可以轻松定位的行号和列号。如果IntelliJ IDEA可以找到错误的位置，它将显示一个链接，您可以单击该链接并导航到编辑器中所需的行。

当“ 构建工具”窗口显示多个错误或警告时，它使您可以从上下文菜单中使用上一个或下一个在上一个或下一个问题之间导航。 Ctrl+Alt+Down和Ctrl+Alt+Up捷径。

如果在同步过程中发生错误，则IntelliJ IDEA将显示一条错误消息。IntelliJ IDEA还会在文本控制台中显示问题描述，并在可能的情况下提供解决方案。
