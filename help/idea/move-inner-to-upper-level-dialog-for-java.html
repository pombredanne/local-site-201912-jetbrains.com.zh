将内部移动到Java的上层对话框

重构|移动
F6

对于在“结构”视图中选择或在编辑器中打开的内部类，将调用“ 移动内部类重构”对话框。

项目

描述

班级名称

如有必要，请使用此字段重命名已移动的内部类。

包裹名字

指定要移动到的类的包名称。

将外部类的实例作为参数传递

如果您要移动的类保留对其先前外部类的访问权限，请选中此复选框。

参数名称

在此指定作为参数传递的外部类实例的名称。

搜索注释和字符串

选择此选项可将更改应用于注释和字符串。

搜索文字出现

选择此选项可将更改应用于项目中包含的文档，HTML，JSP和其他文件。

搜索参考

选择此复选框可将更改应用于引用。此选项仅适用于“移动文件”或“移动程序包”重构。
