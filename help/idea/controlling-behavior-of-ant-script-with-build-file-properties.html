使用构建文件属性控制Ant脚本的行为

您可以控制IntelliJ IDEA执行Ant脚本的方式。如果要在后台执行，更改分配给构建过程的内存量或设置其他执行选项，则需要打开“ 构建文件属性”对话框。本节介绍如何执行此操作。

打开“构建文件属性”对话框

  1. 在“ 蚂蚁构建工具窗口”中 ，选择所需的构建文件。

  2. 请执行以下任一操作：

    • 在所选内容的上下文菜单上，选择“ 属性” 。

    • 请点击蚂蚁构建工具栏上的按钮。

    • 按Alt+Enter 。

有关特定任务，请参考以下过程：

  • 增加构建过程的内存堆 。

  • 在后台执行生成文件 。

  • 定义运行时属性 。

  • 定义额外的Ant类路径 。

  • 定义蚂蚁执行选项 。

  • 定义蚂蚁过滤器 。
