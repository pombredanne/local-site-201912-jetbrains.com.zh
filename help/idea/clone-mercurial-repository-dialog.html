IntelliJ IDEA 2019.3帮助

“克隆Mercurial存储库”对话框

使用该对话框通过从远程存储库下载数据来建立本地存储库。

项目

描述

Mercurial Repository URL

在此文本框中，键入要克隆的远程存储库的URL。

测试

单击此按钮以检查是否已成功建立到远程存储库的连接。

上级目录

在此文本框中，指定要IntelliJ IDEA为本地Mercurial存储库创建文件夹的目录。手动输入路径或单击浏览 然后在打开的对话框中选择所需的目录。

目录名

在此文本框中，输入将存储库克隆到的新文件夹的名称。

父目录不得包含具有指定名称的文件夹。

克隆

单击此按钮开始克隆指定的存储库。
