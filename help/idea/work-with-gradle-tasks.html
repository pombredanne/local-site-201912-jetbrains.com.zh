IntelliJ IDEA 2019.3帮助

Gradle任务

IntelliJ IDEA使您可以在项目中运行，调试和管理Gradle任务。

运行Gradle任务

您可以使用多种方法来运行Gradle任务，例如，从运行任何内容的窗口中使用运行配置来运行它们，或者从上下文菜单中运行它们，甚至可以通过一个运行配置来运行多个任务。

您还可以通过“ 运行任何内容”窗口运行Gradle命令行选项。

在“任何运行”窗口中运行Gradle任务

  1. 在Gradle工具窗口中，在工具栏上，单击 。或者，按Ctrl两次以打开“任何运行”窗口。

  2. 在“ 运行任何内容”窗口中，开始输入要执行的任务的名称。要执行多个任务，请使用空格输入任务名称以分隔每个新任务。或者，向下滚动至“ Gradle任务”部分，然后选择所需的任务。按Enter 。

    如果已链接了项目 ，则在“ 运行任何东西”窗口中的“ Gradle任务”部分中，选择所需的项目，然后在搜索字段中按Space然后添加任务。

  3. IntelliJ IDEA运行指定的任务，并在“ 运行工具”窗口中显示结果。

    IntelliJ IDEA还将任务保存在“ 最近 运行”部分下的“ 运行任何东西”窗口以及Gradle工具窗口中的“运行配置”节点下。

通过运行配置运行Gradle任务

您可以在任务中添加一些其他参数，将其配置为运行配置 ，保存并在需要时在项目中使用该运行配置。

  1. 打开Gradle工具窗口。

  2. 右键单击要为其创建运行配置的任务。

  3. 从上下文菜单中选择创建“任务名称” 。

  4. 在“ 创建运行/调试配置：'任务名称'”中 ，指定任务设置，然后单击“ 确定” 。

    IntelliJ IDEA在“ 运行配置”节点下显示任务。

  5. 双击任务以运行它，或右键单击该任务，然后从上下文菜单中选择“运行” 。

从上下文菜单运行Gradle任务

  1. 打开Gradle工具窗口。

  2. 右键单击要运行的任务。

  3. 从上下文菜单中选择“运行'任务名称'” 。

同时运行多个Gradle任务

您可以为多个任务创建运行配置 。

  1. 选择运行|编辑配置 Shift+Alt+F10 。
    将打开“ 运行/调试配置”对话框。

  2. 在“ 运行/调试配置”对话框中，单击然后选择Gradle添加新配置。

  3. 在“ 运行/调试配置”对话框的右侧的“ 名称”字段中，输入配置的名称。
    例如，指定以下设置：

    • Gradle项目 -单击并选择注册的Gradle项目。

    • 任务 -指定要使用此配置执行的任务。您可以运行多个任务。
      例如，指定clean和build 。

    • VM选项 -您可以自定义VM选项。例如，指定-Xmx3g 。

    • 参数 -您可以指定Gradle命令行参数 。例如，指定--debug 。

  4. 单击确定 。
    创建的配置将添加到“ Gradle项目”工具窗口中的“运行配置”节点。

  5. 双击配置以运行任务，或右键单击配置并选择运行 。

调试Gradle任务和脚本

您可以像调试常规应用程序一样调试Gradle任务和脚本。

调试Gradle脚本任务

  1. 在编辑器中打开build.gradle 。

  2. 设置断点，然后在装订线中右键单击运行图标。从上下文菜单中，选择“ 调试” 。

  3. IntelliJ IDEA启动调试会话，您可以在“ 调试工具”窗口中检查结果。

调试Gradle任务

使用Gradle工具窗口来调试实现以下功能的Gradle任务： org.gradle.process.JavaForkOptions界面，例如test或run 。

  1. 在Gradle工具窗口中，单击项目，打开Tasks节点，然后右键单击要调试的任务。

  2. 从上下文菜单中，选择“ 调试” 。

    启动调试过程，并将该任务添加到“ 运行配置”节点下的最新任务列表中。

为Gradle任务分配快捷方式

IntelliJ IDEA使您可以为Gradle任务分配快捷方式，并通过一次按键即可执行这些任务。您还可以为Gradle运行/调试配置分配一个快捷方式，该快捷方式可以包含多个任务。

  1. 在Gradle工具窗口中，右键单击所需的任务。

  2. 从上下文菜单中，选择分配快捷方式 。

    将打开“ 键盘映射”对话框。

  3. 在“ 键盘映射”对话框的“ Gradle”节点下，导航到您的任务。

  4. 右键单击该任务，然后从打开的列表中选择要分配的快捷方式类型。

  5. 在打开的对话框中，根据快捷方式的类型，配置您的快捷方式，然后单击“ 确定” 。
    在我们的情况下，让我们添加一个键盘快捷键。

    您可以在Gradle工具窗口中看到针对您的任务显示的快捷方式。

在“ 键盘映射”对话框中，您可以添加要为其分配快捷方式的新任务。

  1. 在“ 键盘映射”对话框的“ Gradle”节点下，单击“选择任务以分配快捷方式” 。

  2. 在打开的对话框中，选择所需的任务，然后单击确定 。

    任务将添加到Gradle节点下的列表中。现在，您可以配置快捷方式了 。

您可以创建收藏夹Gradle任务的快速列表 ，并通过关联的快捷方式引用该列表。

配置Gradle任务的运行触发器

IntelliJ IDEA使您可以在项目执行之前运行Gradle任务，或者使用任务激活配置来设置其他条件。

  1. 在Gradle工具窗口中，右键单击Gradle项目。

  2. 从上下文菜单中，选择“ 任务激活” 。

  3. 在“ 任务激活”对话框中，单击 。

  4. 在“ 选择激活阶段”菜单上，选择运行任务的时间，例如，“ 生成之前” ，“ 同步之后”等等。

  5. 在“ 选择任务”菜单上，选择实际任务。

    任务和激活阶段将添加到“ 任务激活”对话框的列表中。您还可以在Gradle工具窗口中看到针对所选任务的激活阶段名称。

或者，在Gradle工具窗口中右键单击要执行的任务时，可以从上下文菜单中选择激活阶段名称。

您还可以创建将依赖于Gradle任务的运行/调试配置。

  1. 从主菜单中，选择运行|编辑配置以打开项目的运行/调试配置。

  2. 在“ 运行/调试配置”对话框的“ 启动前”部分中，单击图标。

  3. 在打开的列表中，选择Run Gradle task 。

  4. 在“ 选择Gradle任务”对话框中，指定项目和要在启动项目之前执行的任务。您可以指定Gradle链接项目或任何其他Gradle项目。请注意，如果未链接您的Gradle项目，则IntelliJ IDEA将使用默认配置（例如，捆绑的Gradle版本）来运行任务。

  5. 单击确定 。
