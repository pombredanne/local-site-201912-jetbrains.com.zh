IntelliJ IDEA 2019.3帮助

Python语言支持

以下内容仅在安装并启用Python插件后才有效。

要在IntelliJ IDEA中开发Python脚本，请下载并安装Python并配置至少一个Python SDK 。

如果您是Windows用户，建议您安装Python for Windows 。

支持的版本

  • Python 2： 2.7版

  • Python 3：从3.5版到3.8版

IntelliJ IDEA提供对Python 3.5和（自2016.3起）Python 3.6的支持，并具有以下支持：

  • PEP-484-类型提示
  • PEP-448-其他拆包概述
  • PEP-492-具有异步和等待语法的协程
  • PEP-526-变量注释的语法
  • PEP-498-文字字符串插值
  • PEP-515-数字文字中的下划线
  • PEP-525-异步发电机
  • PEP-530-异步理解

IntelliJ IDEA中的Python支持

IntelliJ IDEA提供以下功能来帮助使用Python：

  • 专用模块类型 。

  • 能够配置解释器。 。

  • Python控制台 。

  • Python的运行/调试配置以及Python远程调试 。

  • 代码见解

  • 代码检查 。

  • 意向行动 。

  • 代码完成并解决。

  • 内置代码格式化程序和单独的Python 代码样式设置集 。

  • 在Python代码中查找用法 。

  • 测试框架 。

  • 快速文档 。

  • 识别Python 文档注释 。

  • 配置Python调试器 。

  • Python类的UML类图 。
