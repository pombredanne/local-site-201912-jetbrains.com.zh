访问Phing Build工具窗口

  • 在编辑器中打开Phing构建文件，或在“ 项目”工具窗口中选择它，然后从所选内容的上下文菜单中选择“ 添加为Phing构建文件 ”。

  • 选择查看|工具窗口|从主菜单开始构建 。通过编辑器或“项目”工具窗口中的Phing构建文件的上下文菜单打开工具窗口后，即可对其进行访问。

在此工具窗口中：

  • 指定phing.bat文件的位置。

  • 配置Phing构建文件列表，以打包，部署或测试PHP应用程序。

  • 在根据特定配置运行或调试之前，指定要执行的目标。

  • 运行整个构建文件和特定的构建目标。

可以通过工具栏按钮 >和构建文件或目标的上下文菜单使用Phing工具窗口的功能。

性质

单击此按钮以打开“ Phing属性”对话框。在此对话框中，指定phing.bat文件的位置。

如有必要，通过在“ 命令行选项”文本框中指定命令行参数来配置生成过程和输出格式。如果参数集太大而无法容纳在文本框中，请单击按钮或按Shift+Enter并在打开的“ 命令行选项”对话框中键入所需的参数。

仅当构建文件列表不为空时，该按钮才可用。

标记为隐藏

选择此选项可隐藏选定的目标，从而禁止将其显示在构建文件树中。当您有一些目标只能被其他目标调用并且永远不会单独运行时，这可能会有所帮助。

展开特定构建文件的节点或单击“ 全部展开”按钮时， 隐藏的目标不可见 。要删除目标的隐藏状态：

  1. 选择在其中定义了构建文件，然后单击“ 设置 首选项”按钮在工具栏上。

  2. 在打开的“ Phing设置”对话框中，切换到“ 隐藏目标”选项卡，在其中选择了隐藏目标名称旁边的“ 隐藏”复选框。清除旁边您想要的状态隐藏删除的目标的隐藏复选框。

性质

单击此按钮以打开“ Phing属性”对话框。在此对话框中：

  • 指定phing.bat文件的位置。

  • 如有必要，通过在“ 命令行选项”文本框中指定命令行参数来配置生成过程和输出格式。如果参数集太大而无法容纳在文本框中，请单击按钮或按Shift+Enter并在打开的“ 命令行选项”对话框中键入所需的参数。

  • 在“ 隐藏目标”选项卡中，标记目标为隐藏并删除隐藏状态。

    在构建文件中定义的目标中，您可能有一些仅被其他目标调用并且永远不会单独运行的目标。您可以通过将目标标记为隐藏来禁止在构建文件树中显示这些目标。

    展开特定构建文件的节点或单击“ 全部展开”按钮时， 隐藏的目标不可见 。

仅当构建文件列表不为空时，该按钮才可用。
