新资源文件对话框

工程工具窗口| res文件夹的上下文菜单|新增| Android资源文件

使用此对话框为您的Android项目创建资源文件。

项目

描述

文档名称

在此文本框中，指定新资源定义文件的名称。

资源类型

从此下拉列表中，选择应用程序资源类型 。

您可以使用上和下键盘键在“ 文件名”文本框中滚动浏览资源类型列表。

根元素

根据所选资源类型自动填充此字段。

目录名

在此文本框中，指定将在其中存储应用程序资源的文件夹。

IntelliJ IDEA可以根据资源类型和您选择的限定符自动组成文件夹名称。有关详细信息，请参阅创建资源 。

可用的限定词

从此列表中，选择确定特定设备配置的资源限定符 。选择相关的限定词，然后单击 。然后在打开的对话框中指定限定符的值。限定词将添加到“ 选择的限定词”列表中。
