创建OSGi Bundles运行配置

OSGi Bundles运行配置允许运行不依赖于任何已安装框架实例上的特定OSGi框架实现的捆绑软件。

当前，此运行配置类型有限制：

  • 仅核心框架包将被加载。

  • 您的捆绑包不能依赖于框架的任何其他捆绑包。

创建OSGi Bundles运行配置

  • 从主菜单中，选择运行|编辑配置

  • 在“ 运行/调试配置”对话框中，单击然后选择OSGi Bundles 。

  • 在打开的对话框中，选择一个框架实例来运行您的捆绑软件，并指定开始的捆绑软件。

有关更多信息，请参阅创建OSGi捆绑包运行配置 。
