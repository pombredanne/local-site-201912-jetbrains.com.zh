Android开发的先决条件

在开始使用IntelliJ IDEA编写第一个“ Hello，world” Android应用程序之前，请执行以下操作：

  1. 下载并安装JDK 7。

  2. 下载 Android SDK工具。

    Android的官方下载页面提供了不同的下载选项。默认下载选项是包含Android SDK工具的Android Studio。由于您已经拥有带有捆绑的Android插件的IntelliJ IDEA，因此您可以选择较小的下载量，而只需选择Android SDK和平台工具。滚动到“ 其他下载选项”，然后为您的操作系统选择“ 仅SDK工具”软件包。

  3. 运行安装程序以启动“ Android SDK工具设置”向导：

    首先，向导检查是否已安装Java SDK。接下来，选择安装选项并指定目标文件夹。强烈建议Android SDK主目录的路径不包含空格。

  4. 成功完成安装后，您需要使用SDK Manager添加SDK软件包。在Android SDK工具设置向导的最后一页上，选择启动SDK管理器选项，然后点击完成 。

  5. 在Android SDK Manager中 ，选择要安装的软件包（有关详细说明，请参阅添加SDK软件包 ）。

    对于我们教程中涵盖的过程，您可以保留默认选择。如果需要安装更多软件包，可以始终启动SDK Manager 。要调用它，请在主菜单中选择“ 工具” |“工具”。 Android | SDK管理器 。
