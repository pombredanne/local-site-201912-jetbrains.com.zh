Authentication Required

Use this dialog to specify your credentials and gain access to the Subversion repository. The dialog is opened when you add a new repository location, or attempt to browse a repository.

Item

Description

Authentication realm

This read-only area displays the repository name and URL.

User name

By default, this field shows the current user name. You can change the name as required.

Password

Type the password for your Subversion account.

Save credentials

Select this checkbox to preserve the specified user name and password.
