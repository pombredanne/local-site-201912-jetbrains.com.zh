换行返回值对话框

重构|包装返回值

使用此重构可围绕方法的返回值创建包装器类，或使用兼容的现有类作为包装器。

项目

描述

包装退货的方法

此只读字段显示所选方法的名称。

建立新课程

单击此单选按钮创建一个新的包装器类。如果选择此选项，请在下面的字段中指定类和目标包名称。

班级名称

键入新包装器类的名称。

包裹名字

默认情况下，显示当前软件包名称。您可以在文本字段中键入其他程序包名称，或单击省略号按钮并从树视图中选择目标程序包。如果所需的软件包不存在，请单击创建一个新的。

目标目的地目录

使用此字段选择目标目标目录。默认情况下，显示当前目标目录。通常，您会根据当前包选择目标目的地。如果此程序包存在多个根目录，则可以单击箭头按钮，然后从列表中选择“ 保留在同一源根目录中”。在这种情况下，包装将放置在初始类附近。
您可以单击省略号按钮以打开“选择目标目录”窗口。
您可以选择“ 目录结构”选项卡以选择另一个目标目录，或者选择“按邻居类别选择”选项卡以将包装器放置在邻居类别附近，例如，如果要将包装器放置在对类别或三元类附近的util目录中记住确切的包装，将包装纸放在Pair类附近可以节省您的时间。

使用现有课程

单击此单选按钮可以将您选择的现有类用作包装器。

名称

指定所需包装器类的名称。请注意，此类应包含构造函数，该构造函数的参数类型应与所讨论的返回值相同。
您可以在文本字段中输入完全限定的类名称，或单击省略号按钮，然后在“ 选择参数类”对话框中选择所需的类。请注意，您可以从项目类和非项目类中选择所需的包装器类。

包装领域

从字段的下拉列表中选择在指定包装类中遇到的将存储返回值的字段。

内部阶层

单击此单选按钮创建一个内部类。例如，如果您有私有方法，则可能要这样做。在这种情况下，您可以将所有内容放在同一个班级中。

名称

指定内部类的名称。
