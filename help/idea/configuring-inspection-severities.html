更改检查严重性

检查严重性级别指示检测到的代码问题对您的项目的严重程度。每个严重性级别都有其自己的突出显示样式。在IntelliJ IDEA中，有一组预定义的严重性级别：

  • 错误

  • 警告

  • 弱警告

  • 服务器问题

  • 错字

  • 不突出显示，仅修复

在所有范围内更改检查的严重性

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|检验 ，然后选择要修改的配置文件。

  2. 从列表中选择必要的检查。确保已启用它。

  3. 从“ 严重性”列表中，选择一个新的严重性级别。您也可以右键单击检查，然后从上下文菜单中选择严重性级别。

  4. 应用更改并关闭对话框。现在，修改后的检查将在所选配置文件中具有新的严重性级别。

如果可用严重性级别列表缺少所需的严重性级别，则可以创建一个新的严重性级别。

在特定范围内更改检查的严重性

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|检查并选择要在其中进行更改的配置文件。

  2. 从列表中选择必要的检查。确保已启用它。

  3. 从“ 所有作用域”列表中，选择要更改其严重性的作用域。

    IntelliJ IDEA显示两个范围的严重性：所选范围和其他所有范围 。

    要添加一个范围，请单击 。如果要创建新范围，请从范围列表中选择“ 编辑范围顺序 ”，然后单击“确定”。 。

  4. 从列表中选择必要的严重性级别并应用更改。

如果在多个范围中启用检查，并且这些范围中的文件匹配，则IDE将根据列表中它们的顺序处理这些范围。有关更多信息，请参阅更改范围的顺序 。

配置错误突出显示

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|检查并选择要在其中进行更改的配置文件。

  2. 从严重性级别列表中，选择编辑严重性 。

  3. 选择要更改格式的严重性，然后单击“ 编辑设置” |“设置”。颜色和字体 。

  4. 配置新的突出显示规则。使用对话框底部的预览部分。

您还可以在“ 编辑器” | “设置”下的设置中修改突出显示。 配色方案|一般

创建新的严重性级别

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|检查并选择要在其中创建新严重性级别的配置文件。

  2. 单击任何检查，然后从严重性级别列表中选择“ 编辑严重性 ”。

  3. 在“ 严重性编辑器”对话框中，单击“ 并命名新的严重性级别。
