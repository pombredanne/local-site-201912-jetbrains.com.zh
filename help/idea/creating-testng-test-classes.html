创建TestNG测试类

要创建TestNG测试类，可以使用创建测试意图操作 。

您还可以为TestNG测试创建一个新类，然后将必要的代码添加到该类中。本页描述了这种创建TestNG类的方法。

创建一个TestNG测试类

  1. 在“ 项目工具窗口”中 ，右键单击要在其中创建新测试类的目录。

  2. 选择新建|上下文菜单中的Java类 。

  3. 在“ 创建新类”对话框中，指定类名称，然后单击“ 确定” 。

  4. 在编辑器中，为您的测试类编写代码。必要时使用TestNG批注。例如，您可能想注释整个类或单个方法：

    @Test（）公共类testetsng {@DataProvider public Object [] [] data（）{返回new String [] [] {new String [] {“ data1”}，new String [] {“ data2”}}}; } @Test（dataProvider =“ data”）public void test（String d）{Assert.assertEquals（“第一行> \ n第二行”，“第三行\ n第四行”）; }}

    如果添加的注释缺少导入语句，则相应的意图操作将建议您创建一个。

TestNG中的YAML

TestNG支持使用YAML定义测试套件，但是默认情况下它不包含YAML解析器。如果要使用YAML文件，请添加snakeyaml依赖性。该版本应与您使用的TestNG版本相对应。

例如，如果您的项目使用Maven，则添加org.yaml.snakeyaml对pom.xml的依赖。
