准备使用Struts 2

通过Web和Struts 2方面可以与Struts 2集成。这些专用方面包含设置，配置文件路径和验证规则。此信息确定模块的结构，以便IntelliJ IDEA检测如何处理模块内容。

在开始之前，请确保已启用Web和Struts2捆绑的插件。 这些插件默认情况下处于激活状态。如果禁用了插件 ，请按照“ 管理插件”中所述在“ 插件”页面上启用它们。

Struts 2构面只能作为Web构面的子级添加。请注意，模块中仅允许一个Struts 2构面。

创建具有Struts 2构面的模块

当创建具有专用Web和Struts 2方面的Struts 2模块时，IntelliJ IDEA会配置该模块，自动下载并添加所有必需的库。

  1. 打开要向其添加模块的项目，然后选择“ 文件” |“添加” 。新品|模块 。

    结果，“ 新建模块”向导将打开。

  2. 在左侧部分中，选择Java 。

  3. 在右侧部分，指定要使用的JDK 。

  4. 在“ 其他库和框架”下 ，选中“ Web应用程序”复选框，然后从“ 版本”列表中选择Servlet规范的版本 。

  5. 如果要创建部署描述符web.xml文件，请选中“ 创建web.xml”复选框。

  6. 从“ 其他库和框架”列表中，选择Struts 2 。

  7. 您将需要一个实现Struts 2的库。您可以选择使用现有的库，创建和使用新的库，下载库文件（如果计算机上尚不可用）或将库的设置推迟到以后。

  8. 在向导的下一步中，指定模块名称和位置，然后单击完成 。

IntelliJ IDEA如下配置新模块：

  • 使用Web应用程序描述符web.xml创建一个Web节点。

  • 创建Struts 2配置文件struts.xml并将其添加到src节点。

  • 添加所需的Struts 2库。

将Struts 2构面添加到模块

将Struts 2方面添加到现有模块时，IntelliJ IDEA会自动下载并添加必要的库。

Struts 2构面只能作为Web构面的子级添加。如果该模块没有Web界面，请首先添加它。另外，请注意，模块中仅允许一个Struts 2构面。

  1. 在“项目”工具窗口中右键单击所需的模块，然后选择“ 添加框架支持” 。

  2. 在Java EE节点下，选择Web应用程序 。

    如果要创建部署描述符web.xml文件，请选中“ 创建web.xml”复选框。

  3. 选中“ Struts 2”复选框。

    您将需要一个实现Struts 2的库。您可以选择使用现有的库，创建和使用新的库，下载库文件（如果计算机上尚不可用）或将库的设置推迟到以后。

  4. 单击确定 。

创建的Struts 2配置文件struts.xml位于src节点中。

定义验证文件集

启用Struts 2支持后，IntelliJ IDEA会自动定义验证列表。您可以通过添加，编辑和删除文件集来对其进行编辑。

编辑验证文件集

  1. 从主菜单中，选择文件|项目结构 Ctrl+Shift+Alt+S ，然后单击构面 。

  2. 选择Struts 2构面，选择右侧的File Sets选项卡，然后单击 。

    将打开“ 编辑文件集”对话框。

  3. 要将文件添加到文件集中，请选中所需文件旁边的复选框，然后单击确定 。

  4. 要删除文件，请选择该文件，然后单击 。

删除验证文件集

  • 要删除验证文件集，请在“ 文件集”选项卡上将其选中，然后单击 。
