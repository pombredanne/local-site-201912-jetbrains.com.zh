在项目中启用黄瓜支持

先决条件

确保已启用Java的Cucumber和/或Groovy的Cucumber捆绑插件。插件的选择取决于要生成的步骤定义的类型。

创建黄瓜依赖项的两种方法

黄瓜依存关系可以用两种不同的方式定义：

  • 使用Maven依赖项

  • 使用库

使用Maven依赖项

要在Maven项目中启用Cucumber，请遵循以下常规步骤

  1. 从头开始创建项目，然后选择模块类型Maven 。或者，只需添加一个Maven模块 。

    不要从原型创建Maven模块！

  2. 打开以编辑新项目或模块的pom.xml文件。

  3. 在pom.xml中 ，创建并扩展该部分dependencies ，并添加以下代码：

    info.cukes 黄瓜java 测试 1.0.11 info.cukes 黄瓜 java 1.0.11 pom

  4. 运行阶段 package包括所有依赖项。

使用库

要通过库启用Cucumber支持，请遵循以下常规步骤

  1. 在应创建黄瓜测试的项目中，单击 。将打开“ 项目结构”对话框。

  2. 在“ 模块”页面中 ，单击“ 依赖关系”选项卡，然后添加所需的Maven依赖关系：
    1. 请点击 ，然后从子菜单中选择库 。

      如果库已经在本地存在，则在“ 选择库”对话框中，选择所需库版本的cumulage-java （对于Java）或Cucumber-groovy （对于groovy），然后单击添加所选按钮。

      如果该库在本地尚不存在，请单击“ 新建库” ，然后选择“ 来自Maven”的库类型。然后，在“ 从Maven存储库下载库”对话框中，指定Maven坐标（如果已知）或关键字，选择要下载的目标，然后单击“ 确定” 。

      如果库已经在本地存在，则在“ 选择库”对话框中，选择所需库版本的cumulage-java （对于Java）或Cucumber-groovy （对于groovy），然后单击添加所选按钮。

      根据需要配置新库（例如，附加文档URL或源）。

    2. 在“ 作用域”列中，单击箭头以显示下拉列表，然后选择“ 测试” 。

  3. 应用更改并关闭“ 项目结构”对话框。
