在Scala中提取特征

通过Extract Trait重构，您可以将Scala类，对象或特征中的选定成员提取为新特征。IntelliJ IDEA支持def ， val和var类型成员。

提取特征

  1. 在编辑器中，选择要重构的代码成员。

  2. 从主菜单中，选择“ 重构|提取物|特质 。

  3. 在打开的对话框中，输入新的特征名称，然后选择要提取的成员。

    如果需要在“ 要提取的成员”部分中选择所有列出的成员，请按Ctrl+A首先然后Space 。

    单击重构 。

例

让我们创建一个类Calculator用几种方法提取其中一种方法（ mul ）转变为特质。

类计算器{def add（a：Int，b：Int）：Int = a + b def mul（a：Int，b：Int）：Int =匹配{case 0 => 0 case _ => add（b， mul（a-1，b））}}
class计算器扩展乘数{def add（a：Int，b：Int）：Int = a + b}

重构后，IntelliJ IDEA创建一个新特征Multiplier包含mul方法及其定义主体。班级Calculator扩展特质Multiplier 。

特质乘数{this：计算器=> def mul（a：Int，b：Int）：Int =匹配{case 0 => 0 case _ => add（b，mul（a-1，b））}}}

提取特征对话框参考

项目

描述

从中提取特征

该字段包含您要从中提取成员的类，对象或特征的名称。

性状名称

使用此字段输入新性状的名称。

特性包装

使用此字段为新创建的特征添加软件包。我们建议您使用与从中提取原始类，对象或特征相同的包。

成员提取

使用此区域选择要提取的成员。

选中摘录摘要复选框后，新特征将仅包含一个声明，而没有定义主体。例如，在这种情况下，不包含主体的抽象方法将在原始类中被覆盖，并且定义将保留在那里。

清除此复选框后，该方法将被移动到具有其定义的新特征，如示例所示 。
