GUI设计器基础

IntelliJ IDEA中的GUI设计器使您可以使用Swing库组件为应用程序创建图形用户界面（GUI） 。该工具可帮助您加快最频繁的任务：创建要在顶级容器（例如JFrame）中使用的对话框和控件组。使用GUI设计器设计表单时，将创建窗格而不是框架。

使用GUI设计器创建的对话框和控件组可以与您直接使用Java代码创建的组件共存于应用程序中。

使用GUI设计器受到以下限制：

  • GUI Designer不支持非SWING组件的建模。

  • GUI设计器不会为应用程序创建主框架，也不会创建菜单。

有关使用GUI设计器的更多信息，请参见设计GUI。主要步骤 。
