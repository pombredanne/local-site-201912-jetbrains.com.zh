查找用法。班级选择

编辑查找|查找用法
Alt+F7

本节介绍在“ 查找用法”对话框中指定“类用法搜索”和“接口用法搜索”选项的控件。

单击时将打开对话框在“ 显示用法”弹出窗口中，该弹出窗口列出了在插入符号处所有出现的符号。

项目

描述

找

在该区域中，指定要搜索的对象。

  • 用法 -如果选中此复选框， 则按名称搜索该类的所有引用。

  • 方法的用法 -如果选中此复选框，则对所选类方法的所有调用执行搜索。

  • 字段的用法 -如果选中此复选框，则将搜索所选类字段的用法。

  • 派生类 -如果选中此复选框，则对扩展所选类的所有类执行搜索。

    该复选框仅可用于班级使用情况搜索。

  • 实现类 -如果选中此复选框，则对实现所选接口的所有类执行搜索。

    该复选框仅可用于界面使用情况搜索。

  • 派生接口 -如果选中此复选框，则对扩展所选接口的所有接口执行搜索。

    该复选框仅可用于界面使用情况搜索。

一般

在此区域中，使用以下控件配置搜索过程：

  • 搜索文本出现 -如果选中此复选框，则在IntelliJ IDEA中注册的文件中执行搜索。

  • 跳过一种用法的“结果”选项卡 -仅找到 一种用法时 ，选中此复选框可直接导航到找到的用法，而不会显示“ 查找工具”窗口。

范围

在此区域中，指定搜索范围 。从列表中选择一个预定义的范围或单击打开“ 作用域”对话框，您可以在其中定义自定义作用域。

在新标签页中打开

选中此复选框可使每次搜索的结果显示在“ 查找”工具窗口的单独选项卡中。如果清除该复选框，搜索结果将覆盖当前选项卡的内容。
