在ActionScript中提取参数

  1. 在编辑器中，将插入号放置在要用参数替换的表达式内。

  2. 请执行以下任一操作：

    • 按Ctrl+Alt+P 。

    • 选择重构|提取物|主菜单中的参数 。

    • 选择重构|提取物|上下文菜单中的参数 。

  3. 如果当前插入符号位置检测到多个表达式，则出现“ 表达式”列表。在这种情况下，请单击表达式以将其选中。或者，按Up要么Down导航到感兴趣的表达，然后按Enter选择它。

  4. 在“ 提取参数”对话框中 ：

    1. 通常，IntelliJ IDEA会自行设置适当的参数类型。如有必要，您可以从“ 类型”列表中选择其他合适的类型 。

    2. 在名称字段中指定参数名称。

    3. “ 值”字段最初包含您选择的表达式。通常，您不需要更改此设置。

      如果新参数将是可选参数，则指定的值将用作函数定义中的默认参数值。

      如果将新参数作为必需参数引入，则指定的值将添加到函数调用中。

      有关必需和可选参数的信息，请参阅Flex / ActionScript文档中有关功能参数的讨论。

    4. 如果希望新参数为可选参数，请选中“ 可选参数”复选框。

    5. 如果在函数体中找到一个以上的表达式出现，则可以选择用对新参数的引用替换仅选中的出现或所有找到的出现。使用替换所有出现复选框来指定您的意图。

    6. 单击确定 。
