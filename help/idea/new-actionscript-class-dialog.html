“新建ActionScript类”对话框

档案|新增| ActionScript类

使用此对话框可以指定要创建的ActionScript 类或接口的设置。如果您指定了尚不存在的软件包，您也可以创建一个新的软件包 。

项目

描述

名称

指定类或接口的名称。

当光标在该字段中时，您可以使用Up和Down箭头键浏览模板列表的内容。

包

指定应在其中创建类或接口的程序包的标准名称。为此，您可以：

  • 从列表中选择包。

  • 请点击Shift+Enter然后在打开的“选择目标包”对话框中选择包 。

  • 在字段中输入。要启用包名称完成，请按Ctrl+Space 。

该字段可以留为空白。在这种情况下，将在默认的根包中创建类或接口（通常对应于src文件夹）。

如果键入尚不存在的软件包的名称（在这种情况下，名称显示为红色），则会创建相应的软件包。

您可以一次创建多个软件包。例如，如果您键入myPackage.mySubpackage并且这些软件包目前都不存在，这两个软件包myPackage和mySubpackage将被创建。

模板

选择要使用的文件模板 。

  • 类。 选择此模板可创建一个不会扩展其他类或实现接口的类。

  • 与超级类。 选择此模板以创建扩展另一个类和/或实现一个或多个接口的类。

  • 接口。 选择此模板以创建接口。

要编辑现有文件模板或创建新文件模板，请单击Shift+Enter 。“ 文件模板”对话框将打开。

如果所选模板包含值未定义的变量，则“ 下一步”按钮将出现在对话框中，而不是“ 创建”按钮。在这种情况下，如果单击“ 下一步” ，则会打开另一个对话框，您可以在其中设置变量值。（如果使用包含自定义变量的自定义模板 ，则可能会发生这种情况。）

另请参见ActionScript和Flex的预定义文件模板变量和 为MXML组件创建自定义文件模板的示例 。

超类

对于具有Supers的类模板：指定要创建的类应扩展的类。为此，您可以：

  • 请点击Shift+Enter然后在打开的“选择超类”对话框中选择类 。

  • 在字段中输入。要启用班级名称完成，请按Ctrl+Space 。

介面

对于“ 带Supers的类”模板：指定要创建的类应实现的一个或多个接口。

要将接口添加到列表，请指向>>，然后单击Alt+Insert 。在打开的“选择超级接口”对话框中选择接口 。

要从列表中删除不必要的接口，请选择它们并单击Alt+Delete 。
