打包JavaFX应用程序

您可以通过构建相应的工件（Java归档文件）来打包JavaFX应用程序。对于JavaFX应用程序，IntelliJ IDEA提供了专用的工件类型： JavaFx Application 。

构建JavaFX工件

只有Java 8才能将JavaFX应用程序打包到.jar中。

创建一个新的工件配置

IntelliJ IDEA创建用于将应用程序与项目打包在一起的工件。但是，您可以使用自定义设置创建新的工件配置。

  1. 从主菜单中，选择文件|项目结构 Ctrl+Shift+Alt+S然后点击工件 。

  2. 请点击 ，指向JavaFx Application ，然后选择From module'...' 。

    IntelliJ IDEA创建项目配置，并在“ 项目结构”对话框的右侧显示其设置。

  3. 命名新配置。

  4. 切换到Java FX选项卡，然后在“ 应用程序类”字段中，指定main()方法。

  5. 应用更改并关闭对话框。

这是足以打包应用程序的基本配置。您可以在Java FX选项卡上指定其他选项。有关每个选项的详细说明，请参阅Java FX选项卡 。

建立工件

  1. 从主菜单中，选择Build |建立工件 。

  2. 在打开的弹出窗口中，选择所需的工件，然后选择Build 。

默认情况下，工件被生成为\ out \ artifacts \ 。

如果要确保正确构建了工件，请创建一个新的运行配置，然后按照运行打包的应用程序一节中的描述运行它。

疑难排解

  • Error:Java FX Packager: Can't build artifact – fx:deploy is not available in this JDK

    这是一个已知问题： IDEA-200721 。当前，仅Java 8可以将JavaFX应用程序打包在.jar中 。
