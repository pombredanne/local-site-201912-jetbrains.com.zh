调试工具窗口。镜框

通过“ 框架”窗格，您可以访问应用程序的线程列表。您还可以导出到文本文件并自定义线程表示 。

要检查线程，请从窗格顶部的列表中选择它。线程的状态和类型由一个特殊的图标和该线程名称旁边的文字注释指示。对于每个线程，您可以在编辑器中查看堆栈框架，检查框架，在框架之间导航以及自动跳转到框架的源代码。

要检查存储在框架中的值，请使用“调试”工具窗口的“ 变量”窗格 。

在本主题中：

  • 线程状态
  • 线程图标
  • 上下文菜单选项
  • 工具列

线程状态

Java提供了线程状态，以反映程序中线程当前正在发生的情况。

线程状态

描述

监控

线程正在Java监视器上等待。

没有开始

该线程尚未启动。

正在运行

线程处于活动状态并正在运行。

睡眠

线程正在休眠，因为Thread.sleep()要么JVM_Sleep()被称为。

未知

线程状态未知。

等待

线程正在等待Object.wait()要么JVM_MonitorWait()被称为。

松比

线程已完成执行。

上下文菜单选项

项目

描述

落架

使用此命令删除选定的帧 ，即在调试时返回上一帧 。仅当有两个或更多帧时，此选项才可用。

抛出异常

使用此命令可以在不修改代码的情况下从选定位置引发异常 。

强制返回

使用此命令可以在到达return语句之前强制从当前方法返回，而无需执行任何其他指令。如果该方法返回一个值，则将提示您指定它（提供了智能代码完成功能）。如果该方法具有try-finally块，则将提示您选择是否要执行它们。

复制堆栈

使用此命令复制当前线程堆栈跟踪。

添加步进过滤器

使用此命令在打开的对话框中添加步进过滤器。

导出线程

使用此项目可以打开“ 导出线程”对话框 ，该对话框允许您将线程导出到指定的文本文件。

自定义线程视图

使用此项目可以管理“ 框架”选项卡的内容。例如，您可以选择显示线程组，行号等。有关详细信息，请参见对话框说明 。
