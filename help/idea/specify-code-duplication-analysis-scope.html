指定代码重复分析范围

分析找到重复项

使用此对话框可以开始在指定范围内搜索重复的代码片段。

项目

描述

整个项目

选择此选项可以对整个项目进行分析。

模组

选择此选项可使IntelliJ IDEA分析“ 项目”工具窗口中当前选择的模块。

文件

选择此选项可分析当前在“ 项目”工具窗口中选择或在编辑器中打开的文件。

选定的文件

选择此选项可分析“ 项目”工具窗口中当前选定的文件。

未提交的文件

此范围仅适用于受版本控制的项目。
选择此选项可使IntelliJ IDEA仅分析尚未提交到版本控制系统的文件。使用该列表可以进一步限制分析范围。可用的选项有：

  • 全部 -选择此选项可以分析所有变更列表中的文件。

  • 默认 -选择此选项，IntelliJ IDEA的从缺省的变更只分析文件。

自订范围

选择此选项以使用自定义范围。从列表中选择一个预定义的范围，或单击并在打开的“ 范围”对话框中定义范围。

使用特殊的语言定义范围。

包括测试源

选择此复选框以对测试源执行分析。
