Integrate to Branch

View | Tool Windows | Version Control | Repository Tab | Context menu of a changelist or file |
Subversion | Integrate to Branch

Use this dialog to specify the options for integrating changes into a branch.

ItemTooltipDescription
Source branch URLThis read-only field shows the URL address of the source branch.
Target branch URLThis read-only field shows the URL address of the target branch.
Integrate into
working copy		From this list, select the path to the local working copy into which the changes will be integrated.
Ignore whitespaces Select this option if whitespaces are not important.
Try merge, but make no changes Select this option to preview merge results by enabling the --dry-run switch of the svn command. If this option is unchecked, sources are merged silently.
AddClick this button to add a working copy to the list.
RemoveClick this button to remove the selected working copy from the list.
