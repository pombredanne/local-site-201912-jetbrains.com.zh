Time Tracking Tool Window

View | Tool Windows | Time Tracking

ItemTooltip and ShortcutDescription
Open Task
Shift+Alt+N 		Create new task. Refer to the section Opening and Creating Tasks.
Remove TaskDelete selected tasks from the list.
Show closed tasksIf this button is pressed, the list of tasks includes closed tasks. This button works as the second pressing keys Shift+Alt+N.
Auto modeIf this button is pressed, time tracking automatically turns on for the current active task, and the button is disabled.
If this button is not pressed, you have to start timer for the active task yourself.
/ Start/stop timer for the active taskClick this button to start or stop timer for the active task. When the timer is started, the button toggles to (Stop timer for active task).
This button is disabled, when time tracking works in automatic mode (the button is pressed.)
Post work item to bug trackerClick this button send duration of work and optional comment to the bug tracker.
Show time spent from last post of work item
