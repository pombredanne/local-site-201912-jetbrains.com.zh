Accessing module settings

To access module settings

  1. Open the Project Structure dialog.
  2. In the left-hand pane, under Project Settings, click Modules.
  3. In the area under , select the module of interest.

    The Module page opens in the right-hand part of the dialog showing the module settings. Configure the settings as required.

Tips and tricks

To access the module settings, you can select the desired module in the Project tool window, Dependency Viewer, or the Module Dependencies tool window, and then do one of the following:

  • Choose Open Module Settings in the context menu.
  • Press Ctrl+Enter.
  • Press F4.
