Registering IntelliJ IDEA

After you've performed installation and launching, the evaluation period starts. Its duration is 30 days. After that, you can register IntelliJ IDEA. This is how it's done:

To register IntelliJ IDEA

  1. On the main menu, choose Help | Register...
  2. In the dialog box that opens, select the type of activation:

    Depending on your selection, do one of the following:

    • Activation code: paste the activation code to the text area.
    • License server: specify the license server address, or click the button Discover server, if the address is not known.

      If you need to learn more, click the link More info... to open the License server documentation.

    • JetBrains Account: select this option if you have the JetBrains Account and wish to register with it.
  3. Click OK when ready.

The banner appears in the top-right corner of IntelliJ IDEA that says to whom is the copy of IntelliJ IDEA licensed.

Note that you can delete the unnecessary license. To do that, simpy click the button Remove License in the IntelliJ IDEA License Activation dialog.
