Create Android Virtual Device Dialog

This dialog box opens when you first launch a Run/Debug session for an Android module, Emulator is selected as the target device on module creation, and you have no Android Virtual Devices configured.

Use this dialog box to create new emulators.

ItemDescription
NameIn this text box, type the name of the new emulator.
TargetFrom this drop-down list, select the Android platform to have the application built against.
SkinFrom this drop-down list, choose a skin to use for controlling screen dimensions (optional).
Abi TypeFrom this drop-down list, select the Application Binary Interface (ABI) type.
SDCardIf you want to simulate the presence of an SD card in the virtual device, you can create an SD card image by generating a dedicated file and specifying the virtual SD card size (optional).
