Command Line Tool Support: Zend Framework 1

File | Settings | Tools | Command Line Tool Support for Windows and Linux
IntelliJ IDEA | Preferences | Tools | Command Line Tool Support for OS X

The dialog box opens when you click the Add button and choose Zend Framework 1 Tool in the Choose Tool to Add dialog box.
Use the dialog box to configure Zend Framework support in IntelliJ IDEA.

ItemDescription
Path to zf tool In this text box, specify the location of the <Zend-Framework-home>/zf.bat file. Type the path manually or click the Browse button and choose the desired location in the dialog that opens. IntelliJ IDEA parses the contents of the specified file for Zend Framework commands.
