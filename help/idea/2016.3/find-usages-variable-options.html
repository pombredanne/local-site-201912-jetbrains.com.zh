Find Usages. Variable Options

Edit | Find | Find Usages
Alt+F7

This section describes Field, Variable, or Parameter usage search options in the Find Usages dialog.

The dialog opens when you click in the Show Usages pop-up window which lists all the occurrences of the symbol at caret.

ItemDescription
Skip results tab with one usageSelect this check box to be navigated directly to the found usage without the Find tool window displayed, when only one usage is found.
ScopeIn this area, specify the scope of search. Select a pre-defined scope from the drop-down list or click the Browse button to open the Scopes dialog box, where you can define a custom scope.
Open in new tabSelect this check box have the results of each search shown in a separate tab of the Find Results window. If the check box is cleared, the search results will overwrite the contents of the current tab.
