Configuring Joomla! Support

This feature is supported in the Ultimate edition only.

The following is only valid when PHP Plugin is installed and enabled!

In this section:

  • Configuring the Joomla!-specific development environment
  • Changing the Joomla! settings

Configuring the Joomla!-specific development environment

IntelliJ IDEA checks if the development environment is configured properly for Joomla! development.

Any detected inconsistency is reported in the Event Log tool window and as a pop-up. For each discrepancy IntelliJ IDEA suggests a fix. To apply a suggestion, click the link next to the reported event.

Changing the Joomla! settings

You can update the Joomla! settings at any time on the Joomla! Support page.

  1. Open the Settings / Preferences Dialog by pressing Ctrl+Alt+S or by choosing File | Settings for Windows and Linux or IntelliJ IDEA | Preferences for OS X, and click Joomla! Support under Languages and Frameworks.
  2. Make the necessary changes on the Joomla! Support page that opens:
    • To turn off the Joomla! integration, clear the Enable Joomla! integration check box.
    • Change the Joomla! installation directory.
  3. If you did not set code style in the Events Log, then in the Settings/Preferences dialog, click Code Style, and then PHP. On the Code Style: PHP page that opens, update the settings as necessary.
