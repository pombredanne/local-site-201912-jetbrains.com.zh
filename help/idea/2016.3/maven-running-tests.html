Maven. Running Tests

Use this page to configure settings for running JUnit tests using Maven Surefire plugin configuration.

The configuration parameters are activated by default.

You can use these settings for Maven 2 and Maven 3 versions.

ItemDescription
argLineSelect this check box to set arbitrary JVM options on the command line.
systemPropertyVariablesSelect this check box to pass a list of System properties to the JUnit tests.
environmentVariablesUse this check box to set additional environment variables on the command line.
