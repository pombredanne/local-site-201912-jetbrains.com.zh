Using Drupal with IntelliJ IDEA

This feature is supported in the Ultimate edition only.

The following is only valid when PHP Plugin is installed and enabled!

You can use IntelliJ IDEA as an IDE for Drupal development including modules, themes, and core. The supported versions are 6, 7, and 8 .

Before you start Dart development, install and enable the Drupal plugin. The plugin is not bundled with IntelliJ IDEA, but it can be installed from the JetBrains plugin repository as described in Installing, Updating and Uninstalling Repository Plugins and Enabling and Disabling Plugins.

IntelliJ IDEA provides integration between the Symfony2 framework and Drupal 8 while developing Drupal modules and core.
