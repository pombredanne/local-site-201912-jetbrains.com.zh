Validator Tab

View | Tool Windows | Struts Assistant

The tab is is available from the Struts Assistant tool window.

Use this tab to define validation procedures for specific actions.

validator-rules.xml node

ItemName and tooltipDescription
validator-rules.xml The node under which the definitions of standard validation routines are placed.
Context menu (same for all nodes)
F4 Jump to sourceSelect this menu item to switch to the code block in the validator-rules.xml file, text view, that corresponds to the selected item in the tree.
Expand AllClick this button to expand all nodes.
Collapse AllClick this button to collapse all nodes.
Global The node under which the definitions of global validation routines are placed.
Context menu
Add ConstantSelect this menu item to add a constant to the selected validation routine.
Add ValidatorSelect this menu item to add a standard routine validator.
Remove Global/ValidatorSelect this menu item to remove the entire Global node or the selected Validator.
