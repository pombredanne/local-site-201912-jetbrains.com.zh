Debug tool window | Threads | context menu | Export Threads

ItemDescription
Export to fileType the target file to store the thread report.
Use this button to navigate to the desired location in the Select Path dialog.
SaveClick to save the current threads in a text file indicated in the Export to file field.
CopyClick to copy the thread status to the Clipboard.
CancelUse to discard all changes and close the dialog.
