Publishing a Project on GitHub

You can publish your project sources in a GitHub Repository right from IntelliJ IDEA.

Make sure that you have registered your GitHub account credentials in IntelliJ IDEA.

To publish your project sources on GitHub

  1. On the main menu, choose VCS | Import into Version Control | Share Project on GitHub.
    • If you have registered login and password, IntelliJ IDEA establishes connection with GitHub using these credentials.
    • If you have not registered your GitHub credentials in IntelliJ IDEA, the Login to GitHub Dialog opens. Specify your GitHub login and password or create an account there.
  2. Upon establishing connection, the Share Project on GitHub dialog box opens. Specify the name of the repository to store your project sources in. By default, IntelliJ IDEA suggests the name of the current project. Provide a brief description of the project functionality.

    Click the Share button. IntelliJ IDEA initiates creation of the new repository on the GitHub and uploads the project sources to it.
