创建HTML包装器模板对话框

使用此对话框可以为以Web为目标的ActionScript或Flex应用程序配置和创建HTML包装模板。

生成的模板index.template.html将包含一组标记，例如${title} ， ${swf}等在编译期间，这些标记将被替换为适当的值。例如， ${swf}将被替换为.swf文件名。生成的.html包装器文件将与.swf文件具有相同的名称。

另请参阅使用SWF元数据标记控制HTML包装器属性 。

如果选择此选项，则应用程序将在现有Flash Player中运行SWF文件，以将用户升级到最新版本的播放器。
