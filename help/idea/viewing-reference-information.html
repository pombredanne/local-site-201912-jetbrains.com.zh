代码参考信息

定义

在IntelliJ IDEA中，您可以查看项目中在何处以及如何定义符号，例如标签，类，字段，方法或函数。为此，IDE具有“ 快速定义”弹出窗口。

  • 要查看符号的定义，请在编辑器中选择它，然后按Ctrl+Shift+I （或单击查看|快速定义 ）。

或者，使用Ctrl按下键，将光标悬停在任何符号上。IntelliJ IDEA将符号显示为链接，并在工具提示中显示其定义。单击此链接跳转到符号的定义。

点击右上角的图标在“ 查找工具”窗口中打开定义。要打开定义的源代码进行编辑并关闭弹出窗口，请单击图标F4 。要打开定义而不关闭弹出窗口，请单击Ctrl+Enter 。

对于标记语言，IntelliJ IDEA从指定的DTD或模式中检索符号的定义。有关详细信息，请参见HTML和XML 。

参数信息

“ 参数信息”弹出窗口显示方法和函数调用中的参数名称。在编辑器中键入尖括号或从建议列表中选择方法后，IntelliJ IDEA会在1秒（1000毫秒）内自动显示一个弹出窗口，其中包含所有可用的方法签名。

如果弹出窗口已关闭，或者将IDE 配置为不自动显示弹出窗口，则可以显式调用弹出窗口 。为此，请按Ctrl+P （或单击查看|参数信息 ）。

显示完整的方法或功能签名

默认情况下，参数信息弹出窗口显示简单签名。您可以将IDE配置为显示完整的签名，包括方法名称和返回的类型。

  • 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|代码完成 ，然后选择显示完整的方法签名复选框。

配置参数信息弹出窗口

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|代码完成 。

  2. 在“ 以...毫秒为单位显示参数信息弹出窗口”字段中，以毫秒为单位指定弹出窗口应出现的时间。

如果您不希望弹出窗口自动显示，请清除“ 以...毫秒为单位显示参数信息弹出窗口”复选框。

参数提示

参数提示将与代码内联显示在编辑器中，以提高其可读性。它们显示了传递值的方法参数名称，这些值是文字或空值。

这些提示默认情况下处于启用状态。如果您不想看到它们的特定方法，则可以为列表配置例外。

配置参数提示异常列表

默认情况下，例外列表包含许多预定义的方法。您可以从列表中删除方法以查看其提示，也可以添加新方法以在编辑器中隐藏其提示。

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，选择编辑器|插入提示 ，然后单击所需的语言。

  2. 在中间部分的列表中选择“ 参数提示 ”。

    右侧提供了用于配置提示的选项。

  3. 将您不想看到提示的方法添加到“ 黑名单”部分。

  4. 应用更改并关闭对话框。

快速文档

您可以通过“ 快速文档”弹出窗口获取任何符号或方法签名的快速信息 。请注意，该符号必须随附根据Javadoc标记创建的文档注释或SDK中的文档。您还可以为您的项目提供外部文档 ，以便能够在弹出窗口中快速查看它。

  • 要查看插入符号处的文档，请按Ctrl+Q （或单击查看|快速文档 ）。

    你可以按Ctrl+Q再次在弹出窗口和工具栏之间切换。

点击图标以更改字体大小，显示快速文档工具栏或转到源代码。

自动显示快速文档

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般

  2. 在“ 其他”区域中，选中“ 在鼠标移动时显示快速文档”复选框，然后指定延迟（以毫秒为单位），之后应显示弹出窗口。

外部文件

外部文档会在Web浏览器中打开必要的信息，以便您可以导航到相关符号并同时保留信息以备将来参考。在为项目配置外部文档后，您还可以在快速文档弹出窗口中查看它。

配置外部文档路径

要查看外部文档，您需要首先配置文档URL。

  1. 在项目结构对话框中Ctrl+Shift+Alt+S ，选择SDK 。

  2. 如果已配置了多个JDK，则选择必需的JDK版本，然后打开右侧的Documentation Path选项卡。

  3. 点击图标，然后输入外部文档的URL。例如，对于Java 12，键入https://docs.oracle.com/en/java/javase/12/docs/api/ ）。

  4. 应用更改并关闭对话框。

脱机访问外部文档

如果您离线工作，则可以在本地查看外部文档。

  1. 下载所需版本的文档包。该文档包通常以ZIP压缩文件的形式分发，下载后需要解压缩。

    例如，您可以下载官方Java SE Development Kit 8文档并解压缩。

  2. 在项目结构对话框中Ctrl+Shift+Alt+S ，选择SDK 。

  3. 如果已配置了多个JDK，则选择必需的JDK版本，然后打开右侧的Documentation Path选项卡。

  4. 点击图标，并使用下载的文档包指定目录（例如C：\ Users \ john.doe \ Desktop \ docs \ api ）。

  5. 应用更改并关闭对话框。

查看外部文档

  • 要在Web浏览器中的当前插入符号位置查看符号的文档，请按Shift+F1或选择查看|主菜单中的外部文档 。

类型信息

在IntelliJ IDEA中，您可以在弹出窗口中标识任何变量的类型和表达式。如果不需要有关代码元素的完整信息 ，这将很有帮助。

  • 将插入符号放在必要的代码元素上，然后按Ctrl+Shift+P （或从主菜单中选择“ 查看” |“键入信息 ”）。

表达静态数据

IntelliJ IDEA允许您在编辑器中查看有关任何Java表达式的静态已知信息。此信息可帮助您理解带有繁重分支的复杂代码，并详细说明表达式类型信息。

  • 要查看有关Java表达式的静态信息，请将插入标记放在表达式上，然后双击Ctrl+Shift+P 。

在此示例中， Expression静态数据操作显示list实际上是ArrayList ，尽管变量声明为List ，并且在给定点（大小= 0）仍然为空。

生产力提示

将工具提示文本复制到剪贴板

要将工具提示文本复制到剪贴板，请按住Alt （在Linux上，按住Ctrl+Alt ），然后点击工具提示。

查看上下文信息

如果当前方法或类声明不可见，则可以通过按以下工具提示来查看它： Alt+Q 。
