生成对Web服务的调用

IntelliJ IDEA在开发对Web服务（ 严格类型和松散类型）的客户端调用时提供了编码帮助。对强类型 Web服务的调用的编码帮助是特定于Web服务接口和数据类型的，并且基于从WSDL描述符生成的XML-Java绑定 。

IntelliJ IDEA还可以帮助您生成对松散类型的 Apache Axis或XFire Web服务的调用 。在这种情况下，通过专用方面提供了基本的Web服务客户端编码帮助。

生成对强类型Web服务的调用

  1. 在编辑器中打开客户端代码，然后将光标放在应该生成调用的位置。

  2. 选择工具| Web服务|主菜单或WebServices | WebServices客户端代码上下文菜单上的WebServices客户端代码 。

  3. 在打开的上下文菜单中，根据目标Web服务的类型选择相关选项。可用的选项有：
    • 生成Axis的WS调用
    • 生成Axis2的WS调用
    • 生成JAX-WS RI / GlassFish / JWSDP的WS调用
    • 为XFire生成WS调用
    • 生成JAXRPC的WS调用
    • 生成WebSphere的WS调用
  4. 通过使用建议列表和意图操作（如果适用）在模板输入字段中提供相关数据，来完成生成的调用代码存根。有效输入字段以红色框突出显示。

    建议列表的内容由从WSDL描述符生成的Java类确定。

调用生成的松散类型的Web服务

  1. 在编辑器中打开客户端代码，然后将光标放在应该生成调用的位置。

  2. 选择工具| Web服务|主菜单或WebServices | WebServices客户端代码上下文菜单上的WebServices客户端代码 。

  3. 在打开的上下文菜单中，根据目标Web服务的类型选择相关选项。可用的选项有：
    • 生成Axis的WS无类型调用
    • 生成XFire的WS无类型调用
  4. 通过使用建议列表和意图操作（如果适用）在模板输入字段中提供相关数据，来完成生成的调用代码存根。有效输入字段以红色框突出显示。

    建议列表的内容由通过专用方面提供的库确定。
