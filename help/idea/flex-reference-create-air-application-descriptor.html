创建AIR描述符模板对话框

使用此对话框可以为AIR（桌面或移动）应用程序配置和创建应用程序描述符模板。

在编译或打包过程中， 模板的元素将替换为应用程序文件的名称和扩展名.swf 。

  • AIR应用程序描述符
  • 应用属性
  • 行动选项
  • Android标签
  • iOS标签

AIR应用程序描述符

项目

描述

文档名称

指定要生成的描述符模板.xml文件的名称。

夹

指定应该在其中存储生成的描述符模板的文件夹。

使用Shift+Enter在相应对话框中选择文件夹。

AIR版本

指定（选择或键入）目标AIR版本。

应用属性

项目

描述

ID

指定应用程序ID。

名称

指定应用程序名称，即要显示为应用程序标题的文本。

版

在中指定应用程序版本x.x.x格式。

请注意，这不是AIR版本 。

行动选项

项目

描述

行动平台

选择应用程序描述符用于哪个平台（Android或iOS）。

常用选项

选择或取消选择以下屏幕选项 ：

  • 自动定向。 如果选中，则对应于true 。

  • 全屏。 如果选中，则对应于true 。

Android标签

项目

描述

权限

使用复选框启用或禁用相应的Android权限 。

iOS标签

项目

描述

设备

选择支持的设备系列。如果支持iPhone / iPod Touch和iPad，则选择全部 。

高分辨率

如果您的应用程序应使用高屏幕分辨率模式，请选择此选项。
