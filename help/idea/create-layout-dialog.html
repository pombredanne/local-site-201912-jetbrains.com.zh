创建布局对话框

工程工具窗口|具有应用程序类的包的上下文菜单|新增-Android组件

当您创建活动或片段类型的组件时，对话框打开，并选中“ 创建布局文件”复选框。

在此对话框中，指定布局名称，其根元素以及要包含在此布局中的一组限定符。

项目

描述

文档名称

在此文本框中，指定与新组件关联的布局定义文件的名称。仅支持小写字母和数字。

根元素

在此文本框中，指定布局的根元素。按Ctrl+Space获取可用值列表。该值必须是View或ViewGroup类型，有关更多信息，请访问http://developer.android.com/guide/topics/resources/layout-resource.html 。

目录名

在此文本框中，指定将相对于res文件夹存储布局定义的文件夹。

  • 如果您的应用程序不需要与各种Android设备兼容，因此不需要多屏支持，请接受默认的布局子文件夹。

  • 要提供备用资源 ，请指定确定特定设备配置的资源限定符 。将相关的限定词从“ 可用的限定词”列表移至“ 选择的限定词”列表并指定其值。IntelliJ IDEA将所有选定的限定词附加到“ 目录名称”字段，并以短划线字符作为分隔符。

有关详细信息，请参见创建资源 。

可用的限定词

 从此列表中，选择确定特定设备配置的资源限定符 。选择相关的限定词，然后单击 。然后在打开的对话框中指定限定符的值。限定词将添加到“ 选择的限定词”列表中。
