更改EJB类对话框

EJB工具窗口| Bean的上下文菜单|跳转到源更改EJB类

使用对话框更改现有bean的属性 。

对话框的内容是特定于bean类型的。请参阅“ 可用范围”列中的注释。

项目

描述

可用于

此只读字段显示用于生成EJB构成类名称的基本名称。在“ Java EE名称”对话框中配置用于生成EJB名称的规则。

所有豆类

包

在此文本框中，指定Bean所在的软件包的标准路径，或单击“ 浏览”按钮。 并在模块树中选择所需的软件包。

所有豆类

EJB类

此只读字段显示Bean实现类的名称。

所有豆类

讯息侦听器

在此文本框中，指定消息侦听器界面。输入完全限定的名称或单击浏览按钮并从可用接口列表中选择消息侦听器接口。

消息豆

主键类

在此文本框中，指定将用于访问与实体bean关联的数据源的主键的类。

实体豆

CMP版本

此只读字段显示使用的CMP版本。

实体豆

对于BMP实体bean，该字段为只读。

远端介面

如果要配置Bean的远程客户端视图，请选中此复选框。

选中此复选框将启用“ 主页”和“ 远程”文本框。

实体豆

家

在此文本框中，指定远程家庭接口的实现类。如有必要，请单击浏览。 打开“ 选择EJB Home Interface”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。

实体豆

远程

在此文本框中，指定远程接口的实现类。如有必要，请单击浏览。 打开“ 选择EJB远程接口”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。

实体豆

本地接口

如果要配置Bean的本地客户端视图，请选中此复选框。

选中此复选框将启用“ 本地主页”和“ 本地”文本框。

实体豆

当地的家

在此文本框中，为本地home接口指定实现类。如有必要，请单击浏览。 打开“ 选择EJB本地主接口”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。

实体豆

本地

在此文本框中，为本地接口指定实现类。如有必要，请单击浏览。 打开“ 选择EJB本地接口”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。

实体豆
