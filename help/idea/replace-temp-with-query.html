用查询替换温度

用查询替换温度重构可让您将变量的初始值设定项表达式提取到方法中，并将对变量的所有引用替换为对提取的方法的调用。变量的声明将被删除，查询方法可以在其他方法中使用。

代替int size = getActualSize()并在整个代码中使用大小， getActualSize()方法。尽管生成的代码具有更多的调用，但它更整洁，有助于准确识别代码瓶颈在何处出现。

  1. 在编辑器中，将插入符号放在要重构的局部变量的名称上。

  2. 在主菜单上，或在上下文菜单上，选择“ 重构|将temp替换为Query 。

    选择局部变量时，请确保稍后在代码中不修改其值。否则会出现错误信息。

  3. 在打开的对话框中，指定提取方法的名称和其他选项。例如，您可以将方法声明为静态，选择将在提取的方法中使用的参数，并指定可见性选项。

  4. 单击确定 。
