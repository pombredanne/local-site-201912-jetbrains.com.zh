Move File Dialog

Refactor | Move

F6

The Refactor | Move menu command invokes the Move File dialog when a file is selected in the Project tool window or opened in the editor.

Item

Description

To package

Specify the destination package. Type the path manually or click and select the target package in the dialog that opens.

Search for references

Select the checkbox to find and update the references to the file being moved.

Open moved files in editor

Select this checkbox to see the moved file in the editor.
