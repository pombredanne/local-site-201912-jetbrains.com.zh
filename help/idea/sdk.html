开发包

软件开发工具包或SDK ，是为特定软件框架开发应用程序所需的工具的集合。例如，要使用Java开发应用程序，您需要Java SDK（JDK）。SDK包含二进制文件，二进制文件的源代码和源代码的文档。对于Java，SDK还包含注释。

通常，SDK是全局的。这意味着一个SDK可以在多个项目和模块中使用。创建新项目并为其定义SDK后，可以在该项目中配置模块以继承其SDK。您也可以为每个模块分别指定一个SDK。有关更多信息，请参阅更改模块SDK 。

支持的SDK

  • Java SE开发套件（JDK）
  • Java Micro Edition（ME）SDK
  • Android SDK

  • Flex和AIR SDK

  • Google App Engine SDK

  • IntelliJ平台插件SDK （用于开发IntelliJ IDEA插件）

定义一个SDK

定义SDK意味着要让IntelliJ IDEA知道计算机上哪个文件夹中安装了必需的SDK版本。此文件夹称为SDK主目录。

管理全局SDK

  1. 从主菜单中，选择文件|项目结构 Ctrl+Shift+Alt+S 。

  2. 在左侧面板上，找到“ 平台设置”部分，然后选择“ SDK” 。

  3. 要添加新的SDK或新的SDK版本，请单击Shift+Enter ，选择SDK类型并指定其安装目录。

  4. 要删除SDK，请在列表中选择它，然后单击Alt+Delete 。

变更专案SDK

  1. 从主菜单中，选择文件|项目结构 Ctrl+Shift+Alt+S 。

  2. 在左侧面板上，找到“ 项目设置”部分，然后选择“ 项目” 。

  3. 从项目SDK列表中，选择另一个SDK或SDK版本。

    如果尚未在IntelliJ IDEA中定义必需的SDK，请单击“ 新建”并指定其主目录。

更改模块SDK

  1. 从主菜单中，选择文件|项目结构 Ctrl+Shift+Alt+S 。

  2. 在左侧面板上，找到“ 项目设置”部分，然后选择“ 模块” 。

  3. 选择必要的模块，然后单击Dependencies 。

  4. 从“ 模块SDK”列表中，选择要使用的另一个SDK或SDK版本。

    如果尚未在IntelliJ IDEA中定义必需的SDK，请单击“ 新建”并指定其主目录。

如果要模块继承项目SDK，请从“ 模块SDK”列表中选择“ 项目SDK”选项。

Java开发套件（JDK）

要在IntelliJ IDEA中开发应用程序，您需要Java SDK（JDK）。JDK是一个软件包，其中包含库，用于开发和测试Java应用程序的工具（开发工具）以及用于在Java平台上运行应用程序的工具（Java运行时环境— JRE）。

JRE可以与JDK分开获得，但是它不适合应用程序开发，因为它没有必需的组件，例如编译器和调试器。

捆绑的JRE用于运行IDE本身，不足以开发Java应用程序。在开始使用Java开发之前，必须获取并安装独立的JDK 。

IntelliJ IDEA不随JDK一起提供，因此，如果您没有必需的JDK版本，请下载并安装它。请注意，除了JDK之外，某些框架还需要其自己的SDK，例如Android或Grails。

下载一个JDK包

  1. 打开jdk.java.net网站。

    在这里，您可以找到Windows，macOS和Linux的Oracle OpenJDK二进制文件，如.tar.gz和.zip归档文件。

  2. 选择并下载必要的JDK版本。

    如果不确定所需的版本，请选择最新的稳定版本。

  3. 解压缩档案并将Java文件夹放置在有意义的位置，例如：macOS上的/ Library / Java / JavaVirtualMachines或Windows上的C：\ Program Files \ Java 。

示例：设置项目JDK

  1. 从主菜单中，选择文件|项目结构 Ctrl+Shift+Alt+S 。

  2. 在左侧面板上，找到“ 项目设置”部分，然后选择“ 项目” 。

  3. 在“ Project SDK”部分中，单击“ 新建” ，然后指定计算机上Java目录的路径。

  4. 应用更改并关闭对话框。
