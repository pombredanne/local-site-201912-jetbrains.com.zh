IntelliJ IDEA 2019.3帮助

大数据工具

Big Data Tools插件可用于IntelliJ IDEA 2019.2及更高版本。它提供了使用Zeppelin和Spark处理数据的特定功能。您可以创建新的或编辑现有的本地或远程Zeppelin笔记本，执行代码段，预览结果表和图形，并将结果导出为各种格式。

该插件支持许多可与笔记本一起使用的IDE功能：

Scala的编码帮助

  • 代码完成 。

  • 导航：转到声明 。

  • 重命名并提取变量重构 。

  • 显示用法 。

  • 检查和快速修复 ，包括笔记本依赖项的修复。

笔记本功能

  • 浏览， 创建和删除笔记本

  • 添加和删除段落

  • 执行段落

  • 运行SQL语句

  • 预览表格和图表

IntelliJ IDEA中的大数据工具入门

IntelliJ IDEA中大数据处理的基本工作流程包括以下步骤：

配置环境

  1. 安装所需的插件 ：

    • 大数据工具
    • Bash支持
    • 斯卡拉

  2. 在IntelliJ IDEA中创建一个新项目 。

  3. 配置与目标服务器的连接。

使用笔记本

  1. 创建和编辑笔记本。

  2. 执行笔记本。

  3. 分析您的数据：

    • 预览表
    • 配置和预览图

熟悉用户界面

当您安装IntelliJ IDEA的Big Data Tools插件时，将出现以下用户界面元素：

大数据工具窗口

大数据工具窗口出现在最右边的组工具窗口 。

“ 大数据工具”窗口显示由文件夹构成的已配置服务器和用户笔记本的列表。对于服务器的基本操作，请使用窗口工具栏：

项目

描述

将新连接添加到服务器。

刷新与所有已配置服务器的连接。

打开所选服务器的连接设置 。

笔记本编辑器

在笔记本编辑器中，您可以添加和执行Scala和SQL代码段。编辑代码段时，可以使用特定语言可用的所有编码帮助功能。代码警告和错误将在滚动条中的相应代码结构中突出显示。段落执行的结果显示在每个段落下方的预览区域中。

使用笔记本编辑器工具栏进行笔记本的基本操作：

项目

描述

执行笔记本中的所有段落。

停止执行笔记本段落。

清除所有段落的输出预览。

打开“ 解释器绑定”对话框，为所选笔记本配置解释器 。

在浏览器中打开笔记本。

打开模块设置 。

显示小地图，以快速浏览笔记本。

笔记本编辑器工具栏还显示了最后一段执行的状态。
