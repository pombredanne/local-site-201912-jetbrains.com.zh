新侦听器对话框

档案|新增|听众

指定新侦听器的设置。

此页面描述控件。有关如何配置Java EE的信息，请参阅启用Java EE应用程序支持。

项目

描述

名称

侦听器名称。仅用于生成默认的侦听器类名称。

包

应当在其中创建新的侦听器类的包的名称。

请点击 。然后选择一个现有的程序包或单击创建一个新的程序包。

类

侦听器类的全限定名称。

默认的类名称是使用包名称，类名称前缀，侦听器名称和类名称后缀生成的。

类名的前缀和后缀在“ 设置”对话框中设置 。（“ 编辑器” |“代码样式” |“ Java”页面的“ Java EE名称”选项卡上的“ 侦听器类 前缀和后缀”字段。）

如果要指定现有的类，请单击然后在打开的对话框中选择类。

创建Java EE 6带注释的类

如果未选中该复选框，则侦听器类的元素添加到web.xml中 。

如果选中此复选框，则新的侦听器类为@WebListener -带注释，并且不会对web.xml进行任何更改。

如果项目中没有web.xml ，则默认情况下会选中此复选框，并且您不能取消选择它。
