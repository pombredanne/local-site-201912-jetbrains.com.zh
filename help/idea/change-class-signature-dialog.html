更改类签名对话框

重构|更改签名
Ctrl+F6

使用“ 更改类签名”对话框执行“ 更改类签名”重构。

使用可用的控件来管理形式类型参数（请参阅下表）。您可以添加和删除参数以及更改其顺序。

单击“ 重构”以立即执行重构。单击预览以查看可能的更改，然后再实际执行重构。（这些将显示在“ 查找工具”窗口中 。）

图标

捷径

描述

Alt+Insert

使用此图标或快捷方式添加参数。分别在名称和默认值字段中指定参数名称和默认类型。

Alt+Delete

使用此图标或快捷方式删除所选参数。

Alt+Up

使用此图标或快捷方式将所选参数在列表中向上移动一行。

Alt+Down

使用此图标或快捷方式将所选参数在列表中向下移动一行。
