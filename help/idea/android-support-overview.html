IntelliJ IDEA 2019.3帮助

Android支持概述

为了支持Android开发，IntelliJ IDEA提供了专用的Android支持插件。该插件与IDE捆绑在一起，并且默认情况下处于启用状态。

IntelliJ IDEA中的Android支持包括：

  • 用于Android应用程序的基于Gradle的构建系统 。

  • 智能XML编辑器 ，有助于处理布局，清单，资源和其他XML文件。

  • 布局文件的图形编辑器 ：无需手动编辑XML文件即可构建应用程序的设计。

  • 增强了相关文件之间的导航 ：通过装订线图标在布局文件和相关活动，服务和清单文件等之间导航。

  • 自动生成基本布局文件 。

  • Android特定的代码检查 ，包括集成的Android Lint检查。

  • 可以同时在不同设备上预览应用程序UI 。

  • 直接从IDE 访问SQLite数据库 。

  • 内置9补丁编辑器 。

  • 借助资源类型注释，在Android API和您自己的库和API中检测不匹配的资源类型 。

  • 可以根据预定义的规则自动重新排列XML属性 。

  • 专用的Android运行/调试配置 。

  • 专用的Android工件 。
