EJB Module Editor

EJB Tool Window | context menu of a module with an EJB facet | Jump to Source

Use the tabs in this editor to configure the beans contained in a module with an EJB facet, method permissions, and transaction attributes.

If the EJB module editor is invoked from the deployment descriptor, the tab is also available.
