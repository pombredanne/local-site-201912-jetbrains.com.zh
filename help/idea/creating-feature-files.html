创建.feature文件

先决条件

适用于Java的Cucumber应该已下载并安装在您的计算机上！

启用 Cucumber for Java支持后，您可以创建功能文件。

创建特征文件

创建特征文件

  1. 在“项目”工具窗口中，右键单击应在其中创建特征文件的目录。

  2. 从目标目录的上下文菜单中，选择“ 新建” |“新建”。 File ，然后在New File对话框中，键入.feature 。

  3. 在功能文件中，键入您的方案。由于没有步骤定义，因此这些步骤将突出显示为未解决。

  4. 创建步骤定义 。
