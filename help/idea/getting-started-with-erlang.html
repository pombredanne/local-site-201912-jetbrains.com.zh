初步步骤

安装Erlang OTP

设置Erlang环境的第一件事是安装Erlang OTP，这是开发必不可少的一组Erlang库。

视窗

如果您是Windows用户，请下载Erlang OTP软件包并运行安装向导。安装结束后，将安装路径加\ bin添加到PATH环境变量中。

苹果系统

如果您是macOS用户，要安装Erlang OTP，请在终端提示下键入以下命令（确保计算机上已安装Homebrew ）：

高额二郎

如果您更喜欢MacPorts而不是Homebrew，则您的命令行应该有所不同：

高大的erlang + ssl

的Linux

Linux的安装过程与macOS类似，除了brew要么port你必须使用apt-get （Linux软件包管理实用程序）：

安装erlang

您始终可以为任何操作系统下载最新版本的Erlang OTP软件包。

安装钢筋

除了Erlang OTP之外，您还需要Rebar ，这是一个有助于编译和测试Erlang应用程序的构建工具。在计算机上安装它的最简单方法是下载其源代码并在本地构建它：

git clone git：//github.com/rebar/rebar.git $ cd钢筋$ ./bootstrap重新编译：src / getopt ...重新编译：src / rebar_utils ==> rebar（编译）

恭喜你！现在，您在当前工作目录中有一个名为“ rebar”的自包含脚本。将此脚本放在路径中的任何位置，您可以使用钢筋来构建符合OTP的应用程序。

设置IntelliJ IDEA

现在，设置了Erlang OTP和Rebar之后，就该下载并安装IntelliJ IDEA了。请记住，对于Erlang开发，您可以使用IntelliJ IDEA社区版（免费和开源）。

IDE启动后，您会看到其“ 欢迎”屏幕 ，请转至“ 配置” |“配置”。插件 ，然后单击浏览存储库 ，找到并安装Erlang插件：

安装插件后，重新启动IntelliJ IDEA。

配置一个Erlang SDK

配置IntelliJ IDEA还要做的另一件事是添加一个Erlang SDK。

为此，请更改默认项目的结构。通过以下两种方式之一打开默认项目结构：

  • 在“ 欢迎”屏幕上 ，转到“ 配置|项目默认值|项目结构

  • 在主菜单上，选择“ 文件” |“其他”。其他设置|新项目的结构

然后，通过指定Erlang OTP安装目录的路径来添加Erlang SDK。

如果您不知道Erlang OTP的安装位置，请检查以下目录：

  • Windows ： C：\ Program Files \ erl

  • Linux ： / usr / lib / erlang /

  • MacPorts，macOS ： / opt / local / lib / erlang /

  • 自制软件，macOS ： / usr / local / Cellar / erlang /

您可以通过多种方式将项目导入 IntelliJ IDEA的。让我们探索从“欢迎”屏幕导入的问题。
