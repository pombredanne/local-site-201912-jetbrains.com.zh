PHP支持包括：

  • 可以从模板创建PHP文件和类 。

  • 完整的PHP 5.6.x语法支持。

  • 对PHP 7返回类型提示的部分支持。

  • 语法和错误突出显示。

  • 基本的即时代码完成 。

  • 解析度include语句和文件引用，包括对PHAR存档的引用。

    当前项目中所有的PHAR文件以及指定的包含路径都显示在项目工具窗口中的“ 项目视图/库/ PHAR”节点下，并且可以在此处浏览。

  • 完成课程

  • 意向操作和快速修复

  • 围绕代码构造Ctrl+Alt+J和Ctrl+Alt+T 。

  • 代码检查 。

  • 跳转到声明 Ctrl+B 。

  • 重构 ：

    • 改名 Shift+F6 。

    • 移动 F6 。

    • 复制 F5 。

PHP开发支持通过PHP插件提供。 的该插件未与IntelliJ IDEA捆绑在一起，但可以如管理插件中所述从JetBrains插件存储库安装。

本部分描述了一些特定于开发PHP应用程序的过程以及一些配置PHP开发环境所需的初步步骤。

使用PHP开发应用程序

请遵循以下一般步骤：

  1. 配置PHP开发环境 。

  2. 如配置本地PHP解释器和配置远程PHP解释器中所述配置PHP解释 器 。

  3. 从头开始创建项目。在第一页的新建项目向导 ，选择PHP在左侧窗格中，然后选择PHP空项目在右侧窗格中。

    要在本地Web服务器上运行和调试应用程序，请在Web服务器文档根目录下创建项目根目录。因此，您的应用程序源对于本地Web服务器将是“可见的”。

  4. 创建和配置所需的数据源（请参阅管理数据源 ）。

  5. 使用提供的编码帮助来填充应用程序 。

  6. 部署应用程序。

    使用IntelliJ IDEA，您可以灵活地配置PHP应用程序的部署。例如，您可以从一开始就在本地Web服务器上设置PHP项目，或者在本地开发和测试应用程序，然后将其上传到远程Web服务器，依此类推。

  7. 运行应用程序。您可以通过以下几种方式进行操作：

    • 在IntelliJ IDEA中，使用类型为PHP Web Page的运行配置在浏览器中查看应用程序输出。

    • 在IntelliJ IDEA中，使用PHP脚本运行配置在“ 运行工具”窗口中查看应用程序输出。

    • 从IntelliJ IDEA，使用内置的Web服务器 。这种方法节省了您的时间和精力，因为您不需要部署应用程序源。

以下可选步骤可能会有所帮助：

  • 在您的项目中设置单元测试 。

  • 安装和配置调试引擎并指定调试器选项，有关详细信息，请参阅配置调试引擎和调试器 。

  • 调试应用程序。
