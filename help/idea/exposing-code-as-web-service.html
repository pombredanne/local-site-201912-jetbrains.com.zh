将代码公开为Web服务

假设您有一段实现某些功能的代码，并且希望该功能通过特定协议在Web上可用。在这种情况下，您需要将现有代码转换为Web服务并进行部署。该操作将更新Web服务描述符，并在需要时生成其他部署代码。每当更改任何Web服务方法签名或添加或删除新方法时，都需要执行此操作。

公开课程

  1. 将插入号放在编辑器中的班级名称上，然后按Alt+Enter或点击黄色灯泡图标 。

  2. 从建议列表中，选择“ 公开类作为Web服务” 。

  3. 在打开的“ 作为Web服务公开类”对话框中，指定以下内容：

    • Web服务的名称和URL地址。

    • 访问Web服务的公共操作时使用的协议和编码样式。

    • 类型信息，包括名称，操作，参数和构成Web服务接口的数据。

    IntelliJ IDEA会自动将服务描述添加到server-config.wsdd文件中。
