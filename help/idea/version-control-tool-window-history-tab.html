历史记录标签

历史记录选项卡已添加到 版本控制窗口Alt+9在调用显示历史或显示历史以供选择时 。将使用以下名称为每个文件或目录创建一个新选项卡： 历史记录： 。工具栏按钮的设置因版本控制系统而略有不同。

工具栏上所有可用的命令也可从所选内容的上下文菜单中获得。

工具提示和快捷方式

描述

刷新

单击此按钮刷新当前信息。

显示差异
Ctrl+D

单击此按钮可将文件的选定修订版本与“ 差异查看器”中的先前修订版本进行比较。

显示所有受影响的文件
Shift+Alt+A

单击此按钮以打开“修订中受影响的路径”对话框，您可以在其中查看在所选修订中修改的所有文件。

显示所有分支

单击此按钮以显示当前分支以外的分支的更改。

显示详细资料

单击此按钮以显示所选修订的提交消息。

在GitHub中打开

单击此按钮以打开与GitHub上所选提交对应的页面。
