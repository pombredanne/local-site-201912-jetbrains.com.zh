初步步骤

为ActionScript或Flex应用程序开发做准备：

  1. 确保已启用Flash / Flex支持插件。 的默认情况下，插件已激活。如果禁用了插件 ，请按照“ 管理插件”中所述在“ 插件”页面上启用它。

  2. 在计算机上下载并安装Flex或AIR SDK。IntelliJ IDEA提供与Apache Flex ， Adobe Flex ， Adobe AIR和Adobe Gaming SDK的集成。

  3. 在IntelliJ IDEA中注册Flex或AIR SDK 。（您可以将此步骤推迟到以后。创建Flash模块时，您将可以指定SDK。）

  4. 检查常规Flex编译器设置，并在必要时根据需要进行调整。

  5. 使用Flash模块创建一个新项目，或将Flash模块添加到现有项目中。请参阅创建Flash模块 。

    请注意，您可以通过导入Adobe Flash Builder项目来创建项目或模块。

  6. 检查模块的初始配置并进行必要的调整。这些可能包括配置模块内容 ， 添加库 ，修改现有构建配置以及创建其他构建配置等。

    但是，所有这些任务都可以在以后需要时与开发源代码同时执行 。

在IntelliJ IDEA中注册Flex或AIR SDK

在IntelliJ IDEA中注册Flex或AIR SDK

  1. 打开项目结构对话框（例如Ctrl+Shift+Alt+S ）。

  2. 在“ 平台设置”下 ，单击“ SDK” 。

  3. 单击添加新的SDK 在工具栏上，然后单击 添加新的SDK列表中的Flex / AIR SDK 。

  4. 在打开的对话框中 ，选择安装Flex或AIR SDK的目录，然后单击“ 确定” 。

    结果，SDK配置显示在“ 项目结构”对话框右侧的“ SDK”页面上。

  5. 一般情况下，你不需要做的类路径和SOURCEPATH选项卡的任何变化，因为所有必要的库和源已经存在。但是，您可能希望添加外部在线文档，以便在编写代码时能够访问其他参考信息。要做到这一点：

    在文档路径选项卡上，单击 。在打开的对话框中，只需单击确定 。 （IntelliJ IDEA建议的URL是您想要的URL。）

  6. 在“ 项目结构”对话框中单击“ 确定 ”。

配置常规Flex编译器设置

配置常规Flex编译器设置

  1. 打开“ 设置/首选项”对话框（例如， Ctrl+Alt+S ， 要么在工具栏上）。

  2. 在对话框的左侧，在“ 构建，执行，部署”节点下，打开“ 编译器”节点，然后单击“ 动作脚本”和“ Flex编译器” 。

  3. 在“ 编译器”>“ ActionScript和Flex编译器”页面上，根据需要配置设置。

配置模块内容

配置模块内容

通过添加和删除模块内容根目录以及通过将单独的文件夹（在内容根目录内）分配给源文件夹，测试源文件夹以及排除文件夹来配置模块内容。

对于Flash模块，通常，这些任务在“ 项目结构”对话框的“ 模块”页面上执行。

要访问此页面：

  1. 打开项目结构对话框（例如Ctrl+Shift+Alt+S ）。

  2. 在“ 项目设置”下 ，选择“ 模块” 。

  3. 在下面的区域 ，选择所需的Flash模块。
