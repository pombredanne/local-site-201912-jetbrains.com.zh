实现接口或抽象类的方法

如果一个类被声明为实现某个接口或使用抽象方法扩展了一个类，则它必须实现此类接口或类的方法。IntelliJ IDEA为实现的方法创建存根，原始类型的默认返回值，对象的空值。

实施所需的方法

  1. 在代码菜单上，单击实现方法 Ctrl+I 。或者，可以右键单击类文件中的任何位置，然后单击“ 生成”。 Alt+Insert ，然后选择实施方法 。

  2. 选择要实施的方法。如有必要，请选中“ 复制JavaDoc”复选框以为实现的接口或抽象类插入JavaDoc注释。

  3. 单击确定 。

变更方法主体

用于实现方法的代码模板（ Implemented method body ）从File Header include模板（例如${USER} ， ${DATE} ， 等等）

例如，考虑以下文件模板：

#if（$ RETURN_TYPE！=“ void”）返回$ DEFAULT_RETURN_VALUE; #end // TODO（$ USER，$ DATE）：要更改已实现方法的主体，请使用File |设置-编辑器-文件和代码模板。

假设实现的接口包含两个方法，则此模板扩展为以下代码：

@Override public void hunt（）{// TODO（wombat，9/21/12）：要更改已实现方法的主体，请使用File |设置-编辑器-文件和代码模板。} @Override public String sniff（）{返回null; // TODO（wombat，9/21/12）：要更改已实现方法的主体，请使用File |设置-编辑器-文件和代码模板。}
