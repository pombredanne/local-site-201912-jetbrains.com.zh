配置大数据工具环境

在使用大数据工具支持来开始使用笔记本之前，您需要安装所需的插件并配置与服务器的连接。

安装所需的插件

  1. 无论您在IntelliJ IDEA中做什么，都可以在项目中完成 。因此，打开一个现有项目（ File | Open ）或创建一个新项目（ File | New | Project ）。

  2. 在里面 设置/首选项对话框Ctrl+Alt+S ，选择插件|市场 。

  3. 安装以下插件：

    • 大数据工具

    • 斯卡拉

    • Bash支持

    另外，请检查是否已启用Python插件。默认情况下，应该为IntelliJ IDEA Ultimate安装它。

  4. 重新启动IDE。重新启动后，“ 大数据工具”选项卡将出现在工具窗口的最右侧。单击它以打开“ 大数据工具”窗口。

一旦在IDE中启用了Big Data Tools支持，就可以配置与服务器的连接。

配置服务器连接

  1. 在“ 大数据工具”窗口中，单击并选择服务器类型，例如Zeppelin 。将打开“ 大数据工具连接”对话框。

  2. 在“ 大数据工具连接”对话框中，指定以下必需参数：

    • URL ：目标服务器的路径。

    • 登录名和密码 ：您的访问目标服务器的凭据。

    （可选）您可以设置：

    • 名称 ：连接的唯一名称，以区分其他连接。

    • 以匿名身份登录：选择不使用您的凭据即可登录。

    • 每个项目 ：选择以仅对当前项目启用这些连接设置。如果希望此连接在其他项目中可见，请取消选择它。

    • 启用 ：如果出于某些原因要限制使用此连接，请取消选择。默认情况下，启用新创建的连接。

    • Scala版本 ， Spark版本和Hadoop版本 ：这些值是从插件捆绑包派生的。如果需要，请指定任何替代版本值。

    • HTTP基本身份验证 ：选择是否要使用目标服务器提供的NGINX和HTTP基本身份验证机制进行身份验证 。

  3. 单击测试连接以确保所有配置参数均正确。然后单击“ 确定” 。Big Data Tools窗口显示目标服务器的内容。

现在，您已经建立了与服务器的连接，您可以开始使用笔记本了。但是，确保在特定服务器上执行所需的所有库和软件包都已安装并可用，可能是一个好习惯。

配置笔记本依赖项

  1. 从主菜单中，选择文件|项目结构 。

  2. 在“ 项目结构”对话框中，在“ 项目设置”列表中选择“ 模块 ” 。然后，在模块列表中选择任何已配置的连接，然后双击System Dependencies 。

  3. 检查添加的库列表。单击列表，然后开始键入以搜索特定的库。

  4. 如果需要，修改库列表

    • 请点击添加一个新的库。

    • 请点击并指定外部文档的URL。

    • 请点击选择您希望IntelliJ IDEA忽略的项目（文件夹，归档文件和归档文件中的文件夹），然后单击OK 。

    • 请点击从库中删除选定的普通库或还原选定的排除项目。这些物品本身将留在图书馆中。
