提取参数对象对话框

重构|提取物|参数对象

使用此重构可围绕方法的某些选定参数创建包装器类，或使用兼容的现有类作为包装器。

项目

描述

从中提取参数的方法

此只读字段显示所选方法的名称。

参数类别

使用此部分可以指定是要创建新的包装器类，还是要使用现有的包装器类。

建立新课程

单击此单选按钮将方法的参数移至新类。如果选择此选项，请在下面的字段中指定类和目标包名称。默认情况下，显示当前软件包名称。您可以在文本字段中键入其他程序包名称，或单击省略号按钮并从树视图中选择目标程序包。如果所需的软件包不存在，请单击创建一个新的。

创建内部类

单击此单选按钮将方法的参数移动到内部类。如果选择此选项，请在下面的字段中指定类名称。

使用现有课程

单击此单选按钮将方法的参数移至您选择的现有类。您可以在文本字段中输入完全限定的类名称，或单击省略号按钮，然后在“ 选择参数类”对话框中选择所需的类。请注意，您可以从项目类和非项目类中选择所需的包装器类。

要提取的参数

使用本节中的复选框来选择应将方法的哪些参数提取到单独的对象中。

上移/下移

使用这些按钮在列表中移动选定的重新排序参数。

保持方法为委托

如果选中此复选框，则原始方法将保留为新创建方法的委托。
