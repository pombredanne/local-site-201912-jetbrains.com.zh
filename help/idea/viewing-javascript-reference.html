JavaScript文档查找

IntelliJ IDEA向您显示标准JavaScript API，项目中的符号及其依赖项以及外部库中定义的符号的参考。您可以在“ 文档”弹出窗口中查看符号的文档 ，获取有关方法参数的信息，并在外部站点打开文档（如果有指向它们的链接）。

查看插入符号处的文档

  • 按Ctrl+Q或选择查看|从主菜单快速查找文档 。该符号的引用显示在“ 文档”弹出窗口中。

    • 对于项目符号或项目依赖项中的符号，IntelliJ IDEA根据相应的JSDoc注释生成文档。

    • 对于标准JavaScript对象或方法，IntelliJ IDEA从内置TypeScript定义文件 d.ts中显示相应的JSDoc注释。这些文件与IntelliJ IDEA捆绑在一起，并定期更新。

    • 如果在d.ts文件中未找到注释，则IntelliJ IDEA将显示相应MDN文章的摘要。

要查看MDN文档中插入符号的信息

  • 在文档弹出窗口中Ctrl+Q ，请点击底部的链接：

  • 按Shift+F1或选择查看|主菜单中的外部文档 。

IntelliJ IDEA在默认的IntelliJ IDEA浏览器中打开MDN文章。

查看第三方JavaScript库的文档

  1. 下载所需的库或框架，将其配置为外部JavaScript库，并指定其外部文档的链接，有关详细信息，请参阅配置自定义第三方JavaScript库 。

  2. 将插入号置于相关符号处，然后按Shift+F1或选择查看|主菜单中的外部文档 。

查看npm软件包的文档

  • 在package.json文件或require要么import声明，将插入号放在包装名称下，然后按Ctrl+Q

  • 要在浏览器中打开npm官方网站上的文档，请按Shift+F1 。

在工具窗口中查看参考

  • 固定“ 文档”弹出窗口。它将进入“ 文档”工具窗口 ，带有相应的边栏图标和更多控件。

  • 要在“ 文档”弹出窗口和“ 文档工具”窗口之间切换，请按Ctrl+Q按顺序。

自动显示文档

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|代码完成 。

  2. 在打开的“代码完成”页面上，选中“ 显示文档”弹出复选框，并指定经过时间。

要更改快速文档的字体大小，请执行以下一项操作：

  • 请点击在快速文档窗口的右上角，然后移动滑块。

  • 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|常规，然后选择使用Ctrl +鼠标滚轮更改字体大小（缩放）复选框。有关详细信息，请参见有用的编辑器配置 。
