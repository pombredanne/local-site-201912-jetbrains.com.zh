Adding Mnemonics

You can specify a mnemonic character for action controls by placing an ampersand (&) character in front of the desired character in the text property. At runtime, the mnemonic character will be underlined. When you press the corresponding key on the keyboard, this control will be selected.

For components that can define a mnemonic character, there is an intention action Assign mnemonic that keeps track of mnemonics used in the form and suggests a character for the selected component from those not already used.

To escape the ampersand (&) so that it appears as a character in the control, type it twice. For example, Save && Run will render as Save & Run.
