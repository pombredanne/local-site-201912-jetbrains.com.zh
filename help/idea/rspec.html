先决条件

在开始使用Ruby之前，请确保已安装并启用 Ruby插件。

另外，请确保满足以下先决条件：

  • Ruby SDK已下载并安装在您的计算机上。

  • 所需的框架SDK已下载并安装在您的计算机上。

有关详细信息，请参阅其各自的下载和安装页面：

  • 红宝石
  • Ruby on Rails

RSpec 2.x注意

对于RSpec 2.x ，覆盖选项需要特殊配置。在启用代码覆盖率的情况下启动RSpec 2.x测试的Rake任务需要RCOV_OPTS环境变量，应按如下所示进行修改：

＃rspec 2.x需要“ rake”需要“ rspec / core / rake_task” desc“使用rcov运行所有规范” RSpec :: Core :: RakeTask.new（“ test_cov”）做| t | t.rcov = true t.rcov_opts = ENV [“ RCOV_OPTS”]结束

宙斯笔记

如果要在Zeus服务器下运行RSpec测试 ，请确保使用Zeus版本0.13.4.pre2或更高版本。
