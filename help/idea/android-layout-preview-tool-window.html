Android版式预览工具窗口

当您打开布局定义文件以手动模式进行编辑时，将显示工具窗口。它允许您调整布局预览的外观并模拟不同的配置。这可以帮助您将应用程序调整为不同的Android平台，设备，方向，停靠模式，语言环境等。

在这个部分：

  • 工具列
  • 控制项

显示选项菜单

单击此按钮可以配置工具窗口的外观和行为。

  • 对非布局文件隐藏：选择此选项可使IntelliJ IDEA在编辑器中的焦点切换到非布局文件时临时关闭工具窗口。一旦获得相同或另一个布局定义文件的焦点，工具窗口就会自动重新出现。

  • 包括设备框架（如果有）：选择此选项可使预览看起来像将出现在实际设备上。

  • 显示照明效果：选择此选项可显示照明效果，使预览看起来更自然。

    仅当选择了“ 包括设备框架”选项时，此选项才可用。

使用此区域中的控件预览不同配置的布局。

配置以在IDE中呈现此布局

从此列表中，选择要预览和编辑的布局配置，创建新的布局配置，或选择其他预览选项。可用的选项有：

  • 创建横向变化 ：选择此选项可创建布局的横向版本。相应的布局定义文件将在res \ layout-land文件夹中生成。创建此变体后，此菜单选项将替换为“ 切换到布局区域”选项，该选项将打开layout-land \ .xml文件进行编辑。

  • 创建layout-xlarge变体 ：选择此选项可为更大的屏幕尺寸（至少960x720 dp） 创建布局的变体。相应的布局定义文件将在res \ layout-xlarge文件夹中生成。创建此变体后，此菜单选项将替换为“ 切换到layout-xlarge”选项，该选项将打开layout-xlarge \ .xml文件进行编辑。

  • 切换到布局 ：仅当您创建了多个布局版本时，此选项才可用。选择它以返回到原始布局定义文件。

  • 创建其他 ：选择此选项可创建布局的另一个变体。在打开的“ 选择布局目录”对话框中，指定将存储布局定义的文件夹，然后选择确定特定设备配置的资源限定符 。选择相关的限定词，然后单击 。然后在打开的对话框中指定限定符的值。限定词将添加到“ 选择的限定词”列表中。

  • 预览代表样本选择此选项可显示多个设备配置并在最重要的屏幕尺寸上预览布局。

  • 预览所有屏幕尺寸 ：选择此选项可显示多种设备配置并预览所有可用屏幕尺寸的布局。

  • 预览所有语言环境 ：选择此选项可在将要使用您的应用程序的所有语言环境中预览布局。

  • 预览从右到左的布局 ：选择此选项可以并排预览两个方向（从左到右和从右到左）的布局。

  • 预览Android版本 ：选择此选项可预览所有已安装的Android API版本上的布局。

  • 包括预览 ：选择此选项可预览嵌套在另一个布局中的布局。仅当当前布局包含在另一个布局中时，此选项才可用。

  • 预览布局版本 ：选择此选项可显示多个设备配置并预览所有可用版本中的布局。

  • 无 ：选择此选项可返回默认视图。

  • 切换布局模式 ：选择此选项可在不同的预览选项之间切换。

用于渲染布局的虚拟设备

从此下拉列表中，选择一个虚拟或物理设备来预览您的应用程序在该设备上的外观。要添加新的虚拟设备，请选择添加设备定义，然后在打开的Android虚拟设备（AVD）管理器中配置仿真器（有关说明，请参阅管理虚拟设备 ）。

进入下一个状态

从此下拉列表中，选择预览方向（ 纵向或横向 ），UI模式（ 常规，汽车坞，桌面坞，电视，设备 ），然后在夜间 （屏幕变暗）和非夜间 （标准亮度）模式之间切换。有关UI模式的详细信息，请参考UiModeManager 。

主题

单击此按钮从“ 选择主题”对话框中选择一个主题。

不适用

单击此按钮可将布局与活动关联。选择关联使其与当前活动相关联，或与其他活动相关联以显示可供选择的活动列表。

在IDE中使用渲染布局的语言环境

从此下拉列表中，选择一个现有语言环境或为您的应用程序添加语言环境。语言环境是目标国家和语言的组合，可以根据本地规则和偏好显示日期和一些其他数据。您还可以在所有可用语言环境和两个方向（从左到右和从右到左）中预览布局。

在IDE中渲染布局时使用的Android版本

从此下拉列表中选择一个API版本，或使用“ 自动选择最佳”选项来使用最合适的Android版本呈现布局。您还可以预览所有已安装的Android API版本上的布局。
