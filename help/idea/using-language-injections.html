语言注入

语言注入使您可以处理代码中嵌入的其他语言的代码段。当您将一种语言（例如HTML，CSS，XML，RegExp等）注入字符串文字中时，您将获得用于编辑该文字的全面代码帮助。

在开始使用语言注入之前，请确保已启用 IntelliLang插件。

在PHP上下文中，可以使用称为Injectable PHP的特殊PHP方言。该方言类似于PHP，但开头标签，可以省略。可以像其他任何语言一样，将可注入的PHP手动注入到字符串文字中。此外，它会自动插入eval（）调用和Markdown防护代码块中，并将语言标识符设置为php 。

暂时注入语言

  1. 将插入符号放置在要插入语言的字符串文字，标记或属性中，然后按Alt+Enter （或使用意图操作图标 ）。

  2. 选择注入语言或参考，然后选择要注入的语言。

在专用编辑器部分中打开代码片段

  1. 将插入符号放置在注入的代码段内，然后按Alt+Enter （或使用意图操作图标 ）。

  2. 选择编辑片段 。

    IntelliJ IDEA将打开一个专用的编辑器部分，用于使用注入的语言编辑代码。该编辑器提供完整的代码帮助，包括代码完成，检查，意图和代码样式操作。

使用语言注入注释

  • 在目标字符串文字之前添加空白行，然后键入以下注释：

    -语言=

    对于注释，请使用要注入的语言的语法。语言ID通常是直观的，例如SQL，RegExp，XML，HTML。

语言片段可以与充当包装程序的前缀和后缀组合在一起，从而将该片段转换成语法上完整的语言单元。编辑代码时，您只能在片段编辑器中看到前缀和后缀。它们没有显示在主编辑器中。

前缀和后缀可以包含在注入注释中，如下所示：

//语言= 前缀= 后缀=

使用@Language批注

在某些语言中（例如，在Java，Groovy或Kotlin中），您还可以使用@Language("language_ID")注解以注入语言。

  1. 在目标字符串文字之前的空白行上，键入@Language("language_ID") 。

  2. 将插入符号置于注释处，按Alt+Enter然后选择将“注释”添加到classpath 。

    IDE将提示您从Maven下载带有注释的库。

  3. 在搜索字段中，输入org.jetbrains:annotations:16.0.2如果您使用的是JDK 1.8或更高版本。

    对于JDK 1.5、1.6或1.7，键入org.jetbrains:annotations-java5:16.0.2 。

  4. 单击确定 。

  5. 按Alt+Enter然后选择导入类 。

取消注射

  1. 将插入号放置在代码片段上，然后按Alt+Enter （或使用意图操作图标 ）。

  2. 选择取消注入语言或参考 。

要取消语言注入，您还可以删除注入注释或注释。

配置注入规则

您可以在上配置语言注入规则 编辑器| IntelliJ IDEA设置的“ 语言注入”页面Ctrl+Alt+S 。

所有预定义的注入规则都针对内置作用域配置。换句话说，它们是全局的（因此在所有IntelliJ IDEA项目中都可用）。可以为IDE或仅一个项目配置自定义规则。要更改自定义注入的范围，请使用 。

要配置自定义注入规则，请单击添加新规则，或复制预定义规则并更改其设置。

示例：用Java注入SQL

您可以配置设置，以便每次使用某种方法时，IDE都会注入选定的语言，例如SQL。

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，转到编辑器|语言注入 。

  2. 请点击然后选择Java参数 。

  3. 从ID列表中，选择SQL 。

  4. 在“ 类方法”字段中，输入要采用SQL参数的方法。您可以手动输入方法，或单击然后在“ 选择类”对话框中选择方法。

    如果该方法采用多个参数，它们将显示在对话框中。选择您要使用的。

  5. 应用更改并关闭对话框。
