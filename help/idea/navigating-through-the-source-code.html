源代码导航

您可以使用不同的操作和弹出窗口在编辑器中快速浏览代码。有关在编辑器和工具窗口之间导航的详细信息，请查看编辑器基础知识 。

您可以从观看视频教程开始，以检查IntelliJ IDEA中提供了哪些导航选项。

导航与插入符

  • 要向后导航，请按Ctrl + Alt + Left 。要向前导航，请按Ctrl + Alt + Right 。

  • 要导航到最后编辑的位置，请按Ctrl + Shift + Backspace 。

  • 要在编辑器中找到当前插入符的位置，请按Ctrl + M。如果您不想滚动大文件，此操作可能会有所帮助。

    但是，您可以按向上和向下箭头键获得相同的结果。

  • 要突出显示您要查找的插入符号处的单词，请选择“ 编辑” |“其他”。查找|从主菜单中的插入符号中找到Word 。

  • 要查看插入符号当前位于什么元素上，请按Alt + Q。

  • 要在开始和结束代码块大括号之间切换，请按Ctrl + Shift + M。

  • 要在代码块之间导航，请按Ctrl + Shift + [或Ctrl + Shift +] 。

移动插入符号

您可以使用不同的动作通过代码移动插入符号。您还可以配置插入字词和在线中断时插入符号应停止的位置。

  • 要将插入符号移动到下一个单词或上一个单词，请按Ctrl + Right或Ctrl + Left 。

    默认情况下，IntelliJ IDEA将插入符号移动到当前单词的末尾。

    当您将插入符号移动到上一个单词时，插入符号将放置在当前单词的开头。使用这些操作时，可以配置插入标记的位置。

    在里面 设置/首选项对话框Ctrl + Alt + S ，转至编辑器|一般在“ 插入符 移动”部分中，使用“ 按字移动时”和“换行时”选项来配置插入符的行为。

  • 要将插入符号向前移动到下一个段落或向后移动到上一个段落，请按Ctrl + Shift + A并搜索“ 将插入符号向前移动一个段落”或“ 将插入符号向后移动一个段落”操作。

    您也可以选择一个文本，然后将插入符号向前或向后移动到一个段落。按下Ctrl + Shift + A组合键，然后搜索带有选择的段落将插入符号向前移动或带有选择的动作将插入标记向后移动 。

    如果需要，可以为这些操作分配快捷方式。有关详细信息，请参考配置键盘快捷键 。

查找最近的位置

您还可以使用“ 最近的位置”弹出窗口检查您最近查看或更改的代码。

  • 要打开“ 最近的位置”弹出窗口，请按Ctrl + Shift + E。该列表从顶部的最近访问位置开始，并包含代码片段。

  • 在弹出窗口中，使用相同的快捷方式或选中仅显示更改的复选框，以仅查看更改了代码的位置。

  • 要搜索代码段，请在“ 最近的位置”弹出窗口中，开始输入搜索查询。您可以按代码文本，文件名或面包屑搜索。

  • 要从搜索结果中删除位置条目，请按Delete或Backspace 。

    请记住，删除的位置也会从您使用Ctrl + Alt + Left快捷方式访问的条目列表中删除。

使用书签进行导航

  • 要创建匿名书签，请将插入标记放置在所需的代码行中，然后按F11键。

  • 要创建带有助记符的书签，请将插入号放置在所需的代码行中，按Ctrl + F11并选择助记符的数字或字母。

  • 要显示下一个或上一个书签，请在主菜单中选择“ 导航”。书签|下一个书签或导航|书签|上一个书签 。

  • 要打开“ 书签”对话框，请按Shift + F11 。您可以使用此对话框来管理书签，例如，删除书签，对书签排序或为其提供简短说明。

  • 要导航到带有字母助记符的现有书签，请按Shift + F11 ，然后按需要的字母。IntelliJ IDEA使您返回编辑器和相应的书签。

  • 要导航到带有数字助记符的现有书签，请按Ctrl和书签的数字。

  • 每个创建的书签都会反映在“ 收藏夹 Alt + 2”工具窗口中，您也可以使用该窗口导航到书签。

在变更之间导航

如果编辑受版本控制的文件，则IntelliJ IDEA提供了几种来回移动更新的方法。特别是，您可以使用导航命令，键盘快捷键和更改标记。

要导航到上次编辑的位置，请按Ctrl + Shift + Backspace或选择“ 导航|导航”。从主菜单的最后编辑位置 。

  • 按Ctrl + Shift + Alt +向下 / Ctrl + Shift + Alt +向上 。

  • 从主菜单中，选择导航|下一个/上一个更改 。

  • 单击更改标记 ，然后单击要么 。

查看最近的变化

您可以使用“ 最近的更改”弹出窗口查看项目中本地或外部更改的文件列表。如有必要，您可以还原这些更改。

  1. 在主菜单中，选择查看|最近更改 Shift + Alt + C。

  2. 在“ 最近的更改”弹出窗口中，选择所需的文件，然后按Enter在另一个对话框中将其打开，您可以在其中检查更改的内容，并在必要时还原这些更改。

转到声明及其类型

您可以从符号的用法导航到符号的初始声明和符号类型。

  • 将插入号放置在所需的符号上，然后按Ctrl + B。

  • 对于类型声明，请按Ctrl + Shift + B。

前往实施

您可以使用编辑器中的装订线图标或按相应的快捷方式来跟踪类的实现和重写方法。

  • 单击其中一个 / ， / 装订线图标，然后从列表中选择所需的上升或下降类别。

  • 要导航到超级方法，请按Ctrl + U。

  • 要导航到实现，请按Ctrl + Alt + B。

使用“选择范围”弹出窗口进行导航

您可以在“ 项目”工具窗口中自动找到一个类。

  1. 如果在编辑器中打开了该类，请按Alt + F1打开“ 选择范围”弹出窗口。

  2. 在弹出窗口中，选择项目视图 ，然后按Enter 。IntelliJ IDEA在“ 项目”工具窗口中找到目标。

在“项目”工具窗口中找到文件

您可以使用“ 单击打开文件”和“ 始终选择打开的文件”操作来在“ 项目”工具窗口中找到文件。

  1. 在“ 项目”工具窗口中，右键单击“ 项目”工具栏，然后从上下文菜单中选择“ 始终选择打开的文件” 。之后，IntelliJ IDEA将跟踪当前在活动编辑器选项卡中打开的文件，并自动在“ 项目”工具窗口中找到它。

  2. 您也可以选择“ 单击打开文件”选项。在这种情况下，当您在“ 项目”视图中单击文件时，IntelliJ IDEA将自动在编辑器中将其打开。

在错误或警告之间导航

  • 要跳至代码中的下一个或上一个发现的问题，请分别按F2或Shift + F2 。或者，从主菜单中选择“ 导航|导航”。下一个/上一个突出显示的错误 。

    IntelliJ IDEA会在代码发布之前立即将插入号放置。

  • 配置IntelliJ IDEA在代码问题之间导航的方式：它可以在所有代码问题之间跳转，也可以跳过次要问题，并且仅在检测到的错误之间导航。右键单击滚动条区域中的代码分析标记，然后从上下文菜单中选择一种可用的导航模式：

    • 要使IntelliJ IDEA跳过警告，信息和其他次要问题，请选择仅转到高优先级问题 。

    • 要使IntelliJ IDEA在所有检测到的代码问题之间跳转，请选择转到下一个问题 。

在“结构”视图弹出窗口中找到一个代码元素

您可以使用结构视图弹出窗口在正在处理的文件中找到代码元素。

  1. 要打开结构视图弹出窗口，请按Ctrl + F12 。

  2. 在弹出窗口中，找到您需要的项目。您可以开始为IntelliJ IDEA输入元素的名称，以缩小搜索范围。按Enter键返回到编辑器和相应的元素。

    在弹出窗口中，您可以对文件成员进行排序，查看匿名类和继承的成员。

浏览方法

  • 按Alt +向下或Alt +向上 。

  • 为了在视觉上分离代码中的方法， 在里面 设置/首选项对话框Ctrl + Alt + S ，转至编辑器|一般|外观并选择显示方法分隔符选项。

  • 要打开“ 结构”工具窗口，请按Alt + 7 。

使用镜头模式

镜头模式使您无需实际滚动即可预览代码。当您将鼠标悬停在滚动条上时，默认情况下该模式在编辑器中可用。将鼠标悬停在警告或错误消息上时，此功能特别有用。

  • 要禁用镜头模式，请右键单击位于编辑器右侧的代码分析标记，然后在上下文菜单中清除“ 在滚动条悬停时显示代码镜头”复选框。

  • 作为备选， 在里面 设置/首选项对话框Ctrl + Alt + S ，转至编辑器|一般|出现并清除滚动条悬停上的显示代码镜头复选框。

使用面包屑进行导航

您可以使用面包屑浏览源代码，这些面包屑显示当前打开的文件中的类，变量，函数，方法和标记的名称。默认情况下，启用面包屑并将其显示在编辑器的底部。

  • 要更改面包屑的位置，请右键单击一个面包屑，在上下文菜单中选择“ 面包屑”和位置首选项。

  • 要编辑面包屑设置， 在里面 设置/首选项对话框Ctrl + Alt + S ，转至编辑器|一般|面包屑 。

    清除“ 显示面包屑”选项以在编辑器中隐藏面包屑。

使用导航栏导航到文件

使用导航栏作为方便的工具来查找整个项目。

  1. 按Alt + Home激活导航栏。

  2. 使用箭头键或鼠标指针找到所需的文件。

  3. 双击选定的文件，或按Enter在编辑器中将其打开。

查找线或列

  1. 在编辑器中，按Ctrl + G。

  2. 在“ 转到行/列”对话框中，指定行号或列号，或同时指定两者，并用：分隔它们，然后单击“ 确定” 。

  3. 如果您不想在编辑器中看到行号， 在里面 设置/首选项对话框Ctrl + Alt + S ，转至编辑器|一般|外观并清除显示行号复选框。

为了更快地访问，请将快捷方式分配给“ 显示行号”操作。

查找文件路径

  1. 在编辑器中，按Ctrl + Alt + F12或在上下文菜单中，选择“ 在Finder中显示” 。

  2. 在“ 在查找程序中显示”弹出窗口中，选择要在路径查找器中打开的文件或目录，然后按Enter 。

查找最近的文件

您可以使用“ 最近的文件”弹出窗口搜索最近和最近编辑的文件。

  • 要打开带有最近文件列表的“最近文件”弹出窗口，请按Ctrl + E。

  • 要仅查看最近编辑的文件，请再次按Ctrl + E或选中仅显示更改的复选框。

  • 要在弹出窗口中搜索项目，请开始输入搜索查询。IntelliJ IDEA根据您的搜索显示结果。
