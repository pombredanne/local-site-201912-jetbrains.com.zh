从Bnd / Bndtools导入项目

在导入项目之前，请确保已启用捆绑的OSGi插件。

  1. 启动IntelliJ IDEA。

    如果“欢迎”屏幕打开，请单击“ 导入项目” 。

    否则，从主菜单中选择File |新品|现有的项目 。

  2. 在打开的对话框中，选择源，库和其他资产所在的目录，然后单击“ 打开” 。

  3. 在“ 导入项目”对话框中，选择“ 从外部模型导入项目|”。 Bnd / Bndtools ，然后单击下一步 。

  4. 选择要导入的Bnd / Bndtools项目。

    您可以选择“ 导入后打开项目结构”选项，以在导入后立即打开“项目结构”对话框，在此可以完成对项目的配置。

    单击下一步 。

  5. 指定项目SDK，然后单击Finish 。
