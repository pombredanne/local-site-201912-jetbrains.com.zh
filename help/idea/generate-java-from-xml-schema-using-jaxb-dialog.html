使用JAXB对话框从Xml模式生成Java

工具| JAXB |使用JAXB从XML模式生成Java代码

使用此对话框可通过JAXB数据绑定器配置基于XML模式的Java代码存根的生成。

  • 此功能通过WebServices捆绑插件提供，默认情况下启用。如果没有，请按照“ 管理插件 ”部分中所述启用它。

  • 当在活动编辑器选项卡中打开的文件包含XML架构时，菜单项和对话框可用。

项目

描述

JAXB模式/ wsdl / dtd路径

在此字段中，指定要用作生成基础的文件。默认情况下，该字段显示当前文件的完整路径。要使用另一个架构，请单击浏览。 然后在打开的“ 选择用于JAXB的XML模式文件”对话框中选择所需的文件。

输出路径

从此下拉列表中，选择模块源目录以将生成的Java代码存根放入其中。

包前缀

使用此下拉列表可以指定将生成的Java文件放入其中的包。

生成包级注释

请执行以下任一操作：

  • 选中此复选框可生成带有注释的package-info.java 。

  • 清除此复选框可将注释内部化到其他生成的类中。

使用“生成的”注释标记生成的代码

选中此复选框可生成与javax.annotation.Generated注释。

将生成的文件设为只读

选中此复选框可强制XJC绑定编译器将生成的Java源代码标记为只读 。

添加必要的库以便生成的代码进行编译和工作

选择此复选框可将其他JAXB客户端库自动添加到将放置生成的源代码的模块的类路径中。

这些是生成的存根代码所依赖的库。

不产生标题

选中此复选框以通过no-header相应命令的参数。

添加外部绑定文件/目录

选择此复选框以指定外部绑定文件或生成文件所在的输出目录。

状态

在配置生成过程时，查看此只读字段中的信息以跟踪和改善差异。
