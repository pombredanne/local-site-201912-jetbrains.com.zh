从现有代码创建Grails应用程序

如果您已经有Grails项目，则可以围绕其源代码创建IntelliJ IDEA Grails应用程序。IntelliJ IDEA识别结构并创建Grails项目视图。

如果您使用Grails 3，那么您可以简单地为Gradle模型导入项目，因为Grails 3使用了Gradle构建系统。

对于Grails 1/2，请执行以下步骤：

  1. 启动“ 新建项目”向导 。如果IntelliJ IDEA当前未打开任何项目，请在“欢迎”屏幕上单击“ 导入项目 ”。否则，选择文件|新品|从主菜单从“现有来源 ”进行投影。

  2. 在打开的对话框中，选择您的Grails项目，然后单击OK 。

  3. 在向导的下一页上，选择从现有来源创建项目 ，然后单击下一步 。

  4. 在向导的下一页上，选择从现有来源创建项目 ，然后单击下一步 。

  5. 在向导的下一页上，指定Grails项目的名称和位置，然后单击Next 。

  6. 在向导的下一页上，选择要包含在项目中的源根，然后单击下一步 。

  7. 在向导的下一页上，选择将包含在项目中的库，然后单击下一步 。

  8. 在向导的下一页上，查看将添加到项目中的模块结构和依赖项，然后单击下一步 。

  9. 在向导的下一页上，选择项目SDK，然后单击下一步 。

  10. 在向导的下一页，如果检测到框架，则IntelliJ IDEA将显示框架。点击完成 。

IntelliJ IDEA通过其专用结构打开一个Grails项目。此时，如果IntelliJ IDEA在项目中检测到Gradle构建工具，则将显示提示导入Gradle项目的弹出消息。
