播放配置对话框

工具|玩Playframework

工具|仅在以下情况下， “使用Playframework播放”命令和“ 播放配置”对话框才可用：

  • Playframework支持插件已启用。请参阅管理插件 。

  • 当前在IntelliJ IDEA中打开了一个Play应用程序。 （用技术术语来说，目录\ framework \ lib和文件\ framework \ play- .jar包含在代表您的Play应用程序的模块的依赖项中。）

此外，如果在Play框架设置中关闭了“在控制台上运行时显示”选项，则不会显示“ 播放配置”对话框。

使用此对话框可以指定Play框架设置，例如Play框架安装目录和play命令行实用程序的工作目录（Play控制台）。

项目

描述

家

指定Play框架安装目录。

在字段中输入路径，或单击Shift+Enter然后在打开的对话框中选择目录。

工作目录

指定要从其运行play命令行实用程序的命令的目录。通常，这是您的Play应用程序的根目录。

在字段中输入路径，或单击Shift+Enter然后在打开的对话框中选择目录。

在控制台运行时显示

每次访问IntelliJ IDEA中的play命令行实用程序时，选中此复选框即可查看和修改上面讨论的Play框架设置。
