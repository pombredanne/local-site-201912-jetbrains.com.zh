测试框架

IntelliJ IDEA支持使用以下测试框架：

  • JUnit的
  • 测试NG
  • Groovy JUnit
  • 弹性单元
  • MXUnit

仅当安装并启用PHP插件时，以下内容才有效。

  • PHPUnit

以下内容仅在安装并启用Python插件后才有效。

  • Python单元测试
  • pytest
  • Python鼻子测试
  • Python文档测试

  • 毒物 。

  • TwistedTrial

  • BDD框架：

    • 表现得好 。

    • 生菜 。

    • 黄瓜

    有关详细信息，请参阅“ BDD测试框架 ”部分。

以下仅在安装并启用Ruby插件时有效。

  • Test :: Unit用于单元测试，并与Ruby捆绑在一起。

  • Shoulda适用于单元测试，并且在附加了shoulda宝石。这样，将Shoulda测试添加到Test :: Unit框架上。

  • RSpec 。该测试工具支持BDD。 RSpec在附加了rspec宝石。对于Rails应用程序，还需要rspec-rails gem。

  • MiniTest可在连接minitest-reporters gem时使用。最小测试将添加到Test :: Unit框架中。

  • 黄瓜此测试工具支持BDD，并允许使用以人类可读语言（英语或英语中指定的任何其他语言）编写的功能和场景# language:评论。安装和激活黄瓜宝石后，黄瓜将在项目中可用。

在开始使用您选择的测试框架之前，请确保在计算机上安装了所需的框架。有关安装详细信息，请参阅框架文档。

对于每个受支持的测试框架，IntelliJ IDEA提供：

  • 代码完成 ，了解特定的测试框架>。

  • 运行/调试配置 。

  • 能够创建测试 。

  • 能够在测试和测试主题之间导航 。

  • 能够从IDE内运行测试，并在“运行”工具窗口的“ 测试 运行器” 选项卡上查看测试结果。

  • 能够运行目录中的所有测试或功能，特定的测试类，测试用例或功能，单个测试方法或示例。

  • 代码检查 。

有关常见测试过程的详细说明，请参见“ 执行测试 ”部分。
