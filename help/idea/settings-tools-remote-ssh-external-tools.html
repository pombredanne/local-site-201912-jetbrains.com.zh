远程SSH外部工具

档案|设置|工具|适用于Windows和Linux的远程SSH外部工具
IntelliJ IDEA |首选项|工具|适用于macOS的远程SSH外部工具
Ctrl+Alt+S

将需要SSH访问的远程应用程序定义为外部工具，以便能够从IntelliJ IDEA运行它们。

您可以将上下文信息（例如当前选择的文件或您的项目源路径）传递到外部工具，查看工具输出等。

在此页面上定义的工具在“ 工具”菜单和各种上下文菜单中显示为命令。还可以为它们分配键盘快捷方式（请参阅配置键盘和鼠标快捷方式部分）。

描述

添加外部工具的定义。（将打开“ 创建工具”对话框 。）

如果选择了单个工具：删除工具定义。如果选择了一个工具组：删除所选组中所有工具的定义。

编辑所选工具的定义。（将打开“ 编辑工具”对话框 。）

 

在组中将所选工具向上或向下移动一行。（工具的顺序定义了相应菜单中项目的顺序。）

创建所选定义的副本，然后编辑该副本。（将打开“ 复制工具”对话框 。）

选框

使用复选框来启用或禁用工具和工具组。当前未选择的项目在“ 工具”和上下文菜单中不可用。
