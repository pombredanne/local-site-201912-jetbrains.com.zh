运行查询

要运行查询，请创建一个新文件或打开一个现有的SQL文件，将其连接到数据源，然后运行您的代码。在服务工具窗口中，您可以查看数据源（1），连接会话（2）和附加文件（3）。绿灯亮表示您已连接到数据源。

数据库控制台是已经连接到数据源的SQL文件，因此您可以立即键入并运行代码。

从打开的文件中运行语句

在IntelliJ IDEA中，您可以打开并运行一个SQL文件。SQL文件大小的限制为20 MB。当您打开一个大于20 MB的文件时，您只会看到文件的前2.5 MB。

  1. 打开“项目”工具窗口（“视图” |“工具窗口” |“项目”），然后双击一个SQL文件。

  2. 单击您要执行的语句。要运行多个语句的选择，请选择（突出显示）这些语句。

  3. 按Ctrl+Enter或从上下文菜单中选择执行 。

  4. （可选）如果SQL文件未连接到数据源，请在“ 会话”窗口中选择或创建连接会话。有关会话的更多信息，请参阅管理连接会话 。

  5. 在“ 语句”窗口中，按Enter运行选定的语句。您可以在其他条目之间切换以运行另一组语句。您将要执行的语句在编辑器中突出显示。

您一次只能为一个数据源运行单个语句。

从硬盘运行SQL文件

  1. 在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中，右键单击数据源。

  2. 选择“运行SQL脚本” 。

  3. 在“ 选择路径”窗口中，导航到要应用的SQL文件。

为多个数据源运行一个SQL文件

  1. 在“ 项目”工具窗口（“ 视图” |“工具窗口” |“项目” ）中，选择要运行的SQL文件。

  2. 右键单击该SQL文件，选择“运行”。 （或按Ctrl+Shift+F10 ）。

  3. 在“ 执行目标”窗口中，选择必要的数据源或方案。

在数据库控制台中执行语句

创建数据源时，将自动创建数据库控制台 。有关数据库控制台的更多信息，请参见数据库控制台 。

  1. 在数据库工具窗口（视图|工具窗口| 数据库 ）中，单击数据源。

  2. 打开F4或创建（ Ctrl+Shift+F10 | 新控制台 ）数据库控制台。有关使用控制台的更多信息，请参见使用控制台 。

  3. 键入或粘贴要执行的语句。

  4. 按Ctrl+Enter 。或者，单击执行图标在工具栏上。

    如果有多个语句，请选择是执行所有语句还是执行单个语句。建议列表始终包含用于运行所有语句的项目。

新控制台意味着与数据源的新连接。您只能为数据源使用一个连接，并让所有控制台使用同一连接。在“ 启用单一连接模式”中了解有关单一连接模式的更多信息。

生产力提示

从搜索对话框执行文件

  1. 按Shift两次。

  2. 在“ 到处搜索”对话框中，键入文件名。

  3. 使用向上和向下箭头选择文件。

  4. 按Ctrl+Shift+F10 。

  5. 选择要为其运行文件的数据源，然后单击“ 确定” 。

创建数据库控制台

要创建控制台，请在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中使用以下操作之一。

  • 单击数据源，然后选择“ 文件” |“文件”。新品|控制台 。

  • 右键单击数据源，然后选择新建|控制台 。

  • 单击数据源，然后按Alt+Insert ，然后选择控制台 。

  • 单击数据源，然后按Ctrl+Shift+F10 ，然后选择新建控制台 。

新控制台意味着与数据源的新连接。您只能为数据源使用一个连接，并让所有控制台使用同一连接。在“ 启用单一连接模式”中了解有关单一连接模式的更多信息。

打开控制台

  • 在“ 项目”工具窗口（“ 视图” |“工具窗口” |“项目” ）中，导航到“ 临时和控制台” |“工具”。数据库控制台| ，双击控制台文件。

  • 在数据库工具窗口中，单击跳转到控制台图标（ ），然后选择一个控制台。

  • 单击数据源，然后按F4 ，然后选择所需的控制台。

将控制台另存为文件

在控制台文件中，按N/A 。导航到要存储文件的目录。
