IntelliJ IDEA |首选项|工具|适用于macOS的Web服务
Ctrl+Alt+S

使用此对话框可以指定用于生成Web服务的URL地址的默认服务器名称和端口，以及无法直接通过专用方面启用的Web服务引擎的安装目录的路径。

项目

描述

外部引擎

在本节中，指定外部Web服务引擎的安装目录的路径。手动输入路径，或单击浏览 然后在打开的对话框中选择路径。支持以下外部引擎：

  • Glassfish / JAX-WS 2.2 RI / Metro 1。X / JWSDP 2.2

  • Apache Axis 2
  • CXF
  • 老板
  • Xml豆
  • WebSphere 6.x

服务器名称

在此字段中，指定要在URL地址中使用的服务器的默认名称。IntelliJ IDEA建议使用localhost 。

服务器端口

在此字段中，指定要在URL地址中使用的默认端口号。IntelliJ IDEA建议使用8080 。

Web服务URL的前缀路径

在此字段中，指定要在Web服务的URL地址中使用的默认前缀。IntelliJ IDEA建议/ services 。

启动工具时的最大VM堆大小（Mb）

在此字段中，指定启动Web服务时将使用的默认最大堆大小（以Mb为单位）。
