数据库颜色设置对话框

在数据库工具窗口中 ：
选择一个或多个感兴趣的元素，然后从上下文菜单中选择“ 颜色设置” 。

选择颜色 ，然后指定该颜色的使用方式（请参阅递归“ 共享和覆盖” ）。使用外观设置部分中的复选框可以在UI的各个位置启用或禁用数据库颜色。

描述

颜色

单击要用于数据库工具窗口中所选元素的颜色。

如果缺少所需的颜色，请单击“ 自定义” ，然后在打开的对话框中指定颜色。

当前选择的颜色标有其名称。蓝色的 。

如果您不想使用任何颜色，请点击无颜色 。

共享

如果您要与团队共享要更改的颜色，请选中该复选框。（通过版本控制共享数据库颜色设置。）

用技术术语来说，新的颜色（用于选定的一个或多个元素）将存储在

  • .idea / databaseColors.xml或.ipr文件（如果选中此复选框）。

  • .idea / workspace.xml或.iws文件（如果未选中此复选框）。

通常，不是通过版本控制来共享workspace.xml或.iws文件，而其余的IntelliJ IDEA配置文件则是。

有关IntelliJ IDEA配置文件及其共享的更多信息，请参阅项目 。

递归覆盖

如果选中此复选框，则选定的颜色将递归应用于元素本身及其所有从属元素。

如果未选中该复选框，则不会单独更改为从属元素设置的颜色。

外观设置

启用数据库颜色

清除复选框以（临时）禁用UI中所有位置的数据库颜色。

选中该复选框以使用数据库的颜色在数据库工具窗口。

选中复选框以使用数据库颜色作为编辑器选项卡（使用数据编辑器和数据库控制台的输入窗格时）。

在控制台编辑器和网格中

选中复选框以将数据库颜色用于数据库控制台和数据编辑器中的编辑区域。

禁用此选项时，编辑区域如下所示：

启用此选项时，编辑区域的背景颜色与相应的数据源相同的颜色（例如，蓝色）。

在工具栏中

选中该复选框以将数据库颜色用于数据库控制台和数据编辑器的工具栏。

禁用此选项后，工具栏如下所示：

启用此选项后，工具栏背景的颜色与相应的数据源的颜色相同（例如，蓝色）。

没有颜色

如果您不想为所选元素使用任何颜色，请单击此按钮。
