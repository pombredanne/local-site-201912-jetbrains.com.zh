创建侦听器

GUI Designer使您可以创建组件的侦听器。具有侦听器的组件带有特殊图标标记选择该组件时显示。从“表单工作区”和“组件”树视图中的组件，可以快速导航到相应的源代码。

创建一个监听器

  1. 在“表单工作区”或“组件”树视图中选择组件。

  2. 在上下文菜单上，选择“ 创建侦听器” ，或按Ctrl+O 。

  3. 从“ 创建侦听器”弹出菜单中，选择所需的侦听器类型。

  4. 在“选择要替代的方法”对话框中，选择所需的方法。（可选）指定是否要将此方法的JavaDoc注释复制到目标文件，然后按OK 。覆盖指定方法的存根方法将添加到绑定的类中。

  5. 如果要修改存根方法的主体，请编辑“覆盖的方法主体”文件模板：

    在“ 文件模板”对话框中 ，打开“ 代码”选项卡，然后根据需要编辑模板。

从组件导航到侦听器

  1. 在“表单工作区”或“组件”树视图中选择组件。

  2. 在上下文菜单上，选择“ 转到侦听器” 。

  3. 在“ 导航到侦听器”弹出菜单中，单击所需的侦听器，或按Ctrl+Alt+B 。

在表单工作区中，单击并选择所需的监听器。
