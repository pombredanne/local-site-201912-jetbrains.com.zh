代码覆盖率

代码覆盖率使您可以查看在单元测试期间执行了多少代码，因此您可以了解这些测试的有效性。

IntelliJ IDEA中提供以下代码覆盖率运行器：

  • IntelliJ IDEA代码覆盖率运行程序（推荐）。

  • EMMA开源工具包。请注意，作者不再支持EMMA，并且仅当框架验证关闭（通过-noverify的过程）。

  • JaCoCo 。

确保已启用Code Coverage捆绑插件。如果插件被禁用，请按照“ 管理插件”中所述激活它。
