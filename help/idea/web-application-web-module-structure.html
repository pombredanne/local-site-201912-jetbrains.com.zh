填充Web模块

在模块中启用Web开发支持后，IntelliJ IDEA如下设置基本模块结构：

  • 使用index.jsp存根文件创建一个Web文件夹。默认情况下， Web文件夹将是部署后应用程序的根目录，而index.jsp文件将是其主页。

  • 创建一个WEB-INFO子文件夹，其中包含web.xml描述符和一个Action已配置servlet。

  • 如果指定，则在创建项目时创建一个src文件夹。

要填充Web模块，请执行以下常规步骤：

  • 在src文件夹下，创建实现您的应用程序功能的Java类文件。

  • 配置代表用户界面的静态Web内容资源 。

  • 配置应用程序元素 。

  • 指定程序集描述符引用 。
