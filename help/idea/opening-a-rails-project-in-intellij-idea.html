打开和运行Rails项目

在本教程中，我们将向您展示如何克隆现有的Rails应用程序，配置Ruby解释器，安装依赖项以及运行该应用程序。以下应用程序将用于此目的： Sample Rails App 。

先决条件

在使用上述示例Rails项目之前，请执行以下操作：

  • 为您的平台安装Ruby发行版 。

  • 安装并启用Ruby插件 。请参阅管理插件以了解如何使用插件。

  • 安装Node.js和Yarn 。

检出项目

首先，我们需要克隆包含示例应用程序的存储库：

  1. 在“ 欢迎屏幕”上 ，单击“从版本控制获取 ” ，然后选择“ Git” 。

  2. 在调用的“ 克隆存储库”对话框中，将以下存储库地址插入URL字段： https : //github.com/JetBrains/sample_rails_app.git 。

    然后，单击“ 克隆”按钮。

  3. 克隆存储库后，系统将提示您从这些来源创建IntelliJ IDEA项目。

    单击是 。

  4. 在调用的“ 导入项目”向导中，在向导的每个页面上单击“ 下一步” ，保留默认设置。

  5. 在最后一页上，单击“ 完成” 。

配置Ruby SDK

现在，我们需要为该项目配置Ruby解释器：

  1. 转到文件|项目结构 ，或按Ctrl+Shift+Alt+S 。

  2. 在调用的“ 项目结构”对话框中，切换到“ 模块”页面，然后选择Ruby解释器。

    单击确定 。

安装gem依赖项

要安装项目的Gemfile中指定的gem，请执行以下操作：

  1. 按Ctrl两次，然后输入以下命令： gem install bundler 。

    按Enter并等待，直到安装了Bundler gem。

  2. 按Ctrl再次两次，然后开始键入bundle install 。然后，选择bundle install从列表中命令，然后按Enter 。

  3. 在调用的捆绑安装对话框中，单击安装 。

  4. 等到IntelliJ IDEA安装所有gem。

安装JavaScript依赖项

要使用Yarn安装项目的package.json文件中提到的JavaScript依赖项，请执行以下操作：

  1. 按Ctrl再次输入两次并开始输入yarn install 。然后，从列表中选择纱线安装命令，然后按Enter 。

  2. 等到IntelliJ IDEA安装所有gem。

运行迁移

要运行Rails应用程序，首先我们需要迁移数据库：

  1. 按Ctrl两次并输入db：migrate 。选择rake db:migrate在下拉菜单中，然后按Enter 。

  2. 将默认设置保留在调用的“ 执行db：migrate”对话框中，然后单击“ 确定” 。

  3. 等待直到迁移完成。

运行应用程序

要运行我们的Rails应用程序，我们需要创建一个相应的Rails配置：

  1. 按Ctrl+Shift+A然后开始输入编辑配置 。选择编辑配置 ，然后按Enter 。

  2. 在调用的“运行/调试配置”对话框中，单击然后选择Rails 。

  3. 将创建的配置名称更改为development并启用“ 运行浏览器”选项。

    单击确定 。

  4. 按Ctrl两次，然后开始输入创建的配置的名称（ development ）。从列表中选择它，然后按Enter 。

  5. 在“运行工具”窗口中，IntelliJ IDEA将显示准备运行该应用程序的过程。

    ...，然后在浏览器中打开该应用程序。
