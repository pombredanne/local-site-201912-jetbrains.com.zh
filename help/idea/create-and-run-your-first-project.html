创建和运行您的第一个Rails应用程序

本教程将向您展示如何在IntelliJ IDEA中创建和运行最简单的Rails项目。

在开始本教程之前，请执行以下操作：

  • 下载并为您的平台安装Ruby发行版 。

  • 安装并设置 RubyMine。

  • 安装Ruby插件 。

我们将使用安装在macOS上的IntelliJ IDEA执行所有步骤。

下载完成的项目： https : //github.com/rubyminedoc/rails_helloworld

创建一个Rails应用程序

要从头开始创建Rails应用程序，请执行以下操作：

  1. 运行IntelliJ IDEA，然后在“ 欢迎”屏幕上单击“ 创建新项目 ” 。

  2. 在“ 新建项目”对话框中，在左窗格中选择“ Ruby on Rails ”。然后，在Project SDK中选择所需的Ruby解释器，然后选择Rails版本。单击下一步 。

  3. 在下一页上，指定项目名称和位置。点击完成 。

  4. IntelliJ IDEA将创建一个新的Rails应用程序并安装Gemfile中提到的gem依赖项。您可以在“运行”工具窗口中看到此过程。

  5. 安装所有依赖项后，您可以在“项目工具”窗口中看到一个项目结构（ Alt+1 ） 在左边。在右侧，IntelliJ IDEA自动在编辑器中打开主项目文件。

启动网络服务器

现在我们已经有一个功能良好的Rails应用程序。要查看其运行情况，您需要启动Web服务器。执行以下步骤以执行此操作：

  1. 按Ctrl两次，然后开始输入开发 。

  2. 从列表中选择开发运行配置，然后按Enter 。

  3. IntelliJ IDEA将显示准备运行应用程序的过程。

  4. 复制Web服务器使用的0.0.0.0:3000地址，将其插入浏览器的地址栏中，然后按Enter查看Rails的默认信息页面。

创建一个控制器并查看

现在让我们创建一个新页面。为此，您需要创建一个控制器和一个视图。

  1. 按Ctrl两次并开始键入controller 。在调用的列表中，选择rails g controller并按Enter 。

  2. 在调用的“ 添加新控制器”对话框中，将控制器名称设置为“ 欢迎”，并添加一个名为index的操作 。单击确定 。

  3. IntelliJ IDEA将创建一个控制器，视图和其他几个文件。该过程将显示在“运行工具”窗口中。在此窗口中，单击index.html.erb文件。

  4. 在打开的index.html.erb文件中，删除文件中所有现有的代码，并将其替换为以下单行代码：

    您好，Rails！

  5. 要查看结果，请将鼠标指针悬停在查看代码上，然后从弹出窗口中选择所需的浏览器。

  6. 索引页面将在选定的浏览器中打开。

    要将应用程序的主页设置为index ，请打开config / routes.rb文件，并将以下行添加到Rails.application.routes.draw do ...结束块中：

    root 'welcome#index'

    在Rails入门中了解更多信息。
