创建消息文件

以下内容仅在安装并启用Python插件后才有效。

IntelliJ IDEA将语言环境存储在语言环境目录的与语言相关的子目录中。要创建语言环境，请运行makemessages manage.py实用程序的任务。

创建一个消息文件

  1. 在主菜单上，选择“ 工具”。运行manage.py task ，或按N/A 。

  2. 在manage.py任务窗口中，输入makemessages --locale 然后按Enter 。

    对要创建的每个语言环境重复此步骤。

    如果有标记为本地化的字符串，则IntelliJ IDEA将为每个语言环境生成一个目录和django.po文件：

    如果没有这样的字符串，则仅创建一个空的目录结构。
