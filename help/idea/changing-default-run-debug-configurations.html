更改运行/调试配置的模板

对于特定类型的运行/调试配置，可以为一个或多个参数设置默认值，并将其保存为模板以备将来使用。在这种情况下，下次创建该类型的新配置时，对话框的相应字段将已经包含指定的值。

更改默认值不会影响已经存在的运行/调试配置。

设置运行/调试配置的默认值

  1. 在“运行/调试配置”对话框的左侧窗格中，展开“ 模板”节点。

  2. 在“ 模板”节点下选择所需的配置类型。相应的配置模板出现在右侧窗格中。

  3. 在右窗格中指定所需的参数，然后单击“ 应用”以保存模板。
