模块依赖工具窗口

|

每次您执行“ 分析模块依赖关系”命令时，此工具窗口均可用，并显示在模块设置中定义的模块依赖关系。如果在所选模块中遇到任何循环依赖关系，则会在工具窗口中对其进行特殊标记。在此工具窗口中，您可以更改依赖关系的方向，并对源代码执行更详细的分析。

在这个部分：

  • 工具栏按钮
  • 上下文菜单命令

工具栏按钮

项目

描述

单击此按钮关闭当前选项卡。

单击此按钮打开“ 指定依赖关系分析范围”对话框并分析依赖关系 。

单击此按钮更改依赖关系的方向。

单击此按钮以包括测试依赖项。
