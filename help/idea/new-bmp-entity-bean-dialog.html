新建BMP实体Bean对话框

EJB工具窗口| EJB方面的模块的上下文菜单|跳转到源新增-BMP实体Bean

使用此对话框可以创建Bean管理的持久性（BMP）实体Bean 。

项目

描述

在此文本框中，输入用于生成EJB构成类名称的基本名称。在“ Java EE名称”对话框中配置用于生成EJB名称的规则。

包

在此文本框中，指定Bean所在的软件包的标准路径。如有必要，请单击浏览按钮然后在项目树中选择所需的包。

EJB类

在此文本框中，指定Bean实现类的名称。如有必要，请单击浏览按钮打开“ 选择EJB类”对话框，您可以在其中按名称搜索所需的类或在项目树中选择它。

主键类

在此文本框中，指定用于访问与实体bean关联的数据源的主键的类。如有必要，请单击浏览。 打开“ 选择EJB主键类”对话框，您可以在其中按名称搜索所需的类或在项目树中选择它。

CMP版本

此只读字段显示所使用的容器管理的持久性（CMP）版本。

远端介面

如果要配置Bean的远程客户端视图，请选中此复选框。

选中此复选框将启用“ 主页”和“ 远程”文本框。

家

在此文本框中，指定远程家庭接口的实现类。如有必要，请单击浏览。 打开“ 选择EJB Home Interface”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。

远程

在此文本框中，指定远程接口的实现类。如有必要，请单击浏览。 打开“ 选择EJB远程接口”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。

本地接口

如果要配置Bean的本地客户端视图，请选中此复选框。

选中此复选框将启用“ 本地主页”和“ 本地”文本框。

当地的家

在此文本框中，为本地home接口指定实现类。如有必要，请单击浏览。 打开“ 选择EJB本地主接口”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。

本地

在此文本框中，为本地接口指定实现类。如有必要，请单击浏览。 打开“ 选择EJB本地接口”对话框，您可以在其中按名称搜索所需的接口或在项目树中选择它。
