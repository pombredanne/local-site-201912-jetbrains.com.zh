从Java代码生成Xml模式

本主题描述如何在Java类的基础上生成XML Schema ，这涉及将Java类的成员映射到XML Schema的元素。使用IntelliJ IDEA，可以使用JAXB完成此转换。

使用JAXB从Java类生成XML模式

  1. 通过执行以下操作之一，打开“ 使用Java使用JAXB从Java生成XML模式”对话框：

    • 在“ 项目”工具窗口中，选择所需类的名称，然后选择“ Web服务| Web服务”。使用上下文菜单上的JAXB从Java生成XML模式 。

    • 在活动的编辑器选项卡中打开所需的类，然后选择“ 工具” |“其他”。 JAXB |在主菜单上使用JAXB从Java生成XML模式 。

  2. 指定要在生成的模式中反映的方法参数和返回类型：

    • 若要包含所有类方法，请清除“ 包括”参数和以下方法的返回类型复选框。

    • 要选择要涉及的特定方法，请选择“ 包括参数”和“以下方法的返回类型”复选框，然后选择所需方法旁边的“ 添加到JAXB生成”复选框。
