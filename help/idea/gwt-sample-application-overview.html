GWT示例应用程序概述

当您创建具有GWT支持的模块时，IntelliJ IDEA可以生成示例GWT应用程序。该应用程序可帮助您熟悉GWT应用程序的结构。有关GWT基本原理的详细说明，请参阅http://code.google.com/webtoolkit/overview.html 。

IntelliJ IDEA创建一个满足所有GWT要求的简单HelloWorld应用程序。

如您所见，IntelliJ IDEA在包的根目录中生成了一个GWT模块 MySampleApplication.gwt.xml 。相应的程序包布局（客户端和服务器部分）也会自动生成。

  • client ：包含必需的入口点 （ MySampleApplication类）和远程服务接口。

  • server ：包含远程服务的服务器实现。

    创建远程GWT服务时，IntelliJ IDEA会自动生成同步和异步接口（在client部分），创建服务器实现，并根据需要将其注册到GWT模块xml中 。

用于运行应用程序的HTML主机文件位于单独的Web构面中。

IntelliJ IDEA还会在示例应用程序代码中添加注释，以使注释尽可能清楚。
