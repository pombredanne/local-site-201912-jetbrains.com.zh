将构建文件添加到项目

应该将构建文件添加到项目中，以启用运行构建，过滤目标或定义属性。一个项目可以具有多个构建文件。构建文件在Ant工具窗口中按名称分组，例如在每个构建文件的根元素中指定的名称。 。

生成构建文件后，它将自动添加到项目中。如果您手动创建构建文件或重复使用现有文件，则可能需要将其添加到Ant Build Tool Window 。

  1. 在“ 项目”工具窗口中，右键单击生成的构建文件，然后选择“ 添加为Ant构建文件”以在Ant工具窗口中将其打开。

  2. 在“蚂蚁构建”工具窗口中，单击 。

  3. 在“选择路径”对话框中，导航到所需的build.xml文件，然后单击“ 确定” 。

    构建文件应至少具有一个根元素，该根元素要添加到Ant Build工具窗口中。
