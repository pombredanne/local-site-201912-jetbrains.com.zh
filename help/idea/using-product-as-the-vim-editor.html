使用IntelliJ IDEA作为Vim编辑器

以下内容仅在安装并启用IdeaVim插件后才有效。

IdeaVim插件可在IntelliJ IDEA中模拟Vim，包括普通，插入和可视模式，命令行和Ex模式，Vim正则表达式和配置以及其他功能。

安装IdeaVim插件

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，选择插件 。

  2. 在市场中找到IdeaVim插件，然后点击安装 。

  3. 重新启动IntelliJ IDEA。

重新启动IntelliJ IDEA后，将启用Vim仿真，并且编辑器开始在Vim模式下运行。要禁用它，请取消选择工具|主菜单中的Vim Emulator 。

配置快捷方式

  1. Vim和IntelliJ IDEA都是以键盘为中心的。您在IntelliJ IDEA中的按键映射可能与Vim的按键组合冲突。要解决此问题，请为不同的操作选择所需的快捷方式。

    在里面 设置/首选项对话框Ctrl+Alt+S ，选择编辑器| Vim仿真 。

  2. 找到快捷方式和相应的IDE操作，然后选择使用Vim仿真时要如何处理它：

    • 未定义 ：显示一个弹出通知，提示您重新定义IDE快捷方式或在Vim仿真设置中配置处理程序。

    • IDE ：执行与此快捷方式关联的IDE操作。

    • Vim ：将其作为Vim快捷方式处理。

编辑模式

启用Vim仿真后，在普通模式下 ，光标为一个块：

要更改为插入模式 ，请按i ，光标将变为一行：

在此模式下，您可以键入新代码或更改现有代码。您还可以输入其他Vim模式：例如，按r用于替换模式 。

要返回普通模式，请按Escape 。

Vim配置

使用vimrc文件配置Vim。同样， VimIdea插件使用具有相同语法的ideavimrc文件。根据操作系统，在以下位置之一中创建配置文件：

％HOMEPATH％\ _ ideavimrc
〜/ .ideavimrc
〜/ .ideavimrc

IntelliJ IDEA查找ideavimrc的位置取决于user.home JVM选项，默认情况下设置为用户的主目录。有关更多信息，请参阅配置JVM选项 。

如果您的配置中已有vimrc文件，则可以执行以下任一操作：

  • 将vimrc中的配置包括在source您的ideavimrc文件中的命令：

    源〜/ .vimrc

    这样，您还可以在ideavimrc文件中包含其他命令，这些命令将仅应用于IntelliJ IDEA中的Vim仿真，并且不会影响实际的Vim配置。

  • 如果不想使用Vim配置，可以将.vimrc重命名为.ideavimrc 。

  • 如果您不想与实际的Vim配置相比在IntelliJ IDEA中修改您的Vim仿真，请创建ideavimrc作为vimrc的符号链接：

    ln -s“ $ HOME / .vimrc”“ $ HOME / .ideavimrc”
