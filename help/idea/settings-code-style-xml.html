制表符和缩进

使用制表符

  • 如果选中该复选框，则制表符将用于缩进和代码重新格式化。

  • 如果清除该复选框，则IntelliJ IDEA将使用空格而不是选项卡。

智能标签

压痕由两部分组成。一部分来自嵌套代码块，另一部分由对齐确定。

  • 如果选中此复选框，则嵌套产生的零件将同时包含制表符和空格（如果需要），而由对齐方式定义的零件仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组适合指定制表符大小的空格后，将自动用制表符替换，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中可以容纳的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组的元素之间插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

缩进空行

如果选中此复选框，则IntelliJ IDEA会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，则IntelliJ IDEA会删除制表符和空白行。

右边距

使用这些设置可以指定元素右侧所需的页边空白。如果选择默认选项，则将使用全局设置中右边距的值。

包装打字

使用这些设置设置可以指定如何在指定的右页边距中插入已编辑的文本。您可以选择以下选项之一：

  • 默认值 -在这种情况下，IntelliJ IDEA使用全局设置中指定的“ 键入时自动换行”选项。

  • 是 -在这种情况下，将使用指定的右边距中的值。

  • 否 -在这种情况下，此选项已关闭，并且一行可以超出右边界中指定的数字。

保持换行

选中此复选框可使IntelliJ IDEA荣誉行在编辑器中查看XML文件时中断。

在文本中保留换行符

选中此复选框可在编辑器中查看XML文件时使IntelliJ IDEA荣誉行在属性（例如，冗长的描述）中换行。

保持空白行

在此文本框中，指定重新格式化后要保留的最小顺序空白行数。

包装属性

使用此下拉列表确定应如何包装属性行。可用的选项有：

  • 不换行 -如果选择此选项，则不会对代码应用特殊的换行样式。

  • 如果较长则自动换行-选择此选项可使行超出正确的边距并用适当的缩进进行包裹。

  • 如果时间长，则切碎 -选择此选项可使列表中的元素超出右边距的范围，以使每行一个元素具有适当的缩进。

  • 始终换行 -选择此选项可使列表中的所有元素都换行，以使每行具有适当缩进的一个元素。

自动换行

选中此复选框可以根据代码样式设置将长行换行。

对齐属性

选中此复选框可使连续的行中的属性对齐。

保持空白

选中此复选框后，编辑器将保留标签内的所有空格。缩进和换行符也相同。

空间

在此区域中，定义属性和标签名称的空格用法。

  • 在属性中的“ =”周围 ：选中此复选框可在属性中的“ =”符号周围添加空格。

  • 标记名称后 ：选中此复选框可在标记名称后添加空格。

  • 在空标签中 ：选中此复选框可以在空标签中添加空格。

数据

在此区域中，定义MXML文件中CDATA节周围和内部的空格的用法：
  • 周围的空格 ：从下拉列表中选择周围的空格CDATA将得到治疗。

    • 保留 ：重新格式化后，所有空白都将保留不变。

    • 删除（保留标签） ：周围的所有空格CDATA将被删除，标签将保持在同一行。

    • 新行 ：之前和之后将添加新行CDATA 。

  • 将空格保留在内部 ：如果选中此复选框，则空格将在之后保留CDATA[和之前]] 。

安排

通过此选项卡，您可以定义一组规则，这些规则可以根据您的喜好重新排列代码。

项目

描述

匹配规则

使用此区域可将元素顺序定义为规则列表，其中每个规则都有一组匹配项，例如修饰符或类型。

  • ：使用此按钮添加规则。空规则区域打开。

  • ：使用此按钮从列表中删除规则。

  • ：使用此按钮可以编辑现有规则。要查看此按钮，请导航至要编辑的规则，然后单击该按钮。在打开的弹出窗口中，修改规则字段。

  • ：使用这些按钮可以上下移动所选规则。

空规则

使用此区域可以创建新的匹配规则或编辑现有规则。您可以从以下过滤器中进行选择：

  • 输入 ：使用此过滤器为规则选择类或方法。

    请注意，两次单击类型关键字会否定条件。

  • 名称 ：使用此字段在规则中指定条目名称。此过滤器仅匹配条目名称，例如字段名称，方法名称，类名称等。该过滤器支持正则表达式并使用标准语法 。对全名进行匹配。

  • 命名空间 ：使用此字段在规则中指定命名空间。它使您可以指定控制名称空间属性位置的规则。

  • 顺序 ：使用此列表选择规则的排序顺序。当多个元素使用相同的匹配规则时，此选项很有用。在这种情况下，选择“ 保留顺序”将保持与重新排列前相同的顺序，选择“按名称排序”将按名称对具有相同匹配规则的元素进行排序。

  • 别名 ：此选项显示在“ 规则别名定义”对话框中定义的别名 。您可以删除不需要的内容。

从订单列表中选择按名称订购时，将显示此图标。图标表示此规则中的项目按字母顺序排序。

其他设置

使用此区域设置其他排列选项。通过“ 强制重新排列”列表，可以选择影响“ 重新设置 代码”对话框中“ 重新排列条目”复选框的选项。

您可以从以下选项中选择：

  • 使用当前模式（在“重新格式化代码”对话框中切换） ：在这种情况下，“ 重新排列条目”复选框保持活动状态，您可以在“ 重新格式化代码”对话框中对其进行修改。

  • 始终 ：在这种情况下， 重新排列条目复选框处于选中状态，并且变为只读状态。

  • 永不 ：在这种情况下，“ 重新排列条目”复选框已清除并变为只读。

对Android XML文件使用自定义格式设置

使用此复选框可以为Android XML文件设置自定义格式。如果您需要将Android文件的格式设置与其他XML文件不同，或者需要使用特定的Android格式设置选项，那么这可能会有所帮助。
如果未选中此复选框，则将应用默认的XML文件格式。

AndroidManifest.xml

使用此区域可为AndroidManifest.xml设置以下格式：

  • 包装属性 -使用此下拉列表确定应如何包装属性行。请参阅可用选项 。

  • 在第一个属性之前插入换行符 -选中此复选框可在第一个属性之前插入换行符。

  • 在最后一个属性之后插入换行符 -选中此复选框可在最后一个属性之后插入换行符。

  • 对具有相同名称的标签进行分组-选中此复选框可对具有相同名称的标签进行分组。

布局文件

使用此区域可以为布局文件设置以下格式：

  • 包装属性 -使用此下拉列表确定应如何包装属性行。请参阅可用选项 。

  • 在第一个属性之前插入换行符 -选中此复选框可在第一个属性之前插入换行符。

  • 在最后一个属性之后插入换行符 -选中此复选框可在最后一个属性之后插入换行符。

  • 在标签前插入空白行 -选中此复选框可在标签前插入空白行。

价值资源文件和选择器

使用此区域可以为值资源文件和选择器设置以下格式：

  • 包装属性 -使用此下拉列表确定应如何包装属性行。请参阅可用选项 。

  • 在样式声明周围插入换行符 -选中此复选框可在样式声明周围插入换行符。

其他XML资源文件

使用此区域可以为其他XML资源文件设置Wrap属性 。请参阅可用选项 。

  • 在第一个属性之前插入换行符 -选中此复选框可在第一个属性之前插入换行符。默认情况下选中此选项。

  • 在最后一个属性之后插入换行符 -选中此复选框可在最后一个属性之后插入换行符。
