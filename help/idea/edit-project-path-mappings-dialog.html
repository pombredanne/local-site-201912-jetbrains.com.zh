编辑项目路径映射对话框

单击时将打开对话框在各种情况下（例如，在“ 运行/调试配置：Node.js”对话框中，或者在配置远程解释器时），在“ 路径映射”字段旁边显示。

如果在当前的运行/调试配置中，您使用通过SFTP连接可访问或位于Vagrant实例或Docker容器中的解释器，则将从对应的SFTP部署配置Vagrantfile或Dockerfile中自动检索映射，并在对话框中列出。这些映射是只读的。

  • 要添加自定义映射，请单击并分别在“ 本地路径”和“ 远程路径”字段中指定项目中的路径以及远程运行时环境中的相应路径。手动输入路径或单击然后在打开的对话框中选择相关文件或文件夹。

  • 要删除自定义映射，请在列表中选择它，然后单击 。
