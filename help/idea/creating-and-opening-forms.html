创建和打开表格

GUI设计器使开发人员可以创建具有所需布局和绑定类以及对话框的GUI表单。可以将表单创建为仅表单文件，也可以与UI类一起创建。对话框框架包括一个定义基本组件和方法的类，以及一个GUI窗体，其中的组件绑定到UI类的字段。

如GUI设计器概念中所述 ，IntelliJ IDEA中的GUI表单不是Java类。它们是存储在您的项目中的特殊XML文件，扩展名为.form 。

在创建GUI表单和对话框之前，请选择将在其中存储表单的程序包。您可能需要先创建一个程序包。建议将表单和对话框与实现其功能的类存储在同一包中。

创建一个新的GUI表单或对话框

创建一个新的GUI表单

  1. 在主菜单上，选择“ 文件” |“其他”。新的 。

  2. 在弹出菜单上，选择“ GUI表单” 。

  3. 指定新表单的名称，然后选择布局管理器。

  4. 选中选项Create bound class ，然后指定绑定的类名称。

  5. 单击确定 。

创建一个新对话框

  1. 在主菜单上，选择“ 文件” |“其他”。新的 。

  2. 在弹出菜单上，选择“ 对话框” 。

  3. 指定新对话框的名称。

  4. 指定以下选项：

    • 生成main()创建并显示对话框的方法。

    • 生成OK处理程序。

    • 生成取消处理程序。

  5. 单击确定 。

从文件模板创建表单

您可以将任何表单另存为文件模板，从而保留常用的对话框或控件窗格以供以后重用。有关创建和使用文件模板的详细信息，请参阅以下部分：

  • 文件和代码模板

通过GUI表单创建文件模板

  1. 创建一个GUI表单或在GUI设计器中打开它。

  2. 在主菜单上，选择“ 工具”。将文件另存为模板 。“ 文件模板”对话框显示表单的源代码。新模板的扩展名是.form

  3. 指定模板名称，然后单击确定 。

从文件模板创建新的GUI表单

  1. 在“项目”工具窗口中，右键单击要在其中创建新表单的包。

  2. 在上下文菜单上，选择“ 新建” 。子菜单包含所有已定义文件模板的项目，包括GUI表单。

  3. 在上下文菜单上选择所需的GUI表单模板，并在出现提示时为其命名。

创建快照

快照功能使您可以将通过其他方式创建的GUI表单转换为IntelliJ IDEA GUI设计器表单。您所需要做的就是运行您的应用程序，并将正在运行的应用程序的组件树的一部分另存为GUI设计器表单。从那时起，您可以使用GUI设计器来扩展或修改表单。

您可以在GUI设计器中使用布局和代码，就像处理最初使用GUI设计器创建的窗体一样。

拍摄现有对话框的快照

  1. 打开包含要捕获的对话框的源代码的模块。

  2. 选择“ 应用程序运行配置” ，确保已选中“ 启用捕获表单快照 ”选项，然后运行应用程序 。

  3. 在正在运行的应用程序中，执行任何必要的交互操作以打开要捕获的对话框。

  4. 在“项目”工具窗口中，右键单击目标包，然后选择“ 新建” |“新建”。上下文菜单上的“ 表单快照 ”。如果该应用程序尚未运行，则IntelliJ IDEA会提示您运行它，然后准备该应用程序以进行快照：

    出现“创建表单快照”对话框。

  5. 在IntelliJ IDEA的“创建表单快照”对话框中，选择要包含在快照中的顶级组件，输入表单名称，然后单击“ 创建快照”按钮。用这种方法创建的新表单没有绑定到任何类。您必须自己将其绑定到类。

    如果所选节点不是组件的有效容器，则“ 创建快照”按钮将被禁用。

打开现有的GUI表单

打开表单文件的方式与打开源代码或文本文件进行编辑的方式相同。在适当的导航视图中找到文件，选择它，然后执行以下操作之一：

  • 双击

  • 在上下文菜单上选择“ 跳转到源 ”。

  • 按F4 。

每个表格在单独的选项卡中打开，位于源代码文件的编辑器所占用的相同空间中。您可以同时打开源代码/文本文件和表单文件。
