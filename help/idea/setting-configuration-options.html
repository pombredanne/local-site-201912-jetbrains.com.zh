设置配置选项

配置选项包括VM设置，应传递给程序的参数，工作目录，类路径和SDK。有关这些字段的详细说明，请参阅“ 运行/调试配置”对话框。

定义运行/调试配置的配置选项

  1. 单击“ 编辑运行/调试配置”对话框的“ 配置”选项卡。

  2. 在“ 主要类别”字段中，指定包含以下内容的类别： main()方法。为此，请手动键入完全限定的名称，或单击省略号按钮，然后从“ 选择主要类别”对话框中选择所需的类别。

    在“ 选择主类”对话框中，可以使用以下方式之一找到所需的类：

    • 单击项目选项卡，然后选择带有main()项目树视图中的方法。

    • 单击“ 按名称搜索”选项卡，然后开始输入类名称。键入时，可用类的列表会缩小以匹配您的输入。

    单击确定 ，或按Enter准备好了。

  3. 在“ VM选项”字段中，输入可选的VM参数，例如堆大小，垃圾回收选项，文件编码等。如果VM参数行太长，请单击并在编辑器对话框中键入文本。

  4. 在“ 程序参数”字段中，输入应传递给参数的可选参数列表。 main()通过其参数数组的方法。

  5. 在工作目录字段中，指定应用程序在运行时将使用的当前目录。

  6. 在“ 使用模块的类路径和SDK”字段中，从项目中现有的模块列表中选择所需的模块。
