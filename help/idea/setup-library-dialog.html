设置库对话框

使用此对话框可以设置库 。

  • 如果计算机上已经有必需的文件jar，则可以将这些文件放在库中并使用该新库。为此，选择使用库 ，单击创建 ，然后在打开的对话框中选择必要的文件。（使用Ctrl多项选择的键。）

    （可选）单击“ 配置”以编辑新库。（将打开“ 创建库”对话框 。）

  • 您还可以下载必要的文件，并将这些文件用作库。为此，选择下载 。

    （可选）单击“ 配置”以编辑库设置和内容。（将打开“ 下载选项”对话框 。）
