迁移到EJB 3.0

如果要使用EJB 3.0规范的优点，则可以将各种类型的现有bean从EJB 1.x或2.0迁移到EJB 3.0。IntelliJ IDEA建议应用EJB 3.0样式重构，该重构使Bean的结构符合EJB 3.0规范的要求。

要将bean迁移到EJB 3.0规范

  1. 打开EJB工具窗口 。

  2. 选择要迁移的所需模块或bean，然后在上下文菜单上选择Apply EJB 3.0 Style 。

  3. 在显示模块内所有项目列表的“ 应用EJB 3.0样式”对话框中，选择要迁移的项目：

    如果要转换家庭接口而不是删除家庭接口，请选择“ 保留家庭接口”复选框（如果适用）。

  4. 配置环境访问和XML描述符选项。有关选项的详细说明，请参考“ 应用EJB 3.0样式”对话框 。

  5. 尽管您可以单击“ 重构”并立即执行迁移，但建议您首先单击“ 预览” ，然后在“ 查找工具”窗口中浏览临时的迁移结果。

  6. 在“查找工具”窗口中，单击“ 执行重构” 。
