创建一个新的Android项目

  1. 启动“ 新建项目”向导 。如果当前在IntelliJ IDEA中没有打开任何项目，请在“欢迎”屏幕上单击“ 创建新项目 ”：

    否则，选择文件|新品|从主菜单进行项目 。

  2. 在向导的第一页上，在左窗格中选择“ Android ”。在右窗格中，IntelliJ IDEA会自动创建应用程序的名称，公司域，程序包名称和项目位置：

    您可以根据需要编辑指定的字段。在本例中，我们已将应用程序的默认名称更改为HelloDroid 。
    请注意，程序包名称必须具有以下格式： com.xxx.yyy ，其中xxx通常代表您的公司名称，而yyy是应用程序名称。请注意，您可以在此处使用任何名称，但是建议的模式可以大大降低名称与其他应用程序冲突的风险。

  3. 在向导的下一页上，选择将在其中运行和调试应用程序的目标设备。

  4. 在向导的下一页上，为模块添加一个活动。

  5. 在向导的下一页上，自定义所选活动。单击完成以完成向导。

  6. 您的Android项目将使用预定义的项目结构创建：
