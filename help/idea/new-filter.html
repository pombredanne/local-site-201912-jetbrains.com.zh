新建过滤器对话框

档案|新增|过滤

指定新过滤器的设置。

此页面描述控件。有关如何配置Java EE的信息，请参阅启用Java EE应用程序支持。

项目

描述

名称

过滤器名称的“根部分”。

该名称（及其前缀和后缀）将在 web.xml或filter类中的元素@WebFilter注解。

您可以指定过滤器名称的前缀和后缀（“ 设置”对话框中“ 编辑器” |“代码样式” |“ Java”页面的“ Java EE名称”选项卡上的标记“ 前缀”和“ 后缀”字段）。在这种情况下，前缀和后缀会自动添加到指定名称的前面和后面。

包

应在其中创建新过滤器类的包的名称。

请点击 。然后选择一个现有的程序包或单击创建一个新的程序包。

类

过滤器类的全限定名称。

使用包名称，类名称前缀，过滤器名称和类名称后缀生成默认的类名称。

类名的前缀和后缀在“ 设置”对话框中设置 。（“ 编辑器” |“代码样式” |“ Java”页面的“ Java EE名称”选项卡上的“ 过滤器类 前缀和后缀”字段。）

如果要指定现有的类，请单击然后在打开的对话框中选择类。

创建Java EE 6带注释的类

如果未选中该复选框，则过滤器名称-类映射将添加到web.xml中 。

如果选中此复选框，则新的过滤器类别为@WebFilter -带注释，并且不会对web.xml进行任何更改。

如果项目中没有web.xml ，则默认情况下会选中此复选框，并且您不能取消选择它。
