测试：：单元特别说明

以下仅在安装并启用Ruby插件时有效。

Test :: Unit与Ruby 1.8.x捆绑在一起，通常不需要其他调整。如果您正在使用带有Rails 3.x或带有Rails 2.3.x且不带有Rails的IntelliJ IDEA， test-unit gem，您可以“开箱即用”使用此框架。

但是，在某些情况下，IntelliJ IDEA将无法将测试运行器UI插入测试引擎：

  • Ruby 1.9.x，带有有限版本的Test :: Unit
  • Rails 2.3.x和测试单元gem

Ruby 1.9.x，带有有限版本的Test :: Unit

使用完整版本，该版本通过test-unit宝石。如果缺少此gem，则测试结果将显示在控制台中，并且测试运行器UI仅显示错误消息“未找到测试”。

如果要让测试运行程序用户界面显示测试结果的树状视图，请包括test-unit您的Gemfile中的gem。

Rails 2.3.x和测试单元gem

该问题影响所有基于Test :: Unit的测试框架。要解决此问题，您必须手动启用Test :: Unit自定义报告程序，如下所述。
