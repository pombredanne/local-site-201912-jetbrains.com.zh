使用此对话框创建Session Bean 。

项目

描述

在此文本框中，输入用于生成EJB构成类名称的基本名称。在“ Java EE名称”对话框中配置用于生成EJB名称的规则。

包

在此文本框中，指定Bean所在的软件包的标准路径。如有必要，请单击浏览按钮然后在项目树中选择所需的包。

EJB类

在此文本框中，指定Bean实现类的名称。如有必要，请单击浏览按钮打开“ 选择EJB类”对话框，您可以在其中按名称搜索所需的类或在项目树中选择它。
