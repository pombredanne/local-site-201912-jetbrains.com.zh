运行/调试配置：FlexUnit

使用此对话框可以创建或编辑FlexUnit 运行/调试配置 ，使您可以运行或调试FlexUnit测试。

请注意，此运行/调试配置类型不能用于在移动设备上运行或调试测试。要运行或调试针对移动设备的测试，应使用内置的设备模拟器。

  • 主要设定
  • 启动前选项
  • 工具列

主要设定

项目

描述

名称

使用此字段来编辑运行/调试配置的名称。

分享

选中此复选框可通过版本控制共享运行配置。

如果未选中该复选框，则运行配置设置将存储在.idea / workspace.xml或.iws文件中。

如果选中此复选框，则设置将存储在.idea / runConfigurations或.ipr文件中的单独xml文件中。

请参阅项目 。

仅单实例

如果选中此复选框，则一次仅运行配置的一个实例。

构建配置

选择要使用的构建配置 。

请注意，相应的构建配置应在其依赖项之间具有核心FlexUnit库，请参阅测试ActionScript和Flex应用程序 。

测试

选择测试范围并指定相关设置：
  • 全部包装。 选择此选项可运行特定程序包中的所有测试。在“ 程序包”和“ 方法”字段中分别指定合格的程序包名称。

    使用Shift+Enter在“ 选择软件包”和“ 选择测试方法”对话框中选择所需的软件包。

    要在模块的所有软件包中运行所有测试，请将Package字段保留为空。

  • 班级或套房。 选择此选项可运行测试类或套件。在“ 类”和“ 方法”字段中分别指定测试类的标准名称。

    使用Shift+Enter在“ 选择测试类”和“ 选择测试方法”对话框中选择所需的类。

    要在模块的所有软件包中运行所有测试，请将Package字段保留为空。

  • 方法。 选择此选项可运行测试方法。分别在“ 类”和“ 方法”字段中指定测试类的标准名称和方法名称。

    使用Shift+Enter在“ 选择测试类”和“ 选择测试方法”对话框中选择所需的类和方法。

显示测试日志输出

如果要输出测试日志，请选中此复选框，然后从列表中选择日志记录级别。

请注意，将使用mx.logging包中的标准记录器。

与启动

对于Web构建配置：指定用于运行测试的程序。您可以使用：

  • 系统默认应用程序，即与操作系统中的目标文件类型（HTML或SWF）关联的程序。通常，这是一个Web浏览器。对于SWF文件，这也可能是独立的Flash播放器。

  • Web浏览器。

  • Flash播放器。

要选择所需的程序，请单击在启动方式字段的右侧，然后在启动方式对话框中指定程序：

  • 要选择系统默认应用程序，只需单击系统默认应用程序 。

  • 要选择浏览器，请单击浏览器，然后从列表中选择所需的浏览器。

    此外，您可以访问“ Web浏览器”对话框来调整Web浏览器设置。要打开此对话框，请使用在列表旁边。

  • 要选择Flash Player，请单击Flash Player ，然后单击“确定”。 Flash Player字段旁边。然后，在打开的对话框中指定所需Flash播放器的位置。

将SWF文件放置在本地信任的沙箱中

对于Web构建配置：如果要将测试SWF文件注册为受信任的，请选中此复选框。

受信任的SWF文件可以与任何其他SWF文件进行交互。他们可以从任何地方（远程或本地）加载数据。

从技术上讲，将受信任的SWF文件分配给本地受信任的沙箱 。

启动前选项

指定在开始运行/调试配置之前应执行的任务。指定的任务按照它们在列表中出现的顺序执行。

项目

捷径

描述

Alt+Insert 单击此图标可将任务添加到列表中。选择要添加的任务：

  • 运行外部工具。 选择此选项可运行IntelliJ IDEA外部的应用程序。在打开的对话框中，选择应运行的一个或多个应用程序。如果尚未在IntelliJ IDEA中定义必要的应用程序，请添加其定义。有关更多信息，请参阅外部工具和外部工具 。

  • 使。 选择此选项可以编译项目或模块。在制作模块命令将被执行，如果在运行/调试配置中指定一个特定的模块，所述生成项目命令否则。

    如果在编译过程中发生错误，则IntelliJ IDEA不会尝试启动运行/调试配置。

  • 做，没有错误检查。 与Make选项相同，但是IntelliJ IDEA不管编译结果如何，都将尝试启动运行/调试配置。

  • 建立工件。 选择此选项有一个神器内置或假象。在打开的对话框中，选择应构建的工件。

    另请参阅使用工件 。

  • 运行其他配置。 选择此选项可以执行另一个运行/调试配置。在打开的对话框中，选择要运行的配置。

    仅当您在当前项目中已经具有至少一个运行/调试配置时，此选项才可用。

  • 运行Ant目标。 选择此选项可以运行Ant目标。在打开的对话框中，选择要运行的目标。有关更多信息，请参见Ant 。

  • 生成CoffeeScript源映射。 选择此选项可生成CoffeeScript源的源映射。在打开的对话框中，指定CoffeeScript源文件的位置。

    有关更多信息，请参见CoffeeScript 。

  • 运行Maven目标。 选择此选项可以运行Maven目标。在打开的对话框中，选择要运行的目标。

    有关更多信息，请参见Maven 。

Alt+Delete

单击此图标从列表中删除所选任务。

Enter

单击此图标以编辑所选任务。在打开的对话框中进行必要的更改。

Alt+Up

单击此图标可将所选任务在列表中向上移动一行。

Alt+Down

单击此图标可将所选任务在列表中向下移动一行。

显示此页面

选中此复选框可在实际启动运行/调试配置之前显示运行/调试配置设置。
