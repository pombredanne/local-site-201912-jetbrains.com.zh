仅当您的项目处于Mercurial版本控制下时，此选项卡才可用。

它显示所有补丁，并允许您管理Mercurial Queue 。

您可以将修补程序拖放到队列中，以更改其应用顺序。

使用此命令将选定的修补程序及其上方的所有修补程序应用到队列中。所应用的修补程序将在“ 日志”选项卡中可见，并且所做的更改将被注册在存储库的工作副本中。
