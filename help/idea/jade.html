帕格（玉）模板引擎

IntelliJ IDEA与Pug（Jade）模板引擎集成，该引擎将Pug（Jade）文件转换为HTML。

哈巴狗文件标有 ;翡翠文件标有 。

在你开始前

  • 如管理插件中所述，在“ 插件”页面上安装并启用Pug（ex-Jade）和File Watchers存储库插件 。

安装哈巴狗

  • 在嵌入式终端 （ Alt+F12 ），输入：

    npm install pug

编译哈巴狗

要即时编译代码，您需要将模板引擎配置为IntelliJ IDEA File Watcher 。

创建一个Pug / Jade文件观察器

  1. 打开 设置/首选项对话框Ctrl+Alt+S并转到工具|文件监视程序 。

  2. 请点击或按Alt+Insert然后从列表中选择Pug / Jade预定义模板。您的代码将被翻译成JavaScript，并随生成的源映射一起提供 。

  3. 在程序字段中，指定到的路径pug要么jade包。手动输入路径或单击然后在打开的对话框中选择文件位置。

  4. 按照文件监视程序中所述进行操作。

当您打开文件时，IntelliJ IDEA会检查当前项目中是否有适用的文件监视程序。如果已配置但禁用了这种文件监视程序，则IntelliJ IDEA会显示一个弹出窗口，通知您有关已配置的文件监视程序并建议启用它。

如果在当前项目中配置并启用了适用的文件监视程序，则IntelliJ IDEA会在“ 新建监视程序”对话框中指定的事件后自动启动它。

  • 如果选中了“ 自动保存编辑的文件以触发监视程序”复选框，则在对源代码进行任何更改后都会立即调用文件监视程序。

  • 如果清除了自动保存编辑的文件以触发监视程序复选框，则文件监视程序会在保存时启动（ File | Save All ， Ctrl+S ），或从IntelliJ IDEA移动焦点时（禁用帧时）。

配置语法突出显示

您可以根据自己的喜好和习惯配置Pug / Jade-aware语法高亮显示。

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|配色方案|帕格/玉

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
