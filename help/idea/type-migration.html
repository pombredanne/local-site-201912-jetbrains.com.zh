类型迁移

使用类型迁移重构，您可以自动更改成员类型（例如，从整数更改为字符串），以及在整个项目中更改依赖于数据流的类型条目，例如方法返回类型，局部变量，参数等。

它还允许您在数组和集合之间自动转换变量或方法的返回类型。如果发现任何冲突，则IntelliJ IDEA将显示相应的消息。

也可以从UML类图中进行此重构。

  1. 在编辑器中，突出显示插入符或将其置于您要重构的类型。

  2. 按Ctrl+Shift+F6或在主菜单上，选择“ 重构|类型迁移 。

  3. 在打开的对话框中，指定查找用法的新类型和范围。

  4. 预览并应用更改 。
