使用此部分可以配置HTML文件的格式设置选项。更改这些设置时，更改结果将显示在“ 预览”窗格中。

制表符和缩进

使用制表符

  • 如果选中该复选框，则制表符将用于缩进和代码重新格式化。

  • 如果清除该复选框，则IntelliJ IDEA将使用空格而不是选项卡。

智能标签

压痕由两部分组成。一部分来自嵌套代码块，另一部分由对齐确定。

  • 如果选中此复选框，则嵌套产生的零件将同时包含制表符和空格（如果需要），而由对齐方式定义的零件仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组适合指定制表符大小的空格后，将自动用制表符替换，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中可以容纳的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组的元素之间插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

缩进空行

如果选中此复选框，则IntelliJ IDEA会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，则IntelliJ IDEA会删除制表符和空白行。

右边距

使用这些设置可以指定元素右侧所需的页边空白。如果选择默认选项，则将使用全局设置中右边距的值。

包装打字

使用这些设置设置可以指定如何在指定的右页边距中插入已编辑的文本。您可以选择以下选项之一：

  • 默认值 -在这种情况下，IntelliJ IDEA使用全局设置中指定的“ 键入时自动换行”选项。

  • 是 -在这种情况下，将使用指定的右边距中的值。

  • 否 -在这种情况下，此选项已关闭，并且一行可以超出右边界中指定的数字。

保持换行

选中此复选框可使IntelliJ IDEA荣誉行在编辑器中查看HTML文件时中断。

在文本中保留换行符

选中此复选框可在编辑器中查看HTML文件时使IntelliJ IDEA荣誉行在属性（例如，冗长的描述）中换行。

保持空白行

在此文本框中，指定重新格式化后要保留的最小顺序空白行数。

包装属性

使用此下拉列表确定应如何包装属性行。可用的选项有：

  • 不换行 -如果选择此选项，则不会对代码应用特殊的换行样式。

  • 如果较长则自动换行-选择此选项可使行超出正确的边距并用适当的缩进进行包裹。

  • 如果时间长，则切碎 -选择此选项可使列表中的元素超出右边距的范围，以使每行一个元素具有适当的缩进。

  • 始终换行 -选择此选项可使列表中的所有元素都换行，以使每行具有适当缩进的一个元素。

自动换行

选中此复选框可以根据代码样式设置将长行换行。

对齐属性

选中此复选框可使连续的行中的属性对齐。

对齐文字

选中此复选框可使IntelliJ IDEA对齐标签中占几行的文本。

保持空白

选择此复选框可禁止使用制表符替换实际的空格。

空间

在此区域中，定义属性和标记名称的空格用法。

  • 在属性中的“ =”周围 -选中此复选框可在属性中的“ =”符号周围添加空格。

  • 标签名称后 -选中此复选框可在标签名称后添加空格。

  • 在空标签中 -选中此复选框可以在空标签中添加空格。

在前面插入新行

该显示字段显示标签列表，在该列表之前应插入新行。使用按钮在字段旁边或按Shift+Enter打开“ 在标签前插入新行”对话框，您可以在其中编辑标签列表。

之前删除新行

此显示字段显示标签列表，在该列表之前应删除折线。使用按钮在字段旁边或按Shift+Enter打开“在标签前删除换行符”对话框，您可以在其中编辑标签列表。

不要缩进

此显示字段显示了不缩进其子标签的标签列表。使用按钮在字段旁边或按Shift+Enter打开不缩进子级对话框，您可以在其中编辑标签列表。

或者标签尺寸大于

在此文本框中，在不从其缩进其子行的行中指定标签的最小长度。

内联元素

此显示字段显示在源代码中与其他标签在同一行中显示的标签的列表。如果从列表中删除了标记，则在将标记添加到源代码后，编辑器会自动将其移动到新行。使用按钮在字段旁边或按Shift+Enter打开“ 内联元素”对话框，您可以在其中编辑标签列表。

在内部保留空白

这显示字段显示的标签列表中您想要编辑保存空格，而不做任何更改。使用按钮在字段旁边或按Shift+Enter打开“在内部保留空白”对话框，您可以在其中编辑标签列表。

内联内容不要中断

此显示字段显示如果内联标签内容将不包装的标签列表。使用按钮在字段旁边或按Shift+Enter打开“ 如果仅内联内容则不要换行”对话框，您可以在其中编辑标签列表。

生成的引号

选择引号的样式（双引号，单引号或无引号），以便在键入时自动插入HTML属性周围= 。

当使用JavaScript或PHP动态插入HTML且您要始终对JavaScript或PHP字符串使用双引号对以及对HTML使用单引号对以防止出现问题时（例如在复制和粘贴时），这一点很重要。

强制格式

如果选中此复选框，则在代码重新格式化后 ，先前生成的引号将被替换（例如，双引号和单引号）。

安排

在此选项卡中，定义一组规则以根据您的喜好重新排列HTML代码。

匹配规则

使用此区域可将元素顺序定义为规则列表，其中每个规则都有一组匹配项，例如修饰符或类型。

  • -使用此按钮添加规则。空规则区域打开。

  • -使用此按钮从列表中删除规则。

  • -使用此按钮可以编辑现有规则。要查看此按钮，请导航至要编辑的规则，然后单击该按钮。在打开的弹出窗口中，修改规则字段。

  • -使用这些按钮向上或向下移动所选规则。

空规则

使用此区域可以创建新的匹配规则或编辑现有规则。您可以从以下过滤器中进行选择：

  • 类型 -使用此过滤器为规则选择标签或属性。
    请注意，两次单击类型关键字会否定条件。

  • 名称 -使用此字段在规则中指定条目名称。该过滤器仅匹配条目名称，例如标记名称或属性名称。该过滤器支持正则表达式并使用标准语法 。对全名进行匹配。

  • 命名空间 -使用此字段可以在规则中指定命名空间。它使您可以指定控制名称空间属性位置的规则。

  • 顺序 -使用此下拉列表选择规则的排序顺序。当多个元素使用相同的匹配规则时，此选项很有用。在这种情况下，选择“ 保留顺序”将保持与重新排列前相同的顺序，选择“按名称排序”将按名称对具有相同匹配规则的元素进行排序。

从订单列表中选择按名称订购时，将显示此图标。图标表示此规则中的项目按字母顺序排序。
