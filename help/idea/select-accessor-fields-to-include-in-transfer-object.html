描述

使用此按钮可使方法按升序或降序排序。

显示课程

如果bean引用了祖先类，则单击此按钮将显示这些类及其成员。否则，该按钮不可用。

此功能取决于EJB规范。

展开全部

单击此按钮以展开所有节点。

全部收缩

单击此按钮可折叠所有节点。

有问题的实体Bean中的成员列表

从列表中选择所需的方法。使用Ctrl和Shift多选键。

传输对象类名称

在此字段中，为与特定模式匹配的传输对象类指定名称。接受IntelliJ IDEA建议的名称或根据需要进行更改。

在界面中生成getter / setter

选中这些复选框可为所选字段生成访问器方法。您可以接受访问器方法的建议名称，也可以根据需要更改它们。

复制JavaDoc

选中此复选框可以对生成的传输对象添加JavaDoc注释（如果有）。
