描述

默认布局

从列表中，选择默认情况下编辑器的显示方式：编辑器和预览窗格，仅编辑器或仅预览窗格。

自动滚动预览

选中此复选框可自动从源代码中的光标位置滚动到预览中的相应位置。
如果选中此复选框，则按钮在工具栏中被按下。

灰阶

选择此复选框可使用灰度渲染。

未安装PlantUML框架

如果未安装PlantUML框架，则会出现此警告。如果需要在Markdown预览中显示PlantUML图，请单击Install 。

在代码围栏中禁用自动语言注入

选中此复选框可禁用受防护的代码块中的代码注入。

隐藏代码围栏中的错误

选中此复选框可禁用受防护的代码块中的错误突出显示。

自定义CSS

从URI加载

在此处指定所需.css文件的路径。请点击在文件系统中查找文件。

添加CSS规则

键入用于渲染预览的特定CSS。例如，您可以在此处指定背景颜色和字体粗细。
