细胞

单元格的基本操作

您可以在表格单元格中修改值，并在适当的情况下上传文件。

行动

捷径

描述

编辑

 F2

编辑单元格值。或者，选择一个单元格并开始输入新值。要确认您的更改，请按Enter 。

编辑最大化

Shift+Enter

展开单元格进行编辑。要确认您的更改，请按Ctrl+Enter 。

新增一行

Ctrl+Enter

在常规编辑模式下添加新行F2 。

还原所选

Ctrl+Alt+Z

恢复先前的单元格值。

最大化

Ctrl+Shift+Alt+M

展开选定的单元格。

在最大化的单元格中添加新行

Enter

在最大化编辑模式下添加新行（ Shift+Enter ）。要确认您的更改，请按Ctrl+Enter 。

设定默认值

Ctrl+Alt+D

将当前单元格值设置为默认值。有关如何设置默认值的更多信息，请参阅设置单元格的默认值 。

设置NULL值

Ctrl+Alt+N

将当前单元格值设置为NULL值。有关如何设置默认值的更多信息，请参阅设置单元格的默认值 。

载入档案

将文件上传到字段中。右键单击一个单元格，然后选择“ 加载文件” 。

编辑为

F2要么Shift+Enter

将单元格中的值编辑为SQL，HTML，XML或任何其他受支持的语言的片段。

价值完成

Ctrl+Space

打开建议列表。要调用值完成功能，请开始编辑单元格，然后按Ctrl+Space 。该列表包含当前列中与您的输入匹配的值。

将具有相同内容的单元格添加到选择中

Alt+J

将具有相同内容的最近的单元格添加到选择中。要从选择中删除单元格，请按Shift+Alt+J 。

将选择范围从单元格扩展到表格

Ctrl+W

将选择范围从单元格扩展到表格。

  • 按下一次：选择将扩展到一列。

  • 按两次：选择从列切换到行。

  • 按三下：所选内容将扩展到整个表格。

比较细胞

Ctrl+Shift+D

比较一张表中两个或三个单元格的内容。最多选择三个单元格，然后按Ctrl+Shift+D 。或者，右键单击所选内容，然后选择比较单元格 。

设置单元格的默认值

创建新行并且未为某些单元格指定值时，这些单元格将使用各自的默认值填充Ctrl+Alt+D或NULL（ Ctrl+Alt+N ）值。

设置单元格的默认值时，可以从上下文菜单中使用相应的操作（ Set DEFAULT和Set NULL ）。

  1. 在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中，展开一个数据源树。

  2. 右键单击表，然后选择“修改表” Ctrl+F6 。

  3. 在列选项卡上，双击要编辑的列。

  4. 在默认字段中，在列中指定单元格的默认值。要允许使用NULL值，请清除“ 不为null”复选框。

使用快速文档视图

快速文档视图提供了有关所选单元格中值的详细信息。例如，如果一个单元格包含长文本，您将看到整个文本。如果单元格包含图像，您将看到图像的预览。此外，您可以看到当前记录中引用的记录以及引用当前记录的记录。

宾语

文件检视

长文字

如果一个单元格包含长文本，您将看到整个文本。

图片

如果单元格包含图像，您将看到图像的预览。

记录

记录的快速文档视图显示当前记录中引用的记录以及引用当前记录的记录。

您可以切换到转置视图。在转置视图中，行和列互换。

要切换到转置视图，请单击“ 转置视图” 。有关表中的转置视图的更多信息，请参见转置表 。

§

同时编辑多个单元格值

  1. 选择您要编辑的单元格范围。

  2. 开始输入新值。所做的更改将应用于所有选定的单元格。

  3. 要确认值，请按Enter 。要取消编辑，请按Escape 。

将LOB保存在文件中

您可以将单元格中的二进制大对象 （BLOB或LOB）保存到文件中。

  1. 右键单击包含LOB的单元格，然后选择Save LOB 。

  2. 在保存对话框中，指定文件的名称和位置，然后单击确定 。
