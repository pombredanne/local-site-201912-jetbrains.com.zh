IntelliJ IDEA 2019.3帮助

Lightbend Project Starter入门

IntelliJ IDEA使您可以使用Lightbend Project Starter模板。

建立专案

在创建项目之前，请确保已在IntelliJ IDEA中下载并启用了Scala插件。

  1. 如果当前在IntelliJ IDEA中没有打开任何项目，请在“欢迎”屏幕上单击“ 创建新项目 ”。否则，选择文件|新品|项目 。

    结果，“ 新建项目”向导打开。

  2. 在左侧窗格中，选择Scala 。

  3. 在右侧窗格中，选择Lightbend Project Starter 。

    单击下一步 。

  4. 在向导的下一页上，指定项目的信息，选择要打开的模板应用程序（模板描述也可用），然后单击Finish 。

    IntelliJ IDEA将使用选定的模板应用程序创建一个基于sbt的项目。
