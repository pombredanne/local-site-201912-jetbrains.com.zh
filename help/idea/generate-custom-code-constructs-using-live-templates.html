使用实时模板生成自定义代码构造

IntelliJ IDEA为许多常见的代码构造提供了许多预定义的实时模板 。您还可以定义自定义模板，以涵盖特定于您的工作流程的用例。

插入实时模板

  1. 将插入符号放置在您希望模板扩展的位置。

  2. 键入模板缩写，然后按调用键（通常是Tab默认）。或者，在“ 代码”菜单上，单击“ 插入实时模板” Ctrl+J打开建议列表并选择必要的模板。

  3. 如果所选模板需要用户输入，则突出显示相应的字段。输入必要的值，然后按Enter要么Tab完成输入并移至下一个输入字段。完成所有输入字段后，插入符号将移至构造的末尾（或移至$END$标记（如果标记是在模板代码中定义的），则编辑器将返回常规操作模式。

您还可以使用环绕模板包装代码片段。

用实时模板包围代码块

  1. 选择要包含的代码段。

  2. 在代码菜单上，单击环绕带有实时模板 Ctrl+Alt+J打开建议列表并选择必要的模板。

使用实时模板智能完成函数调用中的参数

仅Ultimate Edition支持此功能。

仅当安装并启用PHP插件时，以下内容才有效。

在PHP上下文中，您可以使用 “自动”实时模板，为传递给函数，方法或类构造函数的参数提供完成列表。

要启用参数的智能完成：

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，依次单击“ 编辑器” ，“ 常规 ”和“ 智能键” 。

  2. 选中“ PHP”组中的“ 启用智能功能参数完成”复选框。

要将智能完成模板用于功能：

  • 键入params关键字作为函数调用中的第一个参数。

    IntelliJ IDEA显示一个实时模板，该模板中的参数将使用函数声明中定义的变量名称自动完成。

    要移至下一个参数，请按Enter要么Tab 。要移至上一个参数，请按Shift+Tab 。

    完成列表按以下顺序包含来自本地范围的变量：在附近定义的具有相同类型和相似名称的变量。您始终可以通过按切换到通常的完成模式Ctrl+Space或只键入列表中未包含的任何内容。具有相似名称的变量将自动插入。
