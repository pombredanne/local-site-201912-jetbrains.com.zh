从Eclipse导入项目

  1. 启动IntelliJ IDEA。

    如果“欢迎”屏幕打开，请单击“ 导入项目” 。

    否则，从主菜单中选择File |新品|现有的项目 。

  2. 在打开的对话框中，选择源，库和其他资产所在的目录，然后单击“ 打开” 。

  3. 选择从外部模型导入项目| Eclipse并单击下一步 。

  4. 配置项目：

    • 选择Eclipse项目目录 ：Eclipse工作空间的路径，该工作空间包含要导入的项目。

    • 在.classpath文件附近创建模块文件 ：在每个Eclipse项目的相应项目目录下创建一个新的IntelliJ IDEA模块。指定格式的IntelliJ IDEA项目将在Eclipse工作区或Eclipse项目目录的根目录中创建。

    • 将项目和模块文件保存在 ：如果要在不修改原始Eclipse项目的情况下将项目导入IntelliJ IDEA，请指定IDE将在其中创建.iml文件（ 模块文件 ）的文件夹以及带有配置文件的.idea目录。

      如果保留默认路径，则IntelliJ IDEA在Eclipse项目文件夹中创建.iml文件和.idea目录。

    • 项目文件格式 ：建议您使用基于目录的格式 。

    • 将创建的IntelliJ IDEA模块链接到Eclipse项目文件 ：自动保持Eclipse项目和IntelliJ IDEA模块同步。

    • 检测测试源 ：指定存储测试源的文件夹列表。有关通配符语法，请参见“ 编译器”部分。

    单击下一步 。

  5. 查看在选定的Eclipse工作区中检测到的项目，然后选择要导入IntelliJ IDEA的项目。每个Eclipse项目都将转换为IntelliJ IDEA模块。

    单击下一步 。

  6. 选择要在项目中使用的代码样式配置，然后单击“ 下一步” 。

  7. 指定您要使用的SDK 。

    如果在IntelliJ IDEA中已经定义了必要的SDK，请从左侧列表中选择它。否则，请点击并添加一个新的SDK。

  8. 点击完成 。
