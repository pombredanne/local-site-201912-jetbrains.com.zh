添加框架和技术

对于开发特定于框架的应用程序，IntelliJ IDEA具有多个方面。构面包含库，依赖项和技术，它们为您提供用于配置框架特定设置的其他UI元素。

某些框架仅在IntelliJ IDEA Ultimate中可用。请参考比较矩阵 ，以确保您的IntelliJ IDEA版本支持必要的框架。

IntelliJ IDEA可以识别特定框架典型的文件或目录，并为您添加必需的方面。一旦检测到该构面并添加了构面，IntelliJ IDEA就会通知您缺少的配置，并建议采取必要的措施。

如果未自动检测到构面，则可以手动添加。您可以为一个模块添加多个构面。

手动将构面添加到模块

  1. 在“项目”工具窗口中，右键单击要向其添加构面的模块，然后选择“ 添加框架支持” 。

  2. 从列表中选择必要的框架。根据您的选择，系统可能会提示您配置其他设置（例如，配置库）。

  3. 应用更改并关闭对话框。

禁用框架自动检测

默认情况下，所有支持的框架都启用自动检测。您可以完全禁用框架自动检测，也可以从自动检测中排除单个框架。

  1. 请点击在工具栏上Ctrl+Shift+Alt+S ，然后选择Facets 。

  2. 选择检测 ，然后单击Alt+Insert 。

  3. 从要排除的框架列表中，选择必要的选项。您只能在一个目录或整个项目中禁用对特定框架的自动检测。该列表还允许您禁用特定目录中所有框架的自动检测。

  4. 如果要禁用整个项目中所有框架的自动检测，请取消选中“ 启用框架检测”复选框。

在此对话框中，您也可以删除框架或添加新框架。
