以下仅在安装并启用Ruby插件时有效。

IntelliJ IDEA可帮助您编写在连接shoulda宝石。

如果遵守以下条件之一，则IntelliJ IDEA提供特定于Shoulda的代码：

  • 使用捆绑器。

  • 对于Rails 2。*，shoulda在environment.rb中指定

  • 对于Ruby项目， require 'shoulda'是强制性的。

请参阅Shoulda安装页面以了解有关在Rails应用程序中使用Shoulda的信息。

在项目中启用Shoulda支持后，您可以：

  • 对上下文使用应具备shoda意识的代码完成，并应阻止：

  • 使用Test :: Unit run / debug配置或用户定义的Rake任务运行和调试Shoulda测试。在这两种情况下，IntelliJ IDEA都将在测试运行器UI中运行测试。这样，您可以启动测试用例，应该阻止的目录或所有测试用例。

  • 在“文件结构”视图中查看Shoulda测试：

  • 在Rails应用程序，请查看的测试::单位/早该节点早该测试Rails视图 ：

要启用Shoulda支持，请遵循以下常规步骤

  1. 安装shoulda宝石到您的项目。

  2. 使用Shoulda测试将目录标记为test根目录。
