启用挂毯项目的创建

  1. 打开项目工具窗口。

  2. 通过执行以下任一操作，切换到“ 项目”或“ 包” 视图 ：

    • 从标题栏左侧的列表中选择所需的视图。

    • 如果视图由选项卡表示，请单击相应的选项卡。

  3. 选择应用程序根软件包，然后选择“ 新建|在上下文菜单上打包 。

  4. 在打开的“ 新包装”对话框中，根据要在其下创建的项目类型指定专用包装的名称。

    • 对于Tapestry 页面 的网页 。

    • 组件 Tapestry的组件 。

    • 混入用于挂毯成分混入 。

创建挂毯页面

Tapestry页面通常由实现页面外观的HTML模板和实现页面功能和行为的Java类组成。HTML文件具有与Java文件相同的名称，并且具有.tml扩展名。IntelliJ IDEA生成两个存根， .tml文件包含

和部分。

  1. 打开项目工具窗口。

  2. 通过从标题栏左侧的列表中选择“ Tapestry”项或单击“ Tapestry”选项卡，切换到“ Tapestry”视图。

  3. 右键单击pages节点，然后选择New |挂毯|上下文菜单上的页面 。

  4. 在打开的“ 新建挂毯页面”对话框中，指定页面名称和父文件夹以存储生成的类和模板源。

    IntelliJ IDEA会记住上一个选定的目录，并在您下次创建页面时自动在相应的下拉列表中显示它们。

  5. 自定义页面生成，如有必要：

    • 如果您已经有一个具有指定名称的页面，并且希望IntelliJ IDEA覆盖它，请选中“ 替换现有文件”复选框。

    • 要仅生成Java类存根，请选中“不创建模板”复选框。

创建一个组件

Tapestry组件通常由实现页面外观的HTML模板和实现此项目的行为和功能的Java类组成。HTML文件具有与Java文件相同的名称，并且具有.tml扩展名。IntelliJ IDEA生成两个存根， .tml文件包含一个

部分。

  1. 打开项目工具窗口。

  2. 通过从标题栏左侧的列表中选择“ Tapestry”项或单击“ Tapestry”选项卡，切换到“ Tapestry”视图。

  3. 右键单击components节点，然后选择New |挂毯|上下文菜单上的组件 。

  4. 在打开的“ 新建Tapestry组件”对话框中，指定组件名称和父文件夹以存储生成的类和模板源。

    IntelliJ IDEA会记住最后选择的目录，并在下次创建组件时自动在相应的下拉列表中显示它们。

  5. 自定义组件生成，如有必要：

    • 如果您已经具有指定名称的组件，并且希望IntelliJ IDEA覆盖它，请选中“ 替换现有文件”复选框。

    • 要仅生成Java类存根，请选中“不创建模板”复选框。

创建一个混入

  1. 打开项目工具窗口。

  2. 通过从标题栏左侧的列表中选择“ Tapestry”项或单击“ Tapestry”选项卡，切换到“ Tapestry”视图。

  3. 右键单击mixins节点，然后选择“ 新建|挂毯|混入的上下文菜单上。

  4. 在打开的“ 新建Tapestry混合”对话框中，指定混合名称和父文件夹以存储生成的类。

    IntelliJ IDEA会记住上一个选定的目录，并在下次创建mixin时自动将其显示在下拉列表中。

  5. 自定义mixin生成，如有必要：

    • 如果已经具有指定名称的mixin，并且希望IntelliJ IDEA覆盖它，请选中“ 替换现有文件”复选框。
