创建和编辑运行/调试配置

在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，可用的运行/调试配置显示在“运行”区域的“运行/调试配置”选择器中：

IntelliJ IDEA提供“ 运行/调试配置”对话框作为处理运行/调试配置的工具：创建配置文件或更改默认配置文件。

IntelliJ IDEA建议使用以下方法创建运行/调试配置：

  • 在默认值的基础上手动创建运行配置 ，或使用“ 运行/调试配置”对话框来创建。

  • 保存临时运行配置 。

  • 在模块创建期间指定目标应用程序服务器。IntelliJ IDEA将生成相应类型的运行/调试配置。

创建运行/调试配置

  1. 打开“ 运行/调试配置”对话框：

    • 选择运行|从主菜单编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按Shift+Alt+F10 ，然后按0或从弹出窗口中选择配置，然后按F4 。

  2. 在“ 运行/调试配置”对话框中，单击在工具栏上或按Alt+Insert 。该列表显示了默认的运行/调试配置。选择所需的配置类型。

    右侧窗格中显示的字段显示所选配置类型的默认设置。

    要更改默认运行/调试配置的设置，请展开“ 模板”节点，选择所需的配置类型，然后根据需要对其进行修改。有关详细信息，请参见更改运行/调试配置的模板 。

  3. 对于新的运行/调试配置：

    • 在名称字段中指定其名称。该名称将显示在可用的运行/调试配置列表中。

    • 指定是否要让IntelliJ IDEA检查具有相同运行/调试配置的实例的执行状态。如果要确保当前仅执行运行/调试配置的一个实例，请选中仅单个实例复选框。在这种情况下，当您尝试启动运行/调试配置时，当一个相同类型的实例仍在运行时，将显示一个确认对话框。

      如果在确认对话框中单击“ 确定” ，则将停止运行器的第一个实例，而将其替换。

      如果未选中此复选框，则可以根据需要启动任意数量的运行器实例。结果，每个跑步者将在“ 运行工具”窗口的自己的选项卡中启动 。

    要将现有配置用作模板，请单击“ 复制”按钮创建其副本在工具栏上，然后根据需要进行更改。

  4. 在启动前部分中，定义您是否要编译已修改的源并运行Ant或Maven脚本。

  5. 在“ 配置”选项卡中， 指定包含以下内容的类 ： main()方法，VM选项，程序参数，工作目录和其他特定于配置的设置。

  6. 在“ 日志”选项卡中，指定用于控制运行或调试应用程序时生成的输出日志的选项。

    特别是，指定IntelliJ IDEA是否将标准输出和标准错误输出显示到控制台。

  7. 对于各种服务器，设置部署和启动/连接选项。

  8. 对于应用程序和测试，单击“ 代码覆盖率”选项卡 （例如， 在应用程序运行/调试配置中 ），并指定用于定义代码覆盖率度量以用于测试目的的选项。

  9. 根据配置类型指定其他参数。有关运行/调试配置参数的说明，请参阅“ 运行/调试配置对话框”部分。

  10. 应用更改并关闭对话框。

编辑现有的运行/调试配置

  1. 请执行以下任一操作：

    • 从主菜单中，选择运行|编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按Shift+Alt+F10 ，然后按0以显示“ 编辑配置”对话框，或从弹出窗口中选择配置，然后按F4 。

  2. 在相应的运行/调试配置对话框中，根据需要更改参数。

如果已经在IntelliJ IDEA的早期版本中创建了一个项目，则当该项目已经在同一窗口中打开时，其运行/调试配置可能会丢失。

为避免丢失运行/调试配置，建议您以最新版本打开一次这样的项目，并且仅在将其添加到另一个项目之后再打开。

启用服务工具窗口

您可以在专用工具窗口中一次管理多个运行/调试配置。例如，您可以启动，暂停和停止几个应用程序，跟踪它们的状态，并检查特定于应用程序的详细信息。

  1. 从主菜单中，选择运行|编辑配置或使用工具栏上的选择器。

  2. 在“ 运行/调试配置”对话框中，从左侧列表中选择“ 模板 ”。

  3. 在右侧窗格中，展开“ 服务”部分中的“ 可用配置” 。

  4. 请点击在“ 服务”中的“ 可用配置”部分中，选择所需的运行配置类型。

    您可以一一添加或删除多种配置类型。

    请注意，工具窗口将仅显示已为其创建一个或多个配置的配置类型。

    因此，如果添加项目中不存在任何配置的配置类型，则在创建该类型的配置之前，该类型将不会显示在工具窗口中。

  5. 应用更改并关闭对话框。具有所选配置的“ 服务”工具窗口将打开。

要显示或隐藏“服务”工具窗口，请选择“ 查看” |“隐藏” 。工具窗口|主菜单中的服务 。
