定义运行时属性

使用“ 构建文件属性”对话框的“ 属性”选项卡在运行时将属性传递给构建脚本。指定的值等于在-D命令行启动器的选项。

除了纯值之外，您还可以使用在运行时评估的宏。当您必须将特定路径和其他各种信息传递给构建脚本时，此类宏很有用。宏是用美元符号包围的字符串。“构建文件属性”对话框提供了可供选择的构建文件使用的宏的完整列表，并带有预览，这些预览显示了如何在运行时评估这些宏。

定义运行时属性

  1. 打开“构建文件属性”对话框 。

  2. 在属性选项卡中，单击添加 。

  3. 在“ 名称”列中，键入属性名称。

  4. 在“ 值”列中，键入所需的值。如果使用宏作为属性值，请键入所需宏的名称。如果您不知道名字，请单击 ，从“宏”对话框中选择所需的宏，然后单击“ 确定” 。

  5. 使用“ 添加”和“ 删除”按钮组成属性的完整列表。
