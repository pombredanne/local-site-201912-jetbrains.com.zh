使用Hibernate控制台

使用Hibernate控制台编写和运行HQL查询，分析查询结果以及执行其他相关任务。

  • 先决条件
  • 打开Hibernate控制台
  • 组成HQL查询
  • 导航到类或字段的声明
  • 运行查询
  • 运行参数化查询
  • 运行自动存储的查询
  • 终止查询执行
  • 生成SQL语句和DDL SQL脚本
  • 重新排序栏
  • 隐藏和显示列
  • 恢复初始表视图
  • 使用“结构”视图对数据进行排序以及隐藏和显示列
  • 关闭控制台

另请参见Hibernate Console工具窗口 。

先决条件

为了使Hibernate控制台正常运行，您应该：

  • 将会话工厂与相应的数据源相关联 ，请参阅将持久性单元和会话工厂与数据源相关联 。

  • 如果您使用的自定义实体类/数据库表名称映射未完全反映在您的代码和/或配置文件中，请指定相应的NamingStrategy实现，请参阅将会话工厂与NamingStrategy实现类关联（休眠） 。

打开Hibernate控制台

  1. 打开“ 持久性”工具窗口（例如，“ 视图” |“工具”窗口|“持久性” ）。

  2. 展开“ Hibernate facet”节点。

  3. 选择要为其打开控制台的会话工厂或该会话工厂内的任何节点。

  4. 请执行以下任一操作：

    • 请点击在标题栏上。

    • 从上下文菜单中选择控制台 。

    • 按Ctrl+Shift+F10 。

数据库控制台的输入窗格将作为单独的编辑器选项卡打开。在输入窗格中，您可以编写SQL语句。

执行语句时，“ 服务”工具窗口将打开。服务工具窗口显示可用的连接会话，“ 输出”和“ 结果”选项卡。有关“ 服务”工具窗口的更多信息，请参阅“ 服务工具”窗口 。

如果执行的语句检索数据（例如， SELECT ），结果将显示在“ 结果”标签中，该标签的标题为合格的表名。有关为结果选项卡创建自定义标题的更多信息，请参阅对带有结果的选项卡使用自定义标题 。

在执行参数化语句之前，您会看到“ 参数”窗口，可以在其中指定参数值。

修改控制台设置

  • 打开控制台，然后单击“ 打开数据库设置”图标（ ）。

组成HQL查询

在输入窗格中编写查询时，请使用自动完成功能并突出显示HQL关键字以及对象和属性名称。

另请参见导航至类或字段的声明 。

导航到类或字段的声明

您

组成查询时，查看输入窗格中引用的对象或属性的类或字段的声明有时很有用。要导航到相应的声明，请执行以下任一操作：

  • 将光标放在感兴趣的对象或属性的名称内。然后使用Ctrl+B 。（或者，您可以从主菜单使用“ 导航” |“声明” 。）

  • 按住Ctrl键，然后指向感兴趣的名称。当文本变成超链接时，单击超链接。

结果，必要的源文件在编辑器中打开，并将光标放在相应类的声明或相应字段的getter方法内。

运行查询

要运行当前查询，请执行以下任一操作：

  • 请点击在工具栏上。

  • 按Ctrl+Enter 。

运行参数化查询

如果语句中有参数，则必须在执行语句之前指定参数的值。

要执行参数化的语句，请单击“ 执行”按钮在工具栏上，然后在第二列中输入值。或者，要打开“ 参数”对话框，请单击“ 查看参数”按钮 。

运行自动存储的查询

IntelliJ IDEA存储已运行的所有语句的历史记录。

要打开“ 历史记录”对话框，请单击或按Ctrl+Alt+E 。

  • 要过滤信息，请开始输入搜索查询。

  • 要将查询从“ 历史记录”对话框粘贴到控制台，请在“ 历史记录”对话框的左窗格中双击查询。

  • 要从历史记录中删除记录，请选择记录，然后按Delete 。

终止查询执行

要终止当前一条或多条语句的执行，请单击“ 断开连接”按钮或按Ctrl+F2 。

生成SQL语句和DDL SQL脚本

您可以为您的HQL查询生成SQL语句，并为会话工厂生成DDL SQL脚本：

  • 要生成与当前查询等效的SQL，请执行以下一项操作：

    • 请点击在工具栏上。

    • 按Ctrl+Shift+Enter 。

  • 生成DDL SQL语句CREATE TABLE ， ALTER TABLE和DROP TABLE对于与相应会话工厂关联的所有对象（类），请执行以下操作之一：

    • 请点击在工具栏上，然后选择Generate DDL 。

    • 按Ctrl+Shift+Alt+Enter 。

生成的SQL语句显示在输出窗格中。

将标签固定在查询结果上

如果使用一个和同一选项卡来显示查询结果，并且得到想要保留的结果，则可以将选项卡固定到工具窗口。

  • 右键点击标签，然后选择固定活动标签 。

  • 请点击在工具栏上。

在行的子集之间导航

根据将页面大小限制为参数的值，结果集可能分为几页。例如，您将“ 限制页面大小为”参数设置为100，但是查询返回了200行。您将有两行行，每页上有100行。要在页面之间导航，请使用以下控件：

  • ：导航到结果集的第一页。

  • ：导航到结果集的上一页。（ Ctrl+Alt+Up ）

  • ：导航到结果集的下一页。（ Ctrl+Alt+Down ）

  • ：导航到结果集的最后一页。

设置结果集中的行数

默认情况下，发出查询时，返回的行数限制为500。引入此限制是为了避免过载（例如，当SELECT语句返回一百万行时）。

要更改此限制，请打开设置设置Ctrl+Alt+S并导航到工具|数据库|数据视图 。在限制页面大小为字段中，指定一个新数字。要禁用该限制，请清除“ 限制页面大小为”复选框。

另外，您可以在编辑器中单击“ 更改页面大小”列表，然后选择一个预定义值。

转到指定的行

要导航到具有指定编号的行，请右键单击表格，然后选择“ 转到|行 Ctrl+G 。在转到行对话框中，指定行号，然后单击确定 。

排序数据

要对列中的表数据进行排序，请单击标题行中的单元格。

您可以启用“ 通过ORDER BY排序” 选项，以启用对相应DBMS的数据排序。

另外，您可以在“ 结构”工具窗口（“ 视图” |“工具”窗口|“结构” ）中按列对数据进行排序。在“ 结构”工具窗口中，右键单击列名，然后从“ 排序”菜单中选择一种排序方法。

重新排序栏

要重新排序列，请对标题行中的相应单元格进行拖放。

隐藏和显示列

要隐藏列，请右键单击相应的标题单元格，然后选择“ 隐藏列” 。

要显示隐藏的列：

  1. 请执行以下任一操作：

    • 右键单击标题行中的任何单元格，然后选择“ 列列表” 。

    • 按Ctrl+F12 。

    在出现的列表中，隐藏的列的名称被删除掉。

  2. 选择（突出显示）感兴趣的列名称，然后按Space 。

  3. 按Enter要么Escape关闭列表。

另请参见还原初始表视图和使用结构视图对数据进行排序以及隐藏和显示列 。

恢复初始表视图

请点击重新排序或隐藏列或对数据排序后，可以使用工具栏上的还原初始表视图。结果，数据通常变得不排序，列按最初定义的顺序出现，并显示所有列。

使用“结构”视图对数据进行排序以及隐藏和显示列

使用“结果”窗格时，表结构视图可用作相应的弹出窗口。

结构视图显示所有列的列表，并允许您对数据进行排序以及隐藏和显示列。

要打开结构弹出窗口，请执行以下操作之一：

  • 右键单击表标题行中的单元格，然后选择“ 列列表” 。

  • 按Ctrl+F12 。

在弹出窗口中，选择感兴趣的列，然后执行以下操作之一：

  • 要按此列按升序对数据进行排序，请按Shift+Alt+Up 。

  • 要按降序对数据进行排序，请按Shift+Alt+Down 。

  • 要取消按此列排序，请按Ctrl+Shift+Alt+Backspace 。

  • 要隐藏列（或显示隐藏的列），请按Space 。隐藏的列的名称被删除。

用于对表格数据进行排序的快捷方式（ Shift+Alt+Up ， Shift+Alt+Down和Ctrl+Shift+Alt+Backspace ）可以在“结果”窗格中使用，而无需打开结构视图。

另请参见， 要对列中的表数据进行排序，请单击标题行中的单元格。 ， 隐藏和显示列以及恢复初始表格视图 。

将表格导出到文件

当您将表数据复制到剪贴板或将其保存在文件中时，数据将转换为可用的输出格式之一。作为输出格式，您可以选择SQL INSERT要么UPDATE语句， TSV或CSV ，HTML表或JSON数据。

打开一个表或结果集，单击“ 转储数据”图标（ ），然后选择“ 到文件” 。

将表格导出到剪贴板

在工具栏上，单击“ 转储数据”图标（ ），然后选择“ 到剪贴板” 。

配置数据输出格式和选项

您可以配置如何使用预定义的输出格式，自定义格式和脚本将提取的数据转换为文本。要配置提取数据的输出，请右键单击表中的任何单元格，然后指向Data Extractor 。

在此菜单中，您可以选择输出格式（例如，作为SQL INSERT语句，HTML表或CSV格式）或将数据转换为特定格式的脚本。以下附加选项也可用：

跳过计算列（SQL）
启用以排除具有计算值的列（影响SQL输出格式）。
跳过生成的列（SQL）
启用以排除具有自动增量值的列（影响SQL输出格式）。
添加表定义（SQL）
启用以包括CREATE TABLE语句（影响SQL输出格式）。

此外，您可以配置以下参数：

  • 配置CSV格式。 打开“ CSV格式”对话框，可用于管理定界符分隔的值格式（例如CSV或TSV）。

  • 转到脚本目录。 用脚本打开目录，该脚本将表数据转换为各种输出格式。

将LOB保存在文件中

您可以将单元格中的二进制大对象 （BLOB或LOB）保存到文件中。

  1. 右键单击包含LOB的单元格，然后选择Save LOB 。

  2. 在保存对话框中，指定文件的名称和位置，然后单击确定 。

重新加载表格视图的数据

如果要将在编辑器中看到的数据与数据库内容同步，则需要为表视图重新加载数据。或者，当您想在更改页面大小限制设置后应用它。

要刷新表视图，请考虑以下操作：

  • 单击从磁盘重新加载全部图标在工具栏上。

  • 右键单击该表，然后从上下文菜单中选择“ 重新加载页面 ”。

  • 按Ctrl+F5 。

查看生成结果集的查询

要查看用于生成结果表的查询 ，请单击工具栏上的“ 查看查询 ”。如有必要，您可以选择查询文本并将其复制到剪贴板Ctrl+C 。

要关闭显示查询的窗格，请按Escape 。

关闭控制台

要关闭控制台，请执行以下任一操作：

  • 请点击在工具栏上。

  • 按Ctrl+Shift+F4 。
