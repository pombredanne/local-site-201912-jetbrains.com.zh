浏览Subversion存储库

在检出文件之前，您可以浏览Subversion存储库的内容。Subversion存储库浏览器使您可以添加或放弃存储库位置，查看文件和文件夹的历史记录，检出文件和文件夹，导航至源代码，浏览更改，创建分支或标签，等等。

即使您的项目中未启用Subversion，也始终可以浏览Subversion存储库的内容。您只需要一个有效的用户帐户。

  1. 选择VCS |浏览VCS信息库|从主菜单浏览Subversion存储库 。

  2. SVN存储库工具窗口将打开。

  3. 如果到目前为止尚未指定存储库，请单击在工具栏中，然后在打开的“ 新建存储库位置”对话框中指定存储库URL。

  4. 浏览存储库。如果需要验证，请在首次尝试扩展顶部节点时打开的“ 需要验证”对话框中输入您的凭据。
