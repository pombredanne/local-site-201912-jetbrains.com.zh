创建事件和事件处理程序类

IntelliJ IDEA可以为GWT事件类和GWT事件处理程序接口生成存根。

GWT事件类和事件处理程序接口只能在客户端部分中创建。

创建GWT事件和事件处理程序

  1. 在“ 项目”视图中 ，右键单击客户端程序包。

  2. 从上下文菜单中，选择“ 新建| Google Web工具包| GWT事件和处理程序类 。

  3. 在打开的“ 创建GWT事件和处理程序类”对话框中，指定事件类的名称，事件处理程序接口以及供调用者覆盖的抽象“回调”方法。

    实际上，您不需要手动指定所有三个名称。在“ 事件类名称”文本框中键入事件类名称后 ，IntelliJ IDEA会自动完成处理程序接口的名称和“回调”方法，并分别在“ 处理程序类名称”和“ 处理方法名称”文本框中显示建议。这样可以确保生成的存根将根据GWT Google API库生成。

  4. 当您单击OK时 ，IntelliJ IDEA会为事件类和事件处理程序界面生成存根，并在编辑器中将其打开。根据需要填充生成的存根。
