IntelliJ IDEA 2019.3帮助

运行，调试和测试Groovy

您可以像创建任何其他项目一样创建，导入，测试和运行Groovy应用程序。

在开始使用Groovy之前，请确保已在IntelliJ IDEA中启用了Groovy插件。

创建一个Groovy项目

  1. 在项目向导中 ，选择Groovy 。

  2. 指定以下设置：

    • 项目SDK-指定您的项目SDK。

    • Groovy库 -指定Groovy SDK。 IntelliJ IDEA需要标准的Groovy SDK布局，该布局随http://groovy-lang.org/download.html上的官方发行版一起提供。下载SDK，将其解压缩到任何目录中，然后将此目录指定为库主目录。

    • 其他库和框架 -如果需要，请指定Groovy支持的其他库和框架。

    • Java EE版本 -选择适当的Java EE版本。

  3. 单击下一步 。

  4. 指定您的项目信息，然后单击完成 。

要导入Groovy项目，请遵循“ 导入项目”部分中的步骤。

为了构建Groovy项目，IntelliJ IDEA允许您使用Gradle ， Maven或IntelliJ IDEA自己的生成器 。

检查项目中的Groovy版本

您可以检查IntelliJ IDEA在项目中使用的Groovy SDK版本。

  1. 从主菜单中选择文件|项目结构 Ctrl+Shift+Alt+S 。

  2. 在“ 项目结构”对话框中的“ 平台设置”下 ，选择“ 全局库” 。

    如果需要检查模块中使用的Groovy SDK版本，请选择Modules ，模块的名称，然后单击Dependencies选项卡。

更改项目中现有的Groovy SDK版本

  1. 在项目结构中 Ctrl+Shift+Alt+S对话框，删除Ctrl+Y 全局库和模块依赖项（“ 模块”页面）中的现有Groovy SDK版本。

  2. 删除Groovy SDK后，IntelliJ IDEA将显示一个弹出窗口，建议添加Groovy SDK。

    单击链接并设置一个新的SDK。

  3. 如果由于某种原因IntelliJ IDEA不显示弹出窗口，请在“ 项目”工具窗口中，右键单击项目的名称，然后在上下文菜单中，选择“ 添加框架支持” 。

  4. 在打开的对话框中，从技术列表中选择Groovy并添加一个新的Groovy库 。

    IntelliJ IDEA创建一个全局级别的库。

生成Groovy文档

您可以为构成Groovy语言的Groovy和Java类生成API文档。

  1. 在主菜单中，选择工具|生成GroovyDoc 。

  2. 在打开的对话框中，指定文档的输出目录和其他可用选项 ，然后单击“ 开始” 。

运行Groovy应用程序

输入代码后，可以通过IntelliJ IDEA运行它，或使用交互式Groovy控制台进行快速代码评估。

  1. 在编辑器中打开您的应用程序。

  2. 按Shift+F10执行应用程序。或者，在编辑器的左装订线中，单击图标，然后选择“运行名称” 。

  3. 在“ 运行工具”窗口中查看结果。

要运行Groovy脚本，请从编辑器的上下文菜单中，选择“运行'name'”。 Ctrl+Shift+F10 。

有关部署Groovy应用程序的信息，请参阅使用应用程序服务器 。

调试Groovy应用程序

  1. 在编辑器中打开您的Groovy应用程序。

  2. 在左侧装订线中，为要调试的代码行设置断点。调试器了解Groovy语法，并且可以根据需要在断点上评估表达式。

    有关断点的更多信息，请参见“ 断点”部分。

  3. 如果需要，可以访问“ 运行/调试配置”对话框（“运行” |“编辑配置”）并调整设置，但是通常默认设置足以成功启动并完成调试会话。

  4. 按Shift+F9 。或者，在主工具栏上，单击图标以启动调试过程。

  5. 在“ 调试工具”窗口中评估结果。

    有关如何使用“ 调试工具”窗口中的选项的信息，请参阅“ 调试工具窗口参考” 。

在调试模式下，您可以使用“ 评估表达式”工具评估任何表达式 ，方法是按Alt+F8 。该工具以与编辑器相同的方式提供代码完成功能，因此可以轻松输入任何表达式。

有时，您可能想进入一个特定的方法，而不是第一个将被调用的方法。在这种情况下，使用智能步骤到按Shift+F7选择一种特定的方法。

有关调试的更多详细信息，请参阅“ 调试”部分。

测试Groovy应用程序

您可以使用JUnit 4和JUnit 5测试框架来测试Groovy应用程序。

  1. 在编辑器中打开一个要为其创建测试的类，然后将光标放在包含类声明的行中。

  2. 按Ctrl+Shift+T然后选择创建新测试 。

  3. 在打开的对话框中，指定测试设置，然后单击“ 确定” 。

  4. 在编辑器中打开测试，添加代码并按Ctrl+Shift+F10或右键单击测试类，然后从上下文菜单中选择Run'test name' 。

  5. IntelliJ IDEA自动为测试创建运行/调试配置，但是如果要编辑配置中的设置，请单击“运行” |“调试”。在主菜单上编辑配置 。

  6. 在右侧的“运行/调试配置”对话框中，指定测试套件的设置，然后单击“ 确定” 。该配置具有标准选项，您可以在“ 执行测试”部分中找到更多详细信息。

  7. 在主工具栏上，单击图标以运行测试。

  8. 在“ 运行”工具窗口中评估结果。

您可以在Groovy测试和测试主题之间导航 （ 导航|测试/测试主题 ）。如果尚不存在测试类，则IntelliJ IDEA建议创建一个。
