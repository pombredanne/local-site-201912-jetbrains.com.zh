重构代码

重构是在不创建新功能的情况下改进源代码的过程。重构可帮助您保持代码的牢固， 干燥和易于维护。

调用重构

  1. 选择一个要重构的项目。您可以在“ 项目”工具窗口中选择文件/文件夹，也可以在编辑器中选择表达式/符号。

  2. 按Ctrl+Shift+Alt+T打开可以选择的重构列表。

    或者，您可以使用键盘快捷键进行特定的重构。

如果您需要撤消重构，请按Ctrl+Z 。

预览重构更改

对于某些重构，IntelliJ IDEA允许您在应用更改之前预览更改。

  1. 要查看可能的更改，请在“ 重构预览”对话框中单击“ 预览 ”。

  2. 检查“ 查找工具”窗口中将要进行的更改。您可以排除Delete或删除Ctrl+X您认为不必要的更改。

  3. 单击“ 执行重构”以继续更改。

解决冲突

当IntelliJ IDEA的重构遇到问题时，它会打开一个对话框，其中列出了冲突列表及其简短说明。

  • 单击继续以忽略该问题，然后继续在“ 查找”工具窗口中打开预览。

  • 单击“在视图中显示冲突”以在“ 查找”工具窗口中打开冲突条目，然后进一步处理它们。

    例如，您可以尝试通过按以下操作从重构中排除条目： Delete或单击“ 取消”并返回到编辑器。

配置重构设置

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，选择编辑器|一般

  2. 在“ 常规”页面上的“ 重构”部分中，调整重构选项，然后单击“ 确定” 。

您还可以在“ 编辑器” |“调整”中调整重构意图。用心 。
