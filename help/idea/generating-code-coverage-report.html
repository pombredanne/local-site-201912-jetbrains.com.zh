将覆盖率数据保存到文件

在运行至少一项覆盖率测试之后，代码覆盖率报告将变为可用。

生成报告

  1. 从主菜单中，选择“ 分析|生成覆盖率报告 ，或单击在“ 覆盖率”工具窗口中 。

    请注意，仅当您在覆盖范围内运行必要的测试后，此选项才可用。

  2. 在“ 生成覆盖率报告”对话框中，指定将存储生成的报告的目录，并选择选中“ 在浏览器中打开生成的HTML”复选框。

  3. 点击保存 。

请注意，如果要为在同一窗口中打开的多个项目之一保存代码覆盖率报告，则检查建议的目标位置非常重要，因为IntelliJ IDEA会建议以前使用的位置。
