IDE脚本控制台

IDE脚本控制台可用于编写简单的脚本，以自动执行IntelliJ IDEA功能并提取各种信息。通过访问IntelliJ平台API ，您可以将其视为插件的轻量级替代，后者可以添加或修改IDE的某些行为。

默认情况下，它支持用Kotlin，JavaScript和Groovy编写的脚本。但是，您可以使用任何与JSR 223兼容的脚本语言，例如Python，Ruby，Clojure等。

本要点提供了一些信息和示例。

打开IDE脚本控制台

  1. 在主菜单中，选择工具| IDE脚本控制台 。

  2. 选择所需的脚本语言。

这将在编辑器中打开“ IDE脚本控制台”选项卡，您可以在其中键入代码并使用以下命令执行代码Ctrl+Enter 。

例如，使用以下代码创建Kotlin脚本：

导入com.intellij.openapi.ui。Messages.showInfoMessage变量总和：Long = 0L val arr =“ 35907 77134 453661 175096 23673 29350” .split（“”）arr.forEach {sum + = it.length} showInfoMessage（（sum.toFloat（）/ arr.size）。 toString（），“测试”）

使用鼠标指针或全部选择Ctrl+A然后按Ctrl+Enter运行它。您应该在“ 运行”工具窗口中看到执行每一行的结果，并在弹出的对话框中看到标题为test的对话框，其中包含数组中已编号元素的平均长度。单击确定将其关闭。

如果未选择任何内容，则仅执行带有插入符号的当前行。

脚本存储在consoles / ide下的Configuration目录中 。您还可以在Scratches and Consoles / IntelliJ IDEA Consoles下的Project工具窗口中看到它们。如果在该目录中添加一个名为.profile的文件，然后添加相应语言的名称（例如.profile.groovy ），它将与您运行的任何脚本一起执行。使用配置文件来定义脚本的功能。
