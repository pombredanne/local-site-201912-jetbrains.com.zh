上载2019.1帮助

内存不足异常

除非另有说明，否则此处列出的所有命令都是Windows命令。如果您使用的是Linux或Mac OS服务器，只需更换.bat与.sh 。对于macOS或Linux，请确保以非root用户身份启动Upsource。

  • 上游服务会消耗所有内存

  • 新提交未建立索引

  • 一些（所有）修订卡在“索引编制中”状态

  • Web UI非常慢

解

  1. 停止上载： \bin\upsource.bat stop

  2. 运行以下命令：

    \apps\upsource-frontend\bin\upsource-frontend.bat configure -J-Xmx6000m

  3. 启动上游： \bin\upsource.bat start
