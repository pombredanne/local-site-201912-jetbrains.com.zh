Upsource Cassandra properties

The following JVM options are configured for the Upsource Cassandra process by default. You can change their default values with the configure command or by manually editing the configuration file cassandra.jvmoptions.

See a corresponding step-by-step instruction for a ZIP or Docker installation.

-ea -XX:+HeapDumpOnOutOfMemoryError -XX:HeapDumpPath= -XX:ErrorFile=hs_err_pid%%p.log -Xbootclasspath/a:lib/jamm/jamm-0.3.0.jar -javaagent:lib/jamm/jamm-0.3.0.jar -d64 -Xmx3000 -XX:MaxMetaspaceSize=150m
