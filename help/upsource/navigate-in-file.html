在文件内和文件内导航

不同的视图使您可以选择代码中的各种符号，并通过执行以下描述的操作来深入检查代码库。

在所有三个视图中可用的动作：

  • 显示元素的文档
  • 查找元素的用法
  • 查看用法差异

并排差异视图和文件视图中可用的操作：

  • 显示元素的文档
  • 查找元素的用法
  • 查看用法差异
  • 转到元素声明
  • 查看元素的层次结构
  • 突出显示元素的用法

显示元素的文档

单击元素以显示其文档：

查找元素的用法

  1. 单击元素。

  2. 从弹出窗口中选择查找用法 。该元素的所有用法将在单独的工具窗口中列出：

    您只需单击即可导航至每个列表。

查看用法差异

  1. 单击元素。

  2. 从弹出窗口中选择“ 查看用法差异 ”。将列出文件中该元素的所有用法，并指示在两个修订之间添加了哪些元素，哪些已删除，哪些未更改：

转到元素声明

  1. 单击元素。

  2. 从弹出窗口中选择“ 转到声明 ”。您将被带到该元素的声明。

查看元素的层次结构

  1. 单击元素。

  2. 从弹出窗口中选择“ 查看层次结构 ”。该元素的所有祖先和继承者将在单独的工具窗口中列出：

    您只需单击即可导航至每个列表。

突出显示元素的用法

  1. 单击元素。

  2. 从弹出窗口中选择“ 突出显示文件中的用法” 。该元素在文件中的所有出现都将突出显示：
