Creating a project — Advanced

  • Do not import the following files: Specify patterns for the files that should be excluded from this project.

  • Exclude files from review: Specify patterns for the files that should be excluded from reviews.

  • Default branch: Optionally specify the branch which you want to set as default. Setting the default branch will improve displaying the commits graph, rendering README files, comparing branches, etc.

  • Default charset: Specify the supported charset for text files. Refer to the list of supported encodings.

  • Allow authors to close reviews: When enabled, authors will be allowed to close reviews regardless of the Close Review permission.

  • Allow authors to delete reviews: When enabled, authors will be allowed to delete reviews regardless of the Delete Review permission.

  • Limit who can resolve a discussion: When enabled, only the person that started a discussion will be allowed to resolve it.

  • Default review deadline: A review deadline serves as a reminder to the reviewer to complete the review. The default will be applied to any review when it's created but can be adjusted by a review participant.

    Note that the deadline calendar only counts weekdays and skips weekends.
