File history

This chart shows you how a project evolves over time. Each dot represents a file (hover over it to see its name), horizontal axis shows the date of the latest commit, while vertical axis shows the total number of commits. Knowing which files are edited the most can help you better understand your project. Only the 1000 most edited files are shown.

For large projects, allow some time for the chart to render.
