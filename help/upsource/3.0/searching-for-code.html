Searching for code

Upsource lets you search the source code of any revision for files and text occurrences. In Java, JavaScript, PHP, or Kotlin projects, you can find not just files but even specific classes or methods, both in the source code and in connected libraries.

Open the revision you want to search. The search is available in:

  • Revision view
  • File view

Type your query into the Search in revision search on top:

You can use lowerCamelHumps. For example, to find NonShortCircuitBooleanInspection, type nscbi.
