Web hooks

  • VCS webhooks can be enabled on the VCS side. Use them to get no-delay real-time commit updates from your VCS and to speed up your system by reducing the overall payload.
  • Upsource webhooks can be set up on the Upsource side for each project. Use them to notify external services about events that occur in an Upsource project.
