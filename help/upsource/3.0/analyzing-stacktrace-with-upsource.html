Analyzing stacktrace with Upsource

Compared to IntelliJ IDEA Stacktrace analyzer, Upsource analyzer allows you to find the revision in which the exception you want to analyze has occurred.

To analyze an external stack trace or thread dump

  1. From the main menu, select Analyze | Analyze stacktrace with Upsource.
  2. Paste the external stack trace or thread dump into the text area of the displayed dialog window.
  3. Click OK. The stacktrace is displayed in the Run tool window. File names are highlighted as links.
  4. Click on a file name to open its version in the Editor. You can opt to open a current version of the file. In this case the analyzer will try to find the corresponding source code and display it in the Editor.
