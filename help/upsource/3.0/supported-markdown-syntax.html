Supported Markdown syntax

Upsource supports Markdown in review and discussion comments. Here is the syntax to follow:

  • Emphasis
  • Headings
  • Links
  • HorizontalRules
  • Lists
  • Tables
  • Reply quotation
  • Code
  • Images
