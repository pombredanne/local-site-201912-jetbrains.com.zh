Creating a GitHub project

In the Connect to GitHub form specify the following settings:

Beside GitHub, Upsource supports all Git hosting solutions - be it self-hosting or in the cloud (e.g. BitBucket, Gitlab). A Git project can be configured and created in the Create project page.

  • Repository URL: enter your GitHub repository address (i.g. https://github.com/some-name/some-repo)

  • Authentication method: provide credentials for accessing your repository, or choose Anonymous if authentication is not required.

To set up synchronization with Upsource

  1. Tick Import pull requests as branches and Synchronize comments and pull requests

  2. Create and enable GitHub Authentication module

  3. Acquire an OAuth 2.0 token or enter a personal GitHub token

When you are finished filling out the form, click Create project.
