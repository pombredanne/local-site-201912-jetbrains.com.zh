Revision view with inline diff

In the revision view, you can see the list of changed files, as well as an inline diff view showing modified sections of each changed file in the revision, all on a single page.

To open the revision view from the project home page, click the revision ID or commit message. The files included in the revision will be displayed along with an inline diff under each file:

At the top of each diff pane you can see how many code fragments were added and deleted:

The deleted fragments are highlighted in red while the added ones are in green.

The unchanged blocks of code are hidden by default. You can reveal them by clicking on the cut lines:

From the revision view you can go to a side-by-side diff view or view the entire file - just click a corresponding button.
