Code review statistics

The code review report shows you how many revisions were covered with code reviews, or how many open and closed reviews you have in your project.

The graph at the top displays the relative distribution of revisions and reviews over time for a project. You can filter out the graph by selecting closed or open reviews only.

The tables below let you see and compare the number of open and closed reviews as well as reviewers and authors contribution.
