What is supported for each VCS

This table compares different aspects of support provided by Upsource for version control systems.

VCS Authentication protocols support Branches/streams support Tags support
GIT Anonymous, Username/Password, SSH key (only via ssh:// protocol) All repository branches are included All tags are reported
Perforce Username/Password No No
Subversion (SVN) Anonymous, Username/Password, SSH key (only via svn+ssh:// protocol) Branches are supported via branches specification rules No
Mercurial Implicit (via Mercurial config on server, Anonymous, Username/Password, SSH key (via host configuration only) All open repository branches are included All tags are reported
Externals/submodules are not supported in all VCS types.
