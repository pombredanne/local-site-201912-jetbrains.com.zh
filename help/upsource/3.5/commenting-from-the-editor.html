Commenting from the editor

You can comment on code fragments and participate in discussions right from your IDE editor.

To leave a code selection comment:

  1. Select the fragment you want to discuss and choose Upsource | Leave a Comment from the context menu (or press Ctrl+Alt+Slash

  2. Type in your message into the text field and press OK:

    Your comment will be attached to the latest revision. However, you can optionally attach it to an existing review by ticking the corresponding checkbox and specifying the review:

    If the code you are commenting on or the file altogether has never been part of the review you're specifying - you might get an error, and your comment will not be posted.
