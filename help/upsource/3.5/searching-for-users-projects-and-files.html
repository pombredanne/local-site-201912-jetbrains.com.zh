Searching for users, projects, and files

The omni-search box in the top-right corner lets you find users and projects in your Upsource installation. You can also find files across projects, including deleted files.

The search box is context-dependent: if youâ€™re looking at a particular revision or a review, searches will be limited to that revision (review), but if you perform a search from the entry page, Upsource will present the most relevant results from all projects and branches.

To find a project:

  1. On the Upsource entry page, start entering the project name into the search box.

  2. Select the desired project from the drop-down suggestions list, under Projects.

To find a user:

  1. To search across all projects, go to the entry page. To search within a specific project, go to that project's home page.

  2. Type the user name into the search box.

  3. Select the user from the drop-down suggestions list, under Users.

To find a file:

  1. You can

    • search across all projects
    • limit your search to a specific project
    • limit your search to a specific revision

    by going to a corresponding page

  2. Type the file name into the search box.

  3. Select the file from the drop-down suggestions list.
