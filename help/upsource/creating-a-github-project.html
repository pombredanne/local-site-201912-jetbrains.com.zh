创建一个GitHub项目

除了GitHub之外，Upsource还支持所有Git托管解决方案-无论是自托管还是在云中（例如，BitBucket，Gitlab）。可以在“ 创建项目”页面中配置和创建Git项目。

在“ 连接到GitHub”表单中，指定以下设置：

  • 存储库URL：输入您的GitHub存储库地址（例如https://github.com/some-name/some-repo ）

  • 身份验证方法：提供用于访问您的存储库的凭据，如果不需要身份验证，则选择“ 匿名” 。

与Upsource设置同步

  1. 勾选将提取请求作为分支导入，并同步注释和提取请求 。

  2. 创建并启用GitHub身份验证模块 。

  3. 获取OAuth 2.0令牌或输入个人GitHub令牌 。

填写完表格后，点击创建项目 。
