Resetting a failed project

If an Upsource project has failed to initialize for some reason, in most cases resetting it will solve the problem.

To reset a project:

  1. Click the settings icon on the top right of the screen. The Administration view will open.

  2. A failed project can be recognised by an error message and the Reset project button displayed next to it:

  3. Click Reset project.
