上载2019.1帮助

本地SMTP服务器不适用于Docker安装

无法从上游Docker容器访问本地SMTP。

解

通过添加重新配置Upsource容器以使用主机网络--network host和删除端口映射-p :XXXX 。

  1. 停止上载：docker exec stop

  2. 输入以下命令：

    docker run -it -v ：/ opt / upsource / data \ -v ：/ opt / upsource / conf \ -v ：/ opt / upsource / logs \ -v ：/ opt / upsource / backups \ jetbrains / upsource： \配置--listen-port XXXX \ --base-url http://upsource.mydomain.com:XXXX \ --network host
