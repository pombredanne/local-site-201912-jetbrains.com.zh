云端安装

使用市场上可用的一种云计算服务，可以轻松地在云中安装和运行Upsource。

作为示例，我们将概述Amazon EC2的Upsource安装要求，因为它是当前最受欢迎的云解决方案之一。为了便于服务设置和无故障的上溯源安装，我们也建议您阅读EC2文档 。

  1. 您可以选择任何平台：Linux，Mac或Windows。

  2. 您选择的EC2实例类型应为t2.large或更高，具体取决于存储库的大小。

  3. 配置EC2安全组时 ，添加具有以下参数的自定义规则：

    • 允许的协议：TCP

    • 允许的端口：XXXX（指定默认的上游端口（对于Windows为80，对于Linux和Mac为8080），除非已使用该端口，或者由于其他原因而要使用其他端口。在这种情况下，您还必须为Upsource进行更改（第4步）。

  4. 安装Upsource，然后在启动之前，通过运行以下命令来配置基本URL（最终用户访问您的Upsource安装的URL）和端口：

    • \ bin \ upsource.bat配置--listen-port XXXX --base-url http://upsource.mydomain.com:XXXX

    哪里：

    • --listen-port应与您在步骤3中为EC2规则指定的端口匹配。如果您在步骤3中指定了默认的上游端口，请跳过。

    • --base-url应与您的EC2实例的公共DNS名称匹配。

    这是Windows命令。对于Linux或macOS，只需更换.bat与.sh

    \directory_name应该读为“打开控制台并将目录更改为directory_name在上游主目录下”。

  5. 您已准备好运行Upsource 。
