以只读模式运行上游

在只读模式下，上游用户无法发表评论和编辑评论，因此存储在Cassandra数据库中的评论数据保持不变。尽管如此，代码存储库仍受到监视和更新，并且可以由用户查看。

您可以使用只读模式来最大程度地减少服务器的停机时间，例如，将您的Upsource安装移动到另一台服务器，升级到新版本或执行一些维护任务时。

启用只读模式

  1. 以管理员身份登录到Upsource。

  2. 点击设置图标在屏幕的右上方。“ 管理”视图将打开。

  3. 单击启用只读模式链接。

  4. 如果要通知用户，为什么Upsource处于只读模式，请在文本字段中输入您的解释。登录时，将向用户显示带有您的消息的横幅。

  5. 点击启用 。
