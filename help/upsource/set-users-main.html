以下文档说明了Upsource用户管理背后的概念。

用户管理在Upsource中的工作方式

虽然Upsource在设置过程中为您创建了默认的管理员帐户，但您仍然需要团队中开发人员的用户帐户才能访问和使用Upsource。

除了创建用户帐户，您还需要授予这些帐户权限，以使相应的开发人员能够参与审核过程。

为了使这些任务更有效率，Upsource利用了Hub （服务），该服务为所有JetBrains服务和与其连接的应用程序提供了一个用于用户管理的单入口点。使用集线器，用户只需要一个用户帐户即可访问所有连接的服务，管理员可以从一个位置处理这些服务和用户帐户。
