Searching Upsource

Upsource's powerful search function lets you find virtually everything throughout the installation: projects, users, files and text. Within a specific project, you can filter out the revision list to focus on most relevant entries, using various criteria such as specific author, branch, commit ID and message, file name and time periods.

You can also search for code and text fragments in files, reviews, commit messages, and comments.
