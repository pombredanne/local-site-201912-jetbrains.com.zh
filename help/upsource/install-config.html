安装和配置

有关系统要求和其他重要说明，请查看先决条件 。

上游是您在服务器上安装并运行的本地应用程序。该安装包有两个跨平台发行版：

  • ZIP归档文件 ：一种主流的选项，具有适合大多数情况的简单安装过程。如果不确定要选择哪种发行版，请选择此发行版。有关说明，请参考ZIP安装 。

  • Docker映像 ：允许在隔离的环境（一个Docker容器）中安装和运行Upsource。请参阅： Docker安装以获取指示。

两种发行版均可用于在云中安装和运行Upsource。请参阅云端安装 。

还可以将上游设置为在代理后面工作。请参阅代理配置 。

如果要升级现有的Upsource安装，请按照“ 升级Upsource安装”下概述的步骤进行操作。
