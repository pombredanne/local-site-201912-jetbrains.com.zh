Installing and configuring

For system requirements and other important notes please check out Prerequisites.

Upsource is an on-premises application that you install and run on a server. The installation package is available in two cross-platform distributions:

  • ZIP archive: A mainstream option with a straight-forward installation procedure that suites most cases. If you are not sure which distribution to choose — choose this one. Refer to ZIP installation for instructions.

  • Docker image: Allows to install and run Upsource in an isolated environment — a Docker container. Refer to: Docker installation for instructions.

Both distributions can be used to install and run Upsource in a cloud. See In-cloud installation.

Upsource can be also set up to work behind a proxy. See Proxy configuration.

If you want to upgrade your existing Upsource installation, follow the steps outlined under Upgrading your Upsource installation.
