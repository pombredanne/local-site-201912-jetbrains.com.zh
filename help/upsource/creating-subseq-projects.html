Creating subsequent projects

Creating any subsequent project differs from creating the first project in terms of UI.

To create any subsequent project:

  1. Click the settings icon on the top right of the screen.

  2. The Administration view will open.

  3. Click Create project. The Create Project form will be displayed.

  4. Fill in the Create Project form similar to how you did for your first project.
