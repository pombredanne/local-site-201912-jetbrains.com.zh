Synchronizing a GitLab project

When creating (or editing) a GitLab project, you can synchronize comments and merge requests between Upsource and GitLab.

To set up synchronization with Upsource

  1. Enable OAuth 2.0 Authentication Module in Hub.

  2. When setting up your GitLab project, on the Create Project page, tick Import merge requests as branches and Synchronize comments and merge requests
