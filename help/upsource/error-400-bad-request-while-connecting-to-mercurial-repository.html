错误400：“连接到Mercurial存储库时请求错误”

在管理页面上，您可能会注意到一个Mercurial项目中显示了以下错误：

检查VCS更改是否已完成，但有错误。 jetbrains.vcs.server.api。VcsServiceException：无法完成针对front：front的操作。无法收集front：front的当前修订。 'hg –config extensions.loadcommandscommandpy = / /temp/upsource-analyzer/hg2/load-commands-command.py CMD / /temp/upsource-analyzer/hg2/command.args

对于命令：hg –config ui.interactive = False pull –config extensions.progress = –config

progress.format =主题编号–config progress.delay = 0 –config progress.assume-tty = True

http：// upsource：******@hg.server'命令失败。

stderr：中止：HTTP错误400：错误的请求

解

此错误可能是由于Nginx无法代理大型请求标头引起的。在Nginx配置中增加头大小限制应该可以解决该问题。有关语法和默认值的更多详细信息，请参见Nginx文档 。
