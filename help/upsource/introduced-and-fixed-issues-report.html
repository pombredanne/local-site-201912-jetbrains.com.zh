Introduced and fixed issues report

This report allows you to get a better understanding of how a particular change has affected overall code quality.

Open the review and select a file from the left pane.

Upsource computes the difference between the previous and the current revisions of the file and shows you details of the issues (errors and warnings) that were introduced or fixed in the changes you're looking at:
