将您的上游安装移至另一台服务器

将您的Upsource安装迁移到其他服务器是一项非常简单的任务。

此处列出的命令是Windows命令。如果您使用的是Linux或macOS，只需更换.bat与.sh

  1. 停止上载： \bin\upsource.bat stop

  2. 将整个Upsource主目录移动到新位置。

  3. 在新服务器上，运行以下命令：

    \ bin \ upsource.bat配置--listen-port 1111 --base-url http：//服务器名称：1111 /
    哪里1111是您要使用的端口号， servername是新服务器的名称。

    如果您的基本URL先前已配置为不依赖服务器（例如，已设置为反向代理地址），则不应更改。

  4. 启动上游： \bin\upsource.bat start
