在文件中显示代码检查

Upsource的内置静态分析引擎可提供服务器端代码智能。它支持使用IntelliJ IDEA，Maven或Gradle作为其构建系统的Java，JavaScript和Kotlin项目。还支持PHP，但仅支持IntelliJ IDEA构建系统。

如果您正在使用Upsource查看代码，则可以自定义检查设置，包括选择安全设置。为此，请在IntelliJ IDEA中打开项目，然后导航至“设置” |“设置”。编辑器检查 。选择所需的设置，然后将其保存到“项目默认”配置文件。确保已使用您的项目代码签入Project_Default.xml文件，并且Upsource将使用它来确定要运行的检查。

静态检查可检测到严重的编译器和运行时错误以及不太严重的问题（如拼写错误或无效代码），并立即为您提供每次事件的详细信息。
