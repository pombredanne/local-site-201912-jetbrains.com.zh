Using Upsource to analyse repository activity

Upsource collects and analyses repository statistics and presents it to you in easy-to-view graphs and diagrams.

To go to the Analytics page, click Analytics at the top of your project home page:

  • Commit activity (Commit activity over time)

  • Project treemap (Visualisation of project files based on their sizes and other criteria)

  • File history (Watch how your project evolves over time)

  • Code review (Review coverage and statistics)

  • Reviewers graph (Visualisation of relationships between review participants)
