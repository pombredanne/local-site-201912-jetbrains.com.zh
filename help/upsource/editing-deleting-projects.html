Editing and deleting projects

To edit a project

  1. Click the settings icon on the top right of the screen.

  2. The Administration view will open.

  3. Click the Edit Project button next to the project name. The displayed Edit Project form contains the settings that you entered when you created the project.

  4. Edit the settings and click Save changes (refer to Creating first project for fields descriptions and instructions).

To delete a project

  1. Click the settings icon on the top right of the screen.

  2. The Administration view will open.

  3. Locate the project you are going to delete and click the Delete project button next to it.
