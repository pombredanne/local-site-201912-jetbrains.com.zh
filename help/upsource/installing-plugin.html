安装上游插件

对于所有基于IDEA的IDE，安装过程都非常简单且相似。

要将Upsource插件安装到基于IDEA的IDE中：

  1. 确保您的IDE构建兼容，并在必要时进行更新。请参阅兼容性页面以获取最新信息。

  2. 打开您的IDE并转到“ 文件|设置”或按Ctrl Alt S 。

  3. 在“ 设置”对话框窗口中，从左窗格中选择“ 插件 ”，然后单击“ 安装JetBrains插件..”按钮：

  4. 从显示的列表中，选择“ 上游集成” ，然后单击“ 安装插件” ：

  5. 为了使更改生效，请重新启动IDE。

现在，您需要为Upsource插件提供Upsource服务器URL：

  1. 在IDE的“设置”窗口的左窗格中，选择“ 工具” |“源” |“连接”

  2. 在服务器URL字段中输入地址

  3. 测试连接，然后单击“ 确定” 。

  4. 在显示的浏览器窗口中输入您的上游凭证。现在，您都可以直接从IDE开始使用Upsource code review功能。
