Error: "Some services registered by Product are not valid in target Hub"

This error might occur during the upgrade to version 2018.2 but only when the external Hub configuration is selected.

During the upgrade process, Configuration Wizard throws the following error:

Some services registered by Product are not valid in target Hub

Solution

The issue is caused by outdated records in the Upsource backup. To resolve it and successfully complete Upsource upgrade:

  1. Open the file hub.properties located in <date_specific_backup_directory_you_use_for_upgrade>\conf\internal\

  2. Delete the following lines:

    vcs-settings.service.id=XXXXXXX-XXXX-XXXX-XXXX-XXXXXX vcs-settings.service.secret.hash=XXXXXXXXXXXXXXXXXXXXXXXXXXXXX vcs-settings.service.secret.salt=XXXXXXX
