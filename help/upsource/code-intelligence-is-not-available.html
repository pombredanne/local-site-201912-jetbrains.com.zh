代码情报不可用

基于Maven或Gradle的项目中的代码洞察无法正常工作。代码分析状态指示灯保持灰色。

解

  1. 在创建Upsource项目时 ，请仔细检查您是否已启用代码智能并正确设置属性，包括项目模型的正确路径。必要时进行编辑。

  2. 如果没有帮助，请联系上游支持并描述您的问题。请在查询中包含一个mvn.out或gradle.out文件（对应于Maven或Gradle项目）。

要查找mvn.out（或gradle.out）文件：

  1. 从以下URL下载压缩的生成树： /~generatedTree/:

    缺少代码智能的修订版的ID在哪里。

    上游项目ID通常与项目名称不同。

  2. 在该树中找到mvn.out（或gradle.out）文件。

    如果不存在这样的文件：

  3. 导航到/.idea/model.info

  4. 打开model.info并复制修订ID

  5. 再次下载生成的树，将URL中的修订ID替换为您复制的树。

  6. 在该树中找到mvn.out（或gradle.out）文件。
