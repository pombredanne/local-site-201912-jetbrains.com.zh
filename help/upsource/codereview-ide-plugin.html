使用Upsource插件在IDE中进行代码审查

用于IntelliJ IDEA和其他JetBrains IDE的Upsource code review插件使您无需离开IDE即可参与代码讨论和管理评论。

它使您可以直接从文本编辑器查看和创建评论注释，并提供一个“ 评论”工具窗口，该窗口列出了当前Upsource项目中的评论并可以对其进行管理。

此外，该插件还集成到IDE自己的控件中，例如“ 版本控制”工具窗口和“ 提交更改”对话框。

该插件可在基于IntelliJ平台的以下IDE上运行：

  • IntelliJ IDEA 2016.1以上
  • PhpStorm 2016.1+
  • PyCharm 2016.1+
  • RubyMine 2016.1+
  • WebStorm 2016.1以上
  • AppCode 2016.1以上
  • CLion 2016.1+
  • DataGrip 2016.1+

  • 和Android Studio 2.2
