The following document explains the concept behind Upsource user management.

How user management works in Upsource

While Upsource creates the default admin account for you during setup, you still need user accounts for developers on your team to access and use Upsource.

In addition to creating user accounts, you also need to grant permissions to these accounts in order to enable respective developers to take part in the review process.

To make these tasks more efficient, Upsource utilises Hub - the service that provides a single point of entry for user management across all JetBrains services and applications connected to it. With Hub the user needs only one user account to access all connected services, and the administrator can handle these services and user accounts from one spot.
