Creating and managing projects

What is an Upsource project?

An Upsource project corresponds to a VCS repository. Creating a new project enables you to:

  • Visualize commit history in the VCS repository
  • Browse code from any revision in your project
  • See changes in each revision
  • Create discussions on code and full-fledged code reviews

Read the following articles to learn how to create, edit and remove Upsource projects:

  • Creating your first project
  • Creating subsequent projects
  • Editing and deleting projects

Upsource projects can be also managed from a standalone Hub which allows for user and project management across all JetBrains services and applications connected to it.
