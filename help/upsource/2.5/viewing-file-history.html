Viewing file history

History view

The History view lets you see how a file has evolved since it was created and who has contributed to it.

To open the History view, navigate to the File view and click the History button:

This view displays the list of revisions that have affected the file.

Annotate view

The Annotate view displays line-level references to contributors and revisions that have recently affected specific lines of code in a file.

To open the Annotate view, navigate to the File view and click the Annotate button.

Each reference associated with a specific change contains a revision ID with a link, its author, and a date it took place.
