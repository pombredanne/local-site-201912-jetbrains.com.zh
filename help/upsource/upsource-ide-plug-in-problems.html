Upsource IDE plug-in problems

If you stumble upon a problem while working with or trying to set up Upsource in your IDE, you will need to report it to us to get a prompt help. But first you have to enable advanced logging in the Upsource IDE plugin. This log serves as a starting point for investigating most issues, including:

  • "Test connection failed"

  • "Timeout error"

  • "Upsource request error"

Here is how to proceed:

  1. In you IDE, go to Help | Debug Log Settings

  2. Put the following lines in the displayed dialog box:

    #com.jetbrains.upsource io.socket

  3. Reproduce your issue

  4. Go to Help | Show log in Explorer (Finder)

  5. Copy all idea.log files and send them to Upsource Support Team by creating a ticket through the customer portal or by filing an issue in our issue tracker.
