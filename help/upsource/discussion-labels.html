Discussion Labels

You can apply labels to any discussion to denote priority, category, or any other information that you find useful.

There are several predefined labels to choose from:

  • bug

  • code style

  • enhancement

  • help wanted

The selected label will be displayed:

To remove the label from the discussion, click the minus icon:

You can also create custom labels that will be available across the project. Choose Create new label, type in your label name and select the desired color, then click Create label.

To edit and delete labels from the list, you've got to have project administrator rights. A project administrator can also create new labels centrally, without applying them to discussions.
