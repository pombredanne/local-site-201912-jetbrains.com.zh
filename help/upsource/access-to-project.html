Providing access to a project

You can give access rights to groups or specific users.

To give access to a group:

  1. Click the settings icon on the top right of the screen. The Administration view will open.

  2. Click Users. The Hub's Users view will be displayed in a new window.

  3. In this new window, click the settings icon and select Groups from the drop-down menu. The Groups form will be displayed.

  4. Select the group from the list and click the Access tab.

  5. Click Grant Access then select Role or Permission and a project from a displayed menu.

    Granting a permission to a group is realized by assigning to it a role that contains the specified permission.

To give access to a user:

  1. Click the settings icon on the top right of the screen. The Administration view will open.

  2. Click Users. The Users view will be displayed.

  3. Select the user from the list and click the Access tab.

  4. Click Grant Access then select Role or Permission and a project from the displayed menu.

    Granting a permission to a user is realized by including the user into a group that has the specified access permission.
