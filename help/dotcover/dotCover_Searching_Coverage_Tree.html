Searching Coverage Tree

The coverage tree displayed in the Coverage Results Browser window nor in the Unit Test Coverage window provides a quick-search option for all items included in the tree.

Quick-search in coverage tree supports CamelHumps (identification of compound names composed according to CamelCase, where each part starts with a capital letter) and the asterisk wildcard.

To search the coverage tree

  1. Set the application focus to the Coverage Results Browser window nor in the Unit Test Coverage window.

  2. Start typing a search string.

  3. The Search box will appear under the toolbar displaying your search string and the coverage tree will be filtered according to the search string.

  4. Click Show matching rows only to enable or disable filtering.

Unlike Excluding Nodes from Coverage Snapshot, applying search filters does not recalculate total coverage.
