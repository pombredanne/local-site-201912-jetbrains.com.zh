恢复NuGet软件包

通过此按钮，您可以使用dotCover自己的NuGet恢复器实现来为当前解决方案恢复NuGet软件包。
默认情况下，在每次构建之前也会自动启用NuGet软件包的自动还原。如有必要，您可以通过清除“ 工具” |“ 创建之前还原NuGet软件包”复选框来禁用它。建立 dotCover选项的常规页面。

开始分析
