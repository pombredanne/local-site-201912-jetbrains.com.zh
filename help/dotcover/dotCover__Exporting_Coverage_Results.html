Exporting Coverage Results

dotCover helps process coverage data the way you need. Exporting to several formats, including HTML, XML, JSON, and XML for NDepend enables you to share coverage reports with the rest of the team or pass coverage information to external services.

To export results from coverage snapshot

  1. In the Coverage Results Browser window nor in the Unit Test Coverage window, click Export coverage report .

  2. In the drop-down list that opens, choose a format for export.

  3. In the dialog that appears, specify a file name and location to which the report should be saved.
