推广经理

ReSharper |选项|环境|推广经理

本页列出了扩展管理器对话框用于获取dotCover扩展的所有NuGet软件包源。默认情况下， .NET产品库是主要来源。您还可以指定其他来源-本地或远程文件系统或自定义的NuGet提要-安装针对您的特定需求的个人或公司范围的扩展。

列表中项目的位置定义了它们的优先级。

项目

描述

加

单击此按钮添加新的NuGet软件包源。

编辑

单击此按钮以编辑选定的NuGet软件包源。

去掉

单击此按钮删除所选的NuGet软件包源。

提升

单击此按钮可在列表中向上移动所选的NuGet软件包源，从而提高其优先级。

下移

单击此按钮可在列表中将选定的NuGet软件包源下移，从而降低其优先级。

dotCover的设置保存在设置层中，使您可以针对不同的解决方案进行不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享dotCover设置 。
