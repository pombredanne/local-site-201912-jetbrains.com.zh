在覆盖树中查找代码项

ReSharper |封面找到覆盖树
Ctrl+Alt+K L
ReSharper_LocateInCoverageTree

在Visual Studio编辑器中工作时，可以在代码覆盖树中快速找到代码符号（类型，方法等）。如果要查看当前快照中代码符号的覆盖程度，这可能会有所帮助。

有关覆盖树中符号位置的信息取自当前的覆盖快照。因此，如果没有打开coverage快照，则主菜单和上下文菜单中的相应命令将显示为灰色。您需要运行单元测试的覆盖率分析以启用这些命令。

在覆盖树中定位代码符号

  1. 在Visual Studio编辑器中，将插入标记放置在您要在coverage树中定位的代码符号上。

  2. 按Ctrl+Alt+K L或选择 ReSharper |封面从主菜单中找到“覆盖树 ”。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 该符号将位于当前快照层次结构的 覆盖率结果浏览器窗口或“ 单元测试覆盖率”窗口中。

通过双击覆盖树中的项目可以使用反向导航。
您可以用来浏览代码覆盖率的另一个导航目标是导航到Covering Tests 。
