命令和菜单
隐藏覆盖的Visual Studio菜单项

隐藏被ReSharper功能覆盖的Visual Studio本机菜单项，例如Visual Studio的Organize Usings菜单。

使用Alt + R快捷方式打开ReSharper主菜单

如果您使用Visual Studio 2019或更高版本，则ReSharper菜单将隐藏在Extensions菜单内，因此dotCover允许您按Alt+R快速访问其菜单命令。
注意Alt+R用于某些Visual Studio UI元素，例如，在“ 查找和替换”弹出窗口中，它将替换下一个匹配项。如果以这种方式使用它，或者已将其分配给其他操作，请禁用此选项。

键盘快捷键
ReSharper Ultimate键盘方案

您可以在两个默认键盘快捷方式之间进行切换，也可以在Visual Studio选项（“ 工具” |“选项” |“环境” |“键盘” ）中选择“ 无”以自行配置ReSharper快捷方式。单击“ 应用”以应用选定的方案。有关更多信息，请参见配置键盘快捷方式 。

申请方案

应用所选的键盘快捷方式方案。请注意，所选方案未保存在设置中。

dotCover的设置保存在设置层中，使您可以针对不同的解决方案进行不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享dotCover设置 。
