添加/编辑覆盖率过滤器对话框

编辑覆盖率过滤器|添加过滤器
编辑覆盖率过滤器|编辑

该对话框有助于指定新的coverage过滤器或修改现有的coverage过滤器。单击“ 覆盖范围过滤器”对话框中的相应按钮时，它将打开。

项目

描述

过滤规则
分析代码

选择此选项可将指定的代码项包括在代码覆盖率分析中，并排除所有其他代码项。

不分析代码

选择此选项可从代码覆盖率分析中排除指定的代码项。

... 部件

要包括/排除的程序集或项目的名称。

...课

要包括/排除的类的名称。

... 方法

要包括/排除的方法的名称。

不分析代码

选择此选项可从代码覆盖率分析中排除具有指定属性的代码项

...标有属性

要包括/排除的类的名称。

...在汇编中声明

要包括/排除的方法的名称。

将过滤器应用于
所有解决方案

选择此选项可将过滤器全局应用到计算机上。

当前解决方案

选择此选项可将过滤器仅应用于当前解决方案。

在过滤器中指定代码项名称时，可以使用星号通配符：“ *”（星号）表示零个或多个字符。
例如，您可以使用*.Tests通配符，将从代码分析中排除所有测试项目。
