Options

ReSharper | Options

The Options dialog is the central location for dotCover configuration settings. You can use the search box in the upper left corner to find the desired preference.

The settings are organized in the following categories:

  • Environment
    • General
    • Keyboard & Menus
    • Extension Manager
    • Product Feedback
    • Products & Features
    • Updates
    • Web Proxy Settings
  • Tools
    • Unit Testing
      • Test Runner
      • Test Frameworks
        • NUnit
        • xUnit.net
        • MSTest
    • Build
  • dotCover
    • General
    • Continuous Testing
    • Editor
    • Filtering

Click Manage to access the Settings Layers dialog that helps manage settings layers. For detailed instructions, see Managing and Sharing dotCover Settings.
