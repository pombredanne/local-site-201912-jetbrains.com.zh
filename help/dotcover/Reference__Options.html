ReSharper |选件

“ 选项”对话框是dotCover配置设置的中心位置。您可以使用左上角的搜索框找到所需的首选项。

这些设置分为以下类别：

  • 环境
    • 一般
    • 键盘和菜单
    • 推广经理
    • 产品反馈
    • 产品与功能
    • 更新
    • Web代理设置
  • 工具类
    • 单元测试
      • 测试亚军
      • 测试框架
        • NUnit
        • xUnit.net
        • 微软测试
    • 建立
  • dotCover
    • 一般
    • 连续测试
    • 编辑
    • 筛选

单击管理以访问有助于管理设置层的“设置层”对话框 。有关详细说明，请参阅管理和共享dotCover设置 。
