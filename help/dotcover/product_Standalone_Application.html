dotCover Standalone Application

Start menu | JetBrains dotCover

The dotCover standalone application allows performing the following tasks outside Visual Studio:

  • Run coverage analysis of .NET applications - Analyze button on the Welcome page or File | Cover Application or Ctrl+Alt+K K

  • Open coverage snapshots - Open Snapshot button on the Welcome page or File | Open Coverage Snapshot or Ctrl+Alt+K O - and perform the following tasks with them:
    • Detecting Hot Spots
    • Searching Coverage Tree
    • Excluding Nodes from Coverage Snapshot
    • Exporting Coverage Results

  • Edit global coverage filters - File | Edit Coverage Filters or Ctrl+Alt+K F

  • Configure dotCover options (the subset of options that are not related to Visual Studio) - File | Options
