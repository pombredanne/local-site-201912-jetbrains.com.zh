单元测试范围

在Visual Studio中：
ReSharper | Windows |单元测试范围
Ctrl+Alt+K B

该窗口允许浏览在单元测试coverage运行期间获得的coverage数据 。

每个快照可以两个视图显示，您可以使用工具栏右侧的按钮进行切换：

覆盖树

在此视图中，所有代码项都以树结构分组，您可以研究每个代码项的覆盖范围。

在Coverage列中，dotCover使用三种颜色来显示coverage状态：

  • 绿色 -节点内所覆盖语句的百分比。

  • 红色 -节点内未发现语句的百分比。

  • 灰色 -测试运行期间未覆盖的节点，因为它们没有可执行代码语句。

热点

在此视图中，您可以快速查看快照中的潜在风险区域。要了解更多信息，请参阅检测热点 。

覆盖树

热点

切换热点视图。有关更多信息，请参见检测热点 。

所有测试

如果选中，该树将显示所有单元测试会话的汇总覆盖率结果。

活动会话中的所有测试

如果选中，该树将显示“单元测试会话”窗口中当前选择的会话中所有单元测试的覆盖率结果。

活动会话中的选定测试

如果选中，该树将在“单元测试会话”窗口中显示当前会话中所选测试的覆盖率结果。

使覆盖范围结果与当前项目结构匹配

如果打开此按钮，则dotCover仅显示当前解决方案中包括的部件。有关更多信息，请参见将覆盖率快照与项目结构匹配 。

突出显示代码

切换编辑器中当前coverage快照的代码突出显示。有关更多信息，请参见可视化代码覆盖率 。

如果有多个快照覆盖相同的代码，则突出显示将显示上次测试会话的快照或上次打开突出显示的快照的结果。

将coverage快照另存为

将当前快照保存到.dcvr文件。有关更多信息，请参见保存和打开覆盖率快照 。

覆盖范围结果 删除现有的覆盖范围结果。

仅当查看特定单元测试会话的覆盖结果时，此命令才可用。

出口覆盖率报告

允许您将测试覆盖率报告导出为所选格式之一： 用于NDepend的 HTML，XML，JSON和XML。有关更多信息，请参阅导出承保范围结果

此窗口中的上下文菜单包括命令：

  • 排除选定的节点
  • 排除除所选节点外的所有节点
  • 排除所选节点并创建coverage过滤器
这些命令很容易解释。它们允许将节点从coverage快照呈现中排除 。
