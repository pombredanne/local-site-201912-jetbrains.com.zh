应用范围分析

与您的解决方案中是否有单元测试无关，dotCover允许您分析已执行应用程序的代码覆盖率。在dotCover的控制下运行应用程序并在特定情况下执行该应用程序之后，您将获得一个包含运行语句覆盖率的覆盖率快照 。使用此快照中的数据，您可以深入了解所涵盖应用程序的源代码，并查看在执行的方案期间调用了代码的哪些部分以及未实现的部分。

与单元测试相同，此处的语句覆盖范围并不能保证所覆盖的语句不包含任何错误。但是，这些数据将帮助您分析执行方案中未到达代码的哪些部分。

在dotCover的控制下，有两种运行应用程序的方式：

  • 运行当前解决方案的启动项目
  • 运行外部应用程序或进程

在这两种情况下，都会显示Coverage Controller对话框来指导coverage运行 。
