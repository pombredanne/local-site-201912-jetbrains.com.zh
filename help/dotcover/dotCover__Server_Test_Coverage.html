分析服务器的覆盖率

在远程服务器上执行覆盖的主要方法是使用dotCover Console Runner ：与dotCover捆绑在一起的控制台工具允许您使用命令行提示符运行单元测试或应用程序的覆盖率分析。您还可以在服务器端（例如，在持续集成环境中（在JetBrains TeamCity服务器上或任何第三方CI服务器上 ）使用控制台运行程序 ）。

dotCover Console Runner可以在没有Visual Studio的情况下工作。
