Web代理设置

ReSharper |选项|环境| Web代理设置

在dotCover选项的此页面中，您可以调整dotCover的Internet活动的 Web代理设置。默认情况下，dotCover遵循您的系统设置，如果系统中定义了任何代理服务器，dotCover也会使用它来验证许可证信息，检测更新等。

项目

描述

从Internet下载时使用系统Web代理设置

您可以清除此复选框以覆盖dotCover的Internet连接的系统Web代理设置。

编辑

当您选择使用系统代理服务器设置时，将显示此按钮。
通过单击此按钮，打开“ Internet属性”对话框，您可以在其中更改系统Web代理设置 。

dotCover的设置保存在设置层中，使您可以针对不同的解决方案进行不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享dotCover设置 。
