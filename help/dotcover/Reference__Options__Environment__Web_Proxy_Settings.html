Web Proxy Settings

ReSharper | Options | Environment | Web Proxy Settings

In this page of dotCover options, you can adjust Web proxy settings for dotCover's Internet activities. By default, dotCover follows your system settings and if any proxy server is defined in your system, dotCover will use it too for verifying license information, detecting updates, etc.

Item

Description

Use system Web proxy settings when downloading from Internet

You can clear this check box to override system Web proxy settings for dotCover's Internet connection.

Edit

This button appears when you choose to use system proxy server settings.
By clicking on this button you open the Internet Properties dialog, where you can change system Web proxy settings.

dotCover's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Managing and Sharing dotCover Settings.
