重命名文件对话框

重构|改名
Shift+F6

使用此对话框来重命名文件。

除了重命名文件本身之外，DataGrip还可以查找文件名的用法。如果找到，则您对文件名所做的更改也可以应用于这些用法。

用法被分配到不同的类别，这些类别与您可以打开和关闭的选项相对应。

为文件指定一个新名称。

搜索注释和字符串

如果选中此复选框，DataGrip将在源代码文件中的注释和字符串文字中查找文件名的出现。

搜索文字出现

如果启用此选项，DataGrip将在不包含源代码的文件中查找文件名的出现。这些可能是文本文件，属性文件，HTML文件，文档文件等。

范围

使用此选项可以设置重命名重构的范围。例如，您可以将重构限制为仅最近更改的文件或仅打开的文件，依此类推。您还可以创建可以共享或保留在本地的自定义范围。
