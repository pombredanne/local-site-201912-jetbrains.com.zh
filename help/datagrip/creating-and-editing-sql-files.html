如何使用DDL数据源

步骤1。在SQL文件中定义表

在文件工具窗口（视图|工具窗口|文件）中，右键单击目录，然后选择新建| SQL文件 。指定SQL文件的名称。

打开SQL文件并在我们的文件中定义以下表结构：

CREATE TABLE系列（member_id INT NOT NULL，名称VARCHAR（50），关系VARCHAR（50））；

  1. 用于插入CREATE TABLE语句，有一个预定义的模式。按Ctrl+J然后选择模式tab - new table definition 。

    结果如下：

  2. 输入表格名称family 。

  3. 要从family至col ， 按Tab 。更换col与member_id 。

  4. 键入表定义的其余部分。

第2步。从SQL文件创建DDL数据源

要从SQL文件创建表，请将带有已定义表的SQL文件拖到“ 数据库”工具窗口（“ 视图” |“工具窗口” |“数据库” ）。

第三步定义数据源的数据

现在，我们将创建一个SQL文件，用于向表中添加数据。在步骤2中，将SQL文件定义为数据源，现在在编辑器中提供了表及其列的名称以及有关相应数据类型的信息。

  1. 创建一个SQL文件。

    让我们添加这个简单INSERT声明到文件中：

    插入家族（member_id，姓名，亲戚）值（1，“ Chloe”，“ mother”）；

  2. 添加INSERT通过选择相应的预定义模式来声明： Ctrl+J | ins - insert rows into a table 。

  3. 选择表名family 。

  4. 选择列名称列表member_id, name, relation 。

  5. 指定值时，按Ctrl+P （ 查看|参数信息 ）以查看有关值类型的信息。
