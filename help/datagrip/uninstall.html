Uninstall DataGrip

The proper way to remove DataGrip depends on the method you used to install it.

This guide describes ways to uninstall DataGrip on WindowsmacOSLinux. Use the platform switcher at the top to select another operating system.

Toolbox App

If you installed DataGrip by using the Toolbox App:

  • Open the Toolbox App, click the screw nut icon for the necessary instance, and select Uninstall.

Standalone instance

To remove a standalone DataGrip instance, use the standard procedure for uninstalling applications on your operating system and remove the IDE configuration and system directories.

  1. Open the Apps & features section in the system Settings dialog.

  2. Select the DataGrip app and click Uninstall.

  3. Remove the following directories:

    Syntax
    %HOMEPATH%\.<product><version>\config
    Example
    C:\Users\JohnS\.DataGrip2019.2\config
    Syntax
    %HOMEPATH%\.<product><version>\system
    Example
    C:\Users\JohnS\.DataGrip2019.2\system

  1. Delete the application in the Applications directory.

  2. Remove the following directories:

    Syntax
    ~/Library/Preferences/<product><version>
    Example
    ~/Library/Preferences/DataGrip2019.2
    Syntax
    ~/Library/Caches/<product><version>
    Example
    ~/Library/Caches/DataGrip2019.2

  1. Delete the installation directory.

  2. Remove the following directories:

    Syntax
    ~/.<product><version>/config
    Example
    ~/.DataGrip2019.2/config
    Syntax
    ~/.<product><version>/system
    Example
    ~/.DataGrip2019.2/system

Uninstall silently

If you installed DataGrip silently, you can run the uninstaller with the /S switch as an administrator. The uninstaller is located in the installation directory under bin.

Run cmd (Windows Command Prompt) as administrator, change to the DataGrip installation directory, and run the following:

bin\uninstall.exe /S
