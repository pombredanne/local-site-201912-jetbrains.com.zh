使用“ 文件”工具窗口可以管理与项目以及数据库控制台关联的文件和文件夹。

该工具窗口中的大多数功能都可以通过内容窗格中的上下文菜单命令以及关联的快捷方式来访问。

观看次数

工具窗口为您提供以下视图：

  • 文件视图。对于与项目关联的文件和文件夹，此视图提供的功能类似于文件管理器。它使您可以创建文件和文件夹，在编辑器中打开文件以及执行其他有用的任务。

    该视图还包含“暂存和控制台”文件夹，该文件夹可让您管理创建的暂存文件， 数据库控制台和扩展脚本。

    看到和脚本文件 。

  • 范围视图。包含用户定义的范围 。

从标题栏左侧的列表中选择所需的视图，或者如果视图由选项卡表示，则单击相应的选项卡。

要配置视图，请使用标题栏上下文菜单中的相应选项。还可以通过单击来访问必要的选项在标题栏上。

自动滚动到源

如果启用此选项，DataGrip将自动在编辑器中打开所选项目。否则，您需要双击项目以将其打开。

从源代码自动滚动

如果启用此选项，DataGrip会自动在此窗口的当前视图中找到您在编辑器中打开的文档。

按类型排序

如果关闭此选项，则文件将按文件名的字母顺序进行排序。如果启用此选项，则文件将按其扩展名排序。

该选项不会影响文件夹的顺序。文件夹始终按字母顺序排序。

文件夹总是在最前面

如果启用此选项，则文件夹将显示在文件上方。否则，所有项目均按字母顺序排序，并且文件和文件夹看起来混合在一起。

组选项卡

如果启用此选项，则标题栏左侧会有一个列表，您可以从中选择所需的视图。如果关闭此选项，则视图由显示在标题栏左侧的选项卡表示。

查看模式

此选项使您可以控制工具窗口的一般外观和行为。

搬去

要将工具窗口与其他工具窗口栏关联，请选择此命令，然后选择目标工具窗口栏（“ 顶部” ，“ 左侧” ，“ 底部”或“ 右侧” ）。

调整大小

要通过移动工具窗口的边框来调整其大小，请选择此命令，然后选择必要的“ 拉伸至”选项。

请注意，此命令不适用于浮动模式。

从侧边栏删除

此命令隐藏工具窗口，删除用于显示和隐藏工具窗口的按钮，并从快速访问菜单中删除工具窗口（ 要么 ）。

要再次打开工具窗口（并恢复关联的功能），请使用主菜单：“ 查看” |“更改”。工具窗口| 或按Alt+2 。

内容窗格

内容窗格显示与项目及其内容关联的文件夹，或按DB数据源分组的数据库控制台。

内容窗格项的上下文菜单命令

使用项目的上下文菜单可以访问该项目的所有可用功能。

项目

捷径

描述

附加目录

使用此命令将目录与项目关联。在打开的对话框中选择感兴趣的目录。所选目录及其所有内容实际上将成为项目的一部分。

新

 Alt+Insert

使用此命令或快捷方式在当前目录中创建一个新目录或文件。在Scratches视图中，此命令还允许您创建数据库控制台。

切

 Ctrl+X

使用此命令将选定的一个或多个项目从当前位置移动到剪贴板。

复制

 Ctrl+C

使用此命令将选定的一个或多个项目复制到剪贴板。

复制路径

 Ctrl+Shift+C

使用此命令可将所选项目的完整路径复制到剪贴板。

糊

 Ctrl+V

使用此命令将剪贴板的内容插入到所选位置。

跳转到源

 F4

使用此命令在编辑器中打开选定的文件。如果文件已经打开，则相应的编辑器选项卡将变为活动状态。

查找用法

 Alt+F7

使用此命令查找所选项目的用法。（将打开“ 查找用法”对话框 。）

重构

使用此命令可执行可用于所选项目的重构之一。

例如，您可以重命名，移动和复制文件和目录。

显示缩略图

 Ctrl+Shift+T

使用此命令可以查看位于所选目录中的图像文件的缩略图。（“ 缩略图”工具窗口将打开。）

删除

 Delete

使用此命令删除所选项目。小心使用！

对于“ 文件”视图中的顶级目录，您可以选择物理删除目录或删除目录与项目的关联（“ 分离”选项）。

更改方言（ ）

对于SQL文件和数据库控制台：更改与文件或控制台关联的SQL方言。

运行'

 Ctrl+Shift+F10

对于SQL文件或数据库控制台：执行所选文件或控制台中包含的所有语句。

当地历史

使用此命令可查看所选文件或目录的本地历史记录 ，或为项目的当前版本创建标签。

同步' '

使用此命令将所选项目与其保存在文件系统中的版本同步。

（如果从外部更改文件或目录的内容，则在某些情况下，DataGrip可能不知道相应的更改，除非您使用此命令。）

在资源管理器里显示

使用此命令打开文件浏览器（例如Windows资源管理器或Finder），并在其中显示所选的项目。

文件路径/目录路径

 Ctrl+Alt+F12

该菜单显示从文件系统根目录到所选元素的路径，并以各个目录作为菜单项。

当您在此菜单中选择一个项目（例如，目录）时，将打开文件浏览器（例如，Windows资源管理器或Finder），并在其中显示所选的项目。

与之比较

 Ctrl+D

使用此命令将所选文件或目录与另一个文件或目录进行比较。请参阅文件的“ 差异查看器”和文件夹的“差异查看器” 。

与编辑器比较文件

使用此命令将所选文件与活动编辑器选项卡上打开的文件进行比较。有关文件，请参见“ 差异查看器” 。

将目录标记为排除

使用此命令使DataGrip忽略所选目录。结果，该目录将不会显示在“ 文件”工具窗口中，并且您将无法在DataGrip中使用该目录。
