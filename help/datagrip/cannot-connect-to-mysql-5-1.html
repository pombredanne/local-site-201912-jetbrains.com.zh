DataGrip 2019.2 Help

Cannot connect to MySQL 5.1

  1. Navigate to File | Data Sources Ctrl+Alt+S.

  2. In the Data Sources and Drivers dialog, click the Add icon () and select MySQL.

  3. Click the Driver link and select MySQL for 5.1.

  4. To download JDBC drivers for MySQL 5.1, click the Download link at the bottom of the dialog.

  5. Specify database connection details. Alternatively, paste the JDBC URL in the URL field.

    To set an empty password, right-click the Password field and select Set empty.

  6. To ensure that the connection to the data source is successful, click Test Connection.
