安装Subversion集成插件

要在DataGrip中启用版本控制，请安装Subversion Integration插件。

  1. 打开设置Ctrl+Alt+S并导航到Plugins 。

  2. 单击市场 。

  3. 在搜索字段中，输入Subversion Integration然后按Enter 。

  4. 要安装插件，请单击安装并重新启动DataGrip。

启用Subversion集成后，您可以从DataGrip内部执行基本的Subversion操作。

DataGrip当前支持与Subversion 1.7及更高版本的集成。

DataGrip与Subversion插件捆绑在一起。除此之外，您还需要在计算机上下载并安装命令行客户端。
