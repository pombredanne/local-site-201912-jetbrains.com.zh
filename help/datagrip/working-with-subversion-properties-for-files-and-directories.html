使用文件和目录的Subversion属性

Subversion集成使您可以使用特定于Subversion的属性，而无需离开DataGrip。

定义后，文件或目录的属性将显示在“ SVN属性”视图中。在此视图中，您可以浏览和更改现有属性及其值，或使用工具栏按钮或上下文菜单命令创建新属性。

本节介绍如何：

  • 在DataGrip中查看文件或目录的属性
  • 创建一个新属性，或更改现有属性的值
  • 设置关键字属性
  • 删除属性
  • 解决财产冲突

查看文件或目录的属性

  1. 在“ 文件”工具窗口中 ，在SVN版本控制下选择所需的文件或目录。

  2. 从VCS主菜单或所选内容的上下文菜单中，选择“ Subversion | Subversion”。编辑属性 。SVN属性视图将打开，显示所选文件的属性：

  3. 使用工具栏按钮或上下文菜单命令来创建，编辑或删除属性，如以下过程所述。

如果您希望SVN属性视图在浏览项目或编辑文件时保留其内容，请确保遵循选择按钮没有按下否则，该视图将显示当前选定或编辑的文件的属性。

创建新属性，或为现有属性设置值

  1. 打开“ SVN属性”视图 。

  2. 点击添加按钮在SVN属性视图的工具栏上，或在上下文菜单上选择添加属性命令。出现“ 设置属性”对话框。

  3. 在“ 属性名称”字段中，键入新属性的名称，或从下拉列表中选择一个。

  4. 选择“ 设置属性值”选项，然后在下面的文本区域中指定所需的值。

  5. 要将更改应用于所选目录的所有子目录，请选中“ 递归更新属性”复选框。

  6. 单击确定 。

  • 在“ SVN关键字”对话框中，选中要包含在属性中的关键字。

  • 单击确定 。

删除属性

  1. 选择要删除的属性。

  2. 请点击在工具栏上。

您也可以从“ 设置属性”对话框中删除属性。去做这个：

  1. 在“ 属性名称”字段中，选择要删除的属性。

  2. 选择删除属性单选按钮。

  3. 如果要从所选目录的所有文件和子目录中删除此属性，请选择“ 递归更新属性”选项。
