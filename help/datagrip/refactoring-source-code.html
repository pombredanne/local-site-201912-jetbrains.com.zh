重构代码

重构是在不创建新功能的情况下改进源代码的过程。重构可以帮助您保持代码的牢固和易于维护。

重命名重构

重命名数据库对象

  1. 将插入符号放置在要重命名的对象上。

  2. 导航到重构|改名 Shift+F6 。如果您在函数中编辑参数，则可以开始内联输入新名称，或按Shift+F6两次以调用“ 重命名”对话框。

  3. 在“ 重命名”对话框中，为对象键入一个新名称。

  4. 查看更改，然后单击“ 重构” 。要搜索对象名称的用法，请单击“ 预览” 。

提取重构

提取变量

  1. 将插入符号放在函数中的变量上。

  2. 导航到重构|提取物|变数 （ Ctrl+Alt+V ）。

  3. 选择要提取的表达式的一部分，然后按Enter 。

提取为公用表表达式（CTE）

CTE是一个临时结果集，可以在另一个SELECT，INSERT，UPDATE或DELETE语句中使用。在DataGrip中，您可以提取子查询并将其转换为CTE。

  1. 将插入符号放在要转换的子查询中。

  2. 导航到重构|提取物|子查询为CTE 。

预览重构更改

对于某些重构，DataGrip允许您在应用更改之前预览更改。

  1. 要查看可能的更改，请在“ 重构预览”对话框中单击“ 预览 ”。

  2. 检查“ 查找工具”窗口中将要进行的更改。您可以排除Delete或删除Ctrl+X您认为不必要的更改。

  3. 单击“ 执行重构”以继续更改。
