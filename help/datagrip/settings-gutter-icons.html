Gutter Icons

File | Settings | Editor | General | Gutter Icons for Windows and Linux
DataGrip | Preferences | Editor | General | Gutter Icons for macOS

Use this page to hide or show the icons in the gutter area that invoke actions related to the basic, DataGrip-wide features or to framework- and technology-specific features for all the newly created editors.

The right-hand pane shows all the gutter icons available in DataGrip. The basic, DataGrip-wide features, such as , are displayed at the top of the list under the Common title. Other features are grouped by the frameworks and technologies to which they are related.

  • To have an icon displayed in the gutter area, find the icon or the corresponding action in the list and then select the checkbox next to it.

  • To have an icon hidden, clear the checkbox next to it.
