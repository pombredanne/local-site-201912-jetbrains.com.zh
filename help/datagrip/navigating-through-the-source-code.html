DataGrip 2019.2帮助

源代码导航

您可以使用不同的操作和弹出窗口在编辑器中快速浏览代码。

您可以从观看视频教程开始，以检查DataGrip中可用的导航选项。

查找文件路径

  1. 在编辑器中，按Ctrl+Alt+F12或在上下文菜单中，选择“ 在Finder中显示” 。

  2. 在“ 在查找器中显示”弹出窗口中，选择要在路径查找器中打开的文件或目录，然后按Enter 。

使用“选择范围”弹出窗口进行导航

您可以在“ 文件”工具窗口中自动找到文件。

  1. 如果在编辑器中打开了文件，请按Alt+F1打开“ 选择范围”弹出窗口。

  2. 在弹出窗口中，选择“ 文件视图” ，然后按Enter 。DataGrip在“ 文件”工具窗口中找到您的目标。

使用自动滚动查找文件

您可以使用“ 自动滚动到源”和“ 从源自动滚动”操作在“ 文件”工具窗口中找到文件。

  1. 在“ 文件”工具窗口中，右键单击“ 文件”工具栏，然后从上下文菜单中选择“ 从源代码自动滚动” 。之后，DataGrip将跟踪当前在活动编辑器选项卡中打开的文件，并自动在“ 文件”工具窗口中找到它。

  2. 您还可以选择“ 自动滚动到源”选项。在这种情况下，当您在“ 项目”视图中单击文件时，DataGrip将自动在编辑器中将其打开。

使用书签进行导航

  • 要创建匿名书签，请将插入符号放在所需的代码行中，然后按F11 。

  • 要创建带有助记符的书签，请将插入号放置在所需的代码行中，按Ctrl+F11并选择一个数字或字母作为助记符。

  • 要打开“ 书签”对话框，请按Shift+F11 。您可以使用此对话框来管理书签，例如，删除书签，对书签排序或为其提供简短说明。

  • 要导航到带有字母助记符的现有书签，请按Shift+F11然后按一下您需要的字母。DataGrip使您返回到编辑器和相应的书签。

  • 要导航到带有数字助记符的现有书签，请按Ctrl和书签的编号。

  • 每个创建的书签都会反映在“ 收藏夹”中 N/A工具窗口，您还可以使用该窗口导航到书签。

在错误或警告之间导航

  • 要跳至代码中的下一个或上一个发现的问题，请按F2要么Shift+F2分别。或者，从主菜单中选择“ 导航|导航”。下一个/上一个突出显示的错误 。

    DataGrip会在代码发布之前立即将插入号放置。

  • 配置DataGrip在代码问题之间导航的方式：它可以在所有代码问题之间跳转，也可以跳过次要问题，并且仅在检测到的错误之间导航。右键单击滚动条区域中的代码分析标记，然后从上下文菜单中选择一种可用的导航模式：

    • 要让DataGrip跳过警告，信息和其他次要问题，请选择仅转到高优先级问题 。

    • 要使DataGrip在所有检测到的代码问题之间跳转，请选择转到下一个问题 。

在“结构”视图弹出窗口中找到一个代码元素

您可以使用结构视图弹出窗口在正在处理的文件中找到代码元素。

  1. 要打开结构视图弹出窗口，请按Ctrl+F12 。

  2. 在弹出窗口中，找到您需要的项目。您可以开始为DataGrip输入元素的名称以缩小搜索范围。按Enter返回到编辑器和相应的元素。

    在弹出窗口中，您可以对文件成员进行排序，查看匿名类和继承的成员。

使用面包屑进行导航

您可以使用面包屑浏览源代码，这些面包屑显示当前打开的文件中的视图，别名，字段，表和其他对象的名称。默认情况下，启用面包屑并将其显示在编辑器的底部。

  • 要更改面包屑的位置，请右键单击一个面包屑，在上下文菜单中选择“ 面包屑”和位置首选项。

  • 要编辑面包屑设置， 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|面包屑 。

    清除“ 显示面包屑”选项以在编辑器中隐藏面包屑。

查找线或列

  1. 在编辑器中，按Ctrl+G 。

  2. 在“ 转到行/列”对话框中，指定行号或列号，或同时指定两者，并用：分隔它们，然后单击“ 确定” 。

  3. 如果您不想在编辑器中看到行号， 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|外观并清除显示行号复选框。

为了更快地访问，请将快捷方式分配给“ 显示行号”操作。

查找插入符，编辑的行或编辑的位置

  • 要在编辑器中找到当前插入符号的位置 ，请按Up和Down方向键。

  • 要突出显示您要查找的插入符号处的单词，请选择“ 编辑” |“其他”。查找|从主菜单中的插入符号中找到Word 。

  • 要查看插入符号当前位于什么元素上，请按Alt+Q 。

  • 要查看先前的插入符位置，请按Ctrl+Alt+Left 。要前进，请按Ctrl+Alt+Right 。

  • 要在开始和结束代码块大括号之间切换，请按Ctrl+Shift+M 。

  • 要导航到最后编辑的位置，请按Ctrl+Shift+Backspace 。

移动插入符号

您可以使用不同的动作通过代码移动插入符号。您还可以配置插入字词和在线中断时插入符号应停止的位置。

  • 要将插入符号移动到下一个单词或上一个单词，请按Ctrl+Right要么Ctrl+Left 。

    默认情况下，DataGrip将插入符号移动到当前单词的末尾。使用这些操作时，可以配置插入标记的位置。

    在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般在“ 插入符号移动”部分中，使用“ 按单词移动时”和“换行时”选项来配置插入符号的行为。

  • 要将插入符号向前移动到下一个段落或向后移动到上一个段落，请按Ctrl+Shift+A并搜索将插入符号向前移动段落或将插入符号向后移动段落动作。

    您也可以选择一个文本，然后将插入符号向前或向后移动到一个段落。按Ctrl+Shift+A并搜索带有选择项的将插入符号向前移动段落或带有选择项的将插入符号向前 移动段落 。

    如果需要，可以为这些操作分配快捷方式。有关详细信息，请参考配置键盘快捷键 。

查找最近的文件

您可以使用“ 最近的文件”弹出窗口搜索最近和最近编辑的文件。

  • 要打开带有最近文件列表的“最近文件”弹出窗口，请按Ctrl+E 。

  • 要仅查看最近编辑的文件，请按Ctrl+E再次选择“ 仅显示更改”复选框。

  • 要在弹出窗口中搜索项目，请开始输入搜索查询。DataGrip根据您的搜索显示结果。

查找最近的位置

您还可以使用“ 最近的位置”弹出窗口检查您最近查看或更改的代码。

  • 要打开“ 最近的位置”弹出窗口，请按N/A 。该列表从顶部的最近访问位置开始，并包含代码片段。

  • 在弹出窗口中，使用相同的快捷方式或选中仅显示更改的复选框，以仅查看更改了代码的位置。

  • 要搜索代码段，请在“ 最近的位置”弹出窗口中，开始输入搜索查询。您可以按代码文本，文件名或面包屑搜索。

  • 要从搜索结果中删除位置条目，请按Delete要么Backspace 。

    请注意，删除的位置也会从您通过以下方式访问的条目列表中删除： Ctrl+Alt+Left捷径。

在SQL代码和文件中导航

从引用转到定义

要转到列，表或任何其他对象的定义，请在代码中选择对象名称，然后导航至“ 导航|导航”。宣言 Ctrl+B 。或者，按Ctrl然后单击参考。

从引用转到其DDL定义

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，导航到数据库|一般

  2. 在“ 导航”窗格中，清除“ 在DDL编辑器上首选数据库视图”复选框。

从数据库树视图打开对象的DDL定义

默认情况下，您可以从数据库树视图中打开一个对象的DDL定义（双击该对象）。

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，导航到数据库|一般

  2. 在“ 导航”窗格中，清除“ 在DDL编辑器上优先使用数据编辑器”复选框。

在不同的定义视图之间切换

  1. 将插入符号放在对象上，然后按Alt+F1 。

  2. 从上下文菜单中选择所需的视图。

使用切换器在文件之间切换

使用切换器，您可以在打开的文件，控制台和工具窗口之间切换。

要打开切换器窗口，请按Ctrl+Tab 。不要释放Ctrl键。使用箭头键在“ 切换器”窗口中移动。当您释放Ctrl键，所选项目变为活动状态，并且切换台关闭。

另外，您可以使用Ctrl+Tab列出清单，然后Ctrl+Shift+Tab去上榜。

在MacOS上， Ctrl和箭头键是用于任务控制功能的快捷方式。要使用切换台，请在MacOS系统设置中更改快捷方式。
