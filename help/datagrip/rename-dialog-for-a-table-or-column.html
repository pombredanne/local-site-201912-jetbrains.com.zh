重命名表或列的对话框

重构|改名
Shift+F6

使用此对话框可以重命名表或列。

除了重命名表或列本身之外，DataGrip还可以查找表或列名的用法。如果找到，则您对表或列名所做的更改也可以应用于这些用法。

项目

描述

改名

及其用法

为表或列指定一个新名称。

搜索注释和字符串

如果选中此复选框，DataGrip将在源代码文件的注释和字符串文字中查找表名或列名的出现。

搜索已加载的源

如果选中此复选框，DataGrip将在下载的对象源中查找表或列名称的出现。

SQL脚本

要运行以重命名表或列的语句。如有必要，您可以在此窗格中直接编辑该语句。

重构

执行该语句并立即进行相关更改。

预习

在执行语句之前预览潜在的关联更改。

搜索文字出现

如果启用此选项，DataGrip将在不包含源代码的文件中查找表名或列名的出现。这些可能是文本文件，属性文件，HTML文件，文档文件等。

范围

使用此选项可以设置重命名重构的范围。例如，您可以将重构限制为仅最近更改的文件或仅打开的文件，依此类推。您还可以创建可以共享或保留在本地的自定义范围。
