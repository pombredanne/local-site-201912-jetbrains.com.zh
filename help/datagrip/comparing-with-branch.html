Comparing With Branch

In addition to the common file versions comparison options, the Subversion integration with DataGrip provides a special command that enables you to compare a file from your local working copy with its version in the selected branch.

To compare a file with its version in a specified branch, do the following:

This action is also available in the SVN Repositories browser. Right-click the desired directory and choose the corresponding command from the context menu.

  1. Select the desired file in the Files tool window, or open it in the editor.

  2. From the main VCS menu, or on the context menu of the selection, choose Subversion | Compare with Branch.

  3. In the Compare with Branch popup, select the desired branch. The Compare with Branch dialog in the form of the Differences viewer for files appears.
