查找和替换文件中的文本

DataGrip使您可以在活动的编辑器中查找和替换文本字符串。

搜索当前文件

  1. 在主菜单中，选择“ 编辑” |“选择” 。查找|查找 ，或按Ctrl+F 。搜索窗格显示在活动编辑器的顶部。

  2. 如有必要，请指定搜索选项 。

  3. 在搜索字段中，开始输入搜索字符串：

    键入时，将选择当前光标位置之后搜索字符串的第一个匹配项；其他事件在编辑器中突出显示。此外，匹配的事件在右侧装订线中用条纹标记。

  4. 要搜索多行片段，请单击在搜索框中或按N/A 。

  5. 探索搜索结果 。

替换当前文件

  1. 在主菜单中，选择“ 编辑” |“选择” 。查找|更换或按Ctrl+R 。搜索和替换窗格显示在活动编辑器的顶部。

  2. 如有必要，请指定搜索和替换选项 。

  3. 在搜索字段中，开始输入搜索字符串。键入时，匹配的事件将在编辑器中突出显示，并且“ 替换”弹出对话框在第一个事件出现时打开，建议用空字符串替换当前事件或所有这些事件。

  4. 开始输入替换字符串。

  5. 浏览搜索结果 ，然后使用“替换”对话框中的按钮，根据需要替换出现的位置。请参阅下面的搜索和替换选项 。

处理搜索结果

  • 要启动新搜索，请执行以下一项操作（取决于当前焦点）：

    • 如果编辑器具有焦点，请按Ctrl+F 。

    • 如果搜索字段具有焦点，请按Ctrl+A

    在这两种情况下，都将选择现有的搜索字符串，然后您可以开始输入新的搜索字符串。

  • 要在出现的事件之间跳转，请执行以下一项操作：

    • 按Shift+F3 （跳至上一个选择）或F3 （跳到下一个选择）。

    • 使用要么 搜索窗格中的按钮。

    • 单击装订线条纹。

  • 搜索窗格显示找到的事件数。如果未找到匹配项，搜索窗格将变为红色。

  • 使用最近的搜索历史：在搜索窗格已打开的情况下，单击显示最近条目的列表。

  • 要取消操作并关闭窗格，请按Escape 。

  • 使用多选 （ multiselection ）。例如，如果某个字符串已突出显示为搜索结果，则可以通过单击将该字符串的出现添加到多个选择中Alt+J ，使用从多个选择中删除一个事件Shift+Alt+J ，或使用将所有发现的事件添加到多个选择中Ctrl+Shift+Alt+J 。
