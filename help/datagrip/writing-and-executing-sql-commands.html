SQL语句

在数据库控制台中执行语句

创建数据源时，将自动创建数据库控制台 。有关数据库控制台的更多信息，请参见数据库控制台 。

  1. 在数据库工具窗口（视图|工具窗口| 数据库 ）中，单击数据源。

  2. 打开F4或创建（ Ctrl+Shift+F10 | 新控制台 ）数据库控制台。有关使用控制台的更多信息，请参见使用控制台 。

  3. 键入或粘贴要执行的语句。

  4. 按Ctrl+Enter 。或者，单击执行图标在工具栏上。

    如果有多个语句，请选择是执行所有语句还是执行单个语句。建议列表始终包含用于运行所有语句的项目。

执行参数化语句

如果语句中有参数，则必须在执行语句之前指定参数的值。

要执行参数化的语句，请单击“ 执行”按钮在工具栏上，然后在第二列中输入值。或者，要打开“ 参数”对话框，请单击“ 查看参数”按钮 。

执行一组语句

要执行一组语句，请选择必要的语句，然后单击“ 执行”按钮 。

在控制台中执行所有语句

要在控制台中执行所有语句，请使用Run' '命令（作为Execute命令的替代方法）。

右键单击控制台选项卡或控制台中的任何区域，然后选择“运行” 。

运行'命令有以下限制：

  • 带有参数的语句将被跳过。

  • 检索到的数据SELECT语句未显示。

执行语句的一部分

要执行语句的一部分（例如，子查询），请选择要执行的片段，然后单击“ 执行”按钮 。

查看已执行语句的历史记录

DataGrip存储您已运行的所有语句的历史记录。

要打开“ 历史记录”对话框，请单击或按Ctrl+Alt+E 。

  • 要过滤信息，请开始输入搜索查询。

  • 要将查询从“ 历史记录”对话框粘贴到控制台，请在“ 历史记录”对话框的左窗格中双击查询。

  • 要从历史记录中删除记录，请选择记录，然后按Delete 。

停止运行语句

要终止当前一条或多条语句的执行，请单击“ 断开连接”按钮或按Ctrl+F2 。

将SELECT语句的结果保存到文件中

  1. 右键单击一个SELECT声明。

  2. 选择执行到文件，然后选择输出格式。

  3. 指定文件位置和名称。

将INSERT语句中的数据作为表编辑

  1. 选择INSERT您要编辑的语句。

  2. 右键单击所选内容，然后单击“ 编辑为表格” 。

运行存储过程

存储过程是一组具有指定名称的SQL语句。

  1. 右键单击要执行的存储函数。

  2. 在“ 执行例程”窗口中，键入所有必需的参数值，然后单击“ 确定” 。

启用DBMS_OUTPUT软件包的输出（Oracle）

在Oracle中，DBMS_OUTPUT包使您能够从存储过程，包和触发器发送消息。您可以启用或禁用在输出窗格中显示DBMS_OUTPUT缓冲区的内容。要切换DBMS_OUTPUT选项，请单击Enable SYS.DBMS_OUTPUT按钮在“ 服务”工具窗口的工具栏上。
