从命令行查看文件之间的差异

您可以将DataGrip用作命令行工具来查看文件之间的差异。

  • 在命令行中，运行diff使用以下语法的命令：

    diff path1 path2

所选文件将在“ 差异查看器”中打开。

命令选项如下：

选项

描述

特定于平台的DataGrip启动器。

在macOS和Linux上，建议使用在DataGrip安装期间创建的命令行启动器，或通过选择工具|手动创建一个。在DataGrip主菜单上创建命令行启动器 。有关详细信息，请参阅在macOS / Linux上启用命令行启动器 。

在Windows上，建议将DataGrip可执行文件的路径添加到Path环境变量。

path1

文件本地副本的完整路径。

path2

文件的存储库版本副本的完整路径。

例子

句法
datagrip.exe差异路径1路径2
例
datagrip.exe差异C：\ MyProject \ Readme.md C：\ MyProject \ Readme.md.bak
句法
datagrip diff path1 path2
例
datagrip diff〜/ MyProject / Readme.md〜/ MyProject / Readme.md.bak
