从命令行打开文件

您可以从命令行打开任意文件，文件夹或项目以在DataGrip中进行编辑，还可以选择指定打开文件后插入符号应位于的位置。

  • 在命令行中，运行以下命令：

    [--line ]路径

    另外，您可以使用以下语法：

    路径：

命令选项如下：

选项

描述

特定于平台的DataGrip启动器。

在macOS和Linux上，建议使用在DataGrip安装期间创建的命令行启动器，或通过选择工具|手动创建一个。在DataGrip主菜单上创建命令行启动器 。有关详细信息，请参阅在macOS / Linux上启用命令行启动器 。

在Windows上，建议将DataGrip可执行文件的路径添加到Path环境变量。

path

要打开的文件或文件夹的路径。

  • 如果文件或文件夹属于现有项目，则将打开该项目。

  • 如果该文件不属于任何项目，则将自动为此文件创建一个临时项目。

  • 如果该文件夹不属于任何项目，则将在此文件夹内创建一个新项目。

--line

（可选）打开文件后，脱字符号应位于其上的行号。

例子

句法
datagrip.exe-行路径
替代语法
datagrip.exe路径：
例子
datagrip.exe-第3行C：\ MyProject \ scripts \ numbers.js datagrip.exe C：\ MyProject \ scripts \ numbers.js：3
句法
datagrip-行路径
替代语法
datagrip路径：
例
datagrip-第3行〜/ MyProject / scripts / numbers.js datagrip〜/ MyProject / scripts / numbers.js：3
