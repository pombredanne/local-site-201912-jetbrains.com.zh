管理数据源

数据源是您的应用程序的数据位置。在DataGrip中，您可以定义以下数据源：

  • 数据库数据源：包含数据的可操作数据库（请参阅连接到数据库 ）。

  • 数据定义语言（DDL）文件：包含数据定义语言语句的SQL文件（请参阅添加DDL数据源 ）。

创建数据源

从数据库文件创建数据库数据源

  1. 如果数据库文件位于DataGrip工作文件夹中，请打开“ 文件”工具窗口。否则，请打开文件管理器。

  2. 打开数据库工具窗口（“ 视图” |“工具窗口” |“数据库” ）。

  3. 选择要用作数据源的文件，然后将它们拖到“ 数据库”工具窗口中。为每个文件创建一个单独的数据源。

  4. 如果没有必需的数据库驱动程序文件，请下载它们。请点击在数据库工具窗口的工具栏上。或者，从上下文菜单中选择“ 属性 ”。

  5. 在“ 数据源和驱动程序”窗口中，单击“ 下载”链接。

  6. 为确保成功连接到数据源，请单击“ 测试连接” ，然后单击“ 确定” 。

创建一个DDL数据源

  1. 打开“ 数据库”工具窗口（“ 视图” |“工具窗口” |“数据库” ）。

  2. 单击数据源属性图标（ ），然后选择DDL数据源 。

  3. 指定数据源名称。

  4. 在“ DDL文件”表单中，单击“ 添加”图标（ ），然后选择必要的SQL文件。

另外，您可以将DDL文件从“ 文件”工具窗口或文件管理器拖到“ 数据库”工具窗口。

导出和导入数据源设置

导出数据源设置

  1. 单击文件|导出设置 。

  2. 在“ 导出设置”窗口中，单击全选 。

  3. 在“ 导出设置”窗口中，选择“ 数据库：驱动程序” ，“ 数据库：设置 ”和“ 数据库：全局源” 。您必须至少有一个全局数据源才能在导出列表中拥有数据库：全局源 。

  4. 在“ 将设置导出到”字段中，键入要在其中存储settings.zip文件的文件夹的路径。

settings.zip文件仅包含全局级别可用的数据源。有关如何更改级别的更多信息，请参见将项目级别更改为global 。

导入数据源设置

  1. 单击文件|导入设置 。

  2. 在“ 导入文件位置”窗口中，找到导出的settings.zip文件，然后双击它。

  3. 在“ 选择要导入的组件”窗口中，选择“ 数据库：驱动程序” ，“ 数据库：设置 ”和“ 数据库：全局源” 。

  4. 单击确定 。

  5. 在“ 需要重新启动”窗口中，单击“ 确定”以重新启动IDE。

  6. 打开“ 数据源和驱动程序”窗口，在数据源属性中输入必要的连接凭据。如果数据源在本地或网络驱动器上，请确保此数据源的路径正确。

  7. 要确保成功连接到数据源，请单击“ 测试连接” 。

复制和共享数据源

复制数据源

如果需要在另一个项目中使用相同的数据源，但是对它有不同的要求，例如它的可访问性级别（项目或全局），用户，数据库连接和架构设置等，则可以复制数据源。

  1. 单击文件|数据源 N/A 。

  2. 在“ 数据源和驱动程序”窗口中，选择要复制的数据源。

  3. 单击复制图标，或按Ctrl+D 。

在项目之间共享数据源

DataGrip中的数据源有两个级别：项目级别和全局级别。项目级别意味着您只能从最初创建该数据源的项目中访问数据源。在全球范围内，可为其他项目提供数据源。默认情况下，数据源在项目级别可用。

您不能将全局数据源附加到在其他版本的DataGrip中打开的项目。

要在IDE内的不同项目之间共享数据源，您需要将数据源移到全局级别。

  1. 单击文件|数据来源 。

  2. 在“数据源和驱动程序”窗口中，单击“创建全局图标”（ ），以使数据源在全局一级可用。

在不同的IDE实例之间共享数据源

您可以将有关数据源的信息以XML格式复制到剪贴板。生成的XML不包含密码信息。

您可以将XML代码段发送到另一个IDE（通过Messenger或通过电子邮件）并导入。

  1. 在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中，选择要复制的数据源。

  2. 右键单击所选内容。

  3. 单击数据库工具，然后选择将数据源复制到剪贴板 。

  4. 在另一个工作站上，单击“ 查看” |“其他”。工具窗口|数据库 。

  5. 在数据库工具窗口中，单击新建图标 ，选择从剪贴板导入 。

DataGrip将密码存储在Windows的KeePass存储中以及Linux和MacOS的本机钥匙串中。您可以将IDE配置为将密码存储在KeePass中（ 文件|设置|外观和行为|系统设置|密码 ）。

您可以通过将datasources.xml复制到所需位置来共享项目数据源。默认情况下， datasources.xml与其他配置文件一起存储在项目文件夹中。要查看项目文件夹的路径，请单击“ 文件” |“文件”。打开最近|管理项目 。

自定义数据源

配置数据源设置

  1. 在“ 数据库”工具窗口（“ 视图” |“工具”窗口|“数据库” ）中，选择要更改的数据源。

  2. 单击数据源属性图标（ ）（或按N/A ）。

组数据源

如果您有许多不同的数据库，则可以在“ 数据库”工具窗口（“ 视图” |“工具窗口” |“数据库” ）中对它们的数据源进行分组。要对数据源进行分组，请选择所需的数据源，然后按F6 ，或从上下文菜单中选择“ 移至组 ”。

或者，如果要创建一个新组，只需将一个数据源拖到另一个上即可。如果要将数据源放入现有组中，请将其拖到那里。

在“ 移至组”菜单中，以下选项可用于数据源组：

  • 为数据源创建一个新组。

  • 将数据源移动到根列表。此选项从组中删除数据源。

  • 将数据源移动到现有组。
