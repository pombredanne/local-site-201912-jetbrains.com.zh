生成表和视图的Java实体类

要使用JPA将点对象存储在数据库中，必须定义一个实体类。JPA实体类是一个Java类，被注释为具有表示数据库中对象的能力（POJO（普通Java对象）类）。

您可以选择必要的表和视图并生成相应的JAVA文件。

  1. 右键单击要为其生成Java实体类的数据库对象，然后导航到“ 脚本扩展|脚本扩展”。生成POJOs.groovy 。

  2. 在文件浏览器中，指定要存储JAVA类文件的目录。
