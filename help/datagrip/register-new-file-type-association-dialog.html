注册新文件类型关联对话框

当您尝试在编辑器中打开文件时，对话框打开，但是DataGrip无法通过文件扩展名识别其类型。使用对话框将未知文件扩展名与现有文件类型相关联。

项目

描述

打开匹配文件
在DataGrip中

选择此选项后，DataGrip会将要打开的文件类型视为已识别文件类型之一。从下面的列表中选择相关的类型，该类型显示DataGrip识别的所有文件类型。

文件模式

在此字段中，指定要与所选文件类型关联的文件模式。默认情况下，该字段显示以下模式： *. 。

在关联的应用程序中打开匹配文件

选择此选项后，DataGrip会尝试使用其本机应用程序打开所选文件（如果此应用程序可用）。
