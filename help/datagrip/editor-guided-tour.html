DataGrip 2019.2 Help

Code editor

Code editor, or simply the editor, lets you write your code and perform related tasks.

Editor tabs. Each file or console opens on its own tab. Use Alt+Right and Alt+Left to switch between the tabs, and Ctrl+F4 for closing a tab. More commands for working with the tabs are available in their context menus.

Editing area. In addition to typing, use the commands of the context menu and also of the following menus: Code, Edit, Navigate and Refactor.

indicates that DataGrip has suggestions for the current cursor position. Click or press Alt+Enter to see those suggestions.

Left margin is used to provide additional info. Look at the commands of the associated context menu to see what info can be shown.

The left margin also holds code folding markers that let you collapse ( and ) and expand () the corresponding code blocks.

Code validation marker reflects the quality of your code. Point to the marker to see the summary info.

Navigation stripes let you switch to "interesting" code fragments including the ones that need your attention. Point to a stripe to see a summary for corresponding fragment, e.g. a problem description. Click a stripe to switch to the corresponding fragment. Right-click the scroll bar or the area where it may appear to open the corresponding customization menu.
