代码编辑器

代码编辑器（或简称为编辑器）使您可以编写代码并执行相关任务。

编辑器标签。 每个文件或控制台都在其自己的选项卡上打开。使用Alt+Right和Alt+Left在标签之间切换，以及Ctrl+F4用于关闭标签页。在其上下文菜单中提供了用于使用选项卡的更多命令。

编辑区。 除了键入之外，还可以使用上下文菜单以及以下菜单中的命令： Code ， Edit ， Navigate和Refactor 。

表示DataGrip对当前光标位置有建议。请点击或按Alt+Enter看看那些建议。

左边距用于提供其他信息。查看关联的上下文菜单的命令以查看可以显示哪些信息。

左边距还包含可折叠的代码折叠标记（ 和 ）并展开（ ）相应的代码块。

代码验证标记反映了代码的质量。指向标记以查看摘要信息。

导航条使您可以切换到“有趣的”代码片段，包括需要注意的代码片段。指向条纹以查看相应片段的摘要，例如问题描述。单击条纹以切换到相应的片段。右键单击滚动条或滚动条可能出现的区域，以打开相应的自定义菜单。
