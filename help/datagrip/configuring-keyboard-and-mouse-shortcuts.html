配置键盘快捷键

DataGrip包含预定义的键盘映射，可让您自定义常用的快捷方式。

要查看键盘映射配置，请打开“ 设置/首选项”对话框Ctrl+Alt+S ，然后选择Keymap 。

DataGrip会根据您的环境自动选择预定义的键盘映射。确保它与您正在使用的操作系统匹配，或者选择与您习惯使用的另一个IDE或编辑器（例如Eclipse或NetBeans）中的快捷方式匹配的操作系统。

您无法自定义预定义的键盘映射。修改任何快捷方式时，DataGrip都会创建当前所选键盘映射的副本，您可以对其进行配置。请点击复制选定的键盘映射，重命名，删除或恢复为默认值。

自定义键映射不是其父键映射的完整副本。它从父键映射继承未修改的快捷键，并仅定义已更改的快捷键。有关键盘映射文件的信息，请参见用户定义键盘映射的位置 。

键盘映射表本身是带有相应的键盘和鼠标快捷键以及缩写的操作列表。要按名称查找动作，请在搜索字段中将其键入。如果您知道操作的快捷方式，请单击然后在“ 查找快捷方式”对话框中按组合键 。

添加键盘快捷键

  1. 在“ 设置/首选项”对话框的“ 键盘映射”页面上Ctrl+Alt+S ，右键单击一个动作，然后选择添加键盘快捷方式 。

  2. 在“ 键盘快捷键”对话框中，按必要的组合键。

  3. （可选）选中“ 第二笔画”复选框，以定义具有两个连续键组合的复杂快捷键。

  4. 单击确定以保存快捷方式。

    使用鼠标指针很重要，因为按下Enter而在“ 键盘快捷键”对话框中，则被视为您要用作快捷键的组合键。

您按下的键组合会显示在“ 键盘快捷键”对话框中，如果与现有的快捷键冲突，也会显示警告。

添加鼠标快捷方式

  1. 在“ 设置/首选项”对话框的“ 键盘映射”页面上Ctrl+Alt+S ，右键单击一个动作，然后选择添加鼠标快捷方式 。

  2. 在“ 鼠标快捷方式”对话框中，将鼠标指针移动到中心区域，然后根据需要单击或滚动。

  3. 单击确定以保存快捷方式。

所执行的鼠标操作将显示在“ 鼠标快捷方式”对话框中，并在与现有快捷方式冲突时显示警告。

加上缩写

缩写词可用于快速查找动作而无需快捷键。例如，您可以按Ctrl+Shift+A并键入“ 跳转到颜色和字体”操作的名称，以快速修改当前插入符号位置下的元素的颜色和字体设置。如果为此操作指定缩写（例如JCF ），则可以键入它而不是完整的操作名称。

  1. 在“ 设置/首选项”对话框的“ 键盘映射”页面上Ctrl+Alt+S ，右键单击一个动作，然后选择添加缩写 。

  2. 在“ 缩写”对话框中，键入所需的缩写，然后单击“ 确定” 。

用户定义的键盘映射的位置

所有用户定义的按键映射都存储在DataGrip配置目录中keymaps子目录下的单独配置文件中：

句法
％HOMEPATH％\。 \ config \ keymaps
例
C：\ Users \ JohnS \。DataGrip2019.2 \ config \ keymaps
句法
〜/ Library / Preferences / / keymaps
例
〜/ Library / Preferences / DataGrip2019.2 / keymaps
句法
〜/。 / config / keymaps
例
〜/。DataGrip2019.2 / config / keymaps

每个键盘映射文件仅包含相对于父键盘映射的差异。

您可以使用这些文件与团队成员共享您的自定义键盘映射。要导出您的键盘映射，请复制相应的文件，然后通过将其放置在另一个DataGrip安装上的键盘映射目录中来导入它。

与全局OS快捷方式冲突

预定义的键映射未涵盖所有可能的平台，版本和配置。某些快捷方式可能与全局系统操作和第三方软件的快捷方式冲突。要解决这些冲突，您可以重新分配或禁用冲突的快捷方式。

试用您使用的组合键并进行必要的调整。另外，请确保已在系统上启用功能键，并检查以下默认快捷方式：
