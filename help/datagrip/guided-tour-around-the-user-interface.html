浏览用户界面

在DataGrip中打开项目时，默认用户界面如下所示：

根据插件集和配置设置的不同，IDE的外观和行为可能会有所不同。

编辑

焦点：Escape

使用编辑器读取和编写代码。有关更多信息，请参见编辑器基础 。

导航栏

焦点：Alt+Home

显示/隐藏： 查看|外观|导航栏

顶部的导航栏是数据库和文件工具窗口的快速替代方法，您可以在其中导航数据库结构并打开文件进行编辑。

默认情况下，带有用于打开和保存文件，撤消和重做操作的按钮的主工具栏是隐藏的。要显示它，请选择查看|外观|工具栏

状态栏

显示/隐藏： 查看|外观|状态栏

当您将鼠标指针悬停在状态栏上时，底部的状态栏显示事件消息和动作说明。它还指示整个项目和IDE的状态，并提供对各种设置的访问。下表列出了状态栏上显示的默认图标和元素。根据插件集和配置设置的不同，除默认元素外，还有许多其他元素。

工具视窗

显示/隐藏： 查看|工具视窗

工具窗口提供补充编辑代码的功能。例如，“ 数据库”工具窗口显示您的数据库结构，而“ 服务”工具窗口显示您运行SQL代码时的输出。

默认情况下，工具窗口停靠在主窗口的侧面和底部。您可以根据需要排列它们，取消停靠，调整大小，隐藏等等。右键单击工具窗口的标题或单击标题中的排列选项。

您可以分配快捷方式以快速访问您经常使用的工具窗口。其中一些默认情况下具有快捷方式。例如，要打开数据库工具窗口，请按Alt+1 ，然后打开“ 服务”工具窗口，请按N/A 。要从编辑器跳到最后一个活动工具窗口，请按F12 。

上下文菜单

您可以右键单击界面的各个元素，以查看当前上下文中可用的操作。例如，在数据库工具窗口中右键单击数据库对象以获取与该数据库对象相关的操作，或者在编辑器中右键单击以查看适用于当前代码片段的操作。

这些操作大多数都可以从屏幕顶部或主窗口的主菜单中执行。带有快捷方式的操作在操作名称旁边显示快捷方式。

弹出菜单

弹出菜单可快速访问与当前上下文有关的操作。例如，按Alt+Insert在编辑器中打开“ 生成”弹出窗口，用于基于上下文生成代码结构。

您可以使用常用操作的快速列表来创建自定义弹出菜单。例如，您可以按Ctrl+Shift+Alt+T打开Refactor这个弹出窗口，其中包含与重构相关的预定义快速操作列表。
