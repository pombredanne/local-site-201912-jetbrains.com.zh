支持的应用

dotMemory支持对以下类型的.NET *应用程序进行性能分析：

  • 独立应用程序

  • .NET Core应用程序（独立且依赖于框架的部署）

  • 在ASP.NET开发服务器，IIS和IIS Express下运行的ASP.NET WebForms和ASP.NET MVC Web应用程序

  • Windows服务

  • WCF服务库

  • WinRT桌面应用程序（仅CoreCLR）

  • 任意.NET进程

* .NET Framework 1.0-4.7，.NET Core 1.0-3.0。附件仅适用于.NET Framework 4.0（或更高版本）或.NET Core 3.0 Preview 7（或更高版本）进程。
