分析对象集

对象集是按特定条件分组的多个对象。例如，第2代中的所有对象，或某个其他特定对象保留的所有对象，或特定调用分配的所有对象，等等。
对象集是您调查的主要主题。为了帮助您分析对象集，dotMemory可以在本节中介绍的多种视图中显示它们：

  • 按类型分组
  • 按统治者分组
  • 按类似保留率分组
  • 按实例分组
  • 按创建堆栈跟踪分组
    • 按通话树分组
    • 按通话树分组（冰柱图）
    • 按追溯分组
  • 按世代分组
  • 按最短路径分组

选择一个视图

要为对象集选择某个视图，请在视图列表中单击相应的项目。
