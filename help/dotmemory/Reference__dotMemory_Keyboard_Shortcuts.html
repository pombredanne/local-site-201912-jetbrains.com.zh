dotMemory 2019.2帮助

键盘快捷键

工作空间

捷径

描述

Ctrl+O

从文件导入工作区

Ctrl+S

将当前工作空间导出到文件

Ctrl+Q

关闭当前工作区

Ctrl+W

切换到打开的工作区

Ctrl+H

切换到首页标签

分析

捷径

描述

Ctrl+T

将当前分析（包括分析路径）复制到新选项卡

Ctrl+Alt+T

在新选项卡中开始新分析（分析路径将不会被复制）

快照

捷径

描述

Ctrl+Alt+C

将快照添加到比较区域。仅当您在“ 分析概述”页面上选择快照时，此快捷方式才有效。

对象集和实例

捷径

描述

Ctrl+A

在列表中选择所有对象集/实例。适用于所有具有列表的视图。

Ctrl+C

将视图数据复制到剪贴板。在“ 发送参考”视图中，您可以选择是否复制有关某个对象的数据Ctrl+C或该对象的完整参考路径（ Ctrl+Shift+C ）

Ctrl+L

导航到Visual Studio中的类型声明。在导航到类型声明中了解更多信息

Ctrl+Shift+F

将焦点切换到“ 过滤器”字段。适用于所有具有该字段的视图。

Ctrl+Shift+0..9

用0到9之间的数字标记一个实例 。在“ 实例”视图中工作，所有视图都用于分析实例 。

Ctrl+M

用随机颜色标记实例 。在“ 实例”视图中工作，所有视图都用于分析实例 。

Ctrl+Shift+M

用自定义标记标记实例 。在“ 实例”视图中工作，所有视图都用于分析实例 。

呼叫树

捷径

描述

Ctrl+F

在“ 调用树”视图中找到一个功能。

Ctrl+C

将呼叫树的一部分或追溯轨迹复制到剪贴板。
