dotMemory 2019.2 Help

Keyboard Shortcuts

Workspaces

Shortcut

Description

Ctrl+O

Import a workspace from a file

Ctrl+S

Export current workspace to a file

Ctrl+Q

Close current workspace

Ctrl+W

Switch to the opened workspace

Ctrl+H

Switch to the Home tab

Analysis

Shortcut

Description

Ctrl+T

Copy current analysis including the analysis path to a new tab

Ctrl+Alt+T

Start a new analysis in a new tab (the analysis path will not be copied)

Snapshots

Shortcut

Description

Ctrl+Alt+C

Add a snapshot to the comparison area. This shortcut works only in case you select the snapshot on the Analysis Overview page.

Object sets and instances

Shortcut

Description

Ctrl+A

Select all object sets/instances in a list. Works on all views that have lists.

Ctrl+C

Copy view data to clipboard. In the Outgoing References view, you can choose whether to copy data about a certain object Ctrl+C or full reference path to this object (Ctrl+Shift+C)

Ctrl+L

Navigate to the type declaration in Visual Studio. Learn more in Navigating to Type Declaration

Ctrl+Shift+F

Switch focus to the Filter field. Works on all views that have this field.

Ctrl+Shift+0..9

Mark an instance with a number from 0 to 9. Works in the Instances view and all views used to analyze instances.

Ctrl+M

Mark an instance with a random color. Works in the Instances view and all views used to analyze instances.

Ctrl+Shift+M

Mark an instance with a custom mark. Works in the Instances view and all views used to analyze instances.

Call Tree

Shortcut

Description

Ctrl+F

Find a function in the Call Tree view.

Ctrl+C

Copy a part of call tree or back traces to clipboard.
