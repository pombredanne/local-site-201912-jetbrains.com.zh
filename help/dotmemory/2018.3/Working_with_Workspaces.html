Working with Workspaces

To store the results of your investigations, dotMemory uses workspaces. A workspace, as the name suggests, saves everything you see on the screen: all open analyses including all collected snapshots and analyses paths. In other words, it saves the current state of dotMemory.

After the profiling session ends, the workspace is saved automatically to a default location (you may change it anytime). As workspace files can be very large, we recommend that you automate workspace files cleanup by setting up workspace auto-deletion.

In case you want to copy a workspace to another computer, you can save it as a single archive.
