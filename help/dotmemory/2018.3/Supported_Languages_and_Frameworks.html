Supported Applications

dotMemory supports profiling of the following types of .NET* applications:

  • Standalone applications

  • .NET Core applications (both self-contained and framework-dependent deployments)

  • ASP.NET WebForms and ASP.NET MVC web applications running under ASP.NET Development Server, IIS, and IIS Express

  • Windows services

  • WCF service libraries

  • WinRT desktop applications (CoreCLR only)

  • Silverlight 5 applications

  • Arbitrary .NET processes

* .NET Framework 1.0 to 4.6
