在系统模块的折叠选项卡中， 创建过滤器 ，以限制快照中过滤的功能数量。

项目

描述

从以下模块过滤功能

在该区域中，指定应过滤哪些模块的功能。默认情况下，有多个包含多个模块的组。这些模块无法删除，但可以禁用。
请注意， 模块不等于Assembly 。模块是一个包含可以由CLR执行的内容的文件，而汇编是一组代码模块和其他实现功能单元的资源。尽管在大多数情况下，程序集名称等于不带扩展名的模块名称，但是有时，程序集的功能可以在多个模块中实现。

加

单击此按钮可以从可用框架之一添加新模块。

去掉

单击此按钮删除选定的模块。
