dotTrace 2019.2帮助

键盘快捷键

快照

按键或按键组合

描述

Ctrl+O

打开“ 打开”对话框，以便您可以选择并打开现有快照之一

Ctrl+S

保存当前快照

标签

按键或按键组合

描述

Ctrl+T

在新选项卡中打开选定的节点

Ctrl+Shift+T

在新选项卡中打开选定的节点；如果当前选项卡中多次出现某个功能，则将它们合并

Ctrl+Alt+T

打开当前班级的所有公开成员

Ctrl+F4

关闭活动标签

观看次数

按键或按键组合

描述

Ctrl+Alt+C

切换到呼叫树视图

Ctrl+Alt+P

切换到普通列表视图

Ctrl+Alt+H

切换到热点视图

Ctrl+Alt+B

切换到Back Traces视图

Alt+F2

显示“ 源视图”视图

剪贴板

按键或按键组合

描述

Ctrl+C

将功能签名复制到剪贴板

Ctrl+Shift+C

将子树复制到剪贴板

导航

按键或按键组合

描述

Ctrl+Tab

转到下一个标签

Ctrl+Shift+Tab

转到上一个标签

Ctrl+Right

转到下一个未过滤的函数调用（“下一个我的代码调用”）

Ctrl+Left

转到上一个未过滤的函数调用（“上一个我的代码调用”）

Ctrl+Shift+Right

转到下一个重要电话

Ctrl+Shift+Left

转到上一个重要电话

Ctrl+Down

转到下一个兄弟姐妹

Ctrl+Up

转到上一个同级

Ctrl+Home

转到子树根

搜索

按键或按键组合

描述

Ctrl+F

显示搜索字段，以便您可以找到类型或功能

F3

开始搜索所选功能或转到下一个功能

Shift+F3

转到上一个功能出现

Ctrl+Shift+F3

转到下一个非递归函数出现

Esc

删除突出显示和导航栏

筛选

按键或按键组合

描述

Ctrl+Space

切换子树折叠

Del

优化当前功能的出现

Shift+Del

优化功能的所有实例

Ctrl+Shift+Alt+Del

恢复原始值

Ctrl+Alt+Space

折叠或展开被叫方（在“ 普通列表”中可用）

性质

按键或按键组合

描述

Ctrl+Q

显示“ 属性”对话框，其中包含有关所选功能的信息

书签

按键或按键组合

描述

Ctrl+K,Ctrl+K

设置一个新书签

Ctrl+K,Ctrl+N

导航到下一个书签

Ctrl+K,Ctrl+P

导航到上一个书签

Ctrl+K,Ctrl+W

打开书签对话框

杂

按键或按键组合

描述

Ctrl+Shift+Alt+L

显示日志

Alt+F4

出口
