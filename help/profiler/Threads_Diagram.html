螺纹图

线程图可视化了线程活动和过滤的时间间隔（如果应用了过滤器）。使用该图选择分析所需的线程和时间间隔。

ID
线程ID。

*仅适用于托管线程。

类型
线程名称*或线程类型（如果线程没有名称）。

dotTrace区分以下线程类型：

  • 主要
    启动应用程序的线程。

  • 线程池
    由CLR线程池创建的线程。

  • CLR工人
    由CLR创建的工作线程。

  • 垃圾收集
    执行后台垃圾回收的线程。

  • 终结器
    执行对象完成的线程。

  • 本机
    非托管系统线程。

ms / MB /事件
过滤时间/分配的内存/事件数。例如，如果选择了“未选择”事件，并且未应用其他过滤器，则这将是整个线程寿命。例如，如果您在“ 线程状态”过滤器中选择“ 等待 ”，则此时间将是所有“ 等待时间”间隔的总和。

％
相对于所有选定时间间隔/已分配内存/事件数的过滤时间/已分配内存/事件数的百分比。

线程活动时间表
时间线区分三种线程状态：运行，等待，就绪（与线程状态过滤器相对应）。应用任何过滤器后，时间轴将突出显示已过滤的时间间隔或点事件。
线程时间轴还可以在智能工具提示中显示有关特定时间点的活动的详细信息。在下面了解有关工具提示的更多信息。

智能工具提示

独立查看器和Visual Studio查看器中的线程图都支持智能工具提示。将鼠标悬停在时间轴上的特定时间点上，将显示一个包含当前线程状态，执行的方法以及其他信息的工具提示。
工具提示是上下文相关的。这意味着，如果应用了某些过滤器，则工具提示将包含有关已过滤事件的其他数据。例如，当选择“ 文件操作”过滤器时，工具提示还会显示有关文件操作，文件名和数据大小的信息。

选择线程

选择线程

单击所需的线程。

选择一个或多个线程后，将按线程过滤。所有其他过滤器将仅显示所选线程的数据。

使用它仅分析特定线程。

在某些情况下，选择特定类型的所有线程可能很方便。

按类型选择线程

  1. 打开属于您要选择的类型的任何线程的上下文菜单。

  2. 在上下文菜单中，选择“ 选择所有线程” 。

选择时间间隔和线程

选择时间间隔

将鼠标拖动到所需的时间间隔并按住鼠标左键进行穿线。

此后，将应用“ 选定时间间隔和线程”过滤器。所有其他过滤器将仅显示所选时间范围和线程的数据。

隐藏线程

为了简化分析，您可以隐藏当前不感兴趣的线程（例如，在选定的时间间隔内不执行任何工作）。

隐藏特定线程

  1. 打开要隐藏的线程的上下文菜单。

  2. 在上下文菜单中，选择“ 隐藏线程＃” 。

隐藏（非）选定的线程

  1. 选择线程。

  2. 在上下文菜单中，选择“ 隐藏选定的线程”或“ 隐藏未选定的线程” 。

按类型隐藏线程

  1. 打开属于要隐藏类型的任何线程的上下文菜单。

  2. 在上下文菜单中，选择“ 隐藏所有线程” 。

排序线程

默认情况下，线程列表按其创建顺序排序。通过单击列标题，您可以按其他条件对线程进行排序： ID ， Type ， ms / MB / events ， ％ 。当前用于排序的列标有标志。

在某些情况下，将某个线程放在列表顶部可能会很方便，而与排序顺序无关。使用线程上下文菜单中的“ 移至顶部”选项将任何线程移至顶部。

缩放

放大/缩小图表

请执行以下任一操作：

  • 使用鼠标滚轮。

  • 按Ctrl +按住鼠标左键拖动鼠标。

缩放图表后，将应用“ 可见时间间隔”过滤器。所有其他过滤器将仅显示当前可见间隔的数据。

请注意，“ 选定时间间隔”过滤器始终比“ 可见时间间隔”过滤器具有更高的优先级。

如果在图表上选择了时间间隔，则可以放大到该时间间隔。

放大选定的时间间隔

按Ctrl+O或在上下文菜单中选择“ 缩放到所选内容 ”。

此后，选定的时间范围将显示整个图表。
