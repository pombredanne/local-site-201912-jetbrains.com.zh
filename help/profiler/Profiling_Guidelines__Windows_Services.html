Windows Services

dotTrace allows you to profile any managed Windows service. Note that dotTrace must be run on behalf of administrator.

In order to profile a Windows Service, you should specify the following parameters:

  • Service
    Name of the service you are going to profile. dotTrace scans the installed services automatically.

If Advanced is selected, you are able to additionally specify the following parameters:

  • Arguments
    Optional arguments passed to the service.

  • Start dependent services
    Select this checkbox if the service requires other services to be started.

  • Use environment variables
    Enables you to run the profiled application with specific environment variables. You can specify any number of variables using the list below the option. Note that each variable must be specified on a new line.

After you run profiling, the profiled service will be restarted.
