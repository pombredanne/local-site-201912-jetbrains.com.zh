Application Options

Application options depend on a type of the application you are going to profile. dotTrace supports the following types of applications:

  • Standalone Applications
  • .NET Core Applications
  • Web Applications Using WebDev Server
  • Web Applications Using IIS
  • Web Applications Using IIS Express
  • Windows Services
  • WCF Services
  • Silverlight Applications
  • WinRT Applications
  • .NET Processes
