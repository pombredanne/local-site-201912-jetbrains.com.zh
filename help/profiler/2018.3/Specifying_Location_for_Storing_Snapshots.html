Specifying Location for Storing Snapshots

By default, snapshots are automatically saved to C:\Users\[username]\AppData\Local\JetBrains\Shared\vAny\Sessions. You can change this location to any other folder.

To specify new location for storing snapshots

  1. On the left panel of the dotTrace Home window, select the Preferences tab.

  2. In the Snapshot files location parameter, specify a new location.

  3. Click Save Changes.
