Web Proxy Settings

Performance Viewer | View | Options | Environment | Web Proxy Settings
Timeline Viewer | View | Options | Environment | Web Proxy Settings

In this page of dotTrace options, you can adjust Web proxy settings for dotTrace's Internet activities. By default, dotTrace follows your system settings and if any proxy server is defined in your system, dotTrace will use it too for fetching sources from symbol servers, verifying license information, detecting updates, etc.

Item

Description

Use system Web proxy settings when downloading from Internet

You can clear this check box to override system Web proxy settings for dotTrace's Internet connection.

Edit

This button appears when you choose to use system proxy server settings.
By clicking on this button you open the Internet Properties dialog, where you can change system Web proxy settings.
