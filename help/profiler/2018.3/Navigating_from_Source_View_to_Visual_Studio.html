Navigating from Source View to Visual Studio

You can navigate right to the source code in Visual Studio from Timeline Viewer.

To navigate from Source View to the source code in Visual Studio

  1. Select a method in Call Tree.

  2. Open Source View.

  3. Click Open in Visual Studio in Source View.

    This functionality is not available until the solution that contains the source code is opened in Visual Studio.
