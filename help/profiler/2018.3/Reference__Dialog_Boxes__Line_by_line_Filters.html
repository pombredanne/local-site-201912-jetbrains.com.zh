Profiling Filters

dotTrace Home window | New Session | [application type] | [line-by-line profiling type] | Edit Filters

Using the Profiling Filters dialog you can create filters and narrow down the list of functions that should be profiled line by line.

Item

Description

Add

Click this button to add a new filter.

Edit

Click this button to edit a selected filter from the list.

Remove

Click this button to remove a selected filter from the list.
