Item

Description

Download PDB files from:

Click one of the options to enable downloading PDB and source files from one of the particular servers:

  • Microsoft Reference Source Server

  • Custom source server

For the last option, type source server address where dotTrace should look for PDB and source files.

Otherwise, click Don't use symbol server.

Download sources if possible

Select this check box if you want dotTrace to try and download source files when required. If this check box is not selected, dotTrace will never attempt to download source files.
