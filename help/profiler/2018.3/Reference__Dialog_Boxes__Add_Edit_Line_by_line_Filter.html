Add/Edit Line-by-line Filter

dotTrace Home window | New Session | [application type] | [line-by-line profiling type] | Edit Filters | Add/Edit

The window is used to create and edit method filters for the line-by-line profiling.

Item

Description

Profile code in

Specify assembly, class, or method which must be profiled.

Do not profile code in

Specify assembly, class, or method which must not be profiled.

Do not profile code marked with attribute

The code with the specified attribute will not be profiled.
