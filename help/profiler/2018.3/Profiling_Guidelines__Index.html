Working with Standalone dotTrace

Standalone dotTrace is ideal for applications where the original project or source code is not always available or when you want to run dotTrace without having Visual Studio installed or launched. dotTrace installs in the Windows Start Menu for easy access.

After you launch dotTrace as a standalone application, the dotTrace Home window opens allowing you to configure and start a new profiling session.

This chapter contains details on profiling steps that can be performed using the dotTrace Home window:

  • Starting the profiling session on a local or remote computer.

    You can also run profiling using one of the recent profiling configurations.

  • Working with collected snapshots.
