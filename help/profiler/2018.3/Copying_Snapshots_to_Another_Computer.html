Copying Snapshots to Another Computer

Snapshots are stored on the disk as a number of separate files of various formats. To simplify copying snapshots to another computer, dotTrace introduces the export functionality. This allows you to save a snapshot as a single archive applicable for further distribution.

To export a snapshot to a file

  1. In the dotTrace Home window, do one of the following:

      1. On the left panel, choose Recent | Snapshots.

      2. In the list of snapshots, find a desired snapshot and click the corresponding Export to File button.

      1. On the left panel, choose Recent | Launches.

      2. Find a profiling launch configuration which was used to collect a desired snapshot.

      3. Expand the profiling launch configuration by clicking >.

      4. In the list of snapshots, find a desired snapshot and click the corresponding Export to File button.

  2. Select a folder where you want a snapshot archive to be saved and click OK.

To open the exported snapshot on another computer with installed dotTrace

  • Double-click the snapshot file.
    This will open the snapshot in the corresponding dotTrace viewer.
