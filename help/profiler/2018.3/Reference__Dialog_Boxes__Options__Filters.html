Folding

Performance Viewer | View | Options | Folding
Timeline Viewer | View | Options | Folding

In the Folding tab you define whether to fold calls or not.

Item

Description

Fold filtered calls on tree expansion

Select this check box if you want the filtered function calls to be folded by default.

Fold calls with a very small impact on tree expansion

Select this check box if you want the function calls with a small impact to the application performance to be folded by default.
