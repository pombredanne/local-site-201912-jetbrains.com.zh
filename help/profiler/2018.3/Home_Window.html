Home Window

The Home window is your starting point in dotTrace. Here you can run new and recent profiling sessions, open collected snapshots, and set up dotTrace global settings.

The Home window consists of the following tabs:

  • New Session | Local - Start a new profiling session of a local application.

  • New Session | Remote - Start a new profiling session of an application located on a remote computer.

  • Recent | Snapshots - View a list of snapshots collected in recent profiling sessions. Here you can open the snapshots, delete them, or copy a snapshot to a file convenient for sharing.

  • Recent | Launches - View a list of recent profiling session launches. Here you can open the snapshots, delete them, or copy a snapshot to a file convenient for sharing.

For the information on dotTrace Home dialogs, refer to the Dialogs section.
