Navigating from dotTrace Source Preview to Visual Studio

You can navigate right to the source code in Visual Studio from dotTrace.

To navigate from Source View to the source code in Visual Studio

  1. Select a function and open Source View for this function.

  2. Do one of the following:

    • On the menu bar, choose Edit | Open in Visual Studio.

    • Click Open in Visual Studio in Source View.

    This functionality is not available until the solution that contains the source code is opened in Visual Studio.
