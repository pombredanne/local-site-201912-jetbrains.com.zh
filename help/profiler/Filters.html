Filters

The Filters window contains a number of filters that allow slicing and dicing profiling data by various conditions. The result of filter's work is always a set of time intervals or point events selected by a specific condition.

The Filters window is one of the main analysis instruments in Timeline Viewer.

To open the Filters window

Select the menu item View | Filters.

There are four main filter types:

  • Events
  • Interval Filters
  • Thread State
  • Subsystems
