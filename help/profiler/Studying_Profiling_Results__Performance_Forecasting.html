dotTrace 2019.2帮助

预测效果

对于大型和复杂的应用程序，通常需要花费很多时间来分析它们。分析快照时，您可能会发现似乎很耗时的函数或类。确保优化那些函数和类可以提高整体应用程序性能肯定会有所帮助。Adjust Function选项可让您立即看到在某种程度上优化功能的影响。

如果要排除所选功能或将执行时间减少到几乎为零，则可以轻松实现。

减少函数调用的执行时间为零

  1. 选择一个功能。

  2. 请执行以下任一操作：

    • 在菜单栏上，选择“ 编辑” |“选择” 。调整时间|优化当前实例 。

    • 右键单击该函数，然后单击“ 调整时间|在上下文菜单中优化当前实例 。

    • 按Del 。

将所有函数调用的执行时间减少到零

  1. 选择一个功能。

  2. 请执行以下任一操作：

    • 在菜单栏上，选择“ 编辑” |“选择” 。调整时间|优化所有实例 。

    • 右键单击该函数，然后单击“ 调整时间|在上下文菜单中优化所有实例 。

    • 按Shift+Del 。

如果您打算仅在某种程度上优化所选功能，或者需要对时间调整进行细粒度控制，请使用“ 调整时间”对话框。

调整函数或类的执行时间

  1. 选择您要检查的功能。

  2. 在菜单栏上，选择“ 编辑” |“选择” 。调整时间|调整时间或右键单击该功能，然后单击调整时间|在上下文菜单中调整时间 。

  3. “ 调整时间”对话框打开。

    在此对话框中配置选项。有关更多信息，请参见调整时间 。

  4. 单击确定 。

恢复功能的原始值

  1. 选择调整后的功能。

  2. 请执行以下任一操作：

    • 在菜单栏上，选择“ 编辑” |“选择” 。调整时间|恢复原始值 。

    • 右键单击调整后的功能，然后单击调整时间|。恢复原始值 。

    • 按Ctrl+Shift+Alt+Del 。

恢复当前选项卡中所有可用功能的值

请执行以下任一操作：

  • 在菜单栏上，选择“ 编辑” |“选择” 。调整时间|清除“当前”选项卡中所做的调整 。

  • 右键单击选项卡内的任何位置，然后单击“ 调整时间” |“调整时间”。清除“当前”选项卡中所做的调整 。

要清除当前快照中的所有调整，请打开“概述”视图 ，然后单击“ 调整的功能”部分下的“ 清除所有调整 ”。
