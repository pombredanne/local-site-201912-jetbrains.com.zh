Web代理设置

性能查看器|查看|选项|环境| Web代理设置
时间线查看器|查看|选项|环境| Web代理设置

在dotTrace选项的此页面中，您可以调整dotTrace的Internet活动的 Web代理设置。默认情况下，dotTrace遵循您的系统设置，如果系统中定义了任何代理服务器，dotTrace也会使用它来从符号服务器中获取源，验证许可证信息，检测更新等。

项目

描述

从Internet下载时使用系统Web代理设置

您可以清除此复选框以覆盖dotTrace的Internet连接的系统Web代理设置。

编辑

当您选择使用系统代理服务器设置时，将显示此按钮。
通过单击此按钮，打开“ Internet属性”对话框，您可以在其中更改系统Web代理设置 。
