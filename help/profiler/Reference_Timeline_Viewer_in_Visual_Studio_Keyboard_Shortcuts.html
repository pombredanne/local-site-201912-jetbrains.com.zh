dotTrace 2019.2帮助

键盘快捷键

呼叫树

按键或按键组合

描述

Ctrl+Space

切换子树折叠

Enter

导航到代码中的方法

Ctrl+Enter

当前方法范围

Ctrl+Shift+Enter

合并并显示所有出现的方法

Backspace

如果“调用树”专注于特定的方法出现，这将使“调用树”返回到先前的状态

Ctrl+C

将方法签名复制到剪贴板

Ctrl+Shift+C

将子树复制到剪贴板

源代码

按键或按键组合

描述

Shift+Alt+S

从源代码中的方法声明导航到“ 调用树”，然后按所有出现的方法应用过滤器。
dotTrace 10.0.2中的已知问题 ：安装dotTrace之后，快捷方式将不起作用，直到您重新应用ReSharper的当前键盘方案。为此，在ReSharper中|选项... |环境|键盘和菜单 ，单击“ 应用方案”按钮。
