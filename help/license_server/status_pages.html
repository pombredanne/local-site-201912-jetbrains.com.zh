浮动许可证服务器帮助

状态页

浮动许可证服务器（FLS）管理员可以向服务器发出请求，以获取有关运行状况，当前配置和连接状态的简要信息。如果您有问题或需要完整的信息，请访问FLS仪表板。

健康检查

可从版本＃17043获得

的/health页面显示服务器是否已注册以及是否已连接到JetBrains帐户。在第一种情况下，它会返回200 OK并提出serverUID和lastCallHome 。否则，响应为500 Internal Error问题描述。

lastCallHome显示上次连接到JetBrains服务器的时间。如果服务器无法连接到account.jetbrains.com在一定期限内，FLS停止提供用户票证并尝试每分钟连接到JetBrains帐户，请参阅与JetBrains帐户进行通信 。

  1. 确保FLS服务器正在运行。

  2. 使/health向FLS服务器的请求。例如，如果您的基本网址是http://0.0.0.0:8080http://127.0.0.1:80 ，在浏览器中打开以下内容。

    http://0.0.0.0:8080/health
    http://127.0.0.1:80/health
    您应该看到类似于以下内容的响应：

    {“ serverUID”：“ 2veurqi4td”，“ lastCallHome”：“ 2018年6月2日21:37”}

组态

的/check-configuration页面显示基本信息，例如当前主机和端口，日志文件的位置。

在版本19340中 ，它还显示了FLS Java计算机使用的Max / Total JVM内存。

  1. 确保FLS服务器正在运行。

  2. 使/check-configuration向FLS服务器的请求。例如，如果FLS基本网址是http://0.0.0.0:8080http://127.0.0.1:80 ，在浏览器中打开以下内容。

    http://0.0.0.0:8080/check-configuration
    http://127.0.0.1:80/check-configuration
    您应该看到类似于以下内容的响应：

    由主机和端口定义的基本URL：http://0.0.0.0:8080日志文件的位置：/ opt / license-server-installer / logs最大/总JVM内存：300.00 Mb / 1024.00 Mb
    配置的基本URL：http：// unit-fls主机和端口定义的基本URL：http://127.0.0.1:80日志文件的位置：C：\ ProgramData \ JetBrains \ LicenseService \ logs \ LicenseService配置文件的位置： C：\ ProgramData \ JetBrains \ LicenseService \ conf \ LicenseService最大/总JVM内存：276.00 Mb / 1024.00 Mb

连接

的/check-connection页面显示FLS与JetBrains服务器的连接是否成功。

  1. 确保FLS服务器正在运行。

  2. 使/check-connection向FLS服务器的请求。例如，如果您的基本网址是http://0.0.0.0:8080http://127.0.0.1:80 ，在浏览器中打开以下内容。

    http://0.0.0.0:8080/check-connection
    http://127.0.0.1:80/check-connection
    您应该看到类似于以下内容的响应：

    https://account.jetbrains.com确定https://www.jetbrains.com确定

版

的/check-version页显示您拥有的FLS版本，以及是否有更高版本。

  1. 确保FLS服务器正在运行。

  2. 使/check-version向FLS服务器的请求。例如，如果您的基本网址是http://0.0.0.0:8080http://127.0.0.1:80 ，在浏览器中打开以下内容。

    http://0.0.0.0:8080/check-version
    http://127.0.0.1:80/check-version
    您应该看到类似于以下内容的响应：

    {“ currentVersion”：“ 20308”，“ updateAvailable”：false}
