浮动许可证服务器帮助

配置数据目录

可从内部版本＃19340获得 。

浮动许可证服务器（FLS）将可用许可证列表，UID和注册状态存储在.jb-license-server中 ，这是安装该应用程序的系统用户主目录中的一个隐藏子目录。例如，如果您需要将数据存储在任何加密目录中或想要创建备份，则可以更改默认数据目录。

浮动许可证服务器（FLS）将可用许可证列表，UID和注册状态存储在.jb-license-server中 ，这是启动该应用程序的系统用户的主目录中的隐藏子目录。例如，如果您需要将数据存储在任何加密目录中或想要创建备份，则可以更改默认数据目录。

  1. 使用命令行转到FLS安装目录。

  2. 使用以下命令停止FLS：

    ./bin/license-server.sh停止

  3. FLS的系统用户必须具有所需数据目录的读写权限。

    设定值jb.license-server.base-dir选项到所选数据目录。例如，如果您想使用/home/user/fls ，在命令行中运行：

    ./bin/license-server.sh配置\ --jb.license-server.base-dir = / home / user / fls
    要使FLS保持注册状态，请确保所提及目录中的数据保持不变。

    所选目录的路径必须是绝对路径，而不是相对路径。

  4. 使用以下命令启动FLS：

    ./bin/license-server.sh开始

  5. 检查那个目录.jb-license-server使用命令行在选定目录下创建。由于此目录是隐藏的，因此使用命令ls -la查看所选目录的所有内容。

  1. 使用JetBrains许可服务停止FLS。

    服务Microsoft管理控制台（MMC）（控制面板>管理工具>服务> JetBrains许可服务）

  2. 使用命令行转到FLS安装目录。

  3. FLS的系统用户必须具有所需目录的读写权限。

    设定值jb.license-server.base-dir选项到所选数据目录。例如，如果您想使用C:\Users\JetBrainsLS\Desktop ，在命令行中运行：

    。\ apps \ license-server \ bin \ license-server.bat配置^ --jb.license-server.base-dir = C：\ Users \ JetBrainsLS \ Desktop
    要使FLS保持注册状态，请确保所提及目录中的数据保持不变。

    所选目录的路径必须是绝对路径，而不是相对路径。

  4. 使用JetBrains许可服务启动FLS。

  5. 检查那个目录.jb-license-server使用命令行在选定目录下创建。由于此目录是隐藏的，因此使用命令dir /ah查看所选目录的所有内容。
