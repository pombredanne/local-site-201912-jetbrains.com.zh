浮动许可证服务器帮助

管理许可证

在使用JetBrains帐户 注册浮动许可证服务器（FLS）之后，您可以管理可用于FLS的许可证池。在组织的JetBrains帐户中创建了一个单独的团队，用于管理FLS可用的许可证。

当您在JetBrains帐户中将许可证添加到FLS时，它会转换为浮动许可证，这意味着该许可证可以由多个用户使用，而不是分配给一个特定用户。如果要分配单个许可证，请将其从FLS中删除，然后使用邀请链接或电子邮件。有关管理单个许可证的更多信息，请参阅这些视频教程 。

每当您将许可证移至（或从FLS移出）许可证时，使用此许可证的客户端都将失去对其的访问权限。如果此客户端没有其他有效许可证可以使用，则其IDE将关闭并显示警告，指出没有可用的许可证。

FLS每小时都会连接到JetBrains帐户以获得许可证信息。最新连接的日期和时间显示在FLS网页的底部。使用立即强制链接可以立即启动连接并同步许可证信息。

向FLS添加许可证

  1. 打开您的FLS URL。例如，如果您在默认端口上本地运行它8080 ，打开以下网址： http://0.0.0.0:8080/http://127.0.0.1:80/ 。

  2. 单击添加许可证以打开带有可用许可证列表的JetBrains帐户。

  3. 单击“ 继续”以获取要移至FLS的产品许可证。

  4. 指定通过FLS可用的许可证数量，然后单击“ 移至服务器” 。

  5. 单击“ 返回到许可证服务器”以检查可用于FLS的许可证数量。

从FLS删除许可证

  1. 打开您的FLS URL。例如，如果您在默认端口上本地运行它808080 ，打开以下网址： http://0.0.0.0:8080/http://127.0.0.1:80/ 。

  2. 单击管理许可证以打开您的JetBrains帐户以及可用许可证列表。

  3. 单击“ 继续”以获取要从FLS中删除的产品许可证。

  4. 单击管理许可证以打开分配给FLS的许可证列表。

  5. 选择要从FLS撤消的许可证，打开“ 管理”下拉菜单，然后单击“ 转移到另一个团队” 。

  6. 选择要将所选许可证移至的团队，然后单击“ 转移” 。如果转移成功，这些许可证将列在所选团队中。

  7. 单击“ 返回到许可证服务器”以检查可用于FLS的许可证数量。
