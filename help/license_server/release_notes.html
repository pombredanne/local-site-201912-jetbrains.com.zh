Release notes

Build #21137

Released on: October 24, 2019

  • Updated a Jetty version from 9.3.24 to 9.4.19.

  • Fixed an incorrect displaying of licenses with fallbacks while requesting report.

  • Excluded the situation when priority user could lose the ticket if another priority user made a request.

Build #20308

Released on: July 30, 2019

  • Fixed an error that caused when requesting and downloading reports from Internet Explorer 11.

  • Fixed the FLS denial of service caused by a parallel report requests. Server will occur the 503 error with the message "Server is generating another report at this time. Please return later." if the report cannot be generated, instead of causing FLS performance degradation.

Build #20267

Released on: July 20, 2019

  • Fixed an error that occurred at build #19488 when requesting email report

  • Fixed an incorrect plugin names representation

  • Fixed a problem with the number of licenses presented on FLS dashboard

  • Changed the default logging level of ticket handling from INFO to WARN. To revert it, see Changing logging level

Build #19488

Released on: April 2, 2019

  • Fix unexpected behaviour of virtual hosts configuration parameter for MSI distribution

Build #19340

Released on: March 15, 2019

  • Improvement: custom home folder for usage statistics and registration info

  • Bugfix of R# and ReSharper Ultimate distribution with access config rules

  • Improvement: API endpoint with FLS versions list

  • Improvement: memory usage is added to /check-configuration page

Build #18692

Released on: December 24, 2018

  • Bugfix for the processing of licenses without fallback version, placed on the server

  • Bugfix of the "Maximum length of cell contents" error during a report generating

  • "ticketid" parameter is added to the real-time statistics output

  • Bugfix of the incorrect All Products Pack licenses distribution

  • IP is added as access rules parameter

  • Jetty version update: 9.3.24.v20180605

Build #17955

Released on: September 26, 2018

  • This builds fixes a known issue in sessions sharing logic during the registration process and changes the logic of "New version is available" banner appearance.

Build #17768

Released on: September 5, 2018

  • New user restrictions parameter is added (User restrictions)

  • New API endpoint is added to get the list of all licenses allocated on the license server (User reporting)

Build #17437

Released on: July 30, 2018

  • This build fixes the issue that caused connected products with version RC 2018.2 to each occupy one license every time the license was checked. This could result in one product hoarding licenses.

  • Build #17437 handles release candidate versions correctly

Build #17211

Released on: July 20, 2018

  • This update fixes a known issue with the previously released version (build #17043) where FLS lost connection to JetBrains after server restart.
