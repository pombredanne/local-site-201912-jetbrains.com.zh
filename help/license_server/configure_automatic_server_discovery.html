浮动许可证服务器帮助

自动服务器发现

本主题与系统管理员有关。您可以启用自动服务器发现功能，以使IDE在“许可证激活”对话框中立即找到服务器。它允许最终用户简化授权过程。

  1. 如果您在内部网络中使用DHCP，则还会通过DHCP获得DNS域名（ ）。在这种情况下，只需将DNS TXT记录添加到相应的DNS区域（ ）。例如，如果您的FLS正在运行url=http://fls.company.com ，添加以下记录： _jetbrains-license-server. 。

  2. 使用命令行验证记录：

    挖_jetbrains-license-server。 文本
    nslookup -type = TXT _jetbrains-license-server。

    有效响应应如下所示：

    _jetbrains-license-server。 。3600 IN TXT“ url = http：//fls.company.com”

  3. 要检查配置是否正确，请打开IDE并进入“许可证激活”对话框。选择“ 许可证服务器”以使用FLS激活许可证，然后单击“ 发现服务器” 。配置的地址应自动出现在此对话框中。然后单击“ 激活”以连接到FLS。

自2017年7月以来，基于IntelliJ平台的产品可识别环境变量JETBRAINS_LICENSE_SERVER 。该变量应在最终用户本地配置文件中定义，其值应为指向FLS实例的标准URL。
