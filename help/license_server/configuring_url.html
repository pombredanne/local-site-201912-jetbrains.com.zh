浮动许可证服务器帮助

服务器网址

默认情况下，浮动许可证服务器（FLS）在主机上启动0.0.0.0127.0.0.1和港口808080 。可以在内部进行测试，但是在此地址上运行生产FLS服务器并不安全。您应拥有一台具有专用IP地址（最好是可解析的域名）的物理或虚拟服务器，并配置主机和端口以相应地运行FLS。

  1. 转到FLS安装目录。

  2. 使用以下命令停止FLS：

    ./bin/license-server.sh停止

  3. 将主机和端口指定为FLS配置参数。例如，将FLS主机设置为license-server.domain.com并转至8181 ，运行以下命令：

    ./bin/license-server.sh configure \-监听license-server.domain.com \ --port 8181

  4. 使用以下命令启动FLS：

    ./bin/license-server.sh开始

  1. 停止服务JetBrains许可服务 。

    例如，您可以使用Microsoft管理控制台（MMC）的“ 服务”管理单元。

  2. 使用命令行转到FLS安装目录。

  3. 将主机和端口指定为FLS配置参数。例如，将FLS主机设置为license-server.domain.com并转至8181 ，运行以下命令：

    。\ bin \ license-server.bat配置^ --listen-port 8181 ^ --base-url http://license-server.domain.com:8181

  4. 启动服务JetBrains许可服务 。

自定义上下文路径

如果运行FLS的计算机还运行其他Web应用程序，则可以为FLS指定自定义上下文路径。

  1. 转到FLS安装目录。

  2. 使用以下命令停止FLS：

    ./bin/license-server.sh停止

  3. 设置值--context相对于根上下文的必要上下文路径的参数。

    例如，将上下文路径设置为/licenseServer ，运行以下命令：

    ./bin/license-server.sh配置\ --context / licenseServer

  4. 使用以下命令启动FLS：

    ./bin/license-server.sh开始

  1. 停止服务JetBrains许可服务 。

    例如，您可以使用Microsoft管理控制台（MMC）的“ 服务”管理单元。

  2. 使用命令行转到FLS安装目录。

  3. 更改值--base-url相对于根上下文的必要上下文路径的参数。

    例如，将上下文路径设置为/licenseServer上http://license-server.mydomain.com:8080 ，运行以下命令：

    。\ bin \ license-server.bat配置^ --base-url http://license-server.mydomain.com:8080/licenseServer

  4. 启动服务JetBrains许可服务 。
