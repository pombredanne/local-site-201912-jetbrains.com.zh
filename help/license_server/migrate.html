浮动许可证服务器帮助

移至另一台物理服务器

如果运行FLS的服务器已过时或没有适当的特征，则可以将其移至另一台物理服务器。

  1. 将新的浮动许可证服务器（FLS）副本安装到新位置。

  2. 停止以前和新的FLS版本。

  3. 要将FLS软件移动到另一台计算机，请移动以下文件：

    • 来自的注册数据$HOME/.jb-license-server （或从另一个数据目录（如果已更改），请参阅配置数据目录 ）。

    • 来自的配置数据./conf该目录位于FLS安装目录中。

  4. 从先前的位置删除FLS安装目录。

  5. 如果需要，配置反向代理和自动服务器发现 。

  6. 使用以下命令启动FLS：

    ./bin/license-server.sh开始

  1. 将新的FLS副本安装到新位置。

  2. 停止以前和新的FLS版本。

  3. 要将FLS软件移动到另一台计算机，请移动以下文件：

    • 来自的注册数据%userprofile%\.jb-license-server给用户JetBrainsLS （或其他数据目录，如果您已更改的话）。请参阅将数据目录配置到新计算机上的相同位置。

    • 来自的配置数据%programData%\JetBrains\LicenseService\conf到新计算机上的相同位置。

  4. 确保该服务用户JetBrainsLS对目标计算机上复制的文件具有完全访问权限。

  5. 如果需要，配置反向代理和自动服务器发现 。

  6. 使用服务Microsoft管理控制台（MMC）启动FLS。
