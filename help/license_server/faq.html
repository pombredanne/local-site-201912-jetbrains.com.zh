许可证服务器管理

浮动许可证服务器可以脱机工作吗？

不，它需要Internet连接到JetBrains帐户。对于离线许可选项，请直接使用JetBrains帐户或使用激活码激活产品。有关脱机管理许可证的更多信息，请参阅这些视频教程 。

许可证服务器多久连接一次JetBrains帐户？

浮动许可证服务器每小时连接到JetBrains帐户以获得许可证信息。从上一次成功连接起，服务器最多可以在48小时内不连接到JetBrains帐户的情况下继续颁发许可证。确保浮动许可证服务器至少每两天一次连接到JetBrains帐户。有关更多详细信息，请参阅与JetBrains帐户进行通信 。

应将哪些IP地址添加到防火墙，以允许浮动许可证服务器连接到JetBrains帐户？

IP地址是动态的，没有固定的列表。

浮动许可证服务器是否支持多节点方案进行故障转移？

否，当前浮动许可证服务器不支持故障转移。一次只有一个主机实例可以处理特定的许可证池。但是，您可以为不同的许可证池托管不同的服务器。

可以在浮动许可证服务器主机上使用Java 9吗？

否，不支持Java 9。

我可以在同一主机上运行多个浮动许可证服务器吗？

是的，如果它们是由不同的系统用户启动的。否则，服务器将重写许可证信息和使用情况统计信息。

如何限制对FLS的访问？

在使用您的JetBrains帐户注册浮动许可证服务器后，只有有权访问该帐户的用户才能连接到浮动许可证服务器。此外，您可以使用防火墙过滤网络中浮动许可证服务器的IP地址和端口号。

如何限制许可证分配？

浮动许可证服务器管理员可以创建白名单，黑名单和优先级列表，以限制许可证分配。有关更多信息，请参见用户限制 。

在哪里可以找到服务器的UID？

在浮动许可证服务器网页的右上角。

在哪里可以找到有关FLS版本的信息？

在本文档的“发行说明”页面上。此外，API端点/check-version 。有关更多信息，请参见Version 。

浮动许可证

许可证服务器如何发行和吊销许可证？

浮动许可证服务器消除了手动分配单个许可证密钥的需要。一旦安装了浮动许可证服务器并将其连接到您的JetBrains帐户，它就可以为网络中的客户端应用程序（JetBrains IDE和其他工具）颁发和撤销许可证。每个许可证提供一张票证，一次可以在一台计算机上使用。

启动IDE时，它会向浮动许可证服务器请求许可证。验证之后，如果有可用许可证，服务器将为IDE发出票证。在产品运行时，它每10分钟请求延长许可证。当您停止IDE时，其票证将释放回池中，并可供其他客户端使用。此模型使产品一次可供用户使用的数量与您为团队购买的可用许可证的数量一样。

如果颁发许可证的计算机进入睡眠模式，许可证将如何处理？

即使机器正在休眠，许可证仍在使用中。要发布许可证，请关闭IDE。

如果没有与FLS的连接，是否可以继续使用具有获得的许可证的IDE？

是的，如果产品从服务器收到许可证票，然后失去与服务器的连接，则可以将该产品与该票一起使用最多两天。在与FLS的最后一次连接后48小时，该票证将失效，您将无法使用该产品，直到该产品连接到FLS。此参数不可配置。

如果您在这48个小时内重新启动IDE，票证将立即失效。在这种情况下，您将无法使用该产品，直到该产品连接到Floating License Server。

如果具有获得的许可证的IDE无法连接到浮动许可证服务器，那么该许可证是否可用于其他人？

是的，如果IDE要求延长的时间不超过20分钟，则浮动许可证服务器会将其返回到可用许可证池中。如果IDE在48小时内连接到浮动许可证服务器，则将发生以下情况之一：

  • 如果仍然可用，则已颁发的许可证将被延长。

  • 如果返回的许可证是由另一个客户端获取的，则将向IDE颁发新的许可证。

  • 如果池中没有可用的许可证，IDE将立即停止工作。

如何将许可证移至浮动许可证服务器？

打开您的JetBrains帐户，选择一个或多个许可证，单击“ 管理” ，然后单击“ 转移到另一个团队” 。有关更多信息，请参阅管理许可证 。

如果同时具有产品特定的许可证和所有产品包的许可证，则使用哪个许可证？

对于一个IDE实例，颁发了特定于产品的许可证。如果同一用户启动另一个IDE，并且在浮动许可证服务器上没有其他可用的特定于产品的许可证，但是有所有产品包许可证可用，则将为两个IDE颁发一个“所有产品包”许可证。在这种情况下，产品特定的许可证被吊销，以便其他人可以使用它。

如果服务器上仅剩一个合适的许可证，并且有两个客户端连接，会发生什么情况？

浮动许可证服务器根据先进先出方案处理请求。如果只有一个合适的许可证，则会将其颁发给首先连接并请求该许可证的客户端。第二位用户将收到警告No suitable licenses 。

过时的许可证服务器

什么是过时的许可证服务器？

从内部版本＃16429 （于2018年3月19日发布）开始，仅最新的JetBrains产品支持许可证服务器：高于版本2018.2.1（对于.NET工具而言高于2018.3）。当IDE连接到不支持的许可证服务器版本时，您将收到以下通知： Outdated license server detected 。

有关升级的信息，请参阅此博客文章和“ 升级” 。

在哪里可以看到许可证服务器的版本？

内部版本号显示在“许可证服务器”网页的底部。

新的许可证服务器是否支持旧的IDE

是的，如果您升级到最新的许可证服务器，则可以将其用于所有受支持的JetBrains产品版本。但是，如果您运行的版本早于＃16429 ，则它仅适用于2018.2.1之前的产品版本（.NET工具早于2018.3）。
