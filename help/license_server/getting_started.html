Floating License Server

Latest version: Build #21137

Floating License Server (FLS) is a Java-based application to manage JetBrains product licenses within your company network. Available if you own more than 50 active product subscriptions.

This limit does not apply to classroom licenses. You can use FLS with any number of classroom licenses.

FLS supports all JetBrains Toolbox and Plugins subscription licenses and doesn't support Team Tools.

Start using FLS:

  1. Contact the Sales team to enable FLS for your JetBrains Account.

  2. Check the requirements and download a suitable FLS distribution.

  3. Install and configure FLS.

  4. Register FLS.

  5. Add licenses to FLS.

If you didn't find an answer to your question in this documentation, and you require assistance, contact the JetBrains Support team.
