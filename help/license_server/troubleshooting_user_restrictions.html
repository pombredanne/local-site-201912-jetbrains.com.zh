浮动许可证服务器帮助

用户限制

如果您已配置了限制但它们不起作用，则可能有两个可能的原因导致这种情况发生。

语法错误

问题

FLS尝试在启动/重新启动期间应用访问配置文件。如果找不到文件或语法错误，则将跳过该文件。

解

启动FLS时，您会看到：

  • 找不到文件：

    java.io。FileNotFoundException： /access-config.json（无此文件或目录）
    解决方案：检查在中指定的文件的路径conf/service-config.properties 。将此路径与实际文件的位置进行比较的最简单方法是将其粘贴到浏览器中。修复路径并重新启动许可证服务器进程。

  • 如果文件语法错误：

    错误e：62-无法解析位于文件：/path-to-the-file/access-config.json的配置
    错误e：62-无法解析位于文件：/ path-to-the-file \ access-config.json的配置
    错误说明，例如：
    com.fasterxml.jackson.databind。JsonMappingException：意外字符（']'（代码93））：
    解决方案：检查语法是否正确，然后重新启动过程。使用JSON验证程序很容易检查语法

这些错误也显示在license-server-stdout.log 。 /check-configuration （有关更多详细信息，请参阅Connection ）显示日志和配置文件在FLS副本上的位置。

无法从FLS获取票证

问题

用户发送的LicenseRequest与限制规则不匹配。

解

比较来自的请求./logs/license-server-stdout.log.\logs\license-server-stdout.log遵守规则。它可能不同于主机名，产品或内部版本。
