浮动许可证服务器帮助

注册

安装并启动 FLS后，请使用您的JetBrains帐户注册它，然后在池中添加您希望可用的许可证。您可以注册FLS而不配置FLS仅用于测试目的，因为默认配置不安全。要正确配置FLS，请参阅管理FLS 。

要在将FLS升级或迁移到另一台物理服务器后强制注册，请转到/register URI，例如http://0.0.0.0:8080/registerhttp://127.0.0.1:80/register 。

  1. 打开您的FLS URL。例如，如果您在默认端口上本地运行它808080 ，打开以下网址： http://0.0.0.0:8080/http://127.0.0.1:80/ 。

  2. 单击注册服务器，然后使用您的JetBrains帐户凭据登录。

    用户必须具有公司或团队资料的管理员权限。

  3. 如果您有多个与此JetBrains帐户相关联的团队，则系统将提示您选择要向其注册FLS的团队。

  4. 如果这不是使用此JetBrains帐户注册的第一个FLS实例，系统将提示您创建新的服务器注册或重新使用以前的服务器。

    无法从多个物理服务器运行相同的FLS。确保先前的实例已关闭且未使用。否则，它将自行停止工作并导致获取许可证方面的问题。

  5. 注册FLS后，您可以按照“ 管理许可证”中的说明向其中添加许可证 。
