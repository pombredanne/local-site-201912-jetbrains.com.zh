浮动许可证服务器帮助

代理设定

如果在受限环境中安装了浮动许可证服务器（FLS），则可能需要配置代理设置以使FLS正常运行。

代理相关的系统属性

属性

描述

http.proxyHost

您的代理服务器的主机名。

http.proxyPort

代理服务器侦听的端口号。

https.proxyHost

您的安全代理服务器的主机名。

https.proxyPort

安全代理服务器侦听的端口号。

  1. 使用命令行转到FLS安装目录。

  2. 使用以下命令停止FLS：

    ./bin/license-server.sh停止

  3. 在命令行中运行：

    ./bin/license-server.sh配置\ -J-Dhttps.proxyHost = \ -J-Dhttps.proxyPort =

    哪里是主持人， 是您要使用的代理的端口。

    必须输入proxy_host时不带协议前缀，而只是一个域名。

  4. （可选）如果许可证服务器软件位于需要身份验证的代理后面，请在命令行中运行：

    ./bin/license-server.sh配置\ -J-Dhttps.proxyUser = \ -J-Dhttps.proxyPassword =

    哪里是用户，并且是您要使用的代理所需的密码。

  5. 使用以下命令启动FLS：

    ./bin/license-server.sh开始

  1. 使用服务Microsoft管理控制台（MMC）（控制面板>管理工具>服务> JetBrains许可服务）停止FLS 。

  2. 使用管理员权限打开命令行。

  3. 转到FLS安装目录。

  4. 在命令行中运行：

    。\ apps \ license-server \ bin \ license-server.bat配置^ -J-Dhttps.proxyHost = ^ -J-Dhttps.proxyPort =

    哪里是用户，并且是您要使用的代理所需的密码。

    必须输入proxy_host时不带协议前缀，而只是一个域名。

  5. （可选）如果许可证服务器软件位于需要身份验证的代理后面，请在命令行中运行：

    。\ apps \ license-server \ bin \ license-server.bat配置^ -J-Dhttps.proxyUser = ^ -J-Dhttps.proxyPassword =

    哪里是用户，并且是您要使用的代理所需的密码。

  6. 使用JetBrains许可服务启动FLS。
