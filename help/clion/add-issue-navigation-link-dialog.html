Use this dialog to create an issue pattern and navigation link to a bug tracking system.

Item

Description

Issue ID (regular expression)

In this field, type a regular expression that will be converted to a specific issue id.

Issue link (replacement expression)

In this field, type a regular expression that will be converted to a navigation link to the issue.
