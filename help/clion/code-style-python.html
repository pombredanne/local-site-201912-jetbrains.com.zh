制表符和缩进

项目

描述

使用制表符

  • 如果选中此复选框，则使用制表符：

    • 在按Tab键

    • 缩进

    • 用于代码重新格式化

  • 清除复选框后，CLion使用空格而不是制表符。

智能标签

  • 如果选中此复选框，则由代码块嵌套定义的缩进部分由制表符和（如有必要）空格组成，而由对齐方式定义的缩进部分仅由空格组成。

  • 如果清除此复选框，则仅使用选项卡。这意味着将将一组适合指定制表符大小的空格自动替换为一个制表符，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中包含的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组元素之间插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

缩进空行

如果选中此复选框，则CLion将使空行缩进，就好像它们包含一些代码一样。
如果清除此复选框，CLion将删除制表符和空格。

空间

使用此选项卡可以指定代码中需要空格的位置。要让CLion自动在某个位置插入空格，请在列表中选中该位置旁边的复选框。结果显示在“ 预览”窗格中。

包装和大括号

在此选项卡中，定制代码样式选项，CLion将在重新格式化源代码时应用这些选项。左侧窗格包含异常列表（ 重新格式化时保留 ），以及各种代码结构（列表，语句，操作，注释等）的放置和对齐选项。右侧窗格显示预览。

对齐优先于缩进选项。

右边距（列）

使用“ 硬包装在”字段可以指定元素右侧所需的页边空白空间。如果选择默认选项，则使用全局设置中右边距的值。

包装打字

使用打字设置环绕到指定编辑的文本是怎样安装在指定的硬包装的 。您可以选择以下选项之一：

  • 默认值 ：在这种情况下，CLion使用全局设置中指定的“ 键入时自动换行”选项。

  • 是 ：在这种情况下，CLion使用“ 右边界”字段中指定的值。

  • 否 ：在这种情况下，此选项已关闭，并且一行可以超出右边界中指定的值。

视觉指南

使用“ 视觉参考线”字段来指定多个右边距。您可以保留默认值，也可以输入空白。如果要指定多个边距，请输入用逗号分隔的数字。

重新格式化时保持

使用复选框配置CLion在重新格式化源代码时将产生的异常。例如，默认情况下，“ 换行符”复选框处于选中状态。如果您的代码包含比标准约定短的行，则可以在重新格式化源代码之前通过禁用换行符复选框来转换它们。

空行

使用此选项卡可定义重新格式化后希望CLion保留并插入代码中的空白行的位置和数量。对于每种类型的位置，指定要插入的空白行数。结果显示在“ 预览”窗格中。

保留最大空白行

在此区域中，指定在指定位置重新格式化后要保留的空白行数。

最小空白行

在该区域的文本框中，指定要在指定位置显示的空白行数。

这些设置不会影响第一项之前和最后一项之后的空白行数。
