CLion 2019.3帮助

卸载CLion

删除CLion的正确方法取决于您安装它的方法。

本指南介绍了在Windows macOS Linux上卸载CLion的方法。使用顶部的平台切换器选择其他操作系统。

工具箱应用

如果您通过使用工具箱应用程序安装了CLion：

  • 打开工具箱应用程序，单击所需实例的螺母图标，然后选择“ 卸载” 。

独立实例

要删除独立的CLion实例 ，请使用标准过程在操作系统上卸载应用程序，并删除IDE配置和系统目录 。

  1. 在系统设置对话框中打开“ 应用和功能”部分。

  2. 选择CLion应用程序，然后单击卸载 。

  3. 删除以下目录：

    句法
    ％HOMEPATH％\。 \ config
    例
    C：\ Users \ JohnS \。CLion2019.3 \配置
    句法
    ％HOMEPATH％\。 \系统
    例
    C：\ Users \ JohnS \。CLion2019.3 \ system

  1. 在“ 应用程序”目录中删除该应用程序 。

  2. 删除以下目录：

    句法
    〜/图书馆/首选项/
    例
    〜/图书馆/首选项/CLion2019.3
    句法
    〜/库/缓存/
    例
    〜/库/缓存/CLion2019.3

  1. 删除安装目录。

  2. 删除以下目录：

    句法
    〜/。 /配置
    例
    〜/。CLion2019.3 /配置
    句法
    〜/。 /系统
    例
    〜/。CLion2019.3 /系统

静默卸载

如果您以静默方式安装了CLion ，则可以使用以下命令运行卸载程序： /S切换为管理员。卸载程序位于bin下的安装目录中。

以管理员身份运行cmd （Windows命令提示符），切换到CLion安装目录，然后运行以下命令：

bin \ uninstall.exe / S

卡扣包装

如果将CLion作为快照软件包安装 ，请使用以下命令将其删除：

$ sudo snap删除clion
