参数提示

设置/首选项|编辑器镶嵌提示C / C ++ /目标C

默认状态： 启用

基于Clangd的引擎

参数提示显示传递的参数的函数参数名称。CLion在与代码内联的编辑器中显示它们，以提高其可读性：

参数提示可以帮助您理解函数调用，而无需研究函数签名（您可以使用Parameter Info工具提示或导航到函数声明来完成此操作 ）。

提示要求启用基于Clangd的引擎 （默认情况下启用）。

显示具有多个操作数的参数文字和表达式的参数提示 ：函数调用，lambda，初始化器列表，宏表达式和构造函数（包括通过类似emplace_back的函数调用的构造函数）。当参数用作指定的初始化程序时， 不显示初始化程序的提示。

隐藏和切换参数提示

这些提示默认情况下处于启用状态。要隐藏它们，请转至设置/首选项|编辑器镶嵌提示C / C ++ / Objective-C，然后清除“ 显示参数名称提示”复选框。

您可以分配快捷方式以快速切换提示的开和关。转到设置/首选项| Keymap ，搜索提示 ，并为Toggle参数名称提示操作添加快捷方式 ：

配置参数提示

要隐藏特定情况下的参数提示，请配置例外列表。

  1. 转到设置/首选项|编辑器镶嵌提示C / C ++ / Objective-C ，然后点击黑名单...。

  2. 添加您不想看到提示的方法。

您还可以直接在编辑器中调整提示设置：

  • 按Alt+Enter提示快速禁用当前功能。

  • 右键单击提示，然后从选项列表中选择。
