CLion |首选项|编辑器代码样式|适用于macOS的CMake
Ctrl+Alt+S

使用此部分为CMake文件配置代码样式选项。更改这些设置时，更改结果将显示在“ 预览”窗格中。

从...开始

单击此链接以显示将用作当前语言代码样式基础的语言列表。仅采用适用于当前语言的设置。所有其他设置均不受影响。

单击“ 重置”以放弃更改并返回到初始的代码样式设置集。

制表符和缩进

项目

描述

使用制表符

  • 如果选中此复选框，则使用制表符：

    • 在按Tab键

    • 缩进

    • 用于代码重新格式化

  • 清除复选框后，CLion将使用空格而不是制表符。

智能标签

  • 如果选中此复选框，则由代码块嵌套定义的缩进部分由制表符和（如有必要）空格组成，而由对齐方式定义的缩进部分仅由空格组成。

  • 如果清除此复选框，则仅使用选项卡。这意味着将将一组适合指定制表符大小的空格自动替换为一个制表符，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中包含的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，在表达式，方法声明和方法调用中，指定要在数组元素之间插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

缩进空白行

如果选中此复选框，则CLion将使空行缩进，就好像它们包含一些代码一样。
如果清除此复选框，CLion将删除制表符和空格。

空间

使用此选项卡可以指定代码中需要空格的位置。要让CLion自动在某个位置插入空格，请在列表中选中该位置旁边的复选框。结果显示在“ 预览”窗格中。

包装和大括号

在此选项卡中，定制代码样式选项，CLion将在重新格式化源代码时应用这些选项。左侧窗格包含各种代码结构的放置和对齐选项的列表。右侧窗格显示预览。

对齐优先于缩进选项。

多行时对齐

如果选中此复选框，则代码构造将从下一行的同一列开始。否则，代码构造的位置由当前的缩进级别确定。

空行

使用此选项卡定义重新格式化后希望CLion保留并插入代码中的空白行的位置和数量。对于每种类型的位置，指定要插入的空白行数。结果显示在“ 预览”窗格中。

项目

描述

保留最大空白行

在此区域中，指定在指定位置重新格式化后要保留的空白行数。

其他

命令案例

使用此设置可以更改CMakeLists.txt文件中命令的大小写。注意 ，对于现有代码，所做的更改应跟随“ 重新格式化代码”操作：在主菜单中，选择“代码” | “更改”。重新格式化代码或按Ctrl+Alt+L 。对于新代码，更改将自动应用。同样，该设置分别影响代码完成功能：自动完成功能建议在选定的大小写设置之后执行CMake命令完成操作，而不考虑用于初始键入命令的大小写。

项目

描述

强制命令案例

  • 上方 ：选择此选项可将命令大小写更改为上方。请参阅预览窗格中的示例。

  • 降低 ：选择此选项可将命令大小写更改为较低。

  • 请勿更改 ：默认设置。选择此选项可保持命令大小写不变。
