运行/调试配置：Google测试

使用此页面可以配置Google Test框架的运行/调试选项。

在这里，您可以找到所有运行/调试配置通用的选项说明。

配置选项卡

项目

描述

目标

从列表中选择所需的目标。

目标类型标记有以下图标：

- 所有目标

- 可执行

- 图书馆

测试种类

  • 选择套件/测试选项以运行套件的特定或所有测试。

  • 选择模式选项以运行与特定模式匹配的所有测试。

套房

在此字段中，指定套件的名称。仅当选择“ 套件/测试”选项时，此选项才可用。

测试

从此下拉列表中，选择所需的测试。仅当选择“ 套件/测试”选项时，此选项才可用。

图案

在此字段中，指定模式名称。仅当选择“ 图案”选项时，此选项才可用。

程序参数

在此字段中，以将在命令行中使用的格式键入要传递给程序的参数列表。如有必要，请单击按钮并输入所需的参数。

工作目录

指定用于运行应用程序的工作目录。该目录是所有相对输入和输出路径的起点。默认情况下，该字段包含项目文件所在的目录。要指定另一个目录，请单击并选择目录。

展开列表以查看可用的路径变量 ，您可以将其用作工作目录的路径。在这种情况下，您不需要在“ 环境变量”字段中指定任何其他环境变量。

路径变量的列表可能会因启用的插件而异。
