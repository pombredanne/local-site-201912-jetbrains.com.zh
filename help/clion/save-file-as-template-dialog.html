CLion 2019.3帮助

将文件另存为模板对话框

CLion帮助您从现有文件创建文件模板 。

项目

描述

名称

指定新模板的名称，因为它将显示在“ 文件和代码模板”设置的“ 文件”选项卡中。默认情况下，此字段设置为当前打开的文件的名称。

延期

指定要由新模板创建的文件的扩展名。默认情况下，此字段设置为当前打开文件的扩展名。

范本文字

编辑模板内容。在这里您可以使用：

  • 纯文本。

  • #parse用于模板的指令包括 。

  • 自定义变量。变量名可以通过以下方式直接在模板中定义： #set指令或在文件创建期间。

    请注意，CLion不会提示输入使用定义的Velocity变量的值。 #set 。

  • 变量将被扩展为相应的值${}格式。文件模板变量中介绍了可在文件模板中使用的预定义变量 。

根据样式重新格式化

如果您希望CLion根据“ 代码样式”页面上定义的样式重新格式化生成的存根文件，请选中此复选框。

此选项仅在“ 文件”选项卡中可用。

启用实时模板

选中此复选框可在文件模板中使用实时模板。在这种情况下，您需要将实时模板片段放入Velocity转义语法中。例如：

#[[ $MY_VARIABLE$ $END$ ]]#

这样，您可以指定光标位置。请注意，此处需要使用实时模板变量 。

描述

此只读字段提供有关模板，模板的预定义变量及其工作方式的信息。
