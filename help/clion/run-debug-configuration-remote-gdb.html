运行/调试配置：GDB远程调试

使用此页面为远程GDB调试配置“运行/调试”选项。

单击此处 ，了解所有运行/调试配置共有的选项。

远程GDB配置设置

项目

描述

广东发展银行

默认情况下，该字段建议捆绑GDB多体系结构 。要指定自定义GDB，请通过点击按钮，然后选择包含GDB可执行文件的目录。

“目标远程”参数

在此字段中，指定用于承载调试数据包的介质（串行线路，或使用TCP或UDP的IP网络）。在此处查找有关连接到远程目标的更多详细信息。

符号文件

如果已连接到远程GDB服务器并启动了调试会话，但是程序在到达断点时没有停止，请检查是否正确指定了符号文件和路径映射 。

在此字段中，指定调试符号文件的位置。输入路径或单击按钮，然后从打开的对话框中选择它。

Sysroot

在此字段中，指定sysroot的位置。输入路径或单击按钮，然后从打开的对话框中选择它。

路径映射

在此窗格中，在“ 远程”列中指定目标计算机上的路径（如项目源的路径等），这些路径应映射到主机上的本地路径（在“ 本地”列中指定）。
