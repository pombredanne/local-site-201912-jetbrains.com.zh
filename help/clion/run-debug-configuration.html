运行/调试配置

要在CLion中运行或调试代码，可以使用各种“运行/调试”配置。每个配置都是一组命名的运行/调试启动属性。

CLion基于配置模板生成运行/调试配置。每个模板都有其自己的可自定义设置。当您从某个模板创建新配置时，它是使用模板的设置生成的。

运行/调试配置模板

具有默认运行/调试配置设置的模板在“ 模板”节点下的“ 运行/调试配置”对话框中列出。它们表示创建新的运行/调试配置时使用的设置。

您可以为特定配置类型设置默认设置，该设置将适用于以后创建的该类型的任何运行/调试配置。更改默认值不会影响现有的运行/调试配置。

更改配置模板中介绍了编辑每个类型的默认配置设置的过程。创建和编辑自定义运行/调试配置的过程在“ 创建和编辑配置”中进行了描述。
