CLion 2019.3帮助

蟒蛇

要在CLion中开发Python脚本，请下载并安装Python并配置至少一个Python解释器 。

如果您是Windows用户，建议您安装Python for Windows 。

支持的版本

  • Python 2： 2.7版

  • Python 3：从3.5版到3.8版

CLion中的Python支持

CLion提供以下功能来帮助使用Python：

  • 能够配置本地解释器。

  • Python控制台 。

  • 运行/调试Python的配置。

  • 代码检查。

  • 意向动作。

  • 代码完成并解决。

  • 内置代码格式化程序和单独的Python 代码样式设置集 。

  • 在Python代码中查找用法 。

  • 快速文档 。

  • 配置Python调试器 。
