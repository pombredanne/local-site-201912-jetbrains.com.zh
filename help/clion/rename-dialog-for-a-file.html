重命名文件/目录对话框

重构|改名

Shift+F6

使用此对话框可以重命名文件或目录 。

除了重命名文件/目录本身之外，CLion还可以查找文件/目录名称的用法。如果找到，您所做的更改也可以应用于这些用法。

项目

描述

重命名（ ）及其用法为

指定文件（目录）的新名称。

搜索参考

如果启用此选项，CLion将在代码文件中查找文件/目录名称的出现。

字符串文字是否是引用是由源文件中使用的语言定义的。

搜索注释和字符串

如果选中此复选框，CLion将在源代码文件的注释和字符串文字中查找是否存在文件/目录名称。

范围

使用此选项可以设置重命名重构的范围。例如，您可以将重构限制为仅最近更改的文件或仅打开的文件，依此类推。您还可以创建可以共享或保留在本地的自定义范围。
