提取超类

  1. 在其中一个视图中选择所需的类，或仅在编辑器中将其打开。

  2. 从主菜单或上下文菜单中，选择“ 重构”提取物|超类 。

  3. 在出现的“ 提取超类”对话框中，指定以下信息：

    • 新超类的名称。

    • 成员将包括在超类中

  4. 进行重构。

提取类依赖成员

int d1; class XClass {static const int d2 = 13;整数exRes = d2 * d1; };
