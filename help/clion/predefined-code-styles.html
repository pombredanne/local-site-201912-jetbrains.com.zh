预定义的代码样式

对于C / C ++，Objective-C / C ++和JavaScript ，您可以从预定义的代码样式中复制设置。请注意 ，每种预定义的代码样式仅影响基本设置的某些项目，其余部分保持不变。

在注册表中设置clion.enable.objc.settings项后，可以使用Objective-C / C ++的代码样式选项（有关说明，请参见Objective-C / C ++支持 ）。

  1. 在 设置/首选项对话框Ctrl+Alt+S ，单击“代码样式” ，然后单击语言。

  2. 从右上角选择链接Set from ，然后从列表中选择Predefined style选项。

  3. 在出现的列表中，选择代码样式以从以下位置复制设置：

    • 谷歌
    • 虚拟机
    • 本地数据库
    • t
    • GNU
    • 微软
    • 肉酱

    以下样式仅适用于括号放置：

    • 奥尔曼牙套
    • 铁匠括号
    • K＆R大括号
    • JavaScript标准样式
    • Google JavaScript样式指南
