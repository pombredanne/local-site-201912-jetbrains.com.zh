Predefined Code Styles

For C/C++, Objective-C/C++ and JavaScript, you can copy settings from the predefined code styles. Note that each predefined code style affects only the certain items of base settings and leaves the rest unchanged.

Code style options for Objective-C/C++ are available after setting the clion.enable.objc.settings key in the Registry (see Objective-C/C++ Support for instructions).

  1. In the Settings/Preferences dialog Ctrl+Alt+S, click Code Style, and then click the language.

  2. Select the link Set from in the upper-right corner and select Predefined style option from the list.

  3. In the list that appears, select the code style to copy the settings from:

    • Google
    • LLVM
    • LLDB
    • Qt
    • GNU
    • Microsoft
    • Stroustrup

    The following styles are applicable for braces placement only:

    • Allman braces
    • Whitesmiths braces
    • K&R braces
    • JavaScript Standart Style
    • Google JavaScript Style Guide
