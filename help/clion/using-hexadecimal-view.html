十六进制视图

对于整数类型的数字变量，可以检查十六进制表示形式，也可以检查原始格式。整数类型的十六进制视图可用于所有平台上的GDB和LLDB调试器，包括远程调试用例和WSL 。

目前，此功能是实验性的。

启用十六进制视图

  1. 调用维护弹出窗口：按Ctrl+Shift+Alt+/ ，或致电帮助|查找操作并搜索维护 。选择实验功能 。

  2. 选中cidr.debugger.value.numberFormatting.hex复选框：

  3. 转到设置/首选项|构建，执行，部署|调试器数据视图C / C ++ ，然后选中显示数字的十六进制值复选框。选择显示十六进制值来代替或与原始值一起显示：

现在，十六进制格式显示在“ 调试”工具窗口的“ 变量”窗格和编辑器的内联变量视图中：
