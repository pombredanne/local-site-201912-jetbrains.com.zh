文件模板变量

文件模板可以包含变量，这些变量在应用模板时将被其值替换。变量是一个以美元符号开头的字符串$然后是变量名。变量名称可以选择用大括号括起来。例如： $MyVariable和${MyVariable}是同一变量的不同符号。

预定义模板变量

以下预定义变量可以在文件模板中使用 ：

变量

描述

${CALL_SUPER}

覆盖生成期间的基本函数调用签名

${DATE}

当前系统日期

${DAY}

该月的当前日期

${DEFAULT_RETURN_VALUE}

函数的默认返回值

${DS}

美元符号$ 。此变量用于转义美元字符，因此不会将其视为模板变量的前缀。

${FILE_NAME}

新的C或C ++文件的名称

${HEADER_COMMENTS}

总是返回true并用于测试目的

${HEADER_FILENAME}

为类或源文件生成的头文件的名称

${HOUR}

当前时间

${INCLUDE_GUARD}

防止重复包含特定的头文件

${MINUTE}

当前分钟

${MONTH}

这个月

${MONTH_NAME_FULL}

当月的全名（一月，二月等）

${MONTH_NAME_SHORT}

当前月份名称的前三个字母（一月，二月，依此类推）

${NAME}

新实体的名称（文件，类，接口等）

${ORGANIZATION_NAME}

在项目设置中指定的组织名称（ Ctrl+Shift+Alt+S ）

${NAMESPACE_CLOSE}

重构期间创建的名称空间块的结尾

${NAMESPACE_OPEN}

重构期间创建的名称空间块的开始

${PRODUCT_NAME}

IDE的名称（例如，CLion）

${PROJECT_NAME}

当前项目名称

${RETURN_TYPE}

函数返回值的类型（用于生成新函数）

${SUIT_NAME}

Google测试套件名称

${TEST_NAME}

Google测试名称

${TIME}

当前系统时间

${USER_NAME}

对于Windows和Linux：当前用户的登录名

对于macOS：用户的注册全名

${YEAR}

今年

自定义模板变量

除了预定义的模板变量外，还可以指定自定义变量。如有必要，您可以使用以下命令在模板中定义自定义变量的值#set指示。

例如，如果要使用全名而不是通过预定义变量定义的登录名${USER} ，请使用以下结构：

#set（$ MyName =“ John Smith”）

如果模板中未定义变量的值，则在应用模板时，CLion会要求您指定它。
