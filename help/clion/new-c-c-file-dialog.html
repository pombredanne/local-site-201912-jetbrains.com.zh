新的C / C ++源/头文件对话框

此对话框打开，在项目视图中，当你右键点击一个文件或文件夹，并选择一个从上下文菜单中选择这些选项： 新| C ++类 ， 新增| C / C ++源文件 ， 新增| C / C ++头文件 。

常用选项

项目

描述

名称

在此字段中，输入新类或源/头文件的名称。

类型

如果需要，请从下拉列表中选择所需的文件类型，而不是默认类型。有关文件类型定义的更多详细信息，请参阅“ 代码样式C / C ++帮助”页面。

添加到目标

选中此复选框可将创建的文件添加到所选目标的源文件列表中。

目标窗格

在此窗格中，选择要包含创建的文件的所需目标或多个目标。

CMakeLists.txt文件窗格

在此窗格中，请参阅CMakeLists.txt变量，CLion建议在其中添加创建的文件。

单击此图标以打开“ 代码样式”对话框。
