CLion 2019.3帮助

创建和编辑配置

可用的运行/调试配置显示在导航栏上的切换器中（“ 视图” |“外观” |“导航栏” ）：

单击编辑配置，或选择运行。从主菜单中编辑配置以打开“ 运行/调试配置”对话框。在此对话框中，您可以创建，编辑和管理配置和模板。

运行/调试CMake项目的配置

  • 默认情况下，“ 自动为新的CMake目标生成运行配置”选项处于启用状态 。这意味着CLion使用项目目标中的数据以及CMakeLists.txt脚本中的可执行文件来创建配置。这适用于大多数情况，但有以下限制：

    • 如果已经存在具有相同目标的配置，则CLion将不会创建新的默认配置。

    • 如果已经创建了相同的配置，CLion将不会创建新的默认配置。

    • 如果手动创建的旧配置名称相同，则将其替换为自动创建的新配置。

    • 此外，如果启用了“ 自动删除丢失的CMake目标的运行配置”，则可以为已删除/过时的目标自动删除自动创建的配置。

  • 您始终可以在其中一个可用模板的基础上手动创建配置。

创建运行/调试配置

  1. 打开“ 运行/调试配置”对话框：

    • 选择运行|从主菜单编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按Shift+Alt+F10 ，然后按0或从弹出窗口中选择配置，然后按F4 。

  2. 在“ 运行/调试配置”对话框中，单击在工具栏上或按Alt+Insert 。该列表显示了默认的运行/调试配置。选择所需的配置类型。

    右侧窗格中显示的字段显示所选配置类型的默认设置。

    要更改默认运行/调试配置的设置，请展开“ 模板”节点，选择所需的配置类型，然后根据需要对其进行修改。有关详细信息，请参见更改配置模板 。

  3. 根据所选模板，填写配置设置。例如， CMake应用程序 ， 自定义构建应用程序 ， Gradle / Gradle本机应用程序 。

  4. 在启动之前部分中，定义是否要在运行之前启动任何外部工具或其他构建配置。要跳过构建阶段，请从启动前列表中删除“ 构建” 。

  5. 根据配置类型指定其他参数。有关运行/调试配置参数的说明，请参阅“ 运行/调试配置对话框”部分。

  6. 应用更改并关闭对话框。

编辑现有的运行/调试配置

  1. 请执行以下任一操作：

    • 从主菜单中，选择运行|编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按Shift+Alt+F10 ，然后按0以显示“ 编辑配置”对话框，或从弹出窗口中选择配置，然后按F4 。

  2. 在相应的运行/调试配置对话框中，根据需要更改参数。

如果在早期版本的CLion中创建了一个项目，则当将这样的项目添加到已经在同一窗口中打开的另一个项目时，其运行/调试配置可能会丢失。

为避免丢失运行/调试配置，建议您以最新版本打开一次这样的项目，并且仅在将其添加到另一个项目之后再打开。

启用服务工具窗口

您可以在专用工具窗口中一次管理多个运行/调试配置。例如，您可以启动，暂停和停止几个应用程序，跟踪它们的状态，并检查特定于应用程序的详细信息。

  1. 从主菜单中，选择运行|编辑配置或使用工具栏上的选择器。

  2. 在“ 运行/调试配置”对话框中，从左侧列表中选择“ 模板 ”。

  3. 在右侧窗格中，展开“ 服务”部分中的“ 可用配置” 。

  4. 请点击在“ 服务”中的“ 可用配置”部分中，选择所需的运行配置类型。

    您可以一一添加或删除多种配置类型。

    请注意，工具窗口将仅显示已为其创建一个或多个配置的配置类型。

    因此，如果添加项目中不存在任何配置的配置类型，则在创建该类型的配置之前，该类型将不会显示在工具窗口中。

  5. 应用更改并关闭对话框。具有所选配置的“ 服务”工具窗口将打开。

要显示或隐藏“服务”工具窗口，请选择“ 查看” |“隐藏” 。工具窗口|主菜单中的服务 。
