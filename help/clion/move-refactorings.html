移动并复制

通过移动重构，您可以在项目中移动文件，目录，类成员和声明。这样，CLion会跟踪这些移动并自动更正源代码中对已移动符号的所有引用。

提供以下移动重构：

  • “ 移动文件”重构将文件移动到另一个目录。

  • “ 移动目录”重构将目录移动到另一个目录。

  • “ 移动成员”重构将选定的类成员移动到新的或现有的类。

  • “ 移动声明”重构将顶级类型声明移动到新文件或现有文件。

将文件或目录移动到另一个目录

  1. 在“ 项目”工具窗口中选择文件或目录。

  2. 请执行以下任一操作：

    • 从主菜单中，选择“ 重构|移动 。

    • 从选择的上下文菜单中，选择“ 重构” |“选择” 。移动 。

    • 按F6 。

  3. 在“ 到目录”字段中，指定要将所选文件或文件夹移动到的文件夹。从列表中选择一个现有文件夹，或键入要创建的父文件夹的完整路径。

  4. 要根据重构结果更新对选定文件或文件夹的引用，请选中“ 搜索引用”复选框。

将班级成员移至另一个班级

  1. 在“编辑器”或“结构”视图中，选择要移动的类成员。

  2. 请执行以下任一操作：

    • 从主菜单中，选择“ 重构|移动 。

    • 从选择的上下文菜单中，选择“ 重构” |“选择” 。移动 。

    • 按F6 。

    • 打开的对话框取决于所选符号的类型。

  3. 选择要移动的班级成员，然后指定目标班级名称（现有的或新的）。

  4. 单击“ 移动”以继续，或单击“ 预览”以在实际重构之前检查结果。

    如果发生冲突，当有成员依赖于您移动的成员时，它们将显示在单独的对话框中。

将声明移动到另一个文件或名称空间

  1. 在编辑器或“结构”视图中，选择要移动到另一个文件的声明。

  2. 请执行以下任一操作：

    • 从主菜单中，选择“ 重构|移动 。

    • 从选择的上下文菜单中，选择“ 重构” |“选择” 。移动

    • 按F6

    • 打开的对话框取决于所选符号的类型。

  3. 选择要移动的声明。指定目标文件或名称空间（现有的或新的）：

  4. 单击“ 移动”以继续，或单击“ 预览”以在实际重构之前检查结果。

    如果发生冲突，当有成员依赖于您移动的成员时，它们将显示在单独的对话框中。

复制重构

使用复制重构，您可以将类，文件或目录及其整个结构从一个目录复制到另一个目录。

复制类，文件或目录

  1. 在其中一个视图中选择所需的项目，或在编辑器中将其打开。

  2. 请执行以下任一操作：

    • 从主菜单或选择上下文菜单中，选择“ 重构” |“选择” 。复制 。

    • 按F5 。

    • 在“ 项目”工具窗口中选择所需的类，然后使用以下命令将其拖动到目标位置Ctrl按下。

  3. 在打开的对话框中，指定新名称和目的地，然后单击确定 。
