生成构造函数

CLion可以生成一个构造函数，该构造函数使用相应参数的值初始化特定的类字段。

为一个类生成一个构造函数

  1. 在代码菜单上，单击生成 Alt+Insert 。

  2. 在“ 生成”弹出窗口中，单击“ 构造函数” 。

  3. 如果该类包含字段，请选择要由构造函数初始化的字段，然后单击OK 。

生成获取器和设置器

CLion可以为类中的字段生成访问器和更改器方法（ getter和setter ）。

生成类的获取器和设置器：

  1. 在代码菜单上，单击生成 Alt+Insert 。

  2. 在“ 生成”弹出窗口中，单击以下选项之一：

    • Getter生成访问器方法以获取类字段的当前值。

    • Setter生成用于设置类字段值的增变方法。

    • 使用Getter和Setter生成访问器和更改器方法。

  3. 选择要为其生成getter或setter的字段，然后单击“ 确定” 。
