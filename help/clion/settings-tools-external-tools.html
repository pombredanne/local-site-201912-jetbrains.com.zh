External Tools

File | Settings | Tools | External Tools for Windows and Linux
CLion | Preferences | Tools | External Tools for macOS
Ctrl+Alt+S

Define third-party standalone applications (code generators and analyzers, pre- and post- processors, database utilities, and so on) as External Tools to be able to run them from CLion.

To create and configure external tools, use the Settings / Preferences | Tools | External Tools dialog. Tools defined in this dialog appear as commands in the Tools | External Tools menu and in various context menus. They can also be assigned keyboard shortcuts (see the Configuring Keyboards and Mouse Shortcuts section).

Checkboxes

Use the checkboxes to enable or disable the tools and the tool groups. The items that are not currently selected are not available in the Tools and context menus.
