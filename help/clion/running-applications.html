Run and Rerun Applications

CLion enables running entire applications as well as unit tests.

To see the list of all currently running applications, select Run | Show Running List from the main menu.

If you have configured Before launch tools in a Run/Debug configuration, CLion runs them first, and then will run the application. Otherwise, the program will start immediately.

Note that after you've started a run session, the icon that marks the Run tool window and in the Run/Debug Configuration Selector toggles to to indicate that the run process is active.

If your are working with a CMake project, choose the desired Cmake profile before running your application.

Rerun an application

You can re-run an application if its tab is still opened in the Run window. The program re-runs with the initial settings.

If you want to re-run without loosing focus in the editor tab you are working in, press Shift+F10.

  1. In the Run window, select the tab where the desired application is opened.

  2. In the toolbar of the Run window, click the Rerun button , or press Ctrl+F5.
