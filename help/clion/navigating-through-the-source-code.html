源代码导航

您可以使用不同的操作和弹出窗口在编辑器中快速浏览代码。有关在编辑器和工具窗口之间导航的详细信息，请查看编辑器基础知识 。

您可以从观看视频教程开始，以检查CLion中可用的导航选项。

导航与插入符

  • 要向后导航，请按Ctrl+Alt+Left 。要向前导航，请按Ctrl+Alt+Right 。

  • 要导航到最后编辑的位置，请按Ctrl+Shift+Backspace 。

  • 要在编辑器中找到当前插入符号的位置，请按Ctrl+M 。如果您不想滚动大文件，此操作可能会有所帮助。

    但是，您可以按Up和Down箭头键可以达到相同的效果。

  • 要突出显示您要查找的插入符号处的单词，请选择“ 编辑” |“其他”。查找|从主菜单中的插入符号中找到Word 。

  • 要查看插入符号当前位于什么元素上，请按Alt+Q 。

  • 要在开始和结束代码块大括号之间切换，请按Ctrl+Shift+M 。

  • 要在代码块之间导航，请按Ctrl+Shift+[要么Ctrl+Shift+] 。

移动插入符号

您可以使用不同的动作通过代码移动插入符号。您还可以配置插入字词和在线中断时插入符号应停止的位置。

  • 要将插入符号移动到下一个单词或上一个单词，请按Ctrl+Right要么Ctrl+Left 。

    默认情况下，CLion将插入符号移动到当前单词的末尾。

    当您将插入符号移动到上一个单词时，插入符号将放置在当前单词的开头。使用这些操作时，可以配置插入标记的位置。

    在 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般在“ 插入符 移动”部分中，使用“ 按字移动时”和“换行时”选项来配置插入符的行为。

  • 要将插入符号向前移动到下一个段落或向后移动到上一个段落，请按Ctrl+Shift+A并搜索将插入符号向前移动段落或将插入符号向后移动段落动作。

    您也可以选择一个文本，然后将插入符号向前或向后移动到一个段落。按Ctrl+Shift+A并搜索带有选择项的将插入符号向前移动段落或带有选择项的将插入符号向前 移动段落 。

    如果需要，可以为这些操作分配快捷方式。有关详细信息，请参阅键盘快捷键 。

查找最近的位置

您还可以使用“ 最近的位置”弹出窗口检查您最近查看或更改的代码。

  • 要打开“ 最近的位置”弹出窗口，请按Ctrl+Shift+E 。该列表从顶部的最近访问位置开始，并包含代码片段。

  • 在弹出窗口中，使用相同的快捷方式或选中仅显示更改的复选框，以仅查看更改了代码的位置。

  • 要搜索代码段，请在“ 最近的位置”弹出窗口中，开始输入搜索查询。您可以按代码文本，文件名或面包屑搜索。

  • 要从搜索结果中删除位置条目，请按Delete要么Backspace 。

    请注意，删除的位置也会从您通过以下方式访问的条目列表中删除： Ctrl+Alt+Left捷径。

使用书签进行导航

  • 要创建匿名书签，请将插入符号放在所需的代码行中，然后按F11 。

  • 要创建带有助记符的书签，请将插入号放置在所需的代码行中，按Ctrl+F11并选择一个数字或字母作为助记符。

  • 要显示下一个或上一个书签，请在主菜单中选择“ 导航”。书签|下一个书签或导航|书签|上一个书签 。

  • 要打开“ 书签”对话框，请按Shift+F11 。您可以使用此对话框来管理书签，例如，删除书签，对书签排序或为其提供简短说明。

  • 要导航到带有字母助记符的现有书签，请按Shift+F11然后按一下您需要的字母。CLion使您返回编辑器和相应的书签。

  • 要导航到带有数字助记符的现有书签，请按Ctrl和书签的编号。

  • 每个创建的书签都会反映在“ 收藏夹”中 Alt+2工具窗口，您还可以使用该窗口导航到书签。

前往声明

您可以根据用法来导航至变量，函数，类或类型的声明：

  • 将插入符号放在需要的符号上，然后按Ctrl+B 。

  • 对于类型声明，请按Ctrl+Shift+B 。

  • 您也可以指向符号保持Ctrl按下，然后单击，当它变成超链接时。弹出消息将显示声明信息：

    再次单击以导航到声明。

转到定义

  • 您可以使用以下任一方法来跟踪类的实现和重写功能： / ， / 装订线图标或相应的快捷方式。

  • 按Ctrl+Alt+B导航到插入符号（变量，函数，类）上代码实体的定义。

  • 按Ctrl+U导航到超级定义。

转到标题/源

要在源文件和头文件之间切换，可以使用专用的转到标题/源操作而不是转到相关符号 。

  • 要调用转到标题/源 ，请按F10或致电导航|主菜单中的标题/源 。

    该操作将立即在编辑器中打开一个文件，或显示带有可用选项的弹出窗口：

    在搜索目标文件时， 转到页眉/源使用了一些试探法：

    • 最近切换的文件始终位于列表的顶部。

    • 接下来，使用相同目录中具有匹配名称的文件（如果有）（例如， myfile.h / myfile.cpp ）。

    • 然后，在后台执行基于定义/声明的搜索，并将新项目添加到弹出窗口中。

    • 如果在500毫秒内未找到单个目标，则CLion将显示一个交互式弹出窗口，其中添加了新项目，您可以选择要导航到的位置。

    当前，搜索仅限于直接包含者/包含对象，以避免来自不同目标的相同符号的干扰。

使用“选择范围”导航到“项目”视图中的目标

您可以在“ 项目”工具窗口中自动找到目标。

  1. 如果在编辑器中打开了文件，请按Alt+F1用于“ 选择”弹出窗口。

  2. 在弹出窗口中，选择“ 项目视图” ，然后按Enter 。

    CLion在“ 项目视图”窗口中找到目标。

在“项目”工具窗口中找到文件

您可以使用“ 单击打开文件”和“ 始终选择打开的文件”操作来在“ 项目”工具窗口中找到文件。

  1. 在“ 项目”工具窗口中，右键单击“ 项目”工具栏，然后从上下文菜单中选择“ 始终选择打开的文件” 。之后，CLion将跟踪当前在活动编辑器选项卡中打开的文件，并自动在“ 项目”工具窗口中找到它。

  2. 您也可以选择“ 单击打开文件”选项。在这种情况下，当您在“ 项目”视图中单击文件时，CLion将在编辑器中自动将其打开。

使用面包屑进行导航

面包屑有助于在活动编辑器选项卡中打开的文件中跟踪作用域层次结构内的位置。对于C ++，面包屑表示名称空间，类，结构，函数和lambda：

要跳转到代码中的某个元素，请单击带有其名称的面包屑。

  1. 默认情况下，显示面包屑并将其放在编辑器的底部。要隐藏或重新放置它们，请右键单击面包屑，然后选择“ 面包屑|顶部或面包屑|不显示或在“设置/首选项”中配置其位置。 编辑器一般|面包屑 。

  2. 在相同的面包屑设置对话框中，您还可以通过选择/清除相应的复选框来启用/禁用特定语言的面包屑。

  3. 要配置面包屑突出显示的颜色 ，请转到“设置” /“首选项” |“设置”。编辑器配色方案|常规 ，然后选择编辑器|列表中的面包屑 。

在“结构”视图弹出窗口中找到一个代码元素

您可以使用结构视图弹出窗口在正在处理的文件中找到代码元素。

  1. 要打开结构视图弹出窗口，请按Ctrl+F12 。

  2. 在弹出窗口中，找到您需要的项目。您可以开始为CLion键入元素的名称以缩小搜索范围。按Enter返回到编辑器和相应的元素。

浏览方法

  • 按Alt+Down要么Alt+Up 。

  • 为了在视觉上分离代码中的方法， 在 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|外观并选择显示方法分隔符选项。

  • 要打开“ 结构”工具窗口，请按Alt+7 。

使用镜头模式

镜头模式使您无需实际滚动即可预览代码。当您将鼠标悬停在滚动条上时，默认情况下该模式在编辑器中可用。将鼠标悬停在警告或错误消息上时，此功能特别有用。

  • 要禁用镜头模式，请右键单击位于编辑器右侧的代码分析标记，然后在上下文菜单中清除“ 在滚动条悬停时显示代码镜头”复选框。

  • 作为备选， 在 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|出现并清除滚动条悬停上的显示代码镜头复选框。

查找线或列

  1. 在编辑器中，按Ctrl+G 。

  2. 在“ 转到行/列”对话框中，指定行号或列号，或同时指定两者，并用：分隔它们，然后单击“ 确定” 。

  3. 如果您不想在编辑器中看到行号， 在 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|外观并清除显示行号复选框。

为了更快地访问，请将快捷方式分配给“ 显示行号”操作。

查找文件路径

  1. 在编辑器中，按Ctrl+Alt+F12或在上下文菜单中，选择“ 在Finder中显示” 。

  2. 在“ 在查找器中显示”弹出窗口中，选择要在路径查找器中打开的文件或目录，然后按Enter 。

查找最近的文件

您可以使用“ 最近的文件”弹出窗口搜索最近和最近编辑的文件。

  • 要打开带有最近文件列表的“最近文件”弹出窗口，请按Ctrl+E 。

  • 要仅查看最近编辑的文件，请按Ctrl+E再次选择“ 仅显示更改”复选框。

  • 要在弹出窗口中搜索项目，请开始输入搜索查询。CLion根据您的搜索显示结果。
