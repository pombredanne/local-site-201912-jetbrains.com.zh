CLion 2019.3帮助

Perforce

在你开始前

  1. 按照Perforce官方网站上的说明安装Perforce客户端。

  2. 使用Perforce软件仓库创建一个帐户。

从CLion中的Perforce开始

  1. 使用您的Perforce客户端创建客户端规范。

  2. 使用Perforce客户端从Perforce软件仓库获取源文件。

  3. 一旦本地工作副本在您的计算机上， 请将您的本地目录与Perforce关联 。

    之后，您将能够打开源文件进行编辑，并使用CLion执行与Perforce相关的常规任务。

Perforce支持

  • 启用与CLion的Perforce集成后， Perforce项将出现在VCS菜单以及编辑器和“项目”工具窗口的上下文菜单中。

  • Perforce控件下的文件夹中的文件根据其状态突出显示。有关文件状态突出显示约定，请参见文件状态突出显示。

  • 修改结果显示在 版本控制窗口Alt+9 。

  • 使用Perforce集成时，打开 版本控制窗口Alt+9 ，其中显示以下数据：

    • 根据您通过CLion用户界面指定的设置生成的所有命令。

    • 有关执行生成的Perforce命令的结果的信息消息。

    • 错误消息。
