更改项目根目录

当您打开CMake项目时，CLion会将顶部CMakeLists.txt的位置视为项目根目录 。创建新项目时，CLion还将CMakeLists.txt文件放在根目录下。

但是，您的项目可能具有不同的结构：例如，主CMakeLists.txt脚本可以位于某个子目录中（例如prj_dir / main ）。在这种情况下，请使用工具| CMake |更改项目根命令，以将CLion明确指向您要视为根目录的目录。

请注意， 更改项目根目录不会影响文件的实际位置。另外，它不会自动重新加载项目，因此请确保在更改项目根目录后手动重新加载项目。
