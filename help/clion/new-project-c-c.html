新项目：C / C ++

档案|新项目

对于C和C ++应用程序 ，请使用以下选项：

左窗格

项目

描述

C ++可执行

如果要使用C ++作为项目语言和可执行目标，请选择此选项。

C ++库

如果要使用C ++作为项目语言和库目标，请选择此选项。

C可执行文件

如果要使用C作为项目语言和可执行目标，请选择此选项。

C库

如果要使用C作为项目语言和库目标，请选择此选项。

右窗格

地点

在此文本字段中，输入项目文件夹的路径或单击选择。如果要使用默认参数，则什么也不做。

语言标准

单击此处从下拉列表中选择所需的语言标准。

图书馆类型
（当选择“库”作为目标时可见）

单击此处选择库类型：共享或静态。
