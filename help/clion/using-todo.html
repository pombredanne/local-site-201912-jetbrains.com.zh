TODO评论

有时，您需要标记部分代码以供将来参考：优化和改进，可能的更改，要讨论的问题等。CLion允许您添加特殊类型的注释，这些注释在编辑器中突出显示 ，被索引并在TODO工具窗口中列出。这样，您和您的队友就可以跟踪需要注意的问题。

默认情况下，CLion识别两种模式： TODO和FIXME无论大小写这些模式可在任何受支持文件类型的行注释和块注释内部使用。您可以根据需要修改默认模式或添加自己的模式 。

要创建多行TODO项目，请缩进第一行之后的注释行。如果没有缩进，则将行视为常规注释行。要禁用多行待办事项， 在 设置/首选项对话框Ctrl+Alt+S ，转到编辑器| TODO并清除以下行中的将缩进文本视为同一TODO复选框的一部分。

您可以在“ 待办事项”工具窗口（“ 视图” |“工具”窗口|“待办事项”或“ Alt+6 ）。要跳至源代码中相关注释的位置，请在TODO工具窗口中双击相应的TODO项。要在每次选择相应的TODO项目时自动跳至相关注释的位置，请右键单击TODO工具窗口标题栏，指向“ 查看选项” ，然后选择“ 单击导航” （默认情况下启用此选项）。

添加自定义模式并过滤待办事项

您可以添加自己的模式并过滤列表以仅显示与某些模式匹配的TODO项目。例如，您可以选择在代码中使用OPTIMIZE在“ 待办事项”工具窗口中查看它们时，请更改样式并忽略所有其他类型的待办事项。

  1. 在 设置/首选项对话框Ctrl+Alt+S ，选择编辑器|待办事项 。

  2. 使用正则表达式指定自定义模式。

    例如，为OPTIMIZE项目，单击在“ 待办事项”对话框的“ 模式”部分中，键入以下正则表达式：

    \ boptimize \ b。*

    然后单击确定以保存新模式。

  3. 添加过滤器以将TODO模式分组，并分别在TODO工具窗口中查看相应的TODO项目。

    例如，添加Optimization使用新模式进行过滤，请点击在“ 待办事项”对话框的“ 过滤器”部分中，指定Optimization作为其名称，然后选择要包含在此过滤器中的新模式。

    然后单击确定以保存新的筛选器。

  4. 单击“ 确定”以在“ TODO设置”对话框中应用更改。

  5. 要应用新的过滤器，请在TODO工具窗口中单击并选择优化过滤器。

您为模式选择的图标显示在TODO工具窗口中，以更好地区分各种TODO项目。通过启用模式的区分大小写复选框，可以强制模式仅与指定的大小写匹配。
