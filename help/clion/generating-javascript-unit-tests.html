创建JavaScript单元测试

CLion与大多数流行的JavaScript单元测试框架集成，并为编写JavaScript单元测试提供了丰富的编码帮助。要获得支持框架的编码帮助，请下载所需的框架并将其配置为CLion JavaScript库 。

创建JavaScript单元测试

  1. 在src文件夹旁边创建一个单独的测试文件夹（ File | New | Directory ）。该文件夹的名称取决于您使用的测试框架。

  2. 马克测试作为测试源文件夹 。

  3. 对于每个生产文件，通过在测试文件夹的上下文菜单上选择（ File | New | JavaScript File ，创建一个单独的测试文件。测试文件名的格式取决于您使用的测试框架。
