文件嵌套对话框

当您在“项目”工具窗口中的标题栏的上下文菜单上单击“ 文件嵌套”时，将打开此对话框。

使用此对话框可以配置名称相同但后缀不同的文件的表示形式。CLion可以将此类文件束显示为纯结构，也可以将父文件显示为包含子文件的文件夹（嵌套）。要配置文件嵌套，请设置父文件后缀和子文件后缀之间的对应关系。

CLion提供了一组预定义规则。您可以编辑这些规则以及定义自己的自定义规则。

请注意，嵌套规则仅适用于同一目录中具有相同名称的文件。

如果两个文件的名称匹配一个模式，但是文件存储在不同的目录中，CLion不会在视觉上“移动”它们中的任何一个。

描述

显示与嵌套名称相同的文件

  • 选中此复选框后，CLion会根据列表中的模式识别子文件，并按相应的父项显示它们。

  • 清除复选框后，CLion会在相同级别显示父母和孩子。

Alt+Insert

单击此按钮添加新行，并指定“父”文件后缀和匹配的“子”文件后缀。

Alt+Delete

单击此按钮可从列表中删除所选规则。

重置为默认

单击此按钮可放弃所有自定义模式并重新加载默认规则。
