提取功能

调用“ 提取函数”重构时，CLion将分析所选代码块，并检测作为所选代码段输入的变量和为其输出的变量。

检测到的输出变量用作提取函数的返回值。

int main（）{int x = 15;整数y = 10; int z = x-y;返回0; }

提取功能

  1. 在编辑器中，选择要转换为功能的代码块。

    组成该方法的代码片段不一定必须是一组语句。它也可以是代码中某个地方使用的表达式。

  2. 从主菜单或所选内容的上下文菜单中，选择“ 重构” |“选择” 。提取物|方法或按Ctrl+Alt+M 。

  3. 在打开的“ 提取函数”对话框中，指定新函数的名称。

  4. 指定函数的返回类型。

  5. 如果尚未声明该函数，请选择声明位置：当前位置上方或下方。

  6. 在参数窗格中：

    • 添加新参数或删除不必要的参数

    • 通过单击相应的参数行并输入新的名称和类型来重命名参数或/和更改其类型

    • 重新排序列表中的参数

    • 更改函数的返回类型

  7. 在“ 签名预览”窗格中检查结果，然后单击“ 提取”以创建函数。

    选定的代码片段将替换为函数调用。
