产生选单

生成菜单可帮助您在代码中快速创建标准代码结构。根据当前上下文，插入符号位置或选定的代码，CLion会为您提供要生成的构造的列表。

在一个*.cpp或头文件，可以生成构造函数或析构函数，访问器，覆盖或实现功能。

使用“生成”菜单创建代码构造

  1. 打开要在其中生成代码的文件，然后将插入号放置在相应的位置，例如在实现类时。

  2. 请执行以下任一操作：

    • 电话代码|从主菜单生成 。

    • 右键单击编辑器，然后从上下文菜单中选择“ 生成 ”。

    • 按Alt+Insert 。

  3. 在编辑器中显示的“ 生成”菜单中，选择所需的构造。

    根据选定的项目，CLion会立即插入该结构，或显示一个对话框以供其他输入。
