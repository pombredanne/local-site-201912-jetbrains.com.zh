一般调试程序

  1. 配置调试器设置 。

  2. 对于CMake项目，选择所需的CMake配置文件 。

  3. 如有必要，请创建或修改现有的运行/调试配置 。

  4. 在代码中放置断点 。

  5. 点击调试纽扣或使用其他选项来启动调试会话 。

根据需要暂停或恢复会话。

在调试会话期间，您可以单步执行代码 ， 评估表达式 ，即时更改值， 设置 监视和监视点 。

启动调试会话后， 标记“ 调试工具”窗口的图标切换到指示调试过程处于活动状态。

有用的调试器快捷方式

行动

快捷键

切换断点Ctrl+F8
恢复程序F9
越过 F8
步入 F7

停止

Ctrl+F2
查看断点详细信息/所有断点Ctrl+Shift+F8

插入符号处的调试代码

Shift+F9 （在main方法内），或Shift+Alt+F9
