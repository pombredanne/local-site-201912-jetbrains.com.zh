Clean System Cache

CLion caches a great number of files, therefore the system cache may one day become overloaded. In certain situations the caches will never be needed again, for example, if you work with frequent short-term projects. Also, the only way to solve some conflicts is to clean out the cache.

Cleaning out the system caches causes a complete rebuild of all the projects ever run in the current version of CLion.

Clean the system caches

  • Select File | Invalidate Caches/Restart from the main menu.

    The Invalidate Caches message appears informing you that the caches will be invalidated and rebuilt on the next start. Use buttons in the dialog to invalidate caches, restart CLion, or both.

It is important to note the following:

  • The files are not actually deleted until CLion restarts.

  • Opening and closing a project does not result in deleting any files.
