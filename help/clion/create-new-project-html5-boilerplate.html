New Project: HTML5 Boilerplate

Welcome Screen | Create New Project

In this dialog, create stubs for HTML5 Boilerplate front-end applications. To start using HTML5 Boilerplate in an existing project, download the latest stable release from the HTML5 Boilerplate official website and extract it in your project. Alternatively, open the embedded Terminal (Alt+F12) and type npm install html5-boilerplate at the command prompt.

Item

Description

Location

In this field, specify the path to the project folder where the project-related files will be stored.

Version

From this list, choose the version of the template in accordance to which the stub will be generated. Click to refresh the list of available template versions.
