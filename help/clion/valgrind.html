Valgrind

Use this page to configure Valgrind Memcheck.

Item

Description

Valgrind executable

In this field, specify the path to Valgrind excutable home directory, or click to select it in the dialog that opens.

Analysis options

In this field, specify the required Memcheck options.

Add suppression files:

In this field, specify a list of suppression files which you can use to turn off Valgrind checks when needed. The typical case here, for example, would normally be to suppress OpenSSL or Boost.
