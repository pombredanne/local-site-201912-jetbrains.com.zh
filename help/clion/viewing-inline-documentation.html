快速文档查找

快速文档弹出 Ctrl+Q是一种通用工具，可帮助您在插入符号处获取有关代码元素的更多信息。根据调用快速文档的元素，弹出窗口可以包括：

  • 功能签名的详细信息。

  • 代码文档 （常规注释或Doxygen注释）。

  • 推断类型 ，这对于可能缺少显式类型的现代C ++尤其重要。例如：

  • 宏替换可帮助您更好地理解和调试嵌套宏（例如Boost。测试或捕获宏）。在弹出窗口中，您将看到最终的宏替换格式正确，并带有突出显示的字符串和关键字：

在插入符号处查看符号的快速文档

  • 在主菜单中，选择“ 查看” |“选择”。快速文档查找 。

  • 按Ctrl+Q 。

  • 如果在编辑器设置中选中了“ 在鼠标移动时显示快速文档”复选框，则只需将鼠标悬停在所需的符号上即可。

使用箭头按钮或按Left / Right切换到上一个或下一个文档页面。

更改快速文档的字体大小

  • 请点击并移动滑块。

  • 旋转鼠标滚轮，同时保持Ctrl按键。要使用此功能，需要在常规编辑器设置中使用Ctrl /⌘+鼠标滚轮启用“ 更改字体大小（缩放）” 。
