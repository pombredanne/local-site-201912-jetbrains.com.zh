快速入门指南

0。在你开始前

CLion是跨平台的IDE吗？

是的，您可以在Windows，macOS和Linux上安装和运行CLion（有关在FreeBSD上进行安装的信息，请参阅此说明 ）。

我可以使用哪些编译器和调试器？

CLion支持GCC ， Clang和Microsoft Visual C ++编译器（请注意，可以将MSVC调试作为实验功能 ）。您也可以使用Intel C ++编译器 ，但是请注意，它不受官方支持（请参阅相应的问题 ）。

CLion捆绑了GDB和LLDB调试器，您可以切换到自定义GDB二进制文件（有关详细信息，请参见“ 调试”一章）。

支持哪些构建系统？项目格式是什么？

CLion与CMake构建系统完全集成：您可以无缝地创建，打开，构建以及运行/调试CMake项目 。CMake本身捆绑在CLion中，因此除非您决定使用自定义版本，否则无需单独安装它。

除了CMake，CLion还支持编译数据库和Gradle项目。当前，您不能在CLion中创建这些类型的新项目，但是您可以使用完整的代码洞察力来打开和管理现有项目。

有了Gradle支持，您可以使用新的Gradle C ++项目并作为常规应用程序运行Gradle任务。

除了CMake，CLion还支持编译数据库和Gradle项目。当前，您不能在CLion中创建这些类型的新项目，但是您可以使用完整的代码洞察力来打开和管理现有项目。

编译数据库提供了一种在CLion中开发非CMake和非Gradle项目的方法。您可以为使用另一个系统生成的源创建一个compile_commands.json ，然后在CLion中将该文件作为项目打开（例如，查看它如何帮助管理Makefile应用程序 ）。使用自定义目标，您可以使用外部工具构建此类项目，然后使用自定义配置运行/调试它们。

我需要提前安装任何东西吗？

对于C / C ++项目，CLion使用GCC / G ++，Clang或MSVC工具集。

在Windows上，这意味着如果计划使用Microsoft Visual C ++编译器，则可以在MinGW / MinGW-w64或Cygwin环境， WSL或Visual Studio中进行选择。有关详细信息，请参阅《 快速教程：在Windows上配置CLion》 。

在macOS上，必需的工具可能已经安装。如果不是，请按照在macOS上配置CLion中所述更新命令行开发人员工具 。

在Linux上，可能还预安装了编译器和make 。否则，对于Debian / Ubuntu，请安装build_essentials软件包，并根据需要安装llvm软件包以获取Clang。

是否还支持除C ++之外的其他语言？

是的，CLion完全支持Python ， Objective-C / C ++ ，HTML（包括HTML5），CSS，JavaScript和XML。通过捆绑的插件实现对这些语言的支持，这些插件默认情况下处于启用状态。有关更多详细信息，请参见其他语言的CLion功能 。

您可以安装其他插件来获得CLion支持的更多语言（例如Rust ， Kotlin / Native ， Swift或Markdown ）。查看有价值的语言插件，并在“设置” /“首选项”中浏览“ 插件”页面Ctrl+Alt+S 。

6。构建并运行

运行/调试配置

对于CMake或Gradle项目中的每个目标，CLion创建一个运行/调试配置。它是一个命名的运行/调试设置，包括目标，可执行文件，传递给程序的参数以及其他选项。

运行/调试配置是从模板生成的，例如CMake Application ， Google Test或Remote GDB Debug 。模板是可自定义的：当您编辑模板参数时，您将更改稍后将通过此模板创建的所有配置的默认设置。

可从“ 运行”菜单或配置切换器访问“ 编辑配置”对话框。您可以在此处管理模板以及添加，删除或编辑配置。例如，您可以自定义要在启动之前执行的步骤-调用外部工具（包括远程工具），使用CMake install甚至运行其他配置。

要启动程序，请选择所需的配置，然后使用 运行菜单或按Shift+F10 。或者，通过按调用“ 运行任何内容”对话框Ctrl两次并开始输入配置名称：

8。在运行时进行分析

有些漏洞和错误只能在程序执行期间显示：内存泄漏，未初始化的访问，并发问题，未定义的行为等。为了帮助您解决运行时问题，CLion集成了Valgrind Memcheck和Google Sanitizers 。另外，您可以使用内置的CPU Profiler分析应用程序性能。对于这些工具，CLion提供可视化的输出和便捷的功能，例如导入/导出分析结果的选项。

Valgrind，消毒剂和配置文件的设置位于“ 设置/首选项”下。构建，执行，部署|动态分析工具 。

Valgrind Memcheck

CLion中的Valgrind Memcheck可通过WSL在Linux，macOS和Windows上运行。您需要安装 Valgrind，将CLion指向二进制文件，设置分析标志，然后使用“运行” |“运行”程序运行程序。运行Valgrind Memcheck或 。

Google消毒剂

Linux的某些版本的Clang和GCC支持Google Sanitizers。看看我们有关在CLion中使用消毒剂的详细指南 。

CPU Profiler

CLion的探查器 （可在Linux和macOS上使用）收集应用程序的用户代码和内核代码的性能数据。探查器收集性能指标，并在火焰图和统计视图中将其可视化。要运行它，请调用运行|设定档或使用导航栏中的按钮。
