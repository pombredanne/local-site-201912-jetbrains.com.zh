CLion 2019.3帮助

目标C / C ++

CLion使您能够像使用C或C ++编写的代码一样全面而有效地使用Objective-C / C ++文件。

CLion不提供对Objective-C / C ++的应用程序支持。要构建和运行XCode / Objective-C项目，我们建议您切换到AppCode

您可以将扩展名为.m和.mm的 Objective-C / C ++文件打开，创建和添加到CMake或编译数据库项目中，并使用CLion提供的所有功能进行代码检查，帮助和重构。实时和代码模板也可用于Objective-C / C ++代码。有关更多详细信息，请参见不同语言的CLion功能 。

要配置特定于Objective-C / C ++的设置，请使用“ 设置/首选项”对话框的以下页面Ctrl+Alt+S ：

  • 编辑器检验|目标C

  • 编辑器文件和代码模板 （并选择“ 代码”选项卡）

  • 编辑器实时模板

要为Objective-C / C ++启用代码样式和颜色方案，您需要在注册表中设置clion.enable.objc.settings项。为此，请转到帮助|。在主菜单中找到“操作” （或按Ctrl+Shift+A ）并开始输入Registry ：

从建议的选项列表中选择注册表 ，然后键入clion.enable.objc.settings让CLion查找相应的密钥，然后选择复选框：

因此，“ 设置/首选项”对话框的“ 配色方案”和“代码样式”页面现在涵盖了Objective-C / C ++：
