更改配置模板

对于特定类型的运行/调试配置，可以为一个或多个参数设置默认值，并将其保存为模板以备将来使用。在这种情况下，下次创建该类型的新配置时，对话框的相应字段将已经包含指定的值。

更改默认值不会影响已经存在的运行/调试配置。

设置运行/调试配置的默认值

  1. 在“ 运行/调试配置”对话框的左窗格中，选择“ 模板”节点。

  2. 要禁用自动为新目标生成运行配置 ，请取消选中右窗格的为新CMake目标自动生成运行配置复选框。

  3. 要禁用自动删除缺少或过期目标的运行配置 ，请取消选中“ 自动删除缺少CMake目标的运行配置”复选框。

  4. 要启用新进程的静默启动，请取消选中右窗格的“ 确认进程终止后重新运行”复选框。否则，系统将提示您确认其他进程正在运行时启动新进程。

  5. 在右侧窗格的“ 临时配置限制”字段中，指定要存储并显示在“ 选择运行/调试配置”列表中的临时配置的最大数量。不执行任何操作以保留5个配置的默认值。

  6. 在“运行/调试配置”对话框的左侧窗格中，展开“ 模板”节点。

  7. 在“ 模板”节点下选择所需的配置类型。相应的配置模板出现在右侧窗格中。

  8. 选择所需的配置类型。

  9. 在右窗格中指定所需的参数，然后单击“ 应用”以保存模板。
