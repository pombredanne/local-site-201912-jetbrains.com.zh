提取变量

提取变量重构将所选表达式的结果放入变量中。它声明一个新变量，并将该表达式用作初始化程序。原始表达式将替换为新变量。

您可以自己选择要用变量替换的表达式。您也可以使用智能表达式选择，在这种情况下，CLion将帮助您选择所需的表达式。

此重构适用于C / C ++，Objective-C / C ++，Python和JavaScript 。

就地提取变量

  1. 在编辑器中，选择要用变量替换的表达式。您可以自己执行此操作，也可以使用智能表达式选择功能让CLion帮助您。因此，请执行以下任一操作：
    • 突出显示该表达式。然后选择重构|提取物|主菜单或上下文菜单上的变量 。

      或者，按Ctrl+Alt+V 。

    • 将光标放在表达式之前或之中。选择重构|在主菜单或上下文菜单上引入变量 。或按Ctrl+Alt+V 。

  2. 如果找到一个以上的选定表达式，请在“ 找到多个实例”弹出菜单中选择“ 仅替换此实例”或“ 替换所有实例 ”。要选择所需的选项，只需单击它。或者，使用向上和向下箭头键导航到所需的选项，然后按Enter选择它。

  3. 指定变量的名称。请执行以下任一操作：

    • 从列表中选择建议的名称之一。为此，请双击适当的名称。或者，使用向上和向下箭头键导航到所需的名称，然后Enter选择它。

    • 通过键入来编辑名称。名称将显示在带有红色边框的框中，并在您键入时更改。完成后，按Enter 。

  4. 如果要为变量应用自动类型声明，请在弹出窗口中选择“ 选择自动”复选框

    请注意，IDE会将您的选择保存为默认设置，以供下次尝试进行Extract变量重构时使用。
