CLion 2019.3帮助

主密码/重置主密码对话框

当您在“ 输入主密码”或“ 更改主密码”对话框中单击“ 重置”时 ，将打开此对话框 。

如果重置主密码，则当前存储在密码数据库中的所有密码都将丢失。

指定新的主密码。

您可以将密码设置为空。因此，CLion在访问密码数据库时不会要求您输入密码。但是，在这种情况下，密码的安全性将较低。

要返回上一个对话框，请点击Enter或Change 。

项目

描述

新密码

指定新的主密码。

确认密码

再次输入新的主密码。

使用OS用户凭证加密

仅适用于Windows：如果选中此复选框，则主密码本身将以加密方式存储在密码数据库中。（您的Windows登录凭据将用于加密。）

因此，CLion在访问密码数据库时不会要求您输入主密码。

输入或更改

如果您决定不重设密码（例如，您记得它），请单击此按钮以返回上一个对话框。这可以是“ 输入主密码”或“ 更改主密码”对话框 。
