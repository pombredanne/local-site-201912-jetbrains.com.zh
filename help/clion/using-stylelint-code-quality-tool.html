Stylelint

与克利翁集成Stylelint这样你就可以在IDE中检查你的CSS代码。编辑CSS文件时，Stylelint会自动唤醒，并突出显示检测到的问题 。要在信息弹出窗口中查看问题的描述，请将鼠标指针悬停在“ 验证”侧栏中的条纹上。要检查错误并应用建议的快速修复方法 ，请按Alt+Enter 。

在开始之前，请确保您的计算机上装有Node.js。

安装Stylelint

  • 在嵌入式终端 （ Alt+F12 ），输入：

    npm install -g stylelint

您还可以按照npm和Yarn中的描述，在Node.js和NPM页面上安装stylelint软件包。

激活和配置Stylelint

  1. 在 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” |样式表| Stylelint 。

  2. 在打开的Stylelint页面上，选中“ 启用”复选框以激活Stylelint。之后，对话框中的控件将变为可用。

  3. 在“ 节点解释器”字段中， 指定要使用的Node.js解释器。 如果选择项目别名，CLion将自动从Node.js和NPM页面上的Node解释器字段中使用项目默认解释器 。

    在大多数情况下，CLion会检测项目的默认解释器并填写字段本身。您还可以选择其他已配置的本地解释器或单击并配置一个新的。

  4. 在Stylelint套件字段中， 指定在全局或当前项目中安装的stylelint软件包的位置。

  5. 如果您希望Stylelint在编辑JavaScript文件并选中其中的“样式表”注入时也自动唤醒，请选中“ 处理JS文件”复选框。
