CLion 2019.3帮助

有价值的非捆绑式插件

CLion插件存储库包含1200多个插件，您可以安装它们以丰富现成的CLion功能。这些插件中的一些已经捆绑在CLion中，并且不需要事先采取任何措施即可开始使用它们。但是，还有其他有用和定性的插件可以改善甚至改变CLion中的工作流程。本文提供了一些此类有价值的插件。

请注意，第3方插件未经CLion团队官方测试或支持，因此我们不能保证其稳定性。该编译的选择标准是：下载统计信息，等级，维护状态和用户反馈。

插件按其目标任务分组，每个任务在相应的章节中进行了描述：

  • 学习IDE
  • 熟悉UI
  • 调优代码编辑器
  • 集成更多框架并构建系统
  • 添加语言和文件类型支持
  • 方便调试
  • 获取分析数据
  • 收集统计数据
  • 扩大与VCS相关的行动范围
  • 自动构建，跟踪问题和查看代码
  • 保持创造力和生产力

撤销数据库

为CLion中的Linux C / C ++应用程序集成了Undo的记录，倒带和重放技术。

使用撤消，您可以将程序的执行记录下来成一条指令，以进行进一步的重放和分析。通过这种方式，调试器变得可逆：您可以在代码中前后移动，并准确查看正在发生的事情，从而更容易捕获错误。查看此博客文章，以了解有关在CLion中使用撤消的更多信息。

保持创造力和生产力

另外，如果您希望思维导图以视觉方式构造您的想法，则IDEA思维导图插件将使您可以在CLion中以MMD格式创建和编辑思维导图。使用stackoverflow ，只需右键单击即可在StackOverflow中搜索一行代码或注释。

只是为了好玩，试试Nyan Progress Bar 在等待某些耗时的CLion操作完成时变亮状态栏，或使用Emoji for Git Commits插件在提交中享受表情符号。
