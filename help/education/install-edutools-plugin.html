安装EduTools插件

选择你的IDE

EduTools插件与IntelliJ IDEA ， PyCharm ， WebStorm ， Android Studio和CLion兼容 。

本安装指南介绍了在IntelliJ IDEA PyCharm WebStorm Android Studio CLion中安装EduTools插件所需了解的一切。如果使用其他受支持的IDE，请使用页面顶部的“ 部分”下拉菜单切换到该IDE：

如果您没有安装IntelliJ IDEA，请下载IntelliJ IDEA Edu 。

如果您尚未安装PyCharm，请下载PyCharm Edu 。

在开始之前，请确保已安装Android Studio 。

在开始之前，请确保已安装WebStorm 。

在开始之前，请确保已安装CLion 。

安装EduTools插件

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，选择插件 。

  2. 在“ 插件”对话框中，切换到“ 市场”选项卡。

  3. 在打开的对话框中，搜索EduTools 。

  4. 点击安装 。

  5. 在“ 设置”对话框中单击“ 确定 ”以应用更改，并在出现提示时重新启动IntelliJ IDEA PyCharm WebStorm Android Studio CLion 。

更新已安装的EduTools插件

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，选择插件 。

  2. 在“ 插件”对话框中，切换到“ 更新”选项卡。

  3. 检查EduTools插件是否需要更新。

  4. 点击更新 。

  5. 在“ 设置”对话框中单击“ 确定 ”以应用更改，并在出现提示时重新启动IntelliJ IDEA PyCharm Android Studio CLion 。

卸载EduTools插件

您始终可以通过清除其名称右侧的复选框来禁用EduTools插件。

  1. 在“ 设置/首选项”对话框中Ctrl+Alt+S ，选择插件 。

  2. 在“ 已安装”选项卡上，搜索EduTools 。

  3. 右键单击EduTools插件，然后选择卸载 。

  4. 在“ 设置”对话框中单击“ 确定 ”以应用更改，并在出现提示时重新启动IntelliJ IDEA PyCharm Android Studio CLion 。
