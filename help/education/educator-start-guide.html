教育产品2019.2帮助

教育家入门指南

借助Educational Products，您可以以编码任务的形式学习编程语言，并在基于JetBrains IntelliJ平台的IDE内部获得即时反馈。

教育产品将支持Java ， Kotlin ， Python ， Scala ， JavaScript ， Rust和C ++等更多功能。

选择您的编程语言

本教程将引导您创建带有一组编程任务和集成测试的简单Java Kotlin Python Scala JavaScript Rust C ++课程。您可以使用页面顶部的Section下拉菜单切换到其他编程语言：

先决条件

您可以在安装了EduTools插件的IntelliJ IDEA中创建Java课程。

下载IntelliJ IDEA Edu或如果已经安装了IntelliJ IDEA Community或Ultimate，则安装EduTools插件 。

先决条件

您可以在IntelliJ IDEA或安装了EduTools插件的Android Studio中创建Kotlin课程。

下载带有EduTools和Kotlin插件的IntelliJ IDEA Edu ，或者如果已经安装了IntelliJ IDEA Community或Ultimate，则安装EduTools插件 。
下载Android Studio并安装EduTools插件 。

先决条件

您可以在PyCharm中创建Python课程。

下载PyCharm Edu或如果已经安装了PyCharm Community或Professional，则安装EduTools插件 。

先决条件

您可以在安装了EduTools插件的IntelliJ IDEA中创建一个Scala课程。

下载IntelliJ IDEA Edu或如果已经安装了IntelliJ IDEA Community或Ultimate，则安装EduTools插件 。

先决条件

您可以在安装了EduTools插件的WebStorm ， IntelliJ IDEA Ultimate和PyCharm Professional中创建JavaScript课程。

下载WebStorm ， IntelliJ IDEA Ultimate或PyCharm Professional并附带评估许可证密钥，以进行30天免费试用，并安装EduTools插件 。

如果您是学生或学术人员，请申请我们的教育许可计划，以访问所有JetBrains桌面产品，包括WebStorm。

先决条件

您可以在CLion中创建安装了EduTools和Rust插件的Rust课程。

下载与评估许可证密钥捆绑在一起的CLion以进行30天的免费试用，然后安装EduTools插件 。或者，您可以下载IntelliJ IDEA Edu 。

要了解如何安装Rust插件，请参阅本指南。

您可以在安装了EduTools插件的CLion中创建C ++课程。
