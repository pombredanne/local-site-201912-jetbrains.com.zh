项目

描述

从中提取接口

此只读字段显示包含要从中提取接口的类的源包的名称。

提取界面

选择此选项后，GoLand会提取一个新接口，但不会立即使用它，并且源代码不会更改。

接口名称

输入新接口的名称。如果选择了“ 提取接口”选项，则此字段可用。

提取界面并在可能的情况下使用

选择此选项可提取接口并立即将其应用于源代码，并在“ 查找”工具窗口的专用选项卡中显示建议的更改。

重命名原始类并在可能的情况下使用接口

使用此选项可以重命名原始类，并使其成为新创建的接口的实现。

将实现类重命名为

键入原始类的新名称。如果选择“ 重命名原始类并在可能的情况下使用接口”选项，则此字段可用。

新介面套件

指定新接口的软件包。如有必要，请单击浏览。 然后在打开的对话框中选择目标软件包。

成员组成界面

指定类的方法，以及要包含在新接口中的最终静态字段（常量）。要在界面中包含元素，请选中它旁边的复选框。

JavaDoc / ASDoc

指定要应用于内联文档的操作。可用的选项有：

  • 原样 -选择此选项可将内联文档留在原处。

  • 复制 -选择此选项可将内联文档复制到提取的界面，而不将其从其当前位置删除。

  • 移动 -选择此选项可将内联文档移动到提取的界面，并将其从当前位置删除。
