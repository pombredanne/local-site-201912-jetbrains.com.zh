使用提取和内联重构

使用提取和内联重构，您可以快速清理代码并使代码更加简化或显式。

提取变量

  1. 选择并右键单击以下代码片段： &User{ ID: 1, Name: "Smith",} 。

  2. 导航到重构|提取物|可变的 。

  3. 键入u作为新变量的名称。

提取方法

  1. 选择并右键单击以下代码片段： &User{ ID: 1, Name: "Smith",} 。

  2. 导航到重构|提取物|方法 。

提取一个常数

  1. 在编辑器中，选择要用常量替换的变量的表达式或声明。

  2. 按Ctrl+Alt+C引入一个常数或选择Refactor |提取物|常数 。

    选择要提取为常量的表达式，然后按Enter 。选择要替换的出现次数和要使用的名称。

提取函数或方法

提取方法重构使您可以将可以组合在一起的代码片段，将其移动到单独的方法中，并用对方法的调用替换旧代码。如果代码总是在return语句中退出（所有执行路径都可能导致隐式return语句），则将其原样提取。但是，如果它包含没有返回值的执行路径，则提取的方法将有一个附加的bool标记为return值。该标志用于在呼叫侧执行早期返回。

  1. 在编辑器中，选择要提取的表达式或其部分。您也可以将插入号放置在表达式中，在这种情况下，GoLand会为您提供潜在代码选择的列表。

  2. 按Ctrl+Alt+M或从主菜单中选择“ 重构|提取物|提取方法 。

  3. 输入方法名称，然后按Enter 。
