管理超速套件

GoLand提供了一个工具来选择用于显示或隐藏，添加和删除套件的coverage套件。

选择一个覆盖套件

在您至少运行了一项覆盖测试之后，覆盖套件列表将变为可用。

  1. 从主菜单中，选择运行|显示覆盖率数据 Ctrl+Alt+F6 。

  2. 单击“ 显示所选内容” 。对话框关闭。

    在编辑器中，GoLand打开所选测试套件的测试覆盖率结果。

  3. 要隐藏覆盖率结果，请选中必要类别旁边的复选框，然后单击“ 无覆盖率” 。

上传或删除Coverage Suite

考虑从构建服务器获取包含代码覆盖率信息的文件时的情况。您可以从磁盘加载此文件，并使其可供查看。另外，您可以带些时间检查生产的覆盖套件。

  1. 在主菜单中，选择工具|显示覆盖率数据 Ctrl+Alt+F6 。

  2. 请点击然后在打开的对话框中选择所需的.es文件。

  3. 要从列表和存储中删除套件，请在列表中选择它，然后单击 。
