创建一个Go项目

创建一个Go项目

  1. 选择文件|新品|项目 。

  2. 在“ 新建项目”页面中，选择“ 转到” 。

  3. 在位置字段中，指定要存储项目的位置。

  4. 在“ GOROOT”字段中，指定Go安装的位置。通常，位置是自动定义的。

    要更改或安装新版本的Go SDK，请单击“ 添加SDK”按钮，然后选择“ 本地”以在硬盘驱动器上选择Go SDK版本，或选择“ 下载”以从官方存储库下载Go SDK。有关安装Go SDK的更多信息，请参阅安装Go SDK 。

  5. （可选）切换“ 整个索引索引”复选框：

    • 清除复选框 ：已索引项目文件和供应商的软件包。GOPATH中的软件包未编制索引。可能会提高整体性能。

    • 选中复选框 ：将对GOPATH中的软件包和项目文件进行索引。

  6. 点击创建 。

安装Go SDK

选择Go SDK的本地副本

确保与围棋SDK的文件夹提供的路径包括仓 и 的src的文件夹。

  1. 在“ 新建项目”对话框中，选择“ 转到” 。

  2. 单击添加SDK按钮，然后选择本地 。

  3. 在文件浏览器中，导航到硬盘驱动器上的SDK版本。

  4. 点击打开 。

下载Go SDK

  1. 在“ 新建项目”对话框中，选择“ 转到” 。

  2. 点击添加SDK按钮，选择下载 。

  3. 从版本列表中，选择SDK版本。

  4. 在位置字段中，指定SDK的路径。要使用文件浏览器，请单击“ 浏览”图标 。

  5. 单击确定 。
