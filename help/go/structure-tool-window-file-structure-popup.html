结构工具窗口，文件结构弹出窗口

结构工具窗口

此工具窗口显示当前在编辑器中打开并具有焦点的文件的结构，或在“项目”工具窗口中选择的文件的结构。

对于图，此工具窗口显示图预览。

查看|工具窗口|结构体
Alt+7

文件结构弹出

此弹出窗口显示文件的结构，当前在编辑器中打开该文件并具有焦点。

导航|档案结构
Ctrl+F12

两种视图均有助于快速浏览文件结构 。

本节介绍工具窗口标题栏上的按钮以及标题栏的上下文菜单选项。打开和关闭这些选项，可以隐藏或显示某些类型的元素，并配置它们的显示方式。

标题栏上的按钮在所有语言环境中都是通用的。

  • HTML，XML
  • JavaScript，TypeScript，CoffeeScript

有关“ 结构”工具窗口中文件树视图中的成员图标的信息，请参见“ 符号参考 ”部分。
