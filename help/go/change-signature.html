变更签名

更改签名重构结合了可以应用于函数，方法或方法规范的几种不同的修改。您可以使用此重构来：

  • 更改方法/功能名称

  • 添加，删除和重新排序参数

  • 将默认值分配给新的非可变参数

在更改方法/功能签名时，GoLand会搜索该方法/功能的所有用法，并更新所有可以安全修改以反映更改的方法/功能的调用，实现和替代。重构方法规范时，您会看到一个通知对话框，您可以在其中选择更新所有实现。

更改方法或函数签名

  1. 单击要重构的方法/函数名称。

  2. 按Ctrl+F6 。或者，选择Refactor |从主菜单或上下文菜单中更改签名 。

  3. 在“ 更改签名”对话框中，根据需要对方法/功能签名进行必要的更改：

    • 更改方法/功能名称。要更改名称，请在“ 名称”字段中编辑文本。

    • 管理方法/功能参数。要配置参数，请使用表格和“ 参数”区域中的按钮：

      • 要添加新参数，请单击并在相应的表格行中指定新参数的属性。

      • 要删除参数，请选择任意行，然后单击 。

      • 要重新排列参数，请使用向上键 （ ）和下 （ ）图标。

      • 要声明相同类型的多个变量，请选中“ 重用类型”复选框。相同类型的变量必须一个接一个地声明。所以，代替firstName string, lastName string, age int, id int ， 你会得到firstName, lastName string, age, id int 。

      • 要更改参数的名称或默认值，请在参数表中（分别在名称和默认值字段中）进行必要的更新。

  4. 单击重构 。

要查看预期的更改并在重构之前进行调整，请单击预览 。

添加返回值

  1. 单击功能名称，然后按Ctrl+F6 。或者，选择Refactor |从主菜单或上下文菜单中更改签名 。

  2. 在“ 更改签名”对话框中，单击“ 结果参数”选项卡，然后单击“ 添加”图标（ ）。

  3. 在类型文本字段中，指定返回值。

  4. 单击重构 。

生产力提示

添加缺少的参数

单击签名中的参数，然后按Alt+Enter并选择“更改签名重构”。请参阅GitHub存储库中的demo1.go文件。

更改参数类型

单击签名中的参数，然后按Alt+Enter并选择“更改签名重构”。请参阅GitHub存储库中的demo1.go文件。

更改功能参数

您可以自动推断参数并将其添加到函数调用中，或者在签名中添加缺少的返回参数。
