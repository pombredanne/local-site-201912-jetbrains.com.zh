版本控制工具窗口

查看|工具窗口|版本控制

Alt+9

如果为项目启用了版本控制集成，则此工具窗口可用。

该工具窗口包含多个视图/选项卡，这些视图/选项卡显示VCS相关的信息，并允许您管理变更列表，执行VCS特定的操作，查看其他团队成员所做的更改，等等：

  • 控制台选项卡：此选项卡显示执行VCS相关命令的结果。

  • 本地更改选项卡：此选项卡始终存在，并显示已在本地修改但尚未提交到存储库的文件列表。

  • “历史记录”选项卡：通过VCS | “ 显示历史记录”命令调用时，此选项卡将添加到“版本控制”工具窗口中。 。

  • 集成到“分支信息视图”选项卡：使用指定的更新后运行状态设置运行集成后，此视图可用。

  • 日志选项卡：仅当您使用Git或Mercurial作为版本控制系统时，此选项卡才可用。它显示了提交给本地和远程存储库的所有分支，或特定分支或存储库的所有更改。

  • “存储库”和“传入”选项卡：“ 存储库”选项卡显示在当前项目内VCS根目录下提交到存储库的更改。“ 接收”选项卡显示其他团队成员提交到存储库的更改，而不是在本地检出。

  • 搁架选项卡： 搁置更改或更改列表时，此选项卡将添加到工具窗口。

  • 更新信息选项卡：当本地信息与服务器同步时，此选项卡可用。

标题栏上下文菜单和按钮

您可以右键单击窗口标题栏，然后使用上下文菜单配置其查看模式 ，将窗口与其他工具窗口栏关联，或者调整大小并隐藏窗口。

您还可以使用工具栏按钮：

项目

捷径

描述

单击此按钮可以访问上下文菜单命令的子集，这些命令可让您配置窗口的查看模式 。

Shift+Escape

使用此图标或快捷方式隐藏工具窗口。

当与Alt键，单击此图标将隐藏附加到同一工具窗口栏的所有工具窗口 。
