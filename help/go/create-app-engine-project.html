创建一个App Engine项目

Google App Engine是一项服务和平台，您可以在其中开发和托管Web应用程序。您可以在Google App Engine 官方网站上了解有关Google App Engine的更多信息。

借助App Engine集成，您可以运行和调试Google App Engine应用程序。一个新项目已经包括带有默认配置设置的app.yaml ，以及带有Hello World应用程序的GO文件。在GO文件中自定义基本处理程序或编写您自己的基本处理程序。

通过App Engine集成创建项目

App Engine项目具有预定义的运行/调试配置。要添加其他配置，请在“ 运行/调试配置”对话框中选择“构建 ” 。有关运行/调试配置的更多信息，请参阅使用运行/调试配置 。

  1. 选择文件|新品|项目 。

  2. 在“ 新建项目”页面中，选择App Engine 。

  3. 在“ GOROOT”字段中，指定Go安装的位置。通常，位置是自动定义的。

    要更改或安装新版本的Go SDK，请单击“ 添加SDK”按钮，然后选择“ 本地”以在硬盘驱动器上选择Go SDK版本，或选择“ 下载”以从官方存储库下载Go SDK。有关安装Go SDK的更多信息，请参阅安装Go SDK 。

  4. （可选）切换“ 整个索引索引”复选框：

    • 清除复选框 ：已索引项目文件和供应商的软件包。GOPATH中的软件包未编制索引。可能会提高整体性能。

    • 选中复选框 ：将对GOPATH中的软件包和项目文件进行索引。

  5. 点击创建 。

使用Go 1.9运行时及更低版本创建App Engine

App Engine项目具有预定义的运行/调试配置。要添加其他配置，请在“ 运行/调试配置”对话框中选择Go App Engine 。有关运行/调试配置的更多信息，请参阅使用运行/调试配置 。

  1. 下载并解压缩App Engine的Google Cloud SDK（ 下载Cloud SDK ）。对于1.9运行时版本，请滚动至Previous App Engine SDK部分。

  2. 选择文件|新品|项目 。

  3. 在“ 新建项目”页面中，选择App Engine 。

  4. 点击添加SDK图标（ ）在GOROOT字段附近，然后选择Local 。

  5. 在文件浏览器中，导航到硬盘驱动器上适用于App Engine的Google Cloud SDK的goroot-1.9文件夹。

  6. 单击确定 。

从2019年10月1日开始，使用Go 1.9运行时的新部署将不可用。在2019年10月1日之前升级您的应用程序以使用Go 1.11或Go 1.12。有关Go 1.9运行时的详细信息，请参阅App Engine官方网站 。

使用App Engine应用程序

运行App Engine应用程序

创建App Engine项目时，您可以开始调试。App Engine项目具有预定义的运行/调试配置和默认地址（ http：// localhost：8080 ）。

  1. 单击运行图标Shift+F10

  2. 在“ App Engine”标签中，点击管理服务器地址。

  3. 在浏览器中，单击实例名称（例如， default ）。

调试App Engine应用程序

  1. 在应用程序中设置一个断点。

  2. 单击调试图标Shift+F9

  3. 在“ App Engine”标签中，单击服务器地址。

  4. 在浏览器中，单击实例名称（例如， default ）。

  5. 在“ 调试器”选项卡中，查看可以检查的框架列表。您可以单击框架以查看变量的当前状态。

安装Go SDK

选择Go SDK的本地副本

确保与围棋SDK的文件夹提供的路径包括仓 и 的src的文件夹。

  1. 在“ 新建项目”对话框中，选择“ App Engine” 。

  2. 单击添加SDK按钮，然后选择本地 。

  3. 在文件浏览器中，导航到硬盘驱动器上的SDK版本。

  4. 点击打开 。

下载Go SDK

  1. 在“ 新建项目”对话框中，选择“ App Engine” 。

  2. 点击添加SDK按钮，选择下载 。

  3. 从版本列表中，选择SDK版本。

  4. 在位置字段中，指定SDK的路径。要使用文件浏览器，请单击“ 浏览”图标 。

  5. 单击确定 。
