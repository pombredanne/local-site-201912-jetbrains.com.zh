Configuring content roots

Create a content root

  1. Open settings (Ctrl+Alt+S) and navigate to Project Structure.

  2. In the content roots pane of the Project Structure menu, click the Add Content Root button .

  3. In a file browser, navigate to a folder that you want to attach and click OK.

Remove a content root

  1. Open settings (Ctrl+Alt+S) and navigate to Project Structure.

  2. In the Content roots pane, select the content root that you want to delete.

  3. Click the Remove button and confirm removal.

Excluding files and folders by name patterns

You can make files and folders excluded by specifying the name patterns. Use the Exclude files field to specify the patterns.

The files and folders whose names match at least one of the specified patterns are made excluded. Folders are excluded along with all their contents.
