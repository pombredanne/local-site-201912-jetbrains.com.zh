使用重命名重构

通过重命名重构，您可以在整个代码中更改符号，文件的名称以及对它们的所有引用。

由于仅会影响有限的范围，因此可以轻松地内联重命名局部变量或私有方法。重命名类型，接口或公共方法可能会影响许多文件。在这种情况下，请在重构之前预览潜在的更改。

重命名位置字段

  1. 点击Position领域。

  2. 按Shift+F6然后开始打字JobTitle 。

  3. 按Enter 。

包主要导入“ fmt”类型用户结构{ID int`用户名字符串位置字符串} func main（）{u：=用户{ID：1，用户名：“ User”，位置：“ Job position”，} fmt。Printf（“用户详细信息％＃v \ n”，u）}

重命名数据库字段的重构

如果建立与数据库的连接，则可以直接从查询中重命名数据库字段。

要了解如何将IDE 连接到数据库，请参阅连接数据库 。

重命名工作表

  1. 点击worker表。

  2. 按Shift+F6 。

  3. 在“重命名”对话框中，键入actor 。

  4. 单击重构 。

重命名JSON的重构

GoLand检测字符串中的JSON字段，并在整个代码中重命名该字段的所有出现。

重命名名称字段

  1. 点击name表。

  2. 按Shift+F6 。要调用“ 重命名”对话框，请按Shift+F6两次。

  3. 类型firstName 。

  4. 在“ 重构预览”工具窗口中，预览更改，然后单击“ 执行重构” 。
