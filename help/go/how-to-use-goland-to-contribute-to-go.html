如何使用GoLand为Go贡献力量

GoLand是Go IDE，可用于为您的应用程序编写Go代码。但是，如果您想自己开发Go语言并为编程语言做出贡献，该怎么办？本教程描述了如何配置IDE只需几个步骤。

步骤1。克隆Go资源

  1. 在“欢迎”屏幕上，单击“ 从版本控制中 签 出”，然后选择“ Git” 。

  2. 在“ URL”文本字段中，复制并粘贴以下URL： https://go.googlesource.com/go 。

  3. 在目录文本字段中，定义要存储Go项目的目录。

  4. 单击克隆 。

  5. 在“ 从版本控制签出”对话框中，单击“ 是” 。

  6. 打开设置Ctrl+Alt+S并导航到Go |去模块 。确保已清除“ 启用Go模块（vgo）集成”复选框。

Go来源包括用于测试的go.mod文件。要禁用这些测试，请确保禁用Go模块（vgo）集成。打开设置Ctrl+Alt+S ， 转到Go |转到模块 ，并确认已清除“ 启用转到模块（vgo）集成”复选框。

第2步。编译Go

  1. 在“ 终端工具”窗口（“ 视图” |“工具窗口” |“终端” ）中，通过运行以下命令导航到src目录： cd src 。

  2. 在src目录中，运行以下命令以启动Go编译： ./make.bash （对于Linux和macOS）， make.bat （对于Windows）。

故障排除

错误

可能的解决方案

ERROR: Cannot find . Set $GOROOT_BOOTSTRAP to a working Go tree >= Go .

打开设置Ctrl+Alt+S并导航到工具|码头 。在环境变量字段中，单击浏览按钮并添加以下变量： GOROOT_BOOTSTRAP = 。

第三步将克隆的Go项目设置为Go SDK

如果将克隆的Go项目的根目录设置为该项目的Go SDK，则GoLand将使用此Go SDK进行代码完成，代码导航，标识符定义和其他IDE功能。

  1. 在设置中Ctrl+Alt+S ， 转到Go | GOROOT 。

  2. 点击添加SDK按钮 ，然后选择Local 。

  3. 在文件浏览器中，导航到克隆项目的根目录，然后单击“ 打开” 。

  4. 点击应用 。

第四步。为您的更改创建一个分支

  1. 要在GoLand中创建分支，请导航至VCS | Git |分支，然后选择新建分支 。

  2. 在“ 创建新分支”对话框中，键入新分支的名称，然后单击“ 确定” 。

有关使用VCS的更多信息，请参见版本控制 。

第五步运行测试

完成所有更改并运行为它们创建的测试之后。您还可以运行Go项目中包含的所有测试。

  1. 在“ 终端工具”窗口（“ 视图” |“工具窗口” |“终端” ）中，通过运行以下命令导航到src目录： cd src 。

  2. 在src目录中，运行以下命令以开始Go编译： ./all.bash （对于Linux和macOS）， all.bat （对于Windows）。

第六步提交您的更改

完成更改后，可以将它们提交到自己的fork中，然后向Go团队发送拉取请求，以查看并合并回去。

  1. 导航到VCS |承诺 Ctrl+Alt+K 。

  2. 选择您要提交的所有文件。

  3. 在“ 提交消息”文本字段中，简短介绍您的更改。

  4. 按提交 。

步骤7添加遥控器并推动您的前叉

向存储库中添加一个指向Go分支的新遥控器（例如， git@github.com:/go.git ）。然后推送您的更改并打开Go团队的请求请求。有关创建拉取请求的更多信息，请参阅GitHub上的从派生创建拉取请求 。

  1. 导航到VCS | Git |推 。

  2. 在“ 推送提交”对话框中，单击提交标题中的“ 原始”链接。

  3. 单击定义远程 。

  4. 在名称字段中，输入叉子的名称。

  5. 在“ URL”字段中，输入叉子的URL，然后单击“ 确定” 。
