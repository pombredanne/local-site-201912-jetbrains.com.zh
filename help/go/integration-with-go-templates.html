与Go模板集成

Go有自己的模板引擎，该引擎分为两个包： text / template和html / template 。这些软件包的功能相似，不同之处在于html / template允许用户生成可防止代码注入的HTML代码，使其适合在网页和电子邮件中使用。同样，这些软件包在其他应用程序中提供代码帮助，例如Helm的配置和Kubernetes软件包管理器 。

要在Go模板中使用代码帮助，请确保您的项目文件夹位于GOPATH下（例如go / src / myProject ）。要检查您的GOPATH，请打开设置Ctrl+Alt+S并导航到Go | GOPATH 。
