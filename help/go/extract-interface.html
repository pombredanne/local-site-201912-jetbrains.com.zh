提取接口

使用提取接口重构，可以创建基于所选类型的接口。“ 提取接口”对话框显示方法和规格的列表，可用于创建新的方法规格。

  1. 将插入号放置在要用作新接口基础的类型，接口或方法上。

  2. 选择重构|提取物|接口 。

  3. 在“ 提取接口”对话框中，选择要与接口一起提取的成员。

  4. 在“ 到目录” ，“ 到文件 ”和“ 到包”字段中指定提取位置。

  5. 单击重构 。

要启用“ 预览”按钮，请清除要更改可见性的任何方法的至少一个复选框（“ 导出”列）。
