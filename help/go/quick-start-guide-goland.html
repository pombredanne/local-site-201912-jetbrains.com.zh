快速入门指南

本快速入门指南将向您介绍关键概念，并帮助您开始使用GoLand。

步骤1。在GoLand中打开或创建项目

首次安装GoLand并启动它后，您需要创建一个项目。您在GoLand中所做的一切都是在项目的上下文中完成的。它是编码辅助，批量重构，编码样式一致性和其他功能的基础。

您可以使用三个选项来开始在IDE内的项目上工作：

  • 打开一个现有项目
  • 从版本控制中检出现有项目
  • 创建一个新项目

打开一个现有项目

要打开您存储在计算机上的项目 ，请点击打开项目 。在文件浏览器中，导航到包含项目文件的文件夹，然后单击“ 打开” 。

打开项目后，需要指定Go SDK的位置。您可以指定SDK的本地路径或下载它。要设置Go SDK，请打开设置Ctrl+Alt+S并导航到Go | GOROOT 。单击添加SDK按钮，然后在两个选项之间进行选择：

  • 本地 ：使用本地SDK副本。在文件浏览器中，导航到硬盘驱动器上的SDK版本。

  • 下载 ：下载SDK。在位置字段中，指定SDK的路径。要使用文件浏览器，请单击“ 浏览”图标 。单击确定 。

从版本控制系统克隆现有项目

您也可以从VCS存储或存储库下载源。单击从版本控制中签出，然后在Git（GitHub）或Mercurial之间进行选择：

  • 要从GitHub提取文件，请单击登录GitHub输入您的凭据。输入源的路径，然后将存储库克隆到您的计算机。

  • 要从Mercurial提取文件，请安装并配置Mercurial，然后重新启动GoLand。输入源的路径，然后将存储库克隆到您的计算机。

打开项目后，需要指定Go SDK的位置。您可以指定SDK的本地路径或下载它。要设置Go SDK，请打开设置Ctrl+Alt+S并导航到Go | GOROOT 。单击添加SDK按钮，然后在两个选项之间进行选择：

  • 本地 ：使用本地SDK副本。在文件浏览器中，导航到硬盘驱动器上的SDK版本。

  • 下载 ：下载SDK。在位置字段中，指定SDK的路径。要使用文件浏览器，请单击“ 浏览”图标 。单击确定 。

有关版本控制系统的更多信息，请参见版本控制 。

要使用其他版本控制系统，请安装相应的插件。有关安装插件的更多信息，请阅读《 管理插件》 。

创建一个新项目

  1. 要创建项目，请执行以下操作之一：

    • 选择文件|新品|项目 。

    • 在欢迎屏幕上，单击新建项目 。

  2. 在“ 新建项目”对话框中，选择“ 转到” 。

  3. 在GOROOT字段中，指定Go SDK。您可以选择本地SDK副本或下载它。

    • 要选择 Go SDK 的本地副本 ，请点击Add SDK图标（ ）在GOROOT字段附近，然后选择Local 。在文件浏览器中，导航到硬盘驱动器上的SDK版本。

    • 要下载 Go SDK，请点击添加SDK图标（ ）在GOROOT字段附近，然后选择下载 。从版本列表中，选择SDK版本。在“ 目标目录”字段中，指定SDK的路径。要使用文件浏览器，请单击“ 浏览”图标 。单击确定 。

  4. 点击创建 。

创建一个Go文件

一个新项目没有Go文件。要创建Go文件，请执行以下操作之一：

  • 右键单击项目的父文件夹，然后选择“ 新建” |“新建”。转到文件 。

  • 单击项目的父文件夹，然后按Alt+Insert ，然后选择转到文件 。

  • 单击项目的父文件夹，然后导航到“ 文件” |“文件”。新品|转到文件 。

在“ 新建Go文件”对话框中，键入文件的名称，然后选择是要创建空的Go文件（ Empty file ）还是要使用已定义的Go文件main功能（简单应用程序）。

第2步。探索用户界面

首次启动GoLand时，您会看到“ 欢迎”屏幕对话框。在“ 欢迎屏幕”对话框中，您可以创建和打开项目，从版本控制系统中签出项目，查看文档并配置IDE。

打开项目后，您会看到主窗口分为几个逻辑区域。

  1. 左侧的“ 项目工具”窗口显示您的项目文件。工具窗口是用于访问项目管理，搜索，运行和调试，与版本控制系统集成以及其他任务的窗口。要查看所有工具窗口的列表，请导航至“ 视图” |“视图”。工具窗口 。

  2. 编辑器窗格在右侧，您可以在其中编写代码。它具有用于在打开的文件之间轻松导航的选项卡。

  3. 导航栏位于编辑器上方，还使您可以快速运行和调试应用程序以及执行基本的VCS操作 。

  4. 左装订线是编辑器旁边的垂直条纹。它显示了断点，并提供了导航到定义或声明的便捷方法。另外，您可以使用“ 运行应用程序”图标（ ）运行或调试您的应用程序。

  5. 编辑器右侧的右侧装订线 。GoLand监视代码质量并显示代码检查的结果：错误，警告，错别字和其他问题。装订线顶部的指示器显示了整个文件的代码检查的总体状态。

  6. 状态栏指示项目和整个IDE的状态，并向您显示有关文件编码，行分隔符，检查配置文件以及其他警告和信息消息的信息。

在IDE的左下方，在状态栏中，您可以看到要么纽扣。这些按钮可切换工具窗口弹出窗口的显示。如果将鼠标指针悬停在此按钮上，则会显示当前可用工具窗口的列表。

第三步写你的代码

当您在编辑器中工作时，GoLand会分析您的代码，搜索优化代码的方法，并检测潜在的和实际的问题。以下列表包括一些基本功能和工具，这些功能和工具可能对您的代码编写和提高生产率很有用：

  • 代码完成
  • 产生程式码
  • 实时模板
  • 视察
  • 意向动作

代码完成

GoLand具有两种类型的代码完成功能：

基本代码完成Ctrl+Space帮助您完成可见性范围内的类型，接口，方法和关键字的名称。当您调用代码完成功能时，GoLand会分析上下文并建议从当前插入符号位置可以到达的选择。默认情况下，GoLand在您键入时自动显示代码完成弹出窗口。

智能代码完成Ctrl+Shift+Space过滤建议列表，并仅显示适用于当前上下文的类型。

产生程式码

GoLand提供了多种生成通用代码结构和重复元素的方法，可帮助您提高生产率。这些可以是预定义模板，也可以是自定义模板，它们会根据上下文，各种包装程序和字符自动配对而不同地应用。

在主菜单中，选择代码|生成 Alt+Insert打开带有您可以生成的可用结构的弹出菜单。

您可以生成getter，setter，构造函数，缺少的方法和测试文件。有关Generate动作的更多信息，请参阅使用Generate动作 。

实时模板

通过使用实时模板 ，您可以将常用结构插入代码中。例如，循环，条件，各种声明或打印语句。

要展开代码段，请键入相应的模板缩写，然后按Tab 。继续按Tab从模板中的一个变量跳转到下一个变量 。按Shift+Tab移至上一个变量。

要查看实时模板列表，请打开设置Ctrl+Alt+S并导航到编辑器|实时模板 。

视察

在GoLand中，有一组代码检查可以在编译项目之前检测并更正项目中的异常代码。IDE可以发现并突出显示各种问题，找到死代码，查找可能的错误，拼写问题，并改善整体代码结构。

检查可以扫描所有项目文件中或仅在特定范围内的代码（例如，仅在生产代码中或在修改的文件中）。

每次检查都有一个严重级别 ，即问题可能影响您的代码的程度。严重性在编辑器中以不同的方式突出显示，因此您可以快速区分关键问题和次要问题。GoLand带有一组预定义的严重性级别，使您可以创建自己的严重性级别。

要查看检查清单，请打开设置Ctrl+Alt+S并导航到编辑器|检查 。禁用其中一些，或启用其他功能，并调整每次检查的严重性。您可以决定将其视为错误还是警告。

例如，“ 无法访问的代码检查”检测到无法执行的部分代码。

意向动作

当您在编辑器中工作时，GoLand会分析您的代码，搜索优化代码的方法，并检测潜在的和实际的问题。意图动作涵盖了从错误和警告到优化建议的各种情况。

在GoLand中，您可以使用以下类型的意图动作：

  • 意向动作 ：生成一段代码或为您的代码建议其他变体的操作。通常，将意图操作应用于有效的代码段。通过单击黄色灯泡图标，您可以查看当前上下文中可用的意图动作 。例如，您可以创建一个构造函数，该构造函数生成结构类型的值。或者，将if-else语句替换为等效语句，但条件为否，并替换掉if-else分支。

  • 快速修复 ：建议在代码中解决问题的操作。通常，有意操作将应用于有错误的代码。例如，当您错过表达式末尾的方括号时。或者，当您忘记将值转换为特定类型时。

要查看意图操作列表，请打开设置Ctrl+Alt+S并导航到编辑器|用心 。

  2. 从列表中选择一个动作，然后按Enter 。

第四步。运行和调试代码

您可以通过键盘，上下文菜单，装订线菜单或使用“ 运行/调试”配置在GoLand中运行和调试代码。

运行你的代码

运行您的应用程序Shift+F10 ，单击“ 运行应用程序”图标然后选择运行 。

调试代码

调试从放置将暂停程序执行的断点开始，以便您可以浏览程序数据。只需单击左侧排水沟中要放置断点的行即可。

调试应用程序Shift+F9 ，单击“ 运行应用程序”图标然后选择调试 。然后，逐步执行程序（请参阅“ 运行”菜单或“ 调试工具”窗口中的可用选项）。

有关详细信息，请参见调试部分。
