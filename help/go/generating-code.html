Generating code

GoLand provides multiple ways to generate common code constructs and recurring elements, which helps you increase productivity. These can be either predefined or custom templates which are applied differently based on the context, various wrappers, and automatic pairing of characters.

From the main menu, select Code | Generate Alt+Insert to open the popup menu with available constructs that you can generate.

This chapter includes an overview of the following features that you can use to generate code:

  • Using the Generate action

  • Using Live templates

  • Using the Surround With action

  • Completing paired characters
