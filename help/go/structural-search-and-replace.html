结构搜索和替换

对于结构搜索和替换，GoLand仅支持XML，HTML和JavaScript。

常规的搜索过程没有考虑源代码的语法和语义。即使您使用正则表达式，GoLand仍会将您的代码视为正则文本。结构搜索和替换 （ SSR ）操作使您可以考虑代码结构来搜索代码中的特定代码模式或语法结构。

GoLand根据您创建的搜索模板和您适用的条件查找并替换源代码的片段。

使用现有模板运行结构化搜索

“ 结构搜索”对话框显示所选模板以及该模板中使用的过滤器的值。有关可能的过滤器值的更多信息，请参见搜索模板，过滤器和脚本约束 。

  1. 点击编辑|查找|结构搜索 。

  2. 在“ 结构搜索”对话框中，单击设置按钮 ，然后选择现有模板 。

  3. 在现有模板列表中，双击一个模板。

  4. 点击查看变量过滤器按钮 。

  5. 单击模板中的每个模板变量，然后设置搜索过滤器。要设置过滤器，请单击添加过滤器链接。

    变量是用美元符号括起来的一组字符（例如， $tag$ ）。如果表达式适合用户定义的约束（过滤器），这些约束指定模板变量可能具有的值，则模板变量与源代码表达式匹配。在搜索过程中绑定到模板变量的任何表达式都可以在搜索或替换模板中再次使用。

    GoLand立即在编辑器中突出显示找到的代码实例。

  6. （可选）指定以下选项以进一步缩小搜索范围：

    • 文件类型 -使用列表选择哪些文件类型应属于搜索的一部分（例如XML）。

    • 范围 -指定搜索位置：在项目，模块，目录或自定义范围内。

    • 搜索目标 -您可以选择完全匹配以匹配确切的模板或其他可用的代码元素。

  7. 点击查找 。

您可以在文本过滤器中使用正则表达式。例如，您可以使用[A-Z].*带有选中的“ 区分大小写”复选框的正则表达式，以查找搜索目标的所有大写字母。

保存用于结构搜索的模板

  1. 点击编辑|查找|结构搜索 。

  2. 在“ 结构搜索”对话框中，使用变量和过滤器添加搜索模式。

    变量是用美元符号括起来的一组字符（例如， $tag$ ）。如果表达式适合用户定义的约束（过滤器），这些约束指定模板变量可能具有的值，则模板变量与源代码表达式匹配。在搜索过程中绑定到模板变量的任何表达式都可以在搜索或替换模板中再次使用。

  3. 点击设置按钮 ，然后选择保存模板 。

从结构上替换目标

  1. 从主菜单中，选择编辑|查找|结构上更换 。

  2. 在“ 替换结构”对话框中，将新模板或现有模板添加到搜索和替换模板区域。您可以使用与搜索模板相同的方式保存替换模板。

  3. 如果需要在替换模板中为变量添加过滤器，请在所需的变量处插入插入符号，然后使用过滤器区域来管理过滤器。

    变量是用美元符号括起来的一组字符（例如， $tag$ ）。如果表达式适合用户定义的约束（过滤器），这些约束指定模板变量可能具有的值，则模板变量与源代码表达式匹配。在搜索过程中绑定到模板变量的任何表达式都可以在搜索或替换模板中再次使用。

  4. 在过滤器区域中，根据您选择的过滤器 ，指定条件。

  5. 要缩小替换结果的范围，请选择以下选项：

    • 缩短全限定名 -用短名称和导入替换全限定类名。

    • 重新格式化 -自动格式化替换的代码。

    • 使用静态导入 -尽可能使用静态导入进行替换。

    指定必要的选项后，点击查找 。GoLand在“ 查找工具”窗口中显示结果。

  6. 在“ 查找工具”窗口中，您可以进一步处理结果，一次一地替换找到的项，一次全部替换，或者预览可能的更改。

考虑下面的示例，该示例转换class归因于p标记为小写。在搜索模板字段中，我们创建了一个模板，用于搜索所有p具有大写值的标签（ [A-Z].*并匹配大小写 ） class属性。我们仅将搜索范围缩小到class值（ 搜索目标= b ）。在替换模板字段中，我们创建一个新变量$d$并为其分配一个Groovy脚本（ b.getText().toLowerCase() ）。搜索后，我们选择全部替换以执行替换。

共享搜索模板

您可以通过导出或导入与他人共享搜索模板。

  1. 在“ 结构搜索”对话框（“ 编辑” |“查找” |“结构搜索” ）中， 创建一个新的搜索模板或使用现有的模板 。

  2. 要导出模板，请单击 ，然后选择将模板导出到剪贴板 。GoLand将模板的XML表示添加到剪贴板（按Ctrl+Shift+V查看剪贴板的内容）。您可以在聊天，电子邮件或论坛中与其他开发人员共享此表示。

    要导入模板，请复制（ Ctrl+C ）从任何地方（电子邮件，聊天或论坛）共享的XML代码，然后在“ 结构搜索”对话框中，单击然后选择从剪贴板导入模板 。GoLand将XML代码表示形式转换为包含变量和作用域（如果存在）的模板。
