运行应用程序

GoLand支持运行整个应用程序以及

GoLand利用“ 运行/调试配置”中定义的设置。项目中存在的所有运行配置在“ 选择运行/调试配置”列表中可用。

如果在“运行/调试”配置中启用了运行前启动工具的选项，则GoLand将运行工具，并在成功后运行该应用程序。否则，程序将立即启动。

如果要查看所有当前正在运行的应用程序的列表，请选择“运行” |“运行”。从主菜单显示运行列表 。

请注意，开始运行会话后， 标记“ 运行工具”窗口的图标，并在“运行/调试配置选择器”中切换到指示运行过程处于活动状态。

使用左侧装订线中的图标

在编辑器的左装订线中，单击“ 运行应用程序”图标然后选择“ 在...中运行” 。

使用上下文菜单

在编辑器中右键单击打开的文件或方法，然后选择“运行于” Ctrl+Shift+F10 。

使用运行/调试配置

  1. 打开“ 运行/调试配置”对话框：

    • 选择运行|从主菜单编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按Shift+Alt+F10 ，然后按0或从弹出窗口中选择配置，然后按F4 。

  2. 在“ 运行/调试配置”对话框中，单击“ 添加新配置”图标（ ）或按Alt+Insert 。该列表显示了默认的运行/调试配置。选择所需的配置类型（例如Go build ）。

    右侧窗格中显示的字段显示所选配置类型的默认设置。

  3. 在名称字段中，输入配置名称。

  4. 在“ 文件”字段中，添加要运行的文件的名称（例如， main.go ）。

  5. 根据配置类型指定其他参数。例如，如果您希望代码以sudo特权运行 ，请选中“以sudo运行”复选框。

  6. 应用更改并关闭对话框。

  7. 单击运行纽扣 。

重新运行一个应用程序

如果“ 运行”窗口中的应用程序选项卡仍处于打开状态，则可以重新运行该应用程序。该程序将使用初始设置重新运行。

如果要重新运行而不失去工作位置，请按Shift+F10 。

  1. 在“ 运行”窗口中，选择打开所需应用程序的选项卡。

  2. 在工具栏中，单击“ 重新运行”按钮 ，或按Ctrl+F5 。

  3. 在通知对话框中，单击停止并重新运行 。

停止申请

在工具栏中，单击“ 停止”按钮在工具栏上，或按Ctrl+F2 。同样，“ 停止”按钮在“ 运行”工具窗口中可用。

查看正在运行的应用程序

您可以通过选择运行|列出所有正在运行的应用程序。显示运行清单 。如果没有活动的应用程序，该命令将显示为灰色。
