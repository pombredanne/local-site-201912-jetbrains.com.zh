方案

在这方面选择代码样式方案并根据需要进行更改。每当GoLand生成，重构或重新格式化代码时，都会自动应用代码样式方案设置。

代码样式是在项目级别和IDE级别 （全局）定义的。

  • 在项目级别，设置在“ 项目”方案下分组，该方案是预定义的并以粗体标记。项目样式方案仅适用于当前项目。

    您可以使用“ 复制到IDE”命令将Project方案复制到IDE级别。

  • 在IDE级别，将设置分为预定义的“ 默认”方案（以粗体标记）和用户通过“ 复制”命令创建的任何其他方案（标记为纯文本）。当用户不想在项目中保留代码样式设置并共享它们时，将使用全局设置。

    您可以使用“ 复制到项目”命令将IDE方案复制到当前项目。

定制方案

制表符和缩进

使用制表符

  • 如果选中该复选框，则制表符将用于缩进和代码重新格式化。

  • 如果清除该复选框，GoLand将使用空格而不是制表符。

智能标签

压痕由两部分组成。一部分来自嵌套代码块，另一部分由对齐确定。

  • 如果选中此复选框，则嵌套产生的零件将同时包含制表符和空格（如果需要），而由对齐方式定义的零件仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组适合指定制表符大小的空格后，将自动用制表符替换，这可能会导致中断精细对齐。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中可以容纳的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。
