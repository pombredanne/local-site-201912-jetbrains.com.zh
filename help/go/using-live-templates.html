实时模板

通过使用实时模板 ，您可以将常用结构插入代码中。例如，循环，条件，各种声明或打印语句。

要展开代码段，请键入相应的模板缩写，然后按Tab 。继续按Tab从模板中的一个变量跳转到下一个变量 。按Shift+Tab移至上一个变量。

插入实时模板

  1. 将插入符号放置在您希望模板扩展的位置。

  2. 键入模板缩写，然后按调用键（通常是Tab默认）。或者，在“ 代码”菜单上，单击“ 插入实时模板” Ctrl+J打开建议列表并选择必要的模板。

  3. 如果所选模板需要用户输入，则突出显示相应的字段。输入必要的值，然后按Enter要么Tab完成输入并移至下一个输入字段。完成所有输入字段后，插入符号将移至构造的末尾（或移至$END$标记（如果标记是在模板代码中定义的），则编辑器将返回常规操作模式。

用实时模板包围代码块

  1. 选择要包含的代码段。

  2. 在代码菜单上，单击环绕带有实时模板 Ctrl+Alt+J打开建议列表并选择必要的模板。

实时模板的类型

区分以下类型的实时模板：

  • 简单模板仅包含固定的纯文本。扩展简单模板时，文本将自动插入源代码中，以代替缩写。

    缩写

    扩展到

    imp
    导入（）
    int
    func init（）{}
    main
    func main（）{}

  • 参数化的模板包含允许用户输入的变量 。扩展参数化模板时，变量将由输入字段替换，以供用户手动指定，或者由GoLand自动计算。

    缩写

    扩展到

    forr
    对于键，值：=范围集合{}
    meth
    func（接收方）名称（参数）{}

  • 环绕模板用用户指定的文本包装所选代码块。

后缀代码完成与实时模板相似。它转换当前表达式而不选择它。例如，您可以输入.if在表达式之后调用相应的后缀完成并用if声明。

配置实时模板

要配置实时模板，请打开 编辑器| GoLand设置的“ 实时模板”页面Ctrl+Alt+S 。在“ 实时模板”页面上，您可以查看所有可用的实时模板，对其进行编辑并创建新模板 。

模板根据可以使用它们的上下文进行分组（通常是通过相应的语言）。要将模板移至另一个组，请右键单击该模板，选择Move ，然后选择所需的组名。

如果为活动模板设置了方言，则该活动模板仅在该方言中有效。

每个活动模板均由包含字母数字字符，点和连字符的缩写定义。缩写在一个组中必须唯一，但是相同的缩写可以在不同的组中使用，并根据相应组的上下文扩展到不同的构造。修改后的默认模板的缩写以蓝色字体显示在列表中。

将修改后的模板还原为默认设置

  • 在“ 设置/首选项”对话框的“ 实时模板”页面上，右键单击要还原的模板，然后单击“还原默认值” 。
