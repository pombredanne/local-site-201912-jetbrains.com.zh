通用调试器设置

项目

描述

按字母顺序对值排序

选择此选项可以在“ 调试工具”窗口的“ 变量”窗格中对值进行排序。

在变量视图中启用自动表达式

如果希望GoLand调试器自动评估表达式并在“ 调试工具”窗口的“ 变量”窗格中显示相应的值，请选择此选项。

调试器分析断点附近的上下文（当前语句，前一个语句和后一个语句）。这样做是为了在源代码中找到各种表达式（如果有），例如myvar.myfield 。

如果此类表达式不包含显式方法调用，则调试器将对其进行求值并在“变量”视图中显示相应的值。

编辑

项目

描述

内联显示值

选择此选项以启用“ 内联值视图”功能，该功能允许在编辑器中使用变量的旁边查看变量的值。

显示价值工具提示

选择此选项可在鼠标悬停时在编辑器工具提示中显示变量和表达式的值。

在此上下文中，一个工具提示是一个弹出窗口，它在Debug Tool Window的Variables窗格中提供了替代的，有时是更方便的值表示形式。

如果禁用此选项，则可以按Alt显示一个值。

值工具提示延迟（毫秒）

在“ 调试工具”窗口的“ 变量”窗格中 ，将鼠标指针悬停在对象上的时间与显示带有对象值的工具提示之间的时间（以毫秒为单位）进行指定。

显示有关代码选择的工具提示

选择此选项可启用在编辑器中选择代码片段时显示表达式值的工具提示。
