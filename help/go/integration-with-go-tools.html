使用gofmt格式化Go代码

使用gofmt，您可以在打开的文件或整个Go项目中格式化Go源代码。

要在打开的文件中格式化代码，请转至工具|转到工具|进入fmt文件 。

要格式化当前项目中的代码，请转至工具|转到工具|去fmt项目 。

要在将更改提交到VCS之前格式化代码，请选中Go fmt复选框。有关提交更改的更多信息，请参见

有关更多信息gofmt ，请参阅命令gofmt 。

使用goimports自动添加导入行

  1. 如果您的项目没有goimports ，请在Goimports文件通知窗口中单击go get goimports链接。否则，请打开“ 终端”工具窗口（“ 视图” |“工具窗口” |“终端” ），然后键入以下命令来安装goimports： go get golang.org/x/tools/cmd/goimports 。按Enter 。

  2. 要添加缺少的导入，请导航至“ 工具” |“其他”。转到“工具” ，然后单击“ Goimports文件” 。

goimports工具还会以与以下样式相同的格式设置代码格式gofmt 。有关更多信息goimports ，请参阅命令goimports 。

使用go generate创建或更新Go源文件

  1. 将以下行添加到您的Go代码中： //go:generate command arguments （例如， //go:generate stringer -type=Pill ）

  2. 点击工具|转到工具|去生成文件 。

有关更多信息go generate ，请参阅通过处理源代码和生成代码来 生成Go文件 。

使用Go兽医检查Go代码

查找可能的错误或可疑结构go vet ，单击工具|转到工具|审核文件 。您可以在“终端工具”窗口中查看代码分析的结果。

有关更多信息go vet ，请参阅Command vet 。
