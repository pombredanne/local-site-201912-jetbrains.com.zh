Version control

Use the VCS Operations Popup (Alt+` or VCS | VCS Operations Popup) to quickly invoke any VCS-related commands. The list of actions in the popup depends on the currently enabled VCS.

The list of VCS Operations popup commands is configurable — you can add or remove them on the Appearance and Behavior | Menus and Toolbars page of GoLand settings Ctrl+Alt+S.
