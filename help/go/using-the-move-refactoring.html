使用移动重构

通过移动重构，您可以从代码中获取任何顶级命名标识符，并将其移动到单独的文件或其他包中。

出于示例目的，从Github克隆Delve项目： git clone https://github.com/derekparker/delve 。有关Git的更多信息，请阅读官方Git文档 。

将函数声明移至新文件

  1. 在GoLand中打开Delve项目。

  2. 导航至挖掘| pkg |配置| split.go 。

  3. 右键点击SplitQuotedFields函数声明并选择Refactor |移动 。或者，按F6 。

  4. 在“ 到文件”字段中，键入newsplit 。

  5. 单击重构 。newsplit.go文件是使用我们移动的功能创建的。移动重构不仅会移动声明，还会添加所需的导入，更新对标识符的引用，并清除标识符的先前定义位置，以便您的代码仍然可以工作。

重构前预览更改

  1. 在GoLand中打开Delve项目。

  2. 导航至挖掘| pkg | goversion | go_version.go 。

  3. 右键点击GoVersion键入并选择Refactor |移动 。或者，按F6 。

  4. 指定文件路径和文件名。例如，添加/gover到“ 到目录”字段中的文件路径，然后键入gover在到文件字段中。

  5. 选择AfterOrEqual和IsDevel的标识符GoVersion类型。当标识符为私有且在移动操作期间未导出时，GoLand将显示一条通知。

  6. 点击预览 。在“ 查找工具”窗口中，您可以查看重构更改并排除N/A或删除Ctrl+X重构过程中的代码部分。
