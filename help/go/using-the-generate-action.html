使用生成动作

生成一个构造函数

生成构造函数动作生成一个函数，该函数创建结构类型的值。有关构造函数的更多信息，请参见Effective Go手册中的构造函数和复合文字 。

  1. 将插入符号放在您的代码上，然后按Alt+Insert 。或者，您可以使用意图动作Alt+Enter 。

  2. 在“ 生成”窗口中，选择“ 生成构造函数” 。

  3. 在“ 选择字段”对话框中，选择要在构造函数中使用的字段。

  4. 单击确定 。

生成getter和setter

您可以分别生成getter和setter或使用一个动作来生成这两种方法。有关构造函数的更多信息，请参见Effective Go手册中的Getters 。

  1. 将插入符号放在您的代码上，然后按Alt+Insert 。或者，您可以使用意图动作Alt+Enter 。

  2. 在“ 生成”窗口中，选择“ 获取器和设置器” 。

实施缺失的方法

要在Go中实现接口，您需要实现该接口中列出的所有方法。在GoLand中，您可以自动为这些方法生成模板。

  1. 将插入符号放在您的代码上，然后按Alt+Insert 。

  2. 在“ 生成”窗口中，选择“ 实施方法” （或按Ctrl+I ）。

  3. 在“ 选择类型”窗口中，选择要使用的类型。或者，单击“ 创建类型”以创建新类型。

  4. 在“ 选择要实现的接口”字段中，键入要实现的接口的名称。

  5. 更换panic("implement me")该方法的代码行。

生成一个空的测试文件

  1. 将插入符号放在您的代码上，然后按Alt+Insert 。

  2. 在“ 生成”窗口中，选择“ 清空测试文件” 。

创建带有_test.go后缀的文件。

生成文件，包和功能的测试

  1. 在“ 终端工具”窗口中，键入go get -u github.com/cweill/gotests 。或者，在getests通知窗口中单击go get gotests链接。

  2. 将插入符号放在您的代码上，然后按Alt+Insert 。

  3. 在“ 生成”窗口中，选择“ 测试文件” ，“ 测试软件包 ”或“ 测试功能” 。

有关更多信息gotests ，参见package gotests 。

产生版权

  1. 打开Goland设置Ctrl+Alt+S并导航到编辑器|版权|版权资料 。

  2. 点击添加图标 。

  3. 在“ 创建版权档案”对话框中，键入版权档案的名称。

  4. 在“ 版权文本”（可能包含“速度模板”）字段中，创建或修改版权模板。

  5. 单击确定 。

  6. 将插入符号放在您的代码上，然后按Alt+Insert 。

  7. 在“ 生成”窗口中，选择“ 版权” 。
