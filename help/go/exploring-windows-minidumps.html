探索Windows minidumps

您只能在Microsoft Windows操作系统上创建Windows小型转储。但是您可以在所有受支持的操作系统上分析和调试应用程序。

Windows minidump文件记录程序正在运行时或崩溃时的状态。要读取小型转储文件，您必须具有应用程序的二进制文件和转储文件。

创建您的应用程序的二进制文件

  1. 打开“ 运行/调试配置”对话框：

    • 选择运行|从主菜单编辑配置 。

    • 在导航栏可见（“ 视图” |“外观” |“导航栏” ）下，从运行/调试配置选择器中选择“ 编辑配置 ”。

    • 按Shift+Alt+F10 ，然后按0或从弹出窗口中选择配置，然后按F4 。

  2. 在“ 运行/调试配置”对话框中，单击“ 添加新配置”图标（ ）或按Alt+Insert 。该列表显示了默认的运行/调试配置。选择所需的配置类型（例如Go build ）。

    右侧窗格中显示的字段显示所选配置类型的默认设置。

  3. 在名称字段中，输入配置名称。

  4. 在“ 文件”字段中，添加要运行的文件的名称（例如， main.go ）。

  5. 在“ 输出目录”字段中，指定要在其中存储创建的二进制文件的路径。

  6. 应用更改并关闭对话框。

在Windows上创建minidump文件

  1. 启动该应用程序，并照常使用它。

    您可以运行生成的EXE文件，也可以从GoLand运行应用程序。要从GoLand运行应用程序，请单击“ 运行”图标在主功能附近的装订线中，然后选择Run'go build ' 。

  2. 打开Windows任务管理器Ctrl+Alt+Delete ，单击“ 进程”选项卡。

  3. 右键单击您的应用程序，然后选择创建转储文件 。

在GoLand中打开minidump文件

  1. 导航至运行|打开核心转储 。

  2. 在“ 可执行文件”字段中，指定EXE文件的路径。

  3. 在“ 核心转储”字段中，指定转储文件的路径。

  4. 单击确定 。
