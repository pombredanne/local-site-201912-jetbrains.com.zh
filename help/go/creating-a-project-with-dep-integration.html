通过dep集成创建项目

创建具有Dep集成的项目

通过Dep集成，您可以管理项目中的依赖项。

  1. 安装dep发行版 。

  2. 选择文件|新品|项目 。

  3. 在“ 新建项目”页面中，选择Dep 。

  4. 在位置字段中，指定要在其中存储项目的路径。

  5. 在“ GOROOT”字段中，指定Go安装的位置。通常，位置是自动定义的。

    要更改或安装新版本的Go SDK，请单击“ 添加SDK”按钮，然后选择“ 本地”以在硬盘驱动器上选择Go SDK版本，或选择“ 下载”以从官方存储库下载Go SDK。有关安装Go SDK的更多信息，请参阅安装Go SDK 。

  6. 从Dep可执行文件列表中，选择dep可执行文件的路径。要手动选择dep可执行文件的路径，请单击“ 浏览”按钮（ ），然后浏览至包含dep可执行文件的文件夹。

  7. 点击创建 。

确保在$ GOPATH / src下创建Dep项目。GOPATH变量指向包含bin ， pkg和src子目录的根目录。项目源位于src下。像这样的工具godep和go install对项目源而不是整个GOPATH进行操作 。有关dep项目的更多详细信息，请参阅Go Workspace文档。

如果要为项目源使用任何位置，请考虑使用Go模块集成 。

如果要禁用Dep集成，请清除“ 设置” |“设置”中的“ 启用dep集成”选项。去吧部门

导入依赖

  1. 单击依赖项声明。

  2. 按Alt+Enter然后选择dep sure 。

要查看添加的依赖性，请导航到“ 项目”工具窗口中的供应商目录。

为现有的Dep项目指定Dep可执行文件

  1. 打开设置Ctrl+Alt+S并导航到Go |部门

  2. 确保选中“ 启用dep集成”复选框。

  3. 从Dep可执行文件列表中，选择现有的Dep可执行文件。或者，单击浏览按钮并在硬盘上指定Dep可执行文件的路径。

安装Go SDK

选择Go SDK的本地副本

确保与围棋SDK的文件夹提供的路径包括仓 и 的src的文件夹。

  1. 在“ 新建项目”对话框中，选择“ Dep” 。

  2. 单击添加SDK按钮，然后选择本地 。

  3. 在文件浏览器中，导航到硬盘驱动器上的SDK版本。

  4. 点击打开 。

下载Go SDK

  1. 在“ 新建项目”对话框中，选择“ Dep” 。

  2. 点击添加SDK按钮，选择下载 。

  3. 从版本列表中，选择SDK版本。

  4. 在位置字段中，指定SDK的路径。要使用文件浏览器，请单击“ 浏览”图标 。

  5. 单击确定 。
