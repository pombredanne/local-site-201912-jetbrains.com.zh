JetBrains Rider 2019.2帮助

捷迅

您可以使用已经与JetBrains Rider捆绑在一起的JSHint linter检查代码。

启用JSHint并在JetBrains Rider中配置其行为

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |代码质量工具| JSHint 。

  2. 在打开的“ JSHint”页面上，选中“ 启用”复选框。之后，页面上的所有控件均可用。

  3. 从“ 版本”列表中，选择要使用的工具的版本。 JetBrains Rider与2.9.5版捆绑在一起，默认情况下使用。要下载另一个版本，请从列表中选择它。

  4. 配置JSHint的行为：

    • 要根据配置文件中的规则验证代码，请选中“ 使用配置文件”复选框。

      配置文件是扩展名为.jshintrc的JSON文件，它指定应启用或禁用哪些JSHint选项。JetBrains Rider首先在工作目录中查找.jshintrc文件。如果搜索失败，JetBrains Rider将在父文件夹中搜索，然后在父文件夹中再次搜索。重复该过程，直到JetBrains Rider找到.jshintrc或到达项目根目录为止。若要在项目中未找到.jshintrc时让JetBrains Rider仍运行验证，请指定要使用的默认配置文件。从JSHint官方网站了解有关JSHint配置文件的更多信息。

    • 要手动配置验证，请清除该复选框并在“ 选项”区域中启用相关验证。从JSHint官方网站了解更多信息。
