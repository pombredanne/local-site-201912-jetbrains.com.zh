演示设置

单击以调用选项列表，这些选项使您可以配置如何在“ 日志”视图中显示数据：
  • 紧凑引用视图 ：如果启用此选项，则在折叠视图中显示单个提交的分支引用：

    如果要在一行上展开每个分支引用，请取消选择此选项：

  • 显示标签名称 ：如果您希望除了标签图标之外还显示标签名称，请启用此选项：

    如果禁用此选项，则仍可以通过将鼠标悬停在标签图标上来查看标签名称。

  • 显示长边 ：如果启用此选项，则长分支将完整显示，即使其中没有提交也是如此。如果禁用此选项（默认情况下），则长分支将被向下箭头替换。

  • 折叠线性分支 ：启用此选项可以折叠图形上的所有分支，以便显示虚线而不是连续的提交。
    也可以通过单击折叠单个展开的分支。

  • 扩展线性分支 ：启用此选项可扩展所有折叠的分支，以在图形上显示连续的提交。
    也可以通过单击来展开单个折叠的分支。

  • 突出显示 ：如果要突出显示以下内容，请选择：

    • 我的承诺：粗体

    • 合并提交：变灰

    • 当前分支：蓝色背景

    • 未选择的提交：灰色（仅适用于Git）。未选择的提交是来自选定分支的尚未应用到当前分支的提交。

提交详细信息

显示详细信息时显示此区域选项已启用。

此区域显示在“ 提交”列表中选择的提交的详细信息，例如提交消息，哈希，作者，作者电子邮件的链接，日期，时间，根目录和分支。

请点击Ctrl+A在Windows / Linux上或⌘+A在macOS上选择提交消息，作者或标签，具体取决于您在“ 提交详细信息”窗格中单击的位置。

如果所选提交包含在六个以上的分支中，则仅显示前六个分支，并且显示全部显示链接，您可以单击以展开完整的分支列表。

如果一个提交消息引用了另一个提交，则可以单击此提交的哈希以在日志中跳转到该提交。
