封闭棉短绒

您可以直接从JetBrains Rider配置和使用Closure Linter 。

在你开始前

  1. 按照Python官方网站上的说明下载并安装Python。

  2. 按照Closure官方网站上的说明下载并安装Closure Linter。

启用Closure Linter并在JetBrains Rider中配置其行为

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），单击“ 语言和框架”下的“ JavaScript ”，然后单击“代码质量工具”下的“ 关闭Linter ” 。在打开的Closure Linter页面上，选中Enable复选框。之后，页面上的所有控件均可用。

  2. 指定Closure Linter可执行文件的路径：

    • Windows的\ Scripts \ gjslint.exe

    • / usr / local / bin / gjslint，用于Linux和macOS

  3. 指定先前创建的配置文件的路径。
