Navigate To Assembly Explorer

Navigate | Navigate To | Locate in Assembly Explorer
Alt+` | Locate in Assembly Explorer

This command helps you examine library types (study their members, base types, inheritors, and so on) using the Assembly Explorer view. The command becomes available when you invoke Navigate To on a library symbol in your source code and on decompiled sources. You can invoke this command from the editor, from the Structure window and other tool windows.
