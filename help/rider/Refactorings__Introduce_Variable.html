引入变量重构

重构|变量...
Ctrl+R, V

通过此重构，您可以基于选定的表达式创建新的局部变量或常量，使用表达式对其进行初始化，最后使用对新引入的变量的引用替换该方法中所有出现的表达式。

在下面的示例中，我们使用此重构将同一字符串的两次出现替换为一个变量：

重构后

static void LogError（Exception ex）{控制台。WriteLine（“发生某些事情失败...”）;文件。WriteAllText（@“ c：\ Error.txt”，“出现了某些问题...” + ex）; }
静态无效的LogError（Exception ex）{var message =“发生了某些故障...”；安慰。WriteLine（消息）;文件。WriteAllText（@“ c：\ Error.txt”，消息+前）； }

内联变量重构可以使用反向功能。

用变量替换一个或多个表达式的出现

  1. 在编辑器中选择一个表达式。

  2. 请执行以下任一操作：

    • 按Ctrl+R, V 。

    • 按Ctrl+Shift+R然后选择介绍变量

    • 选择重构|在主菜单中引入变量 。

  3. 如果找到一个以上的选定表达式，JetBrains Rider将显示一个下拉菜单，您可以在其中选择将重构应用于所有实例还是仅对当前实例进行重构。

  4. JetBrains Rider添加了新的局部变量，并建议选择其类型和名称。您可以选择建议之一，也可以键入自己的值。按Tab要么Enter移动到下一个输入位置。 Shift+Tab返回上一个输入位置。
