先决条件

  • 在您的计算机上安装了Mercurial 。

  • 在“设置/首选项”对话框的“ Mercurial”页面上正确指定了Mercurial可执行文件hg.exe的位置。

    如果遵循标准安装过程，则默认位置是/ opt / local / bin或Linux和macOS的/ usr / local / bin ，以及Windows的/ Program Files / TortoiseHG 。

    建议您将Mercurial可执行文件的路径添加到PATH 可变的 。在这种情况下，您只能指定可执行文件名称，而不需要可执行文件位置的完整路径。

  • 当前项目的根目录或目录启用了Mercurial 集成 。

如果要使用远程存储库，请首先创建一个Mercurial托管帐户。您可以通过一对ssh密钥访问远程存储库，或应用Mercurial集成支持的用户名/密码和键盘交互式身份验证方法。

商业支持

  • 启用与JetBrains Rider的Mercurial集成后， Mercurial项将出现在VCS菜单上，以及在Editor和Explorer窗口的上下文菜单中。

  • Mercurial控件下的文件夹中的文件会根据其状态突出显示。有关文件状态突出显示约定，请参见文件状态突出显示。

  • 修改结果显示在“ 版本控制”工具窗口中 。

  • 使用Mercurial集成时，打开“ 版本控制”工具窗口会很有帮助。“ 控制台”选项卡显示以下数据：

    • 根据您通过JetBrains Rider用户界面指定的设置生成的所有命令。

    • 有关执行生成的Mercurial命令的结果的信息消息。

    • 错误消息。
