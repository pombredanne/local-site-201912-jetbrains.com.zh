AngularJS

AngularJS也称为Angular 1是用于开发单页Web应用程序的框架。JetBrains Rider建议针对预定义和自定义的AngularJS感知完成选项ng指令以及控制器和应用程序名称，以及对花括号表达式内的数据绑定的代码见解{{}} 。您可以使用内置的AngularJS实时模板，并在HTML中的控制器名称和JavaScript中的控制器名称之间或之间导航。 ngView要么&routeProvider和模板。对于AngularJS实体，请使用“ 转到符号”导航。

在你开始前

  1. 下载并安装Node.js。

要打开按目录结构组织且没有解决方案文件的应用程序，请按Ctrl+O或选择 档案|打开|从主菜单中打开文件或文件夹... ，然后指定应用程序的根文件夹。

创建一个新的AngularJS应用程序

您可以通过下载AngularJS框架或使用Bower软件包管理器来手动将AngularJS安装在解决方案中。

在空解决方案中手动安装和配置AngularJS

  1. 在http://angularjs.org/下载AngularJS框架。

  2. 将AngularJS配置为JetBrains Rider JavaScript库，以使JetBrains Rider识别AngularJS特定的结构并提供完整的编码帮助：

    1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |图书馆 。将打开“ JavaScript库”页面 。

    2. 在“ 库”区域中，单击“ 添加”按钮。

    3. 在打开的“ 新建库”对话框中，指定库的名称。

    4. 请点击在库文件列表旁边，然后从上下文菜单中选择“ 附加文件”或“ 附加目录 ”，具体取决于您需要单独的文件还是整个文件夹。

    5. 在打开的对话框中选择Angular.js或Angular.min.js或整个目录。JetBrains Rider返回到“ 新库”对话框，其中“ 名称”只读字段显示所选文件或文件夹的名称。

    6. 在“ 类型”字段中，指定已下载并要添加的版本。

      • 如果添加了Angular.js ，请选择“ 调试” 。该版本对开发环境特别是调试很有帮助。

      • 如果添加了缩小的 Angular.min.js ，请选择Release 。该版本在生产环境中很有用，因为文件大小明显较小。

    从配置JavaScript库中了解更多信息。

通过Bower在空项目中安装AngularJS

  1. 安装鲍尔如描述鲍尔 。

  2. 在嵌入式终端 （ Ctrl+Alt+1 ），输入bower install angular将软件包安装为项目依赖项。

您也可以安装angular如“ 管理Bower程序包”中所述，在“设置/首选项”对话框的“ Bower”页上找到该程序包 。

从现有的AngularJS应用程序开始

如果您的项目中已经有Angular源（例如，在bower_components文件夹中），则只需打开项目并开始工作即可。如果这些源未从项目中排除 ，则只需要将AngularJS配置为JavaScript库 。

从版本控制中签出应用程序源

  1. 单击“ 欢迎”屏幕上的“ 从版本控制中签出”，或选择“ VCS |从主菜单中的“ 版本控制”中签出 。

  2. 从列表中选择您的版本控制系统。

  3. 在打开的特定于VCS的对话框中，键入您的凭据和存储库以从中检出应用程序源。
