把手/胡子

在此区域中，配置JetBrains Rider中的车把表达式和Mustache模板的处理。在此页面上指定的设置分别适用于扩展名为.hbs或.mustache的专用车把和Mustache文件。

  • 自动插入结束标记：

    • 选中此复选框后，JetBrains Rider会自动插入第二个右大括号}键入第一个结束句后 ，立即显示Handlebars表达式 。

      JetBrains骑士还发现三重藏匿处 {{{防止转义表达式内的值。在这种情况下，JetBrains Rider会在您键入第一个闭合大括号时自动插入两个闭合大括号。

    • 清除此复选框后，您必须手动键入右大括号和三重隐藏。

  • 启用格式：

    • 选中此复选框可在代码生成，重构或重新格式化期间自动重新格式化Handlebars表达式和Mustache模板 Ctrl+Alt+Enter 。

    • 清除复选框以保留“ 车把”表达式和“ Mustache”模板的原始格式。

  • 打开HTML文件作为把手/胡子：

    • 选中此复选框后，扩展名为.html的文件将被视为Handlebars / Mustache文件，因此JetBrains Rider可以识别和处理Handlebars表达式和Mustache模板 。文件扩展名保持不变，但文件名随图标。

    • 清除此复选框后，仅将扩展名为.hbs的文件视为Handlebars / Mustache文件，并识别并处理其中的Handlebars表达式和Mustache模板 。

  • 留言语言：在此列表中， 选择用于使用样式进行注释的语言。当您按以下内容输入行或注释时Ctrl+Alt+/要么Ctrl+Shift+/ ，JetBrains Rider插入用于所选语言的注释定界符，例如：

    • {{!----}}用于车把

    • /**/用于JavaScript

    • 用于HTML

有关Handlebars表达式和Mustache模板的更多详细信息，请参见http://handlebarsjs.com/ 。

EJS

在此区域中，为您的代码中包含EJS指定分隔符。基于这些定界符，JetBrains Rider将正确识别并处理此类包含物。一对定界符也称为execute tag 。

  • EJS开放定界符：在此字段中，输入将指示EJS包含开始的字符串。默认分隔符是<% 。

  • EJS关闭定界符：在此字段中，输入表示EJS包含结束的字符串。默认分隔符是%> 。
