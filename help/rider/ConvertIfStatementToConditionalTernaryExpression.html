JetBrains Rider 2019.2帮助

代码检查：“ if”语句可以重写为“？：”表达式

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

C＃中的条件？：（三元）运算符是根据简单布尔表达式的值执行两个简单表达式之一的最优雅的方法。

因此，如果JetBrains Rider遇到if...else在这种情况下，它建议将其转换为更简洁的?:操作员。

这是此检查建议的快速修复示例：

次优代码

快速修复后

void Foo（字符串msg1，字符串msg2，bool标志）{if（标志）控制台。WriteLine（msg1）;其他控制台。WriteLine（msg2）; }
void Foo（字符串msg1，字符串msg2，布尔标记）{控制台。WriteLine（flag？msg1：msg2）； }
