转到文件

导航|转到文件
Ctrl+Shift+T

此命令将您导航到其中的任何文件。

请注意，此命令在全局范围内有效，也就是说，您无需将焦点放在编辑器上即可调用它。

在当前解决方案中转到文件

  1. 按Ctrl+Shift+T或选择 导航|文件...从主菜单。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  2. 在出现的“ 输入文件名”弹出窗口中，开始输入项目名称。键入时，查找列表将变窄，并建议与输入的子字符串匹配的名称。

  3. （可选）选择包括非解决方案文件或按Ctrl+Shift+T将外部文件和文件夹包括到搜索结果中。

此处支持不区分大小写的CamelHumps ：例如，您可以通过键入“ LecS”轻松找到LocalElementConflictSearcher.cs 。

您也可以在指定名称时使用通配符：“ *”（星号）表示零个或多个字符； '+'（加号），'：'（分号），'。' （点）和'\'（反斜杠）将类型的完全限定名称分开。
