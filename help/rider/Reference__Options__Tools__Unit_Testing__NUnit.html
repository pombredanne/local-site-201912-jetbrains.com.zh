启用NUnit支持

默认情况下，JetBrains Rider添加操作指示器，使您可以直接从编辑器执行NUnit测试，并自动在解决方案中发现NUnit测试 。清除对应于特定NUnit版本的复选框以禁用此行为。
根据这些设置，JetBrains Rider将使用捆绑的NUnit测试运行程序：v。2.6.4和/或v。3.6.1。

自定义NUnit安装
使用指定的自定义NUnit

如果要使用另一个NUnit版本，请选中此复选框，然后在相应的字段中输入本地NUnit安装的路径。

笔记：

  • JetBrains Rider 2019.2支持NUnit 3.10.1和更高版本。

  • 如果此自定义安装中有任何NUnit插件 ，它们也将被加载。

NUnit 2
加载NUnit插件

如果上面选择了对NUnit 2x的支持，并且您需要将此捆绑版本使用NUnit插件 ，请将这些插件放入 JetBrains Rider安装目录的lib \ ReSharperHost \ NUnit2 \ addins子文件夹（为方便起见显示），然后选择如何加载它们。
