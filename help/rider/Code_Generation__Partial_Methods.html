生成部分方法

代码产生|部分方法
Alt+Insert | 部分方法

部分方法在部分类型的不同部分（不同文件）中定义-一个部分包含方法的签名，而另一部分包含其实现。

生成部分方法命令将帮助您为未实现的部分方法创建实现存根。

根据您的设置，可以用不同的方式生成新成员的存根。例如，他们可以抛出新的NotImplementedException() ，返回默认值或包含将无法编译的代码。您可以在以下位置配置这些首选项和其他首选项 编辑器|的会员生成页面 JetBrains骑士设置Ctrl+Alt+S 。

生成部分类型中的部分方法的实现

  1. 在编辑器中，将插入记号放在要插入部分方法的实现的行的部分类型内。

  2. 按Alt+Insert或选择 代码从主菜单生成...。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 部分方法” 。

  4. 在出现的“ 生成”对话框中，选择要为其创建实现存根的部分方法。

  5. 单击“ 确定”完成向导。
