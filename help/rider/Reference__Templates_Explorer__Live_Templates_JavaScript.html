预定义的JavaScript动态模板

本主题列出了JetBrains Rider 2019.2。中JavaScript的所有预定义实时模板。有关实时模板的更多信息，请参见使用实时模板创建源代码。

模板

细节

ctx

当前文件上下文

范围无处不在

身体

$ CTX $

参量

  • CTX -提供描述当前上下文的项目列表。这包括文件名，包含类型名，名称空间名等。

nguid

插入新的GUID

范围无处不在

身体

$ GUID $

参量

  • GUID -生成新的全局唯一标识符（GUID）

do （导入的JScript代码段）

做... while循环

范围 JS语句

身体

做{$ END $} while（$ COND $）

参量

  • COND -没有宏

  • END -应用模板后的插入符位置。

  • END -应用模板后的插入符位置。

  • END -应用模板后的插入符位置。

for （导入的JScript代码段）

for循环

范围 JS语句（直到ES 5，不在TS中）

身体

for（var $ VAR $ = 0; $ VAR $ <$ LENGTH $; $ VAR $ ++）{$ END $}

参量

  • VAR -建议未使用的名称，该名称可以在评估点用于索引变量。

  • LENGTH -没有宏

  • END -应用模板后的插入符位置。

参量

  • LENGTH -没有宏

  • VARKIND -显示指定的值列表。

  • END -应用模板后的插入符位置。
