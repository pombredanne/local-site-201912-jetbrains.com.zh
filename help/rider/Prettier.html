JetBrains Rider 2019.2帮助

更漂亮

漂亮是格式的.js，.TS， 的CSS，.LESS，.scss，.vue，并以.json代码的工具。使用JetBrains Rider，您可以使用“ 使用更漂亮的格式重新格式化”操作来格式化选定的代码片段以及整个文件或目录。在您将Prettier作为依赖项安装到项目中或在计算机上全局安装后，JetBrains Rider就会添加此操作。

在你开始前

  1. 确保您的计算机上装有Node.js。

  2. 如管理插件中所述，在“ 插件”页面上安装并启用Prettier存储库插件。

安装漂亮的

  • 在嵌入式终端 （ Ctrl+Alt+1 ），键入以下命令之一：

    • npm install --save-dev --save-exact prettier
    • npm install --global prettier

    从Prettier官方网站了解有关安装模式的更多信息。

在JetBrains Rider中配置漂亮的人

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |更漂亮 。

  2. 在打开的“ 漂亮的页面”上，指定漂亮的包的路径，然后选择要使用的Node.js解释器。这可以是本地Node.js解释器，也可以是Windows Subsystem for Linux上的Node.js。

使用Prettier重新格式化代码

  • 在编辑器中选择要重新格式化的代码片段，或者在“ 项目”工具窗口中选择文件或文件夹，然后从上下文菜单中选择“ 使用漂亮的格式重新格式化”。

  • 或者，按Ctrl+Shift+A并从“ 查找操作”列表中选择“ 使用更漂亮的格式重新格式化 ” 。

JetBrains Rider可以将Prettier的配置中的关键代码样式规则应用于JetBrains Rider Code Style设置，以使生成的代码（例如，在重构或快速修复之后）和已使用Prettier处理的代码保持一致的格式。

应用更漂亮的代码样式规则

  • 在启用Prettier的项目中，打开package.json ，然后在选项卡顶部的窗格中单击“ 是 ”。

  • 要重新应用更漂亮的代码样式（在窗格中单击“ 否 ”或修改代码样式后），请按Ctrl+Shift+A并从“ 查找操作”列表中选择“ 应用更漂亮的代码样式规则 ” 。
