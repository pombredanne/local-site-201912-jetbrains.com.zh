语言注入设置对话框：Sql类型注入

档案|设置|编辑器语言注入| | Windows和Linux的SQL类型注入
JetBrains骑士|首选项|编辑器语言注入| |适用于macOS的SQL类型注入

指定数据类型模式和关联的注入语言。

项目

描述

字型

SQL代码中数据类型的正则表达式模式。例如(?i).*DATA对于以结尾的数据类型将是不区分大小写的模式data 。

您可以测试您的模式：单击或按Alt+Enter ，然后选择Check RegExp 。然后在“ 样本”字段中键入要与模式匹配的文本。

语言

要注入到相应类型的字符串值中的语言。

  • ID。 语言ID或名称。

  • 字首。 在相应的字符串值之前要添加的字符序列。

  • 后缀。 在相应的字符串值之后要添加的字符序列。

前缀和后缀是可选的。有关更多信息，请参见 。
