帕格（玉）模板引擎

JetBrains Rider与Pug（Jade）模板引擎集成，该引擎将Pug（Jade）文件转换为HTML。

哈巴狗文件标有 ;翡翠文件标有 。

在你开始前

  1. 下载并安装Node.js。

  2. 如管理插件中所述，在“ 插件”页面上安装并启用Pug（以前的Jade）存储库插件。

另外，如果您需要文件监视程序，请确保已安装并启用了文件监视程序插件。 的插件未与JetBrains Rider捆绑在一起，但可以如管理插件中所述从JetBrains插件存储库安装。

安装翡翠

  • 在嵌入式终端 （ Ctrl+Alt+1 ），输入：

    npm install pug

要即时编译代码，您需要将模板引擎配置为JetBrains Rider File Watcher 。

创建一个Pug / Jade文件观察器

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 工具|文件监视程序 。

  2. 请点击或按Ctrl+N然后从列表中选择Pug / Jade预定义模板。您的代码将转换为JavaScript并随生成的源映射一起提供 。

  3. 在程序字段中，指定到的路径pug要么jade包。手动输入路径或单击然后在打开的对话框中选择文件位置。

  4. 按照文件监视程序中所述进行操作。

当您打开文件时，JetBrains Rider会检查当前项目中是否有适用的文件监视程序。如果已配置但禁用了此类文件监视程序，JetBrains Rider将显示一个弹出窗口，通知您有关已配置的文件监视程序并建议启用它。

如果在当前项目中配置并启用了适用的文件监视程序，JetBrains Rider会在“ 新建监视程序”对话框中指定的事件后自动启动它。

  • 如果选中了“ 自动保存编辑的文件以触发监视程序”复选框，则在对源代码进行任何更改后都会立即调用文件监视程序。

  • 如果清除了自动保存编辑的文件以触发监视程序复选框，则文件监视程序会在保存时启动（ File | Save All ， Ctrl+Shift+S ），或者当您从JetBrains Rider移动焦点时（关闭框架时）。

配置语法突出显示

您可以根据自己的喜好和习惯配置Pug / Jade-aware语法高亮显示。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |配色方案|帕格/玉

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
