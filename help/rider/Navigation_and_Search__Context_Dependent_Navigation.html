JetBrains Rider 2019.2帮助

浏览当前上下文

JetBrains Rider提供了许多功能，可帮助您根据打开的文档及其在其中的插入位置，根据当前上下文导航代码。

导航文件结构

使用JetBrains Rider，您可以使用“结构”窗口导航当前文档的结构，这大大简化了大型文件的导航。该窗口与编辑器同步：切换到另一个编辑器选项卡时，该窗口显示相应文件的结构。

  1. 按N/A或选择查看| Windows工具|结构 。

  2. 当前文档的内容以树形结构显示，您可以浏览并浏览该文档。您可以双击任何符号，以使编辑器中的插入符号切换到相应位置。

  3. 如果代码的某些部分被#region块，区域也会出现在文件结构中。

  4. 要使您在编辑器中更改插入符号位置的方式与“ 文件结构”窗口中突出显示的符号同步，请单击 从源自动滚动 在工具栏上。

  5. 要将您在“ 文件结构”窗口中选择的符号与编辑器中的插入符号位置同步，请单击 自动滚动到源 在工具栏上。

在“结构”窗口中，您可以访问更多操作。要检查可用命令，可以右键单击任何符号。

  • 查找符号的用法 Shift+F12

  • 执行重构 （ Ctrl+Shift+R ）

  • 如果打开的文件包含单元测试，则这些单元测试将用相应的图标标记，以显示其上次执行的状态。您也可以直接在“结构”窗口中运行和调试单元测试 。

要在“结构”窗口中搜索项目，只需开始输入名称或CamelHumps缩写。

使用编辑器面包屑

编辑器面包屑可帮助您快速了解当前文档结构的插入符号以及导航该结构。启用编辑器面包屑后，您将始终在编辑器中始终看到从根代码元素到当前插入符号位置的路径。

要跳至源代码中的父元素，请单击带有其名称的面包屑。如果单击C＃/ VB.NET文件中的名称空间元素，将能够导航到使用该名称空间的所有文件。

默认情况下，面包屑显示在编辑器的底部。要将其显示在顶部或将其完全隐藏，请右键单击面包屑栏，然后选择“ 面包屑” | “面包屑”。顶部或面包屑|不要显示 。

您还可以执行此操作，还可以配置应在哪种语言和颜色上显示编辑器面包屑 编辑器|一般| JetBrains Rider设置的面包屑页面Ctrl+Alt+S 。

在编辑器框架之外查看代码

您可以将鼠标悬停在滚动条上以查看当前可见区域上方或下方的源代码：

您可以通过清除“ 滚动条上的显示代码镜头悬停”复选框来禁用此行为。 编辑器|一般|的外观页面 JetBrains骑士设置Ctrl+Alt+S 。

设置中用于此功能的术语“ 代码镜头”与Visual Studio的CodeLens功能无关 ，在JetBrains Rider中称为“代码视觉” 。

导航到包含声明

使用单个快捷键，您可以在编辑器中从其体内的任何位置跳转到类型或函数的声明。

  1. 将插入号设置为类型或函数体内的任何位置。

  2. 按Ctrl+[ 。

  3. JetBrains Rider会将插入符号设置为类型或函数的声明。

  4. 如果在另一个类型中声明了该类型或函数，则可以重复第二步以导航到包含类型的声明。

在成员和标签之间导航

使用“ 下一个/上一个成员”命令，您可以在代码文件中的成员和嵌套类型之间或标记文件中的标记之间快速导航。

  1. 当焦点位于编辑器中时，按Alt+Down / Alt+Up或选择导航|主菜单中的下一个/上一个成员 。

  2. 插入符号将移至下一个/上一个成员/标签的声明。如果存在内部类型，则插入符号还将在每个内部类型的成员处停止。

浏览编辑器选项卡和工具窗口

您可以像在OS应用程序切换器中一样在编辑器中打开的文件，拆分编辑器选项卡和工具窗口（对接或浮动）之间切换。切换器会弹出一个由两列组成的弹出窗口：左侧显示工具窗口，右侧显示编辑器选项卡。如果当前未打开任何文件，或者只有一个选项卡，则切换器将显示最近打开的文件：

  1. 按Ctrl+Tab然后按住第一个键（ Ctrl在Windows / Linux上或⌃在macOS上），请使用以下键：

    • Left和Right在工具窗口和文件列表之间切换。

    • Up和Down方向键， Tab要么Shift+Tab在两个窗格中上下移动列表。

    • Delete要么Backspace关闭打开所选文件的编辑器选项卡，然后从列表中删除所选文件。

  2. 释放Ctrl / ⌃键。相应的文件或工具窗口将成为焦点，并且切换器弹出窗口消失。

在其他视图中查找当前项目

在编辑器或工具窗口之一中选择文件，类型，成员或其他代码项时，可以在另一个IDE组件中快速找到它。

  1. 按Alt+F1或选择导航|从主菜单中选择“进入 ”。出现“ 选择目标”弹出窗口。

  2. 使用箭头键或鼠标指针选择所需的组件。如果您的目标是“解决方案”窗口，则可以选择所需的视图：

您也可以在一个工具窗口中双击选定的文件或成员，或按Enter在编辑器中打开它。

在解决方案/组装资源管理器中查找当前文档

此便捷的导航功能使您可以在“解决方案资源管理器”中展开树，自动滚动并突出显示与当前文档相对应的项目。当您拥有庞大的解决方案或有限的屏幕区域（迫使您将“解决方案资源管理器”窗口缩小至极限）时，此功能特别有用。

请注意，如果当前解决方案中未包含在编辑器中打开的文件，则此命令将不起作用。

您也可以在使用该符号的源代码中直接在Assembly Explorer中找到一个库符号：插入符号在符号上时，按Alt+`或选择 导航|在主菜单中导航到… ，然后选择在装配体浏览器中定位 。

  1. 按Shift+Alt+L 。

  2. JetBrains Rider将调出“解决方案资源管理器”窗口以聚焦并向您显示该项目。

通过数字导航代码行


“ 转到行”命令使您可以快速跳至当前文档中的任何代码行。

要在编辑器中 显示行号 ，请选择在以下位置显示行号 编辑器|一般|的外观页面 JetBrains骑士设置Ctrl+Alt+S 。

  1. 按Ctrl+G或选择导航|菜单中的行/列 。

  2. 在冒号后输入目标行号和（可选）列号: ，然后单击“ 确定” 。
