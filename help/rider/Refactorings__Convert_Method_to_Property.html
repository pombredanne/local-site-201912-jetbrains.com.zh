将方法转换为属性重构

此重构可帮助您将具有返回值且无参数的方法转换为具有相应属性的属性get访问器和方法void返回类型和属性的一个参数，以及相应的set访问器。

如果存在具有单个访问器的现有属性，并且转换后的方法具有兼容的参数类型或返回类型，则重构可以将第二个访问器添加到现有属性中。

反向功能可通过“ 将属性转换为方法”重构来使用。

考虑以下示例：

重构之前

应用于“ GetFaculty”

然后应用于“ SetFaculty”

学生类{私人弦乐系；公共字符串GetFaculty（）{返回教职； } public void SetFaculty（string f）{faculty = f; }}
学生类{私人弦乐系；公共字符串Faculty {get {return faculty; }} public void SetFaculty（string f）{faculty = f; }}
学生类{私人弦乐系；公共字符串Faculty {get {return faculty; }设置{教师=值； }}}

将方法转换为属性

  1. 将插入号放在编辑器中的声明或方法的用法处，或在“ 结构”窗口中选择它。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择“ 将方法转换为属性”

    • 选择重构|在主菜单中将“ 方法转换为属性 ”。

    将打开“ 将方法转换为属性”对话框。

  3. 输入属性名称的名称保留默认建议。如果转换后的方法适合作为第二个访问器，则可以使用单个访问器键入现有属性的名称。

  4. 要应用重构，请单击下一步 。

  5. 如果未发现冲突，JetBrains Rider将立即执行重构。否则，它会提示您解决冲突 。

    重构完成后，它将在任何地方进行的所有更改（包括其他文件）都注册为一个操作。这样您就可以使用“ 撤消”操作Ctrl+Z只需一次击键即可回滚所有这些更改。
