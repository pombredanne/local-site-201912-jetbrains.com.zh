密码

使用基于目录的设置机制可以保存在此页面上可配置的首选项。

档案|设置|外观和行为系统设置| Windows和Linux的密码
JetBrains骑士|首选项|外观和行为系统设置| macOS的密码
Ctrl+Alt+S

指定JetBrains Rider是否应保存您的密码-您用于访问受密码保护的资源（例如版本控制存储库和数据库）的密码。

当您使用KeePass密码管理器时 ，主密码将用于访问存储各个密码的文件。一旦JetBrains Rider记住您的密码，除非您需要访问密码数据库，否则它将不会再次要求输入密码，包括主密码。

在保存密码下 ，您可以配置密码设置。

项目

描述

在本机钥匙串中

JetBrains Rider仅在macOS和Linux上显示此选项。
选择此选项可使用本机钥匙串存储您的密码。

在KeePass中

选择此选项可使用KeePass密码管理器存储密码。

数据库

此字段显示当前c.kdbx文件的位置。如果需要选择其他位置，请单击在打开的对话框中，选择合适的对话框 。
如果要导入另一个c.kdbx文件，请单击图标，然后从列表中选择导入 。如果要删除c.kdbx文件中的现有密码，请选择清除 。

主密码

使用此字段输入要用于访问c.kdbx文件的密码。JetBrains Rider第一次自动生成密码。

不保存，重启后忘记密码

如果要在重启后删除包含单个密码的c.kdbx文件，请选择此选项。
