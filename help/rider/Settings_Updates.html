Updates

Preferences configurable on this page are saved using the mechanism of directory-based settings.

File | Settings | Appearance and Behavior | System Settings | Updates for Windows and Linux
JetBrains Rider | Preferences | Appearance and Behavior | System Settings | Updates for macOS
Ctrl+Alt+S

This page contains information about the current JetBrains Rider version and availability of a newer version. You can also configure how you want to update JetBrains Rider.

Item

Description

Automatically check for updates for

Select whether you want JetBrains Rider to check for updates automatically and choose an update channel.

  • Early Access Program: Provides all updates, including weekly EAP releases. This channel is not recommended for production development.

  • Beta Releases or Public Previews: Includes stable releases, release candidates, and beta releases. Some updates in this channel may contain minor bugs and feature previews.

  • Stable Releases: Includes only stable releases that are recommended for production.

The list of update channels is only available for stable versions. EAP versions always use the Early Access Program channel.

Learn more about the Early Access Program, or EAP, from http://eap.jetbrains.com/.

Check Now

Check for updates immediately.

Alternatively, you can select Help | Check for Updates (for Windows or *NIX) or JetBrains Rider | Check for Updates (for macOS) from the main menu.

View/edit ignored updates

Show the list of updates that were ignored. These updates will not be suggested until you remove them from the list of ignored updates.
