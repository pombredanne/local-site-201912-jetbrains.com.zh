的HTML

JetBrains Rider为HTML提供了强大的支持，其中包括语法和错误突出显示，根据代码样式进行格式设置，结构验证，代码完成，调试会话期间的实时预览（ Live Edit ）等。

HTML规范可以使用默认HTML语言级别首选项配置 语言和框架| JetBrains Rider设置的“ 模式和DTD”页面Ctrl+Alt+S 。默认情况下，假定使用W3C的HTML 5.0规范。

创建一个HTML文件

  • 从主菜单中，选择文件|单击“ 新建” ，然后从列表中选择“ HTML文件 ”。JetBrains Rider基于HTML文件模板创建一个存根文件，并在编辑器中将其打开。
