JetBrains骑士|首选项|工具|适用于macOS的文件监视程序
Ctrl+Alt+S

启用File Watchers插件后，该页面可用。

File Watcher是一个JetBrains Rider系统，它可以跟踪对文件的更改并运行第三方独立应用程序。JetBrains Rider为许多此类标准流行的第三方工具（ 编译器 ，压缩器，修饰词等）提供了预定义的File Watcher模板。您还可以配置自定义File Watcher以运行任何其他第三方工具。

文件监视程序具有两个专用的代码检查 ：

  • 适用于预定义File Watcher的每个文件都会运行File watcher可用检查。如果项目没有配置相关的File Watcher，JetBrains Rider建议添加一个。

  • 正在运行的File Watcher调用File Watcher 问题检查，并突出显示特定于它的错误。

该页面包括两部分：

  • 当前项目中可用的文件监视程序列表。要激活File Watcher ，请选中它旁边的复选框。如果在运行文件监视程序时发生错误，则会自动禁用文件监视程序。

  • 用于管理此列表的工具栏。

工具提示/
和捷径

描述

加
Ctrl+N

单击此按钮打开“ 选择模板”弹出窗口，然后选择“ 文件监视程序”的相关类型。之后，JetBrains Rider将打开“ 新建监视程序”对话框，以根据当前项目的设置来自定义预定义的文件监视程序 。

编辑
Enter

单击此按钮可在“ 编辑监视程序”对话框中更新所选文件监视程序的设置。此更新仅适用于当前项目File Watcher ，它不会影响预定义的JetBrains Rider级模板。

去掉
Ctrl+R, D

单击此按钮删除选定的File Watcher 。File Watcher不再应用于当前项目中的文件。请注意，此操作不会影响相应的预定义模板，该模板在JetBrains Rider级别仍然可用。

 

上Ctrl+Alt+Up
下Ctrl+Alt+Down

使用这些按钮可以更改列表中File Watcher的顺序。如果启用了多个功能，这将确定启动File Watchers的顺序。

复制

使用此按钮创建所选文件查看器的副本。

进口

单击此按钮导入现有的文件监视程序，并将其添加到可用文件监视程序列表中。

出口

单击此按钮可将所选观察者导出到位于用户主目录下的watchers.xml文件。
