代码检查指标

在本节中，您将找到所有JetBrains Rider的 专有代码检查 ，您可以禁用或更改其严重性水平 。
这些代码检查可帮助您在设计时检测所有打开文件中的代码问题，此外，它们还使您可以查找特定范围内的代码问题 。本节中的主题按语言对代码进行检查：

  • C＃中的代码检查
  • VB.NET中的代码检查
  • XAML中的代码检查
  • ASP.NET中的代码检查
  • HttpHandler或WebService中的代码检查
  • 剃刀中的代码检查
  • 资源文件中的代码检查
  • MSBuild中的代码检查
  • NAnt中的代码检查
