C＃的预定义环绕声模板

本主题列出了JetBrains Rider 2019.2。中C＃的所有预定义环绕模板。有关环绕声模板的更多信息，请参见带有模板的环绕声代码片段。

模板

细节

foreach

foreach块

范围 C＃2.0+语句

身体

foreach（$ COLLECTION $中的$ TYPE $ $ VARIABLE $）{$ SELECTION $$ END $}

参量

  • COLLECTION -在评估变量的位置显示基本代码完成列表

  • TYPE -建议在模板中声明的新变量的类型

  • VARIABLE -在变量声明中执行时（变量名应保留在其中），建议变量名。

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

for

简单的“ for”循环

范围 C＃2.0+语句

身体

for（int $ INDEX $ = 0; $ INDEX $ <$ UPPER $; $ INDEX $ ++）{$ SELECTION $$ END $}

参量

  • INDEX -建议未使用的名称，该名称可以在评估点用于索引变量。

  • UPPER -没有宏

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。
