JetBrains Rider 2019.2帮助

编码协助

JetBrains Rider提供了多种编码辅助功能，使您可以更快地编写和编辑代码，从而提高工作效率并节省时间。

除了本节中介绍的功能外，JetBrains Rider还提供了许多代码生成功能和代码模板 ，可帮助您更快地创建代码。

在这个部分：

  • 代码完成（IntelliSense）
    • 自动完成
    • 基本完成
    • 智能完成
    • 类名完成
    • 生成完成
  • 打字协助
  • 完整陈述
  • 色彩辅助
  • 上下文动作
  • 正则表达式协助
  • C＃语言注入
  • 网络语言中的语言注入
  • 内联参数名称提示
  • 参数信息
  • 快速文档
  • 高亮匹配定界符
  • 重新排列代码元素
  • 重复的行或选择
  • 包含字符串文字的分割线
  • 添加和删除#region块
  • 注释和注释代码
  • 智能粘贴
  • 驼峰

您可以将以下快捷方式用于高级代码编辑助手：
