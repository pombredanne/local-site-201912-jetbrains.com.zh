Navigate To Overriding Members

Navigate | Navigate To | Overriding Members
Alt+` | Overriding Members

This command navigates to all abstract and virtual members of a base class that are overridden in the current class.

The command becomes available when you invoke Navigate To upon a base class entry in a class declaration. It works similar to the Navigate To Implementing Members command.

If there is only one overriding member, JetBrains Rider navigates directly there. If there are several overriding members, each of them is highlighted with blue.

The highlights are also duplicated with %blue% markers on the error stripe on the right. Checking these markers may be helpful if not all highlights are visible on the current screen.
To navigate between the highlighted items, choose Edit | Find | Find Next/Previous in the menu or press F3 / Shift+F3.
When you finish examining the highlights, press Esc to remove them.
