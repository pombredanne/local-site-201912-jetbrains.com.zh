智能完成

代码完成|智能型
Ctrl+Alt+Space

此命令过滤类型，方法和变量的列表以匹配表达式的预期类型。只要JetBrains Rider可以确定表达式的期望类型（例如赋值的右侧，变量初始化程序或return语句），它就可以在任何地方使用。如果可能有几种返回类型，JetBrains Rider会检测适用类型的最近使用符号并在查找列表中自动选择它。Smart Completion还可以建议创建匿名委托，lambda表达式和正则方法，以及为out参数创建局部变量。除此之外，在对象初始化程序中，“智能完成”功能会建议尚未分配值的字段/属性。

调用智能完成

  1. 将插入符号放置在您要键入代码的位置。

  2. 按Ctrl+Alt+Space 。

  3. 从列表中选择一个建议，然后按Enter或开始键入标识符的首字母或其CamelHumps缩写，以缩小建议列表的范围。

您还可以在现有标识符上调用智能完成。在这种情况下，有两种方法可以应用所选的完成建议；按Enter插入并保留现有项目或按Tab替换现有项目。
