JetBrains Rider 2019.2帮助

C＃的预定义文件模板

本主题列出了JetBrains Rider 2019.2。中C＃的所有预定义文件模板。有关文件模板的更多信息，请参见从模板创建文件。

模板

细节

Class

在当前项目中创建一个包含空C＃类的新文件。

范围 C＃项目

身体

$ HEADER $ namespace $ NAMESPACE $ {公共类$ CLASS $ {$ END $}}

参量

  • HEADER -插入在JetBrains Rider设置中指定的文件头。

  • CLASS -评估当前文件名，不带扩展名，所有非字母数字均用下划线替换

  • NAMESPACE -获取当前文件的默认名称空间

  • END -应用模板后的插入符位置。

在当前项目中创建一个包含空C＃接口的新文件。

身体

Test Fixture

测试治具

范围 C＃项目，参考nunit.framework

身体

使用NUnit的$ HEADER $。框架；名称空间$ NAMESPACE $ {[TestFixture]公共类$ CLASS $ {$ END $}}

参量

  • HEADER -插入在JetBrains Rider设置中指定的文件头。

  • CLASS -评估当前文件名，不带扩展名，所有非字母数字均用下划线替换

  • NAMESPACE -获取当前文件的默认名称空间

  • END -应用模板后的插入符位置。

Razor MVC Partial View

剃刀MVC局部视图

范围 Razor（C＃）项目，Razor（C＃）视图文件夹

身体

@model $ model $ $ END $

参量

  • model -预计此时的猜测类型

  • END -应用模板后的插入符位置。

Razor MVC View

剃刀MVC视图

范围 Razor（C＃）项目，Razor（C＃）视图文件夹

身体

@model $ model $ @ {布局= null; } $ title $
$ END $

参量

  • model -预计此时的猜测类型

  • title -没有宏

  • END -应用模板后的插入符位置。

Razor MVC View with Layout

带有布局的Razor MVC视图

范围 Razor（C＃）项目，Razor（C＃）视图文件夹

身体

@model $ model $ @ {ViewBag。标题=“ $ title $”;布局=“ $ _Layout $”; }

$ title $

$ END $

参量

  • model -预计此时的猜测类型

  • title -没有宏

  • _Layout -在评估变量的位置显示智能代码完成列表

  • END -应用模板后的插入符位置。
