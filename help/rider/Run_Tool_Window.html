运行窗口

查看|工具窗口|跑
Alt+4

运行工具窗口显示您的应用程序生成的输出。如果您正在运行多个应用程序，则每个应用程序都将显示在一个以应用运行/调试配置命名的选项卡中。

每个选项卡的外观取决于正在运行的应用程序的类型，并且可以包括其他工具箱和窗格。

运行工具窗口的主工具栏使您可以重新运行，停止，暂停或终止应用程序。下表包含大多数应用程序中常见按钮的说明。

单击此按钮以重新运行当前应用程序。
停止应用程序时出现此按钮 。当应用程序运行时，此按钮切换到 。

停止
Shift+F5

单击此按钮可通过标准在外部终止当前过程shutdown脚本。

单击该按钮一次将调用软终止，从而允许应用程序捕获SIGINT事件并执行正常终止（在Windows上， Ctrl+C事件被模拟）。单击一次按钮后，将其替换为指示随后的单击将导致强制终止应用程序，例如在Unix上SIGKILL已发送。

恢复布局

单击此按钮可以放弃对当前布局的更改，并返回到默认状态。

固定标签

单击此按钮固定或取消固定当前选项卡。您可能需要固定标签，以防止在此窗口中达到最大标签数时自动关闭标签。

关
Ctrl+Shift+F4

单击此按钮以关闭“ 运行工具”窗口的选定选项卡，并终止当前过程。

与剪贴板比较

打开剪贴板与编辑器对话框，使您可以查看编辑器中的选择与当前剪贴板内容之间的差异。此对话框是常规的比较工具 ，使您可以将插入符处的行复制到剪贴板，查找文本，在差异之间导航和管理空白。

描述

上/下堆栈跟踪
Ctrl+Alt+Up Ctrl+Alt+Down

单击此按钮可以在堆栈跟踪中向上或向下导航，并使光标跳至源代码中的相应位置。

使用软包装

单击此按钮可切换输出的自动换行模式。

滚动到最后

单击此按钮导航到堆栈跟踪的底部，并使光标跳至源代码中的相应位置。

打印

单击此按钮可将控制台文本发送到默认打印机。

全部清除

单击此按钮可从控制台中删除所有文本。也可以从控制台的上下文菜单中使用此功能。
