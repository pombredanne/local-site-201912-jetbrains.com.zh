类名完成

Shift+Alt+Space

该命令从当前项目中引用的模块中建议类型和扩展方法，而不管其名称空间是否导入当前文件中。每个建议项的名称空间都显示在建议列表的右侧。一旦选择了非导入类型，就会自动添加所需的名称空间导入指令。

调用类名完成

  1. 将插入符号放置在您要键入代码的位置。

  2. 按Shift+Alt+Space 。

  3. 从列表中选择一个建议，然后按Enter或开始键入标识符的首字母或其CamelHumps缩写，以缩小建议列表的范围。

您还可以在现有标识符上调用类名完成。在这种情况下，有两种方法可以应用所选的完成建议；按Enter插入并保留现有项目或按Tab替换现有项目。

从类名完成中排除项目

您可能希望不建议使用某些符号，例如，如果解决方案中的内容与系统类型相似，请说MyFramework.MyCollections.List ，但实际上并没有使用它。要将此类符号从建议中排除 ，请将其添加到“ 从导入和完成中排除”列表中 编辑器|一般|的自动导入页面 JetBrains骑士设置Ctrl+Alt+S 。 条目的格式为Fully.Qualified.Name ， Fully.Qualified.Name.Prefix* ， 要么*Fully.Qualified.Name.Suffix 。通用类型指定为List`1 。
