代码检查：数组创建中的冗余显式类型

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

从C＃3.0开始，支持隐式类型的数组 。数组实例的类型由数组初始化程序中指定的元素类型确定，因此您无需显式指定数组类型。初始化对象或集合时，此类数组很方便。

JetBrains Rider建议使用快速修复程序，删除多余的类型规范。

次优代码

快速修复后

var x = new int [] {1，2，3};
var x = new [] {1，2，3};
