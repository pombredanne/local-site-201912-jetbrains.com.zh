转到衍生符号

导航|实施方式
Alt+End

此命令获取当前符号的所有派生符号的列表，并让您直接沿继承层次结构导航至其中一个。有一个类似的命令Go to Implementation ，它仅列出实现，而在继承链中没有中间步骤。

除了导航之外，您还可以直观地浏览符号的完整继承层次。

您可以从编辑器，“ 结构”窗口和其他工具窗口中调用此命令。如有必要，可以使用“ 转到基本符号”命令沿相反方向导航。

导航到派生符号

  1. 将插入号放置在编辑器中的符号上，或在“ 结构”工具窗口中选择符号。

  2. 按Alt+End ，选择 导航|实施（S）在主菜单，或点击符号控股Ctrl+Alt键。

  3. 如果该符号只有一个衍生符号，JetBrains Rider会直接导航到该符号。

  4. 如果该符号具有多个派生符号，它们将在下拉列表中列出。 该列表可能包含粗体和常规字体的项目。粗体表示项目具有直接继承，而常规字体用于间接继承。

    您可以执行以下操作之一：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 开始输入以过滤结果。请注意，您可以在此处使用CamelHumps 。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找窗口中打开 ， 按Shift+Enter要么+在数字键盘上。

此命令也可以在“ 导航到”快捷方式中找到Alt+` 。

当类型或成员在当前解决方案中具有继承者时，您可以通过出现特殊图标（例如， ）显示在声明的左侧。您可以单击图标以展开派生符号列表并进行导航。
