JetBrains Rider 2019.2帮助

Stylelint

JetBrains Rider与Stylelint集成在一起，因此您可以从IDE内部检查CSS代码。当您编辑CSS文件时，Stylelint会自动唤醒，并突出显示检测到的问题 。要在信息弹出窗口中查看问题的描述，请将鼠标指针悬停在“ 验证”侧栏中的条纹上。要检查错误并应用建议的快速修复方法 ，请按Alt+Enter 。

在开始之前，请确保您的计算机上装有Node.js。

安装Stylelint

  • 在嵌入式终端 （ Ctrl+Alt+1 ），输入：

    npm install -g stylelint

您还可以按照npm和Yarn中的描述，在Node.js和NPM页面上安装stylelint软件包。

激活和配置Stylelint

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架|样式表| Stylelint 。

  2. 在打开的Stylelint页面上，选中“ 启用”复选框以激活Stylelint。之后，对话框中的控件将变为可用。

  3. 在“ 节点解释器”字段中， 指定要使用的Node.js解释器。这可以是本地Node.js解释器，也可以是Windows Subsystem for Linux上的Node.js。

  4. 在Stylelint套件字段中， 指定在全局或当前项目中安装的stylelint软件包的位置。

  5. 如果您希望Stylelint在编辑JavaScript文件并选中其中的“样式表”注入时也自动唤醒，请选中“ 处理JS文件”复选框。
