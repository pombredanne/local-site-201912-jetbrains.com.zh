Navigate To Generic Substitutions

Navigate | Navigate To | Generic Substitutions
Alt+` | Generic Substitutions

This command helps you determine exactly what types are used to substitute generic type parameters in usages of the generic symbols in the solution. The command becomes available when you invoke Navigate To on a declaration or a usage of any generic type parameter, or or any generic substitution. You can invoke this command from the editor, from the Structure window and other tool windows.

All found substitutions are displayed in the Find Results window, where you can further analyze the results.
