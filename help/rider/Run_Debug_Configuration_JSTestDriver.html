在此字段中，指定运行/调试配置的名称。该名称将帮助您在以后选择编辑它或从“ 运行”弹出窗口中调用它时识别所创建的配置。 Ctrl+Shift+Alt+R 。

通过VCS共享

选中此复选框可将运行/调试配置置于版本控制下，并使其可用于其他团队成员。

共享的运行/调试配置保存在.idea \ runConfigurations文件夹下的单独xml文件中，而本地运行/调试配置保存在.idea \ workspace.xml中 。

允许并行运行

如果选中此复选框，则可以并行启动运行配置多次，而不必重新运行它。每个运行程序都将在“ 运行工具窗口”或“ 调试工具窗口 ”的自己的选项卡中启动 。

禁用后，每次启动运行/调试配置的新实例时，JetBrains Rider都会检查同一运行/调试配置的其他实例是否存在，并显示确认对话框。如果在确认对话框中单击“ 确定” ，则将停止运行器的第一个实例，而将其替换。当启动两个相同类型的运行/调试配置实例时，这会占用过多的CPU和内存资源。
