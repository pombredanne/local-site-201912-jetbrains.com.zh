JetBrains Rider 2019.2帮助

JavaScript Linters

JetBrains Rider集成了最流行的JavaScript代码库，这些代码库可以在不执行代码的情况下检测代码中的问题。安装并启用后，每次打开JavaScript文件时，lint都会自动激活，并在编辑器中报告检测到的错误和警告。

查看问题描述

  • 将鼠标指针悬停在编辑器中突出显示的问题上。JetBrains Rider显示了报告问题的工具提示。

应用快速修复

  • 按Alt+Enter然后从列表中选择相关建议。
