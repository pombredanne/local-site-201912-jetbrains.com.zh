制表符和缩进

使用制表符

  • 如果选中此复选框，则制表符将用于缩进和代码重新格式化。

  • 如果清除该复选框，JetBrains Rider将使用空格而不是选项卡。

智能标签

压痕由两部分组成。一部分来自嵌套代码块，另一部分由对齐确定。

  • 如果选中此复选框，则嵌套产生的零件同时包含制表符和空格（如果需要），而由对齐方式定义的零件仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组适合指定标签大小的空格后，将自动用标签替换，这可能会导致精细对齐中断。

如果选中了“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签尺寸

在此字段中，指定选项卡中可以容纳的空格数。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

连续缩进

在此字段中，指定要在属性值之间插入的空格数（或制表符，如果已选中“ 使用制表符”复选框）。

缩进空白行

如果选中此复选框，JetBrains Rider会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，JetBrains Rider将删除制表符和空白行。

空行

使用此选项卡定义重新格式化后希望JetBrains Rider保留并插入代码中的空白行和位置。结果显示在“ 预览”窗格中。

此选项卡上的设置不会影响第一个项目之前和最后一个项目之后的空白行数。

保留最大空白行

在此区域中，指定重新格式化后要保留的额外空白行数。

最小空白行

在此区域中，配置是否在顶级块和嵌套选择器周围有多余的空行。在每个选项旁边的字段中，指定要保留的最少额外空白行数。

其他

在此选项卡中，指定要在重新格式化时应用的对齐方式，花括号和空格选项。

大括号放置

使用此列表可以指定JetBrains Rider应该在何处放置选择器的左括号。可用的选项有：

  • 在行尾
  • 下一行

对齐值

使用此列表可以指定JetBrains Rider如何对齐属性和值。可用的选项有：

  • 不对齐 ：选择此选项可在属性名称的第一个字符上指定对齐方式。

  • 关于值 ：选择此选项可指定属性值的第一个字符的对齐方式。

  • 在冒号上

将右括号与属性对齐

如果选中此复选框，则选择器的右大括号将放置在属性列表下。

如果未选中此复选框，则选择器的右大括号将放置在选择器下方。

保留单行块

如果选中此复选框，则具有单个属性的块将被限制为一行。

如果未选中此复选框，则每个属性将放置在其自己的行中。

空间

选中此区域中的复选框，以在冒号分隔键和值之后以及选择器的左括号之前添加一个空格。

十六进制颜色

使用此区域配置十六进制颜色语法。您可以从以下检查选项中进行选择：

  • 将十六进制颜色转换为 ：选中此复选框以配置十六进制颜色字母的大小写。您可以选择小写或大写 。

  • 将十六进制颜色格式转换为 ：选中此复选框以配置十六进制颜色格式的长度。您可以选择长格式或短格式 。

在“ 预览”窗格中查看更改。

安排

在此选项卡中，启用CSS属性排序（默认情况下已关闭），然后选择排序顺序。

  • 要启用排序，请选中“ 排序CSS属性”复选框。

  • 选择按名称可将每个块中的所有CSS属性按字母顺序重新排序。JetBrains Rider会忽略特定于供应商的前缀，但会为按字母顺序排序的某些CSS属性保留多个特定于供应商的前缀。

    边框：1px实线； -moz-border-radius：4px; -webkit-border-radius：4px; border-radius：4px;颜色：黑色；

  • 要定义自己的排序策略，请选择“ 自定义顺序”选项，然后在其旁边的文本框中指定所需的排序顺序。

    使用逗号和空格作为分隔符，并跳过特定于供应商的属性，因为JetBrains Rider总是将它们与对应的无前缀的分组。在此类组中，属性按供应商前缀排序，无前缀的属性排在最后。

    自定义订单列表中欢迎使用正则表达式，例如， .*匹配任何属性。

    JetBrains Rider会记住修改后的“ 自定义订单”列表，并在您关闭“自定义排序”然后再次将其打开时将其还原。

对文件中的所有CSS属性进行排序

  • 在主菜单中，选择代码|重新排列代码 。

  • 或者，选择代码|从主菜单显示“重新格式化文件对话框 ”，然后在打开的对话框中选择“ 整个文件”和“ 重新排列代码 ”。

在块内对CSS属性进行排序

  • 选择要排序的块，然后选择代码|从主菜单重新排列代码 。

  • 或者，选择代码|在主菜单上显示“重新格式化文件对话框 ”，然后在打开的对话框中选择“ 重新排列代码 ”。
