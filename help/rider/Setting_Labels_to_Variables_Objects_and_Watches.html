跟踪对象（使对象ID）

您可以使调试器跟踪特定对象，即使它们不在当前上下文中也是如此。

当程序被调试器挂起时，可以使用Make Object ID为当前上下文中的任何对象分配标签。之后，将固定对象，直到该对象被垃圾回收，或者使用Remove Object ID删除标签，或者直到完成调试器会话为止。

如果变量具有标记的对象作为值，则其标签将显示为美元符号$在“调试”窗口视图中。

现在，当标记的对象出现在相同或其他上下文中时，保存该对象的变量将显示为带有相应的标签。当可能有多个相似的对象并且您想区分它们或跟踪特定的对象时，这非常方便。

在调试器视图中区分对象的另一种方法是自定义对象表示 。

除非已标记的对象被垃圾收集，否则即使使用“ 评估”对话框也可以访问它们，即使它们不在当前上下文中Shift+F9 —只要确保程序已挂起并使用带有美元符号的对象标签，就好像它是本地变量名称一样：

分配标签

  1. 在“调试”窗口中选择所需的变量。

  2. 按Ctrl+K, K或从上下文菜单中选择“生成对象ID ”。“ 选择对象标签”对话框打开。

  3. 指定标签名称。使用不带空格的字母数字字符。或者，单击浏览 预览旁边以更改标签颜色。

  4. 单击确定分配标签。

移除标签

  1. 在“调试”窗口中找到保存标记对象的变量。

  2. 按Ctrl+K, K或右键单击该变量， 然后从上下文菜单中选择“ 删除对象IDt ”。
