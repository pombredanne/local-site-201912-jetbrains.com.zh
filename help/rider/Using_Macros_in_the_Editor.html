巨集

宏提供了一种方便的方法来自动执行您在编写代码时经常执行的重复过程。您可以记录，编辑和播放宏，为其分配快捷方式并共享它们。

  • 宏可用于文件中与编辑器有关的操作。

  • 您不能记录按钮的点击，导航到弹出窗口以及访问工具窗口或菜单。

可以使用没有名称的临时宏。对于永久宏，请分配唯一的名称。

录制宏

  1. 打开“ 编辑”菜单，指向“ 宏” ，然后单击“ 开始宏录制” 。

  2. 执行想要记录的必要动作。

  3. 打开“ 编辑”菜单，指向“ 宏” ，然后单击“ 停止宏记录” 。

  4. 在“ 输入宏名称”对话框中，指定新宏的名称，然后单击“ 确定” 。如果宏仅用于临时使用，则可以将该名称保留为空白。

播放宏

  • 要播放临时宏，请打开“ 编辑”菜单，指向“ 宏” ，然后单击“ 播放最后一个宏” 。

  • 要播放命名的宏，请打开“ 编辑”菜单，指向“ 宏” ，然后单击所需的宏名称。

编辑宏

  • 打开“ 编辑”菜单，指向“ 宏” ，然后单击“ 编辑宏” 。

将键盘快捷键绑定到宏

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择Keymap 。

  2. 创建一个新的键盘映射或从键盘映射列表中选择一个键盘映射。

  3. 展开“ 宏”节点，然后选择要为其创建键盘快捷键的宏。

  4. 右键单击该宏，然后在上下文菜单中选择“ 添加键盘快捷键 ”。

  5. 在“ 输入键盘快捷键”对话框中，按要用作快捷键的组合键。击键将立即反映在“ 第一击”字段中。（可选）选中“ 第二笔划”复选框并指定第二笔划。按下按键时，如果已经使用了按键，则“ 预览”字段将显示您按下的按键，而“ 冲突”字段将显示警告。

    注意 ：请使用鼠标指针，因为任何按键都将被解释为快捷方式。

  6. 使用鼠标指针单击“ 确定 ”以创建快捷方式并将其与宏绑定。

  7. 应用更改。
