Predefined Surround Templates for HTML-Like

This topic lists all predefined surround templates for HTML-Like in JetBrains Rider 2019.2. For more information about surround templates, see Surround Code Fragments with Templates

Template

Details

link

Wrap the selection in a hyperlink

Scope HTML and similar tags

Body

<a href="http://$END$">$SELECTION$</a>

Parameters

  • SELECTION - The text selected by the user before invoking the template.

  • END - The caret position after the template is applied.
