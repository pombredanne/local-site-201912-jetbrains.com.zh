安装和设置JetBrains Rider

先决条件

  • 车手系统要求
  • 在macOS上使用Rider的前提条件
  • 在Linux上使用Rider的前提条件。
  • 在没有Visual Studio的Windows上使用Rider的先决条件。

下载并安装JetBrains Rider

要安装JetBrains Rider

  1. 为您的操作系统下载JetBrains Rider 。

  2. 请根据您的操作系统执行以下操作：

    • Windows 安装 ：

      1. 运行您下载的.exe文件。

      2. 请按照安装向导中的说明进行操作。

    • macOS 安装 ：

      1. 打开您下载的.dmg软件包，然后将JetBrains Rider拖到Applications文件夹中。

    • Linux 安装 ：
      1. 如果您当前将％prod％-*。tar.gz文件解压缩到另一个文件夹， Download文件夹不支持文件执行：
        tar xfz％prod％-*。tar.gz -C

        根据文件系统层次结构标准（FHS）的推荐安装位置是/ opt 。要将JetBrains Rider安装到此目录中，请输入以下命令：

        须藤tar xfz％prod％-*。tar.gz -C / opt /

      2. 切换到bin子目录：

        cd /％prod％-* / bin
        例如，
        cd / opt /％prod％-* / bin

      3. 从bin子目录运行。

首次运行JetBrains Rider

导入JetBrains Rider设置

首次启动JetBrains Rider或从先前版本升级JetBrains Rider后，“ 完整安装”对话框将打开，您可以在其中选择是否要导入IDE设置：

您可以稍后使用文件|手动导出和导入设置 。导入设置和文件|在主菜单上的导出设置命令。

如果这是您的JetBrains Rider的第一个实例，请选择“不导入设置”选项。

注册JetBrains Rider

要尝试和评估JetBrains Rider，您可以免费下载并安装其试用版。试用版可使用30天，然后您需要获取并注册许可证。

选择您要如何注册JetBrains Rider：

  • JetBrains帐户 ：如果您有一个JetBrains帐户 ，该帐户可让您访问购买和管理许可证，请选择此选项（请参阅什么是JetBrains帐户？ 了解更多）。

  • 激活码 ：如果您有JetBrains Rider的激活码，请选择此选项，并将其粘贴到文本区域。

  • 许可证服务器 ：选择此选项可通过许可证服务器 Web应用程序注册JetBrains Rider，该应用程序允许您管理浮动许可证并向没有直接Internet访问权限的用户颁发许可证。

您可以稍后通过“ 帮助” |“更改”来更改选择。从主菜单注册 。

可以作为“抢先体验计划”的一部分下载的JetBrains Rider版本不需要任何注册，并附带30天的许可。

自定义您的JetBrains Rider环境

按照“ 自定义JetBrains骑士”向导步骤配置您喜欢的工作环境，或者跳过这些步骤并按相应的按钮保留默认设置。无论如何，您以后都可以更改设置。

  1. 选择用户界面主题。您可以在默认的灯光主题和Darcula主题之间进行选择。

  2. 选择您更喜欢的编辑器配色方案：

    • ReSharper Light-默认的ReSharper方案

    • Visual Studio灯光—类似Visual Studio的方案

    • 默认值—类似于IntelliJ IDEA的方案

  3. 选择键盘映射（快捷方式方案）：

    • Visual Studio键盘映射—类似Visual Studio的方案

    • ReSharper键盘映射—与ReSharper中的“ IntelliJ IDEA”方案相似

    • 车手键盘图—类似于IntelliJ IDEA的方案

    请参阅Windows / Linux和macOS的键盘映射表。

  4. 禁用不必要的默认插件。

    JetBrains Rider附带了各种插件，这些插件提供与不同版本控制系统和应用程序服务器的集成，增加了对各种框架和开发技术的支持等。

    您可以禁用不需要增加JetBrains Rider性能的插件。如有必要，您以后可以在“设置/首选项”对话框中重新启用它们Ctrl+Alt+S在插件下。

    单击每组插件的全部禁用链接以全部禁用它们，或者单击自定义以禁用单个插件。

  5. 获取其他插件。

    从JetBrains Rider插件存储库下载并安装其他插件（未与IDE捆绑在一起）：

在JetBrains Rider中启动项目

完成初始JetBrains Rider配置后，将显示“ 欢迎”屏幕。它允许您：

  • 创建一个新的解决方案或项目
  • 打开现有的解决方案或项目

  • 或从版本控制系统中检出现有项目（从远程存储库克隆）

也可以将现有的项目目录或单独的文件拖到“欢迎”屏幕，然后用JetBrains Rider打开它。

更新JetBrains Rider

  1. 下载最新版本的JetBrains Rider。

  2. 按照下载和安装JetBrains Rider中所述启动安装程序 。

  3. 选择是否要卸载现有的JetBrains Rider版本及其设置，缓存和本地历史记录，然后按照安装向导的说明进行操作。

通过工具箱应用程序管理JetBrains Rider

工具箱应用程序是一个控制面板，可让您从单个访问点管理所有JetBrains开发人员工具，包括JetBrains Rider以及您的项目。它使您可以启动IDE，维护同一工具的不同版本，安装更新并在需要时回滚它们。它还会记住您的JetBrains帐户，并在安装和注册新工具时使用它自动登录。

  1. 下载工具箱应用程序 。

  2. 启动安装文件。

  3. 安装完成后，接受JetBrains隐私策略并登录到您的JetBrains帐户。

在Windows上执行静默安装

无需任何用户界面即可执行静默安装。网络管理员可以使用它在许多计算机上安装JetBrains Rider，并避免打扰其他用户。

要执行静默安装，请使用以下开关运行安装程序：

  • /S ：启用静默安装

  • /D ：指定安装目录的路径

  • /CONFIG ：指定静默配置文件的路径

  • /LOG ：（可选）使用此开关启用在安装过程中记录问题。您需要指定日志文件的路径和名称。

例如：

rider64.exe / S /CONFIG=d:\temp\silent.config /LOG=d:\JetBrains\logs\install.log / D = d：\ IDE \ Rider

静默配置文件

您可以在https://download.jetbrains.com/rider/silent.config下载JetBrains Rider的静默配置文件。

静默配置文件定义用于安装JetBrains Rider的选项。使用默认选项，仅对当前用户执行静默安装mode=user 。如果要为所有用户安装JetBrains Rider，请使用文本编辑器打开静默配置文件，更改安装模式选项的值mode=admin并以管理员身份运行安装程序。

默认的静默配置文件对于每个JetBrains产品都是唯一的。您可以根据需要对其进行修改，以启用或禁用各种安装选项。

静默卸载JetBrains Rider

要以静默方式卸载JetBrains Rider，请使用以下命令运行卸载程序/S切换为管理员。卸载程序位于bin下的安装目录中。

以管理员身份运行cmd （Windows命令提示符），更改为JetBrains Rider安装目录，然后运行以下命令：

bin \ uninstall.exe / S

在Linux上以快速安装软件包安装JetBrains Rider

您可以在Linux上将JetBrains Rider作为独立的快照程序包安装。由于快照会自动更新，因此您的JetBrains Rider安装将始终是最新的。

要使用快照，请按照安装指南中的说明在计算机上安装并运行快照服务。

在Ubuntu 16.04 LTS和更高版本上，此服务已预安装。

JetBrains Rider通过两个渠道分发：

  • 稳定通道仅包括稳定版本。要安装JetBrains Rider的最新稳定版本，请运行以下命令：

    的--classic该选项是必需的，因为JetBrains Rider snap需要像传统打包的应用程序一样具有对系统的完全访问权限。

    $ sudo snap install rider --classic

  • 边缘通道包括EAP版本。要安装JetBrains Rider的最新EAP版本，请运行以下命令：

    $ sudo snap install rider --classic --edge

安装快照后，您可以通过运行rider命令。

要列出所有已安装的快照，请运行sudo snap list 。有关更多信息，请参见Snapcraft文档 。
