JetBrains Rider 2019.2 Help

Version control

Use the VCS Operations Popup (Ctrl+Alt+Q or VCS | VCS Operations Popup) to quickly invoke any VCS-related commands. The list of actions in the popup depends on the currently enabled VCS.

The list of VCS Operations popup commands is configurable — you can add or remove them on the Appearance and Behavior | Menus and Toolbars page of JetBrains Rider settings Ctrl+Alt+S.

You can also use the following shortcuts to invoke global version control commands:

Command

Shortcut

VCS Operations Popup... Ctrl+Alt+Q
Show Version Control window Alt+9
Commit... Ctrl+Shift+Alt+K
Update Project Ctrl+Alt+W
Revert Ctrl+Alt+Z
Push... Ctrl+Shift+K
Next Change Ctrl+Shift+Alt+N
Previous Change Ctrl+Shift+Alt+P
