Refactor This

Refactor | Refactor This...
Ctrl+Shift+R

This command is a shortcut to all JetBrains Rider's refactorings available in the current context. You can invoke this command from almost any place:

  • On the declaration or usage of a symbol in the editor.

  • On the selection in the editor.

  • On the current item or selection in the Solution Explorer.

To view and apply contextually available refactorings

  1. Place the caret at a symbol in the editor or select the symbol in the Structure tool window.

  2. Press Ctrl+Shift+R.

  3. The Refactor This list appears.

  4. Choose an item from the list and click it or press Enter.
