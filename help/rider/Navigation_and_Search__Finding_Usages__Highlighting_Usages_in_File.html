突出显示文件中的用法

编辑查找|突出显示文件中的用法
Shift+Alt+F11

如果要在当前文档中查找符号的用法，则突出显示所有用法并在它们之间跳转可能比在“ 查找结果”窗口中分析用法更方便。请注意，读写访问用法以不同的颜色突出显示。

您可以在不同的符号上调用此命令以同时突出显示几个不同符号的用法。高亮显示一直保留到您按下为止Esc 。

如果您在当前文档中有选择，则此命令的作用与“ 查找 ”相同Ctrl+F —它使您可以搜索文档中所选内容的文本出现。

即使您的插入符号位于声明中以浏览符号用法，也可以调用“ 转到声明” 。

突出显示当前文档中符号的用法

  1. 将插入号放置在编辑器中的符号声明或用法处。

  2. 在主菜单中，选择“ 编辑|查找|在文件中突出显示用法或按Shift+Alt+F11 。

  3. JetBrains Rider用蓝色突出显示所有读取用法，并用红色突出显示所有写入用法。

  4. 高亮部分还会在右侧错误条纹上与蓝色和红色标记重复。如果并非所有亮点在当前屏幕上都可见，则检查这些标记可能会有所帮助。
    要在突出显示的项目之间导航，请选择“ 编辑” |“其他”。查找|在菜单中查找下一个/上一个或按F3 / Shift+F3 。
    完成重点检查后，按Esc删除它们。

您还可以突出显示名称空间符号和表达式的用法（请参阅ReSharper Horizons：突出显示什么的用法？ 有关详细信息）。
