观察变量

如果要在当前框架的上下文中评估多个变量或表达式 ，并同时查看所有变量或表达式 ，则可以为其创建监视 。监视的变量和表达式的值在应用程序的每一步都会更新，但仅在应用程序挂起时可见。与评估表达式不同，活动监视将保存为解决方案的一部分。

您还可以跟踪特定的对象，而不是变量的值和表达式的结果 。

打开“监视”窗格

默认情况下，“ 监视”窗格是隐藏的，并且监视在“ 变量”窗格中显示 。

  • 要单独显示“ 手表”窗格并在其中查看配置的手表，请释放 变量窗格工具栏上的切换按钮。默认情况下，按钮被按下。

  • 要隐藏“ 监视”窗格并在“ 变量”窗格中查看监视，请按 监视窗格的工具栏上的切换按钮。

向手表添加项目

  • 在“ 监视”窗格中，按Insert或点击然后输入变量或表达式的名称。

  • 在“ 变量”窗格上，或在“ 评估”对话框的“结果”视图中，右键单击所需的项目，然后选择“ 添加到监视” 。

  • 在“ 变量”窗格中选择所需的项目，然后将其拖到“ 监视”窗格中。

  • 在编辑器中选择一个变量或表达式，然后将其拖到“ 监视”窗格中，或右键单击它并选择“ 添加到监视” 。

卸下手表

  1. 在“ 手表”窗格中，选择要删除的手表。

  2. 按Ctrl+R, D或从上下文菜单中选择“ 删除监视 ”。
