价值追踪

值跟踪可帮助您调查特定类型的值，并可能确定某个不正确的值可能已传递到程序中的给定点的方式，以及下一步可能传递给它的位置。这可以帮助您调查可能NullReferenceException ，不当行为以及获取不正确值的原因。

在“ 检查结果”窗口中双击任何数据流元素，以在编辑器中导航至该元素。

调查会员的价值来源

  1. 将插入号放在要调查其值的方法，属性，变量，字段等的名称下。

  2. 按Ctrl+Shift+Alt+A然后在“ 检查此”列表中选择“ 值来源 ”。

  3. 在打开的“ 检查结果”窗口中，您可以调查值的可能来源。

调查会员的价值目的地

  1. 将插入号放在要调查其值的方法，属性，变量，字段等的名称下。

  2. 按Ctrl+Shift+Alt+A然后在“ 检查此”列表中选择“ 值目标 ”。

  3. 将打开“ 检查结果”窗口 ，您可以在其中调查该值的可能目的地。

如果节点表示集合的元素，则将其标记为 ;如果存在lambda表达式 ，则将其标记为 。
