JetBrains Rider 2019.2帮助

代码完成（IntelliSense）

为了使您熟悉JetBrains Rider的代码完成功能，以下是一些实现基础知识：

  • 开始输入新标识符后，将显示“ 自动完成 ”建议列表。按时，将显示基本完成的建议列表Ctrl+Space 。

  • 其他两个完成命令， 智能完成 Ctrl+Alt+Space和类名完成 Shift+Alt+Space应用高级算法将更多项目添加到建议列表。

  • 默认情况下，所有JetBrains Rider的完成功能均支持CamelHumps ，即您可以键入化合物名称部分的首字母，并且该名称将出现在建议列表中。

以下是一些其他一般点，它们显示了JetBrains Rider的代码完成功能与Visual Studio的本机IntelliSense有何不同：

  • 键入时，将根据您的输入缩小完成建议列表的范围。

  • 与继承成员相反，强调了类型的直接成员。

  • 当您在列表中选择建议时，JetBrains Rider将显示其摘自XML文档的摘要。

  • 如果适用，Code Completion会建议索引器以及类型成员，并在选择索引器的情况下相应地更正表达式语法。

  • 提供了用于参数和局部变量的自定义图标，以将其与字段区分开。

  • 在现有代码项上使用代码完成功能时，可以通过按插入所选的完成建议Enter或通过按选定的建议替换现有标识符Tab 。您可以通过按Enter，点或其他上下文相关键 ，使用“ 插入选定的建议”来配置此行为， 即使 “ 编辑器” |“ 完成”弹出框中的焦点没有对准复选框， 也可以输入“插入插入建议 ”。一般代码完成设置页面。

  • 声明字段，方法参数或局部变量时，JetBrains Rider会建议一个可能的名称列表，以考虑当前上下文中的其他名称和命名规则 。

在这个部分：

  • 自动完成
  • 基本完成
  • 智能完成
  • 类名完成
  • 生成完成
