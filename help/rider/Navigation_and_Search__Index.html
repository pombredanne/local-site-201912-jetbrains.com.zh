JetBrains Rider 2019.2帮助

导航和搜索

JetBrains Rider使您可以使用声明，成员，方法调用，继承层次结构，错误等，以及诸如操作，设置和工具窗口之类的IDE项，在源代码中找到自己的方式。

导航和搜索的范围不仅限于解决方案的文件：如有必要，您可以在导航或搜索结果中包括引用库中的符号 。

根据您的任务，不同的JetBrains Rider导航和搜索功能集可能会帮助您：

  • 如果需要按名称在解决方案或引用的库中查找任何内容，请研究“ 按名称搜索”部分中的命令。

  • 如果需要转到与当前代码符号相关的任何项目，请研究“ 从符号导航”部分中的命令。

  • JetBrains Rider还提供了通用的“ 导航至”快捷方式，该快捷方式将当前插入符号位置中的所有可用导航选项分组。要了解有关此快捷方式的更多信息，请参见导航到部分。

  • 如果要在当前文件，项目，解决方案或引用的库中查找特定符号的用法，请研究“ 查找用法”部分中的命令。

  • 根据编辑器中当前插入符的位置，JetBrains Rider可以建议许多相关的导航选项。要了解有关这些选项的更多信息，请参见浏览当前上下文 。

此外，JetBrains Rider还提供其他导航方式：

  • 对于在其他地方覆盖和实现的类型和接口，对于实现，覆盖或隐藏其他方法的方法，JetBrains Rider显示装订线图标，这些图标可用于在继承层次结构之间快速导航：

    有关更多信息，请参见转到基本符号和转到派生符号 。

  • 代表当前文件结构的结构窗口保留导航功能。有关更多信息，请参见浏览文件结构 。

  • 帮助您和您的团队伙伴的TODO项目以便捷的方式组织您的工作，并轻松地在源代码中检索需要注意的位置。有关更多信息，请参见TODO列表 。

有关在JetBrains Rider在编辑器中突出显示的代码错误，警告和建议之间进行导航的准则，请参阅导航代码问题 。
