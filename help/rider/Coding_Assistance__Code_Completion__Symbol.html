基本完成

代码完成|基本的
Ctrl+Space

JetBrains Rider的基本完成建议在当前可见性范围内可用的类，方法，变量，关键字和任何其他符号的名称，包括先前导入的扩展方法。完成列表中还提供了许多其他特定于上下文的建议，例如实时模板 ， 代码生成建议 ， 单元测试类别 。
默认情况下，基本完成建议还包括“ 类名完成”中的项目。如果要更改默认行为，请在 编辑器|一般|的代码完成页 JetBrains骑士设置Ctrl+Alt+S 。建议列表类似于“ 自动完成 ”列表。

调用基本完成

  1. 将插入符号放置在您要键入代码的位置。

  2. 按Ctrl+Space 。

  3. 从列表中选择一个建议，然后按Enter或开始键入标识符的首字母或其CamelHumps缩写，以缩小建议列表的范围。

您还可以在现有标识符上调用基本完成。在这种情况下，有两种方法可以应用所选的完成建议；按Enter插入并保留现有项目或按Tab替换现有项目。
