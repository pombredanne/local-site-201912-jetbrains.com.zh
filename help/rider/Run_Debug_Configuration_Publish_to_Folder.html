Run/Debug Configuration: Publish to Folder

Please do not confuse run/debug configurations with build configurations.

Run | Edit Configurations | | Publish to Folder

Use this configuration to publish classic ASP.NET and ASP.NET Core apps to a folder.

Publishing classic ASP.NET apps only works on Windows.
