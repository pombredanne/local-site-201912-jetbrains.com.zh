XPath Search

JetBrains Rider lets you find occurrences of certain XPath expressions in all XML files in a specific scope.

  1. Go to Edit | Find | Find by XPath... on the main menu. The Find by XPath Expression dialog opens.

  2. Type the expression to find and choose the scope for the search. If you specify a custom scope, JetBrains Rider only scans XML files inside source folders.
