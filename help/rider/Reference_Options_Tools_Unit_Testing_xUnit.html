In this page of JetBrains Rider options, you can adjust unit testing settings related to xUnit.net tests.

Item

Description

Enable xUnit.net support

By default, JetBrains Rider adds action indicators allowing you to execute xUnit.net tests right from the editor and automatically discovers xUnit.net tests in your solution. Clear this checkbox to disable this behavior.

xUnit.net config file

With the Auto-detect option will find your xUnit.net configuration file in any of standard locations. If your configuration file is in a custom location, select Specific and specify the path to the configuration file. To learn about the format of the configuration files, see configuring xUnit.net with JSON and configuring xUnit.net with XML
