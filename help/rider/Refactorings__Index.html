JetBrains Rider 2019.2帮助

重构

JetBrains Rider提供了一套广泛的自动化解决方案范围的代码重构功能，使您可以重命名，移动和安全删除符号。引入并内联字段，变量或参数，并轻松进行更多转换。

除了重构之外，JetBrains Rider还提供了一些可用于Context Actions 的较小的本地代码转换 。

重构可用于大多数受支持的语言和项目项。要了解有关特定重构的更多信息，请研究以下部分：

  • 主要重构
  • 项目级重构

执行重构

  1. 将插入符号设置为符号，选择要重构的代码片段，或在工具窗口中选择一个项目。

  2. 请执行以下任一操作：

    • 在主菜单中，选择“ 重构” ，然后选择所需的重构。此菜单中可用的重构列表取决于当前上下文。如果JetBrains Rider无法建议对上下文进行任何重构，则将禁用整个菜单。

    • 在主菜单中，选择“ 重构”。重构此 ，或按Ctrl+Shift+R以显示适用的重构列表，然后选择其中之一。您也可以在选择的上下文菜单中选择“ 重构此 ”。

    • 使用分配给特定重构的默认键盘快捷键（ Windows，Linux /（ macOs ）），或将自定义快捷键分配给您喜欢的重构命令。

  3. 如果所选的重构需要用户输入，则重构向导将打开。请注意，向导的对话框不是模态对话框，因此您可以在向导打开时编辑代码。

  4. 如果重构操作将导致代码冲突（例如重复的名称，可见性冲突等），则在应用重构之前，向导将在最后一步显示冲突列表。对于某些冲突，向导还可以建议快速修复。有关更多信息，请参见解决重构中的冲突 。

  5. 重构完成后，它将在任何地方进行的所有更改（包括其他文件）都注册为一个操作。这样您就可以使用“ 撤消”操作Ctrl+Z只需一次击键即可回滚所有这些更改。

在编辑器中修改代码后，可以立即进行某些重构。有关更多信息，请参见就地重构。

您可以使用以下快捷方式来调用重构：
