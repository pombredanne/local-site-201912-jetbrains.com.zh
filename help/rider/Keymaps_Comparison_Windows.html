键盘快捷键：键盘映射比较（Windows）

在所有预定义的键盘映射中，您很可能会选择以下之一：

  • Visual Studio / Visual Studio（macOS） -建议以下用户使用此键盘映射：

    • 没有其他IDE的经验；

    • 有使用普通Visual Studio的经验；

    • 具有使用Visual Studio + ReSharper的经验，并且在ReSharper中更喜欢“ Visual Studio”键盘方案 。

  • ReSharper / ReSharper（macOS） -此键盘映射建议给具有Visual Studio + ReSharper经验并且在ReSharper中偏爱“ ReSharper 2.x / IntelliJ IDEA”键盘方案的用户使用。

  • 默认（IntelliJ IDEA） / Mac OS X 10.5+（IntelliJ IDEA） —对于具有IntelliJ IDEA或基于JetBrains IntelliJ平台的IDE的经验，并且更喜欢使用'Default'/'的用户，建议使用此键盘映射Mac OS X 10.5+”键盘映射。

要在键盘映射之间切换，请选择“ 文件” |“键盘” 。设置| Windows和Linux或JetBrains Rider上的键盘映射 |首选项|在MacOS 键映射选项，在按键对应选择所需的键映射，然后单击保存 。
