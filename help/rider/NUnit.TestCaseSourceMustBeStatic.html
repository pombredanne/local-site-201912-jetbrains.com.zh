代码检查：NUnit。测试用例源必须是静态的

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

从NUnit 3.x开始，代表源名称的TestCaseSource或ValueSource属性参数必须引用静态字段，属性或方法。例如：

公共类MyTestClass {[TestCaseSource（“ DivideCases”）] public void DivideTest（int n，int d，int q）{断言。AreEqual（q，n / d）; }静态object [] DivideCases = {new object [] {12，3，4}，new object [] {12，2，6}，new object [] {12，4，3}}; }
