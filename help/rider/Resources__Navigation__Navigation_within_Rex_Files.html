JetBrains Rider 2019.2帮助

在Resx文件中导航

JetBrains Rider将帮助您导航和搜索这些文件中的资源。“ 转到文件成员”命令可在.resx文件中完美运行，并允许您通过几次击键跳到任何资源。

在编辑器中浏览资源

  1. 在编辑器中打开一个.resx文件。

  2. 按Alt+\或选择 导航|从主菜单中选择文件成员 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 您将在弹出窗口中看到资源条目列表。您可以开始键入以缩小列表范围。

  4. 当您在列表中看到所需的字符串时，选择它并按Enter或单击它。

JetBrains Rider的“ 结构”窗口可以显示资源文件的结构。它列出了资源文件中的所有资源条目，可视化了资源ID，当前区域性中的资源文本以及资源注释（如果有）。

从文件结构导航到资源

  1. 在编辑器中打开一个.resx文件。

  2. 按N/A或选择查看| Windows工具|结构 。

  3. 在出现的“ 结构”窗口中，选择所需的字符串，然后双击它以在编辑器中找到它。

  4. （可选）您可以使用“ 文件结构”窗口中提供的导航和重构功能。
