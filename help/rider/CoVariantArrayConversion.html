代码检查：协变量数组转换

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

C＃允许您将派生类型的数组引用为基本类型的数组。正如在埃里克·利珀特（Eric Lippert）的帖子中所解释的那样，这种设计有其优点和问题。说到问题，让我们看下面的例子。当我们通过string[] myStrings到ChangeFirstItem(object[] myArray) ，我们仍然有一个字符串数组，而不是对象。因此，当我们尝试将int值分配给第一个数组元素时，我们得到ArrayTypeMismatchException在运行时。JetBrains Rider在此处检测到问题并发出警告。

Test（）{string [] myStrings = {“一个”，“两个”，“三个”}; ChangeFirstItem（myStrings）; //协变量数组转换会导致运行时异常...安慰。WriteLine（myStrings [0]）; } void ChangeFirstItem（object [] myArray）{myArray [0] = 10; // ArrayTypeMismatchException在运行时}

有几种解决此类问题的方法。JetBrains Rider建议使用快速修复程序 ，将初始数组的类型更改为基本类型的数组，或者将接受参数的类型更改为派生类型的数组。

除了这两个修复程序，您还可以利用ToArray()方法，即： myStrings.ToArray
