查找结果窗口

查看|工具窗口|找
Alt+3

此窗口用于显示各种搜索结果 （例如， 符号的用法 ， 公开 符号的 API等 ）。它在不同的选项卡中显示最新搜索的结果以及以前的搜索结果。每个新搜索命令的结果均显示在单独的选项卡中，但同一搜索/导航命令除外。

您可以通过双击这些结果快速打开包含特定搜索结果的文档。

工具栏控件

根据搜索结果的类型，工具栏控件集可能会有所不同。

控制

名称

描述

重新运行
Ctrl+Shift+F5

重复上一次搜索。

固定标签

单击此按钮固定或取消固定当前选项卡。您可能需要固定标签，以防止在此窗口中达到最大标签数时自动关闭标签。

全部展开 / 全部折叠
Ctrl+NumPad Plus / Ctrl+NumPad Minus

展开/折叠当前选项卡中的所有节点。

上一页 / 下一页
Ctrl+Alt+Up / Ctrl+Alt+Down

导航到上一个/下一个项目，并在源代码中滚动。

出口

点击这个按钮以文本格式导出当前在窗口中显示的数据。

自动滚动到源

按下此按钮后，将根据在窗口中选择的项目滚动编辑器中的源代码。

在同一行上合并发生
Ctrl+F

选择此选项后，同一行上的出现将显示为单个项目。

仅显示阅读用法 Ctrl+R

选择此选项时，仅显示读取用法。可用于字段，属性和局部变量的用法。

仅显示写入用法 Ctrl+W

选择此选项时，仅显示写用法。可用于字段，属性和局部变量的用法。

按字母顺序对成员排序

默认情况下，成员按声明顺序排序。您可以单击此按钮以按字母顺序显示成员。

标题栏上下文菜单和按钮

您可以右键单击窗口标题栏，然后使用上下文菜单配置其查看模式 ，将窗口与其他工具窗口栏关联，或者调整大小并隐藏窗口。
