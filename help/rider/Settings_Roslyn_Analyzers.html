Use this settings page to configure .NET compiler platform (Roslyn) analyzers.

Item

Description

Enable Roslyn analyzers

Once enabled and settings are saved, Rider will scan for installed Roslyn analyzers. All code inspections from discovered analysers will be displayed on the settings page and you will be able to change severity levels in the same way as for JetBrains Rider native inspections.

Include Roslyn analyzers in Solution-Wide analysis

This option lets you monitor issues discovered by Roslyn analyzers solution-wide with the solution-wide analysis. For this option to work, you also need to check Enable solution-wide analysis on the Editor | Inspection Settings settings page.
