从全局装配体缓存中打开装配体

JetBrains Rider可以从全局程序集缓存 （GAC）中打开程序集。

从全局程序集缓存中打开程序集

  1. 单击从GAC打开 在Assembly Explorer视图的工具栏上

  2. 在出现的对话框中，选择所需的装配。 您可以通过在“ 搜索”字段中键入部件名称的一部分来批量选择部件项目并过滤出部件。

    使用CamelHumps-程序集名称不同部分开头的大写字母。例如，查找名称包含以下内容的所有程序集Microsoft.VisualStudio.Modeling在GAC程序集列表中，您可以输入mvsm 。

显示其名称和版本的所选装配将添加到“ 装配浏览器”视图中 。您可以通过扩展部件节点及其子节点来探索它们。双击代码符号可反编译并查看相应的代码。

如果您想随时清除反编译缓存，则可以删除％USERPROFILE％\中的数据。Rider2019.2 \ config \ resharper-host \ DecompilerCache 。
