代码检查：不可访问的字段（私有可访问性）

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

C＃编译器会针对未使用的字段发出许多警告：

  • CS0649：字段从未分配给它，并且将始终具有其默认值
  • CS0169：从不使用私有字段
  • CS0414：为私有字段分配了一个常量值，但从未使用过

尽管JetBrains Rider知道所有这些警告并为它们提供设计时通知 ，但它又前进了一步，并检测出编译器警告未涵盖的未使用字段。下面的示例显示，即使为字段分配了非恒定值且没有读取用法，JetBrains Rider也会帮助您检测未使用的成员。

类Foo {private int _a; // CS0414 private int _b; //没有编译器警告，但是JetBrains Rider在此处发出自己的警告public Foo（int value）{_a = 1; _b =值； }}
