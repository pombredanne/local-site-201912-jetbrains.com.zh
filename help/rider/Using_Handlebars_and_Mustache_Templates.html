JetBrains Rider 2019.2帮助

把手和胡子

使用JetBrains Rider，您可以在hbs或mustache文件中使用Handlebars表达式和Mustache模板 。JetBrains Rider根据“设置/首选项”对话框的“ 模板”页面上指定的默认或自定义设置来处理此类文件的内容。

在你开始前

  • 如“ 管理插件”中所述，在“ 插件”页面上安装并启用Handlebars / Mustache存储库插件。

在JetBrains Rider中配置车把和胡须

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |模板 。

  2. 在打开的“ 模板”页面上，切换到“ 把手/胡子”区域。

  3. 自动插入第二个右花括号}在输入第一个结束符时选择“ Handlebars”表达式 ，请选中“ 自动插入结束符”复选框。

    JetBrains骑士还发现三重藏匿处 {{{防止转义表达式内的值。在这种情况下，JetBrains Rider会在您键入第一个闭合大括号时自动插入两个闭合大括号。

  4. 在代码生成，重构或按下时自动重新格式化Handlebars表达式和Mustache模板 Ctrl+Alt+Enter ，请选中启用格式复选框。否则，JetBrains Rider始终保持其原始格式。

  5. 选择用于使用样式进行注释的语言。当您按以下内容输入行或注释时Ctrl+Alt+/要么Ctrl+Shift+/ ，JetBrains Rider插入用于所选语言的注释定界符，例如：

    • {{!----}}用于车把

    • /**/用于JavaScript

    • 用于HTML

  6. 要配置“车把/胡须”语法突出显示， 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |配色方案|车把/小胡子 。选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
