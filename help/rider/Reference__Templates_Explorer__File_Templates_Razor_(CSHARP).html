Razor MVC View with Layout

带有布局的Razor MVC视图

范围 Razor（C＃）项目，Razor（C＃）视图文件夹

身体

@model $ model $ @ {ViewBag。标题=“ $ title $”;布局=“ $ _Layout $”; }

$ title $

$ END $

参量

  • model -预计此时的猜测类型

  • title -没有宏

  • _Layout -在评估变量的位置显示智能代码完成列表

  • END -应用模板后的插入符位置。

Controller

范围 Razor ASP.NET Framework项目

身体

$ HEADER $正在使用系统。网络。Mvc;名称空间$ NAMESPACE $ {公共类$ CLASS $：控制器{// GET public ActionResult Index（）{$ END $ return View（）; }}}

参量

  • HEADER -插入在JetBrains Rider设置中指定的文件头。

  • CLASS -评估当前文件名，不带扩展名，所有非字母数字均用下划线替换

  • NAMESPACE -获取当前文件的默认名称空间

  • END -应用模板后的插入符位置。
