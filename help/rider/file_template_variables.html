文件模板变量

此页面描述了特定于Web的模板，这些模板可在HTML，JavaScript，TypeScript等中使用。.NET专用模板中描述了.NET语言的模板 。

文件模板可以包含变量，这些变量在应用模板时将被其值替换。变量是一个以美元符号开头的字符串$然后是变量名。变量名称可以选择用大括号括起来。例如： $MyVariable和${MyVariable}是同一变量的不同符号。

预定义模板变量

以下预定义变量可以在文件模板中使用 ：

变量

描述

${DATE}

当前系统日期

${DAY}

该月的当前日期

${DS}

美元符号$ 。此变量用于转义美元字符，因此不会将其视为模板变量的前缀。

${HOUR}

当前时间

${MINUTE}

当前分钟

${MONTH}

这个月

${MONTH_NAME_FULL}

当月的全名（一月，二月等）

${MONTH_NAME_SHORT}

当前月份名称的前三个字母（一月，二月，依此类推）

${NAME}

新实体的名称（文件，类，接口等）

${ORGANIZATION_NAME}

在项目设置中指定的组织名称（ N/A ）

${PRODUCT_NAME}

IDE的名称（例如，JetBrains Rider）

${PROJECT_NAME}

当前项目名称

${TIME}

当前系统时间

${USER}

当前用户的登录名

${YEAR}

今年

自定义模板变量

除了预定义的模板变量外，还可以指定自定义变量。如有必要，您可以使用以下命令在模板中定义自定义变量的值#set指示。

例如，如果要使用全名而不是通过预定义变量定义的登录名${USER} ，请使用以下结构：

#set（$ MyName =“ John Smith”）

如果模板中未定义变量的值，则在应用模板时，JetBrains Rider会要求您指定它。
