将类型移到外部作用域重构

重构|移动...
Ctrl+R, O

此重构可帮助您将嵌套类型上移一级。即使嵌套类型使用父类型的成员，所有对应的引用也会自动更新。

将类型移到外部作用域

  1. 通过以下方式之一选择嵌套类型：

    • 在编辑器中，将插入符号设置为嵌套类型的名称。

    • 在解决方案资源管理器中选择一个嵌套类型。

    • 在“ 结构”窗口中选择一个嵌套类型。

  2. 请执行以下任一操作：

    • 按Ctrl+R, O然后选择移至外部范围

    • 按Ctrl+Shift+R然后选择移至外部范围

    • 选择重构|移至主菜单中的“ 外部范围 ”。

    将打开“ 移至外部范围”对话框。

  3. （可选）您可以为该类型指定其他名称。

  4. 要应用重构，请单击下一步 。

  5. 如果未发现冲突，JetBrains Rider将立即执行重构。否则，它会提示您解决冲突 。

    重构完成后，它将在任何地方进行的所有更改（包括其他文件）都注册为一个操作。这样您就可以使用“ 撤消”操作Ctrl+Z只需一次击键即可回滚所有这些更改。
