JetBrains Rider 2019.2帮助

检查一下

Ctrl+Shift+Alt+A

此命令是JetBrains Rider的代码探索和检查功能的快捷方式：

  • 通话追踪
  • 价值追踪
  • 探索类型层次
  • 查找代码问题

您可以从编辑器，“ 结构”窗口和其他工具窗口中调用此命令。根据上下文，此快捷方式可帮助您访问该上下文中可用的所有代码探索功能。

查看上下文可用的代码探索功能

  1. 将插入号放置在编辑器中的符号上，或在“ 结构”工具窗口中选择符号。

  2. 按Ctrl+Shift+Alt+A

  3. 将显示“ 检查此列表”。

  4. 从列表中选择一个项目，然后单击它或按Enter 。
