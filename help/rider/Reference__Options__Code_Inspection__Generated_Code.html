使用基于图层的设置机制可以保存在此页面上可配置的首选项。

档案|设置|编辑器检查设置|为Windows和Linux 生成的代码

Ctrl+Alt+S

在JetBrains Rider设置的此页面上，您可以指定包含自动生成的代码的文件和代码区域。 对于这些项目，JetBrains Rider仅运行那些检查代码的编译器错误和警告的代码检查 。默认情况下，此列表中包括Visual Studio用于生成代码的某些文件掩码和区域，例如* .designer.cs ，但是您可以根据需要更改默认设置。

包含生成的代码的项目列表也可以用于禁用代码样式和清除生成的代码。
