JetBrains Rider 2019.2帮助

密码管理器数据库已更新

当您启动具有先前JetBrains Rider版本配置的新JetBrains Rider版本时，此对话框仅打开一次。

JetBrains Rider使您可以将当前保存的密码数据库转换为新数据库。

项目

描述

密码

使用此字段输入在先前版本的JetBrains Rider中使用的主密码。

兑换

单击此按钮将保存的密码存储在新的密码数据库中。

清除密码

如果单击此按钮，将不会进行任何转换，并且不会保存您的旧密码。如果关闭此对话框而不输入主密码，将执行相同的操作。
