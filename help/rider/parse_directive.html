模板中的可重用内容

此页面描述了特定于Web的模板，这些模板可在HTML，JavaScript，TypeScript等中使用。.NET专用模板中描述了.NET语言的模板 。

包含模板用于定义可重用的代码段（例如，标准标头，版权声明等），以便通过#parse指令。

的#parse指令具有以下语法：

#parse（“ ”）

例如：... #parse("JavaScript File Header.js") 。

可以以这种方式在其他模板可以参照的模板上列出了包括标签 编辑器|文件模板| JetBrains Rider设置的“ 其他语言”页面Ctrl+Alt+S 。

创建包含模板

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |文件模板|其他语言 。

  2. 打开包括选项卡。

  3. 单击创建模板 在工具栏上，并指定包含模板名称，扩展名和正文。

要将可重复使用的模板插入另一个模板

  • 添加#parse指令，其中包含要插入的包含模板的名称。有关示例，请参阅模板语法 。
