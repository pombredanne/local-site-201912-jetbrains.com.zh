XML的上下文操作

JetBrains Rider为XML提供以下上下文操作：

名称

描述

添加模块资格

添加或更新类型引用的模块资格。

折叠空标签

从空的开始和结束标签转换为空标签

将属性转换为嵌套元素

将标签属性转换为嵌套元素。

将CData转换为文本

用转义的内容替换连续的CData元素

将标签转换为属性

将嵌套标签转换为封闭标签的属性。

将文本转换为CData

用CData元素包围文本，如有必要，从文本中转义

展开空标签

将空标签拆分为开始标签和结束标签

删除所有具有指定名称的标签并升级其子级

删除所有具有指定名称的标签，但保留其子标签不变

删除标签并提升其子级

删除标签，但保持其子标签完好无损

删除标签属性

从标签中删除属性

替换所有属性

将文件中具有相同名称的所有属性替换为具有不同名称的当前标记

替换所有标签

用不同标签替换文件中具有相同名称的所有标签

更换标签

用其他标签替换标签
