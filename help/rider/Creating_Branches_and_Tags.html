创建分支和标签

JetBrains Rider允许您根据本地工作副本或其存储库版本创建分支或标签。

要在Subversion存储库中创建分支或标记，请执行以下操作：

  1. 从主菜单中，选择VCS |。颠覆|分支或标签 。或者，在“ SVN存储库”工具窗口中选择源文件夹，然后从上下文菜单中选择“ 分支”或“标记”命令。

  2. 在打开的“ 创建分支或标签”对话框的“ 复制自”部分中，指定将复制到分支或标签的源文件夹。您可以使用本地工作副本或存储库位置。

    如果选择存储库位置作为源：

    • 指定修订作为新分支的基础。这可以是HEAD版本，也可以是具有指定编号的版本。如果选择了“ 指定的”选项，请键入修订号，或单击并在Subversion更改浏览器中找到所需的修订。

  3. 在“ 复制到”部分中，指定将在其中创建分支或标记的目标。如果使用基本URL，请指定新分支或标记的名称。如果您选择在另一个存储库位置创建分支或标记，请键入其URL，或单击然后从“ 选择存储库位置”对话框中选择目标。

  4. （可选）输入评论，然后单击“ 确定” 。
