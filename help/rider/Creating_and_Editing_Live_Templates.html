创建实时模板

此页面描述了特定于Web的模板，这些模板可在HTML，JavaScript，TypeScript等中使用。.NET专用模板中描述了.NET语言的模板 。

以下示例过程说明了如何为广告模板创建模板TODO用当前日期和用户名注释：

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |实时模板|其他语言 。

  2. 选择要在其中创建新实时模板的模板组（例如， other ）。如果未选择模板组，则将实时模板添加到用户组。

  3. 点击添加 ，然后选择实时模板 。

  4. 指定以下值：

    领域

    描述

    示例值

    缩写

    指定字符以扩展模板

    		 todo

    描述

    指定可选描述以标识模板的用途

    		 Insert TODO comment with the current date and username

    范本文字

    用变量指定模板的主体

    		//TODO $DATE$ $USER$: $END$

  5. 单击底部的“ 定义 ”以选择适用的上下文（例如，如果JavaScript应应用于.NET语言源文件，则为JavaScript ）。

  6. 点击编辑变量，然后选择date()功能$DATE$和user()对于$USER$ 。如果同时为两个变量定义 ，还选择“ 跳过” ，然后单击“ 确定” 。

  7. 单击确定以应用更改。

根据当前系统日期和用户名，新创建的todo模板将扩展为类似于以下内容的行：

// TODO 2019-07-02 jsmith：

从代码片段创建新模板

  1. 在编辑器中，选择文本片段以从中创建实时模板。

  2. 选择工具|从主菜单另存为实时模板 。实时模板列表将打开。在此列表中，新创建的模板已添加到用户组。

  3. 指定模板的缩写，可选描述（以标识模板的用途）并修改模板主体。如果模板已定义变量 ，请单击“ 编辑变量”进行配置。

  4. 单击确定以应用更改。

复制现有模板

如果要在多个组中重复使用同一模板，或者要基于另一个模板创建新模板，则可以复制现有模板。

  1. 在编辑器上实时模板页面 设置/首选项对话框（ Ctrl+Alt+S ），选择要复制的模板。

  2. 单击重复 在工具栏上。一个新的模板项将被添加到与原始模板相同的组中，并被选中。

  3. 指定模板的新缩写，可选描述（以标识模板的用途），并在必要时修改模板主体。如果模板已定义变量 ，请单击“ 编辑变量”进行配置。

  4. 单击确定以应用更改。
