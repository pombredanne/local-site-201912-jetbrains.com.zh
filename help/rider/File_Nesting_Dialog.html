文件嵌套对话框

单击文件嵌套时，此对话框打开在解决方案资源管理器的工具栏上。

JetBrains Rider提供了文件类型和用于相关文件类型的典型后缀的默认配置，例如，后缀为.map.ts的文件通常与具有相同名称和扩展名.ts的文件相关。将在解决方案资源管理器中自动分组。
您可以通过单击以下选项来配置这些关系或完全禁用逻辑嵌套在解决方案资源管理器的工具栏上。

以这种方式对文件进行分组不会对您的项目/解决方案配置文件进行任何更改。

请注意，嵌套规则仅适用于同一目录中具有相同名称的文件。

如果两个文件的名称匹配一个模式，但是文件存储在不同的目录中，JetBrains Rider不会在视觉上“移动”它们中的任何一个。
