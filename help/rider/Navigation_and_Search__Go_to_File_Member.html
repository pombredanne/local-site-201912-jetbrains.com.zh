转到文件成员

导航|文件成员
Alt+\

您可以使用此命令快速导航到当前文档中的特定方法，字段，属性（可以是源文件或通过反编译重建的文件）。

导航到文件成员

  1. 按Alt+\或选择 导航|从主菜单中选择文件成员 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  2. 在出现的输入成员名称弹出窗口中，开始输入项目名称。键入时，查找列表将变窄，建议其名称与输入的子字符串匹配的成员。在匹配的成员下方，将出现匹配的文本出现。

  3. （可选）选择“ 包括基本成员”或按Alt+\如果要包括基类的成员，该类其他部分（如果该类是部分类）的成员或其他引用文件的成员。

  4. 请执行以下任一操作：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找窗口中打开 ， 按Shift+Enter要么+在数字键盘上。

如果只想搜索构造函数，请输入new要么ctor （带有尾随空格）在搜索框中作为过滤器。要仅显示具有特定访问修饰符的成员，请键入此修饰符。例如，键入private向私人成员展示

此处支持不区分大小写的CamelHumps ：例如，如果有成员FindAction在文件中，您可以通过在搜索框中键入“ fa”或“ FA”来找到它。
