Visual Basic .NET中的导航和搜索

C＃中所有JetBrains Rider的导航和搜索功能均可用在Visual Basic .NET中也受支持。您可以在“ 导航和搜索”部分的相应主题中找到有关这些功能的详细信息。

在本主题中，您可以找到一些在Visual Basic .NET中使用导航和搜索功能的示例。

档案结构

文件结构有助于概述在编辑器中打开并处于活动状态的当前文件的结构。

在“ 文件结构”窗口中双击一个节点，以导航到所选成员的声明。此外，从上下文菜单中，您可以找到用法并应用重构。

按名称导航

各种“转到...”功能，例如： 转到类 ， 转到符号 ， 转到文件成员 ，等等对于VB.NET也适用。

“随处搜索”使您可以导航到类型，符号，文件，最新文件以及运行配置，操作，工具窗口和首选项。

查找用法

查找用法可帮助您在源代码中找到名称空间，类型，方法等的所有用法。应用“ 查找用法”功能是在解决方案中查找所有代码用法的最快方法。

突出显示文件中的用法

“ 突出显示文件中的用法”功能可帮助您将注意力集中在特定的成员或局部变量及其出现上。请注意，写访问权限以红色突出显示，而读访问权限以蓝色突出显示。

查找依赖于模块的代码

查找依赖于模块的代码有助于查找对特定项目或程序集引用的依赖。
