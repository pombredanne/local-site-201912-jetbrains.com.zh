大括号布局

类型和名称空间声明

属性名称：

[resharper_]csharp_type_declaration_braces ， [resharper_]csharp_brace_style ， [resharper_]type_declaration_braces ，[resharper_]brace_style

可能的值：

  • end_of_line ：行尾（K＆R样式）

  • end_of_line_no_space ：行尾（无空格）

  • next_line ：在下一行（BSD样式）

  • next_line_shifted ：在下一行缩进（铁匠风格）

  • next_line_shifted_2 ：在下一行缩进2（GNU样式）

例子：

行结束

命名空间N {接口I {void foo（）; } C类{}}

end_of_line_no_space

命名空间N {接口I {void foo（）; } C类{}}

next_line

命名空间N {接口I {void foo（）; } C类{}}

next_line_shifted

命名空间N {接口I {void foo（）; } C类{}}

next_line_shifted_2

命名空间N {接口I {void foo（）; } C类{}}
