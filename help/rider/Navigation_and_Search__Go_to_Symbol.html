JetBrains Rider 2019.2帮助

转到符号

导航|符号...
Ctrl+Shift+Alt+T

此命令允许按名称搜索其中的任何代码符号整个解决方案中的符号和文件，以及解决方案项目中引用的所有程序集。

请注意，此命令在全局范围内有效，也就是说，您无需将焦点放在编辑器上即可调用它。

导航到符号

  1. 按Ctrl+Shift+Alt+T或选择 导航|符号...从主菜单。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  2. 在出现的输入符号名称弹出窗口中，开始输入项目名称。键入时，查找列表将变窄，并建议与输入的子字符串匹配的名称。

  3. （可选）选择“ 包括非解决方案符号”或按Ctrl+Shift+Alt+T显示解决方案中引用的库中的匹配项。
    默认情况下，只要您的输入与解决方案中的某些内容匹配，就只显示解决方案项。如果解决方案中输入的内容不匹配，JetBrains Rider会自动开始寻找匹配的库类型。

  4. 请执行以下任一操作：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找窗口中打开 ， 按Shift+Enter要么+在数字键盘上。

每当收集项目以匹配输入的字符串花费大量时间时，JetBrains Rider都会显示一个沙漏图标在搜索字符串的右边。在JetBrains Rider完全加载解决方案之前执行搜索时，预计会有很大的延迟。
