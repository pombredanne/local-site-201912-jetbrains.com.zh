导航|类...

此命令使您可以导航到以下任何类型（类，接口，结构或枚举） 整个解决方案中的符号和文件，以及解决方案项目中引用的所有程序集。

请注意，此命令在全局范围内有效，也就是说，您无需将焦点放在编辑器上即可调用它。

导航到类型

  1. 选择 导航|类...在主菜单中。

  2. 在出现的“ 输入班级名称”列表中，开始输入项目名称。键入时，项目列表会变窄，并建议与输入的子字符串匹配的名称。

  3. （可选）选择“ 包括非解决方案类”以显示解决方案中引用的库中的匹配项。
    默认情况下，只要您的输入与解决方案中的某些内容匹配，就只显示解决方案项。如果解决方案中输入的内容不匹配，JetBrains Rider会自动开始寻找匹配的库类型。

  4. 请执行以下任一操作：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找窗口中打开 ， 按Shift+Enter要么+在数字键盘上。

如果您导航到的项目属于当前解决方案，JetBrains Rider将在编辑器中打开相应的文件，并将插入号放置在符号声明中。如果在引用的库中找到它，JetBrains Rider将反编译相应的程序集并在反编译文件中找到该符号。

此处支持不区分大小写的CamelHumps ：例如，您可以键入“ ch”而不是“ ColorHelper”。如果要按全限定名导航到类型，则可以键入名称的一部分，并用空格或点将它们分开。

您也可以在指定名称时使用通配符：“ *”（星号）表示零个或多个字符； '+'（加号），'：'（分号），'。' （点）和'\'（反斜杠）将类型的完全限定名称分开。
