Navigation

ReSharper enables you to quickly navigate to resource declarations and resource usages. It can display search results in the Find Results window and provides convenient navigation between them.

All navigation actions can be divided into four groups:

  • Navigation within Resx Files
  • Navigation between Code and Resx Files
  • Navigation between Cultures
  • Find Usages
