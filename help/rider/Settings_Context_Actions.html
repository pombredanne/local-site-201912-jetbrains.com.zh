JetBrains Rider 2019.2 Help

Context Actions

Rider uses both layer-based and directory-based settings mechanisms to save preferences configurable on this page.
Preferences that are saved using the directory-based mechanism are marked with the warning icon .

File | Settings | Editor | Context Actions for Windows and Linux
JetBrains Rider | Preferences | Editor | Context Actions for macOS
Ctrl+Alt+S

By selecting or clearing checkboxes in this page, you can compile the list of available Context Actions. When you select a specific context action, you can read its description in the right.
