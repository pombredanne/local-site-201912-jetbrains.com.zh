使用此对话框可以定义模式，以帮助跟踪源代码中的TODO项目。确保源代码中的TODO项插入对支持的文件类型有效的注释中。

描述

图案

在此文本框中，键入描述所需的TODO模式的正则表达式 。

图标

从此下拉列表中，选择图案的图标。

区分大小写

选择此复选框以使图案区分大小写。

字体类型

选中相应的复选框可使图标文本以粗体或斜体显示。

前景

选中此复选框以启用调色板并选择前景色。

背景

选中此复选框以启用调色板并选择背景色。

错误条纹标记

选中此复选框以启用调色板并选择错误条纹的颜色。

特效

选中此复选框以启用效果（下划线，删除线等），并从调色板中选择效果的颜色。
