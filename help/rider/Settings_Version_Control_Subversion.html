使用此页面可以指定要应用于Subversion控制的项目目录的设置

项目

描述

启用互动模式

如果希望JetBrains Rider模仿在交互模式下直接从终端执行Subversion命令时的行为，请选择此选项（对话框将弹出，您可以在其中输入凭据）。这是处理svn + ssh存储库的密码/密码提示以及信任https存储库的无效服务器证书所必需的。

使用自定义配置目录

如果您不想将Subversion配置文件存储在系统默认位置，请选择此选项，并指定自定义目录的路径。

介绍

使用此设置页面可以配置数据演示设置。

项目

描述

准备合并时检查目标子树中的svn：mergeinfo

如果希望JetBrains Rider在合并之前检查目标分支的合并跟踪信息以防止重复，请选择此选项。

在注解中回溯的最大修订版本数

选择此选项可限制在计算注释时要回顾的修订版本数，并指定修订版本数。

在历史记录和注释中显示合并源

如果希望合并源在注释和文件历史记录中可见，请选择此选项。

忽略注释中的空白差异

如果要在注释时忽略空格，请选择此选项，从而获得更有意义的注释，并消除无意义的注释。

网络

使用此设置页面可以配置连接设置。

项目

描述

使用JetBrains Rider常规代理设置作为Subversion的默认设置

如果希望Subversion使用默认的JetBrains Rider代理设置，请选择此选项。

HTTP超时

指定等待建立HTTP连接的秒数。

SSH连接超时

指定等待建立SSH连接的秒数。

SSH读取超时

指定等待响应的秒数。

SSL协议

在此区域中，选择要使用的SSL协议。可用的选项有：

  • 所有

  • SSLv3

  • TLSv1

编辑网络选项

单击此按钮可以在与网络层对话框相关的“ 编辑Subversion”选项中更改Subversion运行时配置文件。

SSH协议

使用此设置页面可配置用于通过隧道SSH协议连接到SVN服务器的设置。

项目

描述

SSH可执行文件

指定SSH客户端的路径。输入可执行文件的名称，或单击“ 浏览”按钮然后在打开的对话框中选择路径。如果未指定， 'ssh'默认情况下使用。

仅当选择了“ 密码”或“ 私钥”选项时，此字段才可用。

用户名

指定用于SSH连接的用户名。如果在存储库URL中明确指定了用户名，则将使用此值，并且将忽略此设置。

仅当选择了“ 密码”或“ 私钥”选项时，此字段才可用。

港口

如果您的服务器正在侦听非标准端口（对于svn + ssh：//，则为22），请修改默认值。

仅当选择了“ 密码”或“ 私钥”选项时，此字段才可用。

密码

如果要使用密码进行SSH身份验证，请选择此选项。

私钥

如果要使用私钥进行SSH身份验证，请选择此选项。

路径

指定私钥的路径。手动输入路径，或单击“ 浏览”按钮然后在打开的对话框中选择路径。

颠覆配置

如果要使用Subversion配置中存储的默认设置进行SSH连接，请选择此选项。

SSH隧道

此字段显示存储在Subversion配置中的SSH隧道设置。您可以修改该值，然后单击“ 更新”按钮将此值写入Subversion配置。

更新资料

单击此按钮以检查Subversion配置并在必要时更新该值，或者将您输入的值写入Subversion配置。

SVN_SSH

此字段显示可在隧道配置中使用的环境变量（默认情况下， SVN_SSH ），并存储在Subversion配置中。
