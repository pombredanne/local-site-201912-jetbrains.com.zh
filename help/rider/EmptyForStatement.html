代码检查：空的“ for”循环是多余的

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

如果一个for循环不包含任何指令，并且其初始化，条件和迭代不使用外部作用域中的变量，JetBrains Rider建议删除此循环，例如：

public void MyMethod（）{for（int i = 0; i <100; i ++）{}}

但是请注意， for带有空主体的循环实际上可以用于遍历外部变量，在这种情况下，JetBrains Rider不会将其突出显示为多余，例如：

公共无效MyMethod（）{int a; var b = 0; for（a = 0; a <100; b + = a ++）{}控制台。WriteLine（“ b =” + b）; }
