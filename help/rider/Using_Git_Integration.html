为了能够使用Git进行版本控制，请确保已启用Git集成插件。 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ）在插件下。

在为现有的本地项目启用Git版本控制或从远程存储库克隆Git项目之前 ，请执行以下操作：

  1. 下载并安装Git。受支持的最低版本是Windows的2.4和Linux和MacOS的1.8.0.0。

  2. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择版本控制|在左窗格中Git并指定Git可执行文件的路径。

  3. 设置远程Git存储库的密码

设置Git遥控器的密码

每次您与远程Git存储库进行交互时（例如，在pull ， update或push操作期间），都需要授权。您可以将JetBrains Rider配置为记住您的密码 ，这样就不必在每次需要授权时都指定您的凭据。

配置密码策略

  1. 在“ 设置 首选项”对话框中Ctrl+Alt+S ，选择外观和行为|系统设置|密码在左边。

  2. 选择您希望JetBrains Rider处理Git远程存储库密码的方式：

    • 在本机钥匙串中 ：选择此选项以使用本机钥匙串存储您的密码。此设置仅适用于MacOS和Linux。

    • 在KeePass中 ：选择此选项以使用KeePass密码管理器存储密码。当您使用KeePass密码管理器时 ，主密码将用于访问存储各个密码的文件。一旦JetBrains Rider记住了您的密码，除非您需要访问密码数据库，否则它不会要求您输入密码。在MasterPassword字段中输入将用于访问c.kdbx文件的密码。

      您可以在数据库字段中更改c.kdbx文件的默认位置。

      要导入c.kdbx文件，请单击并从下拉菜单中选择导入 ，或单击并指定包含密码的本地文件的路径。

      如果要从数据库中删除现有密码，请选择清除 。

    • 不要保存，重新启动后忘记密码 ：如果您希望在关闭JetBrains Rider之后重设密码，请选择此选项。
