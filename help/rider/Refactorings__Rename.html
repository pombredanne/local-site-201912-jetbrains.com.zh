重命名重构

重构|改名...
Ctrl+R, R

通过这种重构，您可以更改解决方案中任何符号或项目的名称。该符号的所有引用和用法将自动更新。

对不同的C＃实体使用重构

这是您可以通过此重构重命名的实体列表：

  • 项目
    您还可以重命名项目文件夹和项目的根名称空间，以及其在解决方案中的所有用法。

  • 命名空间
    您还可以更改其名称空间的嵌套级别。例如，您可以重命名名称空间A.B.C至A.D 。

  • 类型
    如果重命名的类型位于具有相同名称的文件中，JetBrains Rider也会重命名包含的文件。
    例如，如果重命名的类型是名称相似的类型层次结构的一部分， class Foo : IFoo ，您将可以选择是否重命名相关类型。

  • 方法
    如果该方法是继承/实现层次结构的一部分，则相关方法也将重命名。
    如果该方法有重载，则可以选择是否重命名重载。

  • 领域
    如果在具有匹配名称的属性中使用了该字段，则可以选择是否重命名该属性。

  • 属性
    如果该属性的后备字段具有匹配的名称，则您也可以重命名该字段。

  • 参数
    如果该方法是继承/实现层次结构的一部分，或者重载了具有相同名称和类型的参数，则您也可以在相关方法中重命名参数。

  • 局部变量
    JetBrains Rider重命名局部变量而不显示对话框：

  • 事件
  • 代表

用命令调用重构

重命名实体并更新所有引用和用法

  1. 通过以下方式之一选择实体：

    • 在编辑器中，将插入符号设置为实体名称。

    • 在解决方案资源管理器中选择一个实体。

    • 在“ 结构”窗口中选择一个实体。

  2. 请执行以下任一操作：

    • 按Ctrl+R, R 。

    • 按Ctrl+Shift+R然后选择重命名

    • 选择重构|在主菜单中重命名 。

    将打开“ 重命名”对话框。

  3. 输入实体的新名称或使用建议的名称之一。

    考虑到您的命名风格， JetBrains Rider会为实体建议新的名称。

  4. 根据您重命名的实体，选择应该重命名的相关项目，例如，在注释和字符串文字中出现的实体。

  5. 单击下一步 。如果没有与重命名实体相关的内容，JetBrains Rider将应用重构。否则，向导将显示其他步骤：

    • 如果存在具有相似名称的相关实体，则会在以下步骤中列出它们：

      选择要与所选实体一起重命名的相关实体-您可以接受建议的名称或在“ 新名称”列中指定新名称。

    • 如果您以前选择了“在注释中搜索”和“字符串文字”，并且在以下步骤中找到了实体名称，则将在以下步骤中列出它们：选择要重命名JetBrains Rider的文本形式。

  6. 如果未发现冲突，JetBrains Rider将立即执行重构。否则，它会提示您解决冲突 。

    重构完成后，它将在任何地方进行的所有更改（包括其他文件）都注册为一个操作。这样您就可以使用“ 撤消”操作Ctrl+Z只需一次击键即可回滚所有这些更改。

就地执行重构

您可以通过在编辑器中修改实体的声明来重命名实体，然后应用快速修复来调用解决方案范围的重构。

一旦在实体声明中更改了实体名称，该名称周围就会出现一个灰色边框，通知您重构可用。你可以按Alt+Enter在操作列表中找到重构：

您还可以在其他地方应用重构。
