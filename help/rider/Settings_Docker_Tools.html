为了使此页面可用，必须安装并启用Docker集成插件。

指定使用Docker Machine （ Docker Toolbox ）和Docker Compose的设置 。

项目

描述

Docker Machine可执行文件

 docker-machine或docker-machine.exe的实际路径（通常位于Docker Toolbox安装文件夹中）。

默认docker-machine值在以下情况下可以使用：

  • 可执行文件的实际名称是docker-machine 。

  • 文件所在目录的路径包含在Path环境变量。

Docker Compose可执行文件

docker-compose或docker-compose.exe或docker-compose.sh的实际路径（通常位于Docker安装目录的bin文件夹中）。

默认docker-compose值在以下情况下可以使用：

  • 可执行文件的实际名称是docker-compose 。

  • 文件所在目录的路径包含在Path环境变量。
