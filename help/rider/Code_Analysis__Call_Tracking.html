JetBrains Rider 2019.2帮助

通话追踪

呼叫跟踪使您可以查看和浏览代码中的呼叫链。通过JetBrains Rider呼叫跟踪，您可以探索函数，事件，接口和闭包。

在层次结构视图中，JetBrains Rider使用图标来区分不同种类的符号。您可以在“ 符号图标方案”参考页上了解图标的含义。

在“ 检查结果”窗口中双击一个节点，以导航到编辑器中的相应调用。

调查去电

  1. 将插入号放在方法，事件，属性或构造函数的名称处。

  2. 按Ctrl+Shift+Alt+A然后在“ 检查此”列表中选择“ 去电” 。

  3. 在打开的“ 检查结果”窗口中，可以展开成员节点以在其上运行代码分析并显示当前成员调用的所有成员。您还可以展开每个子节点。

调查来电

  1. 将插入号放在方法，事件，属性或构造函数的名称处。

  2. 按Ctrl+Shift+Alt+A然后在“ 检查此”列表中选择“ 来电” 。

  3. 在打开的“ 检查结果”窗口中，您可以展开成员节点以运行代码分析并显示调用当前成员的所有成员。您还可以展开每个子节点。
