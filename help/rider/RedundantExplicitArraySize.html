代码检查：数组创建中的冗余显式数组大小规范

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

如果通过枚举其元素的值来初始化数组，则无需指定其大小，因为该大小由数组初始化器中元素的数量提供。

JetBrains Rider建议使用快速修复程序，以删除对数组大小的明确规定。

次优代码

快速修复后

var x = new int [3] {1，2，3};
var x = new int [] {1，2，3};
