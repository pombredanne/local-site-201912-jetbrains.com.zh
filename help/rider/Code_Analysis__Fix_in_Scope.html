确定范围

大多数快速修复程序都可以修复插入符号下的当前问题，但是其中一些（例如， 删除未使用的指令 ， 将字段设为只读 ， 删除多余的演员表等）也可以自动查找并解决更大范围内的问题。 —在当前文件，当前项目或整个解决方案中。

与“在范围内固定”类似，某些上下文操作也可以在更广泛的范围内应用 。

在范围内应用修复

您可以通过操作列表旁边的小箭头来识别这种修复。

特定范围内的小修正也可以通过代码清理来完成。

在范围内应用修复

  1. 将插入符号设置为JetBrains Rider检查中突出显示的代码问题。

  2. 按Alt+Enter或单击插入符号左侧的动作指示器以打开动作列表 。

  3. 选择您要应用的快速修复。

  4. 单击快速修复程序右侧的箭头，或按向右箭头键以扩展可用范围的列表。如果快速修复程序右侧没有箭头，则此快速修复程序仅适用于本地。

  5. 选择要应用的快速修补程序的范围，然后单击它或按Enter 。

如有必要，您可以使用一个命令撤消这些文件中所有已更改的内容：选择“ 编辑” |“更改”。在主菜单中撤消或按Ctrl+Z 。

修复范围内的命名不一致

修复了不一致的命名的快速修复程序，调用了“ 重命名”重构 ，以确保所有出现的符号在您的解决方案中都被重命名了。
