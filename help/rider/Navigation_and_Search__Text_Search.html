查找和替换文件中的文本

使用JetBrains Rider，您可以像大多数编辑器和IDE一样搜索和替换文本：

  • 要搜索或替换当前文档中的文本 ，请按Ctrl+F / Ctrl+H或选择编辑|查找|在菜单中查找/替换 。

  • 要在所需范围内搜索或替换 ，请按Ctrl+Shift+F / Ctrl+Shift+H或选择编辑|查找|在菜单中的路径中查找/替换 。

在当前文档中搜索文本的另一种选择是按Ctrl+F3在任何单词上查找该单词的所有出现情况 。

如果要查找类型，成员，文件，操作或设置，请使用“在任何地方搜索 （ Ctrl+T要么Shift两次）。
