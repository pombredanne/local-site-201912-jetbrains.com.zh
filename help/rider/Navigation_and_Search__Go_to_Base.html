转到基本符号

导航|超级方法
Alt+Home

该命令使您可以在继承层次结构中导航到当前符号的基本类型或方法。 您可以从编辑器，“ 结构”窗口和其他工具窗口中调用此命令。
使用“ 转到派生符号”和“ 转到实现”命令可以进行相反的导航。

除了导航之外，您还可以直观地浏览符号的完整继承层次。

导航到基本符号

  1. 将插入号放置在编辑器中的符号上，或在“ 结构”工具窗口中选择符号。

  2. 按Alt+Home或选择 导航|主菜单中的“ 超级方法 ”。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 如果所选符号有一个基本符号，JetBrains Rider将直接导航到该符号。

  4. 如果该符号有多个基本符号，它们将在下拉列表中列出。 该列表可能包含粗体和常规字体的项目。粗体表示项目具有直接继承，而常规字体用于间接继承。

    您可以执行以下操作之一：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 开始输入以过滤结果。请注意，您可以在此处使用CamelHumps 。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找窗口中打开 ， 按Shift+Enter要么+在数字键盘上。

此命令也可以在“ 导航到”快捷方式中找到Alt+` 。

如果某个方法替代，实现或隐藏了另一个方法，则可以通过出现特殊图标来查看它（例如， ）显示在方法声明的左侧。将鼠标悬停在该图标上可以查看此方法的基本类型；单击该图标可在方法层次结构中向上导航。

如果您导航到的项目属于当前解决方案，JetBrains Rider将在编辑器中打开相应的文件，并将插入号放置在符号声明中。如果在引用的库中找到它，JetBrains Rider将反编译相应的程序集并在反编译文件中找到该符号。
