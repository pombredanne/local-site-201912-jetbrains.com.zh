Settings/Preferences

File | Settings for Windows and Linux
JetBrains Rider | Preferences for macOS
Ctrl+Alt+S

The settings that pertain to the current project are marked with the icon.

The Settings dialog enables you to control behavior and appearance of JetBrains Rider.

Use the search field in the upper-left part of the dialog to find the options of interest. Alternatively, you can browse the settings using the hierarchical list of categories (groups of settings) underneath the search box.
