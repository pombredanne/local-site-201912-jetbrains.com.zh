JetBrains Rider 2019.2帮助

查找依赖于引用模块的代码

Shift+F12

使用JetBrains Rider，您可以快速找到在特定模块（项目或参考程序集）中使用符号的方式和位置。在大型解决方案中，这将帮助您检测架构问题并减少耦合。

在搜索结果中，您将看到代码中模块类型或类型成员出现的所有位置。

查找依赖于模块的代码

  1. 在解决方案资源管理器的项目的“ 引用”节点中选择一个项目或装配体引用。

  2. 请执行以下任一操作：

    • 按Shift+F12 。

    • 右键单击所选项目，然后在上下文菜单中选择“ 查找相关代码” 。

  3. 搜索结果将显示在“ 查找”窗口中，您可以在其中进一步分析结果 。
