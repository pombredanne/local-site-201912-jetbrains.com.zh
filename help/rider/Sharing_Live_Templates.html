共享实时模板

此页面描述了特定于Web的模板，这些模板可在HTML，JavaScript，TypeScript等中使用。.NET专用模板中描述了.NET语言的模板 。

JetBrains Rider将自定义实时模板组的定义以及添加到预定义模板组的模板存储在自动生成的XML配置文件中。

  • 对于自定义组，该文件包含该组包含的所有模板的定义。

  • 对于已修改的预定义组，该文件仅包含已添加（或已修改）实时模板的定义。

实时模板组配置文件存储在IDE配置目录的模板目录中。通过复制模板目录中的相关文件，您可以在团队成员和多个JetBrains Rider安装之间共享实时模板。此外，您可以在基于IntelliJ平台的所有IDE之间共享实时模板。

如果您的设置是通过“ IDE设置同步”插件同步的 ，则模板存储在IDE配置目录下的jba_config / templates下。

不要将实时模板配置文件复制到项目目录中。

JetBrains Rider提供了用于共享实时模板的导出和导入功能，这可能比手动复制配置文件更为方便。

导出实时模板配置

  1. 选择文件|从菜单中导出设置 。

  2. 在“ 导出设置”对话框中，确保已选中“ 实时模板”复选框，并指定将保存导出的设置的存档的路径和名称。

  3. 单击“ 确定”以基于实时模板配置文件生成文件。您可以与团队成员共享此文件，或将其导入另一个JetBrains Rider安装中。

导入实时模板配置

  1. 选择文件|从菜单导入设置 。

  2. 使用导出的实时模板配置指定存档的路径。

  3. 在导入设置对话框中，选择实时模板复选框，然后单击确定 。

  4. 重新启动JetBrains Rider后，您将在上看到导入的实时模板 编辑器|实时模板| JetBrains Rider设置的“ 其他语言”页面Ctrl+Alt+S 。
