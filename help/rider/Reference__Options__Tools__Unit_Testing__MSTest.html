启用MSTest支持

默认情况下，JetBrains Rider添加操作指示器，使您可以直接从编辑器执行MSTest测试，并自动在解决方案中发现MSTest测试 。清除此复选框可禁用此行为。

选择IDE文件夹

由于MSTest v1不会在没有Visual Studio的情况下分发，因此必须在计算机上的Visual Studio安装中指定IDE文件夹。JetBrains Rider将在此文件夹中找到所有必要的可执行文件。
另一方面，MSTest v2作为NuGet软件包提供 。如果在.NET Core项目中使用MSTest v2，则无需指定IDE文件夹。
但是，经典.NET Framework项目中的MSTest v2仍然需要Visual Studio，因此在这种情况下，您需要指定Visual Studio的IDE文件夹。

测试设定

指定自定义测试设置配置（解决方案中的.testsettings文件）。
