JetBrains Rider 2019.2帮助

在代码和Resx文件之间导航

JetBrains Rider的导航功能（ 转到符号的用法和转到声明 ）可帮助您在.resx文件中的资源实现与代码文件中的用法之间进行导航。

从声明导航到用法

  1. 在编辑器中打开一个.resx文件。

  2. 将插入号放在资源名称上。

  3. 按Shift+Alt+F12或选择 编辑查找|从主菜单显示用法 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  4. JetBrains Rider将打开使用资源的代码文件。如果有两种或多种用法，则可以在列表中选择一种。

从用法导航到声明

  1. 打开一个使用资源的代码文件。

  2. 将插入符号放在资源属性的用法上：

    或代码中使用的资源名称：

  3. 按F12或选择 导航|从主菜单声明或用法 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  4. 从列表中选择一个声明。您可以导航到生成的包装中的属性声明或.resx文件中的资源声明。

您可以从ComponentResourceManager和ResourceManager方法调用相应的资源声明。

你也可以Ctrl单击资源用法以转到其声明。
