元组解构是C＃7.0的功能。 默认情况下，JetBrains Rider基于关联的编译器自动检测C＃版本。但是，您可以显式指定目标C＃版本-按Alt+Enter在解决方案资源管理器中的项目上，并使用“ 项目属性”对话框的“ 应用程序”页面上的“ 语言版本”选择器。

JetBrains Rider可帮助您快速生成超载Deconstruct选定成员的方法。

您可以在上配置通用代码生成首选项 编辑器|的会员生成页面 JetBrains骑士设置Ctrl+Alt+S 。

生成解构器

  1. 在编辑器中，将插入号设置为类型名称，或在要插入解构函数的行的类型内。如果插入符号在类型名称上，则生成的代码将添加到类型声明的开头。

  2. 按Alt+Insert或选择 代码从主菜单生成...。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 析构函数” 。

  4. 在出现的“ 生成”对话框中，选择应在解构函数中使用的类型成员。

  5. 单击“ 确定”完成向导。

您可以调用查找用法 Shift+F12在Deconstruct找到所有相关元组解构的方法。

JetBrains Rider还检查类型是否包含所需的解构函数，如果不包含，则建议基于元组解构的快速修复方法。
