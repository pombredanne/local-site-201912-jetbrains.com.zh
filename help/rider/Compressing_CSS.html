缩小CSS

YUI Compressor是用于缩减CSS代码的工具。 缩小或压缩意味着在不更改源代码功能的情况下删除所有不必要的字符（例如空格，换行和注释）。 在开发和调试期间，这些字符使代码更易于阅读。在生产阶段，它们只会增加要传输的代码的大小。

在开始之前，请确保您的计算机上装有Node.js。

安装和配置YUI Compressor

  • 在嵌入式终端 （ Ctrl+Alt+1 ），输入：

    npm install yuicompressor

压缩CSS代码

要动态压缩代码，您需要将压缩器配置为JetBrains Rider File Watcher 。

创建文件监视程序

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），点击“ 工具”下的“ 文件监视程序 ”。打开的“ 文件监视程序”页面显示已配置的文件监视程序的列表。

  2. 点击或按Ctrl+N然后从列表中选择YUI Compressor CSS预定义模板。

  3. 在“ 程序”文本框中，指定yuicompressor- .jar文件的路径。如果通过Node Package Manager安装了该工具，JetBrains Rider会查找所需文件本身并自动填写字段。否则，请手动输入路径或单击然后在打开的对话框中选择文件位置。

  4. 按照文件监视程序中所述进行操作。

启用缩小文件监视程序后，一旦更改或保存文件监视程序范围内的CSS文件，缩小就会自动开始。

JetBrains Rider使用生成的输出创建一个单独的文件。该文件具有源CSS文件的名称，扩展名为min.css 。在“ 新建监视程序”对话框的“ 刷新的输出路径”字段中定义了生成文件的位置。但是，在Project Tree中 ，默认情况下，它显示在源CSS文件下，该文件现在显示为节点。要更改默认演示文稿，请在“ 项目”工具窗口中配置文件嵌套 。
