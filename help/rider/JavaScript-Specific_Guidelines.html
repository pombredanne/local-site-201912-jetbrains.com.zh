JetBrains Rider 2019.2帮助

的JavaScript

使用JetBrains Rider，您可以使用JavaScript和Node.js开发现代的Web，移动和桌面应用程序。

JetBrains Rider支持JavaScript和TypeScript编程语言， React和Angular框架，并与用于Web开发的各种工具紧密集成。

要打开按目录结构组织且没有解决方案文件的应用程序，请按Ctrl+O或选择 档案|打开|从主菜单中打开文件或文件夹... ，然后指定应用程序的根文件夹。

选择JavaScript语言版本

为了获得可靠和有效的编码帮助，您需要指定默认情况下将在应用程序的所有JavaScript文件中使用的语言版本。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript 。JavaScript页面打开。

  2. 从列表中，选择一种受支持的JavaScript语言版本：

    • ECMAScript 3
    • ECMAScript 5.1
    • JavaScript 1.8.5

    • ECMAScript 6 ：此版本增加了对ECMAScript 2015-2017中引入的功能的支持，以及对该标准的一些最新建议。

    • React JSX ：此版本在ECMAScript 6之上增加了对JSX语法的支持

    • Flow ：此版本增加了对Flow语法的支持。

使用多个JavaScript版本

如果您正在使用同时使用ECMAScript 5.1和ECMAScript或JSX或Flow的较新版本的应用程序，则最简单的方法是从JavaScript页面上的列表中选择整个项目的最高语言版本。例如，如果使用ES5.1和JSX，请启用JSX（因为它是ES5.1和ES6的超集）。

为不同的文件夹配置不同的JavaScript语言版本

  1. 在“ JavaScript”页面上 ，单击 JavaScript语言版本列表旁边。将打开“ JavaScript语言版本”对话框。

  2. 请点击并在打开的对话框中选择您需要自定义语言版本的文件夹。JetBrains Rider使您返回“ JavaScript语言版本”对话框，其中“ 路径”字段中显示所选文件夹。

  3. 从“ 语言”列表中，选择所选文件夹中文件的语言版本。在项目中的所有其他JavaScript文件中，JetBrains Rider将使用在JavaScript页面上选择的版本。

下载npm依赖项

确保您的计算机上装有Node.js。如果您的应用程序使用某些工具，库或框架，请下载所需的软件包。

在空项目中安装软件包

  • 在嵌入式终端 （ Ctrl+Alt+1 ），输入：

    npm install 。

如果您的项目中已经有package.json文件

  • 在编辑器或“ 项目”工具窗口中右键单击package.json文件，然后从上下文菜单中选择“运行'npm install” 。

  • 或者，运行npm install在航站楼 Ctrl+Alt+1 。

为项目依赖项配置代码完成

为了为项目依赖性提供代码完成，JetBrains Rider自动创建一个node_modules库。 在JetBrains Rider中， 库是一个文件或一组文件，除了JetBrains Rider从您编辑的项目代码中检索的功能和方法外，其功能和方法还添加到JetBrains Rider的内部知识中。有关详细信息和示例，请参阅配置JavaScript库和代码完成（IntelliSense） 。

查看参数提示

参数提示显示方法和函数中参数的名称，以使您的代码更易于阅读。 默认情况下，仅对文字或函数表达式的值显示参数提示，而对命名对象则不显示。

配置参数提示

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|外观 。“ 外观”页面打开。

  2. 单击“ 显示参数名称提示”复选框旁边的“ 配置” （默认情况下，该复选框处于选中状态）。

  3. 在打开的“ 配置参数名称提示”对话框中，选中“ 选项”区域中的“ 显示所有参数的名称”复选框。

要隐藏所有类型值的参数提示，请清除“ 显示参数名称提示”复选框。

自动导入JavaScript

JetBrains Rider可以为模块，类，组件和任何其他导出的符号生成导入语句。如果您在项目中使用ES6，则当您完成ES6符号时，JetBrains Rider可以动态添加丢失的导入语句。

对于更早的JavaScript版本或在完成时禁用wheauto-importrt，JetBrains Rider将该符号标记为未解析，并显示工具提示：在按Alt+Enter ，JetBrains Rider显示建议的快速修复：如果可能有多个进口来源，JetBrains Rider将显示建议列表：

从选择JavaScript语言版本中了解更多信息。

在代码完成时添加ES6导入语句

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |一般|自动导入 。将打开“ 自动导入”页面。

  2. 在“ TypeScript / JavaScript”区域中，选中“ 在代码完成时添加ES6导入”复选框。

JetBrains Rider还可以为项目依赖项中定义的符号生成导入语句。自动导入适用于包含TypeScript定义文件的包（例如moment或redux ）或作为ES模块编写的源中的符号。

配置语法突出显示

您可以根据自己的喜好和习惯配置可识别JavaScript的语法突出显示。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |配色方案| JavaScript 。

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。

在浏览器中运行JavaScript

  1. 在编辑器中，打开带有JavaScript参考的HTML文件。此HTML文件不一定必须是实现应用程序起始页面的HTML文件。

  2. 请执行以下任一操作：

    • 选择查看|从主菜单在浏览器中打开或按Alt+F2 。然后从列表中选择所需的浏览器。

    • 将鼠标指针悬停在代码上以显示浏览器图标栏： 单击表示所需浏览器的图标。

调试JavaScript

JetBrains Rider为与Chrome一起使用的客户端JavaScript代码提供了内置调试器。
您还可以在Firefox 36及更高版本中调试客户端JavaScript。但是，强烈建议您使用Chrome或Chrome系列的其他任何浏览器。使用JetBrains Rider，您可以调试在内置服务器，外部服务器或远程服务器上运行的JavaScript应用程序。有关详细信息，请参阅在Chrome中 调试JavaScript和在Firefox中调试JavaScript 。
