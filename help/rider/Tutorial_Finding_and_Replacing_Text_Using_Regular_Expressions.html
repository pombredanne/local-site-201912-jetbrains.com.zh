教程：使用正则表达式查找和替换文本

当您要搜索和替换特定的文本模式时，请使用正则表达式 。它们可以帮助您进行模式匹配，解析，结果过滤等。一旦学习了正则表达式语法，就可以将其用于几乎所有语言。

  1. 按Ctrl+H打开搜索和替换窗格。

    如果您需要搜索和替换多个文件，请按Ctrl+Shift+H 。

  2. 选中“正则表达式”复选框以启用正则表达式语法。如果需要检查regex语法参考，请单击Regex旁边的问号。

  3. 在顶部字段中输入搜索字符串，在底部字段中输入替换字符串。

  4. 当您搜索元字符时，例如.[{()\^$|?*+ ，您需要使用反斜杠将其转义\ ，因此它们可以被识别。

    例如，如果您需要查找. ，类型\.在搜索字段中。

  5. 当您在搜索字段中输入一系列字符时，JetBrains Rider也可以匹配字母大小写。

    例如，如果只想搜索大写字符，请在搜索字段中键入以下内容：

    \ b [AZ]

  6. 如果清除了“区分大小写”复选框，JetBrains Rider将同时搜索大写和小写字母。选中“区分大小写”复选框以匹配指定范围的大小写。

  7. 浏览事件时，JetBrains Rider将显示替换提示，因此您可以在单击“ 替换”按钮之前查看可能的结果。

有关更多详细信息，请参见RegEx语法 。

使用正则表达式捕获组和反向引用

您可以将正则表达式放在方括号内以便对其进行分组。每个组都有一个以1开头的数字，因此您可以在替换模式中引用（反向引用）它们。请注意，组0引用了整个正则表达式。

让我们考虑以下代码：

  1. 打开搜索和替换窗格Ctrl+H 。

  2. 在搜索字段中，输入括号()表示捕获组 ，例如： \stitle="(.*)?"\s*(/>*)

  3. 在替换字段中，以1开头的数字反向引用这些组，例如：

    2元$ 1

  4. JetBrains Rider会根据您的搜索说明突出显示发现的事件，并显示带有替换字符串的提示。

切换字符大小写

您可以使用正则表达式来更改与某些条件匹配的字符的大小写。

  1. 打开搜索和替换窗格Ctrl+H 。确保选中正则表达式复选框。

  2. 在搜索字段中输入搜索模式。

  3. 在替换字段中，根据您要实现的目标，输入以下语法之一：

    • \l将字符更改为小写，直到字符串中的下一个字符。
      例如， Bar变成bar 。

    • \u将字符更改为大写，直到字符串中的下一个字符。
      例如， bar变成Bar 。

    • \L将字符更改为小写，直到文字字符串的末尾\E 。
      例如， BAR变成bar 。

    • \U将字符更改为大写，直到文字字符串的末尾\E 。
      例如， bar变成BAR 。

有关更多详细信息，请参考RegEx语法参考表。

查找并替换字符串

假设您要替换元素中的属性title带有扩展标签 ，其中包含一些在双引号内的任意字符串。

这就是完成的方式。

  1. 在编辑器中打开XML文件后，按Ctrl+H 。“ 替换”窗格显示在编辑器顶部。

  2. 由于您要替换所有title属性，不管其中包含的实际字符串如何，都使用正则表达式。确保选中复选框正则表达式 。因此，您在“ 搜索”和“ 替换”字段中键入的所有内容都将被视为正则表达式。

  3. 在搜索字段中，开始输入描述所有内容的正则表达式title属性。
    \ stitle =“（。*）？“ \ s *（/> *）
    请注意，尽管正则表达式\stitle=".*?"\s*[/>]*匹配的内容title属性，建议在“ 替换”字段中捕获要引用的组。

    对于正则表达式，替换预览显示在工具提示中。

  4. 然后，在“ 替换”字段中，键入以下正则表达式：

    2元$ 1
    哪里$1指第一个捕获组，并且$2指第二个捕获组。

  5. 单击替换 ，或全部替换 。
