 警告
不推荐使用的类型

从8.0开始，不推荐使用“长”类型，因此不建议使用“长”类型（因为该类型的API不使用字符流，而是每次都传输整个值）。在Oracle中，请勿创建包含LONG列的表。请改用LOB列（CLOB，NCLOB，BLOB）。仅支持LONG列以实现向后兼容。

 警告
Postgres：从过程调用中选择

从dblink或返回记录要求类型别名列表的函数中选择。标量函数不需要别名，并且完全禁止键入别名列表。
