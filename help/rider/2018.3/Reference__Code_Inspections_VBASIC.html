Code Inspections in VB.NET

In VB.NET, JetBrains Rider 2018.3 provides two kinds of code inspections: 255 inspections that detect errors such as broken syntax, unresolved symbols, compiler errors, etc. (you cannot configure or disable any of these inspections), and 247 proprietary code inspections, any of which you can disable or change its severity level.
These code inspections help you detect code issues in design time in all open files, and additionally they allow you to find code issues in specific scope.

Configurable VB.NET inspections are listed below, grouped by their categories.
