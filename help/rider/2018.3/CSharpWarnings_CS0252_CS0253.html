Code Inspection: Possible unintended reference comparison

This code inspection warns you about reference comparison in context where value comparison is most likely implied. It helps you detect the following compiler warnings before you compile:

  • CS0252

  • CS0253

Last modified: 25 April 2019
Code Inspection: Operator '==' or operator '!=' with 'Object.Equals(object o)' and 'Object.GetHashCode()' not overridden
