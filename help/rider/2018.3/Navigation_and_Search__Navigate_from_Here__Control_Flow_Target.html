Navigate To Control Flow Target

Navigate | Navigate To | Control Flow Target
Alt+` | Control Flow Target

This command navigates from an editor editor position to the next control flow recipient. It becomes available when you invoke Navigate To at foreach, yield, jump statements (break, continue, goto, return), or throw statements.
