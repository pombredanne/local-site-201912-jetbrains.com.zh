Languages and Frameworks

Preferences configurable on this page are saved using the mechanism of directory-based settings.

File | Settings | Languages and Frameworks for Windows and Linux
JetBrains Rider | Preferences | Languages and Frameworks for macOS
Ctrl+Alt+S

When you select the Languages and Frameworks category in the left-hand pane, its main subcategories are listed in the right-hand part of the dialog.

  • JavaScript
  • Schemas and DTDs
  • ASP.NET (Razor)
  • C# Interactive
  • F# Interactive
  • Node.js and NPM
  • SQL Dialects
  • SQL Resolution Scopes
  • Style Sheets
  • TypeScript
  • XSLT
  • XSLT File Associations
