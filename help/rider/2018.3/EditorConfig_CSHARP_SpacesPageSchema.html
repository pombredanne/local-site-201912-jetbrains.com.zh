Preserve Existing Formatting

Extra spaces inside

Property names:

[resharper_]csharp_extra_spaces, [resharper_]extra_spaces

Possible values:

  • remove_all: Remove all extra spaces

  • leave_tabs: Leave extra tabs

  • leave_multiple: Leave multiple extra spaces

  • leave_all: Leave all extra spaces and tabs

Examples:

Before formatting

After formatting, remove_all

{ var x = foo ( 5 ) ; var xx = foo ( 5 + 6 ) ; var xxx = foo ( 5 + 6 + 7 ) ; var tab = foo ( 5 ) ; var tabx = foo ( 5 + 6 ) ; var tabxx = foo ( 5 + 6 + 7 ) ; }

{ var x = foo(5); var xx = foo(5 + 6); var xxx = foo(5 + 6 + 7); var tab = foo(5); var tabx = foo(5 + 6); var tabxx = foo(5 + 6 + 7); }
