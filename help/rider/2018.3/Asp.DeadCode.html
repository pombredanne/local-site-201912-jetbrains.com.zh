Code Inspection: ASP.NET dead code

You can configure this inspection:

  • disable it to ignore all related issues
  • suppress it in any place in the code to ignore specific issues

  • change its severity level to make the detected issues less or more noticeable

This inspection detects dead ASP.NET code such as empty tags. For example, if a tag that should have a value, such as <script>, is empty, JetBrains Rider suggests removing it:

Suboptimal code

After the quick-fix

<script runat="server"> </script>

// removed
