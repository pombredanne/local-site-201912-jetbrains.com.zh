Go to Line

Navigate | Line/Column
Ctrl+G

The Go to Line command lets you quickly jump to any line of code in the current document.

You can also display line numbers in the editor editor. To do so, select Show line numbers on the Editor | General | Appearance page of JetBrains Rider settings (Ctrl+Alt+S).

  1. Press Ctrl+G or choose Navigate | Line/Column in the menu.

  2. Type the target line number and optionally, a column number after a colon (:), and then click OK.
