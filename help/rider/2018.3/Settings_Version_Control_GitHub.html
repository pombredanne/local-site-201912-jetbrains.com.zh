Ctrl+Alt+S

Use this page to specify your GitHub remote storage account, or create a GitHub account if you do not have one yet.

You can specify several accounts and set the default account for each of your projects. Thus, you won't have to specify which account to use each time you need to connect to GitHub, for example, to create a gist.
