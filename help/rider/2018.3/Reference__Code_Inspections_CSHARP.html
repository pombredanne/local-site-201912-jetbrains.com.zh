Code Inspections in C#

In C#, JetBrains Rider 2018.3 provides two kinds of code inspections: 745 inspections that detect errors such as broken syntax, unresolved symbols, compiler errors, etc. (you cannot configure or disable any of these inspections), and 575 proprietary code inspections, any of which you can disable or change its severity level.
These code inspections help you detect code issues in design time in all open files, and additionally they allow you to find code issues in specific scope.

Configurable C# inspections are listed below, grouped by their categories.

Inspections in this category detect violations of code syntax styles.
