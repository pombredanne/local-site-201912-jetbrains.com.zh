Code Inspection: Resource is overridden with identical value

You can configure this inspection:

  • disable it to ignore all related issues
  • suppress it in any place in the code to ignore specific issues

  • change its severity level to make the detected issues less or more noticeable

If there are several cultures organised in a culture hierarchy, and there is a resource entry that is declared in a culture-specific resource file, which has the same value as in the neutral culture, JetBrains Rider highlights the resource declaration and suggests to remove the redundant override:
