Project-Level Refactorings

Most of JetBrains Rider refactorings target types and type members. When you perform any of these refactorings, the entire solution is always taken into account. JetBrains Rider also provides a set of special project-level refactorings that work like bulk fixes in the scope of a project, solution folder, or the entire solution:

  • Adjust Namespaces
  • Move Types into Matching Files
