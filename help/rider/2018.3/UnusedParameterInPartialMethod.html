Code Inspection: Unused parameter in partial method

This inspection detects unused parameters in the implementation parts of a partial method.

Unused method parameters might indicate incomplete or dead code. Follow JetBrains Riderâ€™s suggestion to remove an unused parameter and update usages.

You can configure this inspection:

  • disable it to ignore all related issues
  • suppress it in any place in the code to ignore specific issues

  • change its severity level to make the detected issues less or more noticeable
