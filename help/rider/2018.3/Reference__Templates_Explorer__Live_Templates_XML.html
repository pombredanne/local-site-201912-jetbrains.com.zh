Predefined Live Templates for XML

This topic lists all predefined live templates for XML in JetBrains Rider 2018.3. For more information about live templates, see Creating Source Code Using Live Templates

Template

Details

ctx

Current file context

Scope everywhere

Body

$CTX$

Parameters

  • CTX - Provides list of items describing current context. This includes file name, containing type name, namespace name, etc.

nguid

Insert new GUID

Scope everywhere

Body

$GUID$

Parameters

  • GUID - Generates new Globally Unique Identifier (GUID)

t

Create tag

Scope XML tags

Body

<$TAG$>$END$</$TAG$>

Parameters

  • TAG - Suggests XML tag name used in current document

  • END - The caret position after the template is applied.
