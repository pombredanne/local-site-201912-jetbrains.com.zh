Predefined Surround Templates for JavaScript

This topic lists all predefined surround templates for JavaScript in JetBrains Rider 2018.3. For more information about surround templates, see Surrounding Code Fragments with Templates

Template

Details

{}

Surround the selection with braces

Scope JavaScript

Body

{ $SELECTION$$END$ }

Parameters

  • SELECTION - The text selected by the user before invoking the template.

  • END - The caret position after the template is applied.
