Subversion

With the Subversion integration enabled, you can perform basic Subversion operations from inside JetBrains Rider.

JetBrains Rider currently supports integration with Subversion 1.7 and above.

JetBrains Rider comes bundled with the Subversion plugin. Apart from it, you also need to download and install the command line client on your machine.
