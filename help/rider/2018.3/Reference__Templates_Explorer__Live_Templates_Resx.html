Predefined Live Templates for Resx

This topic lists all predefined live templates for Resx in JetBrains Rider 2018.3. For more information about live templates, see Creating Source Code Using Live Templates

Template

Details

ctx

Current file context

Scope everywhere

Body

$CTX$

Parameters

  • CTX - Provides list of items describing current context. This includes file name, containing type name, namespace name, etc.

nguid

Insert new GUID

Scope everywhere

Body

$GUID$

Parameters

  • GUID - Generates new Globally Unique Identifier (GUID)

d

Create resource entry

Scope .resx roots

Body

<data name="$name$"> <value>$END$</value> </data>

Parameters

  • name - Suggest tag name used in current document

  • END - The caret position after the template is applied.
