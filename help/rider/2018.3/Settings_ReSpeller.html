Ctrl+Alt+S

Use this page to configure ReSpeller — the JetBrains Rider's spell-checking engine. ReSpeller will look for spelling mistakes in identifiers, comments, and string literals. When it detects a potential spelling mistake or a typo, it will let you know in the same way as other code issues — in design time or when you run code inspection is specific scope.

Item

Description

General

Use options in this section to toggle spell-checking altogether or in specific cases.

Visibility options

Select which kinds of members should be spell-checked.

User words

A newline-separated list of words to be ignored by the spell-checker. Note that the spell-checker is sensitive to case.
