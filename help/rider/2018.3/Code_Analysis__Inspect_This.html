Inspect This

Ctrl+Shift+Alt+A

This command is a shortcut to JetBrains Rider's code exploration and inspection features:

  • Call Tracking

  • Value Tracking

  • Exploring Type Hierarchy

  • Find code issues

You can invoke this command from the editor editor, from the Structure window and other tool windows. Depending on the context, this shortcut helps you access all code-exploration features available in that context.

To view contextually available code exploration features

  1. Place the caret on a symbol in the editor editor or select the symbol in the Structure tool window.

  2. Press Ctrl+Shift+Alt+A

  3. The Inspect This drop-down list appears.

  4. Choose an item from the list and click on it or press Enter.
