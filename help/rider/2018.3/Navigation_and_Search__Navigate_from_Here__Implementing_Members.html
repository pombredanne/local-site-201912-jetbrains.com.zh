Navigate To Implementing Members

Navigate | Navigate To | Implementing Members
Alt+` | Implementing Members

This command navigates to all interface members implemented in the current type. It becomes available when you invoke Navigate To upon an interface entry in a type that implements it:

If there is only one implemented member, JetBrains Rider navigates directly there. If there are several implemented members, each of them is highlighted with blue.

The highlights are also duplicated with markers on the marker bar on the right. Checking these markers may be helpful if not all highlights are visible on the current screen.
To navigate between the highlighted items, choose Edit | Find | Find Next/Previous in the menu or press F3 / Shift+F3.
When you finish examining the highlights, press Esc to remove them.
