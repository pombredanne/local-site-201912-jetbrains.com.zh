Context Actions for HTML

JetBrains Rider provides the following context actions for HTML:

Name

Description

Collapse empty tag

Converts from empty opening and closing tags into empty tag

Convert html entity

Converts html entity from numbered form to named and visa versa

Edit path mapping

Opens path mapping settings for current project

Expand empty tag

Splits a self-closing tag into opening and closing tags

Insert table column after

Insert table column after

Insert table column before

Insert table column before

Insert table row after

Insert table row after

Insert table row before

Insert table row before

Remove tag

Removes tag leaving its content intact

Remove tag attribute

Removes tag attribute

Reorder tag attributes

Reorders tag attributes alphabetically

Replace all tags

Replaces all tags with the same name in the file with different tag

Replace tag

Replaces tag with different tag
