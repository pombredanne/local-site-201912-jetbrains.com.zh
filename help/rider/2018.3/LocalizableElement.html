Code Inspection: Element is localizable

You can configure this inspection:

  • disable it to ignore all related issues
  • suppress it in any place in the code to ignore specific issues

  • change its severity level to make the detected issues less or more noticeable

This inspection detects hard-coded string literals and suggests moving them to resource files. If the same string is already defined in resources, JetBrains Rider will suggest to replace the literal with the existing resource entry.
