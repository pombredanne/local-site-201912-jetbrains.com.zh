注释和注释代码

代码带行/块注释
Ctrl+Alt+/ / Ctrl+Shift+/

使用JetBrains Rider，一个键就足以注释或取消注释代码。

要选择逻辑代码块 ，请按Ctrl+Alt+Right一或多次。

评论和取消评论

要用行注释注释或取消注释代码， 按Ctrl+Alt+/或选择 代码主菜单中的“ 行注释”注释/取消注释

// C＃中的注释行...

如果没有选择，JetBrains Rider将注释当前行，如果注释了该行，则取消注释，然后将插入符号移动到下一行。
根据“ 不要在第一列开始缩进注释”选项，将添加有或没有缩进的注释。您可以分别为不同的语言配置此选项，例如，对于C＃，此选项处于启用状态 编辑器|代码样式| C＃|的选项卡，缩进，对齐页面 JetBrains骑士设置Ctrl+Alt+S 。

如果有选择，JetBrains Rider将注释或取消注释选择范围内的所有行。即使部分选中了一行，注释标记也会插入到该行的开头。

取消注释任何行注释的另一种方法是将插入号设置为此行，按Alt+Enter然后选择取消注释 。

使用块注释进行注释和取消注释

要用块注释注释或取消注释代码， 按Ctrl+Shift+/或选择 代码在主菜单中使用块注释进行注释/取消注释

/ * C＃中的注释代码块...* /

如果没有选择，JetBrains Rider将在当前插入符号位置之后的第一个空白处创建一个空的块注释。
如果插入符号在块注释中，则该块未注释。

如果有选择，JetBrains Rider将在选择的开头和结尾添加块注释标记。插入符号移动到块注释的末尾。

评论任何代码块的另一种方法是选择它，按Alt+Enter然后选择评论选择 。取消注释注释中的代码的相同方法-将注释插入符设置在注释的任何位置，按Alt+Enter然后选择取消注释 。
