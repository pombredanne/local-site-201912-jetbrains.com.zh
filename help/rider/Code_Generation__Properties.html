Alt+Insert | 属性/只读属性

JetBrains Rider将帮助您为要公开的所有字段生成属性。

在下面的示例中，此命令用于生成以下属性： _radius和_center领域。

您还可以使用“ 封装字段”重构将字段转换为属性-将插入符设置为字段，然后按Alt+Enter然后选择封装字段 。

生成将公开选定字段的属性

  1. 在编辑器中，将插入号设置为类型名称，或者在要插入将公开选定字段的属性的行的类型内。如果插入符号在类型名称上，则生成的代码将添加到类型声明的开头。

  2. 按Alt+Insert或选择 代码从主菜单生成...。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 属性” /“只读属性” 。

  4. 在出现的“ 生成”对话框中，选择要为其生成属性的字段。

  5. （可选）使用以下应用于所有生成的属性的控件：

    • 访问权限定义生成的属性的访问权限修饰。

    • 只读定义是否将属性生成为只读。如果选择Automatic ，那么JetBrains Rider将仅为只读字段生成getter，而为其他任何字段生成getter和setter。
      如果您在“ 生成”弹出窗口中选择了“ 只读属性 ”，那么选择器将设置为“ 是” ，但是您可以根据需要进行更改。

    • 虚拟 （如果适用）添加virtual修改生成的属性。

    • 通知属性更改 （如果适用）（如果适用） INotifyPropertyChanged或从支持属性更改通知的类继承（例如，Prism的NotificationObject ），并使用[NotifyPropertyChangedInvocator]属性装饰通知属性更改的方法，您可以选择通过更改通知调用来实现属性。

  6. 单击“ 确定”完成向导。
