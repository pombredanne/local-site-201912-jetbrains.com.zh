代码产生|委派成员
Alt+Insert | 委派成员

使用JetBrains Rider，您可以快速生成将执行委派给可通过私有字段和属性访问的公共成员（方法和属性）的成员-换句话说，以包含类型发布任何私有字段或属性的接口的所需部分。

您可以在上配置通用代码生成首选项 编辑器|的会员生成页面 JetBrains骑士设置Ctrl+Alt+S 。

产生委派成员

  1. 在编辑器中，将插入号设置为类型名称，或在要插入委派成员的行的类型内。如果插入符号在类型名称上，则生成的代码将添加到类型声明的开头。

  2. 按Alt+Insert或选择 代码从主菜单生成...。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 委派成员” 。

  4. 在出现的“ 生成”对话框中，您将看到当前类型的私有属性和字段的列表。您可以展开这些项目以查看其类型的成员。选择这些类型成员中的一些或全部，JetBrains Rider将在当前类型中生成将执行委派给所选类型成员的包装器。

  5. 单击“ 确定”完成向导。
