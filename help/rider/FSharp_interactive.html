F＃互动

要在计算机上启用F＃交互，请执行以下操作：

在单个组件选项卡上启用F＃编译器的情况下安装Visual Studio Build Tools 2017 。如果您已经拥有VS Build Tools或Visual Studio 2017，则可以运行安装程序来修改安装并选择安装F＃编译器。

安装最新的稳定版Mono： https : //www.mono-project.com/download/stable/ 。

按照以下说明安装fsharp软件包： https : //fsharp.org/use/linux/ 。

JetBrains Rider包含一个读取-评估-打印循环（REPL） ，使您无需编译F＃代码即可进行实验。

开始新的F＃交互式会话

  • 在编辑器中选择一个代码块，然后按Ctrl+\或者，按Alt+Enter然后选择将选择发送到F＃交互式 。
    JetBrains Rider将在“ 运行”窗口的新选项卡中启动一个新的交互式会话，并将选定的代码发送到该会话。如果F＃交互式会话已经在运行，则所选代码将附加到该会话中。

  • 选择工具| F＃互动|从菜单启动新的F＃交互式会话 。
    在这种情况下，JetBrains Rider将打开一个空的交互式会话并重置当前的F＃交互式会话（如果已运行）。您还可以随时通过单击重置来清除当前会话在工具栏上。

F＃Interactive启动后，您可以编写任何F＃表达式并按Enter 。

如果成功，F＃Interactive将尝试编译您的代码并执行它。否则，解释器将返回错误消息。

您在会话中执行的所有表达式都记录在历史记录中。要查找先前执行的语句，请使用Up / Down按钮或复制任何最近执行的命令：

要一次执行复制的命令，请取消选中“ 将最近的命令复制到Interactive Editor” ， 语言和框架| F＃的互动页面 JetBrains骑士设置Ctrl+Alt+S 。
