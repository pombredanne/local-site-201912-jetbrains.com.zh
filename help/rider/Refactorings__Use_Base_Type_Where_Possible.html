JetBrains Rider 2019.2帮助

在可能的情况下使用基本类型

这种重构有助于将对类型的引用替换为其对基本类型之一的引用。JetBrains Rider将分析整个解决方案中所有对该类型的引用，并更新可以安全地替换为所选基本类型的引用。（可选）您也可以替换is和typeof操作员。

尽可能使用基本类型

  1. 通过以下方式之一选择类型：

    • 在编辑器中，将插入号设置为类型的名称。

    • 在解决方案资源管理器中选择一种类型。

    • 在“ 结构”窗口中选择一种类型。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择“ 使用基本类型”

    • 选择重构|在主菜单中尽可能使用基本类型 。

    将打开“ 可能时使用基本类型”对话框。

  3. 选择基本类型之一。

  4. （可选）选择是否要替换is操作员和/或typeof操作员。

  5. 要应用重构，请单击下一步 。

  6. 如果未发现冲突，JetBrains Rider将立即执行重构。否则，它会提示您解决冲突 。

    重构完成后，它将在任何地方进行的所有更改（包括其他文件）都注册为一个操作。这样您就可以使用“ 撤消”操作Ctrl+Z只需一次击键即可回滚所有这些更改。
