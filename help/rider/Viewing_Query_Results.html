JetBrains Rider 2019.2帮助

查询结果

通常，当您运行查询时，您会以表格格式接收结果。每个语句显示在“ 服务”工具窗口中的单独选项卡中。例如，如果查询中有两个SELECT语句，则在工具窗口中将看到两个选项卡。

服务工具窗口包括一个控制台列表，这些控制台根据与数据源的连接会话进行分组。

为每个查询打开一个新标签

默认情况下，JetBrains Rider每次运行查询时都会使用结果更新同一选项卡。您可以更改此行为，并在每次运行查询时创建一个选项卡。

  1. 在IDE设置中Ctrl+Alt+S ，转到工具|数据库|一般

  2. 选择“ 在新选项卡中打开结果”复选框，然后单击“ 确定” 。

将自定义标题用于带有结果的选项卡

您可以在查询前的注释部分中定义选项卡标题。在将文本视为标题之后字段中，您可以保留符号或字符的组合，并且此组合之后的任何文本都将被视为标签标题。默认情况下，不使用任何组合，因此后面的任何文本--要么/*被视为标签标题。

要禁用此功能，请打开设置Ctrl+Alt+S ，导航到工具|数据库|常规 ，然后清除“ 为查询前的注释创建结果标题”复选框。

  1. 在设置中Ctrl+Alt+S ，导航到工具|数据库|一般

  2. 在将文本视为标题之后字段中，定义选项卡标题的组合。

编辑结果集中的值

  1. 在结果集中，单击要编辑的单元格值。

  2. 指定一个新值，然后按Enter 。

  3. 要将更改提交到数据库，请单击Submit图标（ ），或按Ctrl+Enter 。

像在数据编辑器中一样 ，您可以编辑结果集中的值，自动提交更改，删除行，搜索文本以及浏览数据。

导出到文件

  • 右键单击％result_set％，选择转储数据|要归档 。

  • 右键单击查询，单击“ 执行到文件”，然后选择要用于导出的文件类型（例如， 逗号分隔（CSV） ）。

  • 在工具栏上，单击“ 转储数据”图标（ ），然后选择“ 到文件” 。

导出到剪贴板

  • 在结果集中或编辑器中选择数据，然后按Ctrl+C 。

  • 在工具栏上，单击“ 转储数据”图标（ ），然后选择“ 到剪贴板” 。

比较两个结果集

  1. 对要比较的表运行查询。

  2. 在“ 服务”工具窗口中，单击“与... 比较”图标（ ），然后选择要添加到差异中的结果集。

  3. 如果需要，在比较对话框中更改“ 公差”参数的值。Tolerance参数定义两个结果集之间允许的最大差异数。例如，如果您希望两行的数据在同一列中不同，则认为两行相等，请输入1在“ 公差”字段中。

导航到相关对象

您可以从表中检索所选对象的行，并将焦点移到该对象。例如，在项目树中选择一个字段，然后按Ctrl+Alt+F7 。焦点移到新选项卡中的相应列。

要导航到结果集中的对象，请选择一个对象，然后转到“ 导航|导航”。相关符号 Ctrl+Alt+F7 。

在相关数据之间导航

您可以在不同类型的相关数据之间导航：

  1. 引用数据 ：当前对象引用的数据。

  2. 引用数据 ：引用当前对象的数据。

  3. 右键单击表中的列，然后导航到“ 转至| | | |”。参考资料 F12或转到|参考资料 Ctrl+Alt+F7 。

  4. 如果出现“ 选择目标”弹出窗口，请选择表和要显示的行：

    • 第一个引用行 ：表中的所有行。选择引用当前行的第一行。

    • 仅引用行 ：仅引用当前行的行。

  5. 按Enter

将标签固定在查询结果上

如果使用一个和相同的选项卡来显示查询结果，并且得到想要保留的结果，则可以将选项卡固定到工具窗口。

  • 右键点击标签，然后选择固定活动标签 。

  • 请点击在工具栏上。

排序数据

要对列中的表数据进行排序，请单击标题行中的单元格。

您可以启用“ 通过ORDER BY排序” 选项，以启用对相应DBMS的数据排序。

另外，您可以在“ 结构”工具窗口（“ 视图” |“工具”窗口|“结构” ）中按列对数据进行排序。在“ 结构”工具窗口中，右键单击列名，然后从“ 排序”菜单中选择一种排序方法。

点击设置图标（ ），然后选择重置视图 。结果，数据变得未排序，列以它们最初定义的顺序出现，并显示所有列。

结构视图中的列操作

结构视图向您显示所选表中所有列的列表。结构视图可在“ 结构”工具窗口（“ 视图” |“工具”窗口|“结构” ）或弹出窗口（“ Alt+\ ）。

右键单击表标题行中的单元格，然后选择“ 列列表” 。

您可以对列执行以下操作：

或者，要对列数据进行排序，请使用上下文菜单。

  • 数据升序排序Shift+Alt+Up或降序（ Shift+Alt+Down ）。

  • 重置排序状态（ Ctrl+Shift+Alt+Backspace ）

  • 隐藏列或显示隐藏列（ Space ）。隐藏的列的名称显示为删除线。

将数据导出到数据库

  1. 确保在数据库工具窗口中将数据库添加为数据源 。

  2. 右键单击表中的任何单元格，然后单击“ 导出到数据库” 。

  3. 指定数据库，目标架构（以使用导出的数据创建新表）或表（以将导出的数据添加到现有表）。

  4. 配置目标表的数据映射和设置。

更改SQL日志的位置

要更改SQL日志的位置，请添加-Didea.system.path属性添加到自定义VM选项。此更改会影响所有日志。

  1. 单击帮助|自定义VM选项 。

  2. 在选项列表的末尾，添加以下行： -Didea.system.path=/path/to/your_log_directory 。
