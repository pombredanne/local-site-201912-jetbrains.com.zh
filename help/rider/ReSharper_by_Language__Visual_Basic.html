JetBrains Rider 2019.2帮助

语言和框架：Visual Basic .NET

JetBrains Rider支持最多VB 14和部分VB 15的Visual Basic .NET。

默认情况下，JetBrains Rider基于关联的编译器自动检测VB.NET版本。但是，您可以显式指定目标VB.NET版本-按Alt+Enter在解决方案资源管理器中的项目上，然后使用“ 项目属性”对话框的“ 应用程序”页面上的“ 语言版本”选择器。
根据当前的VB.NET版本（是自动检测还是显式设置），JetBrains Rider会调整其功能。例如， 代码检查可以检测代码中的位置，使用新的语言功能可以简化这些位置。

VB.NET中的JetBrains Rider功能描述按以下主题分组：

  • 代码检查和快速修复
  • 编码协助
  • 代码样式协助
  • 导航和搜索
  • 重构
  • 代码生成
  • 代码模板
