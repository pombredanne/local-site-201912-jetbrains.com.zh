Quick Lists

Preferences configurable on this page are saved using the mechanism of directory-based settings.

File | Settings | Appearance and Behavior | Quick Lists for Windows and Linux
JetBrains Rider | Preferences | Appearance and Behavior | Quick Lists for macOS
Ctrl+Alt+S

Use this page to. A quick list is a popup that contains a custom set of JetBrains Rider commands and is associated with a keyboard or mouse shortcut. You can create as many quick lists as necessary. Each command, included in a quick list, is identified by a sequential number. Numbering starts from the numerals (0 to 9), and then proceeds with the letters in alphabetical order.

Item

Shortcut

Description

Ctrl+N

Create a new Quick List.

Ctrl+R, D

Delete the selected Quick List.

Display name

Edit the name of the selected Quick List.

Description

Edit the description of the selected Quick List. (The description is optional.)

Ctrl+N

Use this button to add actions to the Quick List. Select the actions in the Add Actions to Quick List dialog that opens.

Use this button to add a separator at the end of the Quick List. (Separators help you organize menu commands in logical groups.)

Ctrl+R, D

Remove selected actions from the Quick List.

Alt+U

Use this button to move the selected item one line up in the list.

Alt+D

Use this button to move the selected item one line down in the list.
