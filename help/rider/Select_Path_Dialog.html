选择路径对话框

使用此对话框选择必要的文件或文件夹。

对话框名称和可用功能取决于您当前正在执行的任务（通常会禁用不适当的功能）。例如，根据情况，您可能只能选择一项或多项。在某些情况下，您可以选择一个或多个文件夹，但无法选择一个或多个文件等。

  • 路径场
  • 使用文件浏览器中的拖放

主要功能

通过工具栏图标（在“ 图标”列中显示）可以访问此对话框中的大多数功能。或者，您可以使用上下文菜单命令（通过右键单击树中的项目访问；在“ 命令”列中列出）或键盘快捷键（“ 快捷方式”列）。

显示或隐藏隐藏的文件和文件夹

使用此图标或命令可以打开或关闭显示隐藏的文件和文件夹。

隐藏或显示路径

 Ctrl+P

使用此命令或快捷方式可以隐藏或显示路径字段 。（该命令位于对话框右侧的工具栏上，并显示为超链接。）

路径场

路径字段（如果未隐藏）位于工具栏下方。此字段显示树中所选项目的路径。

通过使用此字段中提供的路径自动完成功能，您可以快速浏览文件系统以选择所需的文件或文件夹。

要激活路径自动完成功能，请将光标放在该字段中，然后按Ctrl+Space 。开始输入。将出现一个弹出窗口，显示当前目录的内容。从弹出菜单中选择一个项目。继续输入并选择，直到选择了必要的项目。

使用右侧的按钮可显示最近条目的历史记录列表。

使用文件浏览器中的拖放

如果将相应的项目从文件浏览器（资源管理器，Finder等）拖到显示树的区域，则可以快速找到并选择所需的文件或文件夹。
