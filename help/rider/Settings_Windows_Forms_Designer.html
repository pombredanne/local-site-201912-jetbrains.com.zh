Windows Forms Designer

Preferences configurable on this page are saved using the mechanism of layer-based settings.

File | Settings | Tools | Windows Forms Designer for Windows and Linux
JetBrains Rider | Preferences | Tools | Windows Forms Designer for macOS
Ctrl+Alt+S

On this page, you can configure JetBrains Rider's Windows Forms Designer.

Item

Description

Enable Windows Forms Designer

Use this checkbox to disable Windows Forms Designer and open Windows Forms controls in the code editor.

Enable designer exceptions navigation

If this option is enabled and an error happens when starting the designer, JetBrains Rider will automatically navigate to the code that caused this error.
