Predefined Surround Templates for TypeScript

This topic lists all predefined surround templates for TypeScript in JetBrains Rider 2019.2. For more information about surround templates, see Surround Code Fragments with Templates

Template

Details

{}

Surround the selection with braces

Scope JavaScript

Body

{ $SELECTION$$END$ }

Parameters

  • SELECTION - The text selected by the user before invoking the template.

  • END - The caret position after the template is applied.

()

Surround the selection with parentheses

Scope JavaScript

Body

($SELECTION$$END$)

Parameters

  • SELECTION - The text selected by the user before invoking the template.

  • END - The caret position after the template is applied.
