此页面可帮助您配置许多设置，这些设置控制JetBrains Rider如何生成新的ASP.NET代码以及如何重新格式化现有代码 。 您可以调整值并检查页面底部的预览窗格，以查看首选项如何影响代码。

您在此页面上配置的选项在以下情况下适用：

  • 当代码在编辑时或粘贴后自动格式化时 （您可以在 编辑器|一般|的打字协助页面 JetBrains骑士设置Ctrl+Alt+S 。

  • 当JetBrains骑士产生具有代码完成和代码生成功能的新代码，应用代码模板并执行重构 。

  • 重新格式化现有代码时 。
