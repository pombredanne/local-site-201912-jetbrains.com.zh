JetBrains Rider 2019.2 Help

TODO

Preferences configurable on this page are saved using the mechanism of directory-based settings.

File | Settings | Editor | TODO for Windows and Linux
JetBrains Rider | Preferences | Editor | TODO for macOS
Ctrl+Alt+S

On this settings page, you can configure TODO patterns.

Multiline comments

Treat the indented text on the following lines as part of the same TODO

Select the checkbox to use multiline TODO comments.

Patterns

Manage the list of TODO patterns recognized by JetBrains Rider.

Item

Shortcut

Description

Icon

Shows the icon that is assigned to a pattern. This icon appears in the TODO tool window as a marker for the corresponding TODO items.

Case sensitive

Indicates whether the current pattern is case sensitive.

Pattern

Displays the regular expression for a pattern.

Ctrl+N

Open the Add Pattern dialog, where you can create a new TODO pattern.

Enter

Open the Edit Pattern dialog, where you can modify the selected TODO pattern.

Ctrl+R, D

Remove the selected TODO pattern.

Filters

Manage the list of filters to display only specific TODO patterns.

Item

Shortcut

Description

Name

Shows the name of the filter.

Patterns

Shows the names of TODO patterns included in the filter.

Ctrl+N

Open the Add Filter dialog, where you can define a new filter.

Enter

Open the Edit Filter dialog, where you can modify the selected filter.

Ctrl+R, D

Remove the selected filter.
