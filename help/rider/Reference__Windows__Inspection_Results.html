在特定范围内运行代码检查时，此工具窗口将显示代码问题。

此窗口中显示的所有代码问题均具有指示其严重性级别的图标：

  • -错误

  • - 警告

固定标签

单击此按钮固定或取消固定当前选项卡。您可能需要固定标签，以防止在此窗口中达到最大标签数时自动关闭标签。

全部展开 / 全部折叠
Ctrl+NumPad Plus / Ctrl+NumPad Minus

展开/折叠当前选项卡中的所有节点。

自动滚动到源

当按下此按钮时，JetBrains Rider将在窗口的源视图中找到您选择的项目：项目和文件夹-在解决方案资源管理器中；文件-在编辑器中。

标题栏上下文菜单和按钮

您可以右键单击窗口标题栏，然后使用上下文菜单配置其查看模式 ，将窗口与其他工具窗口栏关联，或者调整大小并隐藏窗口。

Shift+Escape

使用此图标或快捷方式隐藏工具窗口。

当与Alt键，单击此图标将隐藏附加到同一工具窗口栏的所有工具窗口 。
