创建分支或标签对话框（Subversion）

VCS |颠覆|创建分支或标签

在此对话框中，设置用于根据本地工作副本或存储库版本创建分支或标记的参数。

项目

描述

复制自

在此部分中，指定要从中创建分支或标记的源文件夹。副本的来源可以从本地工作副本或存储库中获取。

工作副本

单击此选项可基于本地工作副本创建分支或标签。在文本框中输入路径或单击浏览 然后在打开的对话框中选择所需的目录。

储存库位置

单击此选项可在存储库的基础上创建分支或标记。请执行以下任一操作：

  • 在文本框中输入存储库位置的URL。

  • 单击浏览按钮，然后选择源存储库位置。

  • 点击按钮以使用项目主目录。

修订版

在本节中，指定要从中创建分支或标记的源修订。可用的选项有：

  • HEAD-选择此选项可在HEAD版本的基础上创建分支或标签。

  • 已指定 -选择此选项可根据特定版本创建分支或标签。在文本框中手动输入修订号或单击“ 浏览” 然后在打开的“ 更改浏览器”对话框中选择所需的修订。

复制到

使用此部分可以定义分支或标记的目标文件夹。可用的选项有：

  • 分支或标签 -选择此选项可将所选修订复制到特定分支或标签。
    在“ 基本URL”文本框中，指定分支或标记的基本URL。在“ 名称”文本框中，指定新分支的名称。

  • 任何位置 -选择此选项必须将所选修订复制到您选择的位置。在下面的文本框中，指定任何有效位置的URL。手动输入URL或单击浏览 并在打开的“ 选择存储库位置”对话框中指定所需的位置。

评论

在文本区域中键入一些有意义的描述。

切换到新创建的分支或标签

选择以跳到您创建的分支或标签。
