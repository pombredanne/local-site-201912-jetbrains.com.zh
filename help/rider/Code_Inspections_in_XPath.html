冗余类型转换

此检查检查是否有任何类型转换功能string() ， number()要么boolean()是多余的，即参数的类型是否与函数的返回类型相同，或者表达式的期望类型是否为“ any”类型。尽管有时可能会故意使用这种显式转换来强调类型，但是通常可以安全地将其删除。

由XPathView + XSLT支持

 警告
在XPath谓词中使用索引0

此检查检查谓词索引中或与函数比较中是否意外使用了零。 position() 。这几乎总是一个错误，因为在XPath中，索引从1开始而不是从0开始。

例：

//someelement[position() = 0]要么//something[0]
