Navigate from Symbols

From any code symbol, JetBrains Rider can bring you to its declaration, implementations, usages, and more. The scope of navigation is symbols and files in your entire solution as well as all assemblies referenced in the projects of your solution.

If there are multiple navigation locations for your destination, you can open them in the Find Results window and use its features for further navigation (for more information, see analyzing search results ).

The commands that provide navigation from the current code symbol are described in the following topics:

  • Go to Declaration
  • Go to Type Declaration
  • Go to Usages of Symbol
  • Go to Base Symbols
  • Go to Implementation
  • Go to Derived Symbols
  • Navigate To
    • Assembly Explorer
    • Control Flow Target
    • Decompiled Sources
    • Exposing APIs
    • Extension Methods
    • Function Exits
    • Generic Substitutions
    • Implementing Members
    • Member Overloads
    • Metadata View
    • Overriding Members
    • Parameter Declaration
    • Sources from Symbol Files
    • Test Explorer
    • TODO Explorer
  • Find Usages
    • Find Usages of a Symbol
    • Finding Code Dependent on Referenced Modules
    • Highlight Usages in File

These and other commands for navigating from the current symbol are included into the Navigate To shortcut.
