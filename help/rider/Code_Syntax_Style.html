语法样式

代码样式的方面之一是如何使用可互换的语言语法构造。例如，从编译器的角度来看，以下两个方法定义是相同的，但是语法结构的选择使它们看起来有所不同：

[Conditional（“ DEBUG”）] [测试]静态公共无效Foo（String str）{foreach（str ch ch）{系统。安慰。WriteLine（value：ch）; }}
[Test，Conditional（“ DEBUG”）] public static void Foo（string str）{foreach（字符串中的ch）。WriteLine（ch）; }

配置代码语法样式

当您开始使用JetBrains Rider时，它将参考其默认设置，以确保在整个代码库中始终使用可互换的语法构造。但是，您可以根据自己的喜好配置每个个人首选项。有两种方法可以配置代码语法样式：

  • 在上配置您的首选项 编辑器|代码样式| [语言] |的代码样式页面 JetBrains骑士设置Ctrl+Alt+S 。这些修改将使用分层设置的机制进行保存，以便您可以将其应用于所有项目的本地计算机上，或与队友共享。

  • 使用JetBrains Rider定制的EditorConfig代码样式属性在EditorConfig文件中指定首选项，这还将使您在整个团队中拥有一致的代码样式。
    请注意， .editorconfig文件中指定的首选项将覆盖保存在JetBrains Rider设置中的首选项。

通过代码清理应用代码语法样式

要在所需范围内应用代码语法样式，您还可以使用代码清除 。您可以使用默认配置文件“ 完全清理”或“ 重新格式化并应用代码样式”之一运行代码清理 ，或者使用所需的代码语法样式规则子集创建并运行自定义配置文件 。

C＃中代码语法样式的其他操作

在当前选择中应用C＃代码语法样式

  1. 在编辑器中，选择要在其中应用代码语法样式的代码块。

  2. 按Alt+Enter或单击插入符号左侧的动作指示器以打开动作列表 。

  3. 选择在操作列表中应用代码样式 。
