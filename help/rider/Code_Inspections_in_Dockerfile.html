检查

描述

默认严重性

参数数量错误

该检查在Dockerfiles中找到参数数量错误或可疑的命令，并在编辑器中突出显示它们。

 错误
“键=值”对的间距

尽管在Dockerfile文档中没有明确指定，但实际上不允许使用“键=值”对的某些间距组合。ARG，ENV或LABEL命令的规则不同。此检查突出显示已知被拒绝的组合。

 错误
``ADD''/``COPY''语义

此检查突出显示了ADD和COPY命令的参数可能存在的问题。
