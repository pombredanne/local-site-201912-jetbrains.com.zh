Ctrl+E, L

实时模板是JetBrains Rider的Visual Studio代码片段的智能得多的版本。与代码片段不同，JetBrains Rider的模板可以从上下文中预测可以使用哪些变量，字段和类型。在大多数情况下，您无需自己键入任何内容-只需从建议值列表中进行选择即可。

JetBrains Rider为所有受支持的语言提供了大量预定义的实时模板 。预定义模板列表包括用于遍历集合，编写断言方法，创建新数组，输出值，对LINQ查询进行编码，生成自动属性等的模板。

使用实时模板创建代码片段

  1. 在文本编辑器中，将插入符号设置为要从实时模板中插入代码片段的位置。

  2. 请执行以下任一操作：

    • 键入模板快捷方式或其首字母。例如，在C＃方法中，键入itar调用迭代数组模板。

    • 按Ctrl+E, L或选择 代码在主菜单中插入实时模板... ，从建议列表中选择一个模板，然后按Tab要么Enter 。

  3. 如果模板具有可编辑的参数 （即，需要用户输入），JetBrains Rider将在编辑器中部署一个热点会话 ，并将输入位置设置为第一个参数。然后，您可以执行以下操作：

    • 如果JetBrains Rider建议使用当前参数的一些值，请使用Up和Down箭头键浏览建议值列表，或仅键入所需值。

    • 按Tab要么Enter接受该值并移至下一个参数的输入位置。如果这是最后一个参数，则热点会话完成，并且脱字符号移动到为该会话定义的结束位置。

    • 按Shift+Tab将输入焦点移至上一个参数的输入位置。

    • 按Esc退出热点会话。在这种情况下，所有会话参数都将使用默认值初始化。
