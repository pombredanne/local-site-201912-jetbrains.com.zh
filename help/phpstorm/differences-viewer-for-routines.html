Differences viewer for routines

An SQL routine is a set of SQL statements that are stored on the server (for example, procedures or functions). In PhpStorm, you can compare two routines.

  1. In the Database tool window (View | Tool Windows | Database), select two routines.

  2. Right-click the selection and select Compare. Alternatively, press Ctrl+D.
