在源代码中导航

功能

捷径

使用此快捷方式可以...

文件结构弹出

 Ctrl+F12

显示“ 结构”弹出窗口，以快速浏览当前文件。

选择目标

 Alt+F1

将焦点从当前文件，类，方法或对数据源表的引用移至“ 选择目标”弹出菜单中建议的视图。请参阅使用“选择范围”弹出窗口进行导航 。

最近的文件

 Ctrl+E

显示最近打开的文件列表。

最近更改的文件

 N/A

显示“ 查找最近的文件 ”列表。

类型层次结构

 Ctrl+H

浏览所选类的层次结构。

导航到课程

 Ctrl+N

通过在弹出窗口中指定其名称，可以直接导航到项目中的类 。

导航到文件

 Ctrl+Shift+N

通过在弹出窗口中指定文件名，直接导航到项目中的文件 。

导航到最近打开的文件

 Ctrl+E

显示最近打开的文件列表。

导航到最近更改的文件

 N/A

显示最近更新的文件列表。

导航到行

 Ctrl+G

通过指定其编号导航到当前文件中的任何行 。

导航到宣言

 Ctrl+B

导航到插入符号处的符号声明 。

导航到实施

 Ctrl+Alt+B 导航到尖号处该项目的实施。

这种类型的导航使您可以从基本类型或成员跳到其任何最终实现，而跳过继承链中的中间步骤。例如，假设该类MyClass继承抽象类MyAbstractClass ，进而实现接口MyOriginalInterface 。当插入符号位于时，选择转到实施 MyOriginalInterface直接将您带到MyClass 。

导航到类型声明

 Ctrl+Shift+B

导航到插入符号处的符号的类型声明，该符号是变量或方法调用。

导航到超级方法

 Ctrl+U

导航到插入符号下方法的超级方法声明。

导航到测试/测试主题

 Ctrl+Shift+T

导航至插入符号班的测试（如果有），或从测试导航至测试对象。

导航到相关符号

 Ctrl+Alt+Home

在具有复杂关系的文件之间导航。

转到标题/源

 F10

在源文件和头文件之间切换。

导航到下一个方法

 Alt+Down

导航到活动编辑器选项卡中的下一个方法声明。

导航到上一个方法

 Alt+Up

导航到活动编辑器选项卡中的上一个方法声明。

导航到开头括号

 Ctrl+[

导航到当前代码块的开始。

导航到右括号

 Ctrl+]

导航到当前代码块的末尾。

背部

 Ctrl+Alt+Left 撤消上一次导航操作。

在macOS计算机上，您也可以使用三指从右向左滑动手势。

向前

 Ctrl+Alt+Right 重做上一个未完成的导航操作。

在macOS计算机上，您也可以使用三指从左到右滑动手势。

导航到上次出现

 Ctrl+Alt+Up

导航到先前找到的项目。

导航到下一次发生

 Ctrl+Alt+Down

导航到下一个找到的项目。

最后编辑位置

 Ctrl+Shift+Backspace

浏览最新的变更点。

导航到下一个突出显示的错误

 F2

导航到下一个发现的错误/警告。

导航到上一个突出显示的错误

 Shift+F2

导航到先前找到的错误/警告。
