提取字段

提取字段重构使您可以声明一个新字段，并使用所选表达式对其进行初始化。原始表达式将替换为该字段的用法。新领域是使用默认可见性修饰符创建的，该修饰符在代码样式的“代码生成”选项卡上设置。PHP页面的 设置/首选项对话框Ctrl+Alt+S 。

将字段提取到位

  1. 将插入号放置在要提取到字段中的一段代码中。

  2. 按Ctrl+Alt+F或从主菜单中选择“ 重构|提取字段 。

  3. 选择要引入的表达式作为字段。

    如果PhpStorm在您的代码中检测到多个事件，则可以让您指定要替换的事件。

  4. 提供新字段的名称，然后选择要在何处初始化：在其声明中，在当前方法中或在类构造函数中。 当您使用PHPUnit测试时，PhpStorm还可让您在setUp方法。有关在PhpStorm中使用PHPUnit的更多详细信息，请参阅使用PHPUnit测试 。

使用对话框提取字段

如果在“ 常规”页面上清除了启用就地模式复选框的 设置/首选项对话框Ctrl+Alt+S ，“ 提取字段”重构是通过“ 提取字段”对话框执行的 。

  1. 在编辑器中，选择要用字段替换的表达式或变量，或者只是将插入号放置在这样的表达式或变量声明中。

  2. 在主菜单或所选内容的上下文菜单中，选择“ 重构” |“选择” 。提取物|字段 ，或按Ctrl+Alt+F 。

  3. 在打开的“ 提取字段对话框”对话框中，配置重构：

    1. 指定字段名称。

    2. 通过选择Initialize in下的必要选项，指定应在何处初始化新字段。 当您使用PHPUnit测试时，PhpStorm还可让您在setUp方法。有关在PhpStorm中使用PHPUnit的更多详细信息，请参阅使用PHPUnit测试 。

    3. 在“ 可见性”区域中，为新字段选择可见性范围。

    4. 要替换所有出现的所选表达式（如果在类中多次找到所选表达式），请选中“ 替换所有出现”复选框。

    5. 单击确定 。
