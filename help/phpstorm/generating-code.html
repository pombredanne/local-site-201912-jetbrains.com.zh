产生程式码

PhpStorm提供了多种生成常见代码结构和重复元素的方法，可帮助您提高生产率。这些可以是预定义模板，也可以是自定义模板，它们会根据上下文，各种包装程序和字符自动配对而不同地应用。

此外，PhpStorm还提供代码完成和Emmet支持。

在主菜单中，选择代码|生成 Alt+Insert打开带有您可以生成的可用结构的弹出菜单。

生成构造函数

PhpStorm可以生成一个构造函数，该构造函数使用相应参数的值初始化特定的类字段。

为一个类生成一个构造函数

  1. 在代码菜单上，单击生成 Alt+Insert 。

  2. 在“ 生成”弹出窗口中，单击“ 构造函数” 。

  3. 如果该类包含字段，请选择要由构造函数初始化的字段，然后单击OK 。

以下代码片段显示了为类生成构造函数的结果：

class MyClass {public $ field; / ** * MyClass构造函数。* @param $ field * /公共功能__construct（$ field）{$ this-> field = $ field; }}

生成获取器和设置器

PhpStorm可以为您的类中的字段生成访问器和更改器方法（ getter和setter ）。生成的方法只有一个参数。

在PHP上下文中，使用PHP getter / setter文件模板生成getter和setter。默认情况下，如这些模板中所指定，setter会与set前缀，以及带有get要么is根据推断字段类型的前缀boolean要么con-boolean 。前缀是${GET_OR_IS}默认getter模板中的变量。默认模板是在“ 文件”和“代码模板 ”上的“ 代码”选项卡中配置的。

生成类的获取器和设置器：

  1. 在代码菜单上，单击生成 Alt+Insert 。

  2. 在“ 生成”弹出窗口中，单击以下选项之一：

    • Getter生成访问器方法以获取类字段的当前值。

    • Setter生成用于设置类字段值的增变方法。

    • 使用Getter和Setter生成访问器和更改器方法。

  3. 选择要为其生成getter或setter的字段，然后单击“ 确定” 。
