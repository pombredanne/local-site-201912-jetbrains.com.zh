创建测试的运行/调试配置

您可以运行测试（测试用例，测试套件等），可以使用运行/调试配置来运行测试，其方式类似于运行普通应用程序。PhpStorm提供了一个框架，用于为测试目的创建特殊的运行/调试配置，可以将测试指定为目标。

除了常规的测试配置创建之外 ，PhpStorm还支持为特定目标创建测试配置 ，例如，为文件，类或目录中的所有测试，或为任意选择的测试类和方法创建测试配置 。

创建测试配置

  1. 通过执行以下操作之一打开“ 运行/调试配置”对话框：

    • 从主菜单中，选择运行|编辑配置 。

    • 按Shift+Alt+F10然后从上下文菜单中选择“ 编辑配置 ”。

  2. 请点击在工具栏上，然后选择所需的配置类型：

    • PHPUnit的运行PHPUnit测试。

    • Behat用于运行Behat测试。

    • Codeception运行Codeception测试。

    • PHPSpec用于运行PHPSpec规范。

    • JSTestDriver，用于使用JSTestDriver测试JavaScript。请参阅JSTestDriver和运行/调试配置：JSTestDriver 。

    • 噶用于测试使用JavaScript JSTestDriver。请参阅业力和运行/调试配置：业力 。

    • 用于测试Node.js的 NodeUnit 。请参阅测试Node.js和运行/调试配置：NodeUnit 。

    • 用于测试Dart的 DartUnit ，请参阅“运行/调试配置：Dart测试” 。

    自2018年5月起，不推荐使用Nodeunit。有关详细信息，请参见Nodeunit官方网站。

  3. 在打开的对话框中，指定测试范围，配置参数和要在测试执行之前执行的活动。应用更改并关闭对话框。

为特定目标创建测试配置

  1. 在“ 项目”工具窗口中，右键单击所需的测试目录，单个文件或类。

  2. 从选择的上下文菜单中，选择创建运行配置 。

  3. 在打开的“运行/调试配置”对话框中，指定要在测试执行之前执行的配置参数和活动。应用更改并关闭对话框。
