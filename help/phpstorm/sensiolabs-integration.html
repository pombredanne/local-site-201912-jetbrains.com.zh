SensioLabsInsight

SensioLabsInsight是一项质量保证服务，可以分析源代码以查找会降低项目总体质量的问题。它可以分析任何使用PHP开发的应用程序，并且具有专门设计的规则，可以对各种项目类型（例如Symfony应用程序，Laravel网站，Drupal模块等）执行高级分析。每次分析都会生成一份非常详细的报告，其中包含所有发现的问题，从性能问题到潜在的安全漏洞。此外，每个错误都在您自己的代码上下文中进行解释，并包括有关如何修复该错误的全面文档。

SensioLabsInsight通过SensioLabsInsight插件与PhpStorm无缝集成。该插件在编辑器装订线中显示所有检查结果，可方便地导航到完整的SensioLabsInsight报告，等等。在本教程中，我们将研究如何开始使用SensioLabsInsight服务，配置集成以及通过SensioLabsInsight服务提供的其他检查来提高项目代码质量。

您可以观看视频教程，以了解有关SensioLabsInsight与PhpStorm集成的更多信息。

在你开始前

在开始使用SensioLabsInsight之前，请确保已安装并启用 SensioLabsInsight插件。

使SensioLabsInsight与PhpStorm集成

要熟悉SensioLabsInsight，请参阅“ SensioLabsInsight 入门”教程。

安装了SensioLabsInsight插件后，您需要启用与该项目的SensioLabsInsight帐户的集成。创建或打开项目后，PhpStorm将显示一条通知消息，提示您配置插件。

将PhpStorm连接到您的SensioLabsInsight帐户

  1. 请执行以下任一操作：

    • 单击通知消息中的配置链接。

    • 在里面 设置/首选项对话框Ctrl+Alt+S ，单击SensioLabsInsight 。

  2. 在相应的字段中提供您的SensioLabsInsight 用户UUID和API令牌 。您可以在“ 帐户” | “ 帐户”中找到这些值。 API / SDK SensioLabsInsight页面。

    要将这些身份验证设置保存为要在所有其他项目中使用的默认设置，请选中“ 另存为默认API设置”复选框。

  3. 重新启动PhpStorm，以使更改生效。

使用PhpStorm项目配置SensioLabsInsight集成

要为现有项目启用SensioLabsInsight集成，只需确保它指向您在SensioLabsInsight Web界面中使用过的同一VCS存储库即可。要检查项目的当前VCS根配置， 在里面 设置/首选项对话框Ctrl+Alt+S ，单击版本控制 。

如果您已配置与 SensioLabsInsight帐户的连接 ，则该插件将基于远程URL自动与特定项目关联。您可以查看当前选择的项目，或在“设置/首选项”对话框的“ SensioLabsInsight”页面上手动选择一个项目：

如果要创建一个新项目，请按照从头开始创建新项目中所述进行操作。如果从VCS中签出了一个项目，该项目将自动与相关的SensioLabsInsight项目关联（基于其远程URL）；否则，您需要明确选择它。

在PhpStorm中浏览SensioLabsInsight分析结果

配置SensioLabsInsight和PhpStorm之间的集成后，PhpStorm将从SensioLabsInsight服务获取最新报告。所有违反规则的内容都会显示在编辑器装订线中。点击图标以获取有关规则违规的更完整注释，并可以选择直接导航到完整的SensioLabsInsight报告。

在PhpStorm中运行检查

SensioLabsInsight检查与PhpStorm的内置检查集成在一起，这意味着您可以通过Code |检查代码或代码|按名称运行检查...命令Ctrl+Shift+Alt+I 。

您还可以从PhpStorm 状态栏中的SensioLabsInsight面板中运行基于SensioLabsInsight的检查。单击此面板将显示所有与SensioLabsInsight相关的操作。选择“运行代码检查”，然后在“ 检查结果工具”窗口中查看检查结果。

在PhpStorm中更新SensioLabsInsight报告

SensioLabsInsight旨在不断改善分析结果，这意味着更新报告是重要的一步。要从SensioLabsInsight服务器获取最新报告，请在PhpStorm状态栏中的SensioLabsInsight面板中，单击与SensioLabsInsight上的最新分析同步 ：

为了获得最佳性能，请确保将您的SensioLabsInsight项目配置为在每次提交时都进行重新分析。与服务器同步后，PhpStorm插件将自动获取有关当前提交哈希或相应分支的分析报告。
