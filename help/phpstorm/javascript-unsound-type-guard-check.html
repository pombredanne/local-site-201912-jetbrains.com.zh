代码检查：不健全的类型检查

在以下两种情况之一中，不可以使用typeof x类型防护：

  1. typeof x从不对应于指定的值（例如，当x的类型为'string | boolean'时， typeof x ==='number '）

  2. typeof x始终对应于指定的值（例如，当x的类型为'string'时， typeof x === 'string'）


x instanceof在以下两种情况之一中，类型防护可能不健全：

  1. x的类型与A不相关

  2. x的类型是A或A的一个子类型
