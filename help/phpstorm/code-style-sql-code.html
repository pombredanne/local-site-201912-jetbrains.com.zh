项目

描述

放置左括号

在该行的第一个关键字下对齐或缩进左括号。选择“ 在同一行上”以使关键字的开头括号保持在该行上。

放置元素

更改括号中元素的位置。

  • 对齐同一行 ：对齐子句的所有成员，使第一个成员与括号保持在同一行。

  • 不缩排包装 ：将子句的所有成员移至新行，而无需添加缩排。

  • 包装对齐 ：对齐并将子句的所有成员移至新行。

  • 包装缩进 ：添加缩进并将子句的所有成员移至新行。

放置右括号

更改右括号的位置。您可以将右括号相对于左括号（ Under ）放置，或相对于左括号（ Under elements ）放置第一个元素。

包装元素

  • 印章 ：将每个值移到新行。

  • 如果时间长则切碎 ：如果文本超过配置的行长，则将每个值移至新行。要配置行长，请打开设置Ctrl+Alt+S ，导航到编辑器|代码样式 ，然后在N列的硬包装字段中输入所需的长度。

  • 如果较长则自动换行 ：将一段文本分成几行，以使每一行都适合配置的行长 。要配置行长，请打开设置Ctrl+Alt+S ，导航到编辑器|代码样式 ，然后在N列的硬包装字段中输入所需的长度。

放置逗号

移动逗号（ , ）到代码行的开头（ To begin ）或结尾（ To end ）。

“ 自动”选项将分析周围的环境并计算最适合逗号的位置。例如，您出现了三个逗号：开头两个逗号，结尾一个逗号。自动选项会将逗号的第三次出现移动到开头。仅当在单个上下文中具有三个以上案例时，此选项才有效。否则，逗号将保持不变。

在中间选项将一行两行代码连接在一起。

如果在启用“ 开始”选项时要在逗号后禁用新行，请清除“ 包装”选项卡上的“ 换行符”复选框。

在括号内放置空格
