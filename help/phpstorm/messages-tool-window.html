PhpStorm 2019.3 Help

Messages Tool Window

PhpStorm parses the compilation output and displays it in a convenient format in the Messages tool window, enabling you to navigate to the location of relevant problems in the source code.

The Messages tool window is only available if there are messages to show.

Toolbar Buttons

Icon

Tooltip

Description

Refresh Ctrl+F5

Rerun compilation.

Stop

Terminate compilation. This button is enabled when compilation is in progress.

Hide warnings
