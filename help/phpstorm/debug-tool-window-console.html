在控制台窗格中评估变量

在PHP和JavaScript上下文中，您可以在调试会话期间在“ 调试工具”窗口中查看变量的值。这可以通过几种方式完成：

  • 在“ 变量”窗格中选择变量，然后从所选内容的上下文菜单中选择“ 在控制台中评估 ”。

    当您切换到“ 控制台”窗格时，变量名称以绿色显示在>并且其值在下面以蓝色显示。

  • 或者，在以下位置键入任何变量的名称>在控制台中 ，然后按Enter显示其值。

    代码完成由您支配：在键入变量名称时，PhpStorm将显示建议列表。

    请注意，仅当按下“ 使用控制台输入”切换按钮时，此功能才可用 。

  • 要评估先前评估的变量，请使用键盘上的向上和向下箭头找到它，然后按Enter 。

在调试会话期间在控制台窗格中执行代码

在PHP调试会话期间，您不仅可以评估变量，还可以在控制台窗格中更改变量的值，调用PHP函数以及定义其他变量。请注意，此功能不适用于PHPUnit或Behat运行/调试配置。

  1. 启动调试会话（请参阅PHP Debugging Session ），然后切换到“ 调试器”选项卡的“ 控制台”窗格。

  2. 按下使用控制台输入切换按钮在工具栏上。

  3. 在以下位置输入语句或表达式>然后按Enter 。PhpStorm评估您的代码片段，并在输入代码下方显示输出。您可以键入大多数PHP结构，包括类声明，函数声明，变量，表达式等。代码完成随时供您使用：键入时，PhpStorm将显示建议列表。

  4. 键入多行代码片段时，请按Shift+Enter开始新的一行Ctrl+Enter分割线。

  5. 默认情况下，数组和对象的输出包装在var_export()功能并显示在控制台中 。若要隐藏显示的输出，请清除“ 调试”页面上的“ 在调试控制台中显示数组和对象子级”复选框。
