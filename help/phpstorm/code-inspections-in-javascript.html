尾递归

报告尾递归，即函数在返回之前调用自身作为其最后一个动作。尾递归总是可以用循环来代替，这会更快。一些JavaScript引擎执行此优化，而其他JavaScript引擎则不执行。因此，尾递归解决方案在不同环境下可能具有相当不同的性能特征。
