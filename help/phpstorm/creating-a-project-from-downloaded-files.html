PhpStorm 2019.3帮助

从下载的文件创建项目

您可能不时需要更新外部创建的内容并上传到远程主机，并可以通过FTP / SFTP / FTPS协议，网络共享或安装的驱动器进行访问。PhpStorm不支持直接编辑远程主机上的文件。因此，要在PhpStorm中使用远程源，您需要下载它们并将其安排在PhpStorm项目中。要使本地和远程或共享源保持同步，可以使用“ 选项”对话框中的“将更改后的文件自动上传到默认服务器”下拉列表配置自动上传。

PhpStorm根据服务器访问配置与文件所在的远程主机进行交互。您可以在项目创建期间使用适当的现有配置或定义新配置，有关详细信息，请参阅部署应用程序 。
