Command Line Tool Support: Zend Framework 2

File | Settings | Tools | Command Line Tool Support for Windows and Linux
PhpStorm | Preferences | Tools | Command Line Tool Support for macOS

The dialog opens when you click and choose Zend Framework 2 Tool in the Command Line Tools dialog. Use the dialog to configure Zend Framework 2 Tool support in PhpStorm.

The tool works only with local PHP interpreters.

Item

Description

Path to zf.php or zftool.phar

In this field, specify the location of the <Zend-Framework-home>/zf.php file or the zftool.phar archive. PhpStorm parses the contents of the specified file for Zend Framework commands.

Path to PHP executable

In this field, specify the location of the PHP engine installation folder.
