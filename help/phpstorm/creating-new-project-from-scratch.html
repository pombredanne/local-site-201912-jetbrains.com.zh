PhpStorm 2019.3帮助

从头开始创建新项目

  1. 请执行以下任一操作：

    • 在“ 欢迎使用”屏幕上的“ 快速入门”部分下，单击链接创建新项目 。

    • 在主菜单上，选择“ 文件” |“其他”。新品|项目 。

  2. 在打开的“ 选择项目类型”对话框中，在左侧窗格中选择“ PHP空项目 ”。

  3. 在位置文本框中，指定项目名称和位置。

  4. 单击“ 创建”时 ，PhpStorm会生成一个项目存根，并根据您在打开的信息对话框中的选择在当前窗口或新窗口中将其打开：

    有关详细信息，请参阅“ 打开多个项目 ”部分。

创建项目后，按照配置PHP开发环境中所述设置开发环境 。
