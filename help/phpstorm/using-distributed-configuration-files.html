使用分布式配置文件（.htaccess）

PhpStorm在编辑分布式配置文件时提供语法高亮显示，格式设置， 代码完成和文档查找 。

分布式配置文件用于对HTTP Apache Server配置进行基于目录的更改，并且通常名为.htaccess 。

如果您使用的文件名为.htaccess ，则PhpStorm会将其识别为分布式配置文件，并为其提供完整的编码帮助，因此无需您采取其他步骤。

如果要使用其他名称的文件，则需要将此全名或相应的模式与分布式配置文件类型关联。之后，PhpStorm会将名称与指定模式匹配的任何文件视为分布式配置文件，并进行相应处理。

将名称模式与分布式配置文件类型相关联

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，然后选择编辑器|文件类型 。

  2. 从“ 公认的文件类型”列表中选择“ Apache Config ”。

  3. 在“已注册模式”区域中，单击 。

  4. 在打开的“ 添加通配符”对话框中，指定定义分布式配置文件扩展名的模式。

  5. 单击确定 。 PhpStorm将使您返回“ 文件类型”页面，在该页面上将指定的模式添加到“已注册模式”列表中。

    • 要丢弃模式，请在列表中选择它，然后单击 。

    • 要更改模式，请在列表中选择它，然后单击 ，并根据需要在打开的“ 添加通配符”对话框中更新模式。
