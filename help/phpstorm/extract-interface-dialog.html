描述

接口名称

在这个领域里， 指定要创建的接口的名称。默认情况下，PhpStorm建议从中提取接口的当前类的名称。请更新名称，否则PhpStorm将报告错误，并且重构将无法开始。

尽可能用接口替换类引用

  • 清除此复选框后，PhpStorm只会在单独的文件中创建一个接口，并将源类标记为新接口的实现。

  • 选中该复选框后，PhpStorm将创建一个接口，并尝试用PhpDoc注释和方法参数中对该接口的引用替换对源类的引用。建议的更改是强制性显示在“ 查找”工具窗口的“ 重构预览”窗格中的。要提取接口并应用建议的更改，请单击“ 执行重构” 。

命名空间

在此下拉列表中，指定新接口将属于的名称空间。默认情况下，选定源类的名称空间。

目标目的地目录

指定该文件夹用于存储为新界面创建的专用文件。默认情况下，该字段为只读，并显示与根据PSR0标准接口将属于的名称空间相对应的文件夹。

要将具有接口的类存储在另一个文件夹中，请单击F2并指定您选择的文件夹。

会员组成界面

下表显示了当前类中检测到的所有方法。要将方法移至“ 将成员移至类”中指定的接口，请选中该方法旁边的复选框。

php文件

在这方面指定您希望如何处理PhpDoc注释。可用的选项有：

  • 照原样：选择此选项可将文档保留在源类中的位置。

  • 复制：选择此选项可将内联文档复制到提取的界面，而不将其从其当前位置删除。

  • 移动：选择此选项可将内联文档移动到提取的接口，并从其在源类中的当前位置删除。
