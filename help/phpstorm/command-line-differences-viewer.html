Viewing Difference Between Files from Command Line

You can use PhpStorm as a command line tool for viewing differences between files.

  • In the command line, run the diff command using the following syntax:

    <PhpStorm> diff path1 path2

The selected files will be opened in the Differences viewer for files.

The command options are as follows:

Option

Description

<PhpStorm>

The platform-specific PhpStorm launcher.

On macOS and Linux, it is recommended to use the command-line launcher created during the PhpStorm installation, or create one manually by selecting Tools | Create Command-line Launcher on the PhpStorm main menu. Refer to Enable a command-line launcher on macOS/Linux for details.

On Windows, it is recommended to add the path to the PhpStorm executable to the Path environment variable.

path1

The full path to the local copy of the file.

path2

The full path to the repository version copy of the file.

Examples

Syntax
phpstorm.exe diff path1 path2
Example
phpstorm.exe diff C:\MyProject\Readme.md C:\MyProject\Readme.md.bak
Syntax
phpstorm diff path1 path2
Example
phpstorm diff ~/MyProject/Readme.md ~/MyProject/Readme.md.bak
