创建新项目：添加本地服务器

档案|现有文件中的新项目-Web服务器在本地安装，源文件位于其文档根目录下

如果在“ 创建新项目：指定本地服务器”页面上选择了“ 添加新本地服务器”选项，则会打开该页面。在此页面上，定义新的本地Web服务器访问配置。

项目

描述

名称

在此字段中，输入新配置的名称。

Web服务器URL

在此字段中，输入服务器配置根目录的URL地址（ 服务器配置根目录是通过服务器配置可访问的本地或远程服务器上文件树中的最高文件夹。对于就地服务器，它是项目根。）。

请点击在字段中，以确保可以访问指定的服务器根URL地址并指向正确的网页。

HTTP和HTTPS协议均受支持。
