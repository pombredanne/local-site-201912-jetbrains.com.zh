PhpStorm 2019.3帮助

PHP代码质量工具

除了内置的编码帮助，PhpStorm还通过与流行的代码质量工具集成来提供代码样式检查：

  • PHP_CodeSniffer ，

  • PHP混乱检测器

  • PHP CS修复程序 。

这些工具验证您的代码是否与您选择的编码标准保持一致。您可以指定一种预定义的编码标准，也可以使用自己的预先定义的编码标准。

要使用PhpStorm而不是命令行使用代码质量工具，您需要在PhpStorm中注册它们并将其配置为PhpStorm 代码检查 。在PhpStorm中安装并启用后，这些工具既可以即时运行也可以批量运行。
