Opening Files from Command Line

You can open an arbitrary file, folder, or project for editing in PhpStorm from the command line, optionally specifying where the caret should rest after opening the file.

  • In the command line, run the following command:

    <PhpStorm> [--line <number>] path

    Alternatively, you can use the following syntax:

    <PhpStorm> path:<number>

The command options are as follows:

Option

Description

<PhpStorm>

The platform-specific PhpStorm launcher.

On macOS and Linux, it is recommended to use the command-line launcher created during the PhpStorm installation, or create one manually by selecting Tools | Create Command-line Launcher on the PhpStorm main menu. Refer to Enable a command-line launcher on macOS/Linux for details.

On Windows, it is recommended to add the path to the PhpStorm executable to the Path environment variable.

path

The path to the file or the folder to be opened.

  • If the file or folder belongs to an existing project, this project will be opened.

  • If the file does not belong to any project, a temporary project for this file will be created automatically.

  • If the folder does not belong to any project, a new project will be created inside this folder.

--line <number>

(optional) The line number, on which the caret should rest after opening the file.

Examples

Syntax
phpstorm.exe --line <number> path
Alternative syntax
phpstorm.exe path:<number>
Examples
phpstorm.exe --line 3 C:\MyProject\scripts\numbers.js phpstorm.exe C:\MyProject\scripts\numbers.js:3
Syntax
phpstorm --line <number> path
Alternative syntax
phpstorm path:<number>
Example
phpstorm --line 3 ~/MyProject/scripts/numbers.js phpstorm ~/MyProject/scripts/numbers.js:3
