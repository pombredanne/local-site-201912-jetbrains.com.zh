源代码导航

您可以使用不同的操作和弹出窗口在编辑器中快速浏览代码。有关在编辑器和工具窗口之间导航的详细信息，请查看编辑器基础知识 。

导航与插入符

  • 要向后导航，请按Ctrl+Alt+Left 。要向前导航，请按Ctrl+Alt+Right 。

  • 要导航到最后编辑的位置，请按Ctrl+Shift+Backspace 。

  • 要在编辑器中找到当前插入符号的位置，请按Ctrl+M 。如果您不想滚动大文件，此操作可能会有所帮助。

    但是，您可以按Up和Down箭头键可以达到相同的效果。

  • 要突出显示您要查找的插入符号处的单词，请选择“ 编辑” |“其他”。查找|从主菜单中的插入符号中找到Word 。

  • 要查看插入符号当前位于什么元素上，请按Alt+Q 。

  • 要在开始和结束代码块大括号之间切换，请按Ctrl+Shift+M 。

  • 要在代码块之间导航，请按Ctrl+Shift+[要么Ctrl+Shift+] 。

移动插入符号

您可以使用不同的动作通过代码移动插入符号。您还可以配置插入字词和在线中断时插入符号应停止的位置。

  • 要将插入符号移动到下一个单词或上一个单词，请按Ctrl+Right要么Ctrl+Left 。

    默认情况下，PhpStorm将插入符号移动到当前单词的末尾。

    当您将插入符号移动到上一个单词时，插入符号将放置在当前单词的开头。使用这些操作时，可以配置插入标记的位置。

    在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般在“ 插入符 移动”部分中，使用“ 按字移动时”和“换行时”选项来配置插入符的行为。

  • 要将插入符号向前移动到下一个段落或向后移动到上一个段落，请按Ctrl+Shift+A并搜索将插入符号向前移动段落或将插入符号向后移动段落动作。

    您也可以选择一个文本，然后将插入符号向前或向后移动到一个段落。按Ctrl+Shift+A并搜索带有选择项的将插入符号向前移动段落或带有选择项的将插入符号向前 移动段落 。

    如果需要，可以为这些操作分配快捷方式。有关详细信息，请参考配置键盘快捷键 。

查找最近的位置

您还可以使用“ 最近的位置”弹出窗口检查您最近查看或更改的代码。

  • 要打开“ 最近的位置”弹出窗口，请按Ctrl+Shift+E 。该列表从顶部的最近访问位置开始，并包含代码片段。

  • 在弹出窗口中，使用相同的快捷方式或选中仅显示更改的复选框，以仅查看更改了代码的位置。

  • 要搜索代码段，请在“ 最近的位置”弹出窗口中，开始输入搜索查询。您可以按代码文本，文件名或面包屑搜索。

  • 要从搜索结果中删除位置条目，请按Delete要么Backspace 。

    请注意，删除的位置也会从您通过以下方式访问的条目列表中删除： Ctrl+Alt+Left捷径。

使用书签进行导航

  • 要创建匿名书签，请将插入符号放在所需的代码行中，然后按F11 。

  • 要创建带有助记符的书签，请将插入号放置在所需的代码行中，按Ctrl+F11并选择一个数字或字母作为助记符。

  • 要显示下一个或上一个书签，请在主菜单中选择“ 导航”。书签|下一个书签或导航|书签|上一个书签 。

  • 要打开“ 书签”对话框，请按Shift+F11 。您可以使用此对话框来管理书签，例如，删除书签，对书签排序或为其提供简短说明。

  • 要导航到带有字母助记符的现有书签，请按Shift+F11然后按一下您需要的字母。PhpStorm将您带回到编辑器和相应的书签。

  • 要导航到带有数字助记符的现有书签，请按Ctrl和书签的编号。

  • 每个创建的书签都会反映在“ 收藏夹”中 Alt+2工具窗口，您还可以使用该窗口导航到书签。

在变更之间导航

如果您编辑的文件受版本控制 ，则PhpStorm提供了几种更新方式来回移动。特别是，您可以使用导航命令，键盘快捷键和更改标记。

要导航到上次编辑的位置，请按Ctrl+Shift+Backspace或选择导航|从主菜单的最后编辑位置 。

  • 按Ctrl+Shift+Alt+Down / Ctrl+Shift+Alt+Up 。

  • 从主菜单中，选择导航|下一个/上一个更改 。

  • 单击更改标记 ，然后单击要么 。

转到声明及其类型

您可以从符号的用法导航到符号的初始声明和符号类型。

  • 将插入符号放在所需的符号上，然后按Ctrl+B 。

    或者，保持Ctrl按下，将鼠标指针悬停在该符号上。当符号变成超链接时，其声明将显示在工具提示中。单击超链接而不释放键，以在编辑器中打开声明。

  • 对于类型声明，请按Ctrl+Shift+B 。

    或者，保持Ctrl+Shift按下，将鼠标指针悬停在该符号上。当符号变成超链接时，其类型声明将显示在工具提示中。单击超链接而不释放键，以在编辑器中打开符号的类型声明。

考虑以下示例代码片段：

类C {}类D {公共函数f（）{返回新的C； $ a = new D; $ a-> f（）;

  • 在上调用“ 转到类型声明” $a会将插入符号移动到变量的类型， class D 。

  • 在页面上调用“ 转到类型声明 ” f方法中$a->f()将插入符号移到此方法返回的类的声明中， class C 。

  • 在屏幕上调用“ 转到声明 ” f方法中$a->f()会将插入符号移到此方法的声明内class D 。

前往实施

您可以使用编辑器中的装订线图标或按相应的快捷方式来跟踪类的实现和重写方法。

  • 单击其中一个 / ， / 装订线图标，然后从列表中选择所需的上升或下降类别。

  • 要导航到超级方法，请按Ctrl+U 。

  • 要导航到实现，请按Ctrl+Alt+B 。

使用“选择范围”弹出窗口进行导航

您可以在“ 项目”工具窗口中自动找到文件。

  1. 如果在编辑器中打开了文件，请按Alt+F1打开“ 选择范围”弹出窗口。

  2. 在弹出窗口中，选择“ 项目视图” ，然后按Enter 。PhpStorm在“ 项目”工具窗口中找到目标。

在“项目”工具窗口中找到文件

您可以使用“ 单击打开文件”和“ 始终选择打开的文件”操作来在“ 项目”工具窗口中找到文件。

  1. 在“ 项目”工具窗口中，右键单击“ 项目”工具栏，然后从上下文菜单中选择“ 始终选择打开的文件” 。之后，PhpStorm将跟踪当前在活动编辑器选项卡中打开的文件，并自动在“ 项目”工具窗口中找到它。

  2. 您也可以选择“ 单击打开文件”选项。在这种情况下，当您在“ 项目”视图中单击文件时，PhpStorm将自动在编辑器中将其打开。

在错误或警告之间导航

  • 要跳至代码中的下一个或上一个发现的问题，请按F2要么Shift+F2分别。或者，从主菜单中选择“ 导航|导航”。下一个/上一个突出显示的错误 。

    PhpStorm将插入符号放置在代码发布之前。

  • 配置PhpStorm在代码问题之间导航的方式：它可以在所有代码问题之间跳转或跳过次要问题，而仅在检测到的错误之间导航。右键单击滚动条区域中的代码分析标记，然后从上下文菜单中选择一种可用的导航模式：

    • 要使PhpStorm跳过警告，信息和其他次要问题，请选择仅转到高优先级问题 。

    • 要使PhpStorm在所有检测到的代码问题之间跳转，请选择转到下一个问题 。

在“结构”视图弹出窗口中找到一个代码元素

您可以使用结构视图弹出窗口在正在处理的文件中找到代码元素。

  1. 要打开结构视图弹出窗口，请按Ctrl+F12 。

  2. 在弹出窗口中，找到您需要的项目。您可以开始为PhpStorm输入元素的名称以缩小搜索范围。按Enter返回到编辑器和相应的元素。

    在弹出窗口中，您可以对文件成员进行排序，查看匿名类和继承的成员。
