进行静态对话框

重构|设为静态

使用此重构将实例方法转换为静态方法。

如果您当前通过以下方式使用现有对象，则对话框将打开this或访问该类的属性。 如果您的方法不需要任何参数，也不依赖于访问此对象的属性或方法，则将以静默方式获得静态方法，而不会显示任何对话框：

之前

后

类MyClass {私有函数getFormattedDate（）{$ format = getSettings（）['dateFormat'];返回时间（$ format）; }}
class MyClass {静态私有函数getFormattedDate（）{$ format = getSettings（）['dateFormat'];返回时间（$ format）; }}

使用名称添加对象作为参数

选中此复选框通过参数传递对象的实例。在下面的字段中，指定要生成的参数的名称。重构之后，新参数将记录在PHPDoc块中。
类MyClass {私有函数getFormattedDate（）{$ format = $ this-> getSettings（）['dateFormat'];返回时间（$ format）; }}
class MyClass {/ ** * @param MyClass $ instance * @return * /静态私有函数getFormattedDate（$ instance）{$ format = $ instance-> getSettings（）['dateFormat'];返回时间（$ format）; }}

为字段添加参数

如果访问类的属性，请使用此区域将属性的值作为参数传递，而不是访问新创建的静态方法内的对象。

  1. 选择添加字段参数复选框。

  2. 在“ 参数”列表中，其中显示了所有可能的建议参数，选中要传递值的复选框旁边的复选框。

类MyClass {私有函数getFormattedDate（）{$ format = $ this-> timeFormat;返回时间（$ format）; }}
class MyClass {/ ** * @param $ timeFormat * @return int; * /静态私有函数getFormattedDate（$ timeFormat）{$ format = timeFormat;返回时间（$ format）; }}
