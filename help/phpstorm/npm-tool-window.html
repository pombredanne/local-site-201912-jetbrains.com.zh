NPM工具窗口

package.json的上下文菜单-显示npm脚本
查看|工具窗口| npm

使用工具窗口运行npm或Yarn脚本。

在你开始前

  1. 下载并安装Node.js。请注意，还安装了npm，因此，如果要使用它，请按照初步步骤操作。

  2. 要使用纱线，请按照纱线官方网站上的说明进行安装。

运行脚本

在“ 项目”工具窗口中选择package.json文件或在编辑器中打开文件，然后从上下文菜单中选择“ 显示npm脚本”时 ，将打开工具窗口。 一旦您调用npm或Yarn，它就会开始构建在scripts对其进行调用的package.json文件的属性。如果项目中有几个package.json文件，则可以为每个文件构建一个单独的脚本树并运行脚本，而无需删除先前构建的树。每个树显示在单独的节点下。

工具窗口显示脚本输出，报告发生的错误，列出未找到的软件包或插件，等等。最后执行的脚本的名称显示在工具窗口的标题栏上。

要构建脚本树，请执行以下任一操作：

  • 在“ 项目”工具窗口中选择所需的package.json文件，或在编辑器中将其打开，然后从上下文菜单中选择“ 显示npm脚本 ”。

  • 在npm工具窗口中，单击在工具栏上，然后从列表中选择所需的package.json文件。默认情况下，PhpStorm在项目的根目录中显示package.json文件。如果还有另一个package.json文件，请单击“选择package.json” ，然后在打开的对话框中选择所需的package.json文件。PhpStorm在标题上添加了一个新节点，并带有指向所选package.json文件的路径，并在新节点下构建了一个脚本树。

重建一棵树

  • 切换到所需的节点，然后单击在工具栏上。

按名称对树中的脚本进行排序

  • 请点击在工具栏上，从菜单中选择“ 排序依据 ”，然后选择“ 名称” 。
    默认情况下，一棵树以在package.json中定义脚本的顺序显示脚本（选项Definition order ）。

运行一个脚本

  • 双击脚本。

  • 在树中选择脚本，然后按Enter或选择运行
