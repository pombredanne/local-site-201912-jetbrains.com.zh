Code Inspection: Call to 'document.write()'

Reports method calls to document.write() or document.writeln().Most uses of these calls are better performed using explicitDOM calls such as getElementByID() and createElement(). Additionally, the write() and writeln() calls will not work with XMLDOMs, including DOMs for XHTML if viewed as XML. This can result in difficulty to point out bugs.
