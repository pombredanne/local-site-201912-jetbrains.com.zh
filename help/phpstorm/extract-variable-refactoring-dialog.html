Ctrl+Alt+V

使用此对话框提取：

  • PHP或JavaScript上下文中的变量。

  • JavaScript 1.7，JavaScript 1.8.5，ES5和ES6的局部变量。

  • JavaScript常数。

变量声明

选择提取（全局）变量，常量或局部变量：

  • extract var-选择此选项以提取全局变量。

  • 提取常数 -选择此选项以提取常数。

  • 提取局部变量 -选择此选项以提取局部变量。

    提取局部变量选项仅适用于JavaScript 1.7，JavaScript 1.8.5，ES5和ES6。

的JavaScript

介绍变量

当您选择提取字符串文字的一部分时，此选项可用。选择如何在文字中引入提取的变量：

  • 通过嵌入 -例如，将使用Complex（curly）语法引入变量"foo {$bar}" 。

  • 通过串联 -例如，将使用串联运算符（'。'）引入变量。 "foo . $bar" 。
