Custom Plugin Repositories Dialog

This dialog opens when you click Manage Plugin Repositories under in the Plugins.

Use this dialog to manage the list of custom plugin repositories. The repositories are identified by their URLs.

Icon

Shortcut

Description

Alt+Insert

Use this icon or shortcut to add a new repository to the list. Specify the URL of the repository in the dialog that opens. Use the Check Now button to make sure that the specified URL is correct: PhpStorm will try to connect to the repository.

Enter

Use this icon or shortcut to edit the selected plugin repository URL.

Alt+Delete

Use this icon or shortcut to remove the selected URL from the list.
