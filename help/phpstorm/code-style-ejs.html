仅在“ 插件”页面上安装并启用了EJS存储库插件后，此页面才可用，有关详细信息，请参阅管理插件 。

在此页面上，配置要在代码的EJS内含物中应用的格式设置选项。PhpStorm会根据“ 设置”对话框的“ EJS”页面上指定的这些定界符来识别此类包含。默认分隔符为<%和%> 。有关使用EJS的更多详细信息，请参见http://embeddedjs.com/getting_started.html 。

描述

execute标签内的空格

选中此复选框可在开口之间插入空格<%和关闭%>定界符和其中的模板主体。否则，模板主体不会与定界符分开。

execute和get转义标记内的空格

选中此复选框，一方面将模板主体与打开/关闭定界符或转义标签之间插入空格。 =在另一。否则，模板主体将紧跟在定界符和标记之后，并在它们之前。

execute和get unscapescape标签中的空格

选中此复选框，一方面将模板主体与打开/关闭定界符或unescape标签之间插入空格。 ==在另一。否则，模板主体将紧跟在定界符和标记之后，并在它们之前。

生成标签内的空格

选中此复选框，一方面可以在模板主体与打开/关闭定界符之间插入空格，也可以生成标签 %在另一。否则，模板主体将紧跟在定界符和标记之后，并在它们之前。

过滤器标签内的空格

选中此复选框，一方面将模板主体与打开/关闭定界符或过滤器标签之间插入空格。 %=:在另一。否则，模板主体将紧随标签之后并在标签之前。

在“：”之前

选择此复选框可在冒号之前插入空格: 。

在“：”之后

选择此复选框可在冒号后插入空格: 。

链“ |”

选中此复选框可在竖线之前和之后插入空格| 。
