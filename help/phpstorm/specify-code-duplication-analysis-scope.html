Specify Code Duplication Analysis Scope

Code | Locate Duplicates

Use this dialog to launch the search for duplicated code fragments in the specified scope.

Item

Description

Whole project

Select this option to perform the analysis for the whole project.

Module <name>

Select this option to have PhpStorm analyze the module that is currently selected in the Project tool window.

File <name>

Select this option to have PhpStorm analyze the file that is currently selected in the Project tool window or opened in the editor.

Custom scope

Select this option to define a custom scope to apply analysis to. Choose a pre-defined scope from the list or click and define a custom scope using the Scopes dialog.

Use a special language to define a scope.

Include Test Sources

Select this checkbox to perform analysis on the test sources.
