安装AMP软件包

AMP软件包特定于操作系统。最常见的是：

  • 适用于Windows的XAMPP 。

  • LAMP软件包与所使用的Linux发行版兼容。

  • 适用于macOS的MAMP 。

安装过程取决于所使用的操作系统，请遵循提供的安装说明。

如果使用Windows Vista，请避免将程序包安装在Program Files文件夹中。默认情况下，此文件夹是写保护的，这意味着不能在服务器上放置任何文件，PHP引擎无法对其进行进一步处理。

安装和配置AMP软件包

  1. 下载并安装所需的AMP软件包。

  2. 使用AMP控制窗格来启动组件。

    如果Web服务器未启动，则很可能发生端口冲突。默认情况下，Apache HTTP服务器侦听端口80。其他服务（例如Skype）已经可以使用此端口。要解决此问题，请按如下所示更新服务器配置文件：

    • 找到线Listen 80并将其更改为例如Listen 8080 。

    • 找到线ServerName localhost:80并相应地进行更改，在本例中为ServerName localhost:8080 。

    保存配置文件，然后重新启动Web服务器。

  3. 要检查安装，请打开浏览器并输入以下URL地址： http：// localhost： 。出现AMP欢迎页面。

将AMP软件包与PhpStorm集成

  1. 如配置本地PHP解释器中所述配置本地解释器 。

  2. 如配置调试引擎中所述安装和配置调试引擎 。

  3. 在描述MySQL服务器配置集成的MySQL 。

教程：将MAMP与PhpStorm集成

MAMP是macOS软件包，包括Apache HTTP服务器，MySQL数据库，PHP解释器和一些库。单词“ MAMP”是首字母缩写词，其中“ M”代表“ Macintosh”，表示它是为macOS设计的，其他字母代表软件包的组件。

MacOS附带了预安装的PHP解释器和Apache服务器，可以将其配置为根据php.net发布的教程一起正常工作。与PhpStorm相关的配置类似于本教程中描述的配置。由于自行安装其他软件包和配置系统环境可能很棘手，因此本指南介绍了如何在具有完整MAMP软件包的单独环境中轻松地一次完成所有设置。

MAMP是为PHP编程设置环境的可靠而快速的方法。它提供了开发，运行，调试和单元测试PHP应用程序所需的所有组件。MAMP是单独安装和配置Web服务器，PHP引擎，数据库服务器和调试引擎的理想选择。要开始开发，您只需下载并安装MAMP，然后使用MAMP控制面板启动组件。

下载并安装MAMP

从“ MAMP下载”页面下载并安装MAMP。下载MAMP存档后，将其解压缩并运行安装程序。MAMP安装向导非常简单，您只需完成安装步骤即可。

安装MAMP并启动其组件后，我们现在需要告诉IDE这些组件的存储位置和配置方式。

MAMP控制面板

安装后，使用MAMP控制面板执行诸如启动或停止服务器以及更改配置之类的操作。可以在Applications / MAMP文件夹中找到MAMP控制面板应用程序。

启动和停止服务器

要启动Apache和MySQL服务器，请单击“ 启动服务器”按钮。

右上角的图标将变为绿色。服务器启动后，您可以随时单击“ 停止服务器”按钮来停止它们。

起始页 （默认为http：// localhost：8888 / MAMP / ）将在浏览器中打开。该页面包含有关您的环境的信息，并提供有用的链接。

要配置MAMP，请单击“ 首选项...”按钮。

有几个选项：

  • 在“ 启动/停止”选项卡上修改“启动/停止服务器”配置。

  • 在“ 端口”选项卡上指定Apache和MySQL端口。

  • 在“ PHP”选项卡上指定PHP版本和缓存选项。

  • 在“ Web 服务器”选项卡上设置Web服务器文档的根目录。

将MAMP与IDE集成

集成PHP可执行文件

首先，从PhpStorm中的MAMP注册PHP可执行文件。 在里面 设置/首选项对话框Ctrl+Alt+S ，导航到语言和框架| PHP 。

借助PhpStorm，我们可以在IDE中注册多个单独的PHP解释器，具体取决于项目所需的PHP版本或PHP配置。CLI解释器列表中显示所有当前注册的PHP解释器安装：

添加随MAMP安装的解释器

  1. 请点击在列表的右边。“ CLI解释器”对话框打开。

  2. 在左侧窗格中，单击工具栏上的按钮。

  3. 在名称字段中，输入名称以标识当前安装，例如MAMP PHP 7.1 。

  4. 在“ PHP可执行文件”字段中，指定存储php （PHP可执行文件）文件的文件夹。手动输入路径或单击并找到文件。

  5. PhpStorm检查指定的文件夹是否包含PHP可执行文件，检测PHP版本，并将其显示在PHP Info只读字段中。PhpStorm还可以验证调试器的配置是否正确。

现在，我们可以保存这些设置，并为我们的项目选择新添加的PHP解释器：

PhpStorm现在将使用与MAMP捆绑在一起的PHP安装。

集成调试器

PhpStorm提供了功能强大的调试器，可以在其中设置断点并可以在运行时检查变量。有关使用PhpStorm调试器的更多信息，请浏览使用PhpStorm调试：终极指南 。

请注意，如果您使用的是最新的MAMP PRO，则应通过MAMP PRO提供的模板编辑器来编辑httpd.conf ， php.ini和my.cnf文件。要编辑模板文件，请导航至“ 文件” | “模板” 。编辑模板 。当MAMP PRO启动Apache和MySQL服务器时，它将使用这些模板来创建httpd.conf和php.ini文件，从而删除先前创建的文件。

安装并配置调试器后，可以通过执行“ 验证调试引擎的配置”中介绍的步骤来验证调试配置。

集成Apache服务器

PhpStorm与Web，FTP和其他服务器之间的交互由FTP / SFTP Connectivity插件处理，该插件默认启用。IDE使用注册的服务器访问配置中指定的连接设置访问服务器。

创建和管理服务器访问配置

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，单击部署下建立，执行，部署 。

  2. 点击在工具栏上，选择Local或Mounted文件夹 ，然后在打开的对话框中指定MAMP的名称。

  3. 单击OK之后 ，我们可以指定Apache的Web根目录/ Applications / MAMP / htdocs的路径以及Web服务器的URL http：// localhost：8888 。请注意，实际设置取决于您在MAMP配置或Apache配置文件中配置MAMP Apache服务器的方式。

    不要忘记通过单击将服务器标记为默认服务器 ，以便PhpStorm将其与浏览器集成而不是内置的Web服务器一起使用 。通过单击“ 打开” ，我们可以在浏览器中打开MAMP htdocs文件夹，从而验证提供的设置是否正确。

集成MySQL服务器

PhpStorm允许使用数据库工具和SQL插件（默认情况下已安装并启用）连接到MAMP MySQL数据库服务器。在开始之前，请确保MySQL服务器正在MAMP PRO控制面板上运行。如果您使用的是MAMP PRO，还请确保在MAMP PRO的“控制面板”上选中了“ 允许对MySQL的网络访问”复选框。

在IDE的右侧，打开“ 数据库”工具窗口。

在这里，我们可以使用上下文菜单或工具栏按钮创建新的连接或数据源 。我们需要指定将用于连接到数据库的JDBC数据库驱动程序。对于MySQL，我们可以使用驱动程序列表中提供的MySQL Connector驱动程序。PhpStorm不提供这些驱动程序，但是我们可以通过单击链接下载正确的JDBC驱动程序。

接下来，我们需要输入连接详细信息。或者，我们可以在数据库URL字段中提供与MySQL数据库的连接字符串，例如jdbc：mysql：// localhost：8889 /或jdbc：mysql：// localhost：8889 / foodb以连接到foodb数据库。

我们还可以指定MySQL的用户名和密码。默认情况下，MAMP使用root作为用户名和密码。

在用户名，密码或数据库名称中使用UTF-8或Unicode字符时，必须在连接字符串中添加两个附加参数： characterEncoding和useUnicode 。一个示例连接字符串为jdbc:mysql://localhost:8889?characterEncoding=utf8&useUnicode=true 。可能还需要从列表中选择其他JDBC驱动程序，即， com.mysql.jdbc.NonRegisteringDriver 。

定义数据源后，使用“ 测试连接”按钮来验证设置，并确保PhpStorm可以连接到MySQL。

现在，我们可以创建表，运行查询，更新数据等等。有关更多信息，请参见数据库工具和SQL 。

教程：将XAMPP与PhpStorm集成

XAMPP是一个跨平台程序包，由Apache HTTP服务器，MySQL数据库，PHP解释器和Perl解释器组成。单词“ XAMPP”是首字母缩写词，其中“ X”代表“ cross”，意思是“跨平台”，其他字母代表包装组件。

XAMPP是为PHP编程设置环境的可靠而快速的方法。它提供了开发，运行，调试和单元测试PHP应用程序所需的所有组件。XAMPP是单独安装和配置Web服务器，PHP引擎，数据库服务器和调试引擎的不错选择。要开始开发，您只需下载并安装XAMPP，然后使用XAMPP控制面板启动组件。

下载并安装XAMPP

从“ XAMPP下载”页面下载并安装XAMPP。下载XAMPP存档后，将其解压缩并运行安装程序。XAMPP安装向导非常简单，只需执行安装步骤即可。

安装XAMPP并启动其组件后，我们现在需要告诉IDE这些组件的存储位置和配置方式。

XAMPP控制面板

安装后，使用XAMPP控制面板执行诸如启动或停止服务器以及更改配置之类的操作。

启动和停止服务器

要启动特定服务器，请单击其旁边的“ 开始”按钮。

默认情况下，没有任何服务器（例如Apache或MySQL）安装为Windows服务。这意味着每当我们要使用这些服务器时，都必须通过XAMPP控制面板手动启动它们。当服务器作为Windows服务运行时，它将自动与Windows一起运行。

将特定服务器作为Windows服务运行

  1. 以管理员身份打开XAMPP控制面板。

  2. 通过单击“ 停止”按钮来停止要作为Windows服务安装的服务器。

  3. 单击服务器旁边的X ，然后确认服务安装。安装成功后，服务器旁边将显示一个绿色的选中标记，表明服务器现在正在作为Windows服务运行。
