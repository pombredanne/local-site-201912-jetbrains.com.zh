基本编辑

功能

捷径

使用此快捷方式可以...

输入

 Enter 根据上下文：

  • 在查找列表中：选择一个项目。

  • 在编辑器中：输入新行，并在其开始处设置脱字号。

标签

 Tab 在编辑器中：

  • 有选择：缩进选定的行。

  • 如果没有选择：插入一个制表符（或相应数量的空格字符）。

在查找列表中：

  • 插入符号后无代码：选择一个项目。

  • 尖号后面的一些代码：选择一个项目，然后用尖号替换该代码。

删除

 Delete 根据上下文：

  • 在编辑器中：删除选定的符号/块。

  • 在使用情况视图中：排除选定的项目。

  • 在其他视图中：删除选定的项目。

退格键

 Backspace

删除插入符号左侧的字符。

撤消

 Ctrl+Z

撤消上一个操作。

重做

 Ctrl+Shift+Z

重做上次撤消的操作。

切

 Ctrl+X

剪切当前行或选定的代码块到剪贴板。

复制

 Ctrl+C

将当前行或选定的代码块复制到剪贴板。

糊

 Ctrl+V

从剪贴板粘贴到插入符号位置。

从历史粘贴

 Ctrl+Shift+V

将选定的条目从剪贴板粘贴到插入符号位置。

上

 Up

将插入符号向上移动一行。

选择最多

 Shift+Up

将插入符号向上移动一行以选择文本。

下

 Down

将插入符号向下移动一行。

选择下来

 Shift+Down

将插入符号向下移动一行以选择文本。

剩下

 Left

将插入符号向左移动一个字符。

左选

 Shift+Left

将插入符号向左移动一个字符以选择文本。

对

 Right

将插入符号向右移动一个字符。

选择权

 Shift+Right

将插入符号向右移动一个字符以选择文本。

转到页面底部

 Ctrl+Page Down

将插入符号向下移动到页面底部。

转到选择的页面底部

 Ctrl+Shift+Page Down

将插入符号向下移动到页面底部，选择文本。

转到页面顶部

 Ctrl+Page Up

将插入符号移动到页面顶部。

转到带有选择的页面顶部

 Ctrl+Shift+Page Up

将插入符号上移到页面底部，选择文本。

向下翻页

 Page Down

将插入符号向下移动一页。

向下翻页并选择

 Shift+Page Down

将插入符号向下移动一页，选择文本。

向上翻页

 Page Up

将插入符号向上移动一页。

选择上一页

 Shift+Page Up

将插入符号向上移动一页，选择文本。

向下滚动

 Ctrl+Down

向下滚动文本一行。

滚动到中心

 Ctrl+M

将插入符处的一行滚动到屏幕中心。

向上滚动

 Ctrl+Up

向上滚动文本一行。

移至行尾

 End

将插入符号移动到行尾。

移至选择的行尾

 Shift+End

将插入符号移动到行尾，选择文本。

移至生产线起点

 Home

将插入符号移动到行的开头。

移至生产线从选择开始

 Shift+Home

将插入标记移动到行的开头，选择文本。

移至下一个字

 Ctrl+Right

将插入符号移到下一个单词。

移至带有选择的下一个单词

 Ctrl+Shift+Right

将插入符号移动到下一个单词，然后选择它。

移至上一个字

 Ctrl+Left

将插入符号移动到上一个单词。

移至带有选择的上一个单词

 Ctrl+Shift+Left

将插入符号移动到上一个单词，然后选择它。

移至文字结尾

 Ctrl+End

将插入符号移动到文本的末尾。

移至以选择结尾的文本

 Ctrl+Shift+End

将插入符号移动到文本的末尾，然后选择它。

移至文字开始

 Ctrl+Home

将插入符号移动到文本的开头。

移至“从选择开始”。

 Ctrl+Shift+Home

将插入符号移动到文本的开头，然后选择它。

全选

 Ctrl+A

选择在编辑器中打开的整个文本。

删除插入符号处的行

 Ctrl+Y

删除插入符号当前所在的行。

删除到Word End

 Ctrl+Delete

从当前插入符号位置到单词结尾删除单词。

删除到Word Start

 Ctrl+Backspace

从当前插入符号位置到单词start删除单词。

切换插入/覆盖

 Insert

切换插入/覆盖模式。

重复的行或块

 Ctrl+D

复制所选块或插入符号处的行。

拨动套

 Ctrl+Shift+U

切换所选文本块的大小写。

移至代码块末尾

 Ctrl+]

将插入符号移动到当前代码块末端，突出显示块限制。

选择后移至代码块结尾

 Ctrl+Shift+]

将插入符号移动到当前代码块末尾，从初始插入符号位置选择代码。

移至代码块开始

 Ctrl+[

将插入符号移动到当前代码块的开头，突出显示块限制。

从选择开始移至代码块

 Ctrl+Shift+[

将插入符号移动到当前代码块的开头，从初始插入符号位置选择代码。

开始新线

 Shift+Enter

根据当前的缩进级别，在当前的插入标记位置之后，开始新的一行。

在当前行之前开始新行

Ctrl+Alt+Enter

在当前行之前开始新行。

连接线

 Ctrl+Shift+J

将选定的行连接为一个，或将插入符号当前所在的行与下一行连接。

分割线

 Ctrl+Enter

在插入记号所在的位置分割选定的行，将插入记号留在第一行的末尾。

在插入符号处选择单词

 Ctrl+W

从当前插入符号位置开始，选择连续增加的代码块。

在插入符号处取消选择单词

 Ctrl+Shift+W

依次删除该操作所做的选择。

缩进选择

 Tab

将选定的块移动到下一个缩进级别。

缩进选择

 Shift+Tab

将选定的块移动到上一个缩进级别。

自动缩进线

 Ctrl+Alt+I

根据代码样式设置缩进当前行或所选块。
