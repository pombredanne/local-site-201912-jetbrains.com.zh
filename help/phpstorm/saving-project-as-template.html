将项目另存为模板

  • 介绍
  • 创建项目模板
  • 删除项目模板

介绍

您可以将项目另存为模板。如果这样做，您将能够创建基于模板的项目，即与模板项目几乎相同的项目。

您另存为模板的项目将出现在“ 创建新项目”对话框的“ 用户定义”类别中。

可以在“ 管理项目模板”对话框中删除不必要的模板。

创建项目模板

  1. 在PhpStorm中打开所需的项目。

  2. 在主菜单中，选择“ 工具” |“工具”。将项目另存为模板 。

  3. 在打开的项目另存为模板对话框中：

    • 指定模板名称，并可以选择提供简短的模板描述。

删除项目模板

  1. 在主菜单中，选择“ 工具” |“工具”。管理项目模板 。

  2. 在打开的“ 管理项目模板”对话框中，选择要删除的一个或多个模板。

  3. 请点击或按Alt+Delete 。
