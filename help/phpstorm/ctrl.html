PhpStorm 2019.3帮助

Ctrl

本节列出并描述了键盘快捷方式，包括Ctrl键：

  • Ctrl +字母数字键
  • Ctrl +导航键
  • Ctrl +符号键
  • Ctrl +数字键
  • Ctrl +功能（F）键

Ctrl +字母数字键

捷径

功能

使用此快捷方式可以...

Ctrl+A

全选

在活动编辑器中选择整个文本。

Ctrl+B

导航到宣言

可以从任何用法直接导航到元素的声明。

Ctrl+C

复制

将选定的文本复制到剪贴板。

Ctrl+D

重复的行或块

在插入符处复制选定的块或行。

Ctrl+E

最近的文件
最近发现的用法

显示最近打开的文件列表。

当“ 查找”工具窗口具有焦点时，使用此快捷方式显示最近查找用法结果的列表。

Ctrl+F

找

在编辑器中启动文本搜索 。

Ctrl+G

导航到行

导航到当前文件中具有指定编号的行。

Ctrl+H

类型层次结构

浏览所选类的层次结构。

Ctrl+J

插入实时模板

显示以指定前缀开头的实时模板的列表。

Ctrl+M

滚动到中心

将插入符处的一行滚动到屏幕中心。

Ctrl+N

导航到课程

跳转到项目中具有指定名称的类。

Ctrl+O

替代方法

覆盖当前类中的基类方法。

Ctrl+P

参数信息

在插入符号处显示方法调用的参数。

Ctrl+Q
Alt+Mouse Button2

快速文档

在插入符号处显示包含该符号文档的弹出窗口。

在数据库工具窗口中 ：显示一个弹出窗口 ，其中显示create table在插入符号和表的前10行中查询数据库表。

Ctrl+R

更换

调用替换文本对话框。

Ctrl+S

保存全部

保存所有文件和设置。

Ctrl+U

导航到超级方法

导航到尖号处方法的超级方法声明

Ctrl+V

糊

从剪贴板粘贴。

Ctrl+W

在插入符号处选择单词

从插入号处的单词开始，连续选择扩展的文本块。（重复使用此快捷方式来选择表达式。）

Ctrl+X

切

剪切到剪贴板。

Ctrl+Y

删除插入符号处的行

从当前插入符号位置到单词结尾删除一个单词。

Ctrl+Z

撤消

撤消上一个操作。

Ctrl+Shift+Z

重做

重做上次撤消的操作。

Ctrl+Number

导航到书签

导航到带有相应编号的带编号的书签。

Ctrl +导航键

捷径

功能

使用此快捷方式可以...

Ctrl+Tab

切换台

在编辑器中打开的文件和工具窗口之间导航。

Ctrl+Backspace

删除到Word Start

从当前插入符号位置删除单词，直到单词start。

Ctrl+Delete

删除到Word End

从当前插入符号位置到单词结尾删除一个单词。

Ctrl+End

移至文字结尾

将插入符号移动到文本的末尾。

Ctrl+Enter

分割线或未清项目

根据上下文：

  • 在编辑器中：智能地将当前行分为2行，并根据需要移动引号等。

  • 在“工具”窗口中：分别打开一个或多个选定项目的“编辑器”选项卡。

Ctrl+Home

移至文字开始

跳转到文本的开头。

Ctrl+C

复制

将当前行或选定的代码块复制到剪贴板。

Ctrl+Space

基本代码完成

任何类，方法或变量的完整代码。

Ctrl+Page Down

导航到页面底部

将插入符号向下移动到页面底部。

Ctrl+Page Up

导航到页面顶部

将插入符号移动到页面顶部。

Ctrl+Down

向下滚动

将插入符下移一行，以保持句法正确性。

Ctrl+Left

移至上一个字

将插入符号移动到上一个单词。

Ctrl+Right

移至下一个字

将插入符号移到下一个单词。

Ctrl+Up
Ctrl+Shift+Up

向上滚动

将插入符处的行向上移一位，以保持句法正确性。

Ctrl + Ctrl + Home / End

从插入符号位置到当前行的开头/结尾选择文本。

Ctrl +符号键

捷径

功能

使用此快捷方式可以...

Ctrl+[

移至代码块开始

将插入符号移动到当前代码块的开头，突出显示其限制。

Ctrl+]

移至代码块末尾

将插入符号移动到当前代码块的末尾，突出显示其限制。

Ctrl+/
Ctrl+Numpad/

带线注释

注释/取消注释当前行或带有行注释的选定块。

Ctrl+=
Ctrl+NumPad Plus

展开全部

展开所有折叠块。

Ctrl+NumPad -

全部收缩

折叠所有折叠块。

Ctrl +数字键

捷径

功能

使用此快捷方式可以...

Ctrl+Numpad/
Ctrl+/

带线注释

注释/取消注释当前行或带有行注释的选定块。

Ctrl+NumPad Plus
Ctrl+=

展开全部

展开所有折叠块。

Ctrl+NumPad -

全部收缩

折叠所有折叠块。

Ctrl +功能（F）键

捷径

功能

使用此快捷方式可以...

Ctrl+F1

错误说明

在插入符号处显示错误或警告说明。

Ctrl+F3

在插入符号中查找单词

在编辑器中搜索插入符号当前所在的单词。

Ctrl+F6

更改方法签名

重构选定的方法签名并更新所有引用。

Ctrl+F7

在文件中查找用法

开始搜索用法 。

Ctrl+F8

切换断点

在插入符处切换断点。

Ctrl+F9

制作项目

编译项目中所有修改的文件和从属文件。

Ctrl+F11

用助记符切换书签。

打开或关闭带有助记符的书签。

Ctrl+F12

文件结构弹出

在“文件结构”弹出窗口中显示当前文件结构，以进行快速导航。
