代码检查：使用'require（）'代替'import'

何时报告require()用于并帮助替换require()致电import声明。

在内部作用域内启用“ 全部修复”操作来转换Convert require（）以转换所有require()使用“ 全部修复”操作时，将在嵌套函数和语句内调用。

请注意，转换require()内部作用域内的语句import语句可能会导致代码语义的更改。导入语句是静态模块依赖项，并且已被提升，这意味着它们已移至当前模块的顶部。 require()动态调用加载模块。它们可以有条件地执行，并且它们的范围由使用它们的表达式定义。
使用“全部修复”操作清除内部作用域内的Convert require（），以防止在使用“全部修复”操作时在这些复杂情况下进行任何更改。
