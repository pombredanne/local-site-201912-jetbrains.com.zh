项目

描述

括号前

  • 数组初始化括号

    选中此复选框可在数组声明的左括号之前插入一个空格，例如：

    $ array = array（0 =>“零”，1 =>“一”）;

    如果清除该复选框，则不会插入任何空格，并且代码如下所示：

    $ array = array（0 =>“零”，1 =>“一”）;

周围运营商

  • 级联

    选择此复选框可在串联运算符之前和之后插入空格.因此代码如下所示：

    回声“结果是”。 $ i;

    清除复选框后，串联操作符周围不会插入空格，因此代码如下所示：

    echo“结果为”。$ i;

  • 对象访问运算符->

    选中此复选框可在对象访问运算符之前和之后插入空格->因此代码如下所示：

    $ obj-> foo（）-> bar（）;

    清除复选框后，在对象访问运算符周围不会插入空格，因此代码如下所示：

    $ obj-> foo（）-> bar（）;

  • 空合并运算符（??）

    选中此复选框可在空合并运算符之前和之后插入空格??因此代码如下所示：

    清除复选框后，在空合并运算符周围不会插入空格，因此代码如下所示：

    foo（）??酒吧（）;

  • 声明语句中的赋值

    选中此复选框可在工作区中的赋值运算符之前和之后插入空格。 declare语句，因此代码如下所示：

    清除复选框后，赋值运算符周围不会插入空格，因此代码如下所示：

左括号之前

  • 类左括号

    选中此复选框可在类名称和类声明中的大括号之间插入一个空格：

    清除复选框后，不会插入任何空间：

    类Class1 {函数Foo（）}

    §仅当在“ 包装和花括号”选项卡上将类声明中的“花括号”位置设置为“行尾”时，才选择或清除复选框。

中

选中此部分中的复选框可在所选代码代码结构中自动插入空格。

  • 括号

    如果选中此复选框，则始终插入空格。如果清除该复选框，则永远不会插入空格。

  • 变量/表达式周围的括号

    选中此复选框可使PhpStorm仅在方括号包含变量或表达式时才在重新格式化期间在方括号内插入空格。仅当您未通过选择“ 空间”选项卡中“ 内部”节点下的“ 括号”复选框配置未在括号内强制插入空格时，此设置才会影响重新格式化。

    此选项可帮助您使代码保持与WordPress PHP编码标准一致 。

    如果清除此复选框，则不管括号内内容的类型如何，空格插入策略均取决于“ 括号”设置。

在三元运算符中?:

选中此部分中的复选框以在之前和之后自动插入空格? ， 之前和之后: ，以及之间?和:在三元（条件）运算符中 。

其他

选择此部分中的复选框以在逗号，分号，一元NOT运算符之前和之后自动插入空格! ，以及类型强制转换之后。

  • 类型转换后

    当选择该复选框，PhpStorm自动插入的右括号之后的空间中浇铸 ： $fst = (string) $foo;

    如果清除该复选框，则不会插入任何空格，并且强制转换的变量会保留在强制转换上： $fst = (string)$foo;

使用此区域可以创建新的匹配规则或编辑现有规则。您可以从以下过滤器中进行选择：

  • 类型 ：使用此过滤器为规则选择构造函数，方法，字段或属性。
    请注意，两次单击类型关键字会否定条件。

  • 名称 ：使用此字段在规则中指定条目名称。此过滤器仅匹配条目名称，例如字段名称，方法名称，类名称等。该过滤器支持正则表达式并使用标准语法 。对全名进行匹配。

  • 顺序 ：使用此列表选择规则的排序顺序。当多个元素使用相同的匹配规则时，此选项很有用。在这种情况下，选择“ 保留顺序”将保持与重新排列前相同的顺序，选择“按名称排序”将按名称对具有相同匹配规则的元素进行排序。

从订单列表中选择按名称订购时，将显示此图标。图标表示此规则中的项目按字母顺序排序。
