检查

描述

默认严重性

未解决的参考

此检查检查XML语法中的RELAX-NG文件中是否有未解决的对命名模式（“定义”）的引用。对于未解析的引用，提供了QuickFix来创建相应的define-element。

由RELAX-NG支持提供支持

错误

未使用的定义

此检查检查RELAX-NG文件（XML或紧凑语法）中是否有未使用的命名模式（“定义”），并将其突出显示为未使用的符号。相对于来自其他文件的包含来完成此操作，即，如果定义通过另一个文件中的包含来使用，则该定义不会标记为未使用。

由RELAX-NG支持提供支持
