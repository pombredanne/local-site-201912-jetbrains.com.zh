创建新项目：PHP设置

档案|现有文件中的新项目-Web服务器在本地安装，源文件位于其文档根目录下

如果在向导的“ 创建新项目：指定Web路径（本地）”页面上选中了“ 查看PHP设置”复选框，则会打开该页面。使用此页面可确保在指定的本地Web服务器上安装了PHP引擎，并配置了包含路径的列表。该页面显示所选本地Web服务器上安装的PHP引擎版本。如果未检测到PHP引擎，则PhpStorm将显示相应的警告。

包括服务器上的路径

使用此区域中的控件来管理服务器上包含路径的列表：

  • 添加 -单击此按钮可将新条目添加到列表中。

  • 删除 -单击此按钮可从列表中删除所选条目。

  • 浏览 -单击此按钮在相应的对话框中选择所需的文件夹。
