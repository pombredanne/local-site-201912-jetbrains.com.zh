编辑项目路径映射对话框

当您配置可通过SFTP，Vagrant，Docker或WSL访问的远程解释器时，对话框将打开，然后单击 PHP页面上“ 路径映射”字段旁边的的 设置/首选项对话框Ctrl+Alt+S 。

映射是从相应的SFTP部署配置Vagrantfile或Dockerfile中自动检索的，并在对话框中列出。这些映射是只读的。

  • 要添加自定义映射，请单击并分别在“ 本地路径”和“ 远程路径”字段中指定项目中的路径以及远程运行时环境中的相应路径。手动输入路径或单击然后在打开的对话框中选择相关文件或文件夹。

  • 要删除自定义映射，请在列表中选择它，然后单击 。
