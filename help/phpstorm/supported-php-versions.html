PhpStorm 2019.3帮助

支持的PHP版本

借助PhpStorm，您可以在 PHP 5.3 ， PHP 5.4 ， PHP 5.5 ， PHP 5.6 ， PHP 7 ， PHP 7.1 ， PHP 7.2 ， PHP 7.3和PHP 7.4 。本主题概述了每种语言版本可用的编码帮助功能。您可以在语言和框架|设置项目的语言级别。 PHP页面的 设置/首选项对话框Ctrl+Alt+S 。

项目中使用的PHP版本与语言级别之间没有相关性。尽管会自动检测每个解释器的语言版本，但是您仍然可以告诉PhpStorm为您提供与不同语言级别相对应的编码帮助。但是，如果您尝试使用指定语言级别不支持的代码构造，则PhpStorm建议您切换到PHP 快速修复 。

为了能够在PhpStorm中运行PHP代码，您需要配置本地或远程解释器。
