PhpStorm 2019.3帮助

JSON格式

JSON格式通常用于存储数据和配置文件。PhpStorm可帮助您处理JSON文件-检查其语法和格式。在流行的配置文件类型中，由于JSON Schema ，PhpStorm提供了代码完成功能，这是一种用于描述此类文件的结构和内容的特殊格式。您还可以使用自定义JSON模式来启用JSON文件中的代码完成并对其进行验证。

启用JSON5

PhpStorm可以识别许多最受欢迎的JSON标准，包括JSON5 。通过与json5扩展作为JSON5文件，并支持他们这种新语法的默认文件对待PhpStorm。

将JSON5语法扩展到所有JSON文件

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|文件类型 。将打开“ 文件类型”页面。

  2. 在“ 公认的文件类型”列表中，选择JSON5 。

  3. 在“已注册模式”区域中，单击和类型*.json在打开的“ 添加通配符”对话框中。

使用JSON模式存储中的模式

PhpStorm可以从JSON模式存储中自动下载和使用模式，该模式存储许多流行配置文件的模式文件。一旦打开一个名称与可用模式之一（例如tslint.json ）相关联的文件，PhpStorm就会下载并使用该模式。应用的架构的名称显示在状态栏上。

如果您的配置文件具有自定义名称，请在状态栏上单击“ 无JSON模式” ，然后从列表中选择所需的模式，或单击“ 新建模式映射”以打开“ JSON模式映射”页面并配置新的自定义模式 。

JSON模式存储中的模式也可以应用于YAML文件。

默认情况下，启用从JSON模式存储自动下载模式。如果由于某种原因将其关闭，则可以随时再次启用它。

从JSON模式存储启用自动下载模式

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” |模式和DTD |远程JSON模式 。

  2. 选中“ 允许从远程源下载JSON模式”，然后选中“ 使用schemastore.org JSON模式目录”复选框。

PhpStorm捆绑有许多流行的模式。尽管这些架构会定期自动更新，但它们可能仍会过时。

使用捆绑模式的最新版本

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” |模式和DTD |远程JSON模式 。

  2. 选中始终下载最新版本的架构复选框。

使用自定义JSON模式

除了JSON模式存储中的模式外，PhpStorm还允许您配置和使用其他存储中的自定义模式。您可以下载所需的架构并将其存储在项目根目录下，或者指定资源的URL，以便PhpStorm可以自动下载架构。

自定义架构必须符合JSON架构标准 。

配置自定义JSON模式

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” |模式和DTD | JSON模式映射 。

  2. 在显示您先前配置的所有自定义架构的中央窗格中，单击在工具栏上。

  3. 指定架构名称和架构所遵循的架构规范版本 。在“ 架构文件”或“ URL”字段中，指定以前下载的架构文件的位置，或键入可提供所需架构的URL。

    如果指定URL，请确保选中“ 远程JSON模式”页面上的“ 允许从远程源下载JSON模式”复选框。

  4. 创建要针对此架构进行验证的文件或文件夹的列表。该列表可能包含特定文件的名称，整个目录的名称和文件名模式。根据列表，PhpStorm在内部检测要验证的文件。

    要将项目添加到列表中，请单击并指定文件或文件夹的路径或键入文件模式。

PhpStorm仅在当前项目中搜索具有指定名称的文件和文件夹，因此您无需指定文件和文件夹的完整路径。

启用从远程源自动下载JSON模式

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” |模式和DTD |远程JSON模式 。

  2. 选择“ 允许从远程源下载JSON模式” 。

    清除复选框后，围绕JSON Schema的任何网络活动（包括JSON Schema Store中的模式）都将被禁用。

处理架构范围之间的冲突

当文件，文件夹或模式属于两个或多个模式的范围时，就会发生冲突。PhpStorm以两种模式分析范围：

  • 静态分析会检测自定义架构范围内的冲突。如果检测到冲突，则PhpStorm在“ 架构详细信息”窗格中显示警告。要查看重叠的范围，请单击显示详细信息链接。PhpStorm显示带有消息的弹出窗口，其中列出了冲突的作用域和架构：

  • 动态分析检测系统和自定义架构的范围中的冲突。当您打开属于某个范围的文件时，便开始这种类型的分析。如果检测到冲突，PhpStorm将在编辑器选项卡的顶部显示警告：

    单击链接以打开“ JSON架构映射”页面，然后编辑冲突的自定义架构的范围。请注意，您无法编辑系统架构的范围。

配置语法突出显示

您可以根据自己的喜好和习惯来配置JSON感知语法突出显示。

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|配色方案| JSON 。

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
