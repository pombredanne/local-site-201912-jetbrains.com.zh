Command Line Tool Support: Tool Settings

File | Settings | Tools | Command Line Tool Support for Windows and Linux
PhpStorm | Preferences | Tools | Command Line Tool Support for macOS

The dialog opens when you click and choose Custom tool in the Command Line Tools dialog. Use the dialog to configure Custom tool support in PhpStorm.

The tool works only with local PHP interpreters.

Item

Description

Tool path

Specify the location of the tool executable file.

Alias

Type the character string to use in command calls instead of the full path to the tool.
