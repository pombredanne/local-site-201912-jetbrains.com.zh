创建PHP类

PhpStorm假定您的项目中的名称空间的排列符合PSR0标准，并在创建PHP类时强制您根据此标准保留文件夹结构和名称空间层次结构。

当您指定将存储用于实现新类的文件的文件夹时，PhpStorm会自动更新该类的名称空间。

创建PHP类

  1. 请执行以下任一操作：

    • 在项目工具窗口中Alt+1 ，选择要在其中创建新类的目录，然后选择“ 文件” |“目录”。主菜单上的新功能。

    • 右键单击相应的目录，然后从上下文菜单中选择“ 新建 ”。

    • 按Alt+Insert 。

  2. 选择PHP类 。将打开“ 创建新的PHP类”对话框。

  3. 在名称字段中，输入要创建的类的名称。PhpStorm会自动在“ 文件名”字段中填写指定的名称。

  4. 指定用于创建类的名称空间。默认情况下，“ 名称空间”字段显示与从中调用类创建的文件夹相对应的名称空间。

    你可以选择从列表中删除或手动更改名称空间。请记住，这可能会导致加载问题，因为这样的命名空间层次结构将不符合PSR0标准。

    您还可以更改文件夹以存储将实现新类的文件，更改也将反映在“ 名称空间”字段中。

  5. 在“ 模板”列表中，指定用于创建文件的模板。可用选项如下：

    • PhpStorm捆绑模板集：

      • 类
      • 接口
      • 特征

    • 您自己的一组具有PHP文件扩展名的手动创建的文件模板。

      要将此类模板设置为默认模板 ，请选中“ 用作默认模板”复选框。下次调用新类的创建时，将自动选择默认模板。

      有关详细信息，请参考文件和代码模板 。

  6. 从列表中选择文件扩展名。

当您单击OK时 ，将根据所选模板创建一个新类，并自动添加指定的名称空间声明。
