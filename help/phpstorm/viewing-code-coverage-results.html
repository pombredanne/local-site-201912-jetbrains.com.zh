查看代码覆盖率结果

查看代码覆盖率可帮助您检测不受仿真影响的部分源代码。

先决条件

  1. PHPUnit工具已安装在您的计算机上，并已在PhpStorm中启用 。

  2. 在当前的PHP解释器中配置了Xdebug或Zend Debugger调试引擎。

查看代码覆盖率结果

  1. 创建一个测试运行/调试配置 ，在“ 运行/调试配置”列表中选择它，然后单击“覆盖运行”按钮 。

  2. 点击工具|显示代码覆盖率数据 。或者，按Ctrl+Alt+F6 。

  3. 在“ 选择要显示的覆盖套件”对话框中，选中要查看的套件旁边的复选框，然后单击“ 显示所选按钮”。

  4. 在专用的“ 覆盖率”工具窗口中查看覆盖率结果。

  5. 按照执行测试部分中的描述创建缺少的测试 。
