CSS中的代码检查

检查

描述

默认严重性

无效的CSS属性值

此检查将检测非法财产的价值。

错误

属性可以用速记安全地替换

如果CSS规则集包含完整的一组速记属性，则此检查建议将其转换为速记形式。

例如，4个属性： padding-top ， padding-right ， padding-bottom和padding-left可以安全地替换为一个padding属性。

请注意，如果一组常规属性不完整（例如仅3个），则不会显示此检查padding-xxx规则集中的属性），因为切换到速记可能会更改结果。在这种情况下，请考虑“可能用简写形式代替财产”检查。

弱警告

缺少逗号

此检查检测到行尾选择器之间缺少逗号。

警告

缺少度量单位

此检查突出显示未指定度量单位的CSS编号属性值。

警告

未解析的CSS自定义属性

此检查警告有关CSS定制属性变量引用的信息，这些引用无法解析为任何有效目标。

错误

无效的CSS媒体功能

此检查将检测未知的介质功能和非法的介质功能值。

错误

缺少通用字体名称

此检查验证CSS font-family属性包含通用字体名称作为备用替代。

警告

未知的CSS属性

突出显示未知或在错误上下文中使用的CSS属性，并允许将此类属性定义为“ custom”，以避免突出显示为无效属性。

警告

属性可能用速记代替

如果CSS规则集包含许多不构成完整集合的常用属性，则此检查建议将这些属性转换为简写形式。

例如，以下两个属性： outline-color和outline-style可以用一个替换outline一。但是，这样的替换不是100％等效的，因为速记将所有省略的子值重置为其初始状态。在此示例中，切换到outline速记意味着outline-width也设置为其初始值，即medium 。

此检查不能处理全套的速记属性（当切换到速记为100％安全时）。在这种情况下，请参阅“可以用快捷方式安全地替换属性”检查。

不突出显示，仅修复

冗余度量单位

该检查会以指定的测量单位突出显示零值。

警告

缺少分号

此检查在声明末尾检测到缺少的分号。

 残障人士
未使用的CSS选择器

此检查将检测文件中未使用的CSS类或ID。

警告

未知的文件或目录

此检查检查对文件和目录的引用。

错误

未知的CSS规则

此检查检测到未知的CSS规则。

错误

RGB颜色功能代替#HEX表示

此检查将检测定义为CSS rgb（）函数的CSS颜色，并提供将这些值转换为其十六进制表示形式的意图。

 残障人士
无效的CSS元素

此检查检测到分配给CSS属性的不合适的值。例如，无效的测量单位或值超出范围。

错误

无效的CSS函数

此检查将检测未知功能和非法功能参数。

错误

浏览器的属性兼容性

报告指定的浏览器不支持的CSS属性。该检查基于MDN兼容性数据 。

 残障人士
Stylelint

为指定的样式表文件运行Stylelint验证程序。

检查的严重性级别仅适用于“检查代码...”操作。可以在Stylelint配置文件中为每个单独的规则配置编辑器中使用的严重级别。

 残障人士
无效的CSS @charset规则

此检查检测到错误放置的@charset规则和不合适的字符集值。

警告

覆盖的财产价值

此检查突出显示CSS属性，这些属性将在同一规则集块中被覆盖。

警告

未解决的CSS类

此检查警告有关CSS类引用的信息，这些引用无法解析为任何有效的目标。

错误

W3C CSS验证器

此检查使用W3C CSS验证程序库验证CSS。

 残障人士
无效的CSS伪选择器

此检查检测未知和不合适的伪类和伪元素。

错误

负CSS值

此检查检测分配给CSS属性的负值，这些值不应为负值。例如，对象的宽度或高度。

错误

无效的CSS选择器

此检查检测未知的CSS选择器，并提供将其声明为类或ID的功能。

警告

错误放置的CSS @import

此检查检测到放错位置的CSS @import声明。

警告

浮点px长度

该检查检测像素长度的浮动值。

弱警告

#HEX颜色表示而不是RGB颜色功能

此检查将检测定义为十六进制的CSS颜色，并提供快速修复操作以将这些值转换为CSS rgb（）函数调用。
