PhpStorm 2019.3帮助

PHP 7.4支持

PHP 7.4带来了新功能，语法附加功能和错误修正。有关代码迁移的详细信息，请参阅从PHP 7.3.x迁移到PHP 7.4.x指南。

类型属性

借助类型化属性支持 ，您可以为类变量和属性声明类型提示。PhpStorm突出显示并解析键入的属性，并显示类型冲突（如果有）。您可以使用添加声明的类型意图操作Alt+Enter让您根据PHPDoc，默认值或参数类型声明即时添加类型提示。

嵌套三元运算符

PHP 7.4 弃用了左侧的关联性 ?三元运算符。您要么根本不使用嵌套的三元数，要么必须提供括号以显式声明计算顺序。PhpStorm将突出显示不赞成使用的表达式，并为您提供一个快速修复程序以阐明行为：

数值文字分隔符

使用长数字时，可以在数字文字中的任何位置放置下划线分隔符 ，以直观地对其进行格式设置，从而提高代码的可读性。为此，请使用“ 添加数字分隔符”意图操作Alt+Enter 。

协变量返回和协变量参数

使用从父类继承的方法时，可以定义不太具体的参数类型和更具体的返回类型 。也就是说，参数类型可以用其父类型之一替换，而返回类型可以用其子类型替换。

空合并分配运算符

的??= 空合并分配运算符使您可以将空合并和分配操作合并为一个。这样，您可以缩短长代码的构造。PhpStorm检测到??操作员并提供快速修复以减少冗余。

数组表达式中的传播运算符

在PHP 7.4中，您可以使用...在数组表达式内扩展运算符。以前，它仅在函数的参数列表中可用。PhpStorm检查运算符的正确用法，并确保您仅在拆包数组和可遍历的文件时使用 。

箭头功能

在PHP 7.4中编写简单的单行lambda函数时，可以使用短箭头语法： fn(parameter_list) => expr 。短箭头功能引入了来自外部作用域的变量的自动按值绑定。PhpStorm为它们提供完成以及解析声明。

弃用

PHP 7.4不赞成使用几种语言构造。您可以在此处查看弃用的完整列表。PhpStorm重点介绍了这种不赞成使用的用法，并为它们提供了快速修复。

  • 不推荐使用的铸造检查可检测到(real)类型转换，并让您替换为(float)演员表。

  • 不推荐使用的爆破/合并用法检查可检测implode要么join函数以数组作为第一个参数，并以字符串作为第二个参数。

  • 花括号访问语法用法检查可检测不赞成使用花括号来访问数组元素和字符串偏移量的用法 。
