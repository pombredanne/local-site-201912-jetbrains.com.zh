创建Phing构建文件列表

正确的Phing构建文件是带有以下内容的xml文件： 根元素。构建文件是在Phing Build工具窗口之外创建的，通常存储在PHP项目的根文件夹下。在Phing入门中找到有关编写Phing构建文件的更多信息。

访问Phing Build工具窗口

  1. 在编辑器中打开Phing构建文件，或在“ 项目”工具窗口中选择它，然后从所选内容的上下文菜单中选择“ 添加为Phing构建文件 ”。

  2. 选择查看|工具窗口|从主菜单开始构建 。通过编辑器或“项目”工具窗口中的Phing构建文件的上下文菜单打开工具窗口后，即可对其进行访问。

管理构建文件列表

要配置构建文件列表，请执行以下常规步骤

  1. 打开“ Phing Build”工具窗口。

  2. 要将构建文件添加到列表中，请单击添加按钮。 在工具栏上，然后在打开的“ 选择Phing构建文件”对话框中选择所需的xml构建文件。

  3. 要从列表中删除构建文件，请选择文件，然后单击“ 删除”按钮在工具栏上。

  4. 要导航到构建文件的源代码，请选择所需的文件，然后从所选内容的上下文菜单中选择“ 跳转到源代码 ”。
