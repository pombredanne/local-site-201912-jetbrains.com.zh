代码检查：“ array_map”调用可以转换为循环

报告array_map可以替换为的呼叫foreach环。

array_map（php.net）函数接受一个或几个数组作为参数，将回调函数应用于它们的每个元素，并返回一个包含结果元素的新数组。您也可以使用foreach循环（php.net）获得相同的结果。

在以下示例中， myArr数组的值通过使用odd回调函数。该函数从array_filter函数调用或foreach环。
