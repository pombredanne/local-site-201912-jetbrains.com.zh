PhpStorm 2019.3帮助

准备使用WordPress

PhpStorm提供了用于开发和运行WordPress应用程序的专用界面，并提供了WordPress感知的编码帮助，请参阅WordPress特定的编码帮助 。 要在项目中访问此功能，您需要下载WordPress ，在PhpStorm中进行注册，然后在当前项目中激活WordPress集成。

或者，您可以下载WordPress并创建以WordPress为目标的项目的存根。在这种情况下，项目中的WordPress集成将自动激活。

无论是在现有项目中启用WordPress集成，还是创建WordPress应用程序的存根，PhpStorm都会检查开发环境是否已针对WordPress开发正确配置。如果配置不符合要求，PhpStorm将显示带有“ 修复”链接的弹出窗口。

在你开始前

  1. 确保按照配置本地PHP解释器和配置远程PHP解释器中的说明在PHP页面的 PhpStorm中配置了PHP解释 器 。

  2. 确保已安装并启用PHP和WordPress支持插件。

    这些插件默认情况下处于激活状态。如果禁用了插件 ，请按照“ 管理插件”中所述在“ 插件”页面上启用它们。

  3. 如配置远程PHP解释器中所述，将PHP解释器配置为使用WordPress 。请注意， WordPress需要PHP 5.3或更高版本。

下载并安装WordPress

  1. 从https://wordpress.org/download/下载WordPress.zip存档。

  2. 提取档案：

    • 为了能够在本地Web服务器上运行WordPress应用程序，请将提取的文件存储到配置了PHP解释器的服务器的文档根目录中。对于使用Apache HTTPD Web服务器的本地开发环境，请将WordPress.zip解压缩到htdocs文件夹中。

    • 要在就地服务器上运行WordPress应用程序，请将提取的文件存储在项目根目录下。

    • 要在远程服务器上运行WordPress应用程序，请将提取的文件存储在项目中，然后配置将其自动上传到远程服务器的文档根目录。

      了解有关服务器访问配置的更多信息，请参阅配置与Web服务器的同步 。有关配置上载到服务器的信息，请参阅上载和下载文件 。

    • 如果您不想运行WordPress应用程序，而只需要从PhpStorm获得编码帮助，则将提取的文件存储在计算机上的任何位置。在这种情况下，您将必须将安装配置为包含路径 ，请参阅配置包含路径 。

要在命令行模式下运行WordPress，您将需要一组命令行工具，您可以通过使用Composer依赖管理器安装wp-cli / wp-cli软件包或下载wp-cli.phar存档来获取这些工具。有关详细信息，请参阅使用WP-CLI命令行工具 。

在项目中激活WordPress安装

  1. 打开您要启用WordPress集成的项目。

  2. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” | PHP |框架 。

  3. 在打开的“ 框架”页面上，展开WordPress节点，然后选择“ 启用WordPress集成”复选框。

  4. 在WordPress安装路径字段中， 指定安装WordPress的文件夹。此文件夹应包含wp-admin和wp-includes子目录。

  5. 单击“ 应用”以保存WordPress注册，然后单击“ PHP” 。

  6. 在打开的PHP页面上，将WordPress安装文件夹的路径添加到“ 包含路径”列表中：单击按钮，然后在打开的对话框中指定安装文件夹的路径。 在配置包含路径中了解更多信息。

生成WordPress应用程序存根

在项目创建期间，PhpStorm可以生成用于开发WordPress应用程序的项目存根。根据WordPress要求设置项目结构。

  1. 选择文件|新品|项目从主菜单或单击欢迎屏幕上创建新项目 。将打开“ 创建新项目”对话框 。

  2. 在左侧窗格中，选择WordPress插件 。

  3. 在右侧窗格中，指定将在其中创建项目的文件夹。

    在WordPress安装路径字段中， 指定安装WordPress的文件夹。此文件夹应包含wp-admin和wp-includes子目录。单击“ 创建”开始生成项目存根。

  4. 如果新创建的项目不在WordPress安装文件夹中，请将其添加到包含路径列表中，以将其配置为外部库 。

    1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” | PHP 。

    2. 在打开的PHP页面上，将WordPress安装文件夹的路径添加到“ 包含路径”列表中：单击按钮，然后在打开的对话框中指定安装文件夹的路径。 在配置包含路径中了解更多信息。
