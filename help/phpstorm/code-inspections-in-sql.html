报告不带任何“ where”子句的“ delete”或“ update”语句的用法。如果不使用“ where”子句，则“删除”会删除表中的所有数据，而“更新”会覆盖所有表行的值。

警告

不推荐使用的类型

从8.0开始，不推荐使用“长”类型，因此不建议使用“长”类型（因为该类型的API不会使用字符流，但是每次都会传输整个值）。在Oracle中，请勿创建包含LONG列的表。请改用LOB列（CLOB，NCLOB，BLOB）。仅支持LONG列以实现向后兼容。
