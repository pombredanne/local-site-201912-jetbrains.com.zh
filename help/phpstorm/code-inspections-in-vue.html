默认严重性

数据功能

如果Vue组件的data属性不是函数，则此检查将发出警告。

定义组件时， data必须声明为返回初始数据对象的函数，因为将使用相同的定义创建许多实例。如果我们仍然使用普通对象data ，同一对象将在创建的所有实例之间通过引用共享！通过提供data函数，每次创建新实例时，我们都可以简单地调用它以返回初始数据的新副本。

警告

模板/脚本标签重复

此检查警告是否有多个template要么script Vue文件中的标签。

Vue组件规范指示每个* .vue文件最多可以包含一个template要么script一次阻止。
