介绍

PhpStorm提供了“ 数据流分析”功能来帮助您进行代码考古-更好地了解继承的项目代码，解释代码的复杂部分，查找源代码中的瓶颈等。

具体来说，“ 往返于此处的数据流”功能使您能够：

  • 查看分配给变量的值从何而来。

  • 找出变量可能具有的所有可能值。

  • 找出一个表达式\变量\方法参数可以流入的地方。

如果要追溯在插入记号处传递给参数的值，则可以使用“ 分析” |“创建”来查看源代码的切片视图。 数据流到此处命令。此外，使用Analyze |来自“此处的数据流”命令，您可以找到表达式可以流入的位置。每次数据流分析的结果都显示在“ 分析数据流”工具窗口的专用选项卡中。

分析数据流

要分析去往/来自符号的数据流：

  1. 打开所需的文件进行编辑，请参阅使用脱字符导航 。

  2. 将插入号放在要分析的符号上（expression \ variable \ method参数）。

  3. 在主菜单上，或从上下文菜单中，选择“代码” |“其他”。分析数据流到此处或编码|根据您的目的从此处分析数据流 。

  4. 指定分析范围，然后选择是否要忽略来自测试代码的所有值。

  5. 单击确定 。在专用的“ 分析数据流”工具窗口中查看分析结果。
