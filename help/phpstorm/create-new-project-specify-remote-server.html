创建新项目：指定远程服务器

档案|现有文件中的新项目-Web服务器本地安装，源文件位于本地其他位置
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过网络共享或装入的驱动器访问文件
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过FTP / SFTP / FTPS访问文件

在“ 创建新项目：指定本地路径”页面上单击“ 下一步”时，页面打开。在向导的此页面上，选择要使用的远程Web服务器。您可以选择配置的Web服务器或定义新的配置。

项目

描述

添加新的远程服务器

选择此选项可移至“ 创建新项目：添加远程服务器”页面并指定新的Web服务器配置。

使用现有服务器

从以下列表中选择此选项以使用现有的Web服务器配置之一。

名称

此只读字段显示远程服务器配置的名称。

网址

此只读字段显示根据所选配置的服务器文档根目录的URL地址。

本地或挂载文件夹

此只读字段显示已安装驱动器的路径。

如果您选择“ 我的网络服务器在远程主机上”，则可以通过向导首页上的“ 网络共享”或“安装的文件夹”选项访问文件，则该字段可用。

主办

此只读字段以ftp：//格式显示FTP，SFTP或FTPS主机的URL地址： 或ftp：// ：

如果选择“ 我的网络服务器在远程主机上”，则可以通过该向导首页上的“ FTP / SFTP / FTPS”选项访问文件，该字段可用。

不要检查与服务器的HTTP连接

  • 清除此复选框后，PhpStorm将检查指定的URL地址是否确保成功连接到服务器。

  • 选中此复选框后，PhpStorm将移动到向导的下一页，而无需进行任何连接检查。
