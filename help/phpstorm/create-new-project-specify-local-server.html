创建新项目：指定本地服务器

档案|现有文件中的新项目-Web服务器在本地安装，源文件位于其文档根目录下

在“ 创建新项目：选择项目目录”页面上单击“ 下一步”时，页面打开。在向导的此页面上，选择要使用的本地服务器访问配置 。您可以选择现有配置或定义新配置。

项目

描述

添加新的本地服务器

选择此选项可移至“ 创建新项目：添加本地服务器”页面并指定新的Web服务器配置。

使用现有服务器

从以下列表中选择此选项以使用现有的Web服务器配置之一。

名称

此只读字段显示本地服务器访问配置的名称。

网址

此只读字段显示服务器配置根目录的URL地址（ 服务器配置根目录是通过服务器配置可访问的本地或远程服务器上文件树中的最高文件夹。对于就地服务器，它是项目根。）。

不要检查与服务器的HTTP连接

  • 清除此复选框后，PhpStorm将检查指定的URL地址是否确保成功连接到服务器。

  • 选中此复选框后，PhpStorm将移动到向导的下一页，而无需进行任何连接检查。
