代码检查：闭包可以转换为箭头功能

报告可以转换为短箭头功能的匿名功能。PHP 7.4宣布了对短箭头功能的支持。

有关详细信息，请参见PHP RFC：Arrow Functions 2.0（php.net） 。

在下面的示例中，内部的回调匿名函数array_map以两种形式编写：常规和箭头。

正规表格

箭头功能形式

$ myArr = [1,2,3,4,5,6]; $ doubledValues = array_map（function（$ var）{return $ var * 2;}，$ myArr）;
$ myArr = [1,2,3,4,5,6]; $ doubledValues = array_map（fn（$ var）=> $ var * 2，$ myArr）;
