创建新项目：指定Web路径（本地）

档案|现有文件中的新项目-Web服务器在本地安装，源文件位于其文档根目录下

在“ 创建新项目：指定本地服务器”或“ 创建新项目：添加本地服务器”页面上单击“ 下一步”时，页面打开。在此页面上，构成Web路径以通过HTTP访问项目根目录。

项目

描述

项目根的Web路径

在此文本框中，指定相对于服务器配置根URL的URL到所选项目目录的路径（ 服务器配置根目录是通过服务器配置可访问的本地或远程服务器上文件树中的最高文件夹。对于就地服务器，它是项目根。）。

项目网址

此只读字段显示项目根目录可用的完整URL地址。默认情况下，该字段用所选项目根目录的名称填充。当您在“ 项目根目录的Web路径”文本框中键入相对路径时，PhpStorm将动态构建完整的URL地址。

查看PHP设置

选中此复选框，移至“ 创建新项目：PHP设置”页面，然后检查PHP安装。
