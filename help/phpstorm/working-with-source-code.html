PhpStorm 2019.3帮助

使用源代码

当您使用代码时，PhpStorm可确保您的工作轻松自如。它提供了各种快捷方式和功能，可帮助您添加，选择，复制，移动，编辑，折叠，查找出现并保存代码。

有关在编辑器内部进行导航的信息，请参阅《 编辑器基础知识》 。

寻找行动

  • 如果您不记得要使用的操作的快捷方式，请按Ctrl+Shift+A以名称查找任何动作。

您可以使用相同的对话框来查找类，文件或符号。有关更多信息，请参阅按名称搜索目标（在各处搜索） 。

添加新的类，文件，包或暂存文件

  • 在编辑器中，按Ctrl+Alt+Insert添加类，文件或包。

    如果焦点在“ 项目”工具窗口内，并且您想添加新元素，请按Alt+Insert 。

  • 要创建一个新的Scratch文件，请按Ctrl+Shift+Alt+Insert 。

    PhpStorm创建一个可以运行和调试的临时文件。有关更多信息，请参阅暂存文件 。

切换文件的只读属性

如果文件是只读文件，则该文件带有关闭的锁定图标标记在状态栏 ，其编辑器标签或“ 项目工具窗口”中 。如果文件可写，则用打开锁图标标记在状态栏中。

  1. 在编辑器中打开文件，或在“项目”工具窗口中选择它。

  2. 请执行以下任一操作：

    • 从主菜单中，选择文件|将文件设为只读或文件|使文件可写 。

    • 单击状态栏中的锁定图标。

如果版本控制系统设置了只读状态，则建议您使用PhpStorm版本控制集成功能。有关更多信息，请参见版本控制 。

选择代码构造

  • 在编辑器中，将插入符号放置在要选择的项目上，然后按Ctrl+W / Ctrl+Shift+W扩展或缩小您的选择。

    例如，在纯文本文件中，选择从整个单词开始，然后扩展到句子，段落等。

    在PHP文件中，如果从方法调用中选择一个参数开始，它将扩展到所有参数，然后扩展到整个方法，然后扩展到包含此方法的表达式，再扩展到更大的表达式块，依此类推。

  • 如果只需要突出显示支架，则将插入标记立即放在块关闭支架/支架之后或块打开支架/支架之前。

根据大小写选择代码

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|智能钥匙 。

  2. 选中使用“ CamelHumps”单词复选框。

如果要在根据大写字母进行选择时使用双击，请确保在“ 编辑器” | “ 编辑器”中选中“ 荣誉CamelHumps单词...”复选框。一般页面的 设置/首选项对话框Ctrl+Alt+S 。

选择事件

您可以在文件中快速找到所选元素的所有实例，进行多项选择，或选择下一个或上一个实例。

  • 要对文件中的多个事件进行选择，请在代码中选择所需的元素，然后按Alt+J 。按相同的键查找下一个匹配项。

    或者，保持Alt按下，双击或将您要添加到选择中的文本拖到上方。

  • 要将插入符号移至下一个或上一个出现的位置，请按F3要么Shift+F3分别。的F3自动删除上一次出现的选择。

  • 要选择文件中所有出现的内容，请按Ctrl+Shift+Alt+J 。

  • 要查找所选标识符的下一个或上一个匹配项，请按Alt+Wheel up要么Alt+Wheel down 。

复制并粘贴代码

您可以使用标准的快捷方式进行复制Ctrl+C并粘贴Ctrl+V任何选定的代码片段。如果未选择任何内容，则PhpStorm会自动复制插入符号所在的整行。

默认情况下，当您在编辑器中粘贴任何内容时，PhpStorm会执行“智能”粘贴，例如，在注释中粘贴多行会自动将适当的标记添加到您要粘贴的行中。如果只需要粘贴纯文本，请按Ctrl+Shift+Alt+V 。

  • 将插入符号放在线条或符号上，然后按Ctrl+Shift+Alt+C / Ctrl+V 。

  • PhpStorm会跟踪您复制到剪贴板的所有内容。要从历史记录中粘贴，请按Ctrl+Shift+V 。在打开的对话框中，选择您的输入，然后点击粘贴 。

    剪贴板历史记录中存储的默认项目数为100。

在PHP上下文中，PhpStorm也可以自动删除打开和关闭粘贴PHP代码段时的标记。要自定义此行为， 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|智能键页面。

代码行

PhpStorm提供了一些用于操纵代码行的有用快捷方式。

如果您需要撤消或重做更改，请按Ctrl+Z / Ctrl+Shift+Z分别。

  • 要在当前行之后添加一行，请按Shift+Enter 。PhpStorm将插入符号移到下一行。

  • 要在当前行之前添加一行，请按Ctrl+Alt+Enter 。PhpStorm将插入符号移到上一行。

  • 要复制一行，请按Ctrl+D 。

  • 要按字母顺序对整个文件或代码选择中的行进行排序，请从主菜单中选择“ 编辑” |“选择” 。排序行或编辑|反向线 。当您使用属性文件，数据集，文本文件，日志文件等时，这些操作可能会有所帮助。如果您需要为这些操作分配快捷方式，请参阅配置键盘快捷方式以获取更多信息。

  • 要删除行，请将插入号放在所需的行上，然后按Ctrl+Y 。

  • 要连接线，请将插入号放置在要连接其他线的线处，然后按Ctrl+Shift+J 。持续按下键，直到所有需要的元素都被加入。

    您还可以连接字符串文字，字段或变量声明以及语句。请注意，PhpStorm会检查代码样式设置，并消除不必要的空格和冗余字符。

  • 要将字符串文字分为两部分，请按Ctrl+Enter 。

    PhpStorm还提供了一组方便的意图操作来处理字符串文字，例如将字符串拆分为两个串联的字符串或将其转换为字符串。 sprintf呼叫。要查看可用的操作，请将插入号放在字符串文字中，然后按Alt+Enter或点击 。

  • 要注释一行代码，请将插入符号放在适当的行，然后按Ctrl+/ 。

  • 要向上或向下移动一行，请按Shift+Alt+Up要么Shift+Alt+Down分别。

  • 要将XML或HTML标签属性向左或向右移动（交换），请在其上插入插入符号，或选择它，然后按Ctrl+Shift+Alt+Left左或Ctrl+Shift+Alt+Right为了正确。

代码声明

移动语句

  • 在编辑器中，将插入符号放在所需的语句处，然后按Ctrl+Shift+Up向上移动语句或Ctrl+Shift+Down将声明下移。PhpStorm移动选定的语句以执行语法检查。

如果在当前上下文中不允许移动语句，则将禁用操作。

完成当前声明

  • 在编辑器中，按Ctrl+Shift+Enter 。PhpStorm会在结构，切片和其他复合文字中自动插入所需的尾随逗号。插入符号移动到可以开始键入下一条语句的位置。

解开或删除语句

  1. 将插入号放置在要删除或展开的表达式上。

  2. 按Ctrl+Shift+Delete 。

    PhpStorm显示一个弹出窗口，其中包含当前上下文中所有可用的操作。为了更容易区分要提取的语句和要删除的语句，PhpStorm使用了不同的背景色。

  3. 选择一个动作并按Enter 。

代码片段

  • 通过在编辑器中拖动来移动和复制代码片段。

    • 要移动代码片段，请选择它并将其拖到目标位置。

    • 复制代码选择，保持Ctrl按下，将其拖动到目标位置。

      复制操作可能在macOS中不可用，因为它可能与全局OS快捷方式冲突 。

    默认情况下启用拖动功能。要禁用它， 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|常规，然后清除“ 鼠标”部分中的“ 在编辑器中启用拖放功能”复选框。

  • 要在所选代码段的大小写之间切换，请按Ctrl+Shift+U 。

    请注意，当您将切换大小写操作应用于CamelCase名称格式时，PhpStorm会将名称转换为小写。

  • 要注释或取消注释代码片段，请选择它，然后按Ctrl+Shift+/ 。

    要配置注释行为的设置，请使用“ 注释代码”部分中“特定语言”页面“ 设置/首选项” | “ 代码生成”选项卡上“ 注释代码 ”选项卡上的选项。编辑器代码样式 。

多个插入符

您可以同时编辑多行。例如，您可以在多行中添加注释标记，也可以删除单词中多次出现的某些字符。

有关事件的多重选择，请参阅“ 选择事件”部分。

  • 要添加或删除多个插入符，请按住Alt然后单击插入符号的下一个位置。

    或者，要添加多个插入符，可以按Ctrl两次，然后在不释放的情况下，按Up要么Down方向键。您还可以通过按进行多项选择Ctrl并拖动插入符号。

  • 要删除所有添加的插入符号，请按Escape 。

  • 您可以在列中选择多个代码。要启用或禁用列选择模式，请按Shift+Alt+Insert或使用编辑器的上下文菜单。

    当您要同时编辑多行代码时，此模式很有用。将插入符号放置在所需的符号上，然后将其插入或拖移，或者选择一个代码块，然后按指定的快捷方式即可获得相同的结果。

    您可以按时更改插入符号移动的行为Ctrl+Arrows （在Windows和Linux上）和⌥+Arrows （在macOS上）。默认行为是将插入符号移动到当前单词的末尾。要更改此设置，请打开设置Ctrl+Alt+S并导航到编辑器|一般在“ 插入符号移动”部分，从列表中选择适当的行为。

默认情况下，禁用列选择模式。启用后，该模式仅影响当前编辑器，如果关闭并重新打开该编辑器，则该模式将被禁用。

代码折叠

折叠的代码片段显示为阴影椭圆（ ）。如果折叠的代码片段中包含错误，则PhpStorm会以红色突出显示该片段。

要配置默认的代码折叠行为， 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|代码折叠 。

展开或折叠代码元素

  • 要折叠或展开代码片段，请按Ctrl+NumPad - / Ctrl+NumPad Plus 。PhpStorm可以折叠或展开当前代码片段，例如，一个方法。

  • 要折叠或展开所有代码片段，请按Ctrl+Shift+NumPad - / Ctrl+Shift+NumPad Plus 。

    PhpStorm会折叠或展开所选内容中的所有片段，或者，如果未选择任何内容，则会折叠或扩展当前文件中的所有片段，例如文件中的所有方法。

  • 要递归折叠或扩展代码，请按Ctrl+Alt+NumPad - / Ctrl+Alt+NumPad Plus 。PhpStorm会折叠或扩展当前片段以及该片段内的所有从属区域。

  • 要折叠代码块，请按Ctrl+Shift+. 。此操作会使匹配的花括号对之间的代码片段折叠{}并为该片段创建一个自定义折叠区域，并使其“可折叠”。

  • 要折叠或展开当前文件中的文档注释，请在主菜单中选择代码|折叠展开文档注释/折叠文档注释 。

  • 要折叠或展开自定义代码选择，请按Ctrl+. 。

    您可以折叠或展开代码中任何手动选择的区域。

折叠或展开嵌套片段

  • 要扩展当前片段和所有嵌套片段，请按Ctrl+NumPad *, 1 。您可以将当前片段扩展到指定的嵌套级别（从1到5）。

  • 要展开文件中所有折叠的片段，请按Ctrl+Shift+NumPad *, 1 。您可以将折叠后的片段扩展到指定的嵌套级别（从1到5）。

将环绕声与动作配合使用

您可以使用“ 环绕声”操作折叠或展开代码。

  1. 在编辑器中，选择一个代码片段，然后按Ctrl+Alt+T 。

  2. 在弹出菜单中，选择注释或区域...结束区域注释 。

  3. 如果需要，可以像往常一样命名自定义区域并折叠或展开创建的区域Ctrl+NumPad - / Ctrl+NumPad Plus 。

  4. 按Ctrl+Alt+.导航到创建的自定义区域。

禁用代码折叠轮廓

您可以禁用出现在左侧装订线的代码折叠轮廓。

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|代码折叠 。

  2. 清除“ 显示代码折叠轮廓”复选框。

自动保存

PhpStorm自动保存您在文件中所做的更改。保存是由各种事件触发的，例如编译，运行，调试，执行版本控制操作，关闭文件或项目或退出IDE。大多数实际事件是预定义的，无法配置，但是您可以确保所做的更改不会丢失。

要强制保存所有文件，请打开“ 文件”菜单，然后单击“ 全部保存” ，或按Ctrl+S 。

配置自动保存行为

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到外观和行为|系统设置 。

  2. 在“ 同步”下 ，配置以下选项：

    • 在停用框架时保存文件 （当您切换到其他应用程序时）

    • 如果应用程序闲置了N秒，则自动保存文件

用未保存的更改标记文件

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|一般|编辑器标签 。

  2. 选中用星号标记已修改的标签复选框。

还原变更

  • 对于最近的更改，请打开“ 编辑”菜单，然后单击“ 撤消” ，或按Ctrl+Z 。

  • 有关过去更改的更详细视图，请使用“ 本地历史记录” 。

  • 为了最可靠地跟踪所有更改，请使用PhpStorm支持的版本控制系统 。
