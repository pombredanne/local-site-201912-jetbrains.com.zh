PhpStorm 2019.3帮助

触控笔

PhpStorm使您可以将手写笔代码即时转换为CSS。

在你开始前

  1. 确保您的计算机上装有Node.js。

  2. 如管理插件中所述，在“ 插件”页面上安装并启用Stylus存储库插件。

  3. 确保在“ 插件”页面上启用了File Watchers捆绑的插件 ，有关详细信息，请参阅管理插件 。

安装手写笔

  • 在嵌入式终端 （ Alt+F12 ），输入：

    npm install --global stylus

    从Stylus官方网站了解更多信息。

您还可以按照npm和Yarn中的说明 ，在Node.js和NPM页面上安装手写笔软件包。

编写手写笔代码

要即时编译代码，您需要将编译器配置为PhpStorm File Watcher 。

创建文件监视程序

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，单击“ 工具”下的“ 文件监视程序 ”。打开的“ 文件监视程序”页面显示已配置的文件监视程序的列表。

  2. 点击或按Alt+Insert并从列表中选择Stylus预定义模板。

  3. 按照文件监视程序中所述进行操作。

编译代码

当您打开文件时，PhpStorm将检查当前项目中是否有适用的文件监视程序。如果已配置但禁用了这种文件监视程序，则PhpStorm会显示一个弹出窗口，通知您有关已配置的文件监视程序并建议启用它。

如果在当前项目中配置并启用了适用的文件监视程序，则PhpStorm会在“ 新建监视程序”对话框中指定的事件后自动启动它。

  • 如果选中了“ 自动保存编辑的文件以触发监视程序”复选框，则在对源代码进行任何更改后都会立即调用文件监视程序。

  • 如果清除了自动保存编辑的文件以触发监视程序复选框，则文件监视程序会在保存时启动（ File | Save All ， Ctrl+S ）或将焦点从PhpStorm移开（停用帧时）。

PhpStorm使用生成的输出创建一个单独的文件。该文件具有源手写笔文件的名称，扩展名为.css 。在“ 新建监视程序”对话框的“ 刷新的输出路径”字段中定义了生成文件的位置。但是，在项目树中 ，它们显示在源文件下，该文件现在显示为节点。

配置语法突出显示

您可以根据自己的喜好和习惯配置支持Stylus的语法突出显示。

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到编辑器|配色方案|手写笔 。

  2. 选择配色方案，接受从默认设置继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
