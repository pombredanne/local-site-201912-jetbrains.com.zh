从命令行使用PhpStorm功能

除了在PhpStorm中工作之外，还可以从命令行执行以下操作：

  • 从命令行打开文件
  • 从命令行格式化文件
  • 从命令行启动代码检查
  • 从命令行查看文件之间的差异
  • 从命令行合并文件

在Windows上启用命令行启动器

  • 将PhpStorm bin文件夹的路径添加到Path环境变量（例如， C：\ Program Files \ JetBrains \ PhpStorm \ bin ）。配置完后Path环境变量，您可以运行PhpStorm命令。

在Linux和macOS上启用命令行启动器

在macOS和Linux上，创建命令行启动器以将PhpStorm与您的Shell集成。确保创建的启动器脚本在外壳程序的搜索路径内，以便可以在系统范围内调用它。

  1. 在主菜单中，选择“ 工具” |“工具”。创建命令行启动器 。在“ 创建启动器脚本”对话框中，提供启动器的名称和路径，或接受建议的值。

  2. 在PhpStorm外部，将路径和启动脚本的名称添加到您的path 。

在Linux上，将PhpStorm 作为快照程序包安装会自动创建名为phpstorm 。工具|因此，“创建命令行启动器”命令不可用。

从命令行启动PhpStorm

  • 要从命令行启动PhpStorm，请输入在命令行中输入命令，然后按Enter 。 是特定于平台的启动器。
