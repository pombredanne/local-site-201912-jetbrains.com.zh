查看源文件的结构

您可以使用“ 结构”工具窗口检查当前在编辑器中打开的文件的结构 Alt+7或“ 结构”弹出窗口Ctrl+F12 。

默认情况下，PhpStorm显示当前文件的所有类，方法和其他元素。要切换要显示的元素，请单击“ 结构”工具窗口工具栏上的相应按钮。例如：

  • 请点击显示班级字段。

  • 请点击显示常数。

  • 请点击显示继承的成员。

  • 请点击显示包含的文件。

  • 请点击显示lambda。

在“项目”工具窗口中显示班级成员

  • 右键单击“ 项目”工具窗口标题栏，然后从上下文菜单中选择“ 显示成员 ”。
