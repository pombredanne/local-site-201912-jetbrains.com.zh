使用工具箱应用安装

建议使用JetBrains工具箱应用程序安装JetBrains产品的工具。使用它可以安装和维护不同的产品或同一产品的多个版本，包括抢先体验计划 （EAP）版本，在必要时进行更新和回滚，以及轻松删除任何工具。工具箱应用程序维护所有项目的列表，以在正确的IDE和版本中快速打开任何项目。

Windows上的静默安装

无需任何用户界面即可执行静默安装。网络管理员可以使用它在许多计算机上安装PhpStorm，并避免打扰其他用户。

要执行静默安装，请使用以下开关运行安装程序：

  • /S ：启用静默安装

  • /CONFIG ：指定静默配置文件的路径

  • /D ：指定安装目录的路径

    此参数必须是命令行中的最后一个，并且即使路径包含空格也不应包含任何引号。

例如：

PhpStorm-*。exe / S /CONFIG=d:\temp\silent.config / D = d：\ IDE \ PhpStorm

要在安装过程中检查问题，请添加/LOG在日志文件路径和名称之间切换/S和/D参数。安装程序将生成指定的日志文件。例如： PhpStorm-*.exe /S /CONFIG=d:\temp\silent.config /LOG=d:\JetBrains\PhpStorm\install.log /D=d:\IDE\PhpStorm

静默配置文件

您可以在https://download.jetbrains.com/webide/silent.config下载PhpStorm的默认静默配置文件。

静默配置文件定义用于安装PhpStorm的选项。使用默认选项，仅对当前用户执行静默安装mode=user 。如果要为所有用户安装PhpStorm，请使用文本编辑器打开静默配置文件，更改安装模式选项的值mode=admin并以管理员身份运行安装程序。

默认的静默配置文件对于每个JetBrains产品都是唯一的。您可以根据需要对其进行修改，以启用或禁用各种安装选项。

在Linux上作为快照软件包安装

您可以将PhpStorm作为独立的快照软件包安装。由于快照会自动更新，因此您的PhpStorm安装将始终是最新的。

要使用快照，请按照安装指南中的说明安装并运行快照服务。

在Ubuntu 16.04 LTS和更高版本上，此服务已预安装。

PhpStorm通过两个渠道分发：

  • 稳定通道仅包括稳定版本。要安装PhpStorm的最新稳定版本，请运行以下命令：

    的--classic该选项是必需的，因为PhpStorm snap需要像传统打包的应用程序一样具有对系统的完全访问权限。

    sudo snap安装phpstorm --classic

  • 边缘通道包括EAP版本。要安装PhpStorm的最新EAP版本，请运行以下命令：

    sudo snap安装phpstorm --classic --edge

安装快照后，您可以通过运行phpstorm命令。

要列出所有已安装的快照，请运行sudo snap list 。有关其他捕捉命令的信息，请参见Snapcraft文档 。
