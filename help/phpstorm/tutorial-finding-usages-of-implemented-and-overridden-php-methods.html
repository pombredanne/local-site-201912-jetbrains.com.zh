教程：查找已实现和重写的PHP方法的用法

在PHP上下文中，PhpStorm将“ 查找用法”功能应用于已实现和覆盖的方法。让我们考虑以下示例：

  1. 创建一个接口，一个实现该接口的抽象类以及两个扩展该抽象类的类，它们的组织方式如下：

    1. 接口MyInterface与一个foo()方法。

    2. 抽象类MyAbstractClass实现MyInterface 。

    3. 一类MyClass延伸MyAbstractClass ，工具foo()接口所必需，并覆盖父类的方法。

    4. 一类MyClassWithDelegate延伸MyClass并实施foo()与代表。

    5. 的$b和$c调用的变量foo()从MyClass和MyClassWithDelegate分别：

    foo（）; $ c = new MyClassWithDelegate（）; $ c-> foo（）;

  2. 从MyInterface ，为调用查找用法设置 foo()通过按Ctrl+Shift+Alt+F7或选择编辑|查找|从主菜单中找到“用法设置” 。

  3. 在查找用法中。将打开“ 方法选项”对话框，选中“ 包括覆盖/实施方法”复选框，然后单击“ 查找” 。
