安全删除

介绍

通过安全删除重构，您可以安全地从源代码中删除文件。

为了确保删除是安全的，PhpStorm查找正在删除的文件的用法。如果找到了这样的用法，则可以在实际删除符号之前浏览它们并在代码中进行必要的更正。

执行重构

  1. 选择要删除的项目。

  2. 请执行以下任一操作：

    • 按Alt+Delete 。

    • 选择重构|从主菜单或上下文菜单中安全删除 。

    • 选择重构|从主菜单重构 Ctrl+Shift+Alt+T ，然后选择安全删除 。

  3. 在“ 安全删除”对话框中 ，选择必要的选项，然后单击“ 确定” 。

  4. 如果重构可能不安全，则会打开“ 检测到的使用情况”对话框。
    • 查看用法。 单击此按钮以查看要在代码中使用的项目。结果，“ 查找工具”窗口打开。

      分析您的代码并进行必要的更正。然后单击“ 执行重构” 。（如果要从头开始重新运行重构，请点击重新运行安全删除 。PhpStorm将再次检查重构是否安全。）

    • 仍然删除。 单击此按钮可以删除该项目，而无需查看其用法。
