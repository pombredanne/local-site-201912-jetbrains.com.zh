PHP命令行工具

PhpStorm支持运行流行的第三方或用户定义的PHP工具的命令： Symfony 1.1 + ， Symfony2 ， Zend Framework 1 ， Zend Framework 2（ZFTool） ， Yii ， Composer ， Drush 5.8 + ， Laravel和Doctrine （基于Symfony控制台）， WordPress命令行界面 。

在开始之前，请确保“ 插件”页面上的“ 命令行工具支持”插件已启用的 设置/首选项对话框Ctrl+Alt+S 。

目前仅本地解释器支持从PhpStorm运行PHP命令行工具。要在远程环境中运行工具，请使用内置的SSH终端 。

将外部PHP命令行工具与PhpStorm集成

  1. 下载并安装该工具。

  2. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到工具|命令行工具支持 。

  3. 请点击在工具栏上。

  4. 在“ 命令行工具”对话框中，从列表中选择工具，然后指定其可见性级别（ Project或Global ）。

  5. 单击确定时 ，将打开工具设置对话框。在此对话框中，设置工具执行。

  6. 单击“ 确定”以应用更改并返回“ 命令行工具支持”页面。（可选）单击编辑工具属性，或自定义命令集。有关详细信息，请参见自定义工具 。

定制工具

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到工具|命令行工具支持 。

  2. 在“ 命令行工具支持”页面上，在列表中选择工具，然后单击“确定”。 在工具栏上。xml工具描述符在编辑器中打开。

  3. 更新命令的定义（有关详细信息，请参阅定义自己的命令行工具 ）。键入时，将动态检查工具描述符的格式是否正确。

  4. 重新加载命令定义。在“ 命令行工具支持”页面上的 设置/首选项对话框Ctrl+Alt+S ，在列表中选择工具，然后单击在工具栏上。

定义自己的命令行工具

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到工具|命令行工具支持 。

  2. 请点击在工具栏上，然后在打开的“ 命令行工具”对话框中，从“ 选择工具”列表中选择“ 自定义 ”工具 ，然后为其指定可见性级别（“ 项目”或“ 全局” ）。

  3. 在“ 工具设置”对话框中，键入工具的路径和工具别名。

    当您单击OK时 ，PhpStorm会将您带到“ 命令行工具支持”页面，在该页面上将新工具添加到列表中。

  4. 单击“ 确定”接受更改，然后关闭“ 设置/首选项”对话框。

保持工具描述符一致

每次您在xml工具描述符中编辑命令定义时，PhpStorm都会动态检查它的格式是否正确。每次调用命令都会执行完全验证 。

如果检测到任何不一致，则会在“ 运行工具”窗口中显示一条错误消息。

验证工具定义

  1. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到工具|命令行工具支持 。描述符不一致的工具标有 。

  2. 在列表中选择工具，然后单击在工具栏上。xml工具描述符在编辑器中打开。

  3. 根据需要更新命令的定义（有关详细信息，请参阅定义自己的命令行工具 ）。
