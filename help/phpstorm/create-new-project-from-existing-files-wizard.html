现有文件向导中的新建项目

档案|现有文件中的新项目

使用此向导可以围绕驻留在本地或远程主机上的现有文件设置新项目。

该向导为您提供了一种处理分布式项目中特定文件的方法，而无需完全下载或复制它们。PhpStorm在指定位置使用包含项目文件的.idea文件夹创建一个项目目录。

选择方案

在向导的第一页上，选择访问将在其周围建立新项目的文件的方式。

项目

描述

Web服务器安装在本地，源文件位于其文档根目录下。

如果您有本地Web服务器，并且要围绕位于指定为服务器文档根目录的文件夹下方的现有文件或文件夹（例如。\ htdocs下）设置项目，请选择此选项。

单击“ 下一步”将转到“ 创建新项目：选择项目目录”页面。

Web服务器安装在本地，源文件位于本地的其他位置。

如果您有一台本地Web服务器，并且要在其附近建立项目的文件或文件夹也存储在本地，但不在指定为服务器文档根目录的文件夹下，请选择此选项。

单击“ 下一步”将转到“ 创建新项目：指定本地路径”页面。

Web服务器位于远程主机上，可通过网络共享或安装的驱动器访问文件。

选择此选项可使PhpStorm通过网络将文件复制到本地驱动器，并围绕它们建立一个项目。

单击“ 下一步”将转到“ 创建新项目：指定本地路径”页面。

Web服务器位于远程主机上，可通过FTP / SFTP / FTPS访问文件。

选择此选项可使PhpStorm通过FTP，SFTP或FTPS协议从远程服务器下载文件。

单击“ 下一步”将转到“ 创建新项目：指定本地路径”页面。

源文件位于本地目录中，尚未配置任何Web服务器。

如果要使用某个本地目录中的文件而不使用任何Web服务器，请选择此选项。

单击“ 下一步”将转到“ 创建新项目：选择项目目录”页面。
