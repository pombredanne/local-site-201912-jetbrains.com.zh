File types

To view the list of file types recognized by PhpStorm, in the Settings/Preferences dialog Ctrl+Alt+S, go to | Editor | File Types. If a file in your project is marked with the icon, it indicates that PhpStorm can't recognize it. In this case, you can register and configure a new file type.

The set of file types recognized by PhpStorm depends on enabled plugins.
