代码检查：完全合格的名称使用

报告完全合格的类名称，可以通过添加以下名称来缩短该名称use声明。

通过用import语句替换完全限定的类名，可以避免重复长名称空间，从而提高了代码的可读性。有关详细信息，请参见使用名称空间：别名/导入（php.net） 。

在以下示例中， \Foo\Bar\Baz全限定名称将替换为use声明。

命名空间Foo \ Bar {类Baz {}}命名空间A {new \ Foo \ Bar \ Baz（）; }
命名空间Foo \ Bar {类Baz {}}命名空间A {使用Foo \ Bar \ Baz;新的Baz（）; }
