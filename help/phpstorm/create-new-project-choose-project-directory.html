创建新项目：选择项目目录

档案|现有文件中的新项目-Web服务器在本地安装，源文件位于其文档根目录下
档案|现有文件中的新项目-源文件位于本地目录中，尚未配置Web服务器

在向导的此页面上，指定文件的位置，指定项目根目录 ，并配置要创建的项目的结构。项目根目录是所有项目源的父文件夹。默认情况下，此文件夹中的所有子文件夹都被视为源，并且它们的文件参与索引，搜索，解析，代码完成等。使用工具栏上的图标ro更改此状态。

如果您的源位于几个独立的父文件夹中，请在这些文件夹之一中创建一个项目，然后在“设置”对话框的“ 目录”页面上将其他源配置为其他内容根 。

有关配置项目内容的更多详细信息，请参阅“ 内容根目录”和“ 在内容根 目录中 配置文件夹” 。

项目

描述

项目根目录

选择所有项目源的父文件夹，然后单击此图标或在所选内容的上下文菜单上选择“ 项目根目录 ”。PhpStorm用图标，并向其中添加带有设置的.idea目录。

测试来源

单击此图标可使PhpStorm将所选文件夹的内容视为与单元测试有关。或者，在所选内容的上下文菜单上选择“ 资源根 ”。

资源根

单击此图标以使PhpStorm能够完成所选文件夹下资源的相对路径。这意味着可以相对于所选文件夹引用所选文件夹的内容，而不用指定完整路径。

或者，在所选内容的上下文菜单上选择“ 资源根 ”。

排除在外

单击此按钮可使PhpStorm在索引，解析，代码完成等过程中忽略所选目录。或者，在所选内容的上下文菜单上选择“已排除 ”。
