创建新项目：指定Web路径（远程）

档案|现有文件中的新项目-Web服务器本地安装，源文件位于本地其他位置
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过网络共享或装入的驱动器访问文件
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过FTP / SFTP / FTPS访问文件

在“ 创建新项目：选择远程路径”页面上单击“ 下一步”时，页面打开。在此页面上，构成Web路径以通过HTTP访问项目根目录。

项目

描述

项目根的Web路径

在此文本框中，指定相对于Web服务器文档根URL的所选项目目录的路径。

项目网址

此只读字段显示完整的URL地址，通过该URL可以使用项目根目录。默认情况下，该字段用所选项目根目录的名称填充。当您在“ 项目根目录的Web路径”文本框中键入相对路径时，PhpStorm将动态构建完整的URL地址。
