语言注入

语言注入使您可以处理代码中嵌入的其他语言的代码段。当您将一种语言（例如HTML，CSS，XML，RegExp等）注入字符串文字中时，您将获得用于编辑该文字的全面代码帮助。

在开始使用语言注入之前，请确保已启用 IntelliLang插件。

在PHP上下文中，可以使用称为Injectable PHP的特殊PHP方言。该方言类似于PHP，但开头标签，可以省略。可以像其他任何语言一样，将可注入的PHP手动注入到字符串文字中。此外，它会自动插入eval（）调用和Markdown防护代码块中，并将语言标识符设置为php 。

暂时注入语言

  1. 将插入符号放置在要插入语言的字符串文字，标记或属性中，然后按Alt+Enter （或使用意图操作图标 ）。

  2. 选择注入语言或参考，然后选择要注入的语言。

在专用编辑器部分中打开代码片段

  1. 将插入符号放置在注入的代码段内，然后按Alt+Enter （或使用意图操作图标 ）。

  2. 选择编辑片段 。

    PhpStorm将打开一个专用的编辑器部分，以使用注入的语言编辑代码。该编辑器提供完整的代码帮助，包括代码完成，检查，意图和代码样式操作。

使用@lang批注

在PHP上下文中，您还可以使用@lang 注解以注入语言。

  • 提供@lang 目标字符串文字之前的PHPDoc注释。语言ID通常很直观，例如SQL，RegExp，XML，HTML。

在nowdoc / heredoc字符串中插入语言

在PHP背景，你可以注入内部语言nowdoc / 定界符字符串。

  • 开始输入heredoc / nowdoc字符串，并将标记设置为要注入的语言ID。语言ID通常很直观，例如SQL，RegExp，XML，HTML。

使用语言注入注释

  • 在目标字符串文字之前添加空白行，然后键入以下注释：

    //语言=

    对于注释，请使用要注入的语言的语法。语言ID通常是直观的，例如SQL，RegExp，XML，HTML。

取消注射

  1. 将插入号放置在代码片段上，然后按Alt+Enter （或使用意图操作图标 ）。

  2. 选择取消注入语言或参考 。

要取消语言注入，您还可以删除注入注释或注释。

配置注入规则

您可以在上配置语言注入规则 编辑器| PhpStorm设置的“ 语言注入”页面Ctrl+Alt+S 。

所有预定义的注入规则都针对内置作用域配置。换句话说，它们是全局的（因此在所有PhpStorm项目中都可用）。可以为IDE或仅一个项目配置自定义规则。要更改自定义注入的范围，请使用 。

要配置自定义注入规则，请单击添加新规则，或复制预定义规则并更改其设置。
