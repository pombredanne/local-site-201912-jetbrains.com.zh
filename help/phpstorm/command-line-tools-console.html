命令行工具控制台工具窗口

工具|运行命令

Ctrl+Shift+X

工具窗口上标有图标 。使用此工具窗口的选项卡在命令行中键入和运行命令，并验证.xml描述符的结构一致性。有关详细信息，请参见PHP命令行工具 。

从“ 设置/首选项”对话框的“ 命令行工具支持”页面上的“ 控制台编码”列表中选择在工具窗口中使用的字符集。

工具窗口包含以下选项卡和区域：

  • 用于以以下格式键入命令的“ 输入”窗格 。“输入”窗格（字段或弹出窗口）的位置取决于“ 命令行工具支持”页面上的“ 显示控制台”设置。

  • 输出选项卡显示执行命令的结果。该选项卡以最后调用的命令命名。

  • 默认情况下，“ 工具定义文件错误”选项卡处于隐藏状态。仅当在验证期间在.xml描述符中检测到任何结构差异时，才可以访问该选项卡。请参阅如何使工具描述符保持一致？ 有关详细信息。

工具提示和快捷方式

描述

停止

单击此按钮可在不关闭工具窗口的情况下取消命令的执行。

到上一条命令
Ctrl+Alt+Up

单击此按钮导航到上一个命令。

转到下一条命令
Ctrl+Alt+Down

单击此按钮导航到下一个命令。

使用软包装

单击此切换按钮将软包装模式应用于输出。

滚动到最后

单击此按钮导航到以最后调用的命令命名的输出选项卡的底部。

打印

单击此按钮以打印出控制台的内容。单击按钮后，PhpStorm将打开“ 打印”对话框，您可以在其中配置打印过程和输出。

全部清除

单击此按钮可从控制台中删除所有文本。也可以从控制台的上下文菜单中使用此功能。

导出到文本Alt+O

单击此按钮可将执行命令的结果保存在文本文件中。在打开的“ 导出预览”对话框中，指定目标文件，然后单击“ 保存” 。

Ctrl+Shift+F4

单击此按钮以关闭工具窗口。如果一个或多个命令仍在运行，则会打开“ 命令行工具”对话框。通过单击以下按钮之一，指定是要停止它们还是要使其在后台运行：

  • 终止并关闭
  • 关闭而不终止
