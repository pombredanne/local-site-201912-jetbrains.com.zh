  • 如果选中此复选框，则使用制表符：

    • 在按Tab键

  • 清除复选框后，PhpStorm将使用空格而不是选项卡。

缩进

在此字段中，为每个缩进级别指定要插入的空格数（如果选中了“ 使用制表符”复选框，则为制表符 ）。

其他

在此标签中，配置Twig 特殊语法的编码样式。在编辑器中使用Twig模板查看文件时，PhpStorm会根据Symfony2标准自动应用这些设置。
