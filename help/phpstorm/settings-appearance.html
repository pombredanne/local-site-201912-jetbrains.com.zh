选项

描述

背景图像

此选项使您可以为IDE配置自定义背景图像 。

列表中的循环滚动

从最后一项跳到第一项，反之亦然，从而启用滚动列表的功能。

自动将鼠标光标置于默认按钮上

隐藏关于焦点丢失的导航弹出窗口

如果选中此复选框，则当任何其他PhpStorm组件获得焦点时，导航弹出框（ 转到class / file / symbol ）将关闭。

如果未选中此复选框，则导航弹出框将保持更改焦点，并且关闭此类弹出框的唯一方法是按Escape 。

仅按下即可拖放

如果未选中此复选框（默认情况下），则PhpStorm允许使用拖放操作在“ 项目工具窗口”中移动编辑器选项卡，工具窗口按钮，文件和文件夹。

选中此复选框可避免意外移动文件或文件夹或UI组件。因此，拖放操作仅在Alt按键。

工具提示的初始延迟（毫秒）

使用此滑块可指定将鼠标悬停在编辑器中某个项目上方的时间与显示工具提示及其值的时间之间经过的时间。此设置在调试期间特别重要。如果延迟太短，使用鼠标会变得很不方便，因为每次鼠标在屏幕上移动都会带来一些带有所有变量值的工具提示。

抗锯齿

选项

描述

集成开发环境

选择要应用于IDE的哪种抗锯齿模式（包括菜单，工具窗口等）：

  • 子像素 ：建议此选项用于LCD显示器，并利用彩色LCD上的每个像素都由红色，绿色和蓝色子像素组成的事实。这样可以使文本平滑并渲染更多细节。

  • 灰度 ：对于非LCD显示器或垂直放置的显示器，建议使用此选项。它在像素级别处理文本。

  • 无抗锯齿 ：此选项可用于高分辨率显示，其中非抗锯齿字体的渲染速度更快，并且外观可能更好。

编辑

选择要应用于编辑器的抗锯齿模式：

  • 子像素 ：建议此选项用于LCD显示器，并利用彩色LCD上的每个像素都由红色，绿色和蓝色子像素组成的事实。这样可以使文本平滑并渲染更多细节。

  • 灰度 ：对于非LCD显示器或垂直放置的显示器，建议使用此选项。它在像素级别处理文本。

  • 无抗锯齿 ：此选项可用于高分辨率显示，其中非抗锯齿字体的渲染速度更快，并且外观可能更好。

项目

描述

动画窗口

选中此复选框可使未停靠的工具窗口随动画效果一起滑动。

仅当取消停靠工具窗口时，此选项才适用。

显示内存指示

在状态栏上显示内存指示器。

在菜单中禁用助记符

在PhpStorm菜单中隐藏热键的下划线。

禁用控件中的助记符

在PhpStorm控件中隐藏热键的下划线。

在菜单项中显示图标

如果选中此复选框（默认情况下），则图标显示在菜单选项的左侧。

如果未选中此复选框，则显示的菜单选项将不带有图标。

左右并排布局

选中这些复选框后，将针对宽屏显示优化工具窗口的放置方式。

切换布局Ctrl+Click工具窗口之间的拆分器。

这仅适用于位于左侧和右侧的工具窗口，而不适用于PhpStorm窗口的顶部和底部。

平滑滚动

转动鼠标滚轮时，启用逐像素滚动而不是逐行滚动。

这对于macOS和带有某些鼠标设备的Windows有效。

显示工具窗口栏

选择此复选框以显示工具窗口栏。

显示工具窗口编号

选择此复选框可在工具窗口按钮上显示工具窗口快速访问编号。

您可以使用Alt+number快捷方式（无论此设置如何），并在“ 键盘映射”页面上更改快捷方式。

请注意，仅当相应的键绑定具有以下格式时，才会显示工具窗口助记符Alt+number ，在哪里number是1到9之间的整数。如果使用其他键盘快捷键，则不会显示助记符。

允许合并对话框上的按钮

  • 如果选中此复选框，则对话框中的多个选项将组合在带有向下箭头的单个按钮下。

    您可以通过单击列表或按来查看所有合并的命令Shift+Alt+Enter 。

  • 如果未选中此复选框，则按钮将显示在一行中。

宽屏工具窗口布局

如果选中此复选框，则为宽屏显示优化了工具窗口的放置方式。
