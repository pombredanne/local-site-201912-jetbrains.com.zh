创建新项目：指定本地路径

档案|现有文件中的新项目-Web服务器本地安装，源文件位于本地其他位置
档案|现有文件中的新项目-Web服务器位于远程主机上，可通过网络共享或装入的驱动器访问文件
档案|现有文件中的新项目-Web服务器位于远程主机上，可通过FTP / SFTP访问文件

在此页面上，指定要设置的项目的名称以及将项目源下载到的本地文件夹。

项目

描述

项目名

在此文本框中，根据下载的文件键入要设置的项目的名称。

项目本地路径

在此文本框中，指定将在其中创建新项目的本地文件夹。首先指定将在其中创建项目文件夹的父文件夹。手动输入路径或单击浏览 然后在打开的对话框中选择所需的位置。

键入项目名称时，它会添加到项目路径中。

部署选项

在此区域中，指定要使用的下载配置。可用的选项有：

  • 默认 -单击此选项可使PhpStorm根据“ 部署”页面上“ 选项”对话框中指定的设置下载源。

  • 自定义 -单击此选项可在“ 创建新项目：单击部署”对话框中单击“ 下一步”时自定义默认设置。
