创建新项目：添加远程服务器

档案|现有文件中的新项目-Web服务器本地安装，源文件位于本地其他位置
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过网络共享或装入的驱动器访问文件
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过FTP / SFTP / FTPS访问文件

如果在“ 创建新项目：指定远程服务器”页面上选择了“ 添加新的远程服务器”选项，则会打开该页面。在此页面上，指定用于连接和认证到远程服务器的设置的新配置。

项目

描述

可以用来

名称

在此字段中，输入新的远程服务器配置的名称。

FTP，FTPS，SFTP

类型

在此列表中，选择访问服务器的方式。可用的选项有：

  • FTP：选择此选项可使PhpStorm通过FTP 文件传输协议访问服务器。

  • SFTP：选择此选项可使PhpStorm通过SFTP文件传输协议访问服务器。

  • FTPS：选择此选项可使PhpStorm通过SSL上的FTP文件传输协议（ FTPS扩展名）访问服务器。

  • 本地或装入的文件夹：如果Web服务器在本地或装入的文件夹中运行，并且其文档根目录 不是项目根目录的父级，则选择此选项。

  • 就地：如果Web服务器在计算机上运行，项目在其文档根目录下，并且您直接在服务器上进行开发，则选择此选项。

FTP，FTPS，SFTP

FTP / SFTP / FTPS主机

在此字段中，指定要从中下载文件的FTP / SFTP / FTPS服务器的主机名。

FTP，FTPS，SFTP

港口

在此字段中，指定要使用的端口。默认值为：

  • FTP和FTPS为21

  • SFTP为22

FTP，FTPS，SFTP

用户名

在此字段中，输入用于验证服务器的用户名。

FTP，FTPS，SFTP

以匿名身份登录

选中此复选框以使用您的电子邮件地址作为密码启用对服务器的匿名访问 。

FTP，FTPS，SFTP

认证方式

从此列表中，选择客户端身份验证方法。可用的选项有：

  • 密码 -选择此选项可通过密码使用标准身份验证。

  • 密钥对（OpenSSH） -选择此选项可通过密钥对使用SSH身份验证 。

PhpStorm支持根据OpenSSH协议生成的私钥。

SFTP

密码

在此字段中，输入用于验证服务器的密码。

FTP，FTPS，SFTP

私钥文件

在此字段中，指定私钥文件的位置。

FTP，FTPS，SFTP

密码短语

在此字段中，指定您的身份验证密码。

SFTP

保存密码

选择此复选框可使PhpStorm记住指定的密码。

FTP，FTPS，SFTP

保存密码

选择此复选框可使PhpStorm记住指定的密码。

SFTP

测试连接

单击此按钮以检查指定的设置，确保通过FTP / SFTP / FTPS成功连接。

FTP，FTPS，SFTP

根路径

在这个领域里， 指定相对于服务器上根文件夹的服务器配置根 。该文件夹将是可通过当前服务器配置访问的文件夹结构中的最高文件夹。

请执行以下任一操作：

  • 接受默认的/路径，该路径指向服务器上的根文件夹。

  • 手动输入路径。

  • 请点击然后在打开的“选择根路径”对话框中选择所需的文件夹。

  • 单击“ 自动检测”按钮，使PhpStorm检测FTP / SFTP服务器上的用户主文件夹设置，并根据它们设置根路径 。仅在指定凭据后才启用该按钮。

FTP，FTPS，SFTP

自动侦测

单击此按钮可使PhpStorm检测FTP / SFTP / FTPS服务器上的用户主文件夹设置，并根据它们设置根路径 。

FTP，FTPS，SFTP

高级

扩展此区域以指定其他上载设置。对话框中的控件集取决于服务器访问类型，有关详细信息，请参阅“ 部署：连接”选项卡 。

FTP，FTPS，SFTP

TLS

选择调用客户端安全性的方法：

  • 显式 ：应用显式（主动）安全性 。建立连接后，计算机上的FTP客户端会立即向服务器发送命令，以通过默认FTP端口建立安全控制连接。

  • 隐式 ：应用隐式（被动）安全性 。在这种情况下，建立与服务器的连接后会自动提供安全性，该服务器指定了用于安全连接的单独端口。

    请注意，不建议使用隐式方法，不建议使用它。

FTPS

Web服务器URL

在此字段中，指定Web服务器根文件夹的URL地址。请点击在字段中，以确保可以访问指定的服务器根URL地址并指向正确的网页。

HTTP和HTTPS协议均受支持。

FTP，FTPS，SFTP

不要检查与服务器的HTTP连接

  • 清除此复选框后，PhpStorm将检查指定的URL地址是否确保成功连接到服务器。

    HTTP和HTTPS协议均受支持。

  • 选中此复选框后，PhpStorm将移动到向导的下一页，而无需进行任何连接检查。

FTP，FTPS，SFTP
