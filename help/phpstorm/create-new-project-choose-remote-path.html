创建新项目：选择远程路径

档案|现有文件中的新项目-Web服务器本地安装，源文件位于本地其他位置
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过网络共享或装入的驱动器访问文件
档案|现有文件中的新项目-我的Web服务器位于远程主机上，可通过FTP / SFTP / FTPS访问文件

单击“ 创建新项目：指定远程服务器”或“ 创建新项目：添加远程服务器”页面上的“ 下一步”按钮时，页面打开。

该页面显示了服务器配置根目录（ 服务器配置根目录是通过服务器配置可访问的本地或远程服务器上文件树中的最高文件夹。对于就地服务器，它是项目根。）在此页面上，指定将成为项目根目录的文件夹，有关详细信息，请参见创建和管理项目 。

如果适用，通过选择相关文件夹并单击工具栏按钮，在项目根目录下配置文件夹 。另外，您可以稍后在“ 设置/首选项”对话框的“ 目录”页面中进行操作。

描述

项目根

单击此按钮可使所选文件夹成为项目根目录 ，有关详细信息，请参见创建和管理项目 。

请注意以下事项：

  • 指定项目根目录后，其所有子文件夹都显示在绿色背景上。

  • 其他工具栏按钮（ 从下载中排除） ， 资源根 ，并从索引中排除 ）也仅在指定项目根目录后才启用。

从下载中排除

单击此按钮可使PhpStorm在下载过程中跳过项目下的所选文件夹。这些可能是您根本不需要更新的资源，媒体，缓存，临时文件，这些实际上是您在工作中不需要的。

排除是递归应用的。这意味着，如果所选文件夹包含子文件夹，它们也将自动标记为已排除。

资源根

单击此按钮使PhpStorm可以完成所选文件夹下资源的相对路径。

从索引中排除

单击此按钮可使PhpStorm在索引，解析，代码完成等过程中完全忽略所选目录。
