PhpStorm 2019.3帮助

约曼

您可以使用Yeoman工具创建特定于框架的PhpStorm项目。

在你开始前

  1. 下载并安装Node.js。

  2. 如管理插件中所述，在“ 插件”页面上安装并启用Yeoman存储库插件。

在PhpStorm中安装和配置Yeoman

  1. 在嵌入式终端 （ Alt+F12 ），输入：

    npm install -g yo

  2. 在里面 设置/首选项对话框Ctrl+Alt+S ，转到“ 语言和框架” | JavaScript |约曼在打开的Yeoman页面上，指定全局安装的yo软件包的位置，然后选择要使用的Node.js解释器。这可以是本地Node.js解释器，也可以是Windows Subsystem for Linux上的Node.js。

您还可以按照npm和Yarn中的说明将yo软件包安装在Node.js和NPM页面上 。

配置项目存根生成器的列表

  1. 选择文件|新品|项目从主菜单或单击欢迎屏幕上创建新项目 。将打开“ 创建新项目”对话框 。

  2. 在左侧窗格中，选择Yeoman 。

  3. 右侧窗格显示了以前安装的所有Yeoman生成器 。单击安装生成器 。

  4. 在打开的对话框中显示所有可用的生成器软件包，在左侧窗格中选择所需的软件包，然后单击右侧窗格中出现的“ 安装生成器”按钮。您可以一个接一个地安装多个软件包，而无需退出对话框。

    安装结束后，单击“ 关闭”以返回已使用新添加的软件包扩展的生成器列表。

通过生成器创建项目

  1. 选择文件|新品|项目从主菜单或单击欢迎屏幕上创建新项目 。将打开“ 创建新项目”对话框 。

  2. 在左侧窗格中，选择Yeoman 。

  3. 右侧窗格显示了以前安装的所有Yeoman生成器 。要展开列表，请单击Install Generator ，请参阅上面的在PhpStorm中安装和配置Yeoman 。

    从列表中选择所需的生成器，然后单击Next 。

  4. 在启动的“ 新建项目”向导中指定所需的设置。页数及其内容取决于所选的生成器。

  5. 在向导的最后一页上，选中或清除“ 运行npm install＆bower安装”复选框，以指定是否要运行Node Package Manager和Bower来安装开发新项目所需的软件包。

  6. 单击下一步，然后选择在当前窗口或新窗口中打开新项目。
