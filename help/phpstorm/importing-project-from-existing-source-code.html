从现有源代码导入项目

您可以围绕外部创建的现有源代码（即在另一个IDE或专用编辑器中）设置项目。PhpStorm分析代码库，并在.idea目录中添加设置。

当要使用的文件已经在您的计算机上时，您可以通过三种主要方式来设置开发环境：

  • 您的计算机上有一台Web服务器，源位于服务器文档根目录下（例如/ htdocs ），然后直接在服务器上进行开发。

  • Web服务器已安装在您的计算机上，但是源存储在另一个文件夹或已安装的驱动器中。您进行开发，然后与服务器同步。

  • 您的计算机上没有Web服务器，或者现在不打算使用它。源代码在您的计算机上，您只需进行开发即可，可能使用控制台查看输出。

最后，您可能需要更新远程主机上的应用程序。在这种情况下，您还必须下载应用程序源并将其安排在项目中，因为PhpStorm不支持在远程主机上直接编辑文件。PhpStorm假定所有开发，调试和测试都在计算机上完成，然后将代码部署到生产环境中。有关更多详细信息，请参见使用Web服务器和从下载的文件创建项目一节。
