复制代码样式设置

您可以定义与预定义样式不同的代码样式。这些代码样式方案存储在XML文件中，位于用户主目录下的config / codestyles文件夹中。

您可以使用创建的副本来修改代码样式和导出。

如果选择Project以外的其他代码样式方案，则将为项目保存此代码样式。因此，您可以为每个项目分配全局（IDE）代码样式。

创建代码样式设置的副本

  1. 在“ 代码样式”页面中 ，从列表中选择所需的方案，然后单击 。

  2. 选择以下选项之一：

    • 复制到IDE ：选择此选项可将所选方案存储在全局级别。

      PhpStorm将具有指定名称的新代码样式保存在PhpStorm主目录下的config / codestyles / .xml文件中。

    • 复制到项目 ：选择此选项可将所选方案存储在项目级别。

      选定的代码样式保存在文件codeStyleSettings.xml中的.idea目录中。

    • 复制 ：选择此选项可复制所选方案并将其存储在同一级别。

  3. 在“ 方案”字段中，输入新方案的名称，然后按Enter保存更改。

  4. 应用更改。

PhpStorm允许您修改代码样式方案的现有名称，导出或导入代码样式设置。

管理代码样式方案

  1. 在“ 代码样式”页面中 ，从列表中选择所需的方案，然后单击 。

  2. 从列表中选择以下选项之一：

    • 重命名 ：更改所选方案的名称。

    • 导出 ：将您的代码样式设置导出到所需位置。

    • 导入 ：导入PhpStorm XML代码样式设置，JSCS配置文件或Eclipse XML Profile。

  3. 在“ 方案”字段中，输入新方案的名称，然后按Enter保存更改。

  4. 应用更改。
