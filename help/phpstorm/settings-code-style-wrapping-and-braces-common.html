包装和大括号

在此标签中，自定义代码样式选项，PhpStorm将在重新格式化源代码时应用这些选项。左侧窗格包含异常列表（ 重新格式化时保留 ），以及各种代码结构（列表，语句，操作，注释等）的放置和对齐选项。右侧窗格显示预览。

对齐优先于缩进选项。

右边距（列）

使用“ 硬包装在”字段可以指定元素右侧所需的页边空白空间。如果选择默认选项，则使用全局设置中右边距的值。

包装打字

使用打字设置环绕到指定编辑的文本是怎样安装在指定的硬包装的 。您可以选择以下选项之一：

  • 默认值 ：在这种情况下，PhpStorm使用全局设置中指定的“ 键入时自动换行”选项。

  • 是 ：在这种情况下，PhpStorm使用在“ 右边界”字段中指定的值。

  • 否 ：在这种情况下，此选项已关闭，并且一行可以超出右边界中指定的值。

视觉指南

使用“ 视觉参考线”字段来指定多个右边距。您可以保留默认值，也可以输入空白。如果要指定多个边距，请输入用逗号分隔的数字。

重新格式化时保持

使用复选框配置PhpStorm在重新格式化源代码时将产生的异常。例如，默认情况下，“ 换行符”复选框处于选中状态。如果您的代码包含比标准约定短的行，则可以在重新格式化源代码之前通过禁用换行符复选框来转换它们。

包装样式适用于在左侧窗格中指定的各种代码构造（例如，方法调用参数或赋值语句）。

包装方式

从此列表中选择所需的包装样式：

  • 不包裹 ：选择此选项时，不应用特殊的包裹样式。

    选择此选项后，嵌套的对齐方式和花括号设置将被忽略。

  • 如果很长则自动换行：选择此选项可使行超出正确的边距并用适当的缩进包裹。

  • 总是换行 ：选择此选项可使列表中的所有元素都换行 ，以便每行有一个具有适当缩进的元素。

  • 如果时间长，则切短 ：选择此选项可使列表中的元素超出右边距的范围，从而使每行有一个带有适当缩进的元素。

大括号放置样式

使用此列表可以在类声明 ， 方法声明和其他类型的声明中指定左括号的位置。可用的选项有：

  • 行尾 ：选择此选项可将开括号放在声明行的末端。

  • 如果被换行，则换行 ：选择此选项可将开括号放在多行声明行之后的行首。

  • 下一行 ：选择此选项可将开头括号放在声明行之后的行开头。

  • 转移下一行 ：选择此选项可在将声明行转移到相应的缩进高度后，将开括号放在该行。

  • 每次移动下一行 ：选择此选项可在将声明行移动到相应的缩进级别之后将开括号放在该行，并将下一行也移动到下一个缩进级别。
