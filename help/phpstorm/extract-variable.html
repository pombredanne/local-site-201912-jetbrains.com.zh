提取变量

提取变量重构将所选表达式的结果放入变量中。它声明一个新变量，并将该表达式用作初始化程序。原始表达式将替换为新变量（ 请参见下面的示例 ）。

要执行此重构，可以使用：

  • 就地重构 。在这种情况下，您可以在编辑器中直接指定新名称。

  • 重构对话框 ，您在其中指定所有必需的信息。要使此类对话框可访问，必须清除编辑器设置中的启用就地模式复选框。

您可以自己选择要用变量替换的表达式。您也可以使用智能表达式选择 。在这种情况下，PhpStorm将帮助您选择所需的表达式。

此重构也可用于JavaScript和样式表 。

使用就地重构提取变量

  1. 在编辑器中，选择要用变量替换的表达式。您可以自己做，也可以使用智能表达式选择功能让PhpStorm帮助您。因此，请执行以下任一操作：
    • 突出显示该表达式。然后选择重构|提取物|主菜单或上下文菜单上的变量 。

      或者，按Ctrl+Alt+V 。

    • 将光标放在表达式之前或之中。选择重构|在主菜单或上下文菜单上引入变量 。或按Ctrl+Alt+V 。

  2. 如果找到一个以上的选定表达式，请在“ 找到多个实例”弹出菜单中选择“ 仅替换此实例”或“ 替换所有实例 ”。要选择所需的选项，只需单击它。或者，使用向上和向下箭头键导航到所需的选项，然后按Enter选择它。

  3. 指定变量的名称。请执行以下任一操作：

    • 从列表中选择建议的名称之一。为此，请双击适当的名称。或者，使用向上和向下箭头键导航到所需的名称，然后Enter选择它。

    • 通过键入来编辑名称。名称将显示在带有红色边框的框中，并在您键入时更改。完成后，按Enter 。

使用对话框提取变量

如果在“编辑器”设置中清除了“ 启用就地重构”复选框，则通过“ 提取变量”对话框执行“ 引入变量重构”。

  1. 在编辑器中，选择要用变量替换的表达式。您可以自己做，也可以使用智能表达式选择功能让PhpStorm帮助您。因此，请执行以下任一操作：
    • 突出显示该表达式。然后选择重构|提取物|主菜单或上下文菜单上的变量 。

      或者，按Ctrl+Alt+V 。

    • 将光标放在表达式之前或之中。选择重构|在主菜单或上下文菜单上引入变量 。或按Ctrl+Alt+V 。

      从“ 表达式”列表中，选择表达式。为此，请单击所需的表达式。或者，使用向上和向下箭头键导航到感兴趣的表达式，然后按Enter选择它。

  2. 在“ 提取变量对话框”对话框中：

    1. 在“ 名称”字段旁边指定变量名称。您可以从列表中选择建议的名称之一，也可以在“ 名称”框中键入名称 。

    2. 如果找到一个以上的所选表达式，则可以通过选中相应的复选框来选择替换所有找到的表达式。如果只想替换当前事件，请清除替换所有事件复选框。

    3. 对于ActionScript，您可以选择引入常量而不是变量。为此，请选择“ 使常数”复选框。

    4. 单击确定 。
