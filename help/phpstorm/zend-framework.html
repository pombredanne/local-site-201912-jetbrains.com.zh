Command Line Tool Support: Zend Framework 1

File | Settings | Tools | Command Line Tool Support for Windows and Linux
PhpStorm | Preferences | Tools | Command Line Tool Support for macOS

The dialog opens when you click and choose Zend Framework 1 Tool in the Command Line Tools dialog. Use the dialog to configure Zend Framework 1 Tool support in PhpStorm.

Use the dialog to configure Zend Framework support in PhpStorm.

The tool works only with local PHP interpreters.

Item

Description

Path to zf tool

In this field, specify the location of the <Zend-Framework-home>/zf.bat file or <Zend-Framework-home>/zf file. PhpStorm parses the contents of the specified file for Zend Framework commands.
