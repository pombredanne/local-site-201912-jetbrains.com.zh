该页面仅在启用GitHub插件时可用。 的默认情况下，插件已激活。如果禁用了插件 ，请按照“ 管理插件”中所述在“ 插件”页面上启用它。

使用此页面可以指定您的GitHub远程存储帐户，或者如果您还没有该帐户，则创建一个GitHub帐户。

您可以指定多个帐户，并为每个项目设置默认帐户。因此，您无需指定每次需要连接到GitHub时使用的帐户（例如， 创建gist） 。
