为MySQL配置UTF8字符集

创建一个使用utf8字符集的MySQL数据库：

  1. 创建一个新的数据库：

    创建数据库字符集UTF8整理utf8_bin

  2. 打开< TeamCity Data Directory >/config/database.properties ，然后添加characterEncoding属性：

    connectionProperties.characterEncoding = UTF-8

要将现有MySQL数据库的字符集更改为utf8，请执行以下操作：

  1. 关闭TeamCity服务器。

  2. 来自< TeamCity Home >/bin目录，使用maintainDB tool ：

    维护数据库备份-D -F database_backup

    有关此备份的更多详细信息，请参见此处 。

  3. 如上所述，使用utf8作为默认字符集创建一个新数据库。

  4. 修改< TeamCity Data Directory >/config/database.properties通过更改文件connectionUrl属性：

    jdbc：mysql：// /

  5. 将数据导入新数据库：

    维护数据库还原-D -F database_backup -T /config/database.properties

  6. 启动TeamCity服务器。
