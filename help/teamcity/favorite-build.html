TeamCity 2019.1帮助

最喜欢的构建

要轻松访问要监视的内部版本，可以将其标记为收藏。

在此页：

  • 将构建添加到收藏夹
  • 查看您的收藏夹
  • 在收藏夹上配置通知
  • 从收藏夹中删除构建

将构建添加到收藏夹

要将构建手动添加到收藏夹，请使用“ 构建结果”页面上“ 操作”菜单中的适当选项。您最喜欢的版本将标有星号 。

如果在用户配置文件设置中启用了相应的设置，则任何手动触发的版本和个人版本都可以自动添加到收藏夹中 。

您也可以通过选中相应的框将自定义版本添加到收藏夹。

查看您的收藏夹

要查看收藏夹，请在TeamCity Web UI的右上角，单击用户名旁边的箭头，然后选择我的收藏夹构建 。

您也可以使用此页面上的相应链接，在您最喜欢的版本上配置通知。

在收藏夹上配置通知

订阅通知时 ，您可以将所选观看项目的通知或构建配置限制为您最喜欢的构建。

从收藏夹中删除构建

要从收藏夹中删除版本：

  • 在“ 最喜欢的构建”页面或“ 构建结果”页面上，单击星号 。

  • 或者，在“ 构建结果”页面上，使用“ 操作”菜单中的“ 从收藏夹中删除构建”选项。
