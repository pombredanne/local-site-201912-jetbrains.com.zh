配置.NET代码覆盖率

TeamCity使用NCover，PartCover和dotCover覆盖引擎支持.NET代码覆盖。支持通过TeamCity UI进行配置：

  • NUnit构建运行器

  • NAnt运行程序： NAnt任务

  • MSBuild运行程序： 和用于MSBuild的NUnit

对于.NET CLI（dotnet）运行程序以及NUnit 3.x版，唯一受支持的覆盖工具是JetBrains dotCover 。

如果您使用的不是NUnit测试框架，则可以使用JetBrains dotCover控制台运行程序和TeamCity服务消息手动配置覆盖率分析，如“ 手动配置报告覆盖率”中所述 。

有关配置代码覆盖率的详细信息，请参见以下页面：

  • JetBrains dotCover
  • NCover
  • PartCover
  • 手动配置报告范围
