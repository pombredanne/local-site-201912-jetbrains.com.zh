Agent Work Directory

Agent work directory is the directory on a build agent that is used as a containing directory for the default checkout directories.By default, this is the <Build agent home>/work directory.

To modify the default directory location, see workDir parameter in Build Agent Configuration.

Note that TeamCity assumes full control over the directory and can delete subdirectories if they do not correspond to checkout directories of the recent builds.

For more information on handling the directories inside the agent work directory, refer to Build Checkout Directory section.
