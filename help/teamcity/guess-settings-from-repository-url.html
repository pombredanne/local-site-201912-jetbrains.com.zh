从存储库URL猜测设置

TeamCity可以从存储库URL自动发现VCS类型和设置。

配置VCS根目录时 ，从下拉列表中选择“ 从存储库URL猜测”选项并指定URL。 TeamCity将识别支持的版本控制的URL，并将自动创建VCS根目录。创建VCS根目录后，您可以使用“项目设置”或“构建配置设置”页面修改其设置 。

VCS URL格式

VCS

URL格式

吉特

  • http(s)网址

  • git://

  • 类似于Maven的网址： http : //maven.apache.org/scm/git.html

对于SSH身份验证，请首先从URL创建VCS根目录，然后打开其设置以指定要使用的SSH密钥 。或者，手动创建一个Git VCS根目录。

水银

  • http(s)网址

  • 类似于Maven的网址： http : //maven.apache.org/scm/mercurial.html

颠覆

  • http(s)网址

  • +类似于Maven的网址： http ： //maven.apache.org/scm/subversion.html

  • svn://

TFS

推荐的URL格式：

  • http[s]://:/$/要么http[s]://:/tfs//
    例如： http：// tfshost：8080 / tfs / DefaultCollection $ / Project / root

  • 适用于Azure DevOps服务（或以前的Visual Studio Team Services）： https:///要么https:///$/
    例如： https : //username.visualstudio.com/Project

也请参阅相关的博客文章 。

Perforce

  • 类似于Maven的网址： http : //maven.apache.org/scm/perforce.html

  • 与Maven相同，但没有scm前缀，例如：perforce://main:1666/myproject/

金库

  • http(s)来自Vault SourceCode Control网站的URL，其中包含repid参数
    例如： http : //vault-server.example.net/VaultService/VaultWeb/Default.aspx?repid=1709&path=$

CVS

尚无支持

ClearCase

尚无支持
