TeamCity 2019.1帮助

代码检查

TeamCity附带了代码分析工具，该工具能够即时检查您的源代码，查找并报告常见问题和反模式。

以下检查工具与TeamCity捆绑在一起：

  • 检查（IntelliJ IDEA） ：基于IntelliJ IDEA检查运行代码分析。该跑步者进行了600多次Java，HTML，CSS，JavaScript检查。

  • 检查（ReSharper） ：在C＃，VB.NET，XAML，XML，ASP.NET，JavaScript，CSS和HTML代码中收集JetBrains ReSharper 代码分析的结果。检查结果在构建结果页面的“代码检查”选项卡中报告。

TeamCity还可以与外部报告工具集成 。
