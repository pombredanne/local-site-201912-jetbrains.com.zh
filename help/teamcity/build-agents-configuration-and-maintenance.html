查看TeamCity代理商详细信息

TeamCity Web UI的“ 代理”页面提供有关TeamCity代理的全面信息。页面上的选项卡数可能会根据您的代理设置而有所不同。

连接/断开

“ 已连接”和“ 已 断开连接”选项卡按“ 代理程序池”显示代理程序 （默认）。要按字母顺序查看座席，请取消选中“ 按座席分组池”框。
对于每个池，TeamCity都会显示其构建代理的状态。单击池旁边的箭头，将显示池代理及其状态的列表。

通过UI启用/禁用代理

TeamCity仅在启用的代理之间分发构建。通过单击代理名称旁边的状态图标（1），可以通过Web UI手动启用/禁用代理。（可选）您可以告诉TeamCity在一段时间后自动禁用/启用代理并输入您的评论。TeamCity将按照说明进行操作，并显示评论图标（2）。将鼠标悬停在图标上将显示相关信息（3）。
