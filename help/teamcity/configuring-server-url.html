配置服务器URL

当无法从任何其他参数派生URL时，服务器将使用“管理UI”（在“ 管理|全局设置”页面上）配置的服务器URL生成到服务器的链接。这些情况包括通知（电子邮件，Jabber等）以及不在Web请求中执行的其他一些操作。所有生成的链接都将以该URL为前缀。

确保可以通过指定的URL访问服务器。

在大多数情况下，TeamCity可以正确地自动检测其自己的URL并将其设置为Server URL 。但是，有时无法/正确进行自动检测（例如，当TeamCity服务器在Apache代理后面运行时）。在这种情况下，您可以使用以下选项之一指定服务器URL：

  • 关于行政|全局设置页面
