Deployers

Deployer build runners enable TeamCity to upload artifacts to external locations in a number of ways.

The following build runners are available:

  • SMB Upload
  • FTP Upload
  • SSH Upload
  • Container Deployer
  • SSH Exec

Deployer build runners are implemented as a plugin bundled since TeamCity 10.0.. For earlier TeamCity versions, use the stand-alone plugin.
