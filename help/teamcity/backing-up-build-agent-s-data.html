备份Build Agent的数据

备份构建代理程序的数据：

  1. 构建代理配置
    备份< Agent Home Directory >/conf/buildAgent.properties文件。
    您可能还希望备份任何其他已更改的配置文件（在< Agent Home Directory >/conf和< Agent Home Directory >/launcher/conf目录）。

  2. 日志文件
    如果需要Build Agent日志文件（主要用于解决问题或调试目的），请备份< Agent Home Directory >/logs目录。

您可能还希望备份Build Agent Windows服务设置（如果已修改）。
