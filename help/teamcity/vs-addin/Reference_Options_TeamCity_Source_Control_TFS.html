Team Foundation服务器

ReSharper |选项| TeamCity | Team Foundation服务器

在TeamCity加载项选项的此页面中，您可以启用和配置与Team Foundation Server VCS的集成。如果您的解决方案在Team Foundation Server下，则此集成将使您能够执行个人构建，远程运行，预先测试的提交并分析变更集，构建和测试

启用Team Foundation Server支持

选中此复选框，以允许TeamCity外接程序访问Team Foundation Server VCS。如果在Visual Studio中为当前解决方案配置了 TFS支持，则TeamCity加载项将自动检测所有必要的设置。
