对齐多行构造

方法参数

属性名称：

[resharper_]js_align_multiline_parameter ，[resharper_]align_multiline_parameter

可能的值：

true | false

例子：

真正

函数foo（firstParameter，secondParameter）{}

假

函数foo（firstParameter，secondParameter）{}

多个声明

属性名称：

[resharper_]js_align_multiple_declaration ，[resharper_]align_multiple_declaration

可能的值：

true | false

例子：

真正

var变量= 0，变量_2 = 10；

假

var变量= 0，变量_2 = 10；

对齐？：运算符

属性名称：

[resharper_]js_align_ternary ，[resharper_]align_ternary

可能的值：

  • align_all ：对齐

  • align_not_nested ：对齐（嵌套时除外）

  • none ：不对齐

例子：

align_all

var变量= 56 +（condition1吗？result1：condition2？result2：result3）;

align_not_nested

var变量= 56 +（condition1吗？result1：condition2？result2：result3）;

没有

var变量= 56 +（condition1吗？result1：condition2？result2：result3）;
