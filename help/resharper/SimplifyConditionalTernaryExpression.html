代码检查：简化条件三元表达式

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

如果您的三元表达式（形式为a ? b : c ）包含布尔常量b要么c ，可以将其转换为更简单的表达式。例如这个

公共布尔CanVote {get {return myAge> = 16吗？isCitizen：false； }}

可以简化为：

public bool CanVote {get {return myAge> = 16 && isCitizen; }}
