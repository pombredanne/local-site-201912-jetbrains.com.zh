代码检查：私有字段可以转换为局部变量

您可以配置此检查：

  • 禁用它以忽略所有相关问题
  • 在代码中的任何位置都禁止显示它以忽略特定问题
  • 更改其严重性级别，以使检测到的问题更少或更明显

ReSharper可以帮助您遵守局部性原则 ：如果仅在本地范围内分配和使用字段，ReSharper建议将字段转换为局部变量。

首先，这种转换通常会减少内存使用（但是，通过JIT优化可以最大程度地减少这种影响）。其次，它将提高代码的可读性。

此检查仅适用于构造函数。

这是此检查建议的快速修复示例：

次优代码

快速修复后

公共类Foo {private int _index;公共Foo（字符串名称）{_index =名称。LastIndexOf（'。'）;安慰。WriteLine（_index）; }}
公共类Foo {公共Foo（字符串名称）{int索引=名称。LastIndexOf（'。'）;安慰。WriteLine（index）; }}
