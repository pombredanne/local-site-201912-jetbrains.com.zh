在代码和Resx文件之间导航

ReSharper的导航功能（ 转到符号的用法和转到声明 ）可帮助您在.resx文件中的资源实现与代码文件中的用法之间进行导航。

从声明导航到用法

  1. 在XML（文本）编辑器中打开.resx文件。

  2. 将插入号放在资源名称上。

  3. 按Ctrl+Alt+F7或选择 ReSharper |导航|主菜单中符号的用法 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  4. ReSharper将打开使用资源的代码文件。如果有两种或多种用法，则可以在列表中选择一种。

从用法导航到声明

  1. 打开一个使用资源的代码文件。

  2. 将插入符号放在资源属性的用法上：

    或代码中使用的资源名称：

  3. 按Ctrl+B或选择 ReSharper |导航|从主菜单转到声明 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  4. 从列表中选择一个声明。您可以导航到生成的包装中的属性声明或.resx文件中的资源声明。

您可以从ComponentResourceManager和ResourceManager方法调用相应的资源声明。

你也可以Ctrl单击资源用法以转到其声明。

如果您使用资源中的文件（例如图像，图标，音频文件），则可以使用“ 转到相关文件”导航到这些文件。

导航到相关文件

  1. 在XML（文本）编辑器中打开一个.resx文件，然后在文件中的任意位置设置插入符号，或者，只需在解决方案资源管理器中选择该文件。

  2. 在主菜单中，选择“ ReSharper |导航|相关文件或按Ctrl+Shift+Alt+G 。

  3. 在出现的相关文件列表中，单击所需的文件或选择它，然后按Enter 。
