使用突出显示样式对话框添加/编辑/搜索

ReSharper |选项|代码检查|自定义模式添加图案/编辑
ReSharper |查找|用图案搜索

此对话框有助于创建或编辑自定义模式，以检测与特定模式匹配的代码，并在必要时替换此代码。

本主题描述对话框中的控件。要了解如何使用自定义模式搜索和替换代码，请参见结构搜索和替换以及创建自定义代码检查和快速修复 。

该对话框可以以两种模式打开。

  • 使用ReSharper时|查找|在编辑器的上下文菜单中重复的“ 使用模式搜索”命令，该对话框在搜索/替换模式下打开，您可以使用底部的相应按钮从对话框中开始搜索或替换。

  • 当您使用“ 代码检查” | “ 添加” /“ 编辑”按钮时在“ ReSharper”选项的“ 自定义图案”页面上，该对话框在图案编辑模式下打开，提供了用于指定图案描述的其他字段。但是，在此模式下，“ 搜索 / 替换”按钮不可用。

匹配类似的结构 使用此控件可以指定是否匹配相似的构造。
ReSharper认为以下构造类似：

  • 用大括号括起来的单行语句和单行语句。

  • 在模式中以相反的顺序指定的二进制表达式。

  • 模式中指定的表达式以及具有不同操作数顺序的表达式。

  • 用括号括起来的表达式和没有用括号括起来的表达式。

  • 后缀和前缀增量运算符。

  • 具有不同访问修饰符的方法。

抑制键

如果为模式指定了抑制键，则将能够抑制ReSharper检查 ，该检查会检测到模式的使用情况，并带有注释，例如： // ReSharper disable once [SuppressionKey] 。
如果要用单个注释取消对多个模式的检查，则可以为所有这些模式指定相同的取消键。

占位符区域
添加占位符

单击此按钮创建新的占位符。占位符有5种类型：

  • 参数占位符 -方法调用中的一个或多个参数。如有必要，可以指定应匹配的最小或最大数量的参数。

  • 表达式占位符 -一系列运算符和操作数。您可以选择指定此表达式返回的类型。

  • 标识符占位符 -任何符号标识符。您还可以指定用于匹配符号名称的正则表达式。

  • 语句占位符 -以分号或语句块结尾的单行语句。如有必要，可以指定应匹配的最小或最大语句数。

  • 类型占位符 -值类型或引用类型。默认情况下，这种占位符将匹配任何类型，但是您可以显式指定特定类型。

您可以仅在搜索模式区域中键入新的占位符用法，例如$statement$ ，新的占位符将添加到占位符列表，您可以在其中编辑其类型和参数。

单击此按钮后，将打开“ 创建占位符”对话框，您可以在其中选择占位符的类型，指定其名称和参数：

在此对话框中可以使用以下控件：

  • 名称 -指定占位符名称（对于所有占位符）。

  • 限制参数/声明的最小数量-定义占位符出现的最小数量（用于参数和语句占位符）

  • 最大值 -定义占位符出现的最大数量（用于参数和语句占位符）

  • 表达式类型/类型 -从列表中选择一个项目以指定占位符的类型（用于表达式和类型占位符）

  • 或派生类型 -选中此复选框以匹配已选择类型或其派生类型的表达式或类型（用于表达式和类型占位符）

  • 标识符名称regexp —为标识符名称定义正则表达式（对于标识符占位符）

  • 区分大小写 -指定标识符是否区分大小写（对于标识符占位符）

编辑

单击此按钮或双击占位符名称，以使用“ 创建/编辑占位符”对话框编辑选定的占位符 。

去掉

单击此按钮以删除所选的占位符。请注意，您无法删除搜索模式中使用的占位符。

命令按钮

查找 （隐藏替换图案时在查找/替换模式下可用）

单击此按钮执行搜索。

替换 （显示替换图案时在查找/替换模式下可用）

单击此按钮以查找模式出现并指定要替换的模式。

保存

单击此按钮可以使用智能保存逻辑将当前模式保存到目录中。要稍后访问模式目录，请打开“ 代码检查” |“命令检查”。 ReSharper选项的“ 自定义模式”页面。

保存副本 （在图案编辑模式下可用）

单击此按钮将当前模式的副本保存到目录中。如果要创建一个与当前模式相似的新模式，这可能会有所帮助。与“保存”命令类似， 智能保存逻辑在此处适用。
