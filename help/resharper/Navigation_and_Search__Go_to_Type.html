ReSharper |导航|搜索无处不在/键入...
Ctrl+N
ReSharper_GotoType

此命令结合了两个不同的功能：

  • 首先出现的Search Everywhere允许您导航到以下目标：类型，符号，文件，源文件和文本文件中的字符串文字，最近的编辑，最近的文件以及最近查看的方法。建议列表会在您调用此功能后立即出现，并且最初包括您的近期文件和导航的项目。

  • 转到“类型” ，这是您第二次按Ctrl+N或从菜单调用命令，使您可以导航到任何类型。

默认情况下，文本导航已集成到“无处不在搜索”中 -ReSharper将在所有匹配类型，符号和文件之后，在结果列表的末尾显示与查询字符串匹配的文本出现。您可以通过清除“在以下位置将“将文本全部集成到搜索中”来禁用此行为 环境|的搜索和导航页面 ReSharper选项。
如果禁用此选项，则可以通过按导航到文本 Ctrl+N三次。

您可以从导航结果中排除文件，文件夹和文件掩码。为此，请在“ 要跳过的元素”部分中指定这些项目 环境|的搜索和导航页面 ReSharper选项（ Alt+R,,O ）。

使用这些功能，您可以在整个解决方案中的符号和文件，以及解决方案项目中引用的所有程序集。 搜索结果还包括来自“程序集资源管理器”窗口中当前加载的所有程序集的匹配项。

请注意，此命令在全局范围内有效，也就是说，您无需将焦点放在编辑器上即可调用它。

在各处搜索或查找类型

  1. 按Ctrl+N或选择 ReSharper |导航|从主菜单中搜索“无处不在/转到类型... ”。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  2. 在出现的“ 搜索各处”列表中，开始输入项目名称。键入时，项目列表会变窄，并建议与输入的子字符串匹配的名称。

  3. 要进一步缩小项目列表，您可以输入/应用过滤器。例如，您可以输入/me仅显示事件。请注意，您可以在搜索查询之前和之后输入过滤器。

  4. 要切换到“ 转到类型”功能，请单击菜单命令或再次按快捷键。将显示“ 输入类型名称”列表，您可以在其中指定要查找的类型。

  5. （可选）选择“ 包括库类型”或按Alt+N显示解决方案中引用的库中的匹配项。
    默认情况下，只要您的输入与解决方案中的某些内容匹配，就只显示解决方案项。如果解决方案中的输入不匹配，则ReSharper会自动开始寻找匹配的库类型。

  6. 请执行以下任一操作：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找结果中显示 ， 按Shift+Enter要么+在数字键盘上。

如果您导航到的项目属于当前解决方案，则ReSharper在编辑器中打开相应的文件，并将插入符号放置在符号声明中。如果在引用的库中找到它，则ReSharper会根据“ 工具” | “ 工具 ”中定义的设置进行导航。 ReSharper选项的“ 外部源”页面。

ReSharper还可以找到以任何顺序匹配查询部分的项目。例如，搜索exactMatching将匹配isMatchingExactly 。
