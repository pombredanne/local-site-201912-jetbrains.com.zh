C ++的预定义环绕声模板

ReSharper |模板资源管理器|环绕模板| C ++

本主题列出了ReSharper 2019.2中C ++的所有预定义环绕模板。有关环绕声模板的更多信息，请参见带有模板的环绕声代码片段。

模板

细节

()

用括号括起来的选择

范围 C ++表达式，C＃2.0+

身体

（$ SELECTION $$ END $）

参量

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

{}

用大括号括住选择

范围 C ++语句，C＃2.0+

身体

{$ SELECTION $$ END $}

参量

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

forr

反转为

范围 C ++语句

身体

for（int $ INDEX $ = $ MAX $-1; $ INDEX $> = 0;-$ INDEX $）{$ SELECTION $$ END $}

参量

  • INDEX -建议未使用的名称，该名称可以在评估点用于索引变量。 （ proposalIndexVariable ）

  • MAX -没有宏

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

foreach

“基于范围的”循环

范围 C ++语句

身体

为（$ TYPE $ $ VARIABLE $：$ COLLECTION $）{$ SELECTION $$ END $}

参量

  • COLLECTION -在评估变量的位置显示基本代码完成列表（ 完整 ）

  • TYPE -在评估变量的位置显示智能代码完成列表（ completeSmart ）

  • VARIABLE -在变量声明中执行时（变量名应保留在其中），建议变量名。 （ proposalVariableName ）

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

范围 C ++语句

身体

参量

  • EXPR -计算为指定的常数。 （ 常量 ）

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

fori

带整数索引的“ for”循环模式

范围 C ++语句

身体

for（int $ INDEX $ = 0; $ INDEX $ <$ UPPER $; ++ $ INDEX $）{$ SELECTION $$ END $}

参量

  • INDEX -建议未使用的名称，该名称可以在评估点用于索引变量。 （ proposalIndexVariable ）

  • UPPER -没有宏

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

incg

带有护罩的环绕声选择

在行开始时作用域 C ++顶级声明

身体

#ifndef $ FILENAME $ #define $ FILENAME $ $ SELECTION $ #endif // $ FILENAME $

参量

  • FILENAME -以大写字母评估当前文件名，所有非字母数字均用下划线（ getUpperCaseAlphaNumericFileName ）替换

  • DATE -评估文件创建日期和时间（ getCreationTime ）

  • SELECTION -调用模板之前用户选择的文本。

身体

参量

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。
