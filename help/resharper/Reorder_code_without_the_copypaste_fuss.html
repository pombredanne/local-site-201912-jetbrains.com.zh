重新排序代码，而无需进行复制粘贴操作

您是否曾经想过不使用通常的复制粘贴例程对代码成员（例如，类中的成员或只是成员语句）进行重新排序？使用ReSharper，这很容易。

有关详细信息，请参阅重新排列代码元素 。

这里要记住的捷径是Ctrl+Shift+Alt +箭头按钮。按住Ctrl+Shift+Alt结合使用时，箭头键有多种用途，具体取决于插入符号在代码中的位置以及所选的代码块。这里有些例子：

  • 在执行诸如添加之类的操作时，按向左和向右箭头键a + b如果可能，将操作数向左或向右移动。例如，将光标放在b ，按Ctrl+Shift+Alt+Left将表达式更改为b + a 。

  • 按向上和向下箭头键可以上下移动整个语句。注意：如果要在语句后的代码块内移动语句，请使用向右/向左键，而不要使用向上/向下键。

  • 在方法声明中有尖号，向上/向下键（以及Ctrl+Shift+Alt （当然）），使其相对于其他成员声明上下移动。这使您可以按自己的意愿对字段，属性，方法和事件进行重新排序。

由于许多重构都依赖于所选择的确切代码元素，因此ReSharper提供了有用的快捷键，以扩展和缩小选择范围 。扩展可以从单个标识符扩展到整个语句，再扩展到所有语句，再到包含方法，等等。使用Ctrl+W扩展选择范围并Ctrl+Shift+W缩小它。
