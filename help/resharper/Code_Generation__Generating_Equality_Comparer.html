生成平等比较器

ReSharper |编辑生成代码|平等比较器
Alt+Insert | 平等比较器
ReSharper_GenerateEqualityComparer

IEqualityComparer是一个通用的.NET接口，允许对集合实施自定义的相等比较。

为您的类型创建比较器类是为该类型创建Equals（）和GetHashCode（）方法的替代方法 。生成的比较器类将实现IEqualityComparer界面并提供自定义Equals()和GetHashCode() ） 方法。

ReSharper提供了“ 生成相等比较器”命令，以自动生成比较器类。

在下面的示例中，此命令用于根据以下内容生成比较器类： _radius和_center领域。

生成相等比较器类

  1. 在编辑器中，将插入号设置为类型名称，或在要插入相等比较器类的行的类型内。如果插入符号在类型名称上，则生成的代码将添加到类型声明的开头。

  2. 按Alt+Insert或选择 ReSharper |编辑从主菜单生成代码… 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 平等比较器” 。

  4. 在出现的“ 生成”对话框中，选择要在比较器类中使用的字段。

    如果您的类型中没有字段/属性，或者您未选择任何字段/属性，ReSharper会根据您的设置抛出新的NotImplementedException() ，返回默认值或将无法编译的代码放入生成的方法的主体中。您可以在“ 代码编辑” |“配置”中配置设置。 ReSharper选项的“ 成员生成”页面。

    （可选）在对话框中使用以下控件：

    • 字段可以为null-如果您的类型中包含任何可为空的字段或属性，则显示该字段。默认情况下，此复选框处于选中状态，这意味着ReSharper将为所选字段生成空检查。如果不需要空检查，则可以清除此复选框。

    • 字符串比较 -如果您的类型中有任何字符串字段，则会出现。ReSharper将生成string.Compare(string, string, StringComparison)对于选定的字符串，选择器使您可以选择为StringComparison参数生成的值。

    • 通过静态属性公开-将比较器类设为私有，并生成将比较器类暴露给使用者的静态属性。

    • 比较器名称前缀 -允许您指定将在生成的比较器类的名称中使用的前缀。

  5. 单击完成以完成向导。
    您也可以在“ 代码编辑” |“选项”上单击“ 选项”以查看或修改常见的代码生成首选项。 ReSharper选项的“ 成员生成”页面。

要将此代码生成命令绑定到其自己的快捷方式，请转到“ 工具” |“其他”。选项|环境|键盘并为ReSharper_GenerateEqualityComparer命令。
