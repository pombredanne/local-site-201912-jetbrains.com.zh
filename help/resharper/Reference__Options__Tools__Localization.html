本土化

ReSharper |选项|代码编辑|本土化

在ReSharper选项的此页面中，您可以指定新资源名称的模板。当ReSharper使用本地化重构和本地化快速修复自动创建新资源时，将应用此模板。

项目

描述

资源名称模板

您可以在模板中使用纯文本或变量。预定义了以下变量：

  • $TypeName$ —包含可本地化字符串的类型的名称

  • $MemberName$ —包含可本地化字符串的成员的名称

  • $Value$ —作为资源值添加的可本地化的字符串

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
