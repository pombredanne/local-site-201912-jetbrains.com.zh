ReSharper 2019.2帮助

指定许可证信息

ReSharper是一种商业软件产品，受国际法版权保护。产品的每个副本均需要许可证。

对于新用户，ReSharper提供30天的免费评估期。在此期间，您可以享受产品的全部功能，并确定其是否适合您的需求。可以使用“ 许可证信息”对话框中的“ 暂停/继续评估”按钮来暂停和恢复评估期。在评估期间，您将能够看到还剩多少天。

许可证信息

如果您决定在评估期之后使用该产品，则需要在ReSharper网站上获得许可证并以下列方式之一指定许可证信息：

  • 可以通过JetBrains帐户管理JerBrains产品的所有个人和商业许可证，该帐户可以用作指定许可证信息的方式。

  • 可以使用许可证密钥指定所有个人和商业许可证。

  • 对于公司客户，ReSharper允许通过JetBrains许可证服务器存储和分发许可证。该服务器允许企业建立有限数量或无限数量的产品许可证，并在企业网络内分发它们。这种方法的好处是，一旦用户关闭应用程序，他们的许可证便会自动返回服务器，并可供其他用户使用。

您可以在“ 许可证信息”对话框或“ 关于”对话框中检查许可证的状态。

指定您的许可证信息

  1. 在主菜单中，选择“ ReSharper |帮助|许可证信息 。

  2. 在打开的“ 许可信息”对话框中，使用复选框指定一个或多个许可选项（如果您拥有多个以不同方式许可的JetBrains .NET产品，则可能需要几个选项）：

    • 使用JB帐户 -如果您的JetBrains帐户绑定了ReSharper许可证，请选择此选项。
      要指定您的帐户，请单击添加帐户 ，然后在出现的对话框中输入您的帐户凭据，然后点击登录 。成功登录后，ReSharper将显示可用许可证。如有必要，您可以清除不想使用的许可证旁边的复选框。
      请注意，指定JetBrains帐户凭据时应连接到Internet。成功登录后，不需要Internet连接。

    • 使用许可证密钥 -如果您具有ReSharper的许可证密钥，请选择此选项。
      要指定您的许可证密钥，请单击添加许可证密钥。 ，然后在出现的对话框中输入您的用户名和许可证密钥，然后单击添加 。如果许可证有效，您将看到相应的消息。

    • 使用许可证服务器 -如果您的公司使用JetBrains许可证服务器分发许可证，请选择此选项。
      如果许可证服务器在您的本地网络中，则ReSharper通常会检测到该服务器。如果未自动检测到服务器，请单击“ 添加许可证服务器”。 ，然后在出现的对话框中，指定服务器URL并单击添加 。如果服务器上有ReSharper许可证，您将看到相应的消息。

  3. 通过任何许可选项提供有效的产品许可证后，带有许可证详细信息的产品旁边就会出现一个绿色的复选标记。

  4. 单击“ 关闭”以应用选定的许可选项，然后关闭对话框。

来自JetBrains许可服务器的许可票

如果使用的是JetBrains许可证服务器 ，则可以获得两种类型的许可证。默认情况下，服务器为您颁发浮动票证，该票证在您连接到服务器后一直有效。如果您需要脱机工作，则可以获得永久许可证。

浮动许可证票 （默认）

在启动时，ReSharper会在本地网络中找到许可服务器，并请求一个票证，该票证将返回到服务器，并在ReSharper关闭时可供其他用户使用。票证的获取和归还都是自动完成的。

许可证服务器上的许可证数量可以少于ReSharper用户数量。例如，如果有60名工程师使用ReSharper，但是最多只有50名工程师可以同时使用ReSharper，那么许可证服务器上的50个许可证就足够了。

要启动ReSharper，需要通过局域网或VPN连接到许可证服务器。

永久许可证

一旦获得，永久票证将保留在客户端计算机上。从客户端返回之前，其他用户无法使用它。票证的获取和返还都是在客户端上或由许可证服务器管理员手动完成的。

客户端不需要网络连接即可运行ReSharper。

即使客户端不使用永久票证获得的许可证，其他用户也无法使用。

从许可证服务器获取永久许可证票证

  1. 在“ 许可证信息”对话框中 ，检查当前是否使用了来自许可证服务器的浮动票证（应将其写在产品许可证详细信息中）。

  2. 单击许可证服务器选项旁边的“ 请求永久” 。

  3. 在打开的“ 获取永久许可”对话框中，指定您的电子邮件地址，然后点击发送请求 。

  4. 检查您的收件箱，并从许可证服务器中找到带有激活密钥的消息。

  5. 在激活密钥字段中，输入收到的激活密钥，然后单击激活许可证 。

  6. 单击关闭 。请注意，如果您的许可证服务器分发了有限数量的许可证，则您持有的许可证票证将对其他用户不可用，直到您将其退还为止。

发行永久许可证

  • 在主菜单中，选择“ ReSharper |帮助|归还永久许可证 。在确认对话框中，单击是 。

  • 在主菜单中，选择“ ReSharper |帮助|许可证信息 。在打开的“ 许可证信息”对话框中，单击“ 发布永久许可证” 。
