XML中的代码检查

在XML中，ReSharper 2019.2提供了两种代码检查：13种检查，用于检测错误的语法，例如语法错误，无法解析的符号，编译器错误等（您无法配置或禁用这些检查中的任何一种），以及2 专有代码检查 ，您可以禁用或更改其严重性水平 。
这些代码检查可帮助您在设计时检测所有打开文件中的代码问题，此外，它们还使您可以查找特定范围内的代码问题 。

下面列出了可配置的XML检查，按其类别分组。

拼写问题（2次检查）

这些检查可以检测各种情况下的错别字。
