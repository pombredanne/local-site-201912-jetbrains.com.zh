编辑页面

贡献

为了在本地构建站点，您将需要安装Ruby（该站点基于Jekyll ）。建立网站的步骤是：

  1. 安装Ruby
  2. 安装捆绑器-gem install bundler
  3. 克隆此仓库
  4. 初始化并克隆子模块- git submodule init和git submodule update
  5. 下载依赖项（宝石）-rake bootstrap
  6. 建立和预览网站rake preview

这将使Jekyll在以下位置运行站点http://localhost:4001/help/resharper/sdk/
