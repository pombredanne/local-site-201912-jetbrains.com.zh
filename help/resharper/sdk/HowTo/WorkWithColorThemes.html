使用颜色主题

可以根据所选主题自动更改用户界面颜色。 ColorThemeManager零件。例如，我们要更改某些UI控件的背景和前景色：

public class UiThemeViewModel : AAutomation { public IProperty BackgroundColor { get; set; } public IProperty TextColor { get; set; } public UiThemeViewModel(Lifetime lifetime, IColorThemeManager colorThemeManager) { BackgroundColor = new Property(lifetime, "UiThemeViewModel.BackgroundColor") {Value = new SolidColorBrush()}; TextColor = new Property(lifetime, "UiThemeViewModel.TextColor") {Value = new SolidColorBrush()}; var bgColor = colorThemeManager.CreateLiveColor(lifetime, ThemeColor.ToolWindowBackground); bgColor.ForEachValue(lifetime, (lt, color) => { BackgroundColor.Value.Color = color.WpfColor; }); var txtColor = colorThemeManager.CreateLiveColor(lifetime, ThemeColor.ToolWindowForeground); txtColor.ForEachValue(lifetime, (lt, color) => { TextColor.Value.Color = color.WpfColor; }); } }

笔记

  • 的CreateLiveColor的方法ColorThemeManager退货IProperty绑定到特定值ThemeColor 。
  • 的ForEachValue的方法IProperty允许每次更改时分配更新的颜色。与之相比的主要区别Advise方法是ForEachValue通过每个值的生命周期（ lt ）。每次将新值分配给属性时，它都会终止，并允许为每个新值定义一对动作（值后收益-值前收益）。
