2。为插件创建一个NuGet包
编辑页面

1。建立专案

要创建插件项目：

  1. 在Visual Studio中，创建一个类库 （Visual C＃）类型的新项目。
  2. 参考ReSharper SDK。它是通过NuGet分发的，因此，您可以通过包管理器引用SDK：
    • 开启工具| NuGet软件包管理器|管理NuGet软件包以获取解决方案… 。
    • 在打开的窗口中，找到JetBrains。ReSharper。SDK 。
    • 在“ 版本”中 ，选择软件包的版本。它必须与您的插件目标的ReSharper版本匹配。
  3. 点击安装 。
  4. 生成项目以获取.dll文件。

现在，您应该为该插件创建NuGet包 。
