大括号布局

命名空间声明

属性名称：

[resharper_]cpp_namespace_declaration_braces ， [resharper_]cpp_brace_style ， [resharper_]namespace_declaration_braces ，[resharper_]brace_style

可能的值：

  • end_of_line ：行尾（K＆R样式）

  • end_of_line_no_space ：行尾（无空格）

  • next_line ：在下一行（BSD样式）

  • next_line_shifted ：在下一行缩进（铁匠风格）

  • next_line_shifted_2 ：在下一行缩进2（GNU样式）

例子：

行结束

命名空间ns {void a（）; }

end_of_line_no_space

命名空间ns {void a（）; }

next_line

命名空间ns {void a（）; }

next_line_shifted

命名空间ns {void a（）; }

next_line_shifted_2

命名空间ns {void a（）; }

其他牙套

属性名称：

[resharper_]cpp_expression_braces ，[resharper_]expression_braces

可能的值：

  • inside ：圆括号（BSD / K＆R样式）

  • outside ：括号和内部均等（铁匠风格）

  • outside_and_inside ：括号1x，内部2x（GNU样式）

  • none ：无缩进

例子：

内

int a [10] = {1，2，3};

外

int a [10] = {1，2，3};

outside_and_inside

int a [10] = {1，2，3};

没有

int a [10] = {1，2，3};
