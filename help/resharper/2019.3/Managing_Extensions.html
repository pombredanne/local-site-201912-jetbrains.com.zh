ReSharper 2019.3帮助

管理扩展

扩展添加了其他功能，例如其他上下文操作，代码分析器，对编程语言的支持等等。为了简化安装，更新和卸载不同扩展的过程，ReSharper提供了一个管理器，该管理器会自动为您执行所有常规操作。您可以浏览所有可用的扩展，然后单击几次即可安装。

安装扩展

  1. 在主菜单中，选择“ ReSharper |扩展经理 。

  2. 在打开的扩展管理器对话框中，签出可用的扩展。
    此对话框中列出的扩展名来自可用的扩展包来源。默认情况下，只有ReSharper Gallery被启用为软件包源，但是您可以添加自己的软件包源 。

  3. 单击要添加的扩展旁边的“ 安装” ，然后单击对话框底部的“ 安装 ”。

您还可以安装个人或公司范围的扩展程序中的扩展程序，这些扩展程序可以位于任何位置。一旦添加了新的扩展程序包源，就可以从该源安装扩展。本地还是远程都没有区别。因此，如果您在公司中开发了一组扩展，则需要公开分发它们。

ReSharper扩展以NuGet软件包的形式分发。请查阅ReSharper DevGuide，以了解如何创建和打包扩展。

要将源添加到包源列表中

  1. 选择ReSharper |在主菜单或按选项 Alt+R,,O ，然后选择环境|扩展管理器在左侧。

  2. 点击添加 。

  3. 在出现的“ 添加程序包源”对话框中，指定新源的显示名称以及本地或远程文件系统路径或自定义的NuGet feed。

  4. 单击“ 确定”将新项目添加到包来源列表中。

  5. 请点击单击保存以应用修改，并让ReSharper选择将修改保存到何处，或使用“ 保存到”列表将修改保存到特定的设置层。有关更多信息，请参阅管理和共享转发器设置 。
