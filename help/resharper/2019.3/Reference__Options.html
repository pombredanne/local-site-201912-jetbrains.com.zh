Some project-specific properties, for example target C# version, are configured under the ReSharper property group in the Visual Studio's Properties window.

You can also save any ReSharper preferences per project, for more information, see JetBrains .NET Tools Blog: Per Project Settings or How to have different naming styles for my test project

Click Manage to access the Settings Layers dialog that helps manage settings layers. For detailed instructions, see Manage and Share ReSharper Settings.
