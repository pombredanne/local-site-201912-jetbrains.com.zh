Project/PDB Generation Status window

ReSharper | Windows | Pdb Generation Status

This tool window records and displays all events related to generating symbol files and projects. New entries in this window appear in the following cases:

  • you manually generate a PDB file for an assembly.

  • you export an assembly to a Visual Studio project.

If some errors occur during generation, the corresponding error message is displayed.

Toolbar Controls

Control

Name

Description

Cancel generations

Lets you abort selected running generation processes.

Clear list

Clears the list of generation processes. This will also abort all running processes.
