内联类重构

ReSharper |重构|内联|排队…
Ctrl+Alt+N
ReSharper_InlineVariable

这种重构使您可以将一个类合并到另一个类中。删除您合并的类，将其成员移至使用合并的类的类，并相应地更新合并的类的所有用法。

提取类重构重构具有反向功能。

在下面的示例中，我们合并了Painter类（即其字段myColor及其方法InitPainter ） 进入Circle类。

重构之前

重构后

class Painter {private Color myColor; Public Painter（Color c）{myColor = c; InitPainter（myColor）; } private void InitPainter（Color color）{//初始化painter}} class Circle {private Painter myPainter; public Circle（Color c）{myPainter = new Painter（c）; }}
类Circle {private Color myColor; public Circle（Color c）{myColor = c; InitPainter（myColor）; } private void InitPainter（Color color）{//初始化画家}}

内联课程

  1. 将插入号放在表示您要合并的类的属性的名称或用法或字段中。或者，在“ 文件结构”窗口中选择一个属性或字段。

  2. 请执行以下任一操作：

    • 按Ctrl+Alt+N然后选择内联类

    • 按Ctrl+Shift+R然后选择内联类

    • 右键单击并选择“ 重构”。上下文菜单中的内联类 。

    • 选择 ReSharper |重构|内联|内联…在主菜单中。

  3. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。

应用重构后，该属性或该字段将被其类型的成员替换。属性或字段的所有用法都会相应更新。
