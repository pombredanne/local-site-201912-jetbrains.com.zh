NuGet恢复器

ReSharper |选项|工具|建立NuGet恢复器

在ReSharper选项的此页面上，您可以配置ReSharper的NuGet软件包还原程序。

ReSharper还提供了自己的NuGet恢复器的实现，该实现可与所有Visual Studio版本一起使用。默认情况下，ReSharper确保在每次构建之前还原所有相关的NuGet软件包，但是您可以通过清除“ 工具” | “相应工具”上的相应复选框来禁用它。建立 ReSharper选项的常规页面。如果禁用了自动还原，则始终可以通过单击“还原NuGet软件包”来手动运行它在“ 构建和运行”窗口的工具栏上。

项目

描述

用于将NuGet软件包恢复程序消息写入输出的日志级别

如果在“ 工具” |“启用”中启用了“登录到输出”窗口的记录建立 ReSharper选项的“ 常规”页面，您可以使用此选择器指定应记录哪些NuGet恢复器消息。

显示日志

单击此按钮可在当前Visual Studio会话中显示来自NuGet恢复程序的日志消息。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
