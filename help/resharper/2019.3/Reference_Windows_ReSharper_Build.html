生成并运行窗口

ReSharper | Windows |生成并运行

该窗口为您提供两个目的：

  • 它使您可以启动ReSharper Build ，可视化其进度和状态。

  • 它允许您执行运行配置 。

控制

名称

描述

建立

生成当前解决方案或允许您选择其他生成操作： 重建或清理

运行配置选择器

显示活动的运行配置，允许在配置之间进行切换并添加新的配置。有关更多信息，请参见运行配置 。

跑

运行活动的运行配置。

无需构建即可运行

在不构建解决方案的情况下运行活动的运行配置。

除错

开始调试活动配置。

无需构建即可调试

开始调试活动配置而不进行构建。

恢复NuGet软件包

使用此按钮，您可以使用ReSharper自己的NuGet恢复器实现来为当前解决方案恢复NuGet软件包。
默认情况下，在每次构建之前也会自动启用NuGet软件包的自动还原。如有必要，您可以通过清除“ 工具” |“ 创建之前还原NuGet软件包”复选框来禁用它。建立 ReSharper选项的常规页面。

开始分析

使用此下拉按钮，您可以在所需的分析模式下 使用JetBrains dotTrace对选定的运行配置进行 分析 。

设定值

允许您在ReSharper Builder和Visual Studio Builder之间切换，并打开“ 工具” | “ 工具”。 ReSharper构建| ReSharper选项的常规页面。
