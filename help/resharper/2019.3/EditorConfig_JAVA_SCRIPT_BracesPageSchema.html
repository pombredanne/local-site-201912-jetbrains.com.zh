缩进括号，方括号和表达括号

属性名称：

[resharper_]js_indent_pars ，[resharper_]indent_pars

可能的值：

  • inside ：圆括号（BSD / K＆R样式）

  • outside ：括号和内部均等（铁匠风格）

  • outside_and_inside ：括号1x，内部2x（GNU样式）

  • none ：无缩进

例子：

内

a = {1，2，3}; b =（1 + 2）

外

a = {1，2，3}; b =（1 + 2）

outside_and_inside

a = {1，2，3}; b =（1 + 2）

没有

a = {1，2，3}; b =（1 + 2）
