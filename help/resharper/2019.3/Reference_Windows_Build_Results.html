构建结果窗口

ReSharper | Windows |建立结果
ReSharper_ShowBuildResultsBrowserWindow

使用ReSharper生成解决方案时，“ 生成结果”窗口将显示生成错误以及警告（可选）。

您可以通过双击结果快速打开包含构建错误和警告的文档。

工具栏控件

控制

名称

描述

全部展开 / 全部折叠

展开/折叠当前选项卡中的所有节点。

上一页 / 下一页
Shift+F8 / F8

导航到上一个/下一个项目，并在源代码中滚动。

显示预览 Ctrl+P

隐藏或显示窗格，该窗格在使用列表指定的位置（在窗口的底部或右侧）显示所选项目的预览。

出口

点击这个按钮以文本格式导出当前在窗口中显示的数据，或使用下拉选择器以XML或HTML格式导出数据。出现的“ 导出数据”对话框将帮助您将数据保存到文件或将其复制到剪贴板。

显示警告

用于显示或隐藏构建警告。请注意，如果存在构建错误，则构建后始终会隐藏警告，因此要查看警告，构建错误后每次都必须单击此按钮。

通过...分组

允许按不同类别对窗口中的项目进行分组。
