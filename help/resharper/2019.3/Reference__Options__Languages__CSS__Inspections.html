检查浏览器兼容性

这组选项允许您启用或禁用CSS中特定于浏览器的代码检查。

构建CSS样式层次结构

启用此选项后，ReSharper的代码分析引擎将构建CSS样式的层次模型。这可能会导致一些轻微的性能下降，但是如果构建了层次结构，则ReSharper会添加动作指示器 （装订线标记），使您可以查看某些样式是否覆盖其他样式，并在编辑器中直接在层次结构上导航：


无论使用哪种选项，您都可以始终手动构建CSS层次结构。有关更多信息，请参见查看CSS层次结构 。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
