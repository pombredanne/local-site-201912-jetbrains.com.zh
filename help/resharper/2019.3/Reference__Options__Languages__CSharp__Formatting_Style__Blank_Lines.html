空行

ReSharper |选项|代码编辑| C＃|格式样式|空行

此页面使您可以配置ReSharper是否应增加或减少导入指令的名称空间，成员，区域和组周围的空行数。 您可以调整值并检查页面底部的预览窗格，以查看首选项如何影响代码。

“ 保留现有格式”部分中的选项仅在ReSharper 重新格式化现有代码的格式时适用，而“ 空白行”部分包含的选项也会在您键入新代码时生效。

您在此页面上配置的选项在以下情况下适用：

  • 当代码在编辑时或粘贴后自动格式化时 （您可以在 环境|编辑器的“编辑者行为”页面 ReSharper选项（ Alt+R,,O ）。

  • 当ReSharper 产生具有代码完成和代码生成功能的新代码，应用代码模板并执行重构 。

  • 重新格式化现有代码时 。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
