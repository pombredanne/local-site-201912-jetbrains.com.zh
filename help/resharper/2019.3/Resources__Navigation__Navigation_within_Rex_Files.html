ReSharper 2019.3帮助

在Resx文件中导航

如果您更喜欢使用XML编辑器来处理.resx文件，则ReSharper将帮助您导航和搜索这些文件中的资源。“ 转到文件成员”命令可在.resx文件中完美运行，并允许您通过几次击键跳到任何资源。

在编辑器中浏览资源

  1. 在XML（文本）编辑器中打开.resx文件。

  2. 按Ctrl+F12或选择 ReSharper |导航|从主菜单转到文件成员… 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 您将在弹出窗口中看到资源条目列表。您可以开始键入以缩小列表范围。

  4. （可选）您可以勾选一个复选框，以包括相关文件中的成员。

  5. 当您在列表中看到所需的字符串时，选择它并按Enter或单击它。

  6. 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找结果中显示 ， 按Shift+Enter要么+在数字键盘上。

ReSharper的“ 文件结构”窗口可以显示资源文件的结构。它列出了资源文件中的所有资源条目，可视化了资源ID，当前区域性中的资源文本以及资源注释（如果有）。此外，此窗口还指定在其他区域性中是否覆盖资源条目，以及它们是否包含格式占位符。

从文件结构导航到资源

  1. 在XML（文本）编辑器中打开.resx文件。

  2. 按Ctrl+F11或选择 ReSharper | Windows |从主菜单中的文件结构 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在出现的“ 文件结构”窗口中，选择所需的字符串，然后双击它以在编辑器中找到它。

  4. （可选）您可以使用“ 文件结构”窗口中提供的导航和重构功能。
