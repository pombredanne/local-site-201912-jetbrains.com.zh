堆栈跟踪资源管理器选项

ReSharper |选项|工具|堆栈跟踪资源管理器

在ReSharper选项的此页面中，您可以配置Stack Trace Explorer窗口 。

项目

描述

自动打开外部堆栈跟踪

默认情况下，如果您复制包含当前打开的解决方案中的符号的堆栈跟踪，然后将焦点切换到Visual Studio，则ReSharper会自动将其加载到“ 堆栈跟踪资源管理器”窗口并将其带到视图中。如果您不喜欢此行为，请清除“ 工具” | “ 自动打开外部堆栈跟踪”复选框。 ReSharper选项的“ 堆栈跟踪资源管理器”页面。清除此复选框。

换行

选中此复选框，以换行显示堆栈跟踪浏览器窗口中显示的较长行的堆栈跟踪。

跳过长行解析

此选项使您可以禁用对堆栈跟踪中超过指定长度的行的解析。
