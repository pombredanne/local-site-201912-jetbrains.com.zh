在TypeScript中导航和搜索

ReSharper的大多数导航和搜索功能在TypeScript中也受支持。您可以在“ 导航和搜索”部分的相应主题中找到有关这些功能的详细信息。在本节的主要主题中，您还可以找到功能矩阵并检查TypeScript确切支持的功能。

在本主题中，您可以找到一些在TypeScript中使用导航和搜索功能的示例。

导航

要了解整体上TypeScript（尤其是当前符号）所支持的导航功能，请使用“ 导航至上下文”菜单。将插入符号放在符号上，然后按Ctrl+Shift+G

转到文件成员

要通过符号在当前文件中快速找到符号并直接导航至其声明，请按Ctrl+F12 。有关更多信息，请参见转到文件成员 。

转到派生或基本符号

如果您需要查找并直接导航到给定类型的继承者，则可以使用“ 转到派生符号”功能轻松地做到这一点。按Ctrl+Alt+B显示继承者列表。直接继承者以粗体显示在列表顶部。要执行相反的操作：导航到基本符号，请按Ctrl+U 。

档案结构

如果您需要快速查看文件的代码结构，请按Ctrl+F11打开文件结构窗口。您可以展开复合元素，例如类，研究它们并导航到它们的声明。

查找用法

如果一个符号在多个地方使用，并且您想查找其所有用法，请将插入符号放在该符号上，然后按Alt+F7 。
