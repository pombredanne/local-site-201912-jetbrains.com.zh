转到符号类型

ReSharper |导航|符号类型
Ctrl+Shift+T
ReSharper_GotoTypeDeclaration

当“ 转到声明”命令将您导航到声明符号的位置时，“符号类型”命令将导航至当前所选符号类型的声明。例如，如果您有一个变量Foo myVar = new Foo();并在使用时调用此命令myVar ，它将带您进入课程的声明Foo 。如果在函数上调用此命令，它将导航到函数的返回类型。

对于泛型类型的变量，此命令允许您导航到泛型类型声明以及所有泛型参数的声明。例如，假设您有一个类型class MyGeneric和这种类型的变量var myGeneric = new MyGeneric(); 。在以下任何用途上调用此命令myGeneric ，您将可以在MyGeneric ， Author和Book类。

您可以从编辑器，“ 文件结构”窗口和其他工具窗口中调用此命令。

导航到符号类型

  1. 将插入号放置在编辑器中的符号上，或在工具窗口中选择该符号。

  2. 按Ctrl+Shift+T或选择 ReSharper |导航|主菜单中的符号类型 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 如果命令返回单个结果，则ReSharper会直接导航到那里。如果有多个声明（例如，在部分类中），它们将在下拉列表中列出，您可以在其中进行以下操作之一：

    • 单击所需的项目以在编辑器中将其打开。

    • 通过按选择所需的项目Up和Down然后按Enter在编辑器中打开它。

    • 要在“ 查找结果”窗口中查看和分析匹配项的列表， 单击在查找结果中显示 ， 按Shift+Enter要么+在数字键盘上。

此命令也可以在“ 导航到”快捷方式中找到Ctrl+Shift+G 。

如果导航到的项目属于当前解决方案，则ReSharper在编辑器中打开相应的文件，然后将插入符号放置在符号声明中。如果在引用的库中找到它，则ReSharper会根据“ 工具” | “ 工具 ”中定义的设置进行导航。 ReSharper选项的“ 外部源”页面。
