换行和包装

ReSharper |选项|代码编辑| C ++ |格式样式|换行和包装

此页面用于在源代码的某些位置配置换行符。当ReSharper 重新格式化现有代码或键入新代码时，将应用此页面中的选项。 您可以调整值并检查页面底部的预览窗格，以查看首选项如何影响代码。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
