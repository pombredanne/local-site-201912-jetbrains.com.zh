突出显示文件中的用法

ReSharper |查找|突出显示文件中的用法
Ctrl+Shift+F7
ReSharper_HighlightUsages

如果要在当前文档中查找符号的用法，则突出显示所有用法并在它们之间跳转可能比在“ 查找结果”窗口中分析用法更方便。请注意，读写访问用法以不同的颜色突出显示。

您可以在不同的符号上调用此命令以同时突出显示几个不同符号的用法。高亮显示一直保留到您按下为止Esc 。

即使您的插入符号位于声明中以浏览符号用法，也可以调用“ 转到声明” 。

突出显示当前文档中符号的用法

  1. 将插入号放置在编辑器中的符号声明或用法处。

  2. 在主菜单中，选择“ ReSharper |查找|在文件中突出显示用法或按Ctrl+Shift+F7 。

  3. ReSharper用蓝色突出显示所有读取用法，并用红色突出显示所有写入用法。

  4. 高亮部分也与右侧标记栏上的蓝色和红色标记重复。如果并非所有亮点在当前屏幕上都可见，则检查这些标记可能会有所帮助。
    要在突出显示的项目之间导航，请选择ReSharper |查找|转到菜单中的“ 下一个/上一个突出显示 ”或按Ctrl+Alt+PageDown / Ctrl+Alt+PageUp 。或者，单击状态指示器 （应显示 ）在编辑器的右上角之间，以在高光之间跳转。
    完成重点检查后，按Esc删除它们。

您还可以突出显示名称空间符号和表达式的用法（请参阅ReSharper Horizons：突出显示什么的用法？ 有关详细信息）。
