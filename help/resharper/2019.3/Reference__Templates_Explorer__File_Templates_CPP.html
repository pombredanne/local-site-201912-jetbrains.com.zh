C ++的预定义文件模板

ReSharper |模板资源管理器|文件模板| C ++

本主题列出了ReSharper 2019.3中C ++的所有预定义文件模板。有关文件模板的更多信息，请参见从模板创建文件。

模板

细节

Struct

在当前项目中创建包含空C ++ Struct的新头文件和源文件。

范围 C ++项目

身体

$ HEADER $＃pragma一旦构造$ STRUCT $ {$ END $};

参量

  • HEADER -插入ReSharper选项中指定的文件头。 （ cppFileheader ）

  • NAME -评估不带扩展名的当前文件名（ getFileNameWithoutExtension ）

  • STRUCT -评估不带扩展名的当前文件名（ getFileNameWithoutExtension ）

  • PCH -评估为预编译头文件的include指令，如果项目不使用预编译头，则为空字符串。 （ cppPchIncludeDirective ）

  • END -应用模板后的插入符位置。

Class

在当前项目中创建包含空C ++类的新头文件和源文件。

范围 C ++项目

身体

$ HEADER $＃pragma一旦课程$ CLASS $ {public：$ END $};

参量

  • HEADER -插入ReSharper选项中指定的文件头。 （ cppFileheader ）

  • CLASS -评估不带扩展名的当前文件名（ getFileNameWithoutExtension ）

  • NAME -评估不带扩展名的当前文件名（ getFileNameWithoutExtension ）

  • PCH -评估为预编译头文件的include指令，如果项目不使用预编译头，则为空字符串。 （ cppPchIncludeDirective ）

  • END -应用模板后的插入符位置。
