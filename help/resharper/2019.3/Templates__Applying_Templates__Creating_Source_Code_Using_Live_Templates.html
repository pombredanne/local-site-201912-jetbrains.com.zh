使用实时模板创建源代码

ReSharper |编辑插入实时模板…
Ctrl+J
ReSharper_LiveTemplatesInsert

实时模板是ReSharper的Visual Studio代码片段的更为智能的版本。与代码片段不同，ReSharper的模板可以从上下文中预测可以使用哪些变量，字段和类型。在大多数情况下，您无需自己键入任何内容-只需从建议值列表中进行选择即可。

ReSharper为所有受支持的语言提供了大量预定义的实时模板 。预定义模板列表包括用于遍历集合，编写断言方法，创建新数组，输出值，对LINQ查询进行编码，生成自动属性等的模板。

使用实时模板创建代码片段

  1. 在文本编辑器中，将插入符号设置为要从实时模板中插入代码片段的位置。

  2. 请执行以下任一操作：

    • 键入模板快捷方式或其首字母。例如，在C＃方法中，键入itar调用迭代数组模板。

    • 按Ctrl+J或选择 ReSharper |编辑在主菜单中插入实时模板… ，从建议列表中选择一个模板，然后按Tab要么Enter 。

  3. 如果模板具有可编辑的参数 （即，需要用户输入），则ReSharper 会在编辑器中部署热点会话 ，并将输入位置设置为第一个参数。然后，您可以执行以下操作：

    • 如果ReSharper建议使用当前参数的一些值，请使用Up和Down箭头键浏览建议值列表，或仅键入所需值。

    • 按Tab要么Enter接受该值并移至下一个参数的输入位置。如果这是最后一个参数，则热点会话完成，并且脱字符号移动到为该会话定义的结束位置。

    • 按Shift+Tab将输入焦点移至上一个参数的输入位置。

    • 按Esc退出热点会话。在这种情况下，所有会话参数都将使用默认值初始化。

要在完成列表中隐藏模板，或者相反，隐藏所有其他项目，请使用完成过滤器 。

根据特定模板的设置，扩展模板代码后，冗余名称空间引用可能会自动缩短。有关详细信息，请参见创建实时/环绕声模板 。
