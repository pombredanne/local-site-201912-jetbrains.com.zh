C ++ / CLI支持

C ++ / CLI支持依赖于C ++和.NET分析引擎，因此需要ReSharper Ultimate许可证。

除标准ISO C ++语法外，ReSharper还支持Microsoft的C ++ / CLI Specification语法，例如^对于CLR参考， gcnew用于在托管堆上进行分配等。除了代码检查 ， 编码辅助以及导航和搜索等传统功能外，还有一些功能值得一提：

  • 跨语言导航在C ++ / CLI和C＃/ VB.NET之间起作用。但是，尚不支持#using指令和跨语言重构。

  • 您可以使用生成.NET Equals（）和GetHashCode（） Alt+Insert 。

  • 一些C ++ 后缀模板 （ .gcnew和.foreach ）专门用于在上下文中调用时产生C ++ / CLI代码。
