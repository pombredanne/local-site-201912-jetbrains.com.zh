JSON模式目录

ReSharper |选项|代码编辑| JSON | JSON模式目录

在此选项页面上，您可以指定自定义JSON模式目录。这些文件可以是文件，也可以是URL，它们将在后台下载并定期更新。

通过指定的目录，ReSharper可以通过文件掩码为JSON文件找到正确的架构。反过来，该架构允许ReSharper为特定JSON属性提供验证，完成和其他功能。

默认情况下，ReSharper包含来自schemastore.org的JSON API目录。该目录包含几乎所有已知JSON文件格式的架构。默认目录既不能编辑也不能删除，但是您可以通过清除旁边的复选框来禁用它。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
