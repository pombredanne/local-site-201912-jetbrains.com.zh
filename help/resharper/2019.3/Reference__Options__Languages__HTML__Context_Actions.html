Context Actions for HTML

ReSharper provides the following context actions for HTML:

Name

Description

Collapse empty tag

Converts from empty opening and closing tags into empty tag

Convert HTML entity

Converts HTML entity from numbered form to named and visa versa

Edit path mapping

Opens path mapping settings for current project

Expand empty tag

Splits a self-closing tag into opening and closing tags

Replaces tag with different tag
