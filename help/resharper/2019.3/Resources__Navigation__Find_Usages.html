如果需要浏览解决方案中资源条目或资源文件的所有用法，则可以使用“ 查找用法”命令。

查找资源条目的用法

  1. 在代码文件中的资源使用情况下设置插入符号：

    或.resx文件中的资源名称处：

  2. 按Alt+F7或选择 ReSharper |查找|从主菜单中找到用法 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

    • 如果仅找到一种用法，则ReSharper在编辑器中打开相应的代码文件，然后导航到该用法。

    • 找到了几种用法，它们显示在“ 查找结果”窗口中 。

您可以通过按以下方式在解决方案资源管理器中快速找到当前资源文件Shift+Alt+L 。

查找资源文件的用法

  1. 在解决方案资源管理器中选择一个.resx文件。

  2. 按Alt+F7或选择 ReSharper |查找|从主菜单中找到用法 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

    • 如果仅找到一种用法，则ReSharper在编辑器中打开相应的代码文件，然后导航到该用法。

    • 找到了几种用法，它们显示在“ 查找结果”窗口中 。

要在解决方案中查找所有可本地化的字符串，请为相关项目设置Localizable = Yes和Localizable Inspector = Pessimistic ，然后找到任何此类字符串，这些字符串将用下划线突出显示。按Alt+Enter在它上面，然后选择检验[检验名称] |在scope中找到所有此类问题 。
