Root path for references in 'require'

Specifies the root path for relative paths in the require keyword. To specify the path, choose Relative to root folder option for the Relative file reference style in 'require' setting on the Code Editing | TypeScript | Code Style page of ReSharper options.

ReSharper's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Manage and Share ReSharper Settings.
