提取成员以进行部分重构

ReSharper |重构|提取成员以部分…
ReSharper_Type2Partial

此重构可帮助您将选定的类型成员移动到部分类型的另一部分。如有必要，ReSharper可以一次性创建当前类型的一部分，并为此类型创建一个新部分以及一个文件。所有必需的导入都随被移动的成员一起移动。

要使类型成为部分并将所选成员移动到那里

  1. 通过以下方式之一选择类型或成员：

    • 在编辑器中，将插入号设置为类型或成员的名称。

    • 在“ 文件结构”窗口中选择类型或成员。

    或者，也可以在编辑器或工具窗口中选择一个或多个要提取的成员，或在包含要提取的成员的区域中设置插入标记。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择“将成员提取为部分成员”

    • 右键单击并选择“ 重构”。在上下文菜单中将成员提取为部分成员 。

    • 选择 ReSharper |重构|在主菜单中将成员提取为Partial… 。

    将打开“将成员提取为部分”对话框。

  3. 如果当前类型不是局部的，ReSharper将建议创建一个新的零件。否则，您可以选择创建新零件还是选择现有零件之一来移动成员。

  4. 选择应移动到部分类型的新部分的类型成员。如果在选定成员或具有某些成员的区域上调用此重构，则将为您预先选择这些成员。

  5. 要应用重构，请单击下一步 。

  6. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
