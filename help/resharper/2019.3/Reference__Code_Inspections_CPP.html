Code Inspections in C++

In C++, ReSharper 2019.3 provides two kinds of code inspections: 266 inspections that detect errors such as broken syntax, unresolved symbols, compiler errors, and so on (you cannot configure or disable any of these inspections), and 1249 proprietary code inspections, any of which you can disable or change its severity level.
These code inspections help you detect code issues in design time in all open files, and additionally they allow you to find code issues in specific scope.

Configurable C++ inspections are listed below, grouped by their categories.
