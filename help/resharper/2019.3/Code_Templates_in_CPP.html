C ++中的代码模板

ReSharper的所有代码模板功能 C ++也支持。您可以在“ 代码模板”部分的相应主题中找到有关这些功能的详细信息。

实时模板

实时/环绕声模板可用于在现有文件中生成代码，并且（如果存在$SELECTION$模板主体中的参数 ），用于包含模板代码的周围代码片段 ，例如try...catch声明。
每个活动模板都由一个快捷方式 （一个简短的字符串，例如foreach ）标识，您可以在编辑器中直接键入以调用该模板。

ReSharper提供了大量预定义的实时模板 。您也可以在专用编辑器中或直接从现有源代码 创建自定义实时模板 。

文件模板

文件模板用于使用预定义的代码（例如类型声明，单元测试等）创建一个或多个新文件。文件模板没有快捷方式，由它们的描述标识。有关详细信息，请参见从模板 \ 创建文件和创建多文件模板 。

应用模板时，一个文件模板可以创建多个文件。当相关数据保存在不同文件中时，这可能会有所帮助。

有用于类和结构的预定义模板 。您还可以创建新文件和多文件模板。

后缀模板

Postfix模板可帮助您转换已经键入的表达式而不会向后跳-只需在表达式后键入点，然后从完成列表中选择一个模板即可。
