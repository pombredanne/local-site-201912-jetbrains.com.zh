堆栈跟踪资源管理器窗口

ReSharper | Windows |堆栈痕迹
ReSharper |工具|浏览堆栈跟踪
Ctrl+Shift+E

堆栈跟踪资源管理器显示剪贴板中的堆栈跟踪，并将其中的行转换为可单击的链接，以帮助您直接导航到相关代码。有关详细信息，请参见从堆栈跟踪导航到异常 。

默认情况下，如果您复制包含当前打开的解决方案中的符号的堆栈跟踪，然后将焦点切换到Visual Studio，则ReSharper会自动将其加载到“ 堆栈跟踪资源管理器”窗口并将其带到视图中。如果您不喜欢此行为，请清除“ 工具” | “ 自动打开外部堆栈跟踪”复选框。 ReSharper选项的“ 堆栈跟踪资源管理器”页面。
