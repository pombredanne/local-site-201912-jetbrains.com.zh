打开和关闭装配体

使用ReSharper，您可以打开任何.NET程序集，进行反编译并浏览其内容。

您也可以从全局程序集缓存（GAC）中打开程序集，并在特定文件夹中找到程序集 。

支持的程序集文件类型包括：

  • 库（.dll）

  • 可执行文件（.exe）

  • Windows 8元数据文件（.winmd）

  • 封存（.zip）

  • NuGet软件包（.nupkg）

  • Microsoft Visual Studio扩展程序包（.vsix）

打开装配体

  • 点击打开 在“ 程序集资源管理器”工具栏上，然后选择一个程序集文件。

  • 在解决方案资源管理器中，在“ 引用”节点下找到所需的引用程序集，右键单击并在上下文菜单中选择“在程序集资源管理器中查看” 。

打开的装配体将显示其名称和版本，并将其添加到“ 装配体浏览器”窗口中 。您可以通过扩展部件节点及其子节点来探索它。双击代码符号可反编译并查看相应的代码。

除了打开程序集文件之外，您还可以浏览一个包含程序集文件的文件夹 。

由于ReSharper不会对打开的装配进行任何修改，因此可以通过从Assembly Explorer窗口中删除来关闭它们。

关闭装配体

  • 右键单击一个装配，然后在上下文菜单中选择“ 从列表中删除项目” 。

  • 在装配体浏览器中选择一个装配体，然后按Delete 。

  • 要关闭所有部件，请单击“ 清除部件资源管理器”。

如果您想随时清除反编译缓存，请删除％LOCALAPPDATA％\ JetBrains \ Shared \ vAny \ DecompilerCache中的数据 。
