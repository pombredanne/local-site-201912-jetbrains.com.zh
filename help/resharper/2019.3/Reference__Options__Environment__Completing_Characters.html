完成字符

ReSharper |选项|环境|智能感知完成字符

在ReSharper选项的此页面上，您可以设置是否要ReSharper在按下时提交选定的完成项目Space或任何非字母数字字符。另外，您可以定义Enter和Tab键会在完成列表中选择的项目上起作用：两个选择器， Enter键和Tab键使您可以选择以下行为选项之一：

  • 插入 -插入项目而不插入符号后的文本。

  • 替换 -用项目替换插入符号后的文本。

项目

描述

完成太空

取消选中特定语言的复选框，以防止ReSharper在按下时提交所选的完成项目Space 。

不要完成

键入特定语言的任何非字母数字字符且不带空格，例如.;:(以防止ReSharper在按下这些字符时提交所选的完成项目。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
