参数信息

ReSharper |选项|环境|智能感知参数信息

ReSharper选项的此页面允许您针对各种用例调整“ 参数信息”工具提示。

项目

描述

自动以[value]毫秒显示参数信息

此选项定义在键入方法调用时是否自动显示“ 参数信息”工具提示。如果选中此复选框，则在键入方法调用的开头括号后，将显示工具提示。如有必要，您可以指定延迟时间（以毫秒为单位）。

一次显示所有签名

如果选中此复选框，则所有可用签名在“ 参数信息”工具提示中显示为可滚动列表。如果清除此复选框，则工具提示将一次显示一个签名以及签名总数。
签名之间的切换是可以使用相应选项配置的（请参见下文）。

限制候选人人数

仅当选择一次显示所有签名时 ，此选项才可用，它使您可以将显示的签名列表限制为指定的数字。
签名之间的切换是可以使用相应选项配置的（请参见下文）。

注解

使用此选择器可以定义是否在“参数信息”弹出窗口中的注释参数旁边显示代码注释属性 。 ReSharper将显示符号源代码中的 注释和外部添加到库符号中的注释 。
