验证正则表达式对话框

ReSharper |工具|验证正则表达式

顾名思义，此对话框可通过玩各种测试输入来帮助验证正则表达式。

该对话框在两种情况下出现：

  • 如果选择验证源代码中的正则表达式，则对话框将打开并显示当前正则表达式。

  • 如果您从菜单ReSharper |工具|验证正则表达式 ，对话框显示为空，您可以同时指定表达式和测试输入。

该对话框具有以下控件：

项目

描述

正则表达式

您可以在此处查看和编辑您验证的正则表达式。

选件

此组合框允许您设置在此对话框中根据测试输入验证表达式时将使用的正则表达式选项 。

测试输入

您可以在此处键入任何内容来测试您的正则表达式。输入中的匹配项将突出显示。

分别检查线路

如果选中此复选框，则您的正则表达式将应用于测试输入中的每个新行。

火柴

下表列出了测试输入中正则表达式的所有匹配项。您可以单击匹配项以在测试输入中找到它们。

插入

仅当您在源代码中验证正则表达式时，此按钮才可用。在这种情况下，该按钮会将经过验证（或已修改）的表达式粘贴回原始源代码位置，然后关闭对话框。

取消

此按钮关闭对话框。
