ReSharper 2019.3 Help

Navigation and Search in HTML

Most of ReSharper's navigation and search features are also supported in HTML. You can find the detailed information on these features in the corresponding topics of the Navigate and Search section. In the main topic of the section, you can also find the feature matrix and check what exactly is supported in HTML.

In this topic, you can find some examples of using navigation and search features in HTML.

File structure

File Structure provides an overview of the current file.

You can easily navigate to an element in the file by clicking it in the File Structure window.

Navigate To

The Navigate To feature is also available and helps, for example, navigate between HTML and CSS files.
