C ++的预定义实时模板

ReSharper |模板资源管理器|实时模板| C ++

本主题列出了ReSharper 2019.3中C ++的所有预定义实时模板。有关实时模板的更多信息，请参见使用实时模板创建源代码。

enum_to_string

范围 C ++名称空间声明

身体

const char * to_string（$ ENUM $ e）{switch（e）{case $ ENUMERATOR $：返回$ ENUMERATOR_NAME $;默认值：返回“未知”； }}

参量

  • ENUM -调用操作的枚举的名称。 （ cppEnumToStringEnum ）

  • ENUMERATOR -复制每个枚举器的包含模板行，并求值为枚举器的名称。 （ cppEnumToStringEnumerator ）

  • ENUMERATOR_NAME -为每个枚举器复制包含的模板行，并得出带有枚举器名称的字符串。 （ cppEnumToStringEnumeratorName ）

当您对枚举声明调用“ 将枚举转换为字符串上下文”操作时， 将扩展此模板。

doc

文件范本

范围 C ++注释

身体

/ ** * \ brief $ END $ * \ tparam $ TNAME $ $ TDESCRIPTION $ * \ param $ PNAME $ $ PDESCRIPTION $ * \ param $ MNAME $ $ MDESCRIPTION $ * \ return $ RETURN $ * /

参量

  • TNAME -为每个模板参数复制包含的模板行，并求值为模板参数名称（ cppTemplateParameterName ）

  • TDESCRIPTION -计算为指定的常数。 （ 常量 ）

  • PNAME -复制每个函数参数的包含模板行，并求值为参数名（ cppFunctionParameterName ）

  • PDESCRIPTION -计算为指定的常数。 （ 常量 ）

  • MNAME -为每个宏参数复制包含模板的行，并求值为宏参数名称（ cppMacroParameterName ）

  • MDESCRIPTION -计算为指定的常数。 （ 常量 ）

  • RETURN -仅当函数返回类型为非无效时才保留包含模板的行（ cppFunctionReturnValue ）

  • END -应用模板后的插入符位置。

当您在类型或类型成员上调用Document上下文操作时，将扩展此模板。
foreach

“基于范围的”循环

范围 C ++语句

身体

为（$ TYPE $ $ VARIABLE $：$ COLLECTION $）{$ SELECTION $$ END $}

参量

  • COLLECTION -在评估变量的位置显示基本代码完成列表（ 完整 ）

  • TYPE -在评估变量的位置显示智能代码完成列表（ completeSmart ）

  • VARIABLE -在变量声明中执行时（变量名应保留在其中），建议变量名。 （ proposalVariableName ）

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。
