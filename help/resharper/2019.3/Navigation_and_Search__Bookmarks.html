书签

ReSharper |导航|书签|书签…
Ctrl+`
ReSharper_BookmarksBookmarksMenu

如果以后要返回代码中的某个位置，则可以在解决方案或库代码中用书签标记任何代码行。

您可以使用两种书签：

  • 带有助记符的书签 -对于标记您经常引用的代码行最有用，例如当前解决方案中的前10位。您最多可以同时拥有10个编号的书签。它们由左侧编辑器装订线中的以下图标指示：

    对此类书签使用专用的快捷方式： Ctrl+Shift + 用该数字切换书签； Ctrl + 导航到带有该数字的书签。

  • 匿名书签 -数量不限，因此您可以标记任何行，您认为值得在将来的某个时间重新使用。它们由左侧编辑器装订线中的以下图标指示：

    您可以切换这种书签，并从书签弹出窗口中进行导航。 Ctrl+` 。

    如果要标记的代码包含一些未完成的功能，请考虑为该代码添加待办事项注释 ，而不是添加书签。

请注意，左侧装订线上的书签图标对鼠标单击不起作用。要对书签进行操作，请使用相应的快捷方式或菜单命令。

使用助记符添加书签

  1. 将插入号放置在要标记的代码行上。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+ 。

    • 使用弹出窗口切换书签：

      按Ctrl+`或选择 ReSharper |导航|书签|书签…从主菜单。书签弹出窗口将出现，您可以在其中看到哪些数字可用和哪些数字被占用：

      在列表中选择所需的数字，然后按Insert 。

  3. 如果您选择的该数字被另一个书签占用，则该现有书签将被删除并被新的书签替换。

添加匿名书签

  1. 将插入号放置在要标记的代码行上。

  2. 按Ctrl+`或选择 ReSharper |导航|书签|书签…从主菜单。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 出现书签弹出窗口。

  4. 在此弹出窗口中，单击“ 新建匿名书签” ，或选择此项目并按Insert要么Enter 。

导航书签

  • 要导航到带编号的书签，请按Ctrl+

  • 要导航到任何现有书签，请使用“ 书签”弹出窗口。

    按Ctrl+`或选择 ReSharper |导航|书签|书签…从主菜单。

    单击书签或选择此书签，然后按Enter在编辑器中打开标记的代码。

删除书签

  • 要删除当前打开的文档中的所有书签，请选择“ ReSharper |导航|书签|在主菜单中清除文档中的所有书签 。

  • 要删除解决方案中的所有书签，请选择“ ReSharper |导航|书签|清除主菜单中的所有书签 。

  • 要删除带有助记符的书签，请按相应的Ctrl+Shift+插入符号位于此书签标记的行时的快捷方式。

  • 按Ctrl+`或选择 ReSharper |导航|书签|书签…从主菜单。在书签弹出窗口中，选择要删除的书签，然后按Delete 。
