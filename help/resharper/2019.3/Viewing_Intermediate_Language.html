查看中间语言（IL）

ReSharper | Windows | IL查看器
ReSharper_ShowIlViewerWindow

ReSharper允许您使用中间语言（IL）研究源代码和反编译符号。

生成项目后，即可使用源代码的中间语言。
如果在构建后修改了源，则源和IL代码之间的同步可能不正确。再次生成项目后，IL代码将更新。

IL查看器

从编译的程序集中打开符号时，始终可以使用中间语言来学习它们，而与它们在编辑器中的显示方式无关。即使符号显示在通过PDB获取的原始源文件中，ReSharper也会为其找到相关的IL代码。

查看反编译符号的中间语言（IL）

  1. 选择 ReSharper | Windows |主菜单中的IL Viewer 。

  2. 如果您使用的是反编译代码，则还可以调用“ 导航到” Ctrl+Shift+G并选择IL Code 。

  3. 默认情况下，当窗口打开时，IL视图与代码查看器中的插入符号位置同步，并且突出显示匹配的代码。要更改它，请单击 在编辑器中跟踪插入符 在IL Viewer的工具栏上。

在IL Viewer窗口中，将鼠标悬停在指令上时，可以在工具提示中了解IL指令的描述。这些描述来自MSDN和ECMA-335标准（分区III：CIL指令集） 。

当您将尖号设置为代码符号或说明时，该项目的所有用法都会突出显示。代码中的循环可以通过相应的缩进和注释来区分。
