代码检查：NUnit。异步测试方法应返回Task或Task

您可以禁止执行此检查以忽略特定问题 ， 更改其严重性级别以使问题更小或更明显 ，或者将其完全禁用 。

NUnit 3.x中的异步测试方法必须返回Task如果没有返回值，或者Task如果类型的值T返回。在以下两种情况下，异步测试将无法正常工作。

[测试] //警告：异步测试方法无效public async void Test1（）{//做一些等待任务的工作。CompletedTask; } [测试] //警告：NUnit 3.10不支持ValueTask公共异步ValueTask Test2（）{//做一些等待任务返回的任务。FromResult（100）; }

在两种情况下，您都需要重写测试方法，以便它们返回Task要么Task 。
