CSS-制表符和缩进

制表符和缩进

缩进样式

属性名称：

indent_style ，[resharper_]css_indent_style

可能的值：

  • tab ： 标签

  • space ：空间

例子：

标签

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

空间

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

缩进尺寸

属性名称：

indent_size ，[resharper_]css_indent_size

可能的值：

一个整数

例子：

值：0

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

价值：1

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

价值：2

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

标签宽度

属性名称：

tab_width ，[resharper_]css_tab_width

可能的值：

一个整数

例子：

值：0

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

价值：1

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

价值：2

a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

使用制表符缩进时如何对齐

属性名称：

[resharper_]css_alignment_tab_fill_style ，[resharper_]alignment_tab_fill_style

可能的值：

  • use_spaces ：使用空格（看起来与任何标签大小对齐）

  • use_tabs_only ：仅使用标签（不准确）

  • optimal_fill ：混合制表符和空格以实现最佳填充
