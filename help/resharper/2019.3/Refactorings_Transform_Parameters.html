变换参数重构

ReSharper |重构|转换|转换参数…
ReSharper_TransformParameters

此重构可帮助您通过转换参数来快速更改方法签名-例如，摆脱out参数，将参数包装在元组或新类中，依此类推。 —并自动更新解决方案中该方法的所有用法。

ReSharper允许您使用Change Signature重构以其他方式（添加，删除，重命名，重新排序参数，更改返回类型） 更改方法签名 。

重构使您可以执行以下转换：

  • 使用与选定参数对应的类型为公共字段或自动属性创建一个新类，并使用新创建的类代替参数。

  • 用元组对象替换参数。

  • 对于返回的方法void ，转换选定的out方法返回的参数。

  • 对于返回的方法void ，包装多个选择out元组对象的参数，并将其用作方法返回。

  • 对于返回的方法void ，使用与所选字段对应的类型的公共字段或自动属性创建一个新类out参数，并使用新创建的类作为方法返回。

  • 对于非空方法out参数，结合out元组对象或新创建的类中具有返回类型的参数。

在面向.NET Framework 3.5或更低版本的项目中，此重构将不建议将参数转换或将方法返回给元组对象。

转换方法参数

  1. 将插入号放在编辑器中的声明或方法的用法上，或在“ 文件结构”窗口中选择方法。或者，也可以在任何方法参数处设置插入符号。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择“ 变换参数”

    • 右键单击并选择“ 重构”。在上下文菜单中转换参数 。

    • 选择 ReSharper |重构|转换|在主菜单中转换参数… 。

    将打开“ 变换参数”对话框。

  3. 选择要转换的参数。

  4. 根据所选的输入/输出参数，ReSharper启用方法接收和/或方法返回选择器。使用这些选择器选择应如何转换输入/输出参数。如果选择为参数创建新类，则可以在其右侧的字段中指定其名称。

  5. 要应用重构，请单击下一步 。

  6. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
