To-do Explorer

ReSharper | Options | Tools | To-do Items

This page of ReSharper options lists and allows managing patterns for To-do items.

Item

Description

Add

Click to open the Create Pattern dialog and create a new to-do pattern.

Duplicate

Click to create a copy of the selected pattern.

Edit

Click to open the Edit Pattern dialog and modify the selected pattern.

Remove

Click to remove the selected pattern from the list.

ReSharper's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Manage and Share ReSharper Settings.
