MVC View Page with master page

具有母版页的MVC查看页

范围 ASP.NET（VB）项目

身体

<％@页面标题=“ $ Title $”语言=“ VB” Inherits =“系统。网络。Mvc。ViewPage（Of $ ModelType $）“ MasterPageFile =” $ MasterPage $“％> $ CONTENT $

参量

  • Title -没有宏

  • ModelType -此时预期的猜测类型（ guessExpectedType ）

  • MasterPage -在评估变量的位置显示智能代码完成列表（ completeSmart ）

  • CONTENT -在评估变量的位置为母版页内容占位符生成内容（ AspMasterpageContentGenerator ）

MVC View Page without master page

没有母版页的MVC查看页

范围 ASP.NET（VB）项目

身体

<％@页面语言=“ VB” Inherits =“ System。网络。Mvc。ViewPage（Of $ ModelType $）“％> $标题$ $ END $

参量

  • ModelType -此时预期的猜测类型（ guessExpectedType ）

  • Title -没有宏

  • END -应用模板后的插入符位置。

Web control

网页控制

范围 ASP.NET（VB）项目

身体

<％@控制语言=“ VB” ClassName =“ $ ClassName $”％> $ END $

参量

  • ClassName -评估当前文件名，不带扩展名，所有非字母数字都用下划线（ getAlphaNumericFileNameWithoutExtension ）替换

  • END -应用模板后的插入符位置。
