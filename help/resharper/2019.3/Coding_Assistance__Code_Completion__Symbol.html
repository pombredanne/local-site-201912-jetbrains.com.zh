基本完成

ReSharper |编辑完整符号
Ctrl+Space
ReSharper_CompleteCodeBasic

ReSharper的基本完成建议在当前可见性范围内可用的类，方法，变量，关键字和任何其他符号的名称，包括先前导入的扩展方法。完成列表中还提供了许多其他特定于上下文的建议，例如实时模板 ， 代码生成建议 ， 单元测试类别 。
默认情况下，基本完成建议还包括“ 智能完成”和“ 导入符号完成”中的项目 。如果要更改默认行为，请在 环境|智能感知的“完成行为”页面 ReSharper选项（ Alt+R,,O ）。建议列表类似于“ 自动完成 ”列表。

您可以连续按几次基本完成快捷方式，以更改生成完成建议的方式。有关更多信息，请参见Double Completion 。

调用基本完成

  1. 将插入符号放置在您要键入代码的位置。

  2. 按Ctrl+Space 。

  3. 从列表中选择一个建议，然后按Enter或开始键入标识符的首字母或其CamelHumps缩写，以缩小建议列表的范围。

    如有必要，可以通过清除“ 环境” |“ 标识符之间的匹配中间”复选框来禁用CamelHumps 。智能感知 ReSharper选项的“ 完成行为”页面。

您还可以在现有标识符上调用基本完成。在这种情况下，有两种方法可以应用所选的完成建议；按Enter插入并保留现有项目或按Tab替换现有项（如果需要，可以在ReSharper选项的Environment | IntelliSense | Completion Characters页面上更改默认快捷方式）。
