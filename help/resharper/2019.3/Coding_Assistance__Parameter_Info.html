参数信息

ReSharper |编辑参数信息
Ctrl+P
ReSharper_ParameterInfoShow

每当您编写或研究函数调用时，ReSharper都会帮助您查看有关函数所有重载的允许参数的详细信息。在工具提示中，您将看到所有公共签名，包括带有参数的简短签名以及从函数文档中获得的简短说明（如果有）。

您还可以在带有参数名称提示的方法调用中内联显示参数名称，或者可以通过快速文档获得更多该函数的详细信息（ Ctrl+Q ）。

使用参数信息

在您输入参数时，ReSharper会自动突出显示与输入的参数兼容的下一个签名，并将不适用的签名显示为灰色。要研究现有函数调用的替代签名，请将插入号设置在函数括号内，然后按Ctrl+P或选择ReSharper |编辑从主菜单中获取参数信息 。

显示参数信息工具提示时，可以按Ctrl+P / Ctrl+Shift+Alt+Space或使用箭头按钮在签名列表中滚动并显示当前所选签名的摘要。

配置参数信息

默认情况下，键入信息调用的右括号后，就会出现参数信息工具提示。如有必要，您可以使用“ 自动显示参数信息”优先选项来禁用自动显示或在超时后显示工具提示。 环境|智能感知的“参数信息”页面 ReSharper选项（ Alt+R,,O ）。如果禁用了自动外观，则始终可以通过按来显示参数信息工具提示Ctrl+P而您的插入符号位于函数的括号内。

您可以在“ 环境” |“配置”中进一步配置参数信息工具提示。 智能感知 ReSharper选项的“ 参数信息”页面通过以下方式：

  • 默认情况下，可用签名显示在列表中。如有必要，可以限制列表中签名的数量，或使ReSharper一次仅显示一个签名：

    为此，请使用“立即显示所有签名”首选项。

  • 默认情况下，代表以其原始形式显示。如有必要，ReSharper可以用lambda语法显示它们。要启用此行为，请选择相应的复选框。

  • 如果您不想看到功能摘要信息，请清除“ 显示扩展信息”复选框。

  • 默认情况下，ReSharper显示[NotNull]和[CanBeNull] 注解 。如有必要，您可以隐藏或显示所有JetBrains.Annotations使用注释选择器在函数签名中添加属性。

  • ReSharper可以在当前行下方或上方显示参数信息工具提示。要指定所需位置，请打开环境|智能感知单击“ ReSharper”选项的“ 完成行为”页面，单击“ 高级” ，然后使用“ 参数信息”默认位置选择器。
    请注意，即使您选择工具提示的较低位置，也可以在键入时自动显示工具提示并且ReSharper同时在代码完成列表中显示一些建议的情况下，将其显示在上方。当您使用Ctrl+P快捷方式，它始终显示在首选位置。
