代码检查：引入可选参数（私有可访问性）

您可以禁止执行此检查以忽略特定问题 ， 更改其严重性级别以使问题更小或更明显 ，或者将其完全禁用 。

如果重载函数的唯一目的是使用某些参数的默认值调用“实现”函数，则可以在“实现”函数中使用可选参数 （在C＃4.0中引入）来代替此重载。

ReSharper可以帮助您检测此类重载并自动将其删除，从而使“实现”功能的相应参数成为可选参数。

这是此检查建议的快速修复示例：

次优代码

快速修复后

void Foo（object value）{Foo（value，true）; } void Foo（object value，bool flag）{//实现}
void Foo（object value，bool flag = true）{//实现}
