ReSharper 2019.3 Help

Navigation

ReSharper enables you to quickly navigate to resource declarations and resource usages. It can display search results in the Find Results window and provides convenient navigation between them.

All navigation actions can be divided into four groups:

  • Navigation within Resx Files
  • Navigation between Code and Resx Files
  • Navigation between Cultures
  • Find Usages

To find all localizable strings in your solution, set Localizable=Yes and Localizable Inspector=Pessimistic for the relevant projects, then find any such string, which will be highlighted with curly underline. Press Alt+Enter on it and choose Inspection [name of inspection] | Find all issues of this type in scope.
