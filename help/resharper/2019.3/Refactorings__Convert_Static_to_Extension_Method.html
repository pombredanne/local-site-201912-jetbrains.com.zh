将静态转换为扩展方法重构

ReSharper |重构|转换|静态到扩展方法…
ReSharper_Static2ExtensionAction

该重构可帮助您将静态方法转换为扩展方法，前提是该静态方法：

  • 至少有一个参数。

  • 驻留在非通用，非嵌套的静态类中。

反向功能可通过“ 将扩展方法转换为纯静态重构”重构使用。

将静态方法转换为扩展方法

  1. 将插入号放在编辑器中的声明或静态方法的用法处，或在“ 文件结构”窗口中选择它。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择将静态方法转换为扩展

    • 右键单击并选择“ 重构”。在上下文菜单中将“ 静态方法转换为扩展 ”。

    • 选择 ReSharper |重构|转换|静态到扩展方法…在主菜单中。

  3. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
