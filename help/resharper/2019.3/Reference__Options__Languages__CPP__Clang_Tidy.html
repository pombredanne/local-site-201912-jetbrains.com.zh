lang整洁

ReSharper |选项|代码编辑| C ++ | lang整洁

在此页面上，您可以配置Clang-Tidy集成 。

项目

描述

启用clang-tidy支持

使您可以启用或禁用Clang-Tidy集成 。

启用/禁用的clang-tidy检查列表

的-checks Clang-Tidy二进制文件的命令行选项用于传递已启用检查的列表，然后将其附加到从Clang-Tidy配置读取的checks选项的值之后。构造-checks参数，ReSharper接受此值，并在其后附加在“ 代码检查” | “ 代码检查”中关闭的Clang-Tidy检查的名称。 检查严重性选项页面。

其他clang命令行参数

在此字段中指定的参数将附加到传递给Clang-Tidy二进制文件的参数列表中。

Clang-Tidy配置

使用此字段可通过以下方式将具有自定义配置的字符串传递给Clang-Tidy二进制文件： -config命令行选项。

应用Clang-Tidy修复程序后重新格式化更改的代码

默认情况下，ReSharper将重新格式化由Clang-Tidy修复程序更改的代码，以维护代码样式。
您可以禁用此设置以加快Clang-Tidy修复程序的执行速度，并在以后批量重新格式化修改后的代码。

使用外部叮当声

默认情况下，ReSharper使用Clang-tidy 8的内置二进制文件。您可以使ReSharper使用任何其他Clang-tidy二进制文件（版本7或更高版本）。
