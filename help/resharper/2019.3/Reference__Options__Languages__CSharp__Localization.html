ReSharper 2019.3 Help

Localization

ReSharper | Options | Code Editing | C# | Localization

By selecting or clearing checkbox in this page, you can define whether ReSharper considers verbatim strings as localizable.

Item

Description

Don't analyse verbatim string @"Hello world"

If this checkbox is selected, ReSharper does not inspect verbatim strings.

ReSharper's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Manage and Share ReSharper Settings.
