查找结果窗口

ReSharper | Windows |查找结果
Ctrl+Alt+U
ReSharper_ShowFindResults

此窗口用于显示各种搜索结果 （例如， 符号的 用法，外部符号的用法，符号的 API公开等）。它在不同的选项卡中显示最新搜索的结果以及以前的搜索结果。每个新搜索命令的结果均显示在单独的选项卡中，但同一搜索/导航命令除外。

您可以通过双击这些结果快速打开包含特定搜索结果的文档。如果要在Visual Studio的“ 预览”选项卡中打开文件，则可以在“ 环境” | “ 环境” |“复选框”上打勾。 ReSharper选项的“ 搜索和导航”页面。

窗口标签

在窗口中打开搜索结果后，它们将显示在新选项卡中，这些选项卡显示在左侧。如果窗口的宽度允许，先前打开的选项卡将移至右侧，否则它们将自行卸载以节省内存-但您始终可以通过单击选项卡右侧出现的箭头并单击所需的选项卡来还原它们：

为了防止标签页卸载，请点击图钉图标标签标题旁边。

工具栏控件

根据搜索结果的类型，工具栏控件集可能会有所不同。

控制

名称

描述

刷新

更新窗口内容以反映代码中或外部所做的任何更改。

全部展开 / 全部折叠

展开/折叠当前选项卡中的所有节点。

上一页 / 下一页
Shift+F8 / F8

导航到上一个/下一个项目，并在源代码中滚动。

显示预览 Ctrl+P

隐藏或显示窗格，该窗格在使用列表指定的位置（在窗口的底部或右侧）显示所选项目的预览。

出口

点击这个按钮以文本格式导出当前在窗口中显示的数据，或使用下拉选择器以XML或HTML格式导出数据。出现的“ 导出数据”对话框将帮助您将数据保存到文件或将其复制到剪贴板。

在同一行上合并发生

选择此选项后，同一行上的出现将显示为单个项目。

仅显示阅读用法

选择此选项时，仅显示读取用法。可用于字段，属性和局部变量的用法。

仅显示写入用法

选择此选项时，仅显示写用法。可用于字段，属性和局部变量的用法。

筛选器用法

使用此列表，您可以选择要显示的特定用法类型，例如，声明，调用，参数，方法返回等。单击“ 显示所有用法”以启用所有用法，或单击“ 清除过滤器”以禁用所有用法，从而将所有用法隐藏在当前选项卡中。

在图上显示

单击此按钮可在类型依赖关系图视图中显示窗口中显示的所有类型。

按种类切换分组

使用此按钮按用途对搜索结果进行分组-也就是说，目标项目在不同位置的使用方式-例如声明，调用，参数，方法返回等。您可以使用“ 分组依据”选择器进一步细化“种类”分组中的分组 。
您还可以使用过滤器用法 选择器以显示/隐藏特定种类的用法。

通过...分组

允许按不同类别对窗口中的项目进行分组。
