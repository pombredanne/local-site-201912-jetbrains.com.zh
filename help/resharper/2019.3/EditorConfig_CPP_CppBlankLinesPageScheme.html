C++ - Blank Lines

Preserve Existing Formatting

Keep max blank lines in declarations

Property names:

[resharper_]cpp_keep_blank_lines_in_declarations, [resharper_]keep_blank_lines_in_declarations

Possible values:

an integer

Examples:

value: 0

void f() { } void g() { }

value: 1

void f() { } void g() { }

value: 2

void f() { } void g() { }

Keep max blank lines in code

Property names:

[resharper_]cpp_keep_blank_lines_in_code, [resharper_]keep_blank_lines_in_code

Possible values:

an integer

Examples:

value: 0

void f() { g(); g(); }

value: 1

void f() { g(); g(); }

value: 2

void f() { g(); g(); }
