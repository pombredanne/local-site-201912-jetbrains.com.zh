Find Usages

ReSharper provides a set of features that help you study how symbols defined in the solution and in referenced libraries are used in the solution. In this section, you will learn how to:

  • Find usages of a symbol
  • Find usages of related entities
  • Find usages of all symbols defined outside of specific scope
  • Highlight usages in the current file and jump between the occurrences

If multiple usages are found, you can study them in the Find Results window, which provides a lot of ways to analyze the usages.

In middle-sized and large solutions usage search can take a noticeable time. However, if the solution-wide analysis is enabled, this time reduces significantly because a detailed index of all files in solution becomes available.

You can invoke Go to Declaration even if your caret is on a declaration to navigate through symbol usages.

ReSharper also provides the Go to Usages of Symbol command, which can be more convenient if you are looking for a specific usage and want to get to this usage without opening the Find Results window.
