生成代码并完成代码

使用ReSharper生成代码的最简单方法之一就是使用代码完成功能 ，您要做的就是键入几个字符。代码生成建议包括在自动完成和基本完成列表中。

您可以在上配置通用代码生成首选项 代码编辑|的会员生成页面 ReSharper选项（ Alt+R,,O ）。

以下是一些有关如何使用代码完成功能生成代码的示例。

覆盖和实施成员

在下面的示例中，代码完成有助于为基类中的虚拟成员创建替代。开始输入方法名称，您将得到建议以当前类型覆盖它：

接受建议后，具有默认实现的方法主体将在编辑器中展开：

公共类派生：BaseClass {受保护的重写void DoSomething（int x，string s）{基础。DoSomething（x，s）; }}

字段属性

要为字段生成属性，请开始输入字段名称。ReSharper建议根据您的命名样式创建一个具有相应名称的只读或读写属性：

接受建议后，属性主体将在编辑器中展开：

公共类属性{private int _myField; public int MyField {get {return _myField; }设置{_myField =值； }}}

类型构造器

根据当前类型的现有成员，ReSharper在完成列表中建议不同的构造函数。要创建构造函数，请输入ctor 。在完成列表中，您可能会看到以下建议：

  • ctor —不带参数的构造函数

  • ctorf —初始化所有字段的构造函数

  • ctorp —初始化所有自动属性的构造函数

  • ctorfp —初始化所有字段和自动属性的构造函数

在下面的示例中，所有类型的构造函数均可用。

如果您接受ctorfp建议，则构造函数将在编辑器中扩展：

公共类Person {私人字符串年龄；公共字符串名称{get;组; } public Person（字符串年龄，字符串名称）{this.age = age;名称=名称； }}

完成列表中的模板

您的所有实时模板和后缀模板都显示在完成列表中。模板由其快捷方式标识（这是预定义模板的快捷方式列表）。您可以通过相应的图标识别它们 。
