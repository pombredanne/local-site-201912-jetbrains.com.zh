ReSharper 2019.3 Help

Code Inspections in VB.NET

In VB.NET, ReSharper 2019.3 provides two kinds of code inspections: 261 inspections that detect errors such as broken syntax, unresolved symbols, compiler errors, and so on (you cannot configure or disable any of these inspections), and 253 proprietary code inspections, any of which you can disable or change its severity level.
These code inspections help you detect code issues in design time in all open files, and additionally they allow you to find code issues in specific scope.

Configurable VB.NET inspections are listed below, grouped by their categories.
