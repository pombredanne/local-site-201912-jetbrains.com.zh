文化之间的导航

ReSharper使您可以在文化层次结构之间上下移动。它提示哪些资源条目将被覆盖以及在何处被覆盖。在不同区域性中的资源定义之间导航的最快方法是使用显示在资源条目左侧的动作指示符 。

使用行动指标在文化之间导航

  1. 在XML（文本）编辑器中打开.resx文件。

  2. 对于在多种区域性中定义的每个资源，您将在资源条目的左侧看到一个操作指示器。

  3. 将鼠标悬停在动作指示器上，可以查看哪些资源条目被覆盖以及在哪里被覆盖：

  4. 要导航到特定区域性的资源定义，请单击操作指示器并选择所需的区域性。

您可以使用“ 转到派生的符号”命令从文化中立的资源文件中的资源条目导航到特定于文化的资源文件中的相应条目，反之亦然，使用“ 转到基本符号”命令进行导航。

导航到基本资源条目

  1. 在XML（文本）编辑器中打开.resx文件。

  2. 将插入符号设置为资源名称。

  3. 按Ctrl+U或选择 ReSharper |导航|主菜单中的基本符号 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  4. 如果有两个或多个基本资源声明，请在列表中选择其中之一。

导航到派生资源条目

  1. 在XML（文本）编辑器中打开.resx文件。

  2. 将插入符号设置为资源名称。

  3. 按Ctrl+Alt+B或选择 ReSharper |导航|从主菜单派生的符号 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  4. 如果有两个或多个派生条目，请在列表中选择其中之一。

您还可以使用“ 转到实现”命令，该命令将导航到所有区域性中特定资源的所有定义。

导航到资源实施

  1. 在XML（文本）编辑器中打开.resx文件。

  2. 将插入符号设置为资源名称。

  3. 按Ctrl+Shift+Alt+B或选择 ReSharper |导航|从主菜单转到实施 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  4. 如果资源有两个或多个实现，请从列表中选择一种。
