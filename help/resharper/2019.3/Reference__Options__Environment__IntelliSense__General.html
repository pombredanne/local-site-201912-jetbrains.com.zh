项目

描述

使用IntelliSense（代码补全和参数信息）功能

  • ReSharper使您可以在自动IntelliSense弹出窗口中以及在按时使用ReSharper的基本完成 Ctrl+Space 。参数信息还将替换相应的Visual Studio功能。

  • Visual Studio保留了本机Visual Studio的行为，以实现自动IntelliSense弹出式窗口和Ctrl+Space捷径。但是，您仍然可以使用ReSharper的智能完成功能和导入符号完成功能。

  • 自定义IntelliSense与第一个选项相似，但是它允许您指定ReSharper IntelliSense功能适用的语言。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
