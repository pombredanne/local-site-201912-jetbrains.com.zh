Css - Line Breaks

Blank Lines

Keep existing line breaks

Property names:

[resharper_]css_keep_user_linebreaks, [resharper_]keep_user_linebreaks

Possible values:

true | false

Examples:

Before formatting

After formatting, true

a, br, body, p, font, div { color: blue; font: small; background-color:transparent; padding: 0 0 0 0; }
a, br, body, p, font, div { color: blue; font: small; background-color: transparent; padding: 0 0 0 0; }

Before formatting

After formatting, false

a, br, body, p, font, div { color: blue; font: small; background-color:transparent; padding: 0 0 0 0; }
a, br, body, p, font, div { color: blue; font: small; background-color: transparent; padding: 0 0 0 0; }
