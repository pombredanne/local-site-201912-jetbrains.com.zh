CSS-换行符

空行

保留现有的换行符

属性名称：

[resharper_]css_keep_user_linebreaks ，[resharper_]keep_user_linebreaks

可能的值：

true | false

例子：

格式化之前

格式化后为true

a，br，body，p，font，div {color：blue;字体：小；背景颜色：透明；填充：0 0 0 0; }
a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

格式化之前

格式化后为false

a，br，body，p，font，div {color：blue;字体：小；背景颜色：透明；填充：0 0 0 0; }
a，br，body，p，font，div {color：blue;字体：小；背景色：透明；填充：0 0 0 0; }

声明之间的最大空白行

属性名称：

[resharper_]css_keep_blank_lines_between_declarations ，[resharper_]keep_blank_lines_between_declarations

可能的值：

一个整数

例子：

格式化之前

格式化后，值：0

.blue {color：blue}＃error，div.error {color：red}
.blue {颜色：蓝色} #error，div.error {颜色：红色}

格式化之前

格式化后，值：1

.blue {color：blue}＃error，div.error {color：red}
.blue {颜色：蓝色} #error，div.error {颜色：红色}

格式化之前

格式化后，值：2

.blue {color：blue}＃error，div.error {color：red}
.blue {颜色：蓝色} #error，div.error {颜色：红色}
