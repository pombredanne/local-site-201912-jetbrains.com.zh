Web代理设置

ReSharper |选项|环境| Web代理设置

在ReSharper选项的此页面中，您可以调整ReSharper的Internet活动的 Web代理设置。默认情况下，ReSharper会遵循您的系统设置，并且如果系统中定义了任何代理服务器，ReSharper也会使用它来从符号服务器中获取源代码，验证许可证信息，检测更新等。

项目

描述

从Internet下载时使用系统Web代理设置

您可以清除此复选框以覆盖ReSharper的Internet连接的系统Web代理设置。

编辑

当您选择使用系统代理服务器设置时，将显示此按钮。
通过单击此按钮，打开“ Internet属性”对话框，您可以在其中更改系统Web代理设置 。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
