Code Style Sharing

ReSharper | Options | Code Editing | Code Style Sharing

This page of ReSharper options allows you to import code styles from earlier versions. In ReSharper 6.1 and later all code style settings are saved and shared with settings layers. For more information, see Manage and Share ReSharper Settings.

Item

Description

Import Code Styles from earlier versions

This button allows you to import code style settings from ReSharper versions earlier than 6.1. The settings should be exported with the Export command that is available in those versions on this options page.
