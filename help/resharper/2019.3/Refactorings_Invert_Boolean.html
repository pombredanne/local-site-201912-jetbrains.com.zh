ReSharper |重构|反转布尔…
ReSharper_InvertBool

通过此重构，您可以自动修改布尔成员，以便其值取反。如果将此重构应用于布尔字段或属性，则ReSharper允许您指定新名称并反转其所有用法，也就是说，如果您重构private bool isEnabled;至private bool isDisabled; ，ReSharper将确保例如if(isEnabled)变成if(!isDisabled)到处。
如果将此重构应用于布尔方法，则ReSharper允许您选择是反转其内部逻辑，用法还是同时反转两者。

反转布尔方法

  1. 将插入号放在编辑器中的声明或布尔成员的用法处，或在“ 文件结构”窗口中将其选中。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择反转布尔

    • 右键单击并选择“ 重构”。在上下文菜单中反转布尔值 。

    • 选择 ReSharper |重构|在主菜单中反转布尔值… 。

  3. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
