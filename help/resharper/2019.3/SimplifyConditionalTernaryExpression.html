Code Inspection: Simplify conditional ternary expression

You can suppress this inspection to ignore specific issues, change its severity level to make the issues less or more noticeable, or disable it altogether.

If your ternary expression (an expression of the form a ? b : c) contains a boolean constant as either b or c, it can be converted to a much simpler expression. For example, this

public bool CanVote { get { return myAge >= 16 ? isCitizen : false; } }

can be simplified to this:

public bool CanVote { get { return myAge >= 16 && isCitizen; } }
