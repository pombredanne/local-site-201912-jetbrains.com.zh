生成类型构造器

ReSharper |编辑生成代码|建设者
Alt+Insert | 建设者
ReSharper_GenerateConstructor

构造函数生成向导将创建一个非默认构造函数，该构造函数将使用选定字段，属性和自动属性的参数。

所有生成的构造函数都遵循相同的模式，其中：

  • 构造函数中包含的每个字段，属性或自动属性均使用参数进行初始化。

  • 参数的名称是从相应字段或属性的名称派生的。

您可以通过键入快速生成构造函数ctor （无参数）， ctorf （使用用于初始化所有字段的参数）或ctorp （带有初始化所有属性的参数）。

如果存在非默认基类型构造函数，则将必需的参数添加到生成的构造函数中，并传递给基类构造函数。

在下面的示例中，此命令用于生成新的Circle需要两个附加参数进行初始化的构造函数_radius和_center领域。

生成构造函数

  1. 在编辑器中，将插入号设置为类型名称，或在要插入构造函数的行的类型内。如果插入符号在类型名称上，则生成的代码将添加到类型声明的开头。

  2. 按Alt+Insert或选择 ReSharper |编辑从主菜单生成代码… 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 构造器” 。

  4. 在出现的“ 生成”对话框中，选择应在新构造函数中初始化的类型成员。（可选）选择一个或几个基类构造函数。对于每个选定的基本构造函数，将生成一个新的构造函数，该构造函数将调用基本构造函数并另外初始化选定的成员。

    （可选）在对话框中使用以下控件：

    • 访问权限 -允许您为生成的构造函数定义访问权限。

    • 检查参数是否为空 （如果类具有可空类型的字段或属性，则显示）—如果选中此复选框，ReSharper将为每个可空参数生成可配置的空检查 ，例如： if (param == null) throw new ArgumentNullException(nameof(param));

    • 将参数设为可选 -如果选中此复选框，ReSharper将使生成的构造函数的所有参数均为可选，并添加与参数类型相对应的默认值。

  5. 单击完成以完成向导。
    您也可以在“ 代码编辑” |“选项”上单击“ 选项”以查看或修改常见的代码生成首选项。 ReSharper选项的“ 成员生成”页面。

要将此代码生成命令绑定到其自己的快捷方式，请转到“ 工具” |“其他”。选项|环境|键盘并为ReSharper_GenerateConstructor命令。
