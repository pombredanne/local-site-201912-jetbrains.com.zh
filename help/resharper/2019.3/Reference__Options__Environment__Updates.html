Updates

ReSharper | Options | Environment | Updates

This page of ReSharper options lists settings related to ReSharper updates.

Item

Description

Automatically check for ReSharper Ultimate updates

Clear this checkbox if you want to disable notification of new updates.

Participate in the Early Access Program (EAP)

Use this checkbox to enable notifications about new build in ReSharper Early Access Program (EAP). The program provides access to pre-release builds of ReSharper Ultimate, which become available several weeks before each new version is released. Note that:

  • All EAP builds are valid for 30 days from their release date and do not require a license.

  • When installing an EAP build, all previous installations of ReSharper and other JetBrains .NET tools are removed from Visual Studio.

ReSharper's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Manage and Share ReSharper Settings.
