将类型移到外部作用域重构

ReSharper |重构|移动…
F6
ReSharper_Move

此重构可帮助您将嵌套类型上移一级。即使嵌套类型使用父类型的成员，所有对应的引用也会自动更新。

将类型移到外部作用域

  1. 通过以下方式之一选择嵌套类型：

    • 在编辑器中，将插入符号设置为嵌套类型的名称。

    • 在解决方案资源管理器中选择一个嵌套类型。

    • 在“ 文件结构”窗口中选择一个嵌套类型。

    • 在“类视图”中选择一个嵌套类型。

    • 在对象浏览器中选择一个嵌套类型。

    • 在类型依赖关系图中选择一个嵌套类型。

  2. 请执行以下任一操作：

    • 按F6然后选择移至外部范围

    • 按Ctrl+Shift+R然后选择移至外部范围

    • 右键单击并选择“ 重构”。在上下文菜单中移至“外部作用域 ”。

    • 选择 ReSharper |重构|在主菜单中移动...。

    将打开“ 移至外部范围”对话框。

  3. （可选）您可以为该类型指定其他名称。

  4. 选中“ 在注释中搜索”复选框，以更新对注释中类型的引用。

  5. 要应用重构，请单击下一步 。

  6. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
