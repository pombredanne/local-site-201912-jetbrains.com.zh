ReSharper | Options | Code Editing | XML | Editor

This page allows you to adjust ReSharper's XML features in the editor. These are language-specific settings that complement the main set of editor features.

Item

Description

Synchronous changing of matching tag

If this checkbox is selected, ReSharper simultaneously changes the closing tag when you are editing the opening tag of an element and vice versa.

ReSharper's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Manage and Share ReSharper Settings.
