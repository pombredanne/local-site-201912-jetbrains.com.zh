代码检查：“ if”语句可以重写为“？：”表达式

您可以禁止执行此检查以忽略特定问题 ， 更改其严重性级别以使问题更小或更明显 ，或者将其完全禁用 。

C＃中的条件？：（三元）运算符是根据简单布尔表达式的值执行两个简单表达式之一的最优雅的方法。

因此，如果ReSharper遇到if...else在这种情况下，它建议将其转换为更简洁的?:操作员。

这是此检查建议的快速修复示例：

次优代码

快速修复后

void Foo（字符串msg1，字符串msg2，bool标志）{if（标志）控制台。WriteLine（msg1）;其他控制台。WriteLine（msg2）; }
void Foo（字符串msg1，字符串msg2，布尔标记）{控制台。WriteLine（flag？msg1：msg2）； }
