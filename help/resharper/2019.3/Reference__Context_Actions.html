您可以在代码编辑|禁用特定上下文操作。 ReSharper选项的“ 上下文操作”页面。

在本部分中，您可以找到适用于特定语言的上下文操作的完整列表：

  • 与语言无关的上下文动作
  • C＃的上下文操作
  • Visual Basic .NET的上下文操作
  • HTML的上下文操作
  • ASP.NET的上下文操作
  • 剃刀的上下文动作
  • JavaScript的上下文操作
  • TypeScript的上下文操作
  • JSON的上下文操作
  • CSS的上下文操作
  • XML的上下文操作
  • XAML的上下文操作
  • 构建脚本的上下文操作
  • C ++的上下文操作

ReSharper C ++提供了C ++支持， ReSharper C ++是一种专用产品，可以单独安装，也可以与ReSharper或ReSharper Ultimate并排安装。
