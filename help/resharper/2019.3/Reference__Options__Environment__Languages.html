产品展示

如果您的安装包括几种JetBrains .NET平台产品（ReSharper，ReSharper C ++，dotTrace，dotMemory，dotCover），则可以通过禁用当前不使用的资源来节省系统资源。

查看产品文档页面以了解有关它们的更多信息：

  • ReSharper C ++ —针对C ++开发人员的Visual Studio扩展
  • dotTrace —用于.NET性能分析的工具
  • dotMemory —内存分析工具
  • dotCover — .NET单元测试运行程序和代码覆盖率工具
  • dotPeek-一个免费的.NET反编译器
  • TeamCity加载项，可帮助您访问，执行和分析TeamCity服务器上的构建和测试
  • dotMemory Unit —一个单元测试框架，用于检查代码中是否存在内存问题
特征
调试器集成

使用此复选框可以切换调试协助功能。请注意，此选项不会影响没有调试信息（PDB）的调试模块 。

内部化

使用此复选框可以切换国际化辅助功能 。

JavaScript和TypeScript

使用此复选框可以切换JavaScript和TypeScript中的 ReSharper功能支持。

MSBuild和NAnt

使用此复选框可以切换构建脚本中对ReSharper功能的支持 。

导航和搜索

使用此复选框可以通过不加载相应的模块和UI元素来启用或禁用导航和搜索功能。

原虫

使用此复选框可以切换Protobuf（.proto）文件中的ReSharper功能支持 。

ReSharper构建

使用此复选框可以通过不加载相应的模块和UI元素来启用或禁用ReSharper Build 。

转售者

使用此复选框可以切换ReSpeller — ReSharper的拼写检查引擎 。 ReSpeller将在标识符，注释和字符串文字中查找拼写错误。当它检测到潜在的拼写错误或错字时，它将以与其他代码问题相同的方式让您知道— 在设计时或在运行代码检查时是特定的范围 。

单元测试

使用此复选框可以通过不加载相应的模块和UI元素来切换单元测试支持 。

网络语言

使用此复选框可以切换ASP.NET ， ASP.NET MVC和Razor ， HTML和CSS中的 ReSharper功能。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
