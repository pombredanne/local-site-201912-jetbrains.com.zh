HTML的预定义环绕声模板

ReSharper |模板资源管理器|环绕模板|的HTML

本主题列出了ReSharper 2019.3中所有用于HTML的预定义环绕模板。有关环绕声模板的更多信息，请参见带有模板的环绕声代码片段。

模板

细节

tag

范围 HTML标签，Razor标签

身体

<$ TAG $ $ END $> $ SELECTION $

参量

  • TAG -在当前文档中使用的建议标签名称（ proposalTagName ）

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

link

将选择内容包装在超链接中

范围 HTML和类似标签

身体

$ SELECTION $

参量

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。
