架构视图

ReSharper |建筑|显示项目依赖图

架构视图允许浏览项目依赖关系图 ，并将先前获取的依赖关系快照与解决方案的当前状态进行比较。

该视图由解决方案项 ， 图表区域和工具栏组成 。

解决方案

左侧显示的解决方案项目包括解决方案的所有项目以及分组项目。根据集团的选择，这些项目显示为一个普通列表，如果没有分组或者根据分组选项的层次结构。每个项目旁边的复选框定义是否在图中显示该项目。

图表区域

图表区域显示所有带有选定复选框的项目。如果启用了任何分组，则也会显示分组项目。您可以更改布局，展开和折叠分组节点，以及对图执行其他操作。有关更多信息，请参见浏览项目依赖关系图 。

工具栏控件

控制

名称

描述

通过...分组

使用此列表选择将待办事项分组的条件。提供了几个选项：

  • 无 -所有项目均按分组显示。

  • 解决方案文件夹 -项目按解决方案文件夹分组。

  • 文件结构 -项目和解决方案文件夹按文件系统结构分组。

撤消

逐步撤消对图中所做的更改（显示/隐藏的项目）。

重做

撤消上一个撤消操作。

打开/关闭耦合分析

此切换按钮允许启用或禁用在后台执行的耦合分析。默认情况下，此按钮处于打开状态，并且在打开新图后立即开始分析。 耦合分析是一个资源密集型过程，可能会影响大型解决方案的性能。如果在使用图表时遇到任何滞后，可以使用此按钮禁用耦合分析，根据需要调整图表，然后再次启用分析。分析算法首先计算显示项目的耦合，然后处理从视图中隐藏的项目。

隐藏/显示传递参考

使用此切换按钮，您可以显示项目之间的可传递引用，默认情况下将隐藏这些引用以简化图表，而又不会丢失其体系结构。例如，如果您有项目参考A > B > C和A > C ，使用此按钮可隐藏A > C参考。

保存架构图

将依赖项快照保存到文件。有关更多信息，请参见浏览项目依赖关系图 。

显示差异

仅当视图中显示的快照与当前解决方案不同时，才启用此按钮。单击按钮将快照中解决方案的状态与当前状态进行比较。

折叠图

折叠图中的所有元素。

展开图

展开图中的所有元素。

显示可能的路径

什么时候该按钮处于打开状态，您可以查看两个特定项目之间是否具有由当前从视图中隐藏的其他引用项目组成的路径。这些路径以绿色虚线显示。仅当存在隐藏的中间项目以链接当前显示的任何两个项目时，才启用该按钮。有关更多信息，请参阅检测可能的体系结构问题 。

导出到PNG

单击此按钮将图的可见部分另存为PNG图像文件。
