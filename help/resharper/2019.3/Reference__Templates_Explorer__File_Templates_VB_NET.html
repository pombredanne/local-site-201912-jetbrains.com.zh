模板

细节

Class

在当前项目中创建一个包含空VB.NET类的新文件。

范围 VB.NET项目

身体

公开课$ CLASS $ $ END $结束课

参量

  • CLASS -评估当前文件名，不带扩展名，所有非字母数字都用下划线（ getAlphaNumericFileNameWithoutExtension ）替换

  • NAMESPACE -获取当前文件的默认名称空间（ fileDefaultNamespace ）

  • END -应用模板后的插入符位置。
