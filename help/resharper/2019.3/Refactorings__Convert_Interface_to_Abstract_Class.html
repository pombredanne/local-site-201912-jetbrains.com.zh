将接口转换为抽象类重构

ReSharper |重构|转换|索引器方法
ReSharper_Function2Indexer

这种重构将接口转换为抽象类，从而帮助您快速更改一组类和接口之间的层次依赖。

反向功能可用于将抽象类转换为接口重构。

执行此重构后，您可能需要将派生类中的某些成员带到新创建的抽象类中。您可以使用Pull Members Up重构来做到这一点。

将合适的接口转换成抽象类

  1. 通过以下方式之一选择接口：

    • 在编辑器中，将插入符号设置为接口名称。

    • 在解决方案资源管理器中选择一个接口。

    • 在“ 文件结构”窗口中选择一个接口。

    • 在“类视图”中选择一个接口。

    • 在对象浏览器中选择一个接口。

    • 在类型依赖关系图中选择一个接口。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择“ 将接口转换为抽象类”

    • 右键单击并选择“ 重构”。在上下文菜单中将接口转换为抽象类 。

    • 选择 ReSharper |重构|转换|主菜单中的索引方法...。

  3. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
