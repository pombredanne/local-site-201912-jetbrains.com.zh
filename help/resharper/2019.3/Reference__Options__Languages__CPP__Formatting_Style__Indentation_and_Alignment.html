您可以使用此页面上的首选项来指定ReSharper在特定情况下如何缩进和对齐代码。在“ 缩进”组中，您可以定义在特定情况下如何应用制表符和缩进（可以在“ 代码编辑” |“ C ++” |“格式样式” |“制表符和缩进”选项页上进行配置）。在“ 对齐多个结构”组中，可以定义如何对齐跨多行的代码结构（可以在“ 代码编辑” |“ C ++” |“格式设置” |“换行符和换行”页面上指定用于折断长行的规则）。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。

您在此页面上配置的选项在以下情况下适用：

  • 当代码在编辑时或粘贴后自动格式化时 （您可以在 环境|编辑器的“编辑者行为”页面 ReSharper选项（ Alt+R,,O ）。

  • 当ReSharper 产生具有代码完成和代码生成功能的新代码，应用代码模板并执行重构 。

  • 重新格式化现有代码时 。
