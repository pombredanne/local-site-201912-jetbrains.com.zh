生成部分方法

ReSharper |编辑生成代码|部分方法
Alt+Insert | 部分方法
ReSharper_GeneratePartialMethods

部分方法在部分类型的不同部分（不同文件）中定义-一个部分包含方法的签名，而另一部分包含其实现。

生成部分方法命令将帮助您为未实现的部分方法创建实现存根。

根据您的设置，可以用不同的方式生成新成员的存根。例如，他们可以抛出新的NotImplementedException() ，返回默认值或包含将无法编译的代码。您可以在以下位置配置这些首选项和其他首选项 代码编辑|的会员生成页面 ReSharper选项（ Alt+R,,O ）。

生成部分类型中的部分方法的实现

  1. 在编辑器中，将插入记号放在要插入部分方法的实现的行的部分类型内。

  2. 按Alt+Insert或选择 ReSharper |编辑从主菜单生成代码… 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 部分方法” 。

  4. 在出现的“ 生成”对话框中，选择要为其创建实现存根的部分方法。

  5. 单击完成以完成向导。
    您也可以在“ 代码编辑” |“选项”上单击“ 选项”以查看或修改常见的代码生成首选项。 ReSharper选项的“ 成员生成”页面。

要将此代码生成命令绑定到其自己的快捷方式，请转到“ 工具” |“其他”。选项|环境|键盘并为ReSharper_GeneratePartialMethods命令。
