使用ReSharper，您可以基于源代码片段创建实时/周围模板。ReSharper会自动将某些字段转换为模板参数 。

从代码片段创建实时/环绕模板

  1. 在编辑器中，选择一个代码块以从中创建模板。

  2. 在主菜单中，选择 ReSharper |工具|通过选择创建实时模板...。

  3. 在出现的“ 选择新模板的字段和上下文”对话框中，ReSharper建议使用新模板的草稿并自动将一些字段转换为模板参数：

  4. 如果模板参数部分包含任何条目，则可以在“ 模板”文本区域中检查它们的使用位置和替换位置。

  5. 如果您认为某些参数不是必需的，则可以取消选择它们，以便使用原始代码。选择或取消选择参数时，“ 模板”文本视图也会相应更改。如果不确定是否使用建议的模板参数，请研究“ 模板参数”部分。另外，在接下来的步骤中，您将能够将其他参数引入创建的模板中。

  6. 在上下文列表中，选择范围在新的模板将适用。

  7. 单击创建和编辑 。

  8. 新模板将在“ 模板编辑器”中打开，因此您可以按照“ 创建实时/环绕模板”部分中的说明继续进行创建 。

源代码中的模板保存在“此计算机” 设置层中 。如有必要，您可以稍后将其复制到团队共享的设置层或将其导出到文件中来 共享 。研究ReSharper 存储模板的方式可能也很有用。
