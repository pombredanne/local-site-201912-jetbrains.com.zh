内部多余的空间

属性名称：

[resharper_]js_extra_spaces ，[resharper_]extra_spaces

可能的值：

  • remove_all ：删除所有多余的空格

  • leave_tabs ：保留其他标签

  • leave_multiple ：留出多个额外的空间

  • leave_all ：保留所有多余的空格和制表符
