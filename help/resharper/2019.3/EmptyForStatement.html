代码检查：空的“ for”循环是多余的

您可以禁止执行此检查以忽略特定问题 ， 更改其严重性级别以使问题更小或更明显 ，或者将其完全禁用 。

如果一个for循环不包含指令，并且其初始化，条件和迭代不使用外部作用域中的变量，ReSharper建议删除此循环，例如：

public void MyMethod（）{for（int i = 0; i <100; i ++）{}}

但是请注意， for带有空主体的循环实际上可以用于遍历外部变量，在这种情况下，ReSharper不会将其突出显示为多余的，例如：

公共无效MyMethod（）{int a; var b = 0; for（a = 0; a <100; b + = a ++）{}控制台。WriteLine（“ b =” + b）; }
