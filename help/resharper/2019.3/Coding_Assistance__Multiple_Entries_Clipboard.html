从剪贴板历史记录粘贴项目

ReSharper |编辑糊…
Ctrl+Shift+V
ReSharper_PasteMultiple

使用ReSharper，您可以访问剪贴板历史记录并快速粘贴复制到剪贴板的任何最新项目。一旦启动Visual Studio，ReSharper就会开始将复制的项目记录到其剪贴板历史记录中，并在关闭时清除历史记录。

从剪贴板历史记录中粘贴项目

  1. 将插入标记放在要插入最近复制的项目之一的位置。

  2. 按Ctrl+Shift+V或选择 ReSharper |编辑从主菜单粘贴...。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. ReSharper将向您显示自Visual Studio启动以来已复制的项目列表。

  4. 请执行以下任一操作：

    • 开始键入复制元素中的任何子字符串，以显示包含该子字符串的项目。

    • 使用向上和向下键浏览项目，并在弹出窗口中预览它们。

    • 要研究对话框中的剪贴板项目， 单击在模态对话框中显示 ， 按Shift+Enter在数字键盘上。

  5. 如果选择在对话框中研究剪贴板项目，则可以通过按旁边显示的数字或字母来粘贴任何项目。要查看已复制块的全文，可以使用箭头键在它们之间切换或单击它们。要粘贴所选项目，请按Enter或点击粘贴 。

  6. 要清除剪贴板历史记录，请在对话框中单击“ 清除 ”。
