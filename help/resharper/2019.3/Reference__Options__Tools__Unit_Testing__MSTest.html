微软测试

ReSharper |选项|工具|单元测试|测试框架微软测试

在ReSharper选项的此页面中，您可以调整与MSTest测试相关的单元测试设置。

项目

描述

启用MSTest支持

默认情况下，ReSharper添加操作指示器，使您可以直接从编辑器执行MSTest测试，并自动在解决方案中发现MSTest测试 。清除此复选框可禁用此行为。

测试设定

选择ReSharper应该在哪里寻找测试设置：

使用Visual Studio中指定的 .runconfig / .testsettings 。如果启用此选项，ReSharper将在可能的情况下自动检测测试设置。

指定自定义测试设置配置（解决方案中的.testsettings文件）。
当选择非默认设置配置时，“ 编辑...”按钮将处于活动状态，它将打开“ Visual Studio 测试设置”对话框，您可以在其中调整所选的配置。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
