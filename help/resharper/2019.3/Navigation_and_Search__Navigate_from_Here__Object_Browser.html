Navigate To Object Browser

ReSharper | Navigate | Navigate To | Object Browser
Ctrl+Shift+G | Object Browser
ReSharper_NavigateToObjectBrowser

This command helps you locate a library symbol in the Visual Studio's Object Browser window, where you can get details about a symbol, such as: base types or members, members of the type, and so on. The command becomes available when you invoke Navigate To on a library symbol in your source code and on decompiled sources. You can invoke this command from the editor, from the File Structure window and other tool windows.

If necessary, you can make Object Browser the default navigation option for implicit navigation to compiled code, (for example when you go to declaration of System.String). To learn how to do so, see Navigation and Search in Compiled Code.

The Navigate To is available in the Object Browser window, so you can easily navigate further to: Metadata View, source code, decompiled sources, and other destinations.
