代码布局

ReSharper |选项|代码编辑| CSS |格式样式|代码布局

该页面用于配置声明，选择器等的布局方式。当ReSharper 重新格式化现有代码或键入新代码时，将应用此页面中的选项。 您可以调整值并检查页面底部的预览窗格，以查看首选项如何影响代码。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
