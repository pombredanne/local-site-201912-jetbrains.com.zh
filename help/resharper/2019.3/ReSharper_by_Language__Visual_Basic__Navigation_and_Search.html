Visual Basic .NET中的导航和搜索

ReSharper的所有导航和搜索功能均在C＃中可用在Visual Basic .NET中也受支持。您可以在“ 导航和搜索”部分的相应主题中找到有关这些功能的详细信息。

在本主题中，您可以找到一些在Visual Basic .NET中使用导航和搜索功能的示例。

档案结构

文件结构有助于概述在编辑器中打开并处于活动状态的当前文件的结构。

在“ 文件结构”窗口中双击一个节点，以导航到所选成员的声明。此外，从上下文菜单中，您可以应用导航和搜索功能以及重构。

类型层次

类型层次结构有助于概述类型的继承层次结构。

在“ 层次结构”窗口中双击一个节点，以导航到所选类型的声明。此外，使用上下文菜单可以从当前窗口直接应用搜索功能和重构。

按名称导航

各种“转到...”功能，例如：在所有位置搜索/转到类型 ， 转到符号 ， 转到文件成员 ，等等对于VB.NET也适用。

查找用法

查找用法和查找用法高级功能可帮助您在源代码中找到名称空间，类型，方法等的所有用法。应用“ 查找用法”功能是在解决方案中查找所有代码用法的最快方法。

如果需要更灵活的搜索，请使用“ 查找用法”高级功能。例如，它为您提供了查找文本出现并扩展搜索范围的机会。

突出显示文件中的用法

“ 突出显示文件中的用法”功能可帮助您将注意力集中在特定的成员或局部变量及其出现上。请注意，写访问权限以红色突出显示，而读访问权限以蓝色突出显示。

此外，您可以突出显示名称空间的用法。将插入符号放在Imports指令并调用该功能，ReSharper突出显示依赖于名称空间的所有符号。

查找依赖于模块的代码

查找依赖于模块的代码有助于查找对特定项目或程序集引用的依赖。

查找范围外部的符号有助于查找某个范围外部的任何符号的用法。
