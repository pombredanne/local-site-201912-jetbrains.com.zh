本主题列出了ReSharper 2019.3中所有类似HTML的预定义实时模板。有关实时模板的更多信息，请参见使用实时模板创建源代码。

模板

细节

ctx

当前文件上下文

范围无处不在

身体

$ CTX $

参量

  • CTX -提供描述当前上下文的项目列表。这包括文件名，包含类型名，名称空间名等（ context ）。

nguid

插入新的GUID

范围无处不在

身体

$ GUID $

参量

  • GUID -生成新的全局唯一标识符（GUID）（ guid ）

参量

  • END -应用模板后的插入符位置。

创建标签属性

范围 HTML和类似属性

身体

参量

  • ATTR -建议在当前文档的相同标签中使用的属性名称（ proposalAttributeNameByTag ）
