制表符和缩进

缩进样式

属性名称：

indent_style ，[resharper_]protobuf_indent_style

可能的值：

  • tab ： 标签

  • space ：空间

例子：

标签

选项a = {a {x：1} b {y：2}}

空间

选项a = {a {x：1} b {y：2}}

缩进尺寸

属性名称：

indent_size ，[resharper_]protobuf_indent_size

可能的值：

一个整数

例子：

值：0

选项a = {a {x：1} b {y：2}}

价值：1

选项a = {a {x：1} b {y：2}}

价值：2

选项a = {a {x：1} b {y：2}}

使用制表符缩进时如何对齐

属性名称：

[resharper_]protobuf_alignment_tab_fill_style ，[resharper_]alignment_tab_fill_style

可能的值：

  • use_spaces ：使用空格（看起来与任何标签大小对齐）

  • use_tabs_only ：仅使用标签（不准确）

  • optimal_fill ：混合制表符和空格以实现最佳填充

缩进选项

大括号不在选项中

属性名称：

[resharper_]protobuf_not_options_braces ， [resharper_]protobuf_brace_style ， [resharper_]not_options_braces ，[resharper_]brace_style

可能的值：

  • end_of_line ：行尾（K＆R样式）

  • end_of_line_no_space ：行尾（无空格）

  • next_line ：在下一行（BSD样式）

  • next_line_shifted ：在下一行缩进（铁匠风格）

  • next_line_shifted_2 ：在下一行缩进2（GNU样式）

例子：

行结束

消息a {可选int32 a = 1;消息b {可选int32 c = 1; }}

end_of_line_no_space

消息a {可选int32 a = 1;消息b {可选int32 c = 1; }}

next_line

消息a {可选int32 a = 1;消息b {可选int32 c = 1; }}

next_line_shifted

消息a {可选int32 a = 1;消息b {可选int32 c = 1; }}

next_line_shifted_2

消息a {可选int32 a = 1;消息b {可选int32 c = 1; }}
