Navigate To Control Flow Target

ReSharper | Navigate | Navigate To | Control Flow Target
Ctrl+Shift+G | Control Flow Target
ReSharper_NavigateToControlFlowTarget

This command navigates from an editor position to the next control flow recipient. It becomes available when you invoke Navigate To at foreach, yield, jump statements (break, continue, goto, return), or throw statements.
