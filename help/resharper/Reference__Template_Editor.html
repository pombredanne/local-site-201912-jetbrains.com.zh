模板编辑器

模板编辑器有助于创建和编辑代码模板。 模板编辑器将在新的文档选项卡中打开，以便您可以在模板和应用程序代码之间切换以快速测试创建的模板。

模板编辑器提供了突出显示关键字，注释等的基本代码。所有突出显示的颜色都是可配置的。

为模板编辑器配置突出显示颜色

  1. 选择工具|从菜单中选择选项 ，然后转到环境|字体和颜色页面。

  2. 在“ 显示项目”列表中，找到以ReSharper Template Editor开头的项目。

  3. 使用项目前景选择器为所选项目选择颜色。

模板编辑器中的控件可以分为两个逻辑组，在以下两个表中进行了描述：

  • 模板选项 ：常规模板控件。

  • 模板参数 ：模板主体右侧的参数区域。

模板参数

项目

描述

参数名称

在模板主体中定义的模板参数的名称。

选择宏或[ 宏 名称]

该超链接显示或允许选择一个用于计算参数值的宏 。通过单击此超链接，您可以在“ 选择宏”对话框中选择一个宏。 如果所选宏具有宏参数，则在“ 选择宏”对话框中以粗体显示（例如format或type ）。在这种情况下，选择宏后，模板参数下会出现一个文本字段或一个选择器。您可以在此处指定宏参数。

可编辑或可编辑的事件选择器

这些控件使您可以通过以下方式之一使参数可编辑 ：

  • 如果该参数仅在模板中使用一次，请确保选中该复选框。

  • 如果在模板中多次使用了该参数，则会出现一个组合框，您可以在其中选择顺序编号的可编辑事件。此数字指定在使用“ 热点会话”部署模板时将输入焦点设置为哪个参数出现。您可能还会注意到，这些数字以'M'或其他字母作为前缀。这些字母定义应为此参数设置输入焦点的多文件模板的哪个文件。“ M”留在主文件中，并出现在所有模板中。

上移 / 下移

如果选择参数，则可以使用这些按钮更改参数的顺序。 参数列表中的位置定义顺序，在应用模板时，在热点会话期间按顺序切换参数。它仅影响可编辑的参数。
