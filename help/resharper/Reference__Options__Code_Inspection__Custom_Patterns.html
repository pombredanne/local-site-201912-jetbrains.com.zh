自定义模式

ReSharper |选项|代码检查|自定义模式

ReSharper选项的此页面显示“ 结构搜索和替换”的所有自定义模式。您可以单击列表中的模式以在页面底部显示预览。

项目

描述

添加图案

单击此按钮以打开“ 添加/编辑/使用突出显示模式搜索”对话框并创建新的代码模式。

马上搜索

单击此按钮查找或替换出现的所选代码模式。

编辑

单击此按钮打开“ 编辑突出显示图案”对话框，然后编辑所选的代码图案。

去掉

单击此按钮以删除所选的代码模式。

从ReSharper 6及更低版本导入

单击此按钮可以导入使用较旧版本的ReSharper创建并保存在XML文件中的代码模式。

严重级别选择器

严重性级别选择器显示在列表中每个代码模式的旁边。ReSharper会将指定的严重性分配给与此模式匹配的代码块。自定义模式的可用严重性级别与代码检查的严重性级别相同。有关更多信息，请参见创建自定义代码检查和快速修复 。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
