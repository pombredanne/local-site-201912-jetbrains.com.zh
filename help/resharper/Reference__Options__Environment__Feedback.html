产品反馈

ReSharper |选项|环境|产品反馈

安装ReSharper一段时间后，它可能会在系统托盘中调用一个气球，邀请您加入ReSharper反馈计划，以帮助ReSharper开发人员根据匿名使用情况统计信息改进产品。

单击气球时，将出现“ 产品反馈”提示，让您决定是否参加。

此页面有两个目的：

  • 它为参与该计划提出了全面的条件。

  • 它提供了“ 匿名参与反馈计划”复选框，以便随时加入计划或辞职。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
