TypeScript中的代码生成

ReSharper提供了多种生成样板代码的方法。例如，您可以使用未声明的代码符号，并根据用途自动生成这些符号，生成类型成员等。

ReSharper在TypeScript中提供以下代码生成功能：

  • 根据用法生成代码
  • 生成类型成员
    • 生成构造函数
    • 实施/覆盖成员
    • 产生属性

根据您的设置，可以用不同的方式生成新成员的存根。例如，他们可以抛出新的NotImplementedException() ，返回默认值或包含将无法编译的代码。您可以在以下位置配置这些首选项和其他首选项 代码编辑|的会员生成页面 ReSharper选项（ Alt+R,,O ）。

根据用法生成代码

ReSharper允许您在声明代码符号（类，方法，变量等）之前使用它们。当ReSharper检测到未声明的符号时，它会建议一个或多个快速修复 Alt+Enter用于根据用法生成声明，然后根据用法上下文巧妙地调整声明：

生成类型成员

使用ReSharper，您可以快速生成不同的类型成员。要查看当前类型的可用代码生成选项列表，请按Alt+Insert或选择 ReSharper |编辑在主菜单中生成代码… 。

生成构造函数

构造函数生成向导将创建一个非默认构造函数，该构造函数将使用选定字段的参数。

所有生成的构造函数都遵循相同的模式，其中：

  • 构造函数中包含的每个字段都用一个参数初始化。

  • 参数的名称是从相应字段的名称派生的。

实施/覆盖成员

对于继承的类型，ReSharper提供了两个相关的代码生成功能：

  • 实现缺少的成员可实现当前类中缺少的任何接口成员或抽象成员。

  • 替代成员的作用与实施缺少成员的作用相同，但是如果适用，它也会替代虚拟成员。

产生属性

ReSharper将帮助您为要公开的所有字段生成属性。
