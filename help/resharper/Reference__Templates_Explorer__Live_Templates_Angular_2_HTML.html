Angular 2 HTML的预定义实时模板

ReSharper |模板资源管理器|实时模板| Angular 2 HTML

本主题列出了ReSharper 2019.2中Angular 2 HTML的所有预定义实时模板。有关实时模板的更多信息，请参见使用实时模板创建源代码。

模板

细节

ctx

当前文件上下文

范围无处不在

身体

$ CTX $

参量

  • CTX -提供描述当前上下文的项目列表。这包括文件名，包含类型名，名称空间名等（ context ）。

nguid

插入新的GUID

范围无处不在

身体

$ GUID $

参量

  • GUID -生成新的全局唯一标识符（GUID）（ guid ）

创建脚本标签

范围 HTML和类似标签

身体

参量

  • attributeValue -当前html标记属性的建议属性值（ proposalAttributeValue ）

  • END -应用模板后的插入符位置。

a

创建标签属性

范围 HTML和类似属性

身体

$ ATTR $ =“ $ END $”

参量

  • ATTR -建议在当前文档的相同标签中使用的属性名称（ proposalAttributeNameByTag ）

  • END -应用模板后的插入符位置。

ngClass

范围 Angular 2 HTML属性

身体

[ngClass] =“ {'$ cssClass $'：$ expression $}” $ END $

参量

  • cssClass -没有宏

  • expression -显示指定的值列表。 （ 清单 ）

  • END -应用模板后的插入符位置。

ngFor

范围 Angular 2 HTML属性

身体

* ngFor =“让$ item $ of $ list $” $ END $

参量

  • item -建议可以枚举的可见变量（即在foreach循环中用作集合）（ enumerableVariable ）

  • list -没有宏

  • END -应用模板后的插入符位置。

ngIf

范围 Angular 2 HTML属性

身体

* ngIf =“ $ expression $” $ END $

参量

  • expression -没有宏

  • END -应用模板后的插入符位置。

ngModel

范围 Angular 2 HTML属性

身体

[（ngModel）] =“ $ binding $” $ END $

参量

  • binding -没有宏

  • END -应用模板后的插入符位置。

ngRouterLink

范围 Angular 2 HTML属性

身体

[routerLink] =“ ['$ routeName $']” $ END $

参量

  • routeName -没有宏

  • END -应用模板后的插入符位置。

ngStyle

范围 Angular 2 HTML属性

身体

[ngStyle] =“ {'$ style $'：$ expression $}” $ END $

参量

  • style -没有宏

  • expression -没有宏

  • END -应用模板后的插入符位置。
