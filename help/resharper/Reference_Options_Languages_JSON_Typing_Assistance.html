描述

自动修复冒号以排除JSON密钥的引号

默认情况下，当您键入冒号时，ReSharper会聪明地完成您的JSON密钥:符号。例如，在这里输入冒号"someKey: ReSharper将其转换为"someKey" :这样您就可以快速开始输入值。
如果冒号，您可能要清除此复选框:是JSON中键名的一部分。

自动修复逗号以超出JSON值的引号

默认情况下，当您输入逗号时，ReSharper会聪明地完成您的JSON值,符号。例如，您在此处输入逗号"someValue," ReSharper将其转换为"someValue",这样您就可以快速开始输入其他值。
如果使用逗号，则可能要清除此复选框,是JSON中值名称的一部分。
