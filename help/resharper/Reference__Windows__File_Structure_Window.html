文件结构窗口

ReSharper | Windows |档案结构
Ctrl+F11

使用ReSharper，您可以使用“文件结构”窗口浏览当前文档的结构，这大大简化了大型文件的浏览。该窗口与编辑器同步：切换到另一个编辑器选项卡时，该窗口显示相应文件的结构。

文件成员标有符号图标，您可以在“ 环境” |“配置”中配置它们。 ReSharper选项的常规页面。

在“文件结构”窗口中，您可以访问更多操作。要检查可用命令，可以右键单击任何符号。

  • 从符号导航
  • 查找符号的用法 Alt+F7
  • 生成类型成员 Alt+Insert

  • 执行重构 （ Ctrl+Shift+R ）

  • 拖放类型和类型成员以重新排列其顺序。您所做的更改将立即反映在文件中。

  • 如果打开的文件包含单元测试，则这些单元测试将用相应的图标标记，以显示其上次执行的状态。您也可以直接在“文件结构”窗口中运行和调试单元测试 。

要在“文件结构”窗口中搜索项目，只需开始输入名称或CamelHumps缩写。
