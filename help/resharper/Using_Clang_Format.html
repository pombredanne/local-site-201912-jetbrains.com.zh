使用C格式

ReSharper C ++提供了C ++支持， ReSharper C ++是一种专用产品，可以单独安装，也可以与ReSharper或ReSharper Ultimate并排安装。

从2017.2版本开始，在C ++文件中（部分在JavaScript和TypeScript文件中），ReSharper支持以Clang-Format定义的格式样式。默认情况下，Clang格式配置文件会自动用作格式设置设置的来源。

什么是Clang格式，ReSharper如何支持它？

Clang-Format是一种广泛使用的C ++代码格式化程序。由于它提供了在YAML格式的文件（称为.clang格式或_clang格式）中定义代码样式选项的选项，因此这些文件通常成为您保留所有代码样式规则的项目的一部分。要查看Clang-Format样式选项的完整列表和配置文件的示例，请查看Clang-Format文档 。

要在Windows资源管理器中创建.clang格式的文件，请创建一个名为.clang-format的文件。 ，它将自动重命名为.clang-format 。

如果您的项目碰巧在Clang-Format文件中配置了其代码样式，则ReSharper的代码格式化引擎将自动使用其中的大多数样式来格式化C ++文件，其中一些在JavaScript和TypeScript文件中。

解决方案中的Clang格式

默认情况下，启用Clang-Format属性；它们将覆盖ReSharper和Visual Studio选项以及EditorConfig样式中定义的首选项。 如果您希望ReSharper忽略Clang-Format样式的代码格式和代码语法，请清除相应的复选框。 代码编辑|的“常规格式化程序样式”页面 ReSharper选项。

启用Clang格式支持并且有影响当前文件的.clang格式或_clang格式文件时，ReSharper将帮助您了解应用了哪些Clang格式样式以及这些设置的来源：
