检查一下

ReSharper |检查检查这个…
Ctrl+Shift+Alt+A
ReSharper_InspectThis

此命令是ReSharper代码探索和检查功能的快捷方式：

  • 通话追踪
  • 价值追踪
  • 探索类型层次
  • 查找代码问题
  • 类型依赖图

您可以从编辑器，“ 文件结构”窗口和其他工具窗口中调用此命令。根据上下文，此快捷方式可帮助您访问该上下文中可用的所有代码探索功能。

查看上下文可用的代码探索功能

  1. 将插入号放置在编辑器中的符号上，或在工具窗口中选择该符号。

  2. 按Ctrl+Shift+Alt+A或选择 ReSharper |检查从主菜单中检查...。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 将显示“ 检查此列表”。

  4. 从列表中选择一个项目，然后单击它或按Enter 。
