File Formatting Info window

ReSharper | Windows | File Formatting Info
ReSharper_ShowEditorConfigInfo

This window explains formatting style settings for the currently opened file:

  • The source of indentation and its value.

  • The status of auto-detecting indents preference.

  • The status of EditorConfig support and all .editorconfig files that affect the current file.

  • The status of StyleCop support and all settings.stylecop files that affect the current file.
