介绍参数重构

ReSharper |重构|介绍参数…
Ctrl+Alt+P
ReSharper_IntroduceParameter

通过此重构，您可以通过添加新参数将表达式从方法移动到其调用方。表达式的所有出现都被新参数替换；解决方案中对方法的所有调用都将使用新参数进行更新。

内联参数重构可提供反向功能。

为所选表达式引入参数

  1. 在编辑器中，在方法或构造函数中选择一个表达式。

  2. 请执行以下任一操作：

    • 按Ctrl+Alt+P 。

    • 按Ctrl+Shift+R然后选择介绍参数

    • 右键单击并选择“ 重构”。在上下文菜单中引入参数 。

    • 选择 ReSharper |重构|在主菜单中引入参数… 。

  3. 如果找到一个以上的选定表达式，ReSharper将显示一个下拉菜单，您可以在其中选择是将重构应用于所有实例还是仅应用于当前实例。

  4. 在出现的“ 介绍参数”对话框中，输入新参数的名称。

  5. 如果所选表达式是常量或值类型，则可以将表达式用作引入参数的默认值。为此，请选择“ 设为默认”复选框。

  6. 如果所选表达式依赖于其他变量，则这些变量将显示在“ 选择局部变量作为lambda”部分中。选择要在通用委托中使用的变量（在这种情况下，它将作为参数添加）。

  7. 要应用重构，请单击下一步 。

  8. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
