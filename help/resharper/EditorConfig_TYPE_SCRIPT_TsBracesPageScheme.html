TypeScript-大括号布局

大括号布局

类型和模块

属性名称：

[resharper_]js_types_braces ，[resharper_]types_braces

可能的值：

  • end_of_line ：行尾（K＆R样式）

  • end_of_line_no_space ：行尾（无空格）

  • next_line ：在下一行（BSD样式）

  • next_line_shifted ：在下一行缩进（铁匠风格）

  • next_line_shifted_2 ：在下一行缩进2（GNU样式）

例子：

行结束

模块M {类C {公开f：编号； }接口I {f：number; }}

end_of_line_no_space

模块M {C类{公共f：编号； }接口I {f：数字； }}

next_line

模块M {类C {公开f：编号； }接口I {f：number; }}

next_line_shifted

模块M {类C {公开f：编号； }接口I {f：number; }}

next_line_shifted_2

模块M {类C {公开f：编号； }接口I {f：number; }}
