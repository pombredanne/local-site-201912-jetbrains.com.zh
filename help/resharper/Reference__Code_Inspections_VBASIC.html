VB.NET中的代码检查

在VB.NET中，ReSharper 2019.2提供了两种代码检查：261个检查错误的语法检查，例如语法损坏，无法解析的符号，编译器错误等（您无法配置或禁用任何这些检查），以及250 专有代码检查 ，您可以禁用或更改其严重性水平 。
这些代码检查可帮助您在设计时检测所有打开文件中的代码问题，此外，它们还使您可以查找特定范围内的代码问题 。
