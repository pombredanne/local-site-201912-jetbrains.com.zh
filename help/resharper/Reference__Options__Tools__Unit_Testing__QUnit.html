JavaScript测试

ReSharper |选项|工具|单元测试|测试框架JavaScript测试

在ReSharper选项的此页面中，您可以调整JavaScript和TypeScript中的单元测试帮助的首选项。

项目

描述

茉莉花
启用Jasmine支持

默认情况下，ReSharper添加了动作指示器，使您可以直接从编辑器执行Jasmine测试，并自动在解决方案中发现Jasmine测试 。清除此复选框可禁用此行为。

茉莉花版

选择ReSharper测试运行者使用的Jasmine框架版本。

启用自定义HTML功能

使用此选项可允许ReSharper的测试运行器使用与Chutzpah测试运行器兼容的自定义HTML测试工具运行测试。

测试线束路径

HTML样式文件的文件名或路径。使用选择器指定绝对或相对路径。

将默认实现复制到剪贴板

将带有特殊Chutzpah占位符的默认HTML模板复制到剪贴板。

QUnit
启用QUnit支持

默认情况下，ReSharper添加操作指示器，使您可以直接从编辑器执行QUnit测试，并自动在解决方案中发现QUnit测试 。清除此复选框可禁用此行为。

茉莉花版

选择ReSharper测试运行者使用的Jasmine框架版本。

启用自定义HTML功能

使用此选项可允许ReSharper的测试运行器使用与Chutzpah测试运行器兼容的自定义HTML测试工具运行测试。

测试线束路径

HTML样式文件的文件名或路径。使用选择器指定绝对或相对路径。

将默认实现复制到剪贴板

将带有特殊Chutzpah占位符的默认HTML模板复制到剪贴板。

运行测试
网页浏览器

如果选择此选项，则可以指定ReSharper在其中运行测试的浏览器。

幻影

如果选择此选项，ReSharper将使用PhantomJS运行测试。您将需要指定PhantomJS.exe的相对或绝对路径。

命令行参数

指定传递给所选运行时（Web浏览器或PhantomJS）的可选命令行参数。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
