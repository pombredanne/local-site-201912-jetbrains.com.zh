编码协助

ReSharper提供了多种编码辅助功能，使您可以更快地编写和编辑代码，从而提高工作效率并节省时间。

除了本节中介绍的功能外，ReSharper还提供了许多代码生成功能和代码模板 ，可帮助您更快地创建代码。

ReSharper的大多数编码辅助功能都可以在C＃和Visual Basic .NET中使用，某些可用几种语言提供，其他则是特定于语言的。下表列出了所有支持编码的功能和语言/技术。

本节主题中给出的说明和示例介绍了C＃中编码辅助功能的使用。有关可用于特定语言的编码辅助功能的更多信息，请参见按语言划分的ReSharper部分中的相应主题。

ReSharper C ++提供了C ++支持， ReSharper C ++是一种专用产品，可以单独安装，也可以与ReSharper或ReSharper Ultimate并排安装。
