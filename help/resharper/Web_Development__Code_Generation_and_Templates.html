ASP.NET中的代码生成

ReSharper允许您根据相关母版页文件中定义的内容占位符为ASP.NET Web表单和母版页生成内容。

仅当与文件相关的母版页（或基本母版页）定义了一些占位符，而这些占位符在文件中没有相应的内容单元时，此操作才可用。

为占位符生成内容

  1. 在编辑器中，打开.aspx或.master文件，然后将插入符号放置在要生成Content控件的位置。

  2. 按Alt+Insert或选择 ReSharper |编辑从主菜单生成代码… 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”菜单中，选择“占位符的内容” 。

  4. 从母版页的内容生成的 生成向导中，选择ContentPlaceHolder控件的母版的内容占位符页到当前文件生成相关的内容控制。

  5. 作为一个选项，如果ContentPlaceHolder控件包含默认内容，并且您要将其复制到内容页面，则使用“ 复制默认内容” 。

  6. 您可以单击“ 高级” ，然后单击“ 下一步”以准确查看将要创建的控件。

  7. 单击完成以完成向导。

产生中Content基于的标签ContentPlaceHolder在母版页中定义的标签。
