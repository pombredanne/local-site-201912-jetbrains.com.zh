智能完成

ReSharper |编辑智能代码完成
Ctrl+Shift+Space
ReSharper_CompleteCodeSmart

此命令过滤类型，方法和变量的列表以匹配表达式的预期类型。只要ReSharper可以确定表达式的期望类型，它就可以在任何地方使用，例如，赋值的右侧，变量初始化器或return语句。如果可能有几种返回类型，则ReSharper会检测到适用类型的最近使用过的符号，并自动在查找列表中选择它。Smart Completion还可以建议创建匿名委托，lambda表达式和正则方法，以及为out参数创建局部变量。除此之外，在对象初始化程序中，“智能完成”功能会建议尚未分配值的字段/属性。

您可以连续按几次智能完成快捷方式，以更改生成完成建议的方式。有关更多信息，请参见Double Completion 。

默认情况下，C＃和VB.NET中的智能完成项目包含在自动和基本完成的建议列表中。如果要更改默认行为，请使用“ 环境” | “ 环境”中的相应控件。 智能感知 ReSharper选项的“ 完成行为”页面。

调用智能完成

  1. 将插入符号放置在您要键入代码的位置。

  2. 按Ctrl+Shift+Space 。

  3. 从列表中选择一个建议，然后按Enter或开始键入标识符的首字母或其CamelHumps缩写，以缩小建议列表的范围。

    如有必要，可以通过清除“ 环境” |“ 标识符之间的匹配中间”复选框来禁用CamelHumps 。智能感知 ReSharper选项的“ 完成行为”页面。

您还可以在现有标识符上调用智能完成。在这种情况下，有两种方法可以应用所选的完成建议；按Enter插入并保留现有项目或按Tab替换现有项（如果需要，可以在ReSharper选项的Environment | IntelliSense | Completion Characters页面上更改默认快捷方式）。
