CSS-代码布局

代码布局

声明书

属性名称：

[resharper_]css_declarations_style ，[resharper_]declarations_style

可能的值：

  • separate_lines_for_nonsingle ：如果不是简单的话，请放在单独的行上

  • separate_lines ：在单独的行上

  • same_line ：单行格式

例子：

eparate_lines_for_nonsingle

.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }

分隔线

.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }

same_line

.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }

媒体查询

属性名称：

[resharper_]css_media_query_style ，[resharper_]media_query_style

可能的值：

  • same_line ：单行格式

  • separate_lines ：在单独的行上

例子：

same_line

@media（ratio：4/3），（ratio：3/4）{.blue {color：blue; }}

分隔线

@media（ratio：4/3），（ratio：3/4）{.blue {color：blue; }}

选择器

属性名称：

[resharper_]css_selector_style ，[resharper_]selector_style

可能的值：

  • same_line ：单行格式

  • separate_lines ：在单独的行上

例子：

same_line

.blue，.link {color：blue; }

分隔线

.blue，.link {color：blue; }

大括号

属性名称：

[resharper_]css_brace_style ，[resharper_]brace_style

可能的值：

  • end_of_line ：行尾（K＆R样式）

  • end_of_line_no_space ：行尾（无空格）

  • next_line ：在下一行（BSD样式）

  • next_line_shifted ：在下一行缩进（铁匠风格）

  • next_line_shifted_2 ：在下一行缩进2（GNU样式）

例子：

行结束

@media（ratio：4/3）{.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }}

end_of_line_no_space

@media（ratio：4/3）{.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }}

next_line

@media（ratio：4/3）{.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }}

next_line_shifted

@media（ratio：4/3）{.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }}

next_line_shifted_2

@media（ratio：4/3）{.blue，.link {color：blue; } #error，.error {颜色：红色；后组颜色：白色； }}
