JavaScript中的导航和搜索

ReSharper的大多数导航和搜索功能 JavaScript也支持。您可以在“ 导航和搜索”部分的相应主题中找到有关这些功能的详细信息。在本节的主题中，您还可以找到功能矩阵并检查JavaScript到底支持什么。

在本主题中，您可以找到一些在JavaScript中使用导航和搜索功能的示例。

导航

导航命令可用：

  • 在ReSharper中|导航菜单

  • 在编辑器的上下文菜单中

  • 在导航到上下文菜单上

你也可以：

  • 从“ 文件结构”窗口导航

  • 查找符号的用法

档案结构

文件结构有助于概述当前文件的结构。ReSharper会检测所有编程语言通用的代码结构以及JavaScript的某些特定结构，例如模块。所有模块成员仅在模块内可见，因此对应的图标为每个成员显示。

为了在“ 文件结构”窗口中显示匿名函数，因此显示未命名的函数，ReSharper从其声明中检索其名称。ReSharper考虑了XML文档注释并提取了有用的信息。如果未在函数签名中显式指定参数的数据类型，则仍可以在“ 文件结构”窗口中看到它们，因为它们是在XML文档中指定的。您也可以通过在“ 文件结构”窗口中双击成员来导航到成员声明。

前往申报

转到声明 -将插入号放在函数的名称处，然后直接导航到声明该函数的文件。

如果所选项目具有多个声明，则会提示您从列表中选择一个。
注意配对内声明的函数
