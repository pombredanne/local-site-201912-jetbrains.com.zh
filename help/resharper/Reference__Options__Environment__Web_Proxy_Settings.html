Web Proxy Settings

ReSharper | Options | Environment | Web Proxy Settings

In this page of ReSharper options, you can adjust Web proxy settings for ReSharper's Internet activities. By default, ReSharper follows your system settings and if any proxy server is defined in your system, ReSharper will use it too for fetching sources from symbol servers, verifying license information, detecting updates, and so on.

Item

Description

Use system Web proxy settings when downloading from Internet

You can clear this checkbox to override system Web proxy settings for ReSharper's Internet connection.

Edit

This button appears when you choose to use system proxy server settings.
By clicking on this button you open the Internet Properties dialog, where you can change system Web proxy settings.

ReSharper's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Manage and Share ReSharper Settings.
