在CSS中导航和搜索

ReSharper的大多数导航和搜索功能 CSS也支持。您可以在“ 导航和搜索”部分的相应主题中找到有关这些功能的详细信息。在本节的主题中，您还可以找到功能矩阵并检查CSS确切支持的功能。

在本主题中，您可以找到一些在CSS中使用导航和搜索功能的示例。

转到匹配的标签

此导航功能使您可以查找与特定CSS选择器匹配的所有HTML和其他标记标签。要调用此功能，请将插入符号放在CSS选择器上，然后按Ctrl+Shift+G或选择ReSharper |导航|在主菜单中导航至 ，然后选择匹配的标签 。

此操作没有默认的快捷方式。但是，您可以在Visual Studio选项（ 工具|选项|环境|键盘 ）中分配自定义快捷方式。操作别名为： ReSharper_GotoMatchedTags
