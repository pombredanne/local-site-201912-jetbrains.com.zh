ReSharper 2019.2帮助

HTML中的代码检查

在HTML中，ReSharper 2019.2提供了两种代码检查：两次检查，以检测错误的语法，例如语法错误，符号解析错误，编译器错误等（您无法配置或禁用这些检查中的任何一种），以及37 专有代码检查 ，您可以禁用或更改其严重性水平 。
这些代码检查可帮助您在设计时检测所有打开文件中的代码问题，此外，它们还使您可以查找特定范围内的代码问题 。

以下列出了可配置的HTML检查，按其类别分组。

代码通知（2次检查）

此类别将代码检查的严重性级别较小。
