ReSharper 2019.2帮助

通过语言重新整理：Visual Basic .NET

ReSharper支持最多VB 14和部分VB 15的Visual Basic .NET。

默认情况下，ReSharper根据关联的编译器自动检测VB.NET版本。但是，您可以显式指定目标VB.NET版本-右键单击解决方案资源管理器中的项目，从上下文菜单中选择“ 编辑项目项属性” ，然后使用VB.NET语言级别选择器。
根据当前的VB.NET版本（自动检测还是明确设置），ReSharper会调整其功能。例如， 代码检查可以检测代码中的位置，使用新的语言功能可以简化这些位置。

VB.NET中的ReSharper功能描述按以下主题分组：

  • 代码检查和快速修复
  • 编码协助
  • 代码样式协助
  • 导航和搜索
  • 重构
  • 代码生成
  • 代码模板
