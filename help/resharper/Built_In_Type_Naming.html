代码语法样式：内置类型引用

C＃类型关键字和.NET Framework（CLR）类型名称对于内置C＃类型可以互换。如果您希望对内置类型名称使用一种或另一种样式，ReSharper可以帮助您保持自己的偏好。

您可以为静态成员访问维护单独的样式（例如，CLR类型名称- var str = String.Empty; ）以及本地变量，成员和参数（例如，关键字- void Foo(string str){ } 。

ReSharper可以帮助您在现有代码中强制执行C＃内置类型的首选项，并在使用时将您的首选项考虑在内产生具有代码完成和代码生成功能的新代码，应用代码模板并执行重构 。

强制执行C＃内置类型的首选项

ReSharper会检查所有内置类型引用是否符合您的偏好 ，如果不符合您的偏好 ，它将突出显示此类引用并建议相应的快速修复或范围内的修复 。默认ReSharper的首选项表示C＃关键字是首选。

内置类型命名设置不适用于生成的代码。默认情况下，ReSharper提供通常由Visual Studio生成的生成代码（例如Windows Form Designer生成的代码）的文件掩码和区域列表。如有必要，您可以在上编辑生成的项目列表 代码检查|的“生成的代码”页面 ReSharper选项（ Alt+R,,O ）。

在批量模式下对C＃内置类型强制执行首选项的另一种方法是代码清除 。您可以使用默认配置文件“ 完全清理”或“ 重新格式化并应用代码样式”之一运行代码清理 ，或者创建和运行仅针对特定任务的自定义配置文件 ，如下所述。

通过自定义代码清除配置文件对C＃内置类型引用应用首选项

  1. 选择ReSharper |在主菜单或按选项 Alt+R,,O 。

  2. 转至代码清理设置页面： 代码编辑|代码清理 。

  3. 如创建新的自定义清理配置文件部分中所述创建新的配置文件 。在新配置文件的“ 选定的配置文件设置”部分中，选中“ 修复内置类型引用”复选框。（可选）您可以在此配置文件中启用其他代码清除任务。

  4. 请点击单击保存以应用修改，并让ReSharper选择将修改保存到何处，或使用“ 保存到”列表将修改保存到特定的设置层。有关更多信息，请参阅管理和共享转发器设置 。

  5. 选择要在其中执行首选项的范围：

    • 将插入符号设置在文件中的任何位置，以强制您对文件进行首选项。

    • 在解决方案资源管理器中选择一项或多项，以在这些节点及其子项下的文件中强制执行您的首选项。

  6. 请执行以下任一操作：

    • 按Ctrl+Alt+F或选择 ReSharper |编辑从主菜单清除代码 。

    • 右键单击文本编辑器中的任意位置，或右键单击所选内容，然后在上下文菜单中选择“ 清理代码” 。

  7. 在打开的“代码清除”对话框中，选择新创建的配置文件。

  8. 点击运行 。ReSharper将在所选范围内强制执行您的首选项。

如果要在不打开“ 代码清除”对话框的情况下强制执行C＃内置类型的首选项，可以将创建的配置文件绑定到静默清除并通过按以下命令运行它Ctrl+Shift+Alt+F 。您还可以创建一个自定义清除配置文件 ，该配置文件将应用首选项与其他代码样式任务结合在一起。

配置用于引用C＃内置类型的首选项

内置的类型命名首选项是使用基于图层的设置机制保存的。除其他外，此机制使您可以针对不同的解决方案维护不同的首选项，并将这些首选项保持在VCS下，并自动与团队成员共享。

为C＃内置类型配置首选项

  1. 去 代码编辑| C＃|的代码样式页面 ReSharper选项（ Alt+R,,O ）。

  2. 根据您的编码惯例/标准，修改“ 内置类型命名”类别中的设置。

  3. 通过右栏中的“ 使用选择器通知”，您可以设置代码检查的 严重性级别 ，以检测与首选项不同的代码。

  4. 请点击单击保存以应用修改，并让ReSharper选择将修改保存到何处，或使用“ 保存到”列表将修改保存到特定的设置层。有关更多信息，请参阅管理和共享转发器设置 。

从编辑器更改引用C＃内置类型的首选项

  1. 将插入符设置为ReSharper检查中突出显示的代码问题。

  2. 按Alt+Enter或单击插入符号左侧的动作指示器以打开动作列表 。

  3. 在操作列表中，选择“ 检查”“替换内置类型引用...”。配置代码样式 ，然后选择所需的首选项：

  4. 您的更改将使用智能保存逻辑进行保存。

  5. 如果需要将修改后的样式首选项保存在共享设置层中 ，请单击“ 配置代码样式”菜单项或按Enter选择它时。ReSharper将打开代码编辑| C＃| ReSharper选项的“代码样式”页面，您可以在其中根据需要修改首选项，单击“ 保存到” ，然后选择所需的设置层。
