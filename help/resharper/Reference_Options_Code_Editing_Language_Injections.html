Language Injections

ReSharper | Options | Code Editing | Language Injections

On this page, you can view automatic language injections that ReSharper applies in specific cases. All injections are sorted by the injected language. If necessary, you can clear check-boxes to disable specific injections.

ReSharper's settings are saved in settings layers, which enable you to have different settings for different solutions and to share your settings with your team. To learn more, see Manage and Share ReSharper Settings.
