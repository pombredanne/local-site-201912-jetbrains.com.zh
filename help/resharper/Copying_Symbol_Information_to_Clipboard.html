将符号信息复制到剪贴板

ReSharper提供了两个方便的命令，用于将符号信息复制到剪贴板：

  • 复制XML-Doc ID
  • 复制全名

如果您经常使用这些命令，则可以将快捷方式分配给相应的命令（请参阅标题下的命令别名），也可以使用“ 导航”操作 Ctrl+Shift+A 。

复制XML-Doc ID

ReSharper |编辑将XML-Doc ID复制到剪贴板
ReSharper_CopyXmlDocIdToClipboard

当您在任何符号（声明或用法）上调用此命令时，它会将其XML-Doc ID复制到剪贴板，该ID将在代码的XML文档中标识该符号。从其他符号的XML文档注释中引用该符号时，可以使用此ID。有关ID格式的更多信息，请参见处理XML文件（C＃编程指南） 。

例如，对于方法MyMethod声明为

名称空间MyNamespace {类MyClass {void MyMethod（）{}}}

XML-doc iD将是M:MyNamespace.MyClass.MyMethod

复制全名

ReSharper |编辑将标准名称/源浏览器URI复制到剪贴板
ReSharper_CopyFqn

当您在任何符号（声明或用法）上调用此命令时，它会将其标准名称或任何前缀部分复制到剪贴板：

在不同情况下，例如使用反射时，可能需要符号的全限定名称。

如果使用JetBrains Upsource和/或基于Roslyn的源浏览器在线浏览源代码，则可以使用此命令复制URI，以在浏览器中找到当前符号。
如果要使用这些附加功能，请为您在环境|环境中使用的服务配置基本URI。 ReSharper选项的“ 搜索和导航”页面。
