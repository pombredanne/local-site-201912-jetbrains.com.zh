数据提示
使用ReSharper DataTips覆盖VS DataTips

ReSharper可以选择用自己的DataTips替换Visual Studio DataTips，它们有很多区别：

  • 您可以在当前的暂停执行上下文中搜索特定值。为此，只需在焦点位于DataTip中时开始键入即可。请注意，搜索将仅在扩展节点中找到值。

  • 不是在删除鼠标指针时，而是在DataTip之外的任何位置单击时，DataTip消失。

  • 如果一个类型都没有意义ToString()覆盖也不用[DebuggerDisplay]属性，那么在调试器视图中这种类型的对象的表示可能完全没有信息。在ReSharper DataTips中，您可以即时自定义此类对象的表示形式。为此，请在“数据提示”中展开类型，右键单击所需的数据成员，然后选择“ 突出显示属性” ：

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
