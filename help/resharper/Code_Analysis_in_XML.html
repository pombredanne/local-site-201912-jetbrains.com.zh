快速修复的示例

修正XML处理指令

如果处理指令存在问题，ReSharper会通知您并建议快速修复以解决该问题。

应用快速修复后，将根据XML规则修复处理指令。

删除错误的XML注释

如果开始标记中有注释，ReSharper将提供一个快速修复程序以删除该注释。

应用快速修复后，注释将被正确删除。

插入结束标签

如果文档结构不正确并且没有匹配的开始标签，请使用此快速修复程序插入相应的结束标签。

应用此快速修复后，关闭id标签已插入。
