xUnit.net

ReSharper |选项|工具|单元测试|测试框架xUnit.net

在ReSharper选项的此页面中，您可以调整与xUnit.net测试相关的单元测试设置。

项目

描述

启用xUnit.net支持

默认情况下，ReSharper添加操作指示器，使您可以直接从编辑器执行xUnit.net测试，并在解决方案中自动发现xUnit.net测试 。清除此复选框可禁用此行为。

xUnit.net配置文件

使用“ 自动检测”选项，可以在任何标准位置找到xUnit.net配置文件。如果您的配置文件位于自定义位置，请选择“ 特定”，然后指定配置文件的路径。要了解配置文件的格式，请参阅使用JSON 配置xUnit.net和使用XML配置xUnit.net

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
