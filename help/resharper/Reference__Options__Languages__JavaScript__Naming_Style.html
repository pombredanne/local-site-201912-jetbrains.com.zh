命名方式

ReSharper |选项|代码编辑| JavaScript |命名方式

ReSharper选项的此页面定义JavaScript的符号命名规则 。

ReSharper可帮助您为代码中的符号定义，控制和应用命名样式。命名样式是作为一组规则实现的，每个规则都针对具有一组约束的特定标识符。每个规则可以具有一个或多个关联的样式，这些样式定义后缀，前缀，复合词的大写形式，等等。

ReSharper时会考虑这些规则产生具有代码完成和代码生成功能的新代码，应用代码模板并执行重构 。ReSharper还可以帮助您检测和修复违反命名规则的情况。如有必要，可以配置或禁用自动检查命名规则。

项目

描述

编辑

单击此按钮，或双击列表条目，以使用“ 编辑规则设置”对话框编辑选定的命名规则。有关编辑命名规则的详细信息，请参阅“ 修改默认命名规则” 。

缩略语

添加大写的缩写，ReSharper 不应在符号名称中使用它们时警告您 。使用换行符，逗号，分号或空格作为分隔符。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
