探索PDB内容

ReSharper通过生成可读的XML文件来帮助您浏览程序集PDB内容，该文件包含：

  • 源文件名，

  • IL偏移量和源极线之间的映射，

  • 局部变量的名称，

  • 入口点方法（如果存在），

  • 导入范围

  • 一种PDB文件（Windows /便携式/嵌入式），

  • PDB签名，

  • 源服务器信息：SourceLink / SrcSrv。

要研究与程序集关联的PDB内容，请在“ 程序集资源管理器”窗口中右键单击它，然后选择“ 显示PDB内容” 。ReSharper将生成一个XML文件并在编辑器中将其打开。

如果所选的程序集没有PDB，则“ 显示PDB内容”将不可用。

如果您想随时清除反编译缓存，则可以删除％LOCALAPPDATA％\ JetBrains \ Shared \ vAny \ DecompilerCache中的数据 。
