该页面允许您在编辑器中调整ReSharper的XAML功能。这些是特定于语言的设置，是对主要编辑器功能集的补充。

项目

描述

同步更改匹配标签

如果选中此复选框，则ReSharper在编辑元素的开始标记时会同时更改结束标记，反之亦然。

自动在代码完成中插入标签页脚

如果选中此复选框，则在应用任何代码完成后，ReSharper都会插入页脚（关闭）标签。如果未选择，则在关闭开始标签时插入结束标签。

使用智能行为自动完成代码

如果选中此复选框，则ReSharper会将智能完成应用为自动完成代码；如果未选中，则默认情况下将应用基本完成 。
使用XAML文件时，基本完成通常会提供过多的完成建议，而智能完成建议在大多数情况下会更有帮助。
如果选择此选项，则可以使用Ctrl+Space返回到符号完成并获得更多建议。

在代码完成中扩展名称空间别名

如果选中此复选框，则完整名称空间名称将显示在完成列表中，否则，将显示简短别名。

始终在代码完成时创建自封闭标签

如果选中此复选框，则每当代码完成插入新标签时，ReSharper都会插入自关闭标签。否则，将创建一对开始和结束标签

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
