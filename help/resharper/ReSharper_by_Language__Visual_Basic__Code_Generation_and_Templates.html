Visual Basic .NET中的代码生成

Visual Basic .NET也支持C＃中所有ReSharper的代码生成功能。您可以在“ 代码生成”部分的相应主题中找到有关这些功能的详细信息。

生成类型成员

所有成员生成操作均显示在“ 生成”菜单中，该菜单随Alt+Insert捷径。

生成事件订阅

除了常见的成员生成操作集之外，ReSharper还提供了特定于Visual Basic .NET的Generate event subscriptions操作。

ReSharper帮助您在ASP.NET CodeBehind文件中快速生成事件订阅方法。
