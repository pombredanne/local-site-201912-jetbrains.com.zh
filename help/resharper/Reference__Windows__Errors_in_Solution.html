解决方案窗口中的错误/警告

ReSharper | Windows |解决方案错误
ReSharper_ShowErrorsView

此窗口显示在解决方案范围分析期间发现的解决方案范围错误/警告的列表。

您可以选择在解决方案范围的分析中仅监视错误，或同时监视错误和警告。

工具栏控件

控制

名称

描述

分析解决方案中的错误

打开/关闭整个解决方案范围的分析 。

/

暂停/继续分析

如果关闭解决方案范围的分析，然后决定再次打开它 ，则ReSharper将不得不重复一些初始计算，这可能会很耗时。为避免这种情况，您可以暂停并恢复解决方案范围的分析。

包含/排除警告

使用此按钮可以在解决方案范围内的分析中启用/禁用警告。

在解决方案范围的分析中启用警告时，其工作方式如下：只要解决方案中存在错误，就只会显示错误；如果解决方案中存在错误，则仅显示错误。未解决的警告仅在修复了最后一个错误后才会出现。禁用警告后，一旦纠正了最后一个错误，状态栏指示器就会变为绿色。

忽略错误

从发现的问题列表中排除选定的错误/警告（或选定文件中的所有错误/警告）（如果打开了“ 显示忽略的错误”，则将其删除 ）。有关更多信息，请参阅忽略解决方案中的错误 。

停止忽略错误

显示或以前忽略的错误（如果打开“ 显示忽略的错误”，则删除删除线）。有关更多信息，请参阅忽略解决方案中的错误 。

显示忽略的错误

使被忽略的错误可见，但将其删除。

重新分析有错误的文件

启动重新分析先前发现错误的文件的过程。

重新分析所有

启动重新分析解决方案中所有文件的过程。

出口

点击这个按钮以文本格式导出当前在窗口中显示的数据，或使用下拉选择器以XML或HTML格式导出数据。出现的“ 导出数据”对话框将帮助您将数据保存到文件或将其复制到剪贴板。
