语法样式

代码样式的方面之一是如何使用可互换的语言语法构造。例如，从编译器的角度来看，以下两个方法定义是相同的，但是语法结构的选择使它们看起来有所不同：

[Conditional（“ DEBUG”）] [测试]静态公共无效Foo（String str）{foreach（str ch ch）{系统。安慰。WriteLine（value：ch）; }}
[Test，Conditional（“ DEBUG”）] public static void Foo（string str）{foreach（字符串中的ch）。WriteLine（ch）; }

配置代码语法样式

当您开始使用ReSharper时，它将参考其默认设置，以确保在整个代码库中始终使用可互换的语法构造。但是，您可以根据自己的喜好配置每个个人首选项。有两种方法可以配置代码语法样式：

  • 在上配置您的首选项 代码编辑| [语言] |的代码样式页面 ReSharper选项（ Alt+R,,O ）。这些修改将使用分层设置的机制进行保存，以便您可以将其应用于所有项目的本地计算机上，或与队友共享。

  • 使用ReSharper定制的EditorConfig代码样式属性在EditorConfig文件中指定首选项，这还将使您在整个团队中拥有一致的代码样式。
    请注意， .editorconfig文件中指定的首选项将覆盖 ReSharper设置中保存的首选项。

C＃中代码语法样式的其他操作

在C＃中，ReSharper提供了专门的操作Ctrl+Alt+S （ ReSharper |编辑Apply Code Style ）以应用您的代码语法样式。

为现有代码应用C＃代码语法样式

  1. 选择要在其中应用代码样式首选项的范围：

    • 在编辑器中进行选择，以在选择中应用代码样式首选项。

    • 在文件中的任何位置设置插入符号，以在文件中应用代码样式首选项。

    • 在解决方案资源管理器中选择一个或多个项目，以在这些项目及其子项目下的文件中应用代码样式首选项。

  2. 按Ctrl+Alt+S或选择 ReSharper |编辑从主菜单应用代码样式 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

您还可以通过相应的上下文操作在当前选择中应用代码语法样式。

在当前选择中应用C＃代码语法样式

  1. 在编辑器中，选择要在其中应用代码语法样式的代码块。

  2. 按Alt+Enter或单击插入符号左侧的动作指示器以打开动作列表 。

  3. 选择在操作列表中应用代码样式 。
