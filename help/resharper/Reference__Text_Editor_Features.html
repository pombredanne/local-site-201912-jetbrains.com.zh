ReSharper Controls in Text Editor

Besides the variety of dynamic controls produced by ReSharper's coding assistance features (for example completion lists, quick documentation popup), ReSharper adds the following things to the Visual Studio text editor:

  1. Action Bar that displays available ReSharper actions.

  2. Action indicators that notifies if an action is available for a specific code line or the current caret position.

  3. Action list that suggests a list of actions available for the current caret position.

  4. Hot Spot Session that helps completing actions that require user assistance.

  5. Status indicator that helps you to see at once whether the current file has errors or warnings.

  6. Marker bar that by default indicates that a code issue is detected on a specific line.
