ReSharper 2019.2帮助

文本编辑器中的ReSharper控件

除了ReSharper的编码辅助功能（例如完成列表 ， 快速的文档弹出窗口 ）所产生的各种动态控件之外，ReSharper还向Visual Studio文本编辑器中添加了以下内容：

  1. 显示可用的ReSharper操作的操作栏 。

  2. 动作指示器 ，用于通知特定代码行或当前插入符号位置是否有动作。

  3. 动作列表 ，建议可用于当前插入符号位置的动作列表。

  4. 热点会话 ，有助于完成需要用户帮助的操作。

  5. 状态指示器 ，可帮助您立即查看当前文件是否有错误或警告。

  6. 标记栏默认情况下指示在特定行上检测到代码问题。
