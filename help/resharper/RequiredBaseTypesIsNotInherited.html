Code Inspection: Base type is required

You can configure this inspection:

  • disable it to ignore all related issues
  • suppress it in any place in the code to ignore specific issues
  • change its severity level to make the detected issues less or more noticeable

One of the ReSharper annotation attributes is the BaseTypeRequired attribute, which ensures that classes marked with this attribute derive from a specific class. Consider the snippet below:

[BaseTypeRequired(typeof(Soldier))] class FriendlyAttribute : Attribute { } class Soldier { } [Friendly] class Marine { }

In this snippet, the Marine class will generate a warning because the Friendly attribute itâ€™s tagged with expects the class to be derived from Soldier. The following change fixes the problem:

[Friendly] class Marine : Soldier { }
