ReSharper 2019.2帮助

介绍

通过对C＃，VB.NET，XAML，JavaScript，TypeScript，JSON，XML，HTML，CSS，ASP.NET，ASP.NET MVC，Protobuf，NAnt和MSBuild脚本的无与伦比的支持，包括全面的跨语言功能，JetBrains ReSharper将为您提供帮助任何Visual Studio用户都可以编写更好的代码，轻松检查和重构现有的代码库。

另外， ReSharper C ++提供了C ++支持， ReSharper C ++是一种专用产品，可以单独安装，也可以与ReSharper或ReSharper Ultimate并排安装。

您可以花更少的时间进行例行的重复性手动工作，而将精力集中在手头的任务上。一组强大的功能可用于自动错误检查和代码纠正，从而缩短了开发时间并提高了效率。您会发现ReSharper可以通过提高开发人员的工作效率和提高代码质量来迅速收回成本。借助ReSharper，.NET开发人员可以真正体验到我们所说的“开发的动力” 。

如果您从ReSharper 帮助菜单进入此页面，并且正在使用ReSharper Ultimate ，您可能会对Ultimate中包含的其他JetBrains .NET产品的文档感兴趣：

  • dotTrace —用于.NET性能分析的工具
  • dotMemory —内存分析工具
  • dotCover — .NET单元测试运行程序和代码覆盖率工具
  • dotPeek-一个免费的.NET反编译器
  • TeamCity加载项，可帮助您访问，执行和分析TeamCity服务器上的构建和测试
  • dotMemory Unit —一个单元测试框架，用于检查代码中是否存在内存问题

ReSharper功能

ReSharper 2019.2通过提供以下功能来帮助您提高生产力：

  • 代码质量分析
    通过对所有受支持语言的设计时代码检查 ，ReSharper会立即告诉您代码是否包含错误或可以改进。如有必要，您还可以在整个解决方案的任何范围内运行静态代码分析。

  • 修复检测到的代码问题
    ReSharper不仅会在代码出现问题时向您发出警告，而且还提供快速修复以自动消除错误和代码异味。

  • 项目依赖分析
    对于大型解决方案而言，ReSharper不可估量，它可以构建项目层次结构并可视化项目依赖关系图 ，您可以在各个阶段保存并与新的解决方案状态进行比较，而无需编译任何内容。

  • 类型依赖分析
    ReSharper可以快速分析解决方案中类型之间的不同种类的依赖关系，并使用分层视图或类型依赖关系图将它们可视化。

  • 导航和搜索
    您可以立即跳转到代码库中的任何文件 ， 类型或成员 ，或者从特定符号导航到其用法 ， 基础符号和派生符号或实现 。

  • 反编译第三方代码
    集成的反编译器使您可以导航到引用程序集中的代码 ；如果未引用程序集，则可以使用“ 程序集资源管理器”窗口打开并浏览它，该窗口具有与源代码相同的导航功能。

  • 代码编辑助手
    多种代码编辑助手，包括扩展的IntelliSense ，数百种即时代码转换 ， 自动导入名称空间 ， 重新排列代码和显示文档 。

  • 代码生成
    您不必手动编写属性，重载，实现和比较器：使用代码生成操作可以更快地处理样板代码。

  • 安全更改代码库
    享受解决方案范围内的重构，以安全地更改您的代码库。无论您是需要振兴旧代码还是需要整理项目结构，都可以依靠ReSharper。

  • 符合编码标准
    代码格式化 ， 命名样式帮助以及许多其他代码样式首选项 ，可以通过代码清理快速实施这些首选项 ，从而使您摆脱未使用的代码并确保符合编码标准。

  • 更多功能
    ReSharper提供了更多功能： 可扩展模板 ， 正则表达式帮助 ， 国际化帮助 ，强大的单元测试运行器 ，仅举几例。还有一些特定于语言的功能：例如，针对ASP.NET/ASP.NET MVC ， XAML ， JavaScript ， TypeScript或CSS的功能 。

  • 扩展名
    ReSharper扩展，包括成熟的插件，模板集，结构搜索和替换模式等，非常容易发现，安装和更新。有关更多信息，请参见ReSharper插件网页 。

  • 命令行工具
    如果您喜欢ReSharper 检查代码的方式，则可以使用独立的命令行工具在CI服务器或版本控制系统中运行代码检查或查找代码重复项。

支持的语言

ReSharper 2019.2的功能可用以下语言提供（包括语言注入 ）：

  • 直到C＃8.0都完全支持C＃。有关配置C＃版本的更多信息，请参见ReSharper by Language：C＃ 。

  • VB 14和部分VB 15支持Visual Basic .NET 。

  • ReSharper C ++提供了C ++支持， ReSharper C ++是一种专用产品，可以单独安装，也可以与ReSharper或ReSharper Ultimate并排安装。C ++支持包括C，C ++ 03，C ++ / CLI和部分C ++ 11，C ++ 14，C ++ 17，C ++ 20。支持ATL，MFC和COM项目。有关更多信息，请参见按语言分类的ReSharper：C ++ 。

  • HTML最多支持HTML 5。

  • ASP.NET最多支持4.0版本和ASP.NET MVC高达5.0版本。网站项目和Web应用程序项目模型均受支持。

  • 支持JavaScript 直到ECMAScript 2016，包括实验性功能，例如异步/等待，指数运算符以及对象文字/解构中的休息/传播。 jQuery和JSX语法也受支持。

  • 2.0版之前一直支持TypeScript 。

  • CSS最多支持4.0。

  • 直到1.1版都支持XML 。

  • XAML最新支持的Silverlight框架版本是Silverlight 5。ReSharper识别XAML文件中声明的符号并解析引用。

  • 支持MSBuild和NAnt的 构建脚本 。

  • 支持JSON和JSON模式 。

  • Protobuf支持适用于2.x和3.0版本。

要检查支持的语言的功能的详细信息，可以使用ReSharper功能矩阵或研究“ 按语言划分的ReSharper”部分中的相应主题。

产品和许可证

考虑到用户的不同需求，我们提供不同的产品和许可证：

  • 锐化器
    提供除C ++之外所有受支持的语言和技术的功能，包括跨语言功能。

  • ReSharper C ++
    如果您仅使用C ++进行开发，则可以开始享受ReSharper广泛的C ++功能，而无需使用与您不需要的其他语言相关的功能。

  • ReSharper Ultimate许可
    该许可证允许您使用ReSharper，ReSharper C ++以及dotTrace ， dotCover和dotMemory 。

  • ReSharper Ultimate + Rider
    有了此许可证，您还可以使用JetBrains Rider （一个独立的跨平台.NET IDE）。

  • 所有产品包
    顾名思义，该许可使您可以访问所有JetBrains桌面产品。

根据您拥有的许可证，将自动激活该组产品。在评估期内，所有已安装的JetBrains .NET产品均可用。如有必要，您可以在“ 环境” |“禁用”中禁用其中的一些。 ReSharper选项的“ 产品和功能”页面。
