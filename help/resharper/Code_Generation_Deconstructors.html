元组解构是C＃7.0的功能。 默认情况下，ReSharper根据关联的编译器自动检测C＃版本。但是，您可以显式指定目标C＃版本-在解决方案资源管理器中右键单击项目，从上下文菜单中选择“ 编辑项目项属性” ，然后使用C＃语言级别选择器。

ReSharper可帮助您快速生成超载Deconstruct选定成员的方法。

您可以在上配置通用代码生成首选项 代码编辑|的会员生成页面 ReSharper选项（ Alt+R,,O ）。

生成解构器

  1. 在编辑器中，将插入号设置为类型名称，或在要插入解构函数的行的类型内。如果插入符号在类型名称上，则生成的代码将添加到类型声明的开头。

  2. 按Alt+Insert或选择 ReSharper |编辑从主菜单生成代码… 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 析构函数” 。

  4. 在出现的“ 生成”对话框中，选择应在解构函数中使用的类型成员。

  5. 单击完成以完成向导。
    您也可以在“ 代码编辑” |“选项”上单击“ 选项”以查看或修改常见的代码生成首选项。 ReSharper选项的“ 成员生成”页面。

您可以调用查找用法 Alt+F7在Deconstruct找到所有相关元组解构的方法。

ReSharper还检查类型是否包含所需的解构函数，如果不包含，则根据元组解构建议快速修复 。

要将此代码生成命令绑定到其自己的快捷方式，请转到“ 工具” |“其他”。选项|环境|键盘并为ReSharper_GenerateDeconstructor命令。
