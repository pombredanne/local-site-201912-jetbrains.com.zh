创建/编辑图案对话框

ReSharper |选项|工具|待办事项|图案创建/编辑

使用此对话框，您可以在代码中编辑待办事项的模式。

项目

描述

名称

在此字段中，指定图案的名称。

图案

在此字段中，输入将与代码中的模式匹配的正则表达式。

网址

在此字段中，您可以选择指定一个URL，该URL将被添加到与该模式匹配的项目中 ：

图标

在此列表中，在“ 待办事项浏览器”窗口中选择一个图标来表示匹配的待办事项。

区分大小写

选择此复选框，以根据情况查找匹配的待办事项。

在评论中

选择此复选框以查找注释中的待办事项。例：

// TODO：这需要清理

在字符串中

选中此复选框可在字符串文字中查找待办事项。例如：

[Ignore（“ TODO：RSRP-67377”）] [测试] public void test07（）{DoNamedTest（）; }

在标识符中

选择此复选框可在方法，变量，字段和其他标识符的名称中查找待办事项。例如：

抛出新的NotImplementedException（）;
