在本教程中，我们将使用一组标准方法创建一个类型。

在C＃中创建类型或实体可能非常困难。毕竟，我们当中有多少人手工记住了正确的实施方式Equals() ？幸运的是，ReSharper可以为您提供帮助。让我们来看看。
