产生委派成员

  1. 在编辑器中，将插入号设置为类型名称，或在要插入委派成员的行的类型内。如果插入符号在类型名称上，则生成的代码将添加到类型声明的开头。

  2. 按Alt+Insert或选择 ReSharper |编辑从主菜单生成代码… 。或者，您可以按Ctrl+Shift+A ，开始在弹出窗口中键入命令名称，然后在此处选择它。

  3. 在“ 生成”弹出窗口中，选择“ 委派成员” 。

  4. 在出现的“ 生成”对话框中，您将看到当前类型的私有属性和字段的列表。您可以展开这些项目以查看其类型的成员。选择这些类型成员中的一些或全部，ReSharper将在当前类型中生成将执行委托给所选类型成员的包装器。

  5. 单击完成以完成向导。
    您也可以在“ 代码编辑” |“选项”上单击“ 选项”以查看或修改常见的代码生成首选项。 ReSharper选项的“ 成员生成”页面。

要将此代码生成命令绑定到其自己的快捷方式，请转到“ 工具” |“其他”。选项|环境|键盘并为ReSharper_GenerateDelegating命令。
