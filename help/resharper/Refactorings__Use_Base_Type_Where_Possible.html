在可能的情况下使用基本类型

ReSharper |重构|尽可能使用基本类型…
ReSharper_UseBaseTypeAction

这种重构有助于将对类型的引用替换为其对基本类型之一的引用。ReSharper将分析整个解决方案中所有对该类型的引用，并更新可以安全地替换为所选基本类型的引用。（可选）您也可以替换is和typeof操作员。

尽可能使用基本类型

  1. 通过以下方式之一选择类型：

    • 在编辑器中，将插入号设置为类型的名称。

    • 在解决方案资源管理器中选择一种类型。

    • 在“ 文件结构”窗口中选择一种类型。

    • 在“类视图”中选择一种类型。

    • 在对象浏览器中选择一种类型。

    • 在类型依赖关系图中选择一种类型。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择“ 使用基本类型”

    • 右键单击并选择“ 重构”。在上下文菜单中的可能的地方使用基本类型 。

    • 选择 ReSharper |重构|在主菜单中， 在可能的地方使用基本类型 。

    将打开“ 可能时使用基本类型”对话框。

  3. 选择基本类型之一。

  4. （可选）选择是否要替换is操作员和/或typeof操作员。

  5. 要应用重构，请单击下一步 。

  6. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
