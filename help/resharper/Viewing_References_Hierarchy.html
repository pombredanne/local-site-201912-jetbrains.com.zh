探索参考层次

此功能可帮助您探索任何程序集的引用层次结构并跟踪所有程序集依赖项。检测到递归依存关系，并在参考条目的右侧标记字形。

ReSharper还允许浏览解决方案中任何项目的项目引用层次 。

查看装配体的参考层次结构

  1. 在Assembly Explorer窗口中选择一个装配。

  2. 右键单击并在上下文菜单中选择“ 引用层次结构 ”。

  3. 在出现的“ 层次结构”窗口的新选项卡中，可以浏览层次结构或程序集引用：

  4. 要在引用的程序集和引用所选程序集的当前加载的程序集之间切换层次结构， 使用相应的工具栏按钮（ 参考项目 / 参考项目 ）

  5. 您可以进一步探索层次结构中的每个程序集：

    • 单击基于此 重建基于所选装配的层次结构。

    • 双击任何程序集引用以在“程序集 资源管理器”窗口中 浏览该程序集 。
