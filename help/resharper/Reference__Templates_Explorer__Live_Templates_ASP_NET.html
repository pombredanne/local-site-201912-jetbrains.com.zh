foreach

foreach块

范围 ASP.NET标记

身体

<％foreach（$ COLLECTION $中的$ TYPE $$ VARIABLE $）{％> $ SELECTION $ $ END $ <％}％>

参量

  • COLLECTION -建议可以枚举的可见变量（即在foreach循环中用作集合）（ enumerableVariable ）

  • TYPE -建议在模板中声明的新变量的类型（ proposalVariableType ）

  • VARIABLE -在变量声明中执行时（变量名应保留在其中），建议变量名。 （ proposalVariableName ）

  • SELECTION -调用模板之前用户选择的文本。

  • END -应用模板后的插入符位置。

foreach

对于每个块

范围 ASP.NET标记

身体

<％对于每个$ VAR $作为$ COL $中的$ TYPE $％> $ SELECTION $ <％Next％>

参量

  • COL -建议可以枚举的可见变量（即在foreach循环中用作集合）（ enumerableVariable ）

  • TYPE -分析代码并猜测集合元素的类型。 （ guessElementType ）

  • VAR -在变量声明中执行时（变量名应保留在其中），建议变量名。 （ proposalVariableName ）

  • SELECTION -调用模板之前用户选择的文本。
