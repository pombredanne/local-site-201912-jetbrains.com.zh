使方法非静态重构

此重构不适用于静态属性，因为属性没有任何参数可用于从中创建实例。

ReSharper |重构|使方法变为非静态…
ReSharper_MakeNonStatic

通过这种重构，您可以将静态方法转换为原始方法或其他类型的实例方法。可以从作为参数传递给方法的类型之一中选择新实例方法的目标类型。该方法的所有用法，实现和替代都会自动更新。

此重构只能应用于具有至少一个当前解决方案中定义的类型的参数的静态方法。

反向功能在“生成方法/属性静态”重构中可用。

移动实例方法重构提供了与实例方法类似的功能。

将静态方法转换为实例方法

  1. 将插入号放在编辑器中的声明或静态方法的用法处，或在“ 文件结构”窗口中选择它。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择“ 使方法为非静态”

    • 右键单击并选择“ 重构”。在上下文菜单中将方法设为非静态 。

    • 选择 ReSharper |重构|在主菜单中将方法设为非静态… 。

    将打开“ 制作方法为非静态”对话框。

  3. 从列表中选择一个参数。该方法将转换为参数类型的实例方法。

  4. 要应用重构，请单击下一步 。

  5. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
