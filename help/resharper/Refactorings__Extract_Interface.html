提取接口重构

ReSharper |重构|提取物|提取界面…
ReSharper_ExtractInterface

这种重构有助于基于所选类型创建新接口。ReSharper建议选择要转移到新界面的成员。提取后，原始类型将更新以实现新接口。
如果当前类型已经实现任何接口，则也可以将这些接口提取到新接口中。

从类型中提取接口

  1. 通过以下方式之一选择类型：

    • 在编辑器中，将插入号设置为类型的名称。

    • 在解决方案资源管理器中选择一种类型。

    • 在“ 文件结构”窗口中选择一种类型。

    • 在“类视图”中选择一种类型。

    • 在对象浏览器中选择一种类型。

    • 在类型依赖关系图中选择一种类型。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择提取接口

    • 右键单击并选择“ 重构”。在上下文菜单中提取接口 。

    • 选择 ReSharper |重构|提取物|在主菜单中提取接口… 。

    提取接口对话框将打开。

  3. 在新文件中或与原始类型相同的文件中，指定接口的名称及其放置位置。

  4. 选择要转移到新界面的成员。成员列表可以包含：

    • 当前类型的成员

    • 所选类型当前继承/实现的基本类型的成员

    • 所选类型当前实现的接口

  5. 要快速选择所有公共成员，请点击所有公共 。

  6. 如果选择了引用其他成员的成员，则后者以红色突出显示。单击“ 从属”解决冲突，即选择依赖于当前选定成员的任何成员。

  7. 要应用重构，请单击下一步 。

  8. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
