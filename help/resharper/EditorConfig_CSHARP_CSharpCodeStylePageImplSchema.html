C# - CodeStyle

This page lists custom ReSharper EditorConfig properties that you can use for configuring code syntax style rules. Note that some properties apply to one language while others apply to several languages at once. However, for each multi-language property there is a property that can override it for a specific language, for example, default_private_modifier and. csharp_default_private_modifier.

ReSharper also supports most frequently used .NET-coding-convention EditorConfig properties.
