使用ReSharper的好处之一是，它可以向您显示您自己未使用的代码。ReSharper实际上非常聪明：它不仅会指出未在任何地方使用的代码，而且还会检测到更复杂的情况-例如，声明和初始化字段的情况，但除了初始化实际上什么都没有发生对它。
