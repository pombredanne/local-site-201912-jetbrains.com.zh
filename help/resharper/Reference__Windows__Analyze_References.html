分析参考窗口

ReSharper |查找|优化参考…
Ctrl+Alt+Y
ReSharper_OptimizeReferences

调用“ 优化参考”命令时，“ 分析参考”窗口用于显示项目参考。它显示未使用的程序集引用，具有相应用法的已使用程序集引用和固定引用。

窗口标签

在窗口中打开“优化引用”命令的结果时，它们将显示在新选项卡中，这些选项卡显示在左侧。如果窗口的宽度允许，先前打开的选项卡将移至右侧，否则它们将自行卸载以节省内存-但您始终可以通过单击选项卡右侧出现的箭头并单击所需的选项卡来还原它们：

为了防止标签页卸载，请点击图钉图标标签标题旁边。

工具栏控件

控制

名称

描述

刷新

更新窗口内容以反映代码中或外部所做的任何更改。

全部展开 / 全部折叠

展开/折叠当前选项卡中的所有节点。

上一页 / 下一页
Shift+F8 / F8

导航到上一个/下一个项目，并在源代码中滚动。

显示预览 Ctrl+P

隐藏或显示窗格，该窗格在使用列表指定的位置（在窗口的底部或右侧）显示所选项目的预览。

出口

点击这个按钮以文本格式导出当前在窗口中显示的数据，或使用下拉选择器以XML或HTML格式导出数据。出现的“ 导出数据”对话框将帮助您将数据保存到文件或将其复制到剪贴板。

在同一行上合并发生

选择此选项后，同一行上的出现将显示为单个项目。

删除所有未使用的参考

删除所有未使用的程序集引用。

删除所选参考

删除所有选定的程序集引用。

筛选器用法

使用此列表，您可以选择要显示的特定用法类型，例如，声明，调用，参数，方法返回等。单击“ 显示所有用法”以启用所有用法，或单击“ 清除过滤器”以禁用所有用法，从而将所有用法隐藏在当前选项卡中。

通过...分组

允许按不同类别对窗口中的项目进行分组。
