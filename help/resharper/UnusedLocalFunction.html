Code Inspection: Local function never used

You can configure this inspection:

  • disable it to ignore all related issues
  • suppress it in any place in the code to ignore specific issues

ReSharper suggests removing unused local functions to clear up the code. You can remove such a function with a quick-fix or write the code that uses it.
