生成的代码

ReSharper |选项|代码检查|生成的代码

在ReSharper选项的此页面上，您可以指定包含自动生成的代码的文件和代码区域。 对于这些项目，ReSharper仅运行那些检查代码的编译器错误和警告的代码检查 。默认情况下，此列表中包括Visual Studio用于生成代码的某些文件掩码和区域，例如* .designer.cs ，但是您可以根据需要更改默认设置。

包含生成的代码的项目列表也可以用于禁用代码样式和清除生成的代码。

ReSharper的设置保存在设置层中，使您可以为不同的解决方案使用不同的设置，并与团队共享您的设置。要了解更多信息，请参阅管理和共享ReSharper设置 。
