在中显示完成项目

这些单选按钮有助于在代码完成列表中的 Visual Studio IntelliSense和文本编辑器字体之间进行切换。

在Visual Studio 2015及更早版本中，如果在Visual Studio选项（“ 工具” |“选项” |“环境” |“字体和颜色” ）中更改了IntelliSense或“文本编辑器”字体，则ReSharper仅在重新启动Visual Studio后才能匹配新字体。
在Visual Studio 2017和更高版本中，不需要重新启动。

完成列表中有[值]行

设置代码完成列表中的最大行数。

显示会员签名

默认情况下，方法的所有重载都显示为建议列表中没有签名的单个方法名称。选择后，过载列表（如果有）会显示在弹出窗口中，您可以在其中进行研究。要在弹出菜单中切换签名，您可以按Ctrl+P 。如果要在完成列表中分别查看重载方法的每个签名，请选中此复选框。

显示符号类型

使用此复选框可以定义是否在完成列表中显示字段和属性的类型，并在其旁边返回方法的类型。

显示摘要

此复选框确定参数信息是否显示在代码完成列表中所选项目的旁边。

注解

使用此选择器可以定义是否在完成列表中的项目旁边显示代码注释属性 。 ReSharper将显示符号源代码中的 注释和外部添加到库符号中的注释 。
您可以选择仅显示最受欢迎的NotNull或CanBeNull批注属性或所有属性。
