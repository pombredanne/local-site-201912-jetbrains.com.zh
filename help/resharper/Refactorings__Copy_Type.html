复制类型重构

ReSharper |重构|复制类型…
ReSharper_CopyType

通过此重构，您可以将解决方案中定义的类，接口，结构或枚举从一个名称空间复制到另一个名称空间，或将其克隆到同一名称空间中。对于部分类型，即使它们位于单独的文件中，也会复制所有部分。

复制类型

  1. 通过以下方式之一选择类型：

    • 在编辑器中，将插入号设置为类型的名称。

    • 在解决方案资源管理器中选择一种类型。

    • 在“ 文件结构”窗口中选择一种类型。

    • 在“类视图”中选择一种类型。

    • 在对象浏览器中选择一种类型。

    • 在类型依赖关系图中选择一种类型。

  2. 请执行以下任一操作：

    • 按Ctrl+Shift+R然后选择复制类型

    • 右键单击并选择“ 重构”。上下文菜单中的“ 复印类型 ”。

    • 选择 ReSharper |重构|主菜单中的复印类型… 。

    将打开“ 复印类型”对话框。

  3. 为复制的类型指定名称，并为其添加名称空间。

  4. 要应用重构，请单击下一步 。

  5. 如果未发现冲突，则ReSharper会立即执行重构。否则，它会提示您解决冲突 。
