报价风格

属性名称：

[resharper_]js_quote_style ，[resharper_]quote_style

定义是对字符串文字使用单引号（'）还是双引号（“）

可能的值：

  • singlequoted ：撇号：“

  • single_quoted ：撇号：“

  • doublequoted ：Quote：“

  • double_quoted ：Quote：“

陈述

定义语句如何终止

属性名称：

[resharper_]js_termination_style ，[resharper_]termination_style

定义语句如何终止

可能的值：

  • ensure_semicolon ：确保分号

  • drop_semicolon ：删除分号
