Visual Basic .NET的上下文操作

ReSharper为VB.NET提供了以下上下文操作：

名称

描述

向插值添加新参数

向插值添加新参数，并在必要时将字符串转换为插值

添加新的格式项

将新的格式项添加到格式字符串，并将新的参数添加到Format方法调用
有关更多信息，请参见代码分析和字符串文字的帮助器 。

添加其他访问者

将缺少的访问器添加到属性或将任何缺少的访问器添加到事件。

用“值可以为空”属性注释

具有“值可以为空”属性的注释。可以通过外部注释配置属性。
有关更多信息，请参见使用上下文操作添加注释属性 。

用“值不能为空”属性注释

具有“值不能为空”属性的注释。可以通过外部注释配置属性。
有关更多信息，请参见使用上下文操作添加注释属性 。

更改会员可见度

在访问修饰符关键字上可用，并且允许更改声明的访问权限。

在变量循环上将“ For Each”转换为“ For”

可在允许通过索引直接访问其元素的集合上执行的“ For Each”语句上工作。它将变量语句中的“ For Each”语句转换为“ For”。

将“功能”转换为“子”

将“函数”转换为“子”，删除“ As”子句。

将“ If”语句转换为“ Select Case”语句

适用于“ If”语句，该语句将一个表达式的值与一个或多个常量进行比较。它将“ If”语句转换为“ Select Case”语句。

将迭代器转换为方法

使用列表填充或数组返回将迭代器转换为方法或属性

将'Sub'转换为'Function'

通过添加“ As”子句将“ Sub”转换为“ Function”。

转换为隐式类型的局部变量声明

将局部变量声明转换为隐式类型的声明。

转换为字符串插值

将常规字符串转换为插值。

转换为字符串插值

转换字符串。即使可能更改控制流，也可以格式化对字符串插值的调用。

创建不带参数的重载

在没有给定参数的情况下创建重载。调用当前成员。

实现抽象成员

在接口成员或抽象成员上工作。搜索需要实现此成员的类，并允许在其中一个中实现它。

实施虚拟成员

在虚拟成员上工作。搜索可以实现此成员的类，并允许在其中一个或全部中实现它。

从构造函数参数初始化自动属性

（可选）根据构造函数参数创建和初始化自动属性

从构造函数参数初始化字段

（可选）根据构造函数参数创建和初始化字段

插入推断的类型参数

插入方法的推断类型参数。

从表达式语句引入变量

从表达式语句中引入变量

反转'If'语句

反转“ If”语句的保护条件，并相应地交换分支的代码。

通过“对于每个人”进行迭代

在System的表达式上工作。ICollection类型或其他表达式，其值可以使用“ foreach”语句进行迭代。生成一个“ For Each”语句来迭代表达式。

加入局部变量声明和赋值

处理对没有初始化程序就声明的变量的赋值。将声明移至当前点，并将其与分配合并。

将文件名与类型名匹配

重命名当前文件以匹配其中声明的唯一顶级类型的名称。

合并嵌套的“ If”语句

将两个嵌套的“ If”语句合并到单个“ If”语句中，该语句检查原始条件的相符性。

将类型移到另一个文件以匹配其名称

将当前类型移动到另一个文件，以便其名称与该类型的名称匹配。

删除格式项

删除格式字符串项和正确的格式化方法参数
有关更多信息，请参见代码分析和字符串文字的帮助器 。

用后备字段替换自动属性

用利用后备字段的属性替换自动属性。
有关更多信息，请参见使用自动属性。

用自动属性替换支持字段

将具有支持字段的属性替换为自动属性。
有关更多信息，请参见使用自动属性。

反向分配声明

反转赋值语句的源和目标部分。

在局部变量声明中指定类型

在隐式类型的局部变量声明中明确指定类型。

拆分声明清单

适用于声明多个变量的变量声明语句。它将这样的语句分成多个语句，每个语句声明一个变量。

拆分局部变量声明和赋值

将变量初始值设定项从声明语句移动到单独的赋值语句。

分割字串文字

将字符串文字拆分为两个文字。

将“ If”语句拆分为嵌套语句

适用于条件为两个简单条件的合取（“ AndAlso”）的“ If”语句。它将此类“ If”语句转换为两个嵌套的“ If”语句。

环绕格式

用String包围字符串。Format（）调用。

环绕“有”

用“ With”构造包围块。

使用格式字符串

将一些字符串和其他对象的串联转换为字符串的使用。格式化方法。
有关更多信息，请参见代码分析和字符串文字的帮助器 。

使用字符串插值

将几个字符串和其他对象的串联转换为使用字符串插值。
