dotMemory Unit 3.0帮助

支持的框架

支持的.NET Framework版本

当前，dotMemory Unit支持以下.NET Framework版本（仅Windows！）：

  • .NET Framework 3.5、4.0、4.5、4.6或4.7
  • .NET Core 2.0

支持的单元测试框架

dotMemory Unit可以与市场上几乎所有的.NET单元测试框架一起使用。

提供以下框架的支持：

  • MSTest *
  • NUnit *
  • xUnit.net *
  • MBUnit **
  • csUnit **

*可以使用ReSharper单元测试运行程序（需要ReSharper 9.1（或更高版本）或dotCover 3.1（或更高版本））在Visual Studio中运行测试。另一个选择是使用独立的dotMemory Unit启动器运行测试。

**仅可使用独立的dotMemory Unit启动器运行测试。可以通过Visual Studio运行，但前提是您具有针对特定单元测试框架的ReSharper扩展。

重要！
如果您使用的框架不在此列表中，您仍然可以使用dotMemory Unit。您只需要向dotMemory Unit显示测试方法的开始和结束位置。这是由一个特殊的DotMemoryUnitController类。在使用不受支持的单元测试框架中了解更多信息。
