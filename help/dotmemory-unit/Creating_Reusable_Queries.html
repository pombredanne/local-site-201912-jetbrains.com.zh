创建可重复使用的查询

测试断言经常使用相同的dotMemory Unit查询（例如，返回由某种条件选择的一组对象）。在这种情况下，您只需创建一次可重用的查询，即可在代码中的任何位置使用它。静态QueryBuilder类可让您创建返回对象集并按特定条件过滤流量的查询。然后可以将此类查询作为参数传递给ObjectSet。GetObjects或流量。哪里方法。

例

// create reusable query var preset = QueryBuilder.GetObjects(where => where.Generation.Is(Generation.LOH)) .GetObjects(where => where.Namespace.Like("MyNamespace*")); dotMemory.Check(memory => { // use the query Assert.That(memory.GetObjects(preset).ObjectsCount, Is.EqualTo(0)); });

QueryBuilder静态类

允许创建可重用的查询。

QueryBuilder方法

名称描述
GetObjects(Func query): ObjectSetQuery 创建一个查询，该查询按特定条件获取对象的子集。条件由Query应该由lambda表达式返回。传递给lambda的ObjectProperty对象允许创建查询，以按类型，接口和其他参数选择对象。
返回的实例ObjectSetQuery类型。
TrafficQuery(Func query): TrafficQuery 创建一个查询，该查询按特定条件获取内存流量数据。条件由Query应该由lambda表达式返回。传递给lambda的TrafficProperty实例允许创建查询，以按对象类型，接口和其他参数过滤流量数据。
返回的实例TrafficQuery类型。
