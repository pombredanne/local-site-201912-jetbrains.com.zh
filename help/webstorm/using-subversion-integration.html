Subversion

With the Subversion integration enabled, you can perform basic Subversion operations from inside WebStorm.

WebStorm currently supports integration with Subversion 1.7 and later.

WebStorm comes bundled with the Subversion plugin. Apart from it, you also need to download and install the command line client on your machine.
