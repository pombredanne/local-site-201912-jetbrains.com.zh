WebStorm 2019.3帮助

首次运行WebStorm

首次运行WebStorm或从早期版本升级WebStorm时，需要一些步骤才能完成安装，自定义实例并开始使用IDE。

导入WebStorm设置

在“ 完整安装”对话框中，选择是否要导入IDE设置：

如果这是您的WebStorm的第一个实例，请选择“不导入设置”选项。

您可以稍后使用文件|手动导出和导入设置 。导入设置和文件|从主菜单中导出设置命令。

选择用户界面主题

选择是要使用默认的Darcula主题还是Light主题。

创建启动脚本

对于Linux和macOS用户，WebStorm建议通过指定目录或文件名使用命令行来打开项目和分离文件的功能：

  • 选择“ 从命令行创建用于打开文件和项目的脚本”复选框，然后指定要创建的命令行启动器脚本的名称。

    如有必要，您可以稍后使用“ 工具” |“命令行”创建命令行启动器。在WebStorm主菜单上创建命令行启动器命令。

Windows用户可以完成对话框，因为不需要指定命令行工具。以后，要从命令行打开文件或项目，只需将项目或文件名作为命令行参数启动.exe文件即可。

禁用不必要的插件

WebStorm包括用于与不同版本控制系统和应用程序服务器集成的插件，添加了对各种框架和开发技术的支持，等等。为了提高性能，您可以禁用不需要的插件。如有必要，稍后可以在“ 设置/首选项”对话框中启用它们Ctrl+Alt+S在插件下。有关更多信息，请参阅管理插件 。

您可以单击每个插件组的“ 全部禁用”链接以全部禁用它们，或单击“ 自定义”以禁用该组插件中的单个插件。

下载并安装其他插件

如有必要，请从WebStorm插件存储库下载并安装未与IDE捆绑在一起的其他插件 。

在WebStorm中启动项目

在欢迎屏幕上，您可以执行以下操作：

  • 创建一个新项目
  • 从现有本地资源创建项目
  • 打开一个现有的项目或文件
  • 从版本控制系统中检出现有项目

也可以将现有项目目录或单独的文件拖到“欢迎”屏幕，然后在WebStorm中将其打开。
