WebStorm 2019.3 Help

Uninstall WebStorm

The proper way to remove WebStorm depends on the method you used to install it.

This guide describes ways to uninstall WebStorm on WindowsmacOSLinux. Use the platform switcher at the top to select another operating system.

Toolbox App

If you installed WebStorm by using the Toolbox App:

  • Open the Toolbox App, click the screw nut icon for the necessary instance, and select Uninstall.

Standalone instance

To remove a standalone WebStorm instance, use the standard procedure for uninstalling applications on your operating system and remove the IDE configuration and system directories.

  1. Open the Apps & features section in the system Settings dialog.

  2. Select the WebStorm app and click Uninstall.

  3. Remove the following directories:

    Syntax
    %HOMEPATH%\.<product><version>\config
    Example
    C:\Users\JohnS\.WebStorm2019.3\config
    Syntax
    %HOMEPATH%\.<product><version>\system
    Example
    C:\Users\JohnS\.WebStorm2019.3\system

  1. Delete the application in the Applications directory.

  2. Remove the following directories:

    Syntax
    ~/Library/Preferences/<product><version>
    Example
    ~/Library/Preferences/WebStorm2019.3
    Syntax
    ~/Library/Caches/<product><version>
    Example
    ~/Library/Caches/WebStorm2019.3

  1. Delete the installation directory.

  2. Remove the following directories:

    Syntax
    ~/.<product><version>/config
    Example
    ~/.WebStorm2019.3/config
    Syntax
    ~/.<product><version>/system
    Example
    ~/.WebStorm2019.3/system

Uninstall silently

If you installed WebStorm silently, you can run the uninstaller with the /S switch as an administrator. The uninstaller is located in the installation directory under bin.

Run cmd (Windows Command Prompt) as administrator, change to the WebStorm installation directory, and run the following:

bin\uninstall.exe /S

Snap package

If you installed WebStorm as a snap package, use the following command to remove it:

$ sudo snap remove webstorm
