Navigating to source code from the Debug tool window

From any item in the, you can navigate to the corresponding source code.

  • To navigate from an item in the Variables tab to the source code, select Jump to Source from its context menu or press F4.

  • To navigate from a type to the type declaration, select Jump to Type Source from its context menu or press Shift+F4.
