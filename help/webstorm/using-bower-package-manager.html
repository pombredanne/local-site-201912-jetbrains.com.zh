Bower是客户端库和组件的程序包管理器，这些库和组件包含HTML，CSS，JavaScript，字体，图像文件等。您可以在专用的Bower页面上或从内置终端仿真器中的命令行安装，定位，升级和删除Bower软件包，而无需离开WebStorm。

在开始之前，请确保您的计算机上装有Node.js。

安装和配置Bower

安装Bower

  • 在嵌入式终端 （ Alt+F12 ），输入：

    npm install -g bower

您还可以按照npm和Yarn中的说明将Bower软件包安装在Node.js和NPM页面上 。

创建bower.json

  1. 在嵌入式终端 （ Alt+F12 ），输入：

    cd Bower初始化

  2. 回答问题以指定以下基本设置：

    • 要使用的测试框架。

    • 浏览器将被自动捕获。

    • 定义测试文件位置的模式，这些文件将参与测试或从中排除。

从Bower官方网站了解更多信息。

在WebStorm中配置Bower

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |凉亭 。将打开“ 凉亭”页面。

  2. 指定要使用的Node.js解释器。这可以是本地Node.js解释器，也可以是Windows Subsystem for Linux上的Node.js。

  3. 指定Bower软件包和bower.json配置文件的位置。

管理Bower软件包

Bower仅将软件包安装为项目依赖项或开发依赖项 ，请从Bower官方网站了解更多信息。您可以在Bower页面或“ 终端”工具窗口中管理Bower软件包。

在终端中安装软件包

  • 在嵌入式终端 （ Alt+F12 ），键入以下命令之一：

    • bower install --save 将软件包安装为项目依赖项。

    • bower install --save-dev 将软件包安装为开发依赖项。

在Bower页面上处理软件包

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），请转到语言和框架| JavaScript |凉亭 。

  2. 在打开的Bower页面上， Packages区域显示项目中当前安装的所有Bower软件包。

    • 要安装软件包，请单击 ，然后在打开的“ 可用软件包”对话框中，选择所需的软件包，然后点击安装软件包 。

    • 要升级到软件包的最新版本，请在列表中选择它，然后单击 。

    • 要卸载软件包，请在列表中选择它，然后单击 。
