JavaScript。使用范围

档案|设置|语言和框架| JavaScript |库-管理 Windows和Linux的范围
WebStorm |首选项|语言和框架| JavaScript |库-管理 macOS的范围
Ctrl+Alt+S

当您在JavaScript库设置页面上单击“ 管理范围”时，该对话框打开。

使用此对话框可以定义JavaScript库应用于代码完成，突出显示和导航的范围。范围可能涵盖整个项目，目录，甚至单个文件。这有助于使JavaScript代码完成更加精确，并避免建议列表过长。

图书馆

此列显示文件或目录的库（如果适用）。

如果可以为项目树视图的特定节点指定库，请单击所选文件或目录的“ 库”列，然后从可用库列表中选择所需的库。

如果JavaScript库不适用于特定节点，则以灰色字体显示“ N / A”。
