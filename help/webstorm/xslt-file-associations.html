XSLT文件关联

文件关联用于将XSLT文件与其他XML文件关联。当前，这用于三个目的：

  • 增强了XSLT节点选择中元素和属性名称的完成。该完成将提供在关联文档中找到的所有元素名称和标签名称。

  • XSLT节点选择的增强的错误突出显示。如果XSLT脚本已与一个或多个XML文件相关联，则对不属于相关文件的元素名称和属性名称的任何引用都将用警告消息标记 ：“ 标记名'...'不属于文件 ”。

  • 文件关联也用于创建运行配置 。可以从关联文件列表中方便地选择用于转换的XML输入文件。

XSLT文件关联是按项目定义的，并在“设置”对话框的“ XSLT文件关联”页面上进行管理。

在你开始前， 确保在“ 插件”页面上启用了XPath View + XSLT捆绑插件，有关详细信息，请参阅管理插件 。

配置XSLT文件关联

  1. 打开“ 设置/首选项”对话框（ Ctrl+Alt+S ），然后转到语言和框架| XSLT文件关联 。将打开“ XSLT文件关联”页面。

    该页面包含两个窗格：

    • “ 项目XSLT文件”窗格在项目树视图中显示所有XSLT文件，按模块及其内容根分组。

    • “ 关联的文件”窗格显示了与所选XSLT文件关联的所有XML文件。要将新文件添加到关联，请选择相关的XSLT文件，然后单击在工具栏上，然后选择XML文件以添加到打开的对话框中。

      要取消关联，请选择XML并单击在工具栏上。

  2. 要在“ 项目”工具窗口中显示关联的XML文件， 打开“ 设置/首选项”对话框（ Ctrl+Alt+S ），请转到语言和框架| XSLT ，然后选中在项目视图中显示关联文件复选框。之后，WebStorm将在相应的样式表下方显示关联的XML文件。

通过编辑器管理关联

  • 要创建关联，请在编辑器中打开XSLT文件，然后选择“ 文件关联” |“文件关联”。从其上下文菜单中单击“ 添加...” ，然后选择X（HT）ML文件与打开的XSLT关联。

  • 要取消文件的关联，请在“ 文件关联”组中单击其文件名。显示的文件名带有相对于当前文件的相对路径。如果关联文件是任何模块的一部分，则模块名称将包含在方括号中。

  • 您还可以通过Configure ...操作调用关联配置对话框。这将打开关联配置对话框，并预选择编辑器中当前打开的文件。
