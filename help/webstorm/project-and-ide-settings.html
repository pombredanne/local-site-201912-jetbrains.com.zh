项目和IDE设置

有两种类型的设置定义您的首选环境：

  • 项目设置 ，适用于特定项目。 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），它们标有 。

  • 所有项目都通用的IDE设置 ，指的是与项目无关的方面。

项目设定

项目设置与每个特定项目一起存储为.idea文件夹下的一组.xml文件。如果指定模板项目设置 ，则这些设置将自动用于每个新创建的项目。

与项目相关的设置用图标标记在设置/ 首选项对话框中。

IDE设置

IDE设置存储在WebStorm主目录下的专用目录中。WebStorm目录名称由产品名称和版本组成。

例如：

视窗

  • \。WebStormXX \ config包含用户特定的设置。

  • \。WebStormXX \系统 ，用于存储WebStorm数据缓存。

在WindowsXP中是C：\ Documents and Settings \ \ ;在Windows Vista中，它是C：\ Users \ \

的Linux

  • /。WebStormXX / config包含用户特定的设置。

  • 〜/。WebStormXX /系统 ，用于存储WebStorm数据缓存。

苹果系统

  • 〜/ Library / Application Support / WebStormXX包含带有插件的目录。

  • 〜/ Library / Preferences / WebStormXX包含其余的配置设置。

  • 〜/ Library / Caches / WebStormXX包含数据缓存，日志，本地历史记录等。这些文件的大小可能非常重要。

  • 〜/ Library / Logs / WebStormXX包含日志。

目录位置

config目录包含几个子文件夹，这些子文件夹包含带有您的个人设置的.xml文件。通过将这些文件复制到另一个WebStorm安装的相应文件夹中，可以轻松共享您的首选键盘映射，配色方案等。复制之前，请确保WebStorm没有运行，因为它可以在关闭之前清除新传输的文件。

以下是config文件夹下的某些子文件夹以及其中包含的设置的列表。
