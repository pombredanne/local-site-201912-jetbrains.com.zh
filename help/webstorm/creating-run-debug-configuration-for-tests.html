创建测试的运行/调试配置

您可以运行测试（测试用例，测试套件等），可以使用运行/调试配置来运行测试，其方式类似于运行普通应用程序。WebStorm提供了一个框架，用于为测试目的创建特殊的运行/调试配置，可以将测试指定为目标。

创建测试配置

  1. 通过执行以下操作之一打开“ 运行/调试配置”对话框：

    • 从主菜单中，选择运行|编辑配置 。

    • 按Shift+Alt+F10然后从上下文菜单中选择“ 编辑配置 ”。

  2. 请点击在工具栏上，然后选择所需的配置类型：

    自2018年5月起，不推荐使用Nodeunit。有关详细信息，请参见Nodeunit官方网站。

  3. 在打开的对话框中，指定测试范围，配置参数和要在测试执行之前执行的活动。应用更改并关闭对话框。
