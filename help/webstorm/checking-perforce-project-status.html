WebStorm 2019.3帮助

检查Perforce项目状态

除了指示相对于存储库的当前文件状态外 ，WebStorm与Perforce的集成还为您提供了项目文件状态的累积视图。

查看项目文件的状态

  1. 打开所需的项目。

  2. 从主菜单中，选择VCS |。刷新文件状态 。

  3. 打开 版本控制窗口Alt+9并切换到“ 本地更改”标签。

  4. 每个文件的状态由显示文件路径的颜色指示。

刷新文件状态

WebStorm为Perforce控制下的文件状态提供了两种刷新模式。

  • Standard Refresh仅考虑通过WebStorm与Perforce集成进行的更改。因为不需要连接到服务器，所以可以提高性能。但是，这种方法不会让您知道WebStorm外部所做的更改，例如，直接通过p4v客户端应用程序。

  • Force Refresh考虑从WebStorm和任何其他应用程序对项目所做的所有更改，例如，直接通过p4v client进行的更改 。

  1. 打开 版本控制窗口Alt+9并切换到“ 本地更改”标签。

  2. 请执行以下任一操作：

    • 要运行“ 标准刷新” ，请单击“ 刷新”工具栏按钮或按Ctrl+F5 。

    • 要运行“ 强制刷新” ，请单击“ 强制刷新”工具栏按钮 。
