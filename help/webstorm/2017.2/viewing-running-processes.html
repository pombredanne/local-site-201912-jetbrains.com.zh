Viewing Running Processes

WebStorm makes it possible to view all the running applications. The command Show Running List of the menu Run is only enabled if there are active applications. If no applications are active, the command is greyed out.

To view the list of running applications

  • On the main menu, choose Run | Show Running List.

A popup listing all active applications is displayed in the top-right corner of the editor.

Same is also valid for debugging.
