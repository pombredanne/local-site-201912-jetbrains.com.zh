WebStorm License Activation Dialog

To open this dialog: Register on the Welcome screen or Help | Register.

You can evaluate WebStorm Ultimate for 30 days. After that period, you need to buy WebStorm and activate your license.

The upper part of the dialog reflects your WebStorm usage status (e.g. Free evaluation) and, if appropriate, provides related controls (e.g. Buy WebStorm).

The license activation options are in the lower part of the dialog under Activate new license via.

  • WebStorm usage status-related controls
  • License activation options

WebStorm usage status-related controls

ItemDescription
Buy WebStorm Click this button to go to the JetBrains Web site to study the WebStorm purchasing options and to buy a license.
Evaluate for free for 30 days Click this button to start evaluating WebStorm.

License activation options

ItemDescription
License key Select this option if you have a license key. Specify your user name (or company name) and the key. (The corresponding information is in your license certificate email.)

The recommended way of entering the name and the key is by copying the information from the certificate email and then pasting it into the fields (Ctrl+C/Ctrl+V.

If WebStorm rejects the license information, see The key is not accepted for a possible solution.

License server Select this option if there is the WebStorm License Server on your company's intranet. Specify the server URL in the License server address field. For the address to be entered automatically, click the Discover server button.

To find out if your company is using the License Server and what its URL should be, contact your system administrator.

JetBrains Account Select this option if your WebStorm license is linked with your JetBrains Account. Specify your JetBrains Account access credentials.
