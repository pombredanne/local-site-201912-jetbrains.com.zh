Introduction

Any project contains at least one content root created together with the project.

You can create additional content roots as well as remove the unnecessary ones.

Creating a content root

  1. Open the Settings / Preferences Dialog by pressing Ctrl+Alt+S or by choosing File | Settings for Windows and Linux or WebStorm | Preferences for macOS, and click Directories under Project:<project name>.
  2. On the Directories page, click the Add Content Root button .
  3. In the dialog that opens, locate the desired directory and click OK.

Removing a content root

  • Open the Settings / Preferences Dialog by pressing Ctrl+Alt+S or by choosing File | Settings for Windows and Linux or WebStorm | Preferences for macOS, and click Directories under Project:<project name>.
  • On the Directories page, select the content root to be deleted.
  • Click the Remove button .
  • Confirm deletion.

Exluding files and folders by name patterns

You can make files and folders excluded by specifying the name patterns. Use the Exclude files field to specify the patterns.

The files and folders whose names match at least one of the specified patterns are made excluded. Folders are excluded along with all their contents.
