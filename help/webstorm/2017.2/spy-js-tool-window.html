Spy-js Tool Window

The tool window is available only when the Spy-js plugin is enabled. The plugin is activated by default. If the plugin is disabled, enable it on the Plugins page as described in Enabling and Disabling Plugins.

The tool window opens when you launch a run configuration of the type Spy-js or Spy-js for Node.js. As soon as the Spy-js tool captures an event, it shows the event itself, its details, and stack-traces. By clicking an event, you can open the trace file, which is a prettified source file with the script that the event launched.

The tool window consists of a common toolbar and two tabs with panes:

  • Trace Run Tab
  • Trace Proxy Server Tab

Ctrl+F2		Click this button to terminate the current process externally by means of the standard shutdown script.

Clicking the button once invokes soft kill allowing the application to catch the SIGINT event and perform graceful termination (on Windows, the Ctrl+C event is emulated). After the button is clicked once, it is replaced with indicating that subsequent click will lead to force termination of the application, e.g. on Unix SIGKILL is sent.
