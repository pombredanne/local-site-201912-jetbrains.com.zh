Ctrl+Alt+Shift

This section lists and describes the keyboard shortcuts that include the Ctrl+Shift+Alt keys.

ShortcutFunctionUse this shortcut to...
Ctrl+Shift+Alt+CCopy Relative PathCopy a reference (a relative path) of a symbol to the Clipboard.
Ctrl+Shift+Alt+NGo to SymbolNavigate to a symbol with the specified name.
Ctrl+Shift+Alt+HPop up HectorOpen the Highlighting level pop-up window.
Ctrl+Shift+Alt+VPaste SimplePaste the last entry from the Clipboard as plain text.
Ctrl+Shift+Alt+LShow Reformat File DialogShow reformatting dialog.
Ctrl+Shift+Alt+IRun Inspection by NameExecute an inspection by its name.
Ctrl+Shift+Alt+InsertNew Scratch FileCreate a new scratch file with the selected language.
