Extract Method Dialog

Refactor | Extract | Method
Ctrl+Alt+M

In the JavaScript context, the title of the dialog is Extract Function.

Item

Description

Name

In this field, specify the name of the function or method to be generated on the basis of the selected source code.

Declare functional expression

Select this checkbox to have the new function defined through a function expression, for example, new_method = function().

The checkbox is only available when refactoring is invoked in the JavaScript context.

Parameters

In this area, select parameters to be passed to the new method/function.

If any parameter that is critical for the functionality of the new method is not selected, WebStorm will be unable to proceed with the refactoring.

Move Up/Down

Use these buttons to change the order of the parameters.

Signature preview

In this read-only field, view the declaration of the new method.
