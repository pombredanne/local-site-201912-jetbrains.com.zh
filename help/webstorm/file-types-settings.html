描述

没有版权

如果选择此选项，则版权声明不会在所选类型的文件中更新。

使用默认设置

如果选择此选项，则版权声明将根据新项目的默认设置进行更新。

使用自定义格式选项

如果选择此选项，则所选类型的文件中的版权声明将根据下面定义的自定义设置进行更新。

评论类型

在此区域中，指定注释的类型以包含版权声明。可用的选项有：

  • 使用块注释 ：选择此选项可在块注释中包含版权声明。

  • 每行前缀 ：选中此复选框可使版权声明的每一行都以“ 分隔符”字段中定义的字符开头。默认情况下，使用星号。

  • 使用行注释 ：选择此选项可使版权声明以行注释序列表示。

边框

在此区域中，定义如何将版权声明与其他注释分开。

  • 分隔符之前 ：选中此复选框的版权声明前面带有在分隔符字段中定义的一行字符。

  • 之后的分隔符 ：选中此复选框将显示版权声明，并在“ 分隔符”字段中定义一行字符。

  • 分隔符 ：在此字段中，输入将在分隔符字符串中使用的字符，并用作块注释中的前缀或结尾字符。

  • 长度 ：在此字段中，输入spartor行中的字符数。

  • 框 ：选中此复选框可使版权声明的每一行后跟分隔符字段中定义的字符。

  • 在以下位置添加空白行 ：选中此复选框可在版权声明后插入空白行。

相对位置

在此区域中，指定版权声明相对于其他评论的位置。

  • 在其他评论之前 ：选择此选项时，版权声明会插入在其他评论上方。

  • 在其他评论之后 ：选择此选项后，版权声明将插入在其他评论下方。

预习

使用此区域可查看具有您定义的格式的样本版权声明。

文件中的位置

使用此区域可以指定版权声明在文件中的位置。根据文件类型，可用选项包括：

  • HTML，XML

    • 在根标签之前

    • 在Doctype语句之前

  • 对于PHP ， JavaScript ， CSS ， SCSS和SASS文件类型，无法选择。版权声明始终插入文件的顶部。
