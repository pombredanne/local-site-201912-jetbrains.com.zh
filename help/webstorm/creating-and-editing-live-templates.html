创建实时模板

以下示例过程说明了如何为广告模板创建模板TODO用当前日期，用户名和两个扩展到输入字段的用户定义变量进行注释。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），转到“ 编辑器” |实时模板 。

  2. 选择要在其中创建新实时模板的模板组（例如， other ）。如果未选择模板组，则实时模板将添加到用户组。

  3. 请点击 ，然后选择实时模板 。

  4. 在“ 缩写”字段中，指定将用于扩展模板的字符。例如： todo 。

  5. （可选）在“ 描述”字段中，描述模板以供将来参考。例如： Insert TODO comment with the current date, user, and two input fields for the TODO suggestion and for the name of the project where it is relevant.

  6. 在模板文本字段中，使用变量指定模板的主体。例如： //TODO added on $DATE$ by $USER$: consider $todo$ for project $project$

从代码片段创建新模板

  1. 在编辑器中，选择文本片段以从中创建实时模板。

  2. 选择工具|从主菜单另存为实时模板 。实时模板列表将打开。在此列表中，新创建的模板已添加到用户组。

  3. 指定模板的缩写，可选描述（以标识模板的用途）并修改模板主体。如果模板已定义变量 ，请单击“ 编辑变量”进行配置。

  4. 单击确定以应用更改。

复制现有模板

如果要在多个组中重复使用同一模板，或者要基于另一个模板创建新模板，则可以复制现有模板。

  1. 在编辑器上实时模板页面 设置/首选项对话框（ Ctrl+Alt+S ），选择要复制的模板。

  2. 单击重复 在工具栏上。一个新的模板项将被添加到与原始模板相同的组中，并被选中。

  3. 指定模板的新缩写，可选描述（以标识模板的用途），并在必要时修改模板主体。如果模板已定义变量 ，请单击“ 编辑变量”进行配置。

  4. 单击确定以应用更改。
