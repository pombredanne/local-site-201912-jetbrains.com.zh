JSTestDriver服务器工具窗口

查看|工具窗口| JSTestDriver服务器

-启动本地服务器链接。

在此工具窗口中，启动和停止JSTestDriver服务器以对其运行单元测试，并捕获浏览器以在其中执行单元测试。

先决条件

满足以下先决条件后，工具窗口将变为可用：

  1. JSTestDriver插件已启用。 的默认情况下，插件已激活。如果禁用了插件 ，请按照“ 管理插件”中所述在“ 插件”页面上启用它。

  2. 当前项目至少包含一个测试运行程序配置文件 。

工具提示和快捷方式

描述

运行本地服务器

单击此按钮使WebStorm启动默认的JSTestDriver服务器。

停止本地服务器

单击此按钮可使WebStorm停止当前正在运行的JSTestDriver服务器。

使用URL捕获浏览器

此只读字段显示用于访问JSTestDriver的远程控制台的URL地址。复制URL地址，然后在您选择的浏览器中打开它。

图标表示可用的浏览器。与您刚刚打开的浏览器相对应的图标处于活动状态。单击图标准备执行测试。
