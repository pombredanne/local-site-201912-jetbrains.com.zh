在工具窗口中快速搜索

在工具窗口的速度搜索 ，你可以找到并导航到一个文件中或在Project工具窗口中的文件夹，在结构工具窗口中的一员，在版本控制工具窗口一个变更，在TODO列表中的项目，以及更多。

速度搜索仅在展开的节点上执行，如果节点被折叠，则未检测到其下面的匹配项。

在工具窗口中搜索

  1. 选择所需的工具窗口。

  2. 开始键入项目名称（例如，文件名，类，字段和其他名称）。键入时，“ 搜索”字段将显示在工具窗口工具栏上，显示输入的字符，并且元素选择将移至与指定字符串匹配的第一项。字符串的匹配部分突出显示。

  3. 如果几个相邻的项目与模式匹配，请使用键盘上的向上和向下键在它们之间导航。

  4. 按Enter准备好了。结果，在工具窗口中选择了匹配项。紧迫Escape隐藏搜索字段。
