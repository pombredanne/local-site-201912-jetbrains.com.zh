在Node.js应用程序中进行实时编辑

使用实时编辑功能，您对HTML，CSS或JavaScript代码所做的更改将立即显示在浏览器中，而无需重新加载页面。Live Edit仅在Node.js调试会话期间可用。从HTML，CSS和JavaScript的Live Edit中了解更多信息。

Live Edit适用于包含或生成HTML，CSS或JavaScript的其他文件类型。

默认情况下，仅在HTML和CSS文件中启用Live Edit 。

在你开始前

  • 确保在“ 插件”页面上启用了LiveEdit和NodeJS捆绑的插件 ，有关详细信息，请参阅管理插件 。

启用实时编辑

  1. 打开“ 设置/首选项”对话框（ Ctrl+Alt+S ），然后转到“ 构建，执行，部署|调试器实时编辑 。

  2. 选择更改时更新Node.js应用程序 。 指定在编辑器中更改代码与在浏览器中显示此更改之间的时间延迟：接受默认值300 ms或使用相应字段旁边的旋转框指定自定义值。
