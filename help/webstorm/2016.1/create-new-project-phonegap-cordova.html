Create New Project: PhoneGap/Cordova

File | New Project

Welcome Screen | Create New Project

The dialog box opens when you choose PhoneGap/Cordova App from the list in the left-hand pane of the Create New Project dialog.

ItemDescription
Location In this text box, specify the path to the project folder where the project-related files will be stored.
PhoneGap/Cordova In this text box, specify the location of the executable file phonegap.cmd, or cordova.cmd, or ionic.cmd (see Installing PhoneGap/Cordova/Ionic).
