Ctrl+Alt+S

Use this dialog box to configure content roots of a project.

ItemTooltipDescription
Add Content RootClick this button and select a new content root folder in the dialog that opens.
Remove Content EntryClick this button to have the selected root removed from the list.
TestsClick this button to mark the selected folder as a test root.
Excluded Click this button to mark the selected folder as excluded so WebStorm ignores it during indexing, parsing, and code completion.
Resource Root Click this button to enable WebStorm to complete relative paths to resources under the selected folder.
Include Click this button to have the exclusion mark removed from the selected excluded content root.
