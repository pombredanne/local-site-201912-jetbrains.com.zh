Managing Branches

WebStorm supports processing of multiple Git branches using sophisticated checkout and merge strategies, and provides an interface for configuring operations on branches.

To manage branches, use the Branches node and the nested commands. This node is available on:

  • the main VCS menu
  • context menu of the editor
  • VCS operations pop-up
