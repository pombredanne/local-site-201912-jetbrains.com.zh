Navigating Through the Source Code

WebStorm suggests various ways of navigation between the IDE components and within source code. WebStorm's smart editor makes it possible to navigate across the source code using the code structure rather than plain scrolling. You can find your way in the source code using the method calls, declarations, errors, changes etc.
