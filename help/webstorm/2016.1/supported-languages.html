Supported languages

Development of a modern application involves using multiple languages, that is why WebStorm is an IDE for polyglot programming. With the deep understanding of all the subtleties of the source code structure and syntax, WebStorm extends its support to:

  • JavaScript. Refer to the section JavaScript-Specific Guidelines.
  • CoffeeScript. Refer to the section CoffeeScript Support.
  • TypeScript. Refer to the section TypeScript Support.
  • Dart. Refer to the section Dart Support.
  • HTML/XHTML. Refer to the section Markup Languages and Style Sheets.
  • XML. Refer to the section Markup Languages and Style Sheets
  • XSLT.
  • CSS: Less;Sass; Stylus, and Compass.
    Refer to Markup Languages and Style Sheets.

Coding assistance

Coding assistance in WebStorm includes:

  • Syntax and error highlighting. The color attributes are configurable in the Colors and Fonts | <language> pages of the Settings dialog.
  • File templates for the supported languages that enable creating stub classes, scripts etc.
  • Live templates for creating complicated code constructs.
  • Code completion.
  • Code generation.
  • Code folding, formatting, and highlighting.
  • Intention actions and quick fixes.
  • Ability to view code hierarchy.
  • Quick access to the API documentation.
  • Using macros in the editor.
  • Advanced search and replace facilities.
  • Advanced means of navigation.
  • Refactoring.
  • Import Assistance
  • The embedded local terminal where you can execute commands without leaving WebStorm.

Besides editing assistance, WebStorm features a JavaScript debugger.

Debugging for JavaScript applications is supported only in the Chrome browser.
