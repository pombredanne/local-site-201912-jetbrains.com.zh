Browsing Subversion Repository

Prior to checking files out, you can browse the contents of the Subversion repository. The Subversion Repository browser enables you to add or discard repository locations, view history of files and folders, check out files and folders, navigate to source code, browse changes, create branches or tags etc.

Browsing contents of the repository is always available, even when Subversion is not enabled in project. All you need is a valid user account.

To browse contents of a Subversion repository

  1. On the main menu, choose VCS | Browse VCS Repository | Subversion.
