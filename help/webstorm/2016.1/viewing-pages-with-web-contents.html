Viewing Pages with Web Contents

With WebStorm, you can perform two opposite operations:

  • Preview the output of your Web application in the browser to check whether the pages are rendered correctly. WebStorm can display a page preview in a browser of your choice or you can switch between several browsers. WebStorm currently supports previews in the following browsers:
    • Mozilla Firefox
    • Internet Explorer
    • Safari
    • Chrome
    • Opera
  • View the HTML source code of a Web page in the WebStorm editor.
