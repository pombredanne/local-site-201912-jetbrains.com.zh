Supported Version Control Systems

The list of supported integrations of version control systems is determined by the set of currently enabled plugins.

The basic principles of getting started with the supported version control systems (VCS) in WebStorm are rather similar, though some commands and settings are specific and conform to the version control system conventions. In addition to the common information, you can find VCS-specific procedures in the following sections:
