Updates

File | Settings | Appearance and Behavior | System Settings | Updates for Windows and Linux
WebStorm | Preferences | Appearance and Behavior | System Settings | Updates for OS X
Ctrl+Alt+S

Use this page to:

  • Enable automatic update of WebStorm and specify to which kind of release you want it updated.
  • Obtain information about the current WebStorm version and availability of a newer version.
ItemDescription
Check for updates in channelSelect this check box to enable the automatic update function, and select the desired update channel (major version, public preview or beta, EAP).
  • Channel New Major Version releases: this channel includes all WebStorm releases, for example, WebStorm X.Y.Z
  • Channel Beta Releases or Public Previews: this channel includes release candidates (RC).
  • Channel Early Access Program: this channel gets patch from the previous EAP/release version.

    More details about the Early Access Program, or EAP, are available at http://eap.jetbrains.com/.

Use secure connection
  • If this check box is selected, the secure connection protocol (HTTPS) is used.
  • If this check box is cleared, the HTTP protocol is used. Note that the HTTP protocol may be blocked due to security reasons.
By default, the check box is selected.
Check NowClick this button to check for updates immediately.

For the Windows operating system, you can alternatively choose Help | Check for Update on the main menu.
