Ctrl+Alt+Shift

This section lists and describes the keyboard shortcuts that include the Ctrl+Shift+Alt keys.

ShortcutFunctionUse this shortcut to...
Ctrl+Shift+Alt+C Copy ReferenceCopy a reference of a symbol to the Clipboard.
Ctrl+Shift+Alt+N Go to Symbol Navigate to a symbol with the specified name.
Ctrl+Shift+Alt+H Pop-up HectorOpen the Highlighting level pop-up window.
Ctrl+Shift+Alt+S Project StructureOpen Project Structure dialog box.
Ctrl+Shift+Alt+V Paste SimplePaste the last entry from the Clipboard as plain text.
