Messages Tool Window

View | Tool Windows | Messages
Alt+0

WebStorm parses the output and displays it in a convenient format in the Messages window, letting you navigate to the source of the problem whenever possible.

The Messages tool window is only available if there are actually some messages to show.

On this page:

  • Toolbar Buttons
  • Results Context Menu

Toolbar Buttons

ItemTooltip and shortcutDescription
Rerun Ctrl+F5 Click this button to rerun compilation.

The button is not available in Dart projects, see Managing Dart Packages and Assets for details.

Rerun Pub CommandClick this button to run the last executed Pub command.

The button is available only in Dart projects, see Managing Dart Packages and Assets for details.

Pause outputClick this button to pause the compilation process. This button is enabled when compilation is in progress.

The button is not available in Dart projects, see Managing Dart Packages and Assets for details.

Stop/Stop Pub ProcessClick this button to terminate compilation or Pub process. This button is enabled when compilation or a process is in progress.
Close Ctrl+Shift+F4 Click this button to terminate the process and close the console.
