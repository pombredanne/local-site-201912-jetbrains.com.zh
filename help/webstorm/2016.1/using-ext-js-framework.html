Using Ext JS Framework

WebStorm provides you with the ability to create rich JavaScript interfaces in a browser using the Ext JS client-side framework.

To enable the use of Ext JS in your project, perform these general steps:

  • Download the Ext JS.
  • Configure it as a custom WebStorm JavaScript library.
  • Associate the library with the project.
