Template Data Languages

File | Settings | Languages and Frameworks | Template Data Languages for Windows and Linux
WebStorm | Preferences | Languages and Frameworks | Template Data Languages for OS X

To get coding assistance for template data languages, specify where in your project those languages are used.

You can set a language for a file, directory or an entire project.

ItemDescription
File/DirectoryA tree representation of files and directories in your project.
Template data language To set a language for an item (file, directory or the project), click the Template data language cell to the right of that item and select the language.
