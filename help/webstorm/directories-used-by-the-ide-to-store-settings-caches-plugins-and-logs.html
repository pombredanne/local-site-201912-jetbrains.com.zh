WebStorm 2019.3帮助

WebStorm用于存储设置，缓存，插件和日志的目录

IDE文件的位置取决于操作系统和WebStorm版本。

视窗

默认情况下，所有文件都位于此目录下：

Windows Vista，7、8、10

\ Users \ \。

Windows XP

\ Documents and Settings \ \。

例

  • WebStorm 2019.3

    c：\ Users \ John \。WebStorm2019.3 \

请参阅页面高级配置 。

Linux和其他UNIX系统

在用户主目录中可以找到以点开头的产品目录。模式是：

〜/。

〜是主目录的别名，例如/ home / john 。

苹果系统

  • 组态

    〜/图书馆/首选项/

  • 快取

    〜/库/缓存/

  • 外挂程式

    〜/图书馆/应用程序支持/

  • 日志

    〜/图书馆/日志/
