运行应用程序

WebStorm允许运行整个应用程序以及特定的脚本。

WebStorm使用“ 运行/调试配置”中定义的设置。项目中存在的所有运行配置在“ 选择运行/调试配置”列表中可用。从使用运行/调试配置中了解更多信息 。

如果在“运行/调试”配置中启用了运行前启动工具的选项，则WebStorm会运行这些工具，并在成功后运行该应用程序。否则，程序将立即启动。

如果要查看所有当前正在运行的应用程序的列表，请选择“运行” |“运行”。从主菜单显示运行列表 。有关详细信息，请参阅“ 查看运行过程”部分。

请注意，开始运行会话后， 标记“ 运行工具”窗口的图标，并在“运行/调试配置选择器”中切换到指示运行过程处于活动状态。

使用任何运行

使用Run Anything ，您可以快速启动运行配置，npm和yarn脚本，以及打开最近的项目。

  1. 按Ctrl两次或单击在导航栏上。将打开“ 运行任何东西”弹出窗口。

  2. 开始输入要启动的配置的名称。键入时，WebStorm将显示匹配的配置。从列表中选择所需的一个，然后按Enter 。

    运行任何东西也是运行npm或yarn脚本以及重新打开最近项目的一种快速方法。

    • 要查找npm或yarn脚本，请键入 run ，请参阅运行npm和yarn脚本 。

    • 要打开一个项目，输入open ，请参阅打开，重新打开和关闭项目 。

    • 要查看所有可用命令，请键入? 。
