版权

您的项目可能包含您要受版权保护的文件。您可以使用版权资料来代替在每个文件中分别插入版权声明。

配置文件定义了版权声明文本和将合并该文本的一组文件（ 范围 ）。

同一项目中的文件可能需要几个不同的版权声明。在这种情况下，您可以配置多个配置文件并将它们与不同的范围关联。

版权设置是通过VCS共享的，因此必须使用共享作用域 。本地范围不适用于版权配置文件配置。

配置新的版权配置文件

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择编辑器|版权|版权资料 。

  2. 请点击并命名新的配置文件。

  3. 输入版权声明文本。

    您可以输入纯文本，或配置Velocity模板。对于模板，请使用variables ，然后单击Validate以确保已正确配置它。

将配置文件分配给文件范围

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择编辑器|版权 。

  2. 请点击Alt+Insert ，然后从列表中选择一个现有的共享范围。

    您可以根据需要定义新的范围。单击页面下部的“ 选择范围”以添加范围或修改现有范围 。

  3. 从“ 版权”列表中，选择要与范围链接的配置文件。

  4. 应用更改并关闭对话框。

配置版权文本格式

默认情况下，IDE会将块注释粘贴在其他注释之前，并在每行中添加前缀，并在块之后添加空白行。

您可以在设置中更改默认格式：

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择编辑器|版权|格式化 。

    在此页面上，您可以配置所有文件类型的格式。如果要更改特定文件类型的格式 ，请在“ 格式”节点下选择它。

  2. 配置格式选项。使用预览部分来确保新格式看起来符合预期。

将版权文字插入文件

  • 要将文本插入单个文件，请在编辑器中将其打开，然后按Alt+Insert ，然后从弹出窗口中选择版权 。

  • 要将文本插入到一组文件中，请在“项目”工具窗口中右键单击一个节点，然后选择“ 更新版权” 。系统将提示您选择要在哪个范围内更新通知。

    节点可能包含属于不同范围的文件。在这种情况下，将根据分配的配置文件生成版权声明。

    如果节点包含的文件不属于任何范围，则IDE将为其分配默认配置文件 。

更新版权文字

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择编辑器|版权|版权资料 。

  2. 选择必要的配置文件并编辑版权文本。

  3. 在正则表达式中检测评论中的版权字段中，指定一个正则表达式，该正则表达式将用于查找文件中的现有版权声明。

    IDE将使用此表达式在其他注释中标识版权注释。例如，此类注释通常包含单词Copyright 。

  4. 在“ 如果旧版权匹配的情况下允许替换版权”字段中，指定IDE将用于搜索应在其中更新版权文本的文件的正则表达式。

    例如，如果要更新其版权中包含2000-2006年和2010-2016年的所有文件中的版权文本，请指定20[0-1][0-6] 。

  5. 应用更改并关闭对话框。

  6. 在“项目”工具窗口中，右键单击要为其更新文本的节点，然后选择“ 更新版权” 。

设置默认的版权配置文件

默认配置文件的设置将应用于未明确包含在配置文件涵盖的任何范围中的文件。

  1. 在“ 设置/首选项”对话框中（ Ctrl+Alt+S ），选择编辑器|版权 。

  2. 从默认项目版权列表中，选择要用作默认配置文件的配置文件。
