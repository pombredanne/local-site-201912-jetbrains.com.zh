Perforce

启用Perforce集成后，您可以从WebStorm内部执行基本的Perforce操作。

下列主题中提供的信息假定您熟悉Perforce版本控制系统的基础知识。

在你开始前

  1. 按照Perforce官方网站上的说明安装Perforce客户端。

  2. 使用Perforce软件仓库创建一个帐户。

  3. 如管理插件中所述安装并启用Perforce Integration存储库插件。

从WebStorm中的Perforce开始

  1. 使用您的Perforce客户端创建客户端规范。

  2. 使用Perforce客户端从Perforce软件仓库获取源文件。

  3. 一旦本地工作副本在您的计算机上， 请将您的本地目录与Perforce关联 。

    之后，您将能够打开源文件进行编辑，并使用WebStorm执行与Perforce相关的常规任务。

Perforce支持

  • 启用与WebStorm的Perforce集成后， Perforce项将出现在VCS菜单上，以及在Editor和Project Tool Window的上下文菜单中。

  • Perforce控件下的文件夹中的文件根据其状态突出显示。有关文件状态突出显示约定，请参见文件状态突出显示。

  • 修改结果显示在“ 版本控制”工具窗口中 。

  • 使用Perforce集成时，打开“ 版本控制”工具窗口会很有帮助。“ 控制台”选项卡显示以下数据：

    • 根据您通过WebStorm用户界面指定的设置生成的所有命令。

    • 有关执行生成的Perforce命令的结果的信息消息。

    • 错误消息。
