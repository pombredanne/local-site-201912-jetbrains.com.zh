Command Line Tools Input Pane

Although WebStorm provides this tool window for running commands, it is strongly recommended that you use the built-in local Terminal instead.

Tools | Run Command

N/A

The popup is available only when the Command Line Tool Support plugin is installed and enabled as described in Managing plugins.

In this Input pane, type the desired command in the format <tool alias> <command>.

View the results of command execution in the output tab of the dedicated Command Line Tools Console tool window. The tab is named after the last invoked command.
