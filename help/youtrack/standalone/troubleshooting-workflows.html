对工作流程进行故障排除

如果在编写工作流或将工作流附加到项目时遇到问题，请查看以下条件列表，以确定如何解决此情况。

条件

您无法创建操作规则。YouTrack返回错误Action rule can't have same event name as another action rule 。

原因

操作规则的名称与现有命令匹配。操作规则的名称已分配给运行规则的命令。每个服务器的命令必须唯一。

解

使用其他名称作为您的操作规则。

条件

工作流规则针对缺少的问题字段引发错误，但是工作流列表中未显示require setup标志。

原因

该字段未添加到需求部分。

解

将字段添加到需求部分。

原因

在错误的情况下写了需求部分中对该字段的引用。

解

尽管工作流脚本中的需求管理检查不区分大小写，但是工作流区分大小写访问问题字段。将“自定义字段”页面上的字段名称与脚本“ 需求”部分中的引用进行比较。更新需求部分中的引用以匹配用于自定义字段名称的大小写。

条件

更改已应用于自定义脚本，但行为未更改。

原因

仅在更新引用自定义脚本的其他脚本或在应用程序启动时重新加载它们。

解

在引用自定义脚本的脚本中进行较小的更改（例如，添加新行）并保存。
