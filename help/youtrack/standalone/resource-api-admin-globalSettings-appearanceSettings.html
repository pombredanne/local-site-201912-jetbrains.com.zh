Appearance Settings

This resource lets you work with the Visual settings of your YouTrack service.

Resource

/api/admin/globalSettings/appearanceSettings

Returned entity

AppearanceSettings. For the description of the entity attributes, see Supported Fields section.

AppearanceSettings attributes

Represents the Visual settings of the YouTrack service.

This table describes attributes of the AppearanceSettings entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

timeZone

TimeZoneDescriptor

Default application time zone

dateFieldFormat

DateFormatDescriptor

Default format for dates presentation.

logo

Logo

Application logo. Read-only.
