YouTrack Mobile

YouTrack Mobile可作为iOS和Android的本机应用程序下载。每个视图都经过优化，可在移动设备上使用。通过YouTrack Mobile，您可以使用手机或平板电脑执行以下操作：

  • 使用简单的Wiki标记编辑问题详细信息

  • 动态更新字段值

  • 查看标签，问题链接和附件

  • 添加附件

  • 添加评论

要开始使用YouTrack Mobile：

  1. 下载并安装该应用程序。

  2. 在您的移动设备上打开该应用。

  3. 输入您的YouTrack服务器的URL。

  4. 输入您的用户名和密码，然后点击“ 登录”按钮。

    或者，点击通过浏览器登录 。此选项在您的移动浏览器中打开登录页面。

    • 如果您已经登录到YouTrack，则您将通过身份验证并重定向到YouTrack Mobile。

    • YouTrack Mobile将打开到“ 问题”列表。

如果您为帐户启用了双重身份验证，则无法使用用户名和密码登录到YouTrack Mobile。请使用以下选项之一：

  • 使用通过浏览器登录选项。

  • 生成并输入应用程序密码。有关说明，请参阅生成应用程序密码 。

您可以在YouTrack Mobile中使用以下视图和操作（如iOS所示）。

菜单

在菜单视图中，您可以注销YouTrack Mobile。

您还可以访问该应用程序支持的每个视图：

  • 问题
  • 敏捷板

通知列表仅适用于YouTrack InCloud。

问题

在“ 问题”视图中可以执行以下操作：

  • 查看问题列表。筛选出问题以匹配您在搜索框中输入的搜索条件。

  • 点击菜单访问菜单视图。

  • 点击创建以访问“ 新问题”视图。

  • 点击一个问题以在“ 查看问题”视图中将其打开。

编辑搜寻

在“ 编辑搜索”视图中可以执行以下操作：

  • 点击搜索框以编辑查询。

  • 使用小键盘编辑搜索条件。查询完成选项显示在内容区域中。

  • 向上或向下滑动以滚动列表。

  • 点击以选择搜索参数。

  • 点击搜索按钮以应用过滤器。

  • 点击取消以放弃对查询的更改并返回到“ 问题”视图。

新问题

“ 新问题”视图中提供以下操作：

  • 使用键盘输入摘要和描述。

  • 点击从库中选择以从移动设备附加文件。

  • 点按“ 拍摄照片”以使用相机捕获图像并将其附加到问题上。

  • 点击自定义字段以设置或更改其值。您可以水平滚动以查看和更新可用于所选项目中问题的所有自定义字段。

  • 点击创建以报告问题。

查看问题

“ 查看问题”视图中提供了以下操作：

  • 查看完整的问题摘要，描述，评论和字段值。

  • 点击标题中的控件以执行以下操作：

    • 添加“ 星标”标签（观看问题）。

    • 附加文件。

    • 投票给这个问题。

    • 编辑摘要和描述。

  • 点击返回以返回到问题视图。

  • 点击标题中的更多以查看其他操作。

  • 点击更多以显示“ 创建者”和“ 更新者”字段的详细值。

  • 点击评论图标以打开添加评论视图。

  • 点击自定义字段以设置或更改其值。您可以水平滚动以查看和更新可用于所选项目中问题的所有自定义字段。

添加评论

向问题添加评论时，可以执行以下操作：

  • 使用键盘为当前问题添加评论。

  • 点击发送以添加评论。

更新自定义字段

更新自定义字段的值时，可以执行以下操作：

  • 点击自定义字段以更改其值。对于枚举字段，可能的值显示在内容区域中。

  • 向上或向下滑动以滚动列表。

  • 使用搜索框过滤值列表。

  • 点击以选择一个值。

敏捷板

敏捷板视图使您可以显示和处理敏捷板上的问题。

  • 点击标题中的板名称以在板之间切换。

  • 点击冲刺名称，以在所选板的冲刺之间切换。

敏捷板的移动版本支持两个详细级别。点按屏幕右下角的放大镜图标以放大和缩小。

  • 放大以一次显示一列。向右或向左滑动可在各列之间切换。

  • 缩小以显示所有列。

使用以下控件来自定义视图：

  • 点击列名以显示或隐藏列中的卡片。

  • 点击以展开或折叠泳道。根据板配置，可以通过问题，项目或来自自定义字段的值来标识泳道。

使用以下操作与板上的卡进行交互：

  • 点按卡片以在“ 查看问题”视图中打开问题。

  • 按住可将卡拖动到板上的另一个位置。
