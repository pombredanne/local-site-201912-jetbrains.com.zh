全局设置

/ api / admin / globalSettings

退货实体

GlobalSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的子资源

  • / api / admin / globalSettings： 具有特定GlobalSettings的操作

  • / api / admin / globalSettings / systemSettings： 系统设置

  • / api / admin / globalSettings / notificationSettings： 通知设置

  • / api / admin / globalSettings / restSettings： Rest Cors设置

  • / api / admin / globalSettings / appearanceSettings： 外观设置

  • / api / admin / globalSettings / localeSettings： 语言环境设置

  • / api / admin / globalSettings / license： 许可证

GlobalSettings属性

表示应用程序范围的设置。

下表描述了GlobalSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。
