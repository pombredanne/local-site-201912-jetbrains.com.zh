YouTrack Standalone 2019.2帮助

入门

如果您不熟悉YouTrack，这里有一些提示可帮助您找到解决方法。

主导航

首次登录YouTrack时，您会在页面顶部找到主导航。

这些链接和图标可帮助您访问应用程序中的不同页面。链接和图标的可见性基于用户帐户可用的权限。例如，如果您的帐户没有“ 读取项目”权限，则在主导航中看不到“ 项目”链接。“ 管理”菜单中选项的可见性还基于用户帐户可用的权限。

搜索栏

接下来您会注意到的是搜索栏。

使用此功能可以在YouTrack中查找问题。搜索栏具有两个控件：“ 搜索上下文”列表和“ 搜索框” 。这些控件可帮助您仅找到要查找的问题。

  • 要了解如何使用搜索框，请参阅搜索 。

  • 要了解如何使用搜索上下文，请参阅搜索上下文 。

问题清单

在搜索栏下，您会看到一个问题列表。

符合搜索条件的所有问题都显示在此列表中。输入搜索查询时，列表将更新。

目前，实验列表提供了更新版本的问题列表。此处提供的信息适用于该页面的经典版本。要了解实验版本中可用的更改，请参阅重新设计的问题列表 。

问题列表中有一个工具栏，可用于将特定操作应用于一个或多个问题。例如，您可以在列表中选中多个问题的复选框，然后单击“ 添加评论”按钮以一次为所有选定的问题添加评论。

工具栏上还有一个选项，可让您设置列表中问题的详细程度。要扩展单个问题的详细程度，请单击问题右侧的“ 扩展”箭头。

列表下方有一个选项，可让您控制一页上显示的问题数。如果符合搜索条件的问题数量大于每页显示的问题数量，则列表下方还会显示一组导航控件。

  • 要了解有关处理问题的更多信息，请参阅问题 。

  • 要了解如何自定义问题列表，请参阅自定义问题列表 。

侧边栏

在“ 问题”列表的左侧，有一个侧边栏。

边栏包含可用于应用搜索条件和过滤结果的不同选项。

工具栏在“ 问题”列表的实验版中不可用。要按此处所述使用这些控件，请关闭实验性UI。有关说明，请参阅重新设计的问题列表 。

您可以使用侧边栏为常用搜索条件和与您最相关的标签保存快捷方式。

  • 要了解如何自定义边栏，请参阅自定义边栏 。

  • 要了解有关使用过滤器的更多信息，请参见过滤器 。

单期视图

YouTrack提供了单个问题的视图，该视图显示了与问题相关的所有信息。要在此视图中显示问题：

  • 单击“ 问题”列表中的ID或摘要链接。

  • 选中问题复选框，然后按Enter 。

要了解有关此视图的更多信息，请参见单个问题视图 。
