YouTrack Standalone 2019.2帮助

更改运行Windows服务的用户帐户

默认情况下，YouTrack Windows服务由在安装过程中创建的JetBrainsYouTrack用户帐户运行。

您可以将Windows服务重新配置为使用另一个Windows帐户运行YouTrack服务。

  • 在后续升级期间，将重置运行该服务的自定义Windows帐户！因此，您将不得不再次执行自定义过程。

  • 如果您自定义了运行YouTrack Windows服务的Windows帐户，则在下一次升级之前，必须授予JetBrainsYouTrack用户帐户访问该过程中提到的所有位置的权限。否则，升级将失败。

要使用另一个Windows帐户运行Windows服务，请执行以下操作：

  1. 向目标用户授予以下目录的读/写权限：

    • 安装目录。默认情况下，YouTrack安装在C:\YouTrack 。

    • C:\ProgramData\JetBrains\YouTrack

    如果您从以前的版本升级了YouTrack，请为目标用户授予您为以下位置设置的文件夹的读/写权限：

    • 数据目录位置
    • 备份位置
    • 日志位置
    • 临时位置

    您可以在配置文件中找到这些目录的位置：
    %programdata%JetBrains\YouTrack\conf\internal\bundle.properties

  2. 在Windows服务器上打开管理工具>服务窗口。

  3. 停止JetBrains YouTrack服务。

  4. 打开“ 属性”>“登录”对话框。

  5. 将服务用户帐户更改为目标用户帐户。

  6. 启动YouTrack Web服务。
