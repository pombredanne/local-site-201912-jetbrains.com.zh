YouTrack Standalone 2019.2帮助

VCS集成

通过YouTrack，您可以建立与版本控制系统的连接。YouTrack支持与GitHub，GitLab和Bitbucket的直接集成。您还可以在YouTrack与自托管的GitHub Enterprise，GitLab社区版（CE）和GitLab企业版（EE）服务器之间设置集成。

要设置新的集成，请遵循为版本控制系统提供的说明：

GitHub整合

将YouTrack与GitHub或GitHub Enterprise服务器中的存储库集成。

GitLab集成

将YouTrack与GitLab或GitLab CE / EE服务器中的存储库集成。

位桶集成

将YouTrack与Bitbucket中的存储库集成。

Upsource和TeamCity集成支持与其他版本控制系统的连接。

  • 上游支持与Git，Mercurial，Perforce和Subversion中的存储库的连接。

  • TeamCity支持与ClearCase，CVS，Git，Mercurial，Perforce，SourceGear Vault，StarTeam，Subversion，Team Foundation Server和Visual SourceSafe中的存储库的连接。

当管理员将VCS连接到Upsource或TeamCity并在YouTrack中启用相应的集成时，您可以启用此处描述的相同功能。有关更多信息，请参阅上游集成和TeamCity集成 。

将YouTrack与版本控制系统集成时，将启用以下功能：

  • 将更新提交到代码存储库时，请在注释中引用问题ID。VCS集成在更改详细信息中添加了指向YouTrack问题的直接链接。
    注释会自动添加到活动流中针对YouTrack中问题的VCS更改列表。每个VCS更改都包含指向代码存储库中更改详细信息的直接链接。有关更多信息，请参见《 VCS提交中的链接问题》 。

  • 在注释中输入命令以提交，并在VCS中拉取请求。当您推动提交时，该命令将应用注释中引用的所有问题。您可以使用此功能来更新状态或为一个或多个问题添加评论，而无需打开YouTrack。有关更多信息，请参见《 VCS提交》中的“应用命令” 。

  • （仅适用于GitHub）通过将提交哈希值粘贴到YouTrack问题中的摘要，描述，注释或字符串类型的自定义字段中，向存储库添加链接。

“ 管理> VCS集成”页面列出了所有连接到YouTrack的VCS项目和存储库。

工具栏中的以下操作可用：

行动

描述

新的VCS集成

打开“ 新建VCS集成”对话框。

禁用

断开集成项目和VCS存储库之间的连接。配置不会更改，可以随时启用。

编辑

打开侧边栏中所选项目和资源库的集成设置对话框。

删除

从YouTrack删除集成项目的设置。
