YouTrack Standalone 2019.2帮助

单期视图

单个问题视图显示与问题相关的所有信息。这是此问题视图中支持的功能的概述。

实时更新

在单个问题视图中，更新将实时应用于问题。当其他用户对您当前正在查看的问题进行更改时，将立即显示更新。如果将更新添加到活动流的当前在Web浏览器中不可见的部分，则通知会显示在页面底部。

如果将更改应用于在非活动浏览器选项卡中打开的问题，则会通知您有可用的更新，并提示您刷新页面。

搜索栏

搜索栏显示在单个问题视图中。

这将向您显示搜索上下文和查询，这些内容确定了显示哪些问题并可以通过在此视图中的问题之间导航来显示这些问题。

  • 搜索上下文包括一个可见的输入字段，用于过滤列表中的项目。

  • 查询完成将突出显示与当前输入匹配的搜索属性。

  • 属性值以蓝色文本显示。解析为文本搜索的单词以金色显示。

  • 搜索栏右侧以内联方式显示与当前搜索查询匹配的问题的计数器。

当您从“ 问题”列表访问问题时 ，搜索栏中将提供以下选项：

选项

描述

返回搜寻

向后导航以在“ 问题”列表中显示与当前搜索查询匹配的问题 。

上期

导航到列表中与搜索查询匹配的上一个问题。

下期

导航到列表中与搜索查询匹配的下一个问题。

要了解有关搜索查询的更多信息，请参阅搜索 。

问题工具栏

单个问题视图包括一个工具栏，可用于将特定操作应用于当前问题。

在单个问题视图中可以使用以下控件：

控制

描述

打开命令对话框

打开一个对话框，使您可以将一个或多个命令应用于当前问题。有关受支持命令的列表，请参见《 命令参考》 。

链接问题

打开一个列表，您可以从中选择一个问题链接类型。选择问题链接类型时，将打开一个对话框，用于将当前问题链接到一个或多个目标问题。

添加标签

打开可供您使用的标签列表。从列表中选择一个标签，以将标签添加到当前版本中。

分配

打开可以分配问题的用户列表。从列表中选择一个用户以更新“ 受理人”字段中的值。

展示更多

显示其他操作的列表。此菜单还包含工作流支持的自定义操作。

有关详细信息，请参阅显示更多菜单 。

编辑问题

在编辑模式下打开问题。在这里，您可以更改问题的基本属性，例如Project ， Summary和Description 。您还可以设置或更改补充文本字段的值。

显示更多菜单

在“ 显示更多”菜单中可以执行以下操作：

控制

描述

自定义动作

“ 显示更多”菜单中的第一项是工作流支持的自定义操作。每个问题中可用的一组操作由附加到当前项目的工作流确定。有关更多信息，请参阅操作规则 。

克隆问题

创建当前问题的副本。有关更多信息，请参见克隆问题 。

印刷版

在易于打印的页面上打开问题。在浏览器中使用“ 打印”命令来打印页面。

附加档案

打开一个对话框，可让您从本地目录中查找并附加文件。

私下附加文件

打开“私人附加文件”对话框。在这里，您可以选择将所有文件的可见性限制为特定的组和/或用户。

对包含敏感信息的文件使用此选项。

查找类似的问题

在新的浏览器选项卡中打开与当前问题相似的问题列表。问题列表使用looks like搜索属性，以在描述的问题摘要中查找具有相似词语的问题。有关更多信息，请参见查找类似问题 。

重复上一条命令

在输入字段中使用最新命令打开“ 应用命令”对话框。

无干扰模式

工具栏中的“无干扰切换”模式按钮隐藏了标题中的菜单和搜索栏以及自定义字段的侧边栏。这使您可以专注于问题描述和活动流中的内容。

当您的描述包含大量文本时，此视图模式非常有用，因为内容以页面的整个宽度显示。

即使自定义字段被隐藏，您仍然可以使用键盘打开“ 应用命令”对话框，并使用命令来更新值。

标签

在工具栏下方，您将找到当前附加到该问题的标签列表。

  • 要添加标签，请点击工具栏中的添加标签按钮，或使用add tag 命令。

  • 要删除标签，请点击标签内的删除标签图标，或使用remove tag 命令。

或有关标签的更多信息，请参见标签问题 。

发行细节

单个问题视图显示当前问题的详细信息。

页面正文中显示了以下问题详细信息：

项目

描述

问题编号

显示问题的ID。

由...制作

显示报告问题的用户的全名。还显示了报告问题的日期和时间。单击名称以打开该用户的个人资料页面。

更新者

显示上次更新问题的用户的全名。还显示了最后更新的日期和时间。单击名称以打开该用户的个人资料页面。

可见到

显示问题的当前可见性设置。对于可见性受限的问题，锁定图标显示在此标签的左侧。单击链接以设置或更改问题的可见性。

有关更多信息，请参见问题可见性 。

发行摘要

显示问题的摘要。您可以在编辑模式下更新此信息。

问题说明

显示问题的描述。您可以在编辑模式下更新此信息。

快速复印选项

页面左侧的可单击控件使您可以将问题ID和摘要或说明复制到仪表板。

将指针移到要复制的元素上时，将显示这些控件。

星号标签

问题摘要左侧是一个控件，可用于将“ 星号”标签添加到问题中。此操作还将您添加到观察者列表。

再次单击该图标以停止观看问题。

有关更多信息，请参见观看问题 。

选民

问题摘要的右侧是一个控件，您可以通过该控件对问题进行投票并查看选民列表。

  • 单击“ 投票”图标投票。如果您是本期的记者，则此控件将被禁用。

  • 当一个问题获得一票或多票时，“ 投票”图标左侧会显示一个计数器。单击计数器以显示对该问题进行投票的用户列表。

有关更多信息，请参阅投票问题 。

补充文字栏位

单个问题视图支持一种新型的文本自定义字段。这些字段存储一串字符串，当问题显示时，这些字符串将用Markdown语法解释。

此类型的字段不会与其他自定义字段一起显示在边栏中。相反，它们显示在问题说明的下方。

与问题摘要和说明一样，只有在创建或编辑模式下打开问题时，才能更改基于文本的字段的值。编辑这些字段时，输入预览会以Markdown格式显示。有关格式选项的完整列表，请参见Markdown语法 。

否则，使用补充文本字段就像YouTrack中的任何其他自定义字段一样。

  • 要了解如何将字段添加到项目中，请参阅“ 添加自定义字段” 。

  • 要了解有关将价位存储为的字段的更多信息text ，请参阅简单类型 。

发行链接

此面板显示链接到当前问题的问题列表。仅当当前问题链接到至少一个问题时，它才可见。隐藏该面板后，可以使用工具栏中的“ 链接问题”按钮链接到其他问题，也可以使用语法add 使用命令链接到其他问题。

  • 工具栏显示每种问题链接类型的计数器。每个计数器的工具提示将显示使用此链接类型链接到当前问题的问题总数。工具提示还告诉您有多少个未解决的链接问题。

  • 链接到当前问题的所有问题均显示为列表，并按问题链接类型分组。

  • 未解决的问题在列表中显示在较高位置。

  • 面板的高度受到限制。您可以滚动查看更多链接的问题，单击选项卡以按链接类型显示问题，或单击面板底部的展开控件以显示所有链接的问题。

您的YouTrack个人资料中有一个选项，可让您将字段面板移动到页面的左侧。许多参与功能内部测试的用户发现，启用此选项后，当前布局效果最佳。这会将所有重要信息合并到页面左侧，可以一目了然地对其进行扫描。

移动字段面板：

  1. 单击您的头像，然后选择配置文件链接。

  2. 在“ 常规设置”选项卡上，启用左侧选项的“ 显示字段”窗格 。

    • 您的个人资料设置已更新

    • 在单个问题视图中打开新问题或刷新打开的页面时，字段面板显示在问题详细信息的左侧。

快速搜索叠加

现在，您可以执行搜索请求，而无需从当前问题导航到“ 问题”列表。要打开搜索图，请按Ctrl + K （Windows）或⌘ + K （苹果系统）。

在输入字段中输入搜索查询以显示列表或匹配的问题。

有关搜索术语和参数的完整列表，请参阅《 搜索查询参考》 。
