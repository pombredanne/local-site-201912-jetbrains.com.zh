搜索

该模块包含的功能可让您搜索YouTrack中的问题。

功能


搜索

静态搜索（文件夹，查询，用户）

返回与搜索查询匹配的问题。如果未在查询中明确指定排序顺序，则返回的问题将以随机顺序排序。

参量

名称

类型

描述

夹

 WatchFolder

将项目，标签或保存的搜索设置为搜索上下文。如果未提供此参数的值，则搜索将包括所有问题。这等效于在用户界面中将上下文设置为“所有”的搜索。

询问

 string

YouTrack搜索查询。

用户

 User

执行搜索查询的用户帐户。返回的问题列表排除了指定用户无权查看的问题。如果未提供此参数的值，则代表当前用户执行搜索查询。

例

var search = require（'@ jetbrains / youtrack-scripting-api / search'）; ...var query ='for：me状态：{In Progress} issue id：-'+ issue.id; var inProgress = search.search（issue.project，query，ctx.currentUser）; if（inProgress.isNotEmpty（））{//处理发现的问题集。}
