YouTrack Standalone 2019.2帮助

具有特定NotificationSettings的操作

此资源使您可以使用YouTrack服务的通知设置：阅读和配置Email和Jabber服务器设置以启用向用户的通知。

资源资源
/ api / admin / globalSettings / notificationSettings

退货实体

NotificationSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的NotificationSettings 。

  • POST ： 更新特定的NotificationSettings 。

NotificationSettings属性

代表YouTrack服务的“通知”设置。

下表描述了NotificationSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

emailSettings

 电子邮件设置

发送电子邮件的设置。 Read-only 。

jabberSettings

 Jabber设置

用于发送短消息的设置。 Read-only 。

阅读特定的NotificationSettings

阅读YouTrack服务的“通知”设置。

所需权限

需要权限：低级管理员读取

请求语法

GET / api / admin / globalSettings / notificationSettings？{字段}

请求参数

参数

类型

描述

领域

串

响应中应返回的NotificationSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/notificationSettings?fields=emailSettings(isEnabled,host,port,mailProtocol,anonymous,login,from,replyTo),jabberSettings(isEnabled,host,port,login ，服务名称）

样品响应体

{“ emailSettings”：{“ isEnabled”：true，“ port”：587，“ from”：“ youtrack@service.myjetbrains.com”，“ login”：空，“ mailProtocol”：“ SMTP_TLS”，“匿名”： false，“ replyTo”：null，“主机”：null，“ $ type”：“ EmailSettings”}，“ jabberSettings”：{“ isEnabled”：true，“ port”：5222，“ login”：null，“ serviceName” ：“ myjetbrains.com”，“主机”：“ talk.google.com”，“ $ type”：“ JabberSettings”}，“ $ type”：“ NotificationSettings”}

更新特定的NotificationSettings

更改您的YouTrack服务的通知设置。

所需权限

需要低级管理员写权限。

请求语法

POST / api / admin / globalSettings / notificationSettings？{字段}

请求参数

参数

类型

描述

领域

串

响应中应返回的NotificationSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/notificationSettings?fields=emailSettings(isEnabled,from,replyTo),jabberSettings(isEnabled）
