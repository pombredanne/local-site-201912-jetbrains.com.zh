EmailSettings

Represents email settings for this YouTrack installation.

This table describes attributes of the EmailSettings entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

isEnabled

Boolean

true if email notifications are enabled, otherwise false.

host

String

Mail server that is used for notification. Can be null.

port

Int

Mail server port.

mailProtocol

Enumeration

Mail protocol. Supported values are SMTP, SMTPS and SMTP_TLS

anonymous

Boolean

If true, mail server is accessed without authorization.

login

String

Mail server login. Can be null.

sslKey

StorageEntry

SSL key to be used for accessing the mail server. Can be null.

from

String

From address.

replyTo

String

Reply-to address. Can be null.
