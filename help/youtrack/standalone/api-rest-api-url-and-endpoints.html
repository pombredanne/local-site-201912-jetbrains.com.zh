REST API URL和端点

YouTrack中的REST API URL

您可以通过以下网址使用YouTrack的REST API：

/ api /

是您网络环境中YouTrack 服务的基本URL。例如，您拥有公司的服务器www.example.com和一个YouTrack服务。您可以将YouTrack配置为可通过以下方式访问www.example.com/youtrack或者说youtrack.example.com 。

/api/是YouTrack服务的REST API的上下文路径。

对于上面的示例基本URL，REST API URL分别如下：

  • 对于www.example.com/youtrack ：

    https://www.example.com/youtrack/api/

  • 对于youtrack.example.com ：

    https://youtrack.example.com/api/

最后，假设获取当前用户的个人资料的端点URL为：

  • 对于www.example.com/youtrack ：

    https://www.example.com/youtrack/api/admin/users/me

  • 对于youtrack.example.com ：

    https://youtrack.example.com/api/admin/users/me
