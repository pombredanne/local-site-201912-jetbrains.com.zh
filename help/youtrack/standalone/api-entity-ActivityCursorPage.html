ActivityCursorPage

表示一个页面对象，其中包装了发行活动的列表。与活动列表相比，页面的主要优点是游标。该页面提供了边界标记，这些标记允许从页面结束位置开始对活动进行连续迭代。

下表描述了ActivityCursorPage实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

指示页面上返回活动的顺序是从最新到最旧或相反。如果为“ false”，则首先返回与所选过滤器匹配的最早的活动项目。如果为“ true”，则首先返回最新活动。默认情况下，为“ false”。 Read-only 。
