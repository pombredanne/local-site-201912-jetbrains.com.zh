TextFieldValue

Represents a value of the text field. Returns both source and rendered text.

This table describes attributes of the TextFieldValue entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

text

String

The original source text of the custom field. Read-only. Can be null.

markdownText

String

The rendered markdown text of the custom field. Read-only.
