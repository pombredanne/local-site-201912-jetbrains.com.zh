特定用户的操作

此资源使您可以使用YouTrack中的用户设置。

请注意，该资源仅允许您管理用户帐户的YouTrack配置文件。任何与授权或访问管理有关的操作都必须使用Hub REST API来执行。也就是说，对于诸如创建新用户，更新用户凭证，授予或撤消角色和权限等操作，必须使用各自的Hub REST API端点 。

资源资源
/ api / admin / users / {userID}

退货实体

用户 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定用户 。

用户属性

代表YouTrack中的用户。

下表描述了User实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

读取特定用户

阅读特定用户的YouTrack配置文件的设置。

需要权限： Read User Basic 。

所需权限

读取基本信息不需要任何权限，访问所有数据需要具有“读取用户”权限。

请求语法

GET / api / admin / users / {userID}？{字段}

{用户身份}

YouTrack中用户的数据库ID。

样品1

此示例显示具有受限权限的用户的GET请求。

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/users/1-3?fields=name，登录，被禁止，电子邮件，来宾，在线，标签（id，name，issues（idReadable）），savedQueries（名称，问题（idReadable））

样品响应体

{“ savedQueries”：[]，“ tags”：[]，“ online”：false，“ login”：“ jane.doe”，“ banned”：false，“ name”：“ Jane Doe”，“ $ type” ：“用户”}
