VcsChange

表示附加到问题的VCS更改。

扩展BaseEntity 。

性质

名称

类型

描述

变为已移除

 Boolean 只读。 如果为true，则实体在当前交易中被删除。否则为“ false”。

自2017.4.37915起可用

科

 String 只读。 VCS更改已提交到的分支的名称。

自2018.1.38923起可用

changesProcessors

 Set. 只读。 可以从中检索VCS更改的更改处理器的列表。

自2018.1.38923起可用

被创造

 Number 只读。 应用更改的日期，由VCS返回。

自2018.1.39547起可用

日期

 Number 只读。 应用更改的日期，由VCS返回。使用`VcsChange.created`代替。

自2018.1.38923起可用

拿来

 Number 只读。 从更改处理器检索VCS更改的日期。

自2018.1.39547起可用

ID

 Number 只读。 唯一标识符。除版本外，还由某些CI服务器使用。

自2018.1.38923起可用

是新的

 Boolean 只读。 如果为true，则在当前交易中创建实体。否则为“ false”。

自2018.2.42351起可用

文本

 String 只读。 提交消息中包含的注释文本。

自2018.1.38923起可用

用户

 User 只读。 创作VCS更改的用户。

自2018.1.38923起可用

用户名

 String 只读。 VCS返回的更改作者的名称。

自2018.1.38923起可用

版

 String 只读。 更改的版本号。对于基于Git的VCS，修订哈希。

自2018.1.38923起可用
