SSL Keystores

YouTrack lets you import an SSL a keystore to YouTrack. A keystore is a container for public and private key pairs and the certificates that they are associated with. The keystore identifies YouTrack as a client when it tries to connect to a third party.

You can also use a keystore to cache public keys (in the form of certificates) for a third party service. If you have a keystore that contains keys and certificates for an external service, you can upload them on this page. Otherwise, you can upload the certificate and public key on the SSL Certificates page.

When YouTrack Standalone uses an external Hub service, you manage trusted SSL certificates and SSL keystores in Hub, not YouTrack. The SSL Certificates and SSL Keystores options in the Administration menu are redirected to Hub. For information that is specific to an external Hub service, refer to the Hub documentation.

To access SSL keystores that have been imported to your YouTrack server, select SSL Keystores in the Server Settings area of the Administration menu.

The following controls are available on this page:

Control

Description

Import Key Store

Click to import a certificate from your local directory.

View Keystore Properties

Click the name of a keystore to view its properties. Here, you can also edit the name that is assigned to the keystore.

Delete

Click the delete icon to remove a keystore from YouTrack. Use this option to remove keystores that contain certificates that are expired or no longer in use.

Import an SSL Keystore

Before you start, you need a keystore that contains the public and private key pair and the certificates that they are associated with. This file should either be in PKCS12 or JKS format. The private key must be protected with a password. You can generate the keystore using an application like the Java keytool.

To upload a new SSL Client Key:



Global:

  • Low-level Read Administration
  • Low-level Administration

  1. In the Server Settings section of the Administration menu, select SSL Keystores.

  2. Click the Import Keystore button.

  3. In the Import Keystore dialog, enter a name for the certificate.

  4. Click the Choose file button and select the certificate file from your local directory.

  5. Enter the password for the keystore in the Store Password input field.

  6. Click the Import button.

    • The keystore is added to YouTrack.
