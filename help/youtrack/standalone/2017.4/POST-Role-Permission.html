POST Role Permission

POST /admin/role/{role}/permission/{permission}

Add a permission to a role.

POST /rest/admin/role/{role}/permission/{permission}

Parameters:

Name

Type

Description

role

roleByName

Name of the role to be updated.

permission

string

Permission to add to the specified role. (See YouTrack Permissions)

Sample

Request:

POST /rest/admin/role/Tester/permission/CREATE_ISSUE

Response:

Response Code: 200 Response Status: OK
