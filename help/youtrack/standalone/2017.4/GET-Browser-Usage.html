Get browser usage statistics for YouTrack server

Request:

GET /rest/admin/statistics/browserUsage

Response:

<browserUsages> <browserUsage browser="Chrome 4" usage="3" percent="37.50%"/> <browserUsage browser="Firefox 3" usage="3" percent="37.50%"/> <browserUsage browser="MSIE 8" usage="2" percent="25.00%"/> </browserUsages>
