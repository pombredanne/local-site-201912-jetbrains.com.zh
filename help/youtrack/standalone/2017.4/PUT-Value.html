PUT Value

PUT /admin/customfield/bundle/{bundleName}/{fieldValue}

Add new value to custom field enumeration bundle

PUT /rest/admin/customfield/bundle/{bundleName}/{fieldValue}?{description}&{colorIndex}

Parameters:

Name

Type

Description

bundleName

string

Name of an enumeration bundle to add new value to.

fieldValue

string

New field value to add.

description

string

Description for the new value.

colorIndex

integer

Sequential number of a color scheme (background/text color pair) for the new value. See the list of available Color Indices List.

Sample

Request:

PUT /rest/admin/customfield/bundle/RegionEnum/Rostov?description=Empty description&colorIndex=1

Response:

Empty Response. Response Code: 201 Response Message: Created
