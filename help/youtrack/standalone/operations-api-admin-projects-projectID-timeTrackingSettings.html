具有特定ProjectTimeTrackingSettings的操作

此资源使您可以配置特定项目的时间跟踪设置。

资源资源
/ api / admin / projects / {projectID} / timeTrackingSettings

退货实体

ProjectTimeTrackingSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的ProjectTimeTrackingSettings 。

  • POST ： 更新特定的ProjectTimeTrackingSettings 。

ProjectTimeTrackingSettings属性

表示项目的时间跟踪设置。

下表描述了ProjectTimeTrackingSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

已启用

布尔型

什么时候true ，项目中的时间跟踪已启用。除此以外， false 。

估计

 ProjectCustomField

在项目中用于估算的自定义字段。该字段必须附加到项目中。 Can be null 。

所花费的时间

 ProjectCustomField

在项目中用于估算的自定义字段。该字段必须附加到项目中。 Can be null 。

workItemTypes

 WorkItemTypes数组

项目中使用的工作项类型的集合。

项目

 项目

引用这些设置所属的项目。 Read-only 。 Can be null 。

阅读特定的ProjectTimeTrackingSettings

阅读项目的时间跟踪设置。

所需权限

需要“读取项目基本”权限。

请求语法

GET / api / admin / projects / {projectID} / timeTrackingSettings？{字段}

参数

类型
