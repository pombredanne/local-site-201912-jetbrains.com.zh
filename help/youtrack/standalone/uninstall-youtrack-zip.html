Uninstall YouTrack ZIP

To uninstall YouTrack service that is run from the ZIP distribution, you need to remove the data directories of YouTrack and its binaries.

To uninstall YouTrack ZIP from the system:

  1. Stop the running YouTrack service.

  2. Locate the application data folders. You can find actual location of data, logs, temp folders in the <youtrack_home>/conf/internal/bundle.properties file.

  3. Delete the <youtrack_home> directory. This is the directory where you unpacked the ZIP file during installation.

  4. Before you remove the application data, make sure that YouTrack's database and logs are not reused by another YouTrack instance.

  5. If data and logs folders are located outside of the youtrack_home directory, then delete these folders.
