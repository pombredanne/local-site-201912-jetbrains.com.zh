搜索有关Jabber的问题

使用YouTrack，您不仅可以从Jabber客户端中的YouTrack获得通知，还可以通过直接消息与YouTrack进行交互。当您在Jabber客户端中输入搜索查询时，它将返回与您的查询匹配的问题列表。

可让您以这种方式与YouTrack进行交互的功能称为Jabber Bot 。

先决条件

在使用此功能之前，管理员必须为您的YouTrack实例启用Jabber通知并启用Jabber机器人。有关设置说明，请参阅通过Jabber启用通知 。

将您的Jabber帐户添加到YouTrack

要在Jabber客户端中启用个人通知并使用Jabber Bot与YouTrack进行交互，您需要将Jabber帐户添加到YouTrack。

要添加您的Jabber帐户：

需要权限：更新自我

  1. 在YouTrack中访问“个人资料”页面（点击您的头像并选择个人资料 ）。

  2. 单击链接访问您的中心帐户 。

  3. 在Jabber的输入字段中，输入您的Jabber帐户的用户名。

  4. 要确认设置，请单击“ 发送测试消息”消息链接。

    • 测试消息将从Jabber Bot帐户发送到Jabber客户端应用程序中的用户帐户。

  5. 单击保存按钮。

    • Jabber帐户将保存在您的用户个人资料中。

    • 您现在可以在Jabber客户端中接收有关要在YouTrack中发布的更新的通知。

    • 您可以使用Jabber Bot在直接消息中输入搜索查询来搜索问题。

如果要管理发送到Jabber帐户的通知，请单击页面“ 连接的配置文件”部分中的YouTrack链接以返回到YouTrack配置文件。有关详细信息，请参见通知 。

搜索问题

要在Jabber中搜索问题，请输入斜杠，然后输入搜索查询。例如，要获取问题列表，请输入

/for: me project: TST #{In Progress} 。

要查看单个问题的描述，请输入斜杠，然后输入问题ID。
