PeriodFieldFormat

表示格式，其中将期间值显示给用户。

下表描述了PeriodFieldFormat实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

串

期间格式的标识符。可以使用以下格式：

  • FULL ：表示时间段，以星期，天，小时和分钟为单位。例如， 1w 1d 1h 5m 。

  • DAYS_HOURS_MINUTES ：表示时间段，以天，小时和分钟为单位。例如， 6d 1h 5m 。

  • HOURS_MINUTES ：表示时间段，以小时和分钟为单位。例如， 46h 5m 。

  • MINUTES ：表示时间段，以分钟为单位。例如， 2765m 。
