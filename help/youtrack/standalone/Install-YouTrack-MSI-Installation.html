YouTrack Standalone 2019.2帮助

MSI安装

您可以在JetBrains网站上下载YouTrack Standalone MSI发行版。

以下软件与YouTrack Standalone MSI发行版捆绑在一起：

  • Jetty Servlet引擎和HTTP服务器

  • Java SE运行时环境8 x64

MSI发行版不包括JRE x86。如果要在x86环境中安装YouTrack，则需要下载并安装与您的操作系统兼容的Java Runtime Environment。

MSI安装程序需要.NET Framework 2.0或更高版本。

要将YouTrack作为Windows服务安装：

  1. 从JetBrains网站下载MSI发行版。

  2. 打开Windows Installer程序包（MSI文件）。

    • 将打开JetBrains YouTrack设置向导 。

  3. 请按照安装向导中的说明进行操作。

    • 安装向导完成后，基于Web的配置向导将在默认浏览器中打开。

      如果主机上未安装Web浏览器，请在另一个工作站上的Web浏览器中打开安装向导提供的URL。

    安装向导可以检测以前安装的YouTrack Standalone中的数据。有关更多信息，请参见找到先前的安装数据 。

  4. 单击完成按钮以关闭安装向导。

  5. 切换到Web浏览器，然后在基于Web的配置向导中继续进行安装。

  6. 在配置向导中 ，单击“ 设置” 。

  7. 在“ 确认设置”页面上，确认或更改基本系统设置。

    设置

    描述

    基本网址

    最终用户用来访问YouTrack Standalone的URL。

    应用监听端口

    安装后，您的YouTrack Standalone服务器侦听的端口。

    语言

    用户界面中显示的默认语言。

  8. 展开“ 高级设置”部分，以确认或更改YouTrack Standalone存储产品数据的位置。

    设置

    描述

    数据目录位置

    YouTrack Standalone存储内容数据的目录。

    备份位置

    YouTrack Standalone存储数据库备份的目录。

    日志位置

    YouTrack Standalone存储日志文件的目录。

    临时位置

    YouTrack Standalone存储特定于产品的临时文件的目录。

    要更改任何这些文件类型的位置，请单击“ 更改”按钮，然后选择一个新位置。

  9. 要与Jetbrains共享使用情况统计信息，请保持启用匿名发送使用情况统计信息选项。此功能可帮助JetBrains跟踪使用情况统计信息，以使YouTrack更好。我们绝不与任何第三方共享收集的数据。

  10. 完成后，单击“ 下一步” 。

  11. 在“ 选择集线器服务”页上，选择要使用内置还是外部集线器服务。

    哪种集线器服务适合您？

    如果您不打算使用Upsource或TeamCity，则可以选择使用内置的Hub服务。如果您已经安装了JetBrains Upsource，并希望将其与YouTrack集成 ，或者甚至考虑使用此选项，请考虑使用外部Hub服务。对于新的YouTrack安装，我们建议您从内置的Hub服务开始。您以后总是可以将数据迁移到外部集线器服务 。

    根据您选择的服务类型配置以下设置：

    服务

    设定值

    内置集线器

    • 输入并确认默认系统管理员帐户的密码admin 。要将其他用户名分配给管理员帐户，请编辑“ 创建管理员登录名”设置。

    • 取消选择“ 以访客身份登录”选项以禁止访客用户帐户。

    内置集线器服务已配置并自动启动。

    外部集线器

    输入外部集线器安装的基本URL并进行验证。

  12. 完成后，单击“ 下一步” 。

  13. 在“ 确认许可证”页面上，验证许可证名称和密钥。展开“ 许可证限制”部分，以验证适用于当前计划的所有限制。

  14. 完成后，点击完成 。

    • 该配置将应用于您的YouTrack Standalone服务器。

    设置完成之前，请勿在浏览器中关闭页面。当YouTrack Standalone服务器准备就绪时，您将以访客身份登录到YouTrack。如果您禁用以访客身份登录，则会打开登录页面。

    而已。您的YouTrack Standalone实例已安装并可以使用。

现在，您的YouTrack服务已启动并正在运行，您可能要设置项目和访问权限，从另一个跟踪器导入问题，或者微调服务器环境。有关更多信息，请检查本页底部“ 另请参阅”部分中的主题。

找到先前的安装数据

在安装过程中，安装向导可以检测到以前安装的YouTrack Standalone的数据。这些文件可能是由另一个用户卸载或安装的安装遗留的。

您可以从以下选项中选择以继续安装：

  • 执行全新安装。

  • 使用配置文件和先前安装中剩余的数据安装YouTrack Standalone。

除非您熟悉以前的安装的详细信息，否则我们建议您执行全新安装。先前安装版本中的文件将被删除。

故障排除

如果遇到问题并且安装失败，请查看是否符合以下条件之一。

描述

安装失败

条件

安装目录不为空。

原因

的%programdata%\JetBrains\YouTrack目录包含来自先前安装的数据。

解析度

  1. 从该目录中删除所有文件和文件夹。在Windows 7和Windows 8环境中， %programdata%环境变量指向C:\ProgramData默认。要在主机上检查此变量的实际值，请打开“ 命令提示符”窗口并输入以下命令：
    echo %programdata%

  2. 为YouTrack Windows Installer软件包启用详细日志记录。在“ 命令提示符”窗口中，输入以下命令：
    msiexec /i youtrack-xxxxx.msi /l*v install.txt

  3. 重复安装过程。

如果继续遇到安装问题：

  1. 在YouTrack项目中创建问题 。

  2. 附加以下文件，以帮助我们调查和解决问题：

    • 压缩内容logs目录并附加ZIP存档。该目录的默认位置是%programdata%\JetBrains\YouTrack\logs 。如果此目录的位置在“配置向导”中已更改，则可以在logs-dir的属性%programdata%\JetBrains\YouTrack\conf\internal\bundle.properties文件。

    • 压缩内容conf目录并附加ZIP存档。该目录的默认位置是%programdata%\JetBrains\YouTrack\conf 。

    • 附上install.txt文件。
