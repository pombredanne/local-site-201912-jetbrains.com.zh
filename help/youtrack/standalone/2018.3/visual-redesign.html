Visual Redesign

YouTrack 2018.3 delivers an update to the user interface with a clean, fresh look. We've updated the visual appearance of several pages throughout the application with redesigned controls and a polished theme.

The following animation shows the transformation of the view for single issues from its previous design, the redesign that was introduced in the 2018.1 release, and the latest update.

The new theme uses components from the latest version of JetBrains' library for web-based products, Ring UI. By using standard components from this library, you enjoy more consistent and predictable behavior when you use YouTrack.

This library is open source and can be used to build pluggable widgets that look right at home in your dashboards and the new project overview pages. For more information, see Widget Development Guide.
