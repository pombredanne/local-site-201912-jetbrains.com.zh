YouTrack Standalone 2018.3 Help

Watch Issues

YouTrack lets you watch the issues you are interested in. You can also set another user as a watcher for an issue. When notifications are enabled for your YouTrack server, you and other users are automatically notified when changes are applied to the issues you are watching.

You and other users can specify which changes you want to receive notifications for on the Notifications tab of the profile page. For more information, see Notifications.



— Read Issue

To watch an issue:

  • From the Issues list, click the star icon to the left of the issue ID.

  • In single issue view, click the star icon to the left of the issue ID.

  • Open the Apply Command dialog and enter the star me command.



— Read Issue

— Update Watchers

To set another user as a watcher:

  1. Open the issue in single issue view.

  2. Click the > icon in the Watchers section of the sidebar.

  3. Enter the login or full name of the user who you want to add as a watcher. YouTrack filters the list of users as you type.

  4. Select a user from the list.

You can also set another user as a watcher for one or more issues from the Issues list. Simply select the issues in the list and apply the command add watcher <user>.
