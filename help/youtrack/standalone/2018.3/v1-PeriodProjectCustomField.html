PeriodProjectCustomField

Represents a custom field in a project that stores a value as a period type. We use org.joda.time.Period as a base class for period values. While you can read the class documentation at http://joda-time.sourceforge.net/apidocs/org/joda/time/Period.html, please note that we support only class members which use the Period class and primitive types like String and int.

Extends SimpleProjectCustomField.

// to convert period to minutes (or other units) use get* methods: var period = issue.fields.Estimation; var minutes = !period ? 0 : (period.getMinutes() + 60 * (period.getHours() + 8 * period.getDays())); // to create Period instance, use toPeriod function from date-time module: issue.fields.Estimation = dateTime.toPeriod(3 * 3600 * 1000); // 3h in ms issue.fields.Estimation = dateTime.toPeriod('3h'); // short form issue.fields.Estimation = dateTime.toPeriod('2w4d3h15m'); // full form
