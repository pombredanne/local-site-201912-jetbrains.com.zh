External Integrations

YouTrack lets you connect with other applications in ways that enhance your productivity and improve your working processes. There are several applications that you can integrate with YouTrack directly from the Integrations section of the Administration menu. For an overview of these standard integrations, see Integrations.

The integrations that are described in this section are either set up outside YouTrack or are not accessed in the Integrations menu.

HipChat Integration

In older versions of the documentation, we provided instructions for integrating YouTrack with HipChat with a workflow. If you've set up this integration using the sample workflow from the YouTrack Custom Workflow Repository, it still works as previously described. We've removed the setup instructions from the current documentation for the following reasons:

  • Atlassian will discontinue support for Hipchat and is providing a migration path to Slack.

  • The workflow in the repository was written in the external YouTrack Workflow Editor, which is also discontinued.

It's still possible to set up new integrations between YouTrack and HipChat (and other web services) by writing workflows in JavaScript. For details, see Using REST API Methods in JavaScript Workflows.
