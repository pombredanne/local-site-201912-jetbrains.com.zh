Uninstall Docker Container

To uninstall YouTrack service that is run as a Docker container, you just need to remove the data directories on a host machine that are mapped to the data directories inside the container.

Read more about YouTrack-specific data directories.

To uninstall YouTrack Docker from the system:

  1. Stop the running YouTrack service.

  2. Delete the docker container: docker rm <container_id>

  3. Delete the YouTrack-specific data directories: data, conf, logs. Make sure that these directories are not reused by a new instance of YouTrack before you delete them.
