YouTrack Standalone 2018.3 Help

Language Support

YouTrack provides localization support for English, German, French, Japanese, Russian, and Spanish.

  • The System Language setting is available on the Global Settings page. This page is accessible under the Server Settings page of the Administration menu. For more information, see Global Settings.

  • Users can select their personal language preference in their user profile. For consistency across the application, the language that is set at the global level is used for search queries, commands, and custom fields. The page headers, menus, and controls are displayed in the preferred locale. For more information, see YouTrack General Profile Settings.

Additional Language Options

YouTrack provides support for custom translations of the user interface. The following options are available:

  • You can translate the interface files into the language you want to use. For detailed instructions, read the Localization Tutorial.

  • You can download a custom translation from the Custom Localization Repository.
