将新状态添加到捆绑

放置/ rest / admin / customfield / stateBundle / {bundleName} / {stateName}？{isResolved}＆{description}＆{colorIndex}

参数：

名称

类型

描述

bundleName

串

应向其添加新状态的状态束的名称。

stateName

串

新状态的名称。

isResolved

布尔值

将新状态标记为“已解决”。

描述

串

状态说明。

colorIndex

整数

新状态的配色方案（背景颜色/文本颜色对）的顺序编号。查看可用的颜色指数列表
