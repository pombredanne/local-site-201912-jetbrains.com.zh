管理REST API

本节中描述的某些API方法已被弃用。要管理用户，组，角色和权限，请使用Hub REST API 。该API不支持基于cookie的授权，因此应改编现有的脚本或其他客户端以进行Hub OAuth 2.0身份验证。

所有现有的YouTrack REST方法也都支持基于令牌的授权，因此，一旦迁移，您将不需要维护这两种方案。我们建议使用永久令牌进行授权。有关更多信息，请参阅登录到YouTrack 。

要使用Administration REST API方法，用于登录YouTrack的用户帐户必须具有“ 低级管理”权限。
