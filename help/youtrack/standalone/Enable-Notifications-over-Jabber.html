通过Jabber启用通知

YouTrack使您可以通过XMPP与Jabber客户端应用程序进行交互。此功能使您可以启用以下功能：

  • 启用Jabber通知后，用户可以直接在Jabber客户端应用程序中接收通知。每个用户都可以管理他们是否通过Jabber或通过其他渠道接收通知。

  • 启用Jabber Bot后 ，用户可以通过Jabber客户端中的直接消息与YouTrack进行交互。

要通过Jabber启用通知：

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 常规设置” 。

  2. 在通知部分中，为Jabber选择启用选项。

    • 显示了Jabber配置的其他输入字段。

  3. 在输入字段中输入以下值：

    领域

    描述

    Jabber服务器

    输入Jabber服务器的地址。

    Jabber服务器端口

    输入用于XMPP通信的端口。默认端口是5222 。

    Jabber服务名称

    如果需要，请输入Jabber服务的名称。否则，将此字段留空。

  4. 选择启用Jabber Bot复选框。

  5. 单击保存按钮。

    • YouTrack确认设置已更新。

    • 通过Jabber启用通知的用户可以在此频道中接收通知。

要启用Jabber Bot：

  1. 如上一过程所示，通过Jabber启用通知。

  2. 在以下输入字段中输入值：

    领域

    描述

    Jabber用户名

    输入您要用于Jabber Bot的帐户的用户名。使用格式username@domain 。

    Jabber密码

    输入您想要用于Jabber Bot的帐户的密码。

  3. 选择启用Jabber Bot复选框。

  4. 单击保存按钮。

    • YouTrack确认设置已更新。

    • 启用了Jabber通知的用户可以通过直接向Jabber Bot发送消息来搜索Jabber上的问题并将其应用于命令。

要测试连接：

  1. 单击发送测试消息消息按钮。

    将打开“ 测试Jabber”对话框。

  2. 在Jabber服务器中输入收件人的Jabber帐户，而不是Jabber Bot的帐户。您可以使用自己的帐户，也可以使用其他已知用户的帐户。

  3. 单击发送按钮。

    • YouTrack将测试消息发送到Jabber帐户。如果安装程序配置正确，则YouTrack确认消息已成功发送。
