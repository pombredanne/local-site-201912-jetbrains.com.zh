将命令应用于问题

POST /问题/ {问题} /执行

将命令应用于问题。

使用此方法，您可以在单个调用中更新多个问题属性。例如，更新自定义字段，添加标签和链接问题。

POST / rest / issue / {issue} / execute？{命令}＆{comment}＆{group}＆{disableNotifications}＆{runAs}

参数：

名称

类型

描述

问题

issueById

应用命令的问题的ID。

命令

串

适用于指定问题的命令。该命令可以包含一串属性及其值。这意味着您可以使用单个命令更改多个属性。例如，以下命令设置自定义字段的值Type=Bug ， Priority=Critical ， Fix version=5.1 ，并添加regression标签：
Type Bug Priority Critical add Fix versions 5.1 tag regression

有关YouTrack中命令的更多详细信息，请参阅基本描述和命令参考 。

评论

串

要添加到问题的评论。

组

串

组的名称。用于设置评论的可见性设置。

disableNotifications

布尔值

如果true ，该命令将应用而不会发出通知。默认值为false 。

运行

串

登录代表执行命令的用户。

要使用runAs参数，您必须在项目中拥有针对目标问题的“低级管理”权限。有关权限的更多信息，请参见权限参考。
