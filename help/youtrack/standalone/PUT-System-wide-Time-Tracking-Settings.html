PUT系统范围的时间跟踪设置

PUT / rest / admin / timetracking

设置系统范围的时间跟踪设置：一周中工作日的列表以及工作日中的小时数。

PUT / rest / admin / timetracking

请求参数

参数

类型

描述

小时

整数

一个工作日的小时数。

天数

整数

一周中的工作日数。从YouTrack 5.1开始，此设置是根据由workWeek参数。就其本身而言，实际上仅出于兼容性原因而忽略此参数，并保留该参数。

工作周

清单

逻辑结构指定工作日列表（ workDay参数）。

工作日

整数

一周中某天的整数序号，将其视为一个工作日： 0 -周日1 -星期一2 -星期二3 -星期三4 -星期四5 -星期五6 -星期六

样品

请求：

PUT https://example.com/youtrack/rest/admin/timetracking连接：keep-alive Cookie：$ Version = 0; JSESSIONID = axezv08lvrvrzpvvih22ricx; $路径= / 8 1 2 3 4 5

响应：

响应代码：200响应状态：确定
