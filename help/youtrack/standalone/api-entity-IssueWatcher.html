IssueWatcher

Represents a user who subscribed for notifications about an issue.

This table describes attributes of the IssueWatcher entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

user

User

The user who is subscribed to the issue notifications. Read-only. Can be null.

issue

Issue

The original issue that the user subscribed to. Read-only. Can be null.

isStarred

Boolean

true, if the user added the "Star" tag to this issue. Otherwise, false.
