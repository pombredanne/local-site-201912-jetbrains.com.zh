配置网络接口绑定以进行ZIP分发

默认情况下，YouTrack中使用的Jetty servlet引擎和HTTP服务器绑定在0.0.0.0网络接口。出于安全原因，在某些环境中可能会限制此操作。

您可以使用configure命令更改listen-address您的YouTrack服务器的属性。代表运行YouTrack服务的同一OS用户执行此命令。此命令创建配置文件和文件夹。YouTrack服务用户必须有权访问此命令创建的文件和文件夹。

要为ZIP分发自定义网络接口绑定，请执行以下操作：

  1. 停止YouTrack服务 。

  2. 打开操作系统支持的命令行界面。

  3. 在命令行中，将目录更改为/bin 。

  4. 输入以下命令：
    youtrack.sh configure --listen-address=

  5. 启动YouTrack服务 。
