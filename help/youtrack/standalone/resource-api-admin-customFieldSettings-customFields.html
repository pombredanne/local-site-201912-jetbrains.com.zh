自定义字段

该资源使您可以使用自定义字段。在旧的REST API中，我们称它们为“原型”。

资源资源
/ api / admin / customFieldSettings / customFields

退货实体

CustomField 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读CustomFields列表 。

  • POST ： 添加一个新的CustomField 。

支持的子资源

  • / api / admin / customFieldSettings / customFields / {fieldID}： 使用特定CustomField的操作

  • / api / admin / customFieldSettings / customFields / {fieldID} / fieldDefaults： 自定义字段默认值

  • / api / admin / customFieldSettings / customFields / {fieldID} / instances： 项目自定义字段

CustomField属性

代表YouTrack中的自定义字段。

下表描述了CustomField实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

名称

串

自定义字段的名称。 Can be null 。

localizedName

串

如果设置，则此值用于UI中的字段表示。 Can be null 。

fieldType

 字段类型

自定义字段的类型。

isAutoAttached

布尔型

如果true ，那么该字段将自动附加到新项目。

isDisplayedInIssueList

布尔型

如果true ，默认情况下该字段在“问题”列表中可见。

序数

整数

字段编号。

别名

串

逗号分隔的别名列表，可在搜索和命令中使用。 Can be null 。

fieldDefaults

 CustomFieldDefaults

自定义字段的与项目相关的默认设置。对于不同类型的自定义字段，可能包含不同的值。 Read-only 。

hasRunningJob

布尔型

指示此字段是否正在运行后台作业。在这种情况下，某些现场操作可能暂时不可用。 Read-only 。

isUpdateable

布尔型

指示当前用户是否有权更新此字段。 Read-only 。

实例

 ProjectCustomFields的数组

与项目相关的设置。可能包含不同类型的自定义字段的不同值

阅读CustomFields列表

获取系统中所有可用的自定义字段。

请求语法

GET / api / admin / customFieldSettings / customFields？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

应在响应中返回的CustomField属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。

添加一个新的CustomField

在系统中创建一个新的自定义字段。

强制属性： name ， fieldType 。

所需权限

需要更新项目或低级管理员写权限。
