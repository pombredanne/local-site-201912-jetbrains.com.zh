LocaleDescriptor

表示在UI中使用的语言区域设置。

下表描述了LocaleDescriptor实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

地区

串

语言环境代码。 Read-only 。 Can be null 。

语言

串

语言的缩写。 Read-only 。 Can be null 。

社区

布尔型

指示社区是否支持此语言环境。如果为false，则语言环境为官方翻译。 Read-only 。

名称

串

区域名称。 Can be null 。
