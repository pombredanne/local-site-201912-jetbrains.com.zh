LocaleDescriptor

Represents a language locale that is used in UI.

This table describes attributes of the LocaleDescriptor entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

locale

String

Locale code. Read-only. Can be null.

language

String

Language abbreviation. Read-only. Can be null.

community

Boolean

Indicates whether this locale is supported by community. If `false`, the locale is official translation. Read-only.

name

String

Name of the locale. Can be null.
