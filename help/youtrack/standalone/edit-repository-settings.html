编辑存储库设置

如果需要，可以在设置后为VCS集成编辑存储库URL。但是，此操作可能会导致意外的后果。

更新存储库URL时，YouTrack问题和VCS提交之间的链接将更新为使用新的存储库URL。如果您尚未将存储库移至新URL中引用的位置，则到提交的现有链接都将断开。强烈建议您仅在将存储库实际迁移到其他服务器或VCS提供程序时才更新此设置。

可以在应用程序的两个不同部分中执行此操作：

  • 系统管理员可以在“ VCS集成”页面上访问和更新任何VCS集成。可从“ 管理”菜单的“ 集成”部分访问此页面。

  • 项目管理员可以在“ 编辑项目”页面的“ VCS”选项卡上更新这些设置。

此处的说明特定于VCS集成页面。项目管理员可以从第二步开始，直接在其项目的VCS选项卡上执行这些操作。这两个位置都显示相同的Update VCS Integration对话框。

更新存储库URL以进行VCS集成

需要权限：更新项目

  1. 打开管理> VCS集成页面。

  2. 从列表中选择要更新的集成。

    • 集成设置显示在侧栏中。

  3. 单击存储库路径旁边的“ 编辑存储库设置” 。

    • 将打开“ 更新VCS集成”对话框。

  4. 在“ 存储库URL”字段中编辑地址。

  5. 如有必要，更新授权访问存储库的个人访问令牌或凭证。

  6. 单击保存按钮。

    • 新的存储库URL存储在所选VCS集成的设置中。

    • 集成项目中YouTrack问题与VCS提交之间的所有链接都指向新存储库中的提交。
