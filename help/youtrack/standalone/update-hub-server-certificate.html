更新集线器服务器的证书

这些说明仅适用于连接到外部集线器服务的YouTrack安装。

为了通过安全连接与外部集线器服务交换信息，YouTrack将集线器服务器的SSL证书副本存储在其内部密钥库中。如果更新或更新了集线器服务器的SSL证书，则YouTrack可能不再信任与集线器服务的连接。

如果YouTrack无法建立与集线器的信任连接，则YouTrack服务器将无法启动。当您尝试运行YouTrack时，遇到服务器SSL证书不可信错误。要解决此问题，您需要将更新的集线器证书导入到YouTrack的内部密钥库中。通过向YouTrack服务器应用命令来执行此操作。

在某些情况下，您需要在YouTrack中更新中心证书。

  • 您的集线器服务器的SSL证书已过期。您需要用更新的证书替换过期的证书。

  • 集线器URL发生更改，因为集线器服务已迁移到另一个服务器位置。

  • 您之前曾通过纯HTTP连接到集线器服务，并且刚刚安装了证书。

最后两种情况还要求您在YouTrack中更新外部集线器服务的URL。有关特定说明，请参阅更新中心服务器的URL 。

此过程要求您拥有PEM格式的中心证书的副本。此处描述的命令仅导入证书。不需要私钥。

要为外部集线器服务器更新SSL证书，请执行以下操作：

  1. 在YouTrack服务器上打开命令行界面，然后输入以下命令：

    keytool -import -alias hub- -keystore /conf/internal/keystore.jks -file /path/to/imported/certificate

    将命令行参数替换为如下所述的值：

    参数

    描述

    alias

    分配给导入证书的任意别名。您可以输入任何字符串，但是当您分配该证书有意义的值时，它可以帮助您和其他管理员识别该证书的来源。

    keystore

    YouTrack服务器的内部密钥库文件的路径。此路径相对于您的YouTrack主目录。

    更换占位符以及服务器上YouTrack主目录的实际路径。

    • 对于MSI安装， 目录是YouTrack的安装位置。

    • 对于ZIP安装， directory是安装过程中解压缩ZIP分发包的位置。

    如果密钥库文件在指定位置尚不存在，则会创建它。

    file

    集线器服务器的SSL证书的路径。

  2. 出现提示时，输入用于保护密钥库文件的密码。密码存储为additional-keystore-password物业>YouTrack home directory文件。

    • 如果密码正确，则会在控制台中打印SSL证书的属性。

  3. 当提示您确认是否信任证书时，输入yes在控制台中。

    • 如果证书已成功导入，则消息Certificate was added to keystore在控制台中打印。

  4. 重新启动YouTrack 。

    • YouTrack使用密钥库文件中更新的SSL证书向集线器服务器进行身份验证。
