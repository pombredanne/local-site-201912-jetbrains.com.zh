自定义字段的明确值

用例：

使用REST API清除问题自定义字段的值。

请注意，此示例对YouTrack 2019.1和更高版本有效。在以前的版本中，尽管原理相同，但是查询参数和实体属性可能不同。此外，2019.1中引入了许多功能，例如按名称命名实体，但以前的版本中不支持。

摘要

要清除问题自定义字段的值：

  1. 确保目标问题自定义字段的值可以为空。

  2. 使用更新问题请求：

    POST / api / issues / {issueID}？[字段]

  3. 在里面fields request参数指定您要获得响应的字段。例如，字段ID和全值信息。这将帮助您确认所做的更改已应用。

  4. 在请求正文中指定id要么name和$type用于必填字段。通过null对于value 。

  1. 可选的 。为确保目标问题字段的值为空，请发送一个GET请求到服务器：

    curl -X GET \'https://example.myjetbrains.com/youtrack/api/issues/2-7/customFields?fields=id,projectCustomField%28canBeEmpty,field%28name%29%29'\ -H'接受： application / json'\ -H'授权：不记名perm：am9obi5kb2U =。UG9zdG1hbiBKb2huIERvZQ ==。jJe0eYhhkV271j1lCpfknNYOEakNk7'\ -H'Content-Type：application / json'\ -H'cache-control：no-cache'
