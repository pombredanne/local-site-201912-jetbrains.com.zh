用户组

此资源使您可以阅读YouTrack中的用户组和特定用户组的列表。

资源资源
/ api / admin / groups

退货实体

用户组 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读用户组列表 。

UserGroup属性

代表一组用户。

下表描述了UserGroup实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

名称

串

组的名称。 Read-only 。 Can be null 。

ringId

串

集线器中组的ID。使用此ID进行集线器中的操作，以及在YouTrack和集线器之间匹配组。 Read-only 。

usersCount

长

组中的用户数。 Read-only 。

图标

串

组图标的URL。 Read-only 。 Can be null 。

allUsersGroup

布尔型

如果该组包含所有用户，则为true，否则为false。 Read-only 。

teamForProject

 项目

将这个小组设置为团队的项目。退货null ，如果没有这样的项目。 Read-only 。 Can be null 。

阅读用户组列表

阅读YouTrack中的用户组列表。

请求语法

GET / api / admin / groups？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

应在响应中返回的UserGroup属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/groups?fields=id,name,usersCount,teamForProject(name,shortName)&$top=5
