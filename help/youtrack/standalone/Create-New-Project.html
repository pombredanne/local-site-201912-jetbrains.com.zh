建立专案

要开始跟踪新产品或服务的问题，请创建一个新项目。

如果您打算从Jira导入问题，则在导入期间会自动创建相应的YouTrack项目。要直接跳转到正确的表单，请单击“ 创建项目”页面上的“ 从Jira导入”链接。

有关更多信息，请参阅从Jira导入 。

创建项目时，将自动执行以下操作：

  • 您被授予项目中的项目管理员角色。

  • 在常规项目设置中将您设置为项目所有者 。

  • 您已添加到项目团队。此分配授予您分配给项目中项目团队成员的任何角色。

  • 您将添加到项目中问题的“ 受让人”字段的值集中。

  • 已保存的名称为未分配的搜索会以您的用户帐户作为所有者自动生成。此保存的搜索使用查询project: #{Unassigned} 。在项目中创建新问题时，您将自动订阅通知。

    您可以随时在YouTrack个人资料中访问和更新此保存的搜索。有关更多信息，请参见标签和保存的搜索 。

要创建一个项目：

需要权限：创建项目，阅读项目基础

  1. 单击主菜单中的“ 项目”链接。

  2. 在“ 项目”页面上，单击“ 创建项目”按钮。

  3. 在“ 创建项目”页面上，输入名称和唯一的项目ID。

  4. 要使用非1的数字开始编号问题，请在“ 开始数字”输入字段中输入一个值。

  5. 输入项目的可选说明。

  6. 在“ 项目类型”部分中，选择一个项目模板。您可以从一组预定义的项目模板或自定义的项目模板中进行选择。

    • 有关预定义项目类型的描述，请参见项目模板 。

    • 有关自定义项目模板的描述，请参见“ 自定义项目模板” 。

  7. 单击创建项目按钮。

    • 创建具有指定设置的新项目。

    • 将打开项目配置文件页面。在这里，您可以将用户添加到项目团队。

      要配置高级项目设置，请单击“ 编辑项目”链接。

      有关更多信息，请参见配置项目 。

    • 在新项目中，您被授予项目管理员角色。

    • 在常规项目设置中将您设置为项目所有者 。

    • 您已添加到项目团队。该分配将授予您项目中的开发人员角色。

    • 您将添加到项目中问题的“ 受让人”字段的值集中。以后添加到项目团队的所有用户或组也将添加到“ 受让人”字段的值集中。

    • 已保存的名称为未分配的搜索会以您的用户帐户作为所有者自动生成。此保存的搜索使用查询project: #{Unassigned} 。在项目中创建新问题时，您将自动订阅通知。

      您可以随时在YouTrack个人资料中访问和更新此保存的搜索。有关更多信息，请参见标签和保存的搜索 。
