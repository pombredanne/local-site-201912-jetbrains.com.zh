使用特定VersionBundle的操作

该资源提供了带有版本捆绑包的操作。

资源资源
/ api / admin / customFieldSettings / bundles / version / {bundleID}

退货实体

VersionBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的VersionBundle 。

  • POST ： 更新特定的VersionBundle 。

  • DELETE ： 删除特定的VersionBundle 。

VersionBundle属性

代表YouTrack中的一组版本。

扩展BaseBundle

下表描述了VersionBundle实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

价值观

 VersionBundleElements的数组

捆绑软件中包含的版本的集合。

isUpdateable

布尔型

如果true ，则当前登录的用户可以更新此捆绑包。除此以外， false 。 Read-only 。

阅读特定版本

获取具有特定ID的版本捆绑包的属性。

所需权限

需要对使用捆绑软件的字段之一进行读取或更新访问。如果未在任何字段中使用捆绑软件，则至少在一个项目中需要“更新项目”权限。
