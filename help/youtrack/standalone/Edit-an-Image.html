编辑影像

图像是解释困难概念和表达想法的好方法。简单的图形或图表可以帮助您识别和解决问题。

YouTrack包含自己的图像编辑器。此功能使您可以向图像添加插图和注释。图像编辑器是用于交流和协作的便捷工具。

在编辑器中打开图像

您可以打开和注释附加到问题或评论的任何图像。要在编辑器中打开图像文件，请单击图像缩略图上的“ 编辑”图标。

图像编辑器工具

“ 图像编辑器”对话框包含一系列可用来修改图像文件的工具。

工具栏中提供以下控件：

控制

描述

撤消

撤消上一个操作。

重做

重做上一个动作。

明确

放弃当前更改。

移动

在可见区域内移动图像。

作物

裁剪图像。拖动以设置作物边界。拖动手柄以调整作物边界。

铅笔

拖动以在图像上画一条线。要画一条直线，请按住Ctrl （ Control （对于Mac用户）拖动。

突出

拖动以选择要在图像上突出显示的区域。拖动手柄以调整突出显示的区域。

模糊

拖动以选择要在图像上模糊的区域。

评论

单击图像以添加一个标注框，您可以在其中输入文本。拖动手柄以调整标注框的大小。

文档名称

编辑图像文件的名称。

可见到

将附件的可见性限制为特定的组。

连接

保存更改并将图像附加到问题上。

该图像编辑器还支持键盘快捷键。有关快捷方式的列表，请参见图像编辑器 。
