TextCustomFieldActivityItem

表示会影响自定义字段的活动text问题类型。

扩展CustomFieldActivityItem

下表描述了TextCustomFieldActivityItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

类型

描述

目标

 问题

作为已执行操作的目标的实体。 Read-only 。

串

添加

串

领域

 FilterField

过滤器字段。它包含有关目标实体中已修改属性的其他信息。例如，有关链接类型或修改后的自定义字段或捆绑软件类型的信息，等等。 Read-only 。 Can be null 。
