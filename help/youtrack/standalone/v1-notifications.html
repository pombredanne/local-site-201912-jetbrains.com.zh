YouTrack Standalone 2019.2帮助

通知

此模块包含用于发送电子邮件的功能。这些消息独立于用于订阅以在YouTrack中发布更新的通知方案。这些电子邮件消息的所有组件均在此模块支持的功能的参数中定义。但是，邮件仍由连接到YouTrack安装的SMTP服务器发送。

功能


发电子邮件

静态sendEmail（message，message.fromName，message.toEmails，message.subject，message.body，issue）

将电子邮件发送到一个或多个电子邮件地址。如果问题所属的项目使用专用项目“发件人”电子邮件，则将从该地址发送电子邮件。否则，将采用默认的服务器“发件人”地址。

参量

名称

类型

描述

信息

 Object

包含发送所需的所有消息组件的对象。

message.fromName

 string

发件人的可见名称。使用时，指定的值将作为电子邮件中的发件人名称显示。如果发件人电子邮件（无论是从项目设置还是从服务器设置中提取）都包含发件人名称，则该值将被覆盖。如果将此参数保留为空，则发件人的演示由保存在项目或服务器设置中的“发件人”地址确定。

message.toEmails

 Array.

收件人电子邮件地址列表。数组中的第一个电子邮件地址被设置为收件人。所有其他设置为CC。

信息主题

 string

电子邮件主题。

邮件正文

 string

电子邮件正文。

问题

 Issue

电子邮件相关的问题。与单个问题相关的所有电子邮件都合并到一个线程中。
