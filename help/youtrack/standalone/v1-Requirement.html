一组需求中的一个元素

性质

名称

类型

描述

ID

 string

可选的问题ID，用于代替问题要求的名称。

向内的

 string

问题链接类型的内部名称（如果未设置，则等于外部名称）。

登录

 string

可选的登录名，代替用户要求的名称。

多

 boolean

可选标志，默认为false。如果为true，则必填字段必须存储多个值（如果适用）。

名称

 string

字段或实体的可选名称。如果未提供，则使用“ 需求”对象中用于此需求的键（别名）。

向外

 string

问题链接类型的外部名称（对于IssueLinkPrototype要求是必需的）。

类型

string ， Object

实体的数据类型。可以是以下自定义字段类型之一： Build .fieldType， OwnedField .fieldType， State .fieldType， EnumField .fieldType， ProjectVersion .fieldType， User .fieldType， UserGroup .fieldType， Field .dateType， Field .floatType， Field .integerType， 字段 .stringType， 字段 .periodType。也可以是以下系统范围的实体之一： User ， UserGroup ， Project ， Issue ， IssueTag ， SavedQuery ， IssueLinkPrototype 。
