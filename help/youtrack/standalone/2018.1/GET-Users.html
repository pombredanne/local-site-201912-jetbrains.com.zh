GET Users

GET /admin/user

Get a list of all available registered users.

GET /rest/admin/user?{group}&{role}&{project}&{permission}&{onlineOnly}&{start}

Parameters:

Name

Type

Description

group

string

filter by group (groupID)

role

string

filter by role

project

string

filter by project (projectID)

permission

string

filter by permission

onlineOnly

boolean

get only users which are currently online

start

Integer

paginator mode (takes 10 records)

Sample

Request:

HTTP GET: /admin/user

Response:

<userRefs> <user login="guest" url="https://example.com/youtrack/charisma/rest/admin/user/guest"/> <user login="newroot" url="https://example.com/youtrack/charisma/rest/admin/user/newroot"/> <user login="root" url="https://example.com/youtrack/charisma/rest/admin/user/root"/> </userRefs>
