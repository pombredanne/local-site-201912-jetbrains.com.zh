New User Quick Start Guide

If you are new to YouTrack, we recommend that you take a look at the following topics.

Topic

Description

Create and Edit Issues

Learn the basic steps for creating and updating issues.

Attach Files

Flesh out issues by attaching images and files.

Watch Issues

Track the progress of particular issues by adding them to your watch list.

Tag Issues

Categorize and track issues more effectively with tags.

Search

Locate the issues that you want to find quickly and efficiently.

Saved Searches

Save a search query and share it with your team so you can all work from the same page.

Commands

Familiarize yourself with commands that let you update one or more issues at once.

Notifications

Manage how and where you receive notifications for which kind of updates.

In addition to the topics in the documentation, we have a few external resources that you might find useful.
