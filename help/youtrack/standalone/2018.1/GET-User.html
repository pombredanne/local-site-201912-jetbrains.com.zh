GET User

GET /admin/user/{login}

Get user by login name.

GET /rest/admin/user/{login}

Request parameters:

Name

Type

Description

login

string

Login name of a user.

Response parameters:

Name

Description

login

User login.

fullName

Full name of the user.

email

User's email.

jabber

User's jabber.

groupsUrl

Url for getting user's groups.

roleUrl

Url for getting user's own roles.

Sample

Request:

HTTP GET: /admin/user/root

Response:

<user login="root" fullName="root" groupsUrl="http://localhost:8080/charisma/rest/admin/user/root/group/" rolesUrl="http://localhost:8080/charisma/rest/admin/user/root/role/" />
