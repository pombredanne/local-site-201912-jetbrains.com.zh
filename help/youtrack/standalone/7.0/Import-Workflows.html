Import and Export Workflows in the Editor

On this page:

  • Import a Workflow into the YouTrack Workflow Editor
  • Export Workflows from the YouTrack Workflow Editor

In the YouTrack Workflow Editor, you can import and edit a workflow that has been exported from another YouTrack instance or downloaded from the custom workflow repository.

You can also export a workflow and later import it into a YouTrack instance that is not connected to your workflow editor.

Import a Workflow into the YouTrack Workflow Editor

If you want to modify a workflow that is not currently available in the YouTrack instance that is connected to the editor, you need to import it into the YouTrack Workflow Editor.

To import a workflow into the YouTrack Workflow Editor:

  1. From the Workflow menu, select Import Workflow from File.
  2. Select the ZIP file of the workflow you would like to import.
  3. Click the Import button.

    The workflow is imported and available for editing.

Export Workflows from the YouTrack Workflow Editor

If you have a workflow that you want to upload to the custom workflow repository, or are having problems with and want to share with support, you can export the workflow directly from the YouTrack Workflow Editor.

To export a workflow from the YouTrack Workflow Editor:

  1. Right-click a workflow in the list and select Export Workflow to File.
  2. From the Workflow menu, select
  3. Select the location to which you want to export the file.
  4. Click the Save button.

    The workflow is saved in the specified location as a ZIP file.

To export all workflows from the YouTrack Workflow Editor:

  1. From the Workflow menu, select Export Workflow to Disk.
  2. Select the location to which you want to export the workflows.
  3. Click the Save button.

    The workflows are saved in the specified location as ZIP files.
