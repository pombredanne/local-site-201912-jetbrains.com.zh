Add New Build to a Bundle

PUT /admin/customfield/buildBundle/{bundleName}/{buildName}

Add new build to bundle

PUT /rest/admin/customfield/buildBundle/{bundleName}/{buildName}?{description}&{colorIndex}&{assembleDate}

Parameters:

Name Type Description
bundleName string Name of a bundle to add a new build.
buildName string Name of a new build.
description string Build's description.
colorIndex Integer Sequential number of a color scheme (background color/text color pair) for the build.
assembleDate instant Assemble date for the new build.
