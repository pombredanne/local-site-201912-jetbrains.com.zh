POST Issue Link Type

POST /admin/issueLinkType/{name}

Update issue link type

POST /rest/admin/issueLinkType/{name}?{newName}&{outwardName}&{inwardName}&{directed}

Parameters:

NameTypeDescription
namestringissue link type name
newNamestringnew name for link type, when provided link type is renamed
outwardNamestringvisible name for outward link
inwardNamestringvisible name for inward link, not used for not-directed links
directedbooleanif links of this type are directed

Sample

Request:

xmlPOST /rest/admin/issueLinkType/Depend?newName=Depends&outwardName=new depends on&inwardName=new is required for&directed=true

Response:

Empty Response. Response Code: 200 Response Message: OK
