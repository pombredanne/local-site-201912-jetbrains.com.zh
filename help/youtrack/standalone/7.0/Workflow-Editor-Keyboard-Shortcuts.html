Workflow Editor Keyboard Shortcuts

The YouTrack Workflow Editor supports the following keyboard shortcuts:

Editing

ShortcutDescription
Ctrl + SpaceCode completion
Ctrl + WSelect successively increasing code blocks
Ctrl + Shift + WDecrease current selection to previous state
Alt + EnterShow intention actions
Ctrl + Alt + TSurround with...
Ctrl + UpSelect current node or extend selection to parent node
Ctrl + X / Shift + DeleteCut current line or selected block to buffer
Ctrl + C / Ctrl + InsertCopy current line or selected block to buffer
Ctrl + V / Shift + InsertPaste from buffer
Ctrl + DDuplicate current line or selected block
Shift + F5Clone root

Usages and Search

ShortcutDescription
Alt + F7Find usages
Ctrl + Alt + Shift + F7Highlight cell dependencies
Ctrl + Shift + F6Highlight instances
Ctrl + Shift + F7Highlight usages
Ctrl + FFind text

Navigation

ShortcutDescription
Ctrl + B / Ctrl + ClickGo to declaration
Ctrl + NGo to root node
Alt + Left / RightGo to previous/next editor tab
EscGo to editor (from tool window)
Shift + EscHide active or last active window
Ctrl + ERecent nodes popup
Ctrl + Alt + Left / RightNavigate back/forward
Ctrl + F4Close active editor tab
Alt + 2Go to inspector

Refactoring

ShortcutDescription
F6Move
Alt + DeleteSafe delete
Shift + F6Rename

General

ShortcutDescription
Alt + 1 through 0Open corresponding tool window
Ctrl + Shift + F12Toggle maximizing editor
