GET User

GET /admin/user/{login}

Get user by login name.

GET /rest/admin/user/{login}

Request parameters:

NameTypeDescription
loginstringLogin name of a user.

Response parameters:

NameDescription
loginUser login.
fullNameFull name of the user.
emailUser's email.
jabberUser's jabber.
lastAccessLast time when user was logged in (the number of milliseconds since January 1, 1970, 00:00:00 GMT represented by this date).
groupsUrlUrl for getting user's groups.
roleUrlUrl for getting user's own roles.

Sample

Request:

HTTP GET: /admin/user/root

Response:

<user login="root" fullName="root" lastAccess="1430787661" groupsUrl="http://localhost:8080/charisma/rest/admin/user/root/group/" rolesUrl="http://localhost:8080/charisma/rest/admin/user/root/role/" />
