PUT /admin/group/{group}

Create a new user group.

PUT /rest/admin/group/{group}?{description}&{autoJoin}

Parameters:

NameTypeDescription
groupstringName of new user group
descriptionstringDescription of new user group
autoJoinBooleanNewly registered users will be automatically added to this group

Sample

Request:

PUT /rest/admin/group/xda-developers?description=PDAs developers group&autoJoin=false

Response:

xml Response Code: 201 Response Status: Created Location: http://.../rest/admin/group/xda-developers
