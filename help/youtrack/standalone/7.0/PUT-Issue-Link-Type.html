PUT /admin/issueLinkType/{name}

Create new issue link type

PUT /rest/admin/issueLinkType/{name}?{outwardName}&{inwardName}&{directed}

Parameters:

Name Type Description
name string issue link type name
outwardName string visible name for outward link
inwardName string visible name for inward link, not used for not-directed links
directed boolean if links of this type are directed

Sample

Request:

xmlPUT http://localhost:8080/rest/admin/issueLinkType/Displace?outwardName=displaces&inwardName=is%20displaced%20by&directed=true

Response:

Empty Response. Response code: 201 Response message: Created
