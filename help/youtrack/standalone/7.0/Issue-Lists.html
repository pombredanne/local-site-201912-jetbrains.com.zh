Working with ReportsProjects

Issue Lists

There are two additional reports that you can generate in YouTrack. There reports are not listed on the Reports page. Instead, you generate these reports directly from the Reports drop-down list on the issue list.

The following options are available:

OptionDescription
Issues in CSVExports the list of issues as a CSV file. The list of issues is filtered to match the current search query and context. You can use this option to create a list of issues for import into another application or web page.
Issues in HTMLGenerates a list of issues in HTML. The list of issues is filtered to match the current search query and context. You can use this report to generate release notes or a list of recent updates.
