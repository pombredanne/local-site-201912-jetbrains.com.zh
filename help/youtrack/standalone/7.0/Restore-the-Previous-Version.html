Restore the Previous Version

If you are unable to resolve problems with the installation, you can revert to the previous version of YouTrack with a backup of your pre-upgrade database.

To restore the previous version of an MSI installation:

  1. Uninstall the new version.
  2. Delete the contents of the %programdata%\JetBrains\YouTrack directory.
  3. Install the previous version (clean installation). For instructions, see Installing YouTrack with MSI Distribution.
  4. In the Configuration Wizard, choose the Upgrade option.
  5. Select the backup of the previous version as the upgrade source.

To restore the previous version of a ZIP installation:

  1. Delete the contents of the <youtrack_home> directory. The <youtrack_home> directory is the location where the ZIP distribution was unpacked during installation.
  2. Install the previous version. For instructions, see Installing YouTrack with ZIP Distribution.
  3. In the Configuration Wizard, choose the Upgrade option.
  4. Select the backup of the previous version as the upgrade source.

To restore the previous version of a JAR installation:

  1. Remove the new version of the JAR distribution from your server.
  2. Upload the previous version of the JAR distribution to your server.
  3. Run the previous version of the JAR distribution. For instructions, see Installing YouTrack with JAR Distribution.
