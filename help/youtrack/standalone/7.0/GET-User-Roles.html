GET User Roles

GET /admin/user/{login}/role

Get all roles assigned for a user.

GET /rest/admin/user/{login}/role

Request parameters:

NameTypeDescription
loginstringLogin name of a user.

response parameters:

NameDescription
nameRole name.
projectsProjects, that role is applied in.

Sample

Request:

GET /rest/admin/user/root/role

Response:

<userRoles> <userRole name="Admin"> <projects/> </userRole> <userRole name="Observer"> <projects> <projectRef id="E" url="http://localhost:8080/charisma/rest/admin/project/E" /> </projects> </userRole> serRoles>
