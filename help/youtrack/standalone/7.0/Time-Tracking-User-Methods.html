Time Tracking User Methods

This section covers Rest API methods related to time tracking functions of YouTrack:

Permissions

Per-project:

  • Update Issue
  • Read Work Item
  • Update Work Item
  • Delete Existing Work Item
  • Edit Existing Work Item
  • Create New Work Item
  • Get Available Work Items of Issue
  • Import Workitems

Required permissions

To use the methods described in this section, a user should have the following permissions:

  • Read Work Item — to get a list of work items for an issue.
  • Update Work Item — to create, edit or delete his own work items.
  • Update Not Own Work Item — to edit or delete work items created by other users. Please note that this one is an administrator's permission.
