Enable Notifications over Jabber

YouTrack lets you interact with Jabber client applications over XMPP. This feature lets you enable the following functionality:

  • When Jabber notifications are enabled, users can receive notifications directly in a Jabber client application. Each user can manage whether they receive notifications over Jabber or through another channel.
  • When the Jabber Bot is enabled, users can interact with YouTrack through direct messages in a Jabber client.

To enable notifications over Jabber:

  1. In the Server Settings section of the Administration menu, select General Settings.
  2. In the Notifications section, select the Enable option for Jabber.

    — Additional input fields for the Jabber configuration are shown.

  3. Enter the following values in the input fields:
    FieldDescription
    Jabber serverEnter the address of the Jabber server.
    Jabber server portEnter the port used for XMPP communication. The default port is 5222.
    Jabber service nameIf required, enter the name of the Jabber service. Otherwise, leave this field empty.
  4. Select the Enable Jabber Bot check box.
  5. Click the Save button.
    • YouTrack confirms that the settings were updated.
    • Users who have enabled notifications over Jabber can receive notifications in this channel.

Permissions

Global

  • Low-level Administrator

To enable the Jabber Bot:

  1. Enable notifications over Jabber as shown in the previous procedure.
  2. Enter values in the following input fields:
    FieldDescription
    Jabber usernameEnter the username of the account you want to use for the Jabber Bot. Use the format username@domain.
    Jabber passwordEnter the password of the account you want to use for the Jabber Bot.
  3. Select the Enable Jabber Bot check box.
  4. Click the Save button.
    • YouTrack confirms that the settings were updated.
    • Users who have enabled Jabber notifications can search for and apply commands to issues over Jabber with a direct message to the Jabber Bot.

To test the connection:

  1. Click the Send test jabber message button.

    The Test Jabber dialog opens.

  2. Enter the Jabber account of a recipient in the Jabber server — not the account for the Jabber Bot. You can use your own account, or the account of another known user.
  3. Click the Send button.
    • YouTrack sends a test message to the Jabber account. If the setup is configured correctly, YouTrack confirms that the message was sent successfully.
