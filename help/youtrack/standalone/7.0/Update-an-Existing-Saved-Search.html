POST /user/search/{savedQuery}

Update user's saved search

POST /rest/user/search/{savedQuery}?{newName}&{projectShortName}&{visibleForGroup}&{updatableByGroup}&{q}

Parameters:

Name Type Description
savedQuery string Name of the saved search to edit.
newName string New name for the specified saved search.
projectShortName string
projectId
visibleForGroup string Name of a user group, to which the saved search will be visible.
updatableByGroup string Name of a user group, to which should be granted permissions to update the saved search.
q string Search query of the saved search.

Request

POST http://localhost:8081/rest/user/search/Query%20name?newName=New+query+name&visibleForGroup=All+users&updatableByGroup=idea-developers&q=%23Bug Cookie: $Version=0; JSESSIONID=1h2t265efsl30fhu4777pu8z6; $Path=/; jetbrains.charisma.main.security.PRINCIPAL=MTVhMjY2OGUxMTUxMTI5MGMzNjhlZWM5NTJiNGJkNmIyNzc3NjdjZmM3YWU4MTVjYzQwYWE1MWZlYTkwNTcxNzphcHBfZXhjZXB0aW9u; $Path=/
