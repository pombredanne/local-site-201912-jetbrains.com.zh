GET UserGroups

DELETE User Group

DELETE /admin/user/{login}/group/{group}

Remove a user account from a group.

DELETE /rest/admin/user/{login}/group/{group}

Parameters:

Name Type Description
login string Login name of a user account to be removed from a group.
group string Group name to be removed

Sample

Request:

DELETE /rest/admin/user/Alex/group/idea-developers

Response:

Response Code: 200 Response Status: OK
