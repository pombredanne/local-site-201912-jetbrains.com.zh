GET Work Types for a Project

GET /rest/admin/project/{projectId}/timetracking/worktype

Get the list of configured work types for a specific project.

GET /rest/admin/project/{projectId}/timetracking/worktype

Sample

Request:

GET http://example/rest/admin/project/TST/timetracking/worktype Connection: keep-alive Cookie: $Version=0; JSESSIONID=axezv08lvrvrzpvvih22ricx; $Path=/

Response:

HTTP/1.1 200 OK Content-Type: application/xml;charset=UTF-8 Content-Length: 296 Server: Jetty(6.1.23) <workItemTypes> <workType> <name>Development</name> <id>201-0</id> <url> http://example.net/rest/admin/timetracking/worktype/201-0 </url> </workType> <workType> <name>Testing</name> <id>201-1</id> <url> http://example.net/rest/admin/timetracking/worktype/201-1 </url> </workType> <workType> <name>Documentation</name> <id>201-2</id> <url> http://example.net/rest/admin/timetracking/worktype/201-2 </url> </workType> </workItemTypes>
