Get Users Bundle

GET /admin/customfield/userBundle/{bundleName}

Get custom field user bundle by its name

GET /rest/admin/customfield/userBundle/{bundleName}

Parameters:

Name Type Description
bundleName string Name of a user bundle to get from YouTrack.

Request

GET http://localhost:8081/rest/admin/customfield/userBundle/My%20Users Cookie: $Version=0; JSESSIONID=19eie47i6xtd6l37im4l7uxwc; $Path=/; jetbrains.charisma.main.security.PRINCIPAL=NDgxMzQ5NGQxMzdlMTYzMWJiYTMwMWQ1YWNhYjZlN2JiN2FhNzRjZTExODVkNDU2NTY1ZWY1MWQ3Mzc2NzdiMjpyb290; $Path=/
