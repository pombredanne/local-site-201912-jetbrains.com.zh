GET Custom Field Type

GET /admin/customfield/type/{typeName}

Get custom field type by its name

GET /rest/admin/customfield/type/{typeName}

Parameters:

Name Type Description
typeName string Name of the field type to get.

Sample

Request:

GET /rest/admin/customfield/type/enum[*]

Response:

xml<customFieldType name="enum[*]" url="http://youtrack.jetbrains.net/rest/admin/customfield/type/enum%5B*%5D"/>
