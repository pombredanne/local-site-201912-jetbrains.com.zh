Export All Issues from a Project

GET /export/{project}/issues

Get all issues with comments from a project.

GET /rest/export/{project}/issues?{max}

Parameters:

Name Type Description
project projectById Project ID for a project to import issues from.
max Integer Maximum number of issues to be exported. If omitted, 10 issues are exported, by default.

Sample

Request:

GET /rest/export/TP/issues
