Invite New Users to Register

When you invite users to register an account in YouTrack, you have the option to create several accounts at once. The users enter their own display name and password.

Because the accounts are available in the system, you can configure access permissions as soon as the invitations are sent.

To invite one or more users to register for an account in YouTrack:

  1. In the Access Management section of the Administration menu, select Users.
  2. Click the Create User button.
  3. Click the Invite Users button.
  4. Enter a list of space-separated email addresses for the users you wan to invite to register accounts in YouTrack.
  5. Click the Invite button.
    • An email message is sent to users with the specified email addresses that invites them to register their accounts.
    • The profile page of the new user is displayed. If you invited several users at once, the profile page of the first user from the list is displayed.
