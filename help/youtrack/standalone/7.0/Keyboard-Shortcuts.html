Keyboard Shortcuts

YouTrack is a keyboard-centric application and provides enough keyboard support to let you forget about using the mouse in most cases. For almost any operation you need to do, whether it's creating, editing or navigating between issues, commenting, changing attributes — YouTrack provides a shortcut.

These shortcuts are mapped to keys on a keyboard with a functional layout for English. Keys that differ for US and UK variants are not mapped to any shortcuts. If your keyboard has a visual layout for a different language, you can switch the language preference for your operating system to English and use these shortcuts with the default mapping. Alternatively, an administrator can customize the keyboard shortcuts for your YouTrack server. For more information, see Customize Keyboard Shortcuts.

The shortcuts on this page are grouped by the context where they can be applied when you work with issues in YouTrack.

  • Issues List
  • Full Page View
  • Apply Command Dialog
  • Search Box
  • Agile Boards
  • Dashboard
  • Image Editor

Image Editor

Use these shortcuts to edit an image in the image editor.

Windows ShortcutMac ShortcutDescription
Alt + COption + CSelect the crop tool. Drag to set the crop boundary. Drag the handles to adjust the crop boundary.
Alt + BOption + BSelect the brush tool. Drag to draw a line on the image. To draw a straight line, hold down the Ctrl (Control for Mac) key while dragging.
Alt + MOption + MSelect the move tool. Drag to move the image.
Alt + HOption + HSelect the highlight tool. Drag to select an area to highlight on the image. Drag the handles to adjust the highlighted area.
Alt + TOption + TSelect the text tool. Click the image to add a callout box where you can enter text. Drag the handles to adjust the size of the callout box.
