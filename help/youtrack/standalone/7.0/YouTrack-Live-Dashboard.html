Hub Dashboard

This page describes the features of the YouTrack Dashboard. If your YouTrack instance is linked to an external Hub service, you have access to the Hub Dashboard.

Different widgets are available for each of the JetBrains team tools that are integrated with your Hub installation. A description of the Hub Dashboard is available in the Hub Help.

The Dashboard is a convenient way to get an overview of the current status of your tasks and monitor the progress of your team.

The standard search box is displayed at the top of the dashboard. If you want quick access to a list of specific issues, you can enter a search query directly from the dashboard.

The Dashboard is the default start page in YouTrack. To open the Dashboard from any other page in YouTrack, just click the YouTrack logo or the logo of your company if your your instance uses a custom logo.

The Dashboard displays widgets on cards. The YouTrack Dashboard supports the following types of widgets:

  • Markdown Notes Widgets
  • Wiki Notes Widgets
  • Issue Widgets
  • Report Widgets

You can add as many widgets of each type to the Dashboard as you like.

To add a widget to the dashboard:

  1. Click the Add widget button.
  2. Select the type of widget you want to add to the Dashboard.
  3. Configure the settings of the widget.
    • For report widgets, select the report you want to display in the widget.
    • For issue widgets, enter a search query or select a saved search, project, or tag. Click the Create widget button.
    • For note widgets, enter text in the input field. Click the Create widget button.

    — The widget is added to your Dashboard.

Multiple Dashboards

You can create and view multiple dashboards, each with their own set of widgets. This feature lest you set up and monitor dashboards for different projects.

The number of available dashboards is displayed under the name of the active dashboard. The names of the previous and next dashboard on the list are also shown.

Press Shift + Left or Shift + Right to switch between dashboards.

Press Shift + Down to expand the dashboards drop-down list.

To add another dashboard:

  1. Click the down arrow next to the name of your current dashboard and select the Create new dashboard option.

    Alternatively, press Alt + Shift + N on your keyboard.

  2. Enter a name for the new dashboard in the Name Your Dashboard dialog.
  3. Click the Create dashboard button.

    — The new, empty dashboard is added to Hub.

Shared Dashboards

In addition to the dashboards you create yourself, you can view dashboards that have been shared with you.

When a member of your team shares a dashboard with you directly or with a group of which you are a member, you see a link next to the name of your current dashboard. When you click this link, the name of the shared dashboard is displayed, as well as the name of the user who shared the dashboard.

  • To add this dashboard to your list, select the shared dashboard and click the Add to my dashboards button.
  • To remove the notification link from your dashboards, click the Dismiss link in the drop-down list.

    You can still view the shared dashboard even if you have not added it to your list.

Organize a Dashboard

To organize a dashboard, all you have to do is drag the cards around the dashboard and place them in the order you like.

You can size the cards on the dashboard. Drag the edge of the card to set the height and width. The size of the card automatically snaps to the closest intersection of the grid.

To update content of a widget, click the refresh icon in the title bar.

Additional Actions for Dashboards

In addition to the option to add widgets to your dashboards, the following actions are available:

OptionDescription
ShareYou can configure a dashboard and share it with another user or group. For more information, see Share a Dashboard.
RenameYou can change the name of your dashboard at any time. To access this option, click Edit > Rename.
DeleteIf you no longer want to use a dashboard on your list, you can delete it. To access this option, click Edit > Rename.

Additional Actions for Widgets

There are several operations that you can use to customize your Dashboard and share its content with your team.

To open the list of actions that are available for a widget, click the down arrow in the title bar.

If a widget contains information that you want every member of your team to work with, you can share the widget.

To share a widget:

  1. Click the down arrow in the title bar of the widget that you want to share.
  2. Select the Share widget option.
  3. Copy the link in the dialog and send it to your team-mates.

    If the person with whom you shared the widget does not have permission to view the data that is used in the widget, the widget is empty.

You can create a clone of an existing widget on your dashboard. This option comes in handy when you want to add a similar widget with minor changes.

To clone a widget:

  1. Click the down arrow in the title bar of the widget that you want to clone.
  2. Select the Clone widget option.

    — A copy of the widget is added to your Dashboard.

If you want to modify the settings of the widgets on your Dashboard, you can edit them at any time.

To edit an existing widget:

  1. Click the down arrow in the title bar of the widget that you want to edit.
  2. Select the Edit option.
  3. Modify the settings or content of the widget.
    • For report widgets, select the report that you want to display in the widget.
    • For issue widgets, enter a new search query or select a different saved search, project, or tag. Click the Apply button to update the widget.
    • For note widgets, edit the text in the input field. Click the Apply button to update the widget.

    — The widget is updated with the new settings.

If you no longer want to see a widget on your Dashboard, you can remove it.

To remove a widget from the Dashboard:

  1. Click the down arrow in the title bar of the widget that you want to remove from the Dashboard.
  2. Click the Remove option.
  3. Click the Remove button in the confirmation dialog.

    — The widget is removed from your Dashboard.
