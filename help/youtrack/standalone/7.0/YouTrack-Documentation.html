Getting Started

Introduction to YouTrack

Important!

Before you upgrade to YouTrack 7.0, read the upgrade installation guide.

YouTrack 7.0 introduces changes that affect the installation and upgrade procedure.

JetBrains YouTrack is a web-based issue tracking and project management platform. YouTrack is optimized for developers and agile teams.

  • Find issues fast with search queries that use the same code completion features found in JetBrains IDEs.
  • Update issues wherever you work with commands that you apply directly in YouTrack, add to commit messages in a connected VCS, or enter in your Jabber client.
  • Plan and prioritize your product development on an agile board.

New in 7.0

With the release of YouTrack 7.0, you can enjoy the following features and enhancements:

Demo

Click to watch a short overview of new and updated features in YouTrack 7.0.

FeatureDescription
Agile BoardsThe agile board has been redesigned from the ground up. The new board is flexible, powerful, and dynamic. The new board features the following updates:
  • Sprints are no longer linked to fix versions. This change lets you manage sprints and fix versions independently. The ability to link sprints to fix versions was added as an option.
  • You can set a goal for each sprint — the goal keeps the team focused on the same objective.
  • The new board gives you complete control over which issues are assigned to sprints. In the old agile board, new issues that matched a column and swimlane were added to the current sprint automatically. The ability to add new issues to a sprint automatically was added as an option.
  • Many of the features that you can use in the issue list are now available in the backlog. You can filter the list, apply commands, edit custom fields, and use keyboard shortcuts. New functionality lets you prioritize issues in the backlog, add issues as cards or swimlanes to the board, and assign an issue to any sprint.
  • When you add a card to the board, YouTrack adds the values for the column and swimlane to an issue draft. If you accidentally close the issue or close your browser, you can pick up right where you left off. Drafts that were created on a board are added to the list of drafts on the full issue screen.
  • The board updates in real time. Changes to any issue are reflected on the board as soon as they are applied.
  • The interface for creating or editing issues on the board lets you add and update any value that you can set in full page view. Edit custom fields, set visibility, and estimate tasks without leaving the board.
  • The tuning and layout make working with the board comfortable, easy to navigate, and user-friendly.
Gantt ChartThe Gantt chart lets you visualize a set of tasks as a project schedule. The chart shows you what should happen when and illustrates which tasks depend on the completion of others. The Gantt chart can display issues from one or more projects that match specific search criteria.

The Gantt chart supports capacity limitations. Measure how much time is required to finish all of the estimated tasks with a set number of resources. Judge whether you can finish your tasks on time with your current resources and predict how much faster you can complete these tasks when you add resources to the team.

Project TemplatesStart a new project and hit the ground running. Templates for pure issue tracking, Scrum, or Kanban projects are pre-configured to include a custom-tailored set of issue fields, agile boards, and workflows.
Administration UpdatesThe built-in Hub integration has been updated to version 2.5, adding new features to the Administration menu. The latest version of Hub lets you work with the following features:
  • SAML 2.0 — Use your YouTrack profile to log in to external services that support SAML 2.0 authentication.
  • Password strength policy — Set a password policy that is enforced every time users create or change passwords for their YouTrack accounts. Increase the overall security of your system and protect your instance from brute-force attacks.
  • Service-specific permissions — Manage separate sets of permissions by application. The new permissions interface shows you which permissions grant access to issue-related actions in YouTrack versus permissions that are used for access management operations in the built-in Hub service.
  • Delete user profiles — Users can now delete their own Hub profiles, removing their accounts from YouTrack. Administrators can use this feature to delete unused accounts from YouTrack.
New Dashboard OptionsYou can now create multiple dashboards and share a dashboard with a group of users. Using a shared dashboard helps keep your project team on the same page.

You can also create a personal dashboard and get an overview of the issues and metrics that are most important to you.

HelpIf you've worked with an earlier version of YouTrack, you may have noticed that we have moved the documentation to our help center on jetbrains.com. We hope that you enjoy the updated content layout. We also put a lot of effort into making it easier for you to find and digest information that improves your YouTrack experience.

Discontinued Features

The following features have been disabled in YouTrack 7.0, and are no longer supported:

FeatureDescription
BookmarkletDue to changes in computer security standards, the Create YouTrack Issue bookmarklet has been disabled. This feature is no longer available in YouTrack 7.0.
Restricted ModeThe Restricted Mode setting has been applied by default to all YouTrack instances. This means that you can no longer use keywords for viewed issues in search queries. The option to enable this setting has been removed from the user interface.
