PUT /admin/user/{userName}

Create a new user.

PUT /rest/admin/user/{login}

Parameters:

Name Type Description
login string Login name of a user to be created. Required
fullName string Full name of a new user.
email string New user's e-mail address. Required
jabber string Jabber address for a new user.
password string Password for the new user.

Sample

Request:

xmlPUT rest/admin/user/YouTrackUser

Response:

Empty response. Response code: 201 Response message: Created
