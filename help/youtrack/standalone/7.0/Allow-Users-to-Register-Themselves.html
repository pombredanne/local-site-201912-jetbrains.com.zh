Allow Users to Register Themselves

To allow users to register themselves in YouTrack, you must enable registration in the Hub auth module of your YouTrack instance.

To configure the Hub auth module:

  1. In the Access Management section of the Administration menu, select Auth Modules.
  2. Select the Hub module in the list.
  3. Click the Enable registration button.
    • Users can create their own accounts in YouTrack.

Permissions

Global:

  • Update Auth Module
  • Create User

Enable reCAPTCHA

To protect your YouTrack server against bots, you can enable reCAPTCHA on your registration page.

To enable reCAPTCHA on the registration page:

  1. Click the Enable reCAPTCHA button in the Hub auth module settings.
    • Additional parameters for reCAPTCHA setup are displayed.
  2. Click the link to access the reCAPTCHA key management console.
    • The Create reCAPTCHA key page opens.
  3. Register the domains you want to enable for the service. reCAPTCHA is a part of Google services, so you can use your google account to log in. Read the tips provided on the page before creating keys.
  4. When you have created your set of keys, go back to the Hub auth module and enter your Public Key and Private Key in the respective fields in YouTrack.
  5. Save your settings.
    • ReCAPTCHA is enabled for users who create their own accounts in YouTrack.

Permissions

Global:

  • Update Auth Module
