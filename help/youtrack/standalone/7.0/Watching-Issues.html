Watch Issues

YouTrack lets you watch the issues you are interested in. You can also set another user as a watcher for an issue. When notifications are enabled for your YouTrack server, you and other users are automatically notified when changes are applied to the issues you are watching.

You and other users can specify which changes you want to receive notifications for on the Filters and Notifications tab of the profile page. For more information, see Notifications.

Permissions

Per-project:

  • View Watchers

To watch an issue:

  • From the issue list, click the star icon to the left of the issue ID.
  • In full page view, click the star icon to the left of the issue ID.
  • Open the Apply Command dialog and enter the star me command.

To set another user as a watcher:

  1. Open the issue in full page view.
  2. Click the plus icon in the Watchers section of the sidebar.
    • The Apply Command dialog opens. The watcher command is automatically entered in the input field.
  3. Enter the login or full name of the user who you want to add as a watcher. YouTrack filters the list of users as you type.
  4. Select a user from the list.
  5. Click the Apply button.

Permissions

Per-project:

  • Update Watchers

You can also set another user as a watcher for one or more issues from the issues list. Simply select the issues in the list and apply the command as described above.
