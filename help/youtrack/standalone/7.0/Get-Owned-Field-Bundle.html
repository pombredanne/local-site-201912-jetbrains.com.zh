Get Owned Field Bundle

GET /admin/customfield/ownedFieldBundle/{bundleName}

Get owned field bundle by its name

GET /rest/admin/customfield/ownedFieldBundle/{bundleName}

Request parameters:

Name Type Description
bundleName string Name of an owned field bundle to get.

Response parameters:

Name Description
name Name of owned field bundle.
value Value, that bundle contains. For more information about value tag attributes see Get Owned Field.
