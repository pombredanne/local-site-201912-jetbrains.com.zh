YouTrack Standalone 2019.2帮助

管理自定义小部件

“ 自定义小部件”页面使您可以在中央位置管理自定义小部件的状态。在这里，您会看到所有手动上载到YouTrack的小部件，以及来自JetBrains插件存储库的兼容小部件的集合。

要访问此页面，请打开“ 管理”菜单，然后从“ 服务器设置”部分中选择“ 自定义窗口小部件 ” 。

“ 定制小部件”页面的标题中提供了以下操作：

行动

描述

新的小部件

在边栏中打开“ 新建小部件”对话框。在这里，您可以将新的自定义小部件上传到YouTrack。有关说明，请参阅上载新的自定义小部件 。

禁用启用

切换所有选定小部件的状态。有关更多信息，请参见禁用自定义窗口小部件 。

删除

删除所有选定的小部件。有关更多信息，请参见删除自定义窗口小部件 。

更新资料

对于从JetBrains插件存储库安装的窗口小部件，此按钮会将窗口小部件更新为最新的可用版本。

小部件游乐场

打开小部件游乐场页面。在这里，您可以测试正在构建的小部件的功能。有关更多信息，请参见测试自定义窗口小部件 。

细节

在侧边栏中显示和隐藏“ 详细信息”面板。

小部件列表

小部件列表分为两部分。

  • “ 已安装的小部件”部分显示了YouTrack中当前安装的小部件。这可以是已手动上传或从JetBrains插件存储库安装的窗口小部件的组合。

  • “ 存储库中的可用”部分显示了可从JetBrains插件存储库安装的小部件。

    筛选列表以仅显示与您使用的YouTrack版本兼容的窗口小部件。如果您的YouTrack安装已过期，则存储库中可能还有其他小部件，这些小部件仅与更高版本兼容。要查看可用小部件的完整列表，请在线浏览插件存储库 。

将为列表中的每个窗口小部件显示以下属性：

属性

描述

图标

在仪表板上显示为小部件图标的图像。

名称

自定义窗口小部件的名称。此属性显示在仪表板上的“ 添加小部件”下拉列表中。

小部件描述的第一部分显示在名称旁边。要查看完整的描述，请打开侧栏中的“ 详细信息”面板。

小部件状态

名称右侧的徽章集合提供了各种状态指示器。使用以下标志：

  • 已启用 -已启用小部件，并且可在仪表板上使用。

  • 已禁用 —该小部件已安装，但无法使用。有关更多信息，请参见禁用自定义窗口小部件 。

  • 手动上传 -窗口小部件已手动上传到YouTrack。不支持来自插件存储库的自动更新。对小部件的更新也必须手动应用。有关更多信息，请参见更新自定义窗口小部件 。

  • 过时的 -插件存储库中提供了新版本的小部件。可用版本显示在侧栏中。该更新可以直接在“ 自定义小部件”页面上应用。

版

自定义窗口小部件的当前版本。

作者

创建窗口小部件的开发人员的名称和电子邮件地址。

允许的应用

允许自定义窗口小部件从连接的服务中提取和显示信息的应用程序列表。可访问服务的完整列表显示在“ 详细信息”面板中。

上载新的自定义小部件

如果要使用尚未上载到JetBrains插件存储库的窗口小部件，则必须手动将其上载到YouTrack。当YouTrack安装在专用网络上时，您也可以使用此选项安装从资源库下载的小部件。

要上传新的自定义小部件：

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 自定义窗口小部件” 。

  2. 单击新建窗口小部件按钮。

    • 将打开“ 新窗口小部件”对话框。

  3. 将自定义窗口小部件的ZIP文件拖到放置区，或单击放置区以从本地目录中选择ZIP文件。

    • 集线器检查ZIP归档文件，以确保它符合大小限制。这些限制适用于所有上载，包括从JetBrains插件存储库下载的自定义窗口小部件。有关详细信息，请参见大小限制 。

    • YouTrack显示在小部件清单文件中指定的属性。这包括小部件访问的所有服务的列表以及每个应用程序的受支持版本。

  4. 单击上载小部件按钮。

    • 将打开新窗口小部件的“ 定制窗口小部件”页面。

    • 自定义窗口小部件已启用并且可以使用。

从存储库安装窗口小部件

JetBrains插件存储库直接与YouTrack集成。这意味着您可以在不离开YouTrack的情况下从资源库安装小部件。与您的YouTrack安装版本兼容的窗口小部件列表显示在窗口小部件列表的“ 可从存储库中”部分中。

要建立与存储库的连接，您的YouTrack安装必须有权访问Internet。如果您只能从专用网络内部访问YouTrack Standalone安装，则不能直接访问存储库。但是，您仍然可以选择从存储库下载窗口小部件，然后将其手动上载到YouTrack安装中。

要从存储库安装窗口小部件：

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 自定义窗口小部件” 。

  2. 在窗口小部件列表的“ 可从存储库中”部分中选择一个窗口小部件。您可以选择多个小部件。

  3. 单击标题中的“ 安装”按钮。

    • 所选的一个或多个窗口小部件将添加到窗口小部件列表的“ 已安装的窗口小部件”部分。

    • 所选的一个或多个窗口小部件已启用并可供使用。

更新自定义小部件

对于手动上载的小部件和从存储库安装的小部件，更新自定义小部件的过程有所不同。

更新手动上传

扩展自定义窗口小部件的功能后，可以将这些更新应用于现有窗口小部件。这意味着用户可以访问更新的功能，而不必替换其仪表板上的小部件。

窗口小部件列表中的手动上传徽章指示窗口小部件已手动上传，并且也必须手动更新。

要手动更新现有的定制窗口小部件：

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 自定义窗口小部件” 。

  2. 单击您要更新的小部件的名称。

    • 窗口小部件配置文件在“ 自定义窗口小部件”页面中打开。

  3. 单击更新按钮。

    • 将打开“ 上传新版本”对话框。

  4. 将包含自定义窗口小部件的更新版本的ZIP文件拖到放置区，或单击放置区以从本地目录中选择ZIP文件。

    • YouTrack显示在小部件清单文件中指定的属性。这包括小部件访问的所有服务的列表以及每个应用程序的受支持版本。

  5. 单击更新窗口小部件按钮。

    • 更新将应用于现有窗口小部件。

    • 新版本中支持的所有其他功能将立即可用。

从存储库更新窗口小部件

从JetBrains插件存储库中安装的小部件会每周自动更新。如果尚未将存储库中的最新版本应用于您的安装，则可以将更新直接拉到YouTrack中。

窗口小部件列表中的标记已过期 ，表明有可用的新版本的窗口小部件。可用版本显示在侧栏中。

要从存储库更新窗口小部件：

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 自定义窗口小部件” 。

  2. 选择要更新的小部件。如果您有多个标记为已过期的窗口小部件，则可以一次选择并更新它们。

  3. 单击标题中的“ 更新”按钮。

    • 更新将应用于现有窗口小部件。

    • 新版本中支持的所有其他功能将立即可用。

禁用自定义小部件

如果您不再希望用户使用自定义窗口小部件，但又不想从YouTrack删除它，则可以无限期禁用该窗口小部件。

例如，开发人员想要修复的现有窗口小部件存在问题或错误。您可以禁用自定义窗口小部件，以使用户不会遇到当前行为的问题，并在问题已更新为更新版本时启用窗口小部件。

禁用窗口小部件时，在其仪表板上安装了窗口小部件的用户会看到一条消息，指出系统中未找到该窗口小部件。编辑和刷新控件不可见。而是，用户看到了从其仪表板上删除窗口小部件的选项。

禁用自定义窗口小部件：

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 自定义窗口小部件” 。

  2. 在列表中选择要禁用的一个或多个小部件。

  3. 单击禁用按钮。

    • 所选的小部件已禁用。

此操作在自定义窗口小部件的配置文件页面上也可用。

对于禁用的小部件，标题和小部件配置文件页面中的选项分别更改为“ 启用”和“ 启用小部件” 。使用这些控件可以重新激活禁用的窗口小部件。

删除自定义小部件

如果YouTrack中不再支持自定义小部件，则可以将其删除。例如，当自定义窗口小部件显示团队不再使用的应用程序中的信息时。

删除窗口小部件时，在其仪表板上安装了窗口小部件的用户会看到一条消息，指出系统中未找到该窗口小部件。编辑和刷新控件不可见。而是，用户看到了从其仪表板上删除窗口小部件的选项。

禁用自定义窗口小部件：

需要权限：低级管理员写

  1. 在“ 管理”菜单的“ 服务器设置”部分中，选择“ 自定义窗口小部件” 。

  2. 在列表中选择要删除的一个或多个小部件。

  3. 单击删除按钮。

  4. 在对话框中确认操作。

    • 所选的小部件将被删除。

    • 如果该小部件是手动上传的，则将其从YouTrack中完全删除。

    • 如果该窗口小部件是从存储库中安装的，则将从已安装的窗口小部件列表中删除它，并返回到存储库中可用的窗口小部件列表。
