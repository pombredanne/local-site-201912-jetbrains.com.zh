CommandLimitedVisibility

Stores the restricted visibility settings for the command.

Extends CommandVisibility

This table describes attributes of the CommandLimitedVisibility entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

permittedGroups

Array of UserGroups

A list of groups that can access the comment. Read-only.

permittedUsers

Array of Users

A list of users that are allowed to access the comment. Read-only.
