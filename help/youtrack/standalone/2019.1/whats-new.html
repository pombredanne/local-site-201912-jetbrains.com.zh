YouTrack Standalone 2019.1 Help

What's New

We deliver updates and improvements to YouTrack on a continuous basis. Here's an overview of changes that are available in this version.

New Features

Feature

Description

Availability

Display a System-wide Banner

We've added a global setting that displays a banner at the top of every page in YouTrack. This feature replaces the previously supported option to add a system-wide banner by uploading a file to your server. Use this feature to post messages that alert users to system-wide events like performance problems and scheduled maintenance.

2019.1.52545

Favorite Agile Boards

A new control in the list of agile boards let you mark an agile board as a favorite. Your favorite boards are always shown at the top of the list.

2019.1

YouTrack App for Slack

Our custom integration with Slack lets you link your YouTrack Standalone installation to your workspace and get the most out of both applications. Get issue update notifications and copy messages to issue drafts.

2019.1

Community-based Language Support

You now have the option to switch to a language that is supported by the YouTrack user community. If you want to work with YouTrack in a language that isn’t officially supported by JetBrains, you can contribute your own translations.

Updates and Enhancements

Feature

Description

Issue Data Export Options in New Issues List

The options to export issue data in Excel and CSV formats have been added to the experimental version of the Issues list.

2019.1.52545

Experimental Issues List Updates

We've processed your feedback from our initial redesign for the Issues list and have updated the interface accordingly. Most of these updates restore functionality that was supported in the "classic" UI, but there are a few improvements that are unique to the updated version. The latest version supports the following features and enhancements:

  • Fixed-width presentation for values in custom fields.

  • Visual separators between issues and field values.

  • Save action for search queries.

  • Matching issues count.

  • [NEW] Saved searches in smart filters.

  • [NEW] Shift + click multi-select.

2019.1.51759

Enhanced Data Handling in Time Reports

Several changes to the presentation and manipulation of data in time reports help you track and quantify spent time.

  • The option to display work types shows totals for each work type in its own column.

  • A new option in the report settings lets you show values for custom fields when the data is presented using the per issue option in the calculated report. This setting also determines which fields are included in the dataset that is available for download in CSV and Excel format.

  • A new option in the calculated report lets you show remaining (unspent estimation) and total spent time.

Groupings, totals, and views are also applied to the data when you export it in Excel format. This exposes additional points of data for reference in external analytic tools.

2019.1.51759

Relative Parameters for Future Dates

The search query language now supports pre-defined values for date ranges that occur in the future. Use these parameters in your search queries and report settings to find events that are scheduled to happen tomorrow, next week, or next month.

You can also use these parameters to specify dates in commands.

2019.1.51759

App Password Support for Bitbucket Integrations

You can now connect YouTrack to a Bitbucket repository with a username and an app password. This lets you set up integrations with a Bitbucket account that uses two-step verification.

It also means that you don't have to store your personal passwords in the integration settings.

Changes in this release make YouTrack compatible with the latest version of the Bitbucket API.

2019.1.51078

Customizable Fields for Cards on Agile Boards

New options for cards on agile boards let you choose which fields are visible and how the values are presented. This lets you show the same collection of data for each card on your board even when you manage issues in multiple projects.

Also, cards don't have a fixed height anymore — if, for example, the issue description is short or empty, the card height is reduced to compensate. As a result, you see more information on the page before you have to scroll.

2019.1

System-wide Dark Theme

There's a new option in the general settings of your YouTrack profile that lets you apply a dark theme to every page in YouTrack except the Issues list. You can enable the nighttime-friendly version of the Issues list on an experimental basis.

You'll also find a Dark Theme toggle in the menu that opens when you click your avatar.

2019.1

Clone Action for Agile Boards.

This feature adds an option to the settings panel for an agile board that lets you clone the board.

2019.1

Per-user Setting for Period Format

We added a setting to your YouTrack profile that lets you choose how to display values that are stored in period fields. This makes it easier to perform calculations and compare values at a glance.

2019.1

Anonymous SMTP Server Connections

YouTrack can now connect to an SMTP server without sending a login and password. This lets you connect to an SMTP server that does not require authentication.

2019.1

Syntax Highlighting Improvements in Markdown

Improvements to the handling for language-specific syntax highlights in Markdown improve compatibility with similar features in GitHub and add support for code blocks without highlighting.

2019.1

Quick Access to Project Settings

We've added a control that lets you jump directly from an issue to the settings for its project. Users with the Update Project permission can access project settings by clicking the icon next to the project name in the fields panel.

2019.1

Smart Default for Work Item Type

The work item type in new work items defaults to the value that you assigned to your most recent work item. This change helps users who consistently perform the same type of activity to report their work faster.

2019.1

Single Term Search Optimization

Text searches that contain single terms assign higher relevance scores to words that exactly match the specified word form. Search queries that contain single terms in quotation marks only return exact matches for the specified word form.

Discontinued Features

Feature

Description

Availability

New Workflows in the YouTrack Workflow Editor

We no longer support new workflows that are written in the external YouTrack Workflow Editor. We will continue to support workflows that were written in this editor through the first YouTrack release in 2019. After that point in time, we will only support workflows in JavaScript.

If you're still working with old workflows, we encourage you to rewrite them in JavaScript as soon as possible. For details, see Converting Old Workflows to JavaScript.

Removed in 2018.3

Old YouTrack REST API

With the initial publication of endpoints for our current REST API in YouTrack 2018.3, we encourage developers to start migrating from the previous version. Once we have published newer versions for the endpoints that were supported in the previous API, it will be officially deprecated. We will continue to support the old REST API for a fixed period after its deprecation to allow adequate time for migrating applications that use it.

If you have an existing application or integration that uses the old API, we encourage you to familiarize yourself with the updated version. If your application uses endpoints for which an equivalent is not yet published in the new API, join our YouTrack Community in Slack. Let us know what endpoints you’re working with and help us identify which features should be published next.
