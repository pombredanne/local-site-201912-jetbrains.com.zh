POST User Group

POST /admin/user/{login}/group/{group}

Add user account to a group.

POST /rest/admin/user/{login}/group/{group}
Parameters:

Name

Type

Description

login

string

Login name of a user to be added to a group.

group

string

Name of a group to add the user to.

Sample

Request:
POST /rest/admin/user/Alex/group/idea-developers
Response:
Response Code: 200 Response Status: OK
