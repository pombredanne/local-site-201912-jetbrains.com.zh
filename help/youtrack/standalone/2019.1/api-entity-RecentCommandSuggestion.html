RecentCommandSuggestion

Represents command along with the comment that was recently applied by the user.

Extends Suggestion

This table describes attributes of the RecentCommandSuggestion entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

comment

String

Comment that was added along with the command. Read-only. Can be null.

visibility

CommandVisibility

Comment visibility that was used for the recent comment. Read-only.
