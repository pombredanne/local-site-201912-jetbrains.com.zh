IssueWatcher

Represents user that is subscribed for issue notifications.

This table describes attributes of the IssueWatcher entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

user

User

User who is subscribed to issue notifications. Read-only. Can be null.

issue

Issue

Original issue the user subscribed to. Read-only. Can be null.

isStarred

Boolean

true if user is subscribed to this issue via star.
