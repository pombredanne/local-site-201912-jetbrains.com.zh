WatchFolder

Extends IssueFolder

This table describes attributes of the WatchFolder entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

owner

User

User that owns the watch folder. Read-only. Can be null.

visibleFor

UserGroup

Group of users that can see this watch folder. If folder is visible only for its owner, this property is null. Can be null.

updateableBy

UserGroup

Group of users that can update this watch folder. If folder is updatable only by its owner, this property is null. Can be null.
