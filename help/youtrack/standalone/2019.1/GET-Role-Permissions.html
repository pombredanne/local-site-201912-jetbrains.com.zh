GET Role Permissions

GET /admin/role/{role}/permission

Get role permissions. For the list of available permissions, see the YouTrack Permissions page.

GET /rest/admin/role/{role}/permission
Parameters:

Name

Type

Description

role

roleByName

Name of a role.

Request:

GET /admin/role/Admin/permission
