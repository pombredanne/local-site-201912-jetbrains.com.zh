Operations with Specific License

Resource

/api/admin/globalSettings/license

Returned entity

License. For the description of the entity attributes, see Supported Fields section.

Supported methods

  • GET: Read a Specific License.

  • POST: Update a Specific License.

License attributes

Information about the current license.

This table describes attributes of the License entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

username

String

Username used for license. Can be null.

license

String

License key. Can be null.

error

String

License error. If there is no error, this property is empty. Read-only. Can be null.

Read a Specific License

Request syntax

Request parameters

Parameter

Type

Description

fields

string
