UsesMarkupActivityItem

The activity shows the change of the boolean flag denoting either wiki or markdown is used in the target entity as a markup language. If "true", then the markdown is used.If the flag is equal to "false" then a wiki is used.

Extends SingleValueActivityItem

This table describes attributes of the UsesMarkupActivityItem entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

removed

Boolean

Old value of the flag. Read-only. Can be null.

added

Boolean

New value of the flag. Read-only. Can be null.

markup

String

Target entity text rendered using the new value of the markup language. Read-only. Can be null.
