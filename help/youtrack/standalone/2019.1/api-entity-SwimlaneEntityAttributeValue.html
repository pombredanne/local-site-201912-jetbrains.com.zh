SwimlaneEntityAttributeValue

Represents single swimlane in case of AttributeBasedSwimlaneSettings

Extends DatabaseAttributeValue

This table describes attributes of the SwimlaneEntityAttributeValue entity.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

name

String

Name of a swimlane. Can be null.

isResolved

If true, issues in this swimlane are considered to be resolved. Can be updated only for newly created value. Read-only.
