DELETE Role Permission

DELETE /admin/role/{role}/permission/{permission}

Delete a permission from a role.

DELETE /rest/admin/role/{role}/permission/{permission}
Parameters:

Name

Type

Description

role

roleByName

Name of the role to delete a permission from.

permission

string

Permission to be deleted from the role (See YouTrack Permissions)

Sample

Request:
DELETE /rest/admin/role/Tester/permission/CREATE_ISSUE
Response:
Response Code: 200 Response Status: OK
