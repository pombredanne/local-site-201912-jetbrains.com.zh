Operations with Specific NotificationsUserProfile

Resource

/api/admin/customFieldSettings/bundles/user/{bundleID}/individuals/{userID}/profiles/notifications

Returned entity

NotificationsUserProfile. For the description of the entity attributes, see Supported Fields section.

Supported methods

  • GET: Read a Specific NotificationsUserProfile.

  • POST: Update a Specific NotificationsUserProfile.

NotificationsUserProfile attributes

User's data related to notifications.

This table describes attributes of the NotificationsUserProfile entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description
