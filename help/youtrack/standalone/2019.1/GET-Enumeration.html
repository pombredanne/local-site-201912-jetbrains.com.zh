GET Enumeration

GET /admin/customfield/bundle/{bundleName}

Get custom field enumeration bundle by its name

GET /rest/admin/customfield/bundle/{bundleName}
Request parameters:

Name

Type

Description

bundleName

string

Name of the enumeration to get.

Response parameters:

Name

Description

name

Bundle name.

value

Value, that this bundle contains. For more information see GET a Value from a Bundle.

Sample

Request:
GET /rest/admin/customfield/bundle/RegionEnum
Response:
<enumeration name="RegionEnum"> <value>Ural</value> <value>South</value> <value description="Capital of Russia" colorIndex="3">Moscow</value> <value>St. Petersburg</value> </enumeration>
