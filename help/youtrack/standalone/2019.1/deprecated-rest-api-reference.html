Old REST API

This section of the documentation describes resources and methods that are supported in the deprecated YouTrack REST API. This reference is provided only to help you troubleshoot existing client applications that use this API. For new applications, please use resources and methods from the current version.

With the initial publication of endpoints for our current REST API, we encourage developers to start migrating applications from the previous version. Once we have published newer versions for the endpoints that were supported in the previous API, it will be officially deprecated.

If you have an existing application or integration that uses the old API, we encourage you to familiarize yourself with the updated version. If your application uses endpoints for which an equivalent is not yet published in the new API, join our YouTrack Community in Slack. Please be aware of the following limitations when using this reference:

  • This version of the API is no longer in active development. Many of the features that are supported in the current version of YouTrack are not accessible with this version of the API.

  • There are a number of known omissions and gaps in this API reference that will not be addressed. Our efforts are only focused on providing this information for the current API.
