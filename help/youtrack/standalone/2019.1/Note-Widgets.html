Wiki Notes Widgets

A Wiki Notes widget displays whatever message you want to show on your dashboard. This widget lets you use YouTrack Wiki syntax to format your text.

To add a Wiki Notes widget to the Dashboard:

  1. Click the Add widget button on the Dashboard.

  2. From the drop-down list, select Wiki Notes.

    • A card is added to the dashboard with the title Wiki Notes.

  3. Enter an optional widget title.

  4. Enter your note in the input field. Use Wiki markup to format your text. Click the Syntax Help link to view the supported formatting options.

    For a list of supported formatting options, see YouTrack Wiki Syntax

  5. Click the Create widget button.

    • The Wiki Notes widget is added to the dashboard.

    • If you entered a title, your text is displayed in the widget header.
