Update a Build in a Bundle

POST /admin/customfield/buildBundle/{bundleName}/{buildName}

Update build

POST /rest/admin/customfield/buildBundle/{bundleName}/{buildName}?{newName}&{description}&{colorIndex}&{assembleDate}
Parameters:

Name

Type

Description

bundleName

string

Name of a bundle, in which a build should be updated.

buildName

string

Name of a build to update.

newName

string

New name for the build.

description

string

Build's description.

colorIndex

Integer

Sequential number of a color scheme (background color/text color pair) for the build. See the list of available Color Indices List.

assembleDate

instant

Assemble date for the build.
