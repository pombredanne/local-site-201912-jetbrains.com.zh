IssueWatchers

Represents users that are subscribed to notifications about issue.

This table describes attributes of the IssueWatchers entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

hasStar

Boolean

true if the current user added the star tag to this issue. Otherwise, false.

issueWatchers

Array of IssueWatchers

Watchers who subscribed for this particular issue.

duplicateWatchers

Array of IssueWatchers

Watchers who subscribed to the duplicates of this issue. Read-only.
