YouTrack Standalone 2019.1 Help

Issue Lists

There are a few more reports that you can generate in YouTrack. There reports are not listed on the Reports page. Instead, you generate these reports directly from the Reports drop-down list on the Issues list.



The ability to export issue data is only available in the experimental version of the Issues list from version 2019.1.52545. The option to export issues in HTML is not available in the experimental version.

To export issue data as described here, turn off the experimental UI. For instructions, see Redesigned Issues List.

The following options are available:

Option

Description

Export to CSV

Exports the list of issues as a CSV file. The list of issues is filtered to match the current search query and context. You can use this option to create a list of issues for import into another application or web page.

Export to MS Excel

Exports the list of issues in Microsoft Excel format. The XSLX file contains the same data set that is exported as a CSV file. Unlike CSV, exports in Excel format preserve data types for values in custom fields and other issue attributes.

Issues in HTML

Generates a list of issues in HTML. The list of issues is filtered to match the current search query and context. You can use this report to generate release notes or a list of recent updates. For detailed information, see Create Release Notes.
