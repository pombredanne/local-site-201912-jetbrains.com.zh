DateFormatDescriptor

Represents date format.

This table describes attributes of the DateFormatDescriptor entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

presentation

String

Name of the date format. Read-only.

pattern

String

Pattern that is applied when showing date with time. Read-only.

datePattern

String

Pattern that is applied when showing date. Read-only.
