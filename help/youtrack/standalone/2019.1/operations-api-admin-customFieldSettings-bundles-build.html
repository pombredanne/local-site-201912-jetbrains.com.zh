Operations with Specific BuildBundle

This resource provides operations with build bundles.

Resource

/api/admin/customFieldSettings/bundles/build/{bundleID}

Returned entity

BuildBundle. For the description of the entity attributes, see Supported Fields section.

Supported methods

  • GET: Read a Specific BuildBundle.

  • POST: Update a Specific BuildBundle.

  • DELETE: Delete a Specific BuildBundle.

BuildBundle attributes

Represents set of builds in YouTrack.

Extends BaseBundle

This table describes attributes of the BuildBundle entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

values

Array of BuildBundleElements

Collection of builds contained in this bundle.
