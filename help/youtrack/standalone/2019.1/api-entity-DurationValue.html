DurationValue

This table describes attributes of the DurationValue entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

minutes

Int

Number of minutes in the time interval. Read-only.

presentation

String

Presentation of the time interval according to the user settings of duration format and user locale. Read-only.
