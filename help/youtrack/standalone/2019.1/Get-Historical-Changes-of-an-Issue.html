Get Historical Changes of an Issue

GET /issue/{issue}/changes

Get historical changes of an issue.

GET /rest/issue/{issue}/changes
Parameters:

Name

Type

Description

issue

issueById

issueID of an issue, for which you want to get historical changes.

Response parameters:

For information about meaning of tags in response, see Get an issue.
