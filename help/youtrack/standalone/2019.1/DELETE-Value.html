DELETE Value

DELETE /admin/customfield/bundle/{bundleName}/{fieldValue}

Remove value from custom field enumeration bundle

DELETE /rest/admin/customfield/bundle/{bundleName}/{fieldValue}
Parameters:

Name

Type

Description

bundleName

string

Name of an enumeration bundle to delete a value from.

fieldValue

string

A value to delete.

Sample

Request:
DELETE /rest/admin/customfield/bundle/RegionEnum/Moscow
Response:
Enum bundle value [Moscow] deleted.
