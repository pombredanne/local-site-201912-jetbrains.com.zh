REST API Reference

With the publication of endpoints for our current REST API, we encourage developers to start migrating applications from the previous version. Once we have published newer versions for the endpoints that were supported in the previous API, it will be officially deprecated.

If you have an existing application or integration that uses the old API, we encourage you to familiarize yourself with the updated version. If your application uses endpoints for which an equivalent is not yet published in the new API, join our YouTrack Community in Slack.
