Operations with Specific BackupStatus

Resource

/api/admin/databaseBackup/settings/backupStatus

Returned entity

BackupStatus. For the description of the entity attributes, see Supported Fields section.

Supported methods

  • GET: Read a Specific BackupStatus.

BackupStatus attributes

Represents the current status of the backup process.

This table describes attributes of the BackupStatus entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

backupInProgress

Boolean

If true, the backup is currently running. Otherwise, false Read-only.

backupCancelled

Boolean

If true, the last back up operation was cancelled. Otherwise, false. Read-only.

backupError

BackupError

The error that appeared during the previous backup process. Returns null, if there was no error. Read-only. Can be null.

Read a Specific BackupStatus

Request syntax

GET /api/admin/databaseBackup/settings/backupStatus/{statusID}?{fields}

Request parameters

Parameter

Type

Description

fields

string

A list of BackupStatus attributes that should be returned in the response. If no field is specified, only the entityID is returned.
