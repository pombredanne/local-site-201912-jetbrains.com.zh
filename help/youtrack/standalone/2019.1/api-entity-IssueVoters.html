IssueVoters

Represents users that have voted for the issue or its duplicates.

This table describes attributes of the IssueVoters entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

hasVote

Boolean

Indicates whether the user has voted for the issue or one of its duplicates.

original

Array of Users

Users that voted for this particular issue. Read-only.

duplicate

Array of DuplicateVotes

Users who voted for duplicate issues. Read-only.
