YouTrack Standalone 2019.2帮助

将文件附加到问题

用例：

使用REST API将多个文件附加到问题。

请注意，此示例对YouTrack 2019.1和更高版本有效。在以前的版本中，尽管原理相同，但是查询参数和实体属性可能不同。此外，2019.1中引入了许多功能，例如按名称命名实体，但以前的版本中不支持。

摘要

要将文件附加到问题：

  1. 获取问题列表以获取目标问题的实体ID。

  2. 通过将POST请求发送到以下端点，将文件附加到问题：

    / api / issues / {issueID} /附件

一步步

  1. 获取目标项目中的问题列表。

    curl -X GET'https://example.myjetbrains.com/youtrack/api/issues?query=in:NP&$top=3&fields=id,idReadable,summary'\ -H'接受：application / json'\ -H '授权：无记名烫发：cm9vdA ==。MjZGZWI =。WB02vjX0cM2ltLTJXUE3VOWHpJYYNx'\ -H'内容类型：application / json'

    的query request参数使我们可以通过搜索查询过滤问题列表。在此示例中，我们使用查询仅获取属于目标项目的问题（ NP ）。

    对于此特定示例，为了缩短返回的问题列表，我们使用$top=3参数仅获取前三个发现的问题。

    在里面fields request参数，我们指示服务器为找到的每个问题返回以下属性：

    • id和idReadable -数据库实体ID和项目中问题的ID分别。

    • summary -问题的摘要可能有助于确定目标问题。

    在对此请求的响应中，服务器返回以下数据：

    [{“摘要”：“调查任务”，“ idReadable”：“ NP-56”，“ id”：“ 99-356”，“ $ type”：“问题”}，{“摘要”：“营销活动” ，“ idReadable”：“ NP-39”，“ id”：“ 99-303”，“ $ type”：“问题”}，{“ summary”：“发行版”，“ idReadable”：“ NP-77 “，” id“：” 99-500“，” $ type“：”问题“}]

  2. 找到目标问题的ID。在这种情况下，让我们考虑一下NP-77与数据库ID 99-500 。

  3. 要将文件附加到问题上，我们需要发送一个POST请求此端点的附件： /api/issues/{issueID}/attachments 。对于此用例，我们使用以下内容cURL命令：

    curl -v -i -F upload=@/Users/jetbrains/Downloads/youtrack.txt \ -F upload=@/Users/jetbrains/Downloads/youtrack_new.jpeg \ -H'Accept：application / json'\ -H'授权：不记名烫发：cm9vdA ==。MjZGZWI =。WB02vjX0cM2ltLTJXUE3VOWHpJYYNx'\ -H'内容类型：multipart / form-data'\ -X POST'https://example.myjetbrains.com/youtrack/api/issues/99-500/attachments?fields=id,name'

    这是我们使用的请求的关键参数：

    • -F选项使我们可以将所需文件上传到服务器。

      -F upload=@/Users/jetbrains/Downloads/youtrack.txt -应该附加到目标问题的文件的路径。您可以在一个请求中附加多个文件：

      -F upload=@/Users/jetbrains/Downloads/youtrack.txt \ -F upload=@/Users/jetbrains/Downloads/youtrack_new.jpeg

      的@前缀表示以下内容部分是文件。

    • Content-Type: multipart/form-data ：使用此标头附加文件。

    在响应中，服务器返回name和附件的数据库实体ID：

    [{“ name”：“ youtrack.txt”，“ id”：“ 134-31”，“ $ type”：“ IssueAttachment”}，{“ name”：“ youtrack_new.jpeg”，“ id”：“ 134- 32“，” $ type“：” IssueAttachment“}]

就是这样：我们已将所有文件附加到该问题上。

请注意，您不能同时创建一个新问题并为其添加附件。您只能将文件附加到现有问题。
