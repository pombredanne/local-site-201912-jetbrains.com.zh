管理群组访问

使用“ 角色”选项卡，您可以查看和管理组成员可用的权限。组成员根据一个或多个项目中分配给该组的角色继承权限。

您还可以通过将组直接添加到项目团队来使用组来授予组成员对项目的访问权限。有关更多信息，请参见管理项目团队 。

查看组的访问权限

组的访问权限显示在组配置文件的“角色”选项卡上。在这里，您可以查看将哪些角色分配给该组。

  • 该列表显示分配给该组的每个角色，以及授予该组该级别访问权限的项目数。

  • 在每个角色下是授予该角色的项目列表。

  • 边栏显示分配给所选角色的一组权限。

要查看组的访问权限：

需要权限：读取用户基本，读取角色，读取组，读取项目完整

  1. 打开组配置文件的“ 角色”选项卡。

  2. 使用搜索框按角色，项目甚至特定权限过滤列表，以查找组是否具有对资源或操作的所需访问权限。

向组授予角色

您可以直接向组授予角色。授予组成员所有分配给该角色的访问权限。

要将个人角色分配给组：

需要权限：读取用户基本，读取角色，读取组，更新项目

  1. 在管理菜单的访问管理部分，选择组 。

  2. 在组列表中，找到要为其分配角色的组，然后单击其名称以打开组配置文件。

  3. 选择角色选项卡。

  4. 单击授予角色按钮。

  5. 在“ 授予角色”对话框中，选择要分配给组的角色。

  6. 选择要在其中应用角色的项目。如果要将角色应用于YouTrack中的所有项目，请选择“ 全局”项目。

  7. 单击授予角色按钮。

    • 将所选角色授予该组，并将其所有权限授予该组成员的用户。

撤消组中的角色

如果一组用户不再需要角色授予的访问权限，则可以撤消角色分配。

要撤消组中的角色，请执行以下操作：

需要权限：读取用户基本，读取角色，读取组，更新项目

  1. 在管理菜单的访问管理部分，选择组 。

  2. 使用搜索框找到所需的组。

  3. 从列表中选择一个组。

  4. 选择角色选项卡。

  5. 找到要从选定组中删除的角色分配。使用搜索框按名称，项目或权限查找角色。

  6. 在角色分配下，选择在其中授予角色的项目。

  7. 单击撤消角色按钮。

  8. 在确认对话框中确认操作。

    • 该角色将从所选项目的组中撤消。

    • 组的成员将失去分配给项目中角色的权限。
