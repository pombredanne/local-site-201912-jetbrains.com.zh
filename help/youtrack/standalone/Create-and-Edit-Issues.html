创建和编辑问题

如果您的项目团队使用YouTrack，您可能要做的第一件事就是创建一个或两个问题。如果您已经阅读了快速入门指南 ，那么您已经知道它是多么容易。在此页面上，您将找到有关如何创建问题，添加附件以及更新现有问题的更多信息。

创建问题

每当您需要为自己或项目团队的其他成员报告问题或定义任务时，请为其创建问题。

要创建问题：

需要权限：创建问题，更新问题，添加附件

  1. 按Alt + Insert （ Control + N （适用于macOS），或按一下标题中的[ 新发行]按钮。

    • 将打开“ 新问题”页面：

  2. YouTrack通常将项目字段的值设置为您上次报告问题的项目。如果您当前正在使用项目作为搜索上下文，那么将使用该项目。

    要报告另一个项目中的问题，请从标题中的项目列表中选择另一个值。您已标记为收藏夹的项目显示在列表的顶部。

  3. 在相应的输入字段中输入问题摘要和描述。输入描述时，可以使用Markdown设置文本格式。您也可以使用YouTrack Wiki语法设置文本格式，但首选Markdown。

    Description字段下方的文本为您提供了使用Markdown语法的一些提示。要查看格式选项的完整列表，请单击Markdown链接以打开语法列表，然后单击帮助图标以打开“ YouTrack标记参考”页面。有关更多信息，请参见Markdown语法 。

  4. 将值分配给自定义字段。默认的自定义字段包括问题类型，优先级和子系统。提供以下选项：

    • 在字段面板中为每个字段选择值。

    • 打开“ 应用命令”对话框，并在单个命令中设置多个自定义字段的值。要打开“ 应用命令”对话框，请按Ctrl + Alt + J 。

  5. 要将文件附加到问题：

    • 单击附件面板以从本地目录中选择一个或多个文件。

    • 从计算机中拖动文件，然后将其放在附件面板中。

    • 按Ctrl + V从剪贴板粘贴图像。

    有关其他选项，请单击问题工具栏中的“ 显示更多”按钮。有关详细信息，请参阅下面的创建和编辑问题 。

  6. 请执行以下任一操作：

    • 要报告新问题，请按Ctrl + Enter或单击创建按钮。

    • 要关闭“ 新问题”页面而不报告问题，请按Esc或单击关闭 。当前问题另存为草稿。下次创建问题时，可以从“ 草稿”列表工具栏中重新加载草稿 。

附加文件或屏幕截图

有时，词语表达不了全部。要更详细地描述问题，您可以将资源（例如日志文件和屏幕截图）附加到该问题。

要将文件或屏幕快照附加到问题：

需要权限：读取问题，更新问题，添加附件

  1. 单击工具栏中的显示更多按钮。

  2. 选择以下选项之一：

    选项

    描述

    附加档案

    从本地目录中选择一个或多个文件，并将其附加到问题上。

    私下附加文件

    从本地目录中选择一个或多个文件，然后设置附件的可见性。附件包含应仅限于特定用户或组的信息时，请使用此选项。

    附加和编辑图像

    选择，拖放或粘贴图像，然后在图像编辑器中将其打开。有关更多信息，请参见编辑图像 。

有关将文件或屏幕快照附加到问题的更多信息，请参见附加文件 。

编辑现有问题

没有人是完美的。如果要重写问题摘要或描述，则可以随时更改它们。所有更改都会记录并显示在问题历史记录中。有些属性无法更新，例如问题ID，报告者和创建日期。这些属性由YouTrack保存，不能覆盖。

要编辑现有问题：

需要权限：读取问题，更新问题

  1. 在列表中选择一个问题，然后按F2或单击工具栏中的“ 编辑问题”按钮。

    • 该问题在编辑模式下打开。

  2. 更新项目，摘要或描述。您也可以为自定义字段设置新值。

  3. 完成后，单击“ 应用更改”按钮。

您还可以使用命令将更改应用于多个问题。有关更多信息，请参见命令 。
