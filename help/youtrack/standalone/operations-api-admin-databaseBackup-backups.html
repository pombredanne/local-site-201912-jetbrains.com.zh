使用特定BackupFile的操作

此资源使您可以访问数据库备份文件。执行该操作的用户帐户必须具有“ 低级读取管理”权限。

资源资源
/ api / admin / databaseBackup / backups / {fileID}

退货实体

BackupFile 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的BackupFile 。

BackupFile属性

表示备份文件。

下表描述了BackupFile实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。
