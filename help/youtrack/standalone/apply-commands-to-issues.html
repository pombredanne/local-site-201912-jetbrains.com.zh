将命令应用于问题

“ 应用命令”对话框使您可以快速更新问题

问题列表使用较旧版本的“ 应用命令”对话框。

该对话框的较新版本显示在敏捷板，单个问题视图和“ 问题”列表的实验版本上。

这两个对话框通过一组不同的控件支持相同的功能。

每个命令由一个以空格分隔的关键字/值对组成。例如：

  • 要将选定的问题或一组问题分配给自己，请输入for me 。

  • 要将问题分配给自己并将其状态更改为“进行中” ，请输入for me state in progress 。

一旦您开始键入，YouTrack就会显示带有匹配关键字或值的完成列表。对于引用其他问题的链接命令，YouTrack支持全文本搜索。此功能使您可以筛选问题列表以匹配您的搜索条件，而不必指定确切的问题ID。例如，您可以输入duplicates toolbar仅显示包含“工具栏”一词的问题，然后从列表中选择一个问题。

将命令应用于问题

使用命令，可以快速有效地将更改应用于多个问题。

所需的权限因要应用的命令类型而异。

要将命令应用于问题：

需要权限：读取问题，更新问题，更新观察者，创建评论，创建标签或保存的搜索，链接问题，更新工作项

  1. 在列表中选择一个或多个问题。要选择多个问题，请使用箭头键和空格键或按住Shift然后按Up Arrow要么Down Arrow 。

  2. 按Ctrl + Alt + J或任何字母键。

    • 将打开“ 应用命令”对话框。

  3. 在命令输入字段中，输入一个或多个命令。

  4. 在“ 注释”文本框中输入可选注释。

  5. 单击“ 应用”按钮或按Ctrl + Enter 。

    • 将应用命令，并将注释添加到选定的问题。

    要放弃命令和注释，请单击“ 取消”按钮或按Esc 。

有关可应用于问题的命令列表，请参见《 命令参考》 。

静默应用命令

当您静默应用命令时 ，不会通知用户“ silent”命令所做的更改。

当您一次将批量更改应用于多个问题并且不想向团队发送垃圾邮件时，此功能非常有用。例如，当您将问题移至下一发行版本或将问题添加到积压中时，您可以选择静默应用命令。

仅当您有权在项目中静默应用命令时，此选项才可见。要执行此操作，您需要将命令应用于问题的所有权限，以及“ 更新项目”权限。

应用命令的过程因您在YouTrack中应用命令的位置而异。问题列表使用较旧版本的“ 应用命令”对话框。对话框的较新版本显示在敏捷板和单个问题视图上。

要从“问题”列表中静默应用命令：

需要权限：读取问题，更新问题，更新观察者，创建评论，创建标签或保存的搜索，链接问题，更新工作项

  1. 在列表中选择一个或多个问题，或在单个问题视图中打开一个问题。

  2. 按Ctrl + Alt + J或任何字母键。

    • 将打开“ 应用命令”对话框。

  3. 在命令输入字段中，输入命令。

  4. 在“ 注释”文本框中输入可选注释。

  5. 单击“ 静音应用”按钮或按Ctrl + Shift + Enter （Windows）/ Control + Shift + Return （苹果系统）。

    • 将应用该命令，并将注释以静默方式添加到所选问题。

要将命令静默应用于单个问题：

需要权限：读取问题，更新问题，更新观察者，创建评论，创建标签或保存的搜索，链接问题，更新工作项

  1. 在单个问题视图中打开一个问题。

  2. 按Ctrl + Alt + J或任何字母键。

    • 将打开“ 应用命令”对话框。

  3. 在命令输入字段中，输入命令。

  4. 在评论字段中输入可选评论。

  5. 单击“ 应用，恕不另行通知”按钮或按Ctrl + Shift + Enter （Windows）/ Control + Shift + Return （苹果系统）。

    • 将应用该命令，并将注释以静默方式添加到所选问题。

要在敏捷板上静默应用命令：

需要权限：读取问题，更新问题，更新观察者，创建评论，创建标签或保存的搜索，链接问题，更新工作项

  1. 选择板上的一个或多个问题。

  2. 按Ctrl + Alt + J或任何字母键。

    • 将打开“ 应用命令”对话框。

  3. 在命令输入字段中，输入命令。

  4. 在评论字段中输入可选评论。

  5. 单击“ 应用，恕不另行通知”按钮或按Ctrl + Shift + Enter （Windows）/ Control + Shift + Return （苹果系统）。

    • 将应用该命令，并将注释以静默方式添加到所选问题。

样例命令

下表提供了一些常见的YouTrack命令示例：

命令

描述

assigned to me priority major
要么
for me priority major

将选定的问题分配给当前用户，将每个问题的优先级设置为Major 。

state fixed

将所选问题的“ 状态”字段的值设置为固定 。

duplicates TS-2405

在选定的问题中将重复项添加到ID为TS-2405的问题中。还为ID为TS-2405的每个问题添加了一个“按链接复制” 。

type Cosmetics tag fix it this week

将选定的问题与“本周修复”标签相关联，并将“ 类型”字段的值设置为Cosmetics 。

remove tag fix it this week

从所选问题中删除“本周修复”标签。

project TeamCity

将选定的问题移至TeamCity项目。

subsystem Documentation

将选定的问题移至“ 文档”子系统。
