YouTrack Standalone 2019.2帮助

配置项目

创建项目时，YouTrack将打开到项目概述页面。从“ 项目”列表中选择一个项目时，也会显示此页面。

项目概述页面使您可以快速访问项目管理页面。

项目团队

创建项目时，将自动创建项目团队。创建项目的用户将自动添加到团队中。添加到项目团队的用户和组被授予项目中的开发人员角色。有关更多信息，请参见管理项目团队 。

当您向项目团队添加用户和组时，这些更改将与项目中“ 受让人”字段的值集同步。

  • 添加到项目团队的单个用户将添加到项目中“ 受让人”字段的值集。

  • 添加到项目团队的组也将添加到项目中“ 受让人”字段的值集。可以将组中的每个成员设置为项目中问题的受让人。

从项目团队中删除的用户和组不会从“ 受理人”字段的值集中删除。有关更多信息，请参见管理受让人 。

访问项目设置

YouTrack项目是高度可定制的。为了帮助您确保项目支持团队的需求和业务逻辑，您可以从用户界面中的各个位置访问项目设置。这些导航选项仅对在项目中具有“ 更新项目”权限的用户可见。

可从以下位置获得项目设置：

  • 在项目列表中，单击设置图标，然后选择出现在菜单的“ 管理”部分中的任何链接。

  • 在项目概述页面上，单击侧栏的“ 管理”部分中的任何链接。

  • 在单个问题视图中，单击字段面板中项目名称旁边的设置图标，然后选择菜单的“ 管理”部分中出现的任何链接。

常规项目设置

常规选项卡显示定义项目的设置。在这里，您可以编辑设置以添加或更改在创建项目时存储的信息。

您还可以设置用于发送此项目中问题通知的电子邮件地址。

设置

描述

名称

项目的名称。项目名称必须唯一。

ID

项目ID。此值用作分配给该项目的所有问题中问题ID的前缀。项目ID必须是唯一的。

更改项目ID时，YouTrack会将以前用作问题前缀的ID存储在项目中。值的此集合显示在“ 历史ID”设置中，该设置为只读。使用历史项目ID的问题的外部链接将重定向到使用当前ProjectID的问题。

仅当项目具有至少一个历史ID时，“ 历史ID”设置才可见。

商标

存储用作项目徽标的图像。这有助于在“ 项目”列表中区分不同的项目 。

  • 单击图标上传项目的自定义徽标。

  • 单击重置为默认按钮以还原自动生成的图像。

自定义项目模板

启用后，当前项目将用作新项目的自定义模板。该项目将添加到“ 创建项目”页面上的可用项目类型列表中。有关更多信息，请参见定制项目模板 。

标记为自定义项目模板的项目无法以与活动项目相同的方式访问。该项目在搜索上下文中不可用，无法在搜索查询或命令中引用，并且在创建新问题的列表中也不可用。此行为类似于已归档的项目。

如果您有一个活动项目要用作自定义项目模板，则可以先对其进行克隆，然后将克隆的项目标记为模板。要了解如何克隆项目，请参阅克隆项目 。

描述

可选描述，用于解释项目旨在组织或完成的任务。此信息显示在项目概述页面上。您可以使用Markdown格式化项目描述。

此设置支持CommonMark规范中描述的标准Markdown实现。问题字段中支持的许多扩展名在这里不可用。有关详细信息，请参见Markdown语法 。

项目拥有者

负责项目的用户。创建项目的用户将自动设置为项目所有者。

要更改项目所有者，请从下拉列表中选择另一个用户。

发件人地址

用于发送有关此项目中问题的通知的电子邮件地址。这将设置"From:"电子邮件标题中的字段。您可以使用以下格式sender name 或输入不包含发件人姓名的电子邮件地址。如果您未设置值，则会从在YouTrack中设置为全局默认值的地址发送通知。

对于YouTrack InCloud，仅当实例连接到自定义邮件服务器时，此设置才可见。

有关更多信息，请参见发件人和答复地址 。

回覆地址

电子邮件回复发送到的可选地址。这将设置"Reply-To:"由YouTrack发送的邮件的电子邮件标题中的字段。例如，当您用于发送电子邮件通知的地址无法收到回复时，请使用此设置。

如果未设置值，则会将答复发送到发送消息的地址。

  • 如果您在项目中设置“ 发件人”地址的值，则会将答复发送到该地址。

  • 如果将“ 发件人”地址保留为空，则将答复发送到在YouTrack中设置为全局默认地址的地址。

其他项目设定

在“ 编辑项目”页面上，您可以访问项目的所有设置，包括自定义字段，时间跟踪选项，通知模板，集成和工作流。这些设置组织在特定于每个功能的选项卡上。

标签

描述

访问

管理特定用户和组的项目访问权限。有关更多信息，请参见管理项目访问 。

球队

将用户或组添加到项目团队。有关更多信息，请参见管理项目团队 。

领域

自定义项目中的字段和值。YouTrack提供了一组预定义的默认字段，例如State ， Priority和Assignee 。这些字段会自动附加到在YouTrack中创建的任何新项目。您可以编辑或删除大多数预定义字段，也可以创建和附加新的自定义字段。有关更多信息，请参见管理自定义字段 。

VCS

将项目与VCS存储库（例如GitHub，GitLab或BitBucket）集成。有关更多信息，请参阅连接到VCS 。

通知模板

配置项目的通知模板。有关更多信息，请参见编辑通知模板 。

构建服务器集成

选择一个现有连接或向TeamCity服务器添加新连接。您可以配置项目和TeamCity配置之间的映射。有关更多信息，请参见与TeamCity集成 。

时间跟踪

为项目启用时间跟踪，以使您的团队记录问题的工作，跟踪问题的进度并创建时间报告。有关更多信息，请参阅启用时间跟踪 。

工作流程

将工作流程附加到项目。与字段一样，您具有一组预定义的默认工作流程。某些工作流程会自动附加到新项目。在“ 工作流程”选项卡中，您可以查看附加的工作流程列表，分离不必要的工作流程以及激活或停用工作流程规则。有关更多信息，请参见管理工作流 。
