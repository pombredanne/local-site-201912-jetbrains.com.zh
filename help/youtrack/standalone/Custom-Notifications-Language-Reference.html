指令

您使用FTL标记来调用指令。除了内置指令如#list指令（ 内置指令 ），您可以在Freemarker中定义自己的指令。有关更多信息，请参见（《 用户定义的指令手册》 ）。

  • Wiki使用Wiki语法和关键字处理嵌入式文本。指令应用html转义。

    • 参数没有参数

    • 正文处理

例：

@wikisolidhtml <#escape x as x?html> ... <@wiki> <#noescape> $ {comment.text} ...
  • diff呈现文本更改的html元素。用给定的样式标记已删除和已添加的文本部分。
    • 参数：

      • oldText旧版本的文本

      • newText文本的新版本

      • insStyle html文本插入部分的样式

      • 删除部分文字的delStyle html样式

    • 身体没有身体

例：

@diffsolidhtml <@diff oldText=change.before.summary newText=change.after.summary insStyle="background-color: #E6FFE6;" delStyle="background-color: #FFE6E6;"/>
  • stacktrace呈现表示Throwable stacktrace的html组件
    • 参数：

      • Throwable Java类的throwable实例

    • 身体没有身体

例：

@stacktracesolidhtml
<@stacktrace throwable=exception/>
