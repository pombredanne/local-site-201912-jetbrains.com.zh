AOL Auth模块

该身份验证模块使用户可以使用其AOL帐户登录到YouTrack。

启用AOL身份验证

要允许用户使用其AOL帐户登录到YouTrack，请启用AOL身份验证模块。

要启用AOL身份验证：

需要权限：低级管理员写

  1. 在管理菜单的访问管理部分，选择身份验证模块 。

  2. 从添加模块下拉列表中，选择AOL（OpenID） 。

    • AOL身份验证模块是自动创建，启用和配置的。

    • “身份验证模块”页面显示AOL身份验证模块的设置。有关此页面上设置的更多信息，请参阅设置 。

    • 按钮图像设置中存储的图标将添加到登录对话框窗口中。用户可以单击此图标以使用其AOL帐户进行身份验证。

设定值

将模块添加到YouTrack后，将自动配置设置。您不必更改这些设置。要配置用于定义YouTrack如何使用AOL凭据对待新用户帐户的选项，请参阅其他设置 。

领域

描述

类型

显示在YouTrack中启用了第三方身份验证的应用程序或服务的名称。

名称

存储认证模块的名称。使用此设置可以将该模块与“身份验证模块”列表中的其他身份验证模块区分开。

按钮图片

显示用于用户单击以使用其在连接的授权服务中的帐户登录YouTrack的按钮的图像。您可以上传JPG，GIF或PNG文件。图像会自动调整为48 x 48像素。

服务器网址

在YouTrack中存储用于对登录请求进行身份验证的OpenID提供程序的URL。

架构图

使用以下设置将使用OpenID提供程序存储的用户数据映射到YouTrack中的用户帐户。

领域

描述

电子邮件

存储包含电子邮件地址的位置。

电子邮件验证状态

当验证服务未返回电子邮件地址的验证状态时，确定应为YouTrack中的电子邮件地址设置哪个状态。

名字

存储包含用户名字的位置。

姓

存储包含用户姓氏的位置。

全名

存储包含用户全名的位置。

头像

存储包含用户头像的位置。

其他设置

用户首次使用OpenID帐户登录YouTrack时，YouTrack会检查是否存在具有相同电子邮件地址的帐户。如果存在具有相同电子邮件地址的帐户，则AOL凭据将添加到现有帐户中。

使用以下设置来配置YouTrack如何管理用户帐户。这些设置还可以减少空闲连接消耗的处理资源的损失。

选项

描述

用户创建

为使用存储在连接的授权服务中的帐户登录的未注册用户创建YouTrack帐户。YouTrack使用电子邮件地址来确定用户是否已有帐户。

自动加入群组

当用户使用存储在连接的授权服务中的帐户登录时，将用户添加到组中。您可以选择一个或多个组。自动加入组的新用户将继承分配给该组的所有权限。
我们建议您将用户添加到至少一个组中。否则，仅授予新用户当前分配给“ 所有用户”组的权限。

连接超时

设置等待建立与授权服务的连接的时间。默认设置为5000毫秒（5秒）。

读取超时

设置等待从授权服务读取和检索用户配置文件数据的时间。默认设置为5000毫秒（5秒）。

审计

链接到YouTrack中的“ 审核事件”页面。在这里，您可以查看已应用于此身份验证模块的更改列表。
