YouTrack Standalone 2017.2 Help

Work with YouTrack on a Mobile Device

YouTrack provides two options that let you view and update issues on the go.

  • Download a dedicated app for iOS and Android platforms.

  • Open YouTrack in your mobile browser.

Whichever option you choose, tracking issues on your mobile device is comfortable and easy.

YouTrack Mobile

YouTrack Mobile is available for download as a native app for iOS and Android. Each view is optimized for use on a mobile device. YouTrack Mobile lets you perform the following actions with your phone or tablet:

  • Edit issue details by using simple wiki markup

  • Update field values on the fly

  • View tags, issue links, and attachments

  • Add attachments

  • Add comments

To start using YouTrack Mobile:

  1. Download and install the app.

  2. Open the app on your mobile device.

  3. Enter the URL of your YouTrack server.

  4. Enter your username and password, then tap the Log in button. Alternatively, tap Log in via Browser. This option opens the log in page in your mobile browser. If you are already logged in to YouTrack, you are authenticated and redirected to YouTrack Mobile.

    • YouTrack Mobile opens to the issues list.

The following views and actions are available in YouTrack Mobile (as shown on iOS).

View

Actions

Issues

  • View a list of issues. The issues are filtered to match the search criteria that you enter in the search box.

  • Tap Menu to access the Menu view.

  • Tap Create to access the New Issue view.

  • Tap an issue to open it in the View Issue view.

Edit Search

  • Tap the search box to edit the query.

  • Use the keypad to edit the search criteria. Query completion options are displayed in the content area.

  • Swipe up or down to scroll through the list.

  • Tap to select a search parameter.

  • Tap the Search button to apply the filter.

Menu

  • Log out of YouTrack Mobile.

New Issue

  • Use the keypad to enter a summary and description.

  • Tap Create to report the issue.

View Issue

  • View the full issue summary, description, comments, and field values.

  • Tap Back to return to the Issues view.

  • Tap Actions to open the Actions view.

  • Tap more to display detailed values for the Created by and Updated by fields.

  • Tap the Comment icon to open the Add Comment view.

  • Tap a custom field to open the Update Custom Fields view.

  • Swipe left to show additional custom fields.

Actions Menu

  • Access options to edit the issue summary and description, copy the issue URL, or attach an image from your mobile device.

  • Tap the Cancel button to return to the issue.

Add Comment

  • Use the keypad to add a comment to the current issue.

  • Tap Send to add the comment.

Update Custom Fields

  • Tap a custom field to change its value. For enumerated fields, possible values are displayed in the content area.

  • Swipe up or down to scroll through the list.

  • Use the search box to filter the list of values.

  • Tap to select a value.

Responsive Design for Mobile Browsers

The login page and full issue view are customized for mobile browsers. Mobile browser support lets you work with issues on platforms that are not supported by YouTrack Mobile, like Blackberry and Windows Mobile.

With the mobile UI, you can easily perform common tasks. The following views are optimized for a mobile device (as shown on iOS).
