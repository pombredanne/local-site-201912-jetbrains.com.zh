Localization

The language setting for your YouTrack server is set system-wide. When you write workflow rules that include notifications and messages, simply enter the text in the language that is used for your system. For example, if your system language is French, you set the notification that is displayed for a required field as follows:

Due Date.required("S'il vous plaît préciser la date");

The default workflows that are bundled with your YouTrack installation have been translated in all supported languages. These workflows use the l10n expression to create a localized text string. This expression references an ID that is used to display localized strings based on the default language that is used in the system. The ID is displayed in the Inspector panel of the workflow editor.

If you want to modify the text that is used in a default workflow, remove the l10n expression and replace the localized text string with a static string literal in the desired language. This removes the reference to the ID that displays the localized string and lets you set the text directly.
