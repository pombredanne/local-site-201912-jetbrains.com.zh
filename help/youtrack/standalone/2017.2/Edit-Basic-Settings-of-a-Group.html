Edit Basic Group Settings

The basic settings of a user group are displayed on the Settings tab.



Per-project:

  • Read User Group

  • Update User Group

Here, you can edit the following settings:

  • The name of the group.

  • The project to which the group belongs.

  • The image that is used as the avatar for the group.

  • The description of the group.

You can also enable the Auto-join option. When enabled, new users are added to this group automatically.
