GET Issue Link Type

GET /admin/issueLinkType/{name}

Get issue link type by its name.

GET /rest/admin/issueLinkType/{name}

Request parameters:

Name

Type

Description

name

string

issue link type name

Response parameters:

Name

Description

name

Name of issue link type.

outwardName

Label, that outward link will have.

inwardName

Label, that inward link will have.

directed

Whether this link is directed or not.

Sample

Request:

GET /rest/admin/issueLinkType/Depend

Response:

<issueLinkType name="Depend" outwardName="depends on" inwardName="is required for" directed="true"/>
