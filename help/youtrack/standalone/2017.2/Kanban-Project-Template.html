Workflows

In addition to the default workflows, a workflow that is specific to the Kanban board is attached to the project automatically.

The Kanban workflow lets you change the Kanban Stage of a task only when the status for the task is Ready to pull. When the Kanban Stage changes, the status is set to Blocked automatically. For a detailed description of this workflow, see Kanban.
