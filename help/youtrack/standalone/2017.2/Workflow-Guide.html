Workflows

Workflows in YouTrack let you customize and automate the lifecycle of issues in your project. Use the following list of topics to help you locate key pieces of information in this section of the documentation.



From YouTrack 2017.2, you can write and edit workflows in JavaScript without leaving YouTrack. Schedule. Automate. Feel the power. This feature is available on an experimental basis. For more information, see Workflows in JavaScript.
