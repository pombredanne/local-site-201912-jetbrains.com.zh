Custom Workflow Repository

JetBrains has created a dedicated repository for custom workflows. This repository contains ready-to-use workflows that support a range of use cases. You can access the repository in GitHub.

Each workflow includes a description of each rule to help you find the right workflow for your projects.

Using a Workflow from the Repository

The YouTrack workflow repository contains a number of custom workflows that have been written by JetBrains developers and YouTrack users. These workflows support a wide range of use cases. While you're always welcome to write your own workflows, you might find exactly what you're looking for in the repository. You can also download a custom workflow from the repository and adapt it to meet your specific use case.

To use a workflow from the repository:

  1. Download the workflow ZIP file from the repository.

  2. Import the workflow to your YouTrack server and attach it to a project. For instructions, see Import and Export Workflows.

Several custom workflows have already been imported into our YouTrack InCloud test instance. Anyone can register to access this instance and use it as a sandbox. When you register your account, you are given the Global Project Admin role and can test custom workflows in any project.
