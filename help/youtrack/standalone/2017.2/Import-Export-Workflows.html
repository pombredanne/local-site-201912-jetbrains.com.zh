YouTrack Standalone 2017.2 Help

Import and Export Workflows

In YouTrack, you can import a workflow that has been modified in the YouTrack Workflow Editor or downloaded from our custom workflow repository in GitHub.

You can also export a workflow from the YouTrack server. Use this option when you have a workflow that you would like to import into another YouTrack instance. You can also export workflows and share them with the YouTrack support team for troubleshooting.

Workflows are imported and exported as ZIP files. Each ZIP file contains a single workflow.

Import a Workflow to YouTrack

If want to use a custom workflow from the repository, or a workflow that has been exported from another YouTrack server, you need to import the workflow to YouTrack. Project administrators can also import and attach workflows to the projects that they have permission to view and update. For more information, see Manage Workflows.

To import a workflow to the YouTrack server:



Global:

  • Low-level Administration

  1. In the Project-related Settings section of the Administration menu, select Workflows.

  2. Click the Import workflow button.

  3. Select the ZIP file of the workflow you would like to import.

  4. Click the OK button.

    The workflow is imported and can be attached to one or more projects.

Export Workflows from YouTrack

If you want to export a workflow for use in another YouTrack instance, or need to share a workflow with support, you can export it from your YouTrack server.

To export a workflow from YouTrack:



Global:

  • Low-level Administration

  1. In the Project-related Settings section of the Administration menu, select Workflows.

  2. Select one or more workflows from the list. To export multiple workflows, your browser must be configured to allow pop-ups for your YouTrack instance.

  3. Click the Export button.

    The selected workflows are exported from the server and downloaded to your local directory as ZIP files.
