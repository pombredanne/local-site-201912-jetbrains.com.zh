PractiTest Integration

The integration between YouTrack and the PractiTest QA & test management system is provided and supported by H.S. PractiTest. The integration has been verified by the YouTrack QA team.

Integration with PractiTest lets you create issues for failed test cases and add links to existing issues in YouTrack. More information about the integration and setup instructions are available on the YouTrack Integration page of the PractiTest website.

On the YouTrack side, the only thing you need to do is create a project for tracking test issues and configure your custom fields to store the values you want to assign.

Working with the PractiTest Integration

When you have configured the YouTrack integration, the following functionality is enabled in PractiTest:

  • When you encounter an issue during testing, click the Fail & Issue link. Your YouTrack instance opens in a new browser tab or window. If your browser is configured to remember your YouTrack username and password, you are taken directly to the New Issue page.

  • After you create the issue in YouTrack, enter the issue ID in the Issue ID field in PractiTest and click the Link existing issue button. You can enter multiple issue IDs separated by commas or spaces. The YouTrack issue is linked to the test step. You can click the issue ID in PractiTest to open the issue in YouTrack.
