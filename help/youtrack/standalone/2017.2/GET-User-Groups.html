GET User Groups

GET /admin/user/{login}/group

Get all groups the specified user participates in.

GET /rest/admin/user/{login}/group

Parameters:

Name

Type

Description

login

string

Login name of a user.

Sample

Request:

HTTP GET: /admin/user/root/group

Response:

<userGroupRefs> <userGroup name="All Users" url="http://localhost:8080/charisma/rest/admin/group/All Users"/> <userGroup name="test-group" url="http://localhost:8080/charisma/rest/admin/group/test-group"/> <userGroup name="some bla bla" url="http://localhost:8080/charisma/rest/admin/group/some bla bla"/> <userGroup name="very very very long group name" url="http://localhost:8080/charisma/rest/admin/group/very very very long group name"/> </userGroupRefs>
