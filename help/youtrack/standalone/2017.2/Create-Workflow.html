Create a Workflow

Before you can create or edit workflows, you need to download, install, and configure the YouTrack Workflow Editor.

For a complete overview of the process for creating and using a workflow, read the Workflow Tutorial. For detailed samples of workflow rules, see Default Workflows.

To create a workflow:



Global:

  • Low-level Administration

  1. Open the YouTrack Workflow Editor.

  2. Click the button to download the workflows and custom field schema from your YouTrack server.

  3. Create a new workflow and the rules associated with it.

  4. When done, click the button to upload the changes to your YouTrack server.

  5. In YouTrack, open the Administration > Workflow page and verify that the updated workflows have been uploaded.

    • The new workflows are available for use and can be attached to a project. For more information, see Attach Workflows to Projects.
