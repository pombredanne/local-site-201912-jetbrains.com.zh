YouTrack Standalone 2019.2帮助

上游整合

您可以将YouTrack和Upsource都连接到外部集线器服务。通过集线器连接YouTrack和Upsource并正确配置项目后，将启用以下功能：

  • 自动在讨论，代码审查和提交消息中添加指向YouTrack问题的链接。有关更多信息，请参见《 VCS提交中的链接问题》 。

  • 直接从Upsource中的代码审查中创建问题。

  • 查看有关修订的评论或以名称提及问题的评论。

  • 视图提交提到问题的地方。

  • 从VCS提交消息中应用YouTrack命令。有关更多信息，请参见《 VCS提交》中的“应用命令” 。

YouTrack Standalone 6.5+和Upsource 2.0.3+支持此集成。

上游集成不适用于YouTrack InCloud。

启用集成后，活动流上将显示提及YouTrack问题的来自Upsource的提交和代码审查：

有关更多信息，请参见查看和编辑VCS更改 。

连接YouTrack和上游

当您将YouTrack和Upsource都连接到外部集线器服务时，将自动启用Upsource集成。

您的特定设置要求取决于您当前的安装环境。大致过程如下：

  1. 安装和配置外部集线器服务 。

  2. 将您的YouTrack数据迁移到外部集线器服务 。

    安装YouTrack时，可以选择在安装过程中连接到外部集线器服务。

  3. 将您的上游数据迁移到外部集线器服务 。

    安装Upsource时，可以选择在安装过程中连接到外部集线器服务。

  4. 使用集线器中的项目向导可以：

    • 合并您的项目 。

    • 将YouTrack和Upsource项目作为资源添加到Hub项目 。

    如果在任何一项服务中都没有配置任何项目，请使用“项目向导”创建一个新项目，并将YouTrack和Upsource添加为服务。

有关将JetBrains Teamware连接到Hub的更多信息，请参考Hub文档 。

上游整合设定

当YouTrack和Upsource使用相同的外部Hub服务时，将启用Upsource集成，并且可以在任何项目中使用。到“ 上游集成设置”页面的链接已添加到“ 管理”菜单的“ 集成”部分。

此页面显示已作为服务添加到外部集线器服务的上行服务器的列表。该表显示了YouTrack项目的名称，Upsource项目以及每个已连接项目的状态。

此页面上提供以下控件：

控制

描述

拉取新数据

设置从Upsource提取新数据的时间间隔。选择预定义的间隔或指定自定义间隔。要指定自定义间隔，请选择“ 自定义”，然后输入cron表达式。计划不是为多个Upsource服务器独立设置的。相同的日程表将应用于与YouTrack服务器集成的所有Upsource服务器。

跑

从Upsource服务器为选定项目提取新数据。到Upsource的链接被添加到代码审查和提交消息中提到的YouTrack问题中。VCS提交消息中的命令将应用于相应的问题。同步状态在“集成设置”页面上的“ 状态”列中更新。

禁用

在选定的Upsource服务器或项目中禁用Upsource集成。在项目级别使用此选项可基于每个项目禁用集成功能。如果禁用集成，则“ 启用”选项可用。

添加SSL密钥

打开一个对话框，让您选择用于身份验证的SSL密钥。使用此选项可以在YouTrack服务器和Upsource服务器之间建立安全连接。

删除SSL密钥

消除对SSL密钥的依赖关系，以建立与Upsource服务器的连接。仅当SSL密钥已添加到Upsource服务器时，此选项才可用。
