Update an Issue

POST /issue/{issue}

Update summary and description for an issue specified by its issueID.

POST /rest/issue/{issueID}?{summary}&{description}

Parameters:

This method only lets you update the issue summary and/or description.

To update other issue attributes, use the method that applies a command to an issue. For more information, see Apply Command to an Issue.

Name

Type

Description

issueID

string

ID of an issue that should be updated.

summary

string

New summary for the specified issue.

description

string

Updated description for the specified issue.
