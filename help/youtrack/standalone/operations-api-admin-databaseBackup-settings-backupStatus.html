具有特定BackupStatus的操作

此资源使您可以获取备份过程的状态。

资源资源
/ api / admin / databaseBackup / settings / backupStatus

退货实体

BackupStatus 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的BackupStatus 。

BackupStatus属性

表示备份过程的当前状态。

下表描述了BackupStatus实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

backupInProgress

布尔型

如果true ，备份当前正在运行。除此以外， falseRead-only 。

backupCancelled

布尔型

如果true ，上次备份操作被取消。除此以外， false 。 Read-only 。

backupError

 BackupError

在上一个备份过程中出现的错误。退货null ，如果没有错误。 Read-only 。 Can be null 。

读取特定的备份状态

阅读备份过程的状态。

所需权限

需要低级管理员读取权限。

请求语法

GET / api / admin / databaseBackup / settings / backupStatus？{字段}

请求参数

参数

类型

描述

领域

串

响应中应返回的BackupStatus属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.com/youtrack/api/admin/databaseBackup/settings/backupStatus?fields=backupCancelled,backupError(date,errorMessage),backupInProgress,stopBackup

样品响应体

{“ backupInProgress”：错误，“ stopBackup”：错误，“ backupCancelled”：错误，“ backupError”：空，“ $ type”：“ BackupStatus”}
