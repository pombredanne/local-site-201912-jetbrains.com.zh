启动和停止JAR分发

本页介绍如何停止和启动YouTrack Standalone服务器。

以下情况要求您停止YouTrack：

  • 您要更改数据库的位置。

  • 您想要修改YouTrack Standalone服务器的属性或JVM选项。

  • 您要升级YouTrack Standalone安装。

开始发布JAR

要启动YouTrack JAR，请运行以下命令：

java -jar youtrack- .jar

停止分发JAR

要停止YouTrack JAR，请在控制台中按Ctrl+C或在单独的控制台窗口中运行以下命令：

java -jar youtrack- .jar stop
