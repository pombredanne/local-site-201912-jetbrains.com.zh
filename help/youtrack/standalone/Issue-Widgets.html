问题列表小部件

“ 问题列表”窗口小部件显示与搜索查询，保存的搜索，项目或标签匹配的问题列表 。

要将“问题列表”窗口小部件添加到仪表板：

  1. 单击仪表板上的添加小部件按钮。

  2. 从下拉列表中，选择问题列表 。

    • 带有标题“ YouTrack Issues List”的卡将添加到仪表板。

  3. >输入可选的小部件标题。

  4. 通过执行以下操作的组合来配置窗口小部件：

    • 从下拉列表中选择搜索上下文。

    • 在输入字段中输入搜索查询。

    • 在“ 保存的搜索”选项卡上选择一个保存的搜索。

    • 在“ 项目”选项卡上选择一个项目。

    • 在标签选项卡上选择一个标签。

  5. 点击保存按钮

    • “ 问题列表”窗口小部件已添加到仪表板。

    • 如果未指定标题，则卡的标题将设置为所选的搜索查询，保存的搜索，项目或标签。

当您单击窗口小部件的标题时，YouTrack将显示问题列表，其中包含为窗口小部件配置的搜索查询的结果。

刷新问题小部件

小部件内容每10分钟刷新一次。要手动刷新问题窗口小部件，请单击标题栏中的刷新图标。
