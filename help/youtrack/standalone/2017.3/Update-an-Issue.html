Update an Issue

POST /issue/{issue}

Update summary and description for an issue specified by its issueID.

POST /rest/issue/{issueID}?{summary}&{description}

Parameters:

This POST method allows updating issue summary and/or description, only. To update issue fields, please use the method described on the Apply Command to an Issue page.

Name

Type

Description

issueID

string

ID of an issue that should be updated.

summary

string

New summary for the specified issue.

description

string

Updated description for the specified issue.
