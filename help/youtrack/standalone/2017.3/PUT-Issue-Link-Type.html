PUT Issue Link Type

PUT /admin/issueLinkType/{name}

Create new issue link type

PUT /rest/admin/issueLinkType/{name}?{outwardName}&{inwardName}&{directed}

Parameters:

Name

Type

Description

name

string

Issue link type name

outwardName

string

Displayed name for an outward link

inwardName

string

Displayed name for inward link. Is not used for non-directed links

directed

boolean

Defines if links of this type are directed or not

Sample

Request:

PUT http://localhost:8080/rest/admin/issueLinkType/Displace?outwardName=displaces&inwardName=is%20displaced%20by&directed=true

Response:

Empty Response. Response code: 201 Response message: Created
