Custom Widgets

You can now upload custom widgets and add them to your dashboards. Extend the functionality of your dashboards to grab content from external sources or present data from connected services in a unique format.

Custom widgets help you build a consolidated overview of the information that is important to your team. For example:

  • Your Quality Assurance team uses dedicated software for testing. You can build a custom widget that displays a list of failing tests.

  • Your Customer Support Engineers use dedicated software for managing support requests. You can build a custom widget that displays the number of unanswered requests.

Manage the status of each widget in a central location on the Custom Widgets page. We've even built a playground where you can preview and test your widgets.

This feature is available on an experimental basis. To learn how to activate this feature in YouTrack, see Experimental Features.
