Delete Existing Work Item

DELETE /rest/issue/{issue}/timetracking/workitem/{workitemID}

Delete existing work item from specific issue.

DELETE /rest/issue/{issue}/timetracking/workitem/{workitemID}

Parameters:

Name

Type

Description

issue

issue id

Issue id to attach files to

workitemID

string

Unique ID of the existing workitem to edit.
