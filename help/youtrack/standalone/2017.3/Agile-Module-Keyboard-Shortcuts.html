Agile Board Keyboard Shortcuts

YouTrack is a keyboard-centric application and provides enough keyboard support to let you forget about using the mouse in most cases. For almost any operation, whether it's creating, editing or navigating between issues, commenting, changing attributes — YouTrack provides a shortcut.

This page lists the keyboard shortcuts that you can use while working with an agile board.

For the complete list of keyboard shortcuts, see Keyboard Shortcuts. You can also download the shortcuts list in PDF format from the YouTrack web site.
