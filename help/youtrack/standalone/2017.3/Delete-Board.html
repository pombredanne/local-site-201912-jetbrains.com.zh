Delete a Board

In some cases, you might decide to delete a board that you no longer use.

To delete a board:

  1. Open the board that you want to remove.

  2. Click the button to open Board Settings.

  3. Click the Delete board button in the General tab.

    You might want not to rush and do not delete the board, you can do it any time later. Instead, why don't you try to create a new board and see both boards in comparison? Then you can keep the one that you liked the most.

  4. Confirm this action in the dialog.

    • The board is deleted.

When you delete a board:

  • Any mention of the board is deleted from the system. It is not available in search queries, and deleted from the Boards list that is displayed for issues in the full-issue page.

  • All the sprints of the board are deleted. If sprints were linked to an issue field, existing values of the field's set of value are preserved.

  • The saved search that is used for the backlog is not deleted. However, its query becomes broken as the board mentioned in the query does not exist anymore.

  • All other saved searches that mention the board name and sprints require adjustment.
