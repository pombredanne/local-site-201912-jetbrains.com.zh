默认组

YouTrack中有两个默认组： 所有用户和注册用户 。这些组配置有以下默认设置：

名称

自动加入

角色

全部用户

已启用

没有

注册用户

已启用

观察者

全部用户

所有用户帐户将自动添加到“ 所有用户”组。默认情况下，此组在任何项目中均未分配任何角色。

  • 如果要对YouTrack中的所有用户启用特定级别的访问权限，则可以在一个或多个项目中向该组授予角色。

  • 要为所有项目中的所有用户授予特定级别的访问权限，可以在全局项目中向该组授予角色。

如果您不修改“ 注册用户”组的设置，则“ 所有用户”和“ 注册用户”组之间的唯一区别是访客帐户的成员身份。默认情况下，访客帐户不是“ 注册用户”组的成员。

作为管理员，您可以编辑分配给该组的角色和权限，但不能更改组名，禁用“ 自动加入”选项或删除“ 所有用户”组。

注册用户

创建用户帐户后，该帐户将自动添加到“ 注册用户”组中。默认情况下，在全局项目中为该组分配了“ 观察者”角色。

管理员可以编辑分配给该组的角色和权限。与“ 所有用户”组不同，管理员可以修改该组的任何设置，禁用“ 自动加入”选项并删除该组。

例如，如果禁用“ 注册用户”组的“ 自动加入”选项，则新帐户仅添加到“ 所有用户”组。
