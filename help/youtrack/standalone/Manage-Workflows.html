管理工作流程

通过YouTrack中的工作流，您可以自定义和自动化项目中问题的生命周期。作为项目管理员，您可以将现有工作流程附加到项目中。您可以在“ 编辑项目”页面的“ 工作流”选项卡上访问这些设置。

附加工作流程时，可以启用和禁用项目的特定规则。这些设置不会影响系统中的任何其他项目。

系统管理员可以创建工作流程并将其附加到系统中的任何项目中。作为具有“ 读取项目”和“ 更新项目”权限的用户，您还可以访问“ 工作流程”管理页面并创建自己的工作流程。有关更多信息，请参见工作流 。

将工作流程附加到您的项目

在使用工作流程之前，需要将其附加到项目中。

附加工作流程：

需要权限：更新项目

  1. 单击主导航中的“ 项目”链接。

  2. 选择要在其中应用工作流程的项目。

  3. 单击编辑项目链接。

  4. 选择工作流程选项卡。

  5. 单击附加工作流程按钮。

  6. 从列表中选择工作流程。

导入工作流并将其附加到项目

您也可以导入工作流程并将其附加到项目中。对于此过程，您将使用一个ZIP存档，其中包含要在项目中使用的工作流程。这可以是用外部JavaScript编辑器编写的工作流，也可以是从“ 自定义工作流存储库”中下载的工作流 。

从YouTrack 2018.3开始，您将无法再导入在外部YouTrack工作流编辑器中编写的工作流。有关更多信息，请参见将旧工作流程转换为JavaScript 。

导入工作流程并将其附加到项目中：

需要权限：更新项目

  1. 单击主导航中的“ 项目”链接。

  2. 打开您要附加工作流的项目。

  3. 单击编辑项目链接。

  4. 选择工作流程选项卡。

  5. 单击导入工作流按钮。

  6. 选择您要导入的工作流程的ZIP文件。

  7. 单击确定按钮。

    工作流将导入到所选项目中。

停用和激活工作流程规则

将工作流程附加到项目时，所有规则均处于活动状态并已启用。您可以通过禁用和启用特定规则来自定义项目中的工作流程。

要查看工作流程的规则列表，请单击以展开工作流程。已停用的规则将变灰并标记为“不活动” 。

要停用项目中的单个规则：

需要权限：更新项目

  1. 选择“ 编辑项目”页面上的“ 工作流程”选项卡。

  2. 展开工作流程以显示规则列表。

  3. 选择要停用的规则。

  4. 单击停用按钮。

禁用工作流程中的所有规则时，工作流程将自动从项目中分离。

要激活项目中的单个规则：

需要权限：更新项目

  1. 选择“ 编辑项目”页面上的“ 工作流程”选项卡。

  2. 展开工作流程以显示规则列表。

  3. 选择您要激活的规则。

  4. 单击激活按钮。

分离工作流程

如果您不再想要使用工作流程，则可以将其与项目分离。

分离工作流程：

需要权限：更新项目

  1. 选择“ 编辑项目”页面上的“ 工作流程”选项卡。

  2. 选择要分离的工作流程。

  3. 单击分离按钮。

或者，您可以禁用工作流程中的所有规则。工作流程会自动从项目中分离。
