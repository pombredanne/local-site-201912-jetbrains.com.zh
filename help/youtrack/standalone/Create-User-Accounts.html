创建用户帐户

在YouTrack中，可以通过多种方式创建用户帐户：

  • 系统管理员可以手动创建帐户。

  • 系统管理员可以邀请人员进行注册。

  • 用户可以注册自己的帐户。

  • 用户可以使用来自已连接服务的凭据登录到YouTrack。

向YouTrack添加新用户帐户的最常见方法是启用外部身份验证模块。身份验证模块使用户可以使用他们在第三方应用程序中管理的凭据登录到YouTrack。每个身份验证模块都可以让您自动为使用外部凭据登录的未注册用户创建YouTrack帐户。您还可以将新用户添加到特定组中，以自动授予访问权限并为连接的服务分配许可证。有关更多信息，请参阅身份验证模块 。

本页介绍了可用于在YouTrack中创建用户帐户的所有选项。

创建一个用户帐号

要提供对单个用户的访问权限，请在YouTrack中创建一个新的用户帐户。

手动创建用户帐户：

需要权限：“读取用户基本信息”，“创建用户”

  1. 在管理菜单的访问管理部分，选择用户 。

  2. 单击新用户按钮。

  3. 在“ 新用户”对话框中，单击“ 创建用户”按钮。

    • 该对话框显示用于创建新用户帐户的输入字段。

  4. 输入以下设置的值：

    设置

    描述

    全名

    设置新用户的全名。

    电子邮件

    输入新用户的电子邮件地址。该电子邮件地址必须有效。

    密码

    输入新用户帐户的密码。

    确认

    确认新用户帐户的密码。

    强制更改密码

    选择此选项可强制用户更改密码。下次使用这些凭据登录时，系统会提示用户更改密码。

  5. 单击创建按钮。

    • 创建具有指定设置的用户帐户。

    • 新用户的个人资料页面打开。在这里，您可以配置用户帐户的访问权限。

创建帐户后，需要确保用户有权访问项目以及他们与YouTrack一起使用所需的权限集。

  • 选择“ 组”选项卡，然后将用户添加到一个或多个组。有关更多信息，请参见添加和删除成员 。

  • 选择“ 角色”选项卡，并向用户授予组或项目中的一个或多个角色。有关更多信息，请参见管理用户访问 。

邀请新用户注册

在YouTrack中创建用户帐户的一种方法是邀请新用户注册自己的帐户。

邀请用户注册时，实际上是为新用户创建一个帐户和个人资料。受邀用户可以确认帐户并设置自己的密码。

创建帐户的事实使您可以作为管理员在发送邀请之前配置访问权限。

您必须先在YouTrack中启用电子邮件通知，然后才能邀请用户注册。有关更多信息，请参阅全局设置 。

要邀请一个或多个用户在YouTrack中注册帐户，请执行以下操作：

需要权限：“读取用户基本信息”，“创建用户”

  1. 在管理菜单的访问管理部分，选择用户 。

  2. 单击新用户按钮。

  3. 在“ 新用户”对话框中，单击“ 邀请用户”按钮。

    • 该对话框显示一个输入字段，用于输入新用户的电子邮件地址。

  4. 在“ 电子邮件”输入字段中，输入要邀请其在Hub中注册帐户的用户的电子邮件地址列表，以空格分隔。

  5. 单击邀请按钮。

    • 邀请被发送到每个被邀请用户的电子邮件地址。邀请中包含指向登录页面的链接，用户可以在其中设置密码并访问集线器。

允许用户自行注册

要让用户在YouTrack中注册自己的帐户，请在集线器身份验证模块中启用注册。

要配置集线器身份验证模块：

需要权限：低级管理员写

  1. 在管理菜单的访问管理部分，选择身份验证模块 。

  2. 从列表中选择集线器模块。

  3. 将注册选项设置为已启用 。

    • 显示了可用于管理在YouTrack中注册自己的帐户的用户的其他设置。

    • 用户可以在YouTrack中创建自己的帐户。

为了防止您的YouTrack服务器受到机器人的攻击，您可以在注册页面上启用reCAPTCHA。

要在注册页面上启用reCAPTCHA：

需要权限：低级管理员写

  1. 将reCAPTCHA选项设置为Enabled 。

    • 显示了reCAPTCHA设置的其他设置。

  2. 单击链接以访问reCAPTCHA密钥管理控制台。

    • 将打开“ 创建reCAPTCHA密钥”页面。

  3. 使用reCAPTCHA服务注册您的YouTrack域。 reCAPTCHA是Google服务的一部分，因此您可以使用您的Google帐户登录。在注册并生成密钥之前，请阅读页面上提供的提示。

  4. 复制Google中的站点密钥 ，并将其粘贴到集线器身份验证模块中的相应输入字段中。

  5. 在Google中复制密钥 ，然后将其粘贴到集线器身份验证模块中的相应输入字段中。

  6. 单击保存按钮。

    • 为在YouTrack中注册自己的帐户的用户启用了reCAPTCHA验证输入。

启用使用来自外部身份验证提供程序的凭据登录

YouTrack支持使用来自外部身份验证源（例如LDAP或各种OpenID提供程序）的帐户授权用户。要使用户能够使用来自外部提供商的帐户登录，必须启用并配置相应的身份验证模块。

文档的“身份验证模块”部分介绍了受支持的身份验证模块的设置说明。
