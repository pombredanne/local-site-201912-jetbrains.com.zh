LDAP认证模块

LDAP认证模块允许用户使用存储在目录服务中的凭据登录到YouTrack。此身份验证模块已为LDAP预先配置。您可以将模块配置为使用标准LDAP方案或基于SSL的LDAPS。

LDAP认证模块不会从目录服务中导入所有用户帐户。仅当未注册的用户首次登录YouTrack时，YouTrack才会创建一个用户帐户。

启用LDAP身份验证后，YouTrack会为每次登录尝试检查目录服务。从目录服务中删除的用户无法登录到YouTrack。

先决条件

如果要通过SSL连接到目录服务，请在启用身份验证模块之前为LDAPS服务器导入受信任的SSL证书。如果SSL证书和根CA证书之间有任何中间证书，则需要上载包含完整证书链的文件。

LDAP认证模块的设置中不支持导入受信任的SSL证书的选项。相反，您需要访问SSL证书页面并将其导入。有关更多信息，请参见SSL证书 。

启用LDAP认证

要允许存储在目录服务中的用户登录到YouTrack，请启用LDAP身份验证模块。

要启用LDAP身份验证：

需要权限：低级管理员写

  1. 在管理菜单的访问管理部分，选择身份验证模块 。

  2. 从添加模块下拉列表中，选择LDAP 。

    • 侧栏中将打开“ 新模块”对话框。

  3. 在“ 新建模块”对话框中，输入以下设置的值：

    设置

    描述

    名称

    输入身份验证模块的名称。

    服务器

    输入目录服务的服务器地址。对于通过SSL的连接，请将地址的协议部分更改为ldaps 。

    港口

    输入用于与目录服务通信的端口号。

    • 标准LDAP的默认端口是389 。

    • LDAPS的默认端口是636 。

    SSL协议

    选择此选项可通过SSL连接到目录服务。在建立安全连接之前，您需要为LDAPS服务器导入受信任的SSL证书。有关说明，请参阅SSL证书 。

    搜索库

    输入定义用于存储用户帐户的顶级LDAP DN的域组件。例如，如果您的公司使用域mycompany.com ，输入顶级LDAP DN dc=mycompany,dc=com 。

    在此字段中输入的值将添加到LDAP URL中，并且不能包含不安全的字符。

    如果您使用组织单位来管理用户，请为每个组织创建单独的身份验证模块。在搜索库中包括组织单位，以为每个模块创建唯一的LDAP URL。LDAP认证模块不支持LDAP树中的递归搜索。

  4. 单击创建按钮。

    • LDAP认证模块已启用。

    • “ 验证模块”页面显示LDAP验证模块的设置。该模块已预先配置了标准设置，这些设置基于您在“ 添加模块”对话框中提供的信息。有关此页面上设置的更多信息，请参阅设置 。

测试与目录服务的连接

要验证LDAP身份验证模块已连接到您的目录服务，请测试连接。

要测试连接：

  1. 单击测试登录按钮。

  2. 在“ 测试身份验证”对话框中，输入存储在目录服务中的用户的凭据：

    • 在登录字段中，输入domainusername 。

    • 在密码字段中，输入password 。

  3. 单击测试登录按钮。

    • YouTrack在目录服务中搜索指定的用户帐户。如果找到用户，则显示成功通知。如果出现错误，请检查用户凭据和服务器URL。

设定值

使用以下设置来配置YouTrack如何连接到目录服务。

领域

描述

类型

显示在YouTrack中启用了第三方身份验证的目录服务的类型。

名称

存储认证模块的名称。使用此设置可以将该模块与“身份验证模块”列表中的其他身份验证模块区分开。

认证方式

显示模块的当前状态。此状态指示模块当前是启用还是禁用。

服务器网址

在YouTrack中存储用于验证登录请求的目录服务的LDAP URL。

LDAP URL使用以下格式ldap://host:port/DN 。输入存储用户帐户的目录的完整专有名称（DN）。

SSL密钥

选择用于加密YouTrack与目录服务之间的连接的SSL密钥。

当您的LDAP服务需要可信证书来建立与另一服务的连接时，请使用此设置。此列表仅显示已导入到YouTrack中的密钥库。有关更多信息，请参见SSL密钥 。

绑定账户

确定哪个帐户用于LDAP绑定请求。有关更多信息，请参见绑定帐户选项 。

绑定DN

存储用于与目录服务绑定的值。有关更多信息，请参见绑定帐户选项 。

过滤

存储一个表达式，该表达式在LDAP服务中查找特定用户的记录。表达式中的替换变量将替换为在登录页面上作为用户名或电子邮件输入的值。

LDAP引荐

确定LDAP认证模块是忽略还是跟踪来自服务的请求以在LDAP目录中定位其他信息。

  • 如果目录服务将映射到集线器帐户的属性存储在不同的分区中，请启用“ 跟随”选项。

  • 如果用户身份验证不需要遵循重定向，请启用“ 忽略”选项以提高性能。

绑定帐户选项

您可以通过以下两种方式之一配置模块以执行与LDAP服务的绑定请求。所使用的方法取决于为“ 绑定帐户”设置选择的选项。

用于“ 绑定DN”设置的值取决于您为“ 绑定帐户”设置选择的选项。使用以下准则来设置“ 绑定DN”设置的值：

选项

描述

绑定DN设置指南

固定

使用固定帐户绑定到LDAP服务，并代表绑定用户搜索要认证的用户。使用此选项，您可以设置LDAP身份验证模块，并且仍然使用不属于专有名称（DN）的登录名，例如电子邮件地址或令牌。此方法通常也称为搜索+绑定或两步身份验证 。

若要使用此方法，您需要在目录服务器上具有一个特殊帐户，该帐户有权在目录服务中查找其他用户帐户。

输入要用于LDAP绑定请求的用户帐户的完整DN。该帐户必须具有在目录服务中查找其他用户帐户的权限。

使用“ 设置密码”控件可将此帐户的密码存储在YouTrack中。绑定用户的密码存储为纯文本值的哈希。

动态

从登录名导出用户DN，并尝试直接以用户身份绑定到LDAP服务。此方法通常也称为直接绑定 。

使用查询与目录服务绑定。该查询查找要认证的用户的专有名称。用表达式引用用户名。该表达式将替换变量映射到将用户名存储在目录服务中的属性。您选择的属性确定在过滤字符串中使用哪个查询。

在登录页面上作为用户名输入的值在替换替换变量之前会被修剪。如果用户指定域，则将其丢弃。例如，用户名的值为WORKGROUP\smith修剪到smith 。要指定域，请输入域名作为静态值。例如， WORKGROUP\%u 。

属性映射

当YouTrack在LDAP服务中找到与过滤器匹配的记录时，它将从为每个字段指定的LDAP属性中获取值，并将它们复制到YouTrack用户的相应集线器帐户中。使用以下设置来定义过滤条件，并将存储在目录服务中的属性映射到YouTrack中的中心帐户。

领域

描述

全名

映射到LDAP属性，该属性存储值以复制到中心帐户的“ 全名”字段。

登录

映射到LDAP属性，该属性存储要复制到中心帐户中“ 登录”字段的值。

电子邮件

映射到LDAP属性，该属性存储要复制到集线器帐户中“ 电子邮件”字段的值。

贾伯

映射到LDAP属性，该属性存储要复制到Hub帐户中的Jabber字段的值。

VCS用户名

映射到LDAP属性，该属性存储要复制到Hub帐户中的VCS用户名字段的值。

用户管理选项

以下选项位于页面底部。使用这些设置来管理帐户创建和组成员身份。

选项

描述

用户创建

为使用存储在连接的目录服务中的帐户登录的未注册用户启用YouTrack帐户的创建。YouTrack使用电子邮件地址来确定用户是否已有帐户。

所有LDAP身份验证模块都必须允许用户创建。如果拒绝用户创建，则向未注册的用户显示错误。

自动加入群组

当用户使用存储在连接的目录服务中的帐户登录时，将用户添加到组中。您可以选择一个或多个组。自动加入组的新用户将继承分配给该组的所有权限。
我们建议您将用户添加到至少一个组中。否则，仅授予新用户当前分配给“ 所有用户”组的权限。

连接设定

页面此部分中的设置可让您减少空闲连接消耗的处理资源的损失。

设置

描述

连接超时

设置等待建立与授权服务的连接的时间。默认设置为5000毫秒（5秒）。

读取超时

设置等待从授权服务读取和检索用户配置文件数据的时间。默认设置为5000毫秒（5秒）。

组映射

在“ 组映射”选项卡上，可以将LDAP服务中的现有组映射到YouTrack中的组。

如果要将LDAP组映射到YouTrack组，则需要在此auth模块的设置的“ 属性映射”部分中指定存储LDAP组成员身份的Groups属性。

配置组映射后，当用户使用目录服务中管理的帐户登录时，YouTrack将检查LDAP组成员身份。YouTrack对映射到YouTrack组的每个LDAP组执行以下操作：

  • 属于映射的LDAP组成员而不是映射的YouTrack组成员的用户将添加到YouTrack中的组。

  • 不是映射的LDAP组成员的用户，而是映射的YouTrack组成员的用户将从YouTrack的组中删除。

仅当用户使用LDAP身份验证模块登录时，才在YouTrack中应用对LDAP服务中的组成员身份所做的更改。

您可以将多个LDAP组映射到YouTrack中的单个目标组。您不能将LDAP组映射到多个YouTrack组。

要将LDAP组映射到YouTrack中的组，请执行以下操作：

需要权限：低级管理员写

  1. 打开您的LDAP身份验证模块。

  2. 选择“ 组映射”选项卡。

  3. 单击添加映射按钮。

    • 将打开“ 添加映射”对话框。

  4. 在“ LDAP组名称”字段中输入LDAP组的名称 。

    • 如果“ 绑定帐户”选项设置为“ 固定” ，则auth模块将使用目录服务中的绑定帐户查找组。可用组显示在LDAP组名称列表中。

    • 如果将绑定帐户选项设置为动态 ，则目录服务中的组列表对于绑定帐户不可用。要成功映射组，您需要输入与目录服务中显示的完全相同的组名称。

  5. 从YouTrack组列表中选择一个组。

  6. 单击添加按钮。

    • 映射将添加到列表中。
