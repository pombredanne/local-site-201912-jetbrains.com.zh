敏捷板用户界面

YouTrack中的敏捷板具有许多控件，可用于更新问题或修改板的外观。

标头

通过页面顶部的“ 创建”按钮，您可以执行以下操作。

行动

描述

卡

打开一个对话框，用于将问题作为卡添加到板上。

  • 在通过字段中的值标识泳道的板上，会将卡添加到泳道中，该泳道对应于为此字段分配给新问题的值。

  • 在按问题识别泳道的板上，将卡添加到泳道中以归类未分类的卡。

斯威姆兰

打开一个对话框，将问题添加为面板上的泳道。创建后，新的泳道将添加到板的底部。该选项在通过字段中的值标识泳道的敏捷板上不可见。

短跑

打开“ 新建Sprint”对话框。仅当启用了冲刺并且您有权更新当前电路板的设置时，此选项才可见。有关更多信息，请参见创建Sprint 。

板

打开“ 创建敏捷板”页面。有关更多信息，请参阅创建敏捷板 。

标头下方显示以下控件。

控制

描述

董事会名称

显示当前单板的名称。在输入字段中输入一个值以过滤板列表。要查看其他板，请从下拉列表中选择其名称。

过滤

过滤板上的卡。类似于搜索栏，此过滤器支持查询完成。当积压可见时，筛选条件将应用于积压中的问题。按Enter或单击“ 搜索”图标以应用过滤器。单击删除图标以删除过滤器并显示板上的所有卡。

董事会名单

标头中显示当前敏捷板名称的部分也是一个列表。此列表授予对您可以查看和使用的任何敏捷板的访问权限。

此列表显示每个板的名称及其所有者。董事会所有者的右边有一个图标，可让您将相应的董事会添加到收藏夹列表中。您最喜欢的板显示在列表的顶部。当您为已标记为收藏夹的木板单击此图标时，该木板将从收藏夹列表中删除。

工具列

向下滚动面板时，工具栏始终显示在页面顶部。

所有敏捷板的工具栏中均显示以下控件和视觉指示符。

控制

描述

进度指示器

以条形图显示板上每列中的卡总数。将指针移到每个条形上以显示总牌数和分配给每列的值。

估算值

显示板上所有卡的总估算。呈现方式根据在板设置中用作“ 估计字段 ”的自定义字段的数据类型而有所不同。仅当配置“ 估计字段”设置时，此信息才可见。

未解决的问题链接

该链接打开分配给董事会的问题列表，但这些问题与用于定义列和泳道的任何属性都不匹配。这可以帮助您找到已分配值的问题，这些问题不再与标识板上列和泳道的值匹配。然后，您可以选择是要添加值并将其与现有列合并还是添加更多泳道。

卡大小

设置板上卡片的详细程度。此设置是用户特定的，可以为不同的板单独设置。

电视模式

以全屏模式打开面板。电视模式经过优化，可在大型显示器上显示，并且可以比普通模式更远的距离读取。

切换主题

在标准演示文稿和调色板移动的深色主题之间切换。

积压

在页面左侧的待办事项列表中打开问题列表。有关更多信息，请参见处理积压 。

图表

打开配置为跟踪板上进度的图表。显示的图表类型由面板的图表设置决定。如果您有权更新面板设置，则可以使用其他控件重新计算报告，编辑图表设置并打印报告。否则，您只能查看设置并打印报告。

主板设置

打开“ 板设置”面板。如果您没有更新板设置的权限，则会禁用此控件。

以下控件仅在使用sprint的敏捷板上可见。

控制

描述

冲刺名称

显示当前冲刺的名称。要查看不同的Sprint，请从下拉列表中选择其名称。在输入字段中输入一个值以过滤冲刺列表。如果您有权更新当前电路板的设置，则还可以选择“ 新建sprint”选项以将冲刺添加到电路板上。

冲刺赛程

显示当前冲刺的开始和结束日期。仅在为sprint定义了计划后，此控件才可见。如果您有权更新当前电路板的设置，则可以单击下拉菜单更改时间表。否则，将禁用此控件。

冲刺倒计时

显示当前冲刺中剩余的日历天数。该总数包括非工作日。在冲刺的最后一天，此指示器仅显示方格旗。仅在为sprint定义了计划后，此信息才可见。

列

每列的标签对应于自定义字段中的一个值，该值标识板上的列。

每一列的标题中显示以下控件和可视指示符。

控制

描述

显示/隐藏列

单击列标签以折叠或展开板上的列。

在问题列表中打开

单击以在新的浏览器选项卡中打开问题列表。当您将指针移到列标题上方时，此控件可见。筛选“ 问题”列表以显示与所选列的搜索条件匹配的问题。

总估算

在列中显示所有卡的总估算。呈现方式根据在板设置中用作“ 估计字段 ”的自定义字段的数据类型而有所不同。仅当配置“ 估计字段”设置时，此信息才可见。如果未设置评估板的估计字段 ，则此指示器在该列中显示总卡数。将指针移到指示器上，以显示该列中的卡总数。

泳道

板上的每一行代表一条泳道。

  • 对于使用问题识别泳道的板，问题摘要将显示为泳道的标签。

  • 使用自定义字段中的值标识泳道的板将显示存储在所选字段中的值。

所有板卡都可以显示未分类卡的泳道。该泳道可以放在板子的顶部或底部。使用此泳道可以管理不属于特定泳道的问题。

每个泳道中均显示以下控件和可视指示符。

控制

描述

重新订购泳道

拖动以将泳道移动到板上的新位置。

问题编号

单击以在单个问题视图中打开在板上标识泳道的问题。该控件仅在使用问题识别泳道的板上可见。

更新问题

单击以打开在板上查看模式下识别泳道的问题。该控件仅在使用问题识别泳道的板上可见。

显示/隐藏泳道

单击箭头图标或泳道标签以折叠或展开泳道。

卡总数

显示分配给泳道的卡总数。

估算值

显示分配给泳道的所有卡的总估计。呈现方式根据在板设置中用作“ 估计字段 ”的自定义字段的数据类型而有所不同。仅当配置“ 估计字段”设置时，此信息才可见。

添加一张信用卡

打开一个对话框，将问题作为卡片添加到选定的列和泳道。标识相应列和泳道的字段的值会自动设置，并且无法在此对话框中更改。

牌

每张卡代表板上的一个问题。当您将卡拖动到其他列或泳道时，用于标识板的这些组件的字段将在问题中更新。每张卡上显示的信息量由“ 卡大小”滑块中的当前设置确定。

每张卡的视觉呈现均以以下方式变化：

  • 板上的当前卡显示为蓝色边框。在这种状态下，键盘快捷键和命令将应用于当前卡。

  • 在移动到板上的另一张卡之前选择的卡会以蓝色背景显示。在这种状态下，键盘快捷键和命令将应用于所有选定的卡。不考虑当前卡。

页脚

像工具栏一样，页脚始终在敏捷板上可见。

在页脚中显示以下控件和可视指示符。

控制

描述

董事会负责人

显示创建面板的用户的全名。单击链接以打开董事会所有者的用户个人资料。

冲刺目标

显示当前冲刺的目标。

新泳道

打开一个对话框，将问题作为泳道添加到董事会。该控件仅在使用问题识别泳道的板上可见。

当您选择板上的一张或多张卡时，在页脚中显示以下控件。

控制

描述

选定的卡片

显示板上当前选择的卡数。单击x图标以清除选择。

清空选项

清除板上的卡选择。

受让人

使您可以更改所有选定卡的受让人。选择一个用户来设置或更改所有选定卡的受让人。

添加标签

打开标签列表。选择一个标签，将标签添加到所有选定的卡中。

添加链接

打开问题链接类型列表。选择一个问题链接类型以打开“ 应用命令”对话框，并将链接类型添加到所有选定的卡中。

移至下一个冲刺

将所有选定的卡移动到下一个冲刺。该控件仅在启用了sprint的敏捷板上可见。

  • 如果尚未计划下一个冲刺，则此控件将打开“ 新建冲刺”对话框。创建冲刺时，选定的卡将添加到新的冲刺中。

  • 如果已计划下一个冲刺，则此控件将打开“ 应用命令”对话框。将所选问题移至下一个冲刺的命令会自动输入到输入字段中。应用命令将选定的卡片移动到下一个冲刺。

此控件旁边的下拉列表使您可以将选定的卡移动到特定的Sprint或将其返回到待办事项列表。
