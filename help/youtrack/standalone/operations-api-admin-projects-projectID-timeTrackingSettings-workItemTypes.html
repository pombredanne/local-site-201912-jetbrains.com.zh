具有特定WorkItemType的操作

此资源使您可以配置项目的工作项类型。

资源资源
/ api / admin / projects / {projectID} / timeTrackingSettings / workItemTypes / {typeID}

退货实体

WorkItemType 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的WorkItemType 。

  • DELETE ： 删除特定的WorkItemType 。

WorkItemType属性

表示可以分配给工作项目的工作类型。

下表描述了WorkItemType实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

名称

串

工作项类型的名称。 Can be null 。

自动附加

布尔型

如果为true，则在为它启用时间跟踪后，会自动将其添加到项目中。

读取特定的WorkItemType

阅读项目中特定工作项类型的属性。

所需权限

需要读取工作项目或更新工作项目或创建工作项目权限

请求语法

GET / api / admin / projects / {projectID} / timeTrackingSettings / workItemTypes / {typeID}？{字段}

{typeID}

工作项类型的数据库标识。

请求参数

参数

类型

描述

领域

串

应在响应中返回的WorkItemType属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/projects/0-0/timeTrackingSettings/workItemTypes/65-2?fields=id,name,autoAttached

样品响应体

{“ name”：“ Documentation”，“ autoAttached”：false，“ id”：“ 65-2”，“ $ type”：“ WorkItemType”}
