YouTrack Standalone 2019.2帮助

全局默认设置

YouTrack中的每个自定义字段都有一组全局默认设置。这些设置定义：

  • 字段是否自动附加到新项目。

  • 连接到新项目时，字段是私有的还是公共的。

  • 在“ 问题”列表中是否显示字段。自定义字段在“ 问题”列表中以紧凑和详细视图显示。

自动将字段附加到新项目

默认情况下，自动附加字段会附加到任何新项目。

要将字段自动附加到项目：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择所需的字段。

  4. 单击工具栏中的“ 启用自动附加”按钮。

自动附加字段在可用字段列表中标有自动附加徽章。

设置隐私

专用字段仅可用于对项目中具有特定权限的用户查看和/或编辑：分别为读取问题专用字段和更新问题专用字段 。

全局隐私设置仅会影响该字段自动附加到的新项目和该字段手动附加到的现有项目的初始状态。退出该字段已附加到的项目的隐私设置不会受到影响。项目管理员可以随时覆盖初始隐私设置。

默认情况下，将字段设为私有：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 单击您要设为私有的字段名称。

  4. 在工具栏中，单击“ 设为私有”按钮。

有关更多信息，请参见私人和公共发行字段 。

设置可见性

大多数自定义字段的值在“ 问题”列表中以紧凑而详细的视图显示。为了减少视觉混乱，默认情况下，您可以在列表中隐藏自定义字段。在这种情况下，用户只能在以下情况下看到存储在字段中的值：

  • 在单个问题视图中打开问题。

  • 覆盖“ 问题”列表中的默认字段可见性设置。

要在“问题”列表中默认隐藏字段，请执行以下操作：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择所需的字段。

  4. 单击工具栏中的“ 从问题中隐藏”列表按钮。
