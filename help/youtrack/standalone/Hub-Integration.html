YouTrack Standalone 2019.2帮助

集线器集成

YouTrack带有内置的JetBrains Hub服务。集线器服务是YouTrack的组成部分，可提供身份验证，授权和用户管理。无法删除或禁用集线器服务。

集线器集成默认情况下处于启用状态。“ 集线器集成”页面使您可以对集成进行故障排除和监视。

要访问此页面，请在“ 管理”菜单的“ 服务器设置”部分中选择“ 集线器集成 ” 。

在“ 集线器集成”页面的顶部，您会看到有关集线器服务本身的简短说明。您还将看到可用于集线器集成的部署模式的说明。说明指出了YouTrack当前启用的模式。

有关每种部署模式的详细说明，请参阅“ 集线器部署选项” 。

健康指标

“ 集线器集成”页面显示了运行状况指示器的集合，可用于监视集线器集成的状态。使用这些指示器来诊断集线器安装问题。

连接到集线器

这表明YouTrack是否与Hub服务处于活动状态。在正常情况下，连接状态为“ 活动且正在运行” 。

如果您的集线器服务关闭或无响应，此指标将显示可能是问题根源的状况。使用以下列表解决问题：

集线器不信任YouTrack

在大多数情况下，用户已访问连接到集线器的服务列表，并且不信任YouTrack。要解决此问题，请访问集线器中的“ 服务”页面，并信任YouTrack服务。

集线器URL中的未知主机

您的DNS服务无法解析集线器域名。检查您的网络基础结构或联系您的ISP。

无法连接到集线器

存在连接问题。有关详细信息，请参见连接问题 。

与集线器的连接超时

集线器无法访问或存在性能问题。有关详细信息，请参见连接问题 。

集线器事件同步

该指示器显示集线器事件同步例程的当前状态。在正常情况下，状态为活动，正在同步所有内容 。

如果状态显示为“ 未启动” ，请与支持部门联系 。

权限管理

在这里，您可以看到用于访问授权的当前源。在正常情况下，源是集线器权限管理 。如果状态显示为YouTrack权限管理 ，请联系支持 。

中心网址

这是您的中心服务的URL。

如果在安装外部集线器时遇到连接问题，请在运行YouTrack的计算机上打开此URL，以验证是否可以从YouTrack访问集线器服务。

YouTrack服务ID

在这里，您可以找到在Hub中作为连接服务分配给YouTrack的ID。

将服务连接到集线器时，会为其分配一个唯一的服务ID。如果要使用REST API连接到YouTrack，请使用此服务ID进行访问和授权。

如果您怀疑集线器集成无法正常工作，请按照“ 对集线器集成进行故障排除”中的说明进行操作。

同步

为了提高性能，在Hub和YouTrack数据库中都存储了许多对象。这包括用户及其有关组成员身份，许可证，通知设置等的记录。使用通用唯一标识符（UUID）映射这些重复的数据结构。

后台例程会定期检查映射的两面，以消除可能的数据差异。如果您在YouTrack和集线器之间的连接遇到问题，则两个数据库可能不同步。

该操作使用集线器作为数据源同步用户和组的记录。

出口

导出例程会定期查找仅存在于YouTrack中的记录，并将它们导出到中心数据库。

此操作将查找尚未导出的孤立实体，为其分配UUID，然后将其导出到集线器。

运行诊断

该操作进行一系列自检，以查找损坏的映射，识别可疑的数据模式并诊断集线器集成的其他问题。如果找到问题，则诊断例程将提供快速修复操作（如果可用）。Quickfix操作可在页面底部的“ 集成日志”中找到。

还有在后台运行的例行诊断检查。如果后台例程检测到问题，您还将在集成日志中看到有关可用快速修复的建议。

使缓存无效

集线器为YouTrack提供了远程权限缓存。该高速缓存用于检查内存中的用户权限，而无需对集线器进行网络调用。

在集线器中断或类似事件期间，权限缓存可能会损坏。此操作将刷新本地权限缓存，并从集线器重新加载新数据。如果您在YouTrack中遇到与访问权限相关的异常行为，请运行此操作。
