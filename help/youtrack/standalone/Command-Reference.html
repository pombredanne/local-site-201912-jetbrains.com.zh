命令参考

命令基本上是一系列属性和值对，它们可以一次将更改应用于一个或多个问题。在许多情况下，您可以跳过属性，然后输入要应用的值。YouTrack会检测哪些属性使用指定的值，并建议该命令的有效选项。

用于输入命令的格式类似于搜索查询中使用的语法。但是，使用冒号，花括号和数字符号的规则不适用于命令。例如：

  • 要搜索分配给您并与标签YouTrack 2017关联的关键问题，请输入以下搜索查询：

    for: me tag: {YouTrack 2017} #critical 。

  • 要应用将问题分配给自己的命令，添加标签YouTrack 2017 ，并将优先级提升为紧急，请输入以下命令：

    for me tag YouTrack 2017 critical 。

自定义字段

您可以使用命令来设置或更改自定义字段中的值。为受让人字段附加选项在被描述受让人部。自定义字段支持以下选项：

  • 输入格式的命令field namevalue 。

  • 输入value作为单个值。自动完成对话框中显示了使用此值的相应字段。

  • 要更新由工作流中的状态机规则管理的自定义字段的值，请以以下格式输入命令： field nameevent 。

  • 如果自定义字段可以包含一个空值，请输入该空值的名称以清除该字段。

对于存储多个值的自定义字段，您可以使用add和remove命令：

命令

描述

add

将指定的单个值添加到指定的自定义字段。当前存储在该字段中的未指定值不受影响。

remove

从指定的自定义字段中删除指定的单个值。当前存储在该字段中的未指定值不受影响。

如果您具有“ 更新项目”权限，则可以将值添加到自定义字段。此选项显示在对话框的“ 命令预览”部分中。

实际的字段和值根据项目中使用的自定义字段而有所不同。所有自定义字段均支持fieldvalue命令语法。此列表中的其他命令使用分配给默认自定义字段的别名。

受让人

受让人与其他自定义字段类型不同，因为它存储一个user类型。您可以使用几个别名来设置此字段的值。

  • 别名me和my参考当前用户的登录名。您可以将这些别名作为单个值输入，以将选定的问题分配给当前用户。

  • 您可以输入用户login作为单个值，以将选定的问题分配给特定用户。

  • 使用空值的名称（ Unassigned ）清除字段。

  • 使用语法add 分配问题并remove 取消分配问题。

您可以使用命令将链接类型添加到一个或多个问题。要使用命令添加链接类型，请输入链接类型，然后输入目标问题的问题ID。您也可以使用add和remove命令：

命令

描述

将指定的链接类型添加到指定的问题。

从指定的问题中删除指定的链接类型。

以下命令与YouTrack中默认问题链接类型的外部名称和内部名称相对应。自定义链接类型还使用外部名称和内部名称作为命令。

命令

描述

depends on

在所选问题中将“ 依赖于”链接添加到目标问题。还添加了“必需”以链接到目标问题中的选定问题。

duplicates

将“ 重复”链接添加到所选问题中的目标问题。还向目标问题中的选定问题添加一个“复制方式”链接。

is duplicated by

在所选问题中向目标问题添加“通过副本复制” 。还将“重复”链接添加到目标问题中的选定问题。

is required for

在链接到所选问题中的目标问题时需要添加“ 是” 。还向目标问题中的所选问题添加“ 依赖于”链接。

parent for

将父项添加到所选问题中的目标问题的链接。还将链接的子任务添加到目标问题中的所选问题。

relates to

添加“关联” 以链接到所选问题中的目标问题。同时添加“关联” 以链接到目标问题中的选定问题。

subtask of

将链接的子任务添加到所选问题中的目标问题。还添加父项以链接到目标问题中的选定问题。

值

描述

Board

冲刺名称

在指定的敏捷板上将问题分配给具有指定名称的Sprint。您可以将sprint名称替换为{current sprint}将问题分配给当前的Sprint。要了解YouTrack如何识别当前的Sprint，请参阅当前的Sprint 。

对于使用“ 将新问题添加到冲刺”选项的敏捷板，可以使用Default作为在板设置中选择的Sprint名称的别名。

您也可以使用add和remove命令：

命令

描述

add Board

将选定的问题添加到指定敏捷板的指定冲刺中。

add Board

将选定的问题添加到指定的敏捷板。如果为主板启用了sprint，则问题将添加到当前的sprint中。

remove Board

从指定的sprint中删除指定敏捷板的选定问题。

remove

从指定的敏捷板的任何冲刺中删除选定的问题。对于某些板配置，此命令的处理方式有所不同。

  • 对于启用了sprint且使用“将sprint链接到自定义字段”选项的板，不能从板上除去问题。而是将选定的问题分配给与链接的自定义字段的空值相对应的sprint。

  • 对于使用Sprint 卡来匹配查询选项的禁用sprint的板，此命令无效。要从董事会中删除问题，您需要更新其字段，以使该问题不再与搜索查询匹配。
