YouTrack Standalone 2019.2帮助

具有特定敏捷性的运营

该资源使您可以使用REST API在YouTrack中使用敏捷板。

资源资源
/ api / agiles / {agileID}

退货实体

敏捷 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的敏捷 。

  • POST ： 更新特定的敏捷 。

  • DELETE ： 删除特定的敏捷 。

敏捷属性

表示敏捷板配置。

下表描述了Agile实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

名称

串

敏捷板的名称。 Can be null 。

所有者

 用户

敏捷板的所有者。 Can be null 。

visibleFor

 用户组

可以查看此面板的用户组。 Can be null 。

visibleForProjectBased

布尔型

如果为true，则所有可以查看与该板相关的所有项目的人都可以看到该板。

可更新的

 用户组

可以更新主板设置的用户组。 Can be null 。

updateableByProjectBased

布尔型

如果为true，则可以更新关联项目的任何人都可以更新板。

孤儿

布尔型

如果为true，则将孤立的泳道放置在板的顶部。否则，孤儿泳道位于所有其他泳道的下方。

hideOrphansSwimlane

布尔型

为true时，孤儿泳道不显示在板上。

EstimateField

 CustomField

一个自定义字段，用作板的估算字段。 Can be null 。

专案

 一系列项目

与董事会相关的项目的集合。

冲刺

 冲刺数组

与电路板关联的一组Sprint。

currentSprint

 短跑

当前日期的实际冲刺。 Read-only 。 Can be null 。

columnSettings

 列设置

面板的列设置。 Read-only 。

泳道设置

 Swimlane设置

板泳道的设置。 Can be null 。

sprintsSettings

 冲刺设置

董事会冲刺的设置。 Read-only 。

colorCoding

 颜色编码

电路板的颜色编码设置。 Can be null 。

状态

 敏捷状态

董事会状态。 Read-only 。
