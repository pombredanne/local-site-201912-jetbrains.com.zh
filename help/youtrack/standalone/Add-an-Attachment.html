附加档案

您可以将文件直接附加到问题或评论中。您可以上传并附加图像和其他文件类型。将文件附加到问题的选项显示在列表视图和单个问题视图的问题工具栏中。

当您从“ 问题”列表中附加文件时，附件将添加到列表中当前突出显示的问题中。

文件可以直接附加到问题上，也可以添加注释。

  • 附加到问题后，文件仅显示在页面的“ 附件”部分中。

  • 附加到注释的文件显示在问题的“ 附件”部分中，并且也显示为注释文本正下方的缩略图。

本页上的说明描述了如何将文件直接附加到问题。要了解有关将文件附加到注释的更多信息，请参见将文件附加到注释 。

要将文件附加到问题：

需要权限：读取问题，更新问题，添加附件

  1. 点击问题工具栏中的显示更多按钮。

  2. 选择以下选项之一：

    选项

    描述

    附加档案

    打开一个对话框，您可以在其中找到并从本地目录上载文件。附件从问题继承可见性设置。

    有关更多信息，请参见附件的可见性 。

    私下附加文件

    打开一个对话框，您可以在其中选择一个或多个文件，并将附件的可见性限制为特定的组。当文件包含敏感信息（例如用户帐户或日志文件）时，请使用此选项。

    附加和编辑图像

    打开一个对话框，您可以在其中选择，拖放或粘贴剪贴板中的图像。该文件在“ 图像编辑器”中打开。图像编辑器可让您应用各种效果并向图像添加注释。您也可以直接在对话框中设置可见性组。有关更多信息，请参见编辑图像 。

键盘快捷键

您可以使用键盘快捷键将文件附加到问题。这些快捷方式在列表视图和单个问题视图中可用。

Windows快捷方式

Mac捷径

描述

Ctrl + Alt +A

Control + Option +A

打开一个对话框，您可以在其中找到并从本地目录上载文件。附件从问题继承可见性设置。

Ctrl +V

⌘ +V

将剪贴板中的图像附加到问题或评论中。

Ctrl + Alt + Shift +V

Control + Option + Shift +V

打开一个对话框来附加和设置文件的可见性。

拖放

您只需将文件拖到问题上即可附加文件。

此方法适用于任何视图模式。您可以一次拖放多个文件。

  • 在“ 问题”列表中，将一个或多个文件拖到列表中的目标问题。

  • 在单个问题视图中，将一个或多个文件拖到页面中间的任意位置。

糊

您可以将文件复制到剪贴板，然后将其粘贴为问题的附件。此方法使用标准键盘粘贴命令Ctrl + V （ ⌘ + V对于macOS）。

在列表和单个问题视图中，粘贴的文件将附加到问题上。您也可以将图像直接粘贴到注释中。

查看附件

问题附带的所有文件都显示为缩略图。根据缩略图所代表的文件类型，YouTrack提供了不同级别的交互。

文件类型

行为

图片

对于图像文件，缩略图包含图像的预览。

  • 将指针移到缩略图上可显示较大版本的图像预览。

  • 点击缩略图以在灯箱模式下打开图像。

您还可以使用Markdown在问题描述正文或注释文本中内嵌显示附件图像。有关更多信息，请参见图像附件 。

视频

对于MPEG-4，Ogg和WebM格式的视频文件，单击缩略图将打开并在新的浏览器选项卡中播放视频。

您还可以选择使用Markdown将这些格式的视频文件嵌入到问题描述或评论文本中。有关更多信息，请参见嵌入式视频 。

对于所有其他文件类型，单击缩略图会将文件的副本下载到本地目录。

编辑或删除附件

您可以随时更新文件名并更改问题的可见性。

要编辑附件：

需要权限：读取问题，更新问题，添加附件，删除附件

  1. 在单个问题视图中打开问题。

  2. 单击您要更新的附件旁边的编辑图标。

    • 如果附件是图像，则文件在图像编辑器中打开。有关更多信息，请参见编辑图像 。

    • 如果附件不是图像，则会打开“ 编辑附件”对话框。

  3. 使用以下选项来更新附件属性：

    选项

    描述

    可见到

    选择一个组以更改附件的可见性设置。

    新名字

    更改文件名。

    删除

    单击以删除附件。

    取消

    放弃更改并关闭对话框。

  4. 单击“ 保存”按钮以应用更改并关闭对话框。

您可以通过单击单个问题视图中的删除附件图标或如上所述在“ 编辑附件”对话框中删除附件来从问题中删除附件。
