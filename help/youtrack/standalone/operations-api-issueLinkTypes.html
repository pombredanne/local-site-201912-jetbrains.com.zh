具有特定IssueLinkType的操作

此资源提供具有问题链接类型的操作。

资源资源
/ api / issueLinkTypes / {typeID}

退货实体

IssueLinkType 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的IssueLinkType 。

  • POST ： 更新特定的IssueLinkType 。

  • DELETE ： 删除特定的IssueLinkType 。

IssueLinkType属性

代表YouTrack中链接类型的设置。

下表描述了IssueLinkType实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

名称

串

问题链接类型的名称。 Can be null 。

localizedName

串

问题链接类型的本地化名称。它存在，并且只能针对本地化的YouTrack安装中的预定义问题链接类型进行更新。 Can be null 。

sourceToTarget

串

问题链接类型的外部名称。 Can be null 。

localizedSourceToTarget

串

问题链接类型的本地化的外部名称。它存在，并且只能针对本地化的YouTrack安装中的预定义问题链接类型进行更新。 Can be null 。

targetToSource

串

问题链接类型的内部名称。 Can be null 。

localizedTargetToSource

串

问题链接类型的本地化内部名称。它存在，并且只能针对本地化的YouTrack安装中的预定义问题链接类型进行更新。 Can be null 。

指导的

布尔型

如果true ，该链接是定向的。除此以外， false 。

聚合

布尔型

如果true ，该链接表示聚合关系。除此以外， false 。

只读

布尔型

如果true ，链接设置无法更新。除此以外， false 。 Read-only 。

请求语法

{typeID}

问题链接类型的ID。
