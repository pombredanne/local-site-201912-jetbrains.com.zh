拥有的捆绑包

该资源为拥有的字段捆绑包提供操作。

资源资源
/ api / admin / customFieldSettings / bundles / ownedField

退货实体

OwnedBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读OwnedBundles列表 。

  • POST ： 添加一个新的OwnedBundle 。

OwnedBundle属性

代表YouTrack中的一组拥有的值。

扩展BaseBundle

下表描述了OwnedBundle实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

价值观

 OwnedBundleElement数组

拥有的价值观。

isUpdateable

布尔型

如果true ，则当前登录的用户可以更新此捆绑包。除此以外， false 。 Read-only 。

阅读拥有的捆绑包清单

获取所有可用的拥有值捆绑包的列表。

请求语法

GET / api / admin / customFieldSettings / bundles / ownedField？{字段}＆{$ top}＆{$ skip}
