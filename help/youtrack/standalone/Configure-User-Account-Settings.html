YouTrack Standalone 2019.2帮助

更新中心帐户

作为管理员，您可以访问YouTrack中每个注册用户的中心帐户。要为特定用户打开中心帐户，请在“ 用户”列表中选择一个“ 登录名 ”。

Hub帐户的“ 常规”选项卡显示基本信息，例如全名，登录名，头像和电子邮件地址。具有“ 读取自我”和“ 更新自我”权限的用户可以自己查看和编辑此信息。这些权限默认分配给YouTrack中的所有默认角色。这意味着您几乎不需要代表其他用户来管理用户数据。

即便如此，您也可以在Hub帐户的“ 常规”选项卡上访问多个操作。在这里，您可以：

  • 禁止用户 。

  • 删除用户帐户 。

  • 发送请求以验证用户电子邮件地址。如果用户没有至少一个已验证的电子邮件地址，则该用户将不会收到有关恢复忘记密码的说明。

  • 将SSH公钥添加到帐户中以授予对已连接服务的访问权限。

  • 将凭据添加到中心帐户 。

  • 管理现有的登录名。

用户页面上的其余选项卡使您可以管理组成员身份并授予访问权限。

  • 有关组成员资格的更多信息，请参阅管理组成员资格 。

  • 有关访问权限的更多信息，请参见管理用户访问 。

关联的个人资料

“ 连接的配置文件”面板显示指向连接到集线器服务的其他JetBrains应用程序中的用户配置文件的链接。

  • 如果您的YouTrack Standalone安装使用内置的Hub服务，或者您使用的是YouTrack InCloud，则您只会看到指向当前用户的YouTrack配置文件的链接。您的YouTrack服务不支持其他JetBrains应用程序中的连接用户配置文件。

  • 如果您的YouTrack Standalone安装已连接到外部集线器服务，则“ 已连接服务”面板将显示指向每个已连接服务中当前用户配置文件的直接链接。
