YouTrack Standalone 2019.2帮助

配置参数

以下列表包括所有特定于YouTrack的Java启动参数。

要了解在启动YouTrack时如何应用这些参数，请参阅配置JVM选项 。

一般参数

您可以为任何YouTrack安装设置以下参数。

参数

数据类型

默认值

描述

jetbrains.charisma.restoreRootPassword

布尔型

假

将root用户帐户的密码恢复为默认值root 。

jetbrains.youtrack.support.email

电子邮件地址

youtrack-feedback@jetbrains.com

指定用作的电子邮件地址mailto错误屏幕上“ 联系人支持”链接的地址。

jetbrains.youtrack.overrideRootPassword

串

空的

设置新密码root用户帐号。

jetbrains.youtrack.disableCheckForUpdate

布尔型

假

启用和禁用启动时检查软件更新。设置为true ，将禁用检查软件更新的功能。

jetbrains.youtrack.workflow.detachModified

布尔型

假

从项目中分离所有工作流程规则，但未进行任何修改的默认工作流程除外。

jetbrains.dnq.textIndex.minPrefixQueryLength

整数

3

将前缀查询字符串的最小长度设置为一定数量的字符。如果前缀查询和通配符字符串未在搜索查询中返回所有匹配的单词形式，请将此值设置为2 。设置此系统属性会影响YouTrack服务器的性能。值越低，服务器性能越慢。

jetbrains.youtrack.mail限制

整数

0

设置与一起发送的电子邮件数量的每日限制issue.sendMail()工作流程中的方法。设置为0时，没有限制。YouTrack InCloud实例的默认值为100。

jetbrains.youtrack.event.merge.timeout

长

60000

设置在发送电子邮件通知之前，YouTrack等待编译作为摘要应用到问题的更改的时间（以毫秒为单位）。

  • 减小此值可以提高YouTrack分析通知的速度，并且可能会对总体服务器性能产生负面影响。

  • 如果将此值从默认值（一分钟）增加，则可以减少YouTrack发送的通知电子邮件的总数。

jetbrains.youtrack.default.page

串

/仪表板

设置YouTrack服务器的默认起始页。输入到您的基本URL的相对路径，并带有一个斜杠。

jetbrains.mps.webr.i18n.custom-translations

串

空的

指向服务器上的目录，该目录包含用于自定义本地化的文件。

jetbrains.youtrack.webHooksBaseUrl

串

空的

在发送请求以在外部服务中创建Webhook时，为YouTrack服务器设置备用基本URL。

jetbrains.http.request.header.buffer.size

长

102400

最大请求缓冲区大小。如果您在YouTrack日志中看到以下警告/错误，则应该增加该值： URI is too large >102400要么Request header too large

jetbrains.http.response.header.buffer.size

长

102400

最大响应缓冲区大小。如果您在YouTrack日志中看到以下警告/错误，则应该增加该值：Response header too large

认证限制参数

以下JVM选项使您可以在YouTrack中启用或禁用限制。

参数

数据类型

描述

jetbrains.hub.auth.login.throttling。已启用

布尔型

设置集线器身份验证模块的限制功能的状态。如果为true，则启用身份验证限制。
