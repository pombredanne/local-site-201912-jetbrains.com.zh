旧的REST API

文档的此部分描述了不推荐使用的YouTrack REST API支持的资源和方法。提供此参考仅是为了帮助您对使用此API的现有客户端应用程序进行故障排除。对于新的应用程序，请使用当前版本中的资源和方法。

随着我们当前REST API的端点的初始发布，我们鼓励开发人员开始从先前版本迁移应用程序。一旦我们发布了以前的API支持的端点的较新版本，它将正式弃用。

如果您有使用旧API的现有应用程序或集成，建议您熟悉更新的版本。如果您的应用程序使用的端点尚未在新API中发布等效项，请加入Slack中的YouTrack社区 。让我们知道您正在使用哪些端点，并帮助我们确定下一步应该发布哪些功能。

使用本参考时，请注意以下限制：

  • 此版本的API不再处于主动开发中。该版本的API无法访问当前版本的YouTrack支持的许多功能。

  • 此API参考中存在许多已知的遗漏和不足之处，将不予解决。我们的工作仅专注于为当前API提供此信息。
