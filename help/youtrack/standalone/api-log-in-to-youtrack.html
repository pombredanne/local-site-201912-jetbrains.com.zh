YouTrack Standalone 2019.2帮助

登录到YouTrack

YouTrack支持两种身份验证方法，可用于登录和访问外部客户端的数据。

方法

描述

永久令牌授权

永久令牌为脚本和应用程序中的REST API调用启用基于令牌的授权。此功能提高了YouTrack与外部服务集成的安全性。

我们建议您使用永久令牌通过YouTrack REST API进行授权。

OAuth 2.0授权

OAuth 2.0是一种协议，可让您的应用程序从YouTrack请求访问令牌。YouTrack仅支持隐式授权。

我们仅建议您在应用程序需要客户端身份验证时使用OAuth 2.0。
