POST /管理员/用户

创建一个新的用户帐户或更新现有的用户帐户。通过登录名标识用户。如果不存在具有给定登录名的现有用户帐户，则将创建新的用户帐户。

POST / rest / admin / user

参数：

名称

类型

描述

登录

串

要创建的用户的登录名。必填项 。

全名

串

用户全名（可选）。

电子邮件

串

用户电子邮件（新用户必需）。

贾伯

串

用户jabber帐户（可选）。

密码

串

新用户的密码。如果跳过，默认情况下将使用自动生成的密码创建新用户帐户。请注意， 仅当您具有“低级管理”权限时， 才可以使用此参数。
