YouTrack Standalone 2018.4 Help

Uninstall YouTrack

Basically, to uninstall YouTrack, you need to:

  1. Remove the binaries of the application.

  2. Remove the application data of your YouTrack Standalone server.

The exact location of the binaries and application data varies for different OS and distribution type. Please refer to the distribution-specific procedures:

  • Uninstall Docker Container

  • Uninstall YouTrack JAR

  • Uninstall YouTrack ZIP

  • Uninstall YouTrack MSI
