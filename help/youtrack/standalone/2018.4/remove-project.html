Remove a Project

If you have a project that is no longer actively managed, you can remove it from YouTrack. When you remove a project, all of the issues that belong to the project are deleted permanently.



— Read Project Basic

— Delete Project

To remove a project:

  1. Open the Edit Project page of the project that you want to remove.

    • Click the Projects link in the header to open the Projects list.

    • From the Administration menu of the project that you want to clone your project, select General.

  2. Click the Remove project button in the header.

  3. Click the Remove button to confirm this action.

    • The project is removed from YouTrack.

    • All of the issues that were created in the project are deleted permanently.
