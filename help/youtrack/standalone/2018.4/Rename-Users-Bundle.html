Rename Users Bundle

POST /admin/customfield/userBundle/{bundleName}

Rename an existing custom field user bundle.

POST /rest/admin/customfield/userBundle/{bundleName}?{newName}

Parameters:

Name

Type

Description

bundleName

string

A name of user bundle to be renamed.

newName

string

New name for the user bundle.

Sample

Request

POST https://example.com/youtrack/rest/admin/customfield/userBundle/My%20Users?newName=Your+Users Cookie: $Version=0; JSESSIONID=19eie47i6xtd6l37im4l7uxwc; $Path=/; jetbrains.charisma.main.security.PRINCIPAL=NDgxMzQ5NGQxMzdlMTYzMWJiYTMwMWQ1YWNhYjZlN2JiN2FhNzRjZTExODVkNDU2NTY1ZWY1MWQ3Mzc2NzdiMjpyb290; $Path=/

Response

HTTP/1.1 301 Moved Permanently Content-Type: application/xml;charset=UTF-8 Location: https://example.com/youtrack/rest/admin/customfield/userBundle/Your%20Users Content-Length: 0
