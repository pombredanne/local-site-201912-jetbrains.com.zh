YouTrack Standalone 2018.4 Help

REST API Reference

With the release of YouTrack 2018.3 we started to gradually publish new REST API resources and methods. The new REST API is in active development and may be changed. In these terms the gradual publishing of the new resources and methods may be regarded as an Early Access Program.

When working with the new REST API section, keep the following precautions in mind:

  • The API is under development and subject to change.

  • The documentation can be incomplete, and may contain errors.

With the initial publication of endpoints for our current REST API, we encourage developers to start migrating applications from the previous version. Once we have published newer versions for the endpoints that were supported in the previous API, it will be officially deprecated.

If you have an existing application or integration that uses the old API, we encourage you to familiarize yourself with the updated version. If your application uses endpoints for which an equivalent is not yet published in the new API, join our YouTrack Community in Slack.
