YouTrack Standalone 2018.4 Help

What's New

We deliver updates and improvements to YouTrack on a continuous basis. Here's an overview of changes that are available in YouTrack 2018.4.

New Features

Feature

Description

Availability

Custom Project Templates

You can now use an existing project as a template for new projects. Projects that are marked as templates are added to the set of available project types on the Create Project page.

2018.4

Clone Project Action

This action lets you open an existing project and clone it as a new project. A new action lets you clone existing projects. The new project inherits settings from the original, including its project team, access scheme, custom fields, time tracking, workflows, integrations, and agile boards.

2018.4

Timesheet Report

This report shows you how your team allocated their spent time in a calendar-based spreadsheet.

2018.4

Search by User Mentions

You can now search for issues that contain @mention references for specific users. Use the mentions: search attribute to generate reports and lists that mention your account without having to rely on notifications alone to track these issues.

2018.4

IdP-initiated SSO for SAML

We’ve updated the SAML 2.0 authentication module to let you initiate single sign-on requests from the identity provider side.

2018.4

Updates and Enhancements

Feature

Description

Availability

Per-user Setting for Period Format

We added a setting to your YouTrack profile that lets you choose how to display values that are stored in period fields. This makes it easier to perform calculations and compare values at a glance.

2018.4.49852

Anonymous SMTP Server Connections

YouTrack can now connect to an SMTP server without sending a login and password. This lets you connect to an SMTP server that does not require authentication.

2018.4.49852

Favorite Agile Boards

A new control in the list of agile boards let you mark an agile board as a favorite. Your favorite boards are always shown at the top of the list.

2018.4.49168

System-wide Dark Theme

There's a new option in the general settings of your YouTrack profile that lets you apply a dark theme to every page in YouTrack except the Issues list. A nighttime-friendly version of the Issues list will be added in an upcoming release.

You'll also find a Dark Theme toggle in the menu that opens when you click your avatar.

2018.4.49168

Syntax Highlighting Improvements in Markdown

Improvements to the handling for language-specific syntax highlights in Markdown improve compatibility with similar features in GitHub and add support for code blocks without highlighting.

2018.4.48733

Quick Access to Project Settings

We've added a control that lets you jump directly from an issue to the settings for its project. Users with the Update Project permission can access project settings by clicking the icon next to the project name in the fields panel.

2018.4.48733

Aggregation Options for Reports

A new Show totals for option in the settings for several reports lets you show aggregate totals for votes and other fields that store values as an integer, float, or period. Create reports that show the total number of votes, estimations, spent time, and more.

This option is available in all of the issue distribution reports except for Advanced Issue per Project and is also available for the Cumulative Flow report.

2018.4

Issue Link Improvements

Tab-based navigation in the issue links panel lets you quickly find linked issues by link type. We’ve also moved links panel just below the issue summary.

2018.4

Streamlined VCS Integration

We’ve dramatically simplified the process for integrating with a version control system. Integrate your YouTrack projects directly with repositories in GitHub, GitLab, and Bitbucket.

2018.4

Create Issue Macro for Confluence Integration

The latest version of our YouTrack Integration app lets you create issues directly from pages in Confluence.

2018.4

Updated Interface for Project Teams

The new interface for managing project teams lets you add or remove users and groups in a single step. You also have the ability to change the default role that is assigned to the project team.

2018.4

Links to Agile Boards from Project Pages

New links to agile boards are available from the Administration menu for each project in the Projects list and the sidebar for project overviews. These links provide direct access to agile boards that display issues in the related project.

2018.4

Tabular Layout for Spent Time in Activity Stream

Work item data that is shown as spent time in the activity stream is now presented in a series of columns. This improvement helps you scan and calculate the total spent time at a glance.

2018.4

Time Report Improvements

New options in the settings for time reports let you group work items by parent issue, show comments in work items, and show the total spent time collected from subtasks as a summary for parent tasks.

2018.4

Shortcuts for Issue Links

We've added support for two new keyboard shortcuts in single issue view. These let you expand or collapse the panel for linked issues and open a dialog to add links to target issues.

2018.4

Custom Reply-to Address for Notifications

New options added to the Global Settings and Edit Project pages let you specify dedicated email addresses for handling replies to notifications that are sent by YouTrack.

2018.4

RTL Language Support in New Issue Views

The updated views for new and single issues now support languages that are written from right to left. As a result, installations and instances that have enabled this option are no longer restricted to older issue views.

2018.4

New Notification Templates

The templates that are used to generate notifications for integration-related problems are now available on the Notification Templates page. This means that you can customize the message content at the server and project level.

2018.4

Add Cards to Multiple Sprints

A new option in the Card configuration for agile boards lets you remove the restriction that limits cards to single sprints.

2018.4

Relative Parameter for Last Working Day

We've added a pre-defined parameter that lets you search for date values for the last working day. When you use this parameter in search queries instead of yesterday, you can start your work week with a full schedule of tasks to complete.

2018.4

LDAP Group Mappings

You can now map groups from a directory service to groups in YouTrack. These configuration options are available for the Active Directory, LDAP, and Open LDAP auth modules.

2018.4

Allowed Organizations for GitHub Auth Module

A new setting for the GitHub auth module lets you restrict logins to members of specific GitHub organizations. If your YouTrack installation is accessible over the internet, you can let users who belong to your organization sign in with their GitHub accounts.

2018.4

Single Term Search Optimization

2018.4

Clone Agile Board.

This feature adds an option to the settings panel for an agile board that lets you clone the board.

Discontinued Features

Feature

Description

Availability

New Workflows in the YouTrack Workflow Editor

We no longer support new workflows that are written in the external YouTrack Workflow Editor. We will continue to support workflows that were written in this editor through the first YouTrack release in 2019. After that point in time, we will only support workflows in JavaScript.

If you're still working with old workflows, we encourage you to rewrite them in JavaScript as soon as possible. For details, see Converting Old Workflows to JavaScript.

Removed in 2018.3

Old YouTrack REST API

With the initial publication of endpoints for our current REST API in YouTrack 2018.3, we encourage developers to start migrating from the previous version. Once we have published newer versions for the endpoints that were supported in the previous API, it will be officially deprecated. We will continue to support the old REST API for a fixed period after its deprecation to allow adequate time for migrating applications that use it.

If you have an existing application or integration that uses the old API, we encourage you to familiarize yourself with the updated version. If your application uses endpoints for which an equivalent is not yet published in the new API, join our YouTrack Community in Slack. Let us know what endpoints you’re working with and help us identify which features should be published next.
