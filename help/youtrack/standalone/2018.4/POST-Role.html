POST Role

POST /admin/role/{role}

Update an existing role.

POST /rest/admin/role/{role}?{newName}&{description}

Parameters:

Name

Type

Description

role

string

Name of a role to be updated.

newName

string

New name for the specified role.

description

string

New description for the specified role.

Sample

Request:

POST /rest/admin/role/Tester?newName=Testing Engineer&description=Can verify or reopen issues

Response:

Response Code: OK Response Status: 200
