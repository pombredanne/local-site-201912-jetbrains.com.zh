PUT /admin/customfield/versionBundle/{bundleName}/{versionName}

Add new version to bundle

PUT /rest/admin/customfield/versionBundle/{bundleName}/{versionName}?{description}&{colorIndex}&{releaseDate}&{released}&{archived}

Parameters:

Name

Type

Description

bundleName

string

Name of a bundle to add a new version.

buildName

string

Name of a new version.

description

string

Version description.

colorIndex

Integer

Sequential number of a color scheme (background color/text color pair) for the new version. See the list of available Color Indices List

releaseDate

instant

Date of a release for the new version.

released

boolean

Mark the version as released.

archived

boolean

Mark the version as archived.
