Administration REST API

Resources that are described in this section give access to various administrative operations and require administrative permissions for the user account that is used for authorization in REST API calls.

Administrative REST API resource endpoints start with the /api/admin/ prefix.

  • Global Settings
  • Projects
  • Custom Field Settings
  • Issue Link Types
  • System Time Tracking Settings
  • Text Index Settings
  • Telemetry
  • Database Backup
