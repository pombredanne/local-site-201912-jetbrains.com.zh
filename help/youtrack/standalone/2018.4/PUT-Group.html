PUT Group

PUT /admin/group/{group}

Create a new group.

PUT /rest/admin/group/{group}?{description}&{autoJoin}

Parameters:

Name

Type

Description

group

string

Name of the new group

description

string

Description of the new group

autoJoin

Boolean

Defines whether the newly registered users should be automatically added to this group or not

Sample

Request:

PUT /rest/admin/group/xda-developers?description=PDAs developers group&autoJoin=false

Response:

Response Code: 201 Response Status: Created Location: http://.../rest/admin/group/xda-developers
