Sample Custom Widgets

Software Developers from the YouTrack and Hub project teams at JetBrains have published custom widgets to public repositories on GitHub. These projects are open source, which means that you can download the source code and use it as a springboard to develop your own widgets.

Stack Overflow Questions Widget

This widget shows a digest of questions from Stack Overflow that match a specific tag. Clicking a question opens the corresponding page in Stack Overflow.

You can find the source code for this widget in the following locations:

  • GitHub
  • JetBrains Plugin Repository

Agile Board Status

This widget displays the status indicator that tracks the number of issues in each column on an agile board in YouTrack. Clicking a column opens the Issues list and shows the issues that match the query.

You can find the source code for this widget in the following locations:

  • GitHub
  • JetBrains Plugin Repository

Crowdin Project Status

This widget shows the current translation status for each target language in a project that managed in Crowdin. The widget fetches data from the Crowdin service using their REST API.

We use the Crowdin platform to manage localization for the YouTrack UI. The widget helps our test engineers track the status of our translation efforts prior to each release.

You can find the source code for this widget in the following locations:

  • GitHub
  • JetBrains Plugin Repository

Cats Widget

The purrrrrfect widget for adding a little humor to your dashboards. This widget displays one or more cats from the Cat Power Animated collection by Iconka.

You can find the source code for this widget in the following locations:

  • GitHub
  • JetBrains Plugin Repository
