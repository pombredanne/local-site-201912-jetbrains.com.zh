使用特定SavedQuery的操作

此资源使您可以访问和处理YouTrack中保存的搜索。

资源资源
/ api / savedQueries / {queryID}

退货实体

SavedQuery 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的SavedQuery 。

  • POST ： 更新特定的SavedQuery 。

  • DELETE ： 删除特定的SavedQuery 。

SavedQuery属性

表示已保存的搜索。

扩展WatchFolder

下表描述了SavedQuery实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

询问

串

在此搜索中保存的查询。 Can be null 。

问题

 一系列问题

与此保存的搜索相匹配的问题的集合。 Read-only 。

所有者

 用户

创建（因此拥有）watch文件夹的用户。 Read-only 。 Can be null 。

visibleFor

 用户组

可以看到此监视文件夹的用户组。如果该文件夹仅对其所有者可见，则此属性为null。 Can be null 。

可更新的

 用户组

可以更新此监视文件夹的用户组。如果只有文件夹的所有者可以更新它，则此属性为null。 Can be null 。

名称

串

问题文件夹的名称。 Can be null 。

读取特定的SavedQuery

读取具有指定ID的已保存搜索的设置。

所需权限

要查看保存的搜索，用户应该是其作者或与之共享的用户组的一部分。

请求语法

GET / api / savedQueries / {queryID}？{字段}

{queryID}

保存的搜索的数据库ID。

请求参数

参数

类型

描述

应在响应中返回的SavedQuery属性的列表。如果未指定任何字段，则仅entityID返回
