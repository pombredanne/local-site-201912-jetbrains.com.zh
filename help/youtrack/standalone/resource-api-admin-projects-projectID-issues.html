YouTrack Standalone 2019.2帮助

问题

该资源使您可以处理特定项目中的问题。

资源资源
/ api / admin / projects / {projectID} /问题

退货实体

问题 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读问题清单 。

  • POST ： 添加一个新的问题 。

问题属性

代表YouTrack中的问题。

下表描述了Issue实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

idReadable

串

问题编号。 Read-only 。

被创造

长

问题创建时刻的时间戳。 Read-only 。 Can be null 。

更新

长

问题的最新更新时间戳。 Read-only 。 Can be null 。

解决

长

将问题分配给被认为已解决的状态的时刻的时间戳。退货null ，如果问题仍未解决。 Read-only 。 Can be null 。

numberInProject

长

项目中的发行编号。 Read-only 。

项目

 项目

分配问题的项目。 Can be null 。

摘要

串

输入为问题摘要的文本。 Can be null 。

描述

串

输入为问题描述的文本。 Can be null 。

使用Markdown

布尔型

如果为true，则问题描述将解析为Markdown。如果为false，则问题描述将解析为YouTrack Wiki。更改此值不会将标记从一种语法转换为另一种语法。

wikifiedDescription

串

使用Wiki / markdown（包括HTML标记）进行处理后，其在UI中显示的问题描述。 Read-only 。

记者

 用户

报告（创建）问题的用户。 Read-only 。 Can be null 。

更新器

 用户

上次更新问题的用户。 Read-only 。 Can be null 。

draftOwner

 用户

如果问题是草稿，则此草稿的创建者，如果已经报告了问题，则为null。 Read-only 。 Can be null 。

isDraft

布尔型

如果问题是草稿，则为true；如果已经报告，则为false。 Read-only 。

能见度

 能见度

问题的可见性设置。他们描述了谁可以看到此问题。 Can be null 。

票数

整数

此问题的票数。 Read-only 。

评论

 问题评论数组

有关此问题的评论列表。

commentsCount

长

问题中的许多评论。 Read-only 。

标签

 IssueTag数组

添加到问题的标签列表。

链接

 问题链接数组

发布链接（例如，“关联到”，“父代”等）。 Read-only 。

外部问题

 外部问题

在原始第三方系统中对问题或类似对象的引用。 Read-only 。 Can be null 。

customFields

 IssueCustomFields数组

问题中存在的自定义字段的集合。 Read-only 。

选民

 发行投票

包含有关该问题及其重复项目的选民数据的对象。 Read-only 。

观察者

 问题观察者

包含有关用户观看此问题或其重复数据的数据。 Read-only 。

附件

 IssueAttachments数组

问题中的附件列表。

子任务

 发行链接

问题的子任务列表。 Read-only 。

父母

 发行链接

当前版本的父版本。如果问题不是任何问题的子任务，则null 。 Read-only 。

阅读问题清单

获取特定项目中所有可用问题的列表。

请求语法

GET / api / admin / projects / {projectID} /问题？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

应在响应中返回的Issue属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。
