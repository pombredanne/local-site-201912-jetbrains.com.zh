PUT项目

PUT / admin / project / {projectId}

建立新专案

PUT / rest / admin / project / {projectId}？{projectName}＆{startingNumber}＆{projectLeadLogin}＆{description}

参数：

名称

类型

描述

projectId

串

一个项目的所需的唯一标识符被创建。该短名称将用作该项目的发行ID中的前缀。

项目名

串

必填新项目的全名。必须是唯一的。

startingNumber

整数

分配给下一个手动创建的问题的必需编号。

projectLeadLogin

串

必需的要分配为项目所有者的用户的登录名。

描述

串

新项目的可选说明。

样品
