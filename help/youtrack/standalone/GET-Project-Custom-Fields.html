GET Project Custom Fields

GET /admin/project/{projectId}/customfield

Get custom fields used in a project.

GET /rest/admin/project/{projectId}/customfield

Parameters:

Name

Type

Description

projectId

projectByIdNotStrict

ID of a project to get custom fields.

Sample

Request:

GET /rest/admin/project/CMN/customfield

Response:

<projectCustomFieldRefs> <projectCustomField name="Pyatiletka" url="http://youtrack.jetbrains.net/rest/admin/project/CMN/customfield/Plan"/> <projectCustomField name="Region" url="http://youtrack.jetbrains.net/rest/admin/project/CMN/customfield/Region"/> </projectCustomFieldRefs>
