更改YouTrack Windows服务的端口号

当您将YouTrack作为Windows服务运行时，YouTrack使用:80连接端口（如果可用）。否则，YouTrack将使用从以下开始的第一个可用端口:8080 。

要更改MSI安装的连接端口，请执行以下操作：

  1. 以管理员身份打开“ 命令提示符”窗口。

  2. 在命令行中，将目录更改为\bin 。

    默认安装目录位置是C:\YouTrack 。

  3. 输入以下命令：
    net stop youtrack youtrack.bat配置--listen-port 1111 net start youtrack

    这些命令停止服务，将其配置为使用新的HTTP端口值（ 1111 ）并重新启动服务。

  4. 使用具有“ 低级管理”权限的帐户登录到YouTrack。

  5. 打开“ 域设置”页面，并验证基本URL设置正确。
