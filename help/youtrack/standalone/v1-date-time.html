YouTrack Standalone 2019.2帮助

约会时间

该模块包含使您可以处理日期和时间的功能。

功能


后

静态之后（时间戳，持续时间）

返回一个时间戳，该时间戳表示从指定日期到指定时间段之后的时间点。

自2018.2.42881起可用

参量

名称

类型

描述

时间戳记

 number

基本日期值。

持续时间

number ， string ， Period

从自定义字段检索或由toPeriod（）函数返回的持续时间，以数字（毫秒），字符串表示形式或周期表示。字符串表示形式是一系列数值，其后是代表时间跨度的缩写，按降序排列。例如1m3w4d23h58m。

返回值

类型

描述

number

产生的时间戳。

 

之前

静态之前（时间戳，持续时间）

返回一个时间戳，该时间戳表示从指定日期到指定时间段之前的时间点。

自2018.2.42881起可用

参量

名称

类型

描述

时间戳记

 number

基本日期值。

持续时间

number ， string ， Period

从自定义字段检索或由toPeriod（）函数返回的持续时间，以数字（毫秒），字符串表示形式或周期表示。字符串表示形式是一系列数值，其后是代表时间跨度的缩写，按降序排列。例如1m3w4d23h58m。

返回值

类型

描述

number

产生的时间戳。

 

格式

静态格式（时间戳，格式，timeZoneId）

创建Unix时间戳的字符串表示形式。

参量

名称

类型

描述

时间戳记

 number

要格式化为字符串的时间戳。

格式

 string

要应用于输出的日期格式。如果未指定任何值，则系统将提供日期格式。对于归因于当前用户的操作，将应用当前用户配置文件中的日期格式设置。对于归因于工作流用户帐户的操作，将应用全局日期字段格式设置。

timeZoneId

 string

时区的ID。将偏移量应用于以UTC为单位的原始时间戳。如果未指定任何值，则时区由系统提供。对于归因于当前用户的操作，将应用当前用户配置文件中的本地时区设置。对于归因于工作流用户帐户的操作，将应用全局默认时区。有关时区ID的列表，请参见JodaTime 。

返回值

类型

描述

string

指定时间戳记的字符串表示形式。

 

解析

静态解析（dateTimeString，格式，timeZoneId）

解析日期的字符串表示形式以返回Unix时间戳。使用此方法代替JavaScript中的Date.parse（）方法。有关详细说明，请参阅MDN文档 。

参量

名称

类型

描述

dateTimeString

 string

日期的字符串表示形式。

格式

string ， Array.

日期格式，可能与dateTimeString或格式数组匹配。如果提供了数组，则将顺序应用格式，直到成功解析dateTimeString为止。如果未指定任何值，则系统将提供日期格式。对于归因于当前用户的操作，将应用当前用户配置文件中的日期格式设置。对于归因于工作流用户帐户的操作，将应用全局日期字段格式设置。

timeZoneId

 string

dateTimeString所在的时区的ID。仅当与字符串匹配的格式不提供任何时间戳信息时，此参数才有效。如果没有成功匹配字符串的格式或该参数都未提供时区，则系统将提供时区。对于归因于当前用户的操作，将应用当前用户配置文件中的本地时区设置。对于归因于工作流用户帐户的操作，将应用全局默认时区。有关时区ID的列表，请参见JodaTime 。

返回值

类型

描述

number

指定字符串的时间戳表示。
