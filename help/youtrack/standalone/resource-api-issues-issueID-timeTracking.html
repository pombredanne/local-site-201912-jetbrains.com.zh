Issue Time Tracking

This sub-resource lets you read work items of an issue and time tracking status in the issue's project.

Resource

/api/issues/{issueID}/timeTracking

Returned entity

IssueTimeTracking. For the description of the entity attributes, see Supported Fields section.

Supported sub-resources

  • /api/issues/{issueID}/timeTracking: Operations with Specific IssueTimeTracking

  • /api/issues/{issueID}/timeTracking/workItems: Issue Work Items

IssueTimeTracking attributes

Represents time tracking settings in the issue.

This table describes attributes of the IssueTimeTracking entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

workItems

Array of IssueWorkItems

Collection of work items of the issue.

enabled

Boolean

Indicates whether the time tracking is enabled in the project that the issue belongs to. Read-only.
