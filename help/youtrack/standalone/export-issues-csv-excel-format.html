导出问题数据

YouTrack允许您导出问题列表的数据。

导出问题数据的功能仅在版本2019.1.52545中的问题列表的实验版本中可用。如此处所述，要导出问题数据，请关闭实验性用户界面。有关说明，请参阅重新设计的问题列表 。

提供以下选项：

选项

描述

导出为CSV

将问题数据导出到CSV文件（.csv）。

导出到MS Excel

以Microsoft Excel格式（.xlsx）导出问题数据。与CSV不同，Excel格式的导出保留自定义字段和其他问题属性中的值的数据类型。

这些选项可从“ 问题”列表上的“ 报告”下拉菜单中获得。

使用这些选项将问题上载到另一个应用程序，或使用第三方软件来编译自定义报告。

可以导出的最大问题数由“ 全局设置”页面上的“ 要导出的最大问题数”设置确定。默认值为500。如果您要导出的问题数超过了全局限制，请要求管理员增加此值。有关更多信息，请参见System 。

选择导出选项时，将对问题列表进行过滤以匹配当前的搜索查询和上下文。导出文件包含每个问题的以下详细信息：

项目

描述

问题编号

分配给项目中问题的ID。

项目

问题所属的项目。

标签

附加到问题的标签列表。

摘要

问题摘要。

记者

报告问题的用户的全名。

已建立

问题创建的日期和时间。

更新

上次更新问题的日期和时间。

解决

解决问题的日期和时间。

自定义字段

为问题中的每个自定义字段设置的值。

描述

问题说明。

投票数

该问题的当前票数。
