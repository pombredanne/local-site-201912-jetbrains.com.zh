SingleValueActivityItem

描述可以具有单个值的属性的更改。

扩展ActivityItem

下表描述了SingleValueActivityItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

已移除

在后代实体中指定类型

目标实体中属性的旧单一值。 Read-only 。 Can be null 。

添加

在后代实体中指定类型

目标实体中属性的新单一值。 Read-only 。 Can be null 。

作者

 用户

执行操作的用户。 Read-only 。

时间戳记

长

活动的时间戳。 Read-only 。

目标

在后代实体中指定类型

作为已执行操作的目标的实体。 Read-only 。

targetMember

串

目标实体的属性名称，已被修改。 Read-only 。 Can be null 。

领域

 FilterField

过滤器字段。它包含有关目标实体中已修改属性的其他信息。例如，有关链接类型或修改后的自定义字段或捆绑软件类型的信息，等等。 Read-only 。 Can be null 。

类别

 活动类别

活动的类别。 Read-only 。
