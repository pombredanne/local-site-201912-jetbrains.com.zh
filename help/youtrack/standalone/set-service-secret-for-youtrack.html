YouTrack Standalone 2019.2帮助

为YouTrack更新服务秘密

如果您的YouTrack服务被错误地从外部集线器安装中删除，则可能需要手动还原该服务。Hub文档中介绍了此过程的完整说明。

要手动还原服务，您需要在Hub中为YouTrack添加新服务并生成新的服务密钥。然后，您需要获取分配给新服务的ID和新生成的服务密钥，并在YouTrack的配置文件中更新这些值。

要更新您的YouTrack服务的服务ID和密码：

  1. 停止YouTrack服务 。

  2. 找到并打开hub.properties文件。

    • 对于ZIP分发，该文件位于/conf目录。

    • 对于MSI分发，文件位于%programdata%\JetBrains\YouTrack\conf目录。

      您可以找到到的路径conf文件夹中\internal\conf\installation.xml文件。路径被写入属性default-appdata-root 。例如：

      C：\ ProgramData \ JetBrains \ YouTrack

  3. 更新的值youtrack.service.id和youtrack.service.secret属性。例如：

    youtrack.service.id=d1199047-523c-4138-bcb0-5e9032969d95 youtrack.service.secret=gPQ9i7lbdvwExCAF0980QlzitmClE2Y6y4gZRTKjKXKkXPqdpP

  4. 重新启动YouTrack服务 。

此时，您的YouTrack服务应该成功启动。为确保一切正常，删除YouTrack服务之前，请继续按照集线器文档中的说明进行操作。
