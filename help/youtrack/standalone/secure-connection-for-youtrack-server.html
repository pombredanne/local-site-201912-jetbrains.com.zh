YouTrack服务器的安全连接

本节中描述的所有过程仅适用于YouTrack Standalone版本

有几种方法可以在YouTrack Standalone服务器及其客户端之间使用TLS（SSL）设置加密的HTTPS连接。

  • 使用第三方TLS终止反向代理服务器。有关详细信息，请参阅反向代理配置 。

  • 对YouTrack Standalone服务器使用内置的TLS。您可以将YouTrack配置为以HTTPS模式运行：

    • 在配置向导中 ，在安装或升级过程中

    • 在命令行中 ，为现有的YouTrack安装启用HTTPS

无论当前设置如何，您都可以随时从第三方TLS终止代理切换到YouTrack内置TLS，反之亦然 。
