支持的环境

此页面描述了YouTrack的硬件和软件要求。

硬体需求

YouTrack Standalone没有特定的硬件要求。这些要求根据数据库的大小和定期处理的平均事务数而有所不同。使用以下准则来估计服务器的要求。

存储

平均而言，YouTrack需要1 GB的磁盘空间来存储大约5,000个问题。但是，这仅仅是一个粗略的估计。所需空间的确切数量取决于安装问题中附带的文件的实际数量和大小。例如，如果您经常将大型媒体文件附加到每个问题上，则每个问题数据库将占用更多空间。

中央处理器

为了获得最佳性能，我们建议您在至少有两个可用处理器的计算机上运行YouTrack。例如，具有Intel Core i3处理器或类似处理器的计算机应足以启动。较大流量的较大安装可能需要更多处理能力。

记忆

YouTrack至少需要1.5 GB的RAM。

对于分配给运行YouTrack服务的Java虚拟机（JVM）的内存量有最低要求：

  • 元空间的最大内存必须至少为250m。

  • Java堆的最大大小必须至少为1024m。

默认情况下，这些配置参数适用于所有新安装的JVM。

爪哇

YouTrack是需要Java支持的Web应用程序。YouTrack与JRE / JDK版本1.8兼容。JRE / JDK的早期版本不兼容。

  • 如果运行YouTrack JAR安装，则必须在运行YouTrack服务的服务器上安装并运行兼容版本的Oracle Java Runtime Environment（JRE）或等效的Java Development Kit（JDK）。

  • 对于MSI和ZIP安装，JRE的兼容版本与安装包捆绑在一起。

您无需在通过Web界面访问应用程序的客户端计算机上安装Java。

HTTP / 2和TLS

从2018.3版本开始，YouTrack已优化为使用HTTP / 2协议。使用此协议可优化页面加载速度，并有助于支持实时更新。

尽管HTTP / 2标准不需要加密，但是所有主要的客户端实现（Firefox，Chrome，Safari，Opera，IE，Edge）都声明它们将仅支持基于TLS的HTTP / 2。为了确保最佳性能，您需要为YouTrack Standalone安装设置带有TLS（SSL）的加密HTTPS连接。

  • 如果您的YouTrack安装使用内置的TLS ，则默认情况下，服务器和客户端之间的流量使用HTTP / 2。

  • 如果将YouTrack安装在反向代理服务器后面，则强烈建议您使用TLS终止并将服务器配置为使用HTTP / 2连接。为确保优化反向代理服务器的配置以支持YouTrack安装，请遵循特定于服务器类型的准则 。

操作系统

我们尽最大努力确保YouTrack安装在互联网上公共服务器常用平台上的稳定性。但是，我们不能保证该应用程序与所有可能的环境兼容。例如，已知与IBM z / OS操作系统支持的Java运行时环境不兼容。

Windows和macOS的64位版本以及Linux和其他* nix环境的最流行发行版均支持YouTrack。我们不支持在支持生命周期已结束的操作系统版本上进行安装。

网页浏览器

YouTrack在标准的Web浏览器中运行。随着浏览器更新速度的提高，并非总是可以预先认证每个浏览器。我们解决了新版本受支持的浏览器中出现的功能性问题。

我们建议您将浏览器更新到最新版本。您将从最新的安全性和稳定性修补程序中受益。

以下网络浏览器的两个最新稳定版本支持YouTrack Standalone：

  • 铬

  • 火狐浏览器

  • 苹果浏览器

  • 边缘

  • 歌剧

从2019.2版本开始，YouTrack不再支持Internet Explorer。
