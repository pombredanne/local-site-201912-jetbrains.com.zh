External Integrations

YouTrack lets you connect with other applications in ways that enhance your productivity and improve your working processes. There are several applications that you can integrate with YouTrack directly from the Integrations section of the Administration menu. For an overview of these standard integrations, see Integrations.

The integrations that are described in this section are either set up outside YouTrack or are not accessed in the Integrations menu.

Integration

Description

Test Management Tools

Use YouTrack to track and resolve bugs that are reported in your test management tool.

Confluence Integration

Add macros that let you insert links to issues and insert reports into Confluence pages.

HipChat Integration

Post new issues and updates to one or more rooms in HipChat.

Slack Integration

Post notifications to a Slack channel by sending HTTP requests from YouTrack.

A number of third-party developers have built integrations that connect their applications to YouTrack. Check this list to see if any of the applications you use can be integrated with YouTrack. Follow the links to learn more.
