Get Accessible Projects

GET /project/all

Get a list of all accessible projects from the server.

GET /rest/project/all?{verbose}

Parameters:

Name

Type

Description

verbose

boolean

If full representation of projects is returned. If this parameter is false, only short names and id's are returned.

Sample

Request

GET https://example.com/youtrack/rest/project/all Connection: keep-alive Cookie: $Version=0; JSESSIONID=1bl78qpgiu1sf1fb4z1htutlgf; $Path=/; jetbrains.charisma.main.security.PRINCIPAL=NDgxMzQ5NGQxMzdlMTYzMWJiYTMwMWQ1YWNhYjZlN2JiN2FhNzRjZTExODVkNDU2NTY1ZWY1MWQ3Mzc2NzdiMjpyb290; $Path=/
