Update a Comment

PUT /rest/issue/{issue_id}/comment/{comment_id}

Update a comment in an issue.

PUT /rest/issue/{issue_id}/comment/{comment_id} {"text": "updated comment text"}

Parameters:

Name

Type

Description

issue

issueById

ID of an issue, to which the comment belongs.

comment

commentByid

ID of a comment to update.

text

string

The new text of the comment.

Sample

Request

PUT https://example.com/youtrack/rest/issue/HBR-60/comment/42-310 Cookie: $Version=0; JSESSIONID=j9wmm40hwk3t1lp8e11l9p0d4; $Path=/; jetbrains.charisma.main.security.PRINCIPAL=MTVhMjY2OGUxMTUxMTI5MGMzNjhlZWM5NTJiNGJkNmIyNzc3NjdjZmM3YWU4MTVjYzQwYWE1MWZlYTkwNTcxNzphcHBfZXhjZXB0aW9u; $Path=/ {"text": "This is the updated comment"}
