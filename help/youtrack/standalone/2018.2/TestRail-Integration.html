TestRail Integration

Integration between YouTrack and TestRail test case management tool is provided and supported by TestRails developers, the Gurock Software company.

Integration with TestRail test case management software allows you to link test results in TestRails to YouTrack issues. For more information about integration and settings, please refer to the YouTrack Test Management with TestRail page at the official Gurock Software website.

On the YouTrack side, the only thing you need to do is to enable the REST API:

To enable the REST API:



Low-level Admin Read

Low-level Admin Write

  1. In YouTrack web UI open the Administration > Global Settings page.

  2. Scroll down the page to the REST API settings

  3. Enter the URL of your Test Rail server in the Allowed origins input field:

All other integration settings are configured in TestRails.
