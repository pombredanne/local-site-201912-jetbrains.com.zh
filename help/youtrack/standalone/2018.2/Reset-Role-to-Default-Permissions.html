Reset Roles to Default Permissions

YouTrack defines a set of default roles and the default permissions that are assigned to each role. The default permissions are defined separately for YouTrack and the built-in Hub service.

If you have concerns that the permissions for a service are not configured correctly, you can reset the permissions to the default scheme.

To reset the default permissions for a service:



Read Role

Update Role

Read Service

  1. In the Access Management section of the Administration menu, select Roles.

  2. Select any role from the list.

  3. Select the Permissions tab.

  4. Select the application that you want to manage (Hub or YouTrack).

  5. Click the Reset to default button.

    • The list of services that are supported by the application is displayed in the sidebar.

  6. Select the services that you want to reset to the default permission scheme.

  7. Click the Reset to default button.

    • All of the default roles in the selected services are reset to their default permissions.

    • All of the permissions are removed from custom roles in the selected services.
