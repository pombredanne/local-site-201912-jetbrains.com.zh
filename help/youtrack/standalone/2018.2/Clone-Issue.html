Clone Issue

This workflow contains the action rule that is used to clone an issue. This workflow runs when you use the clone command in the Command Window.

This workflow is read only and auto-attached by default. These settings ensure that you can use the clone command for an issue in any project.

For more information, see Clone Issues.
