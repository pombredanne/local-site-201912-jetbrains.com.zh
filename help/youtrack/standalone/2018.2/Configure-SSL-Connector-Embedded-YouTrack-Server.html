YouTrack Standalone 2018.2 Help

Configure SSL Connector for Embedded YouTrack Server

YouTrack 2018.2 introduced the built-in TLS feature that made using the Jetty config file for configuring SSL Connector obsolete. If you used the Jetty configuration file, please switch to the built-in TLS or a third-party reverse proxy with SSL termination.
