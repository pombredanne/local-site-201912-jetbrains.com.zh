State Transition Reports

State transition reports let you track changes in the value of a specific custom field. These reports help you monitor issues that transition to specific stages in your development process.

The following reports are available.

Report

Description

Verified Distribution

Shows the total number of issues that transitioned to a verified state within a specified time frame. The results can be grouped by an issue attribute or the value in a custom field.

Reopened Distribution

Shows the total number of issues that transitioned from a resolved state to an unresolved state within a specified time frame. The results can be grouped by an issue attribute or the value in a custom field.

State Transition

Tracks the change in value for a custom field within a specified time frame. The results can be grouped by an issue attribute or the value in a custom field.
