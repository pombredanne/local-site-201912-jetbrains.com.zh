Secure Connection for YouTrack Server

All procedures that are described in this section are applicable for YouTrack Standalone edition, only.

There are several ways to set up an encrypted HTTPS connection with TLS(SSL) between your YouTrack Standalone server and its clients.

  • Use a third-party TLS-terminating proxy server. See Proxy Configuration for details.

  • Use built-in TLS for YouTrack Standalone server. You can configure YouTrack to run in the HTTPS mode:

    • in Configuration Wizard, during the installation or upgrade

    • from the command line, enabling HTTPS for your existing YouTrack installation

You can switch from a third-party TLS-terminating proxy to the YouTrack built-in TLS and vice-versa at any time regardless of your current settings.
