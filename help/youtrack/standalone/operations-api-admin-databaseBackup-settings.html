具有特定DatabaseBackupSettings的操作

该资源使您可以配置数据库备份的设置。

资源资源
/ api / admin / databaseBackup / settings

退货实体

DatabaseBackupSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的DatabaseBackupSettings 。

  • POST ： 更新特定的DatabaseBackupSettings 。

DatabaseBackupSettings属性

代表YouTrack实例的数据库备份设置。

下表描述了DatabaseBackupSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

地点

串

备份存储文件夹的位置。 Can be null 。

filesToKeep

整数

应该存储的备份文件数。

cronExpression

串

定义自动备份计划的cron表达式。 Can be null 。

archiveFormat

列举

当前选择的备份文件格式。支持的值为TAR_GZ或ZIP。

isOn

布尔型

如果为true，则启用自动常规备份。否则为“ false”。

availableDiskSpace

长

可用磁盘空间的大小。 Read-only 。

notifyedUsers

 用户群

收到有关备份失败的通知的用户列表。

backupStatus

 备份状态

备份的当前状态。 Read-only 。

阅读特定的DatabaseBackupSettings

阅读数据库备份设置。

所需权限

需要低级管理员读取权限。

请求语法

GET / api / admin / databaseBackup / settings？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的DatabaseBackupSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/databaseBackup/settings?fields=archiveFormat，可用磁盘空间，backupStatus（backupCancelled，backupError（date，errorMessage），backupInProgress，stopBackup），isOn，位置，cronExpression，默认位置， filesToKeep，notifiedUsers（登录名，名称，ID）
