仪表板

此页面描述了YouTrack仪表板的功能。如果您的YouTrack实例链接到外部集线器服务，则可以访问集线器中的仪表板。

集线器安装中集成的每个JetBrains团队工具都可以使用不同的小部件。要了解更多信息，请阅读Hub帮助 。

仪表板是一种方便的方法，可用于概述任务的当前状态并监视团队的进度。

标准搜索框显示在仪表板顶部。如果要快速访问特定问题列表，可以直接从仪表板输入搜索查询。

仪表板是YouTrack中的默认起始页。要从YouTrack的任何其他页面打开仪表板，只需单击YouTrack徽标或公司的徽标（如果您的实例使用自定义徽标）。每个用户都有一个名为“ 我的仪表板”的个人仪表板 。您可以编辑，重命名和共享您的个人仪表板，也可以自己保存。还有一个默认仪表板，可用于向来宾用户显示信息。有关更多信息，请参见Guest Dashboard 。

小部件

仪表板在卡上显示小部件。YouTrack仪表板支持以下类型的小部件：

  • Markdown笔记小部件
  • Wiki笔记小部件
  • 问题列表小部件
  • 报告小部件

您可以根据需要将每种类型的小部件添加到仪表板中。

要将小部件添加到仪表板：

  1. 单击添加窗口小部件按钮。

  2. 选择要添加到仪表板的窗口小部件的类型。

  3. 配置窗口小部件的设置。

    • 对于报告窗口小部件，选择要在窗口小部件中显示的报告。

    • 对于问题小部件，输入搜索查询或选择保存的搜索，项目或标签。单击创建窗口小部件按钮。

    • 对于注释小部件，在输入字段中输入文本。单击创建窗口小部件按钮。

    —该小部件已添加到您的仪表板。

多个仪表板

您可以创建和查看多个仪表板，每个仪表板都有自己的小部件集。避免您为不同项目设置和监视仪表板。

当当前仪表板属于您的仪表板列表时，列表中的仪表板数量将显示在当前仪表板的名称下方。还显示列表中上一个和下一个仪表板的名称。

按Shift + Left要么Shift + Right在仪表板之间切换。

按Shift + Down扩展仪表板下拉列表。

要添加另一个仪表板：

  1. 单击当前仪表板名称旁边的向下箭头，然后选择创建新仪表板选项。

    或者，按Alt + Shift + N在键盘上。

  2. 在“ 新建仪表板”对话框中输入新仪表板的名称。

  3. 单击创建仪表板按钮。

    • 新的空白仪表板将添加到YouTrack。

共享仪表板

除了您自己创建的仪表板之外，您还可以查看与您共享的仪表板。

当您的团队成员直接与您或您所属的组共享一个仪表板时，您会在当前仪表板名称旁边看到一个链接。单击此链接时，将显示共享仪表板的名称以及共享仪表板的用户的名称。

  • 要将此仪表板添加到列表中，只需单击星形图标，或选择共享的仪表板，然后单击“ 添加到我的仪表板”按钮。

  • 要从仪表板上删除通知链接，请单击下拉列表中的关闭链接。

    即使尚未将共享仪表板添加到列表中，您仍然可以查看它。

  • 要从列表中删除仪表板，请打开仪表板列表，然后单击星形图标。您仍然可以从列表中访问仪表板，但是该仪表板已从可以使用键盘快捷键或标题中的链接访问的仪表板列表中删除。

当前共享选项在仪表板标题中显示为链接。

  • 如果仪表板与单个用户或组共享，则该用户或组的名称将显示在链接中。

  • 如果仪表板与多个用户或组共享，则该链接显示可以访问仪表板的用户和组的数量。单击链接以显示可以访问仪表板的用户和组的完整列表。

  • 如果其他用户与您共享了仪表板，则该链接中还将显示仪表板所有者的名称。

有关更多信息，请参见共享仪表板 。

重新订购收藏的仪表盘

您添加到列表中的仪表板和共享仪表板显示为收藏的仪表板的集合。您可以使用键盘快捷键快速浏览喜爱的仪表板列表。

为确保这些仪表板按逻辑顺序显示，您可以在此列表中手动对仪表板进行重新排序。

要对收藏的仪表板列表进行重新排序：

  1. 单击标题中的“ 仪表板”链接。

  2. 展开仪表板列表，然后单击“ 重新排列收藏夹仪表板”链接。

    • 将打开“ 重新收藏收藏夹仪表板”对话框。

  3. 将手柄拖到仪表板名称旁边，然后将其放到所需的位置。

  4. 继续拖动，直到每个仪表板按所需顺序显示。

  5. 单击“ 应用”按钮。

    • 自定义订单将应用于您喜欢的仪表板列表。

有关可用于在仪表板界面中浏览的快捷方式的完整列表，请参阅仪表板键盘快捷方式 。

整理仪表板

要组织仪表板，所有您需要做的就是将卡拖到仪表板上，并按照您喜欢的顺序放置。

您可以在仪表板上调整卡的大小。拖动卡的边缘以设置高度和宽度。卡的大小会自动对齐到网格的最近交点。

仪表板的其他操作

除了可以将小部件添加到仪表板的选项之外，还可以执行以下操作：

选项

描述

分享

您可以配置仪表板并与其他用户或组共享。有关更多信息，请参见共享仪表板 。

改名

您可以随时更改仪表板的名称。要访问此选项，请单击编辑>重命名 。

删除

如果您不想再使用列表中的仪表板，则可以将其删除。要访问此选项，请单击编辑>删除 。

小部件的其他操作

您可以使用多种操作来自定义仪表板并与团队共享其内容。要打开可用于窗口小部件的操作列表，请单击标题栏中的向下箭头。

如果窗口小部件包含您希望团队中的每个成员一起使用的信息，则可以共享该窗口小部件。

共享小部件：

需要权限：“读取用户基本信息”，“读取组”

  1. 单击要共享的小部件标题栏中的向下箭头。

  2. 选择共享窗口小部件选项。

  3. 单击“ 共享小部件”对话框中的“ 复制”按钮，然后将其发送给您的队友。

    如果与您共享窗口小部件的人无权查看窗口小部件中使用的数据，则窗口小部件为空。

您可以在仪表板上创建现有小部件的副本。当您想添加具有微小更改的类似小部件时，此选项非常有用。

克隆小部件：

  1. 单击要克隆的小部件标题栏中的向下箭头。

  2. 选择克隆小部件选项。

    • 小部件的副本将添加到您的仪表板。

您可以将小部件从当前仪表板移动到另一个仪表板。

要将小部件移动到另一个仪表板：

  1. 单击要移动到另一个仪表板的小部件标题栏中的向下箭头。

  2. 选择移至另一个仪表板选项。

  3. 在“ 将小部件移动到另一个仪表板”对话框中，选择目标仪表板的名称。

  4. 单击移动按钮。

    • 小部件将移动到所选的仪表板。

如果要修改仪表板上的小部件设置，则可以随时对其进行编辑。

要编辑现有的小部件：

  1. 单击要编辑的小部件标题栏中的向下箭头。

  2. 选择编辑选项。

  3. 修改小部件的设置或内容。

    • 对于报告窗口小部件，选择要在窗口小部件中显示的报告。小部件将显示选定的报告。

    • 对于问题窗口小部件，请输入新的搜索查询或选择其他保存的搜索，项目或标签。单击“ 应用”按钮以更新小部件。

    • 对于注释小部件，请在输入字段中编辑文本。单击“ 应用”按钮以更新小部件。

如果您不再希望在仪表板上看到窗口小部件，则可以将其删除。

要从仪表板上删除窗口小部件：

  1. 单击要从仪表板删除的小部件标题栏中的向下箭头。

  2. 单击删除选项。

  3. 单击确认对话框中的删除按钮。

    —小部件已从仪表板上删除。
