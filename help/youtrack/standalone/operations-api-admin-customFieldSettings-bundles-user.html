特定UserBundle的操作

该资源使您可以使用YouTrack中的用户集。

资源资源
/ api / admin / customFieldSettings / bundles / user / {bundleID}

退货实体

UserBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的UserBundle 。

  • POST ： 更新特定的UserBundle 。

  • DELETE ： 删除特定的UserBundle 。

UserBundle属性

表示包含用户的一组值。您可以将单个用户帐户和用户组添加到集合中。

扩展捆绑

下表描述了UserBundle实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

团体

 用户组数组

属于该捆绑软件的用户组。

个人

 用户群

属于此捆绑包的个人用户。

聚合用户

 用户群

属于此捆绑包的所有用户。 Read-only 。

isUpdateable

布尔型

如果true ，则当前登录的用户可以更新此捆绑包。除此以外， false 。 Read-only 。

读取特定的UserBundle

获取具有特定ID的用户集。

所需权限

需要对使用捆绑软件的字段之一进行读取或更新访问。如果未在任何字段中使用捆绑软件，则至少在一个项目中需要“更新项目”权限。
