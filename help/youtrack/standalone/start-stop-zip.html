开始和停止ZIP安装

本页介绍如何停止和启动YouTrack Standalone服务器。

以下情况要求您停止YouTrack：

  • 您要更改数据库的位置。

  • 您想要修改YouTrack Standalone服务器的属性或JVM选项。

  • 您要升级YouTrack Standalone安装。

如果您使用ZIP发行版安装了YouTrack，则可以从命令行运行YouTrack。您用于启动和停止YouTrack的脚本位于/bin/目录。的 directory是安装过程中解压缩ZIP分发包的位置。

要在macOS或Linux平台上停止和启动YouTrack，请以非root用户身份执行命令。

此页面上的所有命令均适用于Linux和macOS。如果您使用的是Windows服务器，请更换.sh与.bat并使用反斜杠。

在命令行界面中输入命令来停止和启动YouTrack服务。使用以下命令来管理YouTrack Standalone实例的状态：

命令

描述

youtrack.sh run

启动YouTrack服务。使用此命令，控制台将被阻止，直到使用停止服务为止。 youtrack.sh stop命令。

youtrack.sh start

启动YouTrack服务。使用此命令，仅在启动过程中阻止控制台。

youtrack.sh stop

停止YouTrack服务。

youtrack.sh restart

重新启动YouTrack服务。

youtrack.sh help

显示可以在命令行界面中执行的所有命令的列表。
