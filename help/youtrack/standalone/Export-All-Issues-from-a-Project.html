导出项目中的所有问题

GET / export / {project} /问题

从项目中获取所有注释问题。

要使用Export REST API方法，用于登录YouTrack的用户帐户必须具有“ 低级读取管理”权限。

GET / rest / export / {project} /问题？{max}＆{after}

参数：

名称

类型

描述

项目

projectById

导出问题的项目的ID。

最高

整数

导出的最大数量。如果为空，则导出10个问题。

此请求返回的最大问题数还受到YouTrack服务的“ 要导出的最大问题数”设置的值的影响。此设置位于“ 全局设置”页面上。例如，如果您使用max=10000 REST API请求中的参数，但“ 要导出的最大问题数”设置限制为5000个问题，响应最多返回5000个问题。

后

整数

收集要导出的问题列表之前要跳过的问题数。
