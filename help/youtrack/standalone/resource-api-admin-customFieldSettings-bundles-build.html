资源资源
/ api / admin / customFieldSettings / bundles / build

退货实体

BuildBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读BuildBundles列表 。

  • POST ： 添加一个新的BuildBundle 。

BuildBundle属性

代表YouTrack中的一组内部版本。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

阅读BuildBundles列表

获取系统中所有可用的构建集。

请求语法

参数

整数
