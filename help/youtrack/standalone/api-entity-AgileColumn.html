代表单板列的设置

下表描述了AgileColumn实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

介绍

串

列中存储的值的文本表示。 Read-only 。 Can be null 。

isResolved

布尔型

如果列代表问题的已解决状态，则为true。只能针对新创建的值进行更新。 Read-only 。

序数

整数

该列在船上的顺序，从左到右。

父母

 列设置

链接到此列所属的敏捷列设置。 Read-only 。 Can be null 。

wipLimit

 在制品限制

此列的在制品限制。 Can be null 。

fieldValues

 AgileColumnFieldValues的数组

此列表示的字段值。
