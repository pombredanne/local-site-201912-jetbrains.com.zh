设置密码策略

使用集线器身份验证模块，您可以设置一个密码策略，该策略将在用户每次创建或更改其YouTrack帐户密码时强制执行。

为YouTrack设置密码策略时，可以提高系统的整体安全性：

  • 遵守您的应用程序，系统或组织定义的最低密码要求。

  • 保护您的YouTrack实例和连接的服务免受暴力攻击。

  • 为用户提供指导方针，以帮助他们创建易于记忆的强密码。

背景资料

计算机用户必须记住多个应用程序的密码。您可能已经被要求多次创建唯一且符合特定要求的密码。与大多数人一样，您选择的模式将在下次登录时记住。不幸的是，这些模式对于密码破解者来说很明显，因为它们很容易记住。

熵作为密码强度的度量

设置密码策略时，将确定使用集线器身份验证登录的用户创建的密码的最小熵。较强的密码具有较高的熵。

熵是对找到密码所需的猜测次数的估计，以熵位为单位。给密码加上一点熵会使所需的猜测次数加倍。

对于在线暴力攻击，您可以看到强密码极大地提高了系统的安全性。强度很高的密码需要数十年才能破解，而强度较低的密码将在数小时内被发现。如果密码属于允许黑客访问您的数据库的帐户，则他们可以在更短的时间内离线分析它。

单个弱密码可以打开未经授权访问的大门。当您对密码设置更高的要求时，此门将关闭。

创建强密码的准则

通过遵循以下准则，YouTrack用户可以提高其密码强度：

  • 创建一个包含多个单词的密码。

    大写和字符替换使猜测基于字典中单个单词的密码变得更加困难。即使不使用特殊字符，也可以通过使用意外顺序的不相关单词字符串来极大地提高密码的熵。

    当您在已经非常强大的密码中添加另一个单词时，可能要花几个世纪的时间才能猜测出来。

  • 添加一些无法预测的替换以使单词难以识别，或在单词中间插入一个特殊字符。更改单词的顺序以避免逻辑模式。

  • 策略性地包括小写和大写字母字符，数字和符号。例如，您可以将密码短语中的第四个字母大写，而不是每个单词的第一个字母大写。

  • 至少使用12到14个字符。

  • 避免重复字符，键盘模式，字典单词，字母或数字序列，用户名，亲戚或宠物的名字以及传记信息。

  • 避免使用与帐户用户相关联的信息，例如生日或周年纪念日。

当用户在YouTrack中设置或更改密码时，会将密码与超过30,000个常用密码，名称，姓氏和常用英语单词的数据库进行比较并权衡。如果密码包含以上任何一个，或包含此处描述的任何问题，则YouTrack可以识别问题并提供创建更强密码的准则。

YouTrack使用zxcvbn开发的逼真的密码强度估算来测量密码的强度。您可以在相关文章中阅读有关它的更多信息。

使用说明

您可以为集线器身份验证模块设置密码策略，该策略确定用户帐户所需的最低密码强度。

设置密码策略：

需要权限：低级管理员写

  1. 在管理菜单的访问管理部分，选择身份验证模块 。

  2. 选择集线器身份验证模块。

  3. 从“ 密码强度”下拉列表中，选择以下选项之一：

    选项

    描述

    没有政策

    用户可以输入任何密码。

    弱

    密码的最小熵必须为20-26位。

    好

    密码必须具有至少26-33位的熵。

    非常强壮

    密码的熵必须大于33位。

    而已！每次用户为自己的YouTrack帐户创建或更改密码时，都会执行选定的策略。现有密码不符合新策略的用户在登录时要求更改其密码。
