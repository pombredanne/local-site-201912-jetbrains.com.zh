YouTrack Standalone 2019.2帮助

在Linux上将YouTrack JAR作为服务安装

JetBrains开发团队已使用本安装指南中提供的说明在Linux环境中安装和运行YouTrack JAR。

先决条件

要执行此安装，请验证是否满足以下先决条件：

  • 您已经安装了Oracle Java SE Runtime Environment 8或更高版本。这里的说明要求JRE在路径中可用/usr/bin/java 。

  • 任何外部主机名（代理主机名）都可以从安装了YouTrack的实际主机中解析出来。

安装

以下过程描述了如何在Linux或其他* nix操作系统上作为服务安装和运行YouTrack JAR。

要将YouTrack JAR发行版作为服务安装在Linux上，请执行以下操作：

  1. 创建一个用户帐户以运行YouTrack服务。例如， youtrack 。的youtrack用户帐户管理YouTrack数据和服务。

    adduser youtrack-禁用密码

  2. 使用以下命令下载最新版本的JAR发行版：

    wget -O /home/youtrack/youtrack.jar https://download.jetbrains.com/charisma/youtrack- .jar

    更换当前版本号和内部版本号。例如， 2018.2.41826 。

    您可以找到当前版本，并在YouTrack Standalone的下载页面上进行构建 。或者，您可以手动下载JAR文件并将其保存在/home/youtrack/youtrack.jar 。

  3. 使用以下命令设置JAR文件的权限：

    chown youtrack：youtrack /home/youtrack/youtrack.jar

  4. 创建一个systemd服务文件位于以下目录中：/etc/systemd/system/youtrack.service

    将以下内容粘贴到youtrack.service文件：

    [单位]说明= Youtrack要求= network.target之后= syslog.target network.target [服务]类型=简单WorkingDirectory = / home / youtrack ExecStart = / usr / bin / java -jar /home/youtrack/youtrack.jar- -J-Xmx1G 8080 User = youtrack [安装] WantedBy = default.target

    根据需要自定义文件的内容：

    • 如果您希望YouTrack从其他端口号开始，请编辑默认端口8080 。

    • 根据需要添加和修改其他Java参数。

  5. 使用以下命令重新加载systemd守护程序并在系统启动时启用YouTrack服务：

    systemctl守护进程重新加载systemctl启用youtrack.service

  6. 输入以下命令以运行YouTrack服务：

    服务youtrack开始

  7. 打开http://:8080在网络浏览器中，等待应用程序可用。由于启动应用程序需要花费一些时间，因此端口可能会在短时间内无法访问。

安装后，您可以将YouTrack设置为在反向代理服务器之后运行。有关说明，请参阅反向代理配置 。

升级

如上所述安装JAR发行版后，升级到更高版本相对简单。您需要做的只是：

  • 删除JAR文件的先前版本。

  • 下载最新版本。

  • 设置文件的权限。

  • 重新启动服务。

所有这些都可以通过执行以下命令来完成：

mv /home/youtrack/youtrack.jar /home/youtrack/youtrack-old.jar wget -O /home/youtrack/youtrack.jar https://download.jetbrains.com/charisma/youtrack- .jar chown youtrack：youtrack /home/youtrack/youtrack.jar服务youtrack重新启动
