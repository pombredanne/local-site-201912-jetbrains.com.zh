该模块支持标准Java库中的特定字符串实用程序。可用于工作流中的字符串的所有其他方法均来自标准JavaScript库，并且其行为与MDN文档中所述相同 。

功能


getLevenshteinDistance

静态getLevenshteinDistance（str1，str2）

返回两个字符串之间的Levenshtein距离。代表org.apache.commons.lang。StringUtils.getLevenshteinDistance 。

参量

名称

类型

描述

str1

 string

第一个字符串。

str2

 string

与第一个字符串进行比较的字符串。
