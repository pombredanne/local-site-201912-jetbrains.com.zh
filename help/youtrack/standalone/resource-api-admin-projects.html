YouTrack Standalone 2019.2帮助

专案

此资源提供对项目的访问。

资源资源
/ api / admin / projects

退货实体

项目 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读项目清单 。

  • POST ： 添加一个新项目 。

支持的子资源

  • / api / admin / projects / {projectID}： 特定项目的操作

  • / api / admin / projects / {projectID} / timeTrackingSettings： 项目时间跟踪设置

  • / api / admin / projects / {projectID} /问题： 问题

  • / api / admin / projects / {projectID} / customFields： 项目自定义字段

项目属性

代表一个YouTrack项目。

扩展IssueFolder

下表描述了Project实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

startingNumber

长

项目中问题的起始编号。仅在创建新项目期间可以设置此属性。

简称

串

项目的ID。该短名称也是问题ID的前缀。 Can be null 。

描述

串

在项目资料页面上显示的项目描述。 Can be null 。

领导

 用户

设置为项目负责人的用户。 Can be null 。

由...制作

 用户

创建项目的用户。 Can be null 。

问题

 一系列问题

属于项目的所有问题的列表。

customFields

 ProjectCustomField

项目中可用的自定义字段集。

已封存

布尔型

如果项目当前已归档，则此属性为true。

来自电子邮件

串

用于发送项目通知的电子邮件地址。如果未为项目设置“发件人”地址，则返回YouTrack服务器的默认“发件人”地址。

ReplyToEmail

串

用作回复电子邮件以发送项目通知的电子邮件地址。如果未为项目设置，则返回YouTrack服务器的默认地址。 Can be null 。

模板

布尔型

如果true ，该项目是模板。

iconUrl

串

项目图标的URL。 Read-only 。

名称

串

问题文件夹的名称。 Can be null 。

阅读项目清单

获取系统中所有可用项目的列表。

请求语法

GET / api / admin / projects？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

添加一个新项目

创建新项目时，必须提供项目的名称，简称和项目负责人。其他属性是可选的。要创建基于模板（例如Scrum或看板）的项目，您需要添加template端点URL的查询参数。

所需权限

需要“创建项目”权限。

请求语法

POST / api / admin / projects？{字段}＆{template}

请求参数

参数

类型

描述

领域

串

应在响应中返回的Project属性列表。如果未指定任何字段，则仅entityID返回

模板

串

如果template未指定，则新项目将使用默认设置。可选的 。使您可以指定用于新项目的模板。可能的值： scrum ， kanban 。
