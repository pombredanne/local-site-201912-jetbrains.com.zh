新发行视图

用于报告问题的页面被精简，仅包括在YouTrack中创建问题所需的页面。要访问此页面，请单击应用程序标题中的“ 新发行”按钮。这是此问题视图中支持的功能的概述。

项目选择器

页面标题包含一个下拉列表，用于选择报告问题的项目。

YouTrack中分配给问题的许多属性是在项目级别定义的。如果您只能访问一个项目，则此字段的值将自动设置。否则，将为您选择最近创建问题的项目。

以下因素影响列表中显示的项目的顺序。

  • 如果将项目设置为搜索上下文，则该项目将用作新问题的默认值。有关更多信息，请参见项目的搜索上下文 。

  • 接下来，您将在“ 项目”列表中看到已标记为收藏夹的项目 。

  • 在您喜欢的项目之后，所有其他项目都按字母顺序列出。

每个问题都属于一个项目，您永远无法选择一个空值。更改此值时，可用的自定义字段集也会更改。

问题工具栏

工具栏提供对单个问题视图中支持的相同操作的访问。

可以执行以下操作：

行动

描述

打开命令对话框

打开“ 应用命令”对话框。在这里，您可以使用以键盘为中心的命令语法来更新问题草稿。

添加链接

添加从问题草稿到另一个问题的链接。从下拉列表中选择链接类型以打开“ 应用命令”对话框，然后输入目标问题的ID。

添加标签

将标签添加到问题草稿。从列表中选择一个现有标签。在输入字段中输入一个值以过滤标签列表。

分配问题

设置问题草稿中“ 受让人”字段的值。从列表中选择一个用户。在输入字段中输入值以过滤用户列表。

展示更多

打开其他动作的列表。此菜单还提供对工作流支持的自定义命令的访问。

有关详细信息，请参阅显示更多菜单 。

清除草稿

清除问题摘要和描述。将所有自定义字段重置为其默认值。

显示更多菜单

在“ 显示更多”菜单中可以执行以下操作：

控制

描述

自定义动作

“ 显示更多”菜单中的第一项是工作流支持的自定义操作。每个问题中可用的一组操作由附加到当前项目的工作流确定。有关更多信息，请参阅操作规则 。

附加档案

打开一个对话框，可让您从本地目录中查找并附加文件。

私下附加文件

打开“私人附加文件”对话框。在这里，您可以选择将所有文件的可见性限制为特定的组和/或用户。

对包含敏感信息的文件使用此选项。

重复上一条命令

在输入字段中使用最新命令打开“ 应用命令”对话框。

草稿管理

“ 新问题”页面使您可以访问未报告的问题草稿列表。

从草稿下拉列表中，您可以执行以下任何操作：

  • 查看哪些草稿具有标签和附件。

  • 单击创建链接以立即从草稿中创建问题。

  • 单击“ 删除草稿”选项以删除草稿并将其从列表中删除。

  • 单击全部删除按钮删除所有未报告的草稿并从头开始。

有关更多信息，请参见加载发货草稿 。

发行摘要

每个问题都必须包含摘要。

在键入时，YouTrack开始搜索摘要中类似文本的其他问题。这些问题显示在页面的“ 类似问题”部分中。有关更多信息，请参见类似问题 。

问题说明

摘要下方是问题说明的输入字段。如果问题摘要具有足够的描述性，则可以跳过它-说明不是强制性的。但是，当问题更复杂时，通常在更详细地描述问题时有助于更快地解决问题。

键入时，类似问题的列表将更新，以在问题描述中显示具有匹配文本的其他问题。页面的“ 预览”部分向您显示了在创建内容时如何在问题中格式化。

语法菜单使您可以选择以Markdown或YouTrack Wiki语法设置问题描述的格式。Markdown是在YouTrack中格式化文本的推荐选项。YouTrack Wiki标记仍受支持，但随着时间的推移将不建议使用并逐步淘汰。有关格式选项的完整列表，请参见Markdown语法 。

补充文字栏位

新的问题视图支持一种新型的文本自定义字段。这些字段存储一串字符串，当问题显示时，这些字符串将用Markdown语法解释。不支持YouTrack Wiki格式。

此类型的字段不会与其他自定义字段一起显示在边栏中。相反，它们显示在问题说明的下方。

在这些字段中输入值时，输入预览会以Markdown格式显示。有关格式选项的完整列表，请参见Markdown语法 。

附件

“ 新发行”页面上有一个放置区，用于将文件添加为新发行的附件。

  • 单击放置区域以从本地目录中选择并附加文件。

  • 将文件拖到放置区。

  • 使用适用于您的操作系统的粘贴命令来附加剪贴板中的图像。

附加一个或多个文件后，附件将在页面的“ 附件”部分中显示为缩略图。

要了解有关附件的更多信息，请参阅附件 。

问题可见性

在“ 附件”部分下面是一个菜单，可用于设置新问题的可见性。默认情况下， 所有用户组的成员都可以看到该问题。这意味着该组中有权访问所选项目中问题的任何成员都可以查看此问题。

要将新问题的显示范围限制为特定用户和组，请从列表中选择这些用户和组。

要了解有关问题可见性的更多信息，请参见设置问题，评论和附件可见性 。

类似问题

“ 新问题”页面包含一个部分，显示可能与您当前报告的问题相似的问题。也许其他用户遇到了相同的问题，并且已经创建了一个问题来解决。在这种情况下，您可以将注释添加到现有问题中，并避免创建重复项。

输入问题摘要时，YouTrack搜索其他具有相似文本的问题。在问题描述中输入文本时，列表也会更新。

更新的版本包括以下增强功能：

  • 相似问题的列表受到限制，在可滚动列表中最多显示五个问题。这样可以确保该列表不会隐藏显示问题预览的页面部分。

  • 您可以添加Star标记并投票给类似的问题，而无需离开页面。

有关更多信息，请参见查找类似问题 。

自定义字段

侧面板包含附加到当前项目的自定义字段的列表。许多字段被配置为使用默认值。还可以将字段配置为要求输入新问题。

要设置或更改自定义字段的值，请单击该字段，然后从列表中选择一个或多个值或输入一个值。

用户个人资料中的选项可让您决定是在页面的左侧还是右侧显示此面板。有关更多信息，请参见YouTrack常规配置文件设置 。

观察者

自定义字段面板下方是一组Watchers控件。

  • 点击观看问题链接，将自己添加到观看者列表。这还会将Star标签添加到您帐户的问题中。

  • 如果您在项目中具有“ 添加观察者”权限，则可以单击>图标将用户添加到观察者列表中。

有关更多信息，请参见观看问题 。

板子

在“ 监视者”列表下方是董事会的一组控件。

  • 单击>图标查看敏捷板的列表，在报告该问题时将自动将其添加到其中。此列表中的这些板配置为使用过滤器自动添加问题。当前问题与这些板上的过滤器查询相匹配。

  • 单击添加到板子链接以将问题添加到特定板子。该操作将打开“ 应用命令”对话框，其中包含add Board输入字段中的命令。要完成操作，请添加目标板的名称并应用命令。

新问题的行动

“ 新发行”页面上有四个操作按钮。这些操作确定页面上的信息发生了什么。

行动

描述

清除草稿

清除问题摘要和描述。将所有自定义字段重置为其默认值。

创建

报告所选项目中的当前问题。

取消

关闭当前问题，而不向项目报告。问题已保存到您的草稿列表中。

当关闭浏览器选项卡或窗口而不单击“ 创建”按钮时，将执行相同的操作。

删除

关闭当前问题，而不报告给项目或将其保存到草稿列表中。

问题模板

“ 新问题”页面上的另一个选项使您可以生成问题模板而不报告问题。当您要创建具有相同基本属性集的多个问题或为一组用户标准化问题报告时，请使用此选项。

要了解有关问题模板的更多信息，请参阅生成问题模板URL 。
