GET Users

GET /admin/user

Get a list of available registered users. First 10 users are returned by default.

GET /rest/admin/user?{group}&{role}&{project}&{permission}&{start}

Parameters:

Name

Type

Description

group

string

filter by group (groupID)

role

string

filter by role

project

string

filter by project (projectID)

permission

string

filter by permission

start

Integer

paginator mode (returns 10 records)

Sample

Request:

HTTP GET: /admin/user

Response:

<userRefs> <user login="guest" url="https://example.com/youtrack/charisma/rest/admin/user/guest"/> <user login="newroot" url="https://example.com/youtrack/charisma/rest/admin/user/newroot"/> <user login="root" url="https://example.com/youtrack/charisma/rest/admin/user/root"/> </userRefs>
