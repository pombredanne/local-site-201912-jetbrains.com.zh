YouTrack Standalone 2019.2帮助

具有特定外观设置的操作

此资源使您可以使用YouTrack服务的“视觉”设置。

资源资源
/ api / admin / globalSettings / appearanceSettings

退货实体

外观设置 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的AppearanceSettings 。

  • POST ： 更新特定的AppearanceSettings 。

AppearanceSettings属性

表示YouTrack服务的视觉设置。

下表描述了AppearanceSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

时区

 时区描述符

默认申请时区

dateFieldFormat

 DateFormatDescriptor

日期显示的默认格式。

商标

 商标

应用程序徽标。 Read-only 。

阅读特定的外观设置

阅读YouTrack服务的视觉设置。

请求语法

GET / api / admin / globalSettings / appearanceSettings？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的AppearanceSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/appearanceSettings?fields=dateFieldFormat(datePattern,pattern,presentation),timeZone(id,presentation,offset）

样品响应体

{“ timeZone”：{“ presentation”：“莫斯科标准时间（UTC / GMT +3小时）”，“ id”：“ Europe / Moscow”，“ offset”：180，“ $ type”：“ TimeZoneDescriptor”}， “ dateFieldFormat”：{“ pattern”：“ d MMM yyyy HH：mm”，“ presentation”：“默认日期格式（2000年12月31日23:59）”，“ datePattern”：“ d MMM yyyy”，“ $ type” ：“” DateFormatDescriptor“}，” $ type“：” AppearanceSettings“}

更新特定的外观设置

更改您的YouTrack服务的视觉设置。

所需权限

需要低级管理员写权限。

请求语法

POST / api / admin / globalSettings / appearanceSettings？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的AppearanceSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/appearanceSettings?fields=dateFieldFormat(id,presentation),timeZone(id,presentation）
