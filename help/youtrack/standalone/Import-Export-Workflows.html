导入和导出工作流程

在YouTrack中，您可以导入在外部JavaScript编辑器中编写或从GitHub中的自定义工作流程存储库下载的工作流程。

您也可以从YouTrack导出工作流程。当您具有要导入另一个YouTrack实例的工作流程时，请使用此选项。您也可以导出工作流程并与YouTrack支持团队共享以进行故障排除。

工作流作为ZIP文件导入和导出。每个ZIP文件包含一个工作流程。

作为管理员，您可以直接在“ 工作流”管理页面上执行这些操作。项目管理员还可以将工作流导入并附加到他们有权查看和更新的项目中。可从“ 编辑项目”页面的“ 工作流”选项卡访问这些设置。有关更多信息，请参见管理工作流 。

导入工作流程

如果要使用存储库中的自定义工作流程或使用外部JavaScript编辑器编写的工作流程，则需要将该工作流程导入YouTrack。

导入工作流程：

需要权限：更新项目

  1. 在管理菜单的核心功能部分中，选择工作流程 。

  2. 单击导入工作流程按钮。

  3. 选择包含要导入的工作流程的ZIP文件。

  4. 单击确定按钮。

    工作流已导入，可以附加到一个或多个项目。

从YouTrack导出工作流程

如果要导出工作流以在另一个YouTrack实例中使用，或者需要与支持共享工作流，则可以从YouTrack服务器中将其导出。

您还可以导出工作流，以便在支持JavaScript的IDE（如JetBrains WebStorm）中进行编辑。

要从YouTrack导出工作流程：

需要权限：更新项目

  1. 在管理菜单的核心功能部分中，选择工作流程 。

  2. 从列表中选择一个或多个工作流程。要导出多个工作流程，必须将浏览器配置为允许YouTrack实例的弹出窗口。

  3. 单击导出按钮。

    选定的工作流程从服务器导出，并作为ZIP文件下载到本地目录。
