YouTrack Standalone 2019.2帮助

什么是新的

我们不断为YouTrack提供更新和改进。以下是此版本中可用更改的概述。

新的功能

特征

描述

可用性

每个问题类型的状态机工作流

现在，您可以编写状态机规则，这些规则根据指定字段中的当前值强加不同的状态转换集。这使您可以将约束应用于同一项目中不同问题类型的问题状态。

2019.2

通用第二因素认证

当您通过第二个身份验证因素保护YouTrack时，可以将您的帐户与符合FIDO2的硬件设备配对。这为一系列硬件安全密钥和生物识别技术提供了支持。

2019.2

全系统消息标语

我们添加了一个全局设置，该设置在YouTrack的每个页面的顶部显示一个横幅。此功能替换了以前支持的选项，该选项通过将文件上传到服务器来添加系统范围的横幅。使用此功能可以发布消息，提醒用户注意系统范围的事件，例如性能问题和计划的维护。

2019.2

更新和增强

特征

描述

可用性

搜索有关用户帐户的建议

我们优化了确定哪些用户帐户显示为搜索查询，命令，观察者和提及的建议值的逻辑。因此，默认情况下，所有安装类型均支持以前以实验为基础的“ 严格建议”功能。

2019.3.60865

历史项目编号

YouTrack现在将ID的集合存储为项目中以前用作问题前缀的ID。当您更改项目ID时，此项目中存在的所有问题的ID也会更新。

有了此更新，指向使用历史项目ID的问题的外部链接将重定向到使用当前项目ID的问题。这意味着您可以自由更改项目ID，而无需断开任何链接。

2019.2.57829

TeamCity集成的基于令牌的授权

与TeamCity集成的设置现在接受用于授权的令牌，而不是用户名和密码。这意味着您不必在集成设置中存储您的个人密码，并且可以在不中断集成的情况下自由更新密码。

2019.2.57829

持续改进问题清单

问题列表的最新更新提高了自定义字段和标签的可读性。我们还使搜索结果中与您在基于文本的搜索查询中使用的术语相匹配的单词形式重新突出显示。

此更新还从智能过滤器中删除了已保存的搜索，因为现在侧边栏中支持对已保存的搜索的访问。

2019.2.57829

实时更新

现在，“ 问题”列表显示对摘要，描述和自定义字段的更新，而无需刷新页面。当其他用户添加或删除标签时，您还将看到附件，评论和投票的计数器的增量更新以及更改。

2019.2.56187

侧边栏

最新更新使“ 问题”列表上的侧栏重新显示。此实现提供的链接可显示您喜欢的项目，已保存的搜索和标签的问题。

2019.2.56187

报告菜单

现在，可以从应用程序标题中的“ 报告”菜单中使用生成问题列表中的问题的报告的选项。

2019.2.56187

其他用户的永久令牌生成

现在，拥有足够权限的用户可以代表在YouTrack中具有较低权限级别的用户生成永久令牌。这样，支持第三方集成的用户就可以在需要时对运行自动任务的帐户进行身份验证，而不必设置2FA。

2019.2.56187

批处理更新的改进

我们不仅恢复了在“ 问题”列表的实验版本上选择所有问题的选项，而且还改进了YouTrack处理批量更新的方式。

现在，您可以一次安全地更新多达10,000个问题，并可以选择在操作进行时中断操作。

2019.2.55152

增强的控件，可对搜索结果进行排序

已在“ 问题”列表上对搜索结果进行排序的控件已添加到该页面的实验版本中。您可以单击几下添加和删除排序属性，并更改排序方向。

2019.2.55152

扩展支持自定义字段中的存档值

在将数据存储为字段的字段中归档值的选项version现在扩展为存储枚举类型的其他字段中的值。此增强功能使您可以撤消不再想要使用的值，而无需对大量问题应用不必要的更新。

2019.2.55152

新权限

我们添加了权限，使您可以更好地控制哪些用户可以查看，创建或共享报告以及将工作项添加到问题中。我们还隐含了许多权限分配，这使定义具有适当访问权限的自定义角色变得更加容易。

2019.2.55152

问题列表更新

我们将继续对“问题”列表的实验版本应用更新。最新版本为搜索查询，问题计数和字段可见性设置添加了保存操作。

2019.2

必需的两因素身份验证

组设置中的新选项可让您要求组成员为其中心帐户启用两因素身份验证。这可以帮助您保护对YouTrack安装的访问。

2019.2

时间报告中的增强型数据处理

对时间报告中数据的表示和处理进行的几处更改可帮助您跟踪和量化花费的时间。

  • 显示工作类型的选项在其自己的列中显示每种工作类型的总计。

  • 报告设置中的一个新选项使您可以使用计算的报告中的“ 按问题”选项显示数据时显示自定义字段的值。此设置还确定了可以以CSV和Excel格式下载的数据集中包含哪些字段。

  • 计算报告中的新选项可让您显示剩余（未花费的估算）和总花费时间。

当您以Excel格式导出数据时，分组，总计和视图也将应用于数据。这将公开其他数据点，以供外部分析工具参考。

2019.2

未来日期的相对参数

搜索查询语言现在支持将来发生的日期范围的预定义值。在搜索查询和报告设置中使用这些参数，以查找计划在明天，下周或下个月发生的事件。

您还可以使用这些参数在命令中指定日期。

2019.2

应用密码支持Bitbucket集成

现在，您可以使用用户名和应用密码将YouTrack连接到Bitbucket存储库。这使您可以与使用两步验证的Bitbucket帐户建立集成。

这也意味着您不必在集成设置中存储个人密码。

此版本中的更改使YouTrack与最新版本的Bitbucket API兼容。

2019.2

停产的功能

特征

描述

可用性

Internet Explorer支持

从YouTrack版本2019.2开始，我们不再保证在任何版本的Internet Explorer中都具有稳定的性能，并且不再修复只能在此浏览器中显示的错误。如果您当前正在IE 11中使用YouTrack，建议您切换到另一个Web浏览器。

有关YouTrack支持的浏览器的完整列表，请参见Web浏览器 。

在2019.2中删除

YouTrack工作流程编辑器中的新工作流程

我们不再支持在外部YouTrack工作流程编辑器中编写的新工作流程。我们仅在YouTrack 2019.3。发行之前扩展了对该功能的支持。在此之后，我们将仅支持JavaScript中的工作流程。

如果您仍在使用旧的工作流程，建议您尽快用JavaScript重写它们。有关详细信息，请参阅将旧工作流程转换为JavaScript 。

在2018.3中删除

旧的YouTrack REST API

我们将继续公开其他端点，并为新的YouTrack REST API提供示例请求。一旦发布了先前API支持的端点的较新版本，我们将正式弃用旧的REST API。不再使用旧的REST API后，我们将继续提供固定期限的支持，以留出足够的时间来迁移使用它的应用程序。

如果您有使用旧API的现有应用程序或集成，建议您熟悉更新的版本。如果您的应用程序使用的端点尚未在新API中发布等效项，请加入Slack中的YouTrack社区 。让我们知道您正在使用哪些端点，并帮助我们确定下一步应该发布哪些功能。
