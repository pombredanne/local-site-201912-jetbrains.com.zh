VisibilityActivityItem

Represents the changes of properties responsible for visibility restriction. Can be VisibilityGroupActivityItem or VisibilityUserActivityItem

Extends MultiValueActivityItem

This table describes attributes of the VisibilityActivityItem entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

targetMember

String

Returns the "visibility" property that changed in the target entity. Read-only. Can be null.

targetSubMember

String

The name of property refining the targetMember. Returns actual values depending on the VisibilityGroupActivityItem or VisibilityUserActivityItem Read-only. Can be null.

removed

Type is specified in the descendent Entities

The list of entities that are removed from an attribute in the target entity. Read-only.

added

Type is specified in the descendent Entities

The list of entities that are added to an attribute in the target entity. Read-only.

author

User

The user who performed the action. Read-only.

timestamp

Long

The timestamp of the activity. Read-only.

target

Type is specified in the descendent Entities

The entity that is the target of the performed action. Read-only.

field

FilterField

The filter field. It contains the additional information about the modified property in the target entity. For example, the information about a link type, or the type of the modified custom field or bundle, and so on. Read-only. Can be null.

category

ActivityCategory

The category of the activity. Read-only.
