基于Web的工作流程编辑器

要使用JavaScript编写工作流，您不再需要使用外部应用程序。现在，您可以使用内置在YouTrack中的基于Web的编辑器。

基于Web的工作流编辑器仅支持以JavaScript编写的工作流。要编辑在外部YouTrack工作流编辑器中编写的较旧的工作流，必须首先将其转换为JavaScript。

有关更多信息，请参见将旧工作流程转换为JavaScript 。

如果您希望在集成开发环境中编写代码，则仍然可以使用支持JavaScript的外部编辑器，并将工作流程导入YouTrack。

要访问工作流编辑器，请在“ 工作流管理”页面中单击工作流或模块的名称。用JavaScript编写的所有工作流和模块都设置为链接。

在外部YouTrack工作流编辑器中编写的工作流不会在基于Web的编辑器中打开。相反，所选工作流程或规则显示在管理页面的侧栏中。

  • 单击工作流程的名称时，“ 新模块”对话框将在编辑器面板中打开。

  • 当您单击模块的名称时，该模块将直接在工作流编辑器中打开。

标头控件

在新的“ 工作流程”页面的标题中显示以下控件：

工作流程

将您导航回到“ 工作流”管理页面。使用此链接可以查看YouTrack中的所有工作流程，包括基于MPS的工作流程。

工作流程名称

编辑名称

打开“ 编辑工作流程”对话框。在这里，您可以更改工作流程的名称和标题。

删除工作流程

确认此操作后，删除当前选定的工作流程。有关更多信息，请参见删除工作流和规则 。

@ jetbrains / youtrack-workflow-clone-issue

在问题工具栏中包含支持“ 克隆问题”操作的脚本。

侧栏顶部显示以下控件：

创建工作流程

打开“ 新建工作流”对话框。在这里，您可以定义一个新的工作流程并将其添加到列表中。有关说明，请参阅创建工作流 。

按名称过滤列表

打开输入字段以过滤工作流程列表。在这里，您可以过滤列表以显示与输入字段中的子字符串匹配的工作流程和模块。

隐藏侧边栏

在侧栏中隐藏工作流程和规则的列表。使用此控件可以在页面的整个宽度上查看工作流编辑器。

您具有写访问权的工作流在工作流列表中包括以下控件：

控制

描述

编辑

打开“ 编辑工作流程”对话框。在这里，您可以重命名工作流程并为其赋予易于阅读的标题。在当前实现中，“ 标题”属性仅显示在“ 工作流”管理页面上的列表中。

添加模块

将规则或自定义脚本添加到相应的工作流程。选择模板以打开“ 新建模块”对话框，输入名称，然后保存新模块。

删除

删除相应的工作流程。属于工作流程的所有规则都将从它们附加到的任何项目中删除，并且也从YouTrack中删除。

该控件也可用于列表中的各个模块。有关更多信息，请参见删除工作流和规则 。

控制台工具栏

编辑器的底部是一个控制台。控制台显示与当前脚本相关的日志消息。控制台工具栏中提供以下控件：

控制

描述

显示隐藏

切换控制台窗格的可见性。

显示

过滤日志条目以显示全部或仅消息。使用输入字段来筛选与子字符串匹配的日志条目。

过滤

筛选日志条目列表以显示与子字符串匹配的条目。再次单击此控件以隐藏过滤器。

更新资料

重新加载控制台，显示在页面加载后添加的新日志条目。

下载

明确

从控制台中删除所有日志条目。
