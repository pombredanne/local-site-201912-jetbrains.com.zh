GET系统范围的时间跟踪设置

GET / rest / admin / timetracking

获取当前的全系统时间跟踪设置：一周中的多个工作日，一个工作日中的多个小时。

GET / rest / admin / timetracking

响应参数

参数

类型

描述

小时

整数

一个工作日的小时数

天数

整数

一周中的工作日数。从YouTrack 5.1开始，此设置是根据由workWeek参数。

工作周

清单

逻辑结构指定工作日列表（ workDay参数）。

工作日

整数

一周中某天的整数序号，将其视为一个工作日： 0 -周日1 -星期一2 -星期二3 -星期三4 -星期四5 -星期五6 -星期六

样品

请求：

GET https://example.com/youtrack/rest/admin/timetracking连接：keep-alive Cookie：$ Version = 0; JSESSIONID = axezv08lvrvrzpvvih22ricx; $路径= /
