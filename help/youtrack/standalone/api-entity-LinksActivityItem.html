LinksActivityItem

代表issues IssueLink实体的属性。也就是说，每个问题都有links属性，包含一个IssueLink实体数组。它们每个都代表目标问题和问题集合之间的链接。例如，与目标对象链接为“复制人”的一组问题。这些问题列表中的更改表示为LinksActivityItem实体。

扩展MultiValueActivityItem

下表描述了LinksActivityItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。
