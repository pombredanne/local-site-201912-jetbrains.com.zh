YouTrack Standalone 2019.2帮助

YouTrack作为Zendesk的SAML身份提供商

当您将YouTrack服务器配置为Zendesk实例的身份提供程序时，您的用户可以使用其在YouTrack或任何其他身份验证模块中的凭据登录到Zendesk。

此配置还启用单点登录。用户登录到连接到YouTrack的服务之一时，他们将登录到所有连接的服务。

在开始之前，请验证以下先决条件：

  • 您必须在Zendesk和YouTrack中都具有管理权限。

  • 用于为Zendesk配置SAML 2.0的YouTrack管理员帐户的电子邮件地址必须不同于Zendesk实例所有者的电子邮件地址。否则，您可能会在测试SAML配置时陷入重定向循环。

  • YouTrack管理员帐户必须具有经过验证的电子邮件地址。

要配置您的Zendesk实例：

  1. 在您的Zendesk实例中，打开“ 设置”>“安全性”页面。

  2. 选择最终用户选项卡。

    我们描述了如何设置YouTrack来使用SAML对最终用户进行身份验证。但是，您也可以将YouTrack设置为对Zendesk Agent进行身份验证。为此，选择“ 代理”选项卡，并提供与下一步所述相同的参数。

  3. 启用“ 单点登录”选项，选择“ SAML”，然后配置参数：

    参数

    描述

    SAML SSO URL

    将登录URL字段的内容粘贴到YouTrack服务器的“更多设置”>“ SAML2.0”>“设置”页面上。

    证书指纹

    将证书的SHA-256指纹粘贴到YouTrack中的SAML的SSL密钥存储集中。从“ 更多设置”>“ SAML2.0”>“设置”页面上的“ 指纹”字段进行复制。

    远程注销URL

    将“ 登出URL”字段的内容粘贴到YouTrack服务器的“更多设置”>“ SAML2.0”>“设置”页面上。

    IP范围

    来自这些IP地址的请求将始终通过YouTrack进行路由以进行身份验证。默认情况下，我们建议您使用*.*.*.*模式，以通过YouTrack对所有最终用户进行身份验证。

  4. 在YouTrack中，从“ 管理”菜单的“ 访问管理”部分中选择SAML 2.0 。

  5. 选择“ 注册服务提供商”选项卡。

  6. 单击注册服务提供商按钮。

  7. 在对话框中，输入Zendesk实例的参数：

    参数

    描述

    名称

    输入要在YouTrack中为Zendesk实例显示的名称。

    发行人

    使用格式的Zendesk实例名称.zendesk.com 。

    描述

    （可选）输入Zendesk实例的描述。

    消费者网址

    		粘贴您的Zendesk实例的访问消费者服务（ACS）URL。Zendesk中ACS URL的一般格式为https://.zendesk.com/access/saml （区分大小写）。

    在Zendesk中启用SAML时，可以在SAML SSO URL字段的描述中找到特定实例的实际ACS URL。

    YouTrack应该发送LogoutResponse

    确保禁用该选项。
