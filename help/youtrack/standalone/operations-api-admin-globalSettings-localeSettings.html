具有特定区域设置的操作

此资源使您可以使用YouTrack服务的系统语言设置。

资源资源
/ api / admin / globalSettings / localeSettings

退货实体

LocaleSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的LocaleSettings 。

  • POST ： 更新特定的LocaleSettings 。

LocaleSettings属性

表示系统语言设置。

下表描述了LocaleSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

地区

 LocaleDescriptor

YouTrack的当前语言环境。

isRTL

布尔型

指示当前系统语言是否是从右到左。

阅读特定的语言环境设置

阅读您的YouTrack服务的当前系统语言。

请求语法

GET / api / admin / globalSettings / localeSettings？{字段}

请求参数

参数

类型

描述

领域

串

响应中应返回的LocaleSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/localeSettings?fields=locale(id,name,community,locale,language),isRTL

样品响应体

{“ locale”：{“ name”：“ English”，“ locale”：“ en_US”，“ community”：false，“ id”：“ en_US”，“ language”：“ en”，“ $ type”：“ LocaleDescriptor“}，” isRTL“：false，” $ type“：” LocaleSettings“}

更新特定的语言环境设置

更改您的YouTrack服务的系统语言。

必选参数：数据库id目标语言环境。它通常具有与locale属性。

所需权限

需要低级管理员写权限。

请求语法

POST / api / admin / globalSettings / localeSettings？{字段}

请求参数

参数

类型

描述

领域

串

响应中应返回的LocaleSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/localeSettings?fields=locale(id,name,community,locale,language),isRTL

样品申请机构

样品响应体

{“ locale”：{“ name”：“ Deutsch”，“ locale”：“ de_DE”，“ community”：false，“ id”：“ de_DE”，“ language”：“ de”，“ $ type”：“ LocaleDescriptor“}，” isRTL“：false，” $ type“：” LocaleSettings“}
