BundleCustomFieldDefaults

表示捆绑软件字段的字段默认值。

扩展CustomFieldDefaults

下表描述了BundleCustomFieldDefaults实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

canBeEmpty

布尔型

指示此字段在问题中是否可以为空值。

emptyFieldText

串

字段为空时显示为占位符的文本。 Can be null 。

isPublic

布尔型

如果为true，则用户需要具有“ 读取问题”和“ 更新问题”权限才能访问此字段。否则，该字段将被视为私有字段，并且需要“ 读取问题私有字段”和“ 更新问题私有字段”权限。

父母

 CustomField

引用自定义字段。 Read-only 。
