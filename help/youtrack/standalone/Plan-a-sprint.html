计划冲刺

在开始冲刺之前，您的团队通常会开会商定一系列任务，这些任务将是您在这段时间内所做工作的重点。在计划阶段期间或之后，请按照本页上的说明填充新的sprint泳道和卡。

先决条件

请注意，在这一点上，我们假设您已经：

  • 配置主板设置
  • 优先考虑董事会的积压工作
  • 创建了您要计划的冲刺

从待办事项列表中拉出泳道和卡牌

向泳道添加泳道和泳卡的最实用方法是直接从待办事项中解决问题。

要从积压中添加问题，请执行以下操作：

  1. 在sprint下拉列表中选择要计划的sprint。

  2. 点击按钮以查看积压。

  3. 将卡片从待办事项列表拖到董事会，或者在待办事项列表中选择一个或多个问题并将其添加到当前的冲刺中：

    • 点击按钮将相应的问题添加到冲刺。

    • 单击底部工具栏中的“ 添加到冲刺”按钮以打开命令窗口，其中包含预填充的命令以将选定的卡片添加到冲刺中：

  • 用于识别泳道的问题将作为泳道添加到板上。

  • 如果泳道问题具有子任务，则可以选择将其子任务作为卡片添加到泳道。

添加命令问题

有时，您可能想将未积压的问题添加到sprint。在这种情况下，您可以使用命令添加问题。

要使用命令向sprint添加问题：

  1. 打开问题列表。

  2. 使用搜索查询来查找未分配给冲刺的问题。在这种情况下，关键搜索属性为：

    具有：-{Board }
    例如，您想在董事会管理的项目中找到所有未解决的关键问题。以下是用于在CSS和CMS项目中查找未分配给CMS Scrum板的问题的搜索条件：
    #Unresolved＃关键项目：CSS，CMS-{Board CMS Scrum}

  3. 选择要分配给Sprint的所有问题。

  4. 打开“ 应用命令”对话框（只需键入），然后输入命令。使用格式：

    板

    这是您用来将所选问题分配到上一个示例中的Sprint-2 Leads Management sprint板上的命令的命令：

    CMS Scrum Sprint-2潜在客户管理委员会

    • 用于识别泳道的问题将作为泳道添加到板上。

    • 如果将问题链接为泳道问题的子任务，则会将其作为卡片添加到父泳道。否则，问题会出现在泳道中的未分类卡片上。

将新问题添加为泳道

如果需要，您不仅可以将泳道从待办事项移至董事会，还可以在冲刺中直接在董事会上创建泳道问题。

在板上创建新的泳道：

  1. 将板子打开到所需的冲刺。

  2. 请执行以下任一操作：

    • 单击标题中的“ 创建”按钮，然后从列表中选择Swimlane 。

    • 单击页脚中的“ 新建泳道”按钮。

    • 按下键盘快捷键以创建新的泳道。

      视窗：

      Ctrl + Alt + Shift +Insert

      苹果系统：

      Control + Option + Shift +N

    • 将打开“新问题”对话框。

    • 标识泳道的字段的值会自动设置。如果您使用多个值来标识泳道，则列表中的第一个值将分配给该问题。

    • 项目字段也会自动设置。如果您在电路板上管理多个项目，则列表中的第一个项目将分配给该问题。

  3. 在新问题对话框中，输入问题的摘要和描述。您还可以设置自定义字段的值。问题摘要显示为泳道的名称。

  4. 单击添加泳道按钮将问题作为泳道添加到当前的Sprint。如果要在问题创建后向其添加更多信息，请单击“ 添加并打开”按钮。在这种情况下，问题将作为泳道添加到当前的sprint中，并在查看模式下打开。

将新问题添加为卡片

您可以直接在板上创建问题。许多团队使用此选项来创建完成用户故事所需的任务。您可以将卡片添加到任何泳道。如果将泳道用于未分类的卡片，则可以将问题作为孤儿添加到冲刺中。

要将卡添加到板上：

  1. 在您要向其添加任务的泳道中，单击任何列中的图标。

    • 将打开“新问题”对话框。

  2. 在新问题对话框中，输入问题的摘要和描述。您还可以设置自定义字段的值。问题摘要显示为泳道的名称。

  3. 输入新卡的摘要和说明 。如果需要，请自定义其他字段。

  4. 单击添加卡按钮，将新卡添加到所选列的泳道中。如果要在问题创建后向其添加更多信息，请单击“ 添加并打开”按钮。在这种情况下，问题将作为卡片添加到所选泳道中，并在查看模式下打开。

    • 用于定义所选列的值将应用于新发行版中的相应字段。

    • 如果将卡添加到问题标识的泳道，则新问题将链接为泳道问题的子任务。

估算卡

在YouTrack中，我们仅提供任务估算，而不提供泳道估算。泳道（用户故事）的估计是其任务的所有估计的总和。不显示泳道的摘要估算值，并且不会在任何报告中进行计算。

在评估评估板上的任务之前，您应该在评估板设置中指定一个要用作评估的问题字段。在板设置中配置“ 估计字段”时 ，相应的值显示在每张卡的右上角。

默认情况下，所选字段的“空”值显示在任务卡上（例如，“无估计”或为用于估计的字段配置的其他空值）。

要将估算值添加到任务中，请单击“ 估算值字段”并输入一个值。

优先冲刺

设置冲刺的优先级很容易。只需拖动泳道和卡片，将其以所需的顺序放置即可。您还可以使用包含排序属性的搜索查询来过滤板上的卡片。例如，搜索查询sort by: Priority根据“ 优先级”字段中存储的值，自动对每个泳道中的卡片进行排序。

板上泳道和卡片的顺序不会影响“ 问题”列表中显示的自定义顺序，也不会更改“ 优先级”字段的值。优先考虑板上的问题只是视觉上的指示，泳道和板上较高的问题应首先完成。
