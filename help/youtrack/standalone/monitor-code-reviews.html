监控代码评论

当您的YouTrack安装与Upsource集成时，活动流中将提供用于代码审查的其他供稿。这使您可以跟踪与特定问题相关的代码审查中的活动。

代码评论会自动添加到活动流中，以在标题中包含YouTrack问题ID的评论。

将为活动流中的每个项目显示以下信息：

  • 创建代码审阅的用户名。

  • 创建代码审阅的日期和时间。

  • Upsource中代码审查的ID和标题。

  • 已添加为审阅者的用户的化身。当审阅者接受或关注代码审阅中的更改时，其化身将装饰有相应的表情符号。

可以执行以下操作：

  • 单击ID在Upsource中打开代码审查。

  • 将指针移到用户头像上以显示其他信息。
