YouTrack Standalone 2019.2帮助

系统设置

表示影响YouTrack核心功能的系统设置。

下表描述了SystemSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

baseUrl

串

YouTrack安装的基本URL。 Can be null 。

maxUploadFileSize

长

可以上传的文件的最大大小。

maxExportItems

整数

可以导出的最大问题数。

管理员电子邮件

串

服务器管理员的电子邮件。 Can be null 。

allowStatisticsCollection

布尔型

指示是否允许收集使用情况统计信息。

isApplicationReadOnly

布尔型

true ，如果应用程序当前处于只读模式。除此以外， false 。
