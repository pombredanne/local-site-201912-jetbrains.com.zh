命令语法

本页提供有关YouTrack命令语法的BNF描述。

语法

:: = （ ）* :: = | | | | | :: ='评论' :: = :: ='标签'（ | ）| 'untag' :: =（'unstar'|'star'）[ ] :: = :: =（'add'|'remove'）（ | | ） :: =

语法不区分大小写。

有关命令属性的完整列表，请参见《 命令参考》 。

例子

命令

描述

Fixed

将状态字段的值设置为固定 。

relates to IS-007

在选定的问题中添加一个与该问题相关的ID为IS-007的问题的链接。
与ID 相关的链接也已添加到ID为IS-007的问题中。

star me

给当前用户加注星号。

add affects 1.0

添加到枚举“受影响的版本”版本1.0。

tag Todo

将“ Todo”标签添加到所选问题。

remove tag Todo

从所选问题中删除“ Todo”标签。

state To be discussed

将“ 状态”字段的值设置为“待讨论” 。
