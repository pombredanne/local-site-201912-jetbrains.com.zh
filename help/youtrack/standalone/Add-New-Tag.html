新增标签

PUT /用户/标签/ {tagName}

添加新用户的标签

放置/ rest / user / tag / {tagName}？{visibleForGroup}＆{updatableByGroup}＆{untagOnResolve}

参数：

名称

类型

描述

标签名

串

要添加的新标签的名称。

visibleForGroup

串

新标签应在其上可见的组的名称。

可更新组

串

群组的名称，该群组的成员应能够编辑新标签。

untagOnResolve

布尔值

为新标签启用或禁用“解决时取消标签”选项。如果启用，则当问题的状态更改为被视为“已解决”的任何状态（例如，已修复，重复，过时等）时，标签将自动从问题中删除。
