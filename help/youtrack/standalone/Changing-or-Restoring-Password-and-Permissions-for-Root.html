YouTrack Standalone 2019.2帮助

更改或还原root用户的密码和权限

此页面提供有关管理默认“ root”用户帐户的说明。

限制根用户权限

您无法在YouTrack中删除根帐户。如果您希望使用另一个帐户来管理您的YouTrack实例，请正确配置新的管理员帐户并执行以下任何或所有操作：

  • 从根帐户撤消访问权限。

  • 为根帐户设置自定义密码。

  • 禁止root帐户。

采取这些步骤有助于防止root帐户进行未经授权的访问。

重置默认管理员帐户

首次安装YouTrack时，请设置默认管理员帐户的登录名和密码。此帐户在全局项目中被分配了系统管理员角色。

如果您忘记了默认管理员帐户的凭据，或者继承了另一位管理员配置的安装，则可以将该帐户的登录名和密码重置为其默认值。您还可以在还原帐户时为默认管理员设置系统属性。

当启用了双重身份验证并且默认管理员不再有权访问与该帐户配对的身份验证应用程序时，您也可以使用此过程。将默认管理员帐户重置为默认状态也会禁用两因素身份验证。

恢复JAR安装的根密码

如果您忘记了root密码，则可以使用-Djetbrains.charisma.restoreRootPassword=true重新启动YouTrack服务器时的JVM选项。有关更多信息，请参见配置参数 。

如果您从命令行界面运行YouTrack，该命令可能如下所示：

java -Djetbrains.charisma.restoreRootPassword = true -jar youtrack.jar 8080

当YouTrack启动时，root用户密码和权限将重置为其默认值。使用默认凭据登录（ root/root ）。更改系统管理员密码。

恢复MSI安装的根密码

如果您将YouTrack作为Windows服务运行，请使用以下过程。

要恢复您的根密码：

  1. 停止服务。

  2. 导航%programdata%\JetBrains\YouTrack\conf\youtrack.jvmoptions 。如果youtrack.jvmoptions文件丢失，只需通过重命名示例配置文件（ youtrack.jvmoptions.dist ）。

  3. 添加-Djetbrains.charisma.restoreRootPassword=true配置文件的选项。配置文件格式在文件本身中描述。

  4. 照常运行服务。默认的系统管理员凭据被重置为具有密码“ root”的“ root”用户。

  5. 以“ root”用户身份登录YouTrack。更改系统管理员密码。

  6. 停止YouTrack服务。

  7. 删除-Djetbrains.charisma.restoreRootPassword=true配置文件中的选项。否则，每次重新启动YouTrack时都会重置系统管理员密码。

  8. 启动服务。

恢复ZIP安装的根密码

如果您将YouTrack与ZIP安装一起安装，请使用以下过程。

此过程中使用的命令适用于Linux和macOS。对于Windows安装，请更换.sh与.bat并使用反斜杠。

要恢复您的根密码：

  1. 使用以下命令停止YouTrack：

    /bin/youtrack.sh停止

  2. 使用以下命令启动YouTrack：

    /bin/youtrack.sh开始--J-Djetbrains.charisma.restoreRootPassword = true

    请确保在J选项可将JVM属性仅应用于当前运行，而不会影响随后重新启动YouTrack服务。

    默认的系统管理员凭据被重置为具有密码“ root”的“ root”用户。

  3. 以“ root”用户身份登录到YouTrack。更改系统管理员密码。
