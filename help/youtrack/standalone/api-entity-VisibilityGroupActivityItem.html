VisibilityGroupActivityItem

Represents the event when a user adds or removes a group to/from the Visibility settings of the target entity.

Extends VisibilityActivityItem

This table describes attributes of the VisibilityGroupActivityItem entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

targetMember

String

The name of property changed in the target entity. Read-only. Can be null.

targetSubMember

String

The name of property refining targetMember. Read-only. Can be null.

removed

Array of UserGroups

User groups removed from the visibility. Read-only.

added

Array of UserGroups

User groups added to the visibility. Read-only.

author

User

The user who performed the action. Read-only.

timestamp

Long

The timestamp of the activity. Read-only.

target

Type is specified in the descendent Entities

The entity that is the target of the performed action. Read-only.

field

FilterField

The filter field. It contains the additional information about the modified property in the target entity. For example, the information about a link type, or the type of the modified custom field or bundle, and so on. Read-only. Can be null.

category

ActivityCategory

The category of the activity. Read-only.
