Get a State Bundle

GET /admin/customfield/stateBundle/{bundleName}

Get a bundle of states by its name.

GET /rest/admin/customfield/stateBundle/{bundleName}

Parameters:

Name

Type

Description

bundleName

string

Name of a state bundle to get.
