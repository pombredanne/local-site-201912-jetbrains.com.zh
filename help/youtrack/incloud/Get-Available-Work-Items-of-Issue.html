获取可用的发货项目

GET / rest / issue / {issue} / timetracking / workitem /

获取特定问题的可用工作项列表。

GET / rest / issue / {issue} / timetracking / workitem /

参数：

名称

类型

描述

问题

问题编号

您想要获取所有可用工作项列表的问题的问题ID。

响应参数：

对于响应代码中的每个工作项，您将获得以下参数：

参数

类型

描述

串

工作项的唯一ID。此ID应在请求方法中用于编辑或删除工作项。

女士

工作项的日期（自1970年1月1日，格林尼治标准时间00:00:00以来的毫秒数）。创建工作项时，工作项日期的时间部分将替换为当前用户所在时区中午夜的时间戳。

分钟

工作项目的持续时间，以分钟为单位。

串

工作项的描述。

串

工作项作者的用户登录名。另外，我们提供用户个人资料的URL 。

此外，对于每个工作项目，我们都提供唯一的URL来访问该工作项目（例如'URL' 参数的属性）。有关示例，请参见以下部分。
