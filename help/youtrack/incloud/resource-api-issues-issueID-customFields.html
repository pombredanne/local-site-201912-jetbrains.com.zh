退货实体

IssueCustomField 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读IssueCustomFields列表 。

IssueCustomField属性

表示特定问题中自定义字段的值。

的IssueCustomField是问题中所有类型的自定义字段的抽象祖先。自定义字段的实际类型取决于特定项目的设置。

每IssueCustomField实体，无论其类型如何，均具有以下属性：

  • projectCustomField ：对项目中自定义字段设置的引用。

  • value ：分配给问题中自定义字段的实际值。根据字段的类型，此属性可以存储单个值或值的数组。

  • name ：自定义字段的字符串名称。

  • id ：问题自定义字段的实体ID。

  • $type ：问题自定义字段的唯一类型。更新问题中的自定义字段时，必须提供此属性。

由于实现的特定性，为IssueCustomField属性生成的表仅包含name属性。有关支持的属性及其类型，请参见后代实体的描述。
