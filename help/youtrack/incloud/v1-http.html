http

包含通过HTTP交换和传输数据的对象和类的定义。主要类是Connection 。

种类

该模块支持以下对象类型：

类型

描述

连接

用于建立连接并将请求发送到目标站点的主类。

REQUEST_TYPES

支持的HTTP请求类型的集合。请注意，没有将正确的请求类型传递给Connection .doSync方法，而是有一些专用方法与您可以直接调用的每种请求类型相对应。例如，getSync或postSync。

响应

一个为HTTP响应创建定义的类。如果在处理期间发生异常，则响应对象中的大多数属性为空。
