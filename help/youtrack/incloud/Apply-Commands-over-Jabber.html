在Jabber上应用命令

使用YouTrack，您不仅可以从Jabber客户端中的YouTrack接收通知，还可以通过Jabber消息与YouTrack进行交互。在Jabber客户端中输入命令时，该命令将应用于YouTrack中的问题。

避免您以这种方式与YouTrack进行交互的功能称为Jabber Bot 。

先决条件

在使用此功能之前，管理员必须为您的YouTrack实例启用Jabber通知并启用Jabber机器人。有关设置说明，请参阅通过Jabber启用通知 。

将您的Jabber帐户添加到YouTrack

要在Jabber客户端中启用个人通知并使用Jabber Bot与YouTrack进行交互，您需要将Jabber帐户添加到YouTrack。

要添加您的Jabber帐户：

  1. 在YouTrack中访问“个人资料”页面（点击您的头像并选择个人资料 ）。

  2. 单击链接访问您的中心帐户 。

  3. 在Jabber的输入字段中，输入您的Jabber帐户的用户名。

  4. 要确认设置，请单击“ 发送测试消息”消息链接。

    • 测试消息将从Jabber Bot帐户发送到Jabber客户端应用程序中的用户帐户。

  5. 单击保存按钮。

    • Jabber帐户将保存在您的用户个人资料中。

    • 您现在可以在Jabber客户端中接收有关要在YouTrack中发布的更新的通知。

    • 您可以通过在Jabber Bot的直接消息中输入命令来将命令应用于问题。

将命令应用于问题

与Jabber Bot交互时，您可以在问题更新后立即做出反应，并应用命令作为对通知消息的答复。

例如，您收到有关问题A-118已更新的通知。在您的Jabber客户端中，您可以使用以下命令来回复此通知：

assignee me state In testing

该命令适用于在YouTrack中发布A-118。

输入以下格式的命令：

[issueID, issueID, ...] command [comment]

  • 如果您答复特定问题的通知，则可以省略问题ID。

  • 您可以同时将命令应用于一个或多个问题。输入多个问题ID，以逗号或空格分隔。例如， A-1, A-2, A-5要么A-1 A-2 A-5 。

  • 要对问题添加评论，请使用comment关键词。在命令后输入的所有内容都会添加为问题的注释。
