活动光标页面

此资源提供对将其包装到页面对象的问题活动的访问。与活动列表相比，页面的主要优点是游标。该页面提供了边界标记，该边界标记允许从页面结束处继续迭代活动。

退货实体

ActivityCursorPage 。有关实体属性的描述，请参见“ 支持的字段”部分。

ActivityCursorPage属性

表示一个页面对象，其中包装了发行活动的列表。与活动列表相比，页面的主要优点是游标。该页面提供了边界标记，这些标记允许从页面结束位置开始对活动进行连续迭代。
