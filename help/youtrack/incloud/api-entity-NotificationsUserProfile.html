NotificationsUserProfile

代表特定用户的个人资料中的通知设置。

下表描述了NotificationsUserProfile实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

notifyOnOwnChanges

布尔型

指示用户是否收到有关其自身更改的通知。

jabberNotificationsEnabled

布尔型

指示用户是否通过jabber启用了通知。

emailNotificationsEnabled

布尔型

指示用户是否通过电子邮件启用了通知。

提到NotificationsEnabled

布尔型

指示当问题文本中提到通知时，用户是否收到通知。

plicateClusterNotificationsEnabled

布尔型

指示用户是否收到有关问题重复项的通知。

MailboxIntegrationNotificationsEnabled

布尔型

指示用户是否收到邮箱集成产生的通知。

usePlainTextEmails

布尔型

指示用户是否接收纯文本电子邮件。

autoWatchOnComment

布尔型

指示用户发表评论时是否自动订阅问题更新。

autoWatchOnVote

布尔型

指示用户对问题进行投票时是否自动订阅问题更新。

autoWatchOnUpdate

布尔型

指示用户在更新问题时是否自动订阅问题更新。
