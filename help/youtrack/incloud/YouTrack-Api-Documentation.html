YouTrack工作流程API

的@jetbrains/youtrack-scripting-api软件包包括模块的集合。这些模块提供了自定义函数，方法和构造函数，可让您引用和更新YouTrack中的实体。它们还定义了每个实体可用的属性。

提供以下模块：

模组

描述

约会时间

该模块包含使您可以处理日期和时间的功能。

实体

包含工作流程可访问的YouTrack中实体的定义。

http

包含通过HTTP交换和传输数据的对象和类的定义。主要类是Connection 。

通知

此模块包含用于发送电子邮件的功能。这些消息独立于用于订阅以在YouTrack中发布更新的通知方案。这些电子邮件消息的所有组件均在此模块支持的功能的参数中定义。但是，邮件仍然由连接到YouTrack安装的SMTP服务器发送。

搜索

该模块包含的功能可让您搜索YouTrack中的问题。

弦

该模块为标准Java库中的特定字符串实用程序提供支持。可用于工作流中的字符串的所有其他方法均来自标准JavaScript库，并且其行为与MDN文档中所述相同 。

工作流程

该模块包含实用程序功能，可在用户界面中显示与工作流程相关的消息和警告。
