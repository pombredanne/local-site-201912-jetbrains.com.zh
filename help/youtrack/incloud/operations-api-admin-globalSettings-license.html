具有特定许可的操作

此资源使您可以使用YouTrack服务的许可证。

资源资源
/ api / admin / globalSettings / license

退货实体

执照 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的许可证 。

  • POST ： 更新特定许可证 。

许可证属性

代表YouTrack服务的当前许可证。

下表描述了License实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

用户名

串

用于许可证的用户名。 Can be null 。

执照

串

注册码。 Can be null 。

错误

串

许可错误。如果没有错误，则此属性为空。 Read-only 。 Can be null 。

阅读特定许可证

阅读您的YouTrack服务的许可证。
