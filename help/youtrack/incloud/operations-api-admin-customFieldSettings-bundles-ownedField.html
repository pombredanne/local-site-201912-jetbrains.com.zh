使用特定OwnedBundle的操作

该资源为拥有的字段捆绑包提供操作。

资源资源
/ api / admin / customFieldSettings / bundles / ownedField / {bundleID}

退货实体

OwnedBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的OwnedBundle 。

  • POST ： 更新特定的OwnedBundle 。

  • DELETE ： 删除特定的OwnedBundle 。

OwnedBundle属性

代表YouTrack中的一组拥有的值。

扩展BaseBundle

下表描述了OwnedBundle实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

价值观

 OwnedBundleElement数组

拥有的价值观。

isUpdateable

布尔型

如果true ，则当前登录的用户可以更新此捆绑包。除此以外， false 。 Read-only 。

读取特定的OwnedBundle

获取特定的拥有值捆绑。

所需权限

需要对使用捆绑软件的字段之一进行读取或更新访问。如果未在任何字段中使用捆绑软件，则至少在一个项目中需要“更新项目”权限。
