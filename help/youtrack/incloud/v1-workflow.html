该模块包含实用程序功能，可在用户界面中显示与工作流程相关的消息和警告。

功能


校验

静态检查（条件，消息）

检查以确定指定条件是否为真。如果条件不成立，则系统将引发错误。所有更改都回滚到初始状态。错误包含指定的消息，该消息显示在用户界面中。

参量

名称

类型

描述

条件

 boolean

要检查的条件。

信息

 string

在条件为假的情况下向用户显示的错误消息。

例

工作流程检查（issue.fields。受让人&& ctx.currentUser.login === issue.fields。Assignee.login，“只有受让人可以将问题标记为已修复。'）;

 

i18n

静态i18n（messageId）

返回消息的本地化版本。警告：仅供内部使用！仅在将本地化消息的ID和文本字符串存储在应用程序中的默认工作流程中才支持此方法。不支持在自定义工作流中引用此方法。

参量

名称

类型

描述

messageId

 string

消息ID。

返回值

类型

描述

Object

包含与指定消息关联的占位符和本地化字符串的对象。当将此对象传递给需要字符串值的函数时，将在内部调用toString方法，将参数替换为本地化的字符串。

 

信息

静态消息（messageText）

在YouTrack用户界面中显示指定的消息。

参量

名称

类型

描述

messageText

 string

消息文本。
