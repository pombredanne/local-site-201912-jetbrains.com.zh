版本捆绑

该资源提供了带有版本捆绑包的操作。

资源资源
/ api / admin / customFieldSettings / bundles / version

退货实体

VersionBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读VersionBundles列表 。

  • POST ： 添加一个新的VersionBundle 。

VersionBundle属性

代表YouTrack中的一组版本。

扩展BaseBundle

下表描述了VersionBundle实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

价值观

 VersionBundleElements的数组

捆绑软件中包含的版本的集合。

isUpdateable

布尔型

如果true ，则当前登录的用户可以更新此捆绑包。除此以外， false 。 Read-only 。

阅读VersionBundles列表

获取系统中所有可用版本包的列表。

请求语法

GET / api / admin / customFieldSettings / bundles / version？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

响应中应返回的VersionBundle属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。
