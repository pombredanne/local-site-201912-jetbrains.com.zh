设立个人委员会

有时您想跟踪自己的任务，或者在板上查看一组特定的任务。您可以创建一个仅包含您感兴趣的任务的自定义板。

使用“ 个人董事会”模板时，将为您参与的所有项目创建一个个人董事会，并仅显示分配给您的问题。有关详细信息，请参阅个人板 。

无论使用哪种模板，使董事会“个性化”的主要功能是其可见性。您可以限制所创建的任何木板的可见性，因此只有您可以查看它。当您是唯一可以访问该板的人时，如果您在板上添加问题或在冲刺之间移动卡，其他用户将不会收到通知。但是，如果使您的个人面板对其他用户可见，则该组中具有“ 低级管理”权限的用户也可以访问和编辑该面板的设置。

要创建个人板：

  1. 选择敏捷板>创建>板 。

  2. 选择与所需配置匹配的电路板模板 。

  3. 输入电路板的基本设置，包括其名称和要在电路板上管理的项目列表。

  4. 确保可以查看和使用面板以及可以编辑面板设置选项均设置为所有者 。

  5. 为您的板设置默认的积压 。

  6. 完成后，单击创建板按钮。

    • 将创建一个新板，并在板设置中打开“ 常规”选项卡。

  7. 配置板设置以匹配您的个人喜好。要配置不干扰项目团队其他成员的个人董事会，请遵循以下准则：

    • 避免将个人冲刺链接到自定义字段。当您在冲刺之间移动问题时，将通知其他用户，并且当其他用户设置此字段的值时，可以向董事会添加问题。

    • 如果您拥有在个人板上管理的项目的“ 更新项目”权限，请避免添加和编辑用于列和泳道的值。这些更改将应用于这些项目中使用的自定义字段，这可能不是您想要的。

基本上就是这样。现在，您可以根据需要使用个人董事会来计划和监视您的工作。
