标签和保存的搜索

标签和保存的搜索选项卡使您可以查看和管理在YouTrack中创建的标签和保存的搜索。您还可以查看已与您共享的标签和保存的搜索。要访问此页面，请单击您的头像，选择配置文件链接，然后选择标签和保存的搜索选项卡。

标签显示在列表的顶部，然后显示已保存搜索的列表。从列表中选择一个项目时，可以在侧栏中显示所选项目的当前设置。如果标签或保存的搜索与您共享，则所有者也会在边栏中显示。

给问题加注星标时，会生成加注星标。如果您在列表中没有看到“加星标”标签，只需为问题加注星标，它就会自动出现。

标头中提供以下控件：

控制

描述

新标签或保存的搜索

在边栏中打开“ 新建标签 / 新保存的搜索”对话框。在这里，您可以创建新标签或保存的搜索并配置您的个人订阅选项。

删除

删除所选标签或保存的搜索。仅当您选择创建的标签或保存的搜索时，才启用此控件。

搜索标签和搜索

过滤标签和保存的搜索列表。键入时将过滤列表，以显示名称中包含指定文本的项目。

所有

显示所有可用的标签和保存的搜索。这包括您创建的标签和保存的搜索，以及与您共享的标签和保存的搜索。

由我创建

仅显示您自己创建的标签和保存的搜索。

显示/隐藏详细信息

切换详细信息侧栏的显示。

以下选项直接在标签和保存的搜索列表中可用：

选项

描述

固定到侧边栏

启用后，将所选项目添加到“ 问题”列表的侧栏中。从“ 问题”列表中，可以选择标签或保存的搜索以显示分配了相应标签或与保存的搜索匹配的问题的列表。此选项基于每个用户启用。

在解决时取消标记

启用后，在为问题分配了被认为已解决的状态时，会将其从问题中删除。此选项仅适用于标签。

可以查看和使用

设置标签或保存的搜索的可见性。如果您有权编辑该项目，则可以通过从下拉列表中选择一个组来更改可见性。否则，此设置以只读模式显示。

可以编辑

设置有权更新标签或保存的搜索的组。如果您有权编辑该项目，则可以通过从下拉列表中选择一个组来更改此设置。否则，此设置以只读模式显示。

新标签和保存的搜索

此页面提供了用于创建“标签”和“已保存”搜索的控件的替代方法，这些控件可在“ 问题”列表中找到。在此页面上创建标签或保存搜索的主要优点是，您可以配置订阅选项并同时创建项目。

要创建新标签：

需要权限：创建标签或保存的搜索

  1. 单击创建标签或保存的搜索按钮。

    • 侧边栏中将打开“ 新建标签”对话框。

  2. 为新标签配置以下设置：

    设置

    描述

    名称

    输入标签名称。

    标签颜色

    从颜色选择器中选择一种配色方案。

    可以查看和使用

    选择一个组，以授予其成员查看标签的权限并将其应用于问题。

    可以编辑

    选择一个组，以授予其成员更新此标签设置的权限。

    固定到侧边栏

    选择此标签以将其添加到“ 问题”列表中始终可见的标签列表中。此设置基于每个用户应用。

    解决问题后删除标签

    选择此选项可在状态被设置为已解决的状态时自动将其从问题中删除。

  3. 如果要接收有关分配了此标签的问题的通知，请单击“ 订阅”下拉列表以显示订阅选项

  4. 启用要触发通知的事件。

  5. 单击创建按钮。

    • 标记已添加到YouTrack。

    • 如果启用了通知事件，则订阅将添加到您的通知方案中。

要创建新的保存的搜索，请执行以下操作：

需要权限：创建标签或保存的搜索

  1. 单击创建标签或保存的搜索按钮。

    • 侧边栏中将打开“ 新建标签”对话框。

  2. 点击保存的搜索按钮。

    • 侧边栏显示“ 新建保存的搜索”对话框。

  3. 为新保存的搜索配置以下设置：

    设置

    描述

    名称

    输入已保存搜索的名称。

    询问

    输入搜索查询。输入字段支持查询完成。

    可以查看和使用

    选择一个组以授予其成员查看和执行保存的搜索的权限。

    可以编辑

    选择一个组以授予其成员更新此保存搜索的设置的权限。

    固定到侧边栏

    选择以将此保存的搜索添加到“ 问题”列表中始终可见的保存的搜索列表。此设置基于每个用户应用。

  4. 如果您想收到有关与此保存的搜索匹配的问题的通知，请单击“ 订阅”下拉列表以显示订阅选项

  5. 启用要触发通知的事件。

  6. 单击创建按钮。

    • 保存的搜索将添加到YouTrack。

    • 如果启用了通知事件，则订阅将添加到您的通知方案中。

管理标签和保存的搜索

标签和保存的搜索选项卡使您能够在一处管理所有标签和保存的搜索。您还可以查看和更新您有权编辑的标签和保存的搜索的设置。

需要权限：编辑标签或保存的搜索，共享标签或保存的搜索，删除标签或保存的搜索

以下操作可用于您创建或有权编辑的标签：

  • 从列表或边栏中编辑设置。重命名标签，更改其颜色，并更新可以查看和使用或编辑标签的组。

  • 解决问题后，启用或禁用删除标签的选项。

  • 在“ 问题”列表的侧栏中显示的标签列表中，固定或取消固定该标签。

  • 在边栏中的“ 订阅”面板中针对此标签的问题更新您的通知方案。

  • 删除不再使用的标签。

对于您有权查看和使用但不能编辑的标签，可以在边栏中为此标签更新通知方案。

以下操作可用于您创建或有权编辑的已保存搜索：

  • 从列表或边栏中编辑设置。重命名保存的搜索，并更新可以查看和使用或编辑保存的搜索的组。

  • 在“ 问题”列表的侧栏中显示的标签列表中固定或取消固定已保存的搜索。

  • 在边栏中的“ 订阅”面板中，针对与此保存的搜索相匹配的问题更新您的通知方案。

  • 删除不再使用的已保存搜索。

对于您有权查看和使用但没有编辑权限的已保存搜索，可以在边栏中为该已保存搜索更新通知方案。
