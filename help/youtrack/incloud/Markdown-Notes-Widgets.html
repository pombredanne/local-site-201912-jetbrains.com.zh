Markdown笔记小部件

便笺小部件会显示您想要在仪表板上显示的任何消息。此小部件可让您使用Markdown语法设置文本格式。

如果您与其他用户共享仪表板，则此功能特别有用。您可以将此小部件用作公告板来发布对您的团队重要的更新。

注意小部件支持CommonMark规范中描述的标准Markdown实现。问题字段中支持的许多扩展名在这里不可用。有关详细信息，请参见Markdown语法 。

要将Markdown Notes小部件添加到仪表板：

  1. 单击仪表板上的添加小部件按钮。

  2. 从下拉列表中，选择Markdown Notes 。

    • 标题为Markdown Notes的卡将添加到仪表板。

  3. 输入可选的小部件标题。

  4. 在输入字段中输入注释。使用Markdown语法设置文本格式。有关受支持的格式选项的列表，请参见Markdown语法 。

  5. 要预览小部件中的文本，请单击显示预览链接。

  6. 单击创建窗口小部件按钮。

    • Markdown Notes小部件已添加到仪表板。

    • 如果输入标题，则文本将显示在小部件标题中。
