获取具有所有值的问题列表

用例：

使用REST API获取所有自定义字段的问题列表。

请注意，此示例对YouTrack 2019.1和更高版本有效。在以前的版本中，尽管原理相同，但是查询参数和实体属性可能不同。此外，2019.1中引入了许多功能，例如按名称命名实体，但以前的版本中不支持。

摘要

要获取所有自定义字段的问题列表：

  1. 使用“获取问题列表”请求：

    GET / api / issues？[字段]

  2. 在里面fields request参数指定您要获得响应的问题属性，包括其必要的属性customFields 。

一步步

我们将从对请求的逐项描述开始，以获取有关其自定义字段的特定问题。然后，我们将原理外推到问题列表。

解决其自定义字段问题

  1. 要解决所有自定义字段的值问题，请发送GET请求目标问题。您可以使用任一问题idReadable （例如， SP-8 ）或问题的实体id （例如， 2-7 ）以供参考。

    GET / api / issues / {issueID}？[字段]

    进一步了解有问题的运作

  2. 的fields request参数是要在服务器响应中返回的返回实体（在本例中为Issue）属性的逗号分隔列表。

    您可以请求返回实体的任何支持的属性集。有关问题，请参阅支持的属性列表 。

    在本操作方法中，我们要求以下数据：

    ？fields = $ type，id，summary，customFields（$ type，id，projectCustomField（$ type，id，field（$ type，id，name）），value（$ type，avatarUrl，buildLink，color（id），fullName， id，isResolved，localizedName，login，minutes，name，presentation，text））'

    • $type -实体类型。YouTrack返回$type响应中的所有实体，无论您是否在请求中明确指定。

    • id -数据库中返回的实体的唯一ID。

    • summary -问题摘要。

    • customFields -返回的属性Issue实体，其中包含问题自定义字段的集合。这是我们的操作方法的目标。

  3. 注意我们为定制字段获得的属性：

    ？customFields（$ type，id，projectCustomField（$ type，id，field（$ type，id，name）），value（$ type，avatarUrl，buildLink，color（id），fullName，id，isResolved，localizedName，login，minutes ，名称，演示文稿，文本））'

    • projectCustomField -每个IssueCustomField包含一个的设置ProjectCustomField在一个特定的问题。

    • field -分别ProjectCustomField包含一个的设置CustomField在一个特定的项目中。

    • value -而CustomField是最常见的实体IssueCustomField是最特殊的，并且在特定问题中包含字段值。值可以只是一个简单的值（字符串，整数，..），链接到某个实体（链接到“受让人”字段的特定用户）或值的集合（“受影响的版本”字段）。所以，我们得到value所有类型的属性。

    阅读有关自定义字段层次结构的更多信息 。

最终请求如下：

curl -X GET \'https://example.myjetbrains.com/youtrack/api/issues/SP-8?fields=$type,id,summary,customFields($type,id,projectCustomField($type,id,field （$ type，id，name）），value（$ type，avatarUrl，buildLink，color（id），fullName，id，isResolved，localizedName，login，minutes，name，presentation，text））'\ -H'接受： application / json'\ -H'授权：不记名perm：amFuZS5kb2U =。UkVTVCBBUEk = .wcKuAok8cHmAtzjA6xlc4BrB4hleaX'\ -H'Cache-Control：no-cache'\ -H'Content-Type：application / json'

使用自定义字段获取问题列表

要获取每个问题的数据，只需删除{issueID}从端点URL。例如：

curl -X GET \'https://example.myjetbrains.com/youtrack/api/issues?fields=id,summary,customFields(id,projectCustomField(field(name)),value(name））'\ -H'接受：application / json'\ -H'授权：不记名perm：amFuZS5kb2U =。UkVTVCBBUEk = .wcKuAok8cHmAtzjA6xlc4BrB4hleaX'\ -H'Cache-Control：no-cache'\ -H'Content-Type：application / json'
