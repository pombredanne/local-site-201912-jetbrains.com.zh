Google Auth模块

通过此身份验证模块，用户可以使用在Google中管理的电子邮件地址和密码登录到YouTrack。

在YouTrack中启用Google身份验证时：

  • 您的用户使用其Google帐户使用的凭据登录到YouTrack。

  • 您的YouTrack用户需要记住的帐户和密码更少。

  • 具有Google帐户的新用户可以在YouTrack中创建自己的帐户。

启用Google身份验证

要允许具有现有Google帐户的用户登录到YouTrack，请启用Google身份验证模块。

此过程分为三个步骤：

  1. 在YouTrack中生成重定向URI 。当您为Google创建身份验证模块时，YouTrack会生成用于此服务的重定向URI。该URI标识了每个登录到Google的请求的来源。

  2. 在Google API管理器中生成客户端ID和密码 。从YouTrack发送到Google的每个登录请求都包含一个唯一标识符。您存储在身份验证模块中的ID和机密会告诉Google每个登录请求都是经过授权的。

  3. 在YouTrack中启用身份验证模块 。生成了YouTrack用来向Google进行身份验证的信息后，请将这些值复制到YouTrack并启用该模块。

在此过程中，您将在YouTrack和Google API管理器中生成值。

每个步骤的标题告诉您要遵循的应用程序菜单。

在YouTrack中生成重定向URI

首先，打开YouTrack并为Google帐户创建一个身份验证模块。创建身份验证模块时，YouTrack会生成重定向URI以与授权服务一起使用。

要在YouTrack中生成重定向URI，请执行以下操作：

需要权限：低级管理员写

  1. 在管理菜单的访问管理部分，选择身份验证模块 。

  2. 从添加模块下拉列表中，选择Google 。

    • 验证模块页面显示新的Google验证模块的设置。

    • YouTrack会生成一个重定向URI，供您在Google中使用。

  3. 按照页面上的说明复制重定向URI。

  4. 点击链接以访问Google API管理器。

    此设置要求您将值从Google API管理器复制到YouTrack上此页面上的输入字段中。

    为了简化设置，请在新的浏览器选项卡或窗口中打开此链接。

在Google API管理器中生成客户端ID和机密

  1. 登录到您的Google帐户。

  2. 选择或创建一个项目。

  3. 打开API管理器 。

  4. 在左侧，选择凭证 。

  5. 从创建凭据下拉列表中，选择OAuth客户端ID 。

    • 将打开“ 创建客户端ID”页面。

  6. 单击配置同意屏幕按钮。

  7. 在OAuth同意屏幕页面中，输入产品名称，然后点击保存按钮。

    • 您将被重定向到“ 创建客户端ID”页面。

  8. 对于“ 应用程序类型” ，选择“ Web应用程序” 。

    • 显示了用于定义客户端ID的其他输入字段。

  9. 在“ 授权的重定向URI”字段中，将您从“ 身份验证模块”页面中复制的重定向URI粘贴到集线器中。

  10. 单击创建按钮。

    • Google会生成设置集线器模块所需的凭据，并将其显示在弹出窗口中。

在YouTrack中启用身份验证模块

  1. 从Google复制客户ID，然后将其粘贴到YouTrack的客户ID输入字段中。

  2. 从Google复制客户机密，然后将其粘贴到YouTrack的“ 客户机密”输入字段中。

  3. 配置身份验证模块的可选设置。有关更多信息，请参阅可选设置 。

  4. 单击启用模块按钮。

    • Google身份验证模块已启用。

    • 按钮图像设置中存储的图标将添加到登录对话框窗口中。用户可以单击此图标以使用其Google帐户进行身份验证。

描述

类型

显示在YouTrack中启用了第三方身份验证的应用程序或服务的名称。

名称

存储认证模块的名称。使用此设置可以将该模块与“身份验证模块”列表中的其他身份验证模块区分开。

按钮图片

显示用于用户单击以使用Google帐户登录到YouTrack的按钮的图像。

授权重定向URI

显示用于在Google中注册到YouTrack的连接的重定向URI。

服务器网址

当用户使用Google帐户登录时，显示YouTrack向其发送登录请求的服务器的URL。

此字段下方显示的信息可帮助您配置身份验证模块。

客户编号

存储Google用于验证登录请求的标识符。当您配置Web应用程序的授权设置并输入授权的重定向URI时，便会在Google API管理器中生成此值。

客户机密

存储用于验证客户端ID的机密或密码。您会在Google API管理器中生成此值以及客户端ID。

其他设置

以下选项位于页面底部。使用这些设置可以管理YouTrack帐户的创建和组成员身份，并减少空闲连接消耗的处理资源的损失。

选项

描述

用户创建

为使用存储在连接的授权服务中的帐户登录的未注册用户创建YouTrack帐户。YouTrack使用电子邮件地址来确定用户是否已有帐户。

限制网域

将用户帐户的创建限制为具有来自指定域的电子邮件地址的用户。仅当启用“ 创建用户”选项时，此选项才处于活动状态。

自动加入群组

当用户使用存储在连接的授权服务中的帐户登录时，将用户添加到组中。您可以选择一个或多个组。自动加入组的新用户将继承分配给该组的所有权限。
我们建议您将用户添加到至少一个组中。否则，仅授予新用户当前分配给“ 所有用户”组的权限。

连接超时

设置等待建立与授权服务的连接的时间。默认设置为5000毫秒（5秒）。

读取超时

设置等待从授权服务读取和检索用户配置文件数据的时间。默认设置为5000毫秒（5秒）。

审计

链接到YouTrack中的“ 审核事件”页面。在这里，您可以查看已应用于此身份验证模块的更改列表。
