GitLab身份验证模块

GitLab身份验证模块是一个预先配置的OAuth 2.0身份验证模块，该模块允许用户使用其GitLab凭据登录到YouTrack。

启用GitLab身份验证

要允许在GitLab中拥有现有帐户的用户登录到YouTrack，请启用身份验证模块。

此过程分为三个步骤：

  1. 在YouTrack中生成重定向URI 。创建身份验证模块时，YouTrack会生成重定向URI以与授权服务一起使用。此URI标识每个登录请求的源。

  2. 在GitLab中生成一个客户端ID和密钥 。从YouTrack发送的每个登录请求均包含唯一标识符。您存储在身份验证模块中的ID和密码会告诉GitLab授权服务每个登录请求均得到授权。

  3. 在YouTrack中启用身份验证模块 。生成信息后，YouTrack会使用它来通过GitLab授权服务进行身份验证，然后将这些值复制到YouTrack中并启用该模块。

在YouTrack中生成重定向URI

首先，打开YouTrack并为GitLab帐户创建身份验证模块。创建身份验证模块时，YouTrack会生成重定向URI以与授权服务一起使用。

要在YouTrack中生成重定向URI，请执行以下操作：

需要权限：低级管理员写

  1. 在管理菜单的访问管理部分，选择身份验证模块 。

  2. 从“ 新模块”下拉列表中，选择GitLab 。

    • 身份验证模块页面显示新的GitLab身份验证模块的设置。

    • YouTrack生成一个重定向URI，供您在授权服务中使用。

  3. 如果浏览器支持该功能，请使用“ 复制”按钮将重定向URI复制到剪贴板。

生成客户端ID和机密

下一步是在GitLab中为YouTrack注册授权的重定向URI。

要在GitLab中获取客户端ID和密码：

  1. 登录到您的GitLab实例并访问“ 应用程序”管理部分。

  2. 在“ 添加新应用程序”表单中，输入YouTrack服务的名称，然后将生成的重定向URL复制到“ 重定向URI”字段中。

  3. 在“ 作用域”选项中，选择api和read_user 。

  4. 单击保存应用程序按钮。

    • GitLab会生成您用作YouTrack中的客户端ID和客户端密钥的值。

在YouTrack中启用身份验证模块

  1. 从GitLab复制应用程序ID ，并将其粘贴到YouTrack的“ 客户端ID”输入字段中。

  2. 从GitLab复制密钥 ，然后将其粘贴到YouTrack的“ 客户端密钥”输入字段中。

  3. 配置身份验证模块的可选设置。有关更多信息，请参见其他设置 。

  4. 单击保存按钮以应用设置。

  5. 单击启用模块按钮。

    • GitLab身份验证模块已启用。

    • 按钮图像设置中存储的图标将添加到登录对话框窗口中。用户可以单击此图标以在GitLab中使用其凭据登录到YouTrack。

设定值

设置页面的第一部分显示身份验证模块的常规设置。在这里，您还可以找到用于在授权服务中注册YouTrack的重定向URI，以及存储在授权服务中生成的Client ID和Client Secret的输入字段。

设置

描述

类型

显示在YouTrack中为第三方身份验证启用的授权服务的类型。

名称

存储认证模块的名称。使用此设置可以将该模块与“身份验证模块”列表中的其他身份验证模块区分开。

按钮图片

显示用于用户单击以使用其在连接的授权服务中的帐户登录YouTrack的按钮的图像。您可以上传JPG，GIF或PNG文件。图像会自动调整为48 x 48像素。

重定向URI

显示用于将连接注册到授权服务中的YouTrack的授权重定向URI。

客户编号

存储授权服务用来验证登录请求的标识符。当您配置Web应用程序的授权设置并输入授权的重定向URI时，会在授权服务中生成此值。

客户机密

存储用于验证客户端ID的机密或密码。您在授权服务中生成此值以及客户端ID。

授权服务端点

页面此部分中的设置存储GitLab使用的OAuth 2.0端点。

对于预配置的OAuth 2.0模块，将自动设置所选授权服务使用的值。

设置

描述

授权书

存储YouTrack用于通过用户代理重定向从资源所有者获取授权的端点。

代币

存储YouTrack用来交换访问令牌的授权授予的端点。

用户数据

存储用于定位已认证用户的概要文件数据的端点。

电子邮件

用于查找经过身份验证的用户的电子邮件地址的端点仅在电子邮件地址未存储在用户配置文件中时使用

默认电子邮件验证状态

当验证服务未返回电子邮件地址的验证状态时，确定应为YouTrack中的电子邮件地址设置哪个状态。

场图

当GitLab返回用户配置文件响应对象时，来自指定字段路径的值将复制到与每个YouTrack配置文件链接的中心帐户。使用以下设置来定义终结点，该终结点可为已验证用户查找配置文件数据，并将存储在授权服务中的字段映射到YouTrack用户的集线器帐户。

对于GitLab模块，将自动设置值。

使用以斜杠（/）分隔的一系列路径段来指定嵌套对象内部字段的路径。

其他设置使您可以定义请求范围，并选择如何通过服务进行身份验证。

领域

描述

用户身份

映射到存储值的字段，该值将复制到中心帐户的“ 用户ID”属性中。

电子邮件

映射到存储要复制到集线器帐户中“ 电子邮件”字段的值的字段。

验证邮件

映射到存储值的字段，该值将复制到中心帐户的已验证电子邮件属性中。

全名

映射到存储要复制到集线器帐户中的“ 全名”字段的值的字段。

头像

映射到存储图像的字段，以用作集线器帐户中的头像 。

图片网址格式

为ID引用的头像生成图像URL。使用占位符引用存储化身的字段。

范围

设置访问请求的范围。输入范围列表，以空格分隔。

认证方式

确定如何将凭据传递到授权服务。

其他设置

以下选项位于页面底部。使用这些设置可以管理集线器帐户的创建和组成员身份，并减少空闲连接消耗的处理资源的损失。

选项

描述

用户创建

为使用存储在连接的授权服务中的帐户登录的未注册用户启用集线器帐户的创建。YouTrack使用电子邮件地址来确定用户是否已有帐户。

自动加入群组

当用户使用存储在连接的授权服务中的帐户登录时，将用户添加到组中。您可以选择一个或多个组。自动加入组的新用户将继承分配给该组的所有权限。
我们建议您将用户添加到至少一个组中。否则，仅授予新用户当前分配给“ 所有用户”组的权限。

连接超时

设置等待建立与授权服务的连接的时间。默认设置为5000毫秒（5秒）。

读取超时

设置等待从授权服务读取和检索用户配置文件数据的时间。默认设置为5000毫秒（5秒）。

审计

链接到YouTrack中的“ 审核事件”页面。在这里，您可以查看已应用于此身份验证模块的更改列表。
