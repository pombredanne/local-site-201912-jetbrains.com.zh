SprintActivityItem

表示将问题添加到敏捷板冲刺中或从敏捷冲刺中删除时的更改。

扩展MultiValueActivityItem

下表描述了SprintActivityItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

类型

描述

 问题

 冲刺数组

添加

 冲刺数组

 用户

领域

 FilterField

过滤器字段。它包含有关目标实体中已修改属性的其他信息。例如，有关链接类型或修改后的自定义字段或捆绑软件类型的信息，等等。 Read-only 。 Can be null 。

类别

 活动类别

活动的类别。 Read-only 。
