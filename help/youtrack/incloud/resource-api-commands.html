CommandList属性

代表YouTrack中的命令列表和相关注释。可用于应用命令或获取命令建议。

下表描述了CommandList实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

样品1

对命令的最低要求要求您在请求正文中发送查询（要应用的命令）以及至少应由其ID表示的问题集合。对于此示例，最小请求没有fields参数，服务器使用以下命令发送响应200 OK状态和一个空的身体。

样本请求URI

https://example.myjetbrains.com/youtrack/api/commands

样品申请机构

{“查询”：“固定”，“问题”：[{“ id”：“ 2-17”}]}

样品2

以下请求将特定问题分配给当前用户，并添加可见性受到限制的新评论。该命令以静默方式应用，不发送通知。

样本请求URI

https://example.myjetbrains.com/youtrack/api/commands?fields=issues(id,idReadable),query,visibility(permittedGroups(id,name),permittedUsers(id,login））

样品申请机构

{“查询”：“对我来说”，“问题”：[{“ id”：“ 2-15”}]，“沉默”：是，“评论”：“仍然无法复制。”，“可见性”：{ “ $ type”：“ CommandLimitedVisibility”，“ permittedGroups”：[{“ id”：“ 3-2”}]}}

样品响应体

{“ query”：“为我”，“问题”：[{“ idReadable”：“ SP-16”，“ id”：“ 2-15”，“ $ type”：“问题”}]，“可见性” ：{“ permittedGroups”：[{“ id”：“ 3-2”}]，“ $ type”：“ CommandLimitedVisibility”}}
