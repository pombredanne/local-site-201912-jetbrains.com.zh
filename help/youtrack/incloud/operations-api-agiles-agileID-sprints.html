具有特定冲刺的操作

该资源使您可以处理特定敏捷板的sprint。

资源资源
/ api / agiles / {agileID} / sprints / {sprintID}

退货实体

冲刺 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的Sprint 。

  • DELETE ： 删除特定的Sprint 。

  • POST ： 更新特定的Sprint 。

冲刺属性

表示与敏捷板关联的Sprint。每个冲刺可以包含一个或多个项目的问题。

下表描述了Sprint实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

敏捷

 敏捷

冲刺所属的敏捷板。 Read-only 。 Can be null 。

名称

串

冲刺的名称。 Can be null 。

目标

串

冲刺的目标。 Can be null 。

开始

长

冲刺的开始日期作为时间戳。 Can be null 。

完

长

sprint的结束日期作为时间戳。 Can be null 。

已封存

布尔型

指示是否已将Sprint存档。

isDefault

布尔型

如果为true，则与该板上的一列匹配的新问题会自动添加到此sprint中。

问题

 一系列问题

此Sprint中存在的问题。

unresolvedIssuesCount

整数

此Sprint上未解决的问题数。 Read-only 。

previousSprint

 短跑

如果在创建新的Sprint时提供此属性，则该Sprint的所有未解决的问题都将移至新创建的Sprint。

阅读特定的冲刺

获取敏捷板特定冲刺的设置。

所需权限

用户必须具有阅读板的权限。

请求语法

GET / api / agiles / {agileID} / sprints / {sprintID}？{字段}

{sprintID}

冲刺的数据库ID。

请求参数

参数

类型

描述

领域
