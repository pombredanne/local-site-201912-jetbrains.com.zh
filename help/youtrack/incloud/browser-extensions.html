Browser Extensions

Several software companies and members of the user community have developed browser extensions that enhance your experience with YouTrack. Some extensions add functionality that is not available in the core product, while others facilitate integrations with third-party applications.

The following browser extensions support features that aren't otherwise available in YouTrack. Follow the links to learn more.

Extension

Description

YouTrack Issue Cards Printer

This extension for Chrome lets you print a list of issues in card format. You can then arrange these issues on a physical scrum board or use them for other offline activities.

YouTrack Tweaks

The YouTrack Tweaks extension for Chrome lets you customize the look and feel of your agile boards. You can also enable desktop notifications for updates that are applied to issues on an agile board.

To learn more about this extension, read this article in the YouTrack blog.

These extensions support integrations between YouTrack and other web-based applications. Check this list to see if any of the applications you use offer browser extensions that enable integrations with YouTrack. Follow the links to learn more.

Extension

Description

Clockify

The Clockify extension for Chrome lets you track the time you spend on YouTrack issues. You can then edit and manage time entries in the Clockify web app, mobile app, desktop app, or browser extension. All your time tracking data is synced in real time so you can seamlessly switch between devices.

The integration with Clockify is only available for YouTrack InCloud instances that are hosted from the myjetbrains.com domain.

Learn more about Clockify on their official website.

Harvest

The YouTrack support for Harvest Time Tracker supports one-way (Harvest to YouTrack) synchronization for time tracking data.

TMetric

TMetric is a browser extension that you can use to track the time that you spend working on an issue in YouTrack. TMetric offers extensions for Chrome, Firefox, Opera, and Yandex web browsers.

To learn more about this extension, read this article in the YouTrack blog.

Toggl

The Toggl Button: Productivity & Time Tracker extension adds a timer to any web tool. The extension lets you track your productivity in real time and store the data in your Toggl account.

This extension is available for Chrome and Firefox.

To learn more about the extension, check out this blog post.
