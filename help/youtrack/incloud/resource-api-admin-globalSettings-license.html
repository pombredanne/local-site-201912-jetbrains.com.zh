License

This resource lets you work with the license of your YouTrack service.

Resource

/api/admin/globalSettings/license

Returned entity

License. For the description of the entity attributes, see Supported Fields section.

License attributes

Represents the current license of the YouTrack service.

This table describes attributes of the License entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

username

String

Username used for license. Can be null.

license

String

License key. Can be null.

error

String

License error. If there is no error, this property is empty. Read-only. Can be null.
