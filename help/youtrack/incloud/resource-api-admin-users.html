YouTrack InCloud 2019.2帮助

用户数

此资源使您可以使用YouTrack中的用户设置。

请注意，该资源仅允许您管理用户帐户的YouTrack配置文件。任何与授权或访问管理有关的操作都必须使用Hub REST API来执行。也就是说，对于诸如创建新用户，更新用户凭证，授予或撤消角色和权限等操作，必须使用各自的Hub REST API端点 。

资源资源
/ api / admin / users

退货实体

用户 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读用户列表 。

支持的子资源

  • / api / admin / users / {userID}： 特定用户的操作

  • / api / admin / users / {userID} / profiles / timetracking： 时间跟踪用户配置文件

  • / api / admin / users / {userID} / tags： 问题标签

  • / api / admin / users / {userID} / savedQueries： 已保存的查询

  • / api / admin / users / {userID} / profiles / general： 一般用户个人资料

  • / api / admin / users / {userID} / profiles / notifications： 通知用户个人资料

用户属性

代表YouTrack中的用户。

下表描述了User实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。
