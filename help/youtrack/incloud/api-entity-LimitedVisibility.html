LimitedVisibility

Represents visibility limited to several users and/or groups.

Extends Visibility

This table describes attributes of the LimitedVisibility entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

permittedGroups

Array of UserGroups

Members of these groups are allowed to access the entity.

permittedUsers

Array of Users

These users are allowed to access the entity.
