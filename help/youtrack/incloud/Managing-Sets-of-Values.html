管理值集

一组值只是可以存储在自定义字段中的可能值的列表。

在具有枚举类型的自定义字段中使用值集，例如enum[1|*]要么version[1|*] 。默认的自定义字段State ， Priority和Build也是枚举类型。本页介绍如何创建，编辑和自定义这些值集。

这些过程描述了如何配置通用enum[1|*]类型。对于状态，构建，子系统和版本等其他值集，该过程相似，因此未明确描述。

存储用户类型的字段也是使用一组值的枚举类型。这些字段具有您应以特定方式配置的其他属性。要了解有关存储用户类型的字段的更多信息，请参阅管理分配人 。

一组值可以在不同项目的多个自定义字段中使用。不同的项目可以为相同的自定义字段使用独立的值集。

创建一组新值

创建具有枚举类型的字段后，可以定义一组值。

要创建一组新值：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 单击添加新的自定义字段按钮。

    • 侧边栏中将打开“ 新建自定义字段”对话框。

  4. 在“ 新建自定义字段”对话框中，输入“ 字段名称” 。

  5. 在“ 类型”下拉列表中，选择enum 。

  6. 单击添加新字段按钮。

    • 自定义字段已添加到YouTrack。

    • 自定义字段设置显示在侧栏中。

  7. 点击创建新设置按钮

  8. 单击添加值按钮。

    • “ 添加值”对话框打开。

  9. 在“ 添加值”对话框中，输入“ 名称”和可选的“ 描述” ，选择可选的配色方案（背景和文本颜色的组合），然后单击“ 保存”按钮。

  10. 从第7步开始重复，直到添加了要用于自定义字段的所有值。

将值添加到集合

要将值添加到现有集中，可以手动添加新值，也可以从另一组值中复制/合并/替换现有值。

无论您可以在何处更改字段的值，都可以选择向该集合添加新值。项目管理员可以将值添加到项目唯一的字段中，也可以使用一组独立的值。全局项目中的系统管理员和具有项目管理员访问权限的用户可以将值添加到所有项目中的集合中。创建后，新值将添加到问题中。此选项在以下位置可见：

  • 问题列表

  • 单期观点

  • 敏捷板

如果该字段使用一组独立的值，则该值将添加到项目中分配给问题的自定义字段。

您可以在“ 自定义字段设置”页面上向自定义字段添加值。

要将值添加到集合中：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择一个枚举类型的自定义字段。

  4. 如果将自定义字段附加到一个或多个项目，则可以使用以下选项：

    • 所有受影响项目的编辑集 -添加到列表的值在使用此字段的所有项目中均可用。

    • 进行独立复制 -添加到列表中的值仅在当前项目中可用。

    选择适合您目的的选项。

  5. 单击添加值按钮。

    • “ 添加值”对话框打开。

  6. 在以下字段中输入值：

    • 名称（必填）

    • 描述

  7. （可选）选择一种配色方案（背景和文本颜色的组合）。例如，对于“ 优先级”字段中的一组值，您可以选择以红色突出显示“ 关键”和“ 显示停止器”值。

  8. 单击保存按钮。

对集中的值进行排序

默认情况下，集合中的值按添加到字段的顺序手动排序。手动订购时，将为每个值分配一个序数值，以表示其在列表中的位置。

排序顺序会影响YouTrack中的以下行为：

  • 自定义字段的下拉列表中显示的值以指定的排序顺序显示。例如，默认的优先级字段的值从高到低手动排序。代表最高优先级的值（ Show-stopper ）出现在列表的顶部。

  • 在指定值范围的搜索查询中，排序顺序确定哪些值与搜索请求匹配。例如，包含以下内容的搜索请求Priority: Show-stopper .. Major返回的问题包括Show-stopper ， Critical和Major 。

作为管理员，您可以确定多个项目中使用的字段中的值的排序顺序。项目管理员可以通过管理一组值的独立副本来独立设置每个项目的排序顺序。有关更多信息，请参见对集合中的值进行排序 。

您可以在“ 自定义字段设置”页面的“ 字段列表”选项卡上的自定义字段中对值进行排序。排序选项适用于所有存储枚举类型的字段，但不包含存储枚举类型的字段user要么group类型。

  • 对于存储一个user类型，则值将按全名自动按字母顺序排序，当前用户位于列表顶部。

  • 对于存储一个group类型，值按名称的字母顺序排序。

手动排序值

如果要覆盖值在自定义字段中出现的默认顺序，则可以手动使用列表中的值。

要使用手动排序的值：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择一个具有一组可以排序的值的自定义字段。

  4. 如果将自定义字段附加到一个或多个项目，则可以使用以下选项：

    • 编辑所有受影响项目的集 -自定义排序顺序将应用到所有使用此值集的项目中。

    • 进行独立复制 -自定义排序顺序存储在该字段的默认值集中。

    选择适合您目的的选项。

  5. 将值拖到所需的顺序。

    • 自定义订单将自动保存。

按名称对值排序

对于任何支持手动排序的字段，您可以选择按名称自动对值进行排序。值可以按名称升序或降序排序。按名称排序时，添加到集合中的新值将按字母顺序显示。按名称排序有两个选项：

  • 区分大小写的选项按字母顺序从A-Z，然后是a-z排序值。

  • 不区分大小写的选项按字母顺序对值进行排序，而不管第一个字母使用哪种大小写。

当您从手动排序切换到任何其他排序选项时，手动选项将被丢弃。如果您手动恢复为排序值，则必须重新应用自定义顺序。

按名称对值进行排序：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择一个具有一组可以排序的值的自定义字段。

  4. 如果将自定义字段附加到一个或多个项目，则可以使用以下选项：

    • 编辑所有受影响项目的集 -排序顺序将应用到所有使用此值集的项目中。

    • 进行独立复制 -排序顺序存储在该字段的默认值集中。

    选择适合您目的的选项。

  5. 打开排序菜单，然后选择“ 按名称排序”选项（区分大小写或不区分大小写）。

  6. 在“ 更改排序顺序”对话框中确认操作。

    • 如果这些值以前是手动订购的，则手动订购将被丢弃。

    • 集合中的值按名称按字母升序排序。

  7. 如果要按降序对值进行排序，请从下拉列表中选择desc选项。

按版本对值排序

对于存储一个version要么build ，您可以选择按版本对值进行排序。值可以按版本升序或降序排序。使用此选项，数字值将根据语义版本控制规范 （SemVer）自动排序。

此功能最有价值的应用是在搜索查询中。对于许多团队来说，搜索在特定版本或内部版本中已解决的问题的能力有助于为发布发行说明生成准确的列表。

要按版本对值进行排序：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择一个自定义字段来存储version要么build 。

  4. 如果将自定义字段附加到一个或多个项目，则可以使用以下选项：

    • 编辑所有受影响项目的集 -排序顺序将应用到所有使用此值集的项目中。

    • 进行独立复制 -排序顺序存储在该字段的默认值集中。

    选择适合您目的的选项。

  5. 打开排序菜单，然后选择“ 按版本排序”选项。

  6. 在“ 更改排序顺序”对话框中确认操作。

    • 如果这些值以前是手动订购的，则手动订购将被丢弃。

    • 集合中的值按版本升序排序。

  7. 如果要按降序对值进行排序，请从下拉列表中选择desc选项。

编辑一组值

作为管理员，您可以编辑在字段中用于系统中所有项目的一组值。

要在系统范围内编辑一组值：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择使用一组值的字段。

    • 所选自定义字段的详细信息对话框在侧栏中打开。

  4. 单击所有受影响项目的编辑集按钮。

  5. 编辑值集。提供以下选项：

    • 单击添加值按钮以将新值添加到列表。

    • 单击“ 从以下位置复制值”按钮，然后选择一个字段以复制所选字段中使用的值。有关更多信息，请参见从另一个集中复制值 。

    • 单击合并方式按钮，然后选择一个字段以将现有值与所选字段中使用的值合并。有关更多信息，请参见将值与另一组合并 。

从另一组复制值

如果您的自定义字段包含一组要用作其他自定义字段的基础的值，则可以从另一组中复制值。

要从另一组复制值：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择要在其中编辑值列表的字段。

  4. 选择是否要使更改影响所有项目。

  5. 单击“ 从以下位置复制值”按钮，然后选择一个包含要复制的值的字段。

    • 来自所选字段的值将添加到当前值集中。

与另一组合并值

合并两组值时，将获得具有一组通用值的一组值，这些值在合并之前使用该字段的所有项目中都可用。

要将另一组值与当前组合并：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 选择要在其中编辑值列表的字段。

  4. 选择是否要使更改影响所有项目。

  5. 单击合并方式按钮，然后选择要合并的字段。

    • 所选字段中的值与当前值集合并。
