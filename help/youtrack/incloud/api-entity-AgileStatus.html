敏捷状态

显示电路板是否有任何配置问题。

下表描述了AgileStatus实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

有效

布尔型

如果板处于有效状态且可以使用，则为true。 Read-only 。

hasJobs

布尔型

如果为“ true”，则当前正在为该板执行后台作业。在这种情况下，正在运行后台作业时，无法更新板。 Read-only 。

错误

字符串数组

找到此板的配置错误列表。 Read-only 。

警告

字符串数组

找到此主板的与配置有关的警告列表。 Read-only 。
