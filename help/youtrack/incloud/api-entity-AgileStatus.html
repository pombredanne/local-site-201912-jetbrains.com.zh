AgileStatus

Shows if the board has any configuration problems.

This table describes attributes of the AgileStatus entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

valid

Boolean

true if the board is in valid state and can be used. Read-only.

hasJobs

Boolean

If `true`, then a background job is currently being executed for the board. In this case, while a background job is running, the board cannot be updated. Read-only.

errors

Array of Strings

List of configuration errors found for this board. Read-only.

warnings

Array of Strings

List of configuration-related warnings found for this board. Read-only.
