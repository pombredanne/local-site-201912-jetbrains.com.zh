使用特定通知的操作

此资源使您可以读取和更新特定用户的配置文件中的通知设置。

资源资源
/ api / admin / users / {userID} / profiles / notifications

退货实体

NotificationsUserProfile 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的NotificationsUserProfile 。

  • POST ： 更新特定的NotificationsUserProfile 。

NotificationsUserProfile属性

代表特定用户的个人资料中的通知设置。

下表描述了NotificationsUserProfile实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

notifyOnOwnChanges

布尔型

指示用户是否收到有关其自身更改的通知。

jabberNotificationsEnabled

布尔型

指示用户是否通过jabber启用了通知。

emailNotificationsEnabled

布尔型

指示用户是否通过电子邮件启用了通知。

提到NotificationsEnabled

布尔型

指示当问题文本中提到通知时，用户是否收到通知。

plicateClusterNotificationsEnabled

布尔型

指示用户是否收到有关问题重复项的通知。

MailboxIntegrationNotificationsEnabled

布尔型

指示用户是否收到邮箱集成产生的通知。

usePlainTextEmails

布尔型

指示用户是否接收纯文本电子邮件。

autoWatchOnComment

布尔型

指示用户发表评论时是否自动订阅问题更新。

autoWatchOnVote

布尔型

指示用户对问题进行投票时是否自动订阅问题更新。

autoWatchOnUpdate

布尔型

指示用户在更新问题时是否自动订阅问题更新。

读取特定的NotificationsUserProfile

获取特定用户配置文件的通知设置。

所需权限

需要“读取自我”才能读取自己的个人资料，需要“不拥有个人资料”来读取其他用户的个人资料。

请求语法

GET / api / admin / users / {userID} / profiles / notifications？{字段}

请求参数

参数

类型

描述

领域

串

响应中应返回的NotificationsUserProfile属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/users/1-2/profiles/notations？

样品响应体

{“ emailNotificationsEnabled”：true，“ autoWatchOnComment”：true，“ autoWatchOnUpdate”：false，“ autoWatchOnVote”：true，“ notifyOnOwnChanges”：false，“ jabberNotificationsEnabled”：true，“ mailboxIntegrationNotificationsEnabled”：true，“ usePlainTextEmails”：false，“ sayNotificationsEnabled”：true，“ duplicateClusterNotificationsEnabled”：false，“ id”：“ notificationsProfile”，“ $ type”：“ NotificationsUserProfile”}

更新特定的NotificationsUserProfile

更改特定用户配置文件中的通知设置。

所需权限

需要“更新自我”来更新自己的配置文件，需要“更新自己的配置文件”来更新另一个用户的配置文件。

请求语法

POST / api / admin / users / {userID} / profiles / notifications？{字段}

请求参数

参数

类型

描述

领域

串

响应中应返回的NotificationsUserProfile属性列表。如果未指定任何字段，则仅entityID返回

样品

让我们为用户禁用Jabber通知和来自邮箱集成的通知。

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/users/1-2/profiles/notifications?fields=jabberNotificationsEnabled、emailNotificationsEnabled、mailboxIntegrationNotificationsEnabled

样品申请机构

{“ jabberNotificationsEnabled”：否，“ mailboxIntegrationNotificationsEnabled”：否}

样品响应体

{“ emailNotificationsEnabled”：true，“ jabberNotificationsEnabled”：false，“ mailboxIntegrationNotificationsEnabled”：false，“ $ type”：“ NotificationsUserProfile”}
