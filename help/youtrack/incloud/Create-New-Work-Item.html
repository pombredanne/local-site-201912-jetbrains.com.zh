创建新工作项

POST / rest / issue / {issue} / timetracking / workitem

为特定问题创建一个新的工作项目。

POST / rest / issue / {issue} / timetracking / workitem

参数：

名称

类型

描述

问题

问题编号

工作项目添加到的问题ID

在请求代码中，为新工作项目提供以下参数（工作项目指定为标签）：

  1. — Unix Epoch时间格式的新工作项目的日期

  2. —新工作项的持续时间，以分钟为单位

  3. —活动说明

  4. —工作项类型

参数的时间部分将替换为当前用户所在时区中午夜的时间戳。

有关示例代码，请参见下文。
