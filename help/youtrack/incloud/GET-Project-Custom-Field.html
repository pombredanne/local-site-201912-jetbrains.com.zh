获取项目自定义字段

GET / admin / project / {projectId} / customfield / {customFieldName}

通过名称获取项目的自定义字段。

GET / rest / admin / project / {projectId} / customfield / {customFieldName}

请求参数：

名称

类型

描述

projectId

projectByIdNotStrict

项目的ID。

customFieldName

串

要获取的自定义字段的名称。

响应参数：

名称

描述

名称

项目自定义字段的名称。

类型

此自定义字段的类型。

canBeEmpty

强制性二进制参数，用于定义该字段是否可以为空值。

emptyText

自定义字段没有值时显示的文本。

参数

自定义字段的附加参数。例如，附加有自定义字段的捆绑软件。

样品

请求

GET / rest / admin / project / {projID} / customfield / State

响应

已提交
