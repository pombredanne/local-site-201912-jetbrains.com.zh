YouTrack InCloud 2019.2帮助

特定项目的运作

此资源使您可以阅读问题所属的项目，并将问题移至另一个项目。

资源资源
/ api / issues / {issueID} / project

退货实体

项目 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定项目 。

  • PUT ： 更改项目 。

项目属性

代表一个YouTrack项目。

扩展IssueFolder

下表描述了Project实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

startingNumber

长

项目中问题的起始编号。仅在创建新项目期间可以设置此属性。

简称

串

项目的ID。该短名称也是问题ID的前缀。 Can be null 。

描述

串

在项目资料页面上显示的项目描述。 Can be null 。

领导

 用户

设置为项目负责人的用户。 Can be null 。

由...制作

 用户

创建项目的用户。 Can be null 。

问题

 一系列问题

属于项目的所有问题的列表。

customFields

 ProjectCustomField

项目中可用的自定义字段集。

已封存

布尔型

如果项目当前已归档，则此属性为true。

来自电子邮件

串

用于发送项目通知的电子邮件地址。如果未为项目设置“发件人”地址，则返回YouTrack服务器的默认“发件人”地址。

ReplyToEmail

串

用作回复电子邮件以发送项目通知的电子邮件地址。如果未为项目设置，则返回YouTrack服务器的默认地址。 Can be null 。

模板

布尔型

如果true ，该项目是模板。

iconUrl

串

项目图标的URL。 Read-only 。

名称

串

问题文件夹的名称。 Can be null 。

变更专案

更改问题的项目。也就是说，将问题移到另一个项目。

请求语法

放入/ api / issues / {issueID} / project？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的Project属性列表。如果未指定任何字段，则仅entityID返回

样品

要将问题移到另一个项目，必须在请求有效负载中提供目标项目的数据库ID。

样本请求URI

https://example.myjetbrains.com/youtrack/api/issues/SP-8/project?fields=id,name,shortName

样品申请机构

{“ id”：“ 0-14”}

样品响应体

{“ shortName”：“ KNBN”，“ name”：“来自REST的另一个看板项目”，“ id”：“ 0-14”，“ $ type”：“项目”}
