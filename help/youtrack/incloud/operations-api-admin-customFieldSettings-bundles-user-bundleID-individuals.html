捆绑中的个人用户

此资源使您可以使用分别添加到特定用户集（用户捆绑包）的用户帐户。

资源资源
/ api / admin / customFieldSettings / bundles / user / {bundleID} / individuals / {userID}

退货实体

用户 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 捆绑中的个人用户 。

  • DELETE ： 捆绑中的个人用户 。

用户属性

代表YouTrack中的用户。

下表描述了User实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

登录

串

用户的登录名。 Read-only 。 Can be null 。

全名

串

在其个人资料中看到的用户的全名。 Read-only 。 Can be null 。

电子邮件

串

用户的电子邮件地址。 Read-only 。 Can be null 。

jabberAccountName

串

用户的用户名。 Read-only 。 Can be null 。

ringId

串

集线器中用户的ID。您可以使用此ID进行集线器中的操作，以及在YouTrack和集线器之间匹配用户。 Read-only 。 Can be null 。

来宾

布尔型

指示用户是否guest 。 Read-only 。

线上

布尔型

指示用户当前是否在线。 Read-only 。

被禁止

布尔型

指示是否禁止用户。 Read-only 。

标签

 IssueTag数组

属于该用户的标签。 Read-only 。

保存的查询

 保存的查询数组

属于该用户的已保存搜索。 Read-only 。

avatarUrl

串

用户头像的URL。 Read-only 。

型材

GeneralUserProfile |
NotificationsUserProfile |
TimeTrackingUserProfile

用户个人资料列表。 Read-only 。

读取特定用户

从捆绑中获取指定的用户。

所需权限

需要对捆绑软件的读取权限。

请求语法

GET / api / admin / customFieldSettings / bundles / user / {bundleID} / individuals / {userID}？{字段}

{用户身份}

用户的数据库ID。

请求参数

参数

类型

描述

领域

串

应在响应中返回的用户属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/customFieldSettings/bundles/user/8-2/individuals/1-4?fields=name,id,login,guest,banned,online,email,ringId

样品响应体

{“ email”：“ jane.smith@example.com”，“ online”：false，“ ringId”：“ 64cb04d0-f691-4386-900d-39f4c98972f9”，“ guest”：false，“ login”：“ Jane_Smith” ，“禁止”：false，“名称”：“简史密斯”，“ id”：“ 1-4”，“ $ type”：“用户”}

删除特定用户

从捆绑中删除指定的用户。

在使用该捆绑软件的所有项目中都需要“项目更新”权限。如果未在任何字段中使用捆绑软件，则至少在一个项目中需要“更新项目”权限。
