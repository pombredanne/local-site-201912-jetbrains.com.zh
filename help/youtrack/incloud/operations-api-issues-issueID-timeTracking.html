Operations with Specific IssueTimeTracking

This sub-resource lets you read work items of an issue and time tracking status in the issue's project.

Resource

/api/issues/{issueID}/timeTracking

Returned entity

IssueTimeTracking. For the description of the entity attributes, see Supported Fields section.

Supported methods

  • GET: Read a Specific IssueTimeTracking.

IssueTimeTracking attributes

Represents time tracking settings in the issue.

This table describes attributes of the IssueTimeTracking entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

workItems

Array of IssueWorkItems

Collection of work items of the issue.

enabled

Boolean

Indicates whether the time tracking is enabled in the project that the issue belongs to. Read-only.

Read a Specific IssueTimeTracking

Get work items of an issue and time tracking status.

Request syntax

GET /api/issues/{issueID}/timeTracking?{fields}

Request parameters

Parameter

Type

Description

fields

String

A list of IssueTimeTracking attributes that should be returned in the response. If no field is specified,only the entityID is returned

Sample

Sample request URI

https://example.myjetbrains.com/youtrack/api/issues/SNBX-1/timeTracking?fields=draftWorkItem(id),enabled,workItems(created,duration(presentation,minutes),author(name),creator(name),date,id)
