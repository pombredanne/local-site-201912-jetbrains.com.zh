具有特定IssueTimeTracking的操作

此子资源使您可以阅读问题的工作项以及问题项目中的时间跟踪状态。

资源资源
/ api / issues / {issueID} / timeTracking

退货实体

IssueTimeTracking 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的IssueTimeTracking 。

IssueTimeTracking属性

表示问题中的时间跟踪设置。

下表描述了IssueTimeTracking实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

工作项

 IssueWorkItems数组

收集问题的工作项。

已启用

布尔型

指示是否在问题所属的项目中启用了时间跟踪。 Read-only 。
