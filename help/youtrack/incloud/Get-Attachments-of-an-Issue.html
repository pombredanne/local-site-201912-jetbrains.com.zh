获取问题的附件

GET /问题/ {issue} /附件

获取所有问题附件

GET / rest / issue / {issue} / attachment

参数：

名称

类型

描述

问题

issueById

要获取附件的问题的ID。

响应参数：

名称

描述

网址

附件的URL。

名称

附件名称。

authorLogin

已添加附件的用户的登录名。

组

该附件可见的组的名称。

被创造

在YouTrack中添加（创建）附件的时间戳。时间戳是自该日期以来的格林尼治标准时间1970年1月1日以来的毫秒数。
