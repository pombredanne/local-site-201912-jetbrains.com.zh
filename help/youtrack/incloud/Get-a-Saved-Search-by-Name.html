按名称获取保存的搜索

GET / rest / user / search / {savedQuery}

请求参数：

名称

类型

描述

保存的查询

searchName

要获取的已保存搜索的名称。

响应参数：

名称

描述

名称

保存的搜索名称。

projectShortName

此搜索适用的项目的排序名称。

visibleForGroup

可以看到此保存的搜索的用户组。

可更新组

可以更新此保存的搜索的用户组。
