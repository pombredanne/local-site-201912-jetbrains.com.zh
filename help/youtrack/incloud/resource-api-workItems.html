发行工作项目

提供对工作项访问的资源。

资源资源
/ api / workItems

退货实体

IssueWorkItem 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读IssueWorkItems列表 。

IssueWorkItem属性

代表问题中的工作项目。

下表描述了IssueWorkItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

作者

 用户

在工作项中为其分配工作的用户。 Can be null 。

创作者

 用户

将工作项添加到问题的用户。 Read-only 。 Can be null 。

文本

串

工作项描述。 Can be null 。

textPreview

串

解析的描述预览。 Read-only 。

类型

 工作项类型

工作项类型。 Can be null 。

被创造

长

工作项的创建日期。 Can be null 。

更新

长

工作项目的最后更新日期。 Can be null 。

持续时间

 持续时间值

工作项目的持续时间。

日期

长

分配给工作项的日期和时间。存储为UTC中的Unix时间戳。时间部分设置为当前日期的午夜。

问题

 问题

存储工作项所附问题的属性。 Read-only 。

使用Markdown

布尔型

存储注释文本的格式。什么时候true ，则注释将以Markdown格式的文本进行处理。否则，将以YouTrack Wiki格式处理评论。

阅读IssueWorkItems列表

获取属于与搜索查询匹配的问题的所有工作项。如果未提供查询，则返回所有问题的所有工作项。

请求语法

GET / api / workItems？{字段}＆{$ top}＆{$ skip}＆{query}＆{startDate}＆{endDate}＆{start}＆{end}＆{author}＆{creator}

请求参数

参数

类型

描述

领域

串

应在响应中返回的IssueWorkItem属性的列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。

询问

串

发出搜索查询。在此处阅读有关搜索语法的更多信息： 搜索和命令属性

开始日期

日期

工作项日期间隔的开始日期。

结束日期

日期

工作项日期间隔的结束日期。

开始

串

工作项日期间隔的开始时间戳。

结束

串

工作项日期间隔的结束时间戳记。

作者

串

用于按作者过滤工作项的参数。可以为当前登录用户指定数据库ID或登录名或集线器ID或“我”的几个值

创作者

串

用于由创建者过滤工作项的参数。可以为当前登录用户指定数据库ID或登录名或集线器ID或“我”的几个值
