YouTrack InCloud 2019.2帮助

具有特定系统设置的操作

此资源使您可以使用YouTrack服务的“系统”设置。

资源资源
/ api / admin / globalSettings / systemSettings

退货实体

系统设置 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的SystemSettings 。

  • POST ： 更新特定的SystemSettings 。

SystemSettings属性

表示影响YouTrack核心功能的系统设置。

下表描述了SystemSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

baseUrl

串

YouTrack安装的基本URL。 Can be null 。

maxUploadFileSize

长

可以上传的文件的最大大小。

maxExportItems

整数

可以导出的最大问题数。

管理员电子邮件

串

服务器管理员的电子邮件。 Can be null 。

allowStatisticsCollection

布尔型

指示是否允许收集使用情况统计信息。

isApplicationReadOnly

布尔型

true ，如果应用程序当前处于只读模式。除此以外， false 。

阅读特定的系统设置

阅读您的YouTrack服务的系统设置。

所需权限

需要权限：低级管理员读取

请求语法

GET / api / admin / globalSettings / systemSettings？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的SystemSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/systemSettings?fields=baseUrl,maxUploadFileSize,maxExportItems,allowStatisticsCollection,isApplicationReadOnly

样品响应体

{“ maxUploadFileSize”：10485760，“ isApplicationReadOnly”：false，“ maxExportItems”：500，“ allowStatisticsCollection”：true，“ baseUrl”：“ https://example.myjetbrains.com/youtrack”，“ $ type”：“ SystemSettings “}
