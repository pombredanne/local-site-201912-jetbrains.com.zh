TextMarkupActivityItem

代表String -type属性并支持标记： description在Issue或IssueWorkItem中，以及text IssueComment。该实体使您可以在更改后获取呈现的文本。

扩展SimpleValueActivityItem

下表描述了TextMarkupActivityItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。
