使用特定EnumBundle的操作

该资源使您可以使用以下一组值enum类型。

资源资源
/ api / admin / customFieldSettings / bundles / enum / {bundleID}

退货实体

EnumBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的EnumBundle 。

  • POST ： 更新特定的EnumBundle 。

  • DELETE ： 删除特定的EnumBundle 。

EnumBundle属性

代表YouTrack中的一组枚举类型的值。

扩展BaseBundle

下表描述了EnumBundle实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

价值观

 EnumBundleElements数组

枚举中包含的字段值。

isUpdateable

布尔型

如果true ，则当前登录的用户可以更新此捆绑包。除此以外， false 。 Read-only 。

读取特定的EnumBundle

获取一组特定的值enum类型。

所需权限

需要对使用捆绑软件的字段之一进行读取或更新访问。如果未在任何字段中使用捆绑软件，则至少在一个项目中需要“更新项目”权限。
