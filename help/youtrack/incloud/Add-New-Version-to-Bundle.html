将新版本添加到捆绑

放置/ rest / admin / customfield / versionBundle / {bundleName} / {versionName}？{description}＆{colorIndex}＆{releaseDate}＆{released}＆{archived}

参数：

名称

类型

描述

bundleName

串

要添加新版本的捆绑软件的名称。

buildName

串

新版本的名称。

描述

串

版本说明。

colorIndex

整数

新版本的配色方案（背景颜色/文本颜色对）的顺序编号。查看可用的颜色指数列表

发布日期

瞬间

新版本的发布日期。

已发布

布尔值

将版本标记为已发布。

已封存

布尔值

将版本标记为已存档。
