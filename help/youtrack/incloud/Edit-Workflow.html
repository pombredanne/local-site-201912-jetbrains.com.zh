编辑工作流程

您可以修改现有工作流程以使其适应特定用例。您可以从“ 自定义工作流存储库”上载和自定义规则，修改自己的工作流，或编辑YouTrack实例中可用的默认工作流 。

当您将更改应用于默认工作流程时，即使产品升级中包含新版本的默认工作流程，您的更改也会保留下来。您可以随时还原更改并恢复工作流的默认定义。有关更多信息，请参见还原默认工作流 。

当更改保存在工作流编辑器中时，对工作流规则的更新将立即应用到它们所附加的所有项目中。

编辑工作流程：

需要权限：更新项目

  1. 在管理菜单的核心功能部分中，选择工作流程 。

  2. 找到要修改的工作流程，然后在列表中单击其名称。

    • 工作流程将在“ 工作流程”页面中打开。

    • 侧栏中将打开“ 新模块”对话框。

  3. 在这里，您可以添加模块以扩展工作流程，也可以修改现有模块。要在编辑器中打开现有模块，请在列表中单击该模块的名称。

  4. 修改规则或脚本以支持您的用例。

  5. 完成后，单击“ 保存”按钮。

    • 工作流规则的更新将立即应用到它们所附加的所有项目中。

删除工作流程和规则

您可以在YouTrack的“ 管理”区域中删除工作流程。如果要删除整个工作流程，包括其中包含的所有规则和自定义脚本，请遵循此过程。

此操作要求您在工作流当前附加到的所有项目中具有相关权限。

要在YouTrack中删除工作流程，请执行以下操作：

需要权限：更新项目

  1. 在管理菜单的核心功能部分中，选择工作流程 。

  2. 在列表中选择要删除的工作流程。

  3. 单击工具栏中的“ 删除选定的工作流程”图标。

  4. 在确认对话框中确认操作。

    • 所选工作流程将被删除。

请按照以下步骤删除单个规则，而不删除整个工作流程。

此操作要求您在规则当前所属的所有项目中都具有相关权限。

要删除单个工作流程规则：

需要权限：更新项目

  1. 在管理菜单的核心功能部分中，选择工作流程 。

  2. 找到要修改的工作流程，然后在列表中单击其名称。

    • 工作流程将在“ 工作流程”页面中打开。

  3. 在包含要删除的规则的模块旁边，单击“ 删除模块”图标。

  4. 在确认对话框中确认操作。

    • 所选模块已从工作流程中删除。
