ExternalIssue

For an issue that was imported from another service, represents the reference to the issue in the external system.

This table describes attributes of the ExternalIssue entity.

  • To receive an attribute in the response from server, specify it explicitly in the request parameter fields.

  • To update an attribute, provide it in the body of a POST request.

Field

Type

Description

name

String

The name of the external service. Read-only.

url

String

The URL of the external service. Read-only.

key

String

Issue key in the external system. Read-only.
