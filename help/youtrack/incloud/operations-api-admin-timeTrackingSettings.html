具有特定GlobalTimeTrackingSettings的操作

此资源使您可以配置系统范围的时间跟踪设置。

资源资源
/ api / admin / timeTrackingSettings

退货实体

GlobalTimeTrackingSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的GlobalTimeTrackingSettings 。

GlobalTimeTrackingSettings属性

代表服务器的时间跟踪设置。

下表描述了GlobalTimeTrackingSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

workItemTypes

 WorkItemTypes数组

可用工作项类型的集合。 Read-only 。

workTimeSettings

 工作时间设置

服务器的工作计划设置。 Read-only 。

阅读特定的GlobalTimeTrackingSettings

阅读服务的时间跟踪设置。

所需权限

需要“读取工作项”权限。

请求语法

GET / api / admin / timeTrackingSettings？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的GlobalTimeTrackingSettings属性列表。如果未指定任何字段，则仅entityID返回

https://example.myjetbrains.com/youtrack/api/admin/timeTrackingSettings?fields=workItemTypes(name,autoAttached,id),workTimeSettings(daysAWeek,firstDatOfWeek,minutesADay,workDays,id）
