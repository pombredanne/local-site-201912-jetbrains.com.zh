Zendesk整合

通过Zendesk集成，您可以共享带有YouTrack中问题的Zendesk票证。启用和配置集成后，共享的Zendesk票证和在YouTrack中从中创建的问题将自动同步。自定义字段从Zendesk到YouTrack沿一个方向同步。YouTrack问题中对自定义字段的更改不会应用于Zendesk中的共享票证。标记和注释通过两种方式同步。

启用和配置Zendesk集成

要启用和配置Zendesk集成，请按照下列步骤操作：

  • 向YouTrack发送邀请以共享Zendesk实例中的票证。

  • 在YouTrack中接受共享邀请，然后在YouTrack中配置集成设置。

要将邀请发送给YouTrack以共享票证，请执行以下操作：

  1. 在您的Zendesk实例中，打开“ 设置”>“票证”>“票证共享”管理页面。

  2. 点击添加共享邀请 。

  3. 在“与共享票证”对话框中，选择一个第三方系统 。

  4. 在与第三方系统共享票证表单中，输入以下设置的值：

    设置

    描述

    共享网址

    输入以下格式的YouTrack服务器的URL：http:///rest/networkedhelpdesk

    您可以在YouTrack的“ 管理”>“ Zendesk集成”页面上找到并复制共享URL。

    名称

    输入共享协议的名称。

  5. 使用其余设置确定在Zendesk和YouTrack之间同步哪些信息。

  6. 完成后，单击发送邀请按钮。

要继续集成的第二步，请登录到YouTrack服务器。

要接受来自Zendesk的共享邀请并配置集成，请执行以下操作：

需要权限：低级管理员写入，基本项目读取

  1. 在YouTrack中，打开“ 管理”>“ Zendesk集成”页面。如果邀请参数正确，您将看到从Zendesk发送的待处理共享邀请。

  2. 单击共享协议的名称。

    • 集成设置对话框在侧栏中打开。

  3. 在集成设置对话框中，输入以下设置的值：

    接受共享协议后，将无法修改YouTrack项目和状态字段设置。

    设置

    描述

    YouTrack专案

    从Zendesk中的共享票证中选择要在其中创建问题的项目。

    状态栏

    选择在Zendesk中存储故障单状态的自定义字段。

    对群组可见

    如果需要，选择有权查看从共享票证创建的问题的组。

    命令

    输入一个可选命令，该命令将应用于从共享票证创建的问题。

    仅当设置为YouTrack问题的报告者的用户具有足够的权限来更新所选项目中的问题时，才应用此命令。因此，如果您选择使用Zendesk用户作为报告者，则必须确保在YouTrack中创建的新用户成为自动加入组的成员，并具有通过Zendesk集成更新项目中问题的权限。

    设定值

    显示在Zendesk实例中为协议设置的同步设置。如果协议被接受，则显示的参数（注释，标签和自定义字段）将在YouTrack问题和相应的Zendesk票证之间同步。

    记者

    定义将哪个用户设置为从Zendesk票证创建的每个问题的报告者。选择以下选项之一：

    • 将Zendesk用户设置为记者 ：YouTrack尝试找到一个电子邮件帐户与Zendesk用户匹配的用户帐户。如果找不到现有帐户，则会创建一个新的YouTrack用户帐户。

    • 将预定义的用户设置为报告者 ：YouTrack将所选用户设置为由集成创建的所有问题的报告者。提交Zendesk票证的用户未添加到YouTrack。从下拉列表中选择一个现有的用户帐户。

    通过将Zendesk用户设置为报告者选项，将根据您的订阅计划为新用户帐户分配一个YouTrack许可证。如果您已达到每用户许可限制，则新用户提交的Zendesk票证不会通过集成转换为YouTrack问题。

    要继续使用Zendesk集成，您将需要升级您的订阅计划或禁止不活动的用户。

  4. 完成后，单击“ 接受协议”按钮。

    • 共享协议被接受。

    • Zendesk集成已启用并可以使用。

使用Zendesk集成

要与YouTrack 共享票证，请在Zendesk中创建或编辑票证时，在“ 共享”字段中选择共享协议的名称。

提交或更新故障单后，就会在YouTrack中创建问题。

新票在YouTrack中的处理方式如下：

  • 在集成设置中设置的自定义字段中输入Zendesk票证的当前状态。

  • 该问题的报告程序是根据集成设置中的“ 报告程序”选项设置的。

在单期视图中可以看到Zendesk中相应票证的链接。
