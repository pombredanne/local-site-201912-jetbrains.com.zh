备份文件

此资源使您可以访问数据库备份文件。执行该操作的用户帐户必须具有“ 低级读取管理”权限。

资源资源
/ api / admin / databaseBackup / backups

退货实体

BackupFile 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读BackupFiles列表 。

BackupFile属性

表示备份文件。

下表描述了BackupFile实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

请求语法

GET / api / admin / databaseBackup / backups？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

应在响应中返回的BackupFile属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。
