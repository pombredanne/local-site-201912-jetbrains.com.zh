获取项目中的问题

GET / issue / byproject / {project}

按项目获取所有问题

GET / rest / issue / byproject / {project}？{过滤器}＆{after}＆{max}＆{updatedAfter}＆{wikifyDescription}

请求参数：

名称

类型

描述

项目

projectByIdForIssues

从中获取问题的项目的ProjectID。

过滤

串

将过滤器应用于项目中的问题。

后

整数

获取问题列表之前要跳过的许多问题。也就是说，例如，当您在请求中指定after = 12时，那么在响应中您将获得所有与请求匹配的问题，但未找到前十二个问题。

最高

整数

结果中要包含的最大问题数。如果为空，则返回10期。

此请求返回的最大问题数还受到YouTrack服务的“ 要导出的最大问题数”设置的值的影响。此设置位于“ 全局设置”页面上。例如，如果您使用max=10000 REST API请求中的参数，但“ 要导出的最大问题数”设置限制为5000个问题，响应最多返回5000个问题。

更新后

长

按最新更新日期过滤问题。仅返回在指定日期之后已更新的问题。

wikify描述

布尔值

可选的。如果“ true '，则响应中的问题描述将被格式化（即：“ wikified”）。例如，带有此参数的请求网址应如下所示： /rest/issue/jt-23?wikifyDescription=true ”。默认情况下，此参数为“ false ”并跳过了网址。

响应参数：
有关响应中标签含义的信息，请参见“ 获取问题”方法的说明。
