寻找类似问题

当您报告或处理问题时，通常会发现YouTrack中已经存在其他描述相同问题或重复请求的问题。YouTrack提供了一系列功能，可帮助您避免报告重复的问题并链接报告的彼此重复的问题。

避免报告重复问题

减少重复问题的最佳方法是首先避免创建它们。报告问题时，请使用类似问题的列表来

要在报告问题时查找可能的重复项：

  1. 单击标题中的“ 新发行”按钮。

  2. 如果尚不可见，请展开页面的“ 类似问题”部分。

  3. 输入问题摘要。键入时，YouTrack搜索其他类似文本的问题。

  4. 继续问题描述。当您输入问题描述时，类似问题的列表也会更新。

  5. 如果您看到的问题看起来与您当前正在报告的问题潜在重复，请单击链接以在当前选项卡中打开该问题。

    • 如果它重复了您的问题，则可以向现有问题添加其他信息，而不是报告新问题。使用返回按钮返回到您的问题草稿，并通过单击删除按钮将其丢弃。

    • 如果您确定问题不重复您的问题，请使用浏览器中的“后退”按钮返回到草稿并报告问题。

    您也可以添加star标记并为类似问题投票，而无需离开页面。

查找可能的重复项

管理项目中的问题时面临的另一个挑战是查找并链接重复的问题。这可以帮助您避免不同用户同时开始解决同一问题的情况。

YouTrack提供了两个视图，可用于查找和链接重复的问题。每当遇到给您带来麻烦的问题时，请使用这些工具。

要查看可能重复的列表：

  1. 在单个问题视图中打开一个问题。

  2. 将问题ID复制到剪贴板。使用以下键盘快捷键之一：

    • Ctrl + C （Windows / Linux）

    • ⌘ + C （苹果系统）

  3. 在工具栏中，点击显示更多，然后选择查找类似问题 。您还可以使用键盘快捷键：

    • Ctrl + Shift + S （Windows / Linux）

    • Control + Shift + S （苹果系统）

    此操作将在新的浏览器选项卡中打开类似问题的列表。

  4. 在列表中找到并选择重复的问题。

  5. 在工具栏中，点击添加链接 ，然后选择被重复 。

    • 将打开“ 应用命令”对话框，其中包含用于应用所选链接类型的命令。

  6. 从剪贴板粘贴问题ID，然后单击“ 应用”按钮。

    • 原始问题作为重复链接到所选问题。

    • 如果将用于管理重复问题的默认工作流附加到项目，则原始状态字段的值将更改为Duplicate 。该问题被标记为已解决。有关详细信息，请参见重复 。

您还可以选择打开搜索叠加并查找可能的重复项而无需离开单个问题视图。

查找适当的副本：

  1. 在单个问题视图中打开一个问题。

  2. 将问题ID复制到剪贴板。使用以下键盘快捷键之一：

    • Ctrl + C （Windows / Linux）

    • ⌘ + C （苹果系统）

  3. 打开搜索图。使用以下键盘快捷键之一：

    • Ctrl + K （Windows / Linux）

    • ⌘ + K （苹果系统）

  4. 在搜索框中，输入looks like: 。为了加快此过程，请从剪贴板粘贴问题ID。

  5. 如果发现可能的重复项，请单击问题ID或摘要以打开问题。

  6. 在列表中找到并选择重复的问题。

  7. 如果您确定原来的问题是重复的，请单击工具栏中的“ 添加”链接 ，然后选择“ 重复的是” 。

    • 将打开“ 应用命令”对话框，其中包含用于应用所选链接类型的命令。

  8. 从剪贴板粘贴问题ID，然后单击“ 应用”按钮。

    • 原始问题作为重复链接到所选问题。

    • 如果将用于管理重复问题的默认工作流附加到项目，则原始状态字段的值将更改为Duplicate 。该问题被标记为已解决。有关详细信息，请参见重复 。
