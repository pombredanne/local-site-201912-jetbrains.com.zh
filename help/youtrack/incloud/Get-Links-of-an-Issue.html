获取问题的链接

GET / issue / {issue} / link

获取问题的所有链接的列表。

GET / rest / issue / {issue} / link

请求参数：

名称

类型

描述

问题

issueById

我们要获取其链接列表的问题的ID。

响应参数：

名称

描述

typeName

链接类型的名称（请参阅“ 发布链接类型”页面）

typeInward

此链接类型的内部链接的名称

typeOutward

此链接类型的向外链接的名称

目标

目标问题的问题ID

资源

来源问题的问题ID

对于每个问题链接之一source和target问题ID将是issue ，在请求中传递。
