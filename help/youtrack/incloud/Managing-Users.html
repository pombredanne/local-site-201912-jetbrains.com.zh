Users

In the Users section, you can manage accounts for other users and grant them permission to access and work in connected services. To access the Users page in Hub, select Users from the Access Management section of the Administration menu.

The following operations are available:

  • Create User Accounts
  • Merge User Accounts
  • Ban User Accounts
  • Manage the Guest User Account
  • Update YouTrack Profiles
  • Update Hub Accounts
  • Request Password Changes
  • Disable Two-factor Authentication
  • Add Credentials to a Hub Account
  • Manage Group Memberships
  • Manage User Access
  • Manage Authentication Tokens
  • Delete User Accounts
  • Download User Data
  • Anonymize User Data
