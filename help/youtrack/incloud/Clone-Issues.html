克隆问题

通过克隆问题 ，您可以在几秒钟内创建一个问题（或一次多个问题）的完美副本。创建副本时，只需进行必要的更改。例如，您可以更改描述并选择其他优先级。

克隆问题时，YouTrack使用新的问题ID在同一项目中创建所选问题的完美副本。自动设置以下属性：

  • 问题报告者设置为当前用户。

  • 复制除Star标记以外的所有标记。

  • 链接和附件被复制到新的发行。

  • 投票，观察者和评论未复制在新版本中。

要克隆问题：

需要权限：读取问题，创建问题

  1. 在“ 问题”列表中，选择要克隆的一个或多个问题。

  2. 按Ctrl + Alt + J或开始键入命令以打开“ 应用命令”对话框。

  3. 输入命令clone 。

    • 而已。所选的问题将被克隆到同一项目中。

您也可以单击问题工具栏中的“ 显示更多”按钮，然后选择“ 克隆问题”选项。
