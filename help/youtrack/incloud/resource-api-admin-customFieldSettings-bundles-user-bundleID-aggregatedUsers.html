捆绑中的汇总用户

通过此资源，您可以读取特定捆绑软件包含的所有用户的列表，这两个用户都是作为组成员添加的，并且是单独添加的。

资源资源
/ api / admin / customFieldSettings / bundles / user / {bundleID} / aggregatedUsers

退货实体

用户 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 捆绑中的汇总用户 。

用户属性

代表YouTrack中的用户。

下表描述了User实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

阅读用户列表

获取用户捆绑包包含的所有用户的列表。

请求语法

GET / api / admin / customFieldSettings / bundles / user / {bundleID} / aggregatedUsers？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

应在响应中返回的用户属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。
