Time Management Reports

Time management reports help you track the amount of time that is spent working on issues in one or more projects.

The following reports are available.

Report

Description

Time Report

See the total time spent working on one or more projects. Use this report to measure how much time each user spent and the type of work performed.

Gantt Chart

View issues as a linked series of dependent tasks. Use this report to analyze the amount of time that is required to complete all tasks and finish one or more projects. The Gantt chart helps also helps you identify the critical path and track overall progress.

Estimation Report

Track estimations in one or more projects. Group issues by project, assignee, subsystem, and even by sprints on an agile board.

Timesheet Report

This report shows you how users allocated spent time for a collection of issues in a calendar-based spreadsheet. You can show totals per user or per issue on a daily, weekly, or monthly basis.
