自定义脚本

自定义脚本使您可以组织和重用代码块。它们不包含任何特定类型的规则。而是，它们包含可以在其他脚本中使用的代码。

在任何脚本中，都可以通过将它们分配为对象的属性来定义要在其他脚本中使用的任何对象和函数。 exports宾语。

这是一个例子。

首先，我们将创建一个定义简单功能的自定义脚本。

//'math.js'exports.f = function（x）{return x * x-6 * x + 13; }; //这是我们需要的几个常量。exports.lower = 0; exports.upper = 9;
