发送反馈

YouTrack InCloud 2019.2帮助

字段语法

在YouTrack REST API中，默认情况下，当您向资源发送HTTP请求时，服务器仅发送回资源实体的ID。要在服务器的响应中接收属性，必须在fields请求的参数。

让我们看看它如何处理当前用户帐户的示例请求。

样品

不带字段的请求参数

在这里，我们使用默认请求，而没有提供fields参数。

请求

curl -X GET \'https://example.myjetbrains.com/youtrack/api/admin/users/me'\ -H'接受：application / json'\ -H'授权：Bearer perm：am9obi5kb2U =。UG9zdG1hbiBKb2huIERvZQ ==。jJe0eYhhkV271j1lCpfknNYOEakNk7'\ -H'Cache-Control：no-cache'\ -H'Content-Type：application / json'

反应体

作为响应，服务器发送实体ID：

{“ id”：“ 1-2”}

带字段的请求参数

在这里我们使用fields请求参数，用于明确指定应在响应正文中返回实体的哪些属性： id ， login ， name和email当前用户的

请求

curl -X GET \'https://example.myjetbrains.com/youtrack/api/admin/users/me?fields=id，登录，名称，电子邮件'\ -H'接受：application / json'\ -H'授权：不记名perm：am9obi5kb2U =。UG9zdG1hbiBKb2huIERvZQ ==。jJe0eYhhkV271j1lCpfknNYOEakNk7'\ -H'Cache-Control：no-cache'\ -H'Content-Type：application / json'

反应体

作为响应，服务器发送请求的属性：

{“ email”：“ john.doe@example.com”，“ login”：“ john.doe”，“ name”：“ John Doe”，“ id”：“ 1-2”}
