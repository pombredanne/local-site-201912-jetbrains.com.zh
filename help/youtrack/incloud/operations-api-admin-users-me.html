我的用户资料

此资源使您可以在YouTrack中阅读帐户设置和自己的个人资料。

资源资源
/ api / admin / users / me

退货实体

我有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 我的用户个人资料 。

我的属性

代表YouTrack中当前登录的用户。

扩展用户

下表描述了Me实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

类型
