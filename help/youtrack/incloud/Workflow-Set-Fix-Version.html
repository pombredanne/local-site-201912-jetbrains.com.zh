设置修订版本

此工作流程迫使开发人员在解决问题时设置修复版本。

名称

@ jetbrains / youtrack-workflow-set-fix-versions

自动连接

没有

模组

需要修订版本才能解决问题 （变更规则）

用例

该工作流程最初是从提交的请求（ JT-2468 ）中获取的。

提交此问题的用户希望强迫开发人员在解决问题时设置修订版本。

模组

更新问题后，此更改规则将检查状态是否更改为固定 。如果为true，则规则要求在“ 固定版本”字段中设置一个值。

  • 如果在“ 固定版本”字段中设置了值，则该问题将解决为“ 固定” 。

  • 如果固定版本字段为空，则显示警告。问题尚未解决。必填字段突出显示。

需要修复版本才能解决问题
