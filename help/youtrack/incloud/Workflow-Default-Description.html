默认说明

此工作流将默认文本插入到新问题的描述字段中。

名称

@ jetbrains / youtrack-workflow-default-description

自动连接

没有

模组

插入外部用户的默认描述模板 （更改规则）

用例

该工作流为用户提供了用于报告问题的默认描述模板。

模组

当用户开始报告问题时，此模块中的变更规则将检查描述字段。如果描述为空，则插入默认文本。用户开始编写摘要或设置问题字段的值后，就会插入默认文本。如果用户立即在“ 描述”字段中输入文本，则不会插入默认文本。
