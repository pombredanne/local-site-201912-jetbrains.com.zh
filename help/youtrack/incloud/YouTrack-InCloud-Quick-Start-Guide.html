YouTrack InCloud 2019.2帮助

新实例

设置新的YouTrack InCloud实例的过程很简单。无论您是开始免费使用YouTrack还是直接进入商业订阅，您都需要注册。

注册免费计划

要开始使用免费支持最多三个用户的YouTrack，请按照以下步骤操作：

  1. 填写注册表 。所有字段都是必填字段。

    领域

    描述

    实例名称

    		输入您要用于InCloud实例的站点名称。这用作myjetbrains.com域上实例的本地主机名。例如，如果您输入time-travel ，主机名设置为time-travel.myjetbrains.com 。

    如果以后要更改主机名，请与YouTrack支持团队联系或创建一个新实例。

    电子邮件地址

    输入一个有效的电子邮件地址。否则，您将无法确认托管实例的创建。

  2. 选择选项以确认您已阅读并接受服务条款和隐私政策。

  3. 单击免费入门按钮。

    您提交注册数据后，我们会立即向您在注册表格中指定的地址发送一封确认电子邮件。

如果您注册了免费计划，并想了解YouTrack如何扩展到更大的组织，则可以选择开始14天的试用期。免费试用版最多支持100个用户，并为您提供10 GB的磁盘空间。

您可以直接在YouTrack InCloud实例的“ 全局设置”页面上切换到免费试用版。有关更多信息，请参阅切换到试用订阅 。

注册商业订阅

如果您已经决定要使用YouTrack，则可以购买具有商业许可的订阅。

要注册新的商业许可证，请按照下列步骤操作：

  1. 访问jetbrains.com上的“ 购买YouTrack”页面。

    • 如果尚未注册JetBrains帐户，请使用您的电子邮件地址注册一个帐户。

    • 如果您已经拥有JetBrains帐户，请登录。

  2. 单击立即购买按钮。

    • 将打开“ eStore标识”页面。

  3. 确认选择了正确的电子邮件地址和客户进行购买，然后单击“ 下一步” 。

  4. 选择要购买的商业订阅，然后单击与您的首选计费周期相对应的“ 订阅”按钮。

    • 将打开“ eStore订单结帐”页面。

  5. 填写结帐表单上的字段。所有字段都是必填字段。

    • 在您的站点名称字段中，输入要用于InCloud实例的站点名称。这用作myjetbrains.com域上实例的本地主机名。例如，如果您输入time-travel ，主机名设置为time-travel.myjetbrains.com 。

    • 如果要稍后更改主机名，请与YouTrack支持团队联系。

  6. 准备就绪后，单击“下订单”按钮。

    您下订单后，我们将立即处理交易并发送确认电子邮件，该电子邮件已在您的JetBrains帐户中注册。

确认实例创建

注册实例后，需要确认其创建。在这里，您还设置了用户帐户的密码，这是实例的默认管理员帐户。

要确认您的实例创建，请按照下列步骤操作：

  1. 单击电子邮件中的链接，以确认您的YouTrack InCloud实例的创建。

    • YouTrack托管站点的起始页在浏览器中打开。

    • 创建具有请求的URL的新的YouTrack实例。

  2. 设置管理员帐户的密码，然后选择数据中心的位置。选择离您最近的数据中心。

    当您单击保存按钮时，YouTrack实例启动。

  3. 访问您的YouTrack实例。

    • 实例准备好后，单击“将我带到我的YouTrack”按钮。

    • 在登录页面上，输入您的电子邮件地址和密码，然后单击“ 登录”按钮。

    此时，您可能是唯一知道此YouTrack实例存在的人，因此请不要忘记如何登录。

登录时，实例将打开“ 项目”页面，您可以在其中创建第一个项目。有关设置任务的完整列表，请阅读《系统管理员快速入门指南》 。

谁可以访问我的实例？

您的新实例包含两个用户帐户： root和guest 。

  • 您的凭据已分配给root用户帐户。这是您的YouTrack实例的默认管理员帐户。

  • 在全局项目中，此帐户被授予默认的系统管理员角色。

  • 这是新的YouTrack实例中唯一具有管理访问权限的用户帐户。您应该做的第一件事就是为需要管理访问权限的其他用户创建帐户，并为他们分配相似的角色。有关更多信息，请参见访问管理 。

  • 来宾用户帐户也会自动创建，但默认情况下被禁止。此帐户未分配任何角色。

    • 如果要允许匿名访问YouTrack实例，则可以在一个或多个项目中向来宾用户授予角色。

    • 要允许匿名访问所有项目中的问题，可以在全局项目中向来宾用户授予角色。

    有关更多信息，请参阅管理访客用户帐户 。

只要来宾帐户仍然被禁止，您就是唯一有权访问此实例的人。要授予其他人访问权限，您可以为其他用户创建帐户或启用注册，以便用户可以创建自己的帐户。有关更多信息，请参见创建用户帐户 。
