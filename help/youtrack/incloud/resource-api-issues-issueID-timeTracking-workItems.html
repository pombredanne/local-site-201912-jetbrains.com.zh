发行工作项目

此资源使您可以处理特定问题中的工作项。

资源资源
/ api / issues / {issueID} / timeTracking / workItems

退货实体

IssueWorkItem 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读IssueWorkItems列表 。

  • POST ： 添加一个新的IssueWorkItem 。

IssueWorkItem属性

代表问题中的工作项目。

下表描述了IssueWorkItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

作者

 用户

在工作项中为其分配工作的用户。 Can be null 。

创作者

 用户

将工作项添加到问题的用户。 Read-only 。 Can be null 。

文本

串

工作项描述。 Can be null 。

textPreview

串

解析的描述预览。 Read-only 。

类型

 工作项类型

工作项类型。 Can be null 。

被创造

长

工作项的创建日期。 Can be null 。

更新

长

工作项目的最后更新日期。 Can be null 。

持续时间

 持续时间值

工作项目的持续时间。

日期

长

分配给工作项的日期和时间。存储为UTC中的Unix时间戳。时间部分设置为当前日期的午夜。

问题

 问题

存储工作项所附问题的属性。 Read-only 。

使用Markdown

布尔型

存储注释文本的格式。什么时候true ，则注释将以Markdown格式的文本进行处理。否则，将以YouTrack Wiki格式处理评论。

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

整数
