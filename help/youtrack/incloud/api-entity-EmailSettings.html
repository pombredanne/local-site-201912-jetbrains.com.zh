YouTrack InCloud 2019.2帮助

电子邮件设置

代表此YouTrack安装的电子邮件设置。

下表描述了EmailSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

isEnabled

布尔型

如果启用了电子邮件通知，则为true；否则为false。

主办

串

用于通知的邮件服务器。 Can be null 。

港口

整数

邮件服务器端口。

邮件协议

列举

邮件协议。支持的值为SMTP，SMTPS和SMTP_TLS

匿名

布尔型

如果为true，则未经授权访问邮件服务器。

登录

串

邮件服务器登录。 Can be null 。

sslKey

 存储条目

用于访问邮件服务器的SSL密钥。 Can be null 。

从

串

发件人地址。

回复

串

回复地址。 Can be null 。
