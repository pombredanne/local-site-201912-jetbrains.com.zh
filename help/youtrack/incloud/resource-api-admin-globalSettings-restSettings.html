Rest Cors设置

该资源使您可以使用YouTrack服务的“资源共享”设置。

资源资源
/ api / admin / globalSettings / restSettings

退货实体

RestCorsSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

RestCorsSettings属性

表示服务的资源共享（CORS）配置。

下表描述了RestCorsSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

allowedOrigins

字符串数组

跨域请求允许的来源。

allowAllOrigins

布尔型

指示是否允许来自所有来源的请求。
