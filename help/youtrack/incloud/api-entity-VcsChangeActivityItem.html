VcsChangeActivityItem

表示问题的VCSChanges列表中的更新。

下表描述了VcsChangeActivityItem实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

已移除

在后代实体中指定类型

删除的VCS更改列表。 Read-only 。

添加

在后代实体中指定类型

添加的VCS更改列表。 Read-only 。

作者

 用户

VCS更改的作者。如果在获取VCS更改时，YouTrack找不到与其作者匹配的用户，则活动作者设置为VcsUnresolvedUser 。 Read-only 。

长

活动的时间戳。 Read-only 。

在后代实体中指定类型

作为已执行操作的目标的实体。 Read-only 。

targetMember

串

目标实体的属性名称，已被修改。 Read-only 。 Can be null 。

领域

 FilterField

过滤器字段。它包含有关目标实体中已修改属性的其他信息。例如，有关链接类型或修改后的自定义字段或捆绑软件类型的信息，等等。 Read-only 。 Can be null 。

类别

 活动类别

活动的类别。 Read-only 。
