JetBrains IDE Integrations

The YouTrack Integration Plugin extends the functionality of the the Task Management plugin that is built into JetBrains IDEs to provide a tighter integration with YouTrack. This plugin basically lets you interact seamlessly with YouTrack without leaving your IDE.

JetBrains IDEs support native integrations with a host of issue trackers, including YouTrack. You can enable the following integration features without downloading and installing any additional plugins:

Feature

Description

Issue Navigation

Transform issue IDs in change lists and commit messages into direct links to issues in YouTrack.

Time Tracking

Track time spent working on tasks in your IDE and post work items to issues in YouTrack.

Task Management

Open a YouTrack issue as a task in your IDE. This sets the YouTrack issue as the current work context. Changes to the code base are added to a change list that is associated with the issue.

These integrations are available for IntelliJ IDEA, PhpStorm, PyCharm, RubyMine, WebStorm, AppCode, and CLion.
