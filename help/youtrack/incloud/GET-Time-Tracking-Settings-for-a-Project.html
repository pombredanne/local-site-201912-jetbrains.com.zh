获取项目的时间跟踪设置

GET / rest / admin / project / {projectId} / timetracking

获取特定项目的当前时间跟踪设置。

GET / rest / admin / project / {projectId} / timetracking

样品

请求：

GET https://example.com/youtrack/rest/admin/project/TST/timetracking连接：keep-alive Cookie：$ Version = 0; JSESSIONID = axezv08lvrvrzpvvih22ricx; $路径= /

响应：

根据特定项目中时间跟踪参数的当前状态，响应可能有所不同，如下所示：

未为项目启用时间跟踪：

HTTP / 1.1 200 OK内容类型：application / xml; charset = UTF-8内容长度：296

已为项目启用时间跟踪，但尚未配置：

HTTP / 1.1 200 OK内容类型：application / xml; charset = UTF-8内容长度：296

时间跟踪已配置但已禁用：

HTTP / 1.1 200 OK内容类型：application / xml; charset = UTF-8内容长度：296

时间跟踪已启用并已部分配置：

HTTP / 1.1 200 OK内容类型：application / xml; charset = UTF-8内容长度：296
