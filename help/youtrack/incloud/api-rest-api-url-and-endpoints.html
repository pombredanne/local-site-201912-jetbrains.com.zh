REST API URL and Endpoints

REST API URL in YouTrack

YouTrack's REST API is available by the following URL:

<YouTrack Service URL>/api/

<YouTrack Service URL> is the Base URL of your YouTrack InCloud service. Please note that the Base URL for an InCloud service must have the /youtrack context appended. For example, https://example.myjetbrains.com/youtrack.

/api/ is the context path for the REST API of your YouTrack service.

For the sample Base URLs above, REST API URLs are as follows, respectively:

  • For https://example.myjetbrains.com:

    https://example.myjetbrains.com/youtrack/api/

Concluding the example, the endpoint URLs to, let's say, get the profile of the current user are:

  • For www.example.com/youtrack:

    https://www.example.com/youtrack/api/admin/users/me

  • For youtrack.example.com:

    https://youtrack.example.com/api/admin/users/me
