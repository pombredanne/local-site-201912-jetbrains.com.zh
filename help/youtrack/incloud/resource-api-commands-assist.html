/ api / commands / assist

退货实体

CommandList 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • POST ： 命令建议 。

CommandList属性

代表YouTrack中的命令列表和相关注释。可用于应用命令或获取命令建议。

下表描述了CommandList实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

类型

描述

评论

串

要添加到问题的评论。 Can be null 。

能见度

 CommandVisibility

存储随命令添加的注释的可见性设置。

询问

串

存储要应用的命令。 Can be null 。

插入符

整数

当前的插入符号位置。默认情况下，当前命令的结尾。

无声

布尔型

如果为true，则应用该命令而不会通知。默认值为false。

使用Markdown

布尔型

如果为true，则将使用命令添加的注释解析为Markdown格式。否则，注释将解析为YouTrack Wiki格式。

运行

串

登录代表执行命令的用户。要使用runAs参数，您必须在项目中具有目标问题的“低级管理”权限。 Can be null 。

命令

 解析命令数组

问题

建议

 一系列建议
