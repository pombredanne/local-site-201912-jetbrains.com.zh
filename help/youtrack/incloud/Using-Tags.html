标签问题

标签使您可以根据自己的喜好对问题进行分组，而不必考虑其属性。例如，您可以创建一个标签fix it today并将标签添加到多个项目的问题中

您可以将任何问题与不限数量的标签关联。

标签为您提供了一种搜索问题的替代方法。您可以标记任意数量的问题并按标签搜索问题，而不必按受让人，州或项目进行搜索。使用标签可将任意属性和描述性属性添加到任何问题，而不必担心添加新的自定义字段。

创建标签

在开始使用标签之前，需要创建它们。您可以创建和使用个人标签来根据自己的喜好对问题进行分类，也可以创建共享标签以支持内部工作流程。

默认情况下，标签仅对您可见。其他用户无法在边栏中看到您的个人标签。

创建或编辑标签时，请使用设置与一组共享标签。您可以通过两种方式与组共享标签：

  • 使标签对群组可见。小组成员可以在边栏中看到标签，并将标签添加到问题中。

  • 使标签可被组编辑。小组成员可以更新名称，配色方案和“ 解决时取消标记”设置。只有创建标签的用户才能更新可见性设置。

    此设置不会影响标签的可见性。如果您希望组的成员能够编辑标签，请确保他们也是用于可见性设置的组的成员。

目前，实验列表提供了更新版本的问题列表。此处提供的信息适用于该页面的经典版本。要了解如何在问题列表的实验版中添加标签，请参见边栏 。

要创建标签：

需要权限：创建标签或保存的搜索，共享标签或保存的搜索

  1. 单击边栏的“ 标签”部分中的加号图标。

    • 将打开“ 创建标签”对话框。

  2. 输入以下设置的值：

    设置

    描述

    名称

    为标签分配一个名称。

    一律显示在侧栏中

    默认情况下，启用将标签固定在边栏中。

    组可见

    选择一个其标签可见的组。使用此设置可以与其他用户共享标签。

    按组更新

    选择一个其成员有权编辑标签属性的组。确保此组的成员包含在用于可见性设置的组中。

    解决时取消标记

    启用以在问题设置为已解决状态时自动删除标签。

  3. 对标签应用可选的配色方案。使用此设置可以使标记的问题在“ 问题”列表中突出显示。

  4. 单击创建按钮。

为问题添加标签

您可以一次为一个问题添加标签，也可以搜索符合特定条件的问题并立即标记所有问题。

要将标签添加到一个或多个问题：

需要权限：读取问题，更新问题

  1. 在“ 问题”列表中选择一个或多个问题。

  2. 点击工具栏中的添加标签按钮。

  3. 输入要添加到所选问题的标签的名称。当您键入时，YouTrack会过滤标签列表。

  4. 从列表中选择标签。

    • 标记已添加到所有选定的问题。

  5. 单击提交或按Ctrl + Enter 。

您还可以在“ 应用命令”对话框中向事件添加标签。当您通过 “ 添加标签”链接中的“ 使用命令添加标签”选项打开此对话框时， add tag命令是自动输入的。

从问题中删除标签

标签就像临时纹身。您可以将它们涂在夜总会看起来很凉爽，并在早上洗掉。

我们使用设计 ， 文字和文档之类的标签来表示问题需要特定部门的输入，而无需将问题明确分配给团队成员。团队提供所需的输入或反馈后，他们将删除标签。

在两种情况下，用户可以从问题中删除标签：

  • 用户是可以编辑标签的组的成员。如果他们在项目中具有“ 更新问题”权限，则该用户可以删除他们有权编辑的标签。

  • 用户是可以查看和使用标签的组的成员。在这种情况下，具有“ 更新问题”权限的用户可以看到该标签，但不能将其删除。要删除不允许编辑的标签，您需要项目中的“ 更新问题专用字段”权限。添加此功能是为了让管理员删除具有有限编辑权限的标签。

两种情况均不需要“ 编辑标签”或“已保存的搜索”权限。此权限仅确定属于允许编辑标签的组成员的用户是否可以更新其属性。

要从一个或多个问题中删除标签，请执行以下操作：

需要权限：读取问题，更新问题

要删除您无权编辑的标签，您还需要具有更新问题专用字段的权限。

  1. 在“ 问题”列表中选择一个或多个问题。

  2. 打开“ 应用命令”对话框。

  3. 输入命令remove tag 。

  4. 从完成列表中选择要删除的标签（或继续输入标签名称）。

  5. 单击“ 应用”按钮，或按Ctrl + Enter 。

查看与标签相关的问题

列出与标签相关的问题的最简单方法是单击侧栏中的标签名称。如果将初步筛选器设置为搜索上下文，则列表将显示与所选上下文匹配并与所选标签相关联的问题。

要通过标签搜索问题：

需要权限：阅读问题

  1. 输入tag:在搜索框中。

  2. 从完成列表中选择一个标签。

  3. 输入其他搜索条件以优化搜索。例如，要显示用标记的未解决的关键问题fix before end of month ，输入：

    tag: {fix before end of month} state: unresolved priority: critical

    要么

    #{fix before end of month} #unresolved #critical

  4. 单击搜索按钮或按Enter 。

编辑标签设置

您可以根据标签类型来编辑标签设置。

  • 对于您自己创建的标签，您可以随时更新任何标签设置。

  • 对于与您共享的标签，您只能更改名称，颜色方案和“ 解析时取消标记”设置。

要编辑标签设置：

需要权限：编辑标签或保存的搜索

  1. 单击边栏中标签旁边的“ 编辑”图标。

    • 将打开“ 编辑标签”对话框。

  2. 更新您要修改的设置。可见性设置仅显示给创建标签的用户。

  3. 单击保存按钮。

删除标签

只有创建标签的用户才能从YouTrack删除标签。其他用户可以选择通过从边栏中取消固定标签来隐藏标签。

如果您有一个不再使用的个人标签或已经过时的标签，则可以将其从系统中删除。

删除标签：

需要权限：编辑标签或保存的搜索，删除标签或保存的搜索

  1. 单击边栏中标签旁边的“ 编辑”图标。

  2. 在“ 编辑标签”对话框中，单击“ 删除标签”按钮。

  3. 单击确定以确认操作。

    • 标签被删除。该标签已从所有问题中删除。
