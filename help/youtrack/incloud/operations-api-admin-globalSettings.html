具有特定GlobalSettings的操作

提供具有全局应用程序设置的操作。

资源资源
/ api / admin / globalSettings

退货实体

GlobalSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的GlobalSettings 。

  • POST ： 更新特定的GlobalSettings 。

GlobalSettings属性

表示应用程序范围的设置。

下表描述了GlobalSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

系统设置

 系统设置

影响YouTrack核心功能的系统设置。 Read-only 。

通知设置

 通知设置

YouTrack服务的“通知”设置。 Read-only 。

restSettings

 RestCors设置

YouTrack服务的资源共享（CORS）设置。 Read-only 。

 外观设置

YouTrack服务的视觉设置。 Read-only 。

 区域设置

系统语言设置。 Read-only 。

 执照

获取设置。

请求参数

参数

类型
