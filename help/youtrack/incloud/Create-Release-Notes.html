创建发行说明

通过YouTrack，您可以将问题列表导出为HTML页面。您可以使用此页面发布产品的发行说明。列表中每个问题的ID都设置为指向YouTrack中原始问题的链接。

通过应用程序标题中的“ 报告”菜单，可以使用“ 问题”列表的实验版中的HTML格式导出问题选项。要了解更多信息，请参阅“报告菜单” 。

可以导出的最大问题数由“ 全局设置”页面上的“ 要导出的最大问题数”设置确定。默认值为500。如果您要导出的问题数超过了全局限制，请要求管理员增加此值。有关更多信息，请参见System 。

要将问题列表导出到HTML：

需要权限：阅读问题

  1. 单击标题中的“ 问题”链接。

  2. 搜索要在HTML页面上显示的问题。例如，要创建发行说明，您可以使用如下搜索查询：

    于：YouTrack＃功能＃任务＃已验证的修订版本：2017.2
    YouTrack会在YouTrack项目中找到经过验证的状态和功能，这些任务已针对2017.2。版本进行了修复和验证。

  3. 在“ 问题”列表中设置详细程度。此设置确定HTML报告中的默认详细程度。如果将详细信息级别设置为单行视图或紧凑视图 ，则问题ID和问题摘要将显示在HTML报告中。

  4. 从“ 报告”下拉列表中，选择“ HTML中的问题” 。


    • 将生成带有问题列表的HTML页面。

    • 列表中的问题按问题类型分组。

  5. 在“ 查看选项”菜单上，设置HTML报告上的详细程度。提供以下选项：

    选项

    描述

    显示说明

    在HTML报告中显示每个问题的描述。

    显示评论

    显示已添加到每个问题的评论。还显示评论作者和将评论添加到问题的日期。

    您可以通过单击显示问题描述链接来打开或关闭描述 。

  6. 单击链接以查看HTML页面的标题。

而已！您的HTML报告已准备就绪，可以使用。该页面已保存在您的YouTrack服务器上。您可以将此页面的URL添加到您的网站或电子邮件中。
