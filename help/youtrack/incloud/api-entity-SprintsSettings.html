冲刺设置

描述冲刺配置。

下表描述了SprintsSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

isExplicit

布尔型

如果为true，则应将问题手动添加到板上。如果为假，则根据字段的查询和/或值在面板上显示问题。

cardOnSeveralSprints

布尔型

如果为true，则该板的若干个sprint中都可以显示卡。

defaultSprint

 短跑

默认情况下，新卡片会添加到此sprint中。该设置仅在以下情况下适用isExplicit == true 。 Can be null 。

disableSprints

布尔型

如果为true，则敏捷板在UI中没有明显的冲刺。但是，在API中，它看起来像只有一个活动（未存档）冲刺。

显式查询

串

与此查询匹配的问题将显示在面板上。该设置仅在以下情况下适用isExplicit == false 。 Can be null 。

sprintSyncField

 CustomField

根据此字段的值，问题将分配给sprint。该设置仅在以下情况下适用isExplicit == false 。 Can be null 。

hideSubtasksOfCards

布尔型

如果为true，则如果板上的卡的子任务与板查询匹配，则它们将被隐藏。该设置仅在以下情况下适用isExplicit == false 。
