栏位类型

此资源使您可以阅读YouTrack中支持的自定义字段类型的列表。

资源资源
/ api / admin / customFieldSettings / types

退货实体

FieldType 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读FieldTypes列表 。

FieldType属性

代表YouTrack中自定义字段的类型。

下表描述了FieldType实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

类型

描述

串

自定义字段类型的ID。 Read-only 。

阅读FieldTypes列表

获取所有可用的字段类型。

请求语法

GET / api / admin / customFieldSettings / types？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

响应中应返回的FieldType属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。
