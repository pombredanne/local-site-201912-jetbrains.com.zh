YouTrack InCloud 2019.2帮助

YouTrack服务器IP范围

如果您的IT服务提供商默认情况下以不允许IMAP或POP3流量的方式设置防火墙，则您将无法从YouTrack InCloud实例接收通知。为了解决该问题，IT提供商应该知道将要到达与IMAP / POP3的连接的IP地址范围，并创建允许来自这些来源的流量的防火墙规则。

Amazon Web Services（AWS）以JSON格式发布IP地址范围的列表。要了解如何使用此信息并订阅更新，请参阅AWS文档 。

YouTrack InCloud使用以下区域中的服务器：

地区名称

区域

美国西部（加利福尼亚北部）

us-west-1

欧盟（爱尔兰）

eu-west-1

亚太地区（新加坡）

ap-southeast-1

实例的当前数据中心位置显示在“ 全局设置”页面的“ 许可证”部分中。有关更多信息，请参阅许可 。
