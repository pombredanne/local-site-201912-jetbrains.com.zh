YouTrack Server IP Ranges

If your IT service provider has set up your firewall in a way that does not allow IMAP or POP3 traffic by default, you will not be able to receive notifications from your YouTrack InCloud instance. To solve the problem, the IT provider should know a range of IP addresses from which connections to IMAP/POP3 will be coming and create firewall rules that allow traffic from these sources.

Amazon Web Services (AWS) publishes a list of IP address ranges in JSON format. To learn how to use this information and subscribe to updates, refer to the AWS documentation.
