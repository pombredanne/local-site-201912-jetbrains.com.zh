创建工作流程

当您要增强问题跟踪并自动执行项目更改时，请创建工作流程。

作为管理员，您可以在“ 工作流程”页面上的YouTrack中访问工作流程的完整列表。只有在这里您才能将工作流程添加到YouTrack。

要创建工作流程：

需要权限：更新项目

  1. 在管理菜单的核心功能部分中，选择工作流程 。

  2. 单击创建工作流程按钮。

    • 将打开“ 新建工作流”对话框。

  3. 输入名称和标题 。

  4. 单击保存按钮。

    • 工作流将添加到“ 工作流”页面上的列表中。

现在您有了一个新的工作流程，但是工作流程为空。空的工作流程不能附加到项目。

要完成这项工作，您需要为工作流程定义一个或多个规则。

定义工作流程规则：

需要权限：更新项目

  1. 在“ 工作流”列表中找到该工作流 ，然后单击其名称。

    • 工作流程将在工作流程编辑器中打开。

    • “ 新模块”面板在编辑器窗口中打开。

  2. 单击与您要编写的规则类型相对应的模块按钮。例如， Action 。

    • 将打开“ 新模块”对话框。

  3. 输入模块的名称 。

  4. 单击保存按钮。

    • 该模块已添加到当前工作流程中。

    • 所选规则类型的模板将加载到工作流编辑器中。

  5. 在当前模块中定义您的规则。有关特定于规则的准则，请参阅工作流规则类型 。

  6. 完成后，单击“ 保存”按钮。

  7. 继续将模块添加到工作流程中，直到定义了要包含在此工作流程中的所有规则为止。使用添加模块菜单来选择要创建的规则类型。

一旦定义了至少一个规则，就可以使用新的工作流程并将其附加到一个或多个项目中。有关说明，请参阅管理多个项目的工作流 。
