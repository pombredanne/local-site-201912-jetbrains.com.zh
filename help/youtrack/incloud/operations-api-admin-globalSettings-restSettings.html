具有特定RestCorsSettings的操作

该资源使您可以使用YouTrack服务的“资源共享”设置。

资源资源
/ api / admin / globalSettings / restSettings

退货实体

RestCorsSettings 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读特定的RestCorsSettings 。

  • POST ： 更新特定的RestCorsSettings 。

RestCorsSettings属性

表示服务的资源共享（CORS）配置。

下表描述了RestCorsSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

allowedOrigins

字符串数组

跨域请求允许的来源。

allowAllOrigins

布尔型

指示是否允许来自所有来源的请求。

阅读特定的RestCorsSettings

阅读您的YouTrack服务的资源共享设置。

请求语法

GET / api / admin / globalSettings / restSettings？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的RestCorsSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/restSettings?fields=allowAllOrigins,allowedOrigins

样品响应体

{“ allowAllOrigins”：否，“ allowedOrigins”：[]，“ $ type”：“ RestCorsSettings”}

更新特定的RestCorsSettings

更改您的YouTrack服务的资源共享设置。

所需权限

需要低级管理员写权限。

请求语法

POST / api / admin / globalSettings / restSettings？{字段}

请求参数

参数

类型

描述

领域

串

应在响应中返回的RestCorsSettings属性列表。如果未指定任何字段，则仅entityID返回

样品

样本请求URI

https://example.myjetbrains.com/youtrack/api/admin/globalSettings/restSettings?fields=allowAllOrigins,allowedOrigins

样品申请机构

{“ allowedOrigins”：[“ https：//domain.example：8080”，“ https：//internal.domain.example：8342”]}

样品响应体

{“ allowAllOrigins”：否，“ allowedOrigins”：[“ https：//domain.example：8080”，“ https：//internal.domain.example：8342”]，“ $ type”：“ RestCorsSettings”}
