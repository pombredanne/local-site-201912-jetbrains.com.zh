YouTrack InCloud 2019.2帮助

全局设置

此页面描述了YouTrack的全局设置。这些设置在“ 管理”菜单的“ 服务器设置”部分中可用。

需要权限：低级管理员写

执照

“ 全局设置”页面的第一部分提供了有关您当前许可证及其限制的详细信息。

页面此部分中的图形可让您跟踪活动用户数和总磁盘空间是否超过许可证的限制。过度使用的潜在费用显示在图表上方，当前活动用户数和已用磁盘空间旁边。

如果您的YouTrack许可证已过期，请单击“ 升级您的订阅”按钮以访问YouTrack网站并购买另一个许可证。您还可以切换到免费计划，为其他用户购买许可证，或者在月度或年度帐单之间切换。有关更多信息，请参见切换订阅类型 。

系统

系统部分提供的选项可让您配置YouTrack服务器的基本系统设置。

设置

描述

最大上传文件大小

上载附件允许的最大文件大小，以字节为单位。默认情况下，允许的最大文件大小为10Mb（10485760字节）。

导出的最大问题数

可以以CSV，Excel或HTML格式打印或导出的最大问题数。默认情况下，该限制设置为500个问题。要更改它，只需输入一个新值。

设置“ 导出的最大问题数”限制是为了避免应用程序过载。

管理员电子邮件

YouTrack支持团队的主要联系人的电子邮件地址。支持团队使用此信息来确定用户是否有权请求对YouTrack实例进行重大更改。这包括恢复数据库或在实例之间迁移数据的请求。

默认值设置为注册YouTrack InCloud实例的用户的电子邮件地址。当负责管理YouTrack实例的人员与注册该实例或购买许可计划的用户不同时，请更新此设置的值。

数据中心位置

托管YouTrack InCloud实例的数据中心的位置。注册实例时设置此值。如果要将实例移动到其他数据中心，请向YouTrack InCloud支持提交请求。

全系统横幅

启用后，将在每个页面的顶部为YouTrack中的每个用户显示一个系统范围的横幅。要了解有关此功能的更多信息，请参阅显示系统范围的标语 。

视觉效果

“ 视觉”部分提供的选项可让您管理与YouTrack中的视觉演示相关的系统设置。

仅使用商业许可证的YouTrack安装支持“ 应用程序徽标”设置。

设置

描述

应用徽标

为YouTrack设置自定义徽标。自定义徽标显示在系统中所有页面的标题中。要使用自定义徽标，请单击“ 编辑”图标，然后从本地目录中选择一个图像文件。您可以上传JPG，GIF或PNG文件。您的图像会自动调整为250 x 80像素。

要将自定义徽标替换为YouTrack应用程序徽标，请单击设置为默认按钮。

时区

这组控件确定系统范围的默认时区。此处设置的时区用于所有用户的时间显示。用户可以在用户个人资料中更改本地时区。

日期格式

为日期和时间设置系统范围的默认表示。此格式适用于所有使用日期字段类型的自定义字段。

系统语言

您可以将YouTrack服务器的系统语言设置为开箱即用提供的本地化版本之一。YouTrack为英语，法语，德语，日语，俄语和西班牙语提供本地化支持。

还可以使用YouTrack用户社区支持的其他语言。这些语言在列表中带有社区标签。有关更多信息，请参见基于社区的语言支持 。

用户可以在用户个人资料中选择其个人语言偏好。为了确保整个应用程序的一致性，在全局级别设置的语言用于搜索查询，命令和自定义字段。页面标题，菜单和控件显示在首选语言环境中。有关更多信息，请参见YouTrack常规配置文件设置 。

要使文本输入字段和所有下拉列表从右到左对齐，请启用RTL支持选项。有关更多信息，请参见从右到左语言支持 。

键盘快捷键

本节提供了一些选项，可用于设置键盘快捷方式方案。您可以使用默认的快捷方式方案，也可以添加和使用自己的自定义方案。有关更多信息，请参见自定义键盘快捷键 。

资源共享

除了始终启用的常规REST API外，YouTrack还支持跨域资源共享或CORS。CORS支持使您能够对您的YouTrack实例启用JavaScript访问。有关更多信息，请参阅Wikipedia上的跨域资源共享 。您还可以在此处下载CORS使用示例： https : //github.com/vadimgurov/youtrack-cors-samples 。

要启用从站点（来源）的访问，请在页面的“ 资源共享”部分中选择并配置以下两个参数之一：

设置

描述

允许所有来源

启用以允许使用REST API从所有来源进行访问。

允许的来源

输入以行分隔的起源列表，允许跨域请求。通过域名指定每个来源，包括适用的协议和端口号。

通知事项

本部分提供的选项使您可以配置如何将YouTrack中问题的更改通知用户。

电子邮件相关设置

在YouTrack InCloud中，默认情况下启用和配置电子邮件通知。设置页面的此部分使您可以查看用于向用户发送电子邮件通知的配置。

要禁用从YouTrack服务器发送电子邮件通知，请取消选中“ 启用”复选框。

设置

描述

电子邮件

此控件在YouTrack中启用或禁用电子邮件通知。

组态

确定您的YouTrack InCloud实例是否连接到由JetBrains托管的电子邮件服务或外部电子邮件服务器。提供以下选项：

  • 默认值 -您的YouTrack实例已连接到JetBrains托管的电子邮件服务。隐藏了用于连接到外部邮件服务器的其他设置。

  • 自定义 -您的YouTrack实例连接到外部邮件服务器。要了解如何连接到您自己的邮件服务器，请参阅连接到外部邮件服务器 。

SMTP主机

邮件服务器的服务器地址。

SMTP端口

邮件服务器的连接端口。

邮件协议

邮件服务器使用的协议。

匿名连接

启用后，YouTrack无需发送登录名和密码即可连接到SMTP服务器。当您的SMTP服务器不需要身份验证时使用。

启用此选项后，将禁用SMTP登录名和SMTP密码的输入字段。这些设置中存储的值将被忽略。

SMTP登录

用于访问邮件服务器的帐户的用户名。

SMTP密码

用于访问邮件服务器的帐户的密码。

SSL密钥

此列表使您可以选择一个SSL密钥，其中包含受信任的证书及其公共密钥和私有密钥对。当它尝试连接到邮件服务器时，该密钥将YouTrack标识为客户端。

要启用到外部SMTP服务器的安全连接，您需要将其SSL证书导入YouTrack。有关说明，请参阅SSL证书 。

发件人地址

从YouTrack发送电子邮件通知的默认地址。这将设置"From:"电子邮件标题中的字段。您可以使用以下格式sender name 或输入不包含发件人姓名的电子邮件地址。有关更多信息，请参见发件人和答复地址 。

回覆地址

电子邮件回复发送到的可选地址。这将设置"Reply-To:"由YouTrack发送的邮件的电子邮件标题中的字段。例如，当您用于发送电子邮件通知的地址无法收到回复时，请使用此设置。

发送测试电子邮件

此控件使您可以通过将测试消息发送到特定的电子邮件地址来测试与邮件服务器的连接。

与Jabber相关的设置

在YouTrack InCloud中，默认情况下启用并配置通过Jabber进行的通知。设置页面的这一部分使您可以查看用于通过Jabber发送消息的配置以及从中向用户发送通知的系统帐户。

您还可以选择完全覆盖这些设置，并将YouTrack连接到您自己的Jabber服务。有关详细说明，请参阅通过Jabber启用通知 。

在这里，您还可以找到允许用户通过Jabber客户端-Jabber Bot中的直接消息与YouTrack进行交互的选项。有关更多信息，请参见在Jabber上搜索问题和在Jabber上 应用命令 。

要禁用通过Jabber发送通知，请取消选中“ 启用”复选框。

贾伯

此控件在YouTrack中启用或禁用Jabber通知。

组态

确定您的YouTrack InCloud实例是否连接到由JetBrains托管的Jabber服务或外部Jabber服务。提供以下选项：

  • 默认值 -您的YouTrack实例已连接到JetBrains托管的Jabber服务。隐藏了用于连接外部Jabber服务的其他设置。

  • 自定义 -您的YouTrack实例连接到外部Jabber服务。要了解如何连接到自己的Jabber服务器，请参阅“ 通过Jabber启用通知” 。

Jabber服务器

您的Jabber服务的服务器地址。

Jabber服务器端口

Jabber服务的连接端口。

Jabber服务名称

您的Jabber服务的名称。服务名称和地址（主机名）不同时使用。通常，此设置是必需的ejabbered服务器。

Jabber用户名

用于访问Jabber服务的帐户的用户名。使用格式username@domain 。

Jabber密码

用于访问Jabber服务的帐户的密码。

SSL密钥库

此列表使您可以选择一个SSL密钥库，其中包含受信任的证书及其公钥和私钥对。密钥库在尝试连接到Jabber服务时将YouTrack标识为客户端。

要启用与Jabber服务的安全连接，您需要将其SSL证书导入YouTrack。有关说明，请参阅SSL证书 。

使用旧版SSL

此选项禁用SSL连接的证书有效性检查。仅在使用时推荐jabber.org发送通知并遇到与Jabber服务的连接问题。

发送测试戳消息

通过此控件，您可以通过向特定的Jabber帐户发送测试消息来测试与Jabber服务的连接。

启用Jabber机器人

此选项允许用户通过Jabber客户端中的直接消息与YouTrack进行交互。

有关更多信息，请参见实验功能 。
