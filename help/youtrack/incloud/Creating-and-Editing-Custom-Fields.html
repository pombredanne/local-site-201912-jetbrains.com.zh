创建和编辑自定义字段

作为管理员，您可以在YouTrack中创建，编辑和删除海关字段。您可以将自定义字段添加到系统并定义其属性，而无需将其附加到系统中的项目中。在“ 自定义字段设置”页面的“ 字段列表”选项卡上执行此页面上描述的操作。只有具有“ 低级管理员读取”权限的用户才能看到此选项卡。

具有项目管理员角色的用户可以创建自己的自定义字段，并将其附加到自己的项目中。可以在 “ 自定义字段设置”页面的“ 项目”选项卡中的字段或“ 编辑项目”页面的“ 字段”选项卡上执行这些操作。执行这些任务所需的最小权限集是“ 读取项目基本”和“ 更新项目” 。有关特定项目的说明，请参阅“ 管理自定义字段” 。

当您以管理员身份向系统添加字段时，可以将其附加到一个或多个项目，或者让项目管理员选择将该字段附加到他们自己的项目。

创建一个自定义字段

创建自定义字段的过程因字段类型而异。

具有简单数据类型的字段存储单个值。您分配给该字段的数据类型决定了如何在YouTrack中的其他位置使用这些值。例如，当您分配一个字段时， date数据类型，则这些值可用于生成显示过期问题的报告。

要使用简单类型创建自定义字段：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 单击添加新的自定义字段按钮。

    • 侧边栏中将打开“ 新建自定义字段”对话框。

  4. 在自定义字段设置中输入值。有关每个设置的详细说明，请参阅“ 自定义字段设置”。

  5. 单击添加新字段按钮。

    • 自定义字段已添加到YouTrack。

  6. 要设置高级选项，请在列表中选择新的自定义字段。工具栏中提供以下选项：

    • 启用自动附加
    • 设为私人
    • 从问题列表中隐藏

    有关更多信息，请参见全局默认设置 。

具有枚举类型的自定义字段（ enum[* | 1] ），例如State State和Priority还有更多选择。除了用于简单类型的设置之外，您还必须定义可以为该字段选择的可能值的列表。

要创建具有枚举类型的自定义字段，请执行以下操作：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 单击添加新的自定义字段按钮。

    • 侧边栏中将打开“ 新建自定义字段”对话框。

  4. 在自定义字段设置中输入值。有关每个设置的详细说明，请参阅“ 自定义字段设置”。

  5. 对于枚举类型，您可以使用以下选项：

    • 选择新设置选项。创建自定义字段后，定义一组值。

    • 从列表中选择一个字段。选择字段中使用的一组值将添加到新字段中。

  6. 单击添加新字段按钮。

    • 自定义字段已添加到YouTrack。

  7. 要设置高级选项，请在列表中选择新的自定义字段。工具栏中提供以下选项：

    • 启用自动附加
    • 设为私人
    • 从问题列表中隐藏

    有关更多信息，请参见全局默认设置 。

编辑自定义字段

您可以修改自定义字段的设置以控制其可见性，并将其自动附加到新项目中。

要编辑自定义字段：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 选择字段列表选项卡。

  3. 在列表中选择字段。

  4. 编辑侧栏中显示的设置。

  5. 切换工具栏中的高级选项。提供以下选项：

    • 启用自动附加/禁用自动附加
    • 设为私人/公开
    • 从问题列表中隐藏/在问题列表中显示

    有关更多信息，请参见全局默认设置 。

删除自定义字段

删除项目中使用的字段时，将从所有问题中删除该字段。该字段中存储的值将丢失。如果删除从未附加到项目的自定义字段，则将其删除而无需确认。

删除自定义字段：

需要权限：低级管理员写

  1. 在管理菜单的核心功能部分中，选择自定义字段 。

  2. 在列表中选择字段。

  3. 单击工具栏中的垃圾桶按钮。

  4. 单击确认对话框中的删除按钮。
