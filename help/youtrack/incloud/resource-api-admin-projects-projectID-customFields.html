项目自定义字段

此资源使您可以访问和配置特定项目中的自定义字段。

资源资源
/ api / admin / projects / {projectID} / customFields

退货实体

ProjectCustomField 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 阅读ProjectCustomFields列表 。

  • POST ： 添加一个新的ProjectCustomField 。

ProjectCustomField属性

代表特定项目的自定义字段设置。

下表描述了ProjectCustomField实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

领域

 CustomField

描述项目设置的自定义字段。 Read-only 。 Can be null 。

项目

 项目

自定义字段所属的项目。 Read-only 。 Can be null 。

canBeEmpty

布尔型

如果为true，则此字段中的值可以为空，否则为false。

emptyFieldText

串

字段为空时显示为占位符的文本。 Can be null 。

序数

整数

项目中此字段的编号。

isPublic

布尔型

如果为true，则发出公共读取/更新足以访问此字段，否则需要专用读取/更新。

hasRunningJob

布尔型

true ，如果此字段正在运行后台作业。在这种情况下，某些现场操作可能是暂时无法访问的。 Read-only 。

阅读ProjectCustomFields列表

阅读附加到特定项目的自定义字段的列表。

请求语法

GET / api / admin / projects / {projectID} / customFields？{字段}＆{$ top}＆{$ skip}

请求参数

参数

类型

描述

领域

串

应在响应中返回的ProjectCustomField属性列表。如果未指定任何字段，则仅entityID返回

$跳过

整数

可选的 。可让您设置要返回的多个实体，以在返回第一个实体之前跳过。

$ top

整数

可选的 。用于指定响应中返回的最大条目数。
