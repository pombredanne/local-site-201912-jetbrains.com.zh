YouTrack InCloud 2019.2帮助

REST API用例

本部分包含一组页面，描述了如何使用新的REST API在YouTrack中执行不同的操作。

请注意，此示例对YouTrack 2019.1和更高版本有效。在以前的版本中，尽管原理相同，但是查询参数和实体属性可能不同。此外，2019.1中引入了许多功能，例如按名称命名实体，但以前的版本中不支持。

  • 创建问题
  • 创建问题并设置自定义字段
  • 获取具有所有值的问题列表
  • 为问题自定义字段获取价值
  • 为集合增值
  • 更新问题自定义字段
  • 更新文本字段值
  • 自定义字段的明确值
  • 将文件附加到问题
