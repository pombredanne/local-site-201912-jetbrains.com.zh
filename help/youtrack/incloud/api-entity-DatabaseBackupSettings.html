数据库备份设置

代表YouTrack实例的数据库备份设置。

下表描述了DatabaseBackupSettings实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

地点

串

备份存储文件夹的位置。 Can be null 。

filesToKeep

整数

应该存储的备份文件数。

cronExpression

串

定义自动备份计划的cron表达式。 Can be null 。

archiveFormat

列举

当前选择的备份文件格式。支持的值为TAR_GZ或ZIP。

isOn

布尔型

如果为true，则启用自动常规备份。否则为“ false”。

availableDiskSpace

长

可用磁盘空间的大小。 Read-only 。

notifyedUsers

 用户群

收到有关备份失败的通知的用户列表。

backupStatus

 备份状态

备份的当前状态。 Read-only 。
