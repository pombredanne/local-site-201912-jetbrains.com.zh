YouTrack InCloud 2019.2 Help

Storage Limits

Each YouTrack InCloud instance has a set storage limit. For instances that were registered after November 8, 2018, or have been converted to the per-user pricing model, the instance is allocated 1 GB of storage for each user.

  • Images and attachments are stored on the server and count toward the storage limit.

  • Backup copies of your YouTrack database do not count toward the storage limit.

If you exceed the amount of storage that is allocated per user, you are billed for overuse charges. Each additional gigabyte of storage costs the same as a single license for an additional user.

Instances that do not use a commercial license are subject to the following storage limits:

  • For the free plan, the instance is restricted to 3 GB of storage space.

  • For the 14-day trial subscription, the instance is restricted to 10 GB of storage space.
