使用特定StateBundle的操作

该资源为操作提供了一组值state类型。

资源资源
/ api / admin / customFieldSettings / bundles / state / {bundleID}

退货实体

StateBundle 。有关实体属性的描述，请参见“ 支持的字段”部分。

支持的方法

  • GET ： 读取特定的StateBundle 。

  • POST ： 更新特定的StateBundle 。

  • DELETE ： 删除特定的StateBundle 。

StateBundle属性

代表YouTrack中的一组状态。

扩展BaseBundle

下表描述了StateBundle实体。

  • 要在服务器的响应中接收属性，请在request参数中明确指定它fields 。

  • 要更新属性，请在POST请求的正文中提供它。

领域

类型

描述

价值观

 StateBundleElements的数组

这组值包含的状态列表。

isUpdateable

布尔型

如果true ，则当前登录的用户可以更新此捆绑包。除此以外， false 。 Read-only 。

读取特定的StateBundle

阅读具有特定ID的集合中的状态列表。

所需权限

需要对使用捆绑软件的字段之一进行读取或更新访问。如果未在任何字段中使用捆绑软件，则至少在一个项目中需要“更新项目”权限。
