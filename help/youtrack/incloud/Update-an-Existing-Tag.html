更新现有标签

POST /用户/标签/ {标签}

更新用户标签

POST / rest / user / tag / {tag}？{newName}＆{visibleForGroup}＆{updatableByGroup}＆{untagOnResolve}

参数：

名称

类型

描述

标签名

串

要编辑的标签名称。

新名字

串

标签的新名称。

visibleForGroup

串

组名称，指定标签应显示在该组上。

可更新组

串

群组的名称，其成员应能够编辑指定的标签。

untagOnResolve

布尔值

为新标签启用或禁用“解决时取消标签”选项。如果启用，则当问题的状态更改为被视为“已解决”的任何状态（例如，已修复，重复，过时等）时，标签将自动从问题中删除。
