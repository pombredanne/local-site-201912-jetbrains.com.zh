项目概况

YouTrack中的每个项目都有自己的“ 项目概述”页面。要访问此页面，请单击标题中的“ 项目”链接，然后从列表中选择一个项目。

该页面的主要目的是为您和您的团队提供一种更方便的方式来查看与项目相关的信息。对于每个项目，它还充当着陆页或“总部”。

该页面显示项目描述，并提供对敏捷板和项目管理页面的快速访问。它还提供了类似于仪表板的界面，您可以在其中添加小部件，以监视项目中的活动。

项目名称和描述

在页面顶部，您将看到项目名称和描述。此信息来自“ 编辑项目”页面的“设置”选项卡。有关更多信息，请参见常规项目设置 。

在项目名称的右侧，您可以选择将项目添加到收藏夹列表中。这会将项目固定在项目列表的顶部。最喜欢的项目标有星号。

页面此部分的“ 新问题”链接可让您在当前项目中快速创建问题。

小部件

项目概述页面的底部包含一个类似于仪表板的界面，用于显示小部件。在这里，您可以安装小部件，使您可以监视项目中的活动。

项目概述页面支持仪表板上可用的所有小部件，包括YouTrack中已安装的自定义小部件。

要将新的小部件添加到项目概述中：

需要权限：更新项目

  1. 单击添加窗口小部件按钮。

  2. 选择要添加到仪表板的窗口小部件的类型。

  3. 选择要用作小部件数据源的服务。

  4. 配置窗口小部件设置。

  5. 单击创建窗口小部件按钮。

    • 该小部件将添加到项目概述页面。

您还可以将一组小部件从仪表板复制到项目概述。如果您已经设置了专用于项目的仪表板，请使用此选项快速配置项目概述页面。

请注意，用于在项目概述页面上显示小部件的网格不如仪表板上的网格宽。从仪表板导入窗口小部件之后，可能需要调整窗口小部件的大小并重新排列以适合显示。

要从仪表板导入小部件：

需要权限：更新项目

  1. 在页面的“ 小部件”部分中，单击“ 添加小部件”按钮。

  2. 选择从仪表板导入选项。

    • 从另一个仪表板对话框导入小部件打开。

  3. 选择要从中导入的仪表板的名称。仪表板列表仅限于您拥有或已与您所属的组共享的仪表板。

  4. 单击导入小部件按钮。

    • 所选仪表板中存储的所有小部件都将添加到项目概述页面。

如果您具有“ 低级管理”权限，则“ 添加小部件”菜单包含“ 获取更多小部件”选项。单击此链接访问“ 自定义窗口小部件”页面，并从JetBrains插件存储库中浏览其他窗口小部件。有关更多信息，请参见JetBrains插件存储库 。

项目侧栏

项目概述在页面右侧的侧栏中显示了链接的集合。此侧边栏始终可见。

侧栏分为以下几节：

部分

描述

项目徽标

项目徽标显示在侧栏的顶部。徽标取自“ 编辑项目”页面的“设置”选项卡。有关更多信息，请参见常规项目设置 。

敏捷板

本部分显示了敏捷板的列表，这些板显示了相关项目中的问题。该列表仅限于显示当前用户可见的板。

管理

此部分显示指向“ 编辑项目”页面的各个选项卡的链接。使用这些链接可以直接跳到项目设置中的相应选项卡。

这些链接对所有具有“ 读取项目”权限的用户可见，但是，只有具有“ 更新项目”权限的用户才能在链接的页面上编辑项目设置。

关于

此部分显示项目所有者的名称和创建项目的日期。
