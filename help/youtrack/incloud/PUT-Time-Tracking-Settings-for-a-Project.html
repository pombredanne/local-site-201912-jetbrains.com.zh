项目的PUT时间跟踪设置

PUT / rest / admin / project / {projectId} / timetracking

配置特定项目的时间跟踪设置。

PUT / rest / admin / project / {projectId} / timetracking

这是为指定项目中的时间跟踪配置设置的一组XML格式的设置。

样品

请求：

参数块的变化取决于时间跟踪的当前状态及其在项目中的配置：

  • 要在不配置字段的情况下在项目中启用时间跟踪：

    PUT https://example.com/youtrack/rest/admin/project/TST/timetracking连接：keep-alive Cookie：$ Version = 0; JSESSIONID = axezv08lvrvrzpvvih22ricx; $路径= /

  • 要在项目中禁用时间跟踪：

    PUT https://example.com/youtrack/rest/admin/project/TST/timetracking连接：keep-alive Cookie：$ Version = 0; JSESSIONID = axezv08lvrvrzpvvih22ricx; $路径= /

  • 要在项目中启用时间跟踪并配置时间跟踪字段：

    PUT https://example.com/youtrack/rest/admin/project/TST/timetracking连接：keep-alive Cookie：$ Version = 0; JSESSIONID = axezv08lvrvrzpvvih22ricx; $路径= /

  • 更改用于时间估计的字段：

    PUT https://example.com/youtrack/rest/admin/project/TST/timetracking连接：keep-alive Cookie：$ Version = 0; JSESSIONID = axezv08lvrvrzpvvih22ricx; $路径= /

响应：

在响应中，您将获得操作成功或失败的确认。
