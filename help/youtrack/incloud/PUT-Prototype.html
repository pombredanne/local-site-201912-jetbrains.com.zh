YouTrack InCloud 2019.2帮助

PUT原型

PUT / admin / customfield / field / {customFieldName}

创建新的自定义字段原型

放置/ rest / admin / customfield / field / {customFieldName}？{typeName}＆{isPrivate}＆{defaultVisibility}＆{autoAttached}

参数：

名称

类型

描述

customFieldName

串

要创建的自定义字段的名称。

typeName

串

新的自定义字段的类型。有关可能值的列表，请参见字段类型。

isPrivate

布尔值

设置新的自定义字段隐私。

defaultVisibility

布尔值

为新的自定义字段设置默认可见性。

自动附加

布尔值

自定义字段应自动附加到YouTrack中每个新创建的项目。

有关自定义字段设置的详细信息，请参阅“ 自定义字段设置” 。

样品

请求：

放置/ rest / admin / customfield / field / Region？typeName =枚举[*]＆isPrivate = false＆defaultVisibility = true＆autoAttached = false

响应：

栏位类型

的typeName参数接受预定义的值列表。这些值对应于YouTrack中的自定义字段支持的数据类型。

值

栏位类型

string

将单个值存储为字符串。

date

以日期格式存储单个值。

date and time

以日期格式存储单个值。完整的日期和时间显示在自定义字段中。

period

存储代表一段时间的值。

integer

将单个值存储为整数。

float

将单个值存储为浮点数。

enum[1]

存储预定义的一组值中的单个值。

enum[*]

存储一组预定义值中的多个值。

group[1]

存储对单个组的引用。

group[*]

存储对一个或多个组的引用。

user[1]

存储对单个用户帐户的引用。

user[*]

存储对一个或多个用户帐户的引用。

ownedField[1]

存储预定义的一组值中的单个值。值集中的每个值都存储一个可选的owner属性。

ownedField[*]

存储一组预定义值中的多个值。值集中的每个值都存储一个可选的owner属性。

state

存储预定义的问题状态集中的问题状态。仅支持单个值。

version[1]

存储预定义版本列表中的版本。

version[*]

存储预定义版本列表中的一个或多个版本。

build[1]

存储来自一组预定义的内部版本号的内部版本号。

build[*]

从一组预定义的内部版本号中存储一个或多个内部版本号。

有关每种自定义字段类型的详细说明，请参阅“ 自定义字段类型” 。
