创建一个敏捷板

您可以在YouTrack中创建一个敏捷板，并自定义其设置以支持不同的敏捷方法。无论您是使用scrum，看板，还是两者结合，董事会都足够灵活。

您可以与团队创建并共享一个董事会，也可以建立一个个人董事会来跟踪您自己的工作。

权限

在YouTrack中创建敏捷板实际所需的唯一权限是“ 读取问题” 。只要您有权阅读问题，就可以创建一个董事会并监视项目中问题的进度。例如，在项目中具有默认观察员角色的用户可以创建一个木板，并使用它来跟踪问题。但是，此角色具有一组有限的权限。他们与董事会问题互动的能力受到限制。

要创建和配置您和您的团队可用来更新项目中问题的电路板，您需要获得执行更多操作的权限。如果在执行此设置时遇到任何问题，请检查您是否有权访问以下权限：

创建董事会

此过程将引导您完成基本步骤。有关更多详细信息，请浏览本节中的其他主题。

要创建敏捷板：

  1. 单击标题中的“ 敏捷板”链接。

    • 发生两件事之一：

      • 如果您可以访问现有的板，则显示该板或上次查看的板。

      • 如果没有可用的板，则打开“ 创建敏捷板”页面。在这种情况下，请跳至步骤3。

  2. 单击创建按钮，然后选择板 。

  3. 选择一个模板：

    模板

    描述

    Scrum板

    为Scrum开发团队预先配置。自动设置以跟踪一系列计划的sprint上的功能开发。

    看板

    为遵循看板方法的团队预先配置。也是个人董事会的绝佳选择。

    基于版本的板

    预先配置为将sprint链接到存储字段的自定义字段中的值version类型。旨在模拟早期版本的YouTrack中敏捷板的行为，您可以在其中使用查询来过滤板上的卡片的可见性。

    定制板

    使用默认配置生成，您可以调整它以适应任何工作流程。

    个人董事会

    看板式看板，仅显示分配给当前用户的问题。

    有关每个模板的预配置设置列表，请参阅敏捷板模板 。

    • 将打开“ 新板”页面。

  4. 输入新板的名称。

  5. 在板上添加要管理的项目。

  6. 配置单板的可见性设置。选择其成员可以查看和使用董事会并编辑董事会设置的组。

    默认情况下， 可以查看和使用板以及可以编辑板设置设置为所有者 。这意味着新董事会是私有的。但是，如果使该板对其他用户可见，则该组中具有“ 低级管理”权限的用户也可以访问和编辑该板的设置。

  7. 为新板配置积压。您可以创建保存的搜索以用作待办事项，也可以使用现有的搜索。

    要创建新的保存的搜索，请执行以下操作：

    • 选择创建新按钮。

      • 保存的搜索被分配名称Backlog 。如果要为保存的搜索分配其他名称，请在“ 保存的搜索名称”字段中编辑值。

      • 使用以下格式生成对新保存的搜索的查询：

        项目： 具有：-{Board } #Unsolved

        此保存的搜索返回在董事会上管理的项目中未分配给董事会的所有未解决的问题。

        查询中的板名称被标记为不正确，因为尚未创建板本身。创建板后，将立即验证此参数。

    • 如果需要，请修改搜索查询。我们不建议您删除自动生成的查询中建议的任何参数。

    要将现有保存的搜索用于待办事项，请执行以下操作：

    • 选择使用现有选项。

    • 从下拉列表中选择一个保存的搜索。

      您还可以修改积压的已保存搜索，或选择另一个已保存的搜索作为板的积压。有关详细信息，请参见处理积压 。

  8. 单击创建板按钮。

    • 敏捷板创建完成。

    • 保存的搜索（如果是新搜索）将存储在YouTrack中。

    • 该评估板将打开“ 评估板设置”面板。在这里，您可以为主板配置其他设置。
