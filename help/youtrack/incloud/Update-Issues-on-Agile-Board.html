在敏捷板上更新问题

敏捷板使您可以直观地看到问题和任务从创建到完成的过程。随着问题在此过程中的进行，您和您的团队将应用反映您的操作的更新。

本页介绍如何更新敏捷板上的问题。您可以在“ 问题”列表或单个问题视图中应用与对问题相同的更改，但是这些控件的位置对于敏捷板是唯一的。

作为卡发行

您会注意到的第一件事是，问题被转换为交互式卡，您可以在板上随意移动。当您将卡移动到板上的其他位置时，表示新列或泳道的值将在标识板上这些位置的字段中更新。

卡本身上显示用于更新其他问题属性的控件。标题中的“ 大小”滑块会影响交互元素的详细程度和数量。下面的示例显示在大型卡上可见的控件。

您可以将以下操作作为板上的卡片应用于一个或多个问题：

行动

描述

在单个问题视图中打开问题

问题ID被设置为问题的链接。当您单击此链接时，问题将在新的浏览器选项卡中打开。

更新自定义字段

每张卡均显示项目中的前三个自定义字段，不包括板设置和“ 受让人”中使用的字段。要更新字段的值，请单击该字段，然后从列表中选择一个值。使用输入字段来过滤列表中的值。

移除标签

该卡显示的标签数量可以一行显示。要从问题中删除标签 ，请单击“ 删除标签”图标。

分配或重新分配问题

当前受让人的头像显示在每张卡的左下角。未分配的问题改为显示“ 设置受让人”按钮。要分配或重新分配问题，请单击此控件，然后从列表中选择一个用户。使用输入字段来过滤列表中的值。

查看和打开子任务

作为子任务链接到在板上显示为卡的问题的问题在中型和大型卡上可见。

  • 在M和L卡上，未解决的子任务与未解决的子任务的总数以比率显示。点击子任务图标查看子任务的列表。使用输入字段来过滤列表中的值。选择一个项目以查看模式打开板上的子任务。

  • 在XL卡上，每个子任务的问题ID和摘要均设置为链接。解决的子任务的链接被删除并变暗。单击链接以查看模式打开板上的子任务。

添加附件

要将一个或多个文件附加到问题上，请将文件拖到板上的卡上。卡本身成为放置区。

查看和更新估算

如果您的面板显示任务估计，则估计和进度指示器将显示在每张卡的右下角。将指针移到进度指示器上，以显示未用或过期的估计。单击估算值以更新值。

选定问题的选项

您可以在板上选择一个或多个卡。

  • 单击卡片或使用箭头键选择一个问题。当前问题标有蓝色边框。只要您不选择其他问题，更新就会应用于当前问题。

  • 要选择多个问题，请使用Spacebar和方向键或Ctrl +用鼠标单击。选定的问题以蓝色背景标记。当您选择多个问题时，当前问题（带有蓝色边框标记）将不包括在选择中，除非您使用Spacebar要么Ctrl +用鼠标单击。

选择一个或多个问题后，页面底部的工具栏将显示所选问题的数量。工具栏中提供以下选项：

选项

描述

删除选择

单击x图标以取消选择选定的卡。

设定受让人

打开在项目中设置为受让人的用户列表。选择一个用户来设置或更改所有选定卡的受让人。

添加标签

打开标签列表。选择一个标签，将标签添加到所有选定的卡中。

添加链接

打开问题链接类型列表。选择一个问题链接类型以打开“ 应用命令”对话框，并将链接类型添加到所有选定的卡中。

移至下一个冲刺

将所有选定的卡移动到下一个冲刺。

  • 如果尚未计划下一个冲刺，则此控件将打开“ 新建冲刺”对话框。创建冲刺时，选定的卡将添加到新的冲刺中。

  • 如果已计划下一个冲刺，则此控件将打开“ 应用命令”对话框。将所选问题移至下一个冲刺的命令会自动输入到输入字段中。应用命令将选定的卡片移动到下一个冲刺。

此控件旁边的下拉列表使您可以将选定的卡移动到特定的Sprint或将其返回到待办事项列表。

指令

当您选择板上的一张或多张卡时，可以使用命令来更新选定的问题。要输入命令，请按Ctrl + Alt + J ，或者只是开始输入。开始键入时，“ 应用命令”对话框将自动打开。

查看模式

当您双击卡或选择卡并按F3 ，问题将在查看模式下打开。此模式类似于单个问题视图。

在查看模式下，工具栏中的以下选项可用：

控制

描述

明星问题

将您自己添加到观察者列表。如果您已经在观看此问题，则将从列表中将其删除。

打开“应用命令”对话框

打开“ 应用命令”对话框。输入命令以一次更新多个问题属性。

添加链接

添加从当前问题到另一个问题的链接。从下拉列表中选择链接类型。

添加标签

在问题上添加标签。使用输入字段来过滤标签列表。从列表中选择一个项目以将标签添加到问题中。

分配问题

设置问题的受让人 。使用输入字段过滤受让人列表。从列表中选择要设置的用户或受让人。

展示更多

打开带有其他操作的菜单。

编辑问题

在编辑模式下打开问题。在这里，您可以更改问题的基本属性，例如Project ， Summary和Description 。

删除问题

删除问题。

可见到

设置问题的可见性。要限制可见性，请从下拉列表中选择一个组。

“ 显示更多”菜单提供以下选项：

选项

描述

克隆问题

创建问题的副本。分配给当前问题的自定义字段的值将分配给新问题。新问题将作为卡片添加到与原始问题相同的列和泳道中。

印刷版

在易于打印的页面上打开问题。在浏览器中使用“ 打印”命令来打印页面。

新增档案

打开一个对话框来查找和附加文件。

私下添加文件

打开一个对话框来附加文件并限制其可见性。单击以选择文件或将文件拖动到放置区域。从“可见到”下拉列表中选择一个组，以限制附件的可见性。

投票发行

将您的名字添加到选民列表中。如果您已经对该问题进行了投票，那么您的投票将被删除。您不能为自己的问题投票。

重复上一条命令

应用您在“ 应用命令”对话框中输入的最新命令。

在查看模式下，您具有用于将附件添加到问题的其他选项。对于没有附件的问题，将显示放置区域。

  • 单击放置区域以选择并附加文件。

  • 将文件拖到放置区。

  • 按Ctrl + V从剪贴板粘贴图像。

在新标签页中打开所有图像

新增档案

打开一个对话框来查找和附加文件。

其他选项可用于单个附件。将指针移到附件上，然后单击与该选项相对应的图标。

描述

下载

编辑

  • 如果附件是图像文件，则该文件在图像编辑器中打开。在这里，您可以编辑文件名并更改附件的可见性。其他工具可让您添加注释和修改图像。有关图像编辑器的更多信息，请参见“ 编辑图像” 。在您的Web浏览器中打开该问题附带的所有图像。

去掉

选择卡并按F2 ，问题将在编辑模式下打开。除了可以在查看模式下应用的更新之外，还可以通过编辑模式来更新问题摘要和描述 。

要退出查看模式或编辑模式，请按Esc或在问题弹出窗口之外单击。

作为泳道的问题

在将问题用作泳道的敏捷板上，泳道本身就是一个问题。添加到泳道的所有卡都作为子任务链接到代表泳道的问题。

在板上重新排列泳道时，代表每个泳道的问题不会以任何方式修改。一些团队使用该命令来确定应先解决哪些功能或用户存储，但该命令不会影响问题本身中“ 优先级”字段的值。

使用以下操作来更新代表板上泳道的问题：

行动

描述

在单个问题视图中打开问题

问题ID被设置为问题的链接。当您单击此链接时，问题将在新的浏览器选项卡中打开。

在查看模式下打开问题

单击编辑图标以查看模式打开问题。在这里，您可以将更新应用于代表泳道的问题，就像在查看模式下打开卡片时一样。

添加新问题作为泳道

面板底部工具栏中的“ 新建泳道”按钮将打开一个对话框，以显示新问题。输入摘要和说明，然后在对话框中单击“ 新建泳道”按钮以将此问题作为泳道添加到板中。

将问题移至待办事项

要从板上删除泳道并将问题返回到积压，请单击“ 积压”按钮以显示积压，然后将泳道拖动到积压。分配给泳道的所有卡也将返回到积压中。
