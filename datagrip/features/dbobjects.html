数据库对象

内省

数据库视图窗口包含所有数据源及其表，函数和视图等的模式，这些数据分为可折叠和展开的文件夹。

选择要反思的模式

单击m个文本区域中的n个，然后选择工作所需的模式。

对象过滤器

在数据源属性中，您可以定义是否应在数据库树视图中隐藏某些对象。例如，如果您不想看到带有某些前缀的表。

Ctrl + Q快捷键将为您提供有关语法的信息。在图片上，我们排除了所有名称以payment_开头的表。

组数据源

按类型对数据源进行分组，例如数据库供应商。按F6或从上下文菜单中选择“ 移至组... ”。

拖放还可以在数据库浏览器中创建组。要创建一个新组，只需将一个数据源拖到另一个上即可。要将数据源放入现有组，请将其拖放到该组中。

复制数据源

您经常要在另一个项目甚至另一个基于IntelliJ的IDE中使用相同的数据源。选择要共享的数据源，然后转到上下文菜单→“数据库工具”→“将数据源复制到剪贴板” 。

粘贴数据源

要粘贴数据源，请按+ ，然后按从剪贴板导入 。在剪贴板中，您只有XML，因此您甚至可以通过Messenger发送。

为所选对象生成SQL文件

使用SQL生成器（ Ctrl + Alt + G ），可以为数据库对象生成SQL文件。为此，请单击左窗格上的“ 保存”按钮。我们为输出创建了两个布局： 每个对象一个 文件和每个带order的 对象 文件 。

快速表备份

可以通过拖放将表复制到同一数据源。如果您需要在进行任何关键数据操作之前创建表的快速备份，这可能非常方便。

创建/修改表

IDE提供了用于添加或编辑表和表组件（列，索引和约束）的统一UI。该窗口包含定义所有这些组件所需的所有控件。请注意，如果启用了“ 在数据库中执行”，则所有更改将立即应用于数据库。

在数据库树或SQL脚本中的表名称上按Ctrl + F6进行修改。“ 创建/修改表”窗口还具有即时预览，以帮助您使用数据库对象并有效地生成代码。要了解有关创建和修改对象的更多信息，请参见代码生成。

要编辑表中的数据，请按F4键。请参阅数据编辑器 。

编辑源代码

通过按Ctrl + B调用任何数据库对象的“ 编辑”操作，并获取有关过程和函数的DDL语句，或者您可以在数据库浏览器中双击该对象。

装订线（文本编辑器左侧的竖线）将突出显示您所做的更改。在工具栏上单击“ 提交”后，迁移对话框将显示生成的SQL代码以更新源代码。

DataGrip将数据库对象的源代码保留在本地存储中 。这意味着您可以看到已更改的部分源代码，并且能够将所有更改恢复到服务器版本。

此外，源代码存储使您可以通过数据库的整个源代码搜索特定的代码段。

图表

借助DataGrip，您始终可以在有见地的图表中查看表及其关系，可通过按Ctrl + Alt + U或从要可视化的对象的上下文菜单中进行访问。具有主键和外键的列可以从图中隐藏，并且仅在关系线上显示。必要时可以将这些图导出到图像文件。
