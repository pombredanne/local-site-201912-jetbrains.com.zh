Security

User authentication

Upsource integrates with LDAP and many other authentication modules, including Google and GitHub.

User roles

Each user plays one or more roles in your company. The same is supported in Upsource: you can assign different roles to users in different projects. Use standard roles, customize them or create new ones.

Granular permissions

You have complete control over project access: divide users into groups, customize roles and permissions, and set per-project permissions.
