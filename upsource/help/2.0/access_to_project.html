提供对项目的访问

您可以将访问权限授予组或特定用户。

授予访问权限的权限：

  1. 点击设置图标在屏幕的右上方。

  2. “ 管理”视图将打开。

  3. 单击用户 。将显示“ 用户”视图 。

  4. 点击顶部的网上论坛 。将显示“ 组”视图表单。

  5. 从列表中选择组，然后单击访问选项卡。

  6. 单击授予访问权限，然后从显示的菜单中选择角色或权限以及一个项目。

授予用户访问权限：

  1. 点击设置图标在屏幕的右上方。

  2. “ 管理”视图将打开。

  3. 单击用户 。将显示“ 用户”视图 。

  4. 从列表中选择用户，然后单击访问选项卡。

  5. 单击授予访问权限，然后从显示的菜单中选择角色或权限以及一个项目。
