Quais verses do Scala voc costuma utilizar?

O JVM claramente o principal alvo de compilao do Scala, usada por 94%. O Scala.js o alvo de compilao para 8% dos desenvolvedores do Scala e o scala-native, para 7%.

Qual verso do Java voc costuma utilizar?

Mais de um quarto dos desenvolvedores que usam o Scala j usam o Java 11, a verso LTS mais recente.

O uso do Java 7 entre esses desenvolvedores caiu de 10% no ano passado para menos de 0,5%.

Quais dos seguintes frameworks de teste unitrio voc costuma utilizar?

O uso de macros do Whitebox aumentou at 10%. Ainda assim, quase 60% dos entrevistados no usam essas macros ou no as conhecem.

Quais dos seguintes frameworks/bibliotecas voc costuma utilizar para desenvolvimento web?

Qual dos seguintes frameworks/bibliotecas voc costuma utilizar?

A Scalaz e a Cats, duas bibliotecas concorrentes para programao funcional em Scala, esto lado a lado, com 12% e 13%, respectivamente.

Qual dos seguintes editores/IDEs voc utiliza com mais frequncia para o Scala?

71% dos desenvolvedores que usam o Scala trabalham no backend da Web para obter sustento, enquanto 42% so pagos para fazer anlises de dados.

Qual dos seguintes sistemas de compilao voc costuma utilizar?

Muitas pessoas migraram para o sbt 1.0, considerando que sua parcela subiu de 42% para 81% desde o ano passado.

Fato interessante: 58% dos desenvolvedores que usam o Scala participam regularmente de reunies locais.
