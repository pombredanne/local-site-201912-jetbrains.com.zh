简洁

用一行中的getters，setters， equals() ， hashCode() ， toString()和copy()创建一个POJO：

data class Customer(val name: String, val email: String, val company: String)

或使用lambda表达式过滤列表：

val positiveNumbers = list.filter { it > 0 }

要单身吗？创建一个对象：

object ThisIsASingleton {
    val companyName: String = "JetBrains"
}

安全

摆脱那些讨厌的NullPointerExceptions，您知道，十亿美元的错误

var output: String
output = null   // Compilation error

Kotlin保护您避免错误地对可为null的类型进行操作

val name: String? = null    // Nullable type
println(name.length())      // Compilation error

如果您检查类型正确，编译器将自动为您广播

fun calculateTotal(obj: Any) {
    if (obj is Invoice)
        obj.calculateTotal()
}

可互操作的

使用JVM上的任何现有库，因为它们具有100％的兼容性，包括SAM支持。

import io.reactivex.Flowable
import io.reactivex.schedulers.Schedulers

Flowable
    .fromCallable {
        Thread.sleep(1000) //  imitate expensive computation
        "Done"
    }
    .subscribeOn(Schedulers.io())
    .observeOn(Schedulers.single())
    .subscribe(::println, Throwable::printStackTrace)

定位JVM或JavaScript。用Kotlin编写代码，然后决定要部署到的位置

import kotlin.browser.window

fun onLoad() {
    window.document.body!!.innerHTML += "<br/>Hello, Kotlin!"
}
