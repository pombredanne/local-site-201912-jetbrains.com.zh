浮动许可证服务器

最新版本： 内部版本＃21137

浮动许可证服务器 （FLS）是基于Java的应用程序，用于管理公司网络内的JetBrains产品许可证。如果您拥有50个以上有效的产品订阅，则可用。

此限制不适用于教室许可证 。您可以将FLS与任意数量的教室许可证一起使用。

FLS支持所有JetBrains工具箱和插件订阅许可证，并且不支持团队工具 。

开始使用FLS：

  1. 联系销售团队为您的JetBrains帐户启用FLS。

  2. 检查要求并下载合适的FLS发行版。

  3. 安装和配置 FLS。

  4. 注册FLS 。

  5. 向FLS添加许可证 。

如果在本文档中找不到问题的答案，并且需要帮助，请联系JetBrains支持团队 。
