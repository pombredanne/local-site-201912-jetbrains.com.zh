PHP代码编辑器

智能代码编辑器

PhpStorm 为 PHP 提供了一个丰富的智能代码编辑器，后者会实际“获取”您的代码并深刻理解其结构，支持为现代项目和旧项目使用 PHP 5.3、5.4、5.5、5.6、7.0、7.1。 IDE 提供智能代码补全、语法高亮显示、扩展的代码格式化配置、实时错误检查、代码折叠，并支持语音混合等。

语言功能支持

支持最新的 PHP 7，包括支持匿名类、返回类型声明、null 合并运算符、组使用声明和可感知上下文的 lexer，等等。 PHP 7.1 兼容性检查会帮助您将项目更新为最新的PHP版本。 还支持 PHP 5.3、5.4、5.5、5.6 和 7.0。

PHP 重构

Rename和Move重构适用于文件、函数、常量、类、属性、方法、参数以及局部和全局变量。 轻松且安全地进行全局项目变更。 可以立即在原地执行本地变更。

以下重构也可用：Extract Variable / Constant / Field / Method / Interface、Inline variable、Push members Up / Pull members Down、Change signature、Move class to another namespace、Move static member 和 Make Static refactoring。

代码补全

自动代码补全（输入时调用）会在输入时完成对类、方法、变量名和 PHP 关键字以及字段和变量（取决于类型）的常用名的补全。 代码补全也支持 PHP 代码中的数组索引。

代码格式化

具有代码样式配置和其他功能的代码格式化程序可以帮助您编写易于维护的简洁代码。 PhpStorm 提供符合 PSR1/PSR2、Symfony2、Zend、Drupal 和其他标准的内置代码格式化。

代码（重新）排列器

代码（重新）排列器是一款强有力的工具，可以控制现有和生成代码的结构。 例如，您可以指定方法排列在字段后，或者使相关的方法不分离。

PHPDoc 支持

PhpStorm 编辑器遵守代码中的 PHPDoc，并根据注释提供合适的代码补全建议。 您在为代码编辑 PHPDoc 时，变量名和类型会从相应的代码块中自动补全。 PHP 代码重构也会考虑 PHPDocs 以将它们保持为最新状态。

参数提示

利用 PhpStorm 中的参数提示，可以使代码更易读且更易于理解。 为用作方法参数的文字和 null 显示参数提示，有助于您更好地理解特定方法的语义。

代码质量分析

成百上千个代码检查会在您输入时验证代码并检查整个项目是否存在可能的错误或代码异味。

适用于大多数检查的快速修复能够轻松地立即修复或改进代码。 Alt+Enter 可为每个检查显示合适的选项。

PHP Code Sniffer 和 Mess Detector

使用 PHP Code Sniffer（phpcs）和 PHP Mess Detector（phpmd）实时检查您的代码，只需打开 PHP 文件。 每次您在文件中更改内容就会触发检查，因此能够轻而易举地修复由 PHP Code Sniffer 和 PHP Mess Detector 报告的问题。

Twig、Blade 和 Smarty 模板

在处理模板文件时，使用 Twig、Blade 或 Smarty 模板引擎以充分利用高效功能：针对不同结构的可配置语法高亮显示；语法错误高亮显示；函数和特性补全；自动插入成对的大括号、引号和关闭标记，等等。

Phar 包

PhpStorm 可识别 Phar 包内的代码。

智能重复代码检测器

为所有主流支持语言（包括 PHP、HTML 和 CSS）新增。 使用 Code | Locate Duplicates 开始查找，无需复制/粘贴代码。 您将会看到重构的候选项列表 — 通过重构，您可以轻松地避免代码重复。

回调引用

代码编辑器可以理解字符串文字中的回调引用，并提供高级辅助功能，例如用例搜索和重命名重构。

命名空间

PhpStorm 提供命名空间智能处理功能，包括 Auto-Import（自动导入）、Optimize Imports（优化导入）和 folding for imports（折叠导入）等。

高效的项目分析和导航

彻底的源代码分析让 PhpStorm 能够为未注释的代码提供复杂的代码补全，例如，从代码正文扣除函数返回类型时以及从构造函数代码提取返回语句或类属性类型（与声明）时。

文件结构视图与类、方法和调用层次结构视图让您能够实现更快速的代码审查和导航。

数据流分析

数据流分析可以帮助您更好地理解项目代码，解释代码的复杂部分，以及发现源代码中的瓶颈等。

简单的代码导航和搜索

Go to Class/Go to File/Go to Symbol — 这些操作对于快速项目导航至关重要。 使用通配符和缩写词更快速地定位所需项。

Go to declaration — 导航到利用简单的热键或 Ctrl+点击/ ⌘ 点击声明的问题函数、变量或标签所在的位置。

Find Usages — 帮助您定位代码、当前文件或整个项目中任何符号（例如 类、方法和字段，等等）的用例。
