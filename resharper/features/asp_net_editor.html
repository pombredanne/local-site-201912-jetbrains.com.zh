ASP.NET，ASP.NET MVC和ASP.NET Core工具

ReSharper通过一整套代码分析，编码帮助，导航和代码生成功能，可以帮助您有效地使用ASP.NET和ASP.NET MVC项目（包括标记文件）。其中一些在Web应用程序中的工作方式与在C＃或VB.NET项目中完全相同。其他则针对Web开发的特定要求进行了微调。

代码检查和快速修复

无论何时使用ASP.NET标记和代码隐藏文件中使用的C＃代码，您都可以享受完整的ReSharper功能，包括整个代码检查和快速修复 。此外，还有特定于ASP.NET的代码检查和快速修复：例如，ReSharper在aspx页中检测未使用的导入名称空间，未知的符号和实体。

ASP.NET标记文件中也提供了标记栏和状态指示器 ，以帮助您在ReSharper发现的代码问题之间进行导航。

上下文动作

ReSharper提供了许多上下文操作来帮助您解决常见的ASP.NET标记方案。

例如，ASP.NET上下文操作使您可以替换，折叠或删除标签；转换HTML实体；创建事件，功能和属性；插入表格的列和行；添加代码隐藏文件。

导航和搜索

ReSharper的许多导航功能包都可以在ASP.NET中获得。

例如，如果要在当前打开的aspx文件中获得标记项的概述，请按Ctrl + Alt + F以显示“ 文件结构”工具窗口。您也可以使用文件结构派生工具，例如“ 转到文件成员” ，“ 转到下一个/上一个成员 ”和“转到包含声明”。

在ASP.NET项目中方便使用的其他导航操作包括“ 转到声明” （对从引用导航到用户控件和母版页特别有用），以及“从符号到导航”以从ContentPlaceHolder标记导航到其Content对应项，或从母版页引用到使用它的任何Web表单。

转到相关文件

转到相关文件 -Ctrl + Alt + F7-是特定于网络的导航操作，可将您从标记文件转到其引用的任何文件，包括代码隐藏文件，母版页，用户控件，图像，JavaScript和CSS文件，ASP.NET MVC视图和控制器

语法高亮

当使用C＃或VB.NET内联代码在aspx页面中呈现块时，ReSharper的语法突出显示可帮助您：

  • 在ASP.NET MVC调用中轻松发现动作和控制器名称。（阅读有关ASP.NET MVC的此功能和其他ReSharper功能的更多信息。）
  • 利用C＃和VB.NET代码的常规语法突出显示功能。

代码模板

ReSharper使ASP.NET和ASP.NET MVC开发人员能够使用一组20个捆绑的特定于Web的模板生成Web表单，标签和属性：

  • 用于ASP.NET的实时模板 Ctrl + E，L加快了创建新控件，脚本块，标签和属性的速度。
  • 环绕模板 -Ctrl + E，U-帮助包装带有标签，链接或foreach块的文本或代码。
  • 文件模板 -Ctrl + Alt + Ins-有助于创建新的Web表单，用户控件和母版页。

代码生成

ReSharper能够基于引用的母版页中定义的ContentPlaceHolder标签在标记页中生成Content标签。

另一个功能是在ASP.NET代码隐藏文件中生成事件订阅方法。

通过在编辑器中按Alt + Ins ，可以调用这些以及在当前上下文中可用的通用C＃和VB.NET代码生成功能 。

代码完成

代码完成 （包括智能完成 ）可与标签名称和属性值一起使用。 导入符号补全功能可帮助您引用未导入的用户控件而无需提前注册：ReSharper将自动生成Register指令。此外，ReSharper还为JavaScript符号提供了代码完成功能。

自动导入和自动注册

说到自动注册，ReSharper将在您使用完用户控件后立即自动注册您的用户控件，并在使用了属于它的类型后立即自动导入任何名称空间 ，就像在其他类型的项目中一样。

其他编码助手

ReSharper会在您输入开始标记后自动插入匹配的结束标记，或者在您输入属性值的开始引号之后自动插入一个结束引号。不仅如此，当您将插入符放在任意一个分隔符上时，此类匹配的分隔符会突出显示 。

当您具有深层嵌套的标签层次结构时，请依靠“ 扩展/收缩选择”将其从狭窄范围选择到更大范围，反之亦然。进行选择后，可以向上/向下（对于标签）或向左/右（对于属性） 移动它。

国际化

如果在aspx页面中有需要本地化的HTML标记，则可以选择它，然后按Alt + Enter ，然后将字符串移动到资源文件中。但是，这只是ReSharper 国际化能力的一小部分。

ASP.NET MVC支持

ReSharper提供了一组特定于ASP.NET MVC项目的功能。

除了上述功能和通用的C＃/ VB.NET支持以帮助您在控制器中编写代码之外，您还可以在动作和控制器之间导航，享受特殊的语法突出显示和代码完成功能 ，这些功能可以识别字符串文字中的动作引用， 根据使用情况创建动作和控制器，并在aspx和Razor视图引擎中执行更多操作。

ASP.NET Core支持

ReSharper理解ASP.NET Core项目中的项目引用和标记帮助器语法，并提供其核心功能集，包括重构 ， 代码检查和导航以及代码完成 。 在Visual Studio 2015、2017和2019中，.NET Core单元测试还提供单元测试帮助 。
