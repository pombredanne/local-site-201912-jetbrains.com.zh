编码协助

ReSharper提供了多种功能来获取有关代码的其他信息，选择并突出显示代码的结构块，以及应用本地代码转换。

语法高亮

ReSharper扩展了Visual Studio的默认突出显示支持，并以不同语言对代码项进行了自定义突出显示。例如，您可以轻松地区分代码中的局部变量和字段。要打开ReSharper的突出显示，请转到ReSharper |选项|代码检查|设置，然后选择颜色标识符 。每个项目的自定义颜色都是可配置的。

上下文动作

ReSharper的上下文操作通过快速在插入符号下对代码进行微小的更改而为您节省了很多输入时间-例如，您可以更改访问修饰符，生成检查null的代码，将foreach转换为for and foreach只需foreach几次键即可。如果当前插入符位置有一个或多个上下文动作可用，则ReSharper将显示锤子图标 ：您可以单击图标或按Alt + Enter选择适当的操作。

完整陈述

此功能将插入必要的语法元素（花括号，分号等），并使您处于开始下一条语句的位置，从而避免过多地插入插入符。在工作时，请记住此功能的默认快捷键： Ctrl + Shift + Enter 。

Complete Statement（也称为Smart Enter）可在许多情况下使用，包括自动关闭括号，添加分号， if ， while和for语句中完成等等。

语言注入

如果字符串文字（以及类似XML的语言中的标记或属性）包含某种其他形式的语言，例如正则表达式，HTML，CSS，JavaScript，JSON，则ReSharper可以提供代码检查 ， 快速修复 ， 代码摘录中包含了该语言的完成 ， 上下文操作和许多其他特定功能。

当无法自动检测到字符串文字中的形式语言时，ReSharper允许您使用Alt + Enter菜单或通过在字符串文字之前添加以下注释来手动将文字标记为包含特定语言： /*language=javascript|html|regexp|jsregexp|json|css*/

正则表达协助

ReSharper提供了丰富的工具集，可用于C＃，JavaScript和TypeScript中的正则表达式。您可以快速分析现有表达式，查找并修复错误。键入新表达式时，ReSharper会帮助自动完成和验证。

您可以选择ReSharper |工具|在菜单中验证正则表达式 ，以输入各种示例字符串，并查看正则表达式如何与这些字符串匹配。使用此对话框，您可以修正表达式并确保获得期望的匹配。

参数信息

ReSharper中还扩展了Visual Studio的IntelliSense的这一部分。调用方法时，参数信息会在工具提示中显示所有可用的方法签名和参数以及相关文档。键入时，工具提示会自动出现，也可以通过按Ctrl + Shift + Space来显示。在添加新参数时，ReSharper会将所有不兼容的签名涂灰。

突出显示匹配的分隔符

在C＃，JavaScript或TypeScript中，只要将尖号放在右括号/括号/花括号之前/之后，就会突出显示匹配的开头字符，反之亦然（此选项是可配置的）。

在VB.NET，匹配Sub或Function时插入符号内或之后的任何位置被突出显示End Sub或End Function 。每当将插入号放在开始关键字中时，匹配关键字及其行将突出显示（再次，您可以微调此功能）。

在XML中，匹配的标签突出显示，让您看到标签的结构。
如果屏幕上滚动出匹配的代码，则会出现一个适当的弹出窗口，以方便您使用。

导入名称空间

当由于忘记了在源代码中发出相应的名称空间导入指令而无法解析类型名称时，会出现一个小的弹出窗口，提示您要导入一个或多个类型。只需按Alt + Enter ，相应的名称空间import指令将自动插入。同样，您将不会失去当前的插入符号位置。

对于Web应用程序中的标记文件，ReSharper能够根据其用法自动导入用户控件（ ascx文件）。

自动插入定界符

在编辑器中键入( ， [ ， "或'时，会自动插入一个成对的字符（适当时）。可以轻松关闭此功能，但是您不必担心会误插入太多右括号，方括号或引号：如果右括号或引号已存在，则ReSharper不会添加第二个。
在XML中，只要键入一个开头的引号或括号，就会自动出现结尾的引号或括号。

复制行或选择

通过将插入号放在编辑器中的这一行（或分别选择一个块）并按Ctrl + D ，可以快速复制任何代码行（块）。

代码重新排序

ReSharper允许您通过按一个键快速地更改代码元素的顺序-四个键： Ctrl + Shift + Alt +上/下/左/右 。特别是，您可以上下移动成员，对参数和参数重新排序，在块内移动语句，在块内或外移动语句，上下移动XML标签以及左右属性XML，等等。 。

快速文档

要在编辑器中查看特定类，方法/函数或其他符号的文档，请将插入号放在代码符号上，然后按Ctrl + Shift + F1 。该文档显示在一个弹出窗口中，带有指向其他资源的可单击链接，其中最著名的是MSDN文档。

注释/取消注释代码

在编辑器中选择任何代码块，然后按Ctrl + Alt + / （用于行注释）或Ctrl + Shift + / （用于块注释），即可立即注释或取消注释任何代码块。

如果未选择任何阻止代码，也可以使用Ctrl + Alt + /快速注释或取消注释插入符号在编辑器中的代码行。在相同情况下，按Ctrl + Shift + /会在插入符号位置之前和之后立即插入开始块注释（ /* ）和关闭块注释（ */ ），如果插入标记位于注释的代码块内部，则将其删除。

扩展/缩小选择

扩展选择功能有助于依次选择扩展代码块。也就是说，您可以通过将插入号放置在代码中的某个位置并按Ctrl + Alt + Right几次来轻松选择代码中的任何表达式。收缩选择的工作方式相反，可以通过按Ctrl + Alt + Left来应用。

您还可以通过按Ctrl + Shift + [快速选择插入符号的成员，甚至包含其类型。

分割字符串文字

当您必须输入冗长的字符串文字时，ReSharper通过添加必要的引号和加号来帮助您将它们分成几行：当插入符号在字符串文字中时，按Enter或Shift + Enter即可拆分字符串而不会破坏其值。

从剪贴板历史记录中粘贴项目

ReSharper提供了一个特殊的粘贴命令，可通过按Ctrl + Shift + V进行访问，该命令可让您访问剪贴板历史记录并快速将从Visual Studio复制的任何最新项目粘贴到剪贴板。一旦启动Visual Studio，ReSharper就会开始将复制的项目记录到其剪贴板历史记录中，并在关闭时清除历史记录。
